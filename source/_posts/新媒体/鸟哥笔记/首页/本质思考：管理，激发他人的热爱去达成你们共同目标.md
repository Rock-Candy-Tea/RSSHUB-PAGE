                        <section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><a href="https://www.niaogebiji.com/tag/%E8%81%8C%E5%9C%BA" target="_blank">职场</a>，是自我进阶的过程。刚毕业大多数人都是从业务和技能切入，也是我们经常说到的“P专业线”，Profession。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">工作3~5年后，随着<a href="https://www.niaogebiji.com/tag/%E7%BB%84%E7%BB%87" target="_blank">组织</a>的扩张，有人会从专业线跳转到“M<a href="https://www.niaogebiji.com/tag/%E7%AE%A1%E7%90%86" target="_blank">管理</a>线”，Management。 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">专业各有差别，比如，设计能力、文案能力、技术能力，都在某一个领域垂直深入的结果，不同领域对应不同技能包。而管理，却有着共同的底层规律，哪怕行业不同、岗位不同，对应的管理底层逻辑都有相通之处。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">一直被人心心念叨的管理，究竟是什么呢？ </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">本篇，我将系统和你分享对应管理的见解，对于管理新手也许会有所启发。 </p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">管理，是激发他人的热爱去达成你们共同目标。老样子，先来拆解关键词。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">第一，激发。第二，他人。第三，热爱。第四，共同目标。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">激发，对应激励和发现，你要善于发现团队成员的优劣势，合理激励他们去发挥自己的优势；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">他人，管理是组织、有协作分工；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">热爱，和激励相似，激励更多的是外部刺激，比如：增加业绩奖励、年终奖励、薪酬待遇等。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">但热爱不一样，热爱是内部的喜欢和兴趣，如果能让他人找到做事的热爱，那么自发性、积极性、会自然溢出，而不是靠外部鞭策。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">共同目标，不是你的目标，也不是他人的目标，是你们共同想达成的目标，拥有共同目标的信念，管理推进会显得更轻松。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">来看关于管理的两个观点。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">观点一：西方管理学大师彼得·德鲁克曾说过，管理的本质，“最大限度地激发和释放他人的善意”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">观点二：管理=管（事）+ 理（人）。 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">管，是一套制度约束，对事不对人，就要求对事要按制度来办。理，是沟通和人情道理是文化，对人要讲情面，事无情，人有情。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">激发善意，是心理层面的因素，更像儒家文化以“仁”为中心。 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">管事理人，更像是法家和道的结合，法家，以“利”为中心，通过制度驱动人们对事儿控制、完成，以“道”为中心，万事万物均遵循规律，不只是冷冰冰的制度还有人情味儿的道理。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">一个优秀的管理者，一定是通过他人去达成团队的目标。技术大牛，设计大神、文案大师，可能在岗位领域里能大杀四方。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">但是，一旦让专业很强的大神去带团队，很多人却整得一团糟。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">为什么呢？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">因为专业能力强，和管理能力强是两件事，没有正向关联。所以，认识管理我们得从要素去思考， 达成你们共同的目标，是衡量管理的标准。 </p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>第一、专业能力做到80分。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">现实中组织不像《西游记》里的那么魔幻，如果一个领导者只有信念，没有核心技能，大概率很难让团队心服口服。新晋管理者爱犯的一个错误，通过权利职位去施压下属，其实是违背了管理的初衷。 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">迷恋于权威，很难激发人们的善意。有了过硬的专业能力，只为以身作则，积极主动表率。但是，管理者不要跟下属比技能。下属肯定比你强，如果不比你强，说明你没有找到对的人。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>第二、找对的人和做对的事。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">找到对的人 ，放在对的位置，用对的方法，去做对的事。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>什么是对的人和事？</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这就要求管理者能分析组织现状，并且能分析出具体的标准，而不是主观的臆断。是将军还是战士，是勇士还是谋士，不同人有不同的作用，好的管理者，会把对的人在对的时间安排到对的事情上。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>第三、制定计划和目标。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">计划和目标的制定，是每个管理者必备技能包。 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">目标，是你我达成的共识。，是团队不同角色的目标和企业共同目标的融合，去找到员工员工自觉地去做“我要干”的事情，而不是让员工做上面“要我干”的事情。 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">前者是自发的“目标”，后者是被迫的“指标”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>第四、翻译目标和结果。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">把目标翻译成任务，把结果翻译成实现结果的过程。管理者，在团队起承上启下作用，对于公司和项目目标清晰，如何把大目标分解到成员能推行的小目标，以及一个个可执行的任务尤为重要。 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">OKR的管理，就是目标拆解到任务的过程，一个是O，一个是KR，目标是一块，关键结果是一块。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">目标和任务，正如我们常说，“过程做好了，结果一定不会差”，做好哪些过程就是管理者要不断思考的问题。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">前面提到，专业能力和管理能力有区别。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">同样，管理能力和领导能力也有极大的区别，有人说管理力是守江山，微观的对组织的把控；领导力是打江山，宏观的对团队的鞭策和引导。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">有点抽象，举个例子大家很快就能明白。《亮剑》里的李云龙和赵刚，就是典型的领导力和管理力的代表，两者有很多交集，对于完成共同的目标始终不会动摇。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">李云龙，用个人魅力张扬挑战规矩赢人心；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">赵刚，忙于事务管理，克制，让制度和规范在团队里落实。无论张扬还是克制，他们共同的目标永远不会变，同样激发团队善意的动机也不会变。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">比如：李云龙替和尚报仇那段，让团队荷尔蒙爆棚，哪怕受上级处罚也要干你，只为向团队传递一个信号，跟着老子干踏实；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">赵刚，作为政委会分析战况利弊，让团队降低风险、更有纪律的行动避免踩大坑。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">领导者，强调结果，开山鼻祖；管理者，强调效率、控制和规范，优化流程从无序到有序的过程。</p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">一个人，如果自己都管理不好，根本谈不上管理他人、管理团队。最形象的的管理能力提升，是从个人能力的提升跃迁到公众、团队领取的成功。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">管理能力可视化他表现，正如《高效能人士的七个习惯》里，从第一个习惯到第七个习惯的进阶。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">自我管理阶段：积极主动+以终为始+要事第一。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">“积极主动”的做事意识，不管所处那个岗位、那个阶段，都能抱着积极的心态去做事，这是管理者必备的心态。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">“以终为始”的做事方法，做事之前先明确目标，并不断达成一个个目标的过程， 就是以终为始的践行，永不停止对合理化、正确目标的追求。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">“要事第一"的抓重点能力，要事第一是知道什么不该做，能分清轻重缓急，能时刻明确okr的o是什么，这点在管理过程中尤为重要，如果没抓住重要事项，越努力只会和目标偏离得越远。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">他人管理阶段：双赢思维+知己知彼+统合综效。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">“双赢思维”的做事视角，正如《第三选择》里，从你的想法，我的想法，变成你我共同的想法。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这点和第一部分提到“共同目标”观点吻合，通过视角不断切换，找到不同视角的交叉部分，达到共赢而不是互损的结局。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">“知己知彼”的动态观察，从组织来看，要时刻知道自己”有什么，要什么，舍什么”，知道什么路径更适合当下的组织；</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">个体来看，要知道每个人优劣势，让谁在什么位置能发挥最大的擅长，这是知己知彼的践行。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">同样，拿沟通来看，听者站在说话者的角度，去理解他们的思维方式和感受，也是知己知彼践行。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">“统合综效”是系统生态的体现。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>什么叫统合综效？</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">大自然就是统合综效最典型的结果，树灌木、草、微生物、动物在一起，能发挥最好的效能。一个组织，需要参天大树，也有要绿荫小草。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">人也一样，管理者经常拿来隐喻的《西游记》团队，各有利弊就是“综效统合”的案例。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">有善于激发团队善意的管理者唐僧，具有完美型的性格，时刻提醒团队长远目标；有善于解决问题同时爱犯错误的孙悟空，有缺乏责任心但是需要鞭策的猪八戒，有忠诚执行力强但是缺乏创造的沙僧。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">各有优劣势，统合综效最终达成取经的目标，就是组织和管理上的成功。试想一下，如果团队全部是孙悟空这样的角色能行吗？各个单兵作战能力很强，带来的结果却不一定。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">共同管理阶段：不断更新。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">激发善意，物质激励，共同目标…都对应这一个使命，不断更新。团队不断更新拿下结果，个人也不断成长变得强大的过程。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">优秀的人不是管理教唆出来的，是通过组织找到个人热爱，从内到外自我生长的过程。不断更新，使组织无边界变大，使个人终身成长在进阶。</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">最后的话：</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">管理，并不是自己单兵作战能力多厉害。管理，激发他人的热爱去达成你们共同目标。是通过他人，实现你我共同的愿望。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">纵观历史，优秀的管理者技能上往往不是技能最强，看看宋江、刘备、唐僧，再看看他们对组织的协调能力就有所启发。在职场，管理者学会识人达成共同的目标才是最重要。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">比如，员工就像“统合综效”的生态，一个组织各种角色都有。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">8小时在岗，以时间导向的员工；说一坚决不做二，任务导向的员工；按时交付，目标导向中层管理；价值观取得，由衷热爱愿景导向的合伙人。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">优秀管理者，会调动不同角色长处。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">让合适的人做好合适的事，达成你们共同的目标。是值得管理者毕生追求、精进和学习的方向。</p>
