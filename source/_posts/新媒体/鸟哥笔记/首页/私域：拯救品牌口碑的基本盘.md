<div style="height: 6px"></div>
                        <p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: inherit; text-align: inherit; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">品牌营销现在最怕的是什么？</span></p>
<div data-page-id="doxcnNHUIPqSva6DHmo6iZpcLDc" data-docx-has-block-data="true">
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">答：翻车。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">全民互联网时代，人们已经习惯经由网络获取信息、发表意见，输出自己的价值观和道德准则。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">当一个观点在人群中得到广泛认同，它就变成了普世价值。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">普世价值下，对于品牌营销，要避免一切“反普世价值”的选择，否则很容易在营销路上“翻车”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">比如张小泉一心想要营造高端路线，却忽略了市井烟火气，不论是有心还是无意，闹出了“国人几十年的切菜方法都是错的”事件。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在市场这条赛道上，品牌方你追我赶，最重要的不是加速能力，而是稳定与持久。所以，通过优质产品+私域口碑裂变，打造持续的传播能力，做好品牌口碑的基本盘建设，是现阶段品牌营销的必备选择。</p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;">一、私域口碑的终点是“饭圈化”？</h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">传统的广告营销逻辑是企业先大量投放广告，触达用户之后产生品牌效应。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">但在互联网营销时代，品牌营销的逻辑是先获取初期种子用户，通过优质产品和内容赢得口碑，让老用户带新用户，也就是“口碑裂变”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">能够主动帮品牌裂变的用户，一定是其核心用户群体。大量核心群体聚集后，又会形成圈层化。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这里包含着用户彼此之间的身份认同和价值认同。用户会发出“好巧，你也在用这个品牌？”的惊喜感，共同探讨产品使用的心得体会。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>圈层化意味着粉丝经济。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在私域概念还没有兴起之前，品牌的专属用户社区其实是具备私域属性的。比如智能手机行业的小米、魅族都有自己的社区，品牌可以在社区内触达到用户，和用户进行点对点的沟通和交流。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">虽然“饭圈化”在现阶段语境中逐渐变成一个贬义词。但对于品牌来说，这个词则意味着营销的终极目标，代表着坚如磐石的核心群体，而这部分群体是支持品牌的基本盘。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">比如同样是智能手机行业，曾经的“锤粉（锤子手机）”基于对创始人老罗个人魅力和产品理念的支持，愿意在产品遭遇负面舆论时坚持站在品牌方。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">私域的出现增强了品牌方触达用户的频次。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在私域场景中，无论是1v1的私聊还是1vN的社群，品牌和用户之间能够在沟通交流中完成“双向奔赴”，因为私域提供了便捷的技术手段，品牌通过优质服务和产品更容易抓住用户的心。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">所以，私域<a href="https://www.niaogebiji.com/tag/%E5%8F%A3%E7%A2%91%E8%90%A5%E9%94%80" target="_blank">口碑营销</a>的终极目标，是将粉丝饭圈化吗？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">如果品牌只是将某些群体用户“圈”起来，不去管对品牌潜在用户的挖掘，只关注目前的一部分群体，这种“饭圈化”是行不通的。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">真正的饭圈化，是能够准确定位品牌的细分客群，深度了解品牌的特性和用户群体，然后品牌私域中进行精细化运营，形成“潜客——意向用户——初次购买——二次购买——口碑裂变”的营销闭环。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这类营销方式既能够加强与品牌粉丝的情感共鸣，又能够提升品牌与用户之间的粘性，促进转化和口碑裂变。</p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;">二、私域中如何打造口碑营销体系</h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">那么，如何在私域中打造适合企业的口碑营销体系呢？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在《口碑的力量》这本书中，提到了口碑营销的五个“T”要素，分别是：Talkers（谈论者）、Topics（话题）、Tools（推动工具）、Taking Part（参与）、Tracking（跟踪了解）。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">虽然这本书出版时间较久，但关于口碑营销的底层逻辑仍然没有过时。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">接下来，我们在私域场景中来看这五“T”要素如何发挥价值。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">谈论者：找到口碑裂变的核心用户</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">口碑裂变的第一步，就是要找到能够积极分享的人，也就是KOC。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>寻找KOC有两种方式。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>第一种是从原有的公域平台中，将活跃KOC引流至私域。</strong>品牌方可以在美团和大众点评等平台，寻找评价活跃度高的用户，将其引导至企业微信。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">除此之外，还可以在知乎、小红书、抖音等平台，根据不同品牌属性，设置不同的引流链路。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">比如宝岛眼镜就会组织专业的验光团队，在知乎上通过付费咨询的方式，引导客户添加品牌企业微信，持续种草裂变。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>第二种方法就是在品牌自有的私域池中，寻找活跃用户培养成KOC。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">比如品牌可以通过群数据分析，筛选社群内部的活跃用户，引导回复，进行精细化运营。</p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">话题：引导讨论</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">《口碑的力量》中表示，一切口碑都始于出现一个将会传播开来的信息。这个信息并不一定非要是让人眼前一亮，深感震惊的。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">比如打折销售、优质服务、精美包装等信息就足以引起大家讨论。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这部分的重点就是私域中的内容设定。在尘锋系列栏目《跑赢私域》第六期打造高效的内容增长体系中，我们认为，私域中的内容需要达成这样几个目的：</p>
<ul>
<li data-list="bullet">
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">帮助客户了解【自己的需要】——“问题认知”</p>
</li>
</ul>
<ul>
<li data-list="bullet">
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">帮助客户了解【我的企业、品牌和产品】——“信息搜索”</p>
</li>
</ul>
<ul>
<li data-list="bullet">
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">帮助客户了解【为什么我是最好的选择】——“方案评估”</p>
</li>
</ul>
<ul>
<li data-list="bullet">
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">与客户建立【更健康、更优质】的连接关系——“信任、亲近”</p>
</li>
</ul>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">优质的内容一定是简洁、思路清晰，让人能够轻松理解和传播的。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">而一旦用户得到了足以激发口碑谈论者讨论的优质话题内容，下一个挑战就是如何通过工具，来推动裂变传播。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">推动工具：增加内容传播的速度和广度</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在私域中的基础工具三件套是“企业微信+小程序+视频号”，基于这三个工具，可以满足在微信生态内进行裂变传播的需求。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在微信生态外，品牌需要做的是通过私域口碑反哺公域的全域运营。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">全域运营的本质在于对不同渠道，不同平台的整合，在公域和私域之间构建桥梁。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">品牌可以引导粉丝将优质内容发布在微博、小红书等公域平台，将营销能量呈几何级别来放大。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">当口碑形成规模效应后，品牌方最重要的就是能够持续将关于品牌、产品的讨论始终处于新鲜状态，进行“常态化营销”，最好的做法就是保持实时同用户之间的私域沟通和交流。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>参与：融入用户，快速响应</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在私域中，高频次触达用户的另一面就代表着能够随时响应用户的需求。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">对于用户的专业性提问，比如医美、口腔、消费医疗行业，品牌能够做到及时响应。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这就要求，一方面品牌需要注重人才队伍的培养，另一方面，需要部署在社交化客户关系管理中能够沟通提效的工具，如企微SCRM。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">SCRM中的话术库、侧边栏营销素材等组件都能够帮助运营人员沟通提效。专业响应能够赢得用户的尊敬和赞扬，帮助品牌和用户建立长久牢固的关系。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>跟踪了解：了解用户喜好</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">最后一点，就是要能够在<a href="https://www.niaogebiji.com/tag/%E7%A7%81%E5%9F%9F%E8%BF%90%E8%90%A5" target="_blank">私域运营</a>中，及时了解用户的喜好。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">品牌应该要多关注用户之间的沟通和交流，掌握用户对产品、品牌的真实想法。这就要求品牌有用户行为洞察以及数据分析能力。</p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">以上五“T”，代表着私域口碑营销中的五大基本要点。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">当然，优质口碑的核心，一定是建立在优质产品+服务的基础上。如果产品和服务无法跟上用户需求，那无论品牌如何做口碑营销，也只不过是品牌独自的狂欢。</p>
