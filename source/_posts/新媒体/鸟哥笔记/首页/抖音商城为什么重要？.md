<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>增长是大家这两年最关心的话题。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>很多行业都有“金九银十”的说法，消费也不例外。前段时间的走访交流中，不少企业都提到在备战“<a href="https://www.niaogebiji.com/tag/%E6%8A%96%E9%9F%B3921" target="_blank">抖音921</a>好物节”，为秋季的首次增长做一把尝试。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一直都知道抖音电商是近年来不少企业重要的经营阵地，所以我也对大家提到的抖音921好物节，能承载的增长预期充满了好奇。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>抖音921好物节成交额破百万直播间达3950个，实现跃迁的潜力中小商家数达13万家，潜力中小商家在商城的动销店铺数提升21%，<a href="https://www.niaogebiji.com/tag/%E6%8A%96%E9%9F%B3%E5%95%86%E5%9F%8E" target="_blank">抖音商城</a>交易额破千万品牌84个，抖音商城交易额破百万单品达397个，</span></strong><span>这是抖音921好物节交出的首份答卷，多个数据都提到了抖音商城，可见抖音商城在“抖音921好物节”中占了重要位置，因此我对抖音商城做了一些观察，在此与大家分享，希望对你有启发。</span></p>

<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>在抖音921好物节做生意，</span></strong><strong><span>商城是关键场域</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>谈生意增长，必然要关注“货”的排兵布阵。今年5月的第二届抖音电商生态大会上，抖音电商正式宣布由“兴趣电商”升级为“全域兴趣电商”，通过完善“搜索”、“抖音商城”等中心场的建设，打通了货找人、人找货的双向链路，内容场、中心场、营销场，共同构成了现在的全域兴趣电商，帮助商家实现用户沉淀，更好地进行全链路运营。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以前，通过内容场的短视频、直播来实现“货找人”，短视频、直播这些内容，是用户潜在兴趣的激发转化场，用户在这里可以深度种草，高效成交。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>现在，可以通过商城、搜索、店铺橱窗等中心场来实现“人找货”。这是发生用户主动探索行为的场景，用以承接用户已有兴趣的转化，来到这里的用户很多自带购物需求。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这次“抖音921好物节”就围绕中心场推出了“<strong>抖音商城特色玩法</strong>”，包括跨店每满减、超值购和低价秒杀，让有不同的消费者都能找到适合自己需求的物品，而商家可在这些玩法中通过合适的货品组合及合理的运营，来提升品牌转化。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2127302664632c8b88b40002.86622139.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>例如在秒杀频道，商家可以提供高转化能力的福利商品进行拉新引流，是一个高流高转爆单售卖阵地。比如在超值购，商品可在各渠道打上专属标签，在用户消费场景获得更多曝光机会。以及跨店每满减，结合平台补贴，可以满足消费者“低价买到正品”的消费心理。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>抖音商城品牌众多、货品丰富，通过一些组合式策略，商家的货品更容易从“海量”的货品中有更多的展示机会，被更多人看到和搜索，以提升订单转化率。</span></strong></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>中心场何以成为重要转化阵地？</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>可以说，抖音商城成为了选购、转化与复购的关键场域，可以实现用户需求与产品的精准匹配，从而提升商品转化率，同时又承接用户的复购需求，沉淀通过商品成交留下的用户，此外又可以帮助商家针对不同客群落地精细化运营，为商家带来更多复购。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>周大福就是在“抖音921好物节”中表现出色的代表案例。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>周大福在“抖音921好物节”找到了“人找货”的新增量空间。数据显示，周大福抖音商城GMV超2000万，占品牌整体成交GMV 40%以上，品牌馆内累计销售全周期全品类排名TOP1。同时，周大福店铺新增会员环比上涨超600%，其中抖音商城成为品牌新客的重要来源，占比超过90%。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>周大福是如何实现如此亮眼的成绩的呢？具体来看，主要是成功运用了这样一套经营组合拳：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>首先，<strong>通过多梯队价格矩阵，增加货品展示机会</strong>。周大福本身拥有品牌及品类优势，因此在“抖音921好物节”针对抖音商城的不同频道推出了多梯队价格矩阵，为不同属性商品提报了不同的抖音商城特色活动，将有购买意愿的用户精准定向引流至抖音商城的品牌馆内；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其次，<strong>用户点击即进入品牌馆，周大福在品牌馆做好流量承接</strong>，首页聚合爆款产品，统一在品牌馆场景中呈现，让活动信息露出更充分，另一方面给用户更流畅、更一致的购物体验，提高曝光点击和销量；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最后，<strong>公域私域联动，从内容场向中心场导流</strong>：周大福在“抖音921好物节”通过爆款话题、达人矩阵营销等内容场运营策略，比如盘点周大福的爆款首饰、达人种草分享等，加入更加个性、潮流、现代的审美眼光，迅速推动用户关注，并引导进入抖音商城品牌馆，促成种草转化。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/772510790632c8b8906b748.28703446.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于在“抖音921好物节”中的表现，周大福营销总经理陈继娟表示，<strong>中心场能满足更多需求，实现「人找货」及「货找人」的循环链路，实现业务双倍增长。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>周大福对于抖音商城的积极布局，来源于对抖音商城价值的认可——<strong>撬动更多“人找货”，覆盖更多客群。</strong>也就是说，对于珠宝奢品这类商品，用户的决策周期较长，自播不一定能保证被种草的用户第一时间下单，这时，抖音商城这一稳定经营、操作简便的后置购买链路，就可以承接住这部分消费者的延迟购买需求，避免流量散失，对直播间等内容场的转化形成了有力补充。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如今，与周大福一样重视抖音商城建设并屡次实现生意爆发的品牌越来越多。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>他们在抖音618好物节、抖音818发现好物节、抖音921好物节，持续从抖音商城的高速增长中收获了生意增量。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今年6月，<strong>新锐乳业品牌“认养一头牛”借助抖音618好物节的势能，通过抖音商城的中心化玩法组合实现了增长引爆，累计销售额高居6月食品饮料行业TOP1。</strong>这样的成绩让认养一头牛对抖音商城的增量价值更有信心，这次“抖音921好物节”期间，认养一头牛通过积极参加商城秒杀、超值购、品牌馆等商城特色频道，以及达人及自营内容引流商城、店铺会员入会优惠券等组合玩法，成功实现入会率高达66%，成交上涨20%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1891614584632c8b8954c759.40736261.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>9月是家电行业秋季新品发布的重要时机，TCL联合抖音921好物节打造“抖音商城品牌馆TCL大牌日”，结合自身运营和销售模式，借力全域兴趣电商打开了新市场。<strong>TCL在策略上落地精细化消费者运营、打法上以全链路营销提升用户转化、玩法上将内容场和中心场打通实现有效联动、精准引流，在本次“抖音921好物节”成交订单数增加超35%。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/654199868632c8b89e33974.65586777.jpeg" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>好生意，正当时</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这些商家之所以重视中心场，是理解了抖音电商全域兴趣电商生态后的顺势而为。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>过去这一年来，各项数据均表明，用户在抖音电商上的主动购物心智和行为越来越明显，且在快速增长。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今年5月，抖音电商在第二届生态大会表示，抖音电商上“人找货”的行为数据增长得非常快，消费者的兴趣开始向中心场延伸。同比去年4月，抖音电商上有商品意图的搜索行为增长了217%，抖音商城GMV增长了6倍，仅2022年4月，就有64亿次商品意图的搜索，相当于每天2亿次。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>此次“抖音921好物节”，抖音商城交易额破百万单品达到了397个，抖音商城的价值越来越大，抖音商城已成为抖音电商中心化的流量阵地。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一个个平台级活动传来的捷报，是抖音电商与品牌商家们的互相成就，见证抖音电商飞速发展的同时，也验证着抖音商城的长效经营价值。</span><span>而一次次活动的成功落地后，平台的基础设施更加完善，服务经验更加丰富，与品牌和商家的合作路径更加清晰而多元。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不管你做不做抖音电商，今天的品牌，不论新老，都需要快速习得借助内容进行品牌个性化表达的能力。<strong>抖音电商兼具内容和电商属性，对于很多品牌来说，是更好的迭代自己新能力、开启长效经营之地。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因为今天的消费者就是这样来认识品牌、完成交易的。想要被今天的消费者看见和理解，要掌握和适应新时代的传播工具；想做今天消费者的生意，要经营好新时代的电商阵地。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>目前来看，抖音电商正是这样一个平台，而日活快速增长的抖音商城也仍是一块为商家带来生意增量的优质流量池，值得引起商家重视，而做抖音电商最好的时间，就是现在。</span></p>
