<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><a href="https://www.niaogebiji.com/tag/OCPC" target="_blank">OCPC</a>在百度搜索竞价中占据的地位越来越高，越来越多的人开始选择尝试Ocpc。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但竞价员对Ocpc的评价是严重的两极分化：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有的竞价员说Ocpc是爆量神器，线索来得又快又好，效果肉眼可见变好；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>也有竞价员认为Ocpc始终差了点意思，不好控制流量，出不来效果。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为了解决竞价员无从下手优化Ocpc的难题，我们今天准备了一篇堪称全面的Ocpc三大阶段优化的调整重点与思路，让你通过一篇文章即可学会如何应对Ocpc不同阶段的优化。</span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong><span>学习积累期</span></strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一般是建包之后的三到五天，处于系统刚介入、开始积累数据、帮助初步放量的时候，这个阶段的放量能力不会特别强，是属于试探阶段。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>这个阶段我们需要做什么呢？</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>管住手！</span></strong></span><span>不要随意修改出价、不要修改出价模式、账户物料不要做大范围修改、否词等操作需谨慎。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>注：管住手不是指什么操作都不要做，而是指尽量去做负向操作。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不仅需要管住手，学习积累阶段还有包括着建包前的操作：账户结构优化、否词清理、创意优化、样式补全、页面分析与调整。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在这个阶段的关注点在于展现量有没有持续的提高。如果有持续的提高是一个比较好的信号，反映了系统正在学习、流量正在拓展。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/10597572456307a14dae7ef2.05181171.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>注意，这里是没有不允许大家拓词的，当你的展现量上不去，也是可以少部分地去拓词，但是要注意拓词的频率、幅度不超过15%，比如投放包里一共有1000个词，今天拓词加了20000个词，这样的话就会导致比较乱，但加100个、50个词是没问题的。</span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong><span>成熟稳定期</span></strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>判断标准：学习期成功之后，每天有转化、而且成本也不高。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>这时候我们需要做什么呢？</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>定期做拓量操作，日常操作中正向操作比例80%，负向操作比例20%，这项工作是围绕着三点数据展开的：展现量、点击率、转化率。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>只有在我们有展现、消费快的前提下才能够做负向操作，而如果我们本身就没有展现，这时做负向操作就完蛋了！</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>注：正向操作可以简单理解为可以让平台多消费、多赚钱的操作；负向操作可以简单理解为是让平台少消费、少赚钱的操作。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>提升展现量：</span></strong></span><span>拓词。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当小数量展现突然下降时，可以检查账户操作，有没有否词、降价、暂停关键词、删除关键词、删除计划，如果都没有的话，大概率只是流量正常的波动。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>提升点击率：</span></strong></span><span>优化创意、高级样式。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>提升转化率：</span></strong></span><span>优化落地页（建议小范围优化，比如按钮、文案、图片，不建议大动作优化）、换转化不了的词换页面、调整时段和区域、页面增加视频。</span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong><span>衰退期</span></strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>判断标准：展现量连续3天左右呈下降、平均点击价格上升、点击率上升，出现这种情况时就是进入了衰退期。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那我们应该如何延缓衰退期呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>展现量：</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2-3天进行一次拓词，通过关键词规划师、5118输入种子词来进行拓词。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>种子词可以选择转化词、高意向词、下拉词、相关搜索词、人群词。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>虽然人群词和产品的关系不大，但这一人群可以用我们的产品，比如夏天如何快速瘦身（减肥药）、3万块钱做什么生意（招商加盟）。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>拓词之后还是没展现，怎么回事？</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果你发现这些词加在原计划中依旧没有什么展现的话，那么可以把新加的词捡出来，单独地新建一个计划，再把计划直接绑定到投放包里。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>人群词是低价跑吗？</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>人群词是后考虑的。如果通过转化词、高意向词、下拉词、相关搜索词，展现量能够恢复、上升，那我们暂时先人群词放一放，如果加了这些词展现量还是在持续地下降，那么可以专门来一个人群词计划，对它们进行分单元、分词，然后把这个计划直接加到我的投放包中。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>先低价跑cpc，有消费、有展现之后，感觉流量也差不多的时候，稍微收一收，再绑定在投放包。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/18880048246307a14de4c501.76517592.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>建议大家即使能够通过转化词、高意向词、下拉词、相关搜索词让展现量恢复、上升，也可以同时“养”一个人群词，让人群词低价跑cpc，当转化词、高意向词等词呈现了下降趋势，而人群词也稳定了，就可以直接把人群词绑定到投放包中。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>关于人群词还是需要给大家嘱咐一点，如果你的产品不够知名、甚至用户想不到要使用产品，这样的情况下，可以多使用一些人群词。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作为竞价员，我们考虑的角度可能是用户怎么搜索产品，但是我们需要跳出这个思维圈，去思考如果用户在不知道有这个产品的情况下会搜索什么产品、怎么去搜。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>可能有人就要说了“感觉我的行业没有人群词”，任何行业都有人群词，只需要考虑用户为什么购买你的产品。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如“二手家具”，不要局限于二手，还可以跑一跑“便宜家具”“家具批发”，向用户展示二手家具的性价比远超新家具。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这些人群词的转化率肯定不如“二手家具”高，但在缺流量的情况下就可以这么做的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>5天删一次否定关键词，根据我们前面的“中和”原则，我们就需要一边删一边加。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>点击率：</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>暂停最近两周点击率特别低的创意，新建创意。特别低的创意是指展现量特别多，但点击率为0或1的情况等拉低了账户点击率的创意。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过一次性也不要暂停太多，可以一个单元一个单元做，优先做有转化率的、展现多的，一天优化两三个单元基本上就可以了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>没事的时候可以参考同行创意样式，做到创意识别、新建高级样式。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>展现量和点击率是重点延缓衰退的操作。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>价格：</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>价格不是不让动，而是不让下降价格。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当我们的价格合适的时候，就可以动一下价格。但不是看到价格稳定了，真实出价低于预期出价后，开始降低价格。比如出价总是在90左右，看到价格基本稳定后，将价格由原来的120降到100。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这时候一旦降价，展现就容易崩。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而是要提价格，提升3-5%的价格，比如110的成本跑了2周，成本也很合适，那我们现在就可以适当地提升一下价格，改到115，而不能改到98或95。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/8189265416307a14e3a1a53.53411977.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这样做的目的是为了让系统看到一个积极的反馈，让它保持展现的不减少。除了提价格外，我们还可以给计划层级、账户层级加预算，给系统一个积极的信号。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>可以算是给系统一个保护费：</span><strong><span>“我和你合作很愉快，希望你接下来也能多来点流量，价格高一点无所谓。”</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以上就是Ocpc三个阶段优化的调整重点与思路的全部内容，你明白了吗？</span></p>
