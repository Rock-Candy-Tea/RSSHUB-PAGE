<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从今年6月各大直播带货的头部主播偃旗息鼓之后，东方甄选就成为行业的焦点。并且凭借颇具特色的带货模式和出色的产品质量，东方甄选顺利的挤下了交个朋友成为是抖音的直播带货“一哥”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但就在俞敏洪和罗永浩擦肩而过两个月之后，围绕抖音直播带货这块业务，两位新东方的老同事再次做出了不同的选择。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>近日，有网友发现东方甄选低调的推出了独立的电商App，并已经在各大应用平台上线，还有了接近20万的下载量。难道新上位2个月的带货一哥似乎打算单干了？</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>或许还是巧合，在相近的时间里，罗永浩则再次回到了抖音直播间，用一场答题式的综艺直播，来为交个朋友挽回不断下跌的人气。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>说起俞敏洪和东方甄选，他故事和很多再就业的企业家类似。因为教育行业“双减”，俞敏洪不得不选择时下最火爆的，也是变现最快的直播行业。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在将旗下教育业务收缩之后，俞敏洪挑选了新东方中最有天赋档主播的一群老师，开启了自己的直播带货之路。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/769887215630769059fdb25.05376827.jpeg" referrerpolicy="no-referrer"></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>起初的东方甄选以销售精选农产品为主，并且主打不收坑位费，只赚取佣金的合作模式。俞敏洪也多次出现在直播间里，为新项目站台吆喝。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但是在当时直播带货巨头云集的抖音，背靠俞敏洪的个人IP和新东方的品牌势能，也未能让东方甄选一炮而红，而是长期处于不温不火的状态。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>直到今年6月，各家头部主播都离奇隐退时，“东方甄选”直播间突然凭借董宇辉的“双语带货”视频成功破圈，此后，东方甄选直播间迎来爆红。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>6月16日，东方甄选粉丝量突破千万大关；6月29日，东方甄选直播间粉丝突破2000万。目前，东方甄选直播间粉丝已达2400多万。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据新抖截至7月底的数据显示，东方甄选场均销售额达到1968.82万元，近30日直播销售额超过6亿元，多次冲上<a href="https://www.niaogebiji.com/tag/%E6%8A%96%E9%9F%B3%E5%B8%A6%E8%B4%A7" target="_blank">抖音带货</a>榜第一名的位置。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>与此同时，围绕东方甄选的各种话题在这2个月中层出不穷，其中新东方“召回”老师做主播引的新闻更是在网友中刷了一波好评。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然，新东方最终能有多少老师成为带货主播我们不得而知，俞敏洪对东方甄选拒绝all in 抖音却是显而易见的。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>东方甄选从项目上线伊始，就已经在其它平台开设了自己的账号，其视频内容在视频号、快手、小红书、微博等流量平台进行了同步分发。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>只是在其他平台上的账号，东方甄选均没有开设直播业务。新上线的东方甄选App也是除了抖音之外，唯一具有电商功能的平台。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>和俞敏洪的东方甄选不同，抖音是罗永浩置之死地而后生的地方。因此老罗一开始就选择了all in抖音,如今交个朋友日渐走低，老罗也不得不行动起来，再次回归站台。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在8月23日晚的一场直播中，老罗以答题节目主持人的身份，回到了交个朋友直播间，用一场答题式的综艺直播，尝试帮助交个朋友拉高人气。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/1751980910630769062486a5.96164132.jpeg" referrerpolicy="no-referrer"></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>直播当晚，老罗兴冲冲地向观众宣布：今晚的交个朋友直播间，将进行一场有奖知识竞答活动，主持人是自己，而参赛的三位选手，则是朱萧木、文君和王拓。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一个带货的直播间，突然玩起了综艺答题。不用多想就知道，这是老罗在用直播“整活儿”的方式提升直播间的留存，把自己定位为主持人，也可以既不过多干涉其他主播带货，又能用自己引流。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那为什么已经宣布退网的老罗，要回归站台呢？原因也很简单，冲着老罗名气来的朋友，在老罗走后，朋友们也走了。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>交个朋友直播间，在老罗退网的2个月内，面临下滑压力：过去60天，该直播间掉粉4.8万。而俞敏洪的东方甄选，最近60天抖音粉丝增加了518.8万，目前为2467.2万。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/61596927063076906806257.23601683.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除了粉丝数量被东方甄选超越之外，在直播带货的销售额方面，交个朋友在老罗告别后，也已落后于东方甄选。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最近60天，交个朋友直播间，新增直播175场，累计直播销售额6.5亿元；东方甄选直播间，新增直播62场，直播销售额13.1亿元，在直播销售额方面，东方甄选已达到交个朋友的两倍。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>好在布局2年的交个朋友矩阵号，还能出来一战。交个朋友生活家居、交个朋友运动户外、交个朋友酒水食品三个直播间，最近30天直播销售额，分别为2598.7万元、3065万元、2147.3万元。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而东方甄选之图书直播间，最近30天直播销售额1455.2万元。虽然粉丝数量领先交个朋友的矩阵号，但其直播销售额，却落后于交个朋友次级直播间。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>事实上，不管是俞敏洪的东方甄选持续火爆，还是罗永浩的交个朋友热度不再。</span><span>抖音上面谁当一哥，</span><span>一切</span><span>都在抖音的掌握之中</span><span>。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>归根因素是抖音以公域推荐为主，粉丝数多少并不等价决定自己掌握的流量。据了解，“交个朋友”每月购买的抖音信息流费用也是在上千万，只要计算ROI合适，这门生意就可行。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而东方甄选选择多点发展，早在他的个人微信公众号“老俞闲话”中就说过，“基于外部的平台所建立起来的热闹的商业模式，有很强的脆弱性”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>言外之意，俞敏洪及东方甄选已经意识到，过度依赖于抖音平台的流量获取以及交易场景，导致东方甄选几乎所有的交易，都沉淀在抖音电商体系内。东方甄选要想获得长远发展，不能仅仅依靠第三方平台的流量，而是要形成自己的用户群体。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但话又说回来，在独立电商一片哀嚎的当下，东方甄选APP能不能生存下来还是一个问题。眼下，东方甄选乃至俞敏洪、董宇辉都有很多粉丝，凭借他们或许可以在短时间内为自家APP带来一大波流量，但粉丝能不能留下来，则是长远的问题。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>至于老罗回归交个朋友能起到多大作用，其实那场场观176.2万，GMV235.7万DE综艺直播就已经说明了，在超级主播的后时代，老罗只能雪中送炭，却无法锦上添花。</span></p>
