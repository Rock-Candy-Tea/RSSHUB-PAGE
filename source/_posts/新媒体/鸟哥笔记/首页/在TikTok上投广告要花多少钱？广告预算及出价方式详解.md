                        <p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">除了TikTok账号运营、直播卖货，跨境出海企业想利用TikTok推广快速变现，绕不开投放广告这个法子。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">TikTok for Business官方广告代理商YinoLink易诺今天来给卖家们介绍一下TikTok推广时会涉及的广告预算及出价方式选择。</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">一、TikTok for Business广告预算</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">预算是广告主愿意花在投放广告上的金额，需要在<a href="https://www.niaogebiji.com/tag/TikTok%E5%B9%BF%E5%91%8A" target="_blank">TikTok广告</a>推广系列层级设置预算，可以为推广系列和广告组设置预算。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><!-- [if !supportLists]--><strong>1. <!--[endif]-->总预算（Lifetime Budget） </strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">总预算是广告主在投放TikTok广告的整个过程中愿意在推广系列或广告组上花费的最大金额。 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在为广告组设置总预算时，TikTok广告投放系统会尽量在设置的时间范围内分配预算支出，但并不能保证预算会被平均分配至每天。 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><!-- [if !supportLists]--><strong>2. <!--[endif]-->日预算（Daily Budget） </strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">日预算是广告主每天愿意在TikTok广告的推广系列或广告组上花费的上限金额。总预算和日预算设置为上限，成本永远不会超过您设定的预算，并且在投放期间实际花费的金额可能小于设定的预算金额。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">需要注意的是，如果在TikTok广告推广系列层级选择“日预算”，则必须在广告组级别选择“日预算”。如果为推广系列选择“总预算”，则可以在广告组级别选择“日预算”或“总预算”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">YinoLink易诺建议为广告组预算使用日预算，而不是总预算。TikTok广告发布后，广告主无法更改预算类型，将无法从日预算切换到总预算，反之亦然。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><!-- [if !supportLists]--><strong>3. <!--[endif]-->最低预算 </strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">无论广告主设置的是日预算还是总预算，都需要有一个最低预算以确保TikTok广告持续的投放。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">TikTok广告推广系列的日预算和总预算都必须大于50美元。广告组的日预算必须大于20美元。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">总预算将按以下公式计算：最低日预算（20美元）×计划天数</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">例如：如果投放的排期为31天，则广告组的最低预算为20美元×31天=620美元。</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">二、TikTok for Business广告出价方式</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">目前，TikTok for Business广告管理平台提供了四种出价方式，也决定了<a href="https://www.niaogebiji.com/tag/%E5%B9%BF%E5%91%8A%E8%B4%B9" target="_blank">广告费</a>的结算方式。在广告组层级，需要选择您的出价方式，对于出价方式的选择取决于投放TikTok广告的策略和目标。 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><!-- [if !supportLists]--><strong>1. <!--[endif]-->每千次展示费用（CPM）</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">每千次展示费用为广告主愿意为1000次展示支付的价格，TikTok广告系统将投放您的广告，在预算范围内获得最大曝光并触达尽可能多的客户。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">CPM=（总费用/总展示次数）*1,000</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">选择CPM作为出价方式将按照展示次数计费，适用于以“覆盖人数”为推广目标的广告。 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><!-- [if !supportLists]--><strong>2. <!--[endif]-->优化每千次展示费用（oCPM）</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">优化每千次展示费用是广告主愿意为获得1000次展示支付的价格，定向于可能会完成转化事件、成为线索或安装应用的用户。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">oCPM的计算方式、计费方式和CPM一致。相比CPM，oCPM是广告系统经过预估点击率和转化率的分析后，向最有可能发生转化的用户展示广告以提高成效的出价方式。 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">oCPM适用于转化量、应用安装量、线索收集的TikTok推广目标。需要注意的是，目前oCPM是转化量和应用安装量推广目标的默认出价方式。 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><!-- [if !supportLists]--><strong>3. <!--[endif]-->每千次播放费用（CPV）</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">每千次播放费用是广告主愿意为1000次时长为6秒或2秒的视频播放所支付的价格。TikTok广告系统将投放您的视频，使其在您的预算范围内达到尽可能多的播放次数。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">CPV=（总费用/总展示次数）*1,000</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">CPV适用于以“视频播放量”为推广目标的广告。在您的广告组中设置每千次播放费用的出价时，请确保将您所需的每次观看费用乘以1000，并输入总出价作为出价金额。例如，如果您愿意为一次视频播放花费0.05美元，则输入50美元作为CPV出价。 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><!-- [if !supportLists]--><strong>4. <!--[endif]-->单次点击费用（CPC） </strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">单次点击费用是广告主愿意为每次点击支付的费用，TikTok广告系统将尽可能以接近您出价的费用投放广告给最有可能点击它们的用户。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">CPC=总费用/总点击量</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">选择CPC作为出价方式将按照点击计费，适用于以“访问量、应用安装量、转化量”为推广目标的广告。 </p>
