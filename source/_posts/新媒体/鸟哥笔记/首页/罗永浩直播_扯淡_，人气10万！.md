<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“退网”的<a href="https://www.niaogebiji.com/tag/%E7%BD%97%E6%B0%B8%E6%B5%A9" target="_blank">罗永浩</a>又回来了！</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>7月10日晚，宣布退出所有社交平台专心创业的罗永浩再次出现在交个朋友直播间。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然，这倒不是罗老师“食言”，而是当初“退网”时就曾预告还有多场直播合约没有履行完。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>罗永浩此番“复出”，一是为了宣传自己的新公司 Thin Red Line（细红线）；二是为了硬刚直播界只讲内容不讲产品的“歪风”，做一场直播内容实验。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/381440126233b25ace1df6.64458916.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>罗永浩的直播“实验”</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>相较于以往马不停蹄的讲解产品，偶尔金句迸发彰显罗氏脱口秀本色，罗永浩7月10号当天的直播稍显“特别”，竟然学起了东方甄选的侃大山式带货——</span><span><strong>只讲内容，不讲产品。</strong></span></p>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/80194853462ce0dc4c4ec11.68502650.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>以至于前面2个小时的直播，罗永浩只讲了3个产品</span></strong></span><span>，相比东方甄选有过之而无不及。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过不同于“董宇辉式”的诗词歌赋、人生哲理与人情世故，老罗这场以</span><span><strong>“今天聊天为主”</strong></span><span>的直播局更像是一场“时事评论”，从新闻热点到科技行业再到直播行业，涉猎极其广泛。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以至于一场直播下来，话题与争议不断，频频登上微博热搜。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/29152729162ce0dc5031c26.04237195.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>直播中，罗永浩谈近期成为“全民公敌”的钟薛高。表示钟薛高按质量算价格没有哈根达斯贵；价格比哈根达斯便宜的情况下，钟薛高的品质一定是碾压式的好过哈根达斯。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>还说网友拿火烤钟薛高是精神有问题。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>由于公然“站队”钟薛高，以至于不少激进的网友当即宣布脱粉。但向来特立独行的罗永浩显然并不在意这些。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>谈AR创业，老罗在直播中直言不看好苹果做 AR 眼镜。因为平台革命的时候，新世界的霸主永远不是之前旧世界的霸主。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>还谈到收购苹果的路径：等到他们日渐衰落、我们日渐崛起，到了一个形势逆转的转折点，这时候才考虑收购。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>让不少人大呼，那个理想主义者又回来了！</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>由于整场直播大量的内容干货以及关于他新公司的思考和最新进展，再加上经典的罗氏幽默，让不少喜欢罗永浩的网友闻讯赶来支持，迅速拉升了交个朋友直播间的人气，</span><span><strong>实时在线人数7万+，最高时一度逼近10万。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/213935499662ce0dc557b284.18710748.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而这一数据在罗永浩此前的直播中，大多也就维持在2、3万左右。因此，从场观数据来看，老罗的直播实验无疑是成功的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/9225241376233b25c6ec6a4.50995726.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>直播界的“歪风”</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>显然，罗永浩的这场直播实验正是针对当下直播界流行的所谓“歪风”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/205225319662ce0dc597d758.87663174.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在此前7月4号的一场直播中，罗永浩曾直言：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><em><span>现在我们直播界刮起了一股歪风——做直播要讲内容，我认为不对，还是要讲产品。内容会把场观拉上去，但是对直接销售的帮助还是有限的。</span></em><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>由于彼时东方甄选的董宇辉正因为“内容”直播带货而大受欢迎，因此老罗当时的这番话被外界解读为对东方甄选的批评，还登上微博热搜，被不少网友嘲讽为“嫉妒”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以至于罗永浩不得不用此前刚注册的辟谣号澄清“误会”：</span><span><strong>对董宇辉表达敬意；不是针对东方甄选，而是针对行业现象。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/143852616562ce0dc5ed70a1.50339174.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但其实罗永浩确实不是批评东方甄选，因为这并非罗永浩第一次发表这样的观点。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>早在2年前巨量引擎大师课上老罗就曾说过，无论直播风格如何，实际上还是要为卖货服务，直播带货的内核是严肃认真的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>众所周知，自董宇辉爆火之后，关于直播带货的主流观点也发生了一些变化，认为直播带货要讲内容，用内容去带产品。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/69401455062ce0dc68185a5.81480482.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但罗永浩对此的观点却是，靠与带货产品无关的内容吸引来的都是无效流量，对带货本身的价值不大。更直言，在直播间扯淡15分钟，“场观”也许很高，但总收入可能不如3分钟专心讲产品。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>罗永浩的此次直播实验正是为了验证自己的观点。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据笔者通过蝉妈妈小程序查询发现，7月10日，罗永浩的那场直播试验带货成绩是1394万，而此前7月8日、9日，在没有罗永浩参与的情况下带货成绩分别是858万、869万。</span><span><strong>数据差距并不明显。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/123878551162ce0dc6c00fb3.42910840.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而反观东方甄选平均10万+场观，7月8日—10日三天带货成绩维持在1700万—2700万，不但不稳定，转化率确实也不太高。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>某种程度上来看，</span><span><strong>最终的销售数据的确从侧面验证了罗永浩的观点</strong></span><span>——很多网友涌进直播间或许真的并不是为了买东西，而纯粹为了看“内容”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/16900100706233b25e365e39.91683873.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>“讲产品”与“讲内容”之争</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于直播带货到底是应该聚焦“讲产品”还是把重点放在“讲内容”？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在我看来，这不仅是个仁者见仁智者见智的事情，更是一个对于不同直播间、不同品牌应该分开来看的问题。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于像罗永浩这类自带话题与流量的大主播来说，自然可以把直播核心放在“讲产品”上，因为他有足够且庞大的忠实粉丝来撑起整个直播间的日常可观流量。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/165909637262ce0dc7866dd8.79970933.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就像罗永浩自己说的，</span><span><strong>我们直播间40%的人进来啥也不听，直接奔购物车下单就走人。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这就是极具人格魅力的大主播的底气，因为你无论讲什么内容，忠粉始终会买单。某种程度上来说，</span><span><strong>这也就无所谓“讲产品”与“讲内容”之分了。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但对于那些不知名的小主播，还有很多本身就缺乏知名度的自播小品牌来说，选择了罗永浩说的认认真真“讲产品”，那也就是意味着放弃了“看点”与“话题”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>可能这都不是卖不卖的掉货的问题，而是有没有人看的问题了。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/88877480662ce0dc7f05a07.94111234.jpeg" referrerpolicy="no-referrer"></p>
<section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这也是为什么我们在抖音看到有人直播宫斗卖化妆品、有人跑到雪山卖羽绒服，一群中年大叔为了卖衣服尬舞.....</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因为对于他们而言，既没有董宇辉的才气，也没有罗永浩的人格魅力，</span><span><strong>猎奇也就成了唯一的出圈路径。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果能在讲产品的同时巧妙加入内容，也许可以在直播场景中更有效的促进带货的下单。但对于大多数主播和品牌来说，显然没有这个能力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>因此对于罗永浩关于直播带货的“讲产品”论，显然是有适用范围的。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但有一点罗永浩说对了，直播可能1/3重要性是主播和讲解本身，2/3重要性是拼后台供应链能力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>毕竟，无论“讲产品”还是“讲内容”，本质上都是为消费者服务的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而从更宏大的直播行业来看，一个行业的繁荣发展，一定是百花齐放的。没有所谓的“歪风”与“正风”之分。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>因为这本质上是由直播间“人设”决定的。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就像你让以“知识卖货”著称的东方甄选突然一板一眼的在直播间“讲产品”，虽然更有效满足了消费者的购物需求，但对于因“知识”而喜欢他们的大多数粉丝而言，显然是无法接受的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以，这中间顶多是个“讲产品”与“讲内容”两者占比与度的把握问题，不是个非此即彼的矛盾选择题。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因此在我看来，未来的直播带货大概率是按照“内容”与“产品”的路径双线发展。对于还在探索的直播行业来说，东方甄选也只是趟出了千万条通往“罗马”的一条路而已。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>显然，罗永浩只说对了一半！</span></p>
