<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>可口可乐的传奇总裁罗伯特·伍德鲁夫说过——</span></p>
<p><span>“即便可口可乐的工厂被一把火全部烧了，只要我们的商标和Logo还在，我们也能立即在废墟上重建一个可口可乐。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这就是品牌的力量。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然而彼得 · 德鲁克说——</span></p>
<p><span>“ If you can't measure it, you can't improve it”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>没错，品牌作为现代企业中一项重要的战略，如何精确地量化它一直是营销领域的一个难点。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><em><span>“我知道品牌很重要，但貌似是一个虚无缥缈的东西，我讨厌不确定性。”</span></em></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><em><span>“我花钱做了不少高大上的品牌广告，但短期销量没怎么涨，我怎么知道它能不能给我带来长期的增长？”</span></em></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><em><span>“我请了定位专家策划、请了明星代言广告，但只要不做促销，销量就上不去，根本看不到溢价，这是为什么？”</span></em></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>没错，这些普遍的问题都困惑着每一个做品牌建设的营销人和市场人。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>今天，卫夕和大家一起来聊一聊“如何量化品牌力”这个话题——</span></strong></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong>一、量化品牌力：从一个模型说起</strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>8月25日，2022年度凯度BrandZ最具价值中国品牌100强发布会在北京召开。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1738450366308743a7194d1.94229884.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这次发布会的一大亮点是，巨量引擎联合BrandZ共同探索品牌在全链路增长中的价值，现场发布升级版的SCI品牌力模型与品牌力增长度量白皮书。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>该白皮书的核心主题是——“看见确定性增长”，旨在对品牌数字资产进行科学的度量评估，帮助品牌做出全局最优的营销决策，实现健康长效的高增长，让品牌的力量可以真正被看见。</span></p>
<p><span>“我们始终相信，品牌是企业抵御风险的护城河，也是企业实现增长复利的奠基石。在今天，品牌有更多的机会直接接触消费者并和他们沟通互动，如何更敏锐地洞察消费者需求，如何科学度量营销效果，如何灵活优化策略，数字化为品牌带来了全新的机遇与挑战。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>巨量引擎营销与运营副总裁陈都烨在白皮书的前言中写道。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>SCI品牌力模型能帮助品牌知己知彼，实现健康、科学、可持续的高质量增长。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我详细地阅读了这份白皮书，发现里边还是有不少关于品牌建设与经营的干货，它为数字时代品牌力如何科学地衡量、合理地建设提供了一套可行的思路与方法，值得从业者细读。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>下面我会和大家一起来解读白皮书的核心内容。</span></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong>二、SCI品牌力模型：给品牌一把尺</strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>要打造品牌，营销人首先需要解决的是如何度量品牌力，只有能合理的评估和度量品牌力，才可能有的放矢地、有针对性地建设品牌。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在传统时代，品牌度量通常采用的方法是抽样调查，一般从知名度、美誉度、忠诚度等维度衡量<a href="https://www.niaogebiji.com/tag/%E5%93%81%E7%89%8C%E4%BB%B7%E5%80%BC" target="_blank">品牌价值</a>，主要由品牌自身或第三方公司完成。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在数字时代，丰富的线上触点为品牌的全量精确度量提供了多维度的海量数据，如何合理地利用这些不同维度的数据来构建科学的评估模型成为品牌度量的关键。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>在这样的背景下，巨量引擎推出了“SCI品牌力模型”，在这次白皮书中，巨量引擎又对其进行了升级。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>具体而言，“SCI品牌力模型”会从品牌规模（Scale）、品牌效率（Conversion）、品牌形象（Image）三个维度，在认知、种草、行动的三个不同营销阶段，持续评估和追踪品牌的健康度，帮助品牌实现“品牌建设 + 心智种草 +效果转化”三位一体的规模化增长。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/4845680556308743ad921d4.06336282.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>品牌规模代表了品牌资产的量，是品牌的基本盘，体现了品牌在巨量生态内人群资产的体量。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>品牌效率代表了品牌资产的转化效率，效率指标包括人群流转率和内容效率两部分，人群流转率主要评估从消费者到购买者的转化效率，而内容效率，如一个品牌发布内容，能否将产品的优势卖点快速地传达给潜在消费者，最终也会影响人群流转效率。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最后一个指标是品牌形象，它是消费者对品牌的综合印象，体现在行动上就是品牌溢价与复购能力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>SCI品牌力模型是品牌力建设的一把尺子，刻度清晰、逻辑完善，测量结果客观、有效。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>基于抖音集团旗下巨量引擎的海量用户和全链路数据，依托“品效合一”的天然优势，SCI品牌力模型将三个维度精细地落实到了多个指标。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这些指标让品牌在全链路驱动增长的价值能被清晰看见。不仅能量化短期的营销效果，也能衡量长期的品牌效应。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在我看来，巨量引擎的SCI品牌力模型，它代表了国内在营销度量层面的一次非常有意义的探索，为什么这么说呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>主要基于以下两点——</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span>1.短期可度量：一把量化标准的尺</span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为什么它可以成为品牌度量的一把尺？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这背后的核心是全链路的数据，拥有广泛用户基础且高度活跃的抖音生态下的媒体平台已渗透到人们的日常生活和消费场景中，掌握了从触达认知、种草到购买转化的海量决策全链路数据。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>因而SCI品牌力模型具备实现“品效一体”的评估体系，更具有科学归因能力和预测性。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一个新的品牌，要想知道自己的品牌建设如何，在巨量引擎的后台，根据SCI品牌力模型的指引，就可以分维度看到自己在各分项的具体表现以及和行业头部品牌的对比。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如白皮书披露，基于模型的评估发现，各行业内头部品牌与尾部品牌在品牌力建设上已形成较大差距，全行业头尾部品牌组的SCI倍差达94倍，其中品牌规模和品牌形象的差距更大。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从行业视角来看，母婴和个护行业头部品牌的品牌力优势更大，手机和服饰行业竞争相对激烈。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>品牌建设，闭门造车行不通，知己知彼非常重要，建议感兴趣的品牌可酌情去进行一次“品牌力自测”，看一看自家品牌在各个分项上的真实数据与表现。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span>2.长期可指导：一把指导增长的尺</span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>度量是过程，增长才是目的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>SCI品牌力模型在各行业品牌建设过程中都可以被灵活应用，不仅可以指导当下的营销动作，还能对长期的生意增长提供前置性的预测指导。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如何基于SCI品牌力模型实现长期的生意增长很关键，根据白皮书中评估发现，影响生意增长的要素在不同行业呈现出差异化特点。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>因此，结合品牌本身的品牌力建设现状与发展目标，品牌力动能提升路径各有不同。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如，手机行业除了品牌规模建设外，品牌形象的构建对GMV的影响强度很大，因此，提升相关指标才是增长的关键路径。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>再比如，食饮行业高品牌力在品牌规模上优势更大，因此可以重点加强在品牌效率上的提升，加速人群流转效率以及内容互动。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/3625149786308743b481065.19939500.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于<a href="https://www.niaogebiji.com/tag/%E5%93%81%E7%89%8C%E8%90%A5%E9%94%80" target="_blank">品牌营销</a>人而言，SCI品牌力模型所定义的品牌数字资产三大衡量维度会成为品牌数字营销领域的重要指标，将成为品牌CEO、CMO、电商负责人以及数字化营销从业者在品牌营销决策的重要参考工具。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>正如巨量引擎营销科学负责人卢忠浩所言——</span></p>
<p><span>“巨量引擎希望构建一套更加现代和科学的产品能力，通过底层的、实时的数据预演来论证指标的相关性，帮助各类企业在品牌建设过程中都能灵活应用，不仅可以指导当下的营销动作，又能对长期的生意增长提供前置性的预测指导，最终可以提升企业在品牌建设和生意成长过程中的控制感和确定性。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在我看来，这套品牌营销方法论最大的价值在于它第一次真正清晰地量化了品牌数字资产同时提供可以落地提升的方法。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>它的底层逻辑在于——品牌的长期增长来源于可经营消费者人群规模、消费者转化率和单个用户收入（ARPU）值的增长。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>SCI品牌力与生意增长存在正相关性，对企业的短中长期收益均存在持续性影响力，SCI评估体系下，高品牌力组对比低品牌力组在不同生意指标上的表现有显著优越性。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>从这个意义上讲，做好了品牌，就做好了增长。</span></strong></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong>三、品牌力建设：如何应用科学度量实现长期全量增长？</strong></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有了度量和指导的模型，那么具体应该如何应用呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>巨量引擎基于SCI品牌力模型开发了一套可视化的管理工具，让品牌力洞察清晰可落地。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在具体的操作层面，白皮书通过多个案例给出了“三步走”的方案——</span></p>
<p><span>第一步：现状定位诊断与历史趋势分析</span></p>
<p><span>第二步：问题定位和场景化策略</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在我看来，这次白皮书中升级的“SCI品牌力模型”最大的意义在于——让品牌力“看得见、摸得着”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>“看得见”代表可评估、能量化，“摸得着”代表可指导、能增长。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>事实上，品牌建设是一个长期的过程，不能一蹴而就，一方面需要在营销和经营的持续投入形成动能，另一方面是不断积累形成的存量，两者之间不断相互转化形成提升品牌力的双轮驱动模式。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>只有每一次品牌营销都遵循科学的评估与合理的策略，持续地投入，才能撬动存量与动能的“飞轮”，真正让品牌力这个“雪球”不断滚起来。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2021年《财富》的世界500强企业名单中，中国公司占了143家，首次超越美国，成为第一，这是中国企业实力的明证。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然而，在另一个榜单中，我们就要逊色不少：2022年凯度BrandZ最具价值全球品牌排行榜Top100，中国上榜的品牌仅有14个，而美国是63个。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今天，中国拥有超过170万个品牌，在数量上和美国旗鼓相当。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>所以，从这个意义上，中国企业在品牌建设上依然任重道远。</span></strong></p>
