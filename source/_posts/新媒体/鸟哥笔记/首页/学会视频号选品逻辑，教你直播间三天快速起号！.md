<section>
<p><strong><span>1.视频号算法和抖音算法的区别</span></strong></p>
</section>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>视频号和抖音算法有这么几个区别，第一个叫防沉迷系统，一刷抖音基本上一两个小时停不下来，他给你推的基本上都是观众爱看、想看的，就像开盲盒一样，你不知道下一个会看到什么东西，所以会一直刷。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但是视频号，在抖音的基础上加了一个防沉迷系统，官方给我们的意思很明确，我们做视频号的目的不是为了让你们在微信里面长期的泡着，沉迷不出来，而是想增加一下我们微信里面的用户的停留时长，仅此而已。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/211150435662eb391f735ec8.68538135.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>△ 图片来源：嘉宾分享PPT</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>视频号无论直播间，或者短视频也好，主要推流机制是生活记录，还有一些知识分享类型，而且不会给过重的标签，让我们在视频号板块停留过多时间，所以腾讯的想法就很单纯，就是增加微信的停留时长，增加微信的使用率就好。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以基于这个理念，腾讯官方在给我们推送流量，也就基于这个理念，我们再配置我们的素材，还有直播间的时候就舒服很多了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>第二个是内容分发的系统，抖音最开始是做优质内容起的家，但是后来市场把他逼商业化了。所以现在抖音的电商逻辑要大于内容的逻辑，因为它是主要靠电商来变现，内容变现反而是不怎么好，所以出现了这种情况，说抖音现在电商大于内容，但是视频号经过对于抖音的一个总结，还有对抖音的错误方向调整，做了这么一个算法，就是<strong>内容逻辑大于电商逻辑</strong>，也就是说我们如果踏踏实实做内容的话，视频号粉丝用户粘性比抖音高非常多。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>总结来说，抖音的变现是在围绕着产品去变现，但是<strong>视频号变现是围绕着用户、好内容再去做变现</strong>，所以视频号对于违规玩法的容忍度几乎是零。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>第三个不同点就是抖音是根据产品的标签来找用户，就是根据产品来匹配的用户，但是视频号不一样，这是一个本质区别。<strong>视频号是根据用户粘性来找的产品，</strong>所以在视频号儿里面踏踏实实做人设，做一个鲜明的人设，做一个优质内容，这些这些用户在视频号儿里面吃到的红利会比较多。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这几个点，是我们跟官方沟通了好多次之后得出来的，大家如果想做好视频号，建议大家把三个点，深度理解一下。</span></p>

<section>
<p><strong><span>2.视频号使用人群分析</span></strong></p>
</section>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>了</span><span>解完基础的算法和存留机制以后，接下来就是选品。</span><span>在选品之前我们要先分析用户，视频号的整体用户年龄会偏大，主要是40岁以上的女性为主，这是我们综合了大概十几个每天能卖到5万以上的朋友的数据，得出来结果。</span><span>第一个是年龄，主要以40岁以上为主，其中是50到55之间的特别多，然后最高的能到70岁，年龄很高。</span></p>

<section>
<p><strong><span>3.视频号如何选品</span></strong></p>
</section>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>知道了视频号的人群以后才能匹配一个合理选品，这些选品配上这些人群，这种成功概率会大一些。这些选品也是我们身边朋友或包括我们自己在测试过的一些产品。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>直播间的选品逻辑</span></strong><span>，大概就这么几类，第一个是文玩类，檀香茶叶，茶壶。还有茶器这些，在视频号销量比较可观，因为我有一个直播间是在福建泉州，主要做茶器和茶叶，这个赛道一天有5到10万，比较稳定。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/19821620062eb3920496db6.29905045.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>△ 图片来源：嘉宾分享PPT</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>第二个是服装类，我们自己主要做的是T恤，还有基础款服饰。我们楼下一个老哥做的是爸爸服装，我们通过调查发现，买服装人群40％-50％是保安，知道这个人群以后，我们也让这个老哥随机给粉丝做了抽调，为什么你会买这种服装，为什么你会在视频号的直播间买，粉丝给的回答是比较诚恳，第一个我就是保安，一个月也没多少花销，我看你跟我岁数差不多，穿的衣服挺好看，我就想买过来试一下，我半夜没啥事，就习惯性的刷下视频号，不想刷抖音，刷着个直播间，看着挺好就买了，买了之后一穿很舒服，接着就回来买了，是这样的消费状态。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>带大牌元素的服饰，不论是在抖音也好，视频号也好，都是卖的比较好，但是风控也是最严格的，不论哪个平台，只要逮到以后，基本上就是被咔嚓掉了。我们起过两个大牌元素的账号，起号很快，销量也很高，但是面临的风控也是最高。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/175251094162858b8281ee79.88528627.png" referrerpolicy="no-referrer"></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/69010668262eb392099ee62.81861989.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>△ 图片来源：嘉宾分享PPT</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>微信是做私域起家，所以起号的时候会对<span>私域</span>的引流，还有<span>私域</span>的成交要求，在6月18号以前私域和公域配比是按照互动率、转化率以及停留率三个指标去做的匹配，也就是先有互动，再有成交，就会给你匹配公域流量，其中配从最开始1 : 5到一直最高到5 : 1。就是说你私域里面进来100个人，公域里面最早给你匹配20个，数据做的越来越好的时候，能给你从公域100个匹配到私域100个，匹配到公域500个这样的比例。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>6月18号以后整个算法做了一次大调整，由原来互动作为推流指标改成以成交作为推流指标，所以用一些水军或者黄牛三方公司在刷互动起号，这种方式已经不适用，必须要有成交，他更在乎私域的成交金额和数量。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>公域和私域的配比比例基本上没有太大变化，还是这么一个比例，从1 : 5到5 : 1一个比例。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/43482754362eb3920e2e8a0.25131083.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>△ 图片来源：嘉宾分享PPT</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/142739436362eb3921333d33.02720975.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>△ 图片来源：嘉宾分享PPT</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>直播间起号之前，第一个做好指示牌，指示牌告诉粉丝怎么加团，怎么购买，怎么选，规格越细致越好，做成指示牌，做成手牌，因为视频号用户大多数是没有在直播间买过东西，也没有在直播间做过任何动作，所以他们在直播间里面几乎是一个小白。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>第二个，因为是裂变的推流机制，要随时做好裂变的准备。裂变的方式和以前的公众号推广一样，转发朋友圈送小礼品，只不过是把原来线下地推改到了直播间。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>第三个点，直播间的活动，和抖音基本上差不多。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>第四个，就是起号的方法，低价引流很简单，低价引流在视频号的直播间是适用的，视频号的直播间是没有太重标签逻辑。所以在直播间有了低价引流的成交密度，是可以撬动公域流量。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/32149516162eb39217666b5.55614539.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>△ 图片来源：嘉宾分享PPT</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>起号第二天，开播就直接把直播间分享到私域里面，通过做活动，哪怕低价成交，低价引流都可以通过做活动拿私域，把公域流量拉开。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/206001145262eb3921c4ab35.70266087.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>△ 图片来源：嘉宾分享PPT</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>第三天逻辑和第二天，基本上是一样的，先用私域拉动公域，加上一些低价活动，把公域给翘起来，那么三天基本上就可以起来，因为连续两天已经把直播间分到私域，所以第四天，私域的朋友大概率是能刷到直播间。除了私域撬动公域以外，这么一个推理逻辑以外，还有是朋友圈的一个隐形裂变，一个推流方式。大家如果有时间的话，可以仔细看一下直播间广场的界面，会有一个小字，就是两位朋友看过，三位朋友看过，会有这么一个小字，所以说朋友裂变的这种这种机制会存在于视频号的直播里面。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/84255656862eb39220ec264.65678514.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>△ 图片来源：嘉宾分享PPT</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有了这种裂变机制和这种互推机制，到了第四天基本上就不用再分享，他不分享也能看到你的直播间了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这是直播间三天起号的方法和逻辑。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/93076869762858b82bd4854.18600368.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong>直播间流量如何正确运营</strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>要想把视频号做好，需要有深度的思考和长期的心理准备，因为腾讯不比抖音，抖音是一个新的平台，在一个新的机遇或者一个偶然的机遇下面发展的很快，但是腾讯是一个非常成熟的平台，会考虑长期的东西，怎么做、怎么玩。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/145763935162eb39225f0fe7.93882002.png" referrerpolicy="no-referrer"></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>腾讯对于自身生态非常看重，破坏生态必然是一刀切，至少是三类，第一类，是演戏的直播间，公婆矛盾，婆媳矛盾，包括这个夫妻矛盾这些，这些矛盾是的，直播间吵架，演戏，这些直播间到现在来说一个都没有了，被一刀切了。第二个，是那种录播的直播间，这个还没有切完，也在下刀子了，所以腾讯对于这种抄近路比较抵触。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但是，我们进到视频号里面就是两个字搞钱，但是搞钱的前提下是不要伤粉，因为视频号的粉丝确实是比抖音单纯很多，年薪也高很多。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>优质内容是所有平台的刚需，是我们从18年做抖音，一直到现在总结的一个小经验，把内容做好，粘性自然就有了，粘性有了之后，整个的链条或者整个直播间是非常健康的状态，健康往上走是每个人都想要的一个结果。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那什么叫好的内容？三个点，第一个我是谁，第二个我在做什么，第三个大家为什么要买我东西，把这三个问题思考明白了，好的内容就出来了，一个好的内容、好的粘性带来的价值是很常见。</span></p>
