                        <p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">“受不了领导的气！我的领导太苛刻暴躁，我不想伺候了！”</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">职场小白的第一份工作可能伴随着这样的内心活动而仓促结束。2022年的毕业季，首批00后大学毕业生将跨入职场，作为伴随互联网成长起来的00后在面对自己不喜欢的工作，他们有说走就走的勇气。而这份勇气的背后，或者伴随着对于职场的不适应，对于陌生环境的逃避，甚至是对于新同事，工作内容的抗拒。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">职场的生存之道是属于职场小白的必修课，如何克服新入职场的恐惧，快速熟悉陌生环境并漂亮的掀起职场的第一篇章，这是本文与大家讨论的要点。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/558e947d0fa85c4730c411b476645c25.png" alt width="789" height="525" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>首先，职场新人应当学会角色转换。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">从一名大学生到一颗职场螺丝钉，这是两个不同社会组织里的角色定位。心理学有个概念，叫做角色适应性。指个体的角色表现与角色期望达到协调的、良好的状态。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">角色适应特指个体在社会适应的角色转变中的适应过程。这里的社会适应就是上班作为职场人的的交互适应过程。我们要学着忘记学校里的人际规则，行为准则，主动地顺应职场的规则，适应职场的人文和社会环境，最终达到与公司文化和规章制度匹配和谐以及平衡的状态。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">这里新的职场角色，是个全新的角色，要抱着学习的心态，谦卑的心理开始新角色的过程。职场新人可以参考从家里的宝宝到学校学习成为学生的经验，初期多听多看多做，慢慢摸索出新角色的行为模式。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://qnssl.niaogebiji.com/c68a43f6327c0354b275f130925be83f.png" alt width="789" height="525" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">其次，应当具备一定的技能储备，比如以下几个方面：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>1、Office使用能力。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">一份夺人眼球的ppt，一个少有人用的EXCEL 函数，在提升工作效率的同时，可以让人刮目相看。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>2、专业标准的公文书写能力。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">能把事情交代清楚明白的一封邮件，一份标准格式的公司间公文，会让公司的前辈对你器重不已。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>3、出色的文件档案管理能力。 </strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">不论做任何工作，有条理有计划，需要的时候你可以迅速的找到它以及与之相关的文件，这很重要。 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>4、从不喊累的体力。 </strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">身体是革命的本钱。拥有好的体力，才能应对工作中的压力与复杂性，保持生活与工作的平衡。 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>5、多点加分类技能。 </strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">比如PS，摄影特别重要，每个公司都需要这种华而不实的变美能力，当然短视频制作能力或者才艺表演，都会让你在新加入一个公司后留下深刻的印象。 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>最后，提供一些职场厚黑学的实用技巧。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">1、不要质疑公司的决定，对于新人来说，你不了解公司，一切的行为准则以公司规章制度，公司文化为准则，即使不对，也要三思而后行。 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">2、听领导的准没错，除非你的运气很差。领导是你的上司，完成上司交代的任务无可厚非，与此同时，他又是你的领路人，甚至是职场的隐形师傅。出于对于前辈的尊敬，也需要与领导看齐。 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">3、遇到其他公司同事，要积极主动交流。就跟玩游戏做任务一样，多跟NPC交流，可以开展更多的支线隐藏任务，你在公司的路才能走得更宽。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">4、做错事情，不要推卸责任，哪怕对了也要包揽自己的不足之处。因为哪怕你对了，也是你没有足够的能力预判错误的发生，所以承认错误，没有坏处，只有好处，赢得同事和领导的赏识，让他们看到属于你的担当。 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">5、敢作敢为的行动力，哪怕错也能看得见的担当。言必行、行必果是最好的通行证，坦诚相待会为你赢得尊重与信赖。 </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">6、多看一些华而不实的心灵书籍，虽然没有实际有用的结果发生。但是学着迂腐俗气，也是能让你与公司融入一体的有效方法。比如，讨论良好的心态对于工作的帮助；比如谈拍马屁对于领导的作用等。 </p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">结语</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">以上可以帮助我们在职场中不那么暴躁，还没踏入职场就自己把门关掉，耐着性子趟一趟这个浑水，等游得欢快些了，在想着往其他地方游一游。把工作保住，才能更好的适应这个社会，有了一个良好的社会关系，才会慢慢拥有自己想要的生活。</p>                        <div style="height: 1px"></div>
