<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>618依然拉开了帷幕，虽然面临着流量增长的乏力，但这场年中盛世依然吸引了各大品牌纷纷入局。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其中互联网媒介语境下衍生出的“她经济”表现亮眼。据007了解，作为“她经济”重要组成部分的孕产行业成绩突出。比如</span><strong>孕产行业TOP1<a href="https://www.niaogebiji.com/tag/%E5%AB%9A%E7%86%99" target="_blank">嫚熙</a>，在618期间多个产品线贡献了超高GMV，GMV渗透率亮眼！</strong><span>亮眼成绩的背后既是</span><strong>嫚熙在营销上的高举高打，也是其对新生代妈妈营销上消费主张、产品上消费需求和品牌上情感价值三重满足的必然结果。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/202643587762aa131a42ce80.38530308.png" width="555" title="618再破圈，“反重力”的嫚熙何以俘获万千新生代妈妈？" alt="618再破圈，“反重力”的嫚熙何以俘获万千新生代妈妈？" referrerpolicy="no-referrer"></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>嫚熙</span></strong><span>站上孕产赛道C位，俘获用户芳心的原因何在？把时间线拉长，007发现了嫚熙突围登顶的关键——营销上的情感共鸣、产品上的用户共创、品牌上的社会价值。</span></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: center; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong style="mso-bidi-font-weight: normal;"><span>以感同身受制造情感共振</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于品牌而言，差异化的营销从来都是激发品牌声量的关键一步。如果说嫚熙618的强势表现是以产品力为核心，那几乎贯穿整个5月的一场“反常规”营销campaign则为品牌的强势爆发埋下了 “伏笔”。</span><span></span></p>

<p> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为了鼓励更多的妈妈爱自己，也为了传播出更大的舆论声势。在寓意“我爱我”的5月25日，嫚熙不仅推出了一支《合格妈妈统一考试》品牌TVC，还在社交媒体发起#525支持妈妈爱自己#话题，引发大众热议。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>短片用一场考试暗喻女性成为完美妈妈背后的“枷锁”，片中的场景生动还原出妈妈日常面临的压力，真实场景的演绎，加强了观众的代入感，于无形中拉近了品牌与目标消费人群乃至泛人群的心理距离。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>短片最后“另一个”女主面带自信笑容的出现以及被风吹走的</span><span>试卷，嫚熙暗暗给出了答案，更借傅首尔直接亮明品牌此次活动的传播主张——<strong>「反对条条框框的束缚，从爱自己开始」</strong>。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在007看来，从5.8母亲节“唱反调”海报到525推出《合格妈妈统一考试》TVC，嫚熙打破了同质化赞美母亲的营销套路，跳脱以往营销上的“母亲身份”语境，转而从女性“爱自己”这一“非母亲”角色的自我视角出发进行品牌传播，从而以极强的代入感为新生代妈妈们发声，不仅制造出情感的双向共鸣，更打造出母亲营销议题浪潮中的品牌差异度和记忆度。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-role="paragraph"> </p>

<p> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: center; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong style="mso-bidi-font-weight: normal;"><span>产品“反重力”</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: center; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong style="mso-bidi-font-weight: normal;"><span>以用户共创落地品牌关怀</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然，品牌的塑造并非无本之木，最终都要落到产品力的支撑才能获得最具穿透力的用户渗透。站在这个视角审视嫚熙，不难发现其在产品上的创新，在多年的产品研发和迭代中早已植下了基因。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/99371206562aa131c99f481.60092523.png" width="444" title="618再破圈，“反重力”的嫚熙何以俘获万千新生代妈妈？" alt="618再破圈，“反重力”的嫚熙何以俘获万千新生代妈妈？" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: center; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>（图为<strong>嫚熙待产子母包）</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以<strong>嫚熙待产子母包</strong>为例，作为对生育更有准备的一代，新生代的妈妈在产前会对产期用品会尽可能准备充分，避免在生产时因为物品不全而忙乱。所以，在待产包的准备上，这一届的妈妈更加地讲究，她们不仅要多、要全更要好品质！而市面上不少品牌的待产包只是片面追求量多，殊不知使用起来却是十分鸡肋。所以嫚熙在待产包研发过程中始终坚持与孕妈们深度沟通进行“共创”，为她们打造理想的待产包。依托于200万社群粉丝和真实用户的使用反馈，待产包从最开始的10多件到30多件，到现在的51件产品，</span><span>全都源于妈妈们孕产期的经历所需研发，让妈妈可以定制化选择，尽量做到让待产包每一件物品都物尽其用。</span><span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如考虑到妈妈孕期喝水不方便，嫚熙特意增加的一次性可折弯的吸管，</span><span>考虑到妈妈产后会担心吹风，以及当时身体各个方面都会相对敏感，又增加了月子期专用的头巾、袜子用于保暖，还有添加了牙膏、牙刷等生活用品。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>品牌对消费者的关心在于——要比消费者想得更多。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>嫚熙的产品已经不仅停留在对妈妈的考虑，还强化了对新生儿细致入微的关怀。比如在赠送的新生儿小王子礼盒里，给孩子准备了小狐狸安抚公仔，方便妈妈更好地照顾孩子，减少孩子因分离焦虑导致的哭闹；还提供了婴儿连身衣、小方巾、婴儿袜、肚围等，让孩子刚一出生就备受呵护。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>更为重要的是，除了用户共创的产品思维，嫚熙还通过多元化产品发力，以出人性化的产品满足妈妈孕产、哺乳和养育全时期期间全场景的痛点需求。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不得不提的就是<strong>嫚熙的明星产品——太空反重力内衣</strong>。对于孕妈们来说，孕期&哺乳期主要有三个痛点：</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>痛点一：胸围变大、怕下垂以及胸外扩哺乳期涨奶。</span></strong><span>为了解决这3大难题，嫚熙在研发上费尽了心思，更不计成本打样，为的就是给<strong>妈妈们设计出一款穿着舒适不紧勒且防下垂的无缝内衣</strong>。</span></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但早期国内并没有无缝制衣的设备和面料。为此嫚熙花费3个多月从海外采购设备，并专门定制原材料。由此成为中国第一个用全进口的纱线材料的母婴品牌，也是第一个从原材料就开始实现定制的母婴品牌。为了让成品更符合孕期妈妈胸围及胸型，光是打样就打了68 次，过程中不断去调整产品细节不断迭代……最终研发出弹性和延展性极强的太空反重力内衣。面料的弹性能给予内衣20%的扩展空间去适应孕妈胸部的变化;同时，考虑妈妈哺乳需求，贴心地设计了可单手一键解开的哺乳扣。看似一件简单的哺乳内衣，背后全是对妈妈们需求的细腻洞察，也让妈妈能够以更从容地面对孕育期胸部不适，及哺乳穿脱麻烦的问题。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在设计上，嫚熙研发出了<strong>【W反重力黑科技】</strong>，利用啵啵条设计增加软承托力，其承托力是普通内衣的3倍，有效防止下垂，并通过立体【水滴杯专利】设计让聚拢胸形无束缚感，<strong>综合解决妈妈产后哺乳怕胸部下垂或内衣太紧勒导致胸部变型的问题</strong>。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/7590205662aa131d23c6b3.77186869.jpeg" width="444" title="618再破圈，“反重力”的嫚熙何以俘获万千新生代妈妈？" alt="618再破圈，“反重力”的嫚熙何以俘获万千新生代妈妈？" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: center; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>（图为<strong>嫚熙太空反重力内衣）</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>痛点二：市面上现存的内衣往往面料弹性差、勒胸，而且容易堵奶。</span></strong><span>为此嫚熙在面料的使用上不惜投入重金，</span><span>测试了不下20种材料，最终</span><span>采用市面上少有用到服装上的<strong>宇航级太空面料</strong>来增加弹性，而且轻柔舒适0触感，<strong>透气又抗菌，让堵奶、勒胸的问题迎刃而解。</strong></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>痛点三：哺乳内衣好看与舒适性兼顾的难题，市面上的大多数哺乳内衣不仅不好看而且容易闷汗，并不适合24小时穿着。</span></strong><span>为此，嫚熙一方面<strong>邀请日本设计大师藤原灡专为亚洲孕妈身材定制，提升颜值</strong>；另一方面通过一片式无痕贴合设计和哺乳扣设计<strong>有效解决了24小时穿着舒适性</strong>的问题。优质的产品力本质上是对消费者具体场景痛点的逐个击破，显然嫚熙做到了。</span></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: center; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong style="mso-bidi-font-weight: normal;"><span>品牌“反平庸”</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: center; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong style="mso-bidi-font-weight: normal;"><span>以社会价值创造品牌新增量</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>凯文·凯勒在《管理品牌资产》一书中认为，当产品提供的功能性价值足够与竞争品牌产生差异化时，品牌的外延需要以更高的视野与价值维度来塑造品牌。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>反观如今的许多品牌，披着“创新”的外衣，高举着新消费的旗帜“颠覆一切”。然而细看之下，不过是亦步亦趋的邯郸学步，在过度的同质化与概念炒作中沦为平庸。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以此来反观嫚熙的品牌发展之路，在007看来，其一直走在反平庸的路上。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>一是产品上的反平庸。</span></strong><span>产品创新的本质从来不是创新本身，而是想消费者之所想，想消费者之未想。成立多年的嫚熙始终将品牌的极致精神与人文关怀一丝一毫融入对产品的创新改良，为妈妈们带来全新的产品感知。在007看来，这才是真正的于无声处自有声。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>二是品牌上的反平庸。</span></strong><span>只有商业理想没有社会理想的品牌不是好品牌。于嫚熙而言，在营销中纳入妈妈群体的身份困境，让妈妈“爱自己”的主张以一种倡议的姿态进入大众视野，使营销超出了商业的范畴，更有社会意义。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>某种程度上来说，这为品牌注入了社会价值，创造了新的价值增量。这不仅之于品牌自身，于行业而言都指明了一个方向：品牌的价值从来都在纯粹的商业之外，不仅要理解目标群体的消费需求，更要理解目标群体的社会需求，并为之付诸行动！</span></p>
