<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-mpa-powered-by="yiban.io"><span>中国快递<span lang="EN-US">30</span>年，战火从未熄灭。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><a href="https://www.niaogebiji.com/tag/%E9%A1%BA%E4%B8%B0" target="_blank">顺丰</a>、京东物流牢牢把持着高端快件市场的门槛，通达系则紧随其后，吃下更为广阔的中低端市场蛋糕。由此而来的四通八达物流网络，让中国拥有了全世界最为优秀的电商基建。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>浩荡三十年，璀璨一明珠。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作为当之无愧的民营快递龙头企业，</span><span><strong>即将迎来而立之年的顺丰，终于再次摘下了一顶沉甸甸的、新的桂冠。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/99323113621d859ad90f46.00495923.png" referrerpolicy="no-referrer"><span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>顺丰，首次杀进世界<span lang="EN-US">500</span>强</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>昨日，《财富》杂志正式发布了<span lang="EN-US">2022</span>年世界<span lang="EN-US">500</span>强排行榜。这份榜单中，引人注目的点太多，诸如沃尔玛连续第九年成为全球最大公司、亚马逊上升至世界第二位、京东首次进入全球前<span lang="EN-US">50</span>等。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>同时，中国邮政也连续三年进入百强行列。在所有邮政、快递企业中，中国邮政也以<span lang="EN-US">1086.7</span>亿美元营收、<span lang="EN-US">59.8</span>亿美元利润位列双料第一。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然而相比这些熟悉的老面孔，首次杀进世界<span lang="EN-US">500</span>强的顺丰或许更能带给我们惊喜。</span><span><strong>营收<span lang="EN-US">321</span>亿美元、利润<span lang="EN-US">6.6</span>亿美元的顺丰控股，在榜单中位列第<span lang="EN-US">441</span>名，创立近<span lang="EN-US">30</span>年，它终于做到了！</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/27614438562eca6c7246082.54292100.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span> <span>图源：财富世界<span lang="EN-US">500</span>强</span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>回想一年前，由于一季度巨亏<span lang="EN-US">10</span>个亿，年逾半百的王卫亲自向投资者致歉，“首先跟股东做一个道歉，因为我认为第一个季度真的没有经营好。”彼时的他，多少让人感到有些英雄迟暮的悲凉。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一年后的今天，他终于可以直起腰来，扬眉吐气。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不仅如此，顺丰的此次入榜，也代表着除中国邮政以外，中国民营快递企业首次进入世界<span lang="EN-US">500</span>强的席位。</span><span><strong>顺丰的一小步，是中国快递行业的一大步！</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>或许有人会问，凭什么是顺丰，而不是单量更多的通达系？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>让我们用最直观的数据来回答这个问题。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>纵观<span lang="EN-US">2022</span>年中国<span lang="EN-US">500</span>强榜单，除顺丰外，圆通、韵达、德邦、中通、申通的营收分别为<span lang="EN-US">452</span>亿、<span lang="EN-US">417</span>亿、<span lang="EN-US">314</span>亿、<span lang="EN-US">304</span>亿、<span lang="EN-US">253</span>亿元。而顺丰无论在排名还是营收上，都与他们有着断层般的差距。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/134163817162eca6c77bebb9.74146669.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span> 图源：顺丰集团官网微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>顺丰的营收达到了惊人的<span lang="EN-US">2072</span>亿元，这一数字要比另外<span lang="EN-US">5</span>家的营收总和还要高，甚至可以抵得上两个京东物流。</strong></span><span>可以说，顺丰的快递一哥之名，名副其实。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而《财富》在进行世界<span lang="EN-US">500</span>强排名时，便是以企业的营业收入作为最主要指标，这是用以衡量企业增长和成功的最可靠指标，也是顺丰成为中国民营快递第一的根本依据。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>显然，杀进世界<span lang="EN-US">500</span>强并非易事，甚至由于多方面影响出现了上市后首次亏损的顺丰，也曾经历黑暗时刻，一度令市场担忧。但还好，星光不负赶路人，它最终还是做到了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/40300182621d859c4744b0.36029202.png" referrerpolicy="no-referrer"><span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>扭亏为盈，强势逆袭</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一路走来，顺丰的经历像极了班级里的优等生，成绩稳定且始终名列前茅。只是天天被夸的优等生，也偶有考个烂成绩的时候。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span lang="EN-US">2021</span>年，我国快递业务量首次突破<span lang="EN-US">1000</span>亿件，达到一个新的里程碑。而在这一年的开头，顺丰却交出了份业绩大跌的成绩单。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据其<span lang="EN-US">2021</span>年一季度财报显示，该季度顺丰营收<span lang="EN-US">426.2</span>亿元，同比增长<span lang="EN-US">27%</span>，业务量同比增长达<span lang="EN-US">44%</span>，</span><span><strong>但归属于上市公司股东的净利润却亏损<span lang="EN-US">9.89</span>亿元，扣除非经常性损益净亏损<span lang="EN-US">11.34</span>亿元，均同比转盈为亏。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/98706980862eca6c7eaef84.65258428.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span> 图源：顺丰控股<span lang="EN-US">2021</span>年第一季度报告</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>营收、业务量双双增长，净利润却大幅下滑，甚至于为顺丰带来了上市数年后的首次亏损。毫无疑问，这份成绩单实在惨不忍睹，让人大跌眼镜。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此番暴雷下，一季度业绩发布后的次日，顺丰股价直接一字跌停，单日蒸发市值<span lang="EN-US">500</span>亿。正如当时有人评论的那样，“</span><span><strong>顺丰的亏损如同一根沉重的锚，牢牢拖住了顺丰这艘大船，短期内无法轻舟扬帆。</strong></span><span>”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而究其亏损原因，在于顺丰战略层面的失误。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>彼时，定位高端却一心想要扩大市场份额的顺丰率先挑起了“价格战”，企图通过自身服务优势进行降维打击。通过“以价换量”策略，其的确实现了业务量的直线上升。但同时，顺丰的利润空间也在被不断压缩。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>顺丰开局以后，各快递企业纷纷入场开启新一轮战争，甚至于在发件量庞大的兵家必争之地“义乌”，出现了“一元发全国”的极端情况。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/36312730362eca6c8694dd6.39864464.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span> 图源：顺丰集团官方微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>毫无疑问，重资产模式的顺丰难捱这样的后果，</span><strong><span>最<strong>终在监管出面下，这场没有硝烟的战争才终于尘埃落定，顺丰也以单季度巨亏<span lang="EN-US">10</span>亿的代价宣告战争失败。</strong></span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所幸后来，王卫积极反省，以可见的速度进行了顺丰的自我“修复”，终于在第二季度便迅速实现了扭亏为盈，给市场吃了一颗定心丸。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>再到如今，</span><span><strong>以民营快递第一的成绩登上世界<span lang="EN-US">500</span>强，曾经的那个顺丰强势逆袭，它回来了！</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2028633948621d859f7b3c30.12849065.png" referrerpolicy="no-referrer"><span></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>顺丰的目标：星辰大海</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>事实上，顺丰急于扩大市场的背后，是快递行业竞争白热化的缩影。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>随着快递及电商市场规模的快速扩张，大量资本涌入，较前几年，快递行业格局已悄然生变。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>依托于菜鸟网络，通达系不断完善商业模式，逐步形成了“中转直营<span lang="EN-US">+</span>网点加盟”模式，几乎占据了绝大多数中低端电商快递市场。而自<span lang="EN-US">2017</span>年与菜鸟决裂，失去淘系电商平台支持后，作为第三方物流商的顺丰开始增长乏力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在京东平台的中高端电商快递市场，顺丰也面临着京东物流的直接竞争。同样的重资产运营模式、同样的高质量服务，顺丰与京东物流而言，并无直接优势。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/148528252762eca6c921d735.50354555.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span> 图源：极兔速递官方微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除了这些老对手外，极兔速递等新玩家也正虎视眈眈。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span lang="EN-US">2019</span>年，一只凶猛的兔子闯入中国快递市场，搅动风云。与拼多多深度绑定的极兔，仅一年便完成了日单量从<span lang="EN-US">0</span>到<span lang="EN-US">2000</span>万件的快速累积。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>去年<span lang="EN-US">10</span>月，极兔速递以资本换时间，以<span lang="EN-US">68</span>亿元的价格鲸吞下百世快递，直接打破了快递行业的短暂平衡，加快行业洗牌。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>各方挑战下，顺丰尽管仍处龙头，但压力剧增。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>面对这种格局，顺丰最终作出的应对是：</span><span><strong>强化自身优势，不拼价格拼质量，打好快递行业的效率战。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一直以来，顺丰给人的印象都是“速度快”和“服务好”，这也是其立于高端、占据龙头的根本。既然如此，为何不继续深挖自身优点？他们打价格战就去打好了，顺丰的目标，从来只有星辰大海。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因此，在<span lang="EN-US">2021</span>年，顺丰凭借着国内第一大货运机队、高素质末端服务，实现了时效快递全程平均时长缩短近<span lang="EN-US">2</span>小时的战绩。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/197857910262eca6c9d0e704.15771365.png" referrerpolicy="no-referrer"></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>国内站稳跟脚后，国际市场的开拓同样重要。</span></p>
