<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最近有一个朋友在公众号后台私信我：</span><strong><span>“Kevin因为错过校招，以后的工作中怎么准备才能以社招获得去腾讯做产品经理的机会？</span></strong><span>”</span></p>
</section>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其实我曾经也有多次社招面试腾讯产品经理的经历，有1-2年的，也有5年左右的高P，因为和岗位契合度、自己的个人原因就停止了后续流程。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>现在回想起来我认为腾讯的高P面试，可以说是业界里综合评判产品经理技能的最值得借鉴的标准尺度.（去辨别一个优秀的产品经理）</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这里今天说的高P，是指社招腾讯高级产品经理，职位等级在（P9以上，对标阿里P7以上）我分享下社招腾讯高级产品经理的一些经验案例。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>腾讯的高P薪资待遇可以参考下面这张表（仅供参考）</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><img src="https://ngbjimg.xy599.com/195546984162b35b59924444.78467251.png" title="社招腾讯高P（高级产品经理）的面试手册" alt="社招腾讯高P（高级产品经理）的面试手册" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>腾讯的高P（P10以上）我估计在整体员工占比为30%左右，从校园招聘进去要想成为高P，有的人花了10年还没有成功，有的人花了至少5年,所以即使在腾讯内部，高P也是稀缺的存在。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>1.高P社招面试流程长、时间长</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>腾讯的社招面试流程可以有2个月甚至更长。后面在网上也可以看到许多朋友都有已经入职突然收到腾讯offer的情况,所以我建议打算社招的同学不要裸辞去投递腾讯，因为1到2个月空闲时间难免会降低自己的工作敏感度和竞争力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>尤其是高P的产品经理，面试的时候会有一个单独的通道面（面试委员会）也是技术专家进行面试，累计的面试流程可以达到6甚至是8个环节，如果加上开头的HR环节，整个面试可以达到9轮。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如曾经我经历过高P的社招流程为：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>HR电话咨询-业务leader面试-组长面试-总监面试-面试通道-总监面试-HR面试，这个流程是比较理想的情况。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有时候候选人员会存在备胎的情况，需要增加总监交叉面试，在业务部门总监面试上会有隔壁组的总监进行评估。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>业务内面试的进展速度最快，越到后面基本就是一周甚至是2周一次面试，所以跨越的时间就非常长。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>2.高P的面试问题聚焦在产品的点、线、面、球</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>要通过社招腾讯高P的简历筛选，我了解到的前置条件即候选人现在的业务极度对口腾讯用人部门的业务。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>也就是腾讯下一步要做什么，就要找做类似或者一模一样的同学。所以在这个条件下，通过了HR筛选后再展开进入面试环节</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>点：功能怎么做</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>现在依稀还记得，腾讯高P岗位面试在开始的面试主要聚焦简历的业务进行详细介绍，也就是具体功能怎么做，具体的业务背景是什么，甚至是业务流程,同时给出现阶段的判别指标。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这一轮面试应该是产品经理最喜欢的面试内容，毕竟是自己工作具象的描述和复数，最多加一些少量的行业思考。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>线：未来要怎么做</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>随着业务部门面试轮数深入，后面会更加聚焦在产品和业务未来的发展计划，以及产品规划。说白了要讲打算未来怎么做产品迭代和商业化闭环,未来判别的数据指标。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其实这个环节难度就会提升了，难点在于部分回答内容产品经理当前的工作实际上是不涉及的<strong>，是更加需要结合自己的工作经验和履历来表达未来自己业务的发展和评定标准。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>面：资源整合、评判怎么做得更好</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在了解到了具体的功能、业务怎么做后，接下来就聚焦在讨论人员分工、现有资源运用、以及未来商业化的可能节点阶段，产品的竖向发展到横向的T型发展计划</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这一环节的面试是比较难的，</span><span><strong>具体考验的是产品经理产品规划能力、团队资源分工、以及整个产品所处行业背景的赛道，需要产品经理能够综合维度进行考量，同时对市面上的产品商业化有一定的认知。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>球：产品架构与产品定位</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>越到后面的面试，聚焦的内容几乎就和业务不沾边了,聚焦在产品经理的产品架构设计能力，<strong>比如：“面向产品不同用户，进行XX产品的架构设计”，简单陈述设计方法或架构。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>同时针对产品架构，给出当前产品架构的行业类别定位，比如我们如何判别SaaS和PaaS、私域流量和公域流量产品设计的区别。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这个环节难度最高，就考验就是是产品经理通识部分，以及过往履历积累后的经验沉淀了。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>3.高P产品经理的硬性条件</span></strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于腾讯大厂的高P产品经理职位，首先能够通过简历筛选参加社招，我认为已经是产品经理精英中的精英了，能够去社招面试高P，说明你至少达到了3个硬性条件</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>1.毕业时间5-10年</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>越是高P的产品经理，其要求的硬性毕业时间越长，同时产品经理的从业履历不能少于5年。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>2.极度对口的业务或产品</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>越是在腾讯大厂社招，就要求你进去做的产品要和你现在做的产品非常像，要么是直接竞品，要么就是间接可以借鉴参考跨行的竞品。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以如果你所在的公司所处的产品刚好也是腾讯打算做的，那么恭喜你，你就有更多机会通过面试。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>3.有大厂基础履历</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除了以上2点外，还有一点希望候选者也有其他大厂或同类型企业的从业经验。这个就相对比较严格了，所以想社招进入腾讯做高P，建议提前就去一线城市或二线城市，才会有进入大厂履历的机会,以此来曲线救国。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>4.高P的产品经理入职率不是那么高<br></span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>很多人以为拿到了腾讯高P的offer一定很开心吧，毕竟年薪百万，应该只要面试通过了，想都不用想就马上入职。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span></span><span>但实际上相比校园招聘或者刚毕业2-3年左右的社招，高级产品经理的入职率我认为不是那么高，主要原因有2点</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>1.不想出舒适圈</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>5-10年的产品经理生活圈已经定型,很多人可能已经拥有了自己的家庭，也已经定居在某一个城市,想放弃现在所拥有的的去大厂，仅仅是因为一个工作，还是成本过于高。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如我知道的很多高P的产品经理候选者不一定是在深圳或在北京，可能是在成都、杭州这类城市，逃出舒适区只是为钱不是每个人都愿意去的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>2.如今大厂的不确定性</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“钱虽然重要，但身体是革命的本钱”，这个阶段的产品经理放弃去腾讯也是因为力不从心。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>去大厂意味着体力拼搏。和每年校园招聘加入腾讯的同学不一样，5-10年的同学往往都30+，相比同部门里的都是20多左右的年轻人，难免会有劣势。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然工作氛围会更加扁平和年轻化，但就是拼不过的加班时长和体力，怕被现在大环境的不确定性因为低绩效裁掉。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以上就是我聊下自己过往的经历，希望对以后你打算去大厂做高P的产品经理有帮助。</span></p>
