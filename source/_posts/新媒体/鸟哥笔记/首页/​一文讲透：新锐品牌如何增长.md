<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">资本市场遇冷、流量红利消失、头部主播纷纷退场，<a href="https://www.niaogebiji.com/tag/%E6%96%B0%E9%94%90%E5%93%81%E7%89%8C" target="_blank">新锐品牌</a>如何继续保持增长？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">有的人说要加大内容投入、有的人说要加强精细化运营、有的人说要寻找新的流量红利......</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">都好像有道理，但这绝对不是简单复制一下就能解决的，因为增长是一个体系化的大工程。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">腾讯2022年最新发布的《新锐品牌增长研究报告》，体系化地探讨了这个课题。</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">一、新锐品牌增长的3个阶段</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">同样都是新锐品牌，但因为所处的发展阶段不一样，规模体量可能完全不在一个等级，对应增长的策略也就会不一样。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">以GMV为主视角，新锐品牌的发展大致可以分成3个阶段，分别是：孵化期、爆发期和成熟期。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/71370700262b2a47cd932a9.42957688.png" title="​一文讲透：新锐品牌如何增长" alt="​一文讲透：新锐品牌如何增长" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>孵化期-入圈</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">孵化期是品牌经历的第一阶段，此时品牌成立时间一般小于3年，通常处于单品类状态且尚未获得广泛关注，主 要依靠创新产品在小众细分市场中具备一定知名度。由于品牌积累有限，且通常面临营销费用不足的难题，如何 有效触达目标人群，快速融入客群所处圈层，从而俘获首批种子用户并使品牌存活是该阶段的当务之急。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">成功的消费型品牌在该阶段关注GMV快速起量，增长率均值超过300%;体验型品牌前期关注营销生态的孵化， 因此GMV增长起势相对缓慢，一般均值高于200%。目前在该阶段的消费型品牌代表有微笑趣泡、隅田川咖啡 等，体验型品牌包括魔镜、MOODY、珂拉琪等。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>爆发期-扩圈</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在王牌单品取得了可圈可点的市场表现后，新锐品牌将迎来业务增长的第二个时期 — — 爆发期 ，此时品牌通常已成立3-5年，在垂类圈层中拥有了较高的知名度与美誉度，并在种子用户群体中验证了产品与市场需求的匹配 程度以及品牌在目标群体中的竞争力。进入爆发期后，品牌则需要基于种子用户进行延展，触达到更多的目标人 群，持续在人群中提高渗透，从而实现“扩圈”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">处于扩圈阶段的新锐品牌的GMV规模将快速增长，由于基数增大，年均增长率有所放缓，但仍维持在较高水平。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">对消费型品牌而言，经过快速增长之后，拉新难度增大，此阶段增长起势趋缓，通常GMV增长率均值在 50-200%。而体验型品牌随着营销生态的建立，具备更强的业务增长支撑性，相较于消费型GMV增长率更高，均 值大约为100%-200%，典型品牌包括小仙炖、蕉内、观夏等。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>成熟期-固圈</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">扩圈成功后新锐品牌将迎来第三阶段 — — 成熟期 ，此时品牌运转通常超过5年，产品结构 、销售渠道等方面已相对成熟，也具备了精细化的运营能力，经营目标从爆品思维转向对长期价值的追求，品牌力逐步形成并超越产品 本身，成为企业的核心价值。如何以此为基础，提炼关键文化基因，形成品牌文化认同，并对已有用户展开长效运 营实现“固圈”是此阶段的关注重点。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">成熟期内的新锐品牌由于已逐步成为行业头部，GMV规模庞大，增长率将进一步放缓。消费型与体验型品牌前期 的GMV增长率差异至该阶段逐渐拉平，增长率均值低于50%。如国风美妆品牌花西子在成熟期基于前期消费互 动行为建立的用户档案，定期私聊用户，促进复购与品牌信任，实现GMV 50%左右的稳固增长。</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">二、新锐品牌增长策略拆解</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">整体而言，新锐品牌增长策略主要围绕4个方面进行，分别是人群（people）、产品（product）、传播（promotion）和渠道（place）。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">基于品牌所出的发展阶段不同，对应这4个方面的策略也就不一样。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/70862925162b2a47d134309.73535758.png" title="​一文讲透：新锐品牌如何增长" alt="​一文讲透：新锐品牌如何增长" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>孵化期-增长策略</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">对孵化期新锐品牌来说，围绕“入圈”主题快速明确品牌定位，收获初始优质流量是最主要的增长主题。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">由于起步阶段对市场认知有限，识别初始圈层是关键一步，因此需要精准识别并培育首批优质种子用户以支持 初代产品打磨。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在产品策略方面，确立差异化、高辨识度的定位是俘获种子用户的不二选择。通常消费型品牌聚 焦黄金爆品的打造，通过打磨产品本身来塑造核心价值点，而体验型品牌则更关注差异化体验的塑造，通过产品 叠加服务的方式提供优质的消费体验。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">传播方面，为快速切入目标圈层，品牌多选择调性相符的垂类达人布局内容种草，明确传达产品价值点，并通过 高效的转化闭环链路进行快速收割。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">针对渠道策略，消费型品牌多针对性布局可高效触达量级群体且具备转化 动能的流量红利渠道，尤其关注投放与转化效率;体验型品牌则倾向找准并培育适合发展的营销生态，稳步打通 消费者全生命周期运营链路。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>爆发期-增长策略</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">品牌爆发期围绕“扩圈”为增长主题，多渠道投放渗透，实现人群的扩展外延。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">此阶段品牌已积累了一定量级的种子用户，投入产出比或有放缓，因此需发力突破种子用户边界，通过消费行为 分析找准用户关键兴趣点，提高圈层渗透率。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在成功建立市场认知的基础上，品牌可通过产品矩阵拓展抢占消费 者更多钱包份额。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">传播策略方面，通过多平台投放、矩阵式种草的方式升级传播玩法，进一步扩大品牌声量。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">就渠道策略而言，消费型品牌通常积极打造渠道的全域延伸，采用线上线下多元渠道组合的方式，力争最大程度 辐射客群。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">而体验型品牌则积极摸索品牌专属的营销生态，打通链路的各个环节。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>成熟期-增长策略</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在当前的新消费市场环境中，单凭爆款很难将品牌之路走长走远。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">品牌建设从来没有捷径可走，只有持续不断的 投入和沟通，坚持长期主义，才能做到与消费者建立信任，并不断沉淀品牌资产，让新锐品牌在前进的道路中少 走弯路，走得长远。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">所以，对于成熟期新锐品牌来说，巩固品牌认知、精细化长效运营客户的“固圈”策略是保持品牌长青的关键增长主题:</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">此阶段企业致力于长足发展，力求最大化存量用户价值，寻求精细化运营客户的方法。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">品牌需要基于积累用户资 产库判断适用的人群，结合用户消费历史行为建设用户分级体系并制定用户分层管理方法论，提高运营效率。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在单一品类积攒足够的研发经验后，品牌亦可延展关联类别，促成连带销售，延长消费者的品牌消费时间; 同时建 立起较为完整的产品上新标准化流程，指导优质品类集群的搭建。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">为实现品牌形象打造，企业展开高声量、强曝 光的品牌广告投放，持续进军公众认知同时持续精耕私域，打造覆盖用户全生命周期的私域运营，收集消费反馈 并提供产品周边服务，通过服务提升用户尊贵感，巩固与用户群体的紧密情感联系。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">此外适度扩张渠道，广泛进驻大流量电商平台，推进线下渠道部署，根据各渠道表现进行资源分配优化，结合市场反馈进行选择性产品部署，实现布局效益最优。</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">三、新锐品牌增长驱动力</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">上面两部分，我们从新锐品牌的发展趋势出发，探究在当今去中心化、碎片化的流量环境下，新锐品牌在不同发展阶段的增长路径以及增长策略。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">那要实现上述增长策略，哪些企业关键能力可以驱动品牌增长呢？</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">腾讯提炼出了针对新锐品牌增长的四力方法论模型：</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/208008474162b2a47d51a0f1.44763980.png" title="​一文讲透：新锐品牌如何增长" alt="​一文讲透：新锐品牌如何增长" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">接下来，我们针对这“四力”，进行一一讲解。</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">“四力模型”之一：商品力</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">商品是品牌与消费者建立联系的第一道关卡，商品力直接决定了品牌在消费者心智中形成的第一印象，对于所有 新锐品牌来说是在各个发展阶段都需精细打磨的核心竞争力。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">商品力的考量维度可由以下三个方面组成:</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>1. 打磨品牌产品／体验的能力</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">持续优化差异化体验及产品性能是建立消费者心智偏好及所有客户关系运营的基础前提。新锐品牌需在产品上 新和业务发展的过程中不断收集户反馈，反复验证产品/体验的市场竞争力，并及时调整产品定位，确保价值点与 用户诉求的匹配。具体而言，品牌可从以下几个方面进行自测:</p>
<ul>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">商品是否占据了空白利基市场，并具有明显的、可被消费者识别的差异化特征;</p>
</li>
</ul>
<ul>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">产品/服务的价值点是否匹配目标消费者的需求，包括产品功效/性能需求、服务体验需求以及情感需求等;</p>
</li>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">企业是否有标准化的用户洞察、产品研发、产品上新的流程，支持品牌更好打磨商品并实现快速上新。</p>
</li>
</ul>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>2. 制定合理渠道与定价策略的能力</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">好的产品/体验要最终获得消费者认可需要通过制定匹配的渠道与价格策略。品牌可以通过以下维度进行能力评判:</p>
<ul>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">品牌是否针对渠道受众特点匹配了不同的商品，各渠道是否具有差异化的商品定位;</p>
</li>
</ul>
<ul>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">商品价格是否与用户感知价值相匹配，定价是否阻碍了转化;</p>
</li>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">不同渠道的价格策略是否有所区隔，形成互补关系。</p>
</li>
</ul>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>3. 搭建SKU／品类矩阵的能力</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在不同的渠道部署产品货架时，品牌需要代入消费者视角，以消费者可能产生的连带需求为出发点，围绕销量王 牌产品、考虑产品互补属性搭建产品矩阵，确保消费者在第一眼浏览时产生定位不清的混淆感，影响品牌认知的 形成及商品份额之间的互相蚕食; 品牌可以通过以下维度进行能力评判:</p>
<ul>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">不同类型商品货架占比是否合理，如爆款SKU占比、优惠商品占比，非主打类商品占比等;</p>
</li>
</ul>
<ul>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">不同品类和SKU产品的差异化定位在消费者心中是否足够清晰，选购时是否有选择困难;</p>
</li>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">商品组合是否满足渠道受众需求及潜在需求，并在消费者选购时有互补与搭售作用。</p>
</li>
</ul>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">“四力模型”之二：运营力</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">新锐品牌的运营力主要以用户为中心打造，并围绕用户的消费旅程贯穿全生命周期，经由初始的一系列营销活动 所触达的流量，投放举措下的消费转化，购后的留存与复购情况，以及口碑分享等全环节的运营能力。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/146292298262b2a47d7ca840.09029545.png" title="​一文讲透：新锐品牌如何增长" alt="​一文讲透：新锐品牌如何增长" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">一般而言，新锐品牌的运营力主要涵盖以下两个方面:</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>1. 不同渠道的流量获取和转化的能力 </strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">渠道作为品牌面向市场的通道，是营销系统的重要组成部分，对品牌快速获客并实现转化起到关键的助推作用。因此对于新锐品牌而言，发展前期的重点就是找准最优的获客渠道和最佳的传播方式。品牌可以从以下维度理解与评估该能力:</p>
<ul>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">不同渠道(如付费投放渠道、免费渠道、口碑渠道等)是否有机结合并能有效触达目标人群;</p>
</li>
</ul>
<ul>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">传播内容与素材是否迎合受众利益点并具有高传播性;</p>
</li>
</ul>
<ul>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">触达用户同时是否能引导形成闭环转化与用户驻留;</p>
</li>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">流量获取与转化过程中的投入是否满足低成本的要求。</p>
</li>
</ul>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>2. 私域沉淀的用户对品牌好感度和忠诚度培育的能力 </strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">新锐品牌由于其直达消费者的特性，可以大规模触达并深度运营消费者，维持用户的品牌粘性对于持续渗透并巩 固原有圈层至关重要，可成为品牌增长的杠杆。品牌可以从以下维度进行能力评判:</p>
<ul>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">品牌是否拥有可沉淀并在日后反复触达用户的平台;</p>
</li>
</ul>
<ul>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">品牌是否能保持与用户之间的定期互动;</p>
</li>
</ul>
<ul>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">既有运营中是否已形成维持客户驻留的有效手段;</p>
</li>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">私域运营中是否具备覆盖全链路的渠道通路。</p>
</li>
</ul>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">“四力模型”之三：组织力</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">新锐品牌的组织力帮助解决从顶层设计到组织架构，以及团队协同与资源整合等方面的问题，是强劲支撑品牌 建设的管理架构体系，主要具备以下特征:</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>1. 组织模式支持不同阶段的业务需求 </strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在市场迭代迅速的当下，新锐品牌的消费链路、运营模式均产生较大变化，增长打法也推陈出新，当品牌识别完 各个阶段的主要增长主题、策略以及目标后，需要企业架构与团队设置为品牌在用户洞察、关系打造、内容生产 等方面的业务运行提供保障。以下方向可作为品牌理解与评估该能力的维度:</p>
<ul>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">品牌是否理解不同组织模式的适用条件以及优劣势;</p>
</li>
</ul>
<ul>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">品牌是否能够精准判断业务需求与组织支持的匹配标准;</p>
</li>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">高层视角是否具备动态调整架构与团队的决断能力。</p>
</li>
</ul>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>2. 跨部门、职能高效协同 </strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">新锐品牌打破部门边界、去中心化的组织模式，即将团队按照品类/产品进行细分，形成各个独立运营的产品小 组，使得组织更贴近市场的同时匹配效率更优，跨职能的业务沟通更顺畅，员工积极性也有大幅提升。其中对组 织力影响较大的方面通常包括:</p>
<ul>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">现有资源是否能保障团队线上及线下的跨部门协作;</p>
</li>
</ul>
<ul>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">业务相关方的利益分配、考核激励等标准是否能够激励员工参与;</p>
</li>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">高层视角是否能够解决协同机制和利益分歧。</p>
</li>
</ul>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>3. 组织架构敏捷</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">由于新锐<a href="https://www.niaogebiji.com/tag/%E5%93%81%E7%89%8C%E5%8F%91%E5%B1%95" target="_blank">品牌发展</a>迅速，运营模式快速更迭的同时业务规模迅猛扩张，因此对组织架构的配套能力以及适应能 力提出较高挑战。随着新媒体营销方法论的逐渐成熟，市场上出现一大批精细化的专业代运营公司负责具体事 务，包括直播运营、效果广告投放、达人种草、私域门店运营等，而处于快速增长的新锐品牌，则善于通过外部合 作快速识别最佳增长方案，灵活调整资源布局。品牌可通过以下维度进行评估:</p>
<ul>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">品牌是否已经形成较为完善的业务合作的判断逻辑;</p>
</li>
</ul>
<ul>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">品牌是否搭建了清晰的外部引入机制及相应流程;</p>
</li>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">现有供应商的合作资源是否能够覆盖品牌主流业务的范畴。</p>
</li>
</ul>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">具体而言，新锐品牌由于所处不同阶段造成整体目标与增长策略的差异，从而导致组织力的模式也存在不同特点:</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>孵化期:扁平化兼职团队</strong> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">由于公司人员规模小，无需设置过于复杂的组织架构，通常建立扁平化小团队，从而加快工作效率以及服务响应 速度;基于有限的业务体量，岗位设置上通常较精简，通常一人身兼多职，负责与主岗位密切关联的前/后序业务 环节或能力要求接近的工作项。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">一般而言该阶段品牌会针对微信生态设置1~3人的小规模团队，人员兼岗负责多平台运营，多布局在熟悉产品及 服务内容的产品部门或了解媒体特性的运营部门。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>爆发期:小规模专属团队 </strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">成立职能化团队，基于业务职能进行架构细化，设立增长团队负责不同产品的增长目标，打破市场、销售、运营团 队隔阂，同时设置较为完备的支持性职能部门辅助业务推进;根据业务模块进行岗位细化拆分，以专人专岗为主， 各自负责细化领域的业务开展，除少量能力互通的领域或存在一岗多职的情况。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>成熟期:大规模专业团队 </strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">形成精细化团队，搭建小营销前台、大企业中台的共享模式，一般根据区域、行业等维度成立区域公司或事业部， 各自负责与客户接洽密切的营销业务，通过地域优势或行业沉淀，实现客户需求的高效响应;选择具备成本优势 的区域单独设立后续执行环节的业务中台，灵活承接区域/事业部的业务落地;岗位设置上高度细化，岗位与人 员一一对应。</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">“四力模型”之四：产品力</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">为了跟第一个“商品力”有效区分，这里讲的“产品力”，更多可以理解为是企业在做增长操盘过程中应用到的一些产品工具，所以也可以叫做是“工具力”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">孵化期: 善用数据分析工具，助力用户洞察与运营优化</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">由于孵化期阶段的品牌主要关注商品力打磨以及拉新获客，因此数字化产品力提升主要围绕用户画像识别与触 达转化的数据分析与优化，包括受众数据洞察、受众平台偏好、行业渠道变化、行业舆情分布、全KOL平台热门内 容标签、投放效果优化分析等方面。由于此阶段品牌数据资产较少且资源有限，主要依赖外部第三方提供数据分 析，或使用一些门槛较低、简易的自助分析工具。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">如数据工具如智慧零售“腾讯有数”，掌握私域各环节的流量获取能力与转化效率，“腾讯珠玑”帮助公众号内容生产与个性化分发能力提升，腾讯广告互选平台提供KOL种草转化数据等。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">爆发期:加入特色功能模块，打造差异化消费体验</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">随着自有数据的积累，品牌可以展开更多目标消费者的洞察，并应用于更多提升消费者体验的领域。不同行业的 新锐品牌的消费链路与体验需求各不相同，因此需要在常用消费链路与功能的基础之上，加入具有品牌特色的 差异化功能与服务，打造差异化优势，继而盘活营销生态用户留存。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">成熟期:全渠道数字化管理系统，精细用户运营</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">当企业进入成熟期之后，为实现更精细化的用户运营，需要企业建设如CDP(客户数据平台)，SCRM(社会化客户关系管理系统) 等大型营销系统，实现全域数据整合，以支持用户全生命周期的需求。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">针对进入成熟期的新锐品牌，腾讯可定制化提供商业分析中台解决方案，如腾讯营销云可定制化提供企业级SaaS 营销解决方案，帮助企业实现全场景获客、全生命周期线索管理、全旅程自动化营销、全链路数据洞察等目标。</p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">回顾新锐品牌的建设历程，可以看到品牌基于增长主题进行人群、产品、传播、渠道四大要素构成的增长策略部 署，以组织力、运营力、商品力、产品力作为发展的关键驱动能力项，其背后离不开外部产品的生态赋能支持。</p>
