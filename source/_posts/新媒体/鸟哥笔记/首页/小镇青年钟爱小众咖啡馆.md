<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>过去一年，尽管新消费行业有所降温，但咖啡赛道却热度不减。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>前有半年之内在国内开出两家门店的蓝瓶咖啡（</span><span> Blue Bottle Coffee</span><span>），后有2019年进入中国市场的加拿大<a href="https://www.niaogebiji.com/tag/%E5%92%96%E5%95%A1%E5%93%81%E7%89%8C" target="_blank">咖啡品牌</a>Tims中国进入上市倒计时。与此同时，中国邮政、餐饮集团狗不理、运动服装品牌李宁以及华为等，也纷纷跨界入局。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>入局者的增多，带来的最直观变化则是咖啡相关企业数量的增加。天眼查数据显示，“咖啡”相关企业年注册量持续增长，6成以上“咖啡”相关企业成立于5年内。2021年新增注册企业数量近2.6万家，年度注册增速为9.49%。截至目前，2022年新增咖啡相关注册企业1.9万家，同比增长26.6%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>值得关注的是，在逐年增加的咖啡企业中，一线城市以外的咖啡消费正在崛起。据美团美食发布的《2022中国现制咖啡品类发展报告》显示，新一线和三线城市的咖啡门店数量增速最高，接近19%。外卖数据显示，三四五线城市的咖啡订单量同比增长200%以上。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>来自小红书的数据显示，2021年有关“咖啡店创业”的搜索量同比2020年增长311%，其中35%来自三线及以下城市。平台上，关于“县城咖啡”的笔记已超1万篇。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span> </span><img src="https://ngbjimg.xy599.com/213461754563082de2f02af4.98802708.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图/小红书“县城咖啡”相关笔记（左）</span><span>小红书“咖啡创业”相关笔记（右）</span><span>来源/燃次元截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>资本也很快捕捉到了咖啡在下沉市场的快速崛起，地方性咖啡品牌、区域性咖啡品牌逐渐受到资本关注。公开资料显示，浙江宁波新锐咖啡品牌“歪咖啡Y COFFEE”先后于2021年12月和2022年6月完成两笔融资，融资金额达数千万元。2022年5月，诞生于海南的咖啡品牌“小洋侨”千万级融资，同月，起家于长沙的“RUU COFFEE”也获得了融资，融资金额未披露。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>歪咖啡Y COFFEE创始人曹玉志曾公开表示，一线城市咖啡品牌供给趋于饱和，相比之下新一线等更广泛的下沉市场，咖啡消费增长同样旺盛。据悉，歪咖啡Y COFFEE已经在宁波、舟山开出了近20家直营门店。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>艾媒咨询CEO、首席分析师张毅对燃次元表示，此前起家上海的精品咖啡馆受到了资本的青睐，但这类咖啡馆的兴起包含了上海这座城市对咖啡的赤忱，资本助推其快速扩张的同时也面临着巨大的风险。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“反观区域性咖啡连锁品牌，其产品能力、管理能力、业绩以及连锁化能力在本地发展过程中已经得到了初步验证，对于资本来说更加稳健。尤其是考虑到未来上市的可能性，资本会更关注品牌的营收和利润的增长。”张毅强调。</span></p>

data-copy-origin="https://shimo.im"><strong>小镇青年偏爱独立咖啡馆</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>前瞻产业研究院发布的《2022年中国咖啡行业市场现状及发展趋势分析》指出，随着我国经济的发展，三四线城市的消费水平在不断提高，咖啡馆数量也逐渐多了起来，未来咖啡市场将逐渐向三四线城市转移。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如上述报告所说，在沈阳，有一条聚集了100多家特色咖啡店的咖啡小巷——八经街，是当地文艺青年们钟爱的打卡胜地之一。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“我非常喜欢和朋友来这喝咖啡，我觉得这里的咖啡和那些大的连锁咖啡店不是一个味儿，每家店都有自己的特色。”留学归来的90后女生Amy一天至少要喝上两杯咖啡。回国前还在担心没有好喝的咖啡的她，一回到沈阳就被这条飘着咖啡香的巷子所吸引。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“这里的咖啡馆风格非常多元化，其中不乏日式风、韩风，亦或是将画廊与咖啡结合等极具特色的门店。”Amy对燃次元表示，自己回国后的第一批朋友便是在这条巷子里结交的，“工作日到这里来的大多是工作或生活在附近的年轻人，大家更容易建立联结。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>很快巷子里的两家咖啡馆成了Amy和朋友们的“聚点”，“跟老板成为朋友学习咖啡制作，帮客人策划求婚，这里更像是个磁场强烈的社交场，这是大型连锁咖啡馆所不具备的属性。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>阿星的咖啡馆便开在了这条街上。因此，他也见证了这条小巷从四五家咖啡馆到现在超百家咖啡馆的变迁，“在人们之前的认知中，开咖啡店是九死一生的生意。但近两年市场变好了，开店成功率也随即比以前高了。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>阿星表示，以前开店更多的是情怀支撑，现在大部分是盈利的，虽然有的门店利润微薄，但生存已经完全没有问题。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>阿星向燃次元介绍，在沈阳，星巴克体量最大，瑞幸深受喜爱，但在其他城市占一定比重的便利店咖啡在沈阳则几乎隐形，“或许是因为在小镇青年的认知中，购买便利店咖啡不如直接选择雀巢罐装吧。所以本土品牌差异化生存法则就是：和星巴克比价格，和瑞幸比环境，和KFC等比品质。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“最后就形成了一个很有趣的现象：独立咖啡馆成了互不相似的‘亲兄妹’。各家风格大相径庭，但开店运营逻辑却如出一辙，即，文艺化网红化路线。通过探店打卡等方式提高店铺曝光率，通过社交化运作增加产品附加值，提升溢价空间。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>阿星分析道，这是由消费者的消费习惯决定的。除了少数追求咖啡豆不同风味和口味的咖啡“死忠粉”外，大部分消费者，对是商业豆还是精品豆没有那么在意，“咖啡‘好入口’的前提下，他们更愿意为咖啡馆的‘颜值’买单。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>来自江苏淮安的90后全职妈妈大喜，几乎每天都会约朋友到咖啡馆“下午茶”，而她选择咖啡馆的主要标准，就是“颜值高”，以及有打卡的“噱头”，“漂亮的装潢设计、特色摆设、好看的咖啡点心、好玩的标语，以及大大的落地窗……这些都会成为我打卡的理由。”</span></p>

data-copy-origin="https://shimo.im"><span><strong>区域咖啡馆成资本“新宠”</strong></span></span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“咖啡做的是年轻人的生意，而年轻人是各行各业都在抢夺的人群，现在很多行业都在跨界做咖啡，咖啡也在融入更多的业态，市场自然越来越热。”阿星对燃次元表示，在一线城市咖啡品牌供给趋于饱和，三四线乃至县城小镇的咖啡消费，正在迎来爆发式增长的大背景下，立足本地咖啡市场的创业者自然迎来了新机遇。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“只要产品品质达标，门店有风格，再花上一段时间沉淀一部分忠实顾客，店铺拓展到3家以上，基本上就能在当地站住脚。”阿星表示。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>基于此，下沉市场成为众多咖啡玩家的新战场。一方面，知名连锁咖啡品牌快速开店，在下沉市场攻城略地。另一方面，区域性咖啡品牌如雨后春笋般快速增长。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>相较于品牌已经足够成熟且上升空间或已触达天花板的连锁咖啡店来说，区域性咖啡品牌在这一背景下，更受资本关注。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据燃次元不完全统计，2022年第二季度，围绕咖啡品牌、咖啡供应链、上游咖啡原料进行的投融资案不少于12起。具体来看，相较于全国性品牌，区域品牌、咖啡上下游供应商、餐饮数字化服务商、智能设备商等成为了资本更为关注的方向。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>事实上，从去年下半年开始，宁波的歪咖啡，绍兴的Coco.Juliet，长沙的DOC和RUU，东莞的FELICITY ORIGIN，海南的小洋侨，苏州的比星咖啡，广州的.jpg咖啡等地域性咖啡品牌均先后获得融资。其中，4月，FELICITY ORIGIN完成1000万元人民币A轮融资，投资方为福沃基金；7月，比星咖啡完成数千万元天使轮融资，投资方为顺为资本。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>易咖CEO王竞对燃次元表示，中国的咖啡市场正处于增量阶段，未来融资额度会更大。在Manner等头部的品牌逐渐完成融资，同时又有更多的资本想进入的时候，咖啡的上下游，包括供应链企业、智能咖啡机和速溶咖啡品牌等，各个细分赛道逐渐都会获得资本关注，同时也会迎来更多的创业者。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>与此同时，咖啡上下游企业也颇受资本青睐。今年5月，精品咖啡供应链公司乐饮创新获得A轮近5000千万元融资，其主要业务包括为永璞、鹰集等精品速溶咖啡品牌提供设备和工艺；为.jpg、TAG、永夏咖啡等品牌提供闪萃工艺及标准化规模生产设备。6月，智能咖啡机品牌咖啡之翼获得约1亿元人民币战略融资。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然而并非所有经营业绩好的地方品牌都能受到资本的青睐。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>阿星对燃次元表示，沈阳本土经营较好的品牌往往都有一个共性，即，主理人个人风格明确又独特，“品牌与主理人捆绑，品牌人格化后，提供了让顾客心仪的社交场的同时，也保证了一定的流量。不过弊端也很明显，这类品牌的可复制性非常弱。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在燃次元此前发布的《年轻人不喝瑞幸，还能喝什么？》一文中，消费领域资深投资经理王鑫曾对燃次元表示，投资方更看重的是门店是否能够快速复制拓展、盈利能力、产品差异化、供应链优势等条件，具备快速连锁化能力的品牌更受资本青睐。</span></p>

data-copy-origin="https://shimo.im"><span><strong>市场很大，未来在哪？</strong></span></span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>艾媒咨询数据显示，2021年中国咖啡市场规模约3817亿元，已进入高速发展阶段。预计行业将保持27.2%的增长率上升，到2025年中国咖啡市场规模将达1万亿元。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另据伦敦国际咖啡组织统计，中国咖啡消费每年以15-20%的速度增长，远高于全球2%的均速，已成为世界第一大咖啡消费市场。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这也让当下中国咖啡市场处于群雄逐鹿，硝烟四起的状态中。既有星巴克、COSTA、Peet’s Coffee皮爷咖啡、Tims咖啡等国际连锁咖啡巨头持续加码，又有Manner Coffee、M Stand等精品咖啡“新贵”在资本的助力下快速扩张，还有歪咖啡、 DOC咖啡等区域咖啡“新秀”从地方向全国市场进军，更不用提以瑞幸为代表的“新零售”咖啡跑通了“自提+外卖”的本土消费模式，以三顿半、隅田川、永璞等为代表的线上精品冻干咖啡品牌抢夺用户家庭消费场景。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除此之外，蓬勃增长的咖啡赛道还引来了不少跨界选手。继中国石化、中国邮政、同仁堂等品牌跨界卖咖啡之后，运动服装品牌李宁也于近日注册了“宁咖啡NING COFFEE”商标，正式加入咖啡行业赛道。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>天眼查数据显示，截至目前，我国共有16.4万家企业名称或经营范围含“咖啡”且状态为在业、存续、迁入、迁出的“咖啡”相关企业。其中7成以上相关企业为个体工商户，近3成为有限责任公司，注册资本在100万元内的企业占比超8成。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>万亿赛道，高速发展，让国内咖啡市场充满了“想象力”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>王鑫对燃次元表示，仅就现制咖啡赛道而言，目前品牌连锁率和头部集中度较低，市场依旧处于发展初期，这也为现制咖啡的“新势力”们提供了宝贵的机遇，可以通过差异化的品类和品牌定位争取市场份额。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>王竞也表示，无论咖啡馆、咖啡机，还是冻干咖啡等目前在国内都是增量市场，虽增速不同但这几个细分赛道彼此互补。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“中国的咖啡市场目前尚处于从礼品市场向商务市场转变的过程当中，一方面咖啡馆是目前为止普通消费者最容易接受的咖啡消费形式，另一方面无论是北上广深这样的一线城市还是二三线城市，咖啡馆的普及率和覆盖率仍然有非常大的上升空间，所以这个时间点咖啡馆是最容易融资的。”但与此同时，王竞强调，随着时间的推移，咖啡馆也会遇到发展的瓶颈期。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“对于一座城市来说，咖啡馆门店的拓展数量是有上限的，但一座城市的家庭或者消费者，对咖啡机的需求却有着更大的想象空间。同时，随着消费者消费习惯的养生，当咖啡馆对于消费者的市场教育达到一定程度的时候，人手或者每个家庭都拥有一台咖啡机或许就成了新的市场需求。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>王竞表示，现阶段的国内咖啡市场仿佛身处战国时代，群雄割据。“但随着时间的推移，一定会出现一个国产品牌，来一统江湖。这个品牌可能是现在市面上已经有的，也可能是还没诞生的，但一定是一个符合中国人消费习惯的品牌。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>国内消费者的咖啡习惯和国外不一样，口味喜好也不一样。国内的咖啡市场到底会变成什么样，现阶段没人可以判断。对于任何一位已经入局或者想要入局咖啡赛道的玩家来说，只能慢慢推进，才能有更好的应对方法，做出更符合国人消费习惯的咖啡品牌。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过王竞也直言，这或许要等到十年或二十年甚至更久之后才会有结论。</span></p>
