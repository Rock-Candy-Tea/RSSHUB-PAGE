<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最近，抖音出了一个海克斯科技博主，江湖人称“绝命毒师、炼金术士”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>8月13日开始，抖音博主“辛吉飞”20天涨粉超450万，还引起了网友、官媒、餐饮从业者间的一场“舆论乱战”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><a data-miniprogram-appid="wx55778a0d7b3048c4" data-miniprogram-path="pages/NewLive/index" data-miniprogram-nickname="新抖数据" data-miniprogram-type="image" data-miniprogram-servicetype href="https://www.niaogebiji.com/undefined"><img src="https://ngbjimg.xy599.com/4871860546322c412626247.09146631.jpeg" referrerpolicy="no-referrer"></a></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：新抖</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>海克斯科技原本出自游戏《英雄联盟》，指的是一种魔法与科技融合的幻想技术，在“辛吉飞”这里，却成了一种“科技美食”的制作方法。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>视频中，“辛吉飞”向网友系统展示了牛排、燕窝、奶茶等常见食品的制作过程。因为制作过程与化学实验太过相似，“全是科技与狠活”，偏偏不论是制作方法还是食品添加剂的使用都是合法合规的，这给观看视频的网友们造成了极大冲击。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/6905983546322c4128ba233.59070107.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：抖音</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>很快，围绕“辛吉飞”的各种梗开始爆红抖音，诸多网友和博主也参与到了相关讨论中。<strong>截至发稿前，抖音话题#海克斯科技##三花淡奶##科技与狠活#的累计播放次数超7.5亿。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“辛吉飞”是怎么火的？今天我们就来认识下这位舆论中心的海克斯科技博主。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/996074115601215dbd1d874.12950565.png" referrerpolicy="no-referrer"></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong>从烤面筋摊主到海克斯科技博主</strong></span><span></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>回看“辛吉飞”的抖音视频，“辛吉飞”最初是东北沈阳的一位烤面筋摊主。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2019年3月23日，“辛吉飞”在抖音发布了第一条视频，演示了烤面筋切花器的使用方法。此后，他还陆续展示了土耳其烤肉、芝士热狗棒、韩式炸鸡等小吃的制作过程。这个阶段的“辛吉飞”更像是一个小吃从业者的日常分享账号，单条视频获赞很少过千。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/19729320226322c412d0a475.45488667.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：抖音</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2021年4月12日，“辛吉飞”发视频宣布，干了5年的烤面筋生意干黄了，“五年啥也没剩，干出来一堆破烂”。两个月后，“辛吉飞”转换身份，成了一位调料厂家，开始详细分享东北煎焖子、轰炸大鱿鱼等路边摊小吃的制作方法。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这个阶段的“辛吉飞”像是一位美食博主，还有网友请他分享特定小吃的制作方法。其中，烤面筋的制作方法视频成为“辛吉飞”最大的爆款，截至目前获赞超40万。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2919711426322c4131e0ba2.96493683.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：抖音</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>或许是因为天天混迹在食品添加剂市场，像网友调侃的一样“完成了前置任务，开始转职炼金术士”，“辛吉飞”分享的内容逐渐从路边摊小吃变成了“科技美食”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从“辛吉飞”这里，网友们知道了不少“科技美食”的猫腻。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为什么想在自助烤肉店吃回本是做梦？“辛吉飞”说可能是因为商家用了进口冻肉，而南美巴西的进口冻牛肉是20-28元/斤，澳洲新西兰的进口冻牛肉是30元/斤。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/206779836322c41370b945.62590682.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：抖音</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为什么一些餐馆的汤显得特别白？“辛吉飞”说可能是因为商家用了三花淡奶、植脂末等食品添加剂调色，而不是像宣传的那样熬了一晚上。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为什么一些直播间的即食燕窝能卖29.9元的超低价？“辛吉飞”说可能是因为商家一碗燕窝的成本只包括6毛的明胶片、4毛的糖浆、5毛的包装。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此外，炸鸡排的肌肉纤维可能是靠大豆拉丝蛋白，炸虾仁用的可能不是真虾，而是面粉、色素、虾肉香精，冰皮榴莲月饼可能也没什么榴莲……</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2022年7月25日，“辛吉飞”在视频中介绍了某种“黑糖珍珠奶茶”的制作做法：<strong>奶茶里没有奶，而是植脂末和奶香精；茶叶也不是好茶叶，而是茶叶沫子。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>视频下方，一位自称是奶茶店前员工的网友表示这是“非常标准的做法”，还有一位自称前奶茶店主的网友表示，自己用的是好茶叶、新鲜水果和黑白淡奶，但即使用果糖代替蜂蜜也压不下成本，最后只好关门。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/6922618866322c4139fccd1.57826575.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：抖音</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这一波系列内容下来，再配合“辛吉飞”眼歪嘴斜、吃完“科技美食”就要上厕所的夸张演绎，让网友们大为震撼。一位网友留言：“<strong>飞哥一段短视频，比唠叨了我二十年的妈妈还管用，让我戒掉了路边摊小吃。”</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2068046273601215dcef8457.71452222.png" referrerpolicy="no-referrer"></p>
<h1 style="background-image: url('https://qnssl.niaogebiji.com//48aedb8880cab8c45637abc7493ecddd.png'); font-size: 20px; color: #212629; text-align: center; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 16px 20px; background-position: 0px 5px; background-repeat: no-repeat;"><span><strong>一勺三花淡奶</strong></span><span><strong>炸出了餐饮行业的“舆论乱战”</strong></span><span></span></h1>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>事情到这里，“辛吉飞”还仅仅是一位科普方式比较特别的博主。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>转折源于以“中国食品报融媒体”为代表的一批账号对“辛吉飞”提出质疑。批评者认为，“辛吉飞”存在三大问题：<strong>宣扬低价有罪论、贩卖焦虑、传播犯罪方法。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/10355490646322c413d37e08.28501607.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>“中国食品报融媒体”连续发布了两条对“辛吉飞”的批评视频，其中第一条已经删除。图源：抖音</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然而，这些声音反而激发了部分吃瓜群众的逆反心理和战斗热情。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从三鹿奶粉这样轰动全国的食品安全事件，到评论区一个个网友的现身说法，让不少网友相信食品安全问题普遍存在，相信“辛吉飞”说的才是真相。同时，由于“中国食品报融媒体”主理人经营着一个预制菜品牌“三餐食客”，也让部分网友对其观点的客观性产生了质疑。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/15307152996322c414182a55.77604822.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：抖音</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>关于低价有罪论，“辛吉飞”的支持者多次强调反感的不是科技与狠活，而是商家把科技产品当真材实料卖。玩梗过程中，网友更多是提出问题：<strong>肉为什么又便宜又滑嫩？汤为什么又白又香？有科技与狠活吗？</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>关于贩卖焦虑，有网友吐槽，辛吉飞“贩卖焦虑”的方式不过是把符合国标的商品又做了一遍。如果合法合规的东西被人知道了是贩卖焦虑，那有问题的是谁？</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/21150284156322c4145aff35.11176697.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：抖音</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>关于传授犯罪方法，认证为律师的“柴嵩 | 广州刑事律师”反驳道：“如果按照这个理解，中国裁判文书网就是中国最大的教授犯罪的网站。”</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“辛吉飞”和其支持者有两个核心观点：<strong>第一，符合国家标准的不一定是健康的；第二，一些无良商家在用科技产品冒充真材实料。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“辛吉飞”曾在直播中举例说，消费者想象中的蜂蜜可能是蜜蜂在一个山清水秀的地方采花蜜采出来的，但实际上，现在的蜂蜜可能是蜜蜂被蜂农喂了糖产出来的，是科技蜜，是蜂制品。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“中国食品报融媒体”曾在视频中提到，我国蜂蜜的平均合格率为98.2%，但在“辛吉飞”支持者看来，检测合格并不能证明蜂蜜的品质。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/18953533216322c414a9ab94.28673232.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>“中国食品报融媒体”分享的数据。图源：抖音</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>与此同时，更多业内人士也相继加入战场。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有人认为“辛吉飞”是在误导消费者，砸餐饮人的饭碗。“西餐纸老虎（供应链链）”发视频称，三花淡奶就是脱脂浓缩牛奶。评论区有人补充，三花淡奶是很平常的食品添加剂，主要作用是调色。还有人称这是“反智主义”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有人认为“辛吉飞”是在帮助消费者辨别餐饮和食品的真假。“研发主任破解配方”发视频称，这会是对行业的一次洗牌，更有利于用真材实料做餐饮的商家。不少食品商家也在趁机推广自家产品。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有人干脆向“辛吉飞”学习，感叹“各个都使活，我也不搂着了”，爆出更多行业内幕。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>生鲜商家“小李的牛排课”解释了为什么一些消费者能买到超低价牛羊肉。因为一些原本要流到加工厂做罐头、猫粮狗粮的110元/吨的临期牛羊肉，现在流到了直播间。“过期了一年半年你根本看不出来。”</span><span></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“辛吉飞”为什么突然火了？</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从北京的纸箱包子（假），到三鹿的三聚氰胺（真），受各种真真假假的媒体报道，大众对食品安全一直是存有疑虑但不清楚细节。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“辛吉飞”特别的地方在于，<strong>第一次用视频形式，成系列地把各种“科技美食”的幕后制作过程展示了出来。</strong>看完这些视频，网友不禁嘀咕：合法合规的都这么刺激，那不合法不合规的呢？</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><img src="https://ngbjimg.xy599.com/14633034996322c415c80d01.85028211.jpeg" referrerpolicy="no-referrer"></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不可否认，“辛吉飞”的确蹭上了网友对食品安全的焦虑，随着海克斯科技的玩梗增多，不少网友也有点反应过激，觉得什么都是“科技与狠活”。但究其本质，这波流量的来源还是网友对食品安全的不信任。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“中国食品安全报融媒体“分析，网友觉得假冒伪劣产品更多，第一是网友更容易记住假货的新闻报道，第二是食品监管天然存在延后性，监管部门没办法保证所有上市的食品都绝对安全，只能事后处罚。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>直播带货爆火以来，主播们更是屡屡打破“一分价钱一分货”的大众共识，反常识的事情需要理由。主播说是因为优化供应链、减少中间商，但网友也会怀疑是因为偷工减料、滥用添加剂。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/14294521286322c41604add2.97435214.png" referrerpolicy="no-referrer"></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>网友对“科技美食”的愤怒未必全部正确。关于“白汤更有营养”的常识，包括“老爸评测”“中国食品安全网”等账号均表示白汤的脂肪含量极高；关于妖魔化食品添加剂的相关言论，江南大学食品学院教授“食品专业华霄教授”表示<strong>“没有添加剂，就没有食品工业”。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1551316556322c4166611c9.42337508.jpeg" referrerpolicy="no-referrer"></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>近日，抖音还在“辛吉飞”视频下方增加了<strong>“科学看待食品添加剂”</strong>的官方提示。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于“辛吉飞”的内容科学性，本文不做过多评判，但某种意义上讲，“辛吉飞”们的确让观众有了更多判断和选择的视角。很多时候，大众焦虑的不是信息的负面，而是信息的偏颇，<strong>只有充分知情，同时有基于充分知情的自由选择，大众对食品安全的不安才能得到缓解。</strong></span><span></span></p>
