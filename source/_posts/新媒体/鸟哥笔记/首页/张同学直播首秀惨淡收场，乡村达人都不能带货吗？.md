<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">网红的尽头是<a href="https://www.niaogebiji.com/tag/%E7%9B%B4%E6%92%AD%E5%B8%A6%E8%B4%A7" target="_blank">直播带货</a>，张同学也不能免俗地踏入了这条河流。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>走红初期，张同学在接受采访时曾表示，后续会考虑帮村民卖家乡特产。这个承诺在6月25日兑现，张同学以单场GMV325.8万的成绩开启了主播之路。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>从数据来看，对应其粉丝量，张同学的带货成绩算不上亮眼。但从带货品类来看，农产品能够取得这样的成绩，也属实不易。</span></strong><span>毕竟，同样主卖农产品，俞敏洪首播的GMV约481万元，和张同学大致在同一水平区间。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>乡村达人是否只能售卖农产品？如果不是，他们还能往何处发力？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/13363685916010dc4c78b4e6.24352516.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong>325万，什么水平？</strong></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“张同学”真名张凯，辽宁营口大石桥市松树村人，去年11月因记录农村日常生活走红，曾经在抖音单月涨粉超千万，位列涨粉榜第一。截至发稿，其累计粉丝量达到1898.3万。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>去年11月，以勾勒乡村生活图景为内容特色的张同学一夜之间走红网络。除了在抖音创下单月增粉1034.5W的惊人成绩外，更在多个社交平台中掀起了现象级讨论。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/105713742462cc1356c14a76.32797171.jpeg" referrerpolicy="no-referrer"></span></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>@张同学抖音视频截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今年1月22日，张同学进行了直播首秀。开播仅3分钟，直播间就涌进10万人观看。第三方数据平台显示，这场直播总观看人次为676.4万，最高同时在线人数达到26.6万人，涨粉5.6万人。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在这场1小时35分的直播中，张同学对成立农业公司、签约专业MCN机构等传闻做出回应。此前有人质疑张同学签约费高达500万，他坦然表示，不止500万，甚至有的开出1000万、2000万签约费。但他称自己没有签约专业MCN机构，原因是不愿意被束缚，更喜欢随性一点。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>谈及自己对接广告、带货等问题的看法，他表示：<strong>“为了生活接了些广告，属于正常，我不跑偏就行了”，还强调：“暂时不会带货”。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>乡村赛道玩家众多，用户不可避免地会出现审美疲劳。毕竟@张同学能够爆火，和他颇具创意的技术流手法密不可分。但即使是张同学，也没有随时把握住观众的审美变化。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据第三方数据统计，今年以来，张同学的粉丝规模虽很稳定，<strong>但其单条视频的流量数据却持续下滑，已不及之前的十分之一。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/146167675962cc135706c7f3.39748472.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span> 图源：洞见数据研究院</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在其热度下滑的当下，6月25日，张同学进行了直播带货首秀。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作为三农领域的头部达人，<span>张同学主推</span><span>和自身定</span><span>位</span><span>相符的家乡特产，比如<span>大米、海参、裙带菜等</span>。<span></span><span>平均客</span><span>单价为39.98元，属于</span><span>较</span><span>低水</span><span>平</span><span>。</span><span>其中最畅销的商品是单价39.9元的10斤装东北大米，销售额达到73.5万元。</span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在技巧和话术上，张同学带货时表现明显生疏，这和其不适应高速运转的直播间节奏有关。不过</span><span>在长期掉粉的前提下，当天其抖音账号还净增4.3万粉丝，这一定程度上表明，粉丝们并非全然排斥直播带货，用合适的表达形式也能够获得大家的支持。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/165157235662cc135740d686.60289313.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>张同学抖音直播带货截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>同样是首秀，张同学这场带货直播累计观看人数达到839.2万，最高同时在线人数为20.8万，单场涨粉4.3万。</span><strong><span>流量规模上，和第一次聊天直播时旗鼓相当。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span><span>仅从销量表现而言，张同学带货首秀的表现一般。卡思认为，</span><strong>强IP不代表带货能力一样强。</strong><span>带货技巧、控场能力、货品和粉丝画像关联度等，都很重要。</span></span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>观察张同学的直播数据，其</span><span>视频和直播观众画像一致，均以31-40岁男性观众为主，直播观众中男性观众占比甚至达到60%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>众所周知，直播电商行业中，主力消费者是女性，<strong>也许是中年男性粉丝没有足够的购买兴趣，因此没能撑高张同学的GMV。</strong></span><strong>数据显示，这场直播的平均UV价值仅为0.39元，低于行业平均水平；带货转化率为0.97%，同样低于行业平均水平。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/13363685916010dc4c78b4e6.24352516.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>乡村达人，到底该怎么变现</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从2020年下半年迅速崛起到现在，乡村赛道的热闹和繁华我们有目共睹。但在巨大的流量背后，暗流早已涌动。最直观的，便是乡村达人的粉丝增速整体放缓，且已经有个别乡村达人出现掉粉现象。（掉粉、变现难：乡村网红的路不好走了？）</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>真实感和淳朴感是广大看客对乡村题材内容的最大诉求</span></strong><span>。用户既希望他们的内容和人设是真实而不做作的，又希望账号能够保持朴实和纯粹。一旦这两个要素被打破时，达人很容易受到反噬。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>@牛爱芳的小春花就是一个典型的例子。（一场直播掉粉百万：被“杀死”的农村网红）</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>去年10月30日，@牛爱芳的小春花开启了直播带货首秀。直播过后，负面评论此起彼伏，账号也随之开始掉粉。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>卡思曾分析其口碑翻车的原因，除了“打脸”此前的不带货承诺、且直播间商品比其他品牌更贵外，更重要的是被质疑“人设造假”，不仅伪造身份，很多情节都是预设的剧本。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不论真相如何，该账号已然遭受了严重的信任危机。据观测，这场直播过后到发稿前的4个多月内，@牛爱芳的小春花只进行了两场带货直播，销量都不尽人意。而翻看账号近期视频的评论区，画风仍是一片质疑。尤其是面对小春花的怀孕生子过程，不少用户争相指出疑点，认为这又是一场新的戏码。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>截至今年7月7日，@牛爱芳的小春花 抖音粉丝量已降至1705.3万，相比半年前掉粉超过100万。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>目前不断掉粉的张同学面临的困境和牛爱芳有相似之处。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>首先，虽然很多粉丝愿意支持有才华的达人变现，比如在@</span><span>彭传明的评论中，就有不少粉丝在“催促”他变现，并表示一定会为此买单，</span><strong><span>但有些达人迈出变现这一步时，仍会招来诸多质疑和不满，账号也有“变味”风险。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>第二，受账号自身定位的影响，在变现时受到的局限也会相对较多</span></strong><span>。<strong>进行直播带货时，选品眼光和供应链能力尤为重要。</strong></span><span>即使选择了相对“安全”的农产品，调性也和自身账号标签一致，但张同学本次带货成绩依然不算出色，也许和粉丝并未接受其身份转变有一定的关系。</span><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span></span><strong><span>此外，由于复杂规则和高密度竞争，抖音电商已然是一个高门槛的行业。在此进行直播带货，要想冲击更高的成绩，离不开专业的团队。</span></strong><span></span><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>背靠乡村土壤的达人们，难道就注定在风评和变现之间艰难摇摆吗？其实不然。</span></span><span><span> </span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><strong><span>一些乡村达人已通过橱窗带货获得了不错的销量，而他们的产品都与账号特性有着极深的关联度。</span><span>@乡愁 在橱窗中售卖起了多种茶叶，销量最高的一款桂花红茶已售出9.1W；@念乡人周周则在橱窗中售卖自有品牌火锅底料，目前显示的销量也还算可观。</span></strong></span></span><span><span> </span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>而在直播带货领域，也有表现不错的乡村博主值得借鉴。</span></span><span><span> </span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>比如，@麦小登<span>依靠分享生活Vlog走红，</span>如今在抖音吸粉1212.4万<span>，</span>近30天直播7场，预估销售额达到2384.4万。在今年5月-6月期间<span>，她也屡登带货日榜TOP3，6月29日的单场GMV超过1500万。<span></span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span>麦小登能够收获这么亮眼的成绩<span>，离不开其选品结构的多元化<span>。</span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span><span>麦小登的视频观众画像显示，80%以上为女性，且24-30岁的人群占比较高。配合其观众特点，直播间也选择了多元化的货品结构。近30天，</span></span></span></span><span>麦小登直播间销量最高的品类依次</span><span>是日用百货、美妆护肤、厨卫家电、食品饮料和运动户外</span><span>。其销售额最高的4个单品都属于美妆品类，客单价都在139元以上。</span><span></span><span><span><span><span><strong> </strong></span></span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><span><span><span><strong>多元的货品结构帮助麦小登拉高了GMV天花板，也</strong></span></span></span></span><strong>侧面证明了乡村达人并不只能带农产品，而是要根据粉丝画像安排和调整货盘结构。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在大环境日趋严峻的当下，乡村达人的变现之路依然崎岖。</span><span>想要变现，沉淀的时间需要更长。</span><span>只有当内容、IP形成足够的认知度和信任度、达人做好充足的准备之时，变现才显得水到渠成、粉丝的情感也不会受到太大的伤害。</span></p>
