<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>随着暑假的到来，吹着空调，吃着冰镇西瓜，惬意地躺在沙发上打着游戏，相信是很多学生党过暑假的常态。学生党们暑假大量的休闲时间，自然成为品牌抢夺的焦点。这其中，绝对少不了各大游戏品牌的厮杀。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>每年夏天如约而至的“开心消除节”，则是开心消消乐为暑期玩家们打造的一场游戏狂欢。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今年的“开心消除节”，开心消消乐在营销上可谓大手笔，不仅和敦煌博物馆跨界合作，用敦煌的千年文化为品牌赋能，更是上线了一支全新CG大片，让我们看到了开心消消乐恢弘大气、仙气飘飘的一面，重塑了在年轻用户心目中的品牌形象。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><img src="https://ngbjimg.xy599.com/5611320262b974392829e0.57743105.png" referrerpolicy="no-referrer"></strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>游戏萌宠<strong><span>梦幻联动</span></strong>敦煌IP，</span></strong></span><span><strong><span>讲述穿越千年的美丽神话</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>近几年，随着国潮营销的兴起，神秘的敦煌文化不断地走入大众视野，焕发全新的生命力。敦煌壁画、飞天仙女、反弹琵琶、九色鹿等都成为人们耳熟能详的敦煌文化标志符号。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于开心消消乐来说，拥有8亿用户且几乎覆盖所有年龄圈层的影响力，已经足以说明其<a href="https://www.niaogebiji.com/tag/%E5%9B%BD%E6%B0%91%E6%B8%B8%E6%88%8F" target="_blank">国民游戏</a>的地位。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一个是拥有千年历史的文明瑰宝，一个是发行8年的休闲游戏，两个看似毫不相关的IP，却因为都有庞大的群众基础，让它们的联名备受期待和关注。而开心消消乐的这支全新CG大片，也没有让用户的期待落空，反而让我们看到了两个IP相得益彰的融合力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>开心消消乐和敦煌博物馆一起，给我们讲了一个美丽的神话故事。</span></strong></span><span>曾经，西域敦煌藏着一件名为“光之琵琶”的神物，它由文明之光凝聚而成。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/152646064762d7c6a658d4c8.94289493.gif" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但是随着文明失传，神物琵琶也失去了神力，变得黯淡无光。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>于是，莫高窟的九色鹿作为使者，找到了瑶池仙鹤力荐的修复神器之人——开心消消乐的小熊、小鸡、青蛙、狐狸、猫头鹰、河马、魔力鸟等小动物们。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/78442817662d7c6acd3faa0.07927836.gif" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>小动物们千里迢迢到达敦煌，它们找到了隐藏在画、舞、乐、史之中的敦煌之光，帮助“光之琵琶”重新焕发了光芒。“光之琵琶”还出现在了“开心消除节”，奏响了跨时代的华夏文明之音。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/42218510562d7c6b250dee7.60156102.gif" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>修复文化，保护文化、传承文化，在这支CG大片中，我们不难看到这样的品牌主张和格局。</span><span><strong>对于开心消消乐而言，以修复敦煌乐器为名，将品牌和传统文化进行融合，则赋予了品牌更深层的文化内涵和历史厚重感。</strong></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>提升品牌自身的价值感，刷新大众对品牌的认知感，我想，这是开心消消乐和敦煌博物馆进行梦幻联动的目的所在。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><img src="https://ngbjimg.xy599.com/38794396662b974394a3da6.63273158.png" referrerpolicy="no-referrer"></strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>端内联动升级用户体验，</span></strong></span><span><strong><span>端外传播扩散品牌声量</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然，对于游戏品牌，进行IP联动从来不能只停留在创意层面，</span><strong><span>激活游戏用户，提升用户体验，才更能体现游戏品牌跨界合作的营销实力。</span></strong><span>此次开心消消乐，同样在游戏端上线了一系列围绕敦煌IP的玩法。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>首先，和CG大片中小动物们修复“光之琵琶”的方式相呼应，开心消消乐在村内开启了寻梦敦煌之旅，围绕着画、舞、乐、史，推出了</span><span><strong>千年壁画、飞天之美、伎乐盛典、丝绸巡礼</strong></span><span>四条任务路线，用户可以在完成游戏任务的过程中，感受到“触手可及”的千年文化瑰宝，真正实现寓教于乐。</span><span></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在主线任务之外，开心消消乐还设计了一些有趣的互动彩蛋。比如用户可以通过弹奏“光之琵琶”，奏响专属自己的文明乐曲，收获更有仪式感的游戏体验。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/78059339762d7c6bc0ca695.44919574.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然，随着“开心消除节”的开幕，用户福利更是必不可少。</span><span><strong>开心消消乐的策略是，充分利用微博超话这一品牌私域流量阵地，实现和用户更有粘度的互动。</strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>用户可以在超话参加“开心大转盘”活动，每日打卡签到+完成任务，就有机会抽取游戏道具、精美实物周边以及敦煌【玉门关+鸣沙山月牙泉】门票套装等惊喜福利。</span><strong><span>由此，开心消消乐的“开心消除节”就从端内走向了端外，有利于品牌声量的扩大。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/47327271262d7c6bc6ec202.25963291.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于品牌的网络流量和热度，开心消消乐没有坐以待毙。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>互动话题#开心回响千年#登上微博热搜，一系列微博大V或基于CG大片、或基于IP联动、或基于游戏玩法的多角度扩散传播，都助推着开心消消乐的网络传播热度，继而穿透不同圈层的人群，吸引更多人到游戏端内，体验开心消消乐和敦煌文化的碰撞。</span><span></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>和敦煌博物馆的IP合作，增加了开心消消乐的品牌厚度；高人群覆盖率的网络传播，提高了开心消消乐的品牌广度。在广度和宽度之间，开心消消乐俨然是一个会玩的营销高手。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><img src="https://ngbjimg.xy599.com/46978155362b9743fae3289.16804197.png" referrerpolicy="no-referrer"></strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>持续焕新品牌萌团形象，</span></strong></span><span><strong><span>长线夯实品牌节日符号</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>事实上，作为国民游戏，尽管开心消消乐拥有经典的玩法和广泛的群众基础，但在新游戏层出不穷的行业竞争中，开心消消乐一直面临着品牌升级和重塑的营销任务。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>尤其是对游戏体验越来越挑剔的年轻用户，开心消消乐需要做的是，如何在保持自身优势的前提下，为品牌注入更新鲜的活力，实现用户的增长。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“开心消除节”便是开心消消乐的一次成功尝试。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从2020年节日IP的诞生，到2021年和上海美术电影制片厂“开心自有神技”的IP联动，到今年和敦煌博物馆的跨界合作，</span><span><strong>通过“开心消除节”，开心消消乐构建起了和用户对话的全新场域，借此建立了和用户更紧密的联结，也用多元IP不断升级品牌形象，重塑人们对品牌的认知。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在此过程中，我们会发现，开心消消乐的另一个品牌法宝也得到了形象焕新，那就是开心消消乐的官方IP</span><strong><span>消消乐萌萌团</span></strong><span>。作为游戏中的经典形象，这些小熊、小鸡、青蛙等小动物已经成为所有用户的好伙伴。但在萌经济盛行的当下，这些小动物显然有更大的营销价值。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/58170064762d7c6be0dc023.36122112.jpeg" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在IP联动中，消消乐萌萌团的形象就被持续焕新。去年，它们和哪吒、孙悟空一起玩耍，拥有无限神仙技能；今年，它们成了九色鹿的好友，是受命修复敦煌神物的文化保护者。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/212117959362d7c6be8fe0d2.62625143.gif" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>身份和形象不断变化，但消消乐萌萌团可爱、萌趣的形象却烙印在了用户心中，也成为了开心消消乐品牌形象升级的一大功臣。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>相信在开心消消乐的持续努力下，无论是开心消除节，还是消消乐萌萌团，都会成为深入人心的品牌IP符号，助力开心消消乐渗透到更年轻的用户圈层中去。</span></p>
