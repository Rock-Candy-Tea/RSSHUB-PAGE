<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“哈喽，哈喽，我们来咯！”</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>9月20日晚7点，在停播109天后，<a href="https://www.niaogebiji.com/tag/%E6%9D%8E%E4%BD%B3%E7%90%A6" target="_blank">李佳琦</a>终于低调回归。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1601893600632ae56140b1a4.34786374.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不出所料，李佳琦复播的消息很快刷屏了“所有女生”的朋友圈，气氛之热烈堪比过年。有业内人士不淡定地表示：“行业的希望回来了！”</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>现在，让我们用17个细节来回顾一下，李佳琦的920回归夜都发生了些什么。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/996074115601215dbd1d874.12950565.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>过去这段时间，“李佳琦920复出”的消息频繁流传，版本不一，扑朔迷离。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>关于这次的复播，业内也早有传言，有人透露9月初已有不少品牌在跟李佳琦团队签合同。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1409767219632ae5619b6b33.02242874.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但李佳琦开播前，李佳琦的淘宝、微博、抖音等账号，都没有做出对复出的回应。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>直到9月20日下午，宝洁直播间传出的消息成了昨晚回归夜比较靠谱的一次预告。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当天传出的截图显示，宝洁淘宝客服回复确认李佳琦会直播，蕉内也上传了带有李佳琦优惠券的链接。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1945112271632ae561dcb809.60266543.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2068046273601215dcef8457.71452222.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>昨晚19点开播后，李佳琦的直播间一度崩溃。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>19点07分左右，直播信号终于稳定。“好突然”“感动哭了”“终于等到你”等留言在直播间和各大社交媒体刷屏。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>开播半小时，观看590万，开播一小时，观看2700万，到当晚9点14分下播时，观看高达6117万。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>没有任何正式预告和宣传，淘宝直播的精选、热门页面的大部分时间都没有李佳琦的身影，即便如此，李佳琦的“暴力开播”依然拿下了2个多小时超6352万场观的成绩。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1271938081632ae562189083.72633823.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>公开数据显示，这一观看人次超越了618期间李佳琦大部分直播场次。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2012716452601215de253f83.29034753.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>李佳琦在直播里看起来气色不错，嗓音也不像从前透露着嘶哑，有细心的网友发现，“旺旺终于有时间做了个双眼皮”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有网友调侃，“果然休假是最好的医美”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/372550100632ae562466948.79237099.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>停播前，李佳琦多次传出身体不适的消息，在不少直播中李佳琦的状态也透露着疲惫，助播们在直播间占据的时间也逐渐拉长。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/58143460660235253dbd083.00129585.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>直播间还是熟悉的配方，却有了一些不同的味道。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在背景更加简洁的直播间里，李佳琦和助播旺旺依旧默契又热情地介绍着产品和规则，在播到内衣等女性品类时，他依然会把位置留给庆子姐。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过，李佳琦在直播间的口头禅从“OMG，买它”变成了“理性消费，快乐购物”，他多次劝说所有女生们，有需要再买，千万不要为了帮他提高销量而下单。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>即便如此，依然挡不住粉丝们“野性消费”的气势。直播间的十余个链接上架即秒空，买东西依然要拼手速。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1002972054632ae5627ba2d2.49809584.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>6月2日直播间vs9月20日直播间</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/8187367216030c0c85d55e0.47560833.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>李佳琦直播间第一件上架的产品是手机支架，4万件备货于1分钟内售罄。据统计，这场直播只上架了27个链接，主要以日化及护肤品为主。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>消毒液，洗手液，内裤，纸巾，卫生巾，精华液，面霜……面向女性用户刚需的产品满足了多数粉丝的日常购物需求，李佳琦直播间的“常客”夸迪和花西子等国货品牌也陆续出现。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1623954587632ae562b475a4.72662567.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有网友整理了本场直播的清单，在直播前20分钟的产品预告中，这场直播原计划上架约37个产品。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2136373883632ae56401a4a3.96606496.png" referrerpolicy="no-referrer"><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/3201941936030c0c8c50d10.81097853.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>美妆类目不再是“口红一哥”的优先类目。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在一张疑似负责李佳琦直播间选品的工作人员截图里，写到了李佳琦直播间的最新选品条件：国货优先，环保概念产品有先，科技概念产品优先，美妆类目不优先。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>带火了完美日记、花西子和薇诺娜等国货品牌的李佳琦，昨晚在直播中<span>再次</span>提到，希望更多好用的国货品牌被看到。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1399144434632ae564364dc9.11913539.png" referrerpolicy="no-referrer"></p>

<p> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>李佳琦回归，炸出了不少忠粉，甚至有网友开玩笑说：“他今天就是卖屎，我高低也要买点。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>昨晚，《苍兰诀》剧粉和“所有女生”还实现了梦幻联动，网友纷纷表示，“真正的东方清仓回来了”，“东方清仓和他的月光族子民”“被封印的十万将士，在淘宝解封了”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1306825226632ae564ac26f5.38341617.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/3260641406030c44b2df425.32626794.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>李佳琦回来了，一些专门研究李佳琦直播间的账号也复活了，毕竟，停播了三个月现在终于有新素材了！</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如小红书账号“直播间重度爱好者”昨晚欢乐更新，该账号粉丝评论“第一时间想到你，赶紧来你这里看看”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/199121603632ae564d91ae1.76751726.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/15387364756030c560703de3.18560908.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>李佳琦直播时，国内某头部MCN老板发朋友圈说：“不容易，希望大家都能好好地活着……”</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>面对疫情带来的动荡，各行各业都在坚守。再过几个月就到2023了，还有多少人在怀念2019。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有人说：“与其说大家想念李佳琦，不如说大家都太苦了，太需要奇迹了，真希望那些死掉的好时光可以像李佳琦一样突然回来，就像从来没有消失过。”</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/2587821746030c44cbcf1b8.74459222.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于李佳琦的复播，业内人士相继发表观点。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“互联网怪盗团”发文提到：“李佳琦一复出即重返巅峰，再次证明了直播电商还是个人英雄主义的舞台，个人的力量不但不可忽视，而且是最重要的。”</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>火星文化创始人李浩在接受《中国新闻周刊》采访时表示，李佳琦复出有利于淘宝直播发展，结束没有头部主播的尴尬。李佳琦回归对于淘宝直播抢回一些用户是有帮助的。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>上海财经大学电子商务研究所执行所长崔丽丽则表示，现在李佳琦的风格明显有所变化，李佳琦此次复出一定是对大环境有所观察，并经过深思熟虑选品后，开展的一次尝试。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/11981354956030c409384dd5.65151687.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在停播的三个月里，李佳琦的微博、抖音等账号一直处于停更状态。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据新抖数据统计，近90天抖音账号“李佳琦Austin”掉粉72万，但从昨晚李佳琦复播开始，账号粉丝开始迅速回升。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另外值得注意的是，停播期间，李佳琦在抖音的存在感并不弱。与疯狂小杨哥、董宇辉一样，李佳琦的直播切片也得到了非常广泛的传播。</span><span></span></p>
