<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>上线还不到一年时间，<a href="https://www.niaogebiji.com/tag/%E6%8A%96%E9%9F%B3" target="_blank">抖音</a>盒子就要被关停了。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>就在最近，有多方曝出抖音电商旗下的独立APP<a href="https://www.niaogebiji.com/tag/%E6%8A%96%E9%9F%B3%E7%9B%92%E5%AD%90" target="_blank">抖音盒子</a>即将停止运营，负责抖音盒子的员工也将会转岗到其他部门。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/172539012163207572a56780.92226185.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>▲</span><span> </span>图源：新浪微博</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>虽然在消息传出之后，抖音盒子有相关负责人否认了这一传闻。然而必须要承认的是，如今抖音电商依然在一路狂奔，但抖音盒子跟不上时代基本已经成为事实。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>乏善可陈的月活用户数以及对比抖音电商几乎可以忽略的GMV，字节所打造出来的独立电商APP，显然还无法承载自己的梦想。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/9935821603705940eddf2.13315027.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>抖音盒子，凉了</span></strong><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>伴随着流量红利的逐渐消退，内容平台做电商早已不是什么新鲜事。在这之中，抖音便开展的如火如荼。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过由于抖音电商的业务越来越大，为了避免商业化对于原本的内容生态的侵蚀，同时也希望能够从外部获取新的流量，于是抖音盒子这一独立电商APP在2021年底被正式推出。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>最初上线时，抖音盒子的首页主要分为逛街和推荐两个频道，前者定位线上商场，而后者则和抖音类似，利用短视频进行种草。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/95659283863207572f070f5.93047488.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>▲</span><span> </span>图源：抖音盒子APP</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>刚推出时，抖音盒子也确实掀起了一波短暂的热度，而且依靠大量的补贴曾吸引众多优质达人入驻。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>商家更是对抖音盒子纷纷看好，毕竟相较于抖音，抖音盒子上的内容主要由商家来进行生产，也能够通过平台更加准确的传递给消费者。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>只是没过多久，抖音盒子就逐渐被人们所遗忘了。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>虽然相关负责人表示抖音盒子还没有暂停运营，然而作为抖音做独立电商的一个重点目标，如今存在感低迷已经是不争的事实。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这里，通过一组简单的数字对比就可以很明显看出。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如在抖音上有2000万左右粉丝的交个朋友直播间，目前在抖音盒子上的粉丝数只有6000多人，大多数所发布的视频更是几乎0互动。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/52367989463207573402ea4.32716787.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>▲</span><span> </span>图源：抖音&抖音盒子APP截图</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这一现象也让抖音盒子陷入了一种恶性循环：内容没人看又获取不了新流量，使得商家和达人逐渐失去了运营的动力。在这种情况下，抖音盒子的热度只会越来越低。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在安卓应用市场上，抖音盒子的下载量只有1000多万，对比抖音APP高达35亿次的下载量，几乎可以忽略不计。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另一方面，抖音盒子上线将近一年时间，似乎也并没有为抖音APP拉来相应的新增，这或许也是抖音盒子暂停运营的原因之一。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/7051603926037059d0cdda0.93459463.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>失败根源在哪里？</span></strong><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>自从抖音盒子被推出的第1天开始，平台就备受质疑，其中提到最多的一句就是：在抖音上看过的内容，为什么还要在抖音盒子上再看一遍？</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这句话恰恰点出了抖音盒子在运营策略上的模糊。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>事实上，抖音盒子最初的定位是潮流时尚电商平台，和自己的老对手快手不同，抖音盒子的竞争方其实是更加擅长种草的小红书和潮流电商平台得物。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/151407826563207573cfc217.08906311.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>▲</span><span> </span>图源：App Store</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>数据上来看，截止到今年上半年抖音盒子的月活用户数仅仅才过了10万。对比而言，得物月活用户数为1680万，小红书更是高达2亿。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除此之外，不管是初代还是更新过后的版本，抖音盒子的整体体验也和大多数独立电商平台没有太大差别。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>逛街模式就是一个品类没那么齐全的商城，推荐选项则和抖音主平台如出一辙，刷视频的同时就可以进入商品详情页下单。</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然对于抖音盒子来说，潮流电商的定位也过于垂直，抖音电商当中常见的日用品等大众需求在抖音盒子上并不多见。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在潮流电商领域，如今能够存活下来的平台如得物，其实自身的优势主要在于产品。尤其是从2020年以来，得物先后拿到了如卡西欧等诸多大牌的产品首发权，算是真正走在了潮流前列。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过，被称为“时尚电商第一股”的蘑菇街却从成立之初就一直处在亏损状态，营收和利润双双下降，使得这个潮流电商中的老大哥处境越发艰难。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/8618204056320757428b902.10896592.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>▲</span><span> </span>图源：蘑菇街官方微博</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>一方面抖音盒子自身没有做出差异化，而另一方面，潮流电商有限的市场规模对后来者而言也并不友好。</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在大厂纷纷“降本增效”的背景之下，靠烧钱进行补贴显然无法长久持续下去，这个时候如果还要坚持推出一个毫无特色且功能重叠的平台，难度可想而知。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1154105644603705abbbd504.02631435.png" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><strong><span>独立电商梦，还能怎么做？</span></strong><span></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>靠短视频起家的抖音，其实早在2018年就开始发力电商。而其真正走入快车道，则是罗永浩在2020年正式官宣入住</span><span>抖音进行直播带货时期。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在随后的发展过程中，抖音先后上线了抖音小店、抖音商城等，成功打造出了一个电商闭环，还选择和淘宝彻底断开链接。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此刻，抖音做独立电商的目标已经清清楚楚，并且也确实实现了快速发展。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过去年4月份，抖音电商总裁曾公开表示：“不希望电商业务过度依赖抖音主站，而是要主动向外寻找新的增长可能。”只不过在当时，或许还无法预料到脱离主站的平台发展并不顺利。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>事实上，不只是抖音盒子，今年以来字节跳动还有两次被传出关闭独立电商平台。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/179737406363207574952766.67108574.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>▲</span><span> </span>图源：字节跳动官网</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>例如在2月份，字节旗下的跨境女装独立站Dmonstudio在运营100天之后就被宣布关停；4月跨境电商平台Fanno也被曝光出项目组已经解散，APP从4月22日开始便再也没有过更新。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>虽然抖音盒子基本宣告失败，但抖音的独立电商梦并不会随着戛然而止。</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>例如就在今年7月份，抖音又推出了独立电商APP“可颂”，从相关介绍上来看，可颂不管是功能还是服务内容等方面，都是在全方位对标小红书。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/22971535263207574d78c98.92572830.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>▲</span><span> </span>图源：可颂官网</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>只不过想要让电商业务独立的抖音或许还没有真正准备好，供应链问题其实就是抖音电商当下所面临的最大挑战。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不管是阿里还是京东，一直以来都非常重视对于供应链体系的打造。阿里有菜鸟，京东则是花了数十年时间自建配送体系。反观字节跳动，似乎在这一方面并没有太大的动静。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>然而想要真正的做好电商业务，那就不仅仅只是做一个APP就能够解决的。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>一方面想要稳住自己的内容基本盘，并把电商业务尽可能独立出来；另一方面又不愿意投入大量的资源去打造供应链体系。</span></strong></span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在未来的很长一段时间，抖音电商可能依然需要高度依赖成熟的主站。</span></p>
