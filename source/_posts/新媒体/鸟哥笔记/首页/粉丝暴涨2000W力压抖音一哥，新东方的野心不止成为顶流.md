<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: inherit; text-align: inherit;"><a href="https://www.niaogebiji.com/tag/%E6%96%B0%E4%B8%9C%E6%96%B9" target="_blank">新东方</a>搅热“文化直播”，俞敏洪赢麻了？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>新东方直播间爆红出圈的这一个多月里，其股价从暴涨10倍到达最高点33.15港元/股，跌回了今日的17.32港元/股。新东方股价大涨大落，而其直播间却似乎一路水涨船高，董宇辉也从普通主播成了“<span>总裁级主播</span>”。<strong>爆红之后，新东方发生了哪些变化？</strong></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>新东方直播间一度火到什么程度呢</span><span>？</span><span>6月10日，“东方甄选”抖音账号在热搜起飞，<strong>仅用7天就从百万粉丝涨至千万粉丝，直播销售额从几万到几十万，再到过亿，一飙再飙。</strong>尽管在此之前，“东方甄选”直播间开播数月都无甚水花。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>新东方主播们上知天文下知地理，靠着幽默风趣的“高端操作”，“东方甄选”直播间甚至被称为“最有文化的直播间”，自此开始走红破圈。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>截至发稿前，“东方甄选”抖音账号粉丝量已高达2254.5W，超越了抖音一哥罗永浩带队的“交个朋友直播间”</span></strong><span>，其抖音账号粉丝量为1958.5W。而据新抖数据显示，<strong>“东方甄选”直播间在6月份还交出了一份销售额6.81亿元的成绩，成为抖音平台6月唯一一个销售额破6亿的直播间。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/107019516462d70fe6f2aa10.40330264.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“东方甄选”直播间的走红，离不开董宇辉、顿顿等具有鲜明个人特色的主播，以致于不少粉丝常常在直播间留言：“董宇辉老师几点直播？顿顿老师几点来？”等诸如此类的问题。据果酱妹观察，如今的“东方甄选”直播间更是在背景中加上了“主播预告”的标识。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/108949252562d70fe7481598.82622498.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>左为6月中旬截图，右为近日截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>与主播深度绑定，于新东方而言或许是把双刃剑。一边是粉丝文化正在入侵新东方直播间，一边是董宇辉等明星主播能火多久尚未可知。<strong>在流量殆尽之前，新东方需要寻找更多出路。</strong></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>在爆红出圈之前，新东方就有意孵化矩阵，不过那时还未成型。</span></strong><span>相关账号的名称大多与“东方甄选”挂钩，粉丝数量不高且暂未发布作品。<strong>时隔一个多月，新东方正式开启新账号“东方甄选美丽生活”。</strong>据媒体报道，该账号的前身正是“东方甄选之个护美妆”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/108369120762d70fe8459a31.24418309.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>左为6月中旬截图，右为近日截图</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>7月8日，“东方甄选美丽生活”开启首场<a href="https://www.niaogebiji.com/tag/%E7%9B%B4%E6%92%AD%E5%B8%A6%E8%B4%A7" target="_blank">直播带货</a>。不同于“东方甄选”从早上7点至凌晨，超16个小时的直播，“东方甄选美丽生活”平均直播时长仅为近4个小时。根据灰豚数据显示，其近30天内共开启10场直播带货，销售额累计776W。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>即便外界一致认为“东方甄选美丽生活”是新东方进军美妆界的信号，不过就目前来看，“东方甄选美丽生活”的带货商品中，生活厨具类占比30.43%，而美妆个护类占比为26.09%。也就是说，<strong>“东方甄选美丽生活”或许将不仅仅专注于美妆赛道。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>值得注意的是，“东方甄选美丽生活”的主播与“东方甄选”几乎是同一波人，顿顿、杰西等主播轮番出现在直播间进行“双语带货”。只不过，部分转场过来的主播在“东方甄选美丽生活”直播间却不小心翻起了车。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>或许是对于销售产品不熟悉，主播杰西与大琪在讲解某品牌卫生巾时，出现了卡顿，甚至出现“一天用10片，150多片可以连着用半个月”的乌龙讲解。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/154449182862d70fe919b891.46306890.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>至此，新东方的矩阵账号已集合东方甄选、东方甄选之图书、东方甄选美丽生活、新东方直播间、东方甄选自营产品等，带货商品类别主要包含食品百货、图书文娱、生活厨具以及少量的母婴玩具。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除“东方甄选”一骑绝尘外，其他矩阵账号粉丝量最高徘徊在320W左右，新启运营的“东方甄选美丽生活”目前粉丝量为42W，日均涨粉1W+。<strong>与此同时，我们还在新东方直播间看到了多位新主播面孔。然而，部分粉丝对新主播并不买账。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/85769563362d70feb4ec6c9.54728716.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这对于开疆扩土的新东方来说并不是一个好的信号。一方面，现有的主播数量自然无法支撑新东方的矩阵布局，每天长达数小时的直播工作强度，仅靠董宇辉、顿顿等明星主播显然不太现实。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>另一方面，“东方甄选”以“最有文化的直播间”走红于互联网，主播们都是曾经的新东方老师，都有着新东方式的能说会道和文化底色，这是他们有别于其他带货主播并成功吸粉的最大原因。从这一层面上讲，<strong>消费者是在为“知识付费”，对新主播的要求自然也只会更高。自然地，新东方招纳新主播也只会面临更大的考验。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>虽说新东方直播带货大本营驻扎在抖音，但我们在微信上也发现了东方甄选活跃的身影。走红之后，公众号“东方甄选官方号”新增了“直播间金句”的内容，进一步加深自己的“文化直播”印象。此外，公众号“东方甄选会员”则专注商业变现，菜单栏、同名视频号均挂上了“甄选商城”的小程序。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>去年11月，新东方创始人俞敏洪在直播中，宣布将带领数百位新东方老师通过直播带货农产品。<strong>彼时，俞敏洪直播首秀销售仅为11.33W，半路出家的新东方老师们直播了几个月，挨了骂，东西也鲜少有人买。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>直至今年6月，新东方直播间才得以起飞。在奔流不息的流量面前，有人从“东方甄选”直播间获得灵感，尝试开启“文化直播”。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>7月2日，豆神教育旗下的快手账号<strong>“豆选生活”开启古诗词文化带货，主播们也都是核心骨干教师，单场销售额破千万。</strong>不过，该账号的“文化直播”却又略显粗糙，主播们边翻讲解书边介绍商品，讲解历史故事时，更是直接对着电脑看提示。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/176720424262d70fed282bd6.50151597.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>显然，要想复制“东方甄选”这样自成一派的“文化直播”并非易事。就连罗永浩也在直播间表示：“<strong>做直播讲内容是一股歪风，内容会把场观控上去，但是对直接销售的帮助有限。</strong>”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如今风水轮流转，俞敏洪带领的新东方直播间蹿红，俞老师昔日下属罗永浩老师退出“交个朋友直播间”。抖音直播江湖瞬息万变，新东方直播间的神话还能讲多久？</span></p>
