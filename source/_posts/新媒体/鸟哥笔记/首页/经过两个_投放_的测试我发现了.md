<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>你好，我是潮州痞子蔡，我们优化的不仅仅是广告，还有生活！</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>一、公众号发文，再次证明“看脸”是正确的</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>前几天更新的文章，我在第2篇文章用了之前发过的文章，但是更改了标题和封面，尤其是封面，结果推送的第2篇文章的阅读量在次日（一般都是晚上23点多发布）要比其它文章同样条件发布的时候多了不止一倍。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>说明：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>1、标题和封面尤其封面的修改是有效的，尤其封面是原生类封面的时候，这个我是在看其它文章的时候自己作为用户发现的，只是平时很少用，那天突然想着用一下验证一下数据，果然如此。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>虽然说作为运营大多数时候都是知道标题和封面的作用是不小的，但是自己再度尝试，以及用非常原生的图片封面来验证还是相当有趣的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>封面图再次感谢颠木老师：<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/42913485262c7f5177935b2.60411245.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2、阅读量的提升也和公众号现在的推荐算法不同有关，现在的算法和很多年前有了很大的不同。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不仅仅是依靠文章的推送作为流量入口，在阅读文章后的其它文章推荐处也做了“交叉运营”，因此好的标题和封面是能够带来更多的流量。</span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>二、巨量的视频素材投放，“有律可循”</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>上一篇文章《</span><span>素材分析，可以也最好一分为二</span><span>》中我提到分析素材、做好素材来提升投放效果和用户量级，可以观察素材的基础数据，不要仅仅是看素材和最终的转化率、roi等。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果能通过对视频内容的方向把控，以及对基础数据的深度理解，可以比较清楚、容易的做到对素材方向的把控和各类拓展制作。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以我们近期操作的案子为例。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>根据上篇文章讲到，我们第1个视频和第2个视频出现了很大的数据差距，主要是在针对平台用户的特点深度理解后，第2个视频针对目标用户做了素材上的明确瞄准。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>果不其然，第2个视频的数据是第1个的10倍以上。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>近期，我们针对第2个视频再做了两个尝试：<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>a、针对目标用户换了一个运营点进行测试，结果数据不佳：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/28073739462cfc1ac02a165.61130018.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>b、沿用第2个视频的特点进行重新选题制作，数据再次有效：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/158806490462cfc1ac2b4ad1.10672533.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这个数据还在继续涨。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>通过ab两个测试后，我发现了这里面几个有趣的现象：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>（1）在具体一个媒体平台去进行视频素材投放的时候，当抓准了运营点的时候，基本能够预判这个运营点基础上的素材的起量节奏、大致量级；<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这个给投放团队要量、以及视频制作等提供了一个方法，如果无法做出爆量的素材，那是否可以通过批量的这类素材去持续微创新和拿量？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>（2）在（1）的基础上，如果能对视频素材的基础元数据非常熟悉和了解的话，那完全可以在（1）的基础上去尝试制作更加有量的素材，只要围绕平台的用户心理特点、素材的运营点和数据目标等去落实运营就可以实现。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>目标数据的提升，自然能让素材起量。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>（3）如果（1）、（2）乃至可能的（3）、（4）等不同量级的视频素材通过分析、运营、投放和复盘等已经落实有效后，会积累了大量不同量级的视频素材的各种数据，包括不同阶段的播放量数据、点击率数据、用户在视频上的停留时间等数据。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>而这些数据能提供非常有用的价值：</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>c、转化预测：</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>呈现的数据和最终的转化之间的关系，这些关系完全可以关联起来，并且提前做好对效果的预测；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>d、视频起量预测：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此外，还可以提前预测到一些视频素材可能的起量情况。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因为通过上面我提到的几轮测试后，基本能大致判断到媒体的流量池算法中是大致怎么分配流量、以及视频素材到什么阶段就可以“判断死亡”的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于c、d的两个数据预测媒体也是会做预测的，而当对于广告主方来说如果也能做这样的预测，乃至更多的数据预测等，通过这些预测来验证自身对媒体的深度理解，那投放的效果正常来说就不会差到哪里去了。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有条件的团队不妨多试试，探索出来后，其乐无穷！<br></span></p>
