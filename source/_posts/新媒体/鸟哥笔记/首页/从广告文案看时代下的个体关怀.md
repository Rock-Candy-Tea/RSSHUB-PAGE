<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>品牌与消费者之间的桥梁是什么？</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>很多人会说，差一句耳熟能详的广告语。确实，广告语是一个品牌占领用户心智的重要途径。好的广告语能通过真实而具体的细节调动人的感官，使用户产生兴趣，让消费者信赖品牌所传递出来的卖点。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一个品牌从0到1，再从1到100的过程，除了品牌价值观的持续打造，更是广告语的传播，精准触达到了用户心中，从而增强了品牌认知度。但是品牌的价值也会随着品牌的成长，慢慢从广告语回到品牌上。这也就是说，等消费者记不住广告语的时候，或许就是品牌最为强大的时候。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>因为这个时候，消费者记住的是品牌，这也就发挥出了广告语的最大作用。今天的品牌打法与传统的营销方式截然不同，而广告语则反映的是一个时代的社会心理，以及经济发展。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>回看2022年上半年，许多品牌用创意文案/广告语开启新故事、新可能，通过细致入微的文案和镜头语言，呈现出不少独具形象性、感染力、记忆度的年度佳作。接下来，我们一起来盘点2022年上半年，深入人心、共情共鸣的广告语/文案都有哪些吧？</span><span></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从大润发出圈的文案来看，好的文案不一定是要博眼球，但一定是要立于消费者和服务。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果一味追求“打造爆点”，毫无底线的想要“出众”，那么最终“你”会为所谓的“出众”付出代价。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/118423885062d0b169b602c8.31303901.jpeg" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>Campaign：40周年“特刊”：一份手工拼贴出来的《深圳特区报》</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>品牌/广告主：深圳特区报</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>代理商：安知</span></strong></span><span><span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/99829905562d0b169eee330.69013419.jpeg" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一座城，一张报，代表着一个时代的印记。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>每一段镌刻时代印记的岁月都注定不平凡。1982年5月24日，《深圳特区报》在改革开放热土的一间铁皮房里横空出世，从此与深圳经济特区同呼吸、共命运。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>往前追溯，40年上万期的故事和观点，除了保存在特区报档案室中，还散落在无数报人和读者的抽屉里。40年过去了，读者在不断的变化，但是媒体属性依旧如初。尤其是新媒体的出现倒逼传统迈进数字化时代。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>安知通过三分半的短视频，以本土群众最易共情、且能见证深圳发展足迹的版面挑选出来，邀请到新闻当事人用拼贴的方式将报纸的40年进行贯穿，以一张对开的白纸作为画面的主题，一共12块剪报，从邓小平“画了一个圈”之后，深圳有了飞速的发展，在视听还未盛行的40年前，正是报纸将它们挨家挨户的传播，将内容从无到有的拼贴出来。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从采、编、印、读这四个大流</span><span>程里的代表性人物，人物本身的年龄与这四十年相关联。具有年代感、怀旧感。不仅让观者对一张保持从无到有所需要经历的过程有了了解，还将新闻人的匠人精神进行了展现。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这是属于印刷文字的力量，不同字体的新闻使得时间的流逝有了具象的呈现。40年来，《深圳特区报》持续将“深圳奇迹”化作铅字留痕，是这座改革之城忠实的记录者、见证者，也是有力的推动者、建设者。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span> </span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/67331257062d0b16a2ce3c5.10802469.jpeg" width="444" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"> </p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>Campaign：可口可乐抗疫暖心视频《献给每一个你》，真的很暖</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>品牌/广告主：可口可乐</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>代理商：奥美</span></strong></span><span><span></span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img style="display: block; margin-left: auto; margin-right: auto;" src="https://ngbjimg.xy599.com/152324665262d0b16a5eaf62.94254131.png" width="444" referrerpolicy="no-referrer"></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>谁也没想到，上海疫情封控期间可乐竟然会成为「硬通货」。事件的原因还要从上海某小区的“换物接龙”开始，某网友自发将消过毒的12罐可乐放在一楼大厅免费供人取用，谁料物资不减反增。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有人拿走一瓶可乐，拿来一瓶辣酱，有人拿来牛奶，有人用化妆品来换……大厅俨然攥成了一个“小超市”。基于此，可口可乐上线了一则抗疫暖心创意视频《献给每一个你》。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>奥美将镜头聚焦“可乐瓶”的数种存在形态，以此来影射不同群体中相同的“你”的抗疫状态。在仅有的白色背景下用一个个不停变换高、低、大、小、新、旧的可乐瓶来诠释演绎疫情之下，每一个用心守护生活的人。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>人，最重要，最能传递价值的是“情感”。“只要我们在一起 一切都会好起来”的文案更是与2020年的片子遥相呼应，品牌关注世情、人情的责任与情怀从未改变。可口可乐从1886年走来，洞察着时代的情绪，用它的每一个形态和每一帧，表达着每一种状态。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>无论是坚强的、脆弱的，还是个体的、群体的，这个短片向我们传达着疫情中每个努力生活的人都是平凡的英雄。</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>每一张照片都有一个故事。</span></p>
