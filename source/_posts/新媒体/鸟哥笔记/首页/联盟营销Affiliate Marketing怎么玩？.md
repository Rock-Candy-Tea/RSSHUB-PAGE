                        <p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">在企业品牌出海中，<a href="https://www.niaogebiji.com/tag/%E8%81%94%E7%9B%9F%E8%90%A5%E9%94%80" target="_blank">联盟营销</a>作为一种双赢的商业模式，在海外越来越受到广告主欢迎。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">仅以美国为例，Statista数据显示，截止2022年，美国本土企业在联盟营销（Affiliate Marketing）方面的支出将达到82亿美元。Big Commerce博客介绍，在美国，有81%的品牌和84%的发行商已经采用了联盟计划，估计有多达16%的在线订单来自于联盟计划参与者的推广。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">那么，什么是联盟营销？它与网红营销有何不同？</span></p>
<h2 id="toc-1" style="font-size: 18px; background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); background-position: 0px 5px; margin: 25px 0px 20px; line-height: 30px; background-repeat: no-repeat; color: #212629; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; text-align: left; font-weight: 600; background-size: 8px 20px; padding-left: 12px;">一、联盟营销是什么？</h2>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">联盟营销（Affliate marketing）又常被称作伙伴计划（Affiliate Program 或是Referral Program），主要是品牌商通过网路，经由与部落客、网红、个人等推广者的合作，请他们在网络上宣传品牌商的商品或服务（无论是通过自己的网站或是社群平台FB、Email名单等），当通过这些推广者的协助而成功带来转换时，品牌商将会分享利润给这些协助他推广的合作伙伴。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">据Inpander所知，世界上最早的联盟营销是1996年Amazon推出的Associate Program。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">为了让Amazon能够快速地被更多的人认识，他们邀请成员加入伙伴计划、推广平台上的商品，给予相对应的佣金回馈，借此为网站带来大量的流量、声量（知名度），以及营业额。</p>
<p style="margin: 25px 0px 20px; line-height: inherit; background-repeat: no-repeat; background-image: none; background-position: 0px 6px; font-size: 16px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; text-align: inherit; font-weight: inherit;"><strong>1. 活动流程</strong></p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><img class="rich_pages wxw-img aligncenter" style="border: 0px; max-width: 100%; height: auto; display: block; margin-left: auto; margin-right: auto; cursor: pointer;" title="联盟营销Affiliate Marketing怎么玩？【附实操指南】" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2022/05/RzuL0mMYDTUyNLSJk6uO.jpeg" alt="联盟营销Affiliate Marketing怎么玩？【附实操指南】" width="562" height="457" data-action="zoom" referrerpolicy="no-referrer"></p>
<ol style="padding-left: 50px; color: rgba(0, 0, 0, 0.6); font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff;">
<li style="margin-bottom: 5px;">
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">Step 1：</span>推广的人员把你的产品（banner，视频，文字）放在他们的网站上（包括博客，社交渠道等）</p>
</li>
<li style="margin-bottom: 5px;">
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">Step 2：</span>用户点击并形成转化</p>
</li>
<li style="margin-bottom: 5px;">
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">Step 3：</span>这个过程的点击和转化是可被追踪的</p>
</li>
<li style="margin-bottom: 5px;">
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span style="font-weight: bold;">Step 4：</span>Affiliate获取了销售的佣金，你获取了订单</p>
</li>
</ol>
<p style="margin: 25px 0px 20px; line-height: inherit; background-repeat: no-repeat; background-image: none; background-position: 0px 6px; font-size: 16px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; text-align: inherit; font-weight: inherit;"><strong>2. 主要参与者</strong></p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">联盟营销的独特之处在于</span><span style="font-weight: bold;">按照推广效果支付费用</span>，并且需要多个参与者紧密合作才能实现的营销方式，这些参与者包括：</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">1）</span><span style="font-weight: bold;">品牌商</span></p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">通常也称作广告主，也就是拥有产品或者服务需要销售的公司。品牌商需按照联盟营销的实际效果（如销售额、引导数、点击数等）<span style="font-weight: bold;">向联盟会员支付合理的广告费用</span>。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">2）</span><span style="font-weight: bold;">联盟客</span></p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">也就是联盟营销人员，他们通过联盟营销平台选择合适的品牌商并通过播放品牌商广告赚取佣金，轻松地将<span style="font-weight: bold;">把网站访问量变成收益</span>。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">3）</span><span style="font-weight: bold;">联盟平台</span></p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">也就是通常所说的联盟营销平台或者广告联盟，他们实际上是广告主的代理，负责站在广告主的角度，<span style="font-weight: bold;">监督和管理联盟项目的每日运行和策略制定</span>，包括合作伙伴关系、联盟客招募、推广指导和广告活动优化。常见的国外联盟平台有 Commission Junction、ShareASale 等。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">4）</span><span style="font-weight: bold;">消费者</span></p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">就是广告主和联盟客所希望营销的特定用户群体，他们最终会<span style="font-weight: bold;">产生消费行为</span>、或者提交联系方式、订阅公司邮件、试用某项服务、成为一个新客户，等等。</p>
<p style="margin: 25px 0px 20px; line-height: inherit; background-repeat: no-repeat; background-image: none; background-position: 0px 6px; font-size: 16px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; text-align: inherit; font-weight: inherit;"><strong>3. 其他专业术语</strong></p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">附属链接（Affiliate Link）：</span>也称为跟踪链接。这是一个独特的 URL，用于跟踪来自联属会员的点击和转化。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">转换（Conversion）</span>：由商家定义的被视为支付给关联公司的事件。它可以是商家定义的任何内容，但通常是潜在客户或商家产品的购买。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">网络营销联盟（Affiliate Network）</span>：商家和附属公司之间的可选中间人，有助于促进商业活动。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">联盟经理（Affiliate Manager）</span>：与附属公司合作的人，作为与商家的中间人。他们的目标是最大化联盟计划的性能。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">联盟计划（Affiliate Program）</span>：商家向附属公司支付佣金的一种安排。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">优惠（Offer）</span>：产品的特定促销。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">点击率CTR(Click-through Rate)</span>：— 每网页展示次数的附属链接点击百分比。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">转化率CR(Conversion Rate)</span>：转化的点击百分比。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">每次操作成本 CPA（Cost per Action）</span>：一种佣金模式，商家仅向关联公司支付转化费用。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">每次点击成本 CPC（Cost per Click）</span>：一种佣金模式，商家仅向关联公司支付点击费用。它通常与广告横幅一起使用，而不是联属网络营销。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">URL追踪（Postback URL）</span>：— 用于服务器端转化跟踪的 URL；也称为服务器到服务器跟踪。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">像素（Pixel）</span>：跟踪网页查看事件发生时的隐藏图像。在联属网络营销的背景下，像素会跟踪转换发生的时间。今天，像素通常是嵌入在网页上的 JavaScript 代码片段，而不是图像。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">短链（SubID）</span>：作为会员链接的一部分嵌入的文本字符串，由会员定义并用于跟踪会员的转化。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">创意（Creative）</span>：联属会员用来宣传商家产品的广告横幅图像或文字。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">文案（Copy）</span>：有助于推销商家产品的书面文字。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">登陆页面（Landing Page）</span>：会员通过会员链接将流量引至的商家网页。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">广告拦截（Ad Blocker）</span>：可以阻止跟踪联属网络营销点击和转化的网络浏览器技术。此外，它还可以防止广告横幅的显示。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">归因（Attribution）</span>：正确识别哪些附属公司正在产生转化的过程。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">拒付</span><span style="font-weight: bold;">（Charge Back）</span>：导致会员佣金被没收的无效转换。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">佣金（Commission）</span>：会员因产生转化而获得的收入。它也被称为推荐费、发现者费用或赏金。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">网络跟踪器（Cookies</span><span style="font-weight: bold;">）</span>：存储在潜在客户的网络浏览器中以识别特定访问的信息。通过联属网络营销，cookie 用于跟踪联属网络链接点击和潜在转化。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">链接隐藏（Link Cloaking）</span>：使用比商家提供的更简洁的 URL 隐藏附属链接。链接隐藏可用于更好地管理链接并跟踪点击和转化。</p>
<h2 id="toc-2" style="font-size: 18px; background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); padding-left: 12px; background-position: 0px 5px; margin: 25px 0px 20px; line-height: 30px; background-repeat: no-repeat; color: #212629; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; text-align: left; font-weight: 600; background-size: 8px 20px;">二、联盟营销的优势及盈利形式</h2>
<p style="margin: 25px 0px 20px; line-height: inherit; background-repeat: no-repeat; background-image: none; padding-left: 15px; background-position: 0px 6px; font-size: 16px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; text-align: inherit; font-weight: inherit;"><strong>1. 联盟营销的好处</strong></p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">1）易于实施</span></p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">联盟营销相对简单，不需要大量的带宽来实施。一旦品牌建立了为每个会员创建唯一 URL 的机制，他们需要做的就是跟踪每次销售的转化。然而，重要的是要注意，品牌必须成熟，才能支持成功的联属网络营销计划。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">2）只为结果付费</span></p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">联盟营销是一种无风险的策略，因为联盟只有在通过联盟链接进行销售时才能赚钱。与传统广告相反，品牌不为印象或点击付费。无论对附属链接的点击如何，仅在销售时才会支付佣金。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">3）增加网站流量（和销售额）</span></p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">虽然附属链接对搜索引擎排名没有直接影响，但它们确实增加了网页的流量并增加了转化销售的几率。附属公司围绕产品或服务创造的兴奋会产生“光环效应”，增加消费者的信任，并促使更多人搜索您的产品并导航到您的页面。</p>
<p style="margin: 25px 0px 20px; line-height: inherit; background-repeat: no-repeat; background-image: none; padding-left: 15px; background-position: 0px 6px; font-size: 16px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; text-align: inherit; font-weight: inherit;"><strong>2. 盈利形式</strong></p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">有研究指出，全世界的联盟营销产业价值高达65亿，产业横跨零售业、个人财务、游戏产业、旅游业、教育、出版业等。大部分的联盟行销分润方式是以单笔计算，但较大的组织也会根据推广的的程度不同给予不同的佣金抽成，根据商家网站给联盟会员的回报支付方式，联盟网络营销可以分为三种形式：</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">1）按点击数付费（CPC）</span></p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">CPC 即 Cost-Per-Click, 联盟网络营销管理系统记录每个客人在联盟会员网站上点击到商家网站的文字的或者图片的链接（或者Email链接）次数，商家（Merchant）按每个点击多少钱的方式支付广告费。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">2）按引导数付费（CPL）</span></p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">CPL 即 Cost-Per-Lead, 或者或者叫CPA(Cost-Per-Acquisition)。访问者通过联盟会员的链接进入商家网站后，如果填写并提交了某个表单，管理系统就会产生一个对应给这个联属会员的引导（Lead）记录，商家按引导记录数给会员付费。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">3）按销售额付费（CPS）</span></p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">CPS 即 Cost-Per-Sale, 商家只在联盟会员的链接介绍的客人在商家网站上产生了实际的购买行为后（大多数是在线支付）才给联盟会员付费，一般是设定一个佣金比例（销售额的10%到50%不等）。</p>
<h2 id="toc-3" style="font-size: 18px; background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); padding-left: 12px; background-position: 0px 5px; margin: 25px 0px 20px; line-height: 30px; background-repeat: no-repeat; color: #212629; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; text-align: left; font-weight: 600; background-size: 8px 20px;">三、联盟营销VS网红营销，应该怎么选？</h2>
<p style="margin: 25px 0px 20px; line-height: inherit; background-repeat: no-repeat; background-image: none; padding-left: 15px; background-position: 0px 6px; font-size: 16px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; text-align: inherit; font-weight: inherit;"><strong>1. 主要区别</strong></p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">收入分享通常会模糊联盟营销和网红营销之间的界限，但仅仅因为这两种方法都可能涉及创作者佣金，并不意味着薪酬模式的工作方式相同。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">联盟营销</span>几乎专门用于为品牌网站带来流量以完成交易。品牌不会为品牌知名度、增加网络流量或冷潜在客户生成向联盟会员付费。联<span style="font-weight: bold;">盟会员仅因进行销售而获得报酬</span>，这意味着他们必须有动力并不断努力提高转化率。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">在<span style="font-weight: bold;">网红营销</span>中，品牌建立了更长期的KPI，<span style="font-weight: bold;">例如点击率，用户生成的内容和增加的SEO</span>。影响者模型在推广品牌、产品或服务时侧重于内容创建，营销人员与影响者合作，以提高其可信度、真实性和可持续的入站流量。因此，影响者薪酬包括佣金，产品礼品，统一费率费用或这三者的组合。</p>
<p style="margin: 25px 0px 20px; line-height: inherit; background-repeat: no-repeat; background-image: none; padding-left: 15px; background-position: 0px 6px; font-size: 16px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; text-align: inherit; font-weight: inherit;"><strong>2. 网红+联盟，迸发新能量</strong></p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">随着全世界都在把注意力转向社交媒体，消费者现在已经越来越“依赖于”让其他人来帮他们完成购买决策。譬如当红明星、话题人物、某领域的专业人士或是知名美妆美食博主之类，来自他们的评论和推荐会具有广泛的影响力和号召力。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">这也让联盟+红人的新型营销模式，迸发出新的能量：<span style="font-weight: bold;">前者能带来直接收入，而后者在推广品牌信息和认知度方面颇具潜力，通过两者相结合，以接触到更广泛的受众，从而产生更大的ROI。</span></p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">那么，对于出海品牌而言，具体应该如何操作呢？</p>
<h2 id="toc-4" style="font-size: 18px; background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); padding-left: 12px; background-position: 0px 5px; margin: 25px 0px 20px; line-height: 30px; background-repeat: no-repeat; color: #212629; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; text-align: left; font-weight: 600; background-size: 8px 20px;">四、操作流程</h2>
<p style="margin: 25px 0px 20px; line-height: inherit; background-repeat: no-repeat; background-image: none; padding-left: 15px; background-position: 0px 6px; font-size: 16px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; text-align: inherit; font-weight: inherit;"><strong>1. 找到合适的联盟平台</strong></p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">首先，您需要<span style="font-weight: bold;">加入联盟平台</span>，为你的营销活动创建带有活动介绍、产品推广指南等会员招募落地页，并<span style="font-weight: bold;">生成链接和折扣代码</span>，以便您可以跟踪每个联盟产生的收入，并将投资回报率归因于每个单独的执行者以及整个计划。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">常见的联盟平台推荐：</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">1）ShareAsale</span></p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><img class="rich_pages wxw-img aligncenter" style="border: 0px; max-width: 100%; height: auto; display: block; margin-left: auto; margin-right: auto; cursor: pointer;" title="联盟营销Affiliate Marketing怎么玩？【附实操指南】" src="https://cors.zfour.workers.dev/?http://image.woshipm.com/wp-files/2022/05/9Pmvk7cQsYhC9n1yqnGE.png" alt="联盟营销Affiliate Marketing怎么玩？【附实操指南】" width="661" height="251" data-action="zoom" referrerpolicy="no-referrer"></p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;">于2000年在美国成立，拥有225000+个联盟资源媒体，已合作16500个+广告主和品牌，佣金费率视情况而定，一般在5%-20%之间。</p>
<p style="margin: 25px 0px 20px; color: #000000; font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', STHeiti, 'WenQuanYi Micro Hei', Helvetica, Arial, sans-serif; background-color: #ffffff; background-image: none; font-size: 16px; text-align: inherit; font-weight: inherit; line-height: inherit;"><span style="font-weight: bold;">对于商家而言它的核心优势是：</span>商家可自定义注册页面放在独立站footer，用于招募和吸引分销商；能为商家提供免费的分销商招募工具；商家可进行差异化的佣金设置，满足不同的推广需求。</p>
