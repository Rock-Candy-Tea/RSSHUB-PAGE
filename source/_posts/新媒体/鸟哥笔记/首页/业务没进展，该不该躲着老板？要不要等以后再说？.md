<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>最近的文章标题可能老读者有些不适应，我想告诉大家的是，经过这么多年，我觉得杂货铺应该陪读者一起成长，顺带着解决一些新进入行的同学们的问题，雅俗共赏 ，高下相宜。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>我个人并不喜欢标题党，所以我尽量让每篇文章都对得起它的标题和它想传递给大家的信息 。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>这段题记会保持一段时间，阅读时可以快速跳过。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>也可能因为夏天到了，最近80分的社群特别热闹，我觉得这几天应该不会缺选题了。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>有一个典型的职场问题：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我接到了老板交给我的一个业务，或者一个任务，这个业务或者任务表面上看不出难易，但是上手之后才发现有不少问题，甚至有一些是我解决不了的。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在这个时候，我应该怎么做才是正确的？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>通常情况下，作为员工，会有三种反应：<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>第一种，荣辱不惊。</span></strong></span><span>之前是什么汇报节奏，现在依然是什么汇报节奏。</span><span>老板一个礼拜和全员开一次周会，那么我也继续在周会上进行汇报就好；老板这段时间比较忙，那么我就闷着。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>第二种，避开锋芒。</span></strong></span><span>我都没有做出成绩，看到老板要躲着走。上着厕所呢，听到老板说话的声音，赶紧躲到隔间去，等外面没动静了再出来，汇报，那必须报喜不报忧啊，一切尽在掌握，老板你别再问下去了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>第三种，主动出击。</span></strong><span>搞不定就是搞不定，赶紧增加汇报频次，把我的进展和老板同步，把我的问题找老板倾诉，push老板给我资源，拿出决策，让我可以继续推进。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>很显然，只有第三种才是正确的做法。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那么，为什么？</span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我想你一定听过：</span><strong><span><span>「</span><span>事事有着落」</span></span></strong><span>、</span><strong><span>「件件有回应」</span></strong><span>、</span><strong><span>「凡事有交代」</span></strong><span>这几个表述工作状态的短语，但你真的明白这几个短语在说什么，以及为什么这几个短语背后，是老板认定的</span><span><strong>「好员工」</strong></span><span>的标准吗？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果我是老板。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>事事有着落：</span></strong></span><span>我交给小王的事儿，小王一定可以完成。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>这意味着我对小王是有信心的</span></strong></span><span>，我相信小王不会让我失望，一定能帮我搞定我想要做的事儿。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>件件有回应：</strong></span><span>我交给小王的事儿，小王一定会让我知道干成了或者没干成，干成了因为什么，没干成又是什么导致的。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>这意味着我对小王是放心的</span></strong></span><span>，他做事儿一直都能让我超出预期。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong>凡事有交代：</strong></span><span>小王不仅会告诉我他干成没干成，还会在过程中<span>详细地告诉我他的排期、计划、需要的资源、推进的程度，简单地说，小王什么时间做什么事儿，做到什么程度，甚至需要什么帮助，我都知道的一清二楚。</span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>这意味着我对小王是安心的</span></strong><span>，我知道他在干嘛，方向对不对，干成的把握有多大。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但是，最近新入职的小李，好像和小王正好是完全相反的。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>虽然小李最近做的业绩不错，但是小李始终不是特别愿意和我交流，我有时候并不知道他在忙什么，也不知道他需不需要我的帮助，他有时候能把事儿干得特别干净利索，但有时候会拖到最后一刻才让我知道他搞不定，小李是不信任我，还是天生就不爱交流？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>恰好没多久，有一个升职机会，我应该把这个机会给小王还是小李呢？<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>答案显而易见，我会给小王，因为我认为我对小王更放心、更安心、更有信心。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>相比知道你的业务进度、遭遇的阻碍、需要的支持，老板其实并不会把某一次业务或项目的结果看得重如泰山。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以，请你记住，能够让老板放心、安心、有信心的员工才是「好员工」，这也是「事事有着落」、「件件有回应」、「凡事有交代」的底层逻辑。</span><span></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果你已经理解，适时地交流，可以让老板对你更放心、更安心、更有信心，那么你大概率会问：<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如何和老板交流或者汇报才能事半功倍？<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这里有五个原则与你分享。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>第一，汇报要有条理。</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>你可以这样拆解你的汇报流程：<br></span></p>
<ol>
<ol>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>业务/项目最近的进展：讲一讲最近你做了什么，业务或项目到了什么节点，有什么样的阶段成果。</span></p>
</li>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>遭遇的阻碍：说明你在推进过程中遇到了什么样的阻碍，你是否有能力自己跨越它，如果有，写出你的解决方法，请老板帮你查缺补漏；如果没有，直言不讳邀请老板提供帮助。</span></p>
</li>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>需要的资源：继续往下推进，是否需要额外的资源支持，哪些资源你能搞定，哪些资源你搞不定，哪些资源你不知道是否存在，都可以一一列举说明。</span></p>
</li>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>待决策事项：</span></strong></span><span>有些事儿你不能拍板，也不该拍板，这些都可以请领导拍板。譬如，第一次交流或汇报，先让老板拍板能够让你拍板的事儿有哪些。</span></p>
</li>
<li>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span><em><span>下一步计划：</span></em></span></strong><span>接下来你想怎么干，在什么条件下能干成什么样，下一个正式的汇报节点是什么时候。让老板心中有数。</span></p>
</li>
</ol>
</ol>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>千万不要想到哪里就说到哪里，在交流前，你可以列个提纲，在汇报前，建议做个简单的PPT，因为这些可以帮助你在刚开始做这件事儿的时候确保不跑偏。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>第二，讲困难和阻碍最好有数据支撑；</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>老板最怕的是汇报过程中或者交流过程中，员工只是在倒苦水，而不告诉自己这些困难和阻碍会带来什么样的后果。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>其实，有些后果已经在数据中体现了，老板已经看到了，他只是不确定这样的数据变化是什么导致的，那么你完全可以大大方方说给他听；有些结果并未在数据中呈现，甚至都还没有数据支持，那么你可以用这件事儿原本的目标进行折扣估算。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>譬如，A困难如果不解决，可能目标要打个八折，B阻碍如果也不解决，那么八折基础上还要再打五折。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>明确这些数据影响，并不会刺激老板，让他觉得天要塌下来了，而是老板必须知道为什么原先的设想无法实现，他怎么可以帮到你。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>请你记住一个基本事实：</span><span><strong>领导存在的价值之一，就是帮下属解决问题。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><span>第三，保持客观；</span></strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>保持客观的意思特别简单，就是我只说客观事实，不说主观因素。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>譬如，小王和小李都提交了需求给开发组，但是都被开发组以「优先级不足」被驳回了，同样的这件事儿，如何反馈才是客观，如何反馈才是主观呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>小王告诉老板：「开发反馈我们的需求优先级不足，是否可以协助确认一下目前开发组的工作排期，这样我可以见缝插针。」</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这是客观。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>小李告诉老板：「开发不愿意做我们的需求，我都请他们全组喝奶茶了，但是搞不定就是搞不定。」</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这是主观。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>客观的视角会让你保持中立，去想更多解决问题的方法，也让老板更快的去定位问题在哪里，而不会觉得你的事儿没有做到位，只会把问题丢出来。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>第四，提供更多选项。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>交流或汇报的过程中，尽可能让老板做选择题，而不要做填空题或开放题。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>小王和老板说：「开发排期的事儿，还需要您帮我推一推，最好能够在下周三前让开发组给出排期时间，最晚周五下班前必须给出来，否则这个项目就会Delay。」</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这叫选择题，让老板知道你需求的底线，提前建立预期，并且愿意根据你的需求和选项，来帮你推进。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>小李和老板说：「开发排期我搞不定，老板你能去催催吗？开发组的排期越快给出来越好，否则这个项目就会Delay。」</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这叫填空题，什么是你的底线，你要我去催，我也得知道什么情况下会出什么样的问题，你需要的节点是什么才行，否则我是在帮你解决问题，还是帮你打下手呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>小刘和老板说：「开发排期的事儿，开发小组咬得很死，老板你有方法吗？帮我去推推吧。」<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这是开放题，连后果都不说，我作为老板，要怎么知道该不该帮你推，推到什么程度？<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>老板碰到小王，非常开心有这么一员干将；碰到小李，非常后悔怎么招了这么个小皇帝；碰到小刘，那可能就想立刻辞退了事。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>第五，积极思考，学会说「yes and」。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>交流过程中，你要跟随老板给的反馈更加积极的思考，要学会说「yes and」，千万不要说「no but」。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>老板告诉小王：「这个项目，你或许可以寻求一下市场部的投放小组的支持。」</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>小王说：「这太好了，老板，你说我能不能同时找一下产品总监，让他陪我一起去，这样投放小组能更加清晰的理解我们想帮助业务做什么，做到什么程度，他们更容易评估投入的程度。」</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这是「yes and」。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>老板告诉小李：「<span>这个项目，你或许可以寻求一下市场部的投</span><span>放小组的支持。</span>」</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>小李</span><span>说：</span><span>「不行的老板，他们上次项目就拒绝了我，我觉得他们对我这边的项目完全不上心，不如老板你直接和他们总监开个会？</span><span>」</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这是「no but」。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>请问如果你是老板，面对小王和小李，你觉得谁更成熟，更有机会成功达成目标？</span><span></span></p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我在给一家公司做教练的时候，公司有一位特别年轻的运营总监，每次教练完，他都会叫网约车把我送到家门口。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当时他正在被老板夺命追魂天天问，内心非常郁闷，甚至怀疑老板不信任他。于是向我求教，应该怎么办能够解决这个问题。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我说你平时是不是都不主动和老板汇报？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>他说是。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我说那你每周怎么和老板去同步进展呢？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>他说周会。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我给了一个特别简单和实用的建议：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>你每天看到老板的第一件事儿，就是询问他的空余时间，去他办公室和他约好占用他10分钟的时间。你只去同步三件事儿：计划中的每一件事儿的最新进展、接下来你打算做什么、是否碰到问题需要老板帮忙决策。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>保持这个频次2-3周，老板不但不会觉得你烦，相反，他会对你越来越放心，放弃主动问询的做法。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>果然，第三周他和我说，老板现在最喜欢和他说：「想法很好，去干吧。」</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>所以，一定要学会适时同步。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>什么叫适时，就是要找合适的机会占用合理的时间来让老板最关心的问题得到阶段性的反馈。<br></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这个反馈不需要去包装，只需要讲清楚说明白就行。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我希望每个读者都记住，当你负责的业务或项目遭遇挫折时，埋头做鸵鸟是不能解决问题的，相反，你应该主动出击，让老板知道你在做什么，做成什么样了，还需要哪些帮助，有没有下一步计划。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这样，经过一段时间，你一定能成为老板心中的「好员工」。</span><span></span></p>
