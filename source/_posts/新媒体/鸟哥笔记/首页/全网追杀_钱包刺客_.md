<p><span><strong><span>“直面价格敏感，鲜有‘赢家’。”</span></strong></span></p>
</section>
</section>
</section>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>网友将钟薛高“煎烤炸”的同时，引爆了大家对“雪糕刺客”的复杂情绪。事实也是如此，不少网友关注增稠剂等食品问题，也有很多网友聚焦钟薛高的价格。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>为此，数天前，媒体梳理了737款雪糕的价格，发现仅有39款雪糕单支价格在3元以下，超两成雪糕价格在15元以上，最贵的单价高达约99元。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>媒体盘点的这些雪糕中，外国品牌单价大多高于国内本土品牌，有“雪糕贵族”之称的哈根达斯平均单价达57.5元，位居榜首，而钟薛高、中街1946等国产新品牌平均单价都在15元以上，最贵单支超30元。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>“雪糕刺客”扎堆，网友直呼这些“刺客”会伤到钱包，会刺痛人。值得注意的是，相关话题不断扩散，“<a href="https://www.niaogebiji.com/tag/%E9%92%B1%E5%8C%85%E5%88%BA%E5%AE%A2" target="_blank">钱包刺客</a>”的范畴也越来越广，正如一位江西网友所言：“日子越来越难过了，钱越来越难赚，刺痛平凡人的不只是一根钟薛高。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>网友简单一句话，道出了无数消费者的心声，进一步来说，“钱包刺客”屡屡刺痛人，与消费者普遍性的价格敏感密切相关。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/36214633662d82927b031a0.26621861.gif" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span> 更谨慎更敏感 </span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于价格敏感的分析，新零售商业评论认为，着力点在两方面，即“价格”和“敏感”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在“价格”方面，从数十万到数十元乃至几元钱，都是消费者“敏感”的区间。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>比如，消费者如果想买一辆60kWh、续航约500公里的电动车，因为碳酸锂价格飙升，仅碳酸锂成本就需要15000元，是去年初的至少十倍，“对于大量价格敏感的购车者来说，这个金额足以左右买还是不买”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在价格不是那么“刺眼”的商品上，价格敏感同样存在，其中，“国民辣酱”老干妈颇为典型。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>今年3月，媒体称，市场上反馈出来的消息是，一件24瓶装的产品，大概涨价20元，平摊下来，每瓶约涨价1元。老干妈之所以涨价，主要是因为原材料价格上涨，制作老干妈辣酱所用的辣椒、油、大豆等均出现成本上涨的情况。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不要小看“涨价约1元”，它照样能引起消费者“价格敏感”等反应。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于这一点，分析人士江瀚认为，从某种意义上来说，老干妈是一种准必需品，消费者对于此类商品的价格敏感度虽然不如米、面、油等产品，但“这次涨价，其实对于价格敏感型消费者的影响还是非常大的”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在“敏感”方面，包括饮料、洗发水、化妆品、纸巾、烟酒、乳品等多个品类的快速消费品堪称典型案例。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>上个月，贝恩公司与凯度消费者指数发布的《2022年中国购物者报告系列一》显示，2021年疫情相对好转，快速消费品销量呈现小幅度爆发的态势，相比于2020年增长4.1%，创下近4年来新高，但是销售额的增长幅度不及销量，仅有3.1%，说明平均售价有所下降，相比于2020年下降0.9%。而且，这并非平均售价第一次下降，2019~2020年已经下降1%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>销量增加，平均售价却接连下降，在上述报告看来，这是因为，消费者的思维模式和购物行为已经发生变化，“变得更加谨慎，对价格也更加敏感”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>与这份报告的分析可以相互佐证的是，《消费日报》统计了</span><strong><span>2021年消费领域的五大热词，分别是攒钱、平替、国潮、理性、断舍离，“五分之四都在谈克制”。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/67018130262d82927f0a677.52570941.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><em><span>图源消费日报官方平台</span></em></span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>这意味着，消费者形成了普遍性的价格敏感</span></strong><span>，媒体认为，即使是此前被称作“中国消费升级最中坚力量”的中产人士，也概莫能外——受到疫情、经济形势、就业等多种因素影响，“钱越来越难赚”，没有多少钱的消费者会更谨慎、更敏感，有一定经济基础的消费者也会有所克制，变得更理性、更务实。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从这个角度来说，在疫情爆发前，包括钟薛高在内的诸多快速消费品企业全力推动品牌高端化，不断提高商品价格，到疫情持续近三年的当下，仍然没有做出相应调整，被消费者“反噬”也就不难理解了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/70353183262d82928308fa5.89582693.gif" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span> 各有弹性空间 </span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>消费者对价格敏感，在不同的品类上会表现出不同的购买行为。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>《2022年中国购物者报告系列一》调查发现，对于饮料和包装食品，更多消费者选择购买促销产品；对于家庭和个人护理用品，会选择在需要时购买价格较低的产品，或者批量购买。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>一个明显“信号”是，</span><strong><span>越来越多的消费者倾向于在电商大促销活动期间“囤货”。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>数据显示，2021年第四季度快速消费品销量增长6%，相比前两季度实现反弹，背后正是第四季度双11、双12等大促销活动的推动。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>需要说明的是，</span><strong><span>大家更谨慎地花钱，并不是一味购买“便宜货”，而是在价格敏感的驱动下，营造了消费的弹性空间。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于这一现象，凯度消费指数大中华区总经理虞坚直言：“这不一定是消费降级，而是消费者的消费更加理性了。他们会选择品质好但价格更加实惠的产品，中国消费者在价格上涨面前非常务实。”</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>消费者营造属于自己的弹性空间，据新零售商业评论观察，品牌和企业也在竭力打造契合自身需求的弹性空间。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>以小米手机为例，小米近些年一直在进军“高端化”，但在价格敏感的国家和地区，其着力推出最低价格599元的相关机型。结果，这些聚焦价格敏感用户的机型进入“2021年全球最畅销的10款智能手机排行榜”，效果明显。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如果说小米高价、低价并举，那么，新茶饮品牌则是主动远离涨价。今年年初，喜茶、奈雪的茶和乐乐茶等纷纷宣布降价，承诺未来饮品将“告别30元”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除了品牌和企业，在消费弹性空间用力最狠的，当属一些互联网平台。说互联网平台“最狠”，是因为其利用算法，最大限度揣摩消费者的价格敏感度，如果价格敏感度高，算法会建议低价，对于价格敏感度低的顾客，则推荐高价。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>互联网平台此类做法被称为“大数据杀熟”，已经受到国家与地方的监管，去年出台的《平台经济反垄断指南》即明确提出这是违法行为。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当然，打造消费弹性空间，不只是具体的价格，还包括对消费者心理的洞察和掌握。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>举个例子，主打新鲜现卤的热卤具有小吃属性，在从业者看来，消费者对正餐有明确的心理价位，对小吃的预期则相对模糊，价格敏感度较低。这一点，被从业者作为赛道优势予以挖掘。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>即便如此，要稳妥应对消费者普遍性的价格敏感，品牌和企业要做得更多，行得更深。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/145413789662d829286e8144.63156549.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/36811932162d82928a418f7.61367463.gif" referrerpolicy="no-referrer"></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span> 组合拳最难打 </span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>对于品牌和企业来说，应对消费者的价格敏感，一不小心就会进入价格竞争，没有最低价，只有更低价。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>正因如此，在一场以酒类品牌如何应对竞争为主题的论坛上，当当创始人李国庆认为，互联网时代没有价格敏感区域和价格敏感商品，全网比价很透明，“事实上加剧了价格竞争，令价格竞争更加‘残酷’”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>认为没有价格敏感区域和价格敏感商品，李国庆只是提供了“一家之言”，但在消费者普遍价格敏感的背景下，如何直面竞争中的“残酷”，无疑是品牌和企业的重要命题。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>回头看，喜茶、奈雪的茶、乐乐茶等</span><strong><span>新茶饮品牌主动降价，并不是因为它们不想多赚钱，也不是要放弃长久运营的轻奢定位，而是试图用一套“组合拳”来实现高效发展。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>简言之，喜茶们通过供应链、品牌价值、规模优势等多方面控制成本，为产品降价打下基础，争取更多消费者的同时，也希望为自身发展带来更多确定性。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>研究中国购物者价格敏感等消费状况时，贝恩公司与凯度消费者指数也为企业提出一套“组合拳”打法：</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不仅要评估不同价位和城市级别市场所包含的机遇，基于不同的产品定位打造和优化产品、品牌组合，还要在管理成本结构压力的同时构建弹性供应链，更要扩展拼多多、直播、社区团购平台等新兴渠道，长期建立直接面对消费者的能力，整合线上线下渠道，打通线上线下一体化市场通路。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这套“应对方案”乍看起来不大复杂，但要真正实施，达到消费者买账、企业提质增收的效果，谁也不会低估个中难度。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>归根到底，在疫情、经济形势、就业等多种因素影响下，消费者普遍性的价格敏感短期内可能难以改变，品牌和企业的运营“组合拳”也不能立竿见影，需要长期淬炼。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>换句话说，直面价格敏感，目前为止，鲜有“赢家”。</span></p>
