<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-pid="bJBxb0qk">下面就一起来看，企业微信本次又更新了哪些超好用功能吧！</p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">01、可一键群发全部客户，微信客服可接入公众号</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-pid="9WHKRLa_"><strong>1）员工可一键群发内容给全部客户 </strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-pid="8y29VKP6">此前，员工使用群发助手进行群发，可一键选择200位客户，客户数量多的员工，需要操作多次，才能完成全量客户的触达。此次更新后，可一键选择全部客户，大大提升了一线员工的客户触达效率。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-pid="8g6GcL5Q">操作步骤：打开企业微信，点击「工作台」-「群发助手」-「开始群发」，即可一键勾选全部客户，进行群发。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-size="normal"><img src="https://pic1.zhimg.com/80/v2-d6062a11e45ddbc533b147800fb97968_720w.jpg" width="1080" data-size="normal" data-caption data-rawwidth="1080" data-rawheight="810" data-original="https://pic1.zhimg.com/v2-d6062a11e45ddbc533b147800fb97968_r.jpg" data-actualsrc="https://pic1.zhimg.com/v2-d6062a11e45ddbc533b147800fb97968_b.jpg" data-lazy-status="ok" title="企业微信更新：一键群发全量客户，可开通企业商户号！" alt="企业微信更新：一键群发全量客户，可开通企业商户号！" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;" data-pid="9MCx4Uey">图为群发助手功能操作流程</h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>2）微信客服功能接入公众号</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-pid="ay7UssTe">此次更新后，微信客服功能可接入公众号，只需要公众号管理员扫码授权即可接入。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-pid="jjLrS_Zs">接入后，客户向公众号发消息，公众号将自动回复微信客服名片，客户点击名片即可发起咨询；客户点击「联系客服」菜单也可直接发起咨询。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-size="normal"><img src="https://pic3.zhimg.com/80/v2-acc37d3695066607597e57d8ed9ef3fa_720w.jpg" width="1080" data-size="normal" data-caption data-rawwidth="1080" data-rawheight="643" data-original="https://pic3.zhimg.com/v2-acc37d3695066607597e57d8ed9ef3fa_r.jpg" data-actualsrc="https://pic3.zhimg.com/v2-acc37d3695066607597e57d8ed9ef3fa_b.jpg" data-lazy-status="ok" title="企业微信更新：一键群发全量客户，可开通企业商户号！" alt="企业微信更新：一键群发全量客户，可开通企业商户号！" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;" data-pid="7VOjgzQy">图为公众号接入微信客服后效果</h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-pid="cyIKjeln">操作步骤：电脑端管理后台，点击「应用管理」-「微信客服」-「在公众号中接入」-「去接入」，即可在公众号场景下，接入微信客服。此后，公众号的粉丝消息回复，将更加高效、及时。</p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>1）高效创建收款账户</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-pid="GDKbdBX6">在企业微信工作台的「对外收款」处，可创建收款账户。</p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-pid="onCGGHtw"><strong>2）员工使用对外收款功能，可生成收款码</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-pid="MyXbHLqJ">创建收款账户后，企业成员可生成收款码，向客户进行收款，收款金额将自动关联员工业绩金额，可实时查看业绩金额。</p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-pid="TMz6RrrT">当成员业务发生变更时，管理员可将他的客户群批量分配给其他成员继续提供服务；也可将指定客户群分配给其他成员服务。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-pid="JV8Q558q">场景示例：</p>

<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-pid="tdmA5We_">② 将员工A所负责的客户群，统一继承给员工B，进行管理</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;" data-pid="E3k16y7q">操作步骤：点击工作台「客户联系」-「在职继承」，选择需要转接的客户/客户群（可按成员选择，或直接指定客户/客户群），再选择接替成员，即可完成客户或客户群的在职继承。</p>
