<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;">免费模式正从<a href="https://www.niaogebiji.com/tag/C%E7%AB%AF" target="_blank">C端</a>面向<a href="https://www.niaogebiji.com/tag/B%E7%AB%AF" target="_blank">B端</a>。</h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">从“霸王餐”到“注册礼”，再到“免费试”……当C端已经不再对“免费试用”感到陌生，面向B端的企业也不得不开始思考，如何以这一模式扩大用户规模，打破僵局，同时实现双向的<a href="https://www.niaogebiji.com/tag/%E9%99%8D%E6%9C%AC%E5%A2%9E%E6%95%88" target="_blank">降本增效</a>。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">就连大众认知中最难搞定的工业、制造业，也已经有企业开始行动。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">由于托盘是各大工业企业中用处最广的物流单元，且近年来，随着物联网、大数据等技术的普及，托盘能够被赋予的功能也越来越多，不断从基础存储及运输物流延伸至信息追溯、计量单元乃至智慧供应链等领域，有托盘服务企业开始尝第一只“螃蟹”。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">提供专业托盘循环共享服务，创建了中国首个托盘智能共享循环模式的小蚁托盘，在其托盘覆盖城市规模日益增大的基础上，近日推出了“免费托盘”的活动，针对不同行业使用托盘的情况，构建了免费模式的托盘池，从而让各个行业都有了迈出“求变”第一步的可能。</p>
<p><strong>搭建托盘池，背后的“一石二鸟”之意</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">小蚁托盘启动“免费模式”，是以其发展为基础——从2019年开始深耕专业托盘循环服务的小蚁，历经三年，目前已经覆盖到了76个城市，但其中有一大半的城市都是只有货送进去，却没有货送出来，那么不同城市和地区之间的空拉托盘调度运费就成为了最大成本。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">近一年来，油费上涨加上疫情反复，运费涨幅大，空托调度成本更加高。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong> “如果没有突破性的举措，这个成本趋势在3-5年内很难产生根本性变化。”小蚁托盘方面分析。所以，在这个时间节点，小蚁托盘决定推出免费活动，背后有其“一石二鸟”之意，让“免费使用”用户帮忙解决空拉成本问题，其实也在挖掘更多资源，让更多用户参与到循环池中。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">“当循环逐步拓展到更多行业、更多地域之后，小蚁托盘自身的成本，特别是占比最大的空托盘运输成本也会进一步降低，进而使得循环池中用户的托盘使用成本也降低。这就像是一个转动的飞轮，每一个新客户的增加，都在带动整个组织的效率提升。”小蚁托盘的负责人侯凯表示，免费的终点是为了实现共赢。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">也正基于此，小蚁托盘的做法是在限定时间内，指定城市间的带托运输，用户可以免费使用托盘。小蚁托盘根据行业数据和用户情况，搭建了“免费活动”的“托盘循环池”。</p>
<section>
<section>据小蚁托盘方面介绍，由于目前行业内1210型托盘（托盘尺寸为1200*1000mm规格）占比40%以上，具有高流通性，小蚁循环池中的1210型托盘投入数量越来越多，成为了存量最大的；1111型号（托盘尺寸为1100*1100mm规格）紧随其后，此外还有1212型号、1140型号等川字、田字、双面托盘以及定制化组合托盘，托盘池规模庞大，基本上覆盖了大部分行业的需求。</section>
</section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/61994516862bc2550c32a07.77157424.png" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">“1210标准托盘为国标托盘，可使用的行业多，对于我们而言，那意味着在免费模式中容易找到更多的客户使用，使用率越高，运转率越高，单次循环使用成本更低。”侯凯告诉锌刻度，但这一类托盘在循环过程中面临一个阻力——一次性的1210标准托盘，具有使用残值，所以用户自己不用也能卖给其他使用二手托盘的用户，“但是二手托盘往往存在质量问题、货源也不够稳定，所以我们推出免费模式先让客户试用循环托盘，其实相当于给出了常规选项外的第三个选项。”</p>
<p><strong>从石化行业切入，以“托盘池”扩“用户池”</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">小蚁托盘循环之初是从化工、石化行业开启的共享之路，对化工、石化行业尤其了解，所以，在其搭建的托盘池中，不乏石化行业使用较多的1311型号以及化工、酒水、汽车化用1111型号或1140型号。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">“在这些行业我们涉足比较多，客户比较多，所以除了使用最为普遍的1210型号，这两个型号此前的空拉情况也较多。”小蚁托盘方面介绍称。但在免费活动的过程中，空拉的问题迎刃而解，同时也让不少相关行业的客户迈出了使用循环托盘的第一步。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>锌刻度注意到，在小蚁托盘的免费模式下，客户只需要通过其小程序平台查询到有哪些托盘可以免费用，然后电话联系小蚁托盘工作人员进行预约即可。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">值得一提的是，在整个过程中，因为小蚁芯片有空重识别功能，当自动识别到托盘由重变空，意味着托盘使用结束，小蚁托盘的管理平台会识别到信息，不需用户专门派人清点托盘，再通知小蚁回收，小蚁回收团队就可以主动上门回收，缓解用户工作量，提高托盘运转效率。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">广州货多多的孙经理向锌刻度谈到，“我们手里的订单从广州发往嘉兴的路程中，过去没有带托，都是靠人工搬，效率低，但胜在价格便宜。但自从广东出现严重的‘用工荒’情况，目前人工装卸1托（吨）大约需要30元以上（装12-13元/托.吨，卸17-18元/托.吨），华东、华南地区一线城市上浮50-80%，再加上嘉兴的收货人提出了带托运输的需求，我们也就开始关注起了托盘使用。”</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">免费托盘查询界面</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在了解市面上的托盘时，孙经理先后关注过二手废旧托盘和小蚁托盘这类新型环保材料托盘。可前者来源不稳定，质量较差，使用起来存在诸多不确定性，后者质量过硬，还搭载了一套智能终端，但唯独价格偏贵，一度劝退了孙经理。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">就在今年五一节，孙经理偶然看到了小蚁托盘推出的免费使用托盘活动，便积极报名成为了前期用户。使用时间仅一个月，但孙经理的想法却有了很大改变,“货损少多了，而且装卸效率也提升了不少。对于我们来说，购买一批托盘的成本太高，所以一直不敢轻易尝试，但现在大环境变了，免费使用的活动又能够让我们有机会深入了解和体验之后再订购，大家都乐意。”</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">而另一边，在郑州从事果汁生意的樊经理也有相同的体验，“我们本来就是冷链运输，装卸时间长了果汁容易坏，加上搬运工紧张，今年一直在考虑带托运输，看到小蚁托盘的免费使用活动，正好又有从郑州到南京的托盘，就开始使用了。”</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/12412351662bc2551564989.22419856.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;">小蚁托盘带托运输</h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">而事实上，运输过果汁和白酒的托盘，又都会进入石化、化工企业，“郑州的果汁运到南京后，托盘释放出来就会立刻进入化工巨头钟山化工的工厂。货多多的白酒运到嘉兴后，等待着托盘的也是一家上市公司——普利特。”侯凯解释称。</p>
<p><strong>瞄准潜在客户，从产品到数字化循环服务</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">小蚁托盘的“托盘池”中除了常规型号托盘，还包含定制化异型尺寸的组合托盘，这一部分托盘面向的往往是某些特殊行业，例如光伏。这类托盘尺寸非标准型号，往往不能通过租赁获得，而小蚁的介入，更侧重于帮助用户进行循环管理——这也是光伏企业等亟待解决的难题。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">在过去，这类特殊尺寸托盘用户都是一次性采购木托盘，不仅采购成本高而且光伏电厂多建立在地广人稀的偏远地区，很多用一次就坏了，没有坏的，用户自行回收成本又高。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>小蚁托盘采用复合材料，模块化的组合设计更加适合光伏行业：一来不用开模就能生产，很多异形尺寸都可以覆盖；二来便于维修，出现故障只需要找准损坏点，精准更换模块，能够大幅降低使用成本；第三利用小蚁的回收网络，循环使用也可以实现。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">“我们最近交付了小批量的2137MM*1145MM尺寸的托盘，用于清洁能源领域。”小蚁托盘光伏板块销售总监周总对锌刻度透露，“我们的托盘是循环托盘，用户不需要买，而是由我们直接配送到厂，用户使用完成之后再到发电厂回收，用户使用一次成本大约在60-90元。这不需要企业引进设备，只需要改变托盘使用方式即可，很多企业之前想做，但异型尺寸质量要求高，一般循环托盘都做不到，于是在接触到我们之后光伏企业才实现了这个关键点的成本节省。”</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong>而这仅是第一步，小蚁托盘提供的一套具有全程可视化能力的个性化数字解决方案，对于光伏企业完善和升级供应链来说，更称得上是一条捷径。</strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">基于小蚁托盘技术团队自主研发的托盘用智能芯片，能够实时记录运输路线和位置信息。如遇运输延误的情况，用户可以通过微信小程序第一时间得知延误信息，还可以根据需求设置时间偏差区域，及时给上下游发出预警。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/143655838762bc255199d141.59219694.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;">光伏企业晶澳使用小蚁托盘</h4>
<section>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">但B端企业决策链条较长，因此在变革的过程中需要一个捅破窗户纸的助手。而小蚁托盘的“免费活动”，就为企业提供了一次尝试窗户纸的机会。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">显然，从托盘本身，到背后的数字化服务，小蚁托盘此次将免费模式从C端搬到B端，有着双重意义。</p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;">一方面对于小蚁托盘自身而言，可以为客户提供更有竞争力的价格，且将更多潜在客户转化为固定客户；另一方面对于各个行业来看，当越来越多企业迈出“求变”的第一步，必然将加速推动行业变革进程。</p>
</section>
