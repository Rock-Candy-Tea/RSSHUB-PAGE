<section data-role="paragraph" data-width="92%">
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在今年这片屡创高温的盛夏战场，空调巨头格力再次登上了热搜榜。不过这次登上热搜的并非网红“董明珠”，而是一则</span><strong><span>“格力电器停止对河北经销商供货”</span></strong><span>的传闻。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>网传信息显示，格力电器已于近日停止对河北经销商供货，并开始在当地组建新的销售公司。消息一出，舆论哗然。</span><strong><span>在这各地高温关头，正是空调销售最旺季，格力为何与经销商闹掰了？</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/299743138630811c96a2739.43690496.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span>图源：微博APP截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>消息提到，在近日的一次活动中，河北格力总经销商</span><span>（河北新兴格力电器销售有限公司，下称河北格力）</span><span>董事长徐自发宣布“不做格力了”，转做飞利浦。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>哦，原来是一出经销商“叛变”的戏码，</span><strong><span>然而事实真的这么简单吗？</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><img src="https://ngbjimg.xy599.com/6251482662db2e883e36e5.24710351.png" referrerpolicy="no-referrer"></strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>格力也玩“二选一”？</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>综合多方消息了解，此次格力电器对河北经销商的突发停供始于8月16日，而在两天之后，河北格力则发布公告安抚各级经销商，称“即日起，已投款的经销商请尽快做计划并提货，河北公司货源可以保证及时供应”。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>经此公告，河北各级经销商暂时放下了悬着的心。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1362532898630811ca17b850.54412838.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span>图源：家电人</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过事情还没完，8月20日，也就是河北格力发布公告两天后，河北格力董事长徐自发宴请全省主流经销商，并在会上宣布，</span><strong><span>“自今天起退出格力的经营，不干格力了”</span></strong><span>。至此，才有了网络所流传的河北经销商不做格力，转投飞利浦的传言。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>但至此，仍有疑问，河北格力“叛变”或许可以用格力总部断供来解释，那么格力总部断供又师出何名？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>时间再往回拉，就在格力断供河北之前，为限制经销商同时代理销售飞利浦空调，格力电器曾祭出“二选一”政策，要求经销商若代理<a href="https://www.niaogebiji.com/tag/%E6%A0%BC%E5%8A%9B%E7%A9%BA%E8%B0%83" target="_blank">格力空调</a>，便禁止代理飞利浦。也由此，格力总部与河北格力之间的矛盾进一步激化。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在互联网市场，有关“二选一”政策我们见怪不怪。诸如阿里便因“二选一”行为被罚182亿元，美团也因“二选一”被罚34亿元。但在线下市场，“二选一”属实少见。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不过这也不能完全怪格力无情，正如网友所言，</span><strong><span>“这等于利用格力的渠道，推广飞利浦，这是任何厂家都不能够接受的吧！”</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1116707487630811caab9ef3.83151254.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span>图源：微博截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作为与格力电器利益相关的河北格力，转而通过格力渠道代理飞利浦，这不是砸总部的饭碗吗？自家店铺卖竞争对手的空调，“董小姐”当然忍不了。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>显然，如今的格力断供河北，也多是河北经销商在经总部警告后仍不加改变的后果。正如《让子弹飞》中的那句话：</span><strong><span>“给你体面，你不体面，那我只好帮你体面了。”</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而事已至此，仍未完结。河北格力总经销商为何要代理飞利浦？格力与飞利浦之间又有着怎样的瓜葛？背后同样有着诸多故事。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><img src="https://ngbjimg.xy599.com/176078547762db2e8f874706.61176433.png" referrerpolicy="no-referrer"></strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>格力与飞利浦的爱恨情仇</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>格力，中国空调行业龙头企业；飞利浦，则是2019年才进入中国空调市场的新兵。2021年，前者空调业务营收达1317亿元，而后者在年底传出的年度销售目标才仅有30亿。毫无疑问，二者体量完全不在一个量级。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>那</span><strong><span>河北格力为何冒着被格力电器舍弃的风险，也要代理飞利浦？</span></strong><span>这首先便要从一个人说起。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>徐自发，明面上看，他是河北格力总经销商董事长，掌握着格力在河北地区的总代理权。但背地里，它的身份并不简单。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/923262751630811cb43d8e0.04090901.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span>图源：飞利浦空调官方微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2021年年中，飞利浦空调中国市场的主要业务再被转手，由上海飞荔科技发展有限公司承接，并由其持股49%的“南京智浦”负责运营。而上海飞荔背后的实控人，则被认为正是河北格力董事长徐自发。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作为运营方投资人，在利益驱使下，作出力挺飞利浦、甚至不惜与格力决裂的行为也就不足为奇了。当然，徐自发与格力背后同样有着更深层关系，暂且按下不表。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>事实上，不仅徐自发倒戈飞利浦，格力所面临的“叛变”情况要比你想象中更严重。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2021年，原格力电器董事、执行总裁黄辉辞任。作为仅次于董明珠的格力“二号人物”，一直以来，黄辉始终被市场看作是董小姐未来的接班人。但没想到，黄辉在辞任后，很快便转投了飞利浦。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>此外，原格力总裁助理胡文峰、原董明珠秘书刘乙蓉等多人也纷纷离开格力，转投飞利浦的怀抱，出现在飞利浦运营总部的高管名单中。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这一幕，想必也深深刺痛了董明珠的神经。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1467023284630811cbdc69c1.11016764.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span>图源：格力电器官方微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>截止2021年底，飞利浦空调中国运营总部的团队成员共170余人，其中有约8成均曾在格力电器任职。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>如此旧部“倒戈”之痛，难怪董明珠咽不下这口气。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>无论是经销商，还是公司高管纷纷抛格力而去，格力与飞利浦之间早已势同水火。更不要说在格力的渠道上代理飞利浦空调，这无疑是在触碰董明珠的逆鳞。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>如今，格力与飞利浦间的爱恨情仇已捋清晰，但如此多的格力员工转投飞利浦，甚至河北格力经销商老大都不惜与格力决裂的背后，又有着怎样的深层原因？</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><strong><img src="https://ngbjimg.xy599.com/79261823162db2e91cb3478.62060819.png" referrerpolicy="no-referrer"></strong></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong><span>格力渠道变革之殇</span></strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>众所周知，在格力当年从珠海走向全国的道路上，其独特的经销商体系功不可没。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>甚至在2006年，为进一步拥抱供应商，格力电器在进行史上最大股权改制时，格力集团还向经销商团队联手成立的河北京海担保投资有限公司转让了10%的格力电器股份。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>至此，格力与3万家经销商利益高度绑定。</span><strong><span>二者一荣俱荣，一损俱损。</span></strong><span>凭借这一模式，格力多年来坚持线下战略不动摇，创下了空调行业的一个又一个神话。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/335669318630811ccbdb818.58689477.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span>图源：美的空调官方微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>只是如今，</span><strong><span>随着互联网渠道的崛起，美的空调借线上力量实现了对格力的反超，格力一家独大的格局被打破。危机之下，由线下到线上的渠道改革，成为了格力的头等大事。</span></strong><span>这一点，从董明珠不断开始直播带货、削减销售返利、推动新零售模式可见一斑。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>只是在这一转型过程中，曾经为格力攻城略地的3万家线下经销商，如今却成为其渠道变革之殇。在利益受损情形下，经销商与格力之间的裂隙昭然若是，当年牢不可破的联盟已貌合神离。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2020年，原山东格力销售总监苗培成立新公司，投奔美的；今年6月，格力经销商渠道联盟京海互联</span><span>（原京海担保）</span><span>减持1.1亿股格力股份。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而作为河北格力总经销商老大的徐自发，其关联公司作为第一大股东，在京海互联中占有28%的股份。这意味着，</span><strong><span>自此刻起徐自发正式开始从格力抽离，而后，其转投飞利浦。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>尽管在公开场合，董明珠曾多次强调“不抛弃3万家经销商”，要让大家逐步转型，适应线上销售的竞争。但在市场优胜劣汰下，有人掉队才是必然。在其渠道改革之路上，原有的铁杆经销商也因利益问题逐步淡出格力。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/1854815142630811cdb23ff0.38137425.png" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span><span>●</span>图源：格力电器官方微博</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而时至今日，格力的渠道改革还仍未结出果实。据赛迪研究院报告，2021年格力线上渠道收入占比仅在13%至14%左右，明显低于行业平均值52.9%。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>当年以经济利益而相互绑定的伙伴，如今恰因经济利益而分崩离析，不免让人感慨。而这一切也恰如古人言：</span><strong><span>天下熙熙，皆为利来；天下攘攘，皆为利往。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在这场河北格力“叛变”的背后，无外乎“利益”二字罢了。</span></p>
