<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>作为上半年的收尾之月，6月的抖音涨粉榜呈现出了何种面貌？我们一起来看。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/100954074162d56a076c7ff3.10218788.jpeg" referrerpolicy="no-referrer"></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>很显然，整个6月，东方甄选成抖音最大赢家。</span></strong></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>凭借着“双语直播”展现出的深厚文化氛围，@东方甄选不仅单月增粉1950.5W，近30日内的总销售额也高达7.14亿。<strong>而@东方甄选之外，<strong>@董宇辉、</strong>@东方甄选之图书、@俞敏洪三个相关账号均在此次涨粉榜上惊艳亮相</strong>，共同彰显着新东方在<a href="https://www.niaogebiji.com/tag/%E7%9B%B4%E6%92%AD%E5%B8%A6%E8%B4%A7" target="_blank">直播带货</a>之路上的突飞猛进。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/159887436362d56a07ab8295.30881997.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：@东方甄选之图书@董宇辉@俞敏洪</span></h4>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>抖音视频截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>除了大放异彩的新东方家族，6月涨粉榜的整体走势相较前几个而言，并未有太大起色。<strong>从创作者层面来看，依然没有恢复往日的风光。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>增粉量最高的创作者是人气王@疯狂小杨哥，6月共增粉483万。截至发稿时，账号粉丝总量已突破8000万，<strong>现已成为政务号之外、抖音粉丝量最多的达人</strong>。其后，资深健身达人@帕梅拉PamelaReif 以479.4W的增粉量成为榜单TOP4。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/79514994862d56a07d6a818.44856089.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：@帕梅拉PamelaReif抖音视频截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>剩余账号中，有9个账号的增粉量在200W-300W之间，之后的7个账号增粉量均在100W-200W之间。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>从上榜账号类型来看，共覆盖了运动健身、户外、旅行、剧情/搞笑、音乐、美食、颜值、母婴、时尚等十余个品类，其中剧情/搞笑赛道<strong>共有包括@疯狂小杨哥、@张小云、@大狼狗郑建鹏&言真夫妇在内的3个账号上榜；音乐赛道共有@同心乞527、@我杀猪了、@保安宿舍演唱会 3个账号上榜，</strong>优势相对明显，其余品类则分布得较为平均。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>整体来看，<strong>6月涨粉榜中熟面孔与新鲜面孔平分秋色</strong>。<span>@疯狂小杨哥、@大狼狗郑建鹏&言真夫妇（@广东夫妇）、@闲不住的阿俊、@刘畊宏、@董先生、@天元邓刚等账号均是多次上榜，显现出了顽强的生命力；而诸如</span></span><span>母婴账号@薯条来啦、剧情账号@张小云、旅行账号@阿波、知识账号@我是关键、音乐账号@同心乞 527、@我杀猪了、@保安宿舍演唱会等“新鲜面孔”也凭借特色化的内容风格，开始在榜单中崭露头角。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>接下来，我们将针对6月涨粉榜进行一些更为细致的分析。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span><img src="https://ngbjimg.xy599.com/6309740175f28bac1f33465.59120995.png" referrerpolicy="no-referrer"></span></p>
<h2 style="background-image: url('https://qnssl.niaogebiji.com//6cdd60ea0045eb7a6ec44c54d29ed402.png'); font-size: 18px; color: #212629; text-align: left; font-weight: 600; line-height: 30px; margin: 25px 0px 20px; background-size: 8px 20px; background-position: 0px 5px; background-repeat: no-repeat; padding-left: 12px;"><span><strong>@疯狂小杨哥&@广东夫妇：</strong></span><span><strong>老牌账号如何持续焕发生机？</strong></span></h2>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>前面我们提到，此次涨粉榜中有不少熟面孔的身影，在这之中，@疯狂小杨哥和@广东夫妇 可以称之为熟面孔中的老牌账号。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>据观测，这两个账号均是从2018年开始发布作品，如今走到第五个年头，人气依然居高不下。为何他们能不断焕发生机？卡思认为<strong>一方面源于他们在内容上的不断探索，另一方面则源于他们成功转型直播带货、开辟出了一片新天地。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>先看@广东夫妇。账号最早发布的内容是男主人公郑建鹏的个人展示向内容，各项数据都很惨淡。2018年5月6日，他们产出了第一条“老婆整蛊老公”的爆火视频，“女强男弱”的逗趣夫妻互动模式吸引了不少用户关注。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/97187478562d56a081440b5.21458657.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：@广东夫妇 抖音视频截图</span><span></span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这条视频爆火后，他们又发布了一条夫妻合拍的趣味内容，同样受到不少关注。基于此，他们找到了账号的创作方向，确立了合体出镜的内容形式、逐步塑造了包租夫妻、老公朴实“妻管严”、老婆华丽而威风等人设。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>结合着广州人的生活方式、风俗习性，他们创作出了很多具有鲜明个性和风格的内容类型：比如在不同情境之下，通过人前人后两种面貌形成强烈反差的【开门关门】系列；通过极具反差感的穿搭直观展示人物关系，再利用对话实现反差和反转的【走路系列】等。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/38251457262d56a08411637.41213912.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：@广东夫妇 抖音视频截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>靠着一把晃眼的收租钥匙，一口地地道道的广东话，一身不太起眼的行头，以及夫妻二人的完美配合，广东夫妇的账号在2019年10月底时就得以突破1000万。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>2020年年初，由于疫情原因无法出门拍摄，账号也经历了一段时间的不稳定期。直到2月下旬，女儿灵灵和夫妇二人合拍了一条疫情背景下的家庭短剧，古灵精怪的女儿、欢脱无厘头的老公、彪悍强势的老婆，再加之喜气洋洋的合家欢设定，使得账号再次迎来涨粉高峰，仅2020年3月增粉量就直逼1000万，总粉丝量也于3月中旬时突破了3000万。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在此之后，账号的内容形式变得更加丰富，有时是一家三口的家庭互动，有时是夫妻二人的收租日常或沙雕合拍，整体数据表现一直都维持着不错的水平。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/67209827762d56a08773e69.30354875.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：@广东夫妇 抖音视频截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在凭借内容吸引粉丝关注的同时，广东夫妇又在2020年开始了从达人向专业主播进阶的直播带货之路。时至今日，在交出一份又一份漂亮答卷的同时，他们的粉丝量也得以再次增长。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>而分析他们为何在开启直播带货之后人气还能持续上涨，<strong>除了在选品上的严格把控外，更重要的是他们对主播身份的适应程度。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>围观夫妇二人的直播过程，配合着欢快热烈的BGM，郑建鹏和言真会在镜头前密切配合，他们有时会轮流介绍产品，有时会合力出击，一人介绍时，另一人在旁边打辅助、做补充。具体介绍一款产品时，他们除了会熟练介绍产品的功效、适用人群、品牌背景、优惠力度等基本信息以外，<strong>还经常会结合夫妇二人的真实经历，从场景化的角度入手，来让用户更快地接受一款产品。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>不论是镜头表现力、节奏把握能力、还是介绍产品的能力，二人都表现得十分专业，<strong>而夫妇二人的直播风格、默契程度也和视频中呈现的整体风格相似，关联度极高，让人感觉十分亲切。</strong></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>再看人气王@疯狂小杨哥，也是从2018年开始在抖音崭露头角。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>我们目前在其账号中看到的第一支作品，拍摄的是小杨哥被女友抬手的动作吓得“面目全非”，反应过来后又强装镇定的故事，此时小杨哥的“贱男”形象已展露无疑。之后的很长一段时间，账号发布的内容也均是围绕着情侣间的互动展开，虽然也有一定受众，但也逐渐开始显露疲态。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>转折发生在2019年9月，小杨哥发布了一条发生在小杨哥、大杨哥和老爸三人之间的整蛊视频：小杨哥深夜玩游戏害怕被爸爸发现，请大杨哥帮忙盯梢，哥哥却联合爸爸狠整了小杨哥一把，网瘾少年、告密老哥和暴躁老爸的形象顷刻间深入人心。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/84656458262d56a08bbccd9.12879647.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：@疯狂小杨哥抖音视频截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>这条视频的点赞量和评论量相比之前的数据均有了大幅度的提升，小杨哥也就此找准机会，<strong>把视频内容从情侣段子转为家庭段子，从而迎来了账号粉丝量的暴增</strong>。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在后续发布的作品中，主人公小杨哥愈发“疯狂”，视频中他的表情动作日渐夸张，面对突发情况时的反应也越来越让人捧腹；再加上越来越“贱”的大杨哥，对儿子铁血、对老婆唯命是从的杨爸爸以及不爱说话但有极强威信力的杨妈妈，个性鲜明的一家人已然形成了极强的吸引力，再加上每个视频中都有一连串的反转，使得用户在“无语”与“好笑”之间反复横跳，十分上头。</span><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><img src="https://ngbjimg.xy599.com/28477102262d56a09356173.33017260.jpeg" referrerpolicy="no-referrer"></p>
<h4 style="background-image: none; font-size: 14px; color: #818386; text-align: center; font-weight: 300; line-height: 30px; margin: 25px 0px;"><span>图源：@疯狂小杨哥抖音视频截图</span></h4>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>在凭借着一家人嬉笑打骂的逗趣、温馨日常迅速吸引了大量粉丝的关注之后，小杨哥的又一次涨粉高峰则是因为他们开启了直播带货之路。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><strong><span>和一般的带货直播间不同，@疯狂小杨哥走的是一条无法被复制的“反向带货”路。</span></strong><span></span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>围观大杨小杨在直播间的状态，和他们在视频中的搞怪状态几乎毫无差别。两个人经常在直播间毫不留情地“大打出手”，惹得小杨哥发出阵阵嚎叫；此外，整场直播中，除了用“杨式专属风格”全方位展示产品特性外，大小杨更多时候是在和粉丝隔空“对峙”，这之中诞生的金句、爆梗数不胜数。</span><span>比如两人在卖一款纸巾时，粉丝问“没有牙可以买吗”；在卖一款羊奶饮品时，粉丝问“是喜羊羊产的吗”“公羊还是母羊”；卖沙琪玛时，粉丝会问“杀的什么马”…每次直播中几十个回合的推拉下来，兄弟俩往往被“折腾”得暴躁不已，甚至气到在直播间后方打沙包。</span></p>
<p style="background-image: none; font-size: 16px; color: #000000; text-align: inherit; font-weight: inherit; line-height: inherit; margin: 25px 0px 20px;"><span>整体来看，除了在货品端更多地选择了零食饮料、日用百货这种客单价低、较为讨巧的刚需消费品外，@疯狂小杨哥 直播间的最大特点就是趣味性、互动性极强，极大地满足了用户的娱乐消遣需求。</span><span></span></p>
