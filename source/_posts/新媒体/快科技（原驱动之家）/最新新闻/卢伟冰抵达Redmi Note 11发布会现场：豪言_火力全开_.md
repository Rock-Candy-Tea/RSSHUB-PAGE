<p>今天晚上19:00，Redmi Note 11系列将正式发布。</p>
<p>小米集团中国区总裁、Redmi品牌总经理卢伟冰<span style="color:#ff0000;"><strong>已经抵达Redmi Note 11发布会现场，表示“今晚火力全开”。</strong></span></p>
<p>作为Note系列新成员，Redmi Note 11系列至少有两大看点，一是续航。</p>
<p>据报道，<strong>Redmi Note 11系列除了有4500mAh电池版本外，还提供了更大容量电池版本。</strong></p>
<p>卢伟冰表示，这次Redmi对天玑920的调校+极佳的架构设计+MIUI的全面整改优化，<span style="color:#ff0000;"><strong>让Redmi产品续航再上一个大台阶。</strong></span></p>
<p>第二大看点是快充，该机支持120W神仙秒充。目前全球支持120W快充的手机仅有14款，平均售价都在4000元以上，是高端旗舰才会支持的一项技术。这意味着Redmi Note 11系列将是同档位唯一支持120W快充的手机，极具竞争力。</p>
<p>当然，<strong>高刷AMOLED全面屏、WiFi 6、双扬声器、NFC等都没有缺席，</strong>更多细节会在发布会上揭晓。</p>
