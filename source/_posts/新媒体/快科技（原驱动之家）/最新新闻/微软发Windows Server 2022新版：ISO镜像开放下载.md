<p>现在，微软发布了下一个Windows Server长期服务渠道（LTSC）<a class="f14_link" href="https://techcommunity.microsoft.com/t5/windows-server-insiders/announcing-windows-server-preview-build-22538/m-p/3066141?irgwc=1&OCID=AID2200057_aff_7808_1245846__fuist0kwkokfqluj0hdhyki1fm2xobe1lg09lqgo00&tduid=(ir__fuist0kwkokfqluj0hdhyki1fm2xobe1lg09lqgo00)(7808)(1245846)(19519854)(242986)&irclickid=_fuist0kwkokfqluj0hdhyki1fm2xobe1lg09lqgo00" target="_blank">预览版的新版本</a>，并且已经提供一些用户下载使用。</p>
<p>微软以ISO格式提供18种语言的Windows Server长期服务渠道预览版，并以VHDX格式提供仅有的英文版本。</p>
<p>该更新可在public symbol服务器上使用，在可用时，有 Azure 订阅的Insider也可以在Azure市场的微软服务器操作系统预览中测试服务器产品的预览。</p>

<p>Datacenter：2KNJJ-33Y9H-2GXGX-KMQWH-G6H67</p>
<p>Windows Server 2022是服务器客户的下一个长期服务渠道（LTSC）版本，在具体更新频率上，微软将在两到三年内发布一次新版本的操作系统，这将是LTSC版本，同时这些版本与客户端LTSC版本不同，将获得 10 年的支持，包括5年的主流技术支持，然后是5年扩展支持。</p>
<p>Windows Server 2022支持高达48TB的内存和2048个逻辑核心，在64个物理插槽上运行。该操作系统还建立在Windows容器的应用兼容性特征之上，支持 IPv6 和双堆栈，以及用于配置节点的 HostProcess 容器。</p>
