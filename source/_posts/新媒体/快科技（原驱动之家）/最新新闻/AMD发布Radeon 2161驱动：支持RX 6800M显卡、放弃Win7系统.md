<p>微软马上就要发布Windows 11系统了，Win10发布都6年了，Win7系统更是12年前的老古董了。前几天NVIDIA显卡驱动放弃了Win7，现在AMD发布了Radeon Software Adrenalin显卡驱动21.6.1，也跟Win7说再见了。</p>
<p><strong>AMD Radeon Software Adrenalin 21.6.1版显卡驱动主要为本月初发布Radeon RX 6800M笔记本显卡提供支持，</strong>而搭载该显卡的华硕ROG Strix G15、HP OMEN 16等笔记本也已陆续曝光。</p>
<p>此外，新版驱动还为游戏《龙与地下城：黑暗联盟》提供了优化支持，并更新了支持AMD FidelityFX Super Resolution技术的游戏列表，包含《生化危机：村庄》、《孤岛惊魂6》、《魔兽世界：暗影国度》、《Godfall》、《尘埃5》等，具体列表请查看AMD网站。</p>
<p>另外，从此版本开始，<span style="color:#ff0000;"><strong>AMD不再发布Windows 7系统驱动，Windows 7系统用户请使用21.5.2版本。</strong></span></p>
<p><strong>21.6.1版驱动Bug修复列表：</strong></p>
<p>1、关闭游戏并切换主副显示器会导致Radeon FreeSync间歇性的锁定，从而导致系统性能下降或者卡顿。</p>
<p>2、《纪元1800》在DX12模式下启动时会发生崩溃。</p>
<p>3、AMD Cleanup程序删除AMD Chipset/RAID相关驱动文件。</p>
<p>4、与早期驱动相比，部分显卡在《命运2》游戏中性能低于预期。</p>
<p>5、加入AMD用户体验计划后，该服务进程的CPU占用率过高。</p>
<p>6、《无限法则》在DX12模式下开启光线追踪会导致游戏崩溃。</p>
<p><strong>部分已知问题：</strong></p>
<p>1、使用Radeon VII显卡时，《生化危机8：村庄》在游戏开始时可能会挂起或发生TDR。</p>
<p>2、启用增强同步会导致部分游戏或系统黑屏。遇到这一问题的用户请禁用增强同步功能。</p>
<p>3、Radeon RX Vega系列显卡在连接两个分辨率/刷新率差距较大的显示器时会出现闪屏。</p>
<p>4、Radeon performance metrics显示过高或错误的闪存频率。</p>
<p>5、使用Radeon RX 5000/6000系列显卡时，Oculus服务可能会报错。</p>
<p>6、安装Adrenalin软件后若未检测到Ryzen Master，则可能需要重新启动系统。</p>
<p>7、如果在移动系统中出现蓝屏或黑屏，请暂时禁用Windows Hello增强型登录功能。</p>
<p>8、若系统中同时安装了Windows Store和AMD官方两个版本的Radeon软件时，可能会出现驱动程序不匹配的情况，请启动Windows Store版本暂时解决。</p>
<p>9、AMD正在调查一个可能对采用AMD Ryzen 5000系列平台的笔记本电脑的电池寿命产生轻微影响的问题。</p>
<p>10、更新到最新的Radeon软件后，可能会出现蓝屏或黑屏，请禁用Windows Core Isolation（核心隔离）功能。</p>
<p>11、AMD正在调查更新到最新Radeon软件后可能在某些主板上提示D3错误。</p>
<p>12、部分AMD Athlon移动平台可能出现系统性能下降。</p>
<p><strong>Radeon Software Adrenalin 21.5.2 WHQL驱动升级说明</strong></p>
<p>发布Radeon Software Adrenalin 21.6.1的同时，AMD也同时发布了Radeon Software Adrenalin 21.5.2的WHQL版本，更新内容和之前的测试版一样，主要为微软DirectX 12 Agility SDK和Shader Model 6.6提供支持。</p>
<p>AMD显卡用户可以体验到包括光线追踪、采样器反馈、网格阴影、可变速率阴影在内的DX12 Ultimate的所有功能。</p>
<p>此外，新版驱动还为《Days Gone》游戏进行了优化，并修复了之前版本的部分错误，具体如下：</p>
<p>1、使用出厂设置选项升级或者卸载Radeon Software时，若文件夹内含AMD Chipset芯片组驱动文件，则会同时删除。</p>
<p>2、使用Radeon RX 6000系列显卡时，在《赛博朋克2077》中开启光线追踪可能会出现阴影错误。</p>
<p>3、Radeon RX 5000系列显卡在AMD Link中使用HEVC编码时，流媒体优化会间歇性断开。</p>
<p>4、Radeon Software软件中的Ryzen Master按钮会消失或不起作用。</p>
<p>5、使用Ryzen Mobile 4000、5000系列处理器时，会发生FPS日志记录错误。</p>
<p>6、热插拔显示器会导致Radeon Software录制和流媒体功能崩溃。</p>
