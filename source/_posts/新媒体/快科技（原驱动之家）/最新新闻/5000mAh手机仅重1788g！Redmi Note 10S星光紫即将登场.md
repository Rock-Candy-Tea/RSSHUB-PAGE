<p>今天，小米推特官方宣布，Redmi Note 10S新配色星光紫即将登场。</p>
<p>目前Redmi Note 10S有三种配色在售，具体为灰色、蓝色和白色，<span style="color:#ff0000;"><strong>这次加入了全新的星光紫配色。从海报来看，Redmi Note 10S星光紫为渐变设计，颜值出众。</strong></span></p>
<p>核心配置上，<strong>Redmi Note 10S采用6.43英寸AMOLED挖孔屏，搭载联发科Helio G95处理器，配备LPDDR4X内存、UFS 2.2闪存，</strong>后置6400万主摄、800万超广角、200万景深和200万微距四摄，电池为5000mAh，支持33W快充。</p>
<p>官方介绍，Redmi Note 10S 30分钟可以充电50%以上，78分钟就能充满这块5000mAh大容量电池，满足一整天的需求。</p>
<p>值得注意的是，<span style="color:#ff0000;"><strong>Redmi Note 10S还是一款极其轻薄的千元机，机身重量只有178.8g，厚度为8.29mm。</strong></span></p>
<p>目前官方尚未公布其价格，预计新配色与其它版本价格保持一致。PS：Redmi Note 10S在印度起售价为14999印度卢比（约合人民币1300元）。</p>
