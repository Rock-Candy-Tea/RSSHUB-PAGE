<p>昨晚，小米正式在海外市场发布了全新版本的Redmi Note 11和Redmi Note 11S机型，并且还更换了与国内完全不同的配置。</p>
<p><strong>具体来看，Redmi Note 11处理器由原本的天玑900更换为了高通骁龙680，基于6nm工艺制程打造，</strong>采用Kryo 265 CPU架构，最高主频达到了2.4GHz，GPU为Adreno 610。</p>
<p>官方称，这款处理器旨在提供引人入胜的全天体验，包括优化的游戏和采用AI增强型微光捕捉技术的三重ISP。</p>
<p align="center"><a href="https://img1.mydrivers.com/img/20220127/e38dc4779594476b8dff116d6f39f108.jpg" target="_blank"><img alt="千元之王！Redmi Note 11新版发布：换高通芯、起售价不到1200" h="300" src="https://img1.mydrivers.com/img/20220127/s_e38dc4779594476b8dff116d6f39f108.jpg" style="border: black 1px solid;" w="600" referrerpolicy="no-referrer"></a></p>
<p>Redmi Note 11S搭载的是联发科Helio G96处理器，这款处理器为8核设计，包括两个主频为2.05GHz的Cortex-A76内核，支持LPDDR4x和UFS 2.2，支持HyperEngine 2.0 Lite游戏技术，还支持双4G。</p>
<p>除了处理器的变化之外，<span style="color:#ff0000;"><strong>Redmi Note 11新版与Redmi Note 11S均配备了一块90Hz FHD+AMOLED屏幕，这与此前国内的LCD屏幕有很大不同。</strong></span></p>
<p align="center"><a href="https://img1.mydrivers.com/img/20220127/dbd91215157943d8b7e026f6ee5e8b15.jpg" target="_blank"><img alt="千元之王！Redmi Note 11新版发布：换高通芯、起售价不到1200" h="300" src="https://img1.mydrivers.com/img/20220127/s_dbd91215157943d8b7e026f6ee5e8b15.jpg" style="border: black 1px solid;" w="600" referrerpolicy="no-referrer"></a></p>
<p><strong>两者均内置5000mAh超大容量电池，改为33W快充，国内最受青睐的67W快充没了，</strong>双扬声器和3.5mm耳机接口保留。</p>
<p>影像方面，Redmi Note 11S搭载的是108MP的后置主摄，而Redmi Note 11搭载的是50MP的后置主摄。</p>
<p>价格方面，Redmi Note 11售价179美元（约1133.07元人民币）起，Redmi Note 11售价249美元（约1576.17元人民币）起。</p>
