<p>前几天，Redmi已经宣布将于10月28日晚19:00召开新品发布会，正式发布Redmi Note 11系列，官方已经针对该机进行了连续几天的预热。</p>
<p>今天上午，Redmi终于针对大家最关心的参数进行了预热，<span style="color:#ff0000;"><strong>宣布Redmi Note 11将首发联发科天玑920芯片。</strong></span></p>
<p>据悉，天玑920虽然是一款中端芯片，但是却采用了旗舰芯片同样的台积电6nm制程工艺打造，整体的能耗比控制非常优秀，且配备了A78大核，能提供强有力的性能输出，<strong>安兔兔跑分超过50万，逼近高通最强中端芯骁龙778G。</strong></p>
<p align="center"><a href="https://img1.mydrivers.com/img/20211026/4f6f67ec2fb7475d836b57a259bd7ad2.jpg" target="_blank"><img alt="6nm旗舰工艺！Redmi Note 11最神秘参数揭晓：搭载天玑920芯片" h="799" src="https://img1.mydrivers.com/img/20211026/s_4f6f67ec2fb7475d836b57a259bd7ad2.jpg" style="border: black 1px solid;" w="600" referrerpolicy="no-referrer"></a><br>
天玑920跑分超50万 性能直逼骁龙778G</p>
<p>值得一提的是，Redmi Note 11还为这块芯片配备了超大面积VC液冷，配合上6nm的旗舰工艺，能够保证持久稳定的性能输出，<strong>并且支持《王者荣耀》的90帧模式，</strong>表现十分亮眼。</p>
<p>除了采用了旗舰工艺的芯片之外，<span style="color:#ff0000;"><strong>旗舰级别的1亿像素相机也正式回归Note 11系列。</strong></span></p>
<p>据介绍，Redmi Note 11系列1亿像素影像科技拥有三大特性：<strong>高像素、暗光好、更纯净，分辨率高达12000X9000，9合1技术，单像素大至2.1μm，进光更多，在双ISO技术加持下，拍出的夜景更加纯净，噪点更少。</strong></p>

Redmi Note 11搭载1亿像素主摄</p>
<p>官方透露，Redmi Note 11搭载的是三星HM2传感器，这也成为了2K元档最大底、最高像素传感器，此前甚至被友商3K档产品采用，这次Redmi Note 11将用这颗镜头实现影像越级。</p>
<p>另外，<strong>Redmi官微今天还公布了一款联合王一博特调的“浅梦星河”新配色，采用梦幻的玻璃机身，加上独特的闪砂效果，官方称其为“闪光少女风”，</strong>相比此前已经曝光的绿色、紫色，浅梦星河闪耀的外观更适合女性用户。</p>
<p>值得注意的是，Redmi Note系列甚至Redmi品牌，此前都是更加偏向于男性审美，外观方面甚至被成为直男设计，但是此次却更加看重女性用户，推出了这款专属配色，结合潮流小立边设计，<strong>不论是设计还是配色，都堪称是Note系列的巅峰，</strong>应该会受到不少女性新用户的追捧，值得期待。</p>
