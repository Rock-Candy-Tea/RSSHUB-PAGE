<p>2018年8月，三星推出了旗舰定位的Galaxy Note 9，并被Android Authority评为了当年的最佳Android手机，在当时是当之无愧的“机皇”。</p>
<p>对于这款往昔机皇，三星也算得上的下足了功夫，在2020年为Galaxy Note 9推送了基于Android 10的One UI 2.5版本后，一直维持每个季度一次安全更新的频率进行维护。</p>
<p>近日，三星发布了9月的安全更新，并列出了此次更新所支持的全部设备，<span style="color:#ff0000;"><strong>与此前更新不同的是，这次更新名单上删去了Galaxy Note 9。</strong></span></p>
<p>这意味着，<strong>三星正式停止了对于Galaxy Note 9的官方支持，不会再每个季度都收到安全更新，正式退出了舞台。</strong></p>
<p>与此同时，<strong>三星也将Galaxy Note 10系列的更新频率从每月降低到了每季度</strong>，显然，这款Galaxy Note 9的继任者，将是三星下一款放弃维护的旗舰产品。</p>
