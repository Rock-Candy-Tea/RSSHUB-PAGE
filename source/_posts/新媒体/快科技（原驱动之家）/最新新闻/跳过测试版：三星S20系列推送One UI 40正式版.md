<p>根据外媒的消息，三星系统Beta测试版运营经理透露，<span style="color:#ff0000;"><strong>三星Galaxy S20系列，下一步将不会推送One UI 4.0测试版，而是直接推送正式版，预计推送时间在2022年1月开始。</strong></span></p>
<p>这也将是继S21系列之后，下一款获得最新正式版系统的三星手机。</p>
<p><strong>按照三星的推送计划，Galaxy Z Flip 3、Galaxy Z Fold 3、Galaxy Note 20、Galaxy S10等机型，随后也会获得One UI 4.0的稳定版推送。</strong></p>
<p style="text-align: center"><a href="https://img1.mydrivers.com/img/20211226/5ad01f42-01ad-4e01-9048-9c8a3453a01b.png" target="_blank"><img alt="跳过测试版：三星S20系列推送One UI 4.0正式版" h="243" src="https://img1.mydrivers.com/img/20211226/S5ad01f42-01ad-4e01-9048-9c8a3453a01b.png" style="border:black 1px solid" w="600" referrerpolicy="no-referrer"></a></p>
<p>One UI 4.0基于Android 12打造，界面采用了Android 12的Material You设计语言，三星还将内置的大量原生应用进行重新设计，带来了新的功能和交互动画，并且还去掉了广告，体验上会更加丝滑流畅。</p>
<p>在安全性方面，三星Knox也会获得更新，对于骁龙888、Exynos 2100处理器的优化也将更进一步完善。</p>
