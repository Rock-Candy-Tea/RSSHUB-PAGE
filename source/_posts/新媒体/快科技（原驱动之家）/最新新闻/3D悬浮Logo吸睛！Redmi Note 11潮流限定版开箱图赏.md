<p>在上个月的Redmi Note 11系列发布会上，Redmi在中杯、大杯及超大杯三款手机之外，还带来了一款特别拉风的机型：Redmi Note 11潮流限定版。</p>
<p><strong>该机将于11月11日上市发售，由于成本极高，Redmi Note 11潮流限定版的价格也更高，8+256GB版定价2699元</strong>，比同规格Redmi Note 11 Pro+贵了400元。</p>
<p>现在，这款新机已经来到我们评测室，一起来瞧瞧吧。</p>
<p>这款手机基于Redmi Note 11 Pro+打造，全球首发全息悬浮工艺。而且机身有Redmi品牌代言人王一博签名，整机极具辨识度。</p>
<p>据悉，<strong>Redmi Note 11潮流限定版在18cm²的空间中有着11万个微米级Redmi Logo，通过11万个对应的微透镜，最终放大150倍，汇聚为一，呈现出3D悬浮感的立体Redmi Logo。</strong></p>
<p>Redmi Note 11系列，凭借首发联发科天玑920 5G处理器，三星120Hz高刷屏、旗舰级120W神仙秒充，该系列获得众多业内人士与消费者关注。</p>
<p>其中最大亮点无疑是120W神仙秒充，成为目前Redmi充电速度最快的机型，仅15分钟即可充满。</p>
