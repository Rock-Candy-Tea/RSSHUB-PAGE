<p>微软已经暗示接下来的Windows 11重大更新细节，其代号为“Sun Valley 3”，其实在多份微软官方支持文档中提及“Version 23H2”功能更新之后，在最新微软网络直播中提及了“Sun Valley 3”、“SV 3”字眼。</p>
<p><strong>Windows 11 系统中的“refined investments”（精炼投资），表明在明年下半年推出的 Windows 11 23H2 上将会有明显的功能改进。</strong></p>
<p style="text-align: center;"><a href="https://img1.mydrivers.com/img/20220418/27bad7feece74041899b3b59b185b446.png" style="text-align: -webkit-center;" target="_blank"><img alt="微软自曝Windows 11重大更新23H2：功能改进明显" h="363" src="https://img1.mydrivers.com/img/20220418/s_27bad7feece74041899b3b59b185b446.png" style="border: 1px solid black;" w="600" referrerpolicy="no-referrer"></a></p>
<p>日前，在 Windows Server build 25075 中的 appraiserres DLL 文件文件中，将 Windows 11 23H2被称呼为Copper（CU）版本，而第一行字眼中明确显示 Windows Insider Dev频道中的当前版本是代号镍（NI），也就是22H2。</p>
<p>现在谈论明年的新功能还为时过早，但Sun Valley 3很可能会解决新任务栏和平板电脑体验的不足之处。你可能知道，Windows 11 没有一个成熟的任务栏或平板电脑模式，但这可能在明年改变。</p>
