<p>距离 Windows 11 发布已经过去一周，虽然比起苹果发布会，Windows 11 连热搜榜都没能登上，但这并不代表没人关注。</p>
<p>相反，在过去一周里，很多数码爱好者围绕其对 Android 应用的支持、对老机型的升级问题展开了激烈讨论。</p>
<p>与以往不同的是，在这次大版本升级中，微软对老电脑的态度与以往截然不同，这也引发了很多争议。</p>
<p><strong>微软终于抛弃了老电脑</strong></p>
<p>对于很多人来说，第一次使用电脑接触到的就是 Windows 系统，这也是很多人最熟悉的操作系统。</p>
<p>时至今日，尽管面临着苹果和谷歌的强力竞争，但 Windows 还能把握住全球 75% 左右的市场份额，这背后的原因，除了出色的兼容性外，也离不开海量老机型所奠定的坚实基础。</p>
<p>举个有点夸张但又很真实的例子，<strong>有些人把一台电脑从 WinXP 一路用到了 Win10，如果换作其他系统应该很难做到。</strong></p>
<p>然而，随着 Windows 11 的发布，这种“好事”也许不会再延续下去了。</p>
<p style="text-align: center"><a href="https://img1.mydrivers.com/img/20210703/063f17dc-e782-46e6-91dd-d92e89841b5b.png" target="_blank"><img alt="Win11无情抛弃老电脑：微软搞的什么鬼？" h="374" src="https://img1.mydrivers.com/img/20210703/S063f17dc-e782-46e6-91dd-d92e89841b5b.png" style="border: black 1px solid" w="600" referrerpolicy="no-referrer"></a></p>
<p>Windows 11 发布后，微软一改往常对老机型开放的态度，只是在发布会上并没有特别说明，直到很多用户升级时才发现各种问题。</p>
<p>有意思的是，如果单看官方给出的要求，其 1GHz 处理器、4GB 内存、64GB 硬盘容量的最低配置标准并不过分，但大多数用户不了解的是，<strong>这次还要求电脑具备 TPM 2.0 模块，而这也是老机型面临的最大阻碍。</strong></p>
<p>TPM 2.0 标准于 2016 年制定，近两年才被大范围应用到产品上，据微软称，TPM 2.0 可以提供硬件级的隔离和加密，在安全性方面相比之前可以得到更有效的保障。</p>
<p style="text-align: center"><a href="https://img1.mydrivers.com/img/20210703/93e6e772-6b4c-44a3-ad0c-0c26c5565996.png" target="_blank"><img alt="Win11无情抛弃老电脑：微软搞的什么鬼？" h="275" src="https://img1.mydrivers.com/img/20210703/S93e6e772-6b4c-44a3-ad0c-0c26c5565996.png" style="border: black 1px solid" w="600" referrerpolicy="no-referrer"></a></p>
<p>除了 TPM 2.0 外，微软还对 CPU 提出了更严苛的条件，在官方给出的支持列表中，8 代酷睿和 2 代锐龙之前的型号均未出现，再加上 TPM 2.0 的限制，也就是说<strong>只有 2017 年后发布的电脑才能完美运行 Windows 11，但实际上单从性能角度来讲，很多 2017 年之前的产品也能够完美运行，只是微软并没有给出明确支持的态度。</strong></p>
<p>更有意思的是，在预览版系统推送后，很多 2017 年前老电脑通过“民间破解”方法得以升级，于是微软又补充道，尽管这些老型号支持升级，但并不符合 Windows 11 对于安全性和可靠性的要求。</p>
<p>总之，Windows 11 对老机型的支持标准模糊不清，但确实可以感觉到，这次微软想要摆脱老机型的束缚。</p>
<p><strong>旧的不去，新的不来</strong></p>
<p>对微软而言，支持老机型能够维持其庞大的用户体量，但从另一个角度讲，当一个新系统面对海量老硬件时，无疑也加大了适配难度和维护成本。</p>
<p><strong>近几年来，Windows 系统的口碑相比 WinXP 和 Win7 时期下降了很多，不少用户传来了抱怨声，吐槽蓝屏、死机、字体模糊、广告弹窗等现象的越来越多，但实际上这些问题中，Windows 自身原因造成的并不多。</strong></p>
<p>就拿蓝屏来说，除了微软偶尔推送“咖喱味”补丁导致全球用户大批量蓝屏，更多的应该是老硬件不适配、或者驱动程序不兼容这种个性化问题，而老机型往往更容易出现这种现象。</p>
<p style="text-align: center"><a href="https://img1.mydrivers.com/img/20210703/ec613347-ce93-4b18-be2d-76a9df073279.png" target="_blank"><img alt="Win11无情抛弃老电脑：微软搞的什么鬼？" h="374" src="https://img1.mydrivers.com/img/20210703/Sec613347-ce93-4b18-be2d-76a9df073279.png" style="border: black 1px solid" w="600" referrerpolicy="no-referrer"></a></p>
<p><strong>因此，微软提高“最低配置”也是一种无奈之举，虽然很多老硬件因此被淘汰，短时间内也会遭到不少用户的不满，但至少，微软之后可以把精力放在近几年和未来的新硬件上，以此来提升 Windows 电脑的体验，当然，前提是在微软给力的情况下，否则只是美好的空想。</strong></p>
<p>目前，Windows 11 在老硬件适配上依然踌躇不定，用户可以通过修改注册表、或者是替换 dll 文件等方法强制进行更新，但等到秋季正式版面市时，微软很可能加大限制力度。</p>
<p style="text-align: center"><a href="https://img1.mydrivers.com/img/20210703/298cbe74-1bfe-4487-bc6d-3e0f0ea5782e.png" target="_blank"><img alt="Win11无情抛弃老电脑：微软搞的什么鬼？" h="431" src="https://img1.mydrivers.com/img/20210703/S298cbe74-1bfe-4487-bc6d-3e0f0ea5782e.png" style="border: black 1px solid" w="600" referrerpolicy="no-referrer"></a></p>
<p>有意思的是，当我们上手 Windows 11 预览版时，发现系统版本号为“21H2”，即延续了 Win10 时期的命名规则，联想到此前微软曾表示 Windows 10 将是最后一版 Windows，不妨假想如今的 Win11 其实就是 Win10 21H2 版本，只是出于对“升级换代”的渴求，微软做出了重命名这种“违背祖训”的决定。</p>
<p>实际上从功能方面讲，除了支持 Android 应用，你很难再找出能够称之为“代际升级”的改变了，从这一角度来说，改名 Windows 11 也许是为了增强用户的换机动力，毕竟历代大版本更新也许让微软明白了“科技以换名为本”。</p>
<p><strong>不破不立，有舍有得</strong></p>
<p>除去提升“最低配置”，Windows 11 也对更进阶的“推荐配置”做出了更高的硬件要求，比如 5G 网络、HDR 显示器、更好的收录音设备、更多的交互传感器、更大更快的固态硬盘，这些功能在基础之外可以提供更好的体验。实际上每代 Windows 大版本更新都会推动未来几年内的硬件升级，Windows 11 也不例外。</p>
<p>未来几年，随着 Windows 11 装机量的提升，PC 产品可能会迎来更大的变化，尤其是触控很可能会得到率先普及。微软从 Windows 7 开始支持多点触控，Windows 8 更是为触控操作单独设计了 Modern UI，但多年来 Windows 的触控却始终处于一个尴尬境地，毕竟在桌面应用中，键鼠的效率远比触控更高。</p>
<p>但从 Windows 11 开始，随着 Android 应用的引入，触控功能肯定会得到更大的发挥。事实上在 Windows 预览版中，很多 UI 设计也更倾向于触控操作了，例如菜单间距明显变宽，控制中心的控件、资源管理器中常用的复制、粘贴、重命名等操作也被设计为更适合点按的图标形式。</p>
<p style="text-align: center"><a href="https://img1.mydrivers.com/img/20210703/bd1a0040-9045-4158-8c2c-96f9afc6fec6.png" target="_blank"><img alt="Win11无情抛弃老电脑：微软搞的什么鬼？" h="416" src="https://img1.mydrivers.com/img/20210703/Sbd1a0040-9045-4158-8c2c-96f9afc6fec6.png" style="border: black 1px solid" w="600" referrerpolicy="no-referrer"></a></p>
<p>除去新功能的加入，还有一些曾经很重要的功能终被弃用，留给我们印象最深的应该是拥有二十多年历史的 IE 浏览器，这次 IE 浏览器将从 Windows 11 中彻底消失，曾经的它与 Windows 深度捆绑，甚至登上了 Windows 95 的开机画面，微软还因此被扣上了“垄断”的帽子。</p>
<p>直到今天，还有不少网银需要依赖于 IE 浏览器才能运，但随着技术的不断升级迭代，基于 Chromium 内核、性能更出色的 Edge 代替了它。</p>
<p style="text-align: center"><a href="https://img1.mydrivers.com/img/20210703/2a312942-4a15-4f82-ace6-2acec8de70a7.png" target="_blank"><img alt="Win11无情抛弃老电脑：微软搞的什么鬼？" h="342" src="https://img1.mydrivers.com/img/20210703/S2a312942-4a15-4f82-ace6-2acec8de70a7.png" style="border: black 1px solid" w="600" referrerpolicy="no-referrer"></a></p>
<p>就像极力甩掉 IE 浏览器一样，微软如今对于老电脑同样也表现出了相同的态度，只是 IE 有了更好的替代品，但未来 Windows 11 能否达到我们的预期还是一个未知数。</p>
