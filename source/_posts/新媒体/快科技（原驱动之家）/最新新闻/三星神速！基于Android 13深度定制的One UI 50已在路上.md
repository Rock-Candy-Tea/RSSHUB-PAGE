<p>4月9日消息，据<a class="f14_link" href="https://www.sammobile.com/news/samsung-android-13-one-ui-5-0-update-could-begin-testing-by-july/" target="_blank">SamMobile</a>报道，Android 13开发者预览版已经上线，谷歌Pixel机型可以更新尝鲜。</p>
<p>值得注意的是，三星正在紧锣密鼓开发基于Android 13深度定制的One UI 5.0系统。据SamMobile报道，<span style="color:#ff0000;"><strong>三星可能会在7月份开启One UI 5.0 Beta版公测，</strong></span>2022年年底登场。</p>
<p>和往常一样，三星高端机型将会率先尝鲜One UI 5.0系统。<strong>比如Galaxy S22系列、Galaxy Z Fold4、Galaxy Z Flip4等机型将会是首批适配One UI 5.0的机型。</strong></p>
<p>作为新一代操作系统，Android 13最大的升级之处是针对折叠屏、平板电脑等大屏设备进行了适配优化。谷歌强化了界面设计，让更多应用程序可以无缝适配不同尺寸的Android设备，让应用体验更佳。</p>
<p>据悉，<strong>谷歌会在2022年5月份举行的I/O开发者大会上公布Android 13操作系统，</strong>下半年会推送正式版更新。</p>
