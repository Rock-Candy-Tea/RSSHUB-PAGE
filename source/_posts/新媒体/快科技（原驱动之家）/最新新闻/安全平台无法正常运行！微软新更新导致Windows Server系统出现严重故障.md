<p>此前，微软为Windows Server 2022系统推送了编号为KB5007205的更新补丁，<strong>该补丁将修复此前出现在一些高分辨率屏幕上的缩放错误问题。</strong></p>
<p>但近日，有消息称，<span style="color:#ff0000;"><strong>该补丁在Windows Server 2022上造成了一个较为严重的故障，该故障可能会影响到系统的网络安全。</strong></span></p>
<p>据悉，在此次更新后，有用户发觉Windows Server 2022上，只要安装有Windows Server Core，<strong>那么终结点安全平台（Microsoft Defender for Endpoint）就可能出现无法运行的情况。</strong></p>
<p>经了解，<strong>微软的终结点安全平台功能能够帮助企业用户进行网络预防、监测、调查以及响应高级别的威胁</strong>，对于Windows Server 2022这一主要针对企业用户的服务器系统来说极为重要。</p>
<p>Windows Server是微软推出的Windows的服务器操作系统，而Windows Server 2022则是该系统最新的长期服务版本。</p>
<p>好消息是，<strong>微软已经确认了这一Bug，并回应称正在进行解决方案的制定，在之后发布的新版本补丁中，将正式修复这个问题。</strong></p>
