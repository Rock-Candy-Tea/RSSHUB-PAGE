<p>据pricebaba报道，OPPO有两款折叠新机已在欧盟知识产权局 (EUIPO) 完成注册，<strong>分别命名为Find N Flip、Find N Fold。</strong></p>
<p>博主数码闲聊站曾透露，<strong>OPPO正在研发一款竖向内折新机，代号“蜻蜓”，这与OPPO Find N Flip刚好吻合。</strong></p>

<p align="center"><a href="https://img1.mydrivers.com/img/20220805/5d8b7c9495124d37be1d20f91dd68558.jpg" target="_blank"><img alt="OPPO首款竖向折叠旗舰“蜻蜓”定名：OPPO Find N Flip" h="743" src="https://img1.mydrivers.com/img/20220805/s_5d8b7c9495124d37be1d20f91dd68558.jpg" style="border: black 1px solid;" w="600" referrerpolicy="no-referrer"></a></p>
<p>顾名思义，OPPO Find N Flip显然是要与三星Galaxy Z Flip和moto razr 2022展开竞争。</p>
<p>另外，<strong>Find N Flip很有可能搭载高通骁龙最新的骁龙8+旗舰处理器，</strong>在核心配置上与友商看齐。骁龙8+采用台积电4nm，性能和能效较上一代都取得巨大突破。</p>
<p>据悉，<strong>Find N Flip将配备120Hz高刷屏</strong>，采用新型铰链结构，并且将会减少设备厚度和重量，主打轻薄旗舰。</p>
<p>另一款Find N Fold将会是OPPO Find N的继任者，它将采用类似于Galaxy Z Fold的书本式折叠设计。</p>
<p align="center"><a href="https://img1.mydrivers.com/img/20220805/60b763fddb2b48a39a6271de0998b602.jpg" target="_blank"><img alt="OPPO首款竖向折叠旗舰“蜻蜓”定名：OPPO Find N Flip" h="734" src="https://img1.mydrivers.com/img/20220805/s_60b763fddb2b48a39a6271de0998b602.jpg" style="border: black 1px solid;" w="600" referrerpolicy="no-referrer"></a></p>
<p>除了新款折叠旗舰之外，OPPO还在开发OPPO Band 2耳机和搭载骁龙8+的Find X5 Pro，ColorOS 13也有望在今年发布。</p>
