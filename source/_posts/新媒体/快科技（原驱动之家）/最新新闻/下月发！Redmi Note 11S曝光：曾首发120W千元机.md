<p>2021年下半年，Redmi推出了Redmi Note 11系列机型，顶配版<span style="color:#ff0000;"><strong>Redmi Note 11 Pro+首次将120W快充技术做到了2000元以内，</strong></span>这是同价位段充电速度最快的手机，堪称千元快充之王。</p>
<p>如今Redmi即将发布Note系列新成员Redmi Note 11S，据91Mobile报道，<strong>Redmi Note 11S将会在2月底在海外上市发售。</strong></p>
<p>目前关于Redmi Note 11S的细节较少，爆料称Redmi Note 11S采用90Hz高刷新率全面屏，搭载联发科处理器（型号未知），后置主摄一亿像素（108MP），同时配备800万超广角和200万微距。</p>
<p>值得注意的是，Redmi Note 11S可能不会有国行版，<strong>该机可能是一款4G机型，定价预计在1000元左右。</strong></p>
<p>另外，Redmi品牌总经理卢伟冰确认，Redmi Note系列从Note 11系列开始一年两迭代的策略，分别定位“性能小旗舰”和“体验小旗舰”。</p>
<p>考虑到Redmi Note 11系列定位“体验小旗舰”，<strong>下一代产品Redmi Note 12系列将会定位“性能小旗舰”。</strong></p>
