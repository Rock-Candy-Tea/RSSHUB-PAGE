<p>折叠屏能当主力机的时代来了。</p>
<p>“我们想告诉整个行业和消费者，折叠屏再也不是一款试水的产品，而是可以作为主力的旗舰手机所使用。”荣耀CEO赵明近日接受媒体采访时说道。</p>
<p>1月10日，荣耀Magic V作为2022年的开年旗舰正式登场。<strong>它的到来之所以备受关注，不仅在于这是全球首款骁龙8折叠屏旗舰，更让人感兴趣的是，荣耀在折叠屏领域第一次喊出了“一部到位”。而在此之前，折叠屏更多是作为备用机来尝鲜。</strong></p>
<p>2021年可以看作是折叠屏元年，这一年折叠屏手机大热，除了苹果之外，几乎所有主流手机厂商都推出了折叠屏产品。荣耀几乎是最晚进入折叠屏市场的，但赵明依然敢在发布会之前就放话，荣耀Magic V要超越所有折叠屏对手，显然是早已成竹在胸。</p>
<p style="text-align: center"><a href="https://img1.mydrivers.com/img/20220113/be08a9fb-4150-4445-b611-5fe398a1e0cd.jpg" target="_blank"><img alt="荣耀Magic V让折叠屏手机进入主力机时代：Magic UI 6.0才是灵魂" h="399" src="https://img1.mydrivers.com/img/20220113/Sbe08a9fb-4150-4445-b611-5fe398a1e0cd.jpg" style="border:black 1px solid" w="600" referrerpolicy="no-referrer"></a></p>
<p>折叠屏之所以被认为是未来智能手机发展的风向标，主要在于通过交互创新颠覆了智能手机的使用体验，探索了未来手机形态。</p>
<p>但并非所有折叠屏手机都做到了这一点。<strong>市面上很多折叠屏的体验并不尽如人意，而其中的关键，就是缺乏对于折叠屏系统底层的深挖和研究。</strong></p>
<p>在荣耀Magic V发布会上，一开始赵明并未将骁龙8等硬件卖点作为开场，而是花了大量篇幅来深入介绍全新Magic UI 6.0为折叠屏带来的革命性体验。</p>
<p>在笔者看来，硬件只是体验的基础保障，而软件系统才是折叠屏的灵魂和精髓。<span style="color:#ff0000;"><strong>Magic UI 6.0才是荣耀Magic V体验超越所有折叠屏手机的关键所在。</strong></span></p>
<p>想做好体验，从来都不是一蹴而就。尤其是面对折叠屏这样的新形态时候，更是如此。厂商不仅要真正发挥出折叠大屏的优势，做好大小屏切换、分屏的交互和体验，应用的适配和生态，还要在懂用户所需、主动服务上下功夫。</p>
<p style="text-align: center"><a href="https://img1.mydrivers.com/img/20220113/ef4d5cb1-7c16-436f-9f9f-a880aba4efbf.jpg" target="_blank"><img alt="荣耀Magic V让折叠屏手机进入主力机时代：Magic UI 6.0才是灵魂" h="399" src="https://img1.mydrivers.com/img/20220113/Sef4d5cb1-7c16-436f-9f9f-a880aba4efbf.jpg" style="border:black 1px solid" w="600" referrerpolicy="no-referrer"></a></p>
<p>对于大多数购买折叠屏的用户来说，最具吸引力的是在同一款设备上获得了大小屏两种体验，而且可以无缝切换。所以就需要软件系统在交互设计去同时满足上述需求。</p>
<p>当我们打开折叠屏的时候，希望利用大屏幕获得更多的信息展示，而不是手机交互的简单放大。Magic UI 6.0通过研究不同应用的信息展示和交互的差异化特点，将大屏信息的展示总结出三类，我们结合实际场景来看看。</p>
<p><strong>第一类，内容型布局，</strong>让信息的呈现量直接大幅增加，减少用户获取信息的操作频率。</p>
<p><strong>第二类，列表-详情型，</strong>展开大屏通过用分栏同时显示父子层级，提升显示和操作效率。</p>
<p><strong>第三类，简短独立任务，</strong>一些不适合大屏全屏显示的独立任务，会用更轻量的样式，比如居中靠下的显示模式，减少全屏跳转，让用户更易查看和操作。</p>
<p style="text-align: center"><a href="https://img1.mydrivers.com/img/20220113/a328288f-6ddf-4711-ba7e-0ab58226b6c7.jpg" target="_blank"><img alt="荣耀Magic V让折叠屏手机进入主力机时代：Magic UI 6.0才是灵魂" h="687" src="https://img1.mydrivers.com/img/20220113/Sa328288f-6ddf-4711-ba7e-0ab58226b6c7.jpg" style="border:black 1px solid" w="600" referrerpolicy="no-referrer"></a></p>
<p>比如，在我们最长使用的手机淘宝软件上。可以同时显示显示子父级窗口，左边是首页，右边是频道广场，充分利用大屏来显示更多的信息。</p>
<p style="text-align: center"><a href="https://img1.mydrivers.com/img/20220113/da693201-da8b-45bb-9287-33a0847b6462.jpg" target="_blank"><img alt="荣耀Magic V让折叠屏手机进入主力机时代：Magic UI 6.0才是灵魂" h="687" src="https://img1.mydrivers.com/img/20220113/Sda693201-da8b-45bb-9287-33a0847b6462.jpg" style="border:black 1px solid" w="600" referrerpolicy="no-referrer"></a></p>
<p>对于喜欢炒股的用户来说，Magic UI 6.0可以实现全屏展开一屏同看股票列表和K线详情，在股票列表、K线和操作等信息模块间无需频繁切换，显然比一般的直板手机操作效率更加高效。对于短线选手来说，分秒就是金钱，直观和高效比什么都重要。</p>
<p style="text-align: center"><a href="https://img1.mydrivers.com/img/20220113/c69642ae-ff6a-4f07-a4e5-0938017e382b.jpg" target="_blank"><img alt="荣耀Magic V让折叠屏手机进入主力机时代：Magic UI 6.0才是灵魂" h="687" src="https://img1.mydrivers.com/img/20220113/Sc69642ae-ff6a-4f07-a4e5-0938017e382b.jpg" style="border:black 1px solid" w="600" referrerpolicy="no-referrer"></a></p>
<p>随着短视频的流行，越来越多人喜欢拍完视频在手机上剪辑了。但普通的直板手机屏幕尺寸不够，图标和按钮操作起来都不够直观。像剪辑应用OVICUT就是如此，通过Magic UI 6.0的优化之后，应用在Magic V的大屏上放大拉伸，各种工具、适配轨道音轨都显示得更加清晰，使用手指直接触碰、拖拽操作也显得更为方便。</p>
<p>此外，还可以实现多轨道音视频编辑，原始素材一键拖拽，编辑素材分屏调整和管理，并支持4K高清视频导出，效率直接翻倍。</p>
<p>除了为不同为的应用专门为大屏适配外，<strong>分屏体验也直接决定了一款折叠屏的体验优劣。</strong></p>
<p>当下，大多数折叠屏喜欢采用拖拽的方式开启分屏，但缺点很明显，一是用户很难发现，导致使用率很低；二是操作不够便捷，分屏反而成了累赘。</p>
<p><strong>我之所以喜欢Magic V的应用分屏，<span style="color:#ff0000;">是因为Magic UI 6.0的分屏体验是我目前接触过的最直观、最高效的，而且没有之一。</span></strong></p>
<p style="text-align: center"><a href="https://img1.mydrivers.com/img/20220113/817fcd9b-2c37-4321-ad4c-b2cd26467cff.png" target="_blank"><img alt="荣耀Magic V让折叠屏手机进入主力机时代：Magic UI 6.0才是灵魂" h="686" src="https://img1.mydrivers.com/img/20220113/S817fcd9b-2c37-4321-ad4c-b2cd26467cff.png" style="border:black 1px solid" w="600" referrerpolicy="no-referrer"></a></p>
<p>在Magic V上打开一个应用，你就会发现，都有一个横条，就是分屏交互入口，点击即可召唤出分屏导航。你可以一键选择两个App内容的分屏和悬浮位置关系，也可以随时选择需要被全屏的App。</p>
<p>以最常用的左右分屏模式为例，点击按钮后，原应用会靠左显示，右方则会回到桌面。此时，再打开其他应用就会自动进入左右分屏。</p>
<p style="text-align: center"><a href="https://img1.mydrivers.com/img/20220113/29a8fcd8-5fda-424e-97da-bc5d746e0200.png" target="_blank"><img alt="荣耀Magic V让折叠屏手机进入主力机时代：Magic UI 6.0才是灵魂" h="689" src="https://img1.mydrivers.com/img/20220113/S29a8fcd8-5fda-424e-97da-bc5d746e0200.png" style="border:black 1px solid" w="600" referrerpolicy="no-referrer"></a></p>
<p>除此之外，悬浮胶囊功能的加入，可以让当前操作不中断。比如，当微信来消息时，你可以在悬浮窗中快速回复消息或处理临时任务。并且随意调节悬浮窗位置、大小，而且支持同时开启两个悬浮窗。</p>
<p>不管是移动办公，还是社交分享，阅读购物，视频观影，Magic UI 6.0让Magic V都能实现一屏多用，非常利索、干脆。而且逻辑上也不会有任何困扰，即便是初次上手，也没有太大的学习成本。<strong>这在其它折叠屏手机上，是难以体验到的。</strong></p>
<p style="text-align: center"><a href="https://img1.mydrivers.com/img/20220113/3506f98d-3f4a-4125-8f27-fa39c38b0e1a.jpg" target="_blank"><img alt="荣耀Magic V让折叠屏手机进入主力机时代：Magic UI 6.0才是灵魂" h="400" src="https://img1.mydrivers.com/img/20220113/S3506f98d-3f4a-4125-8f27-fa39c38b0e1a.jpg" style="border:black 1px solid" w="600" referrerpolicy="no-referrer"></a></p>
<p>当然，这些都只是Magic UI 6.0智慧交互体验的冰山一角。其它比如地铁出行时自动弹乘车码、可以与PC互联共享操作提高效率、如进出电梯时选择最优网络等等，类似的主动服务不胜枚举。</p>
<p>用赵明的话来说，这不是一个通用的系统，而是为用户定制的个性化操作系统，能够提前预判用户需求。</p>
<p>没有一个能够独特的、带给消费者优秀体验的软件系统和体系，走高端路线是不现实的。而且软件体系也是构筑消费者与众不同体验的一个非常重要的部分，荣耀一定要走出自己的路来。</p>
<p>相比早起折叠屏的炫技，Magic UI 6.0的到来赋予了折叠屏作为主力机使用的能力，更将折叠屏的实用性和智慧体验带到了前所未有的高度，实现了“越用越好用”的体验。而这也正是潜在折叠屏消费者最为看重的一点。<span style="color:#ff0000;"><strong>而Magic UI 6.0的赋能，加骁龙8等顶级旗舰的配置，让荣耀Magic V真正有了作为主力旗舰手机使用的资本。</strong></span></p>
<p>好产品总是会受到市场第一时间的认可。<strong>据了解，荣耀Magic V目前已经开启预售，1月18日上午10:08正式开售。</strong></p>
<p style="text-align: center"><a href="https://img1.mydrivers.com/img/20220113/c5b79a57-9bd8-4495-86af-806508120d5b.png" target="_blank"><img alt="荣耀Magic V让折叠屏手机进入主力机时代：Magic UI 6.0才是灵魂" h="1018" src="https://img1.mydrivers.com/img/20220113/Sc5b79a57-9bd8-4495-86af-806508120d5b.png" style="border:black 1px solid" w="600" referrerpolicy="no-referrer"></a></p>
<p>目前，在第三方平台荣耀Magic V已经有了不少溢价，其中512GB版本已经报出了17999元的高价，相比官价直接上涨了7000元之多。256GB版本也报出了13999元，较9999元-官价高出了4000元。或许，这又将是新一代的理财神器。</p>
