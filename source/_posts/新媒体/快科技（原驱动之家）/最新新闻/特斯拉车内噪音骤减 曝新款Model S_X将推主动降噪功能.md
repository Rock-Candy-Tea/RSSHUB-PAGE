<p>特斯拉电动性能和行驶质感一直都是有口皆碑，<strong>但是因其车辆做工，以及无框车门的应用，特斯拉的车内噪音，却让不少车主不够满意。为此，特斯拉也开始了新的尝试。</strong></p>
<p>12月19日消息，据报道，特斯拉正在为新款 Model S 和 Model X 推出其主动降低道路噪音的功能，这是 2021.44.5 系统更新的功能之一。</p>
<p><strong>官方介绍称：您的车辆现在有能力在粗糙路面上行驶时主动减少低频道路噪音。</strong></p>
<p>为了实现这一目标，该系统使用座椅上的麦克风来测量车厢内的噪音，然后通过扬声器产生抗噪音，在乘员周围创建安静区域。要启用该功能，请点击音频设置 > 选项 > 主动降低道路噪音。</p>
<p>不过，该功能只支持特斯拉的旗舰车型新款 Model S 和 Model X。这是因为该功能需要的硬件，两车的前排座椅上安装了有主动降噪麦克风。目前，Model 3 和 Model Y 还没有配备此种硬件。</p>
<p>特斯拉 Model S 车主迈克尔-霍普伍德分享的视频显示，他在一条崎岖不平的碎石路上行驶时测试了该功能。</p>
<p>在他的驾驶过程中，他觉得启用主动式道路降噪功能后，<strong>车厢内的环境噪音会降低，但其效果似乎不是很明显。</strong></p>
<p>另外，通过施加反向声波，主动降噪能很好地解决低频噪音，而对高频噪音效果非常有限，<strong>而路噪恰好是低频噪音，能够被主动降噪消除。</strong></p>
