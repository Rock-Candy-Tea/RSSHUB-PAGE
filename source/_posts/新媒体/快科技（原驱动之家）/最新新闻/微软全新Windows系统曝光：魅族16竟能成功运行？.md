<p>之前一直有消息称，微软正在秘密开发全新的Windows系统，而它会冠以Windows Core OS的名称，现在来看确实有此事。</p>
<p><strong>国外爆料平台 SlashLeaks 上近日有网友分享了一段关于Windows Core OS 11操作展示，同时还分享了在魅族16上的运行图片。</strong></p>
<p style="text-align: center;"><img alt="微软全新Windows系统曝光：魅族16竟能成功运行？" h="393" src="https://img1.mydrivers.com/img/20210802/9928971149cd4199af662a2c872d66cb.gif" style="text-align: -webkit-center; border: 1px solid black; height: 295px; width: 600px;" w="800" referrerpolicy="no-referrer"></p>
<p><strong>网友在平台上表示，微软的企业软件测试部门已经对Windows Core OS 11的组件进行测试，它基于新内核构建，可与Windows 11界面配合使用。</strong></p>
<p>为了更好地进行测试，测试部门在魅族 16 上安装了该系统。除了麦克风之外一切均正常。消息人士没有提供更多细节，但值得注意的是，同一个人是第一个发布 Windows Core OS 11截图的人，尽管其真实性尚未得到证实。</p>
<p>有传言称，新操作系统将消除Windows 10的大部分遗留代码和兼容性问题。今年早些时候，没有图形界面的早期版本的Windows Core OS被泄露到网上。</p>
