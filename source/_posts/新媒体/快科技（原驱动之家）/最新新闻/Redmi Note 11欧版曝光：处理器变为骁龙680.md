<p>去年下半年，Redmi在中国推出了Redmi Note 11系列，<strong>包括Redmi Note 11、Redmi Note 11 Pro和Redmi Note 11 Pro+等机型，</strong>现在Redmi Note 11即将走出国门。</p>
<p>1月15日消息，据mysmartprice报道，<span style="color:#ff0000;"><strong>Redmi即将在欧洲发布Redmi Note 11。</strong></span></p>
<p>和国行版Redmi Note 11使用联发科处理器不同，<span style="color:#ff0000;"><strong>欧洲版本Redmi Note 11使用的是高通骁龙680处理器。</strong></span></p>
<p>据悉，骁龙680基于6nm工艺制程打造，采用Kryo 265 CPU架构，最高主频达到了2.4GHz，GPU为Adreno 610。</p>
<p>除了搭载骁龙680，Redmi Note 11欧洲版还采用6.5英寸LCD全面屏，最高配备8GB内存、128GB内存，此后置5000万主摄、800万超广角+200万微距三摄，电池容量为5000mAh，支持18W快充。</p>
<p>值得注意的是，mysmartprice曝光了Redmi Note 11欧洲版本的价格，<strong>起售价可能是250欧元（4GB+64GB，约合人民币1800元）。</strong></p>
