<p>很多人吐槽微信没有客服，使用微信遇到难题时无法及时解决。</p>
<p>而现在，“微信客服”终于来了，目前这一功能内测，官网（https://work.weixin.qq.com/kf）已上线。</p>
<p>不过需要注意的是，<strong>这并非大家想的那个客服，不是给微信给用户服务的，而是企业通过微信客服给客户服务的。</strong></p>
<p>目前“微信客服”，用户可以使用企业微信扫码登录。</p>
<p><strong>据官方介绍，用户可以在微信内、外各个场景中接入微信客服，用户可以发起咨询，企业可通过 API 接口回复消息，做好客户服务。</strong></p>
<p>微信客服功能有以下三个亮点：</p>
<p><strong>丰富的接入口</strong></p>
<p>可以在视频号、搜一搜、支付凭证、网页等微信内和微信外场景接入微信客服，接待用户咨询。</p>
<p><strong>微信官方消息通道</strong></p>
<p>咨询记录将出现在微信的“客服消息”中，用户更易发现。</p>
<p><strong>API 收发消息</strong></p>
<p>可通过 API 收发消息，实现多坐席协作、自动回复等功能。</p>
<p align="center"><a href="https://img1.mydrivers.com/img/20210719/7727c43b3fa24bd18056fda2721eb461.png" target="_blank"><img alt="期待已久的“微信客服”来了 官网已上线" h="451" src="https://img1.mydrivers.com/img/20210719/s_7727c43b3fa24bd18056fda2721eb461.png" style="border: black 1px solid;" w="600" referrerpolicy="no-referrer"></a></p>
<p>其实，微信属于腾讯旗下的APP，它的客服自然也就是腾讯客服。</p>
<p>其中腾讯客服电话是0755-83765566，服务时间是早9点到晚22点。而关于微信支付、红包、零钱、支付商户等相关问题，需要拨打95017。</p>
<p>大家在腾讯客服网站（kf.qq.com）的最下面也能找到腾讯不同领域的客服电话。</p>
