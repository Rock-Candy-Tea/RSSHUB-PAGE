<p>今天，小米公司Redmi产品总监王腾为Redmi Note 11系列预热。</p>
<p>王腾表示，<strong>你喜爱的超清影像即将在Note 11系列上回归。</strong></p>
<p>根据官方公布的海报，<span style="color:#ff0000;"><strong>Redmi Note 11系列背部镜头位置印有“108MP”，表明Redmi Note 11系列将会采用一亿像素主摄。</strong></span></p>
<p>去年Redmi在Redmi Note 9 Pro上使用了一亿像素主摄，这是同档位唯一一款采用一亿像素主摄的机型。</p>
<p>随后发布的Redmi Note 10系列无缘一亿像素，<strong>如今一亿像素又来到了Redmi Note 11系列上。</strong></p>
<p>之前Redmi在Note 9系列发布时，其slogan是剑指3000元内手机影像，包括旗舰。如今Redmi Note 11系列再次使用一亿像素，其影像表现值得期待。</p>
<p>该机将于10月28日正式发布，包括Redmi Note 11、Redmi Note 11 Pro和Redmi Note 11 Pro+三款，起售价预计在1000元左右。</p>
