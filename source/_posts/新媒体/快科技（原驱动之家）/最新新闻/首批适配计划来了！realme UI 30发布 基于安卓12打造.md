<p>今天realme的新品发布会，除了推出两款新机和一款手表外，还有全新的系统也一同亮相。</p>
<p><strong>这次realme UI 3.0也一同发布，新系统基于Android12打造，带来全新的立体空间感设计：全新立体图标、重构页面布局、个性化主题、全新AOD显示、全新速览卡片。</strong></p>
<p>官方给出的进程显示，realme UI 3.0首批适配机型包含了：真我GT / GT Neo / GT大师 / GT Neo2系列，会在今年10月起陆续开启尝鲜。</p>
<p>此外，realme UI 3.0带来多项全新特性，更进一步优化用机体验：全新Omoji、全新AI畅速引擎、全新AI动画引擎、全新自由浮窗、全新智能侧边栏。</p>
<p>新系统在隐私方面也进行了升级，realme UI 3.0应用行为记录，权限使用记录可视化；分享隐私保护，抹除照片与视频隐私信息；模糊定位保护，不暴露具体定位信息。</p>
<p>realme UI 3.0智慧融合，带来全新互联体验：生态跨屏互联，打破生态壁垒；车机互联，智慧出行；全新家庭空间，家人联系更紧密。</p>
