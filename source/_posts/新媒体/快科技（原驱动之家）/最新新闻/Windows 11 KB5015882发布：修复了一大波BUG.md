<p>今天，微软推送了Windows 11 KB5015882更新包，类型为“可选”，意味着大家点击下载才会下载，否则不会更新。</p>
<p>根据官方发布说明，<strong>Windows 11现在可让您在焦点辅助开启时收到紧急警报。此功能类似于在您的移动设备上隐藏通知的打扰模式。</strong></p>
<p>除了一些新功能外，微软还修复了当您使用播放和暂停键盘按钮时文件资源管理器可能停止工作的问题。当你右键单击开始菜单的上下文菜单时，导致文件资源管理器停止响应的错误已得到修复。</p>
<p>另一个问题已修复，即当您将鼠标悬停在任务栏上的搜索图标上时无法关闭黑色窗口。</p>
<p>这是完整的变更日志：</p>
<p>1、Microsoft 正在恢复 Windows Autopilot 部署方案。由于硬件的安全问题，此功能已关闭。</p>
<p>2、Microsoft 已修复应用程序使用 UIAutomation() 可能会停止工作的问题。</p>
<p>3、另一个导致启动任务 API 无法工作的错误已得到修复。</p>
<p>4、微软提高了操作系统升级后一键重置的可靠性。</p>
<p>5、Microsoft 已修复租户限制事件日志记录通道无法访问的问题。</p>
<p>6、修复了基于证书的计算机帐户身份验证失败的错误。</p>
<p>7、Microsoft 已修复影响 Arm64EC 代码的问题。</p>
<p>8、Microsoft 已修复 Remove-Item cmdlet 无法与 OneDrive 文件夹正确反应的问题。</p>
<p>9、Microsoft 已修复阻止故障排除工具打开的问题。</p>
<p>10、Microsoft 已修复导致代码完整性在文件被修改后继续信任文件的问题。</p>
<p>11、修复了启用 Windows Defender 应用程序控制时 Windows 停止响应的错误。</p>
