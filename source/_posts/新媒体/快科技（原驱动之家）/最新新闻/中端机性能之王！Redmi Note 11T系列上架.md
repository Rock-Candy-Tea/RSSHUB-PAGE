<p>5月19日消息，小米京东自营旗舰店显示，<strong>Redmi Note 11T系列已上架接受预约，该机将于5月24日正式发布。</strong></p>
<p>预约页面显示，Redmi Note 11T系列至少有两款机型，<span style="color:#ff0000;"><strong>分别是Redmi Note 11T Pro、Redmi Note 11T Pro+，可能还有标准版Redmi Note 11T。</strong></span></p>
<p style="text-align: center;"><a href="https://img1.mydrivers.com/img/20220519/fc5fc6c024144ba1bf0787e5f8b82a59.jpg" style="text-align: -webkit-center;" target="_blank"><img alt="中端机性能之王！Redmi Note 11T系列上架" h="343" src="https://img1.mydrivers.com/img/20220519/s_fc5fc6c024144ba1bf0787e5f8b82a59.jpg" style="border: 1px solid black;" w="600" referrerpolicy="no-referrer"></a></p>
<p>据悉，Redmi Note 11T Pro系列搭载联发科天玑8000系列处理器，这颗芯片基于台积电5nm工艺制程打造，由四颗Cortex A78大核+四颗Cortex A55小核组成，性能强悍。</p>
<p style="text-align: center;"><a href="https://img1.mydrivers.com/img/20220519/cdcebdf676e14fcca2cf0a95c4a9d367.jpg" style="text-align: -webkit-center;" target="_blank"><img alt="中端机性能之王！Redmi Note 11T系列上架" h="157" src="https://img1.mydrivers.com/img/20220519/s_cdcebdf676e14fcca2cf0a95c4a9d367.jpg" style="border: 1px solid black;" w="600" referrerpolicy="no-referrer"></a></p>
<p>此外，Redmi Note 11T Pro系列采用了LCD屏幕，刷新率最高达到了144Hz，预计会支持刷新率自适应调节，系统根据使用场景智能切换刷新率，以达到流畅、省电的效果。</p>
<p>按照Redmi Note系列定位，Redmi Note 11T Pro系列价格可能在1500-2000元之间，这将是同价位段性能最强悍的手机，堪称“中端机性能之王”。</p>
