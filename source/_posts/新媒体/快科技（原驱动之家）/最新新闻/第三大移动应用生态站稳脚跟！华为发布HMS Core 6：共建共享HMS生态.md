<p>今天，华为开发者大会2021在东莞松山湖正式开幕。</p>
<p>华为宣布HMS全球开发者已达510万，集成HMS Core开放能力的全球应用超过17.3万个，HMS生态开发者在2021年前三季度获得的收入增长62%，华为应用市场AppGallery前三季度应用分发3322亿次。</p>
<p>作为全球第三大移动应用生态，华为HMS在全球开发者支持下快速发展，已经站稳脚跟。</p>
<p><strong>过去三年，华为移动服务HMS生态持续走向全球化、全面开放。从2018年，华为应用市场AppGallery、华为音乐等开启全球化，拥有56万注册开发者；</strong></p>
<p>2019年，华为正式提出构建全球移动应用新生态，<span style="color:#ff0000;"><strong>HMS Core 4上线开放了14个能力，发展了130万开发者，华为视频、华为浏览器等加速全球化进程；</strong></span></p>
<p>到2020年，HMS Core 5正式上线，开放7大领域56项能力，并构建了支付、搜索、地图、广告、浏览器五大根服务引擎，搜索、地图、阅读等应用也在全球多国正式上线，增长至230万开发者，<span style="color:#ff0000;"><strong>HMS生态正式成为全球第三大移动应用生态。</strong></span></p>
<p>HMS生态在全场景、多终端等方面也持续发力，支持原生应用、快应用、FA等多种形态的数字服务，并可在手机、智慧屏、PC、智能手表、平板、VR眼镜、车机等多终端进行智慧分发和流转，让全球开发者可以通过更多渠道触及消费者。</p>
<p>现在华为正式发布全新的多终端、跨OS、全场景的华为移动服务核心能力HMS Core 6。</p>
<p style="text-align: center;"><a href="https://img1.mydrivers.com/img/20211022/fa708c4510154296bd5b71f303e9fdb4.jpg" style="text-align: -webkit-center;" target="_blank"><img alt="第三大移动应用生态站稳脚跟！华为发布HMS Core 6：共建共享HMS生态" h="400" src="https://img1.mydrivers.com/img/20211022/s_fa708c4510154296bd5b71f303e9fdb4.jpg" style="border: 1px solid black;" w="600" referrerpolicy="no-referrer"></a></p>
<p>HMS Core 6将华为在图形、媒体、人工智能、应用服务、连接与通信等7大领域积累的技术进一步开放，并带来三个方面的重大升级：支持多终端、支持跨操作系统、支持应用的全场景、一致性体验。</p>
<p>作为领先的多终端、跨操作系统、全场景的移动服务核心能力，HMS Core 6向全球开发者开放了7大领域的69个Kit和21738个API，包括13个跨操作系统能力，开发者基于不同的操作系统开发应用时，都可以集成华为创新的移动服务能力，保证应用体验的一致性。</p>
<p style="text-align: center;"><a href="https://img1.mydrivers.com/img/20211022/d1011fe8b0ba439d89243468885994c0.jpg" style="text-align: -webkit-center;" target="_blank"><img alt="第三大移动应用生态站稳脚跟！华为发布HMS Core 6：共建共享HMS生态" h="337" src="https://img1.mydrivers.com/img/20211022/s_d1011fe8b0ba439d89243468885994c0.jpg" style="border: 1px solid black;" w="600" referrerpolicy="no-referrer"></a></p>
<p>如机器学习MachineLearning kit已经扩展到其他操作系统应用上，让其他操作系统应用也能使用文本识别、OCR、机器翻译等人工智能的能力。</p>
<p>HMS Core 6新增或增强的能力共计31个，其中，3D建模3D Modeling Kit仅通过智能终端的RGB摄像头，即可便捷、高效地为一个目标物体进行3D建模，不仅可以应用在电商、动画制作等领域，未来还可结合3D打印解决远程医疗问题；</p>
<p>视频编辑Video Editor kit除了开放全功能的视频编辑能力，还将花瓣剪辑背后的创新能力逐步开放出来，开发者只需简单集成SDK，就可以让应用获得目标遮挡、目标追踪、滤镜拟合等；</p>
<p>车道级引导能力通过Petal Maps Platform的Map kit、location kit、Site kit等面向全球开发者和行业伙伴陆续开放，让导航App获得更多创新体验；</p>
<p>HDC现场首次使用数字人全程手语直播，即是基于HMS Core 6全新开放的华为手语服务SignPal kit，让沟通无障碍。</p>
<p>HMS Core 6还带来了音频剪辑、DCI版权保护、数字人民币场景创新、智能网络调优等多项新能力与服务。</p>
<p>华为消费者业务云服务总裁、华为云CEO张平安表示：<strong>未来的数字世界是一个生态的世界，全场景智慧时代需要更多的创新，每一位开发者就是创新力量的源泉。</strong></p>
<p>华为希望汇聚全球开发者，开放更多能力与服务，让开发更简单，让创新更高效，<strong>与开发者共建共享共赢HMS新生态，加速万物智联与创新。</strong></p>
