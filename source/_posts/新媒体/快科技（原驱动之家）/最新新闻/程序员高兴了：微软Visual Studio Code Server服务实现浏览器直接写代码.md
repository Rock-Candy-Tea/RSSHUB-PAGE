<p>作为一家以软件开发闻名的商业巨头，微软一直都致力于在程序开发上为用户带来更为便捷的体验。</p>
<p>在2019年，微软曾发布了远程开发扩展，<strong>允许用户通过本地VS Code在Win10及以上系统的Linux子系统、Docker容器、SSH管理的远程物理或虚拟机上远程开发应用。</strong></p>
<p>近日，微软在此基础上，推出了一项新的后端服务Visual Studio Code Server，以及一个可以轻松安装、更新、管理和连接到服务的CLI。</p>
<p>根据微软介绍，<span style="color:#ff0000;"><strong>通过Visual Studio Code Server，用户能够将服务器安装在本地或云端的任何位置，并使用VS Code for Web在浏览器中远程安全访问，且无需设置任何SSH或https。</strong></span></p>
<p align="center"><a href="https://img1.mydrivers.com/img/20220710/b7d8f5dd2d454906a7ebca05184e8ecf.png" target="_blank"><img alt="程序员高兴了：微软Visual Studio Code Server服务实现浏览器直接写代码" h="262" src="https://img1.mydrivers.com/img/20220710/s_b7d8f5dd2d454906a7ebca05184e8ecf.png" style="border: black 1px solid;" w="600" referrerpolicy="no-referrer"></a></p>
<p>对于开发者来说，<strong>这意味着在完成最初的配置工作后，理论上就能够在任何位置，任何性能的电脑中，通过浏览器完成对项目的开发与修改，大大提升了工作的便捷性。</strong></p>
<p>目前，Visual Studio Code Server处在私人预览阶段，想要尝鲜的用户需要填写一份<a class="f14_link" href="https://forms.office.com/Pages/ResponsePage.aspx?id=v4j5cvGGr0GRqy180BHbR2RUg5vzJXJDmTEUlArnJpdUNDEyNkJCQU1LUjgyVFpWRk9GM09SWDlETy4u" target="_blank">注册表单</a>来进行申请，如果通过申请，将在数周内收到反馈邮件。</p>
