<p>今天，Redmi公布了Redmi Note 11系列外观。</p>
<p>如图所示，<strong>Redmi Note 11采用了直角中框设计，与iPhone 13的中框相似。</strong></p>
<p>官方介绍，<span style="color:#ff0000;"><strong>Redmi Note 11之所以要做小立边的大改款，是因为今天的年轻人都有棱有角。</strong></span></p>
<p>Redmi表示，立边更潮流，细节精致好看；立边更自信，给你想要的独特感。</p>
<p>除了在工业设计上有大幅升级外，Redmi Note 11系列的快充也有明显进步。</p>
<p>据爆料，<strong>Redmi Note 11顶配版支持120W快充，这与小米MIX 4的快充功率相一致。</strong></p>
<p>官方海报此前表示Redmi Note 11“一刻满血”，似乎暗示Redmi Note 11系列仅需15分钟就能充至100%，在同档位极具竞争力。</p>
<p>此外，Redmi Note 11采用了高刷新率挖孔屏方案，预计会搭载旗舰处理器，配备双扬声器。</p>
<p>新品将于10月28日发布。</p>
