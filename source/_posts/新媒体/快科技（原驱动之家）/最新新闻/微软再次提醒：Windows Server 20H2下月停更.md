<p>近日，微软再次提醒用户，即将停止对Windows Server 20H2版本的维护支持。</p>
<p>根据公告，<span style="color:#ff0000;"><strong>微软将在8月9日停止为所有Windows Server系统的20H2版本提供安全更新，即将到来的8月“星期二补丁”将会是它接受的最后一次更新。</strong></span></p>
<p>除了终止对Windows Server 20H2的支持外，微软还在公告中提到，<strong>Windows Server半年频道 (SAC)也将在8月9日后停止支持。</strong></p>
<p>与长期服务频道相比，半年频道的特性与功能更新更快，但支持时间仅有8个月，同时在版本、使用用户上也有一定的限制，甚至无法安装图形桌面。</p>
<p>微软建议，<strong>使用Windows Server半年频道的用户迁移到Azure Stack HCI或使用操作系统的长期服务频道</strong>，后者能够为用户提供5年的主流支持与5年的扩展支持。</p>
