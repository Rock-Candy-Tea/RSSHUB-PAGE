<p>在今晚的Redmi Note 11T系列新品发布会上，Redmi Note 11T Pro、Redmi Note 11T Pro+两款机型正式亮相。发布后，有网友表示，Redmi Note 11T Pro、Redmi Note 11T Pro+有何不同？差别在哪里？</p>
<p>从官方公布的参数配置对比来看，两款机型仅在充电和续航上有所不同。</p>
<p align="center"><a href="https://img1.mydrivers.com/img/20220524/073902eadfe04f2b82d277cdec467d5d.png" target="_blank"><img alt="1699元起！Redmi Note 11T Pro/Pro+有何不同：一文看懂" h="286" src="https://img1.mydrivers.com/img/20220524/s_073902eadfe04f2b82d277cdec467d5d.png" style="border: black 1px solid;" w="600" referrerpolicy="no-referrer"></a></p>
<p><span style="color:#ff0000;"><strong>其中Note 11T Pro采用67W超级快充，内置5080mAh大容量电池，而Note 11T Pro+则升级为120W神仙秒充，内置4400mAh容量电池，19分钟即可充满。</strong></span>至于其他硬件配置，两者并无区别。</p>
<p align="center"><a href="https://img1.mydrivers.com/img/20220524/53fbf1c9b0854078b402c442d2ef4f34.png" target="_blank"><img alt="1699元起！Redmi Note 11T Pro/Pro+有何不同：一文看懂" h="331" src="https://img1.mydrivers.com/img/20220524/s_53fbf1c9b0854078b402c442d2ef4f34.png" style="border: black 1px solid;" w="600" referrerpolicy="no-referrer"></a></p>
<p>Note 11T Pro/Pro+采用6.6英寸中孔全面屏，屏幕比例为20.5:9，支持144Hz高刷新率，支持7挡变速调节，采用RGB像素排列，支持硬件级防蓝光、DC调光，同时这块屏幕还获得了DisplayMate A+认证，达成14项屏幕记录。</p>
<p>核心配置上，<strong>搭载联发科天玑8100芯片，芯片基于台积电5nm工艺制程打造，</strong>由4颗主频为2.85GHz的Cortex-A78大核、4颗2.0GHz主频的Cortex-A55组成，GPU为Mali-G610 MC6。</p>
<p>其它配置方面，新机后置6400万AI三摄，配备LPDDR5内存，内置X轴线性马达、杜比立体声双扬声器、多功能NFC3.0等。</p>
<p align="center"><a href="https://img1.mydrivers.com/img/20220524/7407063aa47242208e77f7d0012d8ed6.jpg" target="_blank"><img alt="1699元起！Redmi Note 11T Pro/Pro+有何不同：一文看懂" h="450" src="https://img1.mydrivers.com/img/20220524/s_7407063aa47242208e77f7d0012d8ed6.jpg" style="border: black 1px solid;" w="600" referrerpolicy="no-referrer"></a></p>
<p>价格上，Redmi Note 11T Pro 618到手价1699元起、Redmi Note 11T Pro+ 618到手价1999元起，将于5月31日20:00正式开售。</p>
<p>想要旗舰级快充，追求充电速度，那么选择Redmi Note 11T Pro+即可，而如果你想要更大容量的电池、更持久的续航，选Redmi Note 11T Pro就行。</p>
