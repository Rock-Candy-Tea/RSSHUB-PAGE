<p>上月末，realme官方宣布将在9月正式推出realme UI 4.0；今天，realme软件总监陈雷在微博透露了关于realme UI 4.0版本的新消息。</p>
<p align="center"><a href="https://img1.mydrivers.com/img/20220902/041f8c089b854526bb7b1e98769fe781.png" target="_blank"><img alt="realme UI 4.0在路上了：更新“线路图”下周就发" h="421" src="https://img1.mydrivers.com/img/20220902/s_041f8c089b854526bb7b1e98769fe781.png" style="border: black 1px solid;" w="600" referrerpolicy="no-referrer"></a></p>
<p>陈雷表示，<span style="color:#ff0000;"><strong>realme UI 4.0 的适配机型和顺序还在做最后的评估，下周会公告适配“路线图”。</strong></span></p>
<p>同时，陈雷还透露，<strong>realme GT2 Pro 本月会发布公测版，而realme GT2 大师探索版和realme GT Neo3 则会在一两个月内发布尝鲜版。</strong></p>
<p>据悉，realme目前已经面向海外的realme GT2 大师探索版、realme GT 大师探索版和realme GT2 Pro三款机型，提供了基于Android 13的realme UI 4.0测试版更新，但国内还尚未开启测试。</p>
<p>根据官方介绍，<strong>realme UI 4.0将实现更多的指纹样式，并优化动态壁纸，带来更加丰富的动画效果。</strong></p>
<p>此外，新系统还将对通知栏控制中心进行优化，增加下拉通知栏控制中心的可玩性，增加更多控件。</p>
