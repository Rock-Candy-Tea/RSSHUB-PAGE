<p>近期发售的 Steam Deck 虽然预装的是基于 Linux 的 Steam OS 系统，但是 Valve 一直打算让这款游戏掌机支持 Windows 系统。现在该公司消除安装 Windows 11 的一个主要障碍，将对 Windows 的支持又向前推进了一步。</p>
<p style="text-align: center"><img alt="Steam Deck支持安装Win11：新版固件支持fTPM" h="338" src="https://img1.mydrivers.com/img/20220403/53c71d7c-1d8e-47f8-8369-cb30aefce57b.jpg" style="border: black 1px solid" w="600" referrerpolicy="no-referrer"></p>
<p style="text-align: center"><a href="https://img1.mydrivers.com/img/20220403/010845b6-093c-4311-a17a-3bdbf1dcbf22.jpg" target="_blank"><img alt="Steam Deck支持安装Win11：新版固件支持fTPM" h="590" src="https://img1.mydrivers.com/img/20220403/S010845b6-093c-4311-a17a-3bdbf1dcbf22.jpg" style="border: black 1px solid" w="600" referrerpolicy="no-referrer"></a></p>
<p><strong><span style="color:#ff0000;">在 Steam Deck 最新的操作系统测试版补丁说明中，最值得注意的细节可能是增加了对固件可信平台模块（fTPM）的支持，这使得用户可以在设备上安装 Windows 11。</span>该 BIOS 功能是对 Windows 11 刚推出时安装强制要求TPM 2.0的常见解决方案。</strong></p>
<p>系统要正式支持Windows 11，必须有TPM 2.0，这最初让许多用户感到震惊，因为它将操作系统的兼容性限制在相对较新的CPU上。这一要求可以通过专用硬件或基于软件的 fTPM 来满足。</p>
<p>3 月初，Valve 发布了 Steam Deck 的首批 Windows 驱动，但最初缺乏对 fTPM 的支持意味着该设备直到现在只能运行 Windows 10。</p>
<p><strong>AMD 还没有完成音频驱动，所以运行 Windows 的 Steam Deck 的任何声音都必须通过 USB-C 或蓝牙来实现。</strong></p>
<p>此外，Valve 仍在开发一个具有双启动功能的安装程序，所以目前用户将不得不在 Windows 和 Steam OS 之间做出选择。</p>
