<p>据Redmi手机官微消息，<span style="color:#ff0000;"><strong>Redmi Note 11潮流限定版将会在明天上午10点正式开售，售价2699元（8GB+256GB）。</strong></span></p>
<p>这款Redmi Note 11潮流限定版在外观上与普通版有很大不同，机身背部有YIBO DESIGN的签名logo，灵感是来自于王一博，整体设计非常炫酷。</p>
<p><strong>同时，Redmi Note 11潮流限定版还全球首发了全息悬浮工艺，Logo悬浮机身之上，机身的线条装饰则来自于赛道，配色非常抢眼。</strong></p>
<p>据官方介绍，<strong>Redmi Note 11潮流限定版在18cm²的空间中包含了11万个微米级Redmi Logo，</strong>通过11万个完美对应的微透镜，最终放大150倍，在机身上方呈现出3D悬浮感的立体 Redmi Logo。</p>
<p>除了设计方面之外，Redmi Note 11潮流限定版与Redmi Note 11 Pro+配置完全一致<strong>，<span style="color:#ff0000;">最大的亮点之一就是支持120W快充，这也是同级别唯一一款百瓦快充机型，同时还是市面上最便宜的120W快充之一。</span></strong></p>
<p>官方也将其称之为120W神仙秒充，仅15分钟即可充满，是目前Redmi充电速度最快的机型。</p>
<p>核心方面，Redmi Note 11潮流限定版搭载了联发科天玑920芯片，采用台积电6nm工艺打造，拥有它最新的A78双大核，实现了性能与功耗的绝佳平衡。</p>
