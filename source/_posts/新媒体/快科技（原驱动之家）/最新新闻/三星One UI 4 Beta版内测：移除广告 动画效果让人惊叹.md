<p>手机广告一直是一个比较敏感的话题，大部分用户对此极为反感。但对厂商而言，广告已经成为盈利的重要渠道。</p>
<p>业内人士指出，目前厂商在手机上放广告是出于盈利目的，如何在保证用户体验和获取收益维持平衡是当下手机厂商亟需解决的问题。</p>
<p>作为全球第一大手机品牌，三星做出表率，在自家手机上移除了手机广告。</p>
<p>10月21日消息，据GSMArena报道，三星开始在韩国本土推送One UI 4 Beta 3系统，<span style="color:#ff0000;"><strong>该系统完全移除了广告，预装的应用程序都进行了重新设计，同时改进了动画效果。</strong></span></p>
<p>推特用户@FrontTron反馈，三星在One UI 4 Beta 3上重新设计了天气应用程序，该应用程序彻底移除了广告，焕然一新。</p>
<p>不仅如此，@FrontTron表示，<strong>改进后的<span style="color:#ff0000;">三星One UI 4 Beta 3动画效果让人惊叹</span>，非常顺滑流畅。</strong></p>
<p>据悉，三星One UI 4基于Android 12深度定制，这是三星全新打造的新一代定制操作系统，目前仍然在测试阶段，稳定版后续会推送。</p>
<p>按照惯例，三星Galaxy S系列、Galaxy Z Fold系列以及Galaxy Z Flip系列等高端旗舰机型有望首批适配One UI 4操作系统。</p>
