<p>小米集团中国区总裁、Redmi品牌总经理卢伟冰此前确认，<strong>Redmi Note系列会半年一迭代，分别定位是“性能小旗舰”和“体验小旗舰”。</strong></p>
<p>今年上半年，Redmi推出了Note 11T Pro系列，主打的是“性能小旗舰”，那么今年下半年要发布的<span style="color:#ff0000;"><strong>Redmi Note 12系列定位是“体验小旗舰”。</strong></span></p>
<p>今天，博主@数码闲聊站爆料，<strong>Redmi Note 12系列采用了居中挖孔直屏方案</strong>，材质未知。</p>
<p>此前在Redmi Note 11T Pro系列上，Redmi使用了LCD屏幕，而在Redmi Note 11 Pro系列上，Redmi使用了OLED屏幕，二者都是高刷新率、FHD+分辨率。</p>
<p>另外，Redmi Note 12系列预计影像会有升级。之前博主@数码闲聊站爆料，Redmi会在新机上使用大底主摄，由此猜测，<strong>不排除Redmi Note 12系列率先使用大底主摄的可能。</strong></p>
<p>目前Redmi Note 11 Pro+使用的一亿像素三星HM2，其传感器尺寸为1/1.52英寸，Redmi Note 12系列使用的主摄尺寸可能会超越1/1.52英寸，新品会在今年下半年登场。</p>
