<p>今天，小米公司Redmi产品总监王腾透露，Redmi Note 11顶配版升级的有点猛了。</p>
<p>此前有爆料指出，<strong>Redmi Note 11顶配版支持120W快充。</strong></p>
<p>Redmi Note 11海报文案中出现了“一刻满血”，似乎暗示Redmi Note 11仅需15分钟就能充至100%。</p>
<p>今年小米发布的旗舰中，<span style="color:#ff0000;"><strong>仅有小米MIX 4、小米11T系列支持了120W快充，这些都是小米的高端旗舰手机。</strong></span></p>
<p style="text-align: center;"><a href="https://img1.mydrivers.com/img/20211020/4b94994f3cc1460d99256cf1dddff1e6.jpg" style="text-align: -webkit-center;" target="_blank"><img alt="堆料堪比MIX 4！王腾：Redmi Note 11顶配版有点猛" h="140" src="https://img1.mydrivers.com/img/20211020/s_4b94994f3cc1460d99256cf1dddff1e6.jpg" style="border: 1px solid black;" w="600" referrerpolicy="no-referrer"></a></p>
<p>Redmi Note 11顶配版也将支持百瓦级快充，<strong>意味着Redmi Note 11系列在同档位将脱颖而出</strong>，极具竞争力。</p>
<p>此外，Redmi Note 11系列将采用挖孔屏方案，前置摄像头居中，预计刷新率为120Hz，后置矩阵相机，同时配备了双扬声器。</p>
<p>按照Redmi极致性价比的定位，Redmi Note 11的价格让人期待，该机将于10月28日正式发布，我们拭目以待。</p>
