<p>5月24日晚，Redmi将发布全新Redmi Note 11T系列新机，主题为“真芯英雄”。</p>
<p>卢伟冰曾表示，Redmi Note 11T系列是性能小金刚，刷新中端机性能新战绩，打造用户心中的LCD旗舰直屏。</p>
<p><strong>现在，我们快科技已经收到了Redmi Note 11T 的邀请函，居然是一个铁臂阿童木的手办。</strong></p>
<p>这款手办做工非常精致，并提供了透明、半透明、不透明三种配件，用户可以自己随意变换外形。</p>
<p>《铁臂阿童木》是日本漫画家手冢治虫创作的科幻漫画作品，讲述了未来21世纪的少年机器人阿童木的故事。<strong>阿童木安装了一块芯片，具有和人类一样的心灵，能够产生诸多情感。这个聪明、勇敢、正义的小机器娃娃几乎是人见人爱。</strong></p>
<p>该作品先后多次被改编为动画，其中1963版第一版电视动画是日本第一部电视连续动画，也是中国引进的第一部国外动画。</p>
<p><strong>铁壁阿童木的形象恰好与Redmi Note 11T系列“真芯英雄”的主题相契合。</strong></p>
<p>据了解，Redmi Note 11T系列搭载联发科天玑8000系列旗舰处理器，这是Redmi迄今为止性能最强悍的Note系列手机，这颗芯片基于台积电5nm工艺制程打造，由四颗Cortex A78大核+四颗Cortex A55小核组成，性能强悍。</p>
<p>5月24日，Redmi Note 11T 的发布会马上就要来了，我们拭目以待。</p>
