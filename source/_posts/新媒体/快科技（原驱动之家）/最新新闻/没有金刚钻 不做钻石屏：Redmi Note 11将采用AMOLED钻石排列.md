<p>今天中午，小米Redmi品牌总经理卢伟冰发布微博，称“知道你们关心什么”，<strong><span style="color:#ff0000;">并正式公布Redmi Note 11将会采用AMOLED钻石排列。</span></strong></p>
<p>卢伟冰在微博中介绍称：<strong>钻石排列的像素排布更科学，读书时文字边缘锐利，看电影色彩呈现更富层次感。</strong></p>
<p align="center"><a href="https://img1.mydrivers.com/img/20211022/7acbb9212adc4c7aa32f9f3bd91dbe2c.png" target="_blank"><img alt="没有金刚钻 不做钻石屏：Redmi Note 11将采用AMOLED钻石排列" h="492" src="https://img1.mydrivers.com/img/20211022/s_7acbb9212adc4c7aa32f9f3bd91dbe2c.png" style="border: black 1px solid;" w="600" referrerpolicy="no-referrer"></a></p>
<p>据悉，钻石排列是三星在传统Pentile排列基础上进行改进后设计出的一种OLED屏幕像素排列方式。使用钻石排列的OLED屏幕在像素上以四个像素点为一组，每个像素组之间呈菱形进行排列，这也是这种像素排列方式被称为钻石排列的主要原因。</p>
<p>钻石排列比传统的Pentile排列，RGBW排列等像素排列方式更加科学，能够有效提升屏幕的等效PPI，这使得钻石排列问世不久就获得了硬件厂商与消费者的好评。</p>
<p>从现在透露的信息来看，<strong>Redmi Note 11的屏幕将会支持120Hz高刷新率和360Hz触控采样率</strong>，这使得Redmi Note 11在游戏中将为玩家提供更快，更精准的显示与反应速度。</p>
<p>同时Redmi Note 11在前置摄像头开孔上的控制也相当出色，<strong>仅有2.96毫米的开孔最大程度的保证了用户在日常使用中不会被前置摄像头遮挡内容</strong>，并且保证了机器的美观程度。</p>
<p><strong>除此之外Redmi Note 11还将支持360°光线感应与DCI-P3广色域</strong>，这使得Redmi Note 11在不同的光照条件与使用场景下都将能够保持合适的亮度与优秀的色彩还原度。</p>
<p>这样一块素质优秀的屏幕搭配JBL双扬声器，超长续航时间和“特立独行”的直边设计，相信Redmi Note 11将会是一款值得期待的“小金刚”。</p>
