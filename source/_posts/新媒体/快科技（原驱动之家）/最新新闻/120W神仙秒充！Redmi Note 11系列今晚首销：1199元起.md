<p>10月28日，Redmi Note家族新成员——Note 11系列正式登场，包括Redmi Note 11、Redmi Note 11 Pro、Redmi Note 11 Pro+三款机型。</p>
<p>经过多日等待，<span style="color:#ff0000;"><strong>Redmi Note 11系列将于今晚0点开启首销，售价1199元起。</strong></span></p>
<p>该系列首发联发科天玑920 5G处理器，拥有三星120Hz高刷屏、1亿像素超清主摄，并下放了旗舰级的120W神仙秒充，15分钟即可满血。</p>
<p>具体来看，Redmi Note 11内置5000mAh电池，支持33W快充，62分钟即可充满，Note 11 Pro配备5160mAh电池，支持67W快充，43分钟充满。</p>
<p>而Note 11 Pro+配备4500mAh电池，<strong>独享120W神仙秒充，15分钟充满。</strong></p>
<p>值得一提的是，Note 11系列的120W快充标配充电器，消费者无需额外购买。</p>
<p>据介绍，Redmi Note 11 Pro+采用双电芯+双电荷泵方案，双电芯串联，单个电芯电压降低，充电更安全，而双电荷泵并联，电阻更低，发热更少。</p>
<p>ID设计上，Redmi Note 11系列采用潮流小立边设计，有着犀利、直上直下的线条，机身轮廓硬朗锐利。</p>
<p>在AG磨砂玻璃的加持下，整机触感温润不留指纹，质感绝佳，8.34mm的轻薄机身，更是进一步提升手机握持感。</p>
<p>性能方面，Redmi Note 11系列全球首发天玑920芯片，<strong>该处理器采用旗舰芯同款的台积电6nm工艺，拥有A78双大核，性能与功耗的绝佳平衡。</strong></p>
<p>综合体验部分，Note 11系列保留了“极为先进”的3.5mm耳机孔，搭载JBL对称式立体声扬声器（Note11不支持），支持Hi-Res高解析认证，同时还配备了NFC、红外、X轴线性马达、Wi-Fi 6等配置。</p>
