<p>10月底，Redmi正式发布全新Note 11系列，除带来Redmi Note 11、Redmi Note 11 Pro、Redmi Note 11 Pro+三款机型外。</p>
<p>还推出了一款特别版：<span style="color:#ff0000;"><strong>Redmi Note 11潮流限定版（基于Redmi Note 11 Pro+打造），该机全球首发全息悬浮工艺。</strong></span></p>
<p>经过多日等待，Redmi Note 11潮流限定版将于今日上午10:00正式开售，售价2699元。</p>
<p>ID设计上，Redmi Note 11潮流限定版在18cm²的空间中包含了11万个微米级Redmi Logo，<strong>通过11万个完美对应的微透镜，最终放大150倍，在机身上呈现出3D悬浮感的立体 Redmi Logo。</strong></p>
<p>而机身的线条装饰则来自于赛道，配色十分抢眼。</p>
<p>此外，机身还有Redmi品牌代言人王一博签名，外观辨识度拉满。</p>
<p>值得一提的是，有博主建议Redmi将这种工艺下放到Redmi其它机型。</p>
<p>对此，Redmi品牌总经理卢伟冰表示，Redmi Note 11潮流限定版工艺成本太高了。</p>
<p>核心配置方面，Redmi Note 11潮流限定版搭载联发科天玑920芯片，<strong>配备120W神仙秒充，仅15分钟即可充满，是目前Redmi充电速度最快的机型。</strong></p>
