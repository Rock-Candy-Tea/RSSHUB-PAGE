<p>小米集团中国区总裁、Redmi品牌总经理卢伟冰此前表示，Redmi Note系列会半年一迭代，分别定位是“性能小旗舰”和“体验小旗舰”。</p>
<p>今年上半年，Redmi推出了Note 11T Pro系列，主打的是“性能小旗舰”，<strong>那么今年下半年要发布的Redmi Note 12系列定位是“体验小旗舰”。</strong></p>
<p>今日消息，博主数码闲聊站透露，Redmi Note 12系列是一款冲击双11的新品，<span style="color:#ff0000;"><strong>暗示这款新机会在双11之前登场。网友猜测，Redmi Note 12很有可能会在10月份发布。</strong></span></p>
<p>按照惯例，Redmi Note 12系列至少有两款新品，<strong>包括Redmi Note 12和Redmi Note 12 Pro，也可能有超大杯Redmi Note 12 Pro+。</strong></p>
<p>传闻Redmi Note 12系列使用的芯片名为天玑1080，由台积电代工，安兔兔跑分在50-60万之间，与高通骁龙778G Plus相差不大，后者安兔兔跑分在55万分左右。</p>
<p>此外，爆料称Redmi Note 12系列标配120W有线快充，采用居中挖孔直屏，<strong>后置三颗摄像头，包括5000万主摄、800万超广角和200万副摄。</strong></p>
<p>售价方面，Redmi Note 12定价应该在1000-1500元之间，而Redmi Note 12 Pro定价应该在1500-2000元之间。</p>
