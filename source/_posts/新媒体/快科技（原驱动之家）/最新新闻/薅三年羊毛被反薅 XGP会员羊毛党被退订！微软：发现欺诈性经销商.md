<p>6月19日消息，近日不少网友反馈，此前 100 元 Bug 价薅羊毛购买的三年 Xbox Game Pass 或者 Xbox Game Pass Ultimate 订阅会员现在被微软大面积退订处理。</p>
<p>据微软支持社区工作人员的回答，Xbox 发现了一组欺诈性经销商，他们堆叠了 PC 介绍性优惠的多个实例并将其转换为 Game Pass Ultimate。</p>
<p>为解决此问题，<strong>Xbox 已取消他们以欺诈方式获得的订阅，并在此之前单独通知了受影响的用户。</strong></p>
<p>为什么要取消这些 Xbox Game Pass 订阅？</p>
<p>我们发现了一些通过违反 Microsoft 服务协议的来源获得的成员资格。订阅必须直接从 Microsoft 数字商店或经过验证的第三方零售商处购买。</p>
<p>会员账户数据和游戏历史会怎样？</p>
<p>这都将被保留。您的帐户将保留，但订阅将被停用。</p>
<p>受影响的 Xbox Game Pass 会员该怎么办？</p>
<p>受影响的成员可以随时通过加入 Xbox Game Pass 重新订阅，或经过验证的第三方卖家购买。</p>
<p>这将影响多少成员？</p>
<p><strong>虽然我们没有透露受影响的成员数量，但这是 Xbox Game Pass 社区的一小部分。</strong></p>
<p>据悉，微软社区有网友爆料称，此次被取消XGP的用户，基本都在去年通过系统漏洞“薅羊毛”低价拿到了三年XGPU会员。</p>
<p>该漏洞是利用去年动画网站crunchyroll活动赠送的PC Game Pass试用码，配合1美元的升级漏洞转为最长三年的XGPU会员。</p>
<p><strong>该漏洞还被部分电商商家上架销售，收费普遍在100多元，谋取暴利。</strong></p>
