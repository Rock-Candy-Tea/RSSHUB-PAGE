<p>今天早些时候，微软更新Windows 11的最低系统配置要求，增加对部分旧处理器的支持。</p>
<p>一组Intel第七代处理器被加入了列表，基于这些发现，兼容的64位处理器的名单扩大，包括以下内容：</p>
<p style="text-align: center;"><a href="https://img1.mydrivers.com/img/20210828/4fcb52602aa34bb2a47159f4eca23b84.jpg" style="text-align: -webkit-center;" target="_blank"><img alt="微软更新Windows 11最低系统需求：Intel诸多处理器被列入" h="497" src="https://img1.mydrivers.com/img/20210828/s_4fcb52602aa34bb2a47159f4eca23b84.jpg" style="border: 1px solid black;" w="600" referrerpolicy="no-referrer"></a></p>
<p>酷睿TM X系列、至强W系列</p>
<p>酷睿TM 7820HQ（支持声明性组件化硬件支持应用，DCH）</p>
<p>具体型号包括：</p>
<p>Intel Core i5-7640X</p>
<p>Intel Core i7-7740X</p>
<p>Intel Core i7-7800X</p>
<p>Intel Core i7-7820HQ</p>
<p>Intel Core i7-7820X</p>
<p>Intel Core i7-7900X</p>
<p>Intel Core i7-7920X</p>
<p>Intel Core i9-7940X</p>
<p>Intel Core i9-7960X</p>
<p>Intel Core i9-7980XE</p>
<p><strong>不幸的是，在微软与AMD做了一些额外的测试后，第一代AMD Zen处理器仍然没有入选。除了上述Intel处理器，微软对Windows 11的最低系统要求保持不变，其中包括4GB内存、64GB存储、UEFI、DirectX12 GPU和TPM 2.0芯片。</strong></p>
<p>为了证明其对Windows 11的严格最低规格的合理性，微软强调，其数据显示，不符合最低系统要求的PC无法可靠地运行该操作系统。"不符合最低系统要求的设备在内核模式下的崩溃率高出52%。而符合最低系统要求的设备则有99.8%的无崩溃体验，"Windows团队说。</p>
