<p>去年9月，微软正式下发了Windows Server 2022正式版系统。</p>
<p>类似于Windows 10/11，WinServer同样也有内测预览，本月的最新版Build 25158已经上线，且<a class="f14_link" href="https://www.microsoft.com/en-us/software-download/windowsinsiderpreviewserver?ranMID=24542&ranEAID=TnL5HPStwNw&ranSiteID=TnL5HPStwNw-zgnRRRPIGCgtBhrECcpf8w&epi=TnL5HPStwNw-zgnRRRPIGCgtBhrECcpf8w&irgwc=1&OCID=AID2200057_aff_7593_1243925&tduid=%28ir__11ila9n10gkf6wblq9mzlzttk32xvtlu316r6hfe00%29%287593%29%281243925%29%28TnL5HPStwNw-zgnRRRPIGCgtBhrECcpf8w%29%28%29&irclickid=_11ila9n10gkf6wblq9mzlzttk32xvtlu316r6hfe00" target="_blank">官网还开放下载</a>ISO镜像、VHDX等，注意，下载需要注册Insider会员。</p>
<p>激活序列号如下：</p>
<p>Server Standard: </p>
<p>MFY9F-XBN2F-TYFMP-CCV49-RMYVH</p>
<p>Datacenter: </p>
<p>2KNJJ-33Y9H-2GXGX-KMQWH-G6H67</p>
<p>细心的朋友恐怕发现了，Build 25158的版本号和Win11 23H2的最新预览版保持一致，这是因为它们均基于Sun Valley 3底层开发而来。</p>
<p>不过，<strong>微软强调，预览版严格来说对应的是Windows Server vNext系统，而不是对Server 2022的小修小改。</strong></p>
<p>其中ISO镜像包含18种预言，VHDX虚拟镜像则只提供英文版。</p>
<p>另外，微软还提醒Azure管理员查收Windows Server Vnext Datacenter: Azure Edition build 25120。</p>
