<p>今天，realme印度CEO Madhav Sheth宣布，<span style="color:#ff0000;"><strong>基于Android 12打造的realme UI 3.0即将发布。</strong></span></p>
<p>此前在realme GT Neo2发布会上，realme副总裁徐起就预告realme UI 3.0将会在10月份亮相，<strong>届时会公布realme老机型适配realme UI 3.0计划。</strong></p>
<p>目前realme UI 2.0版本距离发布有将近一年时间，该版本系统基于Android 11深度定制，在流畅度和安全性方面都有不错的表现。</p>
<p>流畅度方面，<strong>realme UI 2.0加入了Quantum量子动画引擎</strong>，对于日常频繁使用的多任务切换、列表滑动等都有了明显的动画效果，在回弹上更为真实，触及时偏向真实物理，有逻辑可循，能给予用户更舒适的反馈。</p>
<p>安全方面，realme UI 2.0带来了互联网隐身+系统分身，支持伪基站识别，能快速准确识别骚扰诈骗电话和钓鱼信息，自动屏蔽伪基站，对响一声来电和空号都能自动拦截，让用户远离骚扰和诈骗。</p>
<p>如今基于Android 12打造的realme UI 3.0版本即将发布，预计realme UI 3.0也将会在设计、流畅度以及安全隐私保护等方面会带来新的惊喜，我们拭目以待。</p>
