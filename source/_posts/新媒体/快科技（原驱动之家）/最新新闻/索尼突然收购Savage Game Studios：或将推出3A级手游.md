<p>今天，<span style="color:#ff0000;"><strong>索尼突然在PlayStation官网发布新闻稿，毫无征兆的宣布已经收购了游戏工作室Savage Game Studios。</strong></span></p>
<p align="center"><a href="https://img1.mydrivers.com/img/20220829/1dc576950e4c4f438bb75ad369cffbd7.png" target="_blank"><img alt="索尼突然收购Savage Game Studios：或将推出3A级手游" h="275" src="https://img1.mydrivers.com/img/20220829/s_1dc576950e4c4f438bb75ad369cffbd7.png" style="border: black 1px solid;" w="600" referrerpolicy="no-referrer"></a></p>
<p>据悉，Savage Game Studios是一家在赫尔辛基和柏林设有办事处的手游开发工作室，由Michail Katkoff、Nadjim Adjir和 Michael McManus领导。</p>
<p>目前，Savage Game Studios并未在官网放出任何参与制作的游戏作品，但根据索尼方面的消息，<strong>该工作室正在研发一款尚未公布的全新3A级服务型动作手游。</strong></p>
<p>PlayStation Studios的负责人Hermen Hulst表示：PlayStation Studios必须继续扩展和多样化我们的产品，超越主机平台，为更多人带来前所未有的新游戏。</p>
<p>不出意外，Savage Game Studios将会加入PlayStation Studios新成立的移动部门，<strong>该部门独立于现有的主机游戏部门，专注于基于全新或现有的PlayStation IP，制作满足移动端的3A级游戏。</strong></p>
<p>当前，索尼与Savage Game Studios都未公布此次收购的价格与合同细节，仅表示工作室的日常运营依旧由此前的管理团队负责。</p>
