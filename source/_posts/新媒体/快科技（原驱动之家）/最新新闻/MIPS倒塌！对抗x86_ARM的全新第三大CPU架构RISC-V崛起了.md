<p>近日，Imagination Technologies宣布推出IMG RTXM-2200，这是其首款实时嵌入式 RISC-V CPU，这是一款高度可扩展、功能丰富的 32 位嵌入式解决方案，具有灵活的设计，适用于各种大容量设备。</p>
<p>IMG RTXM-2200 是 Imagination 的Catapult CPU 系列中首批商用内核之一，此前于 2021 年 12 月发布。</p>
<p>Imagination 的 IMG RTXM-2200 可加速其 RISC-V 产品的扩展，可集成到复杂的 SoC 中，用于一系列应用，包括网络解决方案、数据包管理、存储控制器以及 AI 相机和智能计量的传感器管理。连同其市场领先的 GPU 和 AI 加速器 IP，Imagination 的新 CPU 内核为客户提供创新的异构解决方案。</p>
<p>这种实时嵌入式内核具有高达 128KB 的紧密耦合存储器（指令和数据），用于确定性响应，以及高达 128KB 的 1 级高速缓存大小，可实现稳健的性能。</p>
<p>新 CPU 提供了一系列浮点格式，包括单精度和 bfloat16。后者使制造商能够通过该内核部署 AI 应用程序，而无需额外的芯片。这减少了硅面积，从而在 AI 相机和智能计量应用中实现具有成本效益和优化的设计。</p>
<p>Imagination 首席产品官 Chris Porthouse 表示：“我们很高兴宣布推出 IMG RTXM-2200，这是我们的第一款实时 RISC-V CPU，它巩固了我们对推动 RISC-V 生态系统增长的持续承诺。Imagination 现在拥有独特而广泛的计算 IP 组合，包括 GPU、AI 加速器和现在的 CPU，专为这种创新架构而设计。”</p>
<p>RISC-V International 首席执行官 Calista Redmond 表示：“自Imagination宣布进入 RISC-V CPU IP 市场以来，突显了 RISC-V 架构的扩展特性。</p>
<p>开发人员也利用这种灵活的技术来满足多种市场需求。我们很高兴看到 Imagination 致力于通过交付其 SDK 和支持开源操作系统来推动生态系统的增长。”</p>
<p>ABI Research 首席分析师 James Hodgson 表示：“随着汽车制造商希望利用 RISC-V 的潜在创新和上市时间优势，他们需要一个具备在汽车市场交付能力的 IP 合作伙伴，满足严格的垂直的电源和安全要求。</p>
<p>随着 IMG RTXM-2200 的推出，Imagination 为该市场未来的汽车 RISC-V CPU 奠定了基础，完善了其 GPU 和 AI 加速 IP 产品组合。这使 Imagination 能够很好地支持异构计算，这将证明对于满足汽车行业未来的互联信息娱乐和自动驾驶汽车的雄心壮志至关重要。”</p>
<p>TIRIAS Research 首席分析师史蒂夫·莱布森 (Steve Leibson) 表示：“很高兴看到 Imagination 等成熟的 IP 提供商加入并为 RISC-V 提供了更大的动力。最初强调实时嵌入式应用，可配置的 IMG RTXM-2200 CPU 是该公司先前宣布的 Catapult 系列中的第一个商业 RISC-V 内核，应该会吸引物联网和其他嵌入式市场的 SoC 设计团队。”</p>
<p>凭借 Imagination 在提供复杂 IP 解决方案方面 20 多年的经验，新 CPU 得到快速扩展的开放标准 RISC-V 生态系统的支持，该生态系统通过计算民主化、解锁创新和开放技术继续撼动嵌入式 CPU 行业为各级企业发展。</p>
<p>Imagination 实时嵌入式内核在发布时还附带功能齐全的软件 SDK 和工具包。他们在多个平台上开箱即用，以帮助开发和支持 RISC-V 生态系统。</p>
<p>Imagination 计算副总裁 Shreyas Derashri 表示：“20 多年来，Imagination 一直在创造市场领先的 GPU、AI 加速器和 CPU IP。这使我们处于一个独特的位置，在下一代 SoC 开发方面，我们可以为客户提供强大且具有竞争力的选择。IMG RTXM-2200 RISC-V CPU 汇集了多年的 IP 设计经验和全面的软件支持，使开发人员能够即时轻松地访问，进一步增强了我们的异构产品。”</p>
<p>Catapult Studio 集成开发环境 (IDE) 是基于 Visual Studio 代码的现代尖端 IDE，提供了一个更适合最新开发人员和设计人员使用的熟悉环境。</p>
<p><strong>在 Windows、Ubuntu、CentOS 和 macOS 上运行；</strong></p>
<p>提供完整的 Linux 支持（包括参考引导加载程序、内核和文件系统）；</p>
<p>与 gem5 软件兼容，解锁模拟环境以增强功率和能效测试。</p>
<p>IMG RTXM-2200 CPU 是 Imagination 异构计算解决方案的一部分，为 Imagination IP 提供完整的硬件、软件和调试支持——补充了其市场领先的GPU、AI 加速器和以太网数据包处理器 (EPP)内核。</p>
