<p>在今天（10月27日）上午小米集团中国区总裁、Redmi品牌总经理在微博晒出Redmi Note 11超过18小时的续航时间后，网友便开始了热烈的讨论。而不久后另一位小米高管的微博则再次引爆了话题。</p>
<p>小米集团副总裁常程在今天上午转发了卢伟冰关于Redmi Note 11续航时间的微博，并配文：<span style="color:#ff0000;"><strong>“‘这只是4500mAh版本’...难道意味着还有一个版本?”</strong></span></p>
<p>在这条微博之前就已经有网友对卢伟冰微博中提到的“这只是4500mAh版本”展开了猜测，<strong>而常程的这条微博则几乎做实了Redmi Note 11的电池将不止有4500mAh这一个版本。</strong></p>
<p>根据Redmi Note系列的惯例，Redmi Note 11应该会分为“Redmi Note 11”和“Redmi Note 11 Pro”两个型号进行发售。结合卢伟冰和常程在微博中的暗示，<strong>或许我们可以猜测在Redmi Note 11系列中的Redmi Note 11将会采用4500mAh的电池，而配置和定价更高的“Redmi Note 11 Pro”则会采用一块更大，续航更久的电池。</strong></p>
<p>作为Rebmi Note系列的新机，Redmi Note 11在千元机价位的硬件已经能够说是相当优秀。</p>
<p>Redmi Note 11将搭载全球首发的联发科天玑920处理器，8GB内存，并将采用一块120Hz AMOLED全面屏，支持WiFi 6和NFC功能，同时拥有一块4500mAh大电池和120W超级闪充。</p>
<p>那么将会拥有更大容量电池的“Pro”版本又能在续航以外的地方带给我们多少惊喜呢？</p>
