<p>本月，小米在印度发布小米11i HyperCharge，售价26999印度卢比（约合人民币2300元），该机支持120W超级闪充，仅需15分钟就能充至100%，是印度智能手机市场充电最快的机型。</p>
<p>继小米11i HyperCharge之后，今天下午，小米印度业务负责人Manu Kumar Jain在推特宣布，<span style="color:#ff0000;"><strong>Redmi Note 11S即将发布。</strong></span></p>
<p>如海报所示，Redmi Note 11S采用了类似Redmi Note 11的矩阵相机设计，详细参数官方尚未透露。</p>
<p>此前有爆料称Redmi Note 11S采用90Hz高刷屏，搭载联发科天玑处理器，<span style="color:#ff0000;"><strong>后置一亿像素（108MP）、800万超广角和微距三摄，前置1300万像素。</strong></span></p>
<p>其中<strong>这颗一亿像素主摄为三星HM2，这颗sensor曾被应用到Redmi超大杯旗舰Redmi K40 Pro+上，</strong>它拥有1/1.52英寸大底，单位像素面积为0.7μm，可以通过像素九合一输出1200万像素图片。</p>
<p>从规格不难看出，Redmi Note 11S是Redmi Note 11的小幅升级版，定价有望在千元左右。</p>
