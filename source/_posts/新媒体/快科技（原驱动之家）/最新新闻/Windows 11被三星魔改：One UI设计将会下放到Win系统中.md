<p>和Android手机品牌不同，PC厂商不能随意调整Windows操作系统的界面设计，不过这种局面很快就会有所改变。</p>
<p>10月30日消息，据XDA报道，在今年的三星开发者大会上，<strong>三星宣布与微软的合作进一步深入，<span style="color:#ff0000;">三星笔记本用户可以在PC端体验到One UI的设计风格，以此来保证手机和PC端的体验一致性。</span></strong></p>
<p>据悉，三星很快会在自家笔记本电脑上带去One UI的设计。和Windows 11系统相比，经过三星调整的Windows系统主界面风格更像One UI，<strong>三星称之为“One UI Book 4”。</strong>PS：三星最新版手机操作系统命名为One UI 4。</p>
<p>这次三星Galaxy Book Pro、Galaxy Book 360、Galaxy Book Flex2、Galaxy Book Odyssey等Galaxy Book笔记本上都会体验到这一全新的操作系统。</p>
