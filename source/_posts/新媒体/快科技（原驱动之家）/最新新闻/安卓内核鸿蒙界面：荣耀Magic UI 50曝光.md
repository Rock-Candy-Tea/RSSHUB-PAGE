<p>眼看着距离荣耀Magic 3的发布时间越来越近，与之配套的Magic UI 5.0也终于浮出水面。</p>
<p>今天国内曝料大神放出了疑似搭载荣耀Magic UI 5.0的手机照片，并透露全新的Magic UI 5.0更类似于华为的鸿蒙HarmonyOS，支持于大部分安卓机型多屏协同。</p>
<p>Magic UI 5.0或基于Andorid 11系统开发而来，可以理解安卓系统套上了鸿蒙HarmonyOS的UI，支持大部分鸿蒙HarmonyOS功能，但不支持超级终端功能。</p>
<p style="text-align: center"><img alt="安卓内核鸿蒙界面：荣耀Magic UI 5.0曝光" h="304" src="https://img1.mydrivers.com/img/20210806/85c46b92-0a01-46fd-a1c8-d4bdcb3299ed.jpg" style="border:black 1px solid" w="588" referrerpolicy="no-referrer"></p>
<p>这种换皮鸿蒙的做法在前天发布的麦芒10SE上就出现过，不过麦芒10SE的系统是EMUI12。</p>
<p>华为这样做也有自己的苦衷，一方面芯片危机还没过去，另一方面日子还得过手机还得发，鸿蒙HarmonyOS又不能放弃不做，只能用给新机换UI的方式留住用户，荣耀虽然已经独立，但毕竟与华为藕断丝连，搭载换皮鸿蒙HarmonyOS的安卓系统完全在情理之中。</p>
<p>荣耀Magic 3或搭载高通骁888 Plus处理器，拥有12GB运行内存和512GB存储内存版本，具备IP68防水功能。</p>
<p>据悉，荣耀Magic 3和Magic UI 5.0将在8月12日19:30正式与我们见面。</p>
