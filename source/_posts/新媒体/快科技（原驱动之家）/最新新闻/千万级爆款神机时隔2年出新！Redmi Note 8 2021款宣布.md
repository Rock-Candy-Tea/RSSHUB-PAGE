<p>5月21日消息，小米官方推特宣布，Redmi Note 8 2021款即将发布。</p>
<p>小米介绍，Redmi Note 8在全球销量突破了2500万台，为了庆祝这一里程碑，<strong><span style="color:#ff0000;">我们即将推出Redmi Note 8 2021</span></strong>，敬请期待。</p>
<p>资料显示，<strong>Redmi Note 8系列是Redmi 2019年8月份在小米北京总部发布的千元新机，售价999元起。</strong></p>
<p>时隔将近2年时间，Redmi即将带来新版Redmi Note 8，这款新机已经现身FCC。</p>
<p>和老款相比，Redmi Note 8 2021款最大的升级之处是处理器和快充，<span style="color:#ff0000;"><strong>它搭载的是联发科Helio G85芯片，支持22.5W快充。</strong></span>PS：Redmi Note 8使用的是高通骁龙665处理器，支持18W快充。</p>
<p>此外，Redmi Note 8 2021款电池容量同样是4000mAh，后置主摄为4800万像素，存储空间最大为128GB，出厂预装全新的MIUI 12.5系统（基于Android深度定制）。</p>
<p>从定位来看，Redmi Note 8 2021款是一款千元新机，可能不会亮相国内。</p>
