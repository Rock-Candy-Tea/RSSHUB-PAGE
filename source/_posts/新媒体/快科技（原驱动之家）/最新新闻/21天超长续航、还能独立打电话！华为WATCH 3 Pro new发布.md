<p>7月27日，在HarmonyOS 3及华为全场景新品发布会上，华为WATCH 3 Pro new正式发布。</p>
<p><strong>华为WATCH 3 Pro new是搭载HarmonyOS 3的独立智能高端腕表，支持eSIM独立上网通信，具备独立通讯能力，后续通过OTA升级支持微信手表版、华为手表独立导航应用等，综合体验全面增强。</strong></p>
<p style="text-align: center"><a href="https://img1.mydrivers.com/img/20220727/126dcfe5-7ecd-4e69-a4cf-f15659adec65.jpg" target="_blank"><img alt="21天超长续航、还能独立打电话！华为WATCH 3 Pro new发布" h="400" src="https://img1.mydrivers.com/img/20220727/S126dcfe5-7ecd-4e69-a4cf-f15659adec65.jpg" style="border:black 1px solid" w="600" referrerpolicy="no-referrer"></a></p>
<p>HarmonyOS 3的加持，让华为WATCH 3 Pro new升级了与鸿蒙生态设备的超级联动功能，手表与手机协同，可快速启动遥控拍照；与跑步机、走步机、室内单车等生态设备协同联动，还能充当AITO问界汽车的数字钥匙，实现手表远程控车。</p>
<p><strong>一切为了高端：蓝宝石玻璃镜面、钛金属表体、陶瓷后壳</strong></p>
<p>华为WATCH Pro系列的外观ID以及做工用料一直主打高端，这一次，华为WATCH 3 Pro new又有了升级。</p>
<p style="text-align: center"><a href="https://img1.mydrivers.com/img/20220727/654e36a4-395a-411f-82b2-b798ea77c4ef.jpg" target="_blank"><img alt="21天超长续航、还能独立打电话！华为WATCH 3 Pro new发布" h="400" src="https://img1.mydrivers.com/img/20220727/S654e36a4-395a-411f-82b2-b798ea77c4ef.jpg" style="border:black 1px solid" w="600" referrerpolicy="no-referrer"></a></p>
<p>首先在镜面上，华为WATCH 3 Pro new使用了蓝宝石玻璃材质，1.43英寸的AMOLED高清彩屏不仅显示区域更大，看起来也更加通透。</p>
<p style="text-align: center"><a href="https://img1.mydrivers.com/img/20220727/a07c7e71-acc2-4e7d-9daa-f8585e94a2ba.jpg" target="_blank"><img alt="21天超长续航、还能独立打电话！华为WATCH 3 Pro new发布" h="400" src="https://img1.mydrivers.com/img/20220727/Sa07c7e71-acc2-4e7d-9daa-f8585e94a2ba.jpg" style="border:black 1px solid" w="600" referrerpolicy="no-referrer"></a></p>
<p>表体采用耐用且抗腐蚀的钛金属，同时升级陶瓷后壳，经过42道工序，包括超过1400℃高温烧结后精密抛光，与蓝宝石玻璃镜面一体打磨成型。戴起来温润亲肤，心率监测也更加精准。</p>
<p style="text-align: center"><a href="https://img1.mydrivers.com/img/20220727/86fb8e42-cd90-4df3-bfb5-3ad2f11804b0.jpg" target="_blank"><img alt="21天超长续航、还能独立打电话！华为WATCH 3 Pro new发布" h="400" src="https://img1.mydrivers.com/img/20220727/S86fb8e42-cd90-4df3-bfb5-3ad2f11804b0.jpg" style="border:black 1px solid" w="600" referrerpolicy="no-referrer"></a></p>
<p>整个外观最抢眼的部分，是这个带触觉反馈的3D旋转表冠，细密的齿纹搭配精致红圈，很显高级感。它还支持高精度手指运动识别，支持缩小/放大图形、滑动界面，还能调整参数，同时自带马达震动，体验上有一种很强的精密感。</p>
<p>下键则新增了将电极技术与按键合二为一的ECG按键，支持精准传感心率信息，佩戴者可随时随地进行精准心电分析。</p>
<p><strong>eSIM独立通信、HarmonyOS 3加持：功能像手机一样强大</strong></p>
<p>在跑步、游泳、滑板等户外场景，携带手机多有不便，这时候华为WATCH 3 Pro new的eSIM独立通信功能就派上用场了。用户只需要在手机上开通一号双终端业务，不用去营业厅，就可使用和手机相同的号码，还能共享手机套餐中的语音和流量。</p>
<p>华为WATCH 3 Pro new不仅支持航班、短信提醒、邮件、第三方App、喝水等，还支持微信手表版，微信支付等，用户可通过手机扫码登录微信。在手表上聊微信，不掏手机也可以快速查看、快捷回复消息。还能独立导航、打车，独立下载音乐，碰一碰刷交通卡、门禁卡，功能和手机一样强大。</p>
<p style="text-align: center"><a href="https://img1.mydrivers.com/img/20220727/871ddea7-d27f-4bf4-8e4e-081a6eb80fae.jpg" target="_blank"><img alt="21天超长续航、还能独立打电话！华为WATCH 3 Pro new发布" h="399" src="https://img1.mydrivers.com/img/20220727/S871ddea7-d27f-4bf4-8e4e-081a6eb80fae.jpg" style="border:black 1px solid" w="600" referrerpolicy="no-referrer"></a></p>
<p>很多人期待的独立音乐播放，因为有了eSIM独立通信能力，华为WATCH 3 Pro new也得以实现。它不仅支持在线听歌，还能独立下载音乐，跑步时再也不用不带手机了。</p>
<p>AITO问界M5和M7的车主还能通过华为WATCH 3 Pro new实现远程控车，比如在炎炎夏日，提前打开空调；停车场寻车鸣笛、闪灯找位置。还能抬腕碰车后视镜就可解锁车门、联动车机。</p>
<p>得益于HarmonyOS 3的加持，华为WATCH 3 Pro new还实现了华为超级联动（需后续升级），比如手表与手机协同，支持快速启动遥控拍照；与跑步机、走步机、室内单车等生态设备协同，可启动联动运动。与大屏运动健康协同，可快速启动健身运动等等。</p>
<p>对于华为智能手表的发展来说，超级联动功能的首次加持，将具有里程碑式的意义。据悉，华为WATCH系列产品后续也将会陆续升级HarmonyOS 3。</p>
<p><strong>100+运动类型、ECG心电分析 还有21天超长续航 </strong></p>
<p>运动健康方面，华为WATCH 3 Pro new同样是华为智能手表的集大成之作。</p>
<p>华为WATCH 3 Pro new支持多达100+运动类型，涵盖19种专业运动模式，86种自定义运动模式，同时新增高尔夫球场运动模式。</p>
<p>其中19种专业运动模式，包括12种户外运动（跑步、步行、登山、徒步、越野跑、骑行、开放水域、铁人三项、滑雪、雪板滑雪、越野滑雪、高尔夫），7种室内运动（步行、跑步、骑行、泳池游泳、自由训练、椭圆机、划船机），大家喜欢的主流运动都有支持。</p>
<p style="text-align: center"><a href="https://img1.mydrivers.com/img/20220727/c2a683aa-1a56-4ffe-9e12-e0354ef5aedb.jpg" target="_blank"><img alt="21天超长续航、还能独立打电话！华为WATCH 3 Pro new发布" h="400" src="https://img1.mydrivers.com/img/20220727/Sc2a683aa-1a56-4ffe-9e12-e0354ef5aedb.jpg" style="border:black 1px solid" w="600" referrerpolicy="no-referrer"></a></p>
<p>此外，全球五大卫星定位系统（GPS，北斗，GLONASS，Galileo）它都支持，可将户外运动轨迹生成户外运动路线，还能通过微信的方式分享给好友。户外迷路也不怕，华为WATCH 3 Pro new支持离线导航，迷路的时候可以开启轨迹返航功能，根据导航指示原路返回。</p>
<p>健康监测上，华为WATCH 3 Pro new搭载了全新一代心率监测技术TruSeen 5.0+，数据处理量提升4倍，心率检测更加精准，尤其是运动场景下，准确率更高。华为WATCH 3 Pro new新增ECG心电分析，只需要指尖轻触腕表下键电极，即可采集30秒心电信号，生成心电图报告。</p>
<p>好的智能手表，不只提供健康数据，还应该提供专业健康服务。华为WATCH 3 Pro new不仅加入了中国医疗保健国际交流促进会发起的血管健康研究，基于华为WATCH 3 Pro new高性能的PPG和ECG传感器，可融合检测脉搏波传导速度，精准筛查动脉硬化风险，提供专业健康建议，实现居家主动血管健康管理；它还加入301医院发起的心脏健康研究和睡眠呼吸暂停研究，可精细化识别心律失常，提供个性化指导、心脏管家、跟踪随访等整合管理服务；加入华中科技大学同济医学院附属同济医院和301医院联合发起的呼吸健康研究，基于华为WATCH 3 Pro new呼吸健康监测技术，基于体温、呼吸率、血氧、心率等多参数交叉监测，支持呼吸系统感染早期筛查和风险预警，全方位守护佩戴者身心健康。</p>
<p style="text-align: center"><a href="https://img1.mydrivers.com/img/20220727/2f8bedd7-7325-4a2c-80f2-5a3c73420c73.jpg" target="_blank"><img alt="21天超长续航、还能独立打电话！华为WATCH 3 Pro new发布" h="400" src="https://img1.mydrivers.com/img/20220727/S2f8bedd7-7325-4a2c-80f2-5a3c73420c73.jpg" style="border:black 1px solid" w="600" referrerpolicy="no-referrer"></a></p>
<p>值得一提的是，华为WATCH 3 Pro new还支持跌倒检测及SOS求救模式，还能自动检测洗手动作和时长，只有洗手20秒以上，才会提示洗手成功。续航方面，智能模式典型场景下能够支撑4天，超长续航模式典型场景下支持21天。</p>
<p>如果用两个词形容华为WATCH 3 Pro new，一个是更独立，一个是更智慧。eSIM通信功能让华为WATCH 3 Pro new有了完全独立的使用体验，功能和智能手机一样强大。同时，Harmony OS 3的加持，让手表可以和华为生态设备有效联动，控制智能家居设备以及车机，打造一个围绕手腕的交互中心、智慧中心和健康中心，为用户带来腕上交互新体验、运动健康新突破、智慧生活新升级。</p>
<p>据悉，<strong>华为WATCH 3 Pro new时尚款售价2999元，华为WATCH 3 Pro new尊享款售价3699元，于2022年7月27日21:08正式开售！</strong></p>
