<p>今晚19:00，Redmi卢伟冰将召开一场新品发布会，此次发布会主角正是预热已久的Redmi Note 11系列，官方透露此次将有上百家媒体平台共同直播，堪称科技小春晚。</p>
<p>我们快科技也会为大家进行全程的视频+图文直播，并且还为大家准备了新机抽奖，敬请关注。</p>
<p>需要注意的是，<strong>此前几天Redmi Note 11系列已经公布了多项参数，其中不乏包括120W快充、JBL对称式立体声扬声器、X轴线性马达、Wi-Fi 6等旗舰配置，</strong>但官方还是保留了一个悬念留待发布会上揭晓。</p>
<p><span style="color:#ff0000;"><strong>今天上午，@Redmi红米手机 官微发文称，此次发布会还将公布一项全球首发的技术，</strong></span>但是官方尚未透露任何相关信息，看来只能等到发布会上才能知道了。</p>
<p align="center"><a href="https://img1.mydrivers.com/img/20211028/7819d6e1eb4443d58cbea14dba7d476f.jpg" target="_blank"><img alt="全球首发神秘新技术！Redmi Note 11发布会倒计时：号称体验小旗舰" h="799" src="https://img1.mydrivers.com/img/20211028/s_7819d6e1eb4443d58cbea14dba7d476f.jpg" style="border: black 1px solid;" w="600" referrerpolicy="no-referrer"></a><br>
Redmi Note 11还将全球首发新技术</p>
<p>根据多日的预热来看，<strong>Redmi Note 11系列此次将推出三款机型，其中包括Note 11、Note 11 Pro和Note 11 Pro+，</strong>三者在配置方面呈递增关系，但是在外观上将基本保持一致。</p>
<p>官方透露，Redmi Note 11系列采用了潮流小立边设计，类似iPhone 13的直角边框设计，配合2.5D AG玻璃机身，有着犀利、直上直下的线条，机身轮廓硬朗锐利，搭配上迷雾森林、时光静紫、浅梦星河三款配色，整机的质感瞬间拉满。</p>
<p>Redmi Note 11系列正面还配备了一块三星AMOLED高刷屏，拥有1.75mm超窄边框，2.96mm前摄开孔，<strong>整体屏占比极高，同时还支持120Hz高刷新率，配合上支持Hi-Res、杜比全景声的JBL对称式立体声扬声器，</strong>能提供超高的影音体验。</p>
<p>性能方面，<span style="color:#ff0000;"><strong>Redmi Note 11系列全球首发天玑920芯片，该处理器采用旗舰芯同款的台积电6nm工艺，</strong></span>拥有A78双大核，性能与功耗的绝佳平衡，性能可直逼骁龙778G，支持《王者荣耀》90帧模式。</p>
<p>整体来看，Redmi Note 11系列不仅保留了不错的性能，还在屏幕、外观、音效等各方面带来了更高的体验，是一个全方位进化的产品。</p>
<p>雷军都曾公开称：Note 11 Pro是一款体验小旗舰，小水桶机。</p>
<p>那么问题来了，Redmi Note 11已经有这么多全面的配置了，<strong>你觉得今晚的隐藏挑战究竟是什么呢？</strong></p>
