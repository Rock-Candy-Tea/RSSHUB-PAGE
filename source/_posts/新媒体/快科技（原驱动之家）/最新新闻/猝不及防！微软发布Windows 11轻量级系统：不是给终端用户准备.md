<p><strong>让人没有想到的是，微软悄悄发布了一个轻量级的 Windows 11系统，只是该系统并不是为终端用户准备的</strong>。</p>
<p>该系统基于命令行，称之为“Microsoft Validation OS”，主要针对软硬件供应商、开发人员和技术人员，目的是帮助诊断和解决问题。</p>
<p>如果你想尝试一下，微软Validation OS可以在以Windows Image（.WIM）文件的形式<a class="f14_link" href="https://docs.microsoft.com/zh-cn/windows-hardware/manufacture/desktop/validation-os-overview?view=windows-11&irgwc=1&OCID=AID2200057_aff_7808_1245846__fuist0kwkokfqluj0hdhyki1fm2xvgznzw09lq0c00&tduid=%28ir__fuist0kwkokfqluj0hdhyki1fm2xvgznzw09lq0c00%29%287808%29%281245846%29%2830768654%29%28242986%29&irclickid=_fuist0kwkokfqluj0hdhyki1fm2xvgznzw09lq0c00" target="_blank">下载</a>，适用于ARM64和AMD64架构。</p>
<p>正如你在微软的上述描述中所看到的，Validation OS支持Win32应用程序，这是为了帮助诊断一个基于 Windows 11 的开发中项目。这是因为Win32 API提供了对系统硬件的直接访问，并有各种工具用于诊断。</p>
<p>对于上述系统，微软的介绍是：Microsoft Validation OS 是一个轻量级的、快速的、可定制的基于 Windows 11 的操作系统，你可以在工厂车间使用，在 Windows 设备制造过程中诊断、减轻和修复硬件缺陷。验证操作系统可以启动到命令行环境，以提高工厂车间的可靠性，并支持运行 Win32 应用程序，使从早期硬件引进到零售操作系统和应用程序开发的过渡更加顺利。</p>
