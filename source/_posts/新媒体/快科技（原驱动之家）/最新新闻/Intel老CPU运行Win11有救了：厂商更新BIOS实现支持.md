<p>对于Windows 11的硬件要求，微软目前仍坚持需要Intel 8代或AMD Zen2锐龙及以上的说法。</p>
<p>最近，10款7代处理器被纳入兼容列表，包括Core i5-7640X, Core i7-7740X, Core i7-7800X, Core i7-7820HQ, Core i7-7820X, Core i7-7900X, Core i7-7920X, Core i9-7940X, Core i9-7960X以及Core i9-7980XE。</p>
<p>可以看到只有i7-7820HQ算是正统Kaby Lake，其余都是发烧平台，用户较少。</p>
<p>所以实际上，还有相当多的老用户被抛弃在外。尽管微软表示，老平台可以通过ISO镜像的方式部署Win11，但后续不会发放补丁和系统更新，这会导致蓝屏崩溃、遭受网络攻击的几率大增。</p>
<p>不过，从华硕的动作来看，对于6代、7代酷睿处理器，其有意予以全面支持，至少是在主板TPM2.0这个层面不会卡壳。</p>
<p>日前，<strong>STRIX Z270F GAMING、Z270 MAXIMUS IX FORMULA的BIOS更新中（1501），均明确确认了对6代、7代酷睿处理器兼容Windows 11的支持，且用户更新后无需动手调节UEFI设置，会自动开启TPM 2.0。</strong></p>
<p>其实从Win11允许Win10免费升级来看，微软还是在乎升级率的，所以笔者认为，口头是口头，官方最终还是会为老处理器放行。即便是有所限制，相信民间大神们定会有应对之策予以“破解”。</p>
