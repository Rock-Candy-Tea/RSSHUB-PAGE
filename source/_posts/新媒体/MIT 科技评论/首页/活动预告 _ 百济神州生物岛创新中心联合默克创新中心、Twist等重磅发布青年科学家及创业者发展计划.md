<p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);">由百济神州生物岛创新中心、广州国际生物岛、DeepTech、香港科学园、默克创新中心、研发客以及Twist Bioscience共同合作，将通过“粤港澳大湾区青年科学家及创业者发展计划”链接青年一代生命科学领域科学家及创业者，旨在加速科技创意、科学发展转变成为商业。5月13日，百济神州生物岛创新中心及合作伙伴将通过线下与线上会议（Zoom)结合的方式召开活动启动仪式，详细介绍解读该发展计划、合作模式及系列活动安排。项目覆盖领域将包括下一代突破性疗法、医药研发赋能平台技术、生物医药新兴技术如合成生物等。</p><p><br></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);">活动详情：https://www.mittrchina.com</p><p><br></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"></p><div style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"><img src="https://p26.toutiaoimg.com/origin/tos-cn-i-qvj2lq49k0/c1734d5347f64f5d91c01fd64d48fd6a" referrerpolicy="no-referrer"><a href="http://www.sciphi.cn/article/view/undefined"></a></div>
