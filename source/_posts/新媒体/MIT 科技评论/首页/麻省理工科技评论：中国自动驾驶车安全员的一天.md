<p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"><span style="letter-spacing: 1px;"><span style="color: rgb(89, 89, 89); --tt-darkmode-color: #595959;">麻省理工科技评论（MIT Tech Review）于7 月 27 日发布了百度自动驾驶车安全员的专题采访报道，以下为报道内容：</span></span></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"><span style="letter-spacing: 1px;"><span style="color: rgb(89, 89, 89); --tt-darkmode-color: #595959;"><br></span></span></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"><span style="letter-spacing: 1px;"><span style="color: rgb(89, 89, 89); --tt-darkmode-color: #595959;">刘杨于2021加入中国科技巨头百度，成为一名自动驾驶车的司机，他有一份奇特的工作，每天坐在副驾驶座上，监视自动驾驶车在北京的街道上来往穿梭。</span></span></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"><span style="letter-spacing: 1px;"><span style="color: rgb(89, 89, 89); --tt-darkmode-color: #595959;"><br></span></span></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"><span style="letter-spacing: 1px;"><span style="color: rgb(89, 89, 89); --tt-darkmode-color: #595959;">当刘杨开始这份工作时，他发现很难再回去开自己的车了。这位33岁的北京人说：“我本能地选择了乘客座位。或者当我开车的时候，我希望汽车能自动刹车。”</span></span></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"><span style="letter-spacing: 1px;"><span style="color: rgb(89, 89, 89); --tt-darkmode-color: #595959;"><br></span></span></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"><span style="letter-spacing: 1px;"><span style="color: rgb(89, 89, 89); --tt-darkmode-color: #595959;">如果你住在旧金山、凤凰城或深圳这样的地方，你可能对在城市里漫游的自动驾驶出租车并不陌生。但中国政府要求有人在车内，即使这些人不需要在车里控制车的行动。刘杨百度雇佣的数百名“司机”之一，也就是业内俗称的安全员。</span></span></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"><span style="letter-spacing: 1px;"><span style="color: rgb(89, 89, 89); --tt-darkmode-color: #595959;"><br></span></span></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"><span style="letter-spacing: 1px;"><span style="color: rgb(89, 89, 89); --tt-darkmode-color: #595959;">自动驾驶车的安全员是一种只存在于我们这个时代的职业，是一种不断发展的技术的结果，这种技术足够先进，可以在大多数情况下摆脱司机，但在当前尚不足以说服当局他们可以完全取消人为干预。如今，来自美国、欧洲和中国的自动驾驶公司正在竞相将该技术应用于商业。包括百度自动驾驶子公司阿波罗（Apollo）在内的大多数公司已经开始在公共道路上按需提供自动驾驶出行服务，但仍需要在一些约束条件下运行。</span></span></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"><span style="letter-spacing: 1px;"><span style="color: rgb(89, 89, 89); --tt-darkmode-color: #595959;"><br></span></span></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"><span style="letter-spacing: 1px;"><span style="color: rgb(89, 89, 89); --tt-darkmode-color: #595959;">刘杨拥有人力资源副学士学位，但他一直喜欢开车，在之前的工作中，他是老板的司机。当他听说自动驾驶技术时，他的好奇心驱使他在网上查找相关工作并进行申请。如今，刘杨一周有五天在首钢园“开”一辆自动驾驶车。首钢园占地3.3平方英里，曾是北京的一座发电厂，在2022年冬奥会被开发为冬季项目的比赛场地，同时也被重新开发成一个旅游景点。目前首钢园被指定为自动驾驶车的试验区之一，所以他的乘客通常是在那里工作的员工或周末来访的游客。</span></span></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"><span style="letter-spacing: 1px;"><span style="color: rgb(89, 89, 89); --tt-darkmode-color: #595959;"><br></span></span></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"><span style="letter-spacing: 1px;"><span style="color: rgb(89, 89, 89); --tt-darkmode-color: #595959;">但刘杨也需要考虑下一步的行动，因为他的工作可能在几年内就会被取消。在他19个月的安全员职业生涯中，他经历了几代不同的自动驾驶车车型的更新和政策的变化。2021年4月，百度在首钢园获得了将安全员放在前排乘客座位而不是司机座位的许可，刘杨随后换了个位置，告别了方向盘。今年7月21日，百度公布了第六代量产无人车——Apollo RT6，预计于2023年投入使用。</span></span></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"><span style="letter-spacing: 1px;"><span style="color: rgb(89, 89, 89); --tt-darkmode-color: #595959;"><br></span></span></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"><span style="letter-spacing: 1px;"><span style="color: rgb(89, 89, 89); --tt-darkmode-color: #595959;">以下为《麻省理工科技评论》对刘杨的采访摘要：</span></span></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"><span style="letter-spacing: 1px;"><span style="color: rgb(89, 89, 89); --tt-darkmode-color: #595959;">问：作为一名安全员，你的日常日程安排是怎样的？</span></span></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"><span style="letter-spacing: 1px;"><span style="color: rgb(89, 89, 89); --tt-darkmode-color: #595959;"><br></span></span></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"><span style="letter-spacing: 1px;"><span style="color: rgb(89, 89, 89); --tt-darkmode-color: #595959;"><br></span></span></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"><span style="letter-spacing: 1px;"><span style="color: rgb(89, 89, 89); --tt-darkmode-color: #595959;">问：既然你不开车，那么你在车上负责什么？</span></span></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"><span style="letter-spacing: 1px;"><span style="color: rgb(89, 89, 89); --tt-darkmode-color: #595959;"><br></span></span></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"><span style="letter-spacing: 1px;"><span style="color: rgb(89, 89, 89); --tt-darkmode-color: #595959;">答：当我在驾驶座上时，我观察路况并在出现问题时让车辆转向。当我在副驾驶座上时，我会更关注车辆周围的情况。由于我无法使用方向盘，如果需要人工干预时，我会迅速与基于云的操作员联系。</span></span></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"><span style="letter-spacing: 1px;"><span style="color: rgb(89, 89, 89); --tt-darkmode-color: #595959;"><br></span></span></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"><span style="letter-spacing: 1px;"><span style="color: rgb(89, 89, 89); --tt-darkmode-color: #595959;">问：你刚开始工作时感觉如何？</span></span></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"><span style="letter-spacing: 1px;"><span style="color: rgb(89, 89, 89); --tt-darkmode-color: #595959;"><br></span></span></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"><span style="letter-spacing: 1px;"><span style="color: rgb(89, 89, 89); --tt-darkmode-color: #595959;">答：一开始会感到害怕，因为我不是控制车的人。但实际上是安全的，它根据交通法规自动驾驶。</span></span></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"><span style="letter-spacing: 1px;"><span style="color: rgb(89, 89, 89); --tt-darkmode-color: #595959;">问：你每天有多少乘客？</span></span></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"><span style="letter-spacing: 1px;"><span style="color: rgb(89, 89, 89); --tt-darkmode-color: #595959;"><br></span></span></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"><span style="letter-spacing: 1px;"><span style="color: rgb(89, 89, 89); --tt-darkmode-color: #595959;">答：我每天平均有10到15份订单。每份订单通常接一到两名乘客，每次单程行驶大约持续四到五分钟。</span></span></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"><span style="letter-spacing: 1px;"><span style="color: rgb(89, 89, 89); --tt-darkmode-color: #595959;">答：没有，但是有很多人对这辆车感兴趣，想自己开车。他们会问我是否可以坐在驾驶座上，我会说不行。</span></span></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"><span style="letter-spacing: 1px;"><span style="color: rgb(89, 89, 89); --tt-darkmode-color: #595959;"><br></span></span></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"><span style="letter-spacing: 1px;"><span style="color: rgb(89, 89, 89); --tt-darkmode-color: #595959;">问：由于你每天都被限制在首钢园，你对这个地区了解得如何？</span></span></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"><span style="letter-spacing: 1px;"><span style="color: rgb(89, 89, 89); --tt-darkmode-color: #595959;"><br></span></span></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"><span style="letter-spacing: 1px;"><span style="color: rgb(89, 89, 89); --tt-darkmode-color: #595959;"><br></span></span></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"><span style="letter-spacing: 1px;"><span style="color: rgb(89, 89, 89); --tt-darkmode-color: #595959;">问：你觉得你的工作怎么样？你觉得自己的工作对自动驾驶技术的发展有影响吗？</span></span></p><p style="caret-color: rgb(0, 0, 0); --tt-darkmode-color: #595959;"><br></span></span></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"><span style="letter-spacing: 1px;"><span style="color: rgb(89, 89, 89); --tt-darkmode-color: #595959;">答：我想留在这个行业。我对它寄予厚望，我非常喜欢它。我进入这个行业是因为我很好奇，现在我仍然相信它是先进和强大的。我认为这很有未来感。如果我能有更多的时间来提高自己，我想在技术方面学习更多，比如数据技术。我的另一个想法是，凭借我长期从事这项工作的知识和经验，我可以培训新人，其他行业的人，学习自动驾驶。</span></span></p>
