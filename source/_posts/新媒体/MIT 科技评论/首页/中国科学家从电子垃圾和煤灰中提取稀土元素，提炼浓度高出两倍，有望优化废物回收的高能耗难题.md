--tt-darkmode-color:  #595959;">例如，在结构材料上，稀土可大幅度提高铝合金、镁合金、钛合金的机械性能；在石油化工上，稀土制成的分子筛催化剂可代替硅酸铝催化剂；在陶瓷上，稀土对超导陶瓷、压电陶瓷、导电陶瓷等领域贡献丰富；此外，稀土还是重要的磁性材料。</span></span></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"><span style="letter-spacing: 1px;"><span style="color: rgb(89, 89, 89); --tt-darkmode-color:  #595959;">相比铜、铝等年产数千万吨的大规模使用型金属材料，稀土金属算是一种 “小众” 金属，全球全年产量只有几十万吨。</span><strong><span style="color: rgb(171, 25, 66); --tt-darkmode-color:  #DF285B;">同时，中国又是稀土生产大国，目前的总产量占全球总产量 50% 以上。</span></strong></span></p><p style="caret-color: rgb(0, 0, 0); --tt-darkmode-color:  #595959;">一方面可缓解消耗不可再生材料造成的资源危机，另一方面又能解决固体废弃物排放造成的环境压力。</span></strong></span></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"><span style="letter-spacing: 1px;"><strong><span style="color: rgb(89, 89, 89); --tt-darkmode-color:  #595959;">该工作基于邓兵所在课题组之前开发的一种电热方法。其所在的莱斯大学纳米科学与技术中心詹姆斯·托尔（James Tour）教授团队，曾于 2020 年首次提出闪光焦耳热的方法，能把几乎任何含碳的前驱体快速转化为高质量石墨烯[3]。</span></span></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"><span style="letter-spacing: 1px;"><span style="color: rgb(89, 89, 89); --tt-darkmode-color:  #595959;"><br></span></span></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"><span style="letter-spacing: 1px;"><span style="color: rgb(89, 89, 89); --tt-darkmode-color:  #595959;"><br></span></span></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"><span style="letter-spacing: 1px;"><span style="color: rgb(89, 89, 89); --tt-darkmode-color:  #595959;"><br></span></span></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"><span style="letter-spacing: 1px;"><span style="color: rgb(89, 89, 89); --tt-darkmode-color:  #595959;">此后，鉴于稀土金属的重要性，邓兵转做这类金属的回收。期间，他获悉煤飞灰这种固体废弃物重含有较高含量的稀土金属。</span></span></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"><span style="letter-spacing: 1px;"><span style="color: rgb(89, 89, 89); --tt-darkmode-color:  #595959;"><br></span></span></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"><span style="letter-spacing: 1px;"><span style="color: rgb(89, 89, 89);
