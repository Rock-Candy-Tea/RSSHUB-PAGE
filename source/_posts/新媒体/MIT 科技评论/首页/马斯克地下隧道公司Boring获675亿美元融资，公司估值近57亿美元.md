--tt-darkmode-color:  #595959;">Loop 是 Boring 公司提出的“点对点”地下公共交通系统，而且该系统是高速、全电动、零排放的。按照 Boring公司的说法，要想解决城市的交通拥堵问题，必须将道路 3D 化，而隧道则将最大限度地缓解地表土地的使用压力，而且与现有的交通系统并不冲突。</span></span></p><p style="caret-color: rgb(0, 0, 0); text-align: justify;">（来源：The Boring 公司）</span></div><div style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"><span style="color: rgb(89, 89, 89); font-size: 16px; letter-spacing: 1px; text-align: justify;"><br></span></div><div style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"><span style="color: rgb(89, 89, 89); font-size: 16px; letter-spacing: 1px; text-align: justify;">Boring 公司对于地下隧道网络系统有长远的计划，该公司将目前计划中的城内地下隧道系统称为“Loop”， 而随着钻孔速度和挖掘技术的提高，它们还计划建设城际隧道系统，并称其为“Hyperloop”。</span></div><div><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"><span style="letter-spacing: 1px;"><span style="--tt-darkmode-color:  #A3A3A3;"><br></span><span style="color: rgb(89, 89, 89); --tt-darkmode-color:  #595959;">Boring 公司的 Loop 计划并不是“纸上谈兵”，目前，已经有合作方和在建项目。第一个 Loop 系统位于拉斯维加斯，被称为“Vegas Loop”。2021 年 10 月 20 日，该项目被正式批准，计划在拉斯维加斯建设一个连接 51 个车站、长达 29 英里的隧道网络环路系统。并且，最终预计实现每小时运送 57000 名乘客。</span></span></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"><span style="letter-spacing: 1px;"><span style="--tt-darkmode-color:  #A3A3A3;"><br></span><span style="color: rgb(89, 89, 89); --tt-darkmode-color:  #595959;">目前，该隧道系统已经在拉斯维加斯开始了部分运营：位于拉斯维加斯会议中心和其附近的三个站点之间长达 1.7 英里的隧道已经开通，乘客只要在其中任意一站搭乘一辆特斯拉 SUV，就可以往返于 Loop 隧道系统。</span></span></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"><span style="letter-spacing: 1px;"><span style="--tt-darkmode-color:  #A3A3A3;"><br></span><span style="color: rgb(89, 89, 89); --tt-darkmode-color:  #595959;">此外，Boring 公司还在佛罗里达州的劳德代尔堡（Lauderdale）进行另一个 Loop 项目的筹划，并正在与德克萨斯州圣安东尼奥市的相关部门进行协商，以期开展新的项目。此前，Boring 公司公开宣布过与芝加哥、华盛顿特区和洛杉矶的不同合作项目，但到目前为止，这些项目还尚未启动。</span></span></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"><span style="letter-spacing: 1px;"><span style="--tt-darkmode-color:  #A3A3A3;"><br></span><span style="color: rgb(89, 89, 89); --tt-darkmode-color:  #595959;">目前，Boring 公司与佛罗里达州的劳德代尔堡的项目得到了美国政府的支持，该项目的计划是将市中心和附近的海滩通过隧道连接起来。但是，也有批评者提出了质疑，尤其对于建设成本高昂（每英里 10 亿美元）的地下隧道来说，如果隧道中运行汽车而非火车或地铁，是否能达到真正对隧道的有效利用？</span></span></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"><span style="letter-spacing: 1px;"><span style="--tt-darkmode-color:  #A3A3A3;"><br></span><span style="color: rgb(89, 89, 89); --tt-darkmode-color:  #595959;">对此，Boring 公司表示，该公司正致力于成本更低的隧道开发技术，而且，由于 Loop 计划中车辆不需要在每站停留而减速、加速，也将减少能源的浪费。</span></span></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"><span style="letter-spacing: 1px;"><span style="--tt-darkmode-color:  #A3A3A3;"><br></span><span style="color: rgb(89, 89, 89); --tt-darkmode-color:  #595959;">尽管 Boring 公司 Loop 地下隧道的网络系统相较于现有交通系统进行了重要变革，但这一计划和马斯克对 Boring 公司成立时提出的宏大愿景还有一定差距。</span></span></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"><span style="letter-spacing: 1px;"><span style="--tt-darkmode-color:  #A3A3A3;"><br></span><span style="color: rgb(89, 89, 89); --tt-darkmode-color:  #595959;">此前，马斯克对于 Boring 公司所将建立的交通网络的畅想是，乘客将搭乘专用的自动驾驶汽车，以高达 120 英里/小时的速度，远超现有地面交通的速度，往返于城市隧道之间。</span></span></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"><span style="letter-spacing: 1px;"><span style="--tt-darkmode-color:  #A3A3A3;"><br></span><span style="color: rgb(89, 89, 89); --tt-darkmode-color:  #595959;">但截至目前，Loop项目中的汽车在地下隧道只能以 35 英里/小时的速度行驶，这与特斯拉在地面的行驶速度相同。并且需要人类驾驶员的操控。</span></span></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"><span style="letter-spacing: 1px;"><span style="--tt-darkmode-color:  #A3A3A3;"><br></span><span style="color: rgb(89, 89, 89); --tt-darkmode-color:  #595959;">对于 Loop 和 Hyperloop 隧道系统的开发，Boring 公司也在进行挖掘技术方面的开发。该公司提出的下一代隧道掘进机名为“Prufrock”。据该公司信息，该项目可将传统需要几年建设的大型基础设施项目的工期压缩至几周内便可完成，而且成本预计更低。</span></span></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"><span style="letter-spacing: 1px;"><span style="color: rgb(89, 89, 89); --tt-darkmode-color:  #595959;"><br>Prufrock 隧道掘进机的最新设计版本是 Prufrock-2，设计目标是以每周 1 英里的速度开采，这意味着，可以在一个月内挖掘一条长达约 4 英里、和拉斯维加斯大道长度相同的隧道。而其下一代 Prufrock-3 的目标挖掘速度更快，将达到人类步行速度的 1/10，即 7 英里/天。</span></span></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"><span style="letter-spacing: 1px;"><span style="--tt-darkmode-color:  #A3A3A3;"><br></span><span style="color: rgb(89, 89, 89); --tt-darkmode-color:  #595959;">对于未来的运营，Boring 公司表示，计划最终将过渡到无需融资、完全自主运营的状态，并将逐步降低运营成本。</span></span></p><p style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0);"><span style="letter-spacing: 1px;"><span style="--tt-darkmode-color:  #A3A3A3;"><br></span><span style="color: rgb(89, 89, 89); --tt-darkmode-color:  #595959;">Boring 公司的投资人之一红杉资本曾在博客中表示，Boring 公司目前正处于“拐点”阶段。其在博客中写道：“如果 Boring 公司的系统最终可以实现快速可重用，那么，该公司在未来几年内实现完全自主运营的可能性，将大约是现在的 10 倍。”</span></span></p><p style="caret-color: rgb(0, 0, 0);
