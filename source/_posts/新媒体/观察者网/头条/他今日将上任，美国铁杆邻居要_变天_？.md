</p>
<p class="MsoNormal">
<span style="font-family:宋体;">“马孔多的雨要停一会了吗？”“希望如此。”</span><span></span> 
</p>
<p class="MsoNormal">
<span style="font-family:宋体;">这是</span><span>2022</span><span style="font-family:宋体;">年</span><span>6</span><span style="font-family:宋体;">月</span><span>22</span><span style="font-family:宋体;">日，一段中国网友的对话。几天前的</span><span>6</span><span style="font-family:宋体;">月</span><span>19</span><span style="font-family:宋体;">日夜，在遥远的波哥大，人们走上街头拥抱欢呼庆祝一个重大时刻：哥伦比亚迎来了历史上首位左翼总统——</span><span>62</span><span style="font-family:宋体;">岁的古斯塔沃·佩特罗。</span> 
</p>
<p class="MsoNormal">
<span style="font-family:宋体;">《百年孤独》中马孔多的雨隐晦地表达了哥伦比亚人民饱受的剥削、痛苦和压榨。两个月前的选举后，中国网友也用自己的方式，送上了意味深长的关注与祝福。</span> 
</p>
<p class="MsoNormal" align="center">
<span style="font-family:宋体;"><img src="https://i.guancha.cn/news/external/2022/08/06/20220806164100574.png" title="点击查看大图" class="bigimg" referrerpolicy="no-referrer"><br>
</span> 
</p>
<p class="MsoNormal" align="center">
<span style="font-family:宋体;"><span style="color:#666666;">截图自知乎</span></span> 
</p>
<p class="MsoNormal">
<span style="font-family:宋体;">在两个月前，</span><span style="font-family:宋体;">哥伦比亚的这次史无前例的左右翼对决中，除了首轮竞选当日内政部出动</span>30<span style="font-family:宋体;">万军警确保安全外，据美国左翼政党报纸“争取社会主义和解放党报”称，极力寻求变革的哥民众，有的不惜划船前往被刻意设得很远的投票点；而佩特罗的竞争对手则是一位熟练使用社交媒体的“哥伦比亚特朗普”。</span> 
</p>
<p class="MsoNormal">
<span style="font-family:宋体;">备受关注的古斯塔沃·佩特罗，除了在政治上信奉社会平等、改革、权力和财富共享之外，他曾经的左翼游击队员的身份也吸引着众人的目光。</span><span></span> 
</p>
<p class="MsoNormal">
<span style="font-family:宋体;">作为美国多年“最好朋友”之一，哥伦比亚的“变天”也被视为美国外交的一次挫败。</span><span></span> 
</p>
<p class="MsoNormal">
<span style="font-family:宋体;"><strong>当地时间8月7日下午，古斯塔沃·佩特罗即将宣誓就职哥伦比亚总统，</strong>这位左翼领袖的到来会让“马孔多的雨”停一会吗？<br>
</span> 
</p>
<p class="MsoNormal">
<b><span style="font-family:宋体;">古斯塔沃·佩特罗是谁</span><span></span></b> 
</p>
<p class="MsoNormal">
<span style="font-family:宋体;">经阅读佩特罗的生平公开资料，观察者网查阅到古斯塔沃·佩特罗出生在盛产棉花和牛的加勒比大草原。他的父亲是学校教师，母亲是民族主义政党的成员，家庭条件不算富裕，但是足够生活。</span><span></span> 
</p>
<p class="MsoNormal">
<span style="font-family:宋体;">佩特罗是三个儿子中的老大，在人们的记忆中他性格内向、少言寡语，喜欢穿深色的衣服，酷爱读书。</span><span></span> 
</p>
<p class="MsoNormal">
<span style="font-family:宋体;">佩特罗在其自传中这样描述自己的性格：严肃、内向、不轻易信任他人。他称自己虽然生活在哥伦比亚沿海城市，但是没有沿海人的热情与奔放，更像是“一位安第斯高原人”。他把自己这种个性上的矛盾归结于他的政治倾向。</span><span></span> 
</p>
<p class="MsoNormal">
<span style="font-family:宋体;">在他年幼时期，他随父母搬到了哥伦比亚首都波哥大以北的小镇齐帕奎拉。在那里，佩特罗在一所由教会开办的公立学校度过了自己的学生时代。著名作家加布里埃尔</span><span>·</span><span style="font-family:宋体;">加西亚</span><span>·</span><span style="font-family:宋体;">马尔克斯也曾就读这所学校，也是对他影响最大的人之一。</span><span></span> 
</p>
<p class="MsoNormal">
<span style="font-family:宋体;">据</span><span>BBC</span><span style="font-family:宋体;">采访到的佩特罗的老朋友、同样当过游击队员的何塞</span><span>·</span><span style="font-family:宋体;">奎斯塔（</span><span>José Cuesta</span><span style="font-family:宋体;">）说，年少时的佩特罗好奇心很重，经常去参加工会会议，并在</span><span>17</span><span style="font-family:宋体;">岁时加入了一个以城市、民族主义和社会民主为宗旨的游击队：</span><span>4</span><span style="font-family:宋体;">月</span><span>19</span><span style="font-family:宋体;">日运动（</span><span>M19</span><span style="font-family:宋体;">）。在</span><span>M19</span><span style="font-family:宋体;">游击队中，佩特罗给自己取名奥雷利亚诺，以纪念马尔克斯著名小说《百年孤独》中的上校。</span><span></span> 
</p>
<p class="MsoNormal">
<span style="font-family:宋体;">在右翼政府统治下的哥伦比亚社会分裂，国内环境动荡。史料显示，在</span><span>M19</span><span style="font-family:宋体;">时期的哥伦比亚右翼政府任由准军事组织走私毒品、杀害平民，而此时期诞生的以政治斗争为主的、寻求和平的爱国联盟（</span><span>UP</span><span style="font-family:宋体;">）也因受到民众支持，遭到右翼组织的暗杀。</span><span></span> 
</p>
<p class="MsoNormal">
<span style="font-family:宋体;">动荡的和平之路走得非常艰难，右翼政府虽然一直尝试与哥伦比亚革命武装力量等左翼组织和谈，但实际做出的行为却是为了右翼利益。在右翼总统阿尔瓦罗·乌里韦担任哥伦比亚总统时期（</span><span>2002-2010</span><span style="font-family:宋体;">年），乌里韦被指控为了追求歼灭哥武游击队人员的数量，而杀良冒功，屠杀了超过</span><span>6400</span><span style="font-family:宋体;">名无辜的平民。</span> 
</p>
<p class="MsoNormal" align="center">
<span style="font-family:宋体;"><img src="https://i.guancha.cn/news/mainland/2022/08/06/20220806163856521.jpg" title="点击查看大图" class="bigimg" width="530" referrerpolicy="no-referrer"><br>
</span> 
</p>
<p class="MsoNormal" align="center">
<span style="font-family:宋体;"><span style="color:#666666;">M19攻占哥伦比亚司法大厦。图自BBC</span><br>
</span> 
</p>
<p class="MsoNormal">
<span style="font-family:宋体;">佩特罗在个人自传《</span><span>Una vida, muchas vidas</span><span style="font-family:宋体;">》（一种生活，多个生命）中回忆道：</span><span>M19</span><span style="font-family:宋体;">的男男女女给我留下的第一印象就是坚韧。人们对于斗争的热情让我惊讶，在这里，我无法再使用法律赋予我的名字，我是奥雷里亚诺，与他们一起参与战斗。</span> 
</p>
<p class="MsoNormal">
<span style="font-family:宋体;">成年后的佩特罗进入哥伦比亚外部大学主修经济，并获得奖学金。自此，佩特罗的“斗争生活”就驻扎在首都波哥大。</span><span>2011</span><span style="font-family:宋体;">年，佩特罗担任波哥大市长，据哥媒称，这是哥伦比亚第二重要的职位，是许多希望竞选总统的政治人物必争之位。</span><span></span> 
</p>
<p class="MsoNormal">
<span style="font-family:宋体;">据历史资料显示，</span><span>1985</span><span style="font-family:宋体;">年，</span><span>M19</span><span style="font-family:宋体;">攻占哥伦比亚司法大厦，期间有</span><span>101</span><span style="font-family:宋体;">人死亡。该行动目的是为了抗议1970 <span style="font-family:宋体;">年选举中的欺诈指控。后</span><span>M19</span><span style="font-family:宋体;">组织演变为一个政党，并协助修改了宪法。</span></span> 
</p>
<p class="MsoNormal">
<span>M19</span><span style="font-family:宋体;">游击队组织以反帝、反殖民、建立民主政权为口号，不同于</span><span>“</span><span style="font-family:宋体;">哥伦比亚革命武装力量</span><span>”</span><span style="font-family:宋体;">等立足农村的反政府武装，其成员多有着公开身份，在城市秘密进行</span><span>“</span><span style="font-family:宋体;">武装斗争</span><span>”</span><span style="font-family:宋体;">，曾经还制造过攻占多米尼加大使馆等事件。</span><span></span> 
</p>
<p class="MsoNormal">
<span style="font-family:宋体;">与</span><span>M19</span><span style="font-family:宋体;">有关的佩特罗被警方逮捕后被关押在军营，他说在那里他为掩护战友身份而遭酷刑，后被人们称为“游击队中的英雄”。自传里写道，在游击队中，佩特罗成为一名领导者，负责游击队的社区宣传、新兵招募和组织动员工作，并在这期间大量阅读马克思主义著作，与得到美国支持的右翼政府军斗争周旋。</span><span></span> 
</p>
<p class="MsoNormal">
<span style="font-family:宋体;">在狱中，佩特罗坚决否认自己与袭击司法大厦有关，也否认</span><span>“</span><span style="font-family:宋体;">杀戮</span><span>”“</span><span style="font-family:宋体;">贩毒</span><span>”</span><span style="font-family:宋体;">等指控，称自己是</span><span>“</span><span style="font-family:宋体;">政治领导者而非杀手</span><span>”</span><span style="font-family:宋体;">。最终，当局只能以</span><span>“</span><span style="font-family:宋体;">非法持有枪械</span><span>”</span><span style="font-family:宋体;">罪名判处他</span><span>18</span><span style="font-family:宋体;">个月徒刑。</span> 
</p>
<p class="MsoNormal" align="center">
<span style="font-family:宋体;"><img src="https://i.guancha.cn/news/origin/2022/08/06/20220806163932388.jpg" title="点击查看大图" class="bigimg" width="530" referrerpolicy="no-referrer"> </span> 
</p>
<p class="MsoNormal" align="center">
<span style="font-family:宋体;"><span style="color:#666666;">佩特罗在其自传中反思M19运动给国家和人民带来了深重的灾难</span><br>
</span> 
</p>
<p class="MsoNormal">
<span>1987</span><span style="font-family:宋体;">年，佩特罗被释放。佩特罗自传中写道，在狱中度过的时间里他对于</span><span>“</span><span style="font-family:宋体;">斗争</span><span>”</span><span style="font-family:宋体;">的意识视角发生变化，他认识到</span><span>“</span><span style="font-family:宋体;">武装革命并不是赢得民众支持的最佳策略</span><span>”</span><span style="font-family:宋体;">，下决心寻找</span><span>“</span><span style="font-family:宋体;">另一条斗争道路</span><span>”</span><span style="font-family:宋体;">。</span><span></span> 
</p>
<p class="MsoNormal">
<span style="font-family:宋体;">在查阅哥伦比亚民主进程的资料后，观察者网发现到了</span><span>1990</span><span style="font-family:宋体;">年，当时哥伦比亚国内盛行着一股强烈的寻求和平的风气，除了民族解放军外，所有的游击队都表示愿意为冲突寻求政治解决办法。</span><span>M19</span><span style="font-family:宋体;">游击队同意停止敌对行动，后来开始解散。</span><span>M19 </span><span style="font-family:宋体;">的遣散被认为是哥伦比亚长期冲突历上的历史性成功。彼时的佩特罗已经当选为哥伦比亚众议院的议员。</span><span></span> 
</p>
<p class="MsoNormal">
<span style="font-family:宋体;">但是哥伦比亚革命武装力量和民族解放军对平民的行动破坏了这一和平进程，哥国内求和的步骤被打破，哥伦比亚革命武装力量继续冲突升级活动。</span><span></span> 
</p>
<p class="MsoNormal">
<span style="font-family:宋体;">在佩特罗</span><span>34</span><span style="font-family:宋体;">那年，他收到了死亡威胁，那一年他第一次离开了祖国。</span><span></span> 
</p>
<p class="MsoNormal">
<span style="font-family:宋体;">哥伦比亚政府为了保护这位</span><span>“</span><span style="font-family:宋体;">金盆洗手</span><span>”</span><span style="font-family:宋体;">的前游击队队员，给他在驻比利时大使馆安排了一个低级别的职位。</span><span></span> 
</p>
<p class="MsoNormal">
<span style="font-family:宋体;">佩特罗在自传中回忆欧洲的四年对塑造他的政治理念极为重要：他对发达国家、社会民主党派和知识型社会都有了深入的了解，并希望把哥伦比亚也建成这样的社会。</span><span></span> 
</p>
<p class="MsoNormal">
<span style="font-family:宋体;">在欧洲，他还研究了环境问题。这是继社会正义之后他最为关心的另一个问题。他在自传中透露，由于担心气候带来的影响，他已经</span><span>30</span><span style="font-family:宋体;">年没有开过车。</span><span></span> 
</p>
<p class="MsoNormal">
<span style="font-family:宋体;">佩特罗并不掩饰对自己的高度评价。他在书中这样自我描述：</span><span>“</span><span style="font-family:宋体;">人们像风一样把我从一个地方带到另一个地方，他们使我成为巨人。</span><span>”</span> 
</p>
<p class="MsoNormal">
<span style="font-family:宋体;">这就是他被称为虚荣、向上爬、自命不凡的原因，因为</span><span>“</span><span style="font-family:宋体;">他相信自己是人民的救世主</span><span>”</span><span style="font-family:宋体;">。英</span><span>BBC </span><span style="font-family:宋体;">在</span><span>6</span><span style="font-family:宋体;">月</span><span>25</span><span style="font-family:宋体;">日刊发的一篇人物点评中如是写道。</span><span></span> 
</p>
<p class="MsoNormal">
<span style="font-family:宋体;">左翼人士批评他在知识上傲慢、管理上专制、概念上固执和话语上极端。右翼则谴责他对国家经济的设计愿景、他与查韦斯等人物（查韦斯，前任委内瑞拉总统，任内推行反美主义）的亲密关系以及他参加过游击队的历史。</span> 
</p>
