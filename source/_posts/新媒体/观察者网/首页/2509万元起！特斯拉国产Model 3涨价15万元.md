<p>
（观察者网 讯）据特斯拉中国官网显示，国产Model 3后轮驱动版售价上调1.5万元至25.09万元。值得注意的是，近两年Model 3在中国市场上的售价极少上涨。此前，Model 3的售价从40多万一路降到涨价前的23.59万。
</p>

<p align="center">
<span style="color:#666666;">现在特斯拉中国官网的 Model 3续航和性能截图</span>

<p align="left">
从特斯拉官网新旧两款车型的图片对比来看，首先，特斯拉精简了新款 Model 3 的命名方式，后轮驱动：由“标准续航升级版”改为“Model 3”。双电机全轮驱动：由“Performance 高性能版”改为“Model 3 高性能版”。其次，在续航方面，旧款 Model 3 标注的续航为 468 公里，新款 Model 3 标注的续航为 556 公里。在百公里加速方面，新款 Model 3 则有所退步，由原来的 5.6 秒，降低到了现在的 6.1 秒。新款 Model 3 的最高时速则没有改变，依旧是 225 公里每小时。
</p>
