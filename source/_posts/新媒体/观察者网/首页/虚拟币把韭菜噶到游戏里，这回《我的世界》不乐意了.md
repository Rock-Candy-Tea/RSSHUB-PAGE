<p>本文原创于微信公众号：差评  作者：小发</p><p><img src="https://i.guancha.cn/bbs/2022/07/31/20220731062240642.png?imageView2/2/w/500/format/png" alt="500" referrerpolicy="no-referrer">有没有差友和小发一样，已经越来越<strong>受不了 NFT</strong>（ 非同质化代币 ） 了。</p><p align="center"><img src="https://i.guancha.cn/bbs/2022/07/31/20220731062240267.jpg?imageView2/2/w/500/format/jpg" alt="500" referrerpolicy="no-referrer"></p><p>作为当下区块链技术中的重要“ 虚拟货币 ”，NFT 可以说入侵到我们生活里的方方面面。</p><p>平常社交平台看到满屏的 NFT 新闻、讨论也就算了，大不了我不看，可作为一个游戏佬却发现<strong>连游戏都开始被 NFT 入侵那就真难受了。</strong></p><p>前段时间就有不少公司和团队把主意打到了《 我的世界（ Minecraft ）》身上，利用游戏的世界文件、皮肤包等等进行 NFT 化。</p><p>不过好在这一趋势被《 我的世界 》开发团队 Mojang Studios 给制止了。</p><p>几天前，Mojang Studios 发布官方公告宣布<strong>禁止 Minecraft 客户端和服务器应用程序集成区块链技术，也不得将游戏内容包括世界、皮肤、角色物品或其他 MOD 用于创建任何相关的 NFT。</strong></p><p align="center"><img src="https://i.guancha.cn/bbs/2022/07/31/20220731062240573.png?imageView2/2/w/500/format/png" alt="500" referrerpolicy="no-referrer"></p><p>根据官方公告来看，Minecraft 团队认为如今的 NFT 项目是为了创造数字所有权，会<strong>人为地制造稀缺性和排他性，</strong>这和 Minecraft 本身包容的价值观完全相反。</p><p>另一方面，如今 NFT 项目鼓励暴利，参与其中的人员都有着投资或者投机的心态，这样<strong>会把玩家的注意力从游戏中转移走</strong>，这和 Minecraft 玩家长期的快乐玩法相违背。</p><p>还有一点就是现在的 NFT 团队、公司不少都是草台班子，<strong>动不动就赚钱跑路或者没钱暴雷，</strong>最终受伤害最深的就是每个普通玩家。</p><p align="center"><img src="https://i.guancha.cn/bbs/2022/07/31/20220731062240522.png?imageView2/2/w/500/format/png" alt="500" referrerpolicy="no-referrer"></p><p>这种用 NFT 圈钱跑路的事情甚至已经在 Minecraft 里出现过了。</p><p>在今年年初，有一个叫 Blockverse 的团队在网络上放出话来。说是要打造一个 MC 的 NFT 服务器，玩家可以到这个服务器里边玩边“ 挖矿 ”赚钱。</p><p align="center"><img src="https://i.guancha.cn/bbs/2022/07/31/20220731062240380.png?imageView2/2/w/500/format/png" alt="500" referrerpolicy="no-referrer"></p><p>当然，世界上没有纯“ 白嫖 ”的好事，进这个服务器得<strong>先买他们的 NFT（ 标价 120 美元 ） 产品</strong>当门票。</p><p><img src="https://i.guancha.cn/bbs/2022/07/31/20220731062240712.png?imageView2/2/w/500/format/png" alt="500" referrerpolicy="no-referrer">很多被忽悠了的玩家觉得有搞头，纷纷用上了双十一抢购的手速，好不容易才从 1 万个门票里抢到了一张。</p><p>可正当这些玩家沾沾自喜，准备开始进服“ 挖矿 ”的时候，却发现<strong>服务器压根就不开放</strong>，转过头去联系 Blockverse 团队却发现人家<strong>直接人间蒸发</strong>了。</p><p align="center"><img src="https://i.guancha.cn/bbs/2022/07/31/20220731062240410.png?imageView2/2/w/500/format/png" alt="500" referrerpolicy="no-referrer"></p><p>后来受骗玩家们联合起来一起满世界找 Blockverse 团队人员，好不容易联系到其中一个成员，但他们已经<strong>开始耍无赖</strong>，一个劲说不是不开服务器，是服务器出现了问题。让他们退款，他们又说这 120 万美元收入是他们应得的。这事反正拖到现在也没个后文，被骗玩家们还是只能认栽。</p><p>这次在 Mojang Studios 宣布《 我的世界 》禁止区块链后，NFT 团队们坐不住了，其中有个叫 NFT Worlds 的，本来是准备在《 我的世界 》里大搞特搞，结果一道禁令下来直接破防，发挥了一波 NFT 小作坊的传统艺能。</p><p style="text-align:center;"><img src="https://i.guancha.cn/bbs/2022/07/31/20220731062240991.gif?imageView2/2/w/500/format/jpg" alt="500" referrerpolicy="no-referrer"></p><p>后面眼看顶不住了就在推特上一路追着狂喷，一边指名道姓地说 Mojang Studios 团队让人失望，一边又说这个禁令垃圾，一边又吵吵 NFT 是白莲花根本没有伤害任何《 我的世界 》玩家。</p><p>看看这一波输出，是不是有点“ 退、退、退 ”的意思了。</p><p align="center">大意▼</p><p style="text-align:center;"><img src="https://i.guancha.cn/bbs/2022/07/31/20220731062240390.jpg?imageView2/2/w/500/format/jpg" alt="500" referrerpolicy="no-referrer"></p><p>过了几天，NFT Worlds 好像意识到无论他们怎么说，Mojang Studios 压根都不理他们。</p><p><img src="https://i.guancha.cn/bbs/2022/07/31/20220731062240326.png?imageView2/2/w/500/format/png" alt="500" referrerpolicy="no-referrer">于是他们就又准备搞个大的，说要抛弃《 我的世界 》，联合多个独立制作团队，一起做款<strong>全新的游戏</strong>取代《 我的世界 》。</p><p align="center"><img src="https://i.guancha.cn/bbs/2022/07/31/20220731062240313.png?imageView2/2/w/500/format/png" alt="500" referrerpolicy="no-referrer"></p><p>不过要我说，NFT 天天有，《 我的世界 》可是独此一家，要真能做出能取代《 我的世界 》的游戏，那也犯不着盯着 NFT 那仨瓜俩枣了不是？</p><p>而且《 我的世界 》并<strong>没有“ 永封 ” NFT</strong>，他们在公告提到会持续关注区块链技术的发展情况。所以在将来的某一天，《 我的世界 》用上了 NFT 也是完全可能的事，不过那一定不是今天。</p><p>就像其他领域一样，在<strong>游戏界关于 NFT 的争论也从没停过。</strong></p><p>去年下半年，两大游戏平台 Steam 和 Epic 就曾针对 NFT 分别发出相反的公告。Steam 出于 NFT 发展不成熟考虑，禁止掉了所有与 NFT 有关的游戏、软件。</p><p>Epic 方就比较有意思了，本来Epic首席执行官蒂姆·斯威尼 ( Tim Sweeney ) 也说过“ NFT 领域目前充满骗局”，但当 Steam 明文禁止 NFT 之后，蒂姆立马改口说只要 NFT 游戏遵守法律、披露条款，进行适当的年龄分级，他们来者不拒，果然<strong>敌人的敌人就是朋友啊。</strong></p><p align="center"><img src="https://i.guancha.cn/bbs/2022/07/31/20220731062241212.png?imageView2/2/w/500/format/png" alt="500" referrerpolicy="no-referrer"></p><p>而像育碧很早就对 NFT 感兴趣，而且还在自家游戏《 幽灵行动·断点 》里加入 NFT，结果被玩家一顿臭骂，但育碧也是硬气地表示依旧无所畏惧，立志要走出条新路子来。</p><p>游戏开发商面对 NFT 的态度也是分化得厉害。</p><p><img src="https://i.guancha.cn/bbs/2022/07/31/20220731062241653.png?imageView2/2/w/500/format/png" alt="500" referrerpolicy="no-referrer">激进点的像《 分手厨房 》的开发商 Ghost Town Games，直接宣布<strong>永远不会在游戏里加入 NFT 内容。</strong></p><p align="center"><img src="https://i.guancha.cn/bbs/2022/07/31/20220731062241348.png?imageView2/2/w/500/format/png" alt="500" referrerpolicy="no-referrer"></p><p>而像《 百战天虫 》游戏开发商 Team17，<strong>一度已经进军 NFT 产业</strong>，推出了基于 Worms 视频游戏系列的 NFT，结果转头在游戏社区里被玩家喷了个狗血淋头，一天后就被迫取消计划。</p><p align="center"><img src="https://i.guancha.cn/bbs/2022/07/31/20220731062241874.png?imageView2/2/w/500/format/png" alt="500" referrerpolicy="no-referrer"></p><p>更离谱的是《 潜行者 2 》的开发商 GSC Game World，在发布 NFT 计划一小时后就主动宣布取消计划。</p><p>在小发看来，技术谈不上善恶，但是“ 有用 ”和“ 没用 ”的区别还是有的，眼下的 NFT 技术又不能让游戏变得更好，反而会带来不少风险，火急火燎地吃热豆腐烫了满嘴疮何苦来哉？</p><p><strong>资料内容来源：</strong></p><p>Twitter：@Minecraft @nftworldsNFT @Team17 @Overcookedgame</p><p>3DMGAME：《我的世界》NFT骗局
