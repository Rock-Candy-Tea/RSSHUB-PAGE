<div class="m-b-10"><p><strong>财联社8月25日讯（编辑 刘蕊）</strong>美东时间周三，英伟达公布了令人失望的Q2财报和Q3指引。</p>
<p>从财报来看，公司游戏部门收入已经大幅下滑，并且悲观预计公司营收第三季度还将同比下降17%，导致其股价在盘后交易中下跌约5%。</p>
<p>越来越多分析师们担心，此前支撑英伟达业绩高速增长的游戏和数据中心业务未来将持续降温，英伟达的业绩前景可能堪忧。</p>
<p><strong>英伟达第二季度财报显示：</strong></p>
<ul>
<li><p>二季度公司收入67亿美元，同比增长3%，环比下降19%，大体符合分析师预期，但大大低于英伟达5月份预测的81亿美元。</p>
</li>
<li><p>当季GAAP下每股摊薄收益为0.26美元，比去年同期下降72%，比上一季度下降59%；</p>
</li>
<li><p>调整后每股收益为0.51美元，较上年同期下降51%，较上一季度下降63%，基本符合市场预期。</p>
</li>
</ul>
<p><strong>分部门来看，</strong></p>
<ul>
<li><p><strong>游戏部门的收入为20.4亿美元，同比下降33%，环比降幅达44%；</strong></p>
</li>
<li><p>专业可视化业务报告期内收入为4.96亿美元，同比下降4%，环比降幅达20%；</p>
</li>
<li><p>汽车业务贡献了2.2亿美元的收入，同比增幅为45%，环比大增59%。</p>
</li>
<li><p><strong>数据中心收入维持在38.1亿美元，同比增长61%，环比增速为1%。</strong></p>
</li>
</ul>
<p><strong>英伟达在财报指引中预计，</strong></p>
<ul>
<li><p><strong>第三季度营收为59亿美元，同比下降17%</strong>，远远不及分析师预期的69.2亿美元；</p>
</li>
<li><p>预计三季度调整后毛利润率65.0%±0.5个百分点。</p>
</li>
</ul>
<p><strong>数据中心业务面临降温风险</strong></p>
<p>从上述部门数据来看，<strong>数据中心业务成为支撑英伟达业绩增长的最主要支柱。</strong></p>
<p>然而，在经济衰退风险下，越来越多分析师担忧，未来数据中心市场增长也可能会放缓。</p>
<p>英伟达首席执行官黄仁勋在财报电话会议上表示，<strong>中国云服务提供商第二季度的基础设施投资已经出现大幅放缓，</strong>不过这一逆风因素被美国云市场的强劲增长所抵消。</p>
<p>Summit Insights Group分析师kingai Chan表示：</p>
<blockquote>
<p>“我们认为英伟达可能会看到加密挖矿和数据中心终端市场的进一步下滑。”</p>
</blockquote>
<p>黄仁勋还表示，该公司的数据中心业务还面临着供应链方面的挑战，</p>
<blockquote>
<p>“我们看到了云计算中GPU的大量需求…但本季度我们遇到了大量供应链方面的挑战。”</p>
</blockquote>
<p><strong>游戏业务已经明显疲软</strong></p>
<p>而英伟达的另一项业绩支柱——游戏部门业务，在第二季度出现明显疲软态势。</p>
<p>随着全球多国通胀率飙升至数十年来新高，很多消费者不得不减少了对游戏设备等非必需品的购买，游戏行业因而显示出疲软迹象。</p>
<p>此外，英伟达的GPU芯片被广泛用于加密货币挖矿，然而近期随着加密货币市场的崩溃，英伟达的销售也受到了打击。</p>
<p>自新冠疫情爆发以来，由于居家办公、游戏以及挖矿需求升温，英伟达的芯片销售额也水涨船高，并推升英伟达股价一度飙升至346.35美元的历史新高。然而在当下随着芯片需求逐步降温，英伟达股价年内已经暴跌41%至172.22美元，较此前最高价腰斩。</p>
