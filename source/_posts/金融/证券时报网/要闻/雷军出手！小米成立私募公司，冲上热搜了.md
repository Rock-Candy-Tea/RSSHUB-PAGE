<p style="text-align: left;">中国基金报记者 林雪</p><p style="text-align: left;">年轻人的第一只私募基金要来了？</p><p style="text-align: left;">5月初，“小米私募股权基金管理有限公司”悄然成立，其由小米科技100%控股，背后实控人是雷军，昨天引发网友热议，冲上了微博热搜。</p><p style="text-align: left;">小伙伴们很好奇，未来新成立的小米私募股权基金，将会做哪些业务？当然，其还需要在基金业协会完成私募管理人登记备案，才能对外募资、从事私募股权、创业投资管理。</p><p style="text-align: left;">基金君发现，其实小米掌门人雷军早就开始做私募了，包括顺为资本、湖北小米长江产业投资基金等，投资了不少互联网、科技领域的创业企业，也投了芯片、半导体产业。</p><p style="text-align:center"><img data-type="png" _width="546.51px" class src="https://news.stcn.com/news/202105/W020210509757354738120.png" alt="图片" data-fail="0" draggable="false" data-ratio="0.5923217550274223" data-w="547" oldsrc="W020210509757354738120.png" referrerpolicy="no-referrer"></p><p style="text-align: left;"><strong>小米成立私募股权公司</strong></p><p style="text-align: left;">天眼查信息显示，一家名为“小米私募股权基金管理有限公司”的企业于2021年5月7日成立，注册资本为1亿元人民币，注册地在北京经济技术开发区，公司的法定代表人是林世伟。</p><p style="text-align:center"><img data-type="png" _width="553.74px" class src="https://news.stcn.com/news/202105/W020210509757355207116.png" alt="图片" data-fail="0" draggable="false" data-ratio="0.44223826714801445" data-w="554" oldsrc="W020210509757355207116.png" referrerpolicy="no-referrer"></p><p style="text-align: left;">从股东来看，小米私募股权基金管理有限公司由小米科技有限责任公司全资控股，穿透股权结构，其疑似实际控制人是雷军。</p><p style="text-align: left;">从其经营范围看，这家公司未来将从事私募股权投资基金管理、创业投资基金管理服务（PS：基金君解释一下，私募股权、创业投资基金主要投资于一级市场的初创企业，投资非上市公司的股权，这类投资风险高、周期长，但是一旦成功后收益也会比较丰厚）。</p><p style="text-align: left;">当然，未来小米私募股权基金还需要在中国证券投资基金业协会完成私募基金管理人备案登记以后，才能对外募集资金，从事私募股权、创业投资管理活动等。</p><p style="text-align:center"><img data-type="png" _width="553.74px" class src="https://news.stcn.com/news/202105/W020210509757355529559.png" alt="图片" data-fail="0" draggable="false" data-ratio="0.628158844765343" data-w="554" oldsrc="W020210509757355529559.png" referrerpolicy="no-referrer"></p><p style="text-align: left;">从主要成员看，小米私募股权基金的执行董事、经理是林世伟，他现任小米集团副总裁、首席财务官（CFO）。公开资料显示，林世伟毕业于牛津大学，获得了工程学硕士学位，他具有超过20年的投行业务经验，此前曾担任瑞信亚太区投资银行资本市场部董事总经理及科技、媒体与电信业务主管；他也担任过摩根士丹利董事总经理，负责亚太区科技、媒体与电信行业的资本市场业务。</p><p style="text-align: left;">小米私募股权基金的监事是刘德，现任小米科技联合创始人兼副总裁、智米科技执行董事。资料显示，刘德是美国艺术设计中心学院工业设计硕士，北京理工大学机械工程硕士。曾经担任北京科技大学工业设计系主任、北京新锋锐设计公司创始人兼首席设计师、北京凡朴工业设计公司明斯克航母项目负责人。</p><p style="text-align:center"><img data-type="png" _width="553.74px" class src="https://news.stcn.com/news/202105/W020210509757355927131.png" alt="图片" data-fail="0" draggable="false" data-ratio="0.26895306859205775" data-w="554" oldsrc="W020210509757355927131.png" referrerpolicy="no-referrer"></p><p style="text-align: left;"><strong>小米成立私募公司引发网友热议</strong></p><p style="text-align: left;">5月8日，#小米成立私募公司#冲上了微博热搜，网友炸开锅了，热烈讨论起来。</p><p style="text-align: left;">有网友说，年轻人的第一只基金？</p><p style="text-align: left;">回复说：私募可没几个年轻人够门槛，私募基金都是非公开募集，得是100W以上的有一定的风险承受能力的合格投资者才能购买。</p><p style="text-align:center"><img data-type="png" _width="553.74px" class src="https://news.stcn.com/news/202105/W020210509757356433926.png" alt="图片" data-fail="0" draggable="false" data-ratio="0.9747292418772563" data-w="554" oldsrc="W020210509757356433926.png" referrerpolicy="no-referrer"></p><p style="text-align: left;">还有网友问，雷总基金从业资格证考过了吗？</p><p style="text-align:center"><img data-type="png" _width="553.742px" class src="https://news.stcn.com/news/202105/W020210509757356881262.png" alt="图片" data-fail="0" draggable="false" data-ratio="0.4442446043165468" data-w="556" oldsrc="W020210509757356881262.png" referrerpolicy="no-referrer"></p><p style="text-align: left;">也有网友好奇，不知道小米会聘请谁去当私募经理，给他的第一个基金起个名。</p><p style="text-align:center"><img data-type="png" _width="548.51px" class src="https://news.stcn.com/news/202105/W020210509757357363436.png" alt="图片" data-fail="0" draggable="false" data-ratio="0.8178506375227687" data-w="549" oldsrc="W020210509757357363436.png" referrerpolicy="no-referrer"></p><p style="text-align: left;">还有网友说，雷布斯果然还是想当雷菲特。</p><p style="text-align:center"><img data-type="png" _width="553.51px" class src="https://news.stcn.com/news/202105/W020210509757357744287.png" alt="图片" data-fail="0" draggable="false" data-ratio="0.6552346570397112" data-w="554" oldsrc="W020210509757357744287.png" referrerpolicy="no-referrer"></p><p style="text-align: left;">还有网友将小米做私募跟前不久小米造车联系起来，说新的私募公司专门用来孵化提供小米车需要的各种零部件的中小型未上市公司。</p><p style="text-align:center"><img data-type="png" _width="551.01px" class src="https://news.stcn.com/news/202105/W020210509757358184300.png" alt="图片" data-fail="0" draggable="false" data-ratio="0.33030852994555354" data-w="551" oldsrc="W020210509757358184300.png" referrerpolicy="no-referrer"></p><p style="text-align: left;">但也有网友对此不是很看好。</p><p style="text-align:center"><img data-type="png" _width="553.742px" class src="https://news.stcn.com/news/202105/W020210509757358685170.png" alt="图片" data-fail="0" draggable="false" data-ratio="1.0773381294964028" data-w="556" oldsrc="W020210509757358685170.png" referrerpolicy="no-referrer"></p><p style="text-align: left;"><strong>雷军早就开始做私募了</strong></p><p style="text-align: left;">基金君发现，其实小米掌门人雷军早就开始做私募基金业务了。</p><p style="text-align: left;">基金业协会备案登记信息显示，拉萨经济技术开发区顺为资本投资咨询有限公司成立于2015年1月7日，注册资本1个亿，2015年6月29日在基金业协会登记为私募股权、创业投资基金管理人，目前规模在50-100亿元。</p><p style="text-align:center"><img data-type="png" _width="553.74px" class src="https://news.stcn.com/news/202105/W020210509757359117329.png" alt="图片" data-fail="0" draggable="false" data-ratio="0.944043321299639" data-w="554" oldsrc="W020210509757359117329.png" referrerpolicy="no-referrer"></p><p style="text-align: left;">该私募的法定代表人、董事长、执行董事是雷军，其出资人是雷军认缴75%，张彤认缴25%。目前旗下备案了武汉顺科股权投资合伙企业（有限合伙）等9只私募基金。</p><p style="text-align:center"><img data-type="png" _width="553.74px" class src="https://news.stcn.com/news/202105/W020210509757359622542.png" alt="图片" data-fail="0" draggable="false" data-ratio="0.6064981949458483" data-w="554" oldsrc="W020210509757359622542.png" referrerpolicy="no-referrer"></p><p style="text-align: left;">公开资料显示，顺为资本是雷军与曾经的金山投资人许达来一起组建的一家专注于TMT领域投资的基金，其名取自雷军名言“顺势而为”。</p><p style="text-align: left;">官网介绍，顺为资本团队是由投资行业和互联网行业资深人士组成，具有丰富的风险投资，资本运作和企业经营管理经验，主导投资了超过四百家创业公司。</p><p style="text-align: left;">顺为资本重点关注移动互联网、互联网+、智能硬件、智能制造、<a class="trsHotWords" style="color: #2f67d1;text-decoration: none;" target="_blank" href="http://info.stcn.com/dc/stock/?stockcode=SZ000021" code="000021">深科技</a>、农村互联网等领域与其他行业结合所带来的变革。其所投项目包括小米、华米、爱奇艺、蔚来汽车、云米、御家汇、趣头条、阿姨帮、丁香园等。</p><p style="text-align:center"><img data-type="png" _width="553.74px" class src="https://news.stcn.com/news/202105/W020210509757360090345.png" alt="图片" data-fail="0" draggable="false" data-ratio="0.7454873646209387" data-w="554" oldsrc="W020210509757360090345.png" referrerpolicy="no-referrer"></p><p style="text-align: left;">还有一家湖北小米长江产业投资基金管理有限公司，成立于2017年10月26日，2018年4月2日在基金业协会备案登记为私募股权、创业投资基金管理人，注册资本1000万元，目前管理规模也在50-100亿元。</p><p style="text-align:center"><img data-type="png" _width="553.74px" class src="https://news.stcn.com/news/202105/W020210509757360503244.png" alt="图片" data-fail="0" draggable="false" data-ratio="0.9711191335740073" data-w="554" oldsrc="W020210509757360503244.png" referrerpolicy="no-referrer"></p><p style="text-align: left;">湖北小米长江产业投资基金的实控人及董事长是雷军，其由小米产业投资、湖北省长江经济带产业基金、武汉光谷产业投资基金分别出资80%、15%、5%设立。目前该私募仅备案了一只私募基金“湖北小米长江产业基金合伙企业（有限合伙）”。</p><p style="text-align:center"><img data-type="png" _width="553.74px" class src="https://news.stcn.com/news/202105/W020210509757360920945.png" alt="图片" data-fail="0" draggable="false" data-ratio="0.22021660649819494" data-w="554" oldsrc="W020210509757360920945.png" referrerpolicy="no-referrer"></p><p style="text-align: left;">小米长江产业基金深耕于一级市场，积极投资半导体与集成电路领域，2020年9月，格力集团与小米集团、<a class="trsHotWords" style="color: #2f67d1;text-decoration: none;" target="_blank" href="http://info.stcn.com/dc/stock/?stockcode=SS601998" code="601998">中信银行</a>签署战略合作协议，根据协议，格力集团将出资35.45亿元参与由小米集团发起的小米产业基金的投资和管理。</p><p style="text-align: left;">业内人士表示，小米长江产业基金积极围绕小米产业链布局上下游行业，包括芯片、半导体产业。从其投资情况看，包括华景传感、科韵激光、睿翔讯通、天易合芯、芯来科技、速通半导体等。</p><p style="text-align: left;">因此，关于未来小米私募股权基金的投向，业内人士认为，应该主要也是围绕小米产业链上下游进行投资，积极投向一些创新创业的互联网、科技企业。当然，等到这家机构完成备案，我们拭目以待。</p><p style="text-align: center;"><img data-type="png" _width="553.74px" class
