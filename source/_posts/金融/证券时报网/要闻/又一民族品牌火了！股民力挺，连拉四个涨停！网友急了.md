<p>中国基金报记者 王建蔷</p><p>继鸿星尔克之后，而另一淡出网友视野的民族品牌——贵人鸟也火了。</p><p style="text-align: center;"><strong><a class="trsHotWords" style="color: #2f67d1;text-decoration: none;" target="_blank" href="http://info.stcn.com/dc/stock/?stockcode=SS603555" code="603555">ST贵人</a>连续4涨停</strong></p><p>7月29日，<a class="trsHotWords" style="color: #2f67d1;text-decoration: none;" target="_blank" href="http://info.stcn.com/dc/stock/?stockcode=SS603555" code="603555">ST贵人</a>再度涨停，股价报3.21元/股，总市值50亿元。这已经是该股连续第4个涨停板，期间股价大涨22%。目前的股价也达到了贵人鸟自今年5月以来的最高价。</p><p style="text-align:center"><img data-backh="768" data-backw="554" data-type="png" _width="100%" class src="https://news.stcn.com/news/202107/W020210729594213265590.png" alt="图片" data-fail="0" draggable="false" data-ratio="1.3862815884476534" data-w="554" oldsrc="W020210729594213265590.png" referrerpolicy="no-referrer"></p><p>从近期表现来看，7月23日还在跌停的贵人鸟，自7月26日开盘后，股价呈一字涨停，直至7月28日已获三连板，使得贵人鸟的收盘价格涨幅偏离值累计达到了15%。</p><p>7月28日晚间，面对公司股价连续三涨停，<a class="trsHotWords" style="color: #2f67d1;text-decoration: none;" target="_blank" href="http://info.stcn.com/dc/stock/?stockcode=SS603555" code="603555">ST贵人</a>急发公告回应。</p><p>贵人鸟表示，经自查，贵人鸟未发现对公司股票交易价格产生重大影响的媒体报道或市场传闻，也未涉及热点概念事项，同时也不存在应披露而未披露的重大事项。</p><p style="text-align:center"><img data-backh="463" data-backw="554" data-type="png" _width="100%" class src="https://news.stcn.com/news/202107/W020210729594216532934.png" alt="图片" data-fail="0" draggable="false" data-ratio="0.8357400722021661" data-w="554" oldsrc="W020210729594216532934.png" referrerpolicy="no-referrer"></p><p style="text-align: center;"><strong>因捐款赈灾被网友“野性消费”</strong></p><p>事实上，此次贵人鸟股价连续四涨停，源于其驰援河南灾情的举动。</p><p>面对这次洪灾，7月24日上午10时左右，贵人鸟官方微博发文“众志成城，共渡难关，有困难我们一起扛！”并同时发布了向河南捐赠物质的视频。</p><p>据媒体报道，面对这次河南洪灾，贵人鸟也慷慨解囊捐款5000万（含捐款3000万、物资2000万），且贵人鸟前董事长林天福直接带队驰援郑州。</p><p>于是网友惊呼：这只憨鸟，真的是快破产了还在捐款。这样的良心企业，我们要守护好。几个小时后，“贵人鸟捐款”引发关注并冲上热搜。</p><p style="text-align:center"><img data-backh="401" data-backw="554" data-type="png" _width="100%" class src="https://news.stcn.com/news/202107/W020210729594218093152.png" alt="图片" data-fail="0" draggable="false" data-ratio="1" oldsrc="W020210729594218093152.png" referrerpolicy="no-referrer"></p><p>对此，贵人鸟通过官方微博回应：“没想到有一天贵贵也能上热搜。”</p><p style="text-align:center"><img data-backh="291" data-backw="554" data-type="png" _width="100%" class src="https://news.stcn.com/news/202107/W020210729594222177358.png" alt="图片" data-fail="0" draggable="false" data-ratio="1" oldsrc="W020210729594222177358.png" referrerpolicy="no-referrer"></p><p>除被网友顶上热搜，贵人鸟的直播间也迎来网友“野性消费”。据报道，贵人鸟官方运动旗舰店的直播间在7月25日的观看人数超过了50万人次，而此前的直播观看人数只有一两千。7月28日晚间，贵人鸟直播间的观看人数依然维持在16万左右的水平，且呈现持续上涨趋势。</p><p>飞瓜数据显示，在抖音“贵人鸟官方账号”最近一周直播中销售额达到1743万元，其中最高是在7月24日当日创造了1292万元的销售额。</p><p>不少网友冲向贵人鸟直播间“野性消费”的同时，股民则冲向贵人鸟股票进行“野性投资”。截至今天，贵人鸟连续斩获4个涨停。</p><p style="text-align: center;"><strong>网友直呼：买不进去怎么办</strong></p><p>沉寂许久的贵人鸟股吧同样炸锅了。</p><p>不少股民在股吧里表示，就是要支持这样的企业。还有股民直呼，“买不进去怎么办？”但同时也有一些股民提示道，理性投资，后市可能比较危险。有人甚至认为，已经被ST的贵人鸟或就此涅槃重生。</p><p style="text-align:center"><img data-backh="188" data-backw="554" data-type="png" _width="100%" class src="https://news.stcn.com/news/202107/W020210729594226451002.png" alt="图片" data-fail="0" draggable="false" data-ratio="0.33935018050541516" data-w="554" oldsrc="W020210729594226451002.png" referrerpolicy="no-referrer"></p><p style="text-align:center"><img data-backh="167" data-backw="554" data-type="png" _width="100%" class src="https://news.stcn.com/news/202107/W020210729594229581512.png" alt="图片" data-fail="0" draggable="false" data-ratio="1" oldsrc="W020210729594229581512.png" referrerpolicy="no-referrer"></p><p>据龙虎榜数据，昨天买入金额最大的前5名营业部中有4个位于拉萨的席位，有网友称拉萨帮进场了。</p><p style="text-align: center;"><strong>公司负债35亿</strong></p><p>从财务数据看，贵人鸟目前的状况的确不容乐观。</p><p>作为“A股运动品牌第一股”的贵人鸟，成立于2004年，于2014年1月挂牌上市，巅峰时总市值一度超过400亿元。</p><p style="text-align:center"><img data-backh="256" data-backw="554" data-type="png" _width="100%" class src="https://news.stcn.com/news/202107/W020210729594232506666.png" alt="图片" data-fail="0" draggable="false" data-ratio="1" oldsrc="W020210729594232506666.png" referrerpolicy="no-referrer"></p><p>然而今非昔比，因2018年、2019年连续两年的业绩亏损，贵人鸟被实施退市风险警示。公司2020年度发生净亏损约3.8亿，且已连续3年亏损，贵人鸟也因此被实行了“*ST”。</p><p>对比鸿星尔克的财务数据，贵人鸟显得更为惨淡一些。从2018年至2020年，贵人鸟连续三年亏损，净亏损分别为6.86亿、10.96亿、3.82亿元。</p><p style="text-align:center"><img data-backh="374" data-backw="554" data-type="png" _width="100%" class src="https://news.stcn.com/news/202107/W020210729594233990122.png" alt="图片" data-fail="0" draggable="false" data-ratio="1" oldsrc="W020210729594233990122.png" referrerpolicy="no-referrer"></p><p>今年公司继续亏损，2021年第一季度财报显示，该公司归属于上市公司股东的净资产约-4873.36万元。合并资产负债表显示，截至2021年3月31日，贵人鸟总负债高达35.26亿元，货币资金2268万元。</p><p style="text-align: center;"><strong>刚刚宣布“不做鞋”</strong></p><p>面对经营困境，贵人鸟于今年5月开始重整。</p><p>7月14日，贵人鸟发布了关于优化调整公司经营模式的公告，宣布优化调整公司经营模式，计划在现有运动服装和配饰外协加工采购模式的基础上，将目前公司运动鞋产品自主生产为主的生产模式也调整为外协加工采购模式。经此调整后，贵人鸟不再保留与生产相关的人员和业务。</p><p style="text-align:center"><img data-backh="293" data-backw="554" data-type="png" _width="100%" class src="https://news.stcn.com/news/202107/W020210729594237067058.png" alt="图片" data-fail="0" draggable="false" data-ratio="1" oldsrc="W020210729594237067058.png" referrerpolicy="no-referrer"></p><p>贵人鸟认为，本次调整虽然可能导致公司在外协管控上面临一定的风险，但有利于公司进一步降低生产经营和管理成本，提高公司持续经营能力和盈利能力。</p><p>在重整计划执行完毕后，7月21日，贵人鸟股票停牌一天，后于7月22日复牌。在复牌日开市起，撤销贵人鸟退市风险警示并实施其他风险警示，公司股票简称由“*<a class="trsHotWords" style="color: #2f67d1;text-decoration: none;" target="_blank" href="http://info.stcn.com/dc/stock/?stockcode=SS603555" code="603555">ST贵人</a>”变更为“<a class="trsHotWords" style="color: #2f67d1;text-decoration: none;" target="_blank" href="http://info.stcn.com/dc/stock/?stockcode=SS603555" code="603555">ST贵人</a>”。</p><p>在自身调整经营模式的当口，叠加此次的赈灾事件，贵人鸟能否涅槃重生？</p><p><a class="trsHotWords" style="color: #2f67d1;text-decoration: none;" target="_blank" href="http://info.stcn.com/dc/stock/?stockcode=SS601878" code="601878">浙商证券</a>在研报中指出，鸿星尔克等事件再次验证国潮崛起长期趋势，短期调整不改国货崛起大趋势。短期来看，一方面中国品牌无论<a class="trsHotWords" style="color: #2f67d1;text-decoration: none;" target="_blank" href="http://info.stcn.com/dc/stock/?stockcode=SZ300959" code="300959">线上线下</a>依旧保持了良好的增长速度，海外品牌虽然有所恢复但并未完全恢复至疫情前的增长趋势。同时鸿星尔克由于赈灾火速出圈更加验证了当前消费者对国货的认可。</p>
