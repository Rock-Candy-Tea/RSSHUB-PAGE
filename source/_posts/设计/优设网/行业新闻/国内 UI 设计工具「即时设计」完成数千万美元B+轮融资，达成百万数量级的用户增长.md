<p>国产云端 UI 设计工具「即时设计」近日完成数千万美元B+轮融资。本轮融资由独秀资本(Unicorn Capital Partners）领投，老股东Coatue Management、SIG海纳亚洲、源码资本持续跟投，由高鹄资本担任独家财务顾问。</p>
<p><span class="img-zoom"><img alt="国内 UI 设计工具「即时设计」完成数千万美元B+轮融资，达成百万数量级的用户增长" loading="lazy" class="alignnone size-full wp-image-462041" src="https://image.uisdc.com/wp-content/uploads/2022/06/jd-20220602-1.jpg" width="800" height="534" referrerpolicy="no-referrer"></span></p>
<p><span style="color: #999999"><em>即时设计</em></span></p>
<p>据了解，完成逆势融资的「即时设计」近期动作频频，在产品力提升、用户增长、行业影响等多个方面都成绩不俗。不仅接连完成了产品功能、性能、体验的多次升级，还于近日正式面向全球用户开放了其插件 API，让所有的开发者、设计团队以及企业都能根据自身需求自由定制插件，大幅提升了实用性与扩展性。在产品持续进步的同时，「即时设计」还达成了百万数量级的用户增长。</p>
<h5>逆势融资的背后：填补国产专业<a href="https://www.uisdc.com/tag/%e8%ae%be%e8%ae%a1%e8%bd%af%e4%bb%b6" class="tag_a" target="_blank">设计软件</a>空白，在线实时协作为用户提效</h5>
<p>近年来，随着市场环境与经营环境的变化，国内企业对“效率”的重视达到了前所未有的高度，对于效率工具的需求也被广泛激发。然而受限于技术门槛与工程难度，在专业级生产力工具市场中，国产软件仍有大量空白待填补。</p>
<p><span class="img-zoom"><img alt="国内 UI 设计工具「即时设计」完成数千万美元B+轮融资，达成百万数量级的用户增长" loading="lazy" class="alignnone size-full wp-image-462042" src="https://image.uisdc.com/wp-content/uploads/2022/06/jd-20220602-2.jpg" width="800" height="381" referrerpolicy="no-referrer"></span></p>
<p><span style="color: #999999"><em>即时设计：云端实时协作提效</em></span></p>
<p>在这样的环境下，「即时设计」作为国内首款专业级云端UI设计软件，于2020年9月正式上线。用户们很快发现，「即时设计」不仅是一个UI设计工具，还是一个面向产研团队的在线协作平台。据了解，「即时设计」发展至今，一方面在专业<a href="https://www.uisdc.com/tag/ui%e8%ae%be%e8%ae%a1" class="tag_a" target="_blank">UI设计</a>功能上，已经兼具完善的Sketch基础功能和 Figma 高阶功能，并支持多人实时在线编辑，可以实现直播级同步。另一方面，「即时设计」的产品原型交互连线、内置开发标注切图、实时打点评论反馈等协作功能，可以一站式满足产研团队工作需求，提升远程协同的办公效率。</p>
<p>在中国，Adobe系列、Sketch等本地软件仍然是<a href="https://www.uisdc.com/tag/%e8%ae%be%e8%ae%a1%e5%b8%88" class="tag_a" target="_blank">设计师</a>们的主流工具。而Figma在海外掀起的云端化、协作化浪潮，为「即时设计」带来了在国内的发展带来了巨大的机会。广阔的市场空间，填补国产云端专业设计软件市场空白的先发优势，都成为了「即时设计」得以完成逆势融资的重要因素。</p>
<h5>高速增长的背后：以每天迭代的频率发动产品力引擎，与用户建立共创氛围</h5>
<p>在过去的报道中，「即时设计」创始团队曾表示：“产品力的提升是核心，功能高效的迭代，才能拉动用户数快速增长。”而其近1年来的“成绩单”似乎也印证了这一点。</p>
<p>2022年3月，「即时设计」官宣用户量突破百万，在文章评论区，许多用户的留言都离不开“见证即时设计的成长”。“一个每天都在变得更好用的产品”，几乎成为了「即时设计」在用户心中的共同印象。</p>
<p><span class="img-zoom"><img alt="国内 UI 设计工具「即时设计」完成数千万美元B+轮融资，达成百万数量级的用户增长" loading="lazy" class="alignnone size-full wp-image-462043" src="https://image.uisdc.com/wp-content/uploads/2022/06/jd-20220602-3.jpg" width="800" height="518" referrerpolicy="no-referrer"></span></p>
<p><span style="color: #999999"><em>即时设计：产品功能快速升级</em></span></p>
<p>以「即时设计」近期上线的工作台3.0为例，在察觉到用户们日益增长的文件管理需求后，「即时设计」团队通过对文件列表、文件夹封面、团队目录的全面优化，帮助用户能第一时间在工作台找到自己需要的内容，在受到用户广泛好评的同时，也诠释了效率的提升藏在细节中。这份对用户体贴的长期坚持，为「即时设计」与用户建立起了不同于常规意义上“使用者”与“生产力工具”间的紧密联系。</p>
<p>“我们有很大一部分用户增长是来自于老用户的自发宣传推广”，「即时设计」创始团队透露，并将其归因于一路走来对产品力驱动的坚持，“我们做的毕竟是个用户每天都会与之相处4-5个小时以上的产品。好用不好用是糊弄不了人的。所以我们一直坚持着每天迭代更新的频率，让产品持续进步。”即时设计创始团队还提到，“我们的用户们还每天都非常热心的帮我们提需求、催进度、反馈优化建议，这种与用户共创产品的文化，在产品力提升和用户增长方面都为「即时设计」注入了巨大的能量，也让我在们对未来长期、可持续发展充满信心。”</p>
<p>在商业化方面，「即时设计」也与许多国内知名企业进行了深度的合作和探索，期间所取得的阶段性成果也将会在今后沉淀成更多元的解决方案，为企业客户提供更优质的服务。</p>
<h5>自主化的背后：从“中国版Figma”到“中国的即时设计”</h5>
<p>近年来，随着Matlab封禁国内高校、Figma封禁大疆等事件的发生，专业级软件国产化、自主化的紧迫性被摆上台面。作为国内赛道中，唯一在Figma封禁大疆事件发生前就上线了Figma文件导入功能的产品，「即时设计」为自己引来了许多关注与期待——「即时设计」会扛起国产设计软件自主化的大旗，成为“中国版Figma”么？对此，「即时设计」团队有自己的思考。</p>
<p>“一方面，我们会用最快的速度，力求在产品的综合体验上追评Figma，这里的综合体验包括产品力、插件、社区等等。为了达成这个目标，顺利、高效的融资节奏是必要的，他会让我们更专注于产品力的提升。” 「即时设计」创始团队表示，“另一方面，我们会继续坚持结合国内市场需求，来进行产品创新和功能开发。咱们国内的设计师、产研团队的工作习惯、文化、业务流程等等都与国外不同。我们认为，国内用户真正需要的不是一个中国版Figma，而是一个综合体验不逊于Figma的自主产品。我们希望能早日达成这个目标，先做好一个被用户们广泛认可的、来自中国的「即时设计」，未来再与海外顶尖产品过招、PK。”</p>
<p><span class="img-zoom"><img alt="国内 UI 设计工具「即时设计」完成数千万美元B+轮融资，达成百万数量级的用户增长" loading="lazy" class="alignnone size-full wp-image-462044" src="https://image.uisdc.com/wp-content/uploads/2022/06/jd-20220602-4.jpg" width="800" height="374" referrerpolicy="no-referrer"></span></p>
<p><span style="color: #999999"><em>即时设计：插件生态持续建设</em></span></p>
<p>据介绍，「即时设计」现已正式面向全球用户开放了其插件 API，让所有的开发者、设计团队以及企业都能根据自身需求，充分利用 API 的扩展能力来自由定制插件，不仅能团队内部使用，还可以开放给所有「即时设计」用户。与此同时，「即时设计」的插件社区中已同步上线了数十款插件，这种开放式的生态将使得「即时设计」的工具属性得到质的提升，自身的快速迭代配合着未来海量的第三方插件，真正的全民共创会带给这款设计工具难以想象的全新生命力。</p>
<p>目前，「即时设计」的团队规模已近百人，并保持着75%以上的产研占比。在获得资本青睐、用户认可的当下，这个将精力与资源始终聚焦在“让产品每天都变得更好用”上的团队，也在持续发展壮大。据「即时设计」创始团队透露，预期将在今年内实现团队规模翻倍，希望更多志同道合的伙伴加入，共同实现“与中国设计师们共同定义中国的设计工具”的梦想。</p>
