<p><span class="img-zoom"><img alt="再获两轮数千万美元融资！高速成长的「即时设计」发布全新 2.0 版本" loading="lazy" class="alignnone size-full wp-image-436998" src="https://image.uisdc.com/wp-content/uploads/2021/10/jishi-design-4-20211026-1.jpg" width="800" height="573" referrerpolicy="no-referrer"></span></p>
<p><span style="color: #999999">摘要：国产云端专业 UI 设计工具「即时设计」近日宣布，在多家海内外知名投资机构的连续参与下，公司已完成 A++ 及 B 轮融资。团队于上季度实现 6 位数的用户增长，并在本周正式发布「即时设计 2.0 版本」，后续将持续打造以素材和插件为主的社区，跟更多的企业和产品扩大合作，未来将产品进一步推向全球市场。</span></p>
<p>据悉，近日国产云端 UI 设计工具「即时设计」再次获得数千万美金 A++ 和 B 轮融资。A++ 轮由高瓴创投和 SIG 海纳亚洲联合领投，源码资本、靖亚资本跟投；B 轮由 Coatue Management 领投，高瓴创投、SIG 海纳亚洲、源码资本、蓝驰创投跟投，由元一资本担任独家财务顾问。</p>
<p>在我们与「即时设计」创始团队的采访中了解到，相较于融资而言，「即时设计」的数据增长和产品迭代更为迅速。仅在上个季度就已经实现了 6 位数的用户增长，其余各项用户关键指标也得到了很好的提升，在平台上累计产生的设计项目近百万。</p>
<p>从产品的功能迭代速度来看，即时设计也体现出了自己的“中国速度”，仅在上线后的这一年时间里，就新增了大大小小超百项功能。而就在本周，「即时设计」也发布了它的全新 2.0 版本。</p>
<p><span class="img-zoom"><img alt="再获两轮数千万美元融资！高速成长的「即时设计」发布全新 2.0 版本" loading="lazy" class="alignnone size-full wp-image-436999" src="https://image.uisdc.com/wp-content/uploads/2021/10/jishi-design-4-20211026-2.jpg" width="1080" height="540" referrerpolicy="no-referrer"></span></p>
<p>全新的 2.0 版本除了必备的功能和性能提升以外，更是横跨了以下 7 个方面的内容，在产品和生态上实现了全方位的升级：</p>
<h4>更专业友好的视觉体验</h4>
<p>即时设计 2.0 对主站的 UI 视觉进行了全面升级，更精炼的设计语言、更统一的全局规范，更丰富的动效反馈等大量细节优化让新的界面整体显得更加清爽亲切，更加突显作为专业 UI 计工具的品牌调性，与用户建立了更清晰、更愉悦的品牌共识。</p>
<p>同时，即时设计发布了「你的工具你定义」的自由式概念，不仅支持深色浅色两种模式，还提供了 5 款主题色供用户自由搭配，满足了更多用户的不同视觉需求。</p>
<h4>更完整的全平台覆盖</h4>
<p>「即时设计」致力于打造全平台的无缝设计协作体验，PC 及 Pad 用户都可随时随地通过浏览器直接完成设计创作，同时也发布了 Win、Mac、Linux 等系统客户端，给用户提供更沉浸的操作体验；还可使用 iOS、Android 客户端和本月即将上线的微信<a href="https://www.uisdc.com/topic/%e5%b0%8f%e7%a8%8b%e5%ba%8f" class="tag_a" target="_blank">小程序</a>满足团队协作中的实时预览、远程评审等需求。</p>
<h4>更完善的数据迁移和备份</h4>
<p>「即时设计」全方位提升了不同设计工具用户的迁移、协作以及备份体验，目前已支持导入 Sketch、XD 、JSD 等文件并进行编辑，还可导出为 Sketch 和 JSD 文件满足多工具协作和统一备份以及数据的本地保存等需求；另外，通过开发离线编辑功能，确保了用户在恶劣网络环境下仍能继续工作，网络恢复后也能自动第一时间将数据同步至云端。（注：JSD 为即时设计自有文件格式）</p>
<h4>更强大的核心及特色功能</h4>
<p>凭借着极快的迭代速度，仅用一年时间，「即时设计」的核心功能已经基本追齐 Figma，自动布局、变体、交互动画等深度功能都已完善；同时还结合中国<a href="https://www.uisdc.com/tag/%e8%ae%be%e8%ae%a1%e5%b8%88" class="tag_a" target="_blank">设计师</a>的实际需求，优化了协作交付的体验，推出了云端共享字体、设计资源库、文件状态和项目跟进等一系列特色功能，更加适应国内用户的使用场景和习惯，能让国内设计团队获得更高的工作效率和更好的产出结果。</p>
<h4>更开放的社区生态</h4>
<p>「即时设计」的社区正处于快速发展的阶段，资源广场已发布近 500 套设计资源，累计订阅和使用量破 6 位数，当前也已上线十余款插件用以满足用户更多场景下的深度需求，本季度还会发布更多插件用以满足用户的更多需求。接下来，即时设计会进一步开放社区，并和更多的设计师及团队一起创造国内足够丰富的资源及插件生态。</p>
<h4>更全面的合作生态</h4>
<p>作为一款国产工具，「即时设计」在与国内平台合作方面有着绝对的先天优势，能够给设计师带来更完整的生态体验。目前，即时设计已入驻飞书并即将上线钉钉应用市场，与站酷、UI中国、优设等多个优秀设计社区进行了深度合作，同时还上架了 iconfont 插件，让用户可以直接在工具内使用 <a href="https://www.uisdc.com/tag/icon" class="tag_a" target="_blank">icon</a>font 上的海量图标。<br>
近期，「即时设计」还将与 「字由」完美结合，让更多的设计师能用上没有版权顾虑的丰富字体，提高创作体验。未来，「即时设计」还将进一步扩大各个渠道和维度上的合作，将拥抱更多的合作伙伴和产品生态。</p>
<h4>更符合国内团队的产研协作</h4>
<p>「即时设计」设计+协作的平台形态已经成熟，设计师的设计创作、交互动效，产品经理的页面跳转梳理，同产研团队的在线评审、打点讨论，以及给开发交付的切图导出、参数代码标注等功能需求现在皆可通过即时设计一站式完成，并且实时协同。<br>
而且利用团队文件状态标签、项目跟进和嵌套文件夹等独特功能，配合在线工具云端存储历史版本的优势，足以让任何一个产研团队极大程度地得到完善的数字资产管理和协同效率提升。</p>
<p><span class="img-zoom"><img alt="再获两轮数千万美元融资！高速成长的「即时设计」发布全新 2.0 版本" loading="lazy" class="alignnone size-full wp-image-437000" src="https://image.uisdc.com/wp-content/uploads/2021/10/jishi-design-4-20211026-3.jpg" width="800" height="356" referrerpolicy="no-referrer"></span></p>
<p>如今的「即时设计」团队与一年前相比，也已经有了很大的成长。</p>
<p>据了解，在短短半年时间里「即时设计」增添了数十位年轻有活力的新成员，他们中有许多来自“字节”、“阿里”、“滴滴”、“猎豹”等知名互联网企业，这些有冲劲的新鲜血液们和原有在图形可视化领域深耕六年之久的核心团队融合在一起，产生了极具创造力的化学反应，也极大地推动了「即时设计」的产品进程，大大提升了「即时设计」的产品质量。</p>
<p>公司产研人员占比一直在 85% 以上，以产品力来驱动用户增长也是团队始终坚持和最为擅长的事情。「即时设计」表示，团队也正在向百人团队迈进，期待更多志同道合的伙伴们加入，共同来打造优秀的、能定义未来世界主流标准的 UI 设计工具和产研协作平台。</p>
<p>「Figma」目前在海外 UI 设计工具的使用率中稳居第一，今年也达到了百亿美金的新估值。而在国内的市场环境下，「Sketch」、「XD」、「PS」仍然是 UI 设计师的主要生产力工具，但从这样的客户端工具往云端 UI 设计工具迁移的趋势无论在国内还是在国外都愈发增速明显。</p>
<p>新型设计工具云端化的浪潮将为数千万用户规模的设计师和产研群体，带来新的工作模式和新的使用习惯，这一历史性的新机遇也是「即时设计」不仅会下定决心全力服务好国内市场，同时会在未来坚定进军海外市场的坚实动力。</p>
<p><span class="img-zoom"><img alt="再获两轮数千万美元融资！高速成长的「即时设计」发布全新 2.0 版本" loading="lazy" class="alignnone size-full wp-image-437001" src="https://image.uisdc.com/wp-content/uploads/2021/10/jishi-design-4-20211026-4.jpg" width="800" height="356" referrerpolicy="no-referrer"></span></p>
<p>「即时设计」告诉 36 氪，从「即时设计」2.0 开始，他们的目标不再是做中国的「Figma」，而是要先做成中国的「即时设计」而后成为世界的「即时设计」。</p>
<p>为了这个梦想，目前团队不仅在全力加速开发各项产品和功能，也在不断进行积极扩张。</p>
<p>他们坚信中国的设计师们是世界上最优秀的设计师群体，中国的设计师不仅勤劳、专业而且天生具备国际视野和更加团结的合作精神，在不远的未来「即时设计」一定会和中国的设计师一道走出国门走向全球，下一代的设计工具和设计标准也终将由中国设计师来参与定义。</p>
