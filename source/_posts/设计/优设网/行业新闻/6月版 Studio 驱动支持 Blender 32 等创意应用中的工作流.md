<p>本期“NVIDIA Studio 创意加速”将介绍 3D 艺术家 Daniel Barnes 及其作品 Journey。</p>
<p><span class="img-zoom"><img alt="6月版 Studio 驱动支持 Blender 3.2 等创意应用中的工作流" loading="lazy" class="alignnone size-full wp-image-465306" src="https://image.uisdc.com/wp-content/uploads/2022/06/Daniel-Barnes-20220630-1.jpg" width="900" height="506" referrerpolicy="no-referrer"></span></p>
<p><span style="color: #999999"><em>编辑注：本文是我们每周发布的 NVIDIA Studio 创意加速系列的一部分，该系列旨在介绍一些特邀艺术家、提供创意提示和技巧，并展示 NVIDIA Studio 技术如何加速创意工作流。</em></span></p>
<p>6 月版 NVIDIA Studio 驱动现可供下载，该版本可为各创意应用最新版提供优化的稳定性和可靠性更新。</p>
<p>拥有 NVIDIA RTX GPU 的创作者可享受 Blender 3.2、BorisFX Sapphire 2022.5 发行版和 Topaz Denoise AI 3.7.0 中提供的更快性能和全新功能。</p>
<p>本周，NVIDIA 高级<a href="https://www.uisdc.com/tag/%e8%ae%be%e8%ae%a1%e5%b8%88" class="tag_a" target="_blank">设计师</a> Daniel Barnes 将在“NVIDIA Studio 创意加速”中展示其颇具启发性的艺术作品。Barnes 专注于视觉设计和 3D 内容创作，他将介绍在创作 3D 银河场景 Journey 背后的创意工作流。</p>
<h4>6 月重磅消息：Studio 驱动最新版支持 Blender、Sapphire 和 Denoise AI 更新</h4>
<p>6 月版 Studio 驱动支持最新的 Blender 3.2，该版本在 NVIDIA Omniverse 中支持 OpenVDB 平台。</p>
<p><span class="img-zoom"><img alt="6月版 Studio 驱动支持 Blender 3.2 等创意应用中的工作流" loading="lazy" class="alignnone size-full wp-image-465307" src="https://image.uisdc.com/wp-content/uploads/2022/06/Daniel-Barnes-20220630-2.jpg" width="900" height="506" referrerpolicy="no-referrer"></span></p>
<div style="width: 640px;" class="wp-video"><video class="wp-video-shortcode" id="video-465305-3" width="640" height="360" preload="metadata" controls="controls"><source type="video/mp4" src="https://images.nvidia.cn/cn/youtube-replicates/TnX4_BcFY54.mp4?_=3"><a target="_blank" href="https://link.uisdc.com/?redirect=https%3A%2F%2Fimages.nvidia.cn%2Fcn%2Fyoutube-replicates%2FTnX4_BcFY54.mp4">https://images.nvidia.cn/cn/youtube-replicates/TnX4_BcFY54.mp4</a></video></div>
<p>OpenVDB 提供大规模 3D 索引空间，用户可从 Omniverse 导入和导出大量 USD (Universal Scene Description) 文件，同时让这些文件中的体积对象保持完好无损。3D 艺术家可以迭代更大的文件，加速创意工作流，而无需文件转换或文件压缩。</p>
<p><span class="img-zoom"><img alt="6月版 Studio 驱动支持 Blender 3.2 等创意应用中的工作流" loading="lazy" class="alignnone size-full wp-image-465308" src="https://image.uisdc.com/wp-content/uploads/2022/06/Daniel-Barnes-20220630-3.jpg" width="900" height="506" referrerpolicy="no-referrer"></span></p>
<p><span style="color: #999999"><em>来自场景中室内、游泳池、长凳和室外更大世界的光源融合交织，在 Blender 合成器中打造出令人惊艳的视觉效果，且无需增加重新渲染工作</em></span></p>
<p>Blender 3.2 还新增了“Light Group”功能，让艺术家无需重新渲染，即可在合成器中修改光源的颜色和强度。新的“阴影焦散”支持在物体折射的阴影中选择性地渲染焦散，让画面更加逼真。查看 Blender 3.2 更新的完整概述。</p>
<p><span class="img-zoom"><img alt="6月版 Studio 驱动支持 Blender 3.2 等创意应用中的工作流" loading="lazy" class="alignnone size-full wp-image-465309" src="https://image.uisdc.com/wp-content/uploads/2022/06/Daniel-Barnes-20220630-4.jpg" width="900" height="506" referrerpolicy="no-referrer"></span></p>
<div style="width: 640px;" class="wp-video"><video class="wp-video-shortcode" id="video-465305-4" width="640" height="360" preload="metadata" controls="controls"><source type="video/mp4" src="https://images.nvidia.cn/cn/youtube-replicates/76YB4xB2yQg.mp4?_=4"><a target="_blank" href="https://link.uisdc.com/?redirect=https%3A%2F%2Fimages.nvidia.cn%2Fcn%2Fyoutube-replicates%2F76YB4xB2yQg.mp4">https://images.nvidia.cn/cn/youtube-replicates/76YB4xB2yQg.mp4</a></video></div>
<p>BorisFX Sapphire 2022.5 现已支持多 GPU 系统，可在 Blackmagic 的 DaVinci Resolve 中应用 GPU 加速的视觉效果插件，扩展 GPU 性能，并可将渲染速度最高提升至原来的 6 倍。</p>
<p><span class="img-zoom"><img alt="6月版 Studio 驱动支持 Blender 3.2 等创意应用中的工作流" loading="lazy" class="alignnone size-full wp-image-465310" src="https://image.uisdc.com/wp-content/uploads/2022/06/Daniel-Barnes-20220630-5.jpg" width="900" height="506" referrerpolicy="no-referrer"></span></p>
<p>由 NVIDIA TensorRT 框架加速的 Topaz Denoise AI 可让色彩更加一致，同时减少高光和阴影中的斑点。</p>
<p>Topaz Denoise AI 3.7.0 新增对 NVIDIA TensorRT 框架的支持，这意味着 RTX GPU 用户将从大幅提升的推理速度中受益。使用 RAW 模型降噪功能时，推理的运行速度至高可提升至原来的 6 倍。</p>
<p>下载 GeForce Experience 或 NVIDIA RTX Experience，可获取最新 Studio 驱动版本的通知。</p>
<h4>“旅程 (Journey)”便是奖励</h4>
<p>在本周的“NVIDIA Studio 创意加速”聚焦艺术家活动中，我们将观看来自 NVIDIA 的 Daniel Barnes 分享其 3D 科幻场景 Journey 的创意工作流。</p>
<p>Barnes 的艺术作品从各种电影和动漫作品中汲取灵感，通常会结合视觉方面的各种启示。Journey 从 Barnes 最新痴迷的动漫类型“异世界”中延伸而来。在此类动漫中，主角在另一个世界中醒来，不得不探索全新的未知环境。</p>
<p>Barnes 指出，“这种转世的故事情节可能会令人耳目一新，因为这几乎总是可以与生活和人生的各种初次体验联系起来，或者与希望可以利用后见之明以不同方式让某个特定时刻重新来过有关。”</p>
<p><span class="img-zoom"><img alt="6月版 Studio 驱动支持 Blender 3.2 等创意应用中的工作流" loading="lazy" class="alignnone size-full wp-image-465311" src="https://image.uisdc.com/wp-content/uploads/2022/06/Daniel-Barnes-20220630-6.jpg" width="900" height="506" referrerpolicy="no-referrer"></span></p>
<p><span style="color: #999999"><em>Barnes 在 Adobe Photoshop 中首次将 Journey 以可视化形式展现</em></span></p>
<p>Barnes 会在灵感袭来时勾勒出草图（这一幕通常发生在当地的咖啡店），他会使用 Adobe Photoshop 开始处理 Journey。</p>
<p>借助搭载 RTX 3060 显卡的 GeForce 笔记本电脑，Barnes 可享受经 GPU 加速的快速功能，例如 Scrubby Zoom（可快速缩放并调整精细细节）以及 Flick Panning（可加速画布上的移动，随时随地自由创作）。</p>
<p><span class="img-zoom"><img alt="6月版 Studio 驱动支持 Blender 3.2 等创意应用中的工作流" loading="lazy" class="alignnone size-full wp-image-465312" src="https://image.uisdc.com/wp-content/uploads/2022/06/Daniel-Barnes-20220630-7.jpg" width="900" height="505" referrerpolicy="no-referrer"></span></p>
<p>Barnes 将注意力转向 Autodesk Maya，搭建并画出了 Journey 场景的基础几何形状，且首先会利用现有场景中可复用的元素。Barnes 若有所思地说，“尽可能以智能方式开展工作，这一点绝对没错。” GPU 加速视窗让 Barnes 能快速开展交互式 3D 建模，从而快速设置基础模组。</p>
<p>Barnes 使用定制画笔进行雕刻，进一步在 ZBrush 中细化了的一些 3D 模型。然后，他运行 ZBrush Remesh 功能，通过组合几个现有对象来创建全新的单一网格。这简化了纹理应用，让 Journey 的动画效果制作更加简便。</p>
<p><span class="img-zoom"><img alt="6月版 Studio 驱动支持 Blender 3.2 等创意应用中的工作流" loading="lazy" class="alignnone size-full wp-image-465313" src="https://image.uisdc.com/wp-content/uploads/2022/06/Daniel-Barnes-20220630-8.jpg" width="900" height="506" referrerpolicy="no-referrer"></span></p>
<p><span style="color: #999999"><em>借助 Omniverse Create 的材质预设和照明，Barnes 能够更快、更高效地工作</em></span></p>
<p>然后，Barnes 使用 Omniverse Create 应用组装出物理精准且逼真 3D 场景。在家中，他借助 RTX 3080 显卡提供动力支持的桌面系统，使用内置的 RTX 渲染器在视窗内进行交互式可视化，且速度几乎没有任何下降。即使按实际规模搭建，Create 的材质预设和照明也能让 Barnes 轻松、快速且高效地实现逼真的视觉效果。</p>
<p><span class="img-zoom"><img alt="6月版 Studio 驱动支持 Blender 3.2 等创意应用中的工作流" loading="lazy" class="alignnone size-full wp-image-465314" src="https://image.uisdc.com/wp-content/uploads/2022/06/Daniel-Barnes-20220630-9.jpg" width="900" height="562" referrerpolicy="no-referrer"></span></p>
<p>Barnes 在 Create 中便捷的 Adobe Substance 3D Painter 连接器和 Omniverse NVIDIA vMaterials 库之间来回切换，以便探索、创建和优化纹理，还在 Create 中应用了自己独特的<a href="https://www.uisdc.com/tag/%e9%85%8d%e8%89%b2" class="tag_a" target="_blank">配色</a>方案风格。</p>
<p><span class="img-zoom"><img alt="6月版 Studio 驱动支持 Blender 3.2 等创意应用中的工作流" loading="lazy" class="alignnone size-full wp-image-465315" src="https://image.uisdc.com/wp-content/uploads/2022/06/Daniel-Barnes-20220630-10.jpg" width="900" height="506" referrerpolicy="no-referrer"></span></p>
<p>在 Studio 应用套件中，Omniverse Create 和 NVIDIA Canvas 在开发 Journey 方面发挥了关键作用。</p>
<p>设计基本完成，Barnes 计划导出几个启用了场景深度、额外辉光和雾气的版本来进行合成。</p>
<p><span class="img-zoom"><img alt="6月版 Studio 驱动支持 Blender 3.2 等创意应用中的工作流" loading="lazy" class="alignnone size-full wp-image-465316" src="https://image.uisdc.com/wp-content/uploads/2022/06/Daniel-Barnes-20220630-11.jpg" width="900" height="488" referrerpolicy="no-referrer"></span></p>
<p>现在，只需要一个明亮的天空，作品 Journey 即大功告成。Barnes 借助可供 RTX GPU 用户免费使用的 NVIDIA Canvas 应用的强大 AI 功能，将简单的笔触变成逼真且令人惊艳的画面。Barnes 在短短几分钟内就创造出外层空间，因而可探索更多概念，并省去了搜索背景图片或从头创建背景所需的时间。</p>
<p><span class="img-zoom"><img alt="6月版 Studio 驱动支持 Blender 3.2 等创意应用中的工作流" loading="lazy" class="alignnone size-full wp-image-465317" src="https://image.uisdc.com/wp-content/uploads/2022/06/Daniel-Barnes-20220630-12.jpg" width="900" height="497" referrerpolicy="no-referrer"></span></p>
<p>然后，这位艺术家回到 Adobe Photoshop，增加了渗色和风格方面的一些细节，导入 NVIDIA Canvas 设计的背景，最后导出最终渲染图。</p>
<p><span class="img-zoom"><img alt="6月版 Studio 驱动支持 Blender 3.2 等创意应用中的工作流" loading="lazy" class="alignnone size-full wp-image-465318" src="https://image.uisdc.com/wp-content/uploads/2022/06/Daniel-Barnes-20220630-13.jpg" width="900" height="380" referrerpolicy="no-referrer"></span></p>
