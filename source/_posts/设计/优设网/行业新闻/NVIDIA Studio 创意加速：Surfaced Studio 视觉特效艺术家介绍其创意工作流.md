<p>在 Adobe After Effects 和 Blender 中，由 NVIDIA Studio 提供动力支持的 GPU 加速创意应用和功能可加速创意工作流，呈现震撼效果。</p>
<p><span class="img-zoom"><img alt="NVIDIA Studio 创意加速：Surfaced Studio 视觉特效艺术家介绍其创意工作流" loading="lazy" class="alignnone size-full wp-image-467272" src="https://image.uisdc.com/wp-content/uploads/2022/07/Surfaced-Studio-20220714-1.jpg" width="900" height="478" referrerpolicy="no-referrer"></span></p>
<p><span style="color: #999999">编辑注：本文是我们每周发布的 NVIDIA Studio 创意加速系列的一部分，该系列旨在介绍一些特邀艺术家、提供创意提示和技巧，并展示 NVIDIA Studio 技术如何加速创意工作流。</span></p>
<p>在本周的“NVIDIA Studio 创意加速”中，来自 Surfaced Studio 的视觉特效大师将分享其制作的两个精彩片段 Fluid Simulation 与 Destruction，并介绍其创意工作流。</p>
<p><span class="img-zoom"><img alt="NVIDIA Studio 创意加速：Surfaced Studio 视觉特效艺术家介绍其创意工作流" loading="lazy" class="alignnone size-full wp-image-467272" src="https://image.uisdc.com/wp-content/uploads/2022/07/Surfaced-Studio-20220714-1.jpg" width="900" height="478" referrerpolicy="no-referrer"></span></p>
<div style="width: 640px;" class="wp-video"><video class="wp-video-shortcode" id="video-467271-7" width="640" height="360" preload="metadata" controls="controls"><source type="video/mp4" src="https://blogs.nvidia.cn/wp-content/uploads/2022/07/studio-itns-wk13-fluid-sim-01.mp4?_=7"><a target="_blank" href="https://link.uisdc.com/?redirect=https%3A%2F%2Fblogs.nvidia.cn%2Fwp-content%2Fuploads%2F2022%2F07%2Fstudio-itns-wk13-fluid-sim-01.mp4">https://blogs.nvidia.cn/wp-content/uploads/2022/07/studio-itns-wk13-fluid-sim-01.mp4</a></video></div>
<p>这两个片段具有惊艳的视觉效果，令 Surfaced Studio 在 YouTube 频道上名声大噪。</p>
<p><span class="img-zoom"><img alt="NVIDIA Studio 创意加速：Surfaced Studio 视觉特效艺术家介绍其创意工作流" loading="lazy" class="alignnone size-full wp-image-467273" src="https://image.uisdc.com/wp-content/uploads/2022/07/Surfaced-Studio-20220714-2.jpg" width="900" height="470" referrerpolicy="no-referrer"></span></p>
<div style="width: 640px;" class="wp-video"><video class="wp-video-shortcode" id="video-467271-8" width="640" height="360" preload="metadata" controls="controls"><source type="video/mp4" src="https://blogs.nvidia.cn/wp-content/uploads/2022/07/studio-itns-wk13-deconstruction.mp4?_=8"><a target="_blank" href="https://link.uisdc.com/?redirect=https%3A%2F%2Fblogs.nvidia.cn%2Fwp-content%2Fuploads%2F2022%2F07%2Fstudio-itns-wk13-deconstruction.mp4">https://blogs.nvidia.cn/wp-content/uploads/2022/07/studio-itns-wk13-deconstruction.mp4</a></video></div>
<p>Surfaced Studio 凭借“寓教于乐”的教学风格大获成功，用轻松愉快的节奏介绍关键技术并分享具有创造性的见解，为订阅者带来有趣又难忘的学习体验。</p>
<p>他沉思后说道：“我承认自己是个书呆子，我的教程也充满了蹩脚的笑话，但我很幸运，我发现了一个志同道合的社区，在这里我的艺术创作尝试得到了大家的认可，大家也愿意学习。”</p>
<p><strong>液体创作</strong></p>
<p>在创作 Fluid Simulation 片段的过程中，Surfaced Studio 首先使用 Adobe After Effects 结合两个视频片段：其中一个视频是他假装被一波液体击中，另一个视频则是他的朋友 Jimmy 朝他跑来。随后应用蒙版层使 Jimmy 神奇地消失。</p>
<p><span class="img-zoom"><img alt="NVIDIA Studio 创意加速：Surfaced Studio 视觉特效艺术家介绍其创意工作流" loading="lazy" class="alignnone size-full wp-image-467274" src="https://image.uisdc.com/wp-content/uploads/2022/07/Surfaced-Studio-20220714-3.jpg" width="900" height="506" referrerpolicy="no-referrer"></span></p>
<p><span style="color: #999999">Blender 中的视频播放</span></p>
<p>然后，他对视频片段进行渲染，并将其导入到 Blender 中。这样一来便可将其作为 3D 场景流体模拟的几何参考进行匹配。</p>
<p><span class="img-zoom"><img alt="NVIDIA Studio 创意加速：Surfaced Studio 视觉特效艺术家介绍其创意工作流" loading="lazy" class="alignnone size-full wp-image-467275" src="https://image.uisdc.com/wp-content/uploads/2022/07/Surfaced-Studio-20220714-4.jpg" width="900" height="506" referrerpolicy="no-referrer"></span></p>
<p><span style="color: #999999">使用 Mantaflow 构建的流体模拟与 Surfaced Studio 相撞</span></p>
<p>接着，Surfaced Studio 选择 Mantaflow 流体功能，调整参数后创建逼真的流体模拟。如需初步了解流体模拟技术，请查看他的以下教程：FLUID SIMULATIONS in Blender 2.9 with Mantaflow（使用 Mantaflow 在 Blender 2.9 中构建流体模拟）。流体模拟功能由 GeForce RTX 2070 笔记本电脑 GPU 进行加速，与仅使用 CPU 相比，GPU 对模拟进行烘焙的速度更快。</p>
<p><span class="img-zoom"><img alt="NVIDIA Studio 创意加速：Surfaced Studio 视觉特效艺术家介绍其创意工作流" loading="lazy" class="alignnone size-full wp-image-467276" src="https://image.uisdc.com/wp-content/uploads/2022/07/Surfaced-Studio-20220714-5.jpg" width="900" height="562" referrerpolicy="no-referrer"></span></p>
<div style="width: 640px;" class="wp-video"><video class="wp-video-shortcode" id="video-467271-9" width="640" height="360" preload="metadata" controls="controls"><source type="video/mp4" src="https://blogs.nvidia.cn/wp-content/uploads/2022/07/studio-itns-wk13-fluid-sim-02.mp4?_=9"><a target="_blank" href="https://link.uisdc.com/?redirect=https%3A%2F%2Fblogs.nvidia.cn%2Fwp-content%2Fuploads%2F2022%2F07%2Fstudio-itns-wk13-fluid-sim-02.mp4">https://blogs.nvidia.cn/wp-content/uploads/2022/07/studio-itns-wk13-fluid-sim-02.mp4</a></video></div>
<p>为了以逼真准确的物理特性截取碰撞过程，他们在流体层使用了流体属性效果。这样便可确保仅在几个特定帧内由 Jimmy 发出的液体能飞溅到其动画上。</p>
<p><span class="img-zoom"><img alt="NVIDIA Studio 创意加速：Surfaced Studio 视觉特效艺术家介绍其创意工作流" loading="lazy" class="alignnone size-full wp-image-467277" src="https://image.uisdc.com/wp-content/uploads/2022/07/Surfaced-Studio-20220714-6.jpg" width="900" height="377" referrerpolicy="no-referrer"></span></p>
<p><span style="color: #999999">速度向量可解锁由 Surfaced Studio 配备的 GPU 加速的动态模糊效果</span></p>
<p>此外，速度向量还可以直接在流体模拟中实现动态模糊效果，从而进一步提升视频片段的真实感。</p>
<p>Surfaced Studio 表示：“我情不自禁想为任何想要探索创意空间的创作者推荐 NVIDIA GPU，它已经陪伴我十几年了。”</p>
<p>RTX 2070 笔记本电脑 GPU 为他在 Blender 中的整个 3D 创意工作流提供加速，从流体模拟、动态模糊效果，到动画和网格生成。Blender Cycles RTX 加速的 OptiX 光线追踪可在视图和快如闪电的最终渲染中实现快速的交互式建模。Surfaced Studio 表示，GPU 帮助他们节省了大量时间，以便重新投入到创意工作流中。</p>
<p><span class="img-zoom"><img alt="NVIDIA Studio 创意加速：Surfaced Studio 视觉特效艺术家介绍其创意工作流" loading="lazy" class="alignnone size-full wp-image-467278" src="https://image.uisdc.com/wp-content/uploads/2022/07/Surfaced-Studio-20220714-7.jpg" width="900" height="485" referrerpolicy="no-referrer"></span></p>
<p><span style="color: #999999">注意这些完成 3D 动画所需的层</span></p>
<p>Surfaced Studio 随后在 After Effects 中进行合成，将 GPU 加速的“曲线”效果应用到水中，并根据喜好进行更改形状和添加光照。</p>
<p>接着，使用 BorisFX Mocha AE 插件对 Jimmy 的动作进行转描，或者通过逐帧追踪实时实景镜头来创建视频片段，从而为角色制作动画。这通常是一个漫长的过程，但借助 GPU 加速插件只需片刻即可完成此任务。</p>
<p>然后，使用同样由 GPU 加速的色相/饱和度、亮度和色彩平衡功能结合使用，完成对颜色的润色。</p>
<p>最后，使用 GPU 加速的 NVENC 编码器快速导出最终视频文件。</p>
<p><span class="img-zoom"><img alt="NVIDIA Studio 创意加速：Surfaced Studio 视觉特效艺术家介绍其创意工作流" loading="lazy" class="alignnone size-full wp-image-467279" src="https://image.uisdc.com/wp-content/uploads/2022/07/Surfaced-Studio-20220714-8.jpg" width="900" height="506" referrerpolicy="no-referrer"></span></p>
<div style="width: 640px;" class="wp-video"><video class="wp-video-shortcode" id="video-467271-10" width="640" height="360" preload="metadata" controls="controls"><source type="video/mp4" src="https://images.nvidia.cn/cn/youtube-replicates/1MhAxG9JJPg.mp4?_=10"><a target="_blank" href="https://link.uisdc.com/?redirect=https%3A%2F%2Fimages.nvidia.cn%2Fcn%2Fyoutube-replicates%2F1MhAxG9JJPg.mp4">https://images.nvidia.cn/cn/youtube-replicates/1MhAxG9JJPg.mp4</a></video></div>
<p>如需深入了解 Surfaced Studio 的创意工作流，请观看他的以下教程：Add 3D Fluid Simulations to Videos w/ Blender & After Effects（如何使用 Blender 和 After Effects 在视频中添加 3D 液体模拟）。</p>
<p>Surfaced Studio说：“我经常使用大量第三方插件，包括 Boris FX Mocha Pro、Continuum、Sapphire、Video Copilot Element 3D 和 Red Giant，这些插件都可以在 GPU 加速的助力下大大提升性能。”</p>
<p>GeForce RTX 2070 笔记本电脑 GPU 在这一创意工作流发挥了重要作用，但 Fluid Simulation 片段仅展现了艺术家高超技术的冰山一角。</p>
<p><strong>小心爆炸！</strong></p>
<p>Surfaced Studio 采用与 Fluid Simulation 类似的创意工作流创作 Destruction 片段。将 Blender 中的 3D 场景与 After Effects 中合成的视频片段相结合，并应用了逼真的物理效果。</p>
