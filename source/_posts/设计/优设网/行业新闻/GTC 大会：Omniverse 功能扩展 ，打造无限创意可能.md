<p><span class="img-zoom"><img alt="GTC 大会：Omniverse 功能扩展 ，打造无限创意可能" loading="lazy" class="alignnone size-full wp-image-438837" src="https://image.uisdc.com/wp-content/uploads/2021/11/gtc-20211111-1.jpg" width="1280" height="680" referrerpolicy="no-referrer"></span></p>
<p>NVIDIA GTC 的实时直播热闹非凡，让世界上这些充满智慧和创造力的人才齐聚一堂。他们利用 AI、计算机图形等众多强大技术，让我们的世界正在被改变。</p>
<p>在 GTC 上，我们宣布推出 NVIDIA Omniverse Avatar。这是一个未来技术平台，用于生成交互式 AI 虚拟化身。此外还推出了适用于 <a href="https://www.uisdc.com/tag/nvidia" class="tag_a" target="_blank">NVIDIA</a> Omniverse（可用于 3D 工作流的实时数字孪生仿真和协作平台）的新功能，包括 Omniverse VR、Remote、Showroom 以及新 Omniverse 连接器，一切旨在让 3D 艺术家受益。</p>
<p>此外，Omniverse Enterprise 现已全面推出，让 Omniverse 生态系统可以惠及更多<a href="https://www.uisdc.com/tag/%e8%ae%be%e8%ae%a1%e5%b8%88" class="tag_a" target="_blank">设计师</a>、工程师和创意专业人士。</p>
<p>罗技最近将 NVIDIA Broadcast 的热门降噪功能集成到其高级头戴式耳机和麦克风系列产品中。</p>
<p>这些全新功能均由 NVIDIA Studio 驱动 11 月版提供支持。不仅如此，新版驱动还支持并改进了 NVIDIA Texture Tools、Twinmotion 2022.1、OctaneRender 2021.1、V-Ray 5、Houdini 19 等应用的性能。</p>
<h4>Omniverse Avatar 助力创造虚拟助理</h4>
<p>有了 Omniverse Avatar 这一用于生成交互式 AI 虚拟化身的技术平台，智能虚拟助理的时代即将开启。虚拟化身是采用光线跟踪 3D 图形打造的交互式角色，能够视物、说话，就广泛的主题进行交谈，并理解自然语言意图。</p>
<p><span class="img-zoom"><img alt="GTC 大会：Omniverse 功能扩展 ，打造无限创意可能" loading="lazy" class="alignnone size-full wp-image-438838" src="https://image.uisdc.com/wp-content/uploads/2021/11/gtc-20211111-2.jpg" width="627" height="357" referrerpolicy="no-referrer"></span></p>
<p>其动画效果采用 Omniverse 中的 NVIDIA Video2Face 和 Audio2Face，以及 2D 和 3D AI 驱动的面部动画和渲染技术。</p>
<h4>Omniverse 扩展程序</h4>
<p>NVIDIA Omniverse 的新功能可以进一步助力 3D 艺术家和设计师的协作式创意工作流。</p>
<p>Omniverse VR 即将率先推出完全图像、实时光线追踪 VR 环境。艺术家和设计师将能够在 Omniverse 中使用 Omniverse Create 或 Omniverse View，协作查看和编辑完全光线追踪的 3D VR 场景，并在原生应用中查看所作更改的效果。</p>
<p>借助 Omniverse Remote，艺术家可以连接至 Omniverse 应用并与其交互，还可以通过 iOS 和 Android 设备查看完全光线追踪的素材。而 AR 与虚拟摄像头工作流程（例如面部、身体和手势跟踪）等更多功能即将登陆移动设备。</p>
<p>Omniverse Showroom 现已作为 Omniverse 公测版中的应用，这让非技术用户可以使用 Omniverse 技术演示。现在，3D 艺术家可以更快进入 Omniverse，使用实时物理和渲染技术进行各种尝试。</p>
<p>借助 Omniverse 连接器，艺术家能够跨多个创意应用无缝协作，实时合成场景并将其可视化。来自 Reallusion iClone、Lightmap HDR Light Studio、Replica Studios、RADiCAL 和 Esri ArcGIS CityEngine 的新连接器可供下载。</p>
<p>其他 Omniverse 连接器、扩展程序和内容库即将推出，包括 Adobe Substance 3D 材质插件、Maxon Cinema4D、Clo Virtual Fashion、Daz3D、e-on software PlantFactory、PlantCatalog 和 VUE、Golaem、Notch、Style3D、TwinBru 和对 Wacom 触控笔的支持。</p>
<h4>Omniverse 可供任何规模的组织使用</h4>
<p>NVIDIA Omniverse Enterprise 已度过测试阶段，现已全面推出，可供任何规模的组织用于开展虚拟办公和实时协作。</p>
<p>专为加速 Omniverse 中不同工作流程而设计的应用现已推出，其中令人瞩目的当属 Omniverse Create 和 Omniverse View。借助 Omniverse Create，用户可以通过交互方式实现场景的组合、照明、仿真和渲染；而使用 Omniverse View，用户能以令人惊叹、物理级准确的逼真度查看 3D 设计项目。</p>
<p>Autodesk 3ds Max、Maya、Revit、Epic Games 虚幻引擎及 Trimble SketchUp 等热门连接器现已全面发行。</p>
<h4>罗技和 NVIDIA Broadcast</h4>
<p>罗技已将 NVIDIA Broadcast 的降噪功能集成到其高级头戴式耳机和麦克风系列产品中。</p>
<p><span class="img-zoom"><img alt="GTC 大会：Omniverse 功能扩展 ，打造无限创意可能" loading="lazy" class="alignnone size-full wp-image-438839" src="https://image.uisdc.com/wp-content/uploads/2021/11/gtc-20211111-3.jpg" width="1124" height="768" referrerpolicy="no-referrer"></span></p>
<p>借助降噪功能，用户一键即可清理麦克风信号，消除不需要的背景噪音。有了由 Tensor Core加速的具备强大 AI 功能的 NVIDIA RTX GPU，无论是按键声、恼人的 PC 风扇声还是宠物叫声，均可消除。</p>
<p>拥有 NVIDIA RTX GPU 的玩家，如果下载最新版本的 G HUB，则无需安装单独的工具，便可启用 NVIDIA Broadcast 降噪功能。在 G HUB 应用中选择耳机或麦克风，依次单击麦克风和 NVIDIA 选项卡，然后按照安装步骤操作。此选项与所有启用 Blue VO!CE 的产品兼容，其中游戏耳机包括 G733、PRO X Gaming 和 PRO X Wireless，而麦克风则包括 Blue Yeti X、 Blue Yeti X 魔兽世界版、Blue Yeti 和 Blue Yeti Nano。</p>
<p>如需下载，请访问 <a target="_blank" href="https://link.uisdc.com/?redirect=https%3A%2F%2Fwww.logitechg.com.cn%2Fzh-cn%2Finnovation%2Fg-hub.html">https://www.logitechg.com.cn/zh-cn/innovation/g-hub.html</a>，或在 G HUB 的设置中选择“查看更新”。</p>
<h4>Studio 驱动 11 月版中的更新</h4>
<p>Studio Suite 随着 NVIDIA Texture Tools 3 继续增强，允许用户创建压缩的高质量纹理文件。新 API 允许 3D 艺术家、游戏开发者和渲染软件压缩 GPU 上的纹理，从而大幅减少内存使用量，并加速整个创意工作流。</p>
<p>Twinmotion 2022.1 已添加 RTX 光线追踪。这一技术由 NVIDIA RTX GPU 上搭载的 RT Core 加速。借助驱动对最新更新的支持，建筑师可以用 GeForce RTX 3080 GPU 渲染逼真的图像和视频，速度比 RTX 2080 SUPER GPU 快 75%。</p>
<p><span class="img-zoom"><img alt="GTC 大会：Omniverse 功能扩展 ，打造无限创意可能" loading="lazy" class="alignnone size-full wp-image-438840" src="https://image.uisdc.com/wp-content/uploads/2021/11/gtc-20211111-4.jpg" width="1430" height="611" referrerpolicy="no-referrer"></span></p>
<p>OctaneRender 2021.1 动态模糊场景在 GeForce RTX 30 系列和 Nvidia RTX 专业 GPU 上的速度得到提升。设计师若想获得逼真的反射效果，不再需要采用变通方案，只需使用动态模糊即可实现。</p>
<p><span class="img-zoom"><img alt="GTC 大会：Omniverse 功能扩展 ，打造无限创意可能" loading="lazy" class="alignnone size-full wp-image-438841" src="https://image.uisdc.com/wp-content/uploads/2021/11/gtc-20211111-5.jpg" width="627" height="357" referrerpolicy="no-referrer"></span></p>
<p><span style="color: #999999"><em>图片来源：OctaneRender 2021.1</em></span></p>
<p>V-Ray 5 Update 2 推出了全新功能 V-Ray Decal，支持从任何角度向表面添加贴标，而无需进行额外的 UVW 工作，也不会对任何底层材质造成干扰。此次更新还推出了新的锐化/模糊层。该功能完全由 NVIDIA RTX GPU 提供加速支持，因此专业人士无需离开 3ds Max，即可享有更强的艺术掌控力。</p>
<p>光缓存的新 GPU 加速可以减少最终帧序列中的漏光和闪烁问题。之前仅在 CPU 上计算（即使 GPU 渲染可用）的近似全局照明现在由 NVIDIA RTX GPU 加速，大大加快了最终帧渲染的速度。</p>
<p><span class="img-zoom"><img alt="GTC 大会：Omniverse 功能扩展 ，打造无限创意可能" loading="lazy" class="alignnone size-full wp-image-438842" src="https://image.uisdc.com/wp-content/uploads/2021/11/gtc-20211111-6.jpg" width="627" height="357" referrerpolicy="no-referrer"></span></p>
<p><em><span style="color: #999999">图片来源：Chaos Group</span></em></p>
<p>Houdini 19 发布了其全新 XPU 渲染引擎 Karma 的 Alpha 版本（该版本使用 CPU 和 GPU），将 Optix 带到 NVIDIA RTX GPU。在测试中，一个体积为 30 亿体元的大型序列在 Karma CPU 的小型渲染农场需要三天以上时间才能渲染，而在由 Karma XPU 驱动并搭载 NVIDIA 30 系列 GPU 的计算机上一夜之间便可完成。</p>
<p><span class="img-zoom"><img alt="GTC 大会：Omniverse 功能扩展 ，打造无限创意可能" loading="lazy" class="alignnone size-full wp-image-438843" src="https://image.uisdc.com/wp-content/uploads/2021/11/gtc-20211111-7.jpg" width="627" height="357" referrerpolicy="no-referrer"></span></p>
<p><span style="color: #999999"><em>图片来源：SideFX</em></span></p>
<p>Unity 最近发布新测试版更新，集成了 NVIDIA DLSS，可以提供更高的画质和性能，带来出色体验。经过此次更新，开发者能够在 Unity 创建的游戏和体验中启用 DLSS。了解 24 Entertainment 如何在“永劫无间 (Naraka: Bladepoint)”中使用 DLSS。</p>
<p>这些令人惊艳的创作者应用更新以及 Omniverse 中的改进皆由 Studio 驱动 11 月版 (472.47) 提供支持。该版本现已可供下载。</p>
<p><span class="img-zoom"><img alt="GTC 大会：Omniverse 功能扩展 ，打造无限创意可能" loading="lazy" class="alignnone size-full wp-image-438844" src="https://image.uisdc.com/wp-content/uploads/2021/11/gtc-20211111-8.jpg" width="627" height="357" referrerpolicy="no-referrer"></span></p>
<p>NVIDIA RTX GPU、独家驱动技术和 AI 协同工作，可提升 NVIDIA Studio 系统的性能，并加速 70 多个创意应用。</p>
