<blockquote><p>GitHub Copilot是微软的 代码建议专家系统</p></blockquote><img src="https://cors.zfour.workers.dev/?http://img.jandan.net/news/2018/09/7b50a4e1e7245c57fc5e29a982c96792.jpg!custom" referrerpolicy="no-referrer"><p>微软的 AI代码建议工具 GitHub Copilot 展示了自己的作用，以至于教育工作者可能不得不重新考虑以后要如何考察计算机科学的学生的能力。</p>
<p>GitHub Copilot →<a href="https://github.com/features/copilot"> https://github.com/features/copilot</a></p>
<p>本月早些时候，马萨诸塞大学阿默斯特分校计算机科学教授 Emery Berger 发表了一篇博文，警告教育工作者，考试的时候，“配备Copilot的学生就像带着沙鹰 参加 刀战。”</p>
<p>他担心 Copilot 将使传统的编程练习(计算机科学培训的一部分)变得毫无意义，因为 Copilot 知道所有答案。</p>
<p>“据我所知，Copilot曾经接受过所有介绍性编程任务的专门培训。”Berger 写道。</p>
<p>Berger解释说：“编程解决具体问题，在许多计算机科学课程中扮演着重要角色，尤其是入门课程。如递归算出斐波那契数列的第n项。Copilot会。它不仅可以做到，而且做得很好。”</p>
<p>Copilot与 Stack Overflow 和其他互联网编程资源不同。</p>
<p>“学生可以上网找代码示例。但讲师也会用Google。教师可以轻易查重。”</p>
<p>但Copilot不一样：“它可以生成更具个性的代码。”</p>
<p>结果，Berger 认为，与编程有关的教学法需要改进。“……那就是作弊机。就像有人给你参数，你只需在特定位置键入参数并点击运行，对吗？因此，我认为认为限制学生使用作弊机是合理或负责的……诱惑太大了。”</p>
<p>Berger 承认 Copilot 很有用，并表示开发人员爱它是有道理的。</p>
<p>他说他在伊利诺伊州有一位同事，使用已锁定的计算机进行编程考试，因此学生在受控环境中参加考试。他建议，这些措施以及口试之类的措施可以帮助解决问题。</p>
<p>Berger 还观察到 Copilot 具有积极的方面，例如填写样板文件和实现 API 的能力。</p>
<p>“我不认为记住无数 API 的细节在智力上很有趣。这不是我们真正应该教授或关注的事情。您知道对象的确切格式吗？我不在乎。”</p>
<p>尽管如此，他认为对教育工作者来说，确保学生得到训练很重要，这可能意味着要重新思考作业的设计和期末成绩的构成。</p>
<p>“我想对此保持乐观。但是我认为至少，需要考虑一下。我不认为有很多计算机学科的教育者意识到眼前正发生的学习革命。”</p>
