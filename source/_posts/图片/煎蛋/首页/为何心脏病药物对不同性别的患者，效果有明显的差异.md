<blockquote><p>了解一下X染色体上的基因失活和失活逃逸</p></blockquote><img src="https://cors.zfour.workers.dev/?http://img.jandan.net/news/2020/05/4f9acaaa910a0cd19af2f79d07cd2c50.jpg!custom" referrerpolicy="no-referrer"><p>大多数哺乳动物，包括人类，都有两条性染色体，也就是所谓的X和Y。性染色体通常来自父母，它们在每个细胞中配对为 XX 或 XY。</p>
<p>携带XX 染色体的人通常被认为是女性，而 XY 染色体的人通常被认为是男性。</p>
<p>这些染色体上的基因在发育和功能中起着关键作用——包括对心脏病的影响。</p>
<p>在我(Brian Aguado, Assistant Professor, University of California San Diego)成为一名研究性染色体如何影响心脏的生物医学工程师之前，我在高中科学课上就学过 X 染色体的奇怪功能。</p>
<p>雌性印花布猫几乎总是有橙色和黑色的皮毛斑点，因为定义毛色的基因存在于 X 染色体上。</p>
<p>当一只橙猫与一只黑交配时，从两个亲本那里各继承一个 X 染色体的雌性后代将混合橙色和黑色的毛皮——一个 X 染色体编码橙色毛皮，而另一个 X 染色体编码黑色毛皮。</p>
<p>出于这个原因，通常有一条 X 和一条 Y 染色体的雄性猫有纯橙色或黑色的外套。</p>
<p>这种皮毛颜色的性别差异是如何在生物学上发生的？事实证明，具有 XX 染色体的细胞会经历 X 失活：来自一个亲本的 X 染色体在某些细胞中被关闭，而从另一亲本继承的 X 染色体在其他细胞中被关闭。</p>
<p>在雌性印花布猫的细胞中，如果一个 X 染色体来自具有橙色毛皮的父母，而另一个 X 染色体来自具有黑色皮毛的父母，则 X 失活会导致橙色和黑色皮毛的斑点。</p>
<p>X 失活是因为像猫和人这样的生物体只需要一条 X 染色体就能正常工作。为了确保正确的“剂量”，每个细胞中的一条 X 染色体都被关闭。</p>
<p>但是失活的 X 染色体上的一些基因会逃脱失活并保持开启状态。事实上，人类X染色体上多达三分之一的基因可以逃脱失活，它们被认为在调节健康和疾病方面发挥着作用。</p>
<p>因为 X 失活只发生在具有多条 X 染色体的人身上，像我这样的研究人员一直在研究在第二个 X 上逃脱失活的基因如何影响具有 XX 染色体的人的健康。我们发现，在某些情况下，细胞性别可能是问题的核心。</p>
<p>X染色体逃逸基因部分调节的一种疾病是主动脉瓣狭窄，这是一种控制血液流向身体其他部位的心脏部分变硬和变窄的疾病。这使心脏加倍工作，最终导致心力衰竭。</p>
<p>就像一个人试图用生锈的铰链推开一扇门一样，心脏会感到疲倦。目前没有可用于减缓或停止 AVS 疾病症状的有效药物。</p>
<p>我的实验室研究性染色体如何影响 AVS 等心血管疾病。</p>
<p>先前的研究表明，XX 与 XY 染色体的人的瓣膜可以以不同的方式变硬。</p>
<p>一般来说，XX 染色体的人会增加疤痕，称之为纤维化，而 XY 染色体的人则会增加钙沉积。</p>
<p>鉴于这些差异，我怀疑给每个人服用相同的药物可能不是治疗 AVS 的最佳方法。但是是什么导致了这些差异呢？</p>
<p>总的来说，研究人员认为性激素会导致瓣膜组织硬化的性别差异。事实上，绝经期间降低雌激素水平会加剧心脏纤维化。</p>
<p>然而，对 XX 和 XY 小鼠心血管疾病的研究发现，即使在手术切除产生性激素的生殖器官后，性别差异仍然存在。</p>
<p>我的团队假设，逃过X失活的基因，对于 XX 染色体的人来说，可能导致了瓣膜硬化的这些差异。为了验证这个想法，我们使用水凝胶开发了瓣膜组织的生物工程模型。</p>
<p>与传统的培养皿培养基相比，水凝胶能更好地模拟瓣膜组织的硬度，使我们能够在更接近人体的环境中研究心脏细胞。</p>
<p>我们发现，我们在水凝胶模型上培养的细胞能够复制瓣膜组织中的性别差异——也就是说，具有 XX 染色体的瓣膜细胞比具有 XY 染色体的细胞具有更多的疤痕。</p>
<p>此外，当我们降低逃脱 X 失活的基因的活性时，我们能够减少 XX 染色体细胞中的疤痕。</p>
<p>我们的下一步是使用我们的模型根据细胞性别确定哪种治疗对 AVS 最有效。</p>
<p>我们发现 XX 瓣膜细胞对这些靶向促进瘢痕形成基因的药物的敏感性低于 XY 细胞。然而，专门针对逃避 X 失活的基因的药物对 XX 细胞具有更强的作用。</p>
<p>心血管疾病中的性别和性别差异非常普遍。例如，女性更少被开出心脏病的处方、跨性别者的心脏病发作率高于顺性别者。</p>
<p>通过考虑性染色体，我和我的团队相信可以针对每个人优化治疗策略，而不管细胞“seXX”如何。</p>
