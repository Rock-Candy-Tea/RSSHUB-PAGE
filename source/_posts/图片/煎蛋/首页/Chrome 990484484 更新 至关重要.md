<blockquote><p>修复重大安全隐患的更新</p></blockquote><img src="https://cors.zfour.workers.dev/?http://img.jandan.net/news/2019/01/785ee9a64452fae8f99452d1a539c143.jpg!custom" referrerpolicy="no-referrer"><p>谷歌Chrome 发布了 99.0.4844.84 版，这一更新至关重要，因为它存在可能被网络犯罪分子利用的安全漏洞(“零日漏洞”)。</p>
<p>“谷歌知道 CVE-2022-1096 存在漏洞。”</p>
<p>因此，最新版本已经在全球推出，谷歌表示可能需要几周时间才能覆盖整个用户群。</p>
<p>出于安全考虑，在这个新版 Chrome 带来的所有变化的注册表中，并没有具体说明工程师发现了什么类型的安全漏洞，基本上是为了不给其他网络犯罪分子提供线索，毕竟不同用户的更新有时间差。 Navigator web 还将自动检查新更新并在下一个版本后自动安装它们。</p>
<p>一旦大量 Chrome 用户升级浏览器，谷歌很可能会提供更多详细信息。此安全漏洞标记为 CVE-2022-1096，并被描述为“V8 类型混淆”漏洞。</p>
<p>他们发现的错误与 Chrome V8 JavaScript 引擎中的一个弱点问题有关，该漏洞由一位匿名安全研究人员报告。</p>
<p>尽管谷歌表示它检测到利用计算机上的这种零日攻击，但该公司没有分享有关这些事件的技术细节或其他信息。</p>
<p>“在大多数用户更新修复程序之前，对错误详细信息和链接的访问可能会受到限制。”谷歌表示。</p>
<p>他们补充说：“如果其他项目同样依赖但尚未修复的第三方库中存在漏洞，我们也将保留这些限制。”</p>
<p>在浏览器供应商发布更多信息之前，Google Chrome 用户应该有足够的时间来更新 Chrome 。</p>
