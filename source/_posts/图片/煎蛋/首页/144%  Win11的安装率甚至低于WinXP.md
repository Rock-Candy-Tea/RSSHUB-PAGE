<blockquote><p>早死的 XP 和最近淘汰的 Windows 7 分别为 1.71% 和 4.7%。 Windows 8 仅占 1.99%，而 Windows 10 占据了令人印象深刻的 80.34% 的系统</p></blockquote><img src="https://cors.zfour.workers.dev/?http://img.jandan.net/news/2018/09/f96953561324ec8e2505a3246266a5ca.jpg!custom" referrerpolicy="no-referrer"><p>根据 IT 资产管理平台 Lansweeper 发布的数据，Windows 11 在企业和家庭电脑操作系统中的占有率正以龟速上涨。令微软失望的是，它甚至还没有超过 Windows XP。</p>
<p>Lansweeper 的数据来自对 1000 万台 PC、20% 的企业和 80% 的消费者的扫描。消费者数据来自该公司收购的网络扫描和发现应用 Fing。</p>
<p>从 Microsoft 的角度来看，它是一种严峻的信息。根据 Lansweeper 的说法，过去三个月升级系统可能增加了两倍，但我们认为 Windows 11 的起点过低。</p>
<p>4 月份的快照在 1.44% 的被调查 Windows 机器上安装了 Windows 11。早死的 XP 和最近淘汰的 Windows 7 分别为 1.71% 和 4.7%。 Windows 8 仅占 1.99%，而 Windows 10 占据了令人印象深刻的 80.34% 的系统。</p>
<p>Lansweeper 首席战略官 Roel Decneut 表示：“尽管采用率在一点一点地增加，但很明显，Windows 11 的升级并没有微软希望的那么快，尤其是在商业环境中。许多组织不必购买满足这些条件的新机器，而其他组织则对 Windows 10 的当前存在感到满意，该系统将继续支持到 2025 年。”</p>
<p>由于更严格的硬件要求，接受调查的设备中有一半以上无法升级 Windows 11，因此客户留在原地的行为是可理解的。我们也很难想象许多企业会在 Windows 11 推出至少一年左右并有时间稳定下来之前进行大规模升级。</p>
<p>就趋势而言，消费者在 Windows 11 方面领先于企业也就不足为奇了。该操作系统在消费者份额方面正朝着 2.5% 的方向发展，而在企业方面则勉强超过 1%。</p>
<p>Windows 10 的持续主导地位与 AdDuplex(仅包括 Windows 10 和 11 系统)最近进行的调查相呼应，尽管该报告发现 Windows 11 的增长几乎停滞不前。 AdDuplex 的数据基于使用该公司广告平台的应用程序。</p>
<p>“除非企业有令人信服的理由，否则这种情况可能会在未来继续存在，”Decneut 说。</p>
<p>微软看到了一丝曙光，运行报废操作系统的机器明显下降，从 1 月份的 9.75% 降至 6.6%。尽管该数字包括不受支持的 Windows Server 版本，但 XP 和 Windows 7 仍然占很大比例。</p>
<p>微软本身并没有提供太多数据，但 The Register 要求其采取行动，并将在公司做出回应时进行更新。</p>
