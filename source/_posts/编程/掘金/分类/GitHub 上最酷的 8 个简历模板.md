<p><strong><em>07.</em></strong> <strong>best-resume-ever</strong></p>
<p>使用 Vue 和 Less 快速轻松地制作多份精美简历！使用也相当简单：</p>
<p>1. 克隆这个存储库</p>
<pre><code class="copyable">git clone https://github.com/salomonelli/best-resume-ever.git
<span class="copy-code-btn">复制代码</span></code></pre>
<p>2.运行 npm install  等待几秒钟</p>
<p>3. 在 resume/ 目录中编辑自己的简历：编辑个人信息 data.yml ，将默认的 profile-picture id.jpg 替换为你的照片。将图片重命名为 id.jpg 并将其复制到 resume/ 目录中。</p>
<p>4. 使用 npm run dev 预览恢复，在浏览器中打开  <a href="http://localhost:8080/home%E3%80%82%E8%AF%A5%E9%A1%B5%E9%9D%A2%E5%B0%86%E6%98%BE%E7%A4%BA%E4%B8%80%E4%BA%9B%E7%AE%80%E5%8E%86%E9%A2%84%E8%A7%88%EF%BC%8C%E5%96%9C%E6%AC%A2%E5%93%AA%E4%B8%AA%E5%B8%83%E5%B1%80%E7%82%B9%E5%87%BB%E5%B0%B1%E8%A1%8C%E4%BA%86%E3%80%82" target="_blank" rel="nofollow noopener noreferrer">http://localhost:8080/home。该页面将显示一些简历预览，喜欢哪个布局点击就行了。</a></p>
<p><img src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/1422f2fbeb75410195bf101a403325c4~tplv-k3u1fbpfcp-zoom-1.image" alt loading="lazy" referrerpolicy="no-referrer"></p>

<h5 data-id="heading-1">1. Fork 仓库：<a href="https://github.com/Jayin/resume" target="_blank" rel="nofollow noopener noreferrer">github.com/Jayin/resum…</a></h5>
<h5 data-id="heading-2">2. 修改 resume.json ，替换成你的信息</h5>
<h5 data-id="heading-3">3. 提交修改并 push 到你的 GitHub</h5>
