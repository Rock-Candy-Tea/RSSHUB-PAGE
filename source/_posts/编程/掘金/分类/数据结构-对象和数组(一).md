<h2 data-id="heading-0">对象 Object</h2>
<p>对象 允许我们对一些<code>值</code>(也包括其他对象)进行<code>包装分组</code>, 方便进行<code>复杂的数据结构</code>的构建</p>
<p>举个例子: 我们可以用数组来表示一组每天锻炼的运动条目。但条目不仅包含数字或字符串 -- 每个条目都需要存储活动列表和指示是否运动的布尔值。</p>
<p>理想情况下，我们想将这些分组一起放入单个值中，然后将这些分组的值放入一个日志条目数组中。</p>
<p>类型对象的值是属性的任意集合。</p>
<h2 data-id="heading-1">创建对象</h2>
<p>创建对象的一种方法是使用 <code>大括号</code> 作为表达式。</p>
<pre><code class="hljs language-js copyable" lang="js"><span class="hljs-keyword">let</span> day1 = &#123;
  <span class="hljs-attr">flag</span>: <span class="hljs-literal">false</span>,
  <span class="hljs-attr">exercise</span>: [<span class="hljs-string">"running"</span>，<span class="hljs-string">"study"</span>，<span class="hljs-string">"eat"</span>，<span class="hljs-string">"exercise"</span>]
&#125;;
<span class="hljs-built_in">console</span>.log(day1.flag); <span class="hljs-comment">// → false</span>
<span class="hljs-built_in">console</span>.log(day1.flag1);<span class="hljs-comment">// → undefined</span>

day1.flag1 = <span class="hljs-literal">false</span>;
<span class="hljs-built_in">console</span>.log(day1.flag1);<span class="hljs-comment">//→ false</span>
<span class="copy-code-btn">复制代码</span></code></pre>
<p>在大括号内，有一个以逗号分隔的属性列表。每个属性都有一个名称，后跟冒号和值。</p>
<p>当一个对象在多行上写入时，像示例中那样缩进它有助于提高可读性。对于不是有效绑定名称或有效数字的属性名，必须用引号将其括起来,</p>
<p>如下面对象的第二个属性, 属性名中有空格, 就要用引号括起来</p>
<pre><code class="hljs language-js copyable" lang="js"><span class="hljs-keyword">let</span> work = &#123;
  <span class="hljs-attr">work</span>: <span class="hljs-string">'Went to work'</span>,
  <span class="hljs-string">'coded codes'</span>: <span class="hljs-string">'Coded codes'</span>,
&#125;
<span class="copy-code-btn">复制代码</span></code></pre>
<p>这意味着大括号在 JavaScript 中有两种含义:</p>
<ul>
<li>在语句开头，它们用来开始一个语句块。</li>
<li>在任何其他位置，它们描述一个对象。幸运的是，很少用大括号来开始一个使用对象的语句，因此这两者之间的模糊性不是很大的问题</li>
</ul>
<p>如果读取不存在的属性将返回 undefined</p>
<p>可以使用 <code>=</code> 运算符为属性表达式赋值。如果属性已经存在，它将替换属性的值，如果不存在, 将在对象上自动创建对应的属性</p>
