<p>我们的目的是创建一个自己的项目；第一步是安装 node.js。在安装 node.js 时会同时安装 npm 的包管理工具。我们使用 npm 管理工具去安装 vue cli(vue 脚手架)，最后创建自己的项目。</p>
<h1 data-id="heading-1">node.js</h1>
<ul>
<li>什么是 node.js</li>
</ul>
<blockquote>
<p>简单的说 Node.js 就是运行在服务端的 JavaScript。<br>
Node.js 是一个基于 Chrome JavaScript 运行时建立的一个平台。<br>
Node.js 是一个事件驱动 I/O 服务端 JavaScript 环境，基于 Google 的 V8 引擎，V8 引擎执行 Javascript 的速度非常快，性能非常好。</p>
</blockquote>
<ul>
<li>node.js 能做什么</li>
</ul>
<blockquote>
<p>node.js 给我的感觉就是如果你的 web 特别庞大，有强烈的性能需求是，有特别多的静态资源时，那么 node.js 是合适的。因为它有一个特性就是<code>事件驱动I/O</code>。同时，node.js 它也挺适合用来写爬虫的。<br>

</blockquote>

<h2 data-id="heading-3">vue cli (脚手架)</h2>
<ul>
<li>前言<br>
vue cli 相当于是一个可视化的 vue 管理工具，通过这个官方提供的插件，你可以很轻易的创建你的项目；添加官方提供的插件；添加第三方的前端 UI</li>
<li><a href="https://cli.vuejs.org/" target="_blank" rel="nofollow noopener noreferrer">vue cli 官方参考地址</a></li>
<li>安装 vue cli：</li>
</ul>
<pre><code class="copyable">npm install -g @vue/cli   #npm方式安装vue cli
 OR
yarn global add @vue/cli

<span class="copy-code-btn">复制代码</span></code></pre>
<ul>
<li>创建一个项目：</li>
</ul>
<pre><code class="copyable">vue create my-project
 OR

#查看vue版本
