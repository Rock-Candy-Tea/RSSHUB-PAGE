<h3 data-id="heading-0">前端</h3>
<h4 data-id="heading-1">windows11</h4>
<p>可以直接运行在浏览器里的 windows11 系统！无论是界面还是体验都是对 windows11 的高度还原，还可以浏览网页、打开 VS Code 写代码，足以看出作者的用心。</p>
<p><img src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/d688d54ad19c4de49b951cf4ea2f51c9~tplv-k3u1fbpfcp-watermark.image" alt loading="lazy" referrerpolicy="no-referrer"></p>
<p>这个项目乍一看很厉害，但实际用到的技术并不复杂，都是一些主流的前端技术，通过 iframe 内嵌网页的方式实现敲代码工具之类的高级功能。再加上项目结构规范清晰，很适合学前端的小伙伴学习。</p>
<p>我特地录制了一期视频带大家体验该项目、分享设计思路，欢迎大家观看~</p>

<h4 data-id="heading-2">air-conditioner</h4>
<p>之前火爆朋友圈的云空调项目，其实是一个小网页，因此做到了真正意义上的便携哈哈。</p>
<p><img src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/7d8872f75e4f41349ab117a5539af516~tplv-k3u1fbpfcp-watermark.image" alt loading="lazy" referrerpolicy="no-referrer"></p>
<p>别看只有一个页面，这个前端项目用到的主流技术还真不少，而且作者的脑洞和制作的用心程度着实让人佩服，打开小空调后还会触发真实的空调吹风音效，让我感觉真的凉快下来了。</p>
<p>技术栈：前端 React + TypeScript + Sass + Redux + Material UI</p>
<p>难度：⭐️⭐️⭐️</p>
<p>创意：⭐️⭐️⭐️⭐️⭐️</p>
<p>实用：⭐️</p>
<h4 data-id="heading-3">codelf</h4>
<p>帮助程序员解决变量命名问题的神器，可以输入中英文词汇，自动为你推荐合适的英文变量名称。</p>
<p><img src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/cf8f454359884567b1385e8d9aa81d45~tplv-k3u1fbpfcp-watermark.image" alt loading="lazy" referrerpolicy="no-referrer"></p>
<p>但事实上，他只是用翻译插件先将你的中文输入翻译成英文，然后用英文单词去请求 <code>SearchCode</code> 代码搜索服务，从而获取结果。</p>
<p>基于这个原理，我自己也做了一个表情包搜索神器 HahaLF，可以一键从 “全网” 搜索表情包，代码也开源了~</p>
<p><img src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/9143279f30e748e0aad435948d590086~tplv-k3u1fbpfcp-watermark.image" alt loading="lazy" referrerpolicy="no-referrer"></p>
<p>技术栈：前端 React + Gulp + Animate.css + SearchCode</p>
<p>难度：⭐️⭐️</p>
<p>创意：⭐️⭐️⭐️</p>
<p>实用：⭐️⭐️</p>
<h4 data-id="heading-4">bytemd</h4>
<p>如今前端富文本编辑器有很多，但真正好用又免费的寥寥无几。</p>
<p>bytemd 是字节跳动开源的前端富文本编辑器组件，和掘金的写作风格一致、同时支持 Markdown 和傻瓜式操作、支持多个主题、插件式的设计使得开发者可以按需定制富文本编辑器。</p>
<p><img src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/3ba22685c86a4c0493955330d73b2996~tplv-k3u1fbpfcp-watermark.image" alt loading="lazy" referrerpolicy="no-referrer"></p>
<p>最主要的是它同时支持原生以及各种主流前端框架（甚至还有 Svelte），所以很实用，且适合学习。</p>
<p>技术栈：前端 Vue、React、Svelte + TypeScript + Rollup</p>
<p>难度：⭐️⭐️⭐️⭐️</p>
<p>创意：⭐️⭐️</p>
<p>实用：⭐️⭐️⭐️⭐️</p>
<h3 data-id="heading-5">Java</h3>
<p>网上的 Java 项目千篇一律，像商城、博客、各种管理系统，早就没什么新意了，只能算是学习 Java 时的基础项目。</p>
<p>当然，做的大、做的好、做的精致就另当别论了，但现实往往是功能基础、Bug 层出不穷。所以下面分享 3 个不同方向的、非业务类型的 Java 项目。</p>
<h4 data-id="heading-6">QR-Code-generator</h4>
<p>支持多种主流编程语言的二维码生成器类库，只需编写几行代码，就能快速生成二维码图片。</p>
<p>项目基本没有使用第三方类库，而是纯手写实现二维码生成算法，非常纯净，很适合对算法感兴趣的朋友学习。</p>
<p>再结合前端，做一个艺术二维码生成器也不错哦~</p>
<p><img src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/445a6be3a2f04001af5d543d5881f33b~tplv-k3u1fbpfcp-watermark.image" alt loading="lazy" referrerpolicy="no-referrer"></p>
<p>技术栈：Java 基础 + 算法</p>
<p>难度：⭐️⭐️⭐️</p>
<p>创意：⭐️⭐️⭐️</p>
<p>实用：⭐️⭐️⭐️⭐️</p>
<h4 data-id="heading-7">arthas</h4>
<p>阿里开源的 Java 线上诊断利器，可以帮助开发者分析线上 Java 程序的信息和异常、不重启项目直接热修复问题。</p>
<p><img src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/782785ad158d4376977a7b00e5b6124b~tplv-k3u1fbpfcp-watermark.image" alt loading="lazy" referrerpolicy="no-referrer"></p>
<p>虽然我把它放到了本次的创意项目推荐中，但这却是一个很庞大、很复杂、极其实用的项目，用到了很多 Java 的高级知识（比如探针）。</p>
<p>之前也有很多朋友问我什么项目可以打动面试官，这便是其中之一，不写重复的增删改查业务代码，而是解决实际的技术工程问题。</p>
<p>不过确实很难就是了，可以先从会使用它开始，再去了解它的底层原理，最后自己做一个！</p>
<p>技术栈：Java + Maven + Netty</p>
<p>难度：⭐️⭐️⭐️⭐️⭐️</p>
<p>创意：⭐️⭐️⭐️⭐️</p>
<p>实用：⭐️⭐️⭐️⭐️</p>
<h4 data-id="heading-8">mybatis-generator</h4>
<p>知名 Java 数据库访问框架 MyBatis 官方推出的代码生成器，可以帮助开发者快速生成重复的增删改查代码，减少工作量、提高开发效率。</p>
<p><img src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/66d3ab88bded453fbe2387b940b624b1~tplv-k3u1fbpfcp-watermark.image" alt loading="lazy" referrerpolicy="no-referrer"></p>
<p>如今，代码生成在前后端领域都越来越流行，代码生成平台也层出不穷，是类不错的项目。当我们在抱怨重复写代码的时候，不妨试着把重复的代码抽象出来，编写一套逻辑，用命令一键生成代码。</p>
<p>但怎么更优雅、灵活地生成通用代码，还是有很多门道的，因此，官方开源的生成器很值得去学习。</p>
<blockquote>
<p>顺带一提，MyBatis Plus 也相当好用！</p>
</blockquote>
<p>技术栈：Java + Maven + MyBatis + JavaParser</p>
<p>难度：⭐️⭐️⭐️⭐️⭐️</p>
<p>创意：⭐️⭐️⭐️</p>
<p>实用：⭐️⭐️⭐️⭐️⭐️</p>
<h3 data-id="heading-9">Python</h3>
<h4 data-id="heading-10">thefuck</h4>
<p>程序员的日常工作离不开命令行，当不小心输错或忘记命令时，通常会感觉很火大，忍不住就是一声 f*ck！</p>
<p>因此，有个小机灵鬼开发了 thefuck，自动修正命令行错误的工具。当你输错命令时，再输入一次 fuck，就能自动执行正确的命令了！</p>
<p>以下是一些演示：</p>
<p><img src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/9426180b06f54d61a34f507350175ae1~tplv-k3u1fbpfcp-watermark.image" alt loading="lazy" referrerpolicy="no-referrer"></p>
<p>虽然没什么用，但还是感觉很厉害的样子。</p>
<p>看了下代码，99.9% 纯 Python 实现，目录结构和代码规范都很规整，适合有一定编程基础的朋友学习。</p>
<p>技术栈：Python</p>
<p>难度：⭐️⭐️⭐️</p>
<p>创意：⭐️⭐️⭐️⭐️⭐️</p>
<p>实用：⭐️⭐️</p>

<p><img src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/7f8dd58a0fc647afad5dcfa441512745~tplv-k3u1fbpfcp-watermark.image" alt loading="lazy" referrerpolicy="no-referrer"></p>

<h3 data-id="heading-12">其他</h3>
<h4 data-id="heading-13">askgit</h4>
<p>用 SQL 语言来查询 Git 仓库信息的工具。可以让了解基本 SQL 语法的同学从复杂难记的 Git 查询语言中解脱，更快地对代码仓库进行统计分析。可以在线使用：</p>
<p><img src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/cd98b81e906c4b41a07a97945de6a36c~tplv-k3u1fbpfcp-watermark.image" alt="在线使用" loading="lazy" referrerpolicy="no-referrer"></p>
<p>项目用到了数据库，把 Git 信息存储成数据库中的一张表，从而支持 SQL 语句查询。想做这样一款工具，除了能使用一门编程语言外，对 SQL 和 Git 也要有一定的了解。</p>
<p>技术栈：Go + Sqlite</p>
<p>难度：⭐️⭐️⭐️</p>
<p>创意：⭐️⭐️⭐️⭐️⭐️</p>
