<h3 data-id="heading-2">哪里需要用到栈？</h3>
<p>1.所有需要用到后进先出的场景</p>
<p>例：<br>
十进制转二进制：<br>
短除法将所有余数依次入栈，然后将得到的结果从栈顶依次出栈。每次计算余数后将所得余数存到一个栈中，由于栈先进后出的特点，在依次出栈时最先得到的余数被压在栈底，最后才被取到。</p>
<p><img src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/befff6427c8248e2ae90f1a0427d0e83~tplv-k3u1fbpfcp-watermark.image" alt="短除法.jpg" loading="lazy" referrerpolicy="no-referrer"></p>
<p>括号匹配：<br>
由于括号分为左和右，可以把遍历的左括号入栈，然后遍历所有的右括号，然后与其中的所有左括号相匹配，匹配成功则出栈，最后栈为空时即全部匹配完成，如果最后栈内还有剩余元素，则代表还有括号没有被匹配到。<br>
JS函数运行机制：<br>
JS解释器是通过栈来控制函数调用的顺序，最后调用的函数最先执行完。</p>
<h3 data-id="heading-3">leetcode算法题：</h3>
<h4 data-id="heading-4">20. 有效的括号</h4>
<p>给定一个只包括 '('，')'，'&#123;'，'&#125;'，'['，']' 的字符串 s ，判断字符串是否有效。</p>
<p>有效字符串需满足：</p>
<p>左括号必须用相同类型的右括号闭合。
左括号必须以正确的顺序闭合。</p>

<p>解题思路：</p>
<ol>
<li>运用栈来存放数据。</li>
<li>每次检测到左括号时将其入栈，扫描到右括号时将其与当前栈顶括号匹配，如果匹配成功则将栈顶元素出栈进行下一轮匹配，如果类型不匹配则直接返回false。</li>
<li>所有括号便利完毕后查看栈是否为空栈，空栈则返回true，非空则为false。</li>
</ol>
<p>代码实现：</p>
<pre><code class="copyable">/**

                &#125;else&#123;
                    return false
                &#125;
        &#125;
    &#125;
    return stack.length === 0
