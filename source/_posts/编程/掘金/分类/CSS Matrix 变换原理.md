<p>也叫「点积」，<code>AxB=C</code> 的算法如下</p>
<p><img alt="image.png" class="lazyload" src="https://p1-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/be2e911840cf46e1a1602bba6202294f~tplv-k3u1fbpfcp-watermark.image" data-width="800" data-height="600" referrerpolicy="no-referrer"></p>
<p><img alt="image.png" class="lazyload" src="https://p9-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/89d219bb0be64911be0926326897df9a~tplv-k3u1fbpfcp-watermark.image" data-width="800" data-height="600" referrerpolicy="no-referrer"></p>
<p>规则：</p>
<ol>
<li>当 A 的列数等于  B 的行数时，A 与 B 才可以相乘</li>
<li>C 的行数等于 A 的行数，C 的列数等于 B 的列数，即 AC 同高，BC 同宽。</li>
<li>C(m,n) = A(m,1) x B(1,n) + A(m,2) x B(2,n) + ...</li>
</ol>
