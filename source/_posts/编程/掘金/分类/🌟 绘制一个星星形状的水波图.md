<h3 data-id="heading-1">绘制 Star 水波图</h3>
<blockquote>
<p>默认大家有一定的数学基础</p>
</blockquote>
<p>首先，我们需要获取各个顶点的坐标，然后使用 moveTo (M 命令）和 lineTo（L 命令）把五角星的路径计算出来。
<img alt="image.png" class="lazyload" src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/b689cd2bcd5f4168abdd4b23dc3c270c~tplv-k3u1fbpfcp-zoom-1.image" data-width="800" data-height="600" referrerpolicy="no-referrer">
图 3</p>
<p>看图 3 分析可知：O 点对应的是 (x, y) , A 点坐标 <code>(Math.cos(18 * Math.PI / 180) * w + x, -Math.sin(18 * Math.PI / 180) * w + y)</code> ，再以此计算出各个顶点的坐标即可。完整代码如下：</p>
<pre><code class="hljs language-typescript copyable" lang="typescript">shape: <span class="hljs-function">(<span class="hljs-params">x, y, width, height</span>) =></span> &#123;
  <span class="hljs-keyword">const</span> path = [];
    <span class="hljs-keyword">const</span> w = <span class="hljs-built_in">Math</span>.min(width, height) / <span class="hljs-number">2</span>;

    <span class="hljs-keyword">for</span> (<span class="hljs-keyword">let</span> i = <span class="hljs-number">0</span>; i < <span class="hljs-number">5</span>; i++) &#123;
      path.push([
        i === <span class="hljs-number">0</span> ? <span class="hljs-string">'M'</span> : <span class="hljs-string">'L'</span>,
        (<span class="hljs-built_in">Math</span>.cos(((<span class="hljs-number">18</span> + i * <span class="hljs-number">72</span>) * <span class="hljs-built_in">Math</span>.PI) / <span class="hljs-number">180</span>) * w)  + x,
        (-<span class="hljs-built_in">Math</span>.sin(((<span class="hljs-number">18</span> + i * <span class="hljs-number">72</span>) * <span class="hljs-built_in">Math</span>.PI) / <span class="hljs-number">180</span>) * w) + y,
      ]);
      path.push([
        <span class="hljs-string">'L'</span>,
        (<span class="hljs-built_in">Math</span>.cos(((<span class="hljs-number">54</span> + i * <span class="hljs-number">72</span>) * <span class="hljs-built_in">Math</span>.PI) / <span class="hljs-number">180</span>) * w) / <span class="hljs-number">2</span> + x,
        (-<span class="hljs-built_in">Math</span>.sin(((<span class="hljs-number">54</span> + i * <span class="hljs-number">72</span>) * <span class="hljs-built_in">Math</span>.PI) / <span class="hljs-number">180</span>) * w) / <span class="hljs-number">2</span> + y,
      ]);
    &#125;
    path.push([<span class="hljs-string">'Z'</span>]);
    <span class="hljs-keyword">return</span> path;
