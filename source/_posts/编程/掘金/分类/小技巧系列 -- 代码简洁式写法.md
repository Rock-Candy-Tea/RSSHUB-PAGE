h2&#123;font-size:20px&#125;.markdown-body h3&#123;font-size:18px&#125;&#125;</style><h2 data-id="heading-0">前言</h2>
<p>一个大项目的一个文件的代码就成百上千行，如果采用简洁式写法会使代码变得简洁、更加具有可读性。不仅从细节上提高编程效率，还便于后期查找和修改代码。</p>
<h2 data-id="heading-1">代码简洁化</h2>
<h3 data-id="heading-2">维护一个 Max 值</h3>
<p>使用</p>
<pre><code class="hljs language-js copyable" lang="js"><span class="hljs-built_in">Math</span>.max(item , max)
<span class="copy-code-btn">复制代码</span></code></pre>
<p>代替</p>
<pre><code class="hljs language-js copyable" lang="js"><span class="hljs-keyword">if</span>(item > max) max = item
<span class="copy-code-btn">复制代码</span></code></pre>
<h3 data-id="heading-3">使用 <code>A ? B : C</code> 选择表达式进行判断</h3>
<p>使用</p>
<pre><code class="hljs language-js copyable" lang="js">条件 ? 处理方式 <span class="hljs-number">1</span> : 处理方式 <span class="hljs-number">2</span>
<span class="copy-code-btn">复制代码</span></code></pre>
<p>代替</p>
<pre><code class="hljs language-js copyable" lang="js"><span class="hljs-keyword">if</span>(条件) 处理方式 <span class="hljs-number">1</span> ; <span class="hljs-keyword">else</span> 处理方式 <span class="hljs-number">2</span>
<span class="copy-code-btn">复制代码</span></code></pre>
<p>例如需求背景是：在用户还没成功获取个人头像之前我们需要给用户渲染一个默认头像出来，这个时候代码的简洁式写法为：</p>
<pre><code class="hljs language-js copyable" lang="js"><span class="hljs-keyword">const</span> avatarUrl = getAvatar ? imgUrl : <span class="hljs-string">"@/asserts/img/default-avatar.png"</span>
<span class="copy-code-btn">复制代码</span></code></pre>
<p>值得注意的是：要注意一个等号和两个等号</p>
<h3 data-id="heading-4">模板字符串的应用</h3>
<p>使用</p>
<pre><code class="hljs language-js copyable" lang="js"><span class="hljs-built_in">console</span>.log(<span class="hljs-string">`您已完成 &&#123;FinishOrders&#125; 条订单，还有 &&#123;NotFinishOrders&#125; 条订单未完成！`</span>)

<pre><code class="hljs language-js copyable" lang="js"><span class="hljs-built_in">console</span>.log(<span class="hljs-string">"您已完成"</span> + FinishOrders + <span class="hljs-string">"条订单，还有"</span> + NotFinishOrders + <span class="hljs-string">"条订单未完成！"</span>)
<span class="copy-code-btn">复制代码</span></code></pre>
