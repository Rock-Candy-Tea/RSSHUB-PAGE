<p><strong>1、<a href="https://link.juejin.cn/?target=https%3A%2F%2Fgithub.com%2Fkamranahmedse%2Fdeveloper-roadmap" target="_blank" rel="nofollow noopener noreferrer" title="https://github.com/kamranahmedse/developer-roadmap" ref="nofollow noopener noreferrer">Developer-roadmap</a></strong></p>
<p><img src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/232806469c0f4fe781431edba2b7d654~tplv-k3u1fbpfcp-watermark.image" alt="github-repo-1" loading="lazy" referrerpolicy="no-referrer"></p>
<p>初学者如果想学习前端开发，但是不知道从何学起，推荐查看此仓库。</p>
<p>你可以获得有关开发的所有学习路线，笔者在之前的<a href="https://link.juejin.cn/?target=https%3A%2F%2Fmp.weixin.qq.com%2Fs%2FKItesrF9ajWuOGU2SUIK3A" target="_blank" rel="nofollow noopener noreferrer" title="https://mp.weixin.qq.com/s/KItesrF9ajWuOGU2SUIK3A" ref="nofollow noopener noreferrer">文章</a>中对其进行过翻译。</p>
<p><strong>2、<a href="https://link.juejin.cn/?target=https%3A%2F%2Fgithub.com%2FfreeCodeCamp%2FfreeCodeCamp" target="_blank" rel="nofollow noopener noreferrer" title="https://github.com/freeCodeCamp/freeCodeCamp" ref="nofollow noopener noreferrer">FreeCodeCamp</a></strong></p>
<p><img src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/33f75aed8cce44878d311d5d2bacce89~tplv-k3u1fbpfcp-watermark.image" alt="github-repo-2" loading="lazy" referrerpolicy="no-referrer"></p>
<p>一个友好的社区，为 Web 开发人员提供了大量的视频讲座以及课程资源，可以在其中免费学习编程。</p>
<p><strong>3、<a href="https://link.juejin.cn/?target=https%3A%2F%2Fgithub.com%2FEbookFoundation%2Ffree-programming-books" target="_blank" rel="nofollow noopener noreferrer" title="https://github.com/EbookFoundation/free-programming-books" ref="nofollow noopener noreferrer">Ebook foundation</a></strong></p>
<p><img src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/6df1a0bd6dc84f0b814cdec7b0d550e1~tplv-k3u1fbpfcp-watermark.image" alt="github-repo-3" loading="lazy" referrerpolicy="no-referrer"></p>
<p>一个非常大的计算机电子书资源库，所有的资源均为免费。</p>
<p><strong>4、<a href="https://link.juejin.cn/?target=https%3A%2F%2Fgithub.com%2F30-seconds%2F30-seconds-of-code" target="_blank" rel="nofollow noopener noreferrer" title="https://github.com/30-seconds/30-seconds-of-code" ref="nofollow noopener noreferrer">30 seconds of code</a></strong></p>
<p><img src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/d0b27489974a42cfa206e15b4721a3d2~tplv-k3u1fbpfcp-watermark.image" alt="github-repo-4" loading="lazy" referrerpolicy="no-referrer"></p>
<p>这个仓库涵盖了很多非常重要的主题，具有非常好的视觉效果和优秀的代码片段。不仅仅是 JS！</p>
<p><strong>5、<a href="https://link.juejin.cn/?target=https%3A%2F%2Fgithub.com%2Fthedaviddias%2FFront-End-Checklist" target="_blank" rel="nofollow noopener noreferrer" title="https://github.com/thedaviddias/Front-End-Checklist" ref="nofollow noopener noreferrer">The Front-End Checklist</a></strong></p>
<p><img src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/704b5904132547e5b412d07a4f55e950~tplv-k3u1fbpfcp-watermark.image" alt="github-repo-5" loading="lazy" referrerpolicy="no-referrer"></p>
<p>一份前端清单，此仓库包含了网站投入生产之前需要测试的所有元素的详尽列表，例如 SEO、Accessibility 等等。</p>
<p><strong>6、<a href="https://link.juejin.cn/?target=https%3A%2F%2Fgithub.com%2Fleonardomso%2F33-js-concepts" target="_blank" rel="nofollow noopener noreferrer" title="https://github.com/leonardomso/33-js-concepts" ref="nofollow noopener noreferrer">33-js concepts</a></strong></p>
<p><img src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/b838abdaa76847e180243289a1edecb7~tplv-k3u1fbpfcp-watermark.image" alt="github-repo-6" loading="lazy" referrerpolicy="no-referrer"></p>
<p>每个开发人员都应该知道的 33 个 JavaScript 概念，附有配套的视频。</p>
<p><strong>7、<a href="https://link.juejin.cn/?target=https%3A%2F%2Fgithub.com%2Fgetify%2FYou-Dont-Know-JS" target="_blank" rel="nofollow noopener noreferrer" title="https://github.com/getify/You-Dont-Know-JS" ref="nofollow noopener noreferrer">You Don't Know JS</a></strong></p>
<p><img src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/73ec9a6e3b13404c9a2d1f7b737b875a~tplv-k3u1fbpfcp-watermark.image" alt="github-repo-7" loading="lazy" referrerpolicy="no-referrer"></p>
<p>《你不了解的 JavaScript》，一套非常不错的开源电子书，深入探讨 JavaScript 语言核心机制，已有开发者为其翻译了<a href="https://link.juejin.cn/?target=https%3A%2F%2Fgithub.com%2FJoeHetfield%2FYou-Dont-Know-JS" target="_blank" rel="nofollow noopener noreferrer" title="https://github.com/JoeHetfield/You-Dont-Know-JS" ref="nofollow noopener noreferrer">中文版本</a>。</p>
<p><strong>8、<a href="https://link.juejin.cn/?target=https%3A%2F%2Fgithub.com%2FLeCoupa%2Fawesome-cheatsheets" target="_blank" rel="nofollow noopener noreferrer" title="https://github.com/LeCoupa/awesome-cheatsheets" ref="nofollow noopener noreferrer">Awesome-Cheatsheets</a></strong></p>
<p><img src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/1a19f590e35a49f5a98aa0290fd213a2~tplv-k3u1fbpfcp-watermark.image" alt="github-repo-8" loading="lazy" referrerpolicy="no-referrer"></p>
<p>一个适用于流行编程语言、框架和开发工具的备忘单，将开发常用的 API 等内容进行整合，便于检索。</p>
<p><strong>9、<a href="https://link.juejin.cn/?target=https%3A%2F%2Fgithub.com%2Fpublic-apis%2Fpublic-apis" target="_blank" rel="nofollow noopener noreferrer" title="https://github.com/public-apis/public-apis" ref="nofollow noopener noreferrer">Public APIs</a></strong></p>
<p><img src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/9ca5b1250c9f41368b99f99e3c4ce708~tplv-k3u1fbpfcp-watermark.image" alt="github-repo-9" loading="lazy" referrerpolicy="no-referrer"></p>
<p>一个用于软件和 Web 开发的免费 API 集合列表，可以在此处找到许多项目的提供的公共 API。</p>
<p><strong>10、<a href="https://link.juejin.cn/?target=https%3A%2F%2Fgithub.com%2Fjwasham%2Fcoding-interview-university" target="_blank" rel="nofollow noopener noreferrer" title="https://github.com/jwasham/coding-interview-university" ref="nofollow noopener noreferrer">Coding Interview University</a></strong></p>
<p><img src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/d0a5e1375a564da8baa690b338873356~tplv-k3u1fbpfcp-watermark.image" alt="github-repo-10" loading="lazy" referrerpolicy="no-referrer"></p>
<p>该存储库列出了很多的面试项目，这些项目将帮助你应对各种互联网公司的技术面试。</p>
