<h1 data-id="heading-0">题目描述</h1>
<p>题目大意是这样的：
有<code>num</code>台机器，它们会互发信号<code>singal</code>，发送的信号被对方接收，就能得到10点贡献值。</p>
<p>发信号的规则有以下几条：</p>
<ul>
<li>单发</li>
</ul>
<p>机器1给机器2发一个信号<code>17</code>可以表示成<code>[1, 2, 17]</code>，如果机器2没有该信号数据，则机器2必须接收，并且返回<code>1</code>，并且由于机器1发送信号成功，可以增加<code>10</code>个贡献值；如果机器2有该数据，则不接收，返回<code>0</code>。</p>
<ul>
<li>群发</li>
</ul>
<p>机器2给所有机器群发信号<code>29</code>可以表示成<code>[2, 29]</code>，没有该信号的机器都必须选择接收，假设有n台机器接收，那么返回<code>n</code>，并且机器2的贡献值增加<code>10*n</code>。</p>
<ul>
<li>贡献传递</li>
</ul>
<p>机器1给机器2发送一个信号<code>15</code>，机器2接收，那么机器1贡献值<code>+10</code>；</p>
<p>机器2再将信号<code>15</code>发送给机器3，机器3接收，那么机器1/2贡献值都将<code>+10</code>；</p>
<p>机器3再将该信号发送给机器4，机器4接收，那么机器1/2/3贡献值都<code>+10</code>。</p>
<p>注意：发送不存在的信号，说明系统没有，此时发送者自动会增加该信号。</p>
<p>输入：</p>
<ul>
<li>process 执行流程的二维数组，第一个流程是固定的，用来初始化n台机器：<code>[[n], ...]</code></li>
</ul>
<p>输出：</p>
<ul>
<li>result 返回结果是一个一位数组，数组项是每个流程的返回值，第一个返回结果是固定的：<code>[null, ...]</code></li>
</ul>
<h1 data-id="heading-1">用例一</h1>
<pre><code class="copyable">input
[[3], [1, 2, 17], [2, 29], [2]]

output
[null, 1, 2, 20]

[3] 说明有3台机器，返回[null]
[1, 2, 17] 机器1发送17给机器2，1/2都没有17，此时2接收，并且1/2都有信号17，1贡献值+10，返回1
[2, 29] 机器2群发29给所有机器，机器1/3都没有29，都接收，并且1/2/3都有29，2贡献值+20，返回2
[2] 查询机器2的贡献值，返回20

最终贡献值：
机器1：10
机器2：20
机器3：0
