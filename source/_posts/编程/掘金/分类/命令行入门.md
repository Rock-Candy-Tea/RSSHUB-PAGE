h2&#123;font-size:20px&#125;.markdown-body h3&#123;font-size:18px&#125;&#125;</style><h2 data-id="heading-0">一、文件查看</h2>
<p><strong>注意：先查看自己进入的目录是否正确</strong></p>
<p>1.终止命令的快捷键：ctrl + c</p>
<p>2.查看命令：</p>
<ul>
<li>
<p><code>pwd</code> :查看目录的绝对路径</p>
</li>
<li>
<p><code>ls</code> ：查看目录的内容</p>
</li>
<li>
<p><code>ls + 路径</code>：查看指定目录的内容</p>
</li>
</ul>
<p>3.查看文件内容：</p>
<ul>
<li>
<p><code>cat + 路径</code></p>
</li>
<li>
<p><code>head + 路径</code>：默认查看前十行的内容</p>
<p>可以通过 <code>head 路径 -n(数字) </code>查看前n行内容</p>
</li>
<li>
<p><code>tail + 路径</code>：默认查看后十行的内容</p>
<p>可以通过<code>tail 路径 -n(数字) </code>查看后n行内容</p>
</li>
<li>
<p><code>less + 路径</code>：如果文件内容较多，用less比用cat好，可以翻页查看</p>
</li>
</ul>
<p>4.退出查看:按Q(quit)</p>
<h2 data-id="heading-1">二、文件增加</h2>
<p>1.创建文件：</p>
<ul>
<li>
<p><code>touch 1.txt </code>(创建1.txt文件)</p>
</li>
<li>
<p><code>echo hi > 1.txt</code>（在1.txt文件内增加内容hi）</p>
</li>
<li>
<p><code>echo hi hi >> 1.txt </code>（往1.txt内追加内容hihi）</p>
</li>
<li>
<p><code>echo -e "1\n2" >>1.txt</code>  （往1.txt追加两行内容1、2, \n是换行符号）</p>
</li>
<li>
<p><code>touch 1.txt 2.txt</code>（创建多个文件）</p>
</li>
</ul>
<p>2.创建目录：</p>
<ul>
<li>
<p><code>mkdir a</code>（创建目录a）</p>
</li>
<li>
<p><code>mkdir -p  a/b/c/d/e</code>(创建嵌套的层级目录)</p>
</li>
<li>
<p><code>kdir a b </code>（创建多个目录a b）</p>
</li>
</ul>
<p>3.复制文件：</p>
<ul>
<li><code>cp 1.txt 2.txt</code>（把1.txt文件复制到2.txt）</li>
</ul>
<p>4.复制目录：</p>
<ul>
<li><code>cp -r a b</code></li>
</ul>
<h2 data-id="heading-2">三、文件删除</h2>
<p>1.删除文件：</p>
<ul>
<li><code>rm 1.txt</code> (删除1.txt文件)</li>
</ul>
<p>2.删除目录：</p>
<ul>
<li><code>rm -r a  或者rm - rf a</code>（删除目录a）</li>
</ul>
<h2 data-id="heading-3">四、文件修改</h2>
<p>1.清空文件内容：</p>
<ul>
<li><code>echo "" > 1.txt</code></li>
</ul>
<p>2.移动文件：</p>
<ul>
<li><code>mv  1.txt  目录 </code>（移动1.txt到目录）</li>
</ul>
<p>3.重命名文件：</p>
<p><code>mv  1.txt  2.txt</code></p>
<p>4.更新文件的修改时间：</p>
<p>通过<code>ls -l</code> 查看文件的更新时间</p>
<p><code>touch 文件名</code>  （更新文件的修改时间）</p>
<h2 data-id="heading-4">五、帮助</h2>
<p><code>npm i -g tldr</code>或者<code>yarn global add tldr </code></p>
<p>通过 <code>tldr </code>可以查看常用命令</p>
<h2 data-id="heading-5">六、组合命令</h2>
<p>（1）<code>&&</code>：当前一个命令执行成功后才会执行下一条命令</p>
<p>eg：<code>rm 1.txt && echo 成功</code></p>
<p>（2）<code>;</code>：不管前一个命令是否执行成功 都会执行下一条命令</p>
<p>eg：<code>rm 1.txt;echo 执行完毕</code></p>
<h2 data-id="heading-6">七、把命令变成文件</h2>
<ol>
<li>创建一个文件，后缀无所谓，可以没有</li>
</ol>
<p><code>touch run</code>（创建名为run的文件，没有文件后缀）</p>
<ol start="2">
<li><code>code run</code>（用VSCode打开run文件）</li>
</ol>
<p>在VSCode中输入 :</p>
<p><code>#!/usr/bin/env sh</code> （shebang，指定用什么程序运行当前的脚本）</p>
<p><code>mkdir $1 </code>（$1可接受传递过来的参数）</p>
<p><code>cd $1</code></p>
<p><code>touch index.html</code></p>
<p><code>touch style.css</code></p>
<p><code>touch main.js</code></p>
<p><code>echo -e "<!DOCTYPE html>\n<h1>标题<\h1>"  >> index.html</code></p>
<p>保存并关闭文件</p>
<ol start="3">
<li>加上可执行权限（Windows可跳过）</li>
</ol>
<p><code>chmod + x ./run</code></p>
<ol start="4">
<li>
<p><code>./run xxx</code>（必须加上<code>./</code>,创建名为xxx的目录）或者<code>sh run xxx</code>（创建名为xxx的目录）</p>
</li>
<li>
<p>如果把可执行文件的目录放在<code>path</code>就可以用文件名作为命令</p>
</li>
</ol></div>
