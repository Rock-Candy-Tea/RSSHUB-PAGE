<h2 data-id="heading-1">函数作用域与局部变量</h2>
<p><strong>作用域：</strong> 变量的可访问范围。</p>
<p><strong>函数作用域：</strong> <code>function 函数名()&#123; &#125;</code> 大括号 的范围就是 这个函数的作用域。</p>
<p><strong>局部变量：</strong> 声明在某个函数内部的变量，只能在函数内部使用。</p>
<p><strong>全局变量：</strong> 声明在所有函数外部的变量，可以所有地方使用。</p>
<blockquote>
<p>值得注意的是：JS中，变量的作用域有两种，一种是全局作用域，一种局部作用域。</p>
<p>一个变量是全局变量还是局部变量，主要看变量声明的位置:</p>
<p>声明在函数内部，就是这个函数的局部变量。声明在所有函数外部的变量就是全局变量。</p>
</blockquote>
<h2 data-id="heading-2">匿名函数</h2>
<p><strong>作用：主要用于函数表达式以及自执行函数</strong></p>
<p>顾名思义，匿名函数就是没有函数名。我们使用函数表达式创建函数时，= 号右边就是一个匿名函数对象</p>
<p><img src="https://p9-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/9d9673cf9f3f43eb9f6e7f0a6ee00197~tplv-k3u1fbpfcp-watermark.image" alt="1522679811660.png" loading="lazy" referrerpolicy="no-referrer"></p>
<h2 data-id="heading-3">自执行函数</h2>
<p><strong>用法：</strong> 很简单，两对小括号，一对用来 <strong>包裹匿名函数</strong>，一对用来 <strong>调用匿名函数并传参</strong>，例如：</p>
<pre><code class="copyable">(function (usrName) &#123;
  var regDate = '2210年'
  alert(usrName + '我是匿名函数，被自执行啦~~！当前时间：' + regDate)
&#125;)('小白')
<span class="copy-code-btn">复制代码</span></code></pre>
<p><img src="https://p9-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/34f5dc1dfddf43ab92616cc7e9076653~tplv-k3u1fbpfcp-watermark.image" alt="1522708973322.png" loading="lazy" referrerpolicy="no-referrer"></p>
<p>自执行函数 主要用于创建一个 <code>一次性的函数作用域</code> ，在此作用域内声明的变量，不会和其它作用域内的变量冲突或混淆，大多是以匿名函数方式存在，且立即自动执行。</p>
<h2 data-id="heading-4">arguments的使用</h2>
<p><strong>arguments对象 中存储了传递的所有的实参</strong>，例如：</p>
<pre><code class="copyable">function add()&#123;
    var res = arguments[0] + arguments[1]    // 2 + 5
    return res
&#125;

var result = add(2,5)
alert(result)
<span class="copy-code-btn">复制代码</span></code></pre>
<p>JavaScript中，arguments对象是比较特别的一个对象，实际上是当前函数的一个内置属性。也就是说所有函数都内置了一个arguments对象，arguments对象中存储了传递的所有的实参。arguments是一个伪数组，因此及可以进行遍历。</p>
