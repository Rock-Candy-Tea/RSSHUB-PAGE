h2&#123;font-size:20px&#125;.markdown-body h3&#123;font-size:18px&#125;&#125;</style><h2 data-id="heading-0">项目背景</h2>
<p>在最近得一个项目中，需要根据鼠标滚轮控制页面移动，再根据页面移动的距离产生不同的动画，但滑动很快时，有些动画还没结束就播放其他动画，且页面滚动异常。</p>
<h2 data-id="heading-1">解决思路</h2>
<ol>
<li>首先想的是给鼠标事件加个防抖，如下：</li>
</ol>
<pre><code class="hljs language-javascript copyable" lang="javascript">$(<span class="hljs-string">'body'</span>).mousewheel(<span class="hljs-function"><span class="hljs-keyword">function</span>(<span class="hljs-params">ev,delta</span>)</span>&#123;
  <span class="hljs-built_in">clearTimeout</span>(timer); 
  timer = <span class="hljs-built_in">setTimeout</span>(<span class="hljs-function"><span class="hljs-keyword">function</span>(<span class="hljs-params"></span>)</span>&#123;
    <span class="hljs-comment">//事件处理</span>
  &#125;,<span class="hljs-number">200</span>)
&#125;) 
<span class="copy-code-btn">复制代码</span></code></pre>
<p>但只解决了滑动过快时页面滚动异常问题</p>
<ol start="2">
<li>关于动画还没结束就播放其他动画的问题，想了个设置个状态来控制优化。如下：</li>
</ol>
<pre><code class="hljs language-javascript copyable" lang="javascript"><span class="hljs-keyword">var</span> scrolled = <span class="hljs-literal">true</span>
$(<span class="hljs-string">'body'</span>).mousewheel(<span class="hljs-function"><span class="hljs-keyword">function</span>(<span class="hljs-params">ev,delta</span>)</span>&#123;
  <span class="hljs-keyword">if</span>(scrolled)&#123;
    scrolled = <span class="hljs-literal">false</span>
      <span class="hljs-built_in">clearTimeout</span>(timer); 
      timer = <span class="hljs-built_in">setTimeout</span>(<span class="hljs-function"><span class="hljs-keyword">function</span>(<span class="hljs-params"></span>)</span>&#123;
        <span class="hljs-comment">//每次动画结束后才能响应其他动画</span>
        $(<span class="hljs-string">'html,body'</span>).animate(&#123;<span class="hljs-attr">scrollTop</span>: <span class="hljs-number">0</span>&#125;, <span class="hljs-string">'normal'</span>,<span class="hljs-string">"linear"</span>,<span class="hljs-function"><span class="hljs-keyword">function</span>(<span class="hljs-params"></span>)</span>&#123;
          scrolled = <span class="hljs-literal">true</span>
        &#125;);
      &#125;,<span class="hljs-number">200</span>)
  &#125;
&#125;) 
<span class="copy-code-btn">复制代码</span></code></pre>
<p>在每次动画或移动结束后，scrolled置为true，才能进行下一步的响应。</p></div>
