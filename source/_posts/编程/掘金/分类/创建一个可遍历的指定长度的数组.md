<p>关于<strong>可遍历的指定长度的数组</strong>的定义：</p>
<ul>
<li>可遍历：可以通过 <code>map</code>、<code>forEach</code> 等方法来遍历数组元素。</li>
<li>指定长度：数组拥有指定数量的数组元素。</li>
</ul>
<h2 data-id="heading-1">基础</h2>
<p>创建一个指定长度的数组非常简单，直接在新建数组时初始化数组实例的长度。</p>
<pre><code class="hljs language-js copyable" lang="js"><span class="hljs-keyword">new</span> <span class="hljs-built_in">Array</span>(<span class="hljs-number">5</span>) <span class="hljs-comment">/*等价于*/</span> <span class="hljs-built_in">Array</span>().length = <span class="hljs-number">5</span>;

<span class="hljs-comment">/**
(5) [empty × 5]
length: 5
__proto__: Array(0)
**/</span>
<span class="copy-code-btn">复制代码</span></code></pre>
<p>通过数组的构造函数创建数组实例并初始化长度，只是改变数组实例的 <code>length</code> 属性值，而数组所拥有的实际数组元素依然是空的 <code>[empty × 5]</code>。</p>
<pre><code class="hljs language-js copyable" lang="js"><span class="hljs-keyword">const</span> i = <span class="hljs-number">0</span>;
<span class="hljs-built_in">Array</span>(<span class="hljs-number">5</span>).forEach(<span class="hljs-function">(<span class="hljs-params">item,index</span>)=></span>&#123;<span class="hljs-built_in">console</span>.log(i)&#125;); <span class="hljs-comment">//可以发现 i 并没有输出。</span>
<span class="copy-code-btn">复制代码</span></code></pre>
<blockquote>
<p>※ 考虑到 JavaScript 中数组长度具有动态扩增的特性，因此我们没有必要对数组的长度进行初始化，因为很多判断程序都是基于数组<code>length</code> 属性进行判定，这反而会导致一些意外的麻烦。</p>
</blockquote>
<hr>
<p>最简单粗暴的方式，就是创建数组时手动传入所有数组元素。</p>
<pre><code class="hljs language-js copyable" lang="js"><span class="hljs-built_in">Array</span>(<span class="hljs-number">0</span>, <span class="hljs-number">1</span>, <span class="hljs-number">2</span>, <span class="hljs-number">3</span>, <span class="hljs-number">4</span>);
<span class="hljs-comment">/**
(10) [1, 2, 3, 4, 5
0: 0
1: 1
2: 2
3: 3
4: 4
length: 5
**/</span>
<span class="copy-code-btn">复制代码</span></code></pre>
<p>但此种方式过于极端，所以我们还需要找到更通用的方式：</p>
<h2 data-id="heading-2">ES5，Apply方法与类数组</h2>
<pre><code class="hljs language-js copyable" lang="js"><span class="hljs-built_in">Array</span>.apply(<span class="hljs-literal">null</span>, &#123; <span class="hljs-attr">length</span>: <span class="hljs-number">5</span> &#125;).map(<span class="hljs-function"><span class="hljs-keyword">function</span>(<span class="hljs-params"></span>) </span>&#123;
  <span class="hljs-keyword">return</span> <span class="hljs-number">0</span>;
&#125;);
<span class="copy-code-btn">复制代码</span></code></pre>
<p>第一次看这种方式，还是在<a href="https://cn.vuejs.org/v2/guide/render-function.html#%E7%BA%A6%E6%9D%9F" target="_blank" rel="nofollow noopener noreferrer">Vue</a>的官方文档（不得不赞叹下！)</p>
<p>此种方式有两个要素：</p>
<ol>
<li><code>apply</code> 会将作为参数的数组扩展分解为一个一个普通参数进行传递。</li>
<li><code>&#123;length:5&#125;</code> 并不是一个普通对象，而是一个类数组 (Like Array)。</li>
</ol>
<blockquote>
<p>什么是类数组？</p>
<ul>
<li>可以通过索引访问，并拥有 <code>length</code> 属性的对象</li>
<li>不存在 <code>push</code>、<code>pop</code> 等真实数组才具有的方法。</li>
</ul>

<p>可以说这就是 <code>Like Array</code> 与 <code>apply</code> 方法相遇时产生的美丽巧合。也就是这份巧合，让我们可以替代手动并以自动的方式向数组的构造函数中一次传入多个参数。</p>
<blockquote>
<p>一个题外话，因为类数组具有 <code>length</code> 属性并可通过索引进行遍历的特性，所以才可以用 <code>Array.prototype.slice.call(&#123;0:1,length:1&#125;).</code> 将其转换为真实的数组。</p>

<h2 data-id="heading-3">ES6 的扩展运算符</h2>
<pre><code class="hljs language-js copyable" lang="js">[...Array(<span class="hljs-number">5</span>)]; <span class="hljs-comment">//[undefined, undefined, undefined, undefined, undefined]</span>
<span class="copy-code-btn">复制代码</span></code></pre>
<p>🤔? 如果用扩展运算符来转换一个<code>Like Array</code> 会如何？</p>
<pre><code class="hljs language-js copyable" lang="js">[...&#123;<span class="hljs-attr">length</span>:<span class="hljs-number">5</span>&#125;]; <span class="hljs-comment">//VM146:1 Uncaught TypeError: &#123;(intermediate value)&#125; is not iterable</span>
<span class="copy-code-btn">复制代码</span></code></pre>
<p>执行后，发现直接报错了，这是因为ES6的扩展运算符在其内部实现上，做了一些基于不同情况的判断：</p>
<ol>
<li>如果扩展的数组不是一个空数组，直接 <code>[].concat(arr)</code> 合并要扩展的数组。</li>
<li>如果扩展的数组是一个空数组，则基于该数组的 <code>length</code> 长度进行遍历，循环期间如果数组下标所对应的数组元素为空，则填充 <code>undefined</code>。</li>
<li>如果要扩展的对象不是一个数组，则判断该对象是否具有遍历器(Iterator)接口，否则抛错。</li>
</ol>
<p>了解扩展运算符的基本工作原理后，现在我们就可以创建一个迭代器放入到创建的类数组中：</p>
<pre><code class="hljs language-js copyable" lang="js">[...&#123; <span class="hljs-attr">length</span>: <span class="hljs-number">5</span>, [<span class="hljs-built_in">Symbol</span>.iterator]: <span class="hljs-built_in">Array</span>.prototype[<span class="hljs-built_in">Symbol</span>.iterator] &#125;];
<span class="copy-code-btn">复制代码</span></code></pre>
<p>执行后，成功创建了拥有5个 <code>undefined</code> 元素的数组。</p>

<p>其实还有很多其它方式来实现创建可遍历的指定长度的数组，例如：</p>
<pre><code class="hljs language-js copyable" lang="js"><span class="hljs-built_in">Array</span>.from(&#123; <span class="hljs-attr">length</span>: <span class="hljs-number">5</span> &#125;, <span class="hljs-function">() =></span> <span class="hljs-number">0</span>);
<span class="hljs-built_in">Array</span>.apply(<span class="hljs-literal">null</span>, <span class="hljs-built_in">Array</span>(<span class="hljs-number">5</span>)).map(<span class="hljs-function">() =></span> <span class="hljs-number">0</span>);
<span class="copy-code-btn">复制代码</span></code></pre>

除此之外，Vue 的 <code>v-for="i in 10"</code> 指令也可以支持按照指定的次数来渲染内容。</p></div>
