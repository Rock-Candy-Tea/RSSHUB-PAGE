<h2 data-id="heading-0">43. 最小路径和 (minimum-path-sum)</h2>
<h3 data-id="heading-1">标签</h3>
<ul>
<li>动态规划</li>
<li>中等</li>
</ul>
<h3 data-id="heading-2">题目</h3>
<p><a href="https://leetcode-cn.com/problems/minimum-path-sum/" target="_blank" rel="nofollow noopener noreferrer">leetcode 传送门</a></p>
<p>这里不贴题了，leetcode打开就行，题目大意：</p>
<p>给定一个包含非负整数的 m x n 网格，请找出一条从左上角到右下角的路径，使得路径上的<code>数字总和为最小</code>。说明：每次<strong>只能向下或者向右</strong>移动一步。</p>
<h3 data-id="heading-3">相关知识</h3>
<p>从<a href="https://juejin.cn/post/6937240864561299492/" target="_blank">动态规划</a>这篇我们了解到动态规划的基本步骤是下面三步：</p>
<ol>
<li>寻找最优子结构(状态表示)</li>
<li>归纳状态转移方程(状态计算)</li>
<li>边界初始化</li>
</ol>
<p>然后上篇，我们有两个类似题提前看下，这题就太简单了。
<a href="https://juejin.cn/post/6942784495112159262/" target="_blank">前端算法面试必刷题系列[24]</a></p>
<h3 data-id="heading-4">基本步骤</h3>
<p>接下来我们看下面具体问题</p>
<ol>
<li><strong>状态表示</strong>:</li>
</ol>
<ul>
<li><code>dp(i, j)</code> 表示从左上角出发到 <code>(i, j)</code> 位置的<code>最小路径和</code>。</li>
</ul>
<ol start="2">
<li><strong>状态转移方程</strong>:</li>
</ol>
<ul>
<li>由于我们每一步只能<code>向下</code>或<code>向右</code>移动一步，那么其实只有从<code>(i-1, j)</code>或者<code>(i, j-1)</code>这两处走过来，那么转移方程也很简单 <code>dp[i][j] = min(grid[i-1][j], grid[i][j-1])</code>，表示<code>每条路径取较小的来路进行加和</code></li>
</ul>
<ol start="3">
<li><strong>边界初始化</strong>:</li>
</ol>
<ul>
<li>那么下面就是定边界，我们知道最上面一排和最左边一排，只有一种可能来的路径，<code>最顶时</code>，来的路只有<code>从左边1条</code>，<code>最左边</code>来的路只有<code>从上面1条</code>。所以说，<code>i = 0</code>或<code>j = 0</code>时，直接进行来路加和就行，不用比较大小，因为只有一条。</li>
</ul>
<h3 data-id="heading-5">写法实现</h3>
<pre><code class="hljs language-javascript copyable" lang="javascript"><span class="hljs-keyword">var</span> minPathSum = <span class="hljs-function"><span class="hljs-keyword">function</span>(<span class="hljs-params">grid</span>) </span>&#123;
  <span class="hljs-keyword">let</span> [row, col] = [grid.length, grid[<span class="hljs-number">0</span>].length]
  <span class="hljs-comment">// 可以直接在原来的数组中做原地 DP，不用额外空间</span>
  <span class="hljs-comment">// 跟上题类似，把边界先加和，左边只能从上走，上边只能从左走</span>
  <span class="hljs-keyword">for</span> (i = <span class="hljs-number">1</span>; i < row; i++) &#123;
    grid[i][<span class="hljs-number">0</span>] += grid[i-<span class="hljs-number">1</span>][<span class="hljs-number">0</span>]
  &#125;
  <span class="hljs-keyword">for</span> (j = <span class="hljs-number">1</span>; j < col; j++) &#123;
    grid[<span class="hljs-number">0</span>][j] += grid[<span class="hljs-number">0</span>][j-<span class="hljs-number">1</span>]
  &#125;
  <span class="hljs-comment">// 把剩下的非边界位置用递推公式补齐，据题意取来路较小路径的就行</span>
  <span class="hljs-keyword">for</span> (i = <span class="hljs-number">1</span>; i < row; i++) &#123;
    <span class="hljs-keyword">for</span> (j = <span class="hljs-number">1</span>; j < col; j++) &#123;
      grid[i][j] += <span class="hljs-built_in">Math</span>.min(grid[i-<span class="hljs-number">1</span>][j], grid[i][j-<span class="hljs-number">1</span>])
    &#125;
  &#125;
  <span class="hljs-keyword">return</span> grid[row-<span class="hljs-number">1</span>][col-<span class="hljs-number">1</span>]
&#125;;

<span class="hljs-keyword">let</span> grid = [[<span class="hljs-number">1</span>,<span class="hljs-number">3</span>,<span class="hljs-number">1</span>],[<span class="hljs-number">1</span>,<span class="hljs-number">5</span>,<span class="hljs-number">1</span>],[<span class="hljs-number">4</span>,<span class="hljs-number">2</span>,<span class="hljs-number">1</span>]]

<h2 data-id="heading-6">44. 加一 (plus-one)</h2>
<h3 data-id="heading-7">标签</h3>
<ul>
<li>Array</li>
<li>数学</li>
<li>简单</li>
</ul>
<h3 data-id="heading-8">题目</h3>
<p><a href="https://leetcode-cn.com/problems/plus-one/" target="_blank" rel="nofollow noopener noreferrer">leetcode 传送门</a></p>
<p>这里不贴题了，leetcode打开就行，题目大意：</p>
<p>给定一个由 <strong>整数</strong> 组成的 <strong>非空</strong> 数组所表示的非负整数，在该数的基础上<code>加一</code>。</p>
<p>限制:</p>
<ul>
<li>最高位数字存放在数组的首位， 数组中每个元素只存储单个数字。</li>
<li>你可以假设除了整数 0 之外，这个整数不会以零开头。</li>
</ul>
<p>例子</p>
<pre><code class="copyable">输入：digits = [1,2,3]
输出：[1,2,4]
解释：输入数组表示数字 123。
<span class="copy-code-btn">复制代码</span></code></pre>
<h3 data-id="heading-9">相关知识</h3>
<p>这是个简单题，其实就分三种情况</p>
<ul>
<li><strong>末位无进位</strong>，则末位加一即可，比如 <code>13 => 14</code></li>
<li><strong>末位有进位</strong>，在中间位置进位停止，整体位数不变，需要找到进位的标志:<code>当前位 %10 == 0</code>，则前一位加 1，直到不为 0 为止，比如 <code>199 => 200</code></li>
<li><strong>末位有进位</strong>，并且一直进位到最前方导致结果多出一位，比如 <code>99 => 100</code></li>
</ul>
<h3 data-id="heading-10">基本步骤</h3>
<ol>
<li>数组倒序遍历，若需要进位，先行 +1</li>
<li>判断当前数字是否大于 9， 大于代表需要进位，否则不进位。</li>
<li>遍历结束，还存在进位，则扩充数组，首位补充 1。</li>
</ol>
<h3 data-id="heading-11">写法实现</h3>
<pre><code class="hljs language-javascript copyable" lang="javascript"><span class="hljs-keyword">var</span> plusOne = <span class="hljs-function"><span class="hljs-keyword">function</span>(<span class="hljs-params">digits</span>) </span>&#123;
  <span class="hljs-keyword">const</span> len = digits.length
  <span class="hljs-comment">// 倒序遍历</span>
  <span class="hljs-keyword">for</span> (<span class="hljs-keyword">let</span> i = len - <span class="hljs-number">1</span>; i >= <span class="hljs-number">0</span>; i--) &#123;
    digits[i]++;
    <span class="hljs-comment">// 加一后看 % 10 是否为 0 ，为0说明有进位</span>
    digits[i] = digits[i] % <span class="hljs-number">10</span>;
    <span class="hljs-keyword">if</span> (digits[i] !== <span class="hljs-number">0</span>) &#123;
      <span class="hljs-comment">// 直接输出就行</span>
      <span class="hljs-keyword">return</span> digits;
    &#125;
  &#125;
  <span class="hljs-comment">// 如果遍历到最后都需要进位，说明是要扩充数组</span>
  digits = <span class="hljs-keyword">new</span> <span class="hljs-built_in">Array</span>(len + <span class="hljs-number">1</span>).fill(<span class="hljs-number">0</span>)
  digits[<span class="hljs-number">0</span>] = <span class="hljs-number">1</span>;
  <span class="hljs-keyword">return</span> digits;
&#125;;
