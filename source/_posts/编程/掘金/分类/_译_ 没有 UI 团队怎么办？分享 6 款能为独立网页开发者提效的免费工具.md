<h1 data-id="heading-0">没有 UI 团队怎么办？分享 6 款能为独立网页开发者提效的免费工具</h1>
<p><img alt="头图" class="lazyload" src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/6a866a0a32c849c98ca999ea81d46dce~tplv-k3u1fbpfcp-zoom-1.image" data-width="800" data-height="600" referrerpolicy="no-referrer"></p>
<p>大家好，希望大伙们最近一切都好！在这篇短文中，我将向大家介绍 6 款在搭建网站时为我节省了许多时间的免费工具。希望这些工具也能够帮到你们。</p>
<h2 data-id="heading-1">1. <a href="https://type-scale.com/" target="_blank" rel="nofollow noopener noreferrer">Type Scale</a></h2>
<p>你是否经常为了如何设定页面元素的字体大小而倍感纠结？如果你有这种烦恼，这款软件绝对能帮到你。</p>
<p><img alt class="lazyload" src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/14e5605e9ce44a2c8707dc50c3b271ca~tplv-k3u1fbpfcp-zoom-1.image" data-width="800" data-height="600" referrerpolicy="no-referrer"></p>
<p>只需在网页中输入你的基准字体大小，它就会展示出不同的元素（<code>H1</code>、<code>H2</code>、<code>H3</code> 等等）的字体尺寸。<br>
此外，你还可以从谷歌字体库中挑选不同的字体，并尝试不同的字体大小。</p>
<p>这款工具最棒的点在于，在选择你想要的效果后，它会自动生成对应的 CSS 样式代码，你可以直接把它拷贝到 CSS 文件中。同时，它还会自动导入你所选择的字体库并为 html 设置好 <code>font-family</code> 属性，这样你就不用花时间在不同的文件中定义字体大小了！</p>
<p><img alt class="lazyload" src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/c40723cfa137485e9793f445ace0c364~tplv-k3u1fbpfcp-zoom-1.image" data-width="800" data-height="600" referrerpolicy="no-referrer"></p>
<h2 data-id="heading-2">2. <a href="https://www.happyhues.co/" target="_blank" rel="nofollow noopener noreferrer">Happy Hues</a></h2>
<p>如果你不知道怎样为你的网站进行配色设计的话，这款工具会是你学习的起点。</p>
<p><img alt class="lazyload" src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/de6a56367d354f39b9fbe2ba64a5b2e4~tplv-k3u1fbpfcp-zoom-1.image" data-width="800" data-height="600" referrerpolicy="no-referrer"></p>
<p>它不仅展示了多种绚丽的配色方案，还为不同的配色设计了真实的范例，教你如何把不同的颜色运用到你的项目设计中。</p>
<p>这款工具告诉大家应该如何为不同的元素选择对应的颜色，同时还展示了如何为网页插图选择配色方案，让插图和网页整体的风格更加匹配。</p>
<h2 data-id="heading-3">3. <a href="https://fontjoy.com/" target="_blank" rel="nofollow noopener noreferrer">Fontjoy</a></h2>
<p><img alt class="lazyload" src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/aa55eb287a8f49f5b7a4be6f6f63e514~tplv-k3u1fbpfcp-zoom-1.image" data-width="800" data-height="600" referrerpolicy="no-referrer"></p>
<p>这款工具可以快速生成对应不同元素的一套字体，供你在网页设计中应用。你可以选择你想要的对比度，单击生成就可以得到想要的那套字体了。</p>
<h2 data-id="heading-4">4. <a href="https://csspeeper.com/" target="_blank" rel="nofollow noopener noreferrer">CSS Peeper</a></h2>
<p><img alt class="lazyload" src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/91796fbc8f1d4f05a3b77cf6b62b685d~tplv-k3u1fbpfcp-zoom-1.image" data-width="800" data-height="600" referrerpolicy="no-referrer"></p>
<p>CSS Peeper 是一款 Chrome 拓展程序，有着优秀的用户体验。它能帮助你快速检查不同网页的样式信息，展示网站所使用的颜色和字体。此外，如果你对网站上的某一个特定元素感兴趣，它能够帮助你快速检查出创建对应元素所需的所有 CSS 样式。</p>
<h2 data-id="heading-5">5. <a href="https://getavataaars.com/" target="_blank" rel="nofollow noopener noreferrer">avataaars generator</a></h2>
<p><img alt="Alt Text" class="lazyload" src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/ffaa3d2c6a0743f2a163c4a3dcca4948~tplv-k3u1fbpfcp-zoom-1.image" data-width="800" data-height="600" referrerpolicy="no-referrer"></p>
<p>这款工具可以帮助你生成有趣的拟人化头像。有非常多不同的样式和风格可供你选择，在完成设计后，你还可以快速地将头像素材以多种格式导出。工具现在支持 PNG 与 SVG，还支持直接生成 React 代码或者图片链接。最重要的是，这款工具是完全免费的！</p>
<h2 data-id="heading-6">6. <a href="https://shadows.brumm.af/" target="_blank" rel="nofollow noopener noreferrer">brumm.af</a></h2>
<p><img alt="Alt Text" class="lazyload" src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/c5895b039586496483ded4710686ed7d~tplv-k3u1fbpfcp-zoom-1.image" data-width="800" data-height="600" referrerpolicy="no-referrer"></p>
<p>通过使用这款工具，你可以创造出惊人的 <code>box-shadow</code> 阴影效果。你可以添加多个层级的阴影，并且对每个图层进行调整，从而创造出最自然、最拟真的阴影效果。</p>
