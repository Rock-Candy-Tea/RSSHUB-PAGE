<p>根据微信公众平台的要求，<strong>在开发微信公众号的过程中，必须通过绑定外网域名，才能够拿到微信服务器发出的回调信息</strong>。</p>
<p>已有域名和云服务器的童鞋，通过如下配置，可直接调试本地开发代码，省去 本地开发->上传到服务器->测试发现问题->本地修改->上传服务器 。。。 的一系列繁琐步骤。</p>
<p>如下：</p>
<p>公众号开发设置好自己的域名，验证域名成功；</p>
<p>本地跑起起项目服务（我用的mac自带python 可 <a href="https://link.juejin.cn/?target=https%3A%2F%2Fblog.csdn.net%2FRedaTao%2Farticle%2Fdetails%2F85106216" target="_blank" rel="nofollow noopener noreferrer" title="https://blog.csdn.net/RedaTao/article/details/85106216" ref="nofollow noopener noreferrer">点击这里查看</a>）</p>
<p>调试地址为： <a href="https://link.juejin.cn/?target=http%3A%2F%2Flocalhost%3A8099" target="_blank" rel="nofollow noopener noreferrer" title="http://localhost:8099" ref="nofollow noopener noreferrer">http://localhost:8099</a>;</p>
<p>假设你的域名为 <a href="https://link.juejin.cn/?target=http%3A%2F%2Ftest.com" target="_blank" rel="nofollow noopener noreferrer" title="http://test.com" ref="nofollow noopener noreferrer">test.com</a>;</p>
<h3 data-id="heading-0">1.修改本地host：</h3>
<p>将 <a href="https://link.juejin.cn/?target=http%3A%2F%2Ftest.com" target="_blank" rel="nofollow noopener noreferrer" title="http://test.com" ref="nofollow noopener noreferrer">test.com</a> 映射到 127.0.0.1 。修改/etc/hosts文件（mac系统）;</p>
<h3 data-id="heading-1">2.设置本地代理转发</h3>
<p>因为公众号开发必须80端口，所以需要将80端口转发到如上本地项目的 8099端口；</p>
<p>（可使用nginx 反向代理）</p>
<ol>
<li>
<p>安装nginx；nginx 默认8080端口</p>
</li>
<li>
<p>配置 /usr/local/etc/nginx/nginx.conf </p>
</li>
</ol>
<pre><code class="hljs language-bash copyable" lang="bash">server &#123;
        listen       80;  // 监听80端口
        server_name  localhost;

        <span class="hljs-comment">#charset koi8-r;</span>

        <span class="hljs-comment">#access_log  logs/host.access.log  main;</span>

        location / &#123;
            root   html;
            index  index.html index.htm;
            proxy_pass http://127.0.0.1:8099; // 将80端口转发到 8099 端口
        &#125;
&#125;
<span class="copy-code-btn">复制代码</span></code></pre>
<ol start="3">
<li>此时，输入 localhost:80  应该出现本地项目中的页面；</li>
</ol>
<p>   并且 输入 <a href="https://link.juejin.cn/?target=http%3A%2F%2Ftest.com" target="_blank" rel="nofollow noopener noreferrer" title="http://test.com" ref="nofollow noopener noreferrer">test.com</a>   也出现本地项目中的页面</p>
<p>   访问的走向是：test.com:80 => 127.0.0.1:80 => 127.0.0.1:8099，也就是你的本地代码调试服务。</p>
<p>3、如果需要用真机调试，可以将手机与电脑连接同一个局域网络，然后在电脑开启一个抓包代理（windows用Fiddler2即可，mac可以用Charles），再将手机的wifi代理设置为你的电脑ip及8888端口。这样，在手机上访问网络时，将通过抓包代理到你的电脑上。微信访问 <a href="https://link.juejin.cn/?target=http%3A%2F%2Ftest.com" target="_blank" rel="nofollow noopener noreferrer" title="http://test.com" ref="nofollow noopener noreferrer">test.com</a> 也将最终访问到你的本地代码调试服务。</p></div>
