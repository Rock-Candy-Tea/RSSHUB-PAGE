h2&#123;font-size:20px&#125;.markdown-body h3&#123;font-size:18px&#125;&#125;</style><h3 data-id="heading-0">登录服务器</h3>
<p>方法一、</p>
<pre><code class="hljs language-Linux copyable" lang="Linux">ssh root@IP

// 随后输入密码
<span class="copy-code-btn">复制代码</span></code></pre>
<p>方法二、</p>
<pre><code class="copyable">sudo ssh 云服务器ip地址复制代码
// sudo：赋予权限
<span class="copy-code-btn">复制代码</span></code></pre>
<p>首先输入开机密码，然后输入云服务实例的密码</p>
<h3 data-id="heading-1">如果想要不输入密码直接通过ssh root@的方式登录</h3>
<p>将本地的公钥复制配置到服务器的～/.ssh/authorized_keys文件中</p>
<p>则可以直接通过ssh root@的方式免密码登录</p>
<h3 data-id="heading-2">不需要知道服务器IP进行登录</h3>
<pre><code class="copyable">vim ~/.ssh/config
// 配置
Host hostname #设置的别名
HostName #IP
User root
IdentitiesOnly yes
<span class="copy-code-btn">复制代码</span></code></pre>
<h3 data-id="heading-3">安装nginx和epel-release仓库</h3>
<pre><code class="copyable">yum install epel-release -y

yum install nginx -y
<span class="copy-code-btn">复制代码</span></code></pre>
<p>启动nginx和epel仓库</p>
<pre><code class="copyable">启动nginx：service nginx start

重启nginx：nginx -s reload

关闭nginx：service nginx stop
<span class="copy-code-btn">复制代码</span></code></pre>
<pre><code class="copyable">开启防火墙：systemctl start nginx
<span class="copy-code-btn">复制代码</span></code></pre>

<h3 data-id="heading-7">连接远程仓库</h3>
<pre><code class="copyable">// 初始化

git status　　　　　　　　　　查看工作目录的状态

git push origin master 　　 将本地仓库的文件push到远程仓库(若 push 不成功，可加 -f 进行强推操作)

<span class="copy-code-btn">复制代码</span></code></pre>
<h4 data-id="heading-8">问题一</h4>
<p>在连接远程仓库的时候（git remote add origin <a href="https://link.juejin.cn/?target=mailto%3Agit%40github.com" target="_blank" title="mailto:git@github.com" ref="nofollow noopener noreferrer">git@github.com</a>:yourName/repositoryname.git）</p>
<p>出现ssh: connect to host githab.com port 22: Operation timed out</p>
<p>解决步骤：
22接口不能连接，则尝试一下443接口</p>
<p>首先使用一下命令，尝试一下链接</p>
<pre><code class="copyable">//会出现以上问题
ssh -T git@github.com

// 尝试使用443接口
ssh -T -p 443 git@ssh.github.com
<span class="copy-code-btn">复制代码</span></code></pre>
<p>修改配置文件，默认走443端口</p>
<pre><code class="copyable">// 配置.ssh/config

Host github.com
User 575567292@qq.com
Hostname ssh.github.com
PreferredAuthentications publickey
IdentityFile ~/.ssh/id_rsa
Port 443 // 使用443接口
<span class="copy-code-btn">复制代码</span></code></pre>
<p>至此：已经和远程代码库链接起来</p>
