<h1 data-id="heading-0">611. 有效三角形的个数</h1>
<h3 data-id="heading-1">Title Description</h3>
<p>Given an integer array nums, return the number of triplets chosen from the array that can make triangles if we take them as side lengths of a triangle.</p>
<p> </p>
<p>Example 1:</p>
<pre><code class="copyable">Input: nums = [2,2,3,4]
Output: 3
Explanation: Valid combinations are: 
2,3,4 (using the first 2)
2,3,4 (using the second 2)
2,2,3
<span class="copy-code-btn">复制代码</span></code></pre>
<p>Example 2:</p>
<pre><code class="copyable">Input: nums = [4,2,3,4]
Output: 4
<span class="copy-code-btn">复制代码</span></code></pre>
<p>Constraints:</p>
<p>1 <= nums.length <= 1000
0 <= nums[i] <= 1000</p>
<h3 data-id="heading-2">题目描述</h3>
<p>给定一个包含非负整数的数组，你的任务是统计其中可以组成三角形三条边的三元组个数。</p>
<p>示例 1:</p>
<pre><code class="copyable">输入: [2,2,3,4]
输出: 3
解释:
有效的组合是: 
2,3,4 (使用第一个 2)
2,3,4 (使用第二个 2)
2,2,3
<span class="copy-code-btn">复制代码</span></code></pre>
<p>注意:</p>
<p>数组长度不超过1000。
数组里整数的范围为 [0, 1000]。</p>
<h3 data-id="heading-3">解题思路1</h3>
<h4 data-id="heading-4">排序 + 双指针</h4>
<ul>
<li>
<p>我们使用一重循环枚举 i。当 i 固定时，我们使用双指针同时维护 j 和 k，它们的初始值均为 i；</p>
</li>
<li>
<p>我们每一次将 j 向右移动一个位置，即 j←j+1，并尝试不断向右移动 k，使得 k 是最大的满足nums[k]<nums[i]+nums[j] 的下标。我们将 max(k−j,0) 累加入答案。</p>
</li>
</ul>
<h3 data-id="heading-5">代码</h3>
<pre><code class="copyable">class Solution &#123;
public:
    int triangleNumber(vector<int>& nums) &#123;

        sort(nums.begin(),nums.end());
        
        int ans=0;

        for(int i=2;i<nums.size();++i)&#123;

            int k=0;

            for(int j=i-1;k<j;--j)&#123;
                
                while(k < j&&nums[j]+nums[k]<=nums[i])++k;
                ans+=j-k;
            &#125;

        &#125;
        return ans;
    &#125;
&#125;;

<h3 data-id="heading-6">解题思路2</h3>
<h4 data-id="heading-7">排序 + 二分查找</h4>
<ol>
<li>先排序，再二分查找</li>
<li>查找时，先固定前两个数，然后查第三个数，如果前两个数是a和b，且a<=b，那么第三个数的范围就是：【b,a+b-1】.</li>
<li>用二分法，在nums剩下范围内寻找：大于等于lower的最小值，以及小于等于upper的最大值。</li>
<li>由于二分法特性，在向右查找小于等于upper的最大值时，可能找到的是大于upper的第一个值，最后要比较一下确认是否左退一步。</li>
</ol>
<h4 data-id="heading-8">代码</h4>
<pre><code class="copyable">class Solution &#123;
    public int triangleNumber(int[] nums) &#123;
        int n = nums.length;
        if(n<3) return 0;
        Arrays.sort(nums);
        int ans = 0;
        for(int i = 0; i<n-2;i++)&#123;
            for(int j = i+1; j<n-1; j++)&#123;
                int sum2 = nums[i]+nums[j];
                if(nums[j+1]>=sum2)continue;
                int lower = nums[j+1];
                int upper = sum2-1;
                // 在:j+1~n-1 之间寻找大于等于lower的最小值，返回坐标。
                int minId = searchMin(nums, j+1, n-1, lower);
                // 在:j+1~n-1 之间寻找小于等于upper的最大值，返回坐标。
                int maxId = searchMax(nums, j+1, n-1, upper);
               // System.out.println("i="+i+",j="+j+", lower="+lower+", upper="+upper+",minId="+minId+", maxId="+maxId);
                ans+=1 + maxId - minId;
            &#125;
        &#125;
        return ans;
    &#125;
    int searchMin(int[] nums, int left, int right, int target)&#123;
        while(left<right)&#123;
            int m = (left+right)>>>1;
            if(nums[m]>=target) right=m;
            else left=m+1;
        &#125;
        return left;
    &#125;
    int searchMax(int[] nums, int left, int right, int target)&#123;
        while(left<right)&#123;
            int m = (left+right)>>>1;
            if(nums[m]<=target) left=m+1;
            else right=m;
        &#125;
        if(nums[left]>target)return left-1;
        return left;
    &#125;
