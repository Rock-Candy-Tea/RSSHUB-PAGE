<h2 data-id="heading-1">proto与prototype</h2>
<p>从上一篇的原型继续讲起</p>
<p>所有的函数都会有<code>prototype</code>原型属性</p>
<p>每个对象都会有一个属性<code>__proto__</code>指向构造函数的 <code>prototype</code> 原型对象 ，因为对象的__proto__和它的构造函数的prototype在都是引用的同一个内存地址，因此他们是完全相等的</p>
<pre><code class="hljs language-js copyable" lang="js"><span class="hljs-function"><span class="hljs-keyword">function</span> <span class="hljs-title">Ninja</span>(<span class="hljs-params">name, age</span>)</span>&#123;  
    <span class="hljs-built_in">this</span>.name = name;
    <span class="hljs-built_in">this</span>.age = age;
&#125;

Ninja.prototype.run = <span class="hljs-function"><span class="hljs-keyword">function</span> (<span class="hljs-params"></span>) </span>&#123;
    <span class="hljs-built_in">console</span>.log(<span class="hljs-string">'run'</span>)
&#125;

<span class="hljs-keyword">let</span> Naruto = <span class="hljs-keyword">new</span> Ninja(<span class="hljs-string">'Naruto'</span>, <span class="hljs-number">18</span>)

<p><code>__proto__</code>又叫隐式原型，   <code>prototype</code>又叫显式原型</p>
<p>我们可以通过<code>__proto__</code>去访问（使用）一个对象的构造函数的 <code>prototype</code>原型里面的方法</p>
<h2 data-id="heading-2">原型链</h2>
<blockquote>
<p>当访问一个对象的某个方法时，首先会先在这个对象本身上查找有没有这个方法，如果没有找到，就会去它的__proto__(构造函数的prototype)上查找，如果依旧没有找到就会在构造函数的prototype的__proto__中查找，这样一层一层向上查找就会形成一个链式结构，我们称为<code>原型链</code>。</p>
</blockquote>
<p>原型链的最顶层就是<code>Object.prototype.__proto__ </code>（null）</p>

<h3 data-id="heading-3">思考</h3>
<p>1.<code>Object,Array,Boolean,</code> 这些内置函数对象的__proto__是什么呢</p>
<p>答案是<code>Function.prototype</code> ,这是因为这些构造函数也是由Function去构造出来的</p>
<pre><code class="hljs language-js copyable" lang="js"><span class="hljs-built_in">console</span>.log(<span class="hljs-built_in">Object</span>.__proto__ === <span class="hljs-built_in">Function</span>.prototype)    <span class="hljs-comment">// true</span>
<span class="hljs-built_in">console</span>.log(<span class="hljs-built_in">Array</span>.__proto__ === <span class="hljs-built_in">Function</span>.prototype)     <span class="hljs-comment">// true</span>
<span class="hljs-built_in">console</span>.log(<span class="hljs-built_in">String</span>.__proto__ === <span class="hljs-built_in">Function</span>.prototype)    <span class="hljs-comment">// true</span>

<p>2.<code>Function.__proto__</code>等于什么
答案是<code>Function.prototype</code>， 上文说到对象的__proto__指向这个对象的构造函数的
prototype，那Function是个函数对象，于是便有了</p>
