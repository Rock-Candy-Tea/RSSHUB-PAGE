<h2 data-id="heading-0">产品简介</h2>
<p><strong>汉得开源组件库Choerodon U</strong>I（缩写 C7N UI），拥有开箱即用的高质量 React 组件， 适用于 React 框架项目，符合低代码理念，专为中后台企业项目提高开发效率而生。</p>
<p><strong>2021年07月21日</strong>，汉得开源组件库 Choerodon UI 发布 <strong>1.4.2</strong>，本次更新主要优化动态筛选条样式交互，多组件性能优化等；DataSet等组件也都进行了不同程度的新增、修改和优化，欢迎各位更新体验。</p>
<p><strong>·</strong> <strong>发布版本：1.4.2</strong></p>
<p><strong>·</strong> <strong>发布时间：2021年07月21日</strong></p>
<p><strong>·</strong> <strong>更新范围：DataSet、Table</strong>**、Tooltip、Form、PerformanceTable、Lov 等 Pro 组件。**</p>
<h2 data-id="heading-1">Choerodon UI组件库 1.4.2 版本 特性介绍</h2>
<p>2021年7月28日，Choerodon UI组件库发布了 1.4.2 版本，主要特性有以下内容</p>
<p>■<strong>优化动态筛选条样式交互，多组件性能优化，Pro Table 性能显著提升;</strong></p>
<p><img src="https://p9-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/27398d7f89714ac1a3b8de6a1ee4ddc5~tplv-k3u1fbpfcp-watermark.image" alt loading="lazy" referrerpolicy="no-referrer"></p>
<p>■ <strong>DataSet</strong></p>
<ul>
<li>
<p>新增 combineSort 属性支持组合列排序传参；</p>
</li>
<li>
<p>新增 selectionStrategy 属性和相关方法支持树形选择记录规则；</p>
</li>
<li>
<p>扩展 dataToJSON 属性，Field 新增 json 类型;</p>
</li>
<li>
<p>优化 addField 方法将自动创建已有 Record 对应的 Field；</p>
</li>
<li>
<p>dataToJSON 属性的 all 值将校验所有记录。</p>
</li>
</ul>
<p>■<strong>Tooltip 新增单例模式</strong></p>
<p>■<strong>PerformanceTable</strong></p>
<ul>
<li>
<p>修复滚动条及点击事件问题；</p>
</li>
<li>
<p>新增查询条 queryBar，工具栏 toolbar。</p>
</li>
</ul>
<p>■<strong>Form 修复嵌套表单的 dataSet 属性不起作用的问题</strong></p>
<h2 data-id="heading-2">新增</h2>
<ul>
<li>
<p>configure:新增tableVirtualCell, formatter, timeZone 属性。</p>
</li>
<li>
<p>DataSet:新增selectionStrategy 属性, selectionStrategy,treeSelected值,treeSelect, treeUnSelect 方法。</p>
</li>
<li>
<p>Lov: 新增 showCheckedStrategy 属性。</p>
</li>
<li>
<p>DatePicker: 新增 timeZone 属性。</p>
</li>
<li>
<p>Tooltip: 新增单例模式, 添加 show 和 hide 静态方法。</p>
</li>
<li>
<p>DataSet: dataToJSON 新增 dirty-field, dirty-field-self 类型。</p>
</li>
<li>
<p>DataSet.Field: 新增 json 类型。</p>
</li>
<li>
<p>DataSet: 新增 combineSort 属性支持组合列排序传参。</p>
</li>
<li>
<p>Select: selectAllButton 新增钩子类型用于支持自定义按钮。</p>
</li>
<li>
<p>PerformanceTable: 新增 queryBar, toolbar 属性，用于支持查询条及工具栏。</p>
</li>
<li>
<p>Table: 新增 showHeader 属性。</p>
</li>
</ul>
<h2 data-id="heading-3">优化</h2>
<ul>
<li>
<p>Modal: 支持穿越同域 iframe。</p>
</li>
<li>
<p>DataSet: dataToJSON 属性的 all 值将校验所有记录。</p>
</li>
<li>
<p>DataSet: 优化 addField 方法将自动创建已有 Record 对应的 Field。</p>
</li>
<li>
<p>Radio：优化 label 内容溢出提示。</p>
</li>
<li>
<p>Table：优化性能。</p>
</li>
<li>
<p>Table：优化动态搜索条样式交互。</p>
</li>
<li>
<p>NumberField：step 兼容移动端事件。</p>
</li>
<li>
<p>Progress：优化动画性能。</p>
</li>
<li>
<p>Table.Column：优化货币类型的列默认靠右对齐。</p>
</li>
<li>
<p>Output：优化数字货币显示, 优化空值显示。</p>
</li>
<li>
<p>DataSet.Record: 在没有 field 的情况下， addField 方法会对该 field 已有值进行处理。</p>
</li>
<li>
<p>NumberField: 移除 suffix，prefix 属性支持。</p>
</li>
</ul>
<h2 data-id="heading-4">修复</h2>
<ul>
<li>
<p>Table: 修正 maxHeight 和 minHeight 的计算逻辑。</p>
</li>
<li>
<p>Pagination: 修复最大分页没有设置时，分页选项有超过默认最大100分页无法选择的问题。</p>
</li>
<li>
<p>IconPicker: 修复当点击分页按钮并按钮成禁用状态时，弹窗无法关闭的问题。</p>
</li>
<li>
<p>Table: 修复在孔雀蓝主题下，当Column的editor为function的时候，进入编辑态点击退出后会导致下次进入编辑态组件宽度不正确的问题</p>
</li>
<li>
<p>Table: 修复行内编辑时无法取消编辑的问题。</p>
</li>
<li>
<p>PerformanceTable: 修复滚动条事件。</p>
</li>
<li>
<p>PerformanceTable: 修复表格内点击无效问题。</p>
</li>
<li>
<p>Form: 修复嵌套表单的 dataSet 属性不起作用的问题。</p>
</li>
<li>
<p>Select: 修复自定义弹窗可能无法关闭的问题。</p>
</li>
<li>
<p>TextField: 修复禁用状态下无法显示渲染器渲染的值。</p>
</li>
<li>
<p>Table: 修复双击调整列宽名字中带有.的列宽时报错。</p>
</li>
</ul>
