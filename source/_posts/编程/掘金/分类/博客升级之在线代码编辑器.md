<p>最近打算对个人博客进行优化，主要涉及两个方面，文章编辑器和代码展示分享模块，因此就需要考虑选择什么样的在线编辑器，顺道总结一下在线代码编辑及分享一些可能用得上的优秀代码编辑器。</p>
<p>在线代码编辑器是一个部署在远程服务器上，可通过浏览器访问的在线工具，即可以进行在线编码。低代码平台是最近一两年比较火的概念，此类产品平台少不了在线代码编辑器。在线代码编辑器主要分为两类，一类是具有类似于文本编辑器的基本功能，一类则类似于完整的 IDE。</p>
<p>在本文中，将分别介绍这两种类型。</p>
<p>一些在线代码编辑器专注于一种语言甚至一个框架。例如，有些产品自称是 javascript 在线代码编辑器 或 react 在线编辑器。</p>
<h3 data-id="heading-0">为什么需要在线代码编辑器？</h3>
<p>首先思考一下在什么情况下需要使用在线代码编辑器。</p>
<h4 data-id="heading-1">远程协作</h4>
<p>使用桌面 IDE 设置协作过程可能存在不便，使用在线代码编辑器，可以让其任务变得简单，就像在 Google Docs 中创建文档一样。</p>
<h4 data-id="heading-2">代码分享</h4>
<p>可以使用在线代码编辑器与同事、朋友或全世界分享代码，可以在博客文章、操作指南、文档中使用沙箱嵌入或链接。</p>
<h4 data-id="heading-3">快速部署</h4>
<p>有一些在线代码编辑器集成了主流的主机功能，可以开发一个简单的应用程序，并在几分钟内部署它以进行测试。个人认为这是低代码平台集成在线代码编辑器的主要原因之一。</p>
<h4 data-id="heading-4">在线面试</h4>
<p>使用在线代码编辑器，可以在招聘的过程中快速考察候选人的编码能力。大厂面试基本都使用这种方式考察候选者的编码能力。</p>
<h4 data-id="heading-5">原型制作</h4>
<p>可以快速验证测试一些想法，并从用户那里获得即时反馈。</p>
<h4 data-id="heading-6">学习</h4>
<p>现代 Web 开发中，值得关注最新消息，甚至更好的是，在真实条件下尝试它们。最好的在线代码编辑器可快速完成此操作，而无需进行不必要的设置。</p>
<h3 data-id="heading-7">如何选择在线代码编辑器？</h3>
<p>大多数主流的在线代码编辑器功能都很丰富，包括自动完成编译、git 集成、插件支持、CI/CD 管道等，正是这些功能可以使特定编辑器比其他编辑器更适合。</p>
<p>下面是在选择在线 IDE 或者代码编辑器考虑的条件，但不局限一下条件：</p>
<ul>
<li>运行速度；</li>
<li>是否免费</li>
<li>与代码仓库的集成；</li>
<li>易于设置；</li>
<li>语言支持；</li>
<li>NPM 支持；</li>
<li>终端支持；</li>
<li>文本和布局设置</li>
</ul>
<h3 data-id="heading-8">在线代码编辑器的核心功能</h3>
<p>在线编辑器必须具备的基本功能：</p>
<ul>
<li>分享；</li>
<li>布局设置；</li>
<li>协作；</li>
<li>制表设置；</li>
<li>评论；</li>
<li>结果视图；</li>
<li>基本文件结构</li>
</ul>
<h3 data-id="heading-9">在线代码编辑器的优缺点</h3>
<h4 data-id="heading-10">优点</h4>
<p><strong>零设置</strong>：无需下载、安装和配置 IDE；
<strong>共享和协作</strong>：开始编码，然后将 URL 转发给队友以继续项目开发的工作或协助调试；
<strong>零或很少的成本</strong>：绝大多数在线代码编辑器都有免费版本，足以应付大部分日常任务；</p>
<h4 data-id="heading-11">缺点</h4>
<p><strong>功能有限</strong>：一些代码编辑器甚至缺少被称为代码编辑器的基本功能；
<strong>性能问题</strong>：许多基于 Web 的 IDE 和代码编辑器都存在性能问题；
<strong>没有版本控制</strong>：绝大多数在线代码编辑器没有与 git 工具集成；</p>
<h3 data-id="heading-12">推荐代码编辑器</h3>
<h4 data-id="heading-13">1. Codepen</h4>
<p>项目地址：<a href="https://link.juejin.cn/?target=https%3A%2F%2Fcodepen.io%2F" target="_blank" rel="nofollow noopener noreferrer" title="https://codepen.io/" ref="nofollow noopener noreferrer">codepen.io/</a></p>
<p><img src="https://p6-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/fbc050e8229e4334984e0a37640959d8~tplv-k3u1fbpfcp-watermark.image" alt="image.png" loading="lazy" referrerpolicy="no-referrer"></p>
<p>Codepen 算是在线代码编辑器市场上最受欢迎的工具之一。该平台在某种程度上开始演变为社交网络，类似于 <code>Pinterest</code>。开发者可以在整个平台上分享他们的最佳实践和 Pin 图，并获得社区的批准。</p>
<p>该工具内部有一个完善的搜索，可以发现感兴趣的模板、项目、代码片段和主题，也是一个不错的代码学习平台。</p>
<p>Codepen 的最大的优势是学习和发现新技术及代码最佳实践，浏览、工作和使用 Pens 是了解它们如何构建以及代码如何工作的好方法。</p>
<p>另一个产品细节是代码的呈现模式，在会议、聚会和代码分享讨论中非常有用。</p>
<h4 data-id="heading-14">2. Stackblitz</h4>
<p>项目地址：<a href="https://link.juejin.cn/?target=https%3A%2F%2Fstackblitz.com%2F" target="_blank" rel="nofollow noopener noreferrer" title="https://stackblitz.com/" ref="nofollow noopener noreferrer">stackblitz.com/</a></p>
<p><img src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/cfb376ecd76a47e7a63adf580ef29390~tplv-k3u1fbpfcp-watermark.image" alt="image.png" loading="lazy" referrerpolicy="no-referrer"></p>
<p>Stackblitz 与完整的 IDE 非常相似，特别是 VsCode，该工具是基于 VsCode 代码构建的，具有丰富的功能，可以开始和继续开发完整的全栈应用程序。该工具由 Visual Studio 提供支持。</p>
<p>它会在键入时自动安装依赖项、编译、捆绑和热重载。</p>
<p>导入库对于 Web 开发至关重要，因此 StackBlitz 包含一个浏览器内 npm 客户端，支持一次安装多个包和特定版本，是一个非常不错的功能。</p>
<p>Stackblitz 的杀手锏是离线编辑，Stackblitz 开发了一个浏览器内的网络服务器来实现这一目标。</p>
<p>使用 Stackblitz，可以在单独的窗口中进行预览和编辑，这与 iframe 或其他项目上的小窗口相比非常棒。还支持连接到 Github 并能方便的 导入/导出项目、与 Google Firebase 集成以创建全栈项目。</p>
<h4 data-id="heading-15">3. Playcode</h4>
<p>项目地址：<a href="https://link.juejin.cn/?target=https%3A%2F%2Fplaycode.io%2F" target="_blank" rel="nofollow noopener noreferrer" title="https://playcode.io/" ref="nofollow noopener noreferrer">playcode.io/</a></p>
<p><img src="https://p9-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/b284558c207d40cb99586a6e8f4b698a~tplv-k3u1fbpfcp-watermark.image" alt="image.png" loading="lazy" referrerpolicy="no-referrer"></p>
<p>Playcode 只是一个用于快速原型设计和查看编码结果的简单工具。该工具具有经典的三窗口布局：代码编辑器、控制台和结果视图。该产品具有基本的文件结构，但没有版本控制和其他 IDE 功能。</p>
<p>它仅支持 <code>JavaScript</code>、<code>HTML</code> 和 <code>CSS</code>，还有一个选项可以选择编辑器样式、字体大小和其他文本编辑功能。</p>
<h4 data-id="heading-16">4. JSfiddle</h4>
<p>项目地址：<a href="https://link.juejin.cn/?target=https%3A%2F%2Fjsfiddle.net%2F" target="_blank" rel="nofollow noopener noreferrer" title="https://jsfiddle.net/" ref="nofollow noopener noreferrer">jsfiddle.net/</a></p>
<p><img src="https://p9-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/f771abaed9e64014975f3d94430013ba~tplv-k3u1fbpfcp-watermark.image" alt="image.png" loading="lazy" referrerpolicy="no-referrer"></p>
<p>JSfiddle 是一个简单但非常流行、快速、高效的在线代码编辑器，个人觉得和Codepen类似。该工具允许快速共享代码并将结果可视化预览。这个工具非常流行和简单，并集成了 StackOverflow。</p>
<p>从视觉上看，像大多数平台一样，如Codepen，界面被分为4个区域：用于编写代码的区域、用于编辑CSS、SCSS或sass的区域，用于HTML布局的区域以及用于预览结果的区域。</p>
<p>JSfiddle 有非常好的和完整的文档，以及一个可以进行投票的开发社区。包含 Vue、Jquery、React + JSX和其他语言的代码模板，可以让开发者快速上手。</p>
<p>和类似IDE的在线编辑器相比，JSfiddle 不支持文件和文件夹的概念，并且无法构建 CI/CD 管道，因此主要用于前端展示。</p>
<h4 data-id="heading-17">5. Jsoneditoronline</h4>
<p>项目地址：<a href="https://link.juejin.cn/?target=https%3A%2F%2Fjsoneditoronline.org%2F" target="_blank" rel="nofollow noopener noreferrer" title="https://jsoneditoronline.org/" ref="nofollow noopener noreferrer">jsoneditoronline.org/</a></p>
<p><img src="https://p9-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/687a7b62a8a5493fbf98aa0ef7dcbf76~tplv-k3u1fbpfcp-watermark.image" alt="image.png" loading="lazy" referrerpolicy="no-referrer"></p>
<p>Jsoneditoronline 是一种基于 Web 的工具，用于查看、编辑和格式化 JSON 数据。能够清晰、可编辑的树视图或代码编辑器中格式化的显示 JSON 数据。可以将文档存储在本地或云端。</p>
<p>可以通过链接共享文档、以纯代码或树的形式查看 JSON 文件、进行查询，还可以将文件保存在本地。</p>
<h4 data-id="heading-18">6. Codesandbox</h4>
<p>项目地址：<a href="https://link.juejin.cn/?target=https%3A%2F%2Fcodesandbox.io%2F" target="_blank" rel="nofollow noopener noreferrer" title="https://codesandbox.io/" ref="nofollow noopener noreferrer">codesandbox.io/</a></p>
<p><img src="https://p9-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/9cad8a9cacc949e78e4868c16b467159~tplv-k3u1fbpfcp-watermark.image" alt="image.png" loading="lazy" referrerpolicy="no-referrer"></p>
<p>包括主流前端框架的模板，如 <code>React</code>、<code>Vue</code>、<code>Angular</code> 和 <code>NodeJs</code>、<code>Next</code> 等。</p>
<p>Codesandbox 定位为一个开发平台，社区制作了很多代码模板。此外，可以使用文件结构和依赖项为特定用例创建自己的模板，就像在使用 <code>IDE</code> 中一样。</p>
<p>该工具与 Github 集成，可以轻松创建提交和审查 PR。开箱即用，可以将应用程序部署到 ZEIT 或 Netlify。</p>
<p>Codesandbox 在线代码编辑器中提供了 NPM 支持，此外，还有一个热模块重新加载。当然，每个 <code>sandbox</code> 都有一个安全 URL，支持 HTTPS 共享和反馈。</p>
<p>在协作方面，可以与同事实时协作，就像远程控制一样控制谁可以编辑或观看代码。</p>
<p>此外还有其他的功能，例如：</p>
<ul>
<li>控制台；</li>
<li>测试视图；</li>
<li>内置终端；</li>
<li>Jest 测试的自动运行；</li>
<li>热模块重载；</li>
<li>项目组织；</li>
<li>将代码导出为 zip；</li>
<li>内置 linter (ESlint)</li>
</ul>
<h4 data-id="heading-19">7. TinyMCE</h4>
<p>项目地址：<a href="https://link.juejin.cn/?target=https%3A%2F%2Fwww.tiny.cloud%2F" target="_blank" rel="nofollow noopener noreferrer" title="https://www.tiny.cloud/" ref="nofollow noopener noreferrer">www.tiny.cloud/</a></p>
<p><img src="https://p1-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/2698c7e226f6452c931387a68d68872e~tplv-k3u1fbpfcp-watermark.image" alt="image.png" loading="lazy" referrerpolicy="no-referrer"></p>
<p>TinyMCE是一款易用、且功能强大的所见即所得的富文本编辑器，同类程序有：UEditor、Kindeditor、Simditor、CKEditor、wangEditor、Suneditor、froala等等。目前个人博客后台的编辑器就是集成了 TinyMCE。</p>
<h4 data-id="heading-20">8. StackEdit</h4>
<p>项目地址：<a href="https://link.juejin.cn/?target=https%3A%2F%2Fstackedit.io%2F" target="_blank" rel="nofollow noopener noreferrer" title="https://stackedit.io/" ref="nofollow noopener noreferrer">stackedit.io/</a></p>
<p><img src="https://p9-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/3908ab0128084c91bbbb65394c4492d8~tplv-k3u1fbpfcp-watermark.image" alt="image.png" loading="lazy" referrerpolicy="no-referrer"></p>
<p>StackEdit 是一个在线 Markdown 编辑器，可以同步文件与谷歌驱动器、Dropbox和GitHub，也可以将其作为博客发布到 Blogger、WordPress和Zendesk。</p>
<h4 data-id="heading-21">9. Editor.md</h4>
<p>项目地址：<a href="https://link.juejin.cn/?target=https%3A%2F%2Fpandao.github.io%2Feditor.md%2Findex.html" target="_blank" rel="nofollow noopener noreferrer" title="https://pandao.github.io/editor.md/index.html" ref="nofollow noopener noreferrer">pandao.github.io/editor.md/i…</a></p>
<p><img src="https://p6-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/1921eb666f4b4767a52cc24719723b15~tplv-k3u1fbpfcp-watermark.image" alt="image.png" loading="lazy" referrerpolicy="no-referrer"></p>
<p>Editor.md 是一个开源在线 Markdown 编辑器，支持Markdown转HTML的显示处理，打算将其集成到博客后台中。</p>
<h3 data-id="heading-22">总结</h3>
<p>上面是个人觉得不错的在线编辑器，有些在项目中也使用了，欢迎推荐好用的 Markdown 编辑器。</p></div>
