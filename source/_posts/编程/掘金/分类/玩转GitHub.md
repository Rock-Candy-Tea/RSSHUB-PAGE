<h2 data-id="heading-0">前言</h2>
<p>Hi Coder，我是 CoderStar！</p>
<p>这次主要给大家简单介绍一下我常用的一些Github工具。</p>
<h2 data-id="heading-1">Chrome 插件</h2>
<h3 data-id="heading-2">Octotree</h3>
<p>增强 GitHub 代码审查和探索的浏览器扩展，非常建议大家安装，浏览 Github 项目时不要太爽！</p>
<p><img src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/cd426f67ccbe49c99e62cfa9e0e281a7~tplv-k3u1fbpfcp-watermark.image" alt="Octotree" loading="lazy" referrerpolicy="no-referrer"></p>
<h3 data-id="heading-3">GitHub 黑暗主题</h3>
<p>基于 Atom One Dark 的适用于所有 GitHub 的深色主题。</p>
<p><img src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/0599652d08b2420ebdc8567d32c9934e~tplv-k3u1fbpfcp-watermark.image" alt="github_dark" loading="lazy" referrerpolicy="no-referrer"></p>
<h3 data-id="heading-4">Sourcegraph</h3>
<p>向 GitHub、GitLab 和其他主机添加代码智能：悬停、定义、引用，适用于 20 多种语言。就是让我们可以不使用 IDE 来快速查看代码之间的关系</p>
<p><img src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/2befad5387d3445a9327db290fe3267d~tplv-k3u1fbpfcp-watermark.image" alt="Sourcegraph" loading="lazy" referrerpolicy="no-referrer"></p>
<h3 data-id="heading-5">GitZip for github</h3>
<p>可以将 github 仓库的子目录和文件压缩成 zip 下载。</p>

<h2 data-id="heading-15">在线工具</h2>
<h3 data-id="heading-16">DownGit</h3>
<p><a href="https://link.juejin.cn/?target=https%3A%2F%2Fminhaskamal.github.io%2FDownGit%2F%23%2Fhome" target="_blank" rel="nofollow noopener noreferrer" title="https://minhaskamal.github.io/DownGit/#/home" ref="nofollow noopener noreferrer">DownGit</a></p>

<p><img src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/7c4b54d5432846b5a9a9d2be57095f93~tplv-k3u1fbpfcp-watermark.image" alt="DownGit" loading="lazy" referrerpolicy="no-referrer"></p>
<h3 data-id="heading-17">github1s</h3>
<p><a href="https://link.juejin.cn/?target=https%3A%2F%2Fgithub.com%2Fconwnet%2Fgithub1s" target="_blank" rel="nofollow noopener noreferrer" title="https://github.com/conwnet/github1s" ref="nofollow noopener noreferrer">github1s</a></p>
<p>将 github 仓库前的 github 域名换为 <code>github1s</code>，便能以 VSCode 的形式打开项目，方便查看，但注意不可编辑，是只读的。</p>
<p>如将 <code>https://github.com/Coder-Star/LTXiOSUtils</code> 改为 <code>https://github1s.com/Coder-Star/LTXiOSUtils</code></p>
<p><img src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/8685e846252c42f495d4b7f9bdc71ae1~tplv-k3u1fbpfcp-watermark.image" alt="github1s" loading="lazy" referrerpolicy="no-referrer"></p>
