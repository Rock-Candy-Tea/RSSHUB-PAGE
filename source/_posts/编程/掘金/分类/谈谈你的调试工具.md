<h2 data-id="heading-0">window chrome devtool调试ios</h2>
<p>使用<a href="https://github.com/RemoteDebug/remotedebug-ios-webkit-adapter" target="_blank" rel="nofollow noopener noreferrer">remotedebug-ios-webkit-adapter</a></p>
<p>1.github地址<br>
<a href="https://github.com/RemoteDebug/remotedebug-ios-webkit-adapter" target="_blank" rel="nofollow noopener noreferrer">github.com/RemoteDebug…</a></p>
<p>2.介绍<br>
基于<a href="https://github.com/google/ios-webkit-debug-proxy" target="_blank" rel="nofollow noopener noreferrer">ios-webkit-debug-proxy</a>二次开发的ios-webkit协议适配器</p>
<p>3.注意点<br>
• chrome inspect需配置Discover network targets,点击 Discover network targets后面的configure, 加上remotedebug_ios_webkit_adapter启动的地址<br>
• 访问手机页面后再启动remotedebug_ios_webkit_adapter</p>
<h2 data-id="heading-1">chrome devtool 调试微信网页</h2>
<ol>
<li>跟调试手机普通网页操作一致 chrome devtool调试android真机</li>
<li>对手机微信进行配置<br>
微信访问 <a href="http://debugx5.qq.com/" target="_blank" rel="nofollow noopener noreferrer">debugx5.qq.com/</a>
在配置页面中，找到“信息”标签页，并勾选上它的”打开TBS内核      Inspector调试功能”</li>
<li>大功告成  开始调试</li>
</ol>
<h2 data-id="heading-2">chrome devtool 调试android真机</h2>
<p>1.手机打开usb调试<br>
设置-关于手机-版本号连续点击几次出现-返回设置界面会看到开发者模式选项</p>
<p>2.用usb线连接电脑和手机,手机上弹出usb用于充电的选择框，选择传输文件。弹出允许计算机调试手机，选择允许。</p>
<p>3.chrome浏览器输入 chrome://inspect<br>
选择Devices,勾选Discover USB devices,Discover network targets。
remote target出现手机访问的页面。点击inspect开始调试</p>
<p><img alt="image" class="lazyload" src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/e2ee7bbdf6a44a6cb0f7a2b08aaba391~tplv-k3u1fbpfcp-zoom-1.image" data-width="800" data-height="600" referrerpolicy="no-referrer"></p>
<p>碰到的问题</p>
<ol>
<li>remote target没有出现手机上访问的页面
依次尝试以下方案</li>
</ol>
<ul>
<li>换根usb线，可能usb线只支持充电</li>
<li>电脑下载360手机助手
猜测可能电脑少了一个驱动。360助手应用有集成这个驱动软件</li>
</ul>

<h2 data-id="heading-4">mumu android模拟器调试</h2>
<p>一、抓包
1.下载adb （android debug bridge） （直接解压就行）
2.在安装目录打开cmd 执行adb命令 （我直接将adb拖进vs，用vs编辑器带的cmd窗口操作）
3.连接 mumu 模拟器 adb connect 127.0.0.1:7555 （mumu 模拟器一般占用端口为7555,如果不是自己上网查mumu 怎么查看占用端口）
<img alt="image" class="lazyload" src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/f1d8369d089748d29e9f5b7437b99b35~tplv-k3u1fbpfcp-zoom-1.image" data-width="800" data-height="600" referrerpolicy="no-referrer"></p>
<p>4.调出mumu模拟器 wifi设置 adb shell am start -a android.intent.action.MAIN -n com.android.settings/.wifi.WifiSettings（mumu模拟器本身没有将wifi设置显示出来）
<img alt="image" class="lazyload" src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/716bf9ef66fe48c9baf518afa3a49bc4~tplv-k3u1fbpfcp-zoom-1.image" data-width="800" data-height="600" referrerpolicy="no-referrer">
此时看mumu模拟器 wifi设置就显示出来了
<img alt="image" class="lazyload" src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/61e96fb332cd4b1392b68209d434d008~tplv-k3u1fbpfcp-zoom-1.image" data-width="800" data-height="600" referrerpolicy="no-referrer"></p>
<p>5.接下来就是常规操作了 用fiddler 手机抓包</p>
