h2&#123;font-size:20px&#125;.markdown-body h3&#123;font-size:18px&#125;&#125;</style><h5 data-id="heading-0">1、最长公共前缀</h5>
<p>查找字符串数组中的最长公共前缀。如果不存在公共前缀，返回空字符串 ""。</p>
<p>输入：strs = ["flower","flow","flight"]</p>
<p>输出："fl"</p>
<p>解析： 找到所有字符串的公共前缀，那我们可以以任意一个字符串str来做第一次对比得出的最长公共前缀。</p>
<p>然后以 str 为标准与后面的字符串对比。 对比后得出新的公共前缀 str1，然后 str = str1。 继续往后对比。</p>
<p>下面是代码：</p>
<pre><code class="copyable">var longestCommonPrefix = function(strs) &#123;
  if (strs.length === 0) return ''
  let i = 0
  let str = strs[0]
  let index = 1
  while (index < strs.length) &#123;
    while (str[i] === strs[index][i] && i < str.length) &#123;
      i++
    &#125;
    str = str.substring(0, i)
    i = 0
    index++
  &#125;
  return str
&#125;

<h5 data-id="heading-1">2、最长回文子串</h5>
<p>给你一个字符串 s，找到 s 中最长的回文子串。</p>
<blockquote>
<p>回文子串：从前往后读 === 从后往前读</p>
</blockquote>
<p>输入：s = "babad"</p>
<p>输出："bab"</p>
<p>解释："aba" 同样是符合题意的答案。</p>
<p>解析：</p>
<h6 data-id="heading-2">中心拓展法：</h6>
<blockquote>
<p>有兴趣的同学可以另外去看看动态规划、马拉车算法</p>
</blockquote>
<p>以某点为中心，向两侧进行拓展，假如以a为原点，然后两边拓展是 bab 满足回文子串的规则，那就就继续往两边拓展。 如果 匹配的是 bac ，那么就没有必须继续往两边去拓展了，因为后面匹配出来的串也肯定不是回文子串。</p>
<p>还需要考虑的是：不一定回文子串是奇数的，还有可能是偶数的。所以还有一种可能是以两个原点然后往两边进行拓展。</p>
<blockquote>
<p>时间复杂度：O(n^2) 其中 n 是字符串的长度。长度为 1 和 2 的回文中心分别有 n 和 n-1 个，每个回文中心最多会向外扩展 O(n) 次。</p>
</blockquote>
<blockquote>
<p>空间复杂度：O(1)。</p>
</blockquote>
<p>下面是参考代码：</p>
<pre><code class="copyable">var longestPalindrome = function(s) &#123;
  if (s.length <= 1) return s
  let start = 0
  let end = 0
  for (let i = 0; i < s.length; i++) &#123;
    let len1 = judgeIs(s, i, i) // 奇数情况
    let len2 = judgeIs(s, i, i + 1) // 偶数情况
    let len = Math.max(len1, len2)
    if (len > end - start) &#123;
      start = i - (len - 1) / 2
      end = i + len / 2
    &#125;
  &#125;
  return s.substring(Math.round(start), end + 1)
&#125;

var judgeIs = function (s, left, right) &#123;
  let L = left
  let R = right
  while (L >= 0 && R < s.length && s.charAt(L) === s.charAt(R)) &#123;
    L--
    R++
  &#125;
  return R - L - 1
&#125;

<h5 data-id="heading-3">3、字符串匹配算法：KMP</h5>
<blockquote>
<p>Knuth–Morris–Pratt（KMP）算法是一种改进的字符串匹配算法，它的核心是利用匹配失败后的信息，尽量减少模式串与主串的匹配次数以达到快速匹配的目的。它的时间复杂度是 O(m+n)。</p>
</blockquote>

<p>PMT中的值是字符串的前缀集合与后缀集合的交集中最长元素的长度。</p>
<p>前缀： 字符串：abc的前缀是 &#123;a, ab&#125; 这个集合就是字符串abc的前缀集合</p>
<p>后缀： 字符串：abc的后缀是 &#123;bc, a&#125; 这个集合就是字符串abc的后缀集合</p>
<p>两个集合的交集：&#123;a&#125;, 那么最长元素就是a,长度就是 1</p>
