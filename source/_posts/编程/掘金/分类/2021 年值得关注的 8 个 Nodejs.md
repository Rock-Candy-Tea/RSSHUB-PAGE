<h2 data-id="heading-0">1. Cytoscape.js</h2>
<p><img alt="1.jpeg" class="lazyload" src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/3f41ff2cd0594f68be4c4638febc97bf~tplv-k3u1fbpfcp-watermark.image" data-width="800" data-height="600" referrerpolicy="no-referrer">
网站 <a href="https://js.cytoscape.org/" target="_blank" rel="nofollow noopener noreferrer">js.cytoscape.org/</a></p>
<p>这个用于可视化和图形分析的开源 JavaScript 库实现了丰富的交互功能。选择方形区域、平移、捏拉缩放等功能都是开箱即用。</p>
<p>Cytoscape 可以用于 Node.js 服务端环境完成图形分析任务，也可以在命令行下使用。有兴趣转向数据科学的开发者可以选择参与 Cytoscape 的开发，它的贡献指南和文档都很棒。</p>
<h2 data-id="heading-1">2. PDFKit</h2>
<p>网站 <a href="https://pdfkit.org/" target="_blank" rel="nofollow noopener noreferrer">pdfkit.org/</a></p>
<p>很有用的基于 Node 的 PDF 生成库，有助于创建复杂的 PDF 文件供下载，支持嵌入文本和字体、注解、矢量图形等特性。不过，这个项目的文档不算丰富，给它贡献代码有点困难。</p>
<h2 data-id="heading-2">3. Socket.IO</h2>
<p><img alt="3.png" class="lazyload" src="https://p1-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/eda8652af8f744489307a13baf17f675~tplv-k3u1fbpfcp-watermark.image" data-width="800" data-height="600" referrerpolicy="no-referrer">
网站 <a href="https://socket.io/" target="_blank" rel="nofollow noopener noreferrer">socket.io/</a></p>
<p>提供双向、实时的基于事件的通讯机制，支持所有浏览器设备，也同样注重性能。比如，可以基于它开发一个简单的聊天应用。</p>
<p>服务端收到新消息后会发给客户端，客户端接收事件通知无需再额外发送新请求至服务端。</p>
<p>支持以下有用特性：</p>
<ul>
<li>二进制流</li>
<li>实时分析</li>
<li>文档协作</li>
</ul>
<h2 data-id="heading-3">4. Strapi</h2>
<p><img alt="4.png" class="lazyload" src="https://p9-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/be38577039a540928b3c4ee27ec0899b~tplv-k3u1fbpfcp-watermark.image" data-width="800" data-height="600" referrerpolicy="no-referrer"></p>
<p>网站 <a href="https://strapi.io/" target="_blank" rel="nofollow noopener noreferrer">strapi.io/</a></p>
<p>开源内容管理系统，后端系统通过 REST 风格的 API 提供功能，项目的主要目标是在所有设备上交付结构化的内容。</p>
<p>这个项目支持许多特性，包括内置的邮件系统、文件上传、JSON Web Token 鉴权。基于 Strapi 构建的内容结构非常灵活，可供创建内容分组、定制 API。</p>
<h2 data-id="heading-4">5. Nest</h2>
<p><img alt="5.png" class="lazyload" src="https://p1-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/f3dc28aad97743cc9e10cb67bd60bbeb~tplv-k3u1fbpfcp-watermark.image" data-width="800" data-height="600" referrerpolicy="no-referrer"></p>
<p>网站 <a href="https://nestjs.com/" target="_blank" rel="nofollow noopener noreferrer">nestjs.com/</a></p>
<p>Nest 是很流行的创建高效、可伸缩的服务端应用的新一代框架。底层基于 Express 框架，使用 TypeScript 组合了函数式和面向对象的编程元素。其模块化的架构让你可以很灵活地使用各种库。</p>
<h2 data-id="heading-5">6. Date-fns</h2>
<p><img alt="6.png" class="lazyload" src="https://p6-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/5de7ccc4e5eb4a808af51247ff4f72bc~tplv-k3u1fbpfcp-watermark.image" data-width="800" data-height="600" referrerpolicy="no-referrer"></p>
<p>网站 <a href="https://date-fns.org/" target="_blank" rel="nofollow noopener noreferrer">date-fns.org/</a></p>
<p>date-fns 仍然是在 Node.js 和浏览器环境下处理 JavaScript 日期最简单一致的工具集，也和 browserify、webpack、rollup 等现代模块打包工具配合良好。社区支持非常好，所以支持的本地化区域非常多，各种功能都有详细描述和示例。</p>
<h2 data-id="heading-6">7. SheetJS</h2>
<p><img alt="7.png" class="lazyload" src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/1b1d04770087499d83345743ee2c8d95~tplv-k3u1fbpfcp-watermark.image" data-width="800" data-height="600" referrerpolicy="no-referrer"></p>
<p>网站 <a href="https://sheetjs.com/" target="_blank" rel="nofollow noopener noreferrer">sheetjs.com/</a></p>
<p>这个 Node.js 库可以处理 Excel 电子表格，以及其他相关功能。比如，导出表格、转换 HTML 表格和 JSON 数组为 xlsx 文件。社区很大，贡献指南的文档也很棒。</p>
<h2 data-id="heading-7">8. Express.js</h2>
<p><img alt="8.png" class="lazyload" src="https://p9-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/b58b2950f6b84035ba4532394eec01d0~tplv-k3u1fbpfcp-watermark.image" data-width="800" data-height="600" referrerpolicy="no-referrer"></p>
<p>网站 <a href="https://expressjs.com/" target="_blank" rel="nofollow noopener noreferrer">expressjs.com/</a></p>
<p>这是最流行的 Node.js 开源项目之一，它能够高效处理 HTTP 请求，基于 JavaScript 这一同时适用于服务端和浏览器的语言，因此价值巨大。</p>
<p>它是开发高速、安全的应用的利器。</p>
<p>基本特性：</p>
<ol>
<li>支持不同的扩展和插件</li>
<li>基于 HTTP 方法和 URL 的路由机制</li>
<li>无缝集成数据库</li>
</ol>
