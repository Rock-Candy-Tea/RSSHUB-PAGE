<h4 data-id="heading-1">利用 pagehide 事件在用户刷新页面时将vuex的store存入sessionstorage中，然后在页面加载时，从sessionstorage中获取，replaceState store</h4>

<p><strong>解析代码：</strong></p>
<p>    1. 在App.vue 中的 create() 生命周期中加载，是因为 index.hxml 为 vue 项目默认首页，里面默认引用了 App.vue 根组件，每次页面初次打开、刷新、跳转等都会先加载根组件，在根组件上，进行页面事件的监听，这样就可以通过下面的方式防止 Vuex 中的 state 数据丢失</p>
<p><img src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/e46bd6d6b2b2486b94af2af84ef9a35e~tplv-k3u1fbpfcp-watermark.image" alt loading="lazy" referrerpolicy="no-referrer"></p>
<p>    2. window.addEventListener 事件监听 ，监听 onpagehide 事件（在用户离开网页时触发），本来想使用 beforeunload 事件，但是在ios端不支持，则使用了 pagehide 替代。<img src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/4bcd4b96585844aba8ddc73dd74f60e8~tplv-k3u1fbpfcp-zoom-1.image" alt loading="lazy" referrerpolicy="no-referrer"></p>

<p><img src="https://p1-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/8748f77e05e6422d93224842bfe951e9~tplv-k3u1fbpfcp-watermark.image" alt loading="lazy" referrerpolicy="no-referrer"></p>

<p><strong>beforeunload 事件</strong></p>
<p><img src="https://p1-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/931b6521e27745018e2951d06d8bb47d~tplv-k3u1fbpfcp-watermark.image" alt loading="lazy" referrerpolicy="no-referrer"></p>
<p>    3. 刚开始打开 App时，sessionStorage 里面的数据为空（没有 store）、Vuex 里面的数据为原始状态</p>
<p><img src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/8aa8911be3e74b769ce97c70556b1a2d~tplv-k3u1fbpfcp-zoom-1.image" alt loading="lazy" referrerpolicy="no-referrer"></p>
<p>    4. 当我们在登录页面时，输入账号和密码后，点击登录时，跳转到我的页面，在这一过程中，我们设置了利用了Vuex中的调用异步的方法dispatch去调用 Vuex 里面的 action 方法–doLogin 方法、getUserInfo 方法，这两个方法里面，我们更改了 Vuex 中 state 的状态，这样 state 则保存了最新的数据，即代码效果的数据效果</p>
<p><img src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/624c1f5b5b5b49b0ae581d68012a25cc~tplv-k3u1fbpfcp-zoom-1.image" alt loading="lazy" referrerpolicy="no-referrer"></p>
<p><img src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/62ab10885e204666a13a90f200ae77c6~tplv-k3u1fbpfcp-zoom-1.image" alt loading="lazy" referrerpolicy="no-referrer"></p>
<p>    5. 当完成上面操作后，window.addEventListener 开始监听到 pagehide 页面跳转，这时候则在 sessionStorage 设置了 store 这个key 用来保存 当前 vuex 里面的 state 的状态，注意 sessionStorage.setItem() 里面的key 和 value 都要 " "，所以这里使用JSON.stringify()，把 JSON 对象 转化为 JSON 字符串，从而实现保存，这时的状态是这样的</p>
<p><img src="https://p9-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/2efc4d00313943818208df9142eb1bcf~tplv-k3u1fbpfcp-watermark.image" alt loading="lazy" referrerpolicy="no-referrer"></p>
