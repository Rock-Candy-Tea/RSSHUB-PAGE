<p><img src="https://p9-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/579926b55db64d018e22d0316e5cfe2e~tplv-k3u1fbpfcp-watermark.image" alt="画龙.png" loading="lazy" referrerpolicy="no-referrer"></p>

<p>正好前两天在公司的HWEB大前端分享会上给大家分享了富文本编辑器的一些实践，于是想：</p>
<blockquote>
<p>能否把这条龙插入到富文本编辑器中呢？</p>
</blockquote>
<h1 data-id="heading-1">在富文本编辑器中插入自定义内容</h1>
<p>在<a href="https://juejin.cn/post/6966993945973194765" target="_blank">之前的文章</a>中，给大家分享了如何在Quill中插入自定义的内容，我们一起来回顾下：</p>
<ul>
<li>第一步：自定义工具栏按钮</li>
<li>第二步：自定义Blot内容</li>
<li>第三步：在Quill注册自定义Blot</li>
<li>第四步：调用Quill的API插入自定义内容</li>
</ul>
<p>我们试着按照这个步骤来将龙插入到编辑器中。</p>
<h1 data-id="heading-2">第一步：自定义工具栏按钮</h1>
<p>这个非常简单：</p>
<pre><code class="copyable">const TOOLBAR_CONFIG = [
  [&#123; header: ['1', '2', '3', false] &#125;],
  ['bold', 'italic', 'underline', 'link'],
  [&#123; list: 'ordered' &#125;, &#123; list: 'bullet' &#125;],
  ['clean'],
  ['card', 'divider', 'emoji', 'file', 'tag'],
  ['dragon'], // 新增的
];
<span class="copy-code-btn">复制代码</span></code></pre>
<p>自定义工具栏按钮图标：</p>
<pre><code class="copyable">const dragonIcon = `<svg>...</svg>`;
const icons = Quill.import('ui/icons');
icons.dragon = dragonIcon;
<span class="copy-code-btn">复制代码</span></code></pre>
<p>增加工具栏按钮事件：</p>
<pre><code class="copyable">const quill = new Quill('#editor', &#123;
  theme: 'snow',
  modules: &#123;
    toolbar: &#123;
      container: TOOLBAR_CONFIG,
      handlers: &#123;
        ...
        // 增加一个空的事件
        dragon(value): void &#123;
          console.log('dragon~~');
        &#125;,
      &#125;,
    &#125;
  &#125;,
&#125;);
<span class="copy-code-btn">复制代码</span></code></pre>
<h1 data-id="heading-3">第二步：自定义Blot内容（核心）</h1>
<p>之前的分享提到：</p>
<blockquote>
<p>Quill中的Blot就是一个普通的ES6 Class</p>
</blockquote>
<p>因此我们需要编写一个类。</p>
<p>dragon.ts</p>
<pre><code class="copyable">import Quill from 'quill';

const BlockEmbed = Quill.import('blots/block/embed');

class DragonBlot extends BlockEmbed &#123;
  static blotName = 'dragon';
  static tagName = 'canvas';

  static create(value): any &#123;
    const node = super.create(value);
    const &#123; id, width, height &#125; = value;

    node.setAttribute('id', id || DragonBlot.blotName);
    if (width !== undefined) &#123;
      node.setAttribute('width', width);
    &#125;
    if (height !== undefined) &#123;
      node.setAttribute('height', height);
    &#125;

    // 绘制龙的逻辑，参考大帅老师的文章：https://juejin.cn/post/6963476650356916254
    new Dragon(node);
    
    return node;
  &#125;
&#125;

export default DragonBlot;
<span class="copy-code-btn">复制代码</span></code></pre>
<h2 data-id="heading-4">绘制龙</h2>
<p>绘制龙的逻辑参考大帅老师的文章，这里就不贴代码了，大帅老师的文章里有源码，直接拿来用就可以：</p>
<p><a href="https://juejin.cn/post/6963476650356916254" target="_blank">产品经理：你能不能用div给我画条龙？</a></p>
<blockquote>
<p>需要注意的是大帅老师文章里的龙图片背景不是纯黑的，需要换一张纯黑的图片。</p>
</blockquote>
<h1 data-id="heading-5">第三步：在Quill注册自定义Blot</h1>
<p>有了 DragonBlot，还需要将其注册到 Quill 中才能使用：</p>
<pre><code class="copyable">import DragonBlot from './formats/dragon';
Quill.register('formats/dragon', DragonBlot);
<span class="copy-code-btn">复制代码</span></code></pre>
<h1 data-id="heading-6">第四步：调用Quill的API插入自定义内容</h1>
<p>最后一步，见证奇迹的时刻到了！</p>
<pre><code class="copyable">const quill = new Quill('#editor', &#123;
  theme: 'snow',
  modules: &#123;
    toolbar: &#123;
      container: TOOLBAR_CONFIG,
      handlers: &#123;
        ...
        dragon(value): void &#123;
          console.log('dragon~~');
          const index = this.quill.getSelection().index;
          // 插入自定义内容
          this.quill.insertEmbed(index, 'dragon', &#123;
            id: 'canvas-dragon',
          &#125;);
        &#125;,
      &#125;,
    &#125;
  &#125;,
&#125;);
