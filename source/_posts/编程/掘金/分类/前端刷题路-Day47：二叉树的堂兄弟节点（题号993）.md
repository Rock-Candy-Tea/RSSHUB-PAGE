<h3 data-id="heading-0">二叉树的堂兄弟节点（题号993）</h3>
<h4 data-id="heading-1">题目</h4>
<p>在二叉树中，根节点位于深度 <code>0</code> 处，每个深度为 <code>k</code> 的节点的子节点位于深度 <code>k+1</code> 处。</p>
<p>如果二叉树的两个节点深度相同，但 <strong>父节点不同</strong> ，则它们是一对堂兄弟节点。</p>
<p>我们给出了具有唯一值的二叉树的根节点 <code>root</code> ，以及树中两个不同节点的值 <code>x</code> 和 <code>y</code> 。</p>
<p>只有与值 <code>x</code> 和 <code>y</code> 对应的节点是堂兄弟节点时，才返回 <code>true</code> 。否则，返回 <code>false</code>。</p>

<h4 data-id="heading-3">解释</h4>
<p>这题啊，是笔者想多了。</p>
<p>本以为有多硬的方法，结果就是简单的遍历全部数组，找到每个元素的层级并和<code>x</code>、<code>y</code>进行对比。</p>
<p>这就完事了，而且无所谓遍历的顺序，前序、中序、后序都可以，唯一需要记住的事情就是在遍历的过程中和<code>x</code>、<code>y</code>对比，并且记住层级。</p>
<h4 data-id="heading-4">自己的答案（BFS）</h4>
<pre><code class="copyable">var isCousins = function (root, x, y) &#123;
  var xParent = xLevel = xFound = null
      yParent = yLevel = yFound = null
      stack = [[root, 0]]
  function findXY(node, parent, level) &#123;
    if (node.val === x) &#123;
      [xParent, xLevel, xFound] = [parent, level, true]
    &#125;
    if (node.val === y) &#123;
      [yParent, yLevel, yFound] = [parent, level, true]
    &#125;
  &#125;
  findXY(root, null, 0)
  while (stack.length) &#123;
    const [node, level] = stack.pop()
    if (node.left) &#123;
      stack.unshift([node.left, level + 1])
      findXY(node.left, node.val, level + 1)
    &#125;
    if (node.right) &#123;
      stack.unshift([node.right, level + 1])
      findXY(node.right, node.val, level + 1)
    &#125;
    if (xFound && yFound) break
  &#125;
  return xLevel === yLevel && xParent !== yParent
&#125;;

<p>没啥可说的，就是简单的广度优先搜索，在搜索过程中调用<code>findXY</code>方法来获取比较的结果。</p>

<p>同理可得，也是简单的深度优先搜索，别的没啥。</p>
<h4 data-id="heading-6">自己的答案（DFS+Map）</h4>
<p>上面这两种方法还是有点复杂的，虽然在遍历到<code>x</code>和<code>y</code>就停止了，每次判断确实比较麻烦，而且<code>xParent</code>、<code>xLevel</code>和<code>xFound</code>这些个变量确实太多了，这里可以利用<code>Map</code>或者对象来存储每个节点的相关信息，在遍历完所有的节点后取出<code>x</code>和<code>y</code>节点，再拿它们的值互相对比就完事了。</p>
<pre><code class="copyable">var isCousins = function(root, x, y) &#123;
  var map = new Map()
  function DFS(node, level = 0, parent = null) &#123;
    if (!node) return
    map.set(node.val, &#123; level, parent &#125;)
    DFS(node.left, level + 1, node.val)
    DFS(node.right, level + 1, node.val)
  &#125;
  DFS(root)
  return map.get(x).level === map.get(y).level && map.get(x).parent !== map.get(y).parent

<p>看吧，代码量确实很少，使用广度优先搜索也差不多，反正比上面的两种方法少了很多。</p>
<p>缺点也很明显：</p>
<ol>
<li>新建了<code>Map</code>来存储所有节点的相关信息</li>
<li>需要遍历完所有的节点。</li>
</ol>
<p>但让人意想不到是这种方法简直快得离谱</p>
