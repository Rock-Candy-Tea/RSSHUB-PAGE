h2&#123;font-size:20px&#125;.markdown-body h3&#123;font-size:18px&#125;&#125;</style><p>经典面试题，将两个有序数组合并为一个有序数组。</p>
<h4 data-id="heading-0">1、实现方式：</h4>
<p>（1）while循环遍历两个数组（不破坏原数组，生成新数组）。
直到两个数组的计算的索引（i与j），其中任何一个大于自身数组的长度，终止循环
。</p>
<p>（2）正对比，错位对比。</p>
<p>因为两个数据都是有序的。故，只需按照元素值对比大小，然后取最大的值即可；</p>
<p>如果其中一个数组索引的元素值，小于另外一个数组索引的元素值，将最小的值放到newArr中；</p>
<p>并叠加最小值的索引值（列i++）；</p>
<p>（3）元素值有3个对比（小于、大于、等于）。</p>
<p>小于，取小于的元素值，叠加小于的索引；</p>
<p>大于，取小于的元素值，叠加小于的索引；</p>
<p>等于，两个元素值都取，同时叠加两个索引；</p>
<h4 data-id="heading-1">2、代码</h4>
<pre><code class="copyable">function mergeSort(arr1, arr2) &#123;
    var [i,j] = [0,0];
    let newArr = [];
    while(i < arr1.length || j <arr2.length) &#123;
        if (arr1[i] < arr2[j]) &#123;
            newArr.push(arr1[i]);
            i++
        &#125; else if (arr1[i] > arr2[j]) &#123;
            newArr.push(arr2[j])
            j++
        &#125; else &#123;
            if(arr1[i]) newArr.push(arr1[i]);
            if(arr2[j]) newArr.push(arr2[j]);
            i++;
            j++
        &#125;
    &#125;
    return newArr
&#125;

mergeSort([2,4,6,7], [1,2,3,5,8,9,10,33,44,49]);
