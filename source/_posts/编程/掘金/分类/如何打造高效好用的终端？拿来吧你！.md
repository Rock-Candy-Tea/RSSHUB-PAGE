<p><strong>口味：夏黑葡萄</strong></p>
<p>趁着给手边新电脑配置环境，顺便整理了一波 <code>iTerm2 + oh-my-zsh</code> 的配置笔记，分享给大家。</p>
<p>如果你有一个高效的终端，那么每天早上来到公司后，你只需要以下三步键盘操作，即可开启一天的欢乐撸码时光。</p>
<ol>
<li><code>option + 回车</code>，唤起 iTerm2</li>
<li><code>j work</code>，跳转到工作目录</li>
<li><code>code .</code>, 用 VSCode 打开当前文件夹项目 (VSCode 自带命令行)</li>
</ol>
<p>就是这么高效，那么如何打造呢？拿出 10 分钟，按照下面步骤操作即可。</p>
<p>在这个过程中你可能会遇到各种问题，我尽可能的在文中贴出了相关的解决方案，来帮助你顺利进行下载和配置。</p>
<h2 data-id="heading-0">iTerm2</h2>
<p>首先点击下方链接下载 iTerm2。</p>
<ul>
<li><a href="https://link.juejin.cn/?target=https%3A%2F%2Fiterm2.com" target="_blank" rel="nofollow noopener noreferrer" title="https://iterm2.com" ref="nofollow noopener noreferrer">官网下载链接</a></li>
</ul>
<p><img src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/6ba0cdd4712e43018ea6e4c6631fec43~tplv-k3u1fbpfcp-zoom-1.image" alt loading="lazy" referrerpolicy="no-referrer"></p>
<h3 data-id="heading-1">设置唤醒热键</h3>
<p>下载之后为了更加便捷的唤起 iTerm2 窗口(<strong>实现本文开头的步骤一</strong>)，我们还需要设置热键。</p>
<p>按照 <code>iTerm2 => Preferences => Keys => HotKey</code> 路径找到配置项 HotKey 进行配置。</p>
<p>可以快速唤起终端和隐藏，我设置的是 <code>option + 回车</code>，<code>option + 空格</code>留给了 <a href="https://link.juejin.cn/?target=https%3A%2F%2Fu.tools" target="_blank" rel="nofollow noopener noreferrer" title="https://u.tools" ref="nofollow noopener noreferrer">uTools</a>。uTools 也是我非常喜欢的生产力工具，感兴趣的读者可以自行了解。</p>
<p><img src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/cd5b2382f1684f3a8bb5c688ddd0d86d~tplv-k3u1fbpfcp-zoom-1.image" alt loading="lazy" referrerpolicy="no-referrer"></p>
<h3 data-id="heading-2">常用快捷键</h3>
<ul>
<li>
<p><code>ctrl + u</code> 清除当前行输入</p>
</li>
<li>
<p><code>ctrl + a</code> 快速切换到当前输入的文本头部</p>
</li>
<li>
<p><code>ctrl + e</code> 快速切换到当前输入的文本尾部</p>
</li>
<li>
<p><code>cmd + d</code> 垂直分屏</p>
</li>
<li>
<p><code>cmd + shift + d</code> 水平分屏</p>
</li>
<li>
<p><code>cmd + r</code> 清屏</p>
</li>
<li>
<p><code>cmd + t</code> 打开一个新的 iterm2 Tab</p>
</li>
<li>
<p><code>cmd + 左右方向键</code> 左右切换 Tab</p>
</li>
<li>
<p><code>cmd + n</code> 打开一个新的 iterm2 窗口</p>
</li>
<li>
<p><code>cmd + 回车</code> 全屏切换</p>
</li>
<li>
<p><code>cmd + w</code> 关闭标签</p>
</li>
<li>
<p><code>cmd + f</code> 查找</p>
</li>
<li>
<p><code>cmd + shift + h</code> 查看剪贴板历史</p>
</li>
</ul>

<h3 data-id="heading-4">配置主题</h3>
<p>首先，使用命令 <code>vim ~/.zsh</code> 打开配置文件，然后设置你喜欢的主题，可以去下面两个链接查看社区里的推荐以及官方的皮肤预览列表。</p>
<ul>
<li><a href="https://link.juejin.cn/?target=https%3A%2F%2Fwww.zhihu.com%2Fquestion%2F33277508%2Fanswer%2F58979821" target="_blank" rel="nofollow noopener noreferrer" title="https://www.zhihu.com/question/33277508/answer/58979821" ref="nofollow noopener noreferrer">oh my zsh 哪些主题比较好看、有特点？</a></li>
<li><a href="https://link.juejin.cn/?target=https%3A%2F%2Fgithub.com%2Fohmyzsh%2Fohmyzsh%2Fwiki%2Fthemes" target="_blank" rel="nofollow noopener noreferrer" title="https://github.com/ohmyzsh/ohmyzsh/wiki/themes" ref="nofollow noopener noreferrer">皮肤预览列表</a></li>
</ul>
<pre><code class="hljs language-bash copyable" lang="bash">ZSH_THEME=<span class="hljs-string">"ys"</span>
<span class="copy-code-btn">复制代码</span></code></pre>
<p>最后，执行命令 <code>source ~/.zshrc</code> ，重新执行 zsh 配置文件使配置生效。</p>
<h3 data-id="heading-5">配置插件</h3>
<p>为了实现本文开头的操作，我们还需要下载一些插件，这里也推荐一些日常高频使用的插件，更多插件请前往官网查询。</p>
<h4 data-id="heading-6">git 插件</h4>
<ul>
<li><a href="https://link.juejin.cn/?target=https%3A%2F%2Fgithub.com%2Frobbyrussell%2Foh-my-zsh%2Ftree%2Fmaster%2Fplugins%2Fgit" target="_blank" rel="nofollow noopener noreferrer" title="https://github.com/robbyrussell/oh-my-zsh/tree/master/plugins/git" ref="nofollow noopener noreferrer">官网</a></li>
</ul>
<p>git 插件提供了很多 alias 简化你的 git 命令，在上方链接可以查询。</p>
<h4 data-id="heading-7">autojump 插件</h4>
<p>使用 <code>j + 目录缩写</code>，即可灵活的在任意目录间跳转，终于可以不用 cd 了！</p>
<p><strong>(实现本文开头的步骤二)</strong></p>
<pre><code class="hljs language-bash copyable" lang="bash"><span class="hljs-comment"># 打开 vue3 文件夹</span>
j vue3

<span class="hljs-comment"># 支持多个参数 /user/tong/work/li</span>
j w li
<span class="copy-code-btn">复制代码</span></code></pre>
<h4 data-id="heading-8">zsh-syntax-highlighting</h4>
<ul>
<li><a href="https://link.juejin.cn/?target=https%3A%2F%2Fgithub.com%2Fzsh-users%2Fzsh-syntax-highlighting" target="_blank" rel="nofollow noopener noreferrer" title="https://github.com/zsh-users/zsh-syntax-highlighting" ref="nofollow noopener noreferrer">官网</a></li>
</ul>
<p>语法高亮，官网链接中有安装前后对比示例。</p>
<p>执行下面命令安装。</p>
<pre><code class="hljs language-bash copyable" lang="bash">git <span class="hljs-built_in">clone</span> https://github.com/zsh-users/zsh-syntax-highlighting.git <span class="hljs-variable">$&#123;ZSH_CUSTOM:-~/.oh-my-zsh/custom&#125;</span>/plugins/zsh-syntax-highlighting
<span class="copy-code-btn">复制代码</span></code></pre>
<pre><code class="hljs language-bash copyable" lang="bash">vim ~/.zshrc <span class="hljs-comment"># 输入命令在 zshrc 中配置</span>
plugins=(其他的插件 zsh-syntax-highlighting)
<span class="copy-code-btn">复制代码</span></code></pre>
<p>配置好后保存，并运行下面命令使配置生效。</p>
<pre><code class="hljs language-bash copyable" lang="bash"><span class="hljs-built_in">source</span> ~/.zshrc
<span class="copy-code-btn">复制代码</span></code></pre>
