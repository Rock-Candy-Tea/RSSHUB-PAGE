<h3 data-id="heading-0">面试总是遇到这个问题,那不如自己总结加深一下印象</h3>
<h4 data-id="heading-1">1.如何区分</h4>
<blockquote>
<p>深拷贝与浅拷贝，简单点来说，就是假设B复制了A，当修改A时，看B是否会发生变化，如果B也跟着变了，说明这是浅拷贝 ; 如果B没变，那就是深拷贝。</p>
</blockquote>
<pre><code class="copyable">来看个浅拷贝的例子:
<span class="copy-code-btn">复制代码</span></code></pre>
<pre><code class="hljs language-javascript copyable" lang="javascript"><span class="hljs-keyword">let</span> a=[<span class="hljs-number">0</span>,<span class="hljs-number">1</span>,<span class="hljs-number">2</span>,<span class="hljs-number">3</span>,<span class="hljs-number">4</span>];
<span class="hljs-keyword">let</span> b=a;
<span class="hljs-built_in">console</span>.log(a === b); <span class="hljs-comment">//true</span>
a[<span class="hljs-number">0</span>]=<span class="hljs-number">1</span>;
<span class="hljs-built_in">console</span>.log(a,b); <span class="hljs-comment">// a=[1,2,3,4,5]  b=[1,2,3,4,5]</span>
<span class="copy-code-btn">复制代码</span></code></pre>

<h4 data-id="heading-5">3.实现方法</h4>
<h5 data-id="heading-6">3.1 浅拷贝</h5>
<ol>
<li>Object.assign()</li>
</ol>
<pre><code class="hljs language-javascript copyable" lang="javascript"><span class="hljs-comment">//目标对象</span>
<span class="hljs-keyword">const</span> target = &#123; <span class="hljs-attr">a</span>: <span class="hljs-number">1</span> &#125;; 
<span class="hljs-comment">// 被合并的源对象</span>
<span class="hljs-keyword">const</span> source1 = &#123; <span class="hljs-attr">b</span>: <span class="hljs-number">2</span> &#125;; 
<span class="hljs-keyword">const</span> source2 = &#123; <span class="hljs-attr">c</span>: <span class="hljs-number">3</span> &#125;;
<span class="hljs-comment">// Object.assign()方法的第一个参数是目标对象，后面的参数都是源对象。</span>
<span class="hljs-built_in">Object</span>.assign(target, source1, source2);
<span class="hljs-built_in">console</span>.log(target) <span class="hljs-comment">// &#123;a:1, b:2, c:3&#125;</span>
<span class="copy-code-btn">复制代码</span></code></pre>
<p>Object.assign()拷贝的属性是有限制的，只拷贝源对象的自身属性（不拷贝继承属性），也不拷贝不可枚举的属性（enumerable: false）</p>
<p><strong>注意</strong>: 如果目标对象与源对象有同名属性，或多个源对象有同名属性，则后面的属性会覆盖前面的属性。</p>

<ol start="2">
<li>jQuery.extend()</li>
</ol>
<p>$.extend( [deep ] , target, object1 , [objectN ])</p>
<ol>
<li>deep: 表示是否深拷贝，为true为深拷贝，为false，则为浅拷贝。</li>
<li>target:  Object类型 目标对象，其他对象的成员属性将被附加到该对象上。</li>
<li>object1,objectN(可选): Object类型 第一个以及第N个被合并的对象。</li>
</ol>
<pre><code class="hljs language-javascript copyable" lang="javascript"><span class="hljs-keyword">let</span> a=[<span class="hljs-number">0</span>,<span class="hljs-number">1</span>,[<span class="hljs-number">2</span>,<span class="hljs-number">3</span>],<span class="hljs-number">4</span>],
    b=$.extend(<span class="hljs-literal">true</span>,[],a);
a[<span class="hljs-number">0</span>]=<span class="hljs-number">1</span>;
a[<span class="hljs-number">2</span>][<span class="hljs-number">0</span>]=<span class="hljs-number">1</span>;
<span class="hljs-built_in">console</span>.log(a,b);
