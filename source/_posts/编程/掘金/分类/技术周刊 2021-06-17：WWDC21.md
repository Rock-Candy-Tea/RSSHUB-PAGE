<h1 data-id="heading-0">前端快爆</h1>
<ul>
<li>WebKit 官方博客发布了 WWDC21 中介绍 Safari 更多新特性的<a href="https://webkit.org/blog/11700/webkit-features-in-safari-at-wwdc21/" target="_blank" rel="nofollow noopener noreferrer">演讲视频列表</a>。</li>
</ul>
<p><img src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/45519fab7cc046d1be12367d16288fdc~tplv-k3u1fbpfcp-zoom-1.image" alt="image.png" loading="lazy" referrerpolicy="no-referrer"></p>
<ul>
<li>在 WWDC21 中还发布了测试版的 <a href="https://developer.apple.com/videos/play/wwdc2021/10254/" target="_blank" rel="nofollow noopener noreferrer">Swift 5.5</a>，最令人期待的是支持了 <code>async/await</code> 功能。</li>
</ul>
<p><img src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/c05309067dd54a7e9a67fa0d6a7698dd~tplv-k3u1fbpfcp-zoom-1.image" alt="image.png" loading="lazy" referrerpolicy="no-referrer">
​</p>
<ul>
<li>Chrome Canary 中<a href="https://bugs.chromium.org/p/chromium/issues/detail?id=669058#c17" target="_blank" rel="nofollow noopener noreferrer">初步实验性实现了</a> CSS 关系伪类选择器 <code>[:has()](https://drafts.csswg.org/selectors/#relational)</code>，可以用来<strong>选择父级元素</strong>，目前测试需要通过命令行启动：</li>
</ul>

<span class="copy-code-btn">复制代码</span></code></pre>
<ul>
<li>
<p><a href="https://github.com/mochajs/mocha/releases/tag/v9.0.0" target="_blank" rel="nofollow noopener noreferrer">Mocha 9.0 发布</a>，Mocha 一直是 Node.js 和浏览器中流行的测试框架。新版本的主要改动是：</p>
<ul>
<li>不再支持 Node.js 10.x。</li>
<li>通过默认使用 ECMAScript modules <code>import(test_file)</code> 而不是 CommonJS <code>require(test_file)</code> 来加载测试文件，从而实现 「ES 模块优先」。</li>
</ul>
</li>
<li>
<p><a href="https://bugzilla.mozilla.org/show_bug.cgi?id=737785#c38" target="_blank" rel="nofollow noopener noreferrer">Firefox Nightly 91</a> 实现了无前缀的 CSS <code>[tab-size](https://developer.mozilla.org/en-US/docs/Web/CSS/tab-size)</code> 属性，至此除了 IE 所有的主流浏览器均已<a href="https://caniuse.com/css3-tabsize" target="_blank" rel="nofollow noopener noreferrer">完整支持该属性</a>。</p>
</li>
</ul>
