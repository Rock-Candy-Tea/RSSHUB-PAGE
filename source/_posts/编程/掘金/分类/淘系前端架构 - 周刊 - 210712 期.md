<h3 data-id="heading-1">Remote Repositories</h3>
<p>或许你有过这样一种经历：在 Github 上看到一个感兴趣的项目，想在本地的代码编辑器中看下是怎么实现的，这时你通常会把该仓库 clone 到本地，然后用 VS Code 打开项目，然后浏览代码。这个过程无疑会浪费很多时间。</p>
<p>近日，VS Code 官方发布 <a href="https://link.juejin.cn/?target=https%3A%2F%2Fcode.visualstudio.com%2Fblogs%2F2021%2F06%2F10%2Fremote-repositories" target="_blank" rel="nofollow noopener noreferrer" title="https://code.visualstudio.com/blogs/2021/06/10/remote-repositories" ref="nofollow noopener noreferrer">Remote Repositories</a> 插件，可以在不需要执行 <code>git pull</code> 的情况下，可以直接从 VS Code 中快速浏览、搜索、编辑和提交到任何远程 GitHub 存储库。</p>
<h4 data-id="heading-2">在 VS Code 中查看远程仓库</h4>
<p>Remote Repositories 插件让开发者在本地 VS Code 中查看仓库源码，并且大部分的 VS Code 插件仍然可用：</p>
