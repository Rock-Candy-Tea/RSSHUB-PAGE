<h2 data-id="heading-1">15、算术运算符</h2>
<p>运算符：</p>
<ul>
<li>
<p><code>+ - * / %</code> (二元运算符 双目)</p>
</li>
<li>
<p><code>++ --</code>（一元运算符 单目）</p>
</li>
<li>
<p><code>- - / %</code> 运算符对应的表达式值 ：</p>
<ul>
<li>1、两侧都是数值 正常计算</li>
<li>2、有一侧是纯数字字符串 字符串自动转换成数值 正常计算</li>
<li>3、有一侧是非数字字符串 结果 NaN
NaN ： not a number 不是一个数 NaN 不等于任何数 ，包括他自己</li>
</ul>
</li>
<li>
<p><code>%</code> 模运算符 （求余数）:</p>
<ul>
<li>5%3 === 2 8%4 ===0 9%2 ===1 189%10 === 9 4567%10===7 -1%3 ===-1 3%（-5） === 3</li>
<li>总结 ： 任何数 % 10 === 个位数</li>
</ul>
</li>
<li>
<p><code>+</code> ： 表示 加法运算 和 连接</p>
<ul>
<li>数字和字符串相加 表示 连接 100+"10" - 10 ---> 10000 结果类型 number</li>
</ul>
</li>
<li>
<p><strong>单目</strong> <code>++</code>自增 <code>--</code>自减</p>
<blockquote>
<p>注意 ：自增和自减运算符 操作数必须是变量</p>
</blockquote>
</li>
</ul>
<p><code>var i = 8;</code></p>
<p><code>i++ 或 ++i</code></p>
<ul>
<li>i++和++i 的异同：</li>
</ul>
<blockquote>
<p>相同点 ： 自增变量的值都会自增 1</p>
<p>不同点 ： 表达式的值不同</p>
<p>如果 ++ 在后， 先将自增变量 i 的值赋值给 j ， 然后自增变量 i 再自增 1 （先赋值后自增）</p>
<p>如果 ++ 在前 ，先将自增变量 i 的值自增 1，在将自增变量 i 的值赋值给 整个表达式 j</p>
</blockquote>
<pre><code class="hljs language-js copyable" lang="js"><span class="hljs-keyword">var</span> i = <span class="hljs-number">10</span>;

<h2 data-id="heading-2">16、赋值运算符</h2>
<ul>
<li><code>=</code> 赋值 将右侧的值 赋值给 左侧的变量</li>
</ul>
<blockquote>
<p>注意： = 符号左侧必须是变量 x+y = 10 错误 y = 10-x 正确</p>
</blockquote>
<ul>
<li>复合赋值运算符 ： += -= *= /= %=</li>
<li><code>x+=10</code> 等价 <code>x = x + 10</code></li>
</ul>
<h2 data-id="heading-3">17、关系运算符</h2>
<ul>
<li>
<p>运算符 ： >>= <<= ==等于 ===全等于 !=不等于 !==不全等</p>
</li>
<li>
<p>表达式 ： 3>4 2===3</p>
</li>
<li>
<p>表达式的值 ： true false 表达式成立 为 true，不成立 为假</p>
</li>
<li>
<p>表达式值几种情况 ：</p>
<ul>
<li>1、两侧都是数值 正常比较</li>
<li>2、两侧都是字符 正常比较 根据字符的 ASCII 码 值 进行比较 a--97 A--65 0--48</li>
<li>3、某一侧是纯数字字符串 字符串自动转成数值</li>
<li>4、不能正常比较的结果都是 false</li>
</ul>
</li>
<li>
<p>== 和 === 区别 ：</p>
</li>
</ul>
<blockquote>
<p>== 只比较值是否相等</p>
<p>=== 先 比较类型，再比较值</p>
</blockquote>
<h2 data-id="heading-4">18、逻辑运算符</h2>
<ul>
<li>运算符 ： && 与 || 或 ! 非</li>
<li>表达式 （参与逻辑运算的一般为 布尔类型 ） 3>4 && 6==5 3>4 || 9==9 !true</li>
<li>表达式的值 ：
<ul>
<li><code>&& （且）</code> ： && 两侧都为真 ，结果为 true ， 否则为 false 3>4 && 6==5 -->   false</li>
<li><code>|| （或）</code> : || 两侧都为假 ，结果为 假 ，否则就为真 3>4 || 9==9 --> true</li>
<li><code>! (非)</code> :  <code>!true -->false</code> <code>!false--->true</code></li>
</ul>
</li>
</ul>
<h2 data-id="heading-5">19、条件运算符 (三目运算符)</h2>
<ul>
<li>
<p>运算符 ： ? :</p>
</li>
<li>
<p>表达式 ： 表达式 1 ？ 表达式 2 ： 表达式 3 8<7 ? 89:90 10？1:2</p>
</li>
<li>
<p>表达式的值 ：</p>
<ul>
<li>如果表达式 1 的结果为真，取 表达式 2 的值作为整个条件表达式的值 ，否则 取表达式 3 为整个条件表达式的值</li>
</ul>
</li>
<li>
<p>计算机中表示真假的几种情况 （一般用于判断中）：</p>
<ul>
<li>1、非 0 为真，0 为假</li>
<li>2、所有字符串为真 空串为假 """"?1:2</li>
<li>3、所有对象为真</li>
<li>4、所有函数为真</li>
<li>5、null 和 undefined 为假</li>
</ul>
</li>
</ul>
