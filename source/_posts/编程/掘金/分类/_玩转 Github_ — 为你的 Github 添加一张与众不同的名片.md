<p><img src="https://p6-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/ab47456d911f4b8db89869ede6f35161~tplv-k3u1fbpfcp-watermark.image" alt="image.png" loading="lazy" referrerpolicy="no-referrer"></p>
<h2 data-id="heading-1">添加 Github Profile 名片</h2>
<ul>
<li>第一步：新建一个和你 Github Username 同名仓库</li>
</ul>
<p>比如，我的 Github 是 <strong>luffyZh</strong>，那么就新建一个名为 luffyZh 的仓库，如下图所示：</p>
<p><img src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/eaf016d2cb77477cb6fb5e773b34b6cd~tplv-k3u1fbpfcp-watermark.image" alt="image-20210816204804513的副本.png" loading="lazy" referrerpolicy="no-referrer"></p>
<p>并且，Github 的提示那是相当的友好，当你键入这个仓库名称的时候，官方就已经告诉你了：这是一个特别的仓库，你可以添加一个 <code>README.md</code> 文件作为你的 Github 名片，但是这个仓库你要确保他的权限是 <code>public</code> 并且需要有一个 <code>README.md</code> 文件。</p>
<p>创建好之后，你再回你的首页，你就会发现，这个 <code>README.md</code> 文件的内容已经出现在你的首页了，但是只有文案，那么接下来继续美化名片。</p>
<ul>
<li>第二步：添加相关信息</li>
</ul>
<p>上面我的个人名片大家可以看到，除了基本信息外，还有几个卡片信息，这些才是美化的关键，你可以在 profile 页里添加一些卡片信息，比如仓库状态，常用语言以及个人排名靠前的几个仓库，就像我上面截图的那样。</p>
<p>在这里，给大家推荐一个仓库 <a href="https://link.juejin.cn/?target=https%3A%2F%2Fgithub.com%2Fanuraghazra%2Fgithub-readme-stats" target="_blank" rel="nofollow noopener noreferrer" title="https://github.com/anuraghazra/github-readme-stats" ref="nofollow noopener noreferrer">github-readme-stats</a>，里面包含几个卡片的使用方法以及很多主题配色，相信大家一定能找到一个自己喜爱的主题，然后制作出一个与众不同的名片。</p>
<p><img src="https://p6-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/5ff78e7bcfbb4337935eef920b481e6c~tplv-k3u1fbpfcp-watermark.image" alt="image.png" loading="lazy" referrerpolicy="no-referrer"></p>
