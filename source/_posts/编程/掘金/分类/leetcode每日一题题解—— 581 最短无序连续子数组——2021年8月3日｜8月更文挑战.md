<h3 data-id="heading-1">Title Description</h3>
<p>Given an integer array nums, you need to find one continuous subarray that if you only sort this subarray in ascending order, then the whole array will be sorted in ascending order.</p>
<p>Return the shortest such subarray and output its length.</p>
<p> </p>
<p>Example 1:</p>
<pre><code class="copyable">Input: nums = [2,6,4,8,10,9,15]
Output: 5
Explanation: You need to sort [6, 4, 8, 10, 9] in ascending order to make the whole array sorted in ascending order.
<span class="copy-code-btn">复制代码</span></code></pre>
<p>Example 2:</p>
<pre><code class="copyable">Input: nums = [1,2,3,4]
Output: 0
<span class="copy-code-btn">复制代码</span></code></pre>
<p>Example 3:</p>
<pre><code class="copyable">Input: nums = [1]
Output: 0
<span class="copy-code-btn">复制代码</span></code></pre>
<p> </p>
<p>Constraints:</p>
<p>1 <= nums.length <= 104
-105 <= nums[i] <= 105
 </p>
<p>Follow up: Can you solve it in O(n) time complexity?</p>
<h3 data-id="heading-2">题目描述</h3>
<p>给你一个整数数组 nums ，你需要找出一个 连续子数组 ，如果对这个子数组进行升序排序，那么整个数组都会变为升序排序。</p>
<p>请你找出符合题意的 最短 子数组，并输出它的长度。</p>
<p> </p>
<p>示例 1：</p>
<pre><code class="copyable">输入：nums = [2,6,4,8,10,9,15]
输出：5
解释：你只需要对 [6, 4, 8, 10, 9] 进行升序排序，那么整个表都会变为升序排序。

<p>示例 2：</p>
<pre><code class="copyable">输入：nums = [1,2,3,4]
输出：0

<p>示例 3：</p>
<pre><code class="copyable">输入：nums = [1]
输出：0

<p> </p>
<p>提示：</p>
<p>1 <= nums.length <= 104
-105 <= nums[i] <= 105
 </p>
<p>进阶：你可以设计一个时间复杂度为 O(n) 的解决方案吗？</p>
<h3 data-id="heading-3">解题思路</h3>
<ul>
<li>复制一个新的数组</li>
<li>对复制的数组进行升序排序</li>
<li>从最左往右开始循环，两数组当前位置元素相同，计数变量加一，两数组当前位置元素不相同，从最右往左进行循环</li>
<li>两数组当前位置元素相同，计数变量加一，出现不同直接return</li>
</ul>
<h3 data-id="heading-4">代码</h3>
<pre><code class="copyable">/**
 * @param &#123;number[]&#125; nums
 * @return &#123;number&#125;
 */
var findUnsortedSubarray = function (nums) &#123;

    // 复制一个新的数组
    var num2 = [];
    for (var i = 0; i < nums.length; i++) &#123; num2[i] = nums[i]; &#125;

    // 对复制的数组进行升序排序
    num2.sort((a, b) => &#123;
        return a - b;
    &#125;);

    // 创建计数变量
    var sum = 0;

    // 从最左往右开始循环
    for (var i = 0; i < nums.length;) &#123;
        // 两数组当前位置元素相同，计数变量加一
        if (num2[i] == nums[i]) &#123;
            i++
            sum += 1;
        &#125; else if (num2[i] != nums[i]) &#123;
            // 两数组当前位置元素不相同，从最右往左进行循环
            for (let j = nums.length - 1; j >= 0;) &#123;
                // 两数组当前位置元素相同，计数变量加一
                if (num2[j] == nums[j]) &#123;
                    j--
                    sum += 1;

                &#125; else &#123;
                    // 出现不同直接return
                    return nums.length - sum
                &#125;
            &#125;
        &#125;
    &#125;
    // 如果可以进行到这里说明两个数组相同，return 0
    return 0
&#125;;
