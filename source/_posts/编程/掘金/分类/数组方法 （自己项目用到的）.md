<h3 data-id="heading-3">4. splice操作数组</h3>
<pre><code class="copyable">arr.splice(start,num,arg1,arg2...)
<span class="copy-code-btn">复制代码</span></code></pre>
<p><a href="https://www.jianshu.com/p/9eadb164714a" target="_blank" rel="nofollow noopener noreferrer">www.jianshu.com/p/9eadb1647…</a></p>
<p>是一个很强的数组操作方法，可以增删改</p>
<p>注意：操作的是原数组</p>
<p>a、删除数组元素</p>
<pre><code class="copyable">let arr = [1,2,3,4,5,6,7,8,9]
// 从 下标为 3 的项开始，依次向后删除 2 个元素，返回被删除的元素数组：[4, 5]，原数组改变为：[1, 2, 3, 6, 7, 8, 9]
arr.splice(3,2)
<span class="copy-code-btn">复制代码</span></code></pre>
<p>从 start下标 开始，删除 num 个元素，再在此处插入 arg1，arg2，arg3...</p>
<p>如果只填了 start 却没填 其它参数，则代表删除其后的所有元素</p>
<p>b、删除并插入元素</p>
<pre><code class="copyable">let arr = [1,2,3,4,5,6,7,8,9]
// 从 下标为 3 的项开始，依次向后删除 2 个元素，并将 1, 2, 3 添加到被删除的元素的位置。返回被删除的元素数组：[4, 5]，原数组改变为：[1, 2, 3, 1, 2, 3, 6, 7, 8, 9]
arr.splice(3,2,1,2,3)
<span class="copy-code-btn">复制代码</span></code></pre>
<p>c、单纯插入数组</p>
<pre><code class="copyable">let arr = [1,2,3,4,5,6,7,8,9]
// 从 下标为 3 的项开始，依次向后添加 1, 2, 3 等元素。返回被删除的元素数组：[]，原数组改变为：[1, 2, 3, 1, 2, 3, 4, 5, 6, 7, 8, 9]
arr.splice(3,0,1,2,3)
<span class="copy-code-btn">复制代码</span></code></pre>
