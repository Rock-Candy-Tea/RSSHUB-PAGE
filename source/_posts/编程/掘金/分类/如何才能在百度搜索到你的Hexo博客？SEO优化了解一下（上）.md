<h1 data-id="heading-0">零 前言</h1>
<blockquote>
<p>SEO，即Search Engine Optimization，译为搜索引擎优化。</p>
<p>它是一种利用搜索引擎的规则<strong>提高网站</strong>在有关搜索引擎内的<strong>自然排名</strong>的方式。</p>
</blockquote>
<p>关于Hexo的一切 —— <a href="https://juejin.cn/column/6997777547744247815" target="_blank" title="https://juejin.cn/column/6997777547744247815">我的Hexo专栏</a></p>
<p>继上篇<a href="https://juejin.cn/post/6998140713334472740" target="_blank" title="https://juejin.cn/post/6998140713334472740">Hexo博客域名绑定</a>，这篇我们来聊聊如何才能在百度搜索到你的Hexo博客（依旧是超详细图文教程）。由于内容过长，分为<strong>上下两篇</strong>，主要讲述：</p>
<p><img src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/a3722e676e514b7cb0a070a0ded90806~tplv-k3u1fbpfcp-watermark.image" alt="image-20210820162446863" loading="lazy" referrerpolicy="no-referrer"></p>
<h1 data-id="heading-1">一 解决方案</h1>
<p>因为百度无法爬取GitHub的内容，我们需要找一个<strong>可以被百度爬取</strong>到而且<strong>提供Pages服务</strong>的代码托管平台。</p>
<p>那么我们就只有国内的两家平台可供选择了：<code>Gitee</code> 和 <code>Coding</code> 🙈</p>
<p>虽然这两样都要花钱，但是Gitee自定义域名九十多都可以买服务器了，Coding则是前六个月免费，之后一个月可能才几毛钱（很少人访问的话），理所当然选择Coding平台。</p>
<p>并且我们可以使用<strong>Github + Coding双部署</strong>，国内IP访问国内Coding的地址，国外IP访问Github的地址，当然嫌麻烦可以只用Coding。</p>
<blockquote>
<p><a href="https://link.juejin.cn/?target=https%3A%2F%2Fcoding.net%2F" target="_blank" rel="nofollow noopener noreferrer" title="https://coding.net/" ref="nofollow noopener noreferrer">Coding官网</a></p>
<p>注册后新建Git项目，选择持续部署里的网站托管即可，具体可以查看<a href="https://link.juejin.cn/?target=https%3A%2F%2Fhelp.coding.net%2Fdocs%2Fpages%2Fprice.html" target="_blank" rel="nofollow noopener noreferrer" title="https://help.coding.net/docs/pages/price.html" ref="nofollow noopener noreferrer">帮助中心</a>。</p>
</blockquote>

<li>
<p>将博客同时部署到两个仓库，在根目录配置文件 <code>_config.yml</code> 添加你的项目地址：</p>
<pre><code class="hljs language-yaml copyable" lang="yaml"> <span class="hljs-attr">deploy:</span>

   <span class="hljs-attr">repository:</span> <span class="hljs-string">git@github.com:username/username.github.io.git</span>
   <span class="hljs-attr">branch:</span> <span class="hljs-string">master</span>
 <span class="hljs-string">​</span>
 <span class="hljs-comment"># 你的coding项目地址</span>
 <span class="hljs-bullet">-</span> <span class="hljs-attr">type:</span> <span class="hljs-string">git</span>

   <span class="hljs-attr">branch:</span> <span class="hljs-string">master</span>
   
 <span class="hljs-comment"># 注意两个username可能不同，是你分别注册的用户名</span>

</li>
<li>
<p>然后去<strong>域名控制台</strong>，配置两个解析，在百度项填入你的 <code>Coding</code> 项目地址，在默认项填入你的 <code>github.io</code> 地址：</p>

<h1 data-id="heading-4">四 推送资源</h1>
<p>上一步我们成功的让百度知道了网站的存在，但是百度还不知道我们网站上的内容，所以要向<a href="https://link.juejin.cn/?target=https%3A%2F%2Fziyuan.baidu.com%2Flinksubmit%2Findex" target="_blank" rel="nofollow noopener noreferrer" title="https://ziyuan.baidu.com/linksubmit/index" ref="nofollow noopener noreferrer">百度收录</a><strong>推送我们的内容</strong>。</p>
<p>这里有四种方式：<strong>API提交、主动推送、sitemap、自动推送、和手动提交</strong>。</p>
<p>因为<strong>手动提交</strong>就是把你博客内的所有链接都复制粘贴到<a href="https://link.juejin.cn/?target=https%3A%2F%2Fziyuan.baidu.com%2Flinksubmit%2Findex" target="_blank" rel="nofollow noopener noreferrer" title="https://ziyuan.baidu.com/linksubmit/index" ref="nofollow noopener noreferrer">此界面</a>中进行提交，在此不赘述。</p>

<h2 data-id="heading-7">sitemap</h2>
<ol>
<li>
<p>安装插件：</p>
<pre><code class="copyable"> npm install hexo-generator-sitemap --save 
 npm install hexo-generator-baidu-sitemap --save
<span class="copy-code-btn">复制代码</span></code></pre>
</li>
<li>
<p>安装完成后，我们执行<code>hexo cl</code>和 <code>hexo g -d</code>后，会在根目录下生成 <code>sitemap.xml</code>（其他搜索引擎用） 和 <code>baidusitemap.xml</code> （百度专用）文件。</p>

<h1 data-id="heading-9">五 总结一下</h1>
<p>一般来说，自动方式比手动方式效果要好一点，效率也要更高。</p>
<ul>
<li>自动方式中其中<strong>主动推送</strong>又是最快的一个，每天产出的新链接都可以通过此方式推送给百度收录。</li>
<li><strong>自动推送</strong>最为便捷，当部署代码的页面被访问时就会被自动推送给百度收录。</li>
<li><strong>sitemap</strong>方式提交后，百度会周期性的抓取检查，因此收录速度是最慢的。</li>
</ul>
<p>所以一般我推荐主动+自动配合着使用。</p>
