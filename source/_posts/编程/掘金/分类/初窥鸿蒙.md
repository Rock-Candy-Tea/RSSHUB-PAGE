h2&#123;font-size:20px&#125;.markdown-body h3&#123;font-size:18px&#125;&#125;</style><h2 data-id="heading-0">一、什么是鸿蒙</h2>
<p>鸿蒙即 HarmonyOS ，是华为公司推出的支持手机、平板、智能穿戴、智慧屏、车机等多种终端设备的分布式操作系统，并且它提供了多语言开发的 API，支持 Java、XML、C/C++、JS、CSS、HML（类 html 的鸿蒙自己的标记语言）等开发语言，而且它提供多种响应式布局方案，支持栅格化布局，可以使用同一套代码部署在手机手表平板等多种不同尺寸屏幕的设备上。</p>
<h2 data-id="heading-1">二、开发准备</h2>
<h3 data-id="heading-2">2.1 环境安装</h3>
<blockquote>
<p>开发鸿蒙软件需要用到 HUAWEI DevEco Studio，它提供了模板创建、开发、编译、调试、发布等服务。</p>
</blockquote>
<p>1、登录 <a href="https://developer.harmonyos.com/cn/home" target="_blank" rel="nofollow noopener noreferrer">HarmonysOS 应用开发门户</a>，点击右上角注册按钮，注册开发者帐号。</p>
<p>2、进入 <a href="https://developer.harmonyos.com/cn/develop/deveco-studio" target="_blank" rel="nofollow noopener noreferrer">HUAWEI DevEco Studio 产品页</a>，登录华为开发者账号后下载 DevEco Studio 安装包并进行安装。</p>
<p>3、启动 DevEco Studio，根据工具引导下载 HarmonyOS SDK。
<img src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/13e28858a6d74582980df9511e35bea2~tplv-k3u1fbpfcp-zoom-1.image" alt="下载HarmonyOS SDK" loading="lazy" referrerpolicy="no-referrer"></p>
<p>4、下载 HarmonyOS SDK 成功后会进入欢迎页，点击欢迎页中的 Configure > Settings 打开设置窗口，点击 Apparance&behavior > System settings > HarmonyOS SDK，选中 JS SDK 进行下载。
<img src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/0d7ed7c9afaa409681bce701c89b3ed1~tplv-k3u1fbpfcp-zoom-1.image" alt="下载JS SDK" loading="lazy" referrerpolicy="no-referrer"></p>
<p>至此开发环境安装完成。</p>
<h3 data-id="heading-3">2.2 新建项目</h3>
<p>点击菜单栏中的 File > New > New Project
<img src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/9cb1c0440094458cb5b07a89365713d5~tplv-k3u1fbpfcp-zoom-1.image" alt="新建项目" loading="lazy" referrerpolicy="no-referrer">
选择需要开发的项目设备，然后选择 Empty Featrue Ability(JS) 后，点击 Next，此时会出现项目的信息配置
<img src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/1b16d75259ab4ad59bb8b7f2644fd43d~tplv-k3u1fbpfcp-zoom-1.image" alt="项目信息配置" loading="lazy" referrerpolicy="no-referrer">
点击 Finish，一个新的项目就被创建出来了。</p>
<h3 data-id="heading-4">2.3 项目目录</h3>
<p><img src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/d361a48785e947df8d7e7dc109703b04~tplv-k3u1fbpfcp-zoom-1.image" alt="项目目录" loading="lazy" referrerpolicy="no-referrer"></p>
<p>使用 JS SDK 进行开发的话，需要关注的是 entry > src > main > js 文件夹，其中：</p>
<ul>
<li>
<p>i18n 目录下存放的是多语言的 json 文件</p>
<blockquote>
<p>en-US.json 为英文模式展示的内容
zh-CN.json 为中文模式展示的内容</p>
</blockquote>
</li>
<li>
<p>pages 下存放的是项目的多个页面，每个页面都由 hml、js 和 css 组成</p>
<blockquote>
<p>hml 文件定义了页面的布局、页面中用到的组件，以及这些组件的层级关系
js 文件定义了页面的业务逻辑，比如数据绑定、事件处理等
css 文件定义了 index 页面的样式</p>
</blockquote>
</li>
<li>
<p>app.js 中存放的是全局的 js 逻辑和 app 的生命周期管理</p>
</li>
</ul>
<p>除此之外，还可以自己创建 common 目录用于存放公共资源文件，比如：公共样式和公用方法。</p>
<h3 data-id="heading-5">2.4 生命周期</h3>
<p>生命周期分为应用的生命周期以及页面的生命周期</p>
<p><img src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/0ea269d117ef4ac6b94391b2a48bae8c~tplv-k3u1fbpfcp-zoom-1.image" alt="生命周期" loading="lazy" referrerpolicy="no-referrer"></p>
<p>其中，应用的生命周期主要分为应用创建时调用的 onCreate，以及应用销毁时触发的 onDestroy。</p>
<p>而页面的生命周期分为：</p>
<ul>
<li>onInit：页面数据准备完成时触发；</li>
<li>onReady：页面编译完成时触发；</li>
<li>onShow：页面展示时触发；</li>
<li>onHide：页面被隐藏时触发；</li>
<li>onDestroy：页面被销毁时触发；</li>
</ul>
<p>由于JS UI只支持应用同时运行并展示一个页面，因此当应用从页面 A 跳转到页面 B 时，首先触发页面 A 的 onHide、onDestroy 函数，然后依次调用页面 B 的 onInit、onReady、onShow 函数来初始化和显示页面 B。</p>

<h3 data-id="heading-9">3.3 媒体组件</h3>
<p>媒体组件暂时只有 <strong>video</strong> 组件一个，除了智能穿戴设备不支持外，手机、平板、智慧屏设备均支持该组件，它为设备提供了视频播放功能。</p>
<h3 data-id="heading-10">3.4 画布组件</h3>
<p>画布组件展示只有 <strong>canvas</strong> 组件一个，手机、平板、智慧屏、智能穿戴设备均支持该组件，它为设备提供了自定义绘制图形的能力。</p>
<h3 data-id="heading-11">3.5 栅格组件</h3>





















<table><thead><tr><th>组件名</th><th>描述</th></tr></thead><tbody><tr><td>grid-container</td><td>栅格布局容器根节点</td></tr><tr><td>grid-row</td><td>grid-row 是栅格布局容器 grid-container 的子容器组件，使用 flex 横向布局，排列每个 grid-col 容器，justify-content 与 align-items 默认为 flex-start，支持折行显示</td></tr><tr><td>grid-col</td><td>grid-row 的子容器组件</td></tr></tbody></table>
<p>栅格系统有 Margins, Gutters, Columns 三个属性：</p>
<ul>
<li>Margins：用于控制元素距离屏幕最边缘的距离</li>
<li>Gutters：用来控制元素和元素之间的距离关系</li>
<li>Columns：用来辅助布局的主要定位工具，不同的屏幕尺寸匹配不同的 Columns 数量来辅助布局定位，它会根据实际设备的宽度和 Columns 数量自动计算每一个 Columns 的宽度</li>
</ul>
<p><img src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/fa2e115969b549c1ba18dedf58a9b73e~tplv-k3u1fbpfcp-zoom-1.image" alt="栅格系统" loading="lazy" referrerpolicy="no-referrer"></p>
<p>不同的设备根据水平宽度 px，显示不同数量的栅格数：</p>
<p>xs : 0px < 水平分辨率 < 320px：2 Columns 栅格；</p>
<p>sm : 320px <= 水平分辨率 < 600px：4 Columns 栅格；</p>
<p>md : 600px <= 水平分辨率 < 840px：8 Columns 栅格；</p>
<p>lg : 840px <= 水平分辨率：12 Columns 栅格。</p>
<h2 data-id="heading-12">四、HML语法</h2>
<p>HML（HarmonyOS Markup Language）是一套类 HTML 的标记语言，通过组件，事件构建出页面的内容。页面具备事件绑定、数据绑定、列表渲染、条件渲染和逻辑控制等能力。</p>
