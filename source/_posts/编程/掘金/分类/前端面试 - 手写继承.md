<h2 data-id="heading-1">Object.create()</h2>
<blockquote>
<p><code>Object.create()</code> 方法创建一个新对象，使用现有的对象来提供新创建的对象的__proto__。 （请打开浏览器控制台以查看运行结果。）</p>
</blockquote>
<h2 data-id="heading-2">Object.assign()</h2>
<blockquote>
<p><code>Object.assign() </code> 方法用于将所有可枚举属性的值从一个或多个源对象分配到目标对象。它将返回目标对象。</p>
</blockquote>
<h2 data-id="heading-3">Function.prototype.call()、Function.prototype.apply()、Function.prototype.bind()</h2>
<p>区别：</p>
<ul>
<li>call()、apply()直接执行函数，bind()返回新函数，需要手动调用。</li>
<li>call()接受参数列表，apply()接受参数数组</li>
</ul>
<h1 data-id="heading-4">手写继承</h1>
<h2 data-id="heading-5">构造函数继承</h2>
<p>缺点：原型链上的方法和属性没办法继承</p>
<pre><code class="copyable">function Person() &#123; ... &#125; 
Person.prototype.name = 'Person'
function Student() &#123; 
    Person.call(this) // 执行Person，将this指向当前环境 
    ...
&#125;

const s1 = new Student()
console.log(s1.name) // undefined

<h2 data-id="heading-6">原型链继承</h2>
<p>缺点：实例化原型对象的属性是引用类型的时候，数据互相影响</p>
<pre><code class="copyable">function Person() &#123;
    this.arr = [1, 2, 3, 4]
&#125; 
function Student() &#123; ... &#125; 
Student.prototype = new Person() // 将构造函数的原型对象指向父函数的实例对象

const s1 = new Student()
const s2 = new Student()
s1.arr.push(5)

console.log(s1.arr) // [1, 2, 3, 4, 5]
console.log(s2.arr) // [1, 2, 3, 4, 5]

<h2 data-id="heading-7">组合继承</h2>
<p><a href="https://link.juejin.cn/?target=https%3A%2F%2Fdeveloper.mozilla.org%2Fzh-CN%2Fdocs%2FWeb%2FJavaScript%2FReference%2FGlobal_Objects%2FFunction%2Fapply" target="_blank" rel="nofollow noopener noreferrer" title="https://developer.mozilla.org/zh-CN/docs/Web/JavaScript/Reference/Global_Objects/Function/apply" ref="nofollow noopener noreferrer">参考文档</a></p>
<p>⚠️ 重点：</p>
<ul>
<li>执行父函数时，this 的指向问题</li>
<li>子函数原型对象指向，需构造新对象</li>
<li>修正指向</li>
</ul>
<pre><code class="copyable">// 继承单个对象
function Person() &#123; ... &#125; 
function Student() &#123; 
    Person.call(this) // 执行Person，将this指向当前环境 ... 
&#125; 
Student.prototype = Object.create(Person.prototype) // 利用Object.create创建新对象
Student.prototype.constructor = Student // 修正指向 

// 继承多个对象【混入】 
function Person() &#123; ... &#125; 
function Behavior() &#123; ... &#125; 
function Student() &#123; 
    Person.call(this) Behavior.call(this) 
    ... 
&#125; 
Student.prototype = Object.create(Person.prototype) 
Object.assign(Student.prototype, Behavior.prototype) // Object.create和Object.assign都是生成一个新的对象，来避免引用类型相互影响
Student.prototype.constructor = Student // 修正指向
<span class="copy-code-btn">复制代码</span></code></pre></div>
