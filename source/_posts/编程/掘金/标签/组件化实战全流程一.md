h2&#123;font-size:20px&#125;.markdown-body h3&#123;font-size:18px&#125;&#125;</style><h1 data-id="heading-0">组件仓库创建</h1>
<p><strong>1.创建远端仓库 以码云为例</strong></p>
<ul>
<li>创建索引仓库</li>
</ul>
<p><img src="https://p1-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/88732ce10dfb4622b5dbcbd31442b19e~tplv-k3u1fbpfcp-watermark.image" alt="image.png" loading="lazy" referrerpolicy="no-referrer"></p>
<ul>
<li>创建组件仓库</li>
</ul>
<p><img src="https://p9-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/77dc78ed07a048fa8918523d18d867ef~tplv-k3u1fbpfcp-watermark.image" alt="image.png" loading="lazy" referrerpolicy="no-referrer">
<strong>2.索引库关联本地</strong></p>
<pre><code class="copyable">cd ~/.cocoapods/repos
pod repo add GTCompSpec https://gitee.com/guochangzheng/gtcomp-spec.git
<span class="copy-code-btn">复制代码</span></code></pre>
<p><strong>3.创建文件夹</strong></p>
<pre><code class="copyable">GTCompTest
<span class="copy-code-btn">复制代码</span></code></pre>
<p><strong>4.cd文件夹到目录下</strong></p>
<pre><code class="copyable">cd ~/Desktop/GTCompTest
<span class="copy-code-btn">复制代码</span></code></pre>
<p><strong>5.创建本地组件模板</strong></p>
<pre><code class="copyable">pod lib create GTCompDemo
<span class="copy-code-btn">复制代码</span></code></pre>
<p><img src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/68240c1a88a945798c216a5333128ff1~tplv-k3u1fbpfcp-watermark.image" alt="image.png" loading="lazy" referrerpolicy="no-referrer">
创建完成之后会自动打开一个demo工程</p>
<ul>
<li>配置.podspec文件</li>
</ul>
<p><img src="https://p6-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/ea4a4a9eee7e4a2e87cdcc5767ea76d5~tplv-k3u1fbpfcp-watermark.image" alt="image.png" loading="lazy" referrerpolicy="no-referrer"></p>
<p>**6.添加远端组件仓库 **</p>
<pre><code class="copyable">#添加远程仓库
cd ~/Desktop/GTCompTest/GTCompDemo
git remote add origin https://gitee.com/guochangzheng/gtcomp-demo.git 
<span class="copy-code-btn">复制代码</span></code></pre>
<p><strong>7.本地组件库代码提交</strong></p>
<pre><code class="copyable">git push -u origin master #第一次可能会报错可尝试用 git push -u origin master -f 可能会覆盖远程的修改
git add . #记得后面一定要有 .
git commit -m "创建我的组件"
git push -u origin master
git tag '0.1.0' #注意:这里的tag号必须和.podSpec文件的版本号一致
git push --tags
<span class="copy-code-btn">复制代码</span></code></pre>
<p><strong>8.组件库验证</strong></p>
<pre><code class="copyable">本地验证
pod lib lint --use-libraries --allow-warnings
远端验证
pod spec lint --use-libraries --allow-warnings
<span class="copy-code-btn">复制代码</span></code></pre>
<p><strong>9.将spec 文件提交到本地的私有仓库，然后再push到远程仓库</strong></p>
<pre><code class="copyable">pod repo push [xxxSpec] [xxx].podspec --use-libraries --allow-warnings
例如：
pod repo push GTCompSpec GTCompDemo.podspec --use-libraries --allow-warnings
<span class="copy-code-btn">复制代码</span></code></pre>
<p><strong>10.查看私有仓库</strong></p>
<pre><code class="copyable">pod search GczMyTest
<span class="copy-code-btn">复制代码</span></code></pre>
