<h3 data-id="heading-1">注册主命令</h3>
<p>什么是主命令？</p>
<p>比如 <code>vue create app</code></p>
<p>vue 就是主命令，create 是子命令，app 就是option参数（这里指项目名称）</p>
<p>怎么实现？</p>
<ul>
<li>初始化项目</li>
</ul>
<p>其实就是创建个文件夹，切到目录<code>npm init -y </code>初始一下，完事～</p>
<p>(因为我叫阿远，我直接ayuan，嘻嘻～)</p>
<pre><code class="copyable">mkdir ayuan-cli
cd ayuan-cli
npm init -y
<span class="copy-code-btn">复制代码</span></code></pre>
<ul>
<li>创建index.js文件</li>
</ul>
<p>在项目中创建index.js,然后随便加一句话</p>
<pre><code class="copyable">console.log('hello~')
<span class="copy-code-btn">复制代码</span></code></pre>
<ul>
<li>修改package.json</li>
</ul>
<p>重点来了哈,在package.json中加上这句话</p>
<pre><code class="copyable">"bin": &#123;
  "ayuan": "index.js"
&#125;
<span class="copy-code-btn">复制代码</span></code></pre>
<p>再切到命令行，输入</p>
<pre><code class="copyable">npm link
<span class="copy-code-btn">复制代码</span></code></pre>
<p>其实就是把项目直接安装到本地，不需要发布到npm，再安装，非常的好用哈！</p>
<p><img src="https://p9-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/4b25a7dbdee842cbb89431b02262c9a1~tplv-k3u1fbpfcp-watermark.image" alt="C9401EA4B6F7E6053271A5F2B2CF49F4.jpg" loading="lazy" referrerpolicy="no-referrer">
这就完成命令注册！</p>
<p>不信，你试试！</p>
<p>提示一定是从</p>
<pre><code class="copyable">command not found: ayuan
<span class="copy-code-btn">复制代码</span></code></pre>
<p>到</p>
<pre><code class="copyable">syntax error near unexpected token `'hello~''
<span class="copy-code-btn">复制代码</span></code></pre>
<p>(我都替你测好几次了！)</p>
<h3 data-id="heading-2">使用node运行</h3>
<p>好了，出现上述的提示，其实就是咱们系统不知道他要用什么环境运行</p>
<p>怎么告诉他？在index.js的第一行，一定是第一行哈，加下面一句话</p>
<pre><code class="copyable">#!/usr/bin/env node
<span class="copy-code-btn">复制代码</span></code></pre>
<p>像这样！</p>
<p><img src="https://p6-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/137b7abad3224633b02e7742e1937357~tplv-k3u1fbpfcp-watermark.image" alt="image.png" loading="lazy" referrerpolicy="no-referrer"></p>
<p>你再切到命令行，输入<code>ayuan</code>,或者你用bin的注册的命令,一定会输出<code>hello~</code></p>
<p>如果不行，那就是你上述步骤做错了！（哎，我不管）</p>
<p><img src="https://p9-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/937246321e414ed689576e713b86634a~tplv-k3u1fbpfcp-watermark.image" alt="6F2B55C0EE5E6D0B1736FBFB69F0B5AD.jpg" loading="lazy" referrerpolicy="no-referrer"></p>
<h3 data-id="heading-3">注册参数命令事件</h3>
<p>注册参数命令事件，这其实可拆两步<code>解析参数</code>和<code>注册参数事件</code></p>
<ul>
<li>解析参数</li>
</ul>
<p>好，先说第一步，解析参数</p>
<p>比如，我们知道 vue-cli, 创建项目命令是<code>vue create project</code></p>
<p>其实可以认为<code>create</code>和<code>project</code>都是参数</p>
<p>怎么获取？可以使用<code>process.argv</code></p>
<p>我可以在这贴下文档（不然说我胡说八道）</p>
<p><img src="https://p6-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/ba41f9ca694343bda10bb1bf5d46a4ac~tplv-k3u1fbpfcp-watermark.image" alt="image.png" loading="lazy" referrerpolicy="no-referrer"></p>
<p>这样我们不就直接用<code>process.argv.slice(2)</code>就可获取参数了，注意是数组</p>
<ul>
<li>注册参数事件</li>
</ul>
<p>当然得使用<code>commander</code>库(vue-cli、create-react-app也在用哦)
可以贴下 vue-cli源码</p>
<pre><code class="copyable">npm install commander -S
<span class="copy-code-btn">复制代码</span></code></pre>
<p>安装完，快速coding</p>
<pre><code class="copyable">// 引入 Command
const &#123; Command &#125; = require('commander')
const program = new Command()

// 获取 package.json 版本
const version = require('./package.json').version

program.version(version) // 注册版本
    .usage('<command> [options]') // 使用命令格式
    
program.parse(process.argv); // 解析参数，一定放最后    
<span class="copy-code-btn">复制代码</span></code></pre>
<p>这时候，就可以输入</p>
<pre><code class="copyable">ayuan --version
<span class="copy-code-btn">复制代码</span></code></pre>
<p>看看有无版本号输出，有得话说明commander使用成功</p>
<p>接下就是注册命令</p>
<pre><code class="copyable">program.command('init <name>')
    .option('-d, --default', '默认配置', false) // 第一是想要注册参数， 第二是参数是描述， 第三是默认值
    .action((name, options) => &#123;
        console.log(name)
        console.log(options)
    &#125;)
<span class="copy-code-btn">复制代码</span></code></pre>
<p>这行代码一定要放在</p>
<p><img src="https://p6-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/8161cb54677f49699beda1f3ab51066a~tplv-k3u1fbpfcp-watermark.image" alt="image.png" loading="lazy" referrerpolicy="no-referrer"></p>
<p>不然代码不会生效哈！</p>
<p>我们使用<code>ayuan init test</code> 和 <code>ayuan init test -d</code></p>
<p>就会很轻松看到</p>
