                                                                                            <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">Bytebase 是一个基于网络、零配置、无依赖的数据库 Schema 变更和版本控制管理工具，适用于开发人员和 DBA。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">Bytebase 0.11.0 发布，更新内容如下：</p> 
<h3 style="margin-left:.6em; margin-right:0; text-align:start">新功能：</h3> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><strong>SQL 编辑器，提供了如下一些功能（预览版）：</strong></p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>带智能提示和右键菜单的 SQL 编辑器</li> 
 <li>执行 SQL 语句 (仅允许 SELECT ) ，并以表格形式展现执行结果</li> 
 <li>通过左侧树形面板，在不同数据连接间进行切换。</li> 
 <li>左下面板展示表的 Schema 信息</li> 
</ul> 
<h3 style="margin-left:.6em; margin-right:0; text-align:start">改进：</h3> 
<ul style="margin-left:0; margin-right:0"> 
 <li>为 kbar 提供了一条快速入门。通过 cmd/ctrl + k 快捷键可以打开 kbar 命令面板，提供了快捷的导航功能。</li> 
 <li>把 VCS 模块重构成了插件架构。可以相对容易地支持其他 VCS 系统，比如 GitHub, BitBucket。</li> 
</ul> 
<h3 style="margin-left:.6em; margin-right:0; text-align:start">错误<strong>修复：</strong></h3> 
<ul style="margin-left:0; margin-right:0"> 
 <li>修复了 GitOps workflow 时，发生数据库 Schema 变更后，无法把最新的 Schema 回写到 Git 仓库的问题。</li> 
</ul> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">更多详情可查看：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fbytebase%2Fbytebase%2Freleases%2Ftag%2F0.11.0" target="_blank">https://github.com/bytebase/bytebase/releases/tag/0.11.0</a></p>
