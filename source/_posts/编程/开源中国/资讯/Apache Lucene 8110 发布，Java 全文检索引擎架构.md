                                                        <p>Apache Lucene 8.11.0 已发布，Lucene 是完全用 Java 编写的高性能、功能齐全的全文检索引擎架构，提供了完整的查询引擎和索引引擎、部分文本分析引擎。目的是为软件开发人员提供一个简单易用的工具包，以方便地在目标系统中实现全文检索的功能，或者是以此为基础建立起完整的全文检索引擎。</p> 
<p>此版本增加了多项新功能，以及其他优化和错误修复。</p> 
<p>更新亮点：</p> 
<ul> 
 <li>在累计所有文档的 facet count 计数时，facet 支持忽略已删除的文档</li> 
 <li>支持同时运行 CheckIndex</li> 
 <li>显着提升 MemoryIndex 在内存中存储检索条数的 RAM 效率，并减少 IndexWriter 内部检索条数簿记中的一些 RAM 开销</li> 
 <li>使用 3-ways 分区改进 IntroSorter</li> 
 <li>Make QueryCache respect Accountable queries.</li> 
 <li>……</li> 
</ul>
