                                                        <p>Go 1.19 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Ftip.golang.org%2Fdoc%2Fgo1.19" target="_blank">发布</a>了第一个 RC 版本，正式版预计在 8 月推出。</p> 
<p><img src="https://static.oschina.net/uploads/space/2022/0714/160404_X1UL_2720166.png" referrerpolicy="no-referrer"></p> 
<p><strong>主要变化</strong></p> 
<ul> 
 <li>针对“泛型”的 Bugfix</li> 
 <li>更新内存模型，与 C、C++、Java、JavaScript、Rust 和 Swift 使用的内存模型保持一致</li> 
 <li>Go 1.19 在启动时将默认提高打开文件的限制值</li> 
 <li>Go 1.19 race detector 将升级到 v3 版 thread sanitizer</li> 
 <li>Go 1.19 增加"unix" build tag</li> 
 <li>在 Linux 上，Go 正式支持 64 位龙芯 CPU 架构 (GOOS=linux, GOARCH=loong64)</li> 
 <li>Go 1.19 的 Go 编译器使用 jump table 重新实现了针对大整型数和 String 类型的 switch 语句，平均性能提升 20% 左右</li> 
</ul>
