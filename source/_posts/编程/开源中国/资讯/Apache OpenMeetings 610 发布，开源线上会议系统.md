                                                                                            <p>Apache OpenMeetings 6.1.0 现已发布。Apache Openmeetings 提供视频会议、即时通讯、白板、协作文档编辑等组件软件工具，使用 Kurento Media Server 的 API 功能来实现远程传输和流传输。</p> 
<p><strong>主要更新内容</strong></p> 
<ul> 
 <li>bug 修复 
  <ul> 
   <li>音视频测试应用程序不适用于 Safari</li> 
   <li>文件下载失败</li> 
   <li>工具显示不正确</li> 
   <li>会话意外终止</li> 
   <li>房间关闭时显示无效对话框</li> 
   <li>源工件包含冗余文件</li> 
   <li>使用户在自己的会议室和演示室中成为默认主持人</li> 
   <li>限制谷歌分析字段以避免溢出</li> 
   <li>演示室中的空用户列表问题</li> 
   <li>OM 不适用于 Android Chrome</li> 
   <li>将语言更改为日语时部分日历未翻译</li> 
   <li>日历选择器无法同步到大日历</li> 
   <li>从日历收到的电子邮件链接不起作用</li> 
  </ul> </li> 
 <li>新功能 
  <ul> 
   <li>添加通过 Admin->Config 选择主题的能力</li> 
  </ul> </li> 
 <li>优化 
  <ul> 
   <li>优化管理员权限</li> 
   <li>优化上传文件图标的功能</li> 
   <li>支持配置 Kurento::WebRtcEndpoint.CertificateKeyType</li> 
   <li>在管理->组中标记默认组</li> 
  </ul> </li> 
</ul>
