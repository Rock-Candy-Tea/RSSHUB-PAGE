                                                                                            <p>ShareX 13.6.0 发布了。ShareX 是一个开源截图工具，可捕获或记录屏幕的任何区域，并一键共享。 它还允许将图像、文本或其他类型的文件上传到超过 80 个支持的存储服务上。</p> 
<ul> 
 <li>ShareX 现在使用 .NET Framework 4.8，如果系统上没有这个框架将会自动下载</li> 
 <li>更新 Inno Setup 到 6.2.0，它具有现代化的图形</li> 
 <li>更新 Inno Setup Dependency Installer 脚本，它现在使用新的内置 Inno Setup 下载器来下载 .NET Framework</li> 
 <li>增加了 "Glow" 图像效果，可以在屏幕截图周围增加渐变的光效</li> 
 <li>删除了主窗口中的社交按钮工具栏和新闻按钮</li> 
 <li>增加了 Twitter 按钮，它将取代以前的新闻面板，如果你想收到 ShareX 的相关新闻或提示，请务必关注 Twitter 账户</li> 
 <li>增加了 Discord 按钮，可以加入官方的 Discord 服务器，获得社区的支持</li> 
 <li>增加了 "菜单字体 "主题选项，并将菜单和上下文菜单的默认字体大小改为 9.75pt</li> 
 <li>在主窗口的上下文菜单中增加了 Bing 视觉搜索按钮，这既是一个图像搜索，也是一个 OCR 服务</li> 
 <li>在主窗口上下文菜单中增加了快速图像组合按钮。 
  <ul> 
   <li>你必须通过按住Ctrl或Shift从主窗口选择多个图像，然后选择多个缩略图，才能在右键菜单中看到这些按钮。</li> 
  </ul> </li> 
 <li>当截图时，窗口的标题和活动窗口的进程名称将作为元标签保存在历史中 
  <ul> 
   <li>这将大大改善 ShareX 历史窗口中的截图搜索</li> 
  </ul> </li> 
 <li>增加了一个自动调整 JPEG 质量的选项，以适应目标文件大小下的最佳质量</li> 
 <li>重新组织了工具菜单，将类似的工具组合在一起，并在它们之间设置分隔符</li> 
 <li>Tweet 信息从工具菜单移至上传菜单</li> 
 <li>改进了自定义上传器的错误信息，以包括更具体的错误信息</li> 
 <li>为自定义上传器参数值字段添加了多行支持</li> 
 <li>在自定义上传器中允许空的 multipart/form-data 值</li> 
 <li>在响应窗口增加了复制 URL 按钮</li> 
 <li>在历史窗口增加了简单的搜索栏，可以通过文件名、窗口标题和进程名搜索</li> 
 <li>图片历史窗口的搜索栏现在也支持搜索元标签</li> 
 <li>在历史窗口的工具栏上增加了高级搜索按钮，可以切换具有更多过滤选项的面板</li> 
 <li>由于过滤选项被移到了高级搜索中，图片预览现在填充了右边的位置</li> 
 <li>在历史窗口的工具栏上增加了 "复制统计信息到剪贴板" 按钮</li> 
 <li>删除了 <a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fge.tt" target="_blank">ge.tt</a> 文件上传器，因为它已经不再使用了</li> 
</ul> 
<p>更多详情可查看：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgetsharex.com%2Fchangelog%2F" target="_blank">https://getsharex.com/changelog/</a></p>
