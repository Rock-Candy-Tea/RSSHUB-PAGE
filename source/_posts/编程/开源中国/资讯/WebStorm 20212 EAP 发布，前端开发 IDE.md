                                                        <p>WebStorm 2021.2 EAP 现已发布，这是其 2021 年的第二个 EAP 版本。</p> 
<p><img alt height="800" src="https://oscimg.oschina.net/oscnet/up-5bb00dff89e3ad659354568f17da7ab8b2f.png" width="1600" referrerpolicy="no-referrer"></p> 
<h4>浏览器页面将在保存时重新加载</h4> 
<p>WebStorm 现在可以根据用户对相应的 HTML 文件或链接的 CSS 和 JavaScript 文件所做的修改，自动重新加载在浏览器中打开的页面。只要在 IDE 中自动或使用 Ctrl+S 保存更改，或在外部对文件进行更改，就会触发重新加载。</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-79c5c9dfe5d7c27561b9c01dfce0ca21482.gif" referrerpolicy="no-referrer"></p> 
<h4><span style="background-color:#ffffff"><span style="color:#182026">基于函数调用的参数类型的补完</span></span></h4> 
<p><span style="background-color:#ffffff"><span style="color:#182026">对于函数体中的参数，WebStorm 现在将根据函数调用中使用的类型显示完成建议。</span></span></p> 
<p><span style="background-color:#ffffff"><span style="color:#182026"><img alt height="700" src="https://oscimg.oschina.net/oscnet/up-2f80d0ce3e157e4b3802c82e7bd6a8a45d3.png" width="1400" referrerpolicy="no-referrer"></span></span></p> 
<h4><span style="background-color:#ffffff"><span style="color:#182026">添加箭头功能的新动作</span></span></h4> 
<p><span style="background-color:#ffffff"><span style="color:#182026">WebStorm 添加了一个新的实时模板，以帮助用户更快地添加箭头函数，如下所示。</span></span></p> 
<p><span style="background-color:#ffffff"><span style="color:#182026"><img alt height="700" src="https://oscimg.oschina.net/oscnet/up-a504b0415d8fed0589f830b3e7e415d6872.gif" width="1400" referrerpolicy="no-referrer"></span></span></p> 
<h4><span style="background-color:#ffffff"><span style="color:#182026">优化 “Preferences/Settings” 对话框</span></span></h4> 
<p><span style="background-color:#ffffff"><span style="color:#182026">现在在 Preferences/Settings 对话框中，有一个新的高级设置节点。这个节点包含一些额外的配置选项，它们中的大多数是从注册表中转移过来的。</span></span></p> 
<p><span style="background-color:#ffffff"><span style="color:#182026"><img alt height="700" src="https://oscimg.oschina.net/oscnet/up-f06d8e188e984d96aeef6a45f0cd51c2838.png" width="1400" referrerpolicy="no-referrer"></span></span></p> 
<p><span style="background-color:#ffffff"><span style="color:#182026">此外，用户现在可以在 Preferences/Settings 对话框中打开的部分之间来回导航。</span></span></p> 
<p><span style="background-color:#ffffff"><span style="color:#182026"><img alt src="https://oscimg.oschina.net/oscnet/up-73111d4c6b968976c2af8ab8a88e0720349.gif" referrerpolicy="no-referrer"></span></span></p> 
<h4><span style="background-color:#ffffff"><span style="color:#182026">自动缓存和日志清除</span></span></h4> 
<p><span style="background-color:#ffffff"><span style="color:#182026">现在，在每次重大更新之后，IDE会自动清理最后一次更新时间超过 180天 的任何缓存和日志目录。系统设置和插件目录将保持不变。要手动触发这一过程，请在主菜单中选择 "Help | Delete Leftover IDE Directories..."。</span></span></p>
