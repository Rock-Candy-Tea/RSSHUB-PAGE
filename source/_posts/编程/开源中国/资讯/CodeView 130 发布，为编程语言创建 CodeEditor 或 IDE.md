                                                                                            <p>CodeView 1.3.0 现已发布。CodeView 是一个Android 库，可以轻松创建一个支持任何语言和主题的 <span style="background-color:#ffffff; color:#24292f">Code editor<span> </span></span>或 IDE。具有自动完成、自动缩进、片段和更多的功能。</p> 
<p>具体更新内容如下：</p> 
<ul> 
 <li>提高 drawing line number 性能</li> 
 <li>改进自动完成实现以修复多尺寸下拉弹出位置</li> 
 <li>添加<code>setMaxSuggestionsSize</code>以限制 suggestion 的数量</li> 
 <li>添加<code>setAutoCompleteItemHeightInDp</code>获取自动完成列表项以更改下拉最终大小</li> 
 <li>添加对自动配对完成的支持</li> 
 <li>改进自动缩进实现并能够支持 python 缩进（Issue <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FAmrDeveloper%2FCodeView%2Fissues%2F9" target="_blank">#9</a>）</li> 
</ul>
