                                                                                            <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">Microsoft PowerToys 是 Windows 系统实用程序，供高级用户调整和简化其 Windows 体验，可最大限度地提高生产力。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">在上个版本中（v0.58），大部分用于原生在 ARM64 上运行的工作都已完成。在 v0.59 版本的发布周期中，完成了原生 ARM64 的构建，并发布了第一个构建。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">以下是其中的一些亮点：</p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>在 ARM64 上原生运行的工作已经完成，并发布了一个构建</li> 
 <li>Power Rename 现在在 WinUI 3 上运行</li> 
 <li>Keyboard Manager 现在最多允许 4 个修饰键作为快捷键，并得到了一些质量方面的修正</li> 
 <li>升级了 Windows App SDK 运行时至 1.1.0，修复了 UAC 关闭时设置无法启动的问题，并提高了性能</li> 
 <li>Windows App SDK 运行时二进制文件正与 PowerToys 一起提供，这应该可以解决 WinAppSDK 报告的安装问题</li> 
</ul> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">已知问题</p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>有报告称，用户无法打开设置窗口。这是由一些应用程序的不兼容引起的，如果受此影响，请查看链接中的问题，以验证所提出的解决方案是否有效。（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmicrosoft%2FPowerToys%2Fissues%2F18015" target="_blank">链接</a>）</li> 
</ul>
