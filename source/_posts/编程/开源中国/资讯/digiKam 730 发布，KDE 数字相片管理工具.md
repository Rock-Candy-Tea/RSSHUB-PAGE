                                                                                            <p>digiKam 是 KDE 桌面环境的影像管理和编辑程序，支持所有主要图像格式，并可以组织目录为基础的照片收藏，或按日期、时间、或标签的动态相册。用户还可以对图像添加标题和注释，搜索他们和透过智能文件夹保存搜索。添加插件还可以输出到 Flickr 的相册、Gallery2、谷歌地球的 KML 文件、Simpleviewer、刻录成光盘或创建 Web 画廊。</p> 
<p>近日，digiKam 7.3.0 正式发布，本次更新的部分内容如下：</p> 
<p><strong>新功能：</strong></p> 
<ul> 
 <li>在 ExifTool 基础上增加新的元数据查看器；</li> 
 <li>更好地支持天体照片 FITS 图像格式，包括支持元数据提取以填充数据库；</li> 
 <li>更好地支持 MPO 图像格式，包括支持元数据提取以填充数据库；</li> 
 <li>Digital Scanner 插件现在可以在 Windows下使用；</li> 
 <li>在所有支持 Base Media File 元数据读写（HEIF、AVIF）的捆绑包中使用最新的 Exiv2 0.27.4；</li> 
 <li>在所有捆绑包中使用最新的 KF5 5.80 框架；</li> 
 <li>更新内部 Libpgf 至 7.21.7 (18-02-2021)；</li> 
 <li>从快照 20210622 更新内部 Libraw；</li> 
 <li>更新内部 DNG SDK 到最新的 1.5 版本，XMP SDK 到最新的 5.7.0 版本；</li> 
 <li>DNGWriter 现在使用 ExifTool 来后期处理元数据；</li> 
 <li>内部 Libraw 使用 DNG SDK 1.5 来处理 8 bits DNG 文件；</li> 
 <li>查找重复的项目现在可以使用多核并行处理；</li> 
 <li>插件：新的工具可以将项目导出到 iNaturalist 网络服务；</li> 
 <li>插件：幻灯片和演示工具现在允许移动项目到垃圾桶；</li> 
 <li>插件：为 Showfoto 新增了将 RAW 文件批量转换为 DNG 的工具；</li> 
</ul> 
<p><strong>Bug 修复：</strong></p> 
<ul> 
 <li>Yolov3 在 png 图片上无法正常工作；</li> 
 <li>重命名不支持逗号和分号；</li> 
 <li>检查新版本时没有找到更新版本的问题；</li> 
 <li>从 digiKam 7.1 升级到 7.2 后，digiKam 无法启动；</li> 
 <li>Libksane 无法为 Windows 交叉编译；</li> 
 <li>导入菜单中没有从扫描仪获取图像的选项；</li> 
 <li>digiKam 因 seg 错误而崩溃；</li> 
 <li>digiKam 使用数字扫描仪时崩溃；</li> 
 <li>digiKam 在试图保存扫描的图像时崩溃；</li> 
 <li>标签管理器工具条上的图标按钮无响应；</li> 
</ul>
