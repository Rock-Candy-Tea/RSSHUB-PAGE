                                                                                            <p>KUI for Vue v3.2.10 已经发布，基于 Vue 2.0 的 UI 组件库</p> 
<p>此版本更新内容包括：</p> 
<ul> 
 <li>新增 Skeleton 组件</li> 
 <li>新增 Descriptions 组件</li> 
 <li>修复 Page 组件翻页bug</li> 
 <li>新增分组功能 优化 Icon</li> 
 <li>新增一些图标</li> 
 <li>图标支持Sprite 渲染模式</li> 
 <li>支持自定义线宽</li> 
 <li>Progress 新增 strokeHeight 可设置进度条高度</li> 
 <li>新增 strokeWidth 可设置圆形精度条宽度</li> 
 <li>新增 gapDegree 可设置dashboard开口大小</li> 
 <li>新增 strokeLinecap 可设置线条节点型状</li> 
 <li>新增可自定义子节点</li> 
 <li>修复 Table 组件一些细节问题</li> 
 <li>Spin 组件新增防抖属性</li> 
</ul> 
<p>文档：<a href="https://gitee.com/link?target=https%3A%2F%2Fk-ui.cn" target="_blank">https://k-ui.cn</a></p> 
<p>详情查看：<a href="https://gitee.com/chuchur/kui-vue/releases/v3.2.10">https://gitee.com/chuchur/kui-vue/releases/v3.2.10</a></p>
