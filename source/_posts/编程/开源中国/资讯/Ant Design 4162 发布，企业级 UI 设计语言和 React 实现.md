                                                        <p>Ant Design 4.16.2 发布了。Ant Design 是一套企业级的 UI 设计语言和 React 实现，使用 TypeScript 构建，提供完整的类型定义文件，自带提炼自企业级中后台产品的交互语言和视觉风格、开箱即用的高质量 React 组件与全链路开发和设计工具体系。</p> 
<p>此版本更新内容如下：</p> 
<ul> 
 <li>修复 Menu <code>horizontal</code> 模式下的自动省略布局会被 <code>float</code> 破坏的问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F30879" target="_blank">#30879</a></li> 
 <li>Typography 
  <ul> 
   <li>修复 Typography.Paragraph 的 <code>copyable</code> 会将 React.Fragment 当做数组处理的问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F30869" target="_blank">#30869</a> </li> 
   <li>修复 Typography 自定义 <code>font-size</code> 时省略宽度不正确的问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F30840" target="_blank">#30840</a></li> 
  </ul> </li> 
 <li>修复 Progress <code>type="circle"</code> 有时会报错 <code>Rendered more hooks than during the previous render</code> 的问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F30855" target="_blank">#30855</a></li> 
 <li>优化 Tooltip 内容的宽度。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F30822" target="_blank">#30822</a></li> 
 <li>Table 
  <ul> 
   <li>优化 Table 中全选复选框的状态显示逻辑。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F30811" target="_blank">#30811</a></li> 
   <li>修复 Table 列筛选和排序时 <code>column.title</code> 设置 <code>onClick</code> 和 <code>ellipsis</code> 无效的问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F30799" target="_blank">#30799</a></li> 
  </ul> </li> 
 <li>修复 Dropdown 条目包含超链接时的颜色样式，另外现在点击图标也能触发超链接效果。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F30802" target="_blank">#30802</a></li> 
 <li>移除重复的通配符 reset 样式以优化样式渲染性能。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F30660" target="_blank">#30660</a></li> 
 <li>TypeScript 
  <ul> 
   <li>调整 Form, Input, Locale, PageHeader 组件类型定义使其更精确。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F30776" target="_blank">#30776</a></li> 
  </ul> </li> 
</ul>
