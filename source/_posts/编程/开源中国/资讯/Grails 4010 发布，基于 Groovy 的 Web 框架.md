                                                                                            <p>Grails 是一套用于快速 Web 应用开发的老牌开源框架，它基于 Groovy 编程语言，并构建于 Spring、Hibernate 和其它标准 Java 框架之上，从而为大家带来一套能实现超高生产力的一站式框架。</p> 
<p>Grails 4.0.10 已经发布，本次更新信息如下：</p> 
<h4>变化</h4> 
<ul> 
 <li>修正 GrailsResourceUtils.getPathFromBaseDir 方法的实现；</li> 
 <li>修复了 compileGroovyPages 的 GSP 配置，请确保在 PR 合并后以最新的 grails-gsp 为目标这样才会有效；</li> 
 <li>交互式 Rebase 工作流程；</li> 
 <li>include 标签的 url 是用冗余的 contextPath 计算的；</li> 
 <li>配置自动 Rebase 工作流程；</li> 
</ul> 
<h4>依赖项升级</h4> 
<ul> 
 <li>将 Grails GSP 从 4.0.2 升级到 4.0.3；</li> 
</ul>
