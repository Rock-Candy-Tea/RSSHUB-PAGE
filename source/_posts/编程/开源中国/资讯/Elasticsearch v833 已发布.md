   <p style="margin-left:0px">Elasticsearch 是一个基于 Lucene 库的搜索引擎。它提供了一个分布式、支持多租户的全文搜索引擎，具有 HTTP Web 接口和无模式 JSON 文档。Elasticsearch 基于 Java 开发，并在 SSPL + Elastic License 双重授权许可下作为开源软件发布。</p> 
   <p style="margin-left:0px">Elasticsearch 8.3.3 现已发布，该版本更新内容如下：</p> 
   <h3 style="margin-left:0px"><strong>Bug修复</strong><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Felastic%2Felasticsearch%2Fedit%2F8.3%2Fdocs%2Freference%2Frelease-notes%2F8.3.3.asciidoc" target="_blank"><strong>编辑</strong></a></h3> 
   <p><strong>基础设施/核心</strong></p> 
   <ul> 
    <li>添加 <code>build_flavor</code> 到 <span style="color:#212529">info</span> API REST 响应 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Felastic%2Felasticsearch%2Fpull%2F88336" target="_blank">#88336</a>（问题：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Felastic%2Felasticsearch%2Fissues%2F88318" target="_blank">#88318</a>）</li> 
   </ul> 
   <p><strong>映射</strong></p> 
   <ul> 
    <li>仅在运行脚本时强制执行最大值限制 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Felastic%2Felasticsearch%2Fpull%2F88295" target="_blank">#88295</a></li> 
   </ul> 
   <p><strong>监控</strong></p> 
   <ul> 
    <li>将 cgroup 内存字段切换为关键字 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Felastic%2Felasticsearch%2Fpull%2F88260" target="_blank">#88260</a></li> 
   </ul> 
   <p><strong>打包</strong></p> 
   <ul> 
    <li>修复 Docker 位置参数传递 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Felastic%2Felasticsearch%2Fpull%2F88584" target="_blank">#88584</a></li> 
   </ul> 
   <p><strong>安全</strong></p> 
   <ul> 
    <li>确保 <code>CreateApiKey</code> 创建新文档 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Felastic%2Felasticsearch%2Fpull%2F88413" target="_blank">#88413</a></li> 
   </ul> 
   <h3 style="margin-left:0px"><strong>增强功能</strong></h3> 
   <p><strong>安全</strong></p> 
   <ul> 
    <li>在 OIDC 反向通道中关闭空闲连接的新设置 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Felastic%2Felasticsearch%2Fpull%2F87773" target="_blank">#87773</a></li> 
   </ul> 
   <h3 style="margin-left:0px"><strong>升级</strong></h3> 
   <p><strong>打包</strong></p> 
   <ul> 
    <li>升级到 OpenJDK 18.0.2+9 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Felastic%2Felasticsearch%2Fpull%2F88675" target="_blank">#88675</a>（问题：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Felastic%2Felasticsearch%2Fissues%2F88673" target="_blank">#88673</a>）</li> 
   </ul> 
   <p>更新公告：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.elastic.co%2Fguide%2Fen%2Felasticsearch%2Freference%2F8.3%2Frelease-notes-8.3.3.html" target="_blank">https://www.elastic.co/guide/en/elasticsearch/reference/8.3/release-notes-8.3.3.html</a></p>
