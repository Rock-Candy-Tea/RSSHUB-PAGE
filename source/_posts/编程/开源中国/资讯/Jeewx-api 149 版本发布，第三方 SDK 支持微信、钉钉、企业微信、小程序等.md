                                                                                            <h3 style="margin-left:0; margin-right:0; text-align:left">项目介绍</h3> 
<blockquote> 
 <p style="margin-left:0; margin-right:0">JEEWX-API 是一款 JAVA 版的微信开发 SDK，支持微信公众号、小程序、微信企业号、支付宝生活号 SDK 和微博 SDK。你可以基于她，快速的傻瓜化的进行微信开发、支付窗和微博开发。 基于 jeewx-api 开发，可以立即拥有简单易用的 API，让开发更加轻松自如，节省更多时间。</p> 
</blockquote> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><strong>当前版本</strong>：v1.4.9 | 2022-09-02</p>

<h3 style="margin-left:0; margin-right:0; text-align:left">升级日志</h3> 
<blockquote> 
 <p style="margin-left:0; margin-right:0">第三方 APP 开发 SDK，支持微信、钉钉、企业微信、小程序等</p> 
</blockquote> 
<ul style="list-style-type:disc; margin-left:0; margin-right:0"> 
 <li>【钉钉】新增 OAuth2 登录 API</li> 
 <li>【企业微信】新增 OAuth2 登录 API</li> 
 <li>【钉钉】新增 ActionCard 消息</li> 
 <li>【企业微信】新增 TextCard 消息</li> 
 <li>【企业微信】新增手机号获取 userid 接口</li> 
 <li>【钉钉】修复 JSON 转换 java 泛型类不正确的问题</li> 
 <li>新增钉钉 API——DepartmentAPI（部门）</li> 
 <li>新增钉钉 API——MessageAPI（消息）</li> 
 <li>新增钉钉 API——UserAPI（用户）</li> 
 <li>新增钉钉 API——BaseAP</li> 
 <li>钉钉和企业微信支持发送 Markdown 消息</li> 
 <li>钉钉新增获取全部部门接口，包含返回失败的信息</li> 
 <li>钉钉和企业微信支持网页授权登录获取到的 code 来获取用户详情</li> 
 <li>支持发送文本卡片消息</li> 
 <li>fastjson 挖矿漏洞问题</li> 
 <li>升级底层依赖与 jeecgboot 同步</li> 
 <li>群发上传媒体资源接口地址换成最新 https 的</li> 
 <li>sun.misc.BASE64Decoder 过时方法改造</li> 
 <li>删除百度云 API 代码</li> 
</ul>
