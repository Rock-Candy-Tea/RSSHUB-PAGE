                                                                                            <p>OBS Studio 是一款免费且开源的用于视频录制以及直播串流的软件，提供实时高性能的视频/音频捕捉与混合，以及无限的场景模式帮助用户通过自定义实现无缝转换。</p> 
<p>OBS Studio 27.2.1 发布，该版本更新内容包括：</p> 
<ul> 
 <li>修正了一个对某些用户来说会引发系统完全冻结的问题</li> 
 <li>提高从浏览器源码构建的最新 Chromium 的性能</li> 
 <li>修正了使用某些旧的 Windows 版本的用户无法以很高的比特率进行流式传输问题</li> 
 <li>修正了一些相机在 Linux 上不能正确解码的问题</li> 
 <li>修正了一个常见的程序内快捷键有时会停止工作的问题</li> 
 <li>修正了 Windows 8.1 上桌面音频捕获停止工作的错误</li> 
 <li>修正了 SRT 和 RIST 在未设置 URL 或禁用 RIST URL 的媒体源时使 OBS 崩溃的问题</li> 
 <li>修正了 Linux 上 PipeWire 捕获的内存泄漏问题</li> 
 <li>再次更新了 AOM/SVT 编码器</li> 
 <li>修正了 macOS 上的一个问题，即 FFmpeg 的依赖文件名不必要地复杂，破坏了一些第三方插件</li> 
 <li>修正了在 macOS 上，如果 OBS 从某些路径启动，浏览器插件无法加载的问题</li> 
 <li>……</li> 
</ul>
