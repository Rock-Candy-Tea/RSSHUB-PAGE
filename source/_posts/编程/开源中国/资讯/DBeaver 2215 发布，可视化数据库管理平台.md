                                                                                            <p>DBeaver 是一个免费开源的通用数据库工具，适用于开发人员和数据库管理员。</p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">DBeaver 22.1.5 现已发布，更新内容如下：</p> 
<ul style="margin-left:0; margin-right:1.5em"> 
 <li>SQL editor：脚本连接关联问题已修复</li> 
 <li>Data editor： 
  <ul style="margin-left:0; margin-right:0"> 
   <li>Data editor 中的主要内存泄漏已修复</li> 
   <li>分组面板清理问题已解决</li> 
   <li>Hex editor 现在 respect 列的只读状态</li> 
   <li>查找/替换对话框现在 respect focused control</li> 
  </ul> </li> 
 <li>General： 
  <ul style="margin-left:0; margin-right:0"> 
   <li>连接配置加载和编辑问题已修复</li> 
   <li>对话框按钮布局在不同的对话框中得到修复</li> 
   <li>认证配置文件配置 UI 已修复</li> 
  </ul> </li> 
 <li>SQL Server： 
  <ul style="margin-left:0; margin-right:0"> 
   <li>已为 Azure 数据库修复了视图定义读取</li> 
   <li>nvarchar/nchar 列长度检测问题已修复</li> 
  </ul> </li> 
 <li>MySQL：数字模式名称的问题已修复</li> 
 <li>Oracle： 
  <ul style="margin-left:0; margin-right:0"> 
   <li>Trigger DDL 已修复（CREATE 关键字）</li> 
   <li>数据类型 DDL 生成已修复</li> 
  </ul> </li> 
 <li>PostgreSQL：复合数据类型的过滤器已修复</li> 
</ul>
