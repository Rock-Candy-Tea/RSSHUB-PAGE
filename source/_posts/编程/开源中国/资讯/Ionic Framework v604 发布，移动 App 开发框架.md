                                                                                            <p><span style="background-color:#ffffff; color:#000000">Ionic Framework 是一个开源的移动应用程序开发框架，拥有强大的跨平台 UI 工具包，通过使用 HTML、CSS 和 JavaScript 就能够构建高质量原生的 iOS、Android 和 Progressive Web App。</span></p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">Ionic Framework v6.0.4 正式发布，本次更新内容如下：</p> 
<ul style="margin-left:0; margin-right:0"> 
 <li><strong>accordion:</strong><span> </span>内容内的项目现在正确显示边框<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fionic-team%2Fionic-framework%2Fissues%2F24618" target="_blank">#24618</a>)</li> 
 <li><strong>angular:<span> </span></strong>具有 null 值的 routerLink 适用于 Angular 13(<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fionic-team%2Fionic-framework%2Fissues%2F24622" target="_blank">#24622</a>) </li> 
 <li><strong>col:</strong><span> </span>col 在非浏览器环境下运行不再报错 (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fionic-team%2Fionic-framework%2Fissues%2F24603" target="_blank">#24603</a>) </li> 
 <li><strong>datetime:</strong><span> </span>使用 h23 的日期时间语言环境将尊重最大时间范围 (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fionic-team%2Fionic-framework%2Fissues%2F24610" target="_blank">#24610</a>) </li> 
 <li><strong>datetime:</strong><span> </span> timepicker popover 将相对于点击目标定位 (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fionic-team%2Fionic-framework%2Fissues%2F24616" target="_blank">#24616</a>) </li> 
 <li><strong>input:</strong><span> </span>ion-input 接受任何字符串值 (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fionic-team%2Fionic-framework%2Fissues%2F24606" target="_blank">#24606</a>) </li> 
 <li><strong>item:</strong><span> </span>标签文本与输入文本对齐 (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fionic-team%2Fionic-framework%2Fissues%2F24620" target="_blank">#24620</a>)</li> 
 <li><strong>item:</strong><span> </span><span style="color:#2e3033">匹配<span> </span></span>material design<span> </span><span style="color:#2e3033">字符计数器</span>(<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fionic-team%2Fionic-framework%2Fissues%2F24335" target="_blank">#24335</a>)</li> 
 <li><strong>menu:</strong><span> </span>使用菜单和叠加层捕获焦点不再导致错误 (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fionic-team%2Fionic-framework%2Fissues%2F24611" target="_blank">#24611</a>)</li> 
 <li><strong>modal:</strong><span> </span>当底部工作表被拖动时，原生键盘将关闭 (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fionic-team%2Fionic-framework%2Fissues%2F24642" target="_blank">#24642</a>)</li> 
 <li><strong>range:</strong><span> </span>在 ion-range 上设置 dir ，以启用现在支持的 rtl 模式 (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fionic-team%2Fionic-framework%2Fissues%2F24597" target="_blank">#24597</a>) </li> 
 <li><strong>react:</strong><span> </span>setupIonicReact 不再在 SSR 环境中崩溃 (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fionic-team%2Fionic-framework%2Fissues%2F24604" target="_blank">#24604</a>) </li> 
 <li><strong>searchbar:</strong><span> </span>在 ion-searchbar 上设置 dir 以启用现在支持的 rtl 模式 (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fionic-team%2Fionic-framework%2Fissues%2F24602" target="_blank">#24602</a>)</li> 
 <li><strong>segment:</strong><span> </span>在 ion-segment 上设置 dir 以启用现在支持的 rtl 模式 (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fionic-team%2Fionic-framework%2Fissues%2F24601" target="_blank">#24601</a>) </li> 
 <li><strong>spinner:</strong><span> </span>确保变换不会覆盖圆形动画  (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fionic-team%2Fionic-framework%2Fissues%2F24643" target="_blank">#24643</a>) </li> 
 <li><strong>toast:</strong><span> </span>在 toast 可见时允许输入焦点 (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fionic-team%2Fionic-framework%2Fissues%2F24572" target="_blank">#24572</a>)</li> 
 <li><strong>toggle:</strong><span> </span>在 ion-toggle 上设置 dir 以启用现在支持的 rtl 模式 (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fionic-team%2Fionic-framework%2Fissues%2F24600" target="_blank">#24600</a>)</li> 
</ul>
