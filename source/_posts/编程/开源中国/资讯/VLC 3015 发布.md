                                                        <p>VLC 3.0.15 现已发布，VLC 是一款自由、开源的跨平台多媒体播放器及框架，可播放大多数多媒体文件，以及 DVD、音频 CD、VCD 及各类流媒体协议。</p> 
<p><strong>主要更新内容</strong></p> 
<ul> 
 <li>修复了使用 D3D11 搜索时的延迟</li> 
 <li>修复了用非 ASCII 字符打开 DVD 文件夹的问题</li> 
 <li>使用广播流修复 ASF 回归</li> 
 <li>修复了寻找特定 MP4 内容时丢失音频的问题</li> 
 <li>修复了与字幕重叠的搜索和音量滑块</li> 
 <li>改进了自由字体轮廓</li> 
 <li>修复了 Windows XP 上的 HTTPS 支持</li> 
 <li>添加 Opus 和 Alac 波形格式映射</li> 
 <li>修复 Windows XP 的 GnuTLS 支持</li> 
</ul> 
<p>详情请查看<a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fwww.videolan.org%2Fvlc%2Freleases%2F3.0.15.html" target="_blank">更新公告</a>。</p>
