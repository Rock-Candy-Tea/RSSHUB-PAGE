                                                        <p>MX Linux 21 首个 Beta 版本<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fmxlinux.org%2Fblog%2Fmx-21-beta-1-now-available-for-testing-purposes%2F" target="_blank">已发布</a>。</p> 
<p>更新亮点：</p> 
<ul> 
 <li>引入新的、以及更新应用程序</li> 
 <li>新的安装程序分区选择区域，如果 lvm 卷已经存在，包括一些 lvm 支持</li> 
 <li>新的 UEFI 实时系统启动菜单。现可以从引导菜单和子菜单中选择实时引导选项（持久性等），而不是使用以前的控制台菜单</li> 
 <li>Xfce 4.16 
  <ul> 
   <li>默认情况下，admin task 支持使用用户密码 (sudo)，可在 mx-tweak->Other 选项卡中进行切换</li> 
   <li><span style="color:null"><span style="background-color:null">许多小的配置变化，特别是在面板上使用新的默认面板插件</span></span></li> 
  </ul> </li> 
</ul> 
<p>另外值得注意的是，官方希望主要版本号可遵循基于年份的规则，因此原本的 MX Linux 20 已被跳过，直接从 MX Linux 19 跳到了 MX Linux 21。</p> 
<p><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fmxlinux.org%2Fblog%2Fmx-21-beta-1-now-available-for-testing-purposes%2F" target="_blank">下载地址</a></p> 
<p>MX Linux 是基于 Debian 稳定分支的面向桌面的 Linux 发行，它是 antiX 及早先的 MEPIS Linux 社区合作的产物。它采用 Xfce 作为默认桌面环境，是一份中量级操作系统，并被设计为优雅而高效的桌面与如下特性的结合：配置简单、高度稳定、性能可靠、占用空间不大也不小。</p>
