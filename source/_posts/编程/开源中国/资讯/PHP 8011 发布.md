                                                                                            <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">PHP 8.0.11 现已<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.php.net%2Farchive%2F2021.php%232021-09-23-2" target="_blank">发布</a>，这是一个安全版本，修复了 CVE-2021-21706；<span style="color:#333333">官方鼓励所有 PHP 8.0 用户升级到此版本。具体更新内容如下：</span></p>

 <li><span style="color:#333333">Core：</span> 
  <ul style="margin-left:0; margin-right:0"> 
   <li><span style="color:#333333">​​​​</span>修复了<span style="color:#333333"> bug </span><a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fbugs.php.net%2F81302" target="_blank">#81302</a>（移除 stream filter 后的 stream position）。</li> 
   <li>修复了<span style="color:#333333"> bug </span><a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fbugs.php.net%2F81346" target="_blank">#81346</a>（Non-seekable streams 在写入后不更新位置）。</li> 
   <li>修复了<span style="color:#333333"> bug </span><a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fbugs.php.net%2F73122" target="_blank">#73122</a>（串联字符串时整数溢出）。</li> 
  </ul> </li> 
 <li><span style="color:#000000">GD</span> 
  <ul style="margin-left:0; margin-right:0"> 
   <li>修复了<span style="color:#333333"> bug </span><a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fbugs.php.net%2F53580" target="_blank">#53580</a>（在调整大小的过程中，gdImageCopyResampled 导致颜色改变）。</li> 
  </ul> </li> 
 <li>Opcache：

   <li>修复了<span style="color:#333333"> bug </span><a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fbugs.php.net%2F81353" target="_blank">#81353</a>（带有预加载和静态绑定闭包的 segfault）。</li> 
  </ul> </li>

  </ul> </li> 
 <li><span style="color:#333333">Standard：</span> 
  <ul style="margin-left:0; margin-right:0"> 
   <li>修复了<span style="color:#333333"> bug </span><a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fbugs.php.net%2F71542" target="_blank">#71542</a>（disk_total_space 不适用于相对路径）。</li> 
   <li>修复了<span style="color:#333333"> bug </span><a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fbugs.php.net%2F81400" target="_blank">#81400</a>（dns_get_record() results 中未终止的字符串）。</li>

  </ul> </li>
