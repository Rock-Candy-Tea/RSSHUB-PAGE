 <div style="text-align:left"> 
  <p style="margin-left:0; margin-right:0; text-align:left"><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Feguidedog.net%2Flinux.php" target="_blank">eGuideDog Linux</a> 是一个针对中国盲人研发的 Linux 发行版。eGuideDog Linux 0.9 基于 Ubuntu Linux 22.04 MATE修改，主要修改包括：</p> 
  <ul style="list-style-type:disc; margin-left:0; margin-right:0"> 
   <li> <p style="margin-left:0; margin-right:0">预装 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Feguidedog.net%2Fcn%2Fekho_cn.php" target="_blank">Ekho 语音合成软件</a></p> </li> 
   <li> <p style="margin-left:0; margin-right:0">预装 ibusreader 输入法读屏软件</p> </li> 
   <li> <p style="margin-left:0; margin-right:0">预装 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fcto.eguidedog.net%2Fnode%2F268" target="_blank">eGuideDog 工具箱</a></p> </li> 
   <li> <p style="margin-left:0; margin-right:0">做了一些中文和读屏相关的预配置</p> </li> 
  </ul> 
  <p style="margin-left:0; margin-right:0; text-align:left">下载链接和安装指南请访问 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Feguidedog.net%2Flinux.php" target="_blank">eGuideDog Linux 的主页</a>获取。</p> 
  <p style="margin-left:0; margin-right:0; text-align:left">对研发过程感兴趣的朋友可以访问：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fcto.eguidedog.net%2Fnode%2F1202" target="_blank">eGuideDog Linux 0.9 开发日志</a>，打包的脚本源代码放在<a href="https://gitee.com/hgneng/eguidedog-linux/tree/master/ubuntu22.04">Gitee</a>。</p> 
 </div> 
</div> 
<p>特别鸣谢“暮然回首”、“冰心”、“小草莓”给该系统开发提供的意见反馈。</p>
