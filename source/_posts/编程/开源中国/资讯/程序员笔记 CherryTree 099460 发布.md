                                                                                            <p>CherryTree 0.99.46 现已发布。CherryTree 是一个支持无限层级分类的笔记软件，Python 编写，支持富文本编辑和代码高亮，支持 Linux 和 Windows 平台。数据采用 sqlite 或 XML 存储，支持密码保护，支持从 NoteCase、KeepNote、Knowit、Tomboy、TuxCards、Treepad、Leo 等笔记软件导入数据。</p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><img height="342" src="https://static.oschina.net/uploads/space/2022/0119/070828_IWDo_4937141.png" width="500" referrerpolicy="no-referrer"></p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><img height="342" src="https://static.oschina.net/uploads/space/2022/0119/070833_Gy9G_4937141.png" width="500" referrerpolicy="no-referrer"></p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">此版本更新内容如下：</p> 
<ul> 
 <li>修复了节点名称和标签搜索中的时间创建/修改过滤器</li> 
 <li>将默认的键盘快捷键 Ctrl+Period 改为 Ctrl+Backslash</li> 
 <li>修复了在 Windows 和双屏幕上恢复窗口位置的问题</li> 
 <li>修复了在 Windows 上从代码框复制并粘贴到富文本中不需要的附加字符 </li> 
 <li>在  Windows上导出到 txt 时设置了预期的行尾 [CR][NL]，而不是 [NL]</li> 
 <li>在富文本的右键菜单中增加了 strip trailing spaces 的操作</li> 
 <li>添加命令行选项，以传递密码来打开加密文件</li> 
 <li>添加阿拉伯语</li> 
 <li>添加了匈牙利语</li> 
 <li>更新了法语、俄语、哈萨克语等语言</li> 
</ul>
