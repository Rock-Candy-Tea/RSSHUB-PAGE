                                                                                            <p>Qt 6.4 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Flists.qt-project.org%2Fpipermail%2Fdevelopment%2F2022-September%2F042999.html" target="_blank">发布</a>了 RC 版本，将在 9 月 29 日左右发布稳定版本。</p> 
<p>Qt 6.4 的新模块包括 Qt HTTP Server 和 Qt Quick 3D Physics ，但两者均不稳定。</p> 
<p>Qt HTTP Server 模块可通过可选的 TLS 支持轻松地将 HTTP 服务器嵌入到应用程序中。但文档中提及，它更多是面向嵌入式应用的本地/基于 LAN 的 Web 服务，不具备面向 Internet 的 Web 服务器的稳健性和安全性。</p> 
<p>Qt Quick 3D Physics 提供了一个高级物理模拟 API，它支持模拟交互式刚体，以及用于检测重叠的静态网格和非碰撞体。每个模拟物体都可以有自己的物理属性，如质量、密度和摩擦力。<br> <br> Qt 6.4 还为 Qt Core 添加了许多 API、对 Qt Quick 进行了许多增强，QSSlServer 已添加到 Qt Network 作为仅通过 TLS 通信的服务器、Qt Quick 3D 提供了光照贴图烘焙支持的预览、改进了 Qt 的阴影渲染 Qt WebEngine 的快速 3D 和 HTML5 文件系统访问 API。</p> 
<p>有关 Qt 6.4 更改的更多详细信息，请参阅<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdoc-snapshots.qt.io%2Fqt6-dev%2Fwhatsnew64.html" target="_blank">文档</a>。</p>
