                                                                                            <p>概述：</p> 
<p>Pear Admin 社区拥抱 Python 生态，基于 Flask 1.12 推出 Flask Admin 后台管理系统，得益于Python 的生态广泛应用，通过使用本系统，即可快速构建你的功能业务；</p> 
<p>项目旨在为 Python 开发者提供一个后台管理系统，成为您构建信息管理系统，物联网后台....等等应用时灵活，简单的快速开发平台；</p> 
<p>版本说明</p> 
<p>Common [ 通用版本 ] -- 基于 Mysql 的 RBAC 设计实现</p> 
<p>Simple [ 简洁版本 ] -- 基于 Yml 配置的权限管理</p>

<p>功能列表：</p> 
<ul> 
 <li>用户管理：用户是系统操作者，该功能主要完成系统用户配置。</li> 
 <li> 权限管理：配置系统菜单，操作权限，按钮权限标识等。</li> 
 <li> 角色管理：角色菜单权限分配。</li> 
 <li> 操作日志：系统正常操作日志记录和查询；系统异常信息日志记录和查询。</li> 
 <li> 登录日志：系统登录日志记录查询包含登录异常。</li> 
 <li> 服务监控：监视当前系统CPU、内存、磁盘、python版本,运行时长等相关信息。</li> 
 <li> 文件上传: 图片上传示例</li> 
</ul>
