                                                        <p>React 18.2.0 现已发布，具体更新内容如下：</p> 
<p><span style="color:#24292f"><strong>React DOM</strong></span></p> 
<ul> 
 <li>提供一个组件堆栈作为<code>onRecoverableError</code>的第二个参数。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffacebook%2Freact%2Fpull%2F24591" target="_blank">#24591</a></li> 
 <li>修复 hydrating into document 导致不匹配时出现空白页的问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffacebook%2Freact%2Fpull%2F24523" target="_blank">#24523</a></li> 
 <li>修复 false positive hydration errors with Suspense。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffacebook%2Freact%2Fpull%2F24480" target="_blank">#24480</a> 和 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffacebook%2Freact%2Fpull%2F24532" target="_blank">#24532</a></li> 
 <li>修复 Safari 中添加 iframe 时被忽略的<code>setState</code>问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffacebook%2Freact%2Fpull%2F24459" target="_blank">#24459</a></li> 
</ul> 
<p><span style="color:#24292f"><strong>React DOM Server</strong></span></p> 
<ul> 
 <li>将有关服务器错误的信息传递给客户端。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffacebook%2Freact%2Fpull%2F24551" target="_blank">#24551</a> 和 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffacebook%2Freact%2Fpull%2F24591" target="_blank">#24591</a></li> 
 <li>允许在中止 HTML 流时提供原因。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffacebook%2Freact%2Fpull%2F24680" target="_blank">#24680</a></li> 
 <li>尽可能消除 HTML 中无关的文本分隔符。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffacebook%2Freact%2Fpull%2F24630" target="_blank">#24630</a></li> 
 <li>不允许<code><title></code>元素内的 complex children 与浏览器约束匹配。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffacebook%2Freact%2Fpull%2F24679" target="_blank">#24679</a></li> 
 <li>通过显式设置<code>highWaterMark</code>为<code>0</code>来修复某些工作环境中的缓冲。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffacebook%2Freact%2Fpull%2F24641" target="_blank">#24641</a></li> 
</ul> 
<p><span style="color:#24292f"><strong>Server Components (Experimental)</strong></span></p> 
<ul> 
 <li>在 Server Components 中添加对<code>useId()</code>的支持。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffacebook%2Freact%2Fpull%2F24172" target="_blank">#24172</a></li> 
</ul>
