                                                                                            <p><span style="background-color:#ffffff; color:#333333">Git for Windows 2.35.3 现已发布，主要更新内容如下：</span></p> 
<p style="text-align:start"><strong>New Features</strong></p> 
<ul> 
 <li>随附 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fgit%2Fgit%2Fblob%2Fv2.35.3%2FDocumentation%2FRelNotes%2F2.35.3.txt" target="_blank">Git v2.35.3</a>。</li> 
</ul> 
<p style="text-align:start"><strong><span><span><span><span><span style="color:#24292f"><span><span><span><span><span><span><span><span><span><span><span style="background-color:#ffffff"><span><span><span>Bug 修复</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></strong></p> 
<ul> 
 <li>指示如何使用网络共享路径的<code>%(prefix)</code>建议<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fgit-for-windows%2Fgit%2Fpull%2F3790" target="_blank">已更新</a>，以使用适当数量的 slashes。</li> 
 <li>使用 Windows Subsystem for Linux (WSL) 时，对<code>safe.directory</code>和<code>%(prefix)</code>的使用进行了<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fgit-for-windows%2Fgit%2Fpull%2F3791" target="_blank">各种修复。</a></li> 
</ul>
