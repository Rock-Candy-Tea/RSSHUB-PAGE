                                                        <h4 style="text-align:left">项目简介</h4> 
<p>Pear Admin Think 基于 thinkphp6 的快速开发平台，通过简单的代码生成功能，即可快速构建你的功能业务，努力成为最顺手的轮子。</p> 
<h4 style="text-align:left">相关链接</h4> 
<p>开源地址：<a href="https://gitee.com/pear-admin/Pear-Admin-Think">前往</a></p> 
<p>官网地址：<a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fwww.pearadmin.com" target="_blank">前往</a></p> 
<h4 style="text-align:left">更新内容</h4> 
<ul> 
 <li>增加数据签名验证。</li> 
 <li>增加语法糖扩展。</li> 
 <li>增加成功失败404模板。</li> 
 <li>增加配置文件读改。</li> 
 <li>增加多文件上传。</li> 
 <li>增加工具包。</li> 
 <li>修改系统配置项。</li> 
 <li>更新登录验证方式。</li> 
 <li>更新JSON输出格式。</li> 
 <li>同步更新最新版pear-layui。</li> 
 <li>修复菜单为url报错,修改验证方式。</li> 
 <li>更新路由与入口兼容方法。</li> 
 <li>去除配置表，通过文件配置读改。</li> 
 <li>模型，服务，验证器，事件，工具包 统一放入common文件进行管理。</li> 
 <li>修改设计思路，移动目录结构，简化代码。</li> 
 <li>本次更新，变动较大。请使用最新版本！</li> 
</ul>
