                                                        <p>Taro 3.2.14 发布了。Taro 是一个开放式跨端跨框架解决方案，支持使用 React/Vue/Nerv 等框架来开发微信/京东/百度/支付宝/字节跳动/ QQ 小程序/H5 等应用。此版本更新内容包括：</p> 
<h4>修复</h4> 
<p><strong>小程序</strong></p> 
<ul> 
 <li>兼容 <code>vue@3.1.2</code>，by <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fb2nil" target="_blank">@b2nil</a> 。</li> 
 <li>修复支付宝小程序中 Vue3 使用 <code>Swiper</code> 时出现空白页的问题，fix <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FNervJS%2Ftaro%2Fissues%2F9618" target="_blank">#9618</a></li> 
 <li>修复 Vue3 生命周期触发问题，fix <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FNervJS%2Ftaro%2Fissues%2F9614" target="_blank">#9614</a></li> 
 <li>优化反向转换</li> 
</ul> 
<p><strong>H5</strong></p> 
<ul> 
 <li>修复 <code>Video</code> 点击全屏在 IOS 下无效的问题。</li> 
 <li>修复 Taro 在低端机安卓 4.4 上面白屏的问题。</li> 
</ul> 
<p><strong>RN</strong></p> 
<ul> 
 <li>修复 <code>hoverStyle</code> 类型问题。</li> 
 <li>更新 <code>types</code> 版本。</li> 
 <li>生成 <code>sourcemap</code> 输出目录。</li> 
 <li>修复启动遇到报错无提示问题。</li> 
</ul> 
<p><strong>Typings</strong></p> 
<ul> 
 <li>修复升级 TypeScript 4.3.2 后编译报错。</li> 
 <li>补充背景音频 <code>BackgroundAudioManager</code> 中的 <code>onSeeking</code> 和 <code>onSeeked</code> 方法声明。</li> 
</ul> 
<p>更新说明：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FNervJS%2Ftaro%2Freleases%2Ftag%2Fv3.2.14" target="_blank">https://github.com/NervJS/taro/releases/tag/v3.2.14</a></p>
