                                                        <p><span style="background-color:#ffffff; color:#333333">Taichi（太极）0.8.9 已经发布，这是专为高性能计算机图形学设计的编程语言。</span></p> 
<p><img height="287" src="https://static.oschina.net/uploads/space/2022/0105/070330_1REO_4937141.gif" width="512" referrerpolicy="no-referrer"></p> 
<div style="margin-left:0; margin-right:0"> 
 <div style="margin-left:0; margin-right:0"> 
  <div style="margin-left:0; margin-right:0"> 
   <p style="margin-left:0; margin-right:0"><span style="color:#333333">此版本亮点内容如下：</span></p> 
   <ul style="margin-left:0; margin-right:0"> 
    <li><strong>Android</strong> 
     <ul style="margin-left:0; margin-right:0"> 
      <li>在 GGUI 中添加对 Android 的初始支持 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ftaichi-dev%2Ftaichi%2Fpull%2F3845" target="_blank">#3845</a> )</li> 
     </ul> </li> 
    <li><strong>Bug 修复</strong> 
     <ul style="margin-left:0; margin-right:0"> 
      <li>使用 cuda 11 及以上版本时查询设备属性 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ftaichi-dev%2Ftaichi%2Fpull%2F3930" target="_blank">#3930</a> )</li> 
      <li>修复 ttf 路径（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ftaichi-dev%2Ftaichi%2Fpull%2F3931" target="_blank">#3931</a>）</li> 
     </ul> </li> 
    <li><strong>Language and syntax</strong> 
     <ul style="margin-left:0; margin-right:0"> 
      <li>对于 ti.kernel 初始矩阵参数的支持（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ftaichi-dev%2Ftaichi%2Fpull%2F3905" target="_blank">＃3905</a>）</li> 
      <li>尽可能启用矩阵字段元素的动态索引（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ftaichi-dev%2Ftaichi%2Fpull%2F3865" target="_blank">#3865</a>）</li> 
     </ul> </li> 
    <li><strong>Miscellaneous</strong> 
     <ul style="margin-left:0; margin-right:0"> 
      <li>支持 logging on Android platforms ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ftaichi-dev%2Ftaichi%2Fpull%2F3849" target="_blank">#3849</a> )</li> 
     </ul> </li> 
    <li><strong>Refactor</strong> 
     <ul style="margin-left:0; margin-right:0"> 
      <li>删除所有出现的 print_preprocessed 和 print_ast（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ftaichi-dev%2Ftaichi%2Fpull%2F3911" target="_blank">＃3911</a>）</li> 
      <li>弃用 exceptionhook 并完全删除 _taichi_skip_traceback ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ftaichi-dev%2Ftaichi%2Fpull%2F3902" target="_blank">#3902</a> )</li> 
     </ul> </li> 
    <li><strong>Tests</strong> 
     <ul style="margin-left:0; margin-right:0"> 
      <li>添加初始 AOT CPP 测试 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ftaichi-dev%2Ftaichi%2Fpull%2F3850" target="_blank">#3850</a> ) ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ftaichi-dev%2Ftaichi%2Fpull%2F3899" target="_blank">#3899</a> )</li> 
      <li>添加初始 AOT CPP 测试 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ftaichi-dev%2Ftaichi%2Fpull%2F3850" target="_blank">#3850</a> )</li> 
     </ul> </li> 
   </ul> 
   <p style="margin-left:0; margin-right:0">更多详情可查看：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ftaichi-dev%2Ftaichi%2Freleases%2Ftag%2Fv0.8.9" target="_blank">https://github.com/taichi-dev/taichi/releases/tag/v0.8.9</a> </p>
