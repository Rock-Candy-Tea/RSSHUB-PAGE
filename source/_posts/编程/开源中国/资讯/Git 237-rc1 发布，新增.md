                                                                                            <p>Git 2.37 上周发布了<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Flore.kernel.org%2Fall%2Fxmqqv8syloqp.fsf%40gitster.g%2F" target="_blank">最新 RC 版本</a>。</p> 
<p>2.37 没有添加“杀手锏”级别的新功能，但与之前的版本相比有许多增量工作，主要变化包括：</p> 
<ul> 
 <li>为 <em>git --version </em>引入替代方案 <em>git -v</em><span style="color:#121212">，以及</span> <em>git --help </em>的替代方案<span style="color:#121212"> </span><em>git -h</em></li> 
 <li><span style="background-color:#ffffff"><span style="color:#121212">提供给外部 diff 命令的临时文件现在</span></span>会在同一 basename 下的临时目录内生成</li> 
 <li> <p>使用 C 语言重写 <em>git add -i</em><span style="background-color:#ffffff"><span style="color:#121212"> 功能，</span></span>现在终于可以被所有用户默认使用</p> </li> 
 <li> <p>针对传输和打包功能进行初步的代码重构</p> </li> 
 <li> <p>多项性能优化，修复内存泄露问题，以及其他和性能相关的工作</p> </li> 
 <li> <p>修复多项错误</p> </li> 
</ul> 
<p><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Flore.kernel.org%2Fall%2Fxmqqv8syloqp.fsf%40gitster.g%2F" target="_blank">发布公告</a>。</p>
