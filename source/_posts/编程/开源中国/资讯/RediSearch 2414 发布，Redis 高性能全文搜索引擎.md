                                                                                            <p>RediSearch 是 RedisLabs 团队开发的一个高性能全文搜索引擎，可作为一个 Redis Module 运行在 Redis 上。</p> 
<p>RediSearch 2.4.14 发布了，这是 RediSearch 2.4 的维护版本，带来了一些改进和修复。</p> 
<p><strong>改进：</strong></p> 
<ul> 
 <li>VectorSimilarity <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FRedisAI%2FVectorSimilarity%2Freleases%2Ftag%2Fv0.3.2" target="_blank"> v0.3.2</a> 升级到版本</li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FRediSearch%2FRediSearch%2Fpull%2F2955" target="_blank">#2955</a> 在 <span style="color:#24292f">prefix </span>查询期间添加超时 (MOD-3949)</li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FRediSearch%2FRediSearch%2Fpull%2F2957" target="_blank">#2957</a> 从 <span style="color:#24292f">prefix </span>列表中删除具有多索引的案例</li> 
</ul> 
<p><strong>修复：</strong></p> 
<ul> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FRediSearch%2FRediSearch%2Fpull%2F2937" target="_blank">#2937</a> 遇到过期文档后返回 <code>NULL</code> 响应 (MOD-3515)</li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FRediSearch%2FRediSearch%2Fpull%2F2962" target="_blank">#2962</a>  AOF 预加载时崩溃 (MOD-3951)</li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FRediSearch%2FRediSearch%2Fpull%2F2986" target="_blank">#2986</a> 与架构前缀相关的内存泄漏</li> 
</ul>
