                                                        <p><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.chiark.greenend.org.uk%2F%7Esgtatham%2Fputty%2F" target="_blank">PuTTY 0.76 现已发布</a>。PuTTY 是一款集成虚拟终端、系统控制台和网络文件传输为一体的自由开源程序。它支持多种网络协议，包括 SCP，SSH，Telnet，rlogin 和原始的套接字连接，它也可以连接到串行端口。其软件名字“PuTTY”并没有特殊含义。</p> 
<p>v0.76 是一个错误修复和安全版本。它修复了 0.75 中的错误；还增加了一个新的配置选项，作为对恶意的或被破坏的 SSH 服务器的认证提示欺骗的额外防御。</p> 
<p>具体更新内容如下：</p> 
<ul> 
 <li>如果服务器允许你以简单的方式进行身份验证，则放弃 SSH 连接的新选项。</li> 
 <li>错误修复：使用“Use system colours”选项时，Windows PuTTY 崩溃。</li> 
 <li>错误修复：在 Windows 上使用 MIT Kerberos 和 Restart Session 时会崩溃。</li> 
 <li>错误修复：Windows PuTTY 在联系 Pageant 后泄露了 named pipes。</li> 
 <li>错误修复：当你按住滚动条箭头按钮的时间足够长以“key-repeat”时，Windows PuTTY 没有更新窗口。</li> 
 <li>错误修复：通过“Change Settings”重新配置用户调色板的操作延迟。</li> 
 <li>错误修复：服务器调色板重新配置有时会丢失。</li> 
 <li>错误修复：读取截断的私钥文件时可能会发生 tight loop。</li> 
 <li>错误修复：Windows Pageant GUI 密钥列表未显示密钥长度。</li> 
</ul>
