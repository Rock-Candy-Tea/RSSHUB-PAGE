                                                                                            <p>SourceGit v5.0 已经发布，开源的 Git 客户端。</p> 
<p>此版本更新内容包括：</p> 
<p>[CHANGELOG]</p> 
<ol> 
 <li>Welcome起始页与标签页同步显示仓库的名称、书签等变化</li> 
 <li>修复Welcome起始页分组图标显示错乱的问题</li> 
 <li>页面标签增加右键菜单</li> 
 <li>修复启用Filter的分支/标签删除后历史记录无法显示的问题</li> 
</ol> 
<h1>SourceGit</h1> 
<p>开源的Git客户端，仅用于Windows。</p> 
<h2>特点</h2> 
<ul> 
 <li>永久免费+开源</li> 
 <li>轻量级，Windows 10下软件小于2M，无需安装，直接运行</li> 
 <li>启动速度、加载速度快（相对于SmartGit，SourceTree等，加载同数量的日志）</li> 
 <li>一次性显示最近20000条提交历史</li> 
 <li>中英双语并提供明暗两种主题</li> 
 <li>覆盖常用GIT指令</li> 
 <li>分支线路图</li> 
 <li>高级指令图形操作 
  <ul> 
   <li>SUBMODULES</li> 
   <li>SUBTREES</li> 
   <li>ARCHIVE</li> 
   <li>PATCH/APPLY</li> 
   <li>FILE HISTORIES</li> 
   <li>BLAME</li> 
   <li>REVISION DIFF</li> 
  </ul> </li> 
</ul>
