                                                                                            <p><span style="color:#333333">Apache ECharts 是一个使用 JavaScript 实现的开源可视化库，可以流畅的运行在 PC 和移动设备上，兼容当前绝大部分浏览器（IE8/9/10/11，Chrome，Firefox，Safari等），底层依赖矢量图形库 ZRender，提供直观，交互丰富，可高度个性化定制的数据可视化图表。</span></p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><span style="color:#333333">Apache ECharts 5.3.2 现已发布，具体更新内容如下：</span></p> 
<ul> 
 <li>[Feature] [line] 允许<code>areaStyle.origin</code>将数字作为 input。</li> 
 <li>[Feature] [scatter]<code>scale</code>支持数字。</li> 
 <li>[Feature] [tree] 在 tree click event parameters 添加<code>collapsed</code></li> 
 <li>[Fix] [marker] 修复 marker state 模糊后不恢复的问题。</li> 
 <li>[Fix] [gauge] 当值为 0 且启用 progress.roundCap 时，进度条可能意外地变成圆形的问题。</li> 
 <li>[[Fix] [polar] 修复在半径轴中启用<code>splitLine</code>时图表抛出的错误。</li> 
 <li>[Fix] [labelLine] 修复<code>labelLine</code>无法隐藏的问题。</li> 
 <li>[Fix] [dataZoom] 修复当数据包含空值时，<code>dataZoom</code>意外地显示在顶部的问题。</li> 
 <li>[Fix] [heatmap] 修复 heatmap charts 中的意外 gaps。</li> 
 <li>[Fix] [toolbox] 修复 toolbox title 可能在图表之外。</li> 
 <li>[Fix] [toolbox] toolbox 在图标上悬停时不进入强调状态。</li> 
 <li>[Fix] [toolbox] [dataview] 修复使用默认文本视图时意外的 scrollbar 和 outline。</li> 
 <li>[Fix] [visualMap] 修复部分文本样式在<code>visualMap</code>上无法工作的问题。</li> 
 <li>[Fix] [sanky] 如果颜色 illegal，则回退为黑色。</li> 
 <li>[Fix] [types] 修复错误的 jpg 而不是 jpeg 图像导出类型。</li> 
</ul>
