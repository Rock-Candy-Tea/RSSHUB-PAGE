                                                        <p>Apache SkyWalking 是分布式系统的 APM（应用性能监控）工具，尤其是为微服务、云原生和基于容器（Docker、Kubernetes、Mesos）的架构而设计。</p> 
<p>Apache SkyWalking 8.9.0 正式发布，相比 8.8.1 版本，这个版本包含一些新功能、错误修复和改进。其中值得注意的变化包括：</p> 
<ul> 
 <li>E2E 测试迁移到 e2e-v2</li> 
 <li>支持 JDK 16 和 17</li> 
 <li>增加 arm64 架构的 Docker 镜像</li> 
 <li>优化端点依赖性</li> 
 <li>新的时间范围选择器和时间范围设置</li> 
 <li>恢复 zipkin-receiver 插件</li> 
 <li>增强有关数据报告和查询协议的文档</li> 
 <li>在时区设置中支持 +/- hh:mm</li> 
 <li>优化全局设置</li> 
 <li>修复端点依赖的当前端点</li> 
 <li>在全局设置弹出窗口中添加版本</li> 
 <li>优化日志页面样式</li> 
 <li>修复事件的查询条件</li> 
 <li>将 Kubernetes Java 客户端升级到 14.0.0</li> 
 <li>添加 OpenSearch 1.2.0 以测试和验证它是否有效</li> 
 <li>将 grpc-java 升级到 1.42.1 并将 protoc 升级到 3.17.3 以允许使用原生 Mac osx-aarch_64 工件</li> 
 <li>修复 TopologyQuery.loadEndpointRelation 错误</li> 
 <li>……</li> 
</ul>
