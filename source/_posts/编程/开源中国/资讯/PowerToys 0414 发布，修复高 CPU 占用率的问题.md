                                                        <p>就在几天前，微软发布了 PowerToys 0.41.3，它增加了一个新的功能可以使用户的电脑保持唤醒状态。然而，0.41.3 似乎出现了唤醒状态下 CPU 占用率高的问题，因此微软又推送了 0.41.4 版本。</p> 
<p>PowerToys 0.41.4 的更新日志如下：</p> 
<ul> 
 <li>#11898 - 选择搜索结果时 PowerToys Run 崩溃；</li> 
 <li>#11912 - 唤醒状态时的 CPU 使用率很高；</li> 
 <li>#11955 - 在 Windows 启动时 PowerToys 运行出错；</li> 
 <li>#11964 - 在系统托盘中为 Awake 添加 "PowerToys"；</li> 
 <li>#12031 - FancyZones 无法正确启动；</li> 
 <li>#12044 - 图像大小调整为 1x1；</li> 
</ul> 
<p>更多详情可查看：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmicrosoft%2FPowerToys%2Freleases%2Ftag%2Fv0.41.4" target="_blank">https://github.com/microsoft/PowerToys/releases/tag/v0.41.4</a></p>
