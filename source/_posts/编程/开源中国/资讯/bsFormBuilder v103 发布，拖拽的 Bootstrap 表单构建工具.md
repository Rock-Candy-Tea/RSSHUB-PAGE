<div class="content">
                                                                    
                                                        <p style="color:#333333; margin-left:0; margin-right:0; text-align:left">bsFormBuilder v1.0.3 发布了，bsFormBuilder 是一个基于 Bootstrap  v4.x 表单构建工具。这次主要是新增了 3 个新的组件，并提供了演示站点：<a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fbsformbuilder.jpress.cn" target="_blank">http://bsformbuilder.jpress.cn</a></p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><strong>bsFormBuilder 的特点：</strong></p> 
<ul style="list-style-type:disc; margin-left:0; margin-right:0"> 
 <li>1、基于 Bootstrap (v4.x) + JQuery，简单易用</li> 
 <li>2、拖动的 html 组件，支持通过 Json 自定义扩展</li> 
 <li>3、组件的属性面板，支持通过 Json 自定义扩展</li> 
 <li>4、支持导出 html 和 json，自己通过 json 来构建 UI 页面</li> 
 <li>5、支持导入 json 到 bsFormBuilder，进行二次编辑</li> 
 <li>6、丰富的 API，方便二次开发和扩展</li> 
 <li>7、支持 “模板” 功能，可以选择已有模板进行二次开发</li> 
 <li>8、内置轻量的 html 渲染引擎，速度极快，极好用～～</li> 
</ul> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><img alt="bsFormBuilder" src="https://gitee.com/fuhai/bs-form-builder/raw/master/assets/images/bsformbuilder.png" referrerpolicy="no-referrer"></p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left"> </p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><strong>bsFormBuilder v1.0.3 更新内容如下：</strong></p> 
<ul> 
 <li>新增：时间、日期、上传文件 <span>3 </span>个新的组件</li> 
 <li>新增：新增演示站点 <span><a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fbsformbuilder.jpress.cn" target="_blank">http://bsformbuilder.jpress.cn</a></span></li> 
 <li>修复：个别属性在某些情况下可能无法正确渲染的问题</li> 
 <li>优化：删除 <span>_invokeRenderBody </span>方法，用 <span>_renderTemplate </span>代替</li> 
 <li>优化：<span>tips </span>组件，配置让其显示 <span>tag </span>属性</li> 
 <li>优化：优化大量的代码注释，使之更加便于阅读</li> 
 <li>优化：移除 <span>bower</span>，在某些网络下可能无法正常编译的问题</li> 
</ul> 
<p> </p>
