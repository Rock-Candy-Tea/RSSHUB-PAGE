                                                                                            <p>Linus Torvalds 发布了Linux 5.19 ， 作为 Linux 内核的最新稳定版本。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Flore.kernel.org%2Flkml%2FCAHk-%3Dwgrz5BBk%3DrCz7W28Fj_o02s0Xi0OEQ3H1uQgOdFvHgx0w%40mail.gmail.com%2FT%2F%23u" target="_blank">发布公告中</a>还提到发布工作是在一台运行 Apple M2 SoC 的 Arm64 架构 Apple MacBook 上进行的。</p> 
<p>借助 <a href="https://www.oschina.net/p/asahi-linux">Asahi Linux</a> 的力量，Linus <strong>首次在 ARM64 硬件</strong>上发布新的 Linux 内核。（Asahi Linux 致力于将 Linux 系统移植到苹果芯片的硬件上，具体可查看文章《 <a href="https://www.oschina.net/news/203465/asahi-linux-apple-m2-initial">Asahi Linux 已实验性支持 Apple M2</a>》）</p> 
<p>ARM64 硬件一直被用来运行 Linux ，但至今仍难以用于 Linux 的开发。Linus 这次也只是进行 Linux 5.19 版本的测试构建和启动、发布工作，而没有进行功能开发等实际工作。</p> 
<p>此外，这是 Linus Torvalds 第三次使用苹果的硬件进行 Linux 开发，第一次是在年代久远的 ppc970（IBM 古董处理器） 机器上进行 powerpc 开发，第二次是十多年前的第一代 Macbook Air。</p> 
<p>Linux 5.19 带来了许多新功能，比如：</p> 
<ul> 
 <li><a href="https://www.oschina.net/news/198517/loongarch-merged-linux-5-19" target="news">合并龙芯 LoongArch CPU 架构</a></li> 
 <li><a href="https://www.oschina.net/news/197756/linux-5.19-fat-file-creation-time" target="news">支持报告 FAT32 文件的创建时间</a></li> 
 <li><a href="https://www.oschina.net/news/190021/linux-5-19-amd-brs" target="news">引入 AMD “BRS” 分支采样功能</a></li> 
 <li><a href="https://www.oschina.net/news/198236/riscv-update-for-linux-5-19" target="news">RISC-V ：支持在 64 位系统运行 32 位二进制文​​件</a></li> 
 <li><a href="https://www.oschina.net/news/198582/linux-5-19-zstd-firmware" target="news">支持使用 Zstd 压缩固件</a></li> 
 <li><a href="https://www.oschina.net/news/197831/linux-5-19-drm-driver-updates" target="news">获得近 50 万行新的图形驱动程序代码</a></li> 
</ul> 
<p>除此之外还有一些不错的性能改进，详情可查看<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Flore.kernel.org%2Flkml%2FCAHk-%3Dwgrz5BBk%3DrCz7W28Fj_o02s0Xi0OEQ3H1uQgOdFvHgx0w%40mail.gmail.com%2FT%2F%23u" target="_blank">公告邮件</a>。</p>
