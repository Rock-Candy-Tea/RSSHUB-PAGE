<div class="content">
                                                                                            <p>AlmaLinux 8.6 Beta <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Falmalinux.org%2Fblog%2Falmalinux-86-sky-tiger-beta-now-available%2F" target="_blank">已发布</a>，代号"Sky Tiger"。</p> 
<blockquote> 
 <p style="color:#333333; margin-left:0; margin-right:0; text-align:left">AlmaLinux 是开放源码的、社区驱动的项目，它从红帽企业版 Linux (RHEL) 的源码编译而来。AlmaLinux 跟 RHEL 8 完全在二进制上兼容，它由 CloudLinux OS 的创建者打造。AlmaLinux 团队承诺永久免费提供 AlmaLinux 操作系统，项目永久开源且不采取任何限制，不收取任何费用，支持至 2029 年。</p> 
</blockquote> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-05fd1022e2192f5b87349fb5b4683f15eaf.png" referrerpolicy="no-referrer"></p> 
<p>此次更新来自上游 RHEL 8.6 Beta 的最新变化，其中包括对 Stratis Storage 的 Web 控制台管理支持、支持 Cockpit Web 控制台中的 pass-through 设备、增加新的 RHEL 配置角色，以及其他各种改进。</p> 
<p><strong>AlmaLinux 8.6 Beta 部分更新内容</strong></p> 
<ul> 
 <li>改进 Web 控制台</li> 
 <li>引入全新的系统角色，简化系统管理</li> 
 <li>安全更新包括接收来自 SCAP 安全指南、OpenSCAP 和其他软件包的上游版本更新以及其他改进</li> 
 <li>升级软件包 
  <ul> 
   <li>PHP 8.0</li> 
   <li>Perl 5.32</li> 
   <li>log4j 2</li> 
   <li>container-tools 4.0</li> 
  </ul> </li> 
</ul> 
<p><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwiki.almalinux.org%2Frelease-notes%2F8.6-beta.html%23beta-release" target="_blank">详细说明查看 release note</a>。</p> 
<p>AlmaLinux 上个月还发布了 <a href="https://www.oschina.net/news/192154/almalinux-9-0-beta-released">AlmaLinux 9 Beta</a>，Red Hat Enterprise Linux 9 正式版即将推出。</p>
