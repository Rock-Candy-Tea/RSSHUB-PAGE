                                                                                            <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">Universal Media Server 是一个兼容 DLNA，UPnP 和 HTTP/S 的媒体服务器，支持所有主要操作系统，包括 Windows，Linux 和 macOS 版本，几乎不需要配置就可以流式传输或转码许多不同的媒体格式。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">自 10.20.0 以来的变化：</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">常规：</p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>改进了电视剧的 API 匹配</li> 
 <li>语言改进</li> 
 <li>修正了数据库无法从序列化变化中恢复的问题</li> 
 <li>修正了自定义渲染器配置未被加载的问题</li> 
 <li>修复了 Docker 上的 MEncoder 和 MPlayer</li> 
 <li>修复了 UMS 关闭时配置被覆盖的问题</li> 
 <li>修正了当文件夹名称相似时将目录设置为完全播放的问题</li> 
 <li>修复了扫描过程中状态栏中的 peakaboo 问题</li> 
 <li>修正了数据库升级的错误</li> 
</ul> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">通过 Crowdin 的翻译更新：</p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>德语</li> 
 <li>葡萄牙语</li> 
</ul> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">依赖性：</p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>将 FFmpeg 升级到最新版本</li> 
 <li>将 h2 从 2.1.210 升级到 2.1.212</li> 
 <li>将 icu4j 从 70.1 升级到 71.1</li> 
 <li>将 JRE 从 8u322 升级到 8u332</li> 
 <li>将 maven-antrun-plugin 从 3.0.0 升级到 3.1.0</li> 
 <li>将 maven-javadoc-plugin 从 3.3.2 升级到 3.4.0</li> 
 <li>将 maven-site-plugin 从 3.11.0 升级至 3.12.0</li> 
 <li>将 MediaInfo 从 21.09 升级到 22.03</li> 
 <li>将 oshi-core 从 6.1.5 升级到 6.1.6</li> 
 <li>将 surefire-version 从 3.0.0-M5 调至 3.0.0-M6</li> 
</ul>
