                                                        <p>Visual Studio Code 1.59 现已发布，其中一些主要<strong>亮点内容如下</strong>：</p> 
<h4>工作台</h4> 
<ul> 
 <li>扩展</li> 
</ul> 
<p>改进了关于调整大小的扩展视图。默认宽度的扩展视图显示了所有详细信息（以前未显示图标、评级和安装计数）。当视图缩小时，会显示一个较小的扩展图标，当其宽度进一步减小时，图标和评级将被隐藏。</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-4d7d9b920761dc140c6f303a14786b5912b.gif" referrerpolicy="no-referrer"></p> 
<p>扩展程序视图现在在扩展程序上显示一个自定义的悬停，包括扩展的完整描述和其他有用的信息，如为什么一个扩展被禁用或推荐。</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-897fa2064836fd1661e20f22dc87d49683f.png" referrerpolicy="no-referrer"></p> 
<p>现在可以在扩展窗格中的新运行时状态选项卡中检查扩展的运行时状态。状态信息包括其激活时间、是否在启动时激活以及是否有任何警告或错误。</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-e194abea2382b0426234deccdc8ab70b40a.png" referrerpolicy="no-referrer"></p> 
<p>“扩展面板详细信息”选项卡现在显示类别、资源链接和其他信息，例如扩展发布和更新日期。选择一个类别将在“扩展”视图中显示该类别中的其他扩展。</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-ea7cc71a2600205fb9735cd5d7027daf3e6.png" referrerpolicy="no-referrer"></p> 
<ul> 
 <li>设置编辑器</li> 
</ul> 
<p>设置编辑器现在支持对象验证。验证会检查直接编辑 JSON 文件时可能引入的类型错误。</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-5ea405aa95ccb77fc34f21d1e75b2a96a08.png" referrerpolicy="no-referrer"></p> 
<h2>编辑器</h2> 
<ul> 
 <li>在折叠区域之间导航</li> 
</ul> 
<p>有一些新命令可以将光标位置设置为相应的折叠：</p> 
<ol> 
 <li>转到下一个折叠 (editor.gotoNextFold)</li> 
 <li>转到上一个折叠 (editor.gotoPreviousFold)</li> 
 <li>转到父折叠 (editor.gotoParentFold)</li> 
</ol> 
<p>这些命令目前没有默认键绑定。</p> 
<ul> 
 <li>内联建议改进</li> 
</ul> 
<p>改变了内联建议的呈现方式。这不仅修复了很多错误，而且还使自动换行能识别内联建议。此外，现在支持非尾随位置的多行内联建议。</p> 
<h4>终端</h4> 
<ul> 
 <li>跨窗口拖放终端</li> 
</ul> 
<p>将终端从一个窗口的选项卡列表或编辑器区域拖放到另一个窗口的选项卡列表、编辑器区域或面板中。</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-cf1238040419b9dfb7e19937320254f971f.gif" referrerpolicy="no-referrer"></p> 
<ul> 
 <li>下划线和删除线支持</li> 
</ul> 
<p>集成终端现在支持下划线和删除线属性。例如，Git 可以配置为使用这些新属性：</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-377033355781f127e48dc203f3acef3fb9f.png" referrerpolicy="no-referrer"></p> 
<h4>调试</h4> 
<ul> 
 <li>改进了编辑器标题中的运行/调试按钮</li> 
</ul> 
<p>调试按钮 的下拉按钮现在有两个单击区域，一个用于默认操作（左），另一个用于下拉（右），在所选时运行该操作并将其记住为新的默认值。</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-88b7cd764b7a733233af9eebb24edb670d7.gif" referrerpolicy="no-referrer"></p> 
<p>此外，还包括一些针对扩展的更新。更多详细内容，请查看<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fcode.visualstudio.com%2Fupdates%2Fv1_59" target="_blank">更新公告</a>。</p>
