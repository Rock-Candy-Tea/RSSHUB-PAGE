                                                                                            <p>Visual Studio Code 是一个开源的代码编辑器，支持 IntelliSense、调试、Git 和代码片断。可在 Windows、Linux 和 macOS 上下载使用。支持常见的脚本和编程语言，还可以通过安装扩展来获得更多语言和功能的支持。微软发布了 1.62 版本的第二次更新，解决了以下问题：</p> 
<ul> 
 <li>由于 $GIT_ASKPASS 中的空字符串，导致 Git 操作挂起</li> 
 <li>解决了在 1.62.1 版本中 shell 环境失败的问题</li> 
 <li>WSL 扩展安装失败</li> 
 <li>使用 contentText 的 setDecorations 在非 ASCII 行上无法呈现</li> 
 <li>当问题出现在第一行时，自定义任务 problemMatcher 的结果没有出现在问题面板上</li> 
 <li>在 Notebooks 中从未被提示选择内核</li> 
 <li>在 1.62.0 及以后的版本中，语义高亮会闪烁</li> 
 <li>Editors - pathsToEditors 应该忽略文件夹</li> 
 <li>使用括号对着色器插件打开大文件时发生崩溃</li> 
 <li>由于终端的否决，<code>lose Remote Connection</code> 被阻止</li> 
</ul>
