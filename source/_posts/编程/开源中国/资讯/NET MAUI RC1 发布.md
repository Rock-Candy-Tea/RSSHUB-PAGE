                                                                                            <p>.NET MAUI <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdevblogs.microsoft.com%2Fdotnet%2Fdotnet-maui-rc-1%2F" target="_blank">发布</a>了首个 RC 版本。目前 SDK 的 API 已完成，其他库可进行更新，并为 GA 版本的兼容性做好准备。</p> 
<blockquote> 
 <p>.NET MAUI 是一个跨平台框架，用于使用 C# 和 XAML 创建原生移动和桌面应用。使用 .NET MAUI，开发者可以创建在 Android、iOS、macOS 和 Windows 从单个共享代码库运行的应用。</p>

<p><strong>.NET MAUI RC 版本包含什么？</strong></p> 
<p>作为一个支持多平台的应用程序构建框架，.NET MAUI 包含了适用于 Android、iOS、macOS 和 Windows 这些平台的 SDK。开发者除了通过 .NET MAUI 最大化代码共享和生产力之外，还可以直接将它们与 C# 一起搭配使用。</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-e4c80c585b5d9ba71dbb2d005fec9a76665.png" referrerpolicy="no-referrer"></p> 
<p>.NET MAUI 附带了 40 多种布局和控件，它们针对在桌面和移动平台上构建自适应 UI 进行了优化。开发者还可以整合 Blazor 组件或整个 Blazor 应用程序，以便在桌面和移动设备上提供与在 Web 上相同的体验。</p> 
<p><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fdotnet%2Fmaui%2Freleases%2Ftag%2F6.0.300-rc.1" target="_blank">详情更新内容查看 Changelog</a>。</p> 
<p>如需获取 .NET MAUI RC1，将 Visual Studio 2022 预览版更新至<strong> </strong><span style="background-color:#ffffff; color:#333333"><strong>17.2 Preview 3</strong>。</span></p>
