                                                                                            <p>WebStorm 2022.1.4 已发布，这是 WebStorm 2022.1的第四个错误修复更新版本。</p> 
<p style="margin-left:0px">可以通过使用 Toolbox 应用程序、直接从 IDE 安装或从<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.jetbrains.com%2Fwebstorm%2Fdownload%2Fother.html" target="_blank">网站</a>下载来更新到 v2022.1.4 。</p> 
<p style="margin-left:0px">以下是 v2022.1.4 中最值得注意的修复：</p> 
<ul> 
 <li>WebStorm 不在 macOS High Sierra ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-296489" target="_blank">IDEA-296489</a> ) 上崩溃。</li> 
 <li>修复了<em>Commit</em>工具窗口的 <em>Before Commit </em>部分中的 <em>Run Git hooks </em>复选框( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-264817" target="_blank">IDEA-264817</a> )。</li> 
 <li>修复了导致目录的外部差异工具发生故障的问题 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-295565" target="_blank">IDEA-295565</a> )。</li> 
 <li>修复了导致 <em>提交详细信息 </em>窗格中文本闪烁的问题 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-284246" target="_blank">IDEA-284246</a> )。</li> 
</ul> 
<p style="margin-left:0px">有关 WebStorm 2022.1.4 中解决的 Bug 的完整列表，请参阅<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Farticles%2FWEB-A-233538363%2FWebStorm-221600813-Release-Notes" target="_blank">发行说明</a>。</p>
