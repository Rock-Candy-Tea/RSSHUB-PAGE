                                                                                            <p><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fmp.weixin.qq.com%2Fs%2FgTltRTbFkAwMUs7CYA5zpw" target="_blank">CutefishOS 0.7 已发布</a>，主要更新内容如下：</p> 
<p style="text-align:start"><strong>设置</strong></p> 
<ul> 
 <li>Wifi 支持连接隐藏网络</li> 
 <li>通知勿扰模式</li> 
 <li>触摸板新增自然滚动选项</li> 
 <li>系统更新图形用户界面</li> 
</ul> 
<p style="text-align:start"><strong>文件管理器</strong></p> 
<ul> 
 <li>U盘安全移除</li> 
 <li>卸载磁盘分区</li> 
 <li>UI 细节改善</li> 
 <li>可运行应用快捷方式 (Desktop 文件)</li> 
 <li>支持快捷键 Ctrl + Z 撤销操作</li> 
 <li>创建文件夹新交互</li> 
 <li>应用启动菜单</li> 
 <li>新增右键卸载</li> 
</ul> 
<p style="text-align:start"><strong>桌面</strong></p> 
<ul> 
 <li>桌面后台增加字体配置，解决在部分场景下的字体显示异常</li> 
 <li>登陆界面支持高分辨率</li> 
 <li>支持将文件拖拽到程序坞回收站</li> 
 <li>新增 GTK 主题，使 GTK 应用程序与 CutefishOS 风格更加统一</li> 
</ul> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-48c3a4a58cf22ae8c6e075e26bc86c293e5.png" referrerpolicy="no-referrer"></p> 
<p><strong>Cutefish Wine</strong></p> 
<p style="color:#111111; margin-left:0; margin-right:0; text-align:start">Wine 是能够使 Windows 程序在 Unix & Linux 上运行的应用程序的兼容层，Cutefish Wine 做了以下优化：</p> 
<ul> 
 <li>容器化运行，无需安装 32 位架构软件包，不会给系统带来依赖问题</li> 
 <li>适配 CutefishOS 高分屏设置，无需自己手动配置</li> 
 <li>新增微信 Wine 支持，安装完成后一键打开<br> 安装方法:<br> sudo apt update<br> sudo apt install cutefish-wine cutefish-wechat</li> 
</ul> 
<p style="text-align:start"><strong>Bug 修复</strong></p> 
<ul> 
 <li>首次打开桌面，程序坞回收站状态异常</li> 
 <li>文件管理器键盘首字母搜索滚到当前项</li> 
 <li>修复文件管理器回收站没有实时更新</li> 
</ul> 
<p>下载地址：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fcn.cutefishos.com%2Fdownload" target="_blank">https://cn.cutefishos.com/download</a></p> 
<p>根据官网的介绍，Cutefish 是一款具有现代风格设计与实用的 Linux 桌面操作系统，可以满足在各种场景下的使用需求。</p> 
<p><img src="https://oscimg.oschina.net/oscnet/up-6e023c06c5453d0b0c1bf4f4fd874ec9314.png" referrerpolicy="no-referrer"></p> 
<p><span style="background-color:#ffffff; color:#333333">从整体外观来看，CutefishOS 的风格与 macOS 比较接近，它的 Dock 底栏、图标风格、控制中心以及小部件等元素都受到了 macOS 的影响。CutefishOS 的桌面环境 CuteFishDE 使用了 KDE 框架、Qt 和 KDE Plasma 5，似乎还借鉴了深度操作系统的桌面环境（DDE）、MATE 和 JingOS。</span></p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><img alt src="https://oscimg.oschina.net/oscnet/up-6e023c06c5453d0b0c1bf4f4fd874ec9314.png" referrerpolicy="no-referrer"></p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><img alt src="https://oscimg.oschina.net/oscnet/up-f074cf0423d006695f9344b910a7f39ff71.png" referrerpolicy="no-referrer"></p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><img alt src="https://oscimg.oschina.net/oscnet/up-fd3290402335c51e4ed58e4818cb22c1e04.png" referrerpolicy="no-referrer"></p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">应用方程序面，CutefishOS 预装了小部分 KDE 软件，例如文本编辑器 Kate、截图工具 Spectacle 和 Muon Package Manager，以及 Firefox、VLC 和 MPV 等知名应用。</p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">CutefishOS 的桌面环境还预装了一些原生的 CuteFish 应用程序，包括文件管理器、计算器、终端和设置中心。此外，CutefishOS 顶部拥有全局菜单，它是一个应用程序所有功能的集合，非常方便程序功能与系统整合。</p>
