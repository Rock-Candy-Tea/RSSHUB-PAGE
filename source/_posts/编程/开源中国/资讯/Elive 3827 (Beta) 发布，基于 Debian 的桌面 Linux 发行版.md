                                                        <p>Elive 3.8.27 Beta 版本已发布。</p> 
<p><strong>更新亮点</strong></p> 
<ul> 
 <li>改进安装程序的兼容性，支持通过 Windows 系统进行引导</li> 
 <li>Web 浏览器的临时文件运行在 RAM 中，大幅提升了 SSD 磁盘或 USB 在持久模式下的寿命</li> 
 <li>优化 Vim 配色方案</li> 
 <li>可正确检测到 NVIDIA 显卡，并安装额外的驱动程序</li> 
 <li>改进 Macbooks<span> 上的风扇支持</span></li> 
 <li><strong>Wallpapers switching</strong><span> </span>demo 回归，并且在 E16 桌面上提供</li> 
 <li>改进 Tmux，提供更好的默认配置</li> 
 <li>将底层的 Debian<span> 升级至 </span>10.12</li> 
 <li>更新 Terminology<span> 终端应用，带来了新特性</span></li> 
 <li>优化内核稳定性</li> 
</ul> 
<p><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.elivecd.org%2Freleases%2Fbeta%2Felive-3-8-27-beta-released%2F" target="_blank">详情查看发布公告</a>。</p> 
<hr> 
<p>Elive，或者称为 Enlightenment live CD，是基于 Debian 的桌面 Linux 发行及自启动运行光盘，其特色在于使用 Enlightenment 窗口管理器。除了进行预配置并适合于日常桌面应用外，它还包含了“Elpanel”，一份面向简易系统及桌面管理的控制中心。</p>
