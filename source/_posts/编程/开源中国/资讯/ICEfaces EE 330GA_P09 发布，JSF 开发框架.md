                                                                                            <p><span style="background-color:#ffffff; color:#333333">ICEfaces EE 3.3.0.GA_P09 发布了，ICEfaces 是一个基于 Ajax 的 JSF 开发框架（JavaServer Faces）。ICEfaces 原本是一个商业产品，现已基于 Mozilla Public License 协议开源。它提供一整套完整的 Java EE 应用程序开发组件，能够帮助开发人员用纯 Java 快速开发瘦客户端胖互联网应用程序(Rich Internet Applications:RIA)。</span></p> 
<p><strong><span style="background-color:#ffffff; color:#333333">主要更新内容</span></strong></p> 
<ul> 
 <li><span style="background-color:#ffffff; color:#333333">jQuery Security Patches</span> 
  <ul> 
   <li><span style="background-color:#ffffff; color:#333333">更新到 jQuery 1.12.4 和 jQuery Migrate 1.4.1</span></li> 
   <li><span style="background-color:#ffffff; color:#333333">将已知 jQuery 漏洞的安全补丁应用于 ICEfaces 的分叉 jQuery 库</span></li> 
  </ul> </li> 
 <li><span style="background-color:#ffffff; color:#333333">ace:autoCompleteEntry</span> 
  <ul> 
   <li><span style="background-color:#ffffff; color:#333333">​​​​​​​添加了“alt”、“dir”、“lang”、“maxlength”、“size”和“title”传递属性</span></li> 
  </ul> </li> 
 <li><span style="background-color:#ffffff; color:#333333">ace:comboBox</span> 
  <ul> 
   <li><span style="background-color:#ffffff; color:#333333">​​​​​​​ace:comboBox 和 ace:selectMenu 在 Safari 上无法正确呈现</span></li> 
  </ul> </li> 
 <li><span style="background-color:#ffffff; color:#333333">ace:dataTable</span> 
  <ul> 
   <li><span style="background-color:#ffffff; color:#333333">​​​​​​​ace:tabSet 中的延迟加载 ace:dataTable 加载过快</span></li> 
   <li><span style="background-color:#ffffff; color:#333333">固定列不会在 Firefox 上滚动</span></li> 
  </ul> </li> 
 <li><span style="background-color:#ffffff; color:#333333">ace:dateTimeEntry</span> 
  <ul> 
   <li><span style="background-color:#ffffff; color:#333333">​​​​​​​添加了 “标题” 传递属性</span></li> 
  </ul> </li> 
</ul>
