<h1>vxe-table</h1> 
<p>一个基于 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.npmjs.com%2Fpackage%2Fvue" target="_blank">vue</a> 的 PC 端表格组件，支持增删改查、虚拟滚动、懒加载、快捷菜单、数据校验、树形结构、打印导出、表单渲染、数据分页、虚拟列表、弹窗、自定义模板、渲染器、贼灵活的配置项、扩展接口等...</p> 
<ul> 
 <li> <p>设计理念</p> 
  <ul> 
   <li>面向现代浏览器，高效的简洁 API 设计</li> 
   <li>模块化表格、按需加载、扩展接口</li> 
   <li>为单行编辑表格而设计，支持增删改查及更多扩展，强大的功能的同时兼具性能</li> 
  </ul> </li> 
 <li> <p>计划</p> 
  <ul> 
   <li> v1.0 基于 vue2.6+，支持所有主流的浏览器，实现表格的一切实用的功能</li> 
   <li> v2.0 基于 vue2.6+，支持所有主流的浏览器，同时兼具功能与性能</li> 
   <li> v3.0 基于 vue2.6+，支持现代浏览器并保留兼容 IE11，提升渲染性能</li> 
   <li> v4.0 基于 vue3.0+，只支持现代浏览器，不支持 IE</li> 
  </ul> </li> 
</ul>
