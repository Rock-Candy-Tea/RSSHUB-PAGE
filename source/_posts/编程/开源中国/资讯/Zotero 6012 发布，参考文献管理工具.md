                                                                                            <p>Zotero 是一个参考文献管理软件，也是一个免费 Firefox 和 Google 浏览器扩展。Zotero 可以协助用户收集、管理及引用研究资源，包括期刊、书籍等各类文献和网页、图片等。</p> 
<p>目前，Zotero 6.0.12 已发布， 该版本更新内容如下：</p> 
<p><strong>功能更改或添加</strong></p> 
<ul> 
 <li> <p>在 PDF 阅读器工具栏中添加了用于 PDF 内链接的 Back button（也可通过<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.zotero.org%2Fsupport%2Fkb%2Fkeyboard_shortcuts%23pdf_reader" target="_blank">键盘快捷键</a>使用）</p> </li> 
 <li> <p>添加了“Create Book Section”和“Create Book from Book Section”上下文菜单选项</p> </li> 
 <li> <p>允许使用鼠标滚轮滚动标签</p> </li> 
</ul> 
<p><strong>已修复的错误</strong></p> 
<ul> 
 <li> <p>修复了库之间独立附件的复制</p> </li> 
 <li> <p>修复了从 Zotero RDF 导入独立附件的问题</p> </li> 
 <li> <p>合并重复项时保留嵌入的 PDF 注释</p> </li> 
 <li> <p>修复了某些引文样式（自 6.0.10 起）未出现的事件标题（例如，会议名称）</p> </li> 
 <li> <p>修复了 Audio Recording 未在引文中显示为作者的问题（自 6.0.10 起）</p> </li> 
 <li> <p>不要在经典引文对话框中为多个来源预先选择 Act</p> </li> 
 <li> <p>修复了从 ScienceDirect 下载大型 PDF 的问题</p> </li> 
 <li> <p>修复了启动时未选择最后选择的 PDF tab</p> </li> 
 <li> <p>修复了重启后现有的 PDF tab 不会被重复用于同一文件的问题</p> </li> 
 <li> <p>将 annotations 拖到 notes 时不要创建额外的换行符</p> </li> 
 <li> <p>修复了 library vie 中尚未选择的提要的高级搜索问题</p> </li> 
 <li> <p>修复了 single-field creators 的 Shift-Tab 中断</p> </li> 
 <li> <p>其他错误修复</p> </li> 
</ul> 
<p><span style="background-color:#ffffff; color:#333333">更新说明：</span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.zotero.org%2Fsupport%2Fchangelog" target="_blank">https://www.zotero.org/support/changelog</a> </p>
