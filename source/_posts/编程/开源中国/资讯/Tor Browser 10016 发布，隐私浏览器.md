                                                        <p>Tor Browser 10.0.16 现已发布。Tor Browser 是一个基于 Firefox ESR (Firefox with extended support) 的 Web 浏览器，默认配置通过 Tor 和 Vidalia 实现了个人隐私保护和匿名。</p> 
<p>这个版本将 Firefox 更新到 78.10esr。此外，Tor 浏览器 10.0.16 将 NoScript 更新到 11.2.4，并增加了缅甸语的本地化。</p> 
<p>提醒：Tor Browser 将在今年晚些时候停止支持版本 2 的 onion services，因此请尽快将你的服务迁移并将你的书签更新到版本 3 的 onion services。</p> 
<p>完整更新日志如下：</p> 
<ul> 
 <li>Windows + OS X + Linux： 
  <ul> 
   <li>更新 Firefox 到 78.10.0esr；</li> 
   <li>更新 NoScript 到 11.2.4；</li> 
   <li>Bug 40007：更新 Moat 的域名前置配置；</li> 
   <li>Bug 40390：添加缅甸语为新的语言环境；</li> 
   <li>Bug 40408：不允许在所有网页内容中使用 SVG Context Paint。</li> 
  </ul> </li> 
</ul>
