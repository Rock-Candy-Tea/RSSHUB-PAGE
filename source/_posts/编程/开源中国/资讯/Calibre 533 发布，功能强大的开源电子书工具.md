                                                        <p>Calibre 开源项目是 Calibre 官方出的电子书管理工具。它可以查看，转换，编辑和分类所有主流格式的电子书。Calibre 是个跨平台软件，可以在 Linux、Windows 和 macOS 上运行。</p> 
<p>Calibre 5.33 正式发布，此次更新内容如下：</p> 
<h3>新功能</h3> 
<ul> 
 <li>允许改变 Calibre 库的图标，在 Calibre 中右击库的图标，选择 "改变这个库的图标"。</li> 
 <li>Comments 编辑器：如果三个工具栏都适合的话，可以使用单行。</li> 
 <li>编辑书籍：允许通过将 HTML 文件拖放到另一个 HTML 文件上来合并文件。</li> 
 <li>Kobo 驱动程序：增加对列出购买的有声读物的支持。</li> 
 <li>编辑元数据：封面和格式栏现在为封面图片提供尽可能多的垂直空间。</li> 
 <li>添加书籍对话框：当下次使用对话框时添加非图书文件类型，预先选择 "所有文件" 过滤器</li> 
 <li>calibre-server --manage-users：允许在服务器运行时管理用户，同时增加自动改变用户账户限制的操作。</li> 
 <li>EPUB 3 元数据：如果书中包含一个 "副标题"，在阅读元数据时将其附加到主标题上。</li> 
</ul> 
<h3>错误修复</h3> 
<ul> 
 <li>电子书查看器：修复启动时在 "加载章节" 处偶尔出现的挂起现象</li> 
 <li>获取书籍：更新 Kobo 插件以适应网站变化</li> 
 <li>批量编辑元数据对话框：修复改变搜索模式后重置其他字段的问题</li> 
 <li>电子书查看器：修复在使用带连字符的 HTML IDs 的书籍中高亮部分的排序错误。</li> 
 <li>PDF 输出：修复在 macOS 上使用某些字体时，在行尾断开长字的选项导致在行尾呈现方框的问题。</li> 
 <li>Google 元数据插件：当按 ISBN 搜索时，如果没有发现的结果，重试使用一个替代的查询语法。</li> 
 <li>5.33.2 修复了几个回归问题，这些问题破坏了 Windows 上弹出的评论编辑器对话框中的工具栏和编辑元数据对话框中的下载元数据按钮的渲染，以及 Windows 上从 MTP 设备中读取文件的问题。</li> 
</ul>
