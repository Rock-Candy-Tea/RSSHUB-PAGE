                                                        <p>youtube-dl 是一个小型的命令行工具，用于下载视频。虽然它最初仅支持 <a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fyoutube.com%2F">YouTube.com</a>，但它目前已支持许多其他平台，如 Anitube、<a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Farchive.org%2F">Archive.org</a>、Bloomberg、CBS、Discovery、dropbox、eHow、flickr、Google+、MTV、MyVideo、NBC、SoundCloud、Southpark、Steam 和 Vimeo 等。它可以保存视频 MP4 和其他提供的格式，也可以只提取音轨。</p> 
<p>youtube-dl 2021.04.26 正式发布，本次更新内容如下：</p> 
<p>提取器：</p> 
<ul> 
 <li>[xfileshare] 增加对 <a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fwolfstream.tv%2F" target="_blank">wolfstream.tv</a> 的支持 (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fytdl-org%2Fyoutube-dl%2Fpull%2F28858" target="_blank">#28858</a>)</li> 
 <li>[francetvinfo] 改进视频 ID 提取 (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fytdl-org%2Fyoutube-dl%2Fissues%2F28792" target="_blank">#28792</a>)</li> 
 <li>[medaltv] 修复提取功能 (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fytdl-org%2Fyoutube-dl%2Fpull%2F28807" target="_blank">#28807</a>)</li> 
 <li>[tver] 重定向所有下载到 Brightcove (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fytdl-org%2Fyoutube-dl%2Fpull%2F28849" target="_blank">#28849</a>)</li> 
 <li>[go] 改进视频 ID 提取 (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fytdl-org%2Fyoutube-dl%2Fissues%2F25207" target="_blank">#25207</a>, <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fytdl-org%2Fyoutube-dl%2Fpull%2F25216" target="_blank">#25216</a>, <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fytdl-org%2Fyoutube-dl%2Fissues%2F26058" target="_blank">#26058</a>)</li> 
 <li>[youtube] 修复 lazy 提取器 (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fytdl-org%2Fyoutube-dl%2Fissues%2F28780" target="_blank">#28780</a>)</li> 
 <li>[bbc] 从 INITIAL_DATA 提取描述和时间戳 (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fytdl-org%2Fyoutube-dl%2Fpull%2F28774" target="_blank">#28774</a>)</li> 
 <li>[cbsnews] 修复对 Python < 3.6 的提取 (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fytdl-org%2Fyoutube-dl%2Fissues%2F23359" target="_blank">#23359</a>)</li> 
</ul>
