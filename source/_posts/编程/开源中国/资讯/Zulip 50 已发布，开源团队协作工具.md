<div class="content">
                                                                    
                                                        <p>Zulip 是一个<span style="background-color:#ffffff; color:#40485b">开源团队协作工具，一款专为实时和异步对话而设计的现代团队聊天应用程序，</span><span style="background-color:#ffffff; color:#333333">支持快速搜索、拖放文件上传、图像预览、组私人消息、可听通知、错过电子邮件消息提醒与桌面应用等。</span></p> 
<p>最新<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fzulip%2Fzulip%2Freleases%2Ftag%2F5.0" target="_blank">发布的</a> Zulip Server 5.0 是一个主要版本，自去年 5 月的 4.0 以来，项目中合并了超过 7000 个新提交，新功能包括状态表情符号、将主题标记为已解决、改进的流和权限管理等内容，下面未该版本的部分改动：</p> 
<ul> 
 <li> <p style="margin-left:0; margin-right:0"><span><span><span><span><span><span><span><span><span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fzulip.com%2Fhelp%2Fresolve-a-topic" target="_blank">解决主题</a>功能允许将主题标记为 ✔ 已完成状态。</span></span></span></span></span></span></span></span></span></p> </li> 
 <li> <p style="margin-left:0; margin-right:0"><span style="background-color:#ffffff; color:#35373a">文本编辑框经过重新设计，外观更简洁，支持粗体、斜体和超链接功能，支持插入全球时间、支持</span><a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fzulip.com%2Fhelp%2Fresize-the-compose-box" target="_blank">全屏</a>功能。</p> </li> 
 <li> <p style="margin-left:0; margin-right:0">可以在状态消息旁边<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fzulip.com%2Fhelp%2Fstatus-and-availability%23set-a-status" target="_blank">设置状态 emoji</a>：</p> </li> 
</ul> 
<p style="margin-left:0; margin-right:0"><img alt height="344" src="https://oscimg.oschina.net/oscnet/up-7479c637af5a6ebf40aec880db1c685787c.png" width="400" referrerpolicy="no-referrer"></p> 
<ul> 
 <li><span style="background-color:#ffffff; color:#24292f">重新设计流的创建过程，界面更加简洁</span></li> 
 <li>重新设计的“完整用户资料”小部件可以显示用户的流和用户组订阅</li> 
 <li>组织管理员现在可以为加入组织的新用户配置默认的个人偏好设置</li> 
 <li><span style="background-color:#ffffff; color:#35373a"><span> </span>Zulip UI 拥有数百项改进</span></li> 
 <li>如果目标消息已移动到新的流或主题，对话的永久链接也会正确重定向</li> 
 <li>添加了在 ARM 平台（包括 Mac M1）上安装的支持。</li> 
 <li>删除了对 Ubuntu 18.04 的支持</li> 
 <li>...</li> 
</ul> 
<p style="margin-left:0px; margin-right:0px">有关该版本的更多详细内容可在<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fblog.zulip.com%2F2022%2F03%2F29%2Fzulip-5-0-released%2F" target="_blank">官方博客</a>中阅览。</p>
