                                                        <p>NativeScript 8.0.7 现已发布。NativeScript 使用户能够直接从 JavaScript 访问 native API。目前，该框架为丰富的移动开发提供了 iOS 和 Android 运行时，并可用于多种不同的用例。</p> 
<p>此版本主要更新内容如下：</p> 
<p><strong>Bug Fixes</strong></p> 
<ul> 
 <li> <p><strong>animations：</strong>在视图卸载后停止 keyframe animations ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FNativeScript%2FNativeScript%2Fissues%2F9421" target="_blank">#9421</a> ) ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FNativeScript%2FNativeScript%2Fcommit%2F04381fa3e74e9c277970fb282223abedd937b4c1" target="_blank">04381​​fa</a> )</p> </li> 
 <li><strong>box-shadow：</strong>支持“none”（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FNativeScript%2FNativeScript%2Fissues%2F9418" target="_blank">#9418</a>）（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FNativeScript%2FNativeScript%2Fcommit%2F263c920cb8a4b22e257033006ce585ec32ca2605" target="_blank">263c920</a>）</li> 
 <li><strong>core：</strong>在异步条件下保护不稳定的页面/帧初始化上下文（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FNativeScript%2FNativeScript%2Fissues%2F9428" target="_blank">#9428</a>）（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FNativeScript%2FNativeScript%2Fcommit%2Fd3bc4d5b82879ccb83483f99585127d981b43b38" target="_blank">d3bc4d5</a>）</li> 
 <li><strong>core：</strong>Utils.queueMacrotask 的类型和 Observable 上的静态方法（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FNativeScript%2FNativeScript%2Fissues%2F9425" target="_blank">#9425</a>）（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FNativeScript%2FNativeScript%2Fcommit%2Fd589ac600063416f3725c0abf8dfc88f556166db" target="_blank">d589ac6</a>）</li> 
</ul> 
<p><strong>Features</strong></p> 
<ul> 
 <li><strong>WebView：</strong>允许通过 loadStarted 拦截 JS 绑定的窗口框架导航（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FNativeScript%2FNativeScript%2Fissues%2F9430" target="_blank">#9430</a>）（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FNativeScript%2FNativeScript%2Fcommit%2F3806b85fcef01ad34327dda9cc2c88619b522c74" target="_blank">3806b85</a>）</li> 
</ul>
