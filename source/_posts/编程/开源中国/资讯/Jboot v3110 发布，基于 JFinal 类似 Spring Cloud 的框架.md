  <p style="box-sizing: inherit; margin: 0px 0px 20px; line-height: inherit; text-align: left;">Jboot 一个更简单的分布式、微服务框架。</p> 
  <p style="box-sizing: inherit; margin: 0px; line-height: inherit; text-align: left;">Jboot是一个基于 JFinal、JFinal-Undertow、Dubbo、Seata、Sentinel、ShardingSphere、Nacos 等开发的微服务框架，帮助开发者降低微服务、分布式开发门槛。爽爽开发，快乐生活。</p> 
 </blockquote> 
 <p style="box-sizing: inherit; margin: 0px 0px 20px; line-height: inherit; color: rgb(51, 51, 51); font-family: -apple-system, system-ui, "Apple Color Emoji", "Segoe UI Emoji", "Segoe UI Symbol", "Segoe UI", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei", "Helvetica Neue", Helvetica, Arial, sans-serif; font-size: 16px; font-style: normal; font-variant-ligatures: normal; font-variant-caps: normal; font-weight: 400; letter-spacing: normal; orphans: 2; text-align: left; text-indent: 0px; text-transform: none; white-space: normal; widows: 2; word-spacing: 0px; -webkit-text-stroke-width: 0px; background-color: rgb(255, 255, 255); text-decoration-thickness: initial; text-decoration-style: initial; text-decoration-color: initial;"><span style="box-sizing: inherit; background-color: rgb(255, 255, 255); color: rgb(51, 51, 51);">到目前为止，Jboot 已经开源超过了 5 年的时间，迭代了 200+ 个版本，已经被超过 1000+ 公司在使用，其中包含了多个知名的上市公司，我们了解到的多个使用 Jboot 开发的产品，用户量超过 1亿 以上。</span></p> 
 <p style="box-sizing: inherit; margin: 0px 0px 20px; line-height: inherit; color: rgb(51, 51, 51); font-family: -apple-system, system-ui, "Apple Color Emoji", "Segoe UI Emoji", "Segoe UI Symbol", "Segoe UI", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei", "Helvetica Neue", Helvetica, Arial, sans-serif; font-size: 16px; font-style: normal; font-variant-ligatures: normal; font-variant-caps: normal; font-weight: 400; letter-spacing: normal; orphans: 2; text-align: left; text-indent: 0px; text-transform: none; white-space: normal; widows: 2; word-spacing: 0px; -webkit-text-stroke-width: 0px; background-color: rgb(255, 255, 255); text-decoration-thickness: initial; text-decoration-style: initial; text-decoration-color: initial;"><strong style="box-sizing: inherit; font-weight: 700;">Jboot v3.11.0 更新内容如下：</strong></p> 
 <div style="box-sizing: inherit; color: rgb(51, 51, 51); font-family: -apple-system, system-ui, "Apple Color Emoji", "Segoe UI Emoji", "Segoe UI Symbol", "Segoe UI", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei", "Helvetica Neue", Helvetica, Arial, sans-serif; font-size: 16px; font-style: normal; font-variant-ligatures: normal; font-variant-caps: normal; font-weight: 400; letter-spacing: normal; orphans: 2; text-indent: 0px; text-transform: none; white-space: normal; widows: 2; word-spacing: 0px; -webkit-text-stroke-width: 0px; background-color: rgb(255, 255, 255); text-decoration-thickness: initial; text-decoration-style: initial; text-decoration-color: initial; text-align: start;"> 
  <ul> 
   <li style="box-sizing: inherit; line-height: 1.875em; margin-top: 0px;">新增：实验性新增 Restful 支持，默认关闭，可以通过 "jboot.web.pathVariableEnable=true" 开启，感谢 @没牙的小朋友</li> 
   <li style="box-sizing: inherit; line-height: 1.875em; margin-top: 0px;">新增：shiro 配置添加 filter 参数，自定义shiro相关的filter，感谢 @orangeJ</li> 
   <li style="box-sizing: inherit; line-height: 1.875em; margin-top: 0px;">新增：JbootCache 新增通过前缀了使用 "多实例" 的支持</li> 
   <li style="box-sizing: inherit; line-height: 1.875em; margin-top: 0px;">新增：Model、Db 编码动态设置默认数据源的支持</li> 
   <li style="box-sizing: inherit; line-height: 1.875em; margin-top: 0px;">新增：DatasourceConfig 新增表前缀配置的支持</li> 
   <li style="box-sizing: inherit; line-height: 1.875em; margin-top: 0px;">优化：在shiro拦截器初始化时处理对应方法的注解，取消对路由、actionKey的倚赖，感谢 @orangeJ</li> 
   <li style="box-sizing: inherit; line-height: 1.875em; margin-top: 0px;">优化：修改 JbootActionMapping 拼写错误</li> 
   <li style="box-sizing: inherit; line-height: 1.875em; margin-top: 0px;">优化：AttachmentManager 新增多实例的支持</li> 
   <li style="box-sizing: inherit; line-height: 1.875em; margin-top: 0px;">优化：优化缓存前缀配置的方法名，否则可能会造成误解</li> 
   <li style="box-sizing: inherit; line-height: 1.875em; margin-top: 0px;">优化：同时修改 "jboot.web.escapeParas"配置为 "jboot.web.escapeParasEnable"</li> 
   <li style="box-sizing: inherit; line-height: 1.875em; margin-top: 0px;">修复：修正现有swagger实现中，会忽略ActionKey注解中的URL的问题，感谢 @没牙的小朋友</li> 
  </ul> 
  <p style="box-sizing: inherit; line-height: 1.875em; margin-top: 0px;"> </p>
