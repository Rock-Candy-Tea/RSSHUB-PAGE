                                                                                            <p>Borax v3.5.5 已经发布，Python 3 开发工具集合。</p> 
<p>此版本更新内容包括：</p> 
<ul> 
 <li><code>borax.calendars.festivals2</code> 
  <ul> 
   <li><code>FestivalLibrary.iter_month_daytuples</code> 新增 <code>return_pos</code> 参数，可返回日期位置</li> 
   <li><code>Festival</code> 新增 <code>list_days_in_future</code> / <code>list_days_in_past</code> 函数</li> 
  </ul> </li> 
 <li><code>borax.calendars.utils</code> 
  <ul> 
   <li>修正 <code>ThreeNineUtils</code> 内部计算错误的BUG</li> 
  </ul> </li> 
 <li><code>borax.htmls</code> 
  <ul> 
   <li><code>html_tag</code> 函数新增 width / height 参数</li> 
   <li>修正 <code>html_tag</code> 函数style参数解析空值的BUG</li> 
  </ul> </li> 
</ul> 
<p>详情查看：<a href="https://gitee.com/kinegratii/borax/releases/v3.5.5">https://gitee.com/kinegratii/borax/releases/v3.5.5</a></p>
