                                                        <p>Jedis 是 Redis 的一个 Java 客户端库，Jedis 4.0.0 RC1 发布，更新内容如下：</p> 
<h3>重要变化</h3> 
<ul> 
 <li>大多数 SortedSet 方法被改为返回 Java <code>List</code> 而不是 <code>Set</code>。</li> 
 <li>将 RedisCommandObjects 更名为 CommandObjects，将 RedisClusterCommandObjects 更名为 ClusterCommandObjects。</li> 
</ul> 
<h3>新功能</h3> 
<ul> 
 <li>实现了 JedisSharding</li> 
 <li>实现了 ClusterPipeline 和 ShardedPipeline</li> 
</ul> 
<h3>维护</h3> 
<p>恢复了 Jedis 3 sendCommand 方法</p>
