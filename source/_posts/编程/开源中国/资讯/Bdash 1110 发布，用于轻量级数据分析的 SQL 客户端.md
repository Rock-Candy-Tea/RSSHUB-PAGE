                                                        <p>Bdash 是一个用于轻量级数据分析的简单 SQL 客户端，适用于 Windows、macOS 和 Linux 操作系统。</p> 
<p>Bdash 1.11.0 正式发布，自 1.10.0 以来的更新内容包括：</p> 
<ul> 
 <li>更新 BigQuery SDK；</li> 
 <li>使 bdash 服务器的 url 路径规范成为非必要的；</li> 
 <li>分享到 gist 时增加图表的宽度；</li> 
 <li>向 bdash 服务器发送数据源信息；</li> 
 <li>在 bdash 服务器上实现共享；</li> 
 <li>分享到 bdash 服务器时增加图表的宽度</li> 
</ul>
