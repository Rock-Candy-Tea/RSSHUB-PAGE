                                                        <p>Electron 12.0.8 现已发布，主要更新内容如下： </p> 
<p><strong>Fixes</strong></p> 
<ul> 
 <li>修复了<code>webFrame</code>spell checker API 在沙盒渲染器中调用时崩溃的问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Felectron%2Felectron%2Fpull%2F29086" target="_blank">＃29086 </a></li> 
 <li>修复了启用<code>nodeIntegrationInSubFrames</code>时可能引发的<code>illegal access error</code>问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Felectron%2Felectron%2Fpull%2F29169" target="_blank">＃29169 </a></li> 
 <li>修复了没有背景页的扩展可能无法访问文件的问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Felectron%2Felectron%2Fpull%2F29172" target="_blank">＃29172 </a></li> 
 <li>修复了 void 函数<code>Menu.setApplicationMenu</code>在某些平台上会返回一个值的问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Felectron%2Felectron%2Fpull%2F29128" target="_blank">＃29128 </a></li> 
 <li>修复了 macOS 11 中 vibrancy view 的 corner radius。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Felectron%2Felectron%2Fpull%2F29071" target="_blank">＃29071 </a></li> 
</ul>

</ul>
