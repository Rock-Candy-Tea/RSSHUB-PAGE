  <div style="margin-left:0; margin-right:0"> 
   <p style="margin-left:0; margin-right:0"><span style="color:#333333">Mastodon 3.4.4 发布了。Mastodon 是一个免费的开源社交网络程序，一个商业平台的替代方案，避免了单个公司垄断沟通的风险。无论选择哪个服务器，都可以与其他人进行互动。通过运行自己的 Mastodon 实例，可无缝地参与到社交网络中。</span></p> 
   <p style="margin-left:0; margin-right:0"><span style="color:#333333"><strong>此版本修复以下 bug</strong></span></p> 
   <ul style="margin-left:0; margin-right:0"> 
    <li><span style="color:#2e3033">修正当用户挂起一个已经被阻止的规范电子邮件时的错误</span></li> 
    <li><span style="color:#2e3033">修复管理员界面中长配置文件字段溢出的问题</span></li> 
    <li><span style="color:#2e3033">修正 WebFinger 请求返回空文档时的混乱错误</span></li> 
    <li>修复使用 OpenStack Swift 上传远程媒体有时会失败的问题 </li> 
    <li>修复注销链接在 Safari 中不起作用的问题</li> 
    <li>修复了媒体模式的“打开”链接，而不是在 Web UI 中关闭模式</li> 
    <li>修复从 Web UI 中的模态回复</li> 
    <li>修复<span> </span><code>mastodon:setup</code><span> </span>在某些情况下命令崩溃的问题</li> 
   </ul> 
   <h3 style="margin-left:.6em; margin-right:0"><strong>安全</strong></h3> 
   <ul style="margin-left:0; margin-right:0"> 
    <li><span style="color:#2e3033">修正了从非跟踪用户中过滤 DM 的问题</span></li> 
    <li><span style="color:#2e3033">修正了在 web 中递归 toots 的处理</span></li> 
   </ul> 
   <h3 style="margin-left:.6em; margin-right:0"><span style="color:#2e3033">升级说明</span></h3> 
   <p style="margin-left:0; margin-right:0">这是一个向后移植，所以它不适用于<code>git pull</code><span> </span>。请用<span> </span><code>git fetch && git checkout v3.4.4</code><span> </span>。</p>
