                                                        <p>Apache Maven 3.8.1 发布了。Apache Maven 是一个项目管理和构建工具。基于项目对象模型（POM）的概念， Maven 可以从中心位置管理项目的构建、报告和文档。</p> 
<p>此版本更新内容如下：</p> 
<p><strong>Bug </strong></p> 
<ul> 
 <li>[MNG-7128] - 改进在构建 POM 中定义的 blocked repository 时的错误信息。</li> 
</ul> 
<p><strong>New Feature</strong></p> 
<ul> 
 <li>[MNG-7116] - 增加对 external:http:* 镜像选择器的支持</li> 
 <li>[MNG-7117] - 增加对 blocking mirrors 的支持</li> 
 <li>[MNG-7118] - 默认屏蔽外部 HTTP 资源库</li> 
</ul> 
<p><strong>Dependency upgrade</strong></p> 
<ul> 
 <li>将 Maven Wagon 升级至 3.4.3</li> 
 <li>将 Maven Resolver 升级至 1.6.2</li> 
</ul>
