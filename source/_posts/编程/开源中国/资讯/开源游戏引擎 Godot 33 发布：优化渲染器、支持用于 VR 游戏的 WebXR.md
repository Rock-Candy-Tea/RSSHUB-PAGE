                                                                                            <p><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgodotengine.org%2Farticle%2Fgodot-3-3-has-arrived" target="_blank">Godot 3.3 </a>已正式发布，其开发团队表示，此版本原本是作为 3.2 分支的 3.2.4 版本更新，但由于在开发过程中增加了许多新功能，因此团队决定直接将这个版本作为稳定分支 3.x 的更新版本。尽管团队目前的工作重点放在即将推出的 Godot 4.0 版本上，但他们依旧希望为使用者提供一个成熟而强大的 3.x 分支来开发游戏，并将该版本作为过渡版本。</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-36422c5070a21e77587aa060a081f33f525.png" referrerpolicy="no-referrer"></p> 
<p>Godot 3.3 与 Godot 3.2.x 兼容，开发团队推荐所有使用 3.2.x 的用户进行升级。值得注意的是，3.2.x 不会再发布任何新版本，Git 上的版本已重置为<code>3.2.3-stable</code>状态。</p> 
<p>下载地址：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgodotengine.org%2Fdownload" target="_blank">https://godotengine.org/download</a></p> 
<p>Godot 3.3 更新亮点：</p> 
<ul> 
 <li>提供基于 Web 版本 Godot 编辑器的初始构建版本</li> 
 <li>改进 HTML5 导出支持</li> 
 <li>在 Apple M1 Mac 上提供针对 macOS ARM64 的官方构建</li> 
 <li>对多线程 API 进行现代化处理</li> 
 <li>统一的 2D 批处理</li> 
 <li>引入新的 CPU light-mapper</li> 
 <li>以及其他渲染改进</li> 
</ul> 
<p>除此之外，Godot 3.3 在物理功能方面也有所增强，以及改进 3D 编辑器、引入 OpenXR 插件、对虚拟现实 (VR) 游戏的 WebXR 支持。由于 MP3 专利已过期并且解决了其他障碍，Godot 3.3 还提供了 MP3 音频加载/播放支持。</p> 
<p>详情访问<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgodotengine.org%2Farticle%2Fgodot-3-3-has-arrived" target="_blank">发布公告</a>进行查看。</p>
