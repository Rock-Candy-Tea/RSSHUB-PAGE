                                                                                            <p>Shutter 是一个功能丰富的屏幕截图程序，适用于基于 Linux 的操作系统。用户可以对一个特定的区域、窗口、整个屏幕，甚至是一个网站进行截图。并且可以在截图后对图片进行适当编辑，然后上传到图片托管网站，所有这些操作都能在一个窗口内完成。</p> 
<p>Shutter 0.98 正式发布，自 0.96 版本以来修复了众多 bug：</p> 
<h3>修复：</h3> 
<ul> 
 <li>在启动 Shutter 和实例已经在运行时显示主窗口；</li> 
 <li>在选择模式下，允许移动选择区域；</li> 
 <li>在选择模式下，允许通过双击来触发屏幕截图；</li> 
 <li>修复 Print 按钮；</li> 
 <li>当通过 FTP 上传截图时，现在可以将 URL 复制到剪贴板上；</li> 
 <li>修复 "Torn paper"和 "Watermark" 插件；</li> 
</ul> 
<h3>修复崩溃问题：</h3> 
<ul> 
 <li>修复了在设置 "包括光标" 的情况下进行屏幕截图时的崩溃问题；</li> 
 <li>修复了在编辑器中使用文本注解工具时的崩溃问题；</li> 
 <li>在 Wayland 上启动时；</li> 
 <li>当使用一个没有专门的 Pictures/ 目录的 DE 时；</li> 
</ul> 
<h3>其他：</h3> 
<ul> 
 <li>在 Gnome 上不显示托盘图标，使得在主窗口关闭/隐藏后无法与应用程序交互； 
  <ul> 
   <li>如果 libappindicator 不可用，并且传统的托盘图标无法正常显示，关闭窗口现在会终止应用程序，而不是将其隐藏到不存在的托盘中；</li> 
  </ul> </li> 
 <li>复制到剪贴板无法正常运作；</li> 
 <li>在多个屏幕上捕捉全屏无法运作；</li> 
</ul>
