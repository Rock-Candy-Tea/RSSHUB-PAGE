                                                                                            <p>vxe-table 4.2.0 已经发布，vue 表格解决方案</p> 
<p>此版本更新内容包括：</p> 
<ul> 
 <li>重要更新，存在兼容性变动！！！</li> 
 <li>table 
  <ul> 
   <li>fix 修复多个实例重复安装警告问题</li> 
   <li>（v4保留兼容）参数 row-key 改为 row-config.useKey</li> 
   <li>（v4保留兼容）参数 row-id 改为 row-config.keyField</li> 
   <li>（v4保留兼容）参数 column-key 改为 column-config.useKey</li> 
   <li>（v4保留兼容，建议调用方法获取）事件 checkbox-change 删除参数 &#123;records, reserves, indeterminates&#125;</li> 
   <li>（v4保留兼容，建议调用方法获取）事件 checkbox-all 删除参数 &#123;records, reserves, indeterminates&#125;</li> 
   <li>（v4保留兼容，建议调用方法获取）事件 checkbox-range-start 删除参数 &#123;records, reserves&#125;</li> 
   <li>（v4保留兼容，建议调用方法获取）事件 checkbox-range-change 删除参数 &#123;records, reserves&#125;</li> 
   <li>（v4保留兼容，建议调用方法获取）事件 checkbox-range-end 删除参数 &#123;records, reserves&#125;</li> 
   <li>增加事件 keydown-start</li> 
   <li>增加事件 keydown-end</li> 
   <li>优化虚拟滚动</li> 
   <li>优化工具提示</li> 
  </ul> </li> 
 <li>column 
  <ul> 
   <li>（v4保留兼容）参数 title-help 更改为 title-prefix</li> 
   <li>增加参数 title-prefix.useHTML</li> 
  </ul> </li> 
 <li>select 
  <ul> 
   <li>（v4保留兼容）参数 option-key 改为 option-config.useKey</li> 
   <li>（v4保留兼容）参数 option-id 改为 option-config.keyField</li> 
  </ul> </li> 
 <li>form-item 
  <ul> 
   <li>增加参数 title-prefix.useHTML</li> 
   <li>增加参数 title-suffix.useHTML</li> 
  </ul> </li> 
</ul> 
<p>详情查看：<a href="https://gitee.com/xuliangzhan_admin/vxe-table/releases/4.2.0">https://gitee.com/xuliangzhan_admin/vxe-table/releases/4.2.0</a></p>
