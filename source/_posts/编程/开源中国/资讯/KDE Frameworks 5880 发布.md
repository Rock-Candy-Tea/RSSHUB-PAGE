                                                                                            <p><span style="background-color:#ffffff; color:#333333">KDE Frameworks 5.88.0 已正式发布。KDE Frameworks 是 Qt 的 83 个附加库，它在成熟的、经过同行评审和测试的库中提供了各种常见的功能，并且具有友好的许可条款。</span></p> 
<p><strong><span style="background-color:#ffffff; color:#333333">主要更新内容</span></strong></p> 
<ul> 
 <li>Attica 
  <ul> 
   <li>添加安卓 CI</li> 
  </ul> </li> 
 <li>BluezQt 
  <ul> 
   <li>添加 GATT 客户端功能</li> 
   <li>生成 pkgconfig 文件</li> 
  </ul> </li> 
 <li>Breeze Icons 
  <ul> 
   <li>使用对 QString 的引用来避免复制，修复警告</li> 
   <li>添加 RTL 注销图标</li> 
   <li>某些图标缺少配色方案使用</li> 
   <li>修复用于安装彩色图标的暗安装目录</li> 
   <li>使一些 22 px 的地方图标实际上再次变为 22</li> 
   <li>添加 Kongress 图标修复网格对齐</li> 
   <li>修复 16px im-user-offline 图标的颜</li> 
   <li>将所有 /preferences 移至图标，将其从 icons_dark 中删除</li> 
  </ul> </li> 
 <li>Extra CMake Modules 
  <ul> 
   <li>交叉编译时修复 Wayland_DATADIR</li> 
   <li>将 NDK 解决方法扩展到版本 22</li> 
   <li>msvc：修复 __cplusplus 值</li> 
   <li>ecm_add_test: 添加 -DQT_FORCE_ASSERTS 默认编译标志</li> 
  </ul> </li> 
 <li>KDE Doxygen Tools 
  <ul> 
   <li>处理大写的目录名称</li> 
   <li>删除泛型类型定义中的空格</li> 
  </ul> </li> 
 <li>KCalendarCore 
  <ul> 
   <li>Event 和 FreeBusy 中的脏字段和更新观察者</li> 
   <li>事件中的脏字段和更新观察者</li> 
   <li>IncidenceBase 中的脏更新</li> 
   <li>将 IncidencePrivate 和 IncidenceBasePrivate 移至标题</li> 
   <li>简化 IncidenceBasePrivate 构造函数</li> 
   <li>将关联类从 Foo::Private 重命名为 FooPrivate</li> 
   <li>cmake：使用 libical 的 cmake 脚本的端口</li> 
  </ul> </li> 
</ul> 
<p>详情请查看<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fkde.org%2Fannouncements%2Fframeworks%2F5%2F5.88.0%2F" target="_blank">更新公告</a>。</p>
