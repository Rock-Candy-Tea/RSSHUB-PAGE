                                                        <p>Apache Fineract 1.5.0 现已发布了。Apache Fineract 是一个开源的系统，用于核心银行系统平台化建设。为创业者、金融机构和服务提供商提供了一个可靠、健壮的、可负担得起的金融服务解决方案。</p> 
<p><strong>主要更新内容</strong></p> 
<ul> 
 <li>从 ThitsaWorks CB 获取信用报告</li> 
 <li>自述文件和其他文件不受 Spotless 覆盖</li> 
 <li>Swagger fineract-client 将作为单独的模块</li> 
 <li>Travis CI 实际上没有建立新的多模块项目结构</li> 
 <li>从 Joda API 迁移到 java.time</li> 
 <li>电子邮件服务配置默认为端口 25，而不考虑指定的端口号</li> 
 <li>Swagger CodeGen WARN i.s.c.v.g.java.AbstractJavaCodegen - New（保留字）不能作为模型名称。重命名为 ModelNew</li> 
 <li>Swagger UI 应该针对其所在的服务器运行 "试运行 "查询</li> 
 <li>修复信用局贷款产品映射模块损坏的问题</li> 
 <li>修复 ProcessorHelper 错误处理可能导致 NullPointerException 的问题</li> 
 <li>SDK Swagger Java Client 生成的实用程序已损坏，应从 JAR 中删除</li> 
 <li>使用新的 Swagger Client API（“ Fineract SDK”）进行集成测试</li> 
 <li>将信用报告上传到 ThitsaWorks CB</li> 
 <li>将 Google ErrorProne 代码质量扫描仪工具从 2.4.0 升级到 2.5.1</li> 
</ul>
