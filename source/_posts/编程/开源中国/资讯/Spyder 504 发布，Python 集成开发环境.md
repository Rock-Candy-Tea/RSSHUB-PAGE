                                                        <p>Spyder 5.0.4 现已发布。Spyder 是一个强大的交互式 Python 语言开发环境，提供高级的代码编辑、变量监视、交互测试、调试等特性，支持包括 Windows、Linux 和 OS X 系统。</p> 
<p><strong>主要更新内容</strong></p> 
<ul> 
 <li>按程序增加新的布局</li> 
 <li>修复 IPython 7.24.0 的调试器</li> 
 <li>修复加载复杂的第三方插件</li> 
 <li>修复重新启动内核时的错误</li> 
 <li>“import sys” 似乎在控制台启动时不起作用</li> 
 <li>内核死亡或重新启动或从 IPython 控制台创建日志条目后记录错误</li> 
 <li>调试器被 IPython 7.24.0 损坏</li> 
 <li>Spyder 因启动时未捕获的 FileNotFoundError 而崩溃</li> 
 <li>按退格键时崩溃</li> 
 <li>PYTHONPATH 管理器不能处理路径中的 unicode 字符</li> 
 <li>标题栏在 macOS 上不使用深色模式</li> 
 <li>关闭控制台时出现 AttributeError</li> 
 <li>重新启动内核时显示 “Warning, no such comm”</li> 
 <li>仍会为 Windows 安装程序打印 UMR 消息</li> 
 <li>右键点击编辑器选项卡会选择错误的选项卡</li> 
 <li>Spyder 5 安装问题：没有找到 QcoreApplication</li> 
</ul>
