  <p>渠成企业软件百宝箱 1.5 版本发布, 命令行工具q功能增强, 用户可以通过命令行工具快速安装应用市场应用。</p> 
  <p>我们的官网提供<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.qucheng.com%2Fmarket-browse.html" target="_blank">渠成软件百宝箱的试用</a>，您可以在应用市场选取喜欢的应用体验安装与试用，同时，我们还提供每一款应用的演示服务，让您快速体验不同的应用。</p> 
  <h1>一、新增功能</h1> 
  <h2>1.1 命令行工具q功能增强</h2> 
  <ul> 
   <li> <p>初始化集群默认安装禅道开源版本</p> </li> 
   <li> <p><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.qucheng.com%2Fbook%2FInstallation-manual%2F25.html" target="_blank">app子命令新增支持安装应用</a></p> </li> 
  </ul> 
  <p><img alt="image" src="https://www.qucheng.com/file.php?f=202209/f_38c68f1a9ba227b8b5e8225f68859f61&t=png&o=&s=&v=1662444046" referrerpolicy="no-referrer"></p> 
  <h2>1.2 PostgreSQL备份还原</h2> 
  <ul> 
   <li> <p>新增对PostgreSQL的备份还原支持</p> </li> 
  </ul> 
  <h1>二、1.5版本详细更新列表</h1> 
  <h2>2.1 新特性</h2> 
  <h3>命令行安装应用</h3> 
  <ul> 
   <li> <p>增加<code>q</code>命令行支持初始化平台时默认安装禅道开源版</p> </li> 
   <li> <p><code>q app</code>子命令增强, 新增子命令<code>install</code>和<code>market</code>, 支持安装指定应用或者从应用市场安装</p> </li> 
  </ul> 
  <h3>PostgreSQL备份还原</h3> 
  <ul> 
   <li> <p>新增对PostgreSQL的备份还原支持，用户可对使用了PostgreSQL的应用做完整备份，如SonarQube。</p> </li> 
  </ul> 
  <h2>2.2 新增应用</h2> 
  <ul> 
   <li> <p>上架<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.qucheng.com%2Fmarket-view-108.html" target="_blank">OpenLDAP</a></p> </li> 
   <li> <p>更新<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.qucheng.com%2Fmarket-view-99.html" target="_blank">ZincSearch</a>到0.3.2</p> </li> 
   <li> <p>更新<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.qucheng.com%2Fmarket-view-89.html" target="_blank">Gitea</a>到1.17.2</p> </li> 
   <li> <p>更新<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.qucheng.com%2Fmarket-view-102.html" target="_blank">Drawio</a>到20.2.8</p> </li> 
   <li> <p>更新<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.qucheng.com%2Fmarket-view-105.html" target="_blank">Umami</a>到1.37.0</p> </li> 
   <li> <p>更新<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.qucheng.com%2Fmarket-view-93.html" target="_blank">YApi</a>到1.9.5</p> </li> 
  </ul> 
  <h2>2.3 变更</h2> 
  <ul> 
   <li> <p>优化命令行日志展示</p> </li> 
   <li> <p>优化了服务管理页面的性能</p> </li> 
   <li> <p>调整平台管理员账号初始化流程</p> </li> 
  </ul> 
  <h2>2.4 Bug修复</h2> 
  <ul> 
   <li> <p>修复了渠成平台和渠成官网中，应用市场按上架时间排序不正确的问题</p> </li> 
  </ul>
