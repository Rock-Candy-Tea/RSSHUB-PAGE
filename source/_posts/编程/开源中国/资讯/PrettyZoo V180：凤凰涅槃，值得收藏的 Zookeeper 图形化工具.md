                                                        <p>Hi，又见面了，PrettyZoo 这次版本的 Release 周期比以往要短，但是改动却一点也不少。</p> 
<p>新版本直接将整个底层的 UI 库迁移到了 <a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fwww.jfoenix.com%2F" target="_blank">Jfoenix</a>，将按钮、对话框、消息提醒全部改为了 material design 风格的控件。当然，新功能也不会落下，这次支持了断线后自动重连，再也不用担心 session 过期后导致的断线。</p> 
<p>最后，矫情一下，算是抒发一下自己这一年半来坚持开源的心情吧</p> 
<blockquote> 
 <p>不经意回首，走过的路不长也不短</p> 
 <p>说短吧，也不过一年半载</p> 
 <p>说长吧，又不止一年半载</p> 
 <p>......</p> 
</blockquote> 
<p style="text-align:start"><span style="color:#333333">简介：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fvran-dev%2FPrettyZoo">PrettyZoo</a></span><span style="color:#333333"> 是一款基于 JavaFX 实现的 Zookeeper 桌面图形化工具，不仅颜值极高，而且功能齐全，支持 Windows / Mac / Linux 等主流平台。</span></p> 
<h3 style="text-align:start">v1.8.0 发布记录</h3> 
<ol> 
 <li>Feature：迁移底层 UI 库到 <a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fwww.jfoenix.com%2F" target="_blank">Jfoenix</a>，全新 Material Design 控件</li> 
 <li>Feature：移除不必要的动画效果，为后续的统一设计作准备</li> 
 <li>Feature：统一文案语言为英文，为后续的国际化作准备</li> 
 <li>Feature：支持断线自动重连</li> 
 <li>Bug fix：更新节点数据后 meta data 没有同步更新</li> 
 <li>Bug fix：在创建新的 server 时列表仍然时选中状态</li> 
</ol>
