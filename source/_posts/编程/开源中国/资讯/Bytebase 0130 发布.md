                                                                                            <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">Bytebase 是一个基于网络、零配置、无依赖的数据库 Schema 变更和版本控制管理工具，适用于开发人员和 DBA。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">Bytebase 0.13.0 发布，更新内容如下：</p> 
<h3 style="margin-left:.6em; margin-right:0; text-align:start">新功能</h3> 
<ul style="margin-left:0; margin-right:0"> 
 <li>SQL 编辑器支持工作表共享</li> 
 <li>引入一个新的“工作表（Sheet）”的概念，代替了旧的“归档查询”</li> 
</ul> 
<h3 style="margin-left:.6em; margin-right:0; text-align:start">增强</h3> 
<ul style="margin-left:0; margin-right:0"> 
 <li>可搜索的成员选择器</li> 
 <li>数据库标签相关功能现在可用于标准模式项目</li> 
 <li>当检测到 Schema Drift 时显示 Schema 模式对比</li> 
 <li>为 MySQL 和 Clickhouse 增加快速启动功能</li> 
</ul> 
<h3 style="margin-left:.6em; margin-right:0; text-align:start">错误修复</h3> 
<ul style="margin-left:0; margin-right:0"> 
 <li>防止 SQL 编辑器的 XSS 攻击</li> 
 <li>过滤获取用户通知时的不正当授权</li> 
</ul> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">更多详情可查看：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fbytebase%2Fbytebase%2Freleases%2Ftag%2F0.13.0" target="_blank">https://github.com/bytebase/bytebase/releases/tag/0.13.0</a></p>
