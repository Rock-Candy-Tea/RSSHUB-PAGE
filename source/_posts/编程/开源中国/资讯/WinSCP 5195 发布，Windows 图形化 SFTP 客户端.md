                                                                                            <p>WinSCP 是一个 Windows 环境下使用的 SSH 的开源图形化 SFTP 客户端，同时支持 SCP 协议，它的主要功能是在本地与远程计算机间安全地复制文件，并且可以直接编辑文件。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">目前，WinSCP 5.19.5 已经发布。该版本具体更新内容如下：</p> 
<ul> 
 <li>复制文件时与 Google Cloud S3 API 的兼容性。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwinscp.net%2Ftracker%2F2038" target="_blank">2038</a></li> 
 <li>删除隐式现有目录时与 Google Cloud S3 API 的兼容性。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwinscp.net%2Ftracker%2F2042" target="_blank">2042</a></li> 
 <li>翻译更新：土耳其语。</li> 
 <li>当 OpenSSH 8.8（或更高版本）服务器拒绝密钥时，记录对 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwinscp.net%2Ftracker%2F1952" target="_blank">1952</a> 的引用。</li> 
 <li>错误修复：在下载从剪贴板粘贴的文件时，将新内容复制到剪贴板时崩溃。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwinscp.net%2Ftracker%2F2036" target="_blank">2036</a></li> 
 <li>错误修复：同步核对清单上的 Browse Remote Directory 命令没有找到带空格的文件。</li> 
 <li>错误修复：Extension ZIP 和 Upload 不适用于驱动器根目录中的文件。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwinscp.net%2Ftracker%2F2039" target="_blank">2039</a></li> 
 <li>错误修复：控制台窗口上的路径在不适合时没有缩短。</li> 
</ul> 
<p>更新说明：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwinscp.net%2Feng%2Fdocs%2Fhistory%235.19.5" target="_blank">https://winscp.net/eng/docs/history#5.19.5</a> </p>
