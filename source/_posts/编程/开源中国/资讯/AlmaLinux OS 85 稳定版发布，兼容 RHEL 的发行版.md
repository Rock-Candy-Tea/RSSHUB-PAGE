<div class="content">
                                                                                            <p>AlmaLinux OS 8.5 稳定版<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Falmalinux.org%2Fblog%2Falmalinux-os-85-stable-now-available%2F" target="_blank">已发布</a>。</p> 
<blockquote> 
 <p>AlmaLinux 是开放源码的、社区驱动的项目，它从红帽企业Linux (RHEL) 的源码编译而来。AlmaLinux 跟 RHEL 8 完全在二进制上兼容，它由 CloudLinux OS 的创建者打造。</p> 
</blockquote> 
<p>下载地址：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fmirrors.almalinux.org%2Fisos.html" target="_blank">https://mirrors.almalinux.org/isos.html</a></p> 
<p>发布公告称，AlmaLinux OS 8.5 是适用于 x86_64 和 ARM 架构的稳定版本（代号 <span style="background-color:#ffffff; color:#2c3e50">Arctic Sphynx</span>），目前已准备好用于生产环境的安装，并支持用户的所有计算需求和工作负载。</p> 
<p>AlmaLinux OS 8.5 的树莓派镜像版本已升级到最新版，Live 镜像、云和容器镜像版本的更新正在进行中。</p> 
<p>AlmaLinux OS 8.5 包括对容器工具的功能和改进，以减少不便之处并使构建和部署过程更加灵活、支持 OpenJDK 17、用于强化和安全合规性的额外 OpenSCAP 配置文件、新系统角色、用于 NTP 的网络时间安全性 (NTS) ，以及其他添加和增强功能。</p> 
<p><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwiki.almalinux.org%2Frelease-notes%2F8.5.html%23changelog" target="_blank">更新日志</a></p> 
<ul> 
 <li> <p>增加新的 module streams</p> 
  <ul> 
   <li>nginx 1.20</li> 
   <li>Node.js 16</li> 
   <li>Ruby 3.0</li> 
  </ul> </li> 
 <li> <p>升级组件</p> 
  <ul>

   <li>Squid 4.15</li> 
   <li>Mutt 2.0.7</li> 
  </ul> </li> 
 <li> <p>升级编译器</p> 
  <ul> 
   <li>GCC Toolset 11</li> 
   <li>LLVM Toolset 12.0.1</li> 
   <li>Rust Toolset 1.54.0</li> 
   <li>Go Toolset 1.16.7</li> 
  </ul> </li> 
 <li> <p>在网络安装模式下将自动选择最近的镜像</p> </li> 
 <li> <p>添加新的仓库</p> 
  <ul> 
   <li>ResilientStorage</li> 
   <li>Plus</li> 
  </ul> </li> 
</ul>
