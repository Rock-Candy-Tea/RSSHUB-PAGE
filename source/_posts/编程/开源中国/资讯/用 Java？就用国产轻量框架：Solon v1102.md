                                                                                            <h4>相对于 Spring Boot 和 Spring Cloud 的项目：</h4> 
<ul> 
 <li>启动快 5 ～ 10 倍。 <strong>（更快）</strong></li> 
 <li>qps 高 2～ 3 倍。 <strong>（更高）</strong></li> 
 <li>运行时内存节省 1/3 ~ 1/2。 <strong>（更少）</strong></li> 
 <li>打包可以缩小到 1/2 ~ 1/10；比如，90Mb 的变成了 9Mb。 <strong>（更小）</strong></li> 
 <li>基于服务 name 进行注册发现 与 k8s svc 相互对应</li> 
 <li>支持 Service Mesh 架构部署方案</li> 
 <li>支持 jdk8, jdk11, jdk17</li> 
</ul> 
<h4>本次更新：</h4> 
<ul> 
 <li>完善 solon.serialization.fastjson2 插件</li> 
 <li>增加 solon.config 配置项（与 solon.extend.config 同）</li> 
 <li>增加 nami.Invocation::target 用于获取代理</li> 
 <li>增加 solon.schedule 插件动态修改调度时间的支持</li> 
 <li>优化 solon-test 插件的注解处理</li> 
 <li>优化 solon.data 插件 tag 的缓存时间处理</li> 
 <li>插件 solon.config.yaml 升级 snakeyaml 为：1.30</li> 
 <li>插件 solon.serialization.snack3 升级 snack3 为：3.2.35</li> 
 <li>插件 solon.serialization.fastjson 升级 fastjson 为：1.2.83</li> 
 <li>插件 solon.serialization.fastjson2 升级 fastjson2 为：2.0.12</li> 
 <li>插件 solon.boot.undertow 升级 undertow 为：2.2.19.Final</li> 
 <li>插件 nami.channel.socketd.rsocket 升级 rsocket 为：1.1.2</li> 
 <li>插件 nami.channel.socketd.netty 升级 netty 为：4.1.68.Final</li> 
 <li>插件 nami.channel.http.hutool 升级 hutool 为：5.8.5</li> 
 <li>插件 mybatis-solon-plugin 升级 mybatis 为：3.5.10</li>
