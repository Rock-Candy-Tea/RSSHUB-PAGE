                                                                                            <p>微软 Edge 98 正式发布，新版本带来了几个功能更新，如网页浏览时增强的安全模式、便利的多用户个性化选项，以及新的覆盖式滚动条。完整的功能更新列表如下：</p> 
<ul> 
 <li>增强网络安全性：微软 Edge 浏览器新增了浏览器安全优先的浏览模式，在浏览网页时给用户提供额外的保护。管理员可以将组策略应用于终端用户的桌面（Windows、macOS 和 Linux），以帮助防止零日漏洞。以下组策略支持这种浏览模式： 
  <ul> 
   <li>EnhanceSecurityMode</li> 
   <li>EnhanceSecurityModeBypassListDomains</li> 
   <li>EnhanceSecurityModeEnforceListDomains</li> 
  </ul> </li> 
 <li>对于设置了多个 Edge 配置文件的用户，现在可以创建一个站点列表，这些站点会自动将用户切换到对应的配置文件。例如，始终使用学校的特定 Edge 配置文件访问学校的网站。如果要避免使用其他配置文件访问该站点，可以对其进行设置，以便在打开网站时自动切换到正确的配置文件。</li> 
 <li>弃用 WebRTC 的 Plan B SDP 语义：这一变化弃用了名为 Plan B 的 SDP dialect。该 SDP 格式将被 Unified Plan 所取代，后者是一种符合规范和跨浏览器兼容的 SDP 格式。</li> 
 <li>Microsoft Edge 中添加了覆盖式滚动条：Edge 已经更新了一个采用了基于覆盖式设计的滚动条（浮动的小部件）。用户可以在 edge://flags 中打开这个功能。</li> 
</ul> 
<p>除了功能更新，Edge 98 还带来了一些重要的安全更新：</p> 
<ul> 
 <li>CVE-2022-23261：Edge 篡改漏洞</li> 
 <li>CVE-2022-23262：Edge 特权提升漏洞</li> 
 <li>CVE-2022-23263：Edge 特权提升漏洞</li> 
</ul>
