                                                                                            <p>NetworkManager 1.40 近日正式发布，这个软件主要用于配置 Linux 下的有线和无线网络接口。</p> 
<p>NetworkManager 1.40 更新内容包括：</p> 
<ul> 
 <li>现在支持 Multi-Path TCP（MPTCP） 
  <ul> 
   <li>NetworkManager 现在支持将 IP 地址配置为 MPTCP 端点，如果 Linux 内核支持，MPTCP 处理将自动启用。</li> 
  </ul> </li> 
 <li>放弃了长期不用的内部 systemd DHCPv4 客户端，它早就被 nettools n-dhcp4 实现所取代</li> 
 <li>改进了无效 DHCP 选项的日志信息</li> 
 <li>支持配置 IPv6 MTU</li> 
 <li>读取 /proc/cmdline 选项</li> 
 <li>改进了运营商检测</li> 
 <li>其他各种改进</li> 
</ul>
