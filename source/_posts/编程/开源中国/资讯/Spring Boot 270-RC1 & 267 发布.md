                                                                                            <p>Spring Boot 发布了两个版本更新：2.7.0-RC1 和 2.6.7。</p> 
<p><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fspring.io%2Fblog%2F2022%2F04%2F21%2Fspring-boot-2-7-0-rc1-is-now-available" target="_blank"><strong>2.7.0-RC1</strong></a></p> 
<p>值得关注的变化：</p> 
<ul> 
 <li>新增用于 Kafka Retry Topics 的自动配置</li> 
 <li>新增用于在 GraphQL 中提供 RSocket 支持的自动配置</li> 
 <li><span style="background-color:#ffffff; color:#24292f">在 GraphQlTesterAutoConfiguration 中使用自动配置的 JSON 编解码器</span></li> 
 <li><span style="background-color:#ffffff; color:#24292f">支持 Elasticsearch RestClientBuilder 自动配置，无需 RestHighLevelClient</span></li> 
 <li><span style="background-color:#ffffff; color:#24292f">支持 CNB platform API 0.9</span></li> 
</ul> 
<p><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fspring-boot%2Fwiki%2FSpring-Boot-2.7.0-RC1-Release-Notes" target="_blank">详情查看 release note</a>。</p> 
<p><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fspring.io%2Fblog%2F2022%2F04%2F21%2Fspring-boot-2-6-7-available-now" target="_blank"><strong>2.6.7</strong></a></p> 
<p>此版本主要是修复 bug：</p> 
<ul> 
 <li><span style="background-color:#ffffff; color:#24292f">当附加属性具有空值时，bootBuildInfo 失败并出现 NullPointerException</span></li> 
 <li><span style="background-color:#ffffff; color:#24292f">ApplicationAvailabilityBean 没有线程安全</span></li> 
 <li><span style="background-color:#ffffff; color:#24292f">spring-boot-properties-migrator 的 Neo4j 用户名属性替换提示不正确</span></li> 
</ul> 
<p><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fspring-boot%2Freleases%2Ftag%2Fv2.6.7" target="_blank">详情查看 release note</a>。</p>
