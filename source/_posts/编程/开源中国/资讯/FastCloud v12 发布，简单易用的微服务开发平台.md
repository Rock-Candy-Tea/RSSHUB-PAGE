                                                                                            <h2 style="margin-left:0; margin-right:0; text-align:left">项目说明</h2> 
<ul style="list-style-type:disc; margin-left:0; margin-right:0"> 
 <li>FastCloud 是采用 Spring Cloud Alibaba、SpringSecurity Oauth2.0、Spring Cloud Gateway、SpringBoot、Nacos、Redis、Mybatis-Plus 等框架，开发的一套 SpringCloud 快速开发平台，使用门槛极低，且采用 MIT 开源协议，完全免费开源，可免费用于<strong>商业项目</strong>等场景。</li> 
 <li>开发文档：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fmaku.net%2Fdocs%2Ffast-cloud" target="_blank">https://maku.net/docs/fast-cloud</a></li> 
 <li>演示环境：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdemo.maku.net%2Ffast-cloud" target="_blank">https://demo.maku.net/fast-cloud</a></li> 
</ul> 
<h2 style="margin-left:0; margin-right:0; text-align:left">更新日志</h2> 
<ul> 
 <li>新增达梦数据库支持，更好的服务国内企业</li> 
 <li>新增定时任务，可轻松解决定时任务需求</li> 
 <li>新增 api 模块，用于处理远程接口调用</li> 
 <li>新增新模块开发DEMO，新业务开发更简单</li> 
 <li>优化项目结构，代码结构更合理</li> 
 <li>优化版本管理，版本管理更合理</li> 
 <li>优化按需加载配置，本地启动前端更快</li> 
 <li>升级 springboot 到 2.6.10</li> 
 <li>升级 vite 到 3.0.4</li> 
</ul>
