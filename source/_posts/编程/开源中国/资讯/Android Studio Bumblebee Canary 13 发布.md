<div class="content">
                                                                    
                                                        <p><span style="background-color:#ffffff; color:#333333">Android Studio Bumblebee | 2021.1.1 Canary 13 现已发布，此次更新内容主要为 bug 修复 。</span></p> 
<p><strong><span style="background-color:#ffffff; color:#333333">主要更新内容</span></strong></p> 
<ul> 
 <li><span style="background-color:#ffffff; color:#333333">Android Gradle 插件</span> 
  <ul> 
   <li>修复<span style="background-color:rgba(255, 255, 255, 0.95); color:#202124">使用经过测试的代码中的内嵌函数时，ASM API 会使测试失败的问题</span></li> 
  </ul> </li> 
 <li><span style="background-color:rgba(255, 255, 255, 0.95); color:#202124">应用更改</span> 
  <ul> 
   <li><span style="background-color:rgba(255, 255, 255, 0.95); color:#202124">修复在 API 级别 30 模拟器上进行增量构建后，应用会崩溃并出现 KotlinReflectionNotSupportedError 的问题</span></li> 
  </ul> </li> 
 <li><span style="background-color:rgba(255, 255, 255, 0.95); color:#202124">Databinding</span> 
  <ul> 
   <li><span style="background-color:rgba(255, 255, 255, 0.95); color:#202124">修复 ​​​​​​​Databinding 无法与平台工具 31.0.3 配合使用以及无法再使用 platform-tools/api/api-versions.xml 的问题</span></li> 
  </ul> </li> 
 <li><span style="background-color:rgba(255, 255, 255, 0.95); color:#202124">lint</span> 
  <ul> 
   <li><span style="background-color:rgba(255, 255, 255, 0.95); color:#202124">​​​​​​​修复更新到 AGP 7.1.0-alpha08 后，lint 的 SourceTransformationTestMode.performEdits 中出现 StringIndexOutOfBoundsException 的问题</span></li> 
   <li>修复<span style="background-color:rgba(255, 255, 255, 0.95); color:#202124">在 AGP 7.0 中，lint 会失败并显示 "Can't initialize detector" 的问题</span></li> 
   <li><span style="background-color:rgba(255, 255, 255, 0.95); color:#202124">修复 "针对 Lint 检查的 lint 检查" 未运行的问题</span></li> 
   <li>修复<span style="background-color:rgba(255, 255, 255, 0.95); color:#202124">通过 Gradle 运行时，Studio 中的测试失败不会显示测试输出的问题</span></li> 
  </ul> </li> 
</ul>
