                                                        <p>SourceGit v3.9 已经发布，开源的 Git 客户端</p>
<p>此版本更新内容包括：</p>
<p>[CHANGELOG]</p> 
<ol> 
 <li>支持使用配置的第三方工具查看Diff</li> 
 <li>增加WinMerge工具支持</li> 
 <li>优化内存使用</li> 
 <li>增加快捷键 4.1. <code>CTRL + TAB</code> 标签页轮换 4.2. <code>CTRL + T</code> 打开新标签页 4.3. <code>CTRL + W</code> 关闭当前标签页 4.4. <code>CTRL + [1-9]</code> 切换到指定标签页 4.5. <code>CTRL + F</code> 如果当前页为仓库页，打开搜索 4.6. <code>F5</code> 如果当前页为仓库页，刷新</li> 
 <li>增加<code>git subtree</code>第一版实现</li> 
</ol>
<p>详情查看：<a href="https://gitee.com/sourcegit/SourceGit/releases/v3.9" blank="_target">https://gitee.com/sourcegit/SourceGit/releases/v3.9</a></p>
