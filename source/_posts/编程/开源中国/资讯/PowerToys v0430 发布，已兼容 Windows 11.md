                                                        <p>Microsoft PowerToys 是 Windows 系统实用程序，供高级用户调整和简化其 Windows 体验，可最大限度地提高生产力。</p> 
<p>PowerToys v0.43.0 发布，更新内容如下：</p> 
<h3>更改日志：</h3> 
<ul> 
 <li>更改了系统托盘图标的提示文本，使其在单行上以兼容 Windows 11；</li> 
 <li>修正了 Color Picker 的转义行为，以便只有弹出窗口在活动时才会关闭；</li> 
 <li>改变了 URI PowerToys Run 插件，默认启动 HTTPS 而不是 HTTP；</li> 
 <li>增加了从 PowerToys Run 执行系统命令时的确认对话框；</li> 
</ul> 
<h3>常规：</h3> 
<ul> 
 <li>Image Resizer 设置中的尺寸列表视图的新 UI；</li> 
 <li>修正了安装/更新过程中的 FileInUse 错误；</li> 
 <li>修正了 PowerToys 运行设置中的切换开关，使其正确显示；</li> 
 <li>修正了主题颜色改变时，标题文本不更新的问题；</li> 
 <li>为 GIF 图片添加了 Name/Alt 文本；</li> 
 <li>扩展了 Report Bug 工具，以收集更丰富的诊断信息；</li> 
 <li>修正了屏幕阅读器功能，以停止在设置中读取隐藏的文本；</li> 
 <li>为 Welcome to PowerToys 窗口的 GIF 图像添加了 Name 和 alt 文本属性</li> 
</ul> 
<h3>Awake：</h3> 
<ul> 
 <li>修正了右键单击 Awake 应用图标时菜单的错误；</li> 
 <li>修正了定时保持唤醒时 CPU 的高使用率；</li> 
 <li>修正了 Awake 图标对通知栏的干扰；</li> 
 <li>增加了遥测功能来收集 Awake 的设置和日志；</li> 
</ul> 
<h3>Color Picker：</h3> 
<ul> 
 <li>删除了 element Name 属性中的私有 Unicode 字符;</li> 
 <li>Name 属性中没有类名;</li> 
 <li>为屏幕阅读器修正了颜色阴影的编号；</li> 
 <li>ControlType 和 LocalizedControlType 不再同时设置为 Custom；</li> 
</ul>
