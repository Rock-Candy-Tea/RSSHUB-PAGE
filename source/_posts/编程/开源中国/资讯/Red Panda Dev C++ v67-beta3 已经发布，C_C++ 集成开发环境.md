                                                        <p>Red Panda Dev C++ v6.7-beta3 已经发布，C/C++ 集成开发环境</p> 
<p>此版本更新内容包括：</p> 
<p>VERSION 6.7-BETA3 APR 2021</p> 
<ul> 
 <li>变化：编译选项对话框中的“静态链接所有库”选项改为缺省打开</li> 
 <li>修正：当devcpp启动后，立即F10运行已编译好的.exe文件，会提示找不到dll文件错误</li> 
 <li>修正：当gdb返回的监视变量内容格式不正确时，devcpp会出错</li> 
 <li>功能增强：在文件浏览器中点击无内容区域会清除选中项</li> 
 <li>功能增强：文件浏览器增加右键菜单</li> 
 <li>修正：如果gcc所在的文件夹不在Dev-Cpp文件夹下，且其路径中包含空格或非ASCII字符，则在Dev-C++启动时会提示gcc路径错误</li> 
 <li>改变：删除编辑器选项对话框中的“搜索文本”选项</li> 
 <li>改变：将编辑器对话框中的“缺省文件类型”选项，移动到杂项页中</li> 
 <li>功能增强：在编辑器选项对话框中增加“显示代码折叠线”选项（缺省为开)</li> 
 <li>功能增强：在编辑器选项对话框中增加“显示缩进提示线”和“缩进提示线颜色”选项</li> 
 <li>当前编译器器选项中的gcc或g++路径不正确时，自动停止后台的语法错误检查，而不再报错</li> 
</ul> 
<p>详情查看：<a href="https://gitee.com/royqh1979/Dev-CPP/releases/v6.7-beta3">https://gitee.com/royqh1979/Dev-CPP/releases/v6.7-beta3</a></p>
