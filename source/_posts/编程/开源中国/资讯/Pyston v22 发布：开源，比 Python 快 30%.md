                                                        <p>Pyston v2.2 已正式发布，Pyston 是 Python 的高度兼容实现，自称比 Python 更快。官方表示，v2.2 版本比以前的版本要快得多，重要的它现在是开源的。同时，开发团队还合并了 CPython 的许多更改，其现在基于 CPython 3.8.8。</p> 
<p>Pyston 方面指出，在其 Web 服务器基准测试中，Pyston v2.2 比现有的 Python 快 30％，这相较以前的性能有了很大的提高。支持 Pyston v2.2 的基础技术与早期版本相同，但开发团队调整和优化了更多的领域，并进行了许多的速度提升，尤其是在 JIT 和属性缓存机制方面。</p> 
<p>一个值得注意的变化是，团队决定删除许多 Python 支持的很少使用的调试功能，此举导致速度实现了 2% 的提升。</p> 
<blockquote> 
 <p>这对我们来说是非常了不起的：在世界所有运行 Python 的计算机中，有 2% 在执行调试检查。我们已经禁用了这些检查，并将自己定位为"optimized build"，类似于没有调试信息的二进制文件。那些仍然想要调试功能的人可以使用现有 Python 的"debug build"，因为它们是可以互换的。关于我们在 Pyston v2.2 中删除的功能的完整列表，请看我们的 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fpyston%2Fpyston%2Fwiki" target="_blank">wiki</a>。</p> 
</blockquote> 
<p>此外，在将 Pyston v2.2 开源后；官方也已将旧存储库存档，以减少混乱。</p> 
<p>详情可查看官方公告：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fblog.pyston.org%2F2021%2F05%2F05%2Fpyston-v2-2-faster-and-open-source%2F" target="_blank">https://blog.pyston.org/2021/05/05/pyston-v2-2-faster-and-open-source/</a></p>
