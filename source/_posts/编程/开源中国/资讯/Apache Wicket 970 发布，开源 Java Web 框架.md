                                                        <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">Apache Wicket 是一个开源的面向 Java 组件的 Web 应用框架，为政府、商店、大学、城市、银行、电子邮件提供商等成千上万的 Web 应用和网站提供支持。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">Wicket 的开发中使用了语义版本，因此与 9.0.0 相比，9.7.0 版本没有出现 API 中断。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">9.7.0 的更新内容包括：</p> 
<h3 style="margin-left:.6em; margin-right:0; text-align:start">错误修复</h3> 
<ul style="margin-left:0; margin-right:0"> 
 <li>用户指南：'determinate' 不是动词，应改为 'to determine'</li> 
 <li>FilePageStore 在 Windows 下失败</li> 
</ul> 
<h3 style="margin-left:.6em; margin-right:0; text-align:start">改进</h3> 
<ul style="margin-left:0; margin-right:0"> 
 <li>添加一些 Web-socket 改进</li> 
 <li>方便使用 InSessionPageStore 作为持久性存储</li> 
 <li>使一些与数据化相关的组件更易于重复使用</li> 
</ul> 
<h3 style="margin-left:.6em; margin-right:0; text-align:start">任务</h3> 
<ul style="margin-left:0; margin-right:0"> 
 <li>更新快速启动程序所使用的密钥存储。</li> 
</ul>
