   <p><span style="color:#333333">Vitest v0.23.4 现已发布，它是一个基于 Vite 的原生快速单元测试框架。具体更新内容如下</span></p> 
   <h3><strong>特征</strong></h3> 
   <ul> 
    <li>使用 vi.mock 时提供有用的信息  <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fvitest-dev%2Fvitest%2Fpull%2F2047" target="_blank">#2047</a></li> 
    <li><strong>基准</strong>：仅模式 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fvitest-dev%2Fvitest%2Fpull%2F2039" target="_blank">#2039</a></li> 
   </ul> 
   <h3><strong>Bug修复</strong></h3> 
   <ul> 
    <li>将 outputFile 移动到基准测试，允许 --reporter 进行基准测试  <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fvitest-dev%2Fvitest%2Fpull%2F2037" target="_blank">#2037</a></li> 
    <li><strong>cli</strong> : 转发节点 wasm 标志 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fvitest-dev%2Fvitest%2Fpull%2F2041" target="_blank">#2041</a></li> 
    <li><strong>覆盖范围</strong>：<code>istanbul-reports</code>支持<code>projectRoot</code>   <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fvitest-dev%2Fvitest%2Fpull%2F2051" target="_blank">#2051</a></li> 
   </ul>
