                                                        <p>Ghost 是一个开源建站平台，可以使用它创建自己的网站，有点类似于 WordPress 。不过现阶段的 WordPress 无论是构架还是设计都已经有些臃肿，  而 Ghost 使用起来很简单，点几下就可以完成添加 logo、创建导航菜单和注入代码等操作。 Ghost 的编辑环境基于 Markdown ，所有已发布的内容都会存放在一个易于搜索的存档中，可供快速查找。</p> 
<p>目前 Ghost <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fghost.org%2Fchangelog%2F5%2F" target="_blank">发布</a>了 5.0 版本，该版本带来如下改动：</p> 
<ul> 
 <li>Ghost 的最后 2 个主要版本引入了会员、高级订阅和电子邮件通讯。Ghost 5.0 通过支持自定义高级层、多个新闻通讯、特别优惠、详细的受众细分和扩展分析，扩展了这些功能。</li> 
 <li>升级了现有主题，并引入了一些强大的新主题，包括一个特别适合新闻网站的主题，名为 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fheadline.ghost.io%2F" target="_blank">Headline</a> 。</li> 
 <li><span style="color:#333333">在 Ghost 中添加了主题的设计设置，以及电子邮件通讯设置。</span></li> 
 <li><span style="color:#333333">生产堆栈升级： Ubuntu 20、Node 16、MySQL8</span></li> 
</ul>
