                                                                                            <p>PhpStorm 2021.2.1 是 2021.2 版本的第一个小更新。你可以通过 Toolbox、使用 IDE 本身或从<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.jetbrains.com%2Fphpstorm%2Fdownload%2F" target="_blank">网站</a>下载更新的版本来进行安装。</p> 
<h3><strong>此版本中值得注意的更新</strong></h3> 
<ul> 
 <li>允许在内置服务器和 HTML 预览中“在保存时重载”和“更改时重载”之间进行切换</li> 
 <li>修正：在 "jit" 模式下，CSS 类的补全在 Tailwind 2.1 中不能很好地工作 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FWEB-50318" target="_blank">[web-50318]</a></li> 
 <li>Vue 3：支持新的 <script setup> <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FWEB-49000" target="_blank">[WEB-49000]</a></li> 
 <li>对无 break 的 swithc case 使用 PSR-2 注释建议 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FWI-32123" target="_blank">[WI-32123]</a></li> 
 <li>修正：从 WSL2 打开项目时冻结 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-273398" target="_blank">[IDEA-]273398</a></li> 
 <li>修正：在 2021.2 中没有显示 Splash <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-270204" target="_blank">[IDEA-]270204</a></li> 
 <li>支持基于 IteratorAggregate<User> 的推理 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FWI-61438" target="_blank">[WI-61438]</a></li> 
 <li>修正：添加的属性没有 <code>$</code> 符号<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FWI-34398" target="_blank">[WI-34398]</a></li> 
 <li>默认启用 breadcrumbs <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FWI-59237" target="_blank">[WI-59237]</a></li> 
 <li>支持模板的 phpstan-* 注释 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FWI-57959" target="_blank">[WI-57959]</a></li> 
</ul>
