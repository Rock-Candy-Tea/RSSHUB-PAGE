                                                                                            <p><span style="color:#000000">PHP 8.2.0 的第一个候选版本 RC 1 现已</span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.php.net%2Farchive%2F2022.php%232022-09-01-4" target="_blank"><span style="color:#000000">发布</span></a><span style="color:#000000">，带来了各种错误修复，包括可能的崩溃修复。PHP CLI 的内置 Web 服务器也对其静态资源的处理进行了各种改进。</span></p> 
<p><span style="color:#000000">根据</span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.phoronix.com%2Fnews%2FPHP-8.2-RC1-Released" target="_blank"><span style="color:#000000">介绍</span></a><span style="color:#000000">，早期的 PHP 8.2 开发版本包括：</span></p> 
<ul> 
 <li><span style="color:#000000">增加了对在 traits 中定义常量的支持</span></li> 
 <li><span style="color:#000000">Random code 中的 extension-specific Exceptions/Errors</span></li> 
 <li><span style="color:#000000">更新了内置 Web 服务器的 MIME 类型表</span></li> 
 <li><span style="color:#000000">减少了各种函数返回的字符串的内存占用</span></li> 
 <li><span style="color:#000000">初步支持 macOS Instrument 的 JIT 性能分析生成</span></li> 
 <li><span style="color:#000000">新的 Random 扩展</span></li> 
 <li><span style="color:#000000">初步支持 ARM64 的 Windows 上的交叉构建</span></li> 
 <li><span style="color:#000000">将 JIT 缓冲区 Opcache 分配到靠近 PHP .text 段，以允许使用直接的 IP 相关调用和跳转</span></li> 
 <li><span style="color:#000000">PHP Sockets 的新选项</span></li> 
 <li><span style="color:#000000">支持新 Curl 选项</span></li> 
 <li><span style="color:#000000">新的 ZipArchive 方法</span></li> 
 <li><span style="color:#000000">以及其他性能改进...</span></li> 
</ul> 
<p><span style="color:#000000">预计未来至少还将会有五个候选版本，下一个版本即第二个候选版本 (RC 2)，计划于 2022 年 9 月 15 日发布；而 PHP 8.2.0 GA 版本预计在 11 月 24 日左右。</span></p>
