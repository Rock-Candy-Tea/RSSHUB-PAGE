                                                        <p>Bootstrap v5.1.2 正式发布，值得关注的更新内容如下：</p> 
<h3>更新亮点</h3> 
<ul> 
 <li>通过重新安排使用负数的 <code>calc()</code> 函数，临时修补了 postcss-values-parser 的问题。这应该可以恢复在 <code>create-react-app</code> 中导入和编译 Bootstrap Sass 的功能</li> 
 <li>为小的和大的 <code>.form-select</code> 添加了 <code>border-radius</code></li> 
 <li>为按钮添加了 <code>align-self: center</code>，以改善在 Flex 容器中的渲染</li> 
 <li>修正了崩溃问题，该问题阻碍了在同级子节点之间的切换</li> 
 <li>更新了 JS Sanitizer，以在 <code>SAFE_URL_PATTERN</code> 中添加 <code>sms</code></li> 
 <li>改进了 <code>.img-fluid</code> 的相关文档</li> 
 <li>为我们的文档中的表单 switches 添加了 <code>role="switch"</code></li> 
 <li>实施了 GitHub issue 表单，以取代我们以前的 issue 模板</li> 
</ul>
