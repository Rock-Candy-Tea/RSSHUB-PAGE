                                                                                            <p>FileZilla Server 是一个免费开源的 FTP 和 FTPS 服务器，能够提供与服务器的安全加密连接。Filezilla Server 没有实现对 SFTP（SSH 文件传输协议）的支持。</p> 
<p>FileZilla Server 1.2.0 发布，更新内容如下：</p> 
<h3>新功能</h3> 
<ul> 
 <li>创建了 Debian 和 macOS 软件包</li> 
 <li>为挂载点引入了更精细的访问控制</li> 
</ul> 
<h3>错误修正和改动：</h3> 
<ul> 
 <li>小的可用性变化，以设置 Let's Encrypt 证书</li> 
 <li>MSW：不允许文件和目录以空格或点结尾</li> 
 <li>如果一个用户被禁用，相应的会话现在会被踢出</li> 
 <li>Let's Encrypt 证书现在只有在使用中才会被更新</li> 
 <li>UI 现在允许你编辑用户信息，即使他们被禁用</li> 
 <li>修正了对 Windows 上 UNC 路径名的支持</li> 
 <li>修正了一些回归问题</li> 
 <li>Autoban 现在可以在不同的登录会话中共享状态，这使其能按预期工作</li> 
 <li>UI 现在可以在日志中清楚地显示它与 FileZilla FTP 服务器的连接情况</li> 
</ul> 
<p>更多详情可查看：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Ffilezilla-project.org%2F" target="_blank">https://filezilla-project.org/</a></p>
