                                                                                            <p>DBeaver 是一个免费开源的通用数据库工具，适用于开发人员和数据库管理员。</p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">DBeaver 22.1.2 现已发布，更新内容如下：</p> 
<ul style="margin-left:0; margin-right:1.5em"> 
 <li>Data editor： 
  <ul style="margin-left:0; margin-right:0"> 
   <li>数组和结构数据类型的可视化得到修复和改进</li> 
   <li>添加了类似控制台的查询结果查看器</li> 
   <li>修复了聚合面板中对列别名的支持</li> 
   <li>字典面板现在支持分页</li> 
   <li>固定列滚动已修复</li> 
   <li>二进制/十六进制数据查看器已修复（在 MacOS 上崩溃）</li> 
   <li>消除了字符串编辑器面板中的主要内存泄漏问题</li> 
  </ul> </li> 
 <li>SQL editor： 
  <ul style="margin-left:0; margin-right:0"> 
   <li>DDL 查询的参数绑定已修复</li> 
   <li>查询错误突出显示得到改进（+现在可以禁用）</li> 
   <li>查询格式化程序已修复（对于从数字开始的标识符）</li> 
   <li>改进了紧凑型格式化程序</li> 
   <li>物化视图创建/更改查询的自动完成问题已修复</li> 
  </ul> </li> 
 <li>Data transfer： 
  <ul style="margin-left:0; margin-right:0"> 
   <li>自定义 SQL 查询的目标文件名生成已修复</li> 
   <li>已为已保存的任务修复设置保存/加载</li> 
  </ul> </li> 
 <li>General improvements： 
  <ul style="margin-left:0; margin-right:0"> 
   <li>消除了保存 preferences 后的虚拟错误消息</li> 
   <li>改进了元数据编辑器中不适用列的 UI</li> 
   <li>主窗口最小化问题已在 MacOS 上修复</li> 
   <li>修复了许多小的 ui 错误</li> 
  </ul> </li> 
 <li>Clickhouse：修复了数据编辑器的问题</li> 
 <li>CocroachDB：添加了序列支持</li> 
 <li>H2：模式序列读取已修复</li> 
 <li>Hive：view source editor 已修复（分号问题）</li> 
 <li>Kylin：改进了方言支持</li> 
 <li>MySQL、MariaDB：改进了权限编辑器 UI</li> 
 <li>Oracle：VARCHAR2 现在是默认的字符串数据类型</li> 
 <li>PostgreSQL：数据库刷新已修复（活动模式标记问题）</li> 
 <li>Snowflake：BEGIN TRANSACTION 查询执行的问题已修复</li> 
 <li>SQL Server： 
  <ul style="margin-left:0; margin-right:0"> 
   <li>Table DDL：重复索引声明的问题已修复</li> 
   <li>GO 分隔符评估问题已修复（也适用于 Sybase）</li> 
   <li>Database name proposal 已修复</li> 
   <li>执行计划可视化已修复</li> 
  </ul> </li> 
</ul>
