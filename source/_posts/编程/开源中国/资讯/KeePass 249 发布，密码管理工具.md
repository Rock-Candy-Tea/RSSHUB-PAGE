                                                                                            <p>KeePass 是一款密码管理工具，可以帮助用户记住电子邮件、主页 FTP、论坛等用户名和密码。KeePass 将密码保存在高度加密的数据库中，不让其他人和其他应用程序所识别。</p> 
<p>KeePass 2.49 正式发布，该版本更新内容如下：</p> 
<h3><strong>新功能：</strong></h3> 
<ul> 
 <li>增加了可选的 "自动类型-已启用 "主条目列表列（默认关闭，可在 "视图"→"配置列" 中打开）；</li> 
 <li>增加了可选的 "自动类型-序列 "主条目列表列（默认关闭，可在 "视图"→"配置列 "中打开）；</li> 
 <li>增加了对导入 Keeper 16.0.0 JSON 文件的支持；</li> 
 <li>为一些控件增加了工具提示；</li> 
 <li>对话框横幅现在可以被屏幕阅读器读取；</li> 
 <li>为一些控件添加了无障碍名称；</li> 
 <li>可访问性/自动化控制树中的控件现在按其视觉位置排序；</li> 
 <li>增加了基于操作系统的屏幕阅读器参数的自动屏幕阅读器检测；</li> 
 <li>各种可访问性的增强；</li> 
 <li>增加了 <code>UIFlags</code>，用于在关闭/锁定数据库时不清除快速搜索词列表;</li> 
 <li>增加了 <code>UIFlags</code>，用于在安全桌面上启用 IME;</li> 
 <li>增加了自定义选项，以覆盖为帮助页面打开的 URL</li> 
</ul> 
<h3><strong>改进：</strong></h3> 
<ul> 
 <li>在大多数列表视图中，工具提示现在显示的时间更长</li> 
 <li>条目字符串移动命令现在可以一次移动多个字符串</li> 
 <li>删除了条目对话框中的条目字符串按钮 "移动"</li> 
 <li>在条目详情视图和 "配置列" 对话框中重新排序字段/列，以提高与条目对话框的一致性；</li> 
 <li>当在 .NET 4.7 或更高版本上运行时，富文本框现在实例化了 Rich Edit 4.1（或更高版本）控件，它支持各种新功能；</li> 
 <li>在主窗口的条目详情视图和内部数据查看器中改进了链接化</li> 
 <li>内部数据查看器的十六进制视图中的 URL 不再被链接；</li> 
 <li>改进了各种对话框中的标签顺序；</li> 
 <li>改进了临时文件/目录的删除性能</li> 
 <li>改进了一些图标的质量</li> 
 <li>在 HTML 导出/打印对话框中，当组名不可用时，"组名" 选项现在被禁用</li> 
 <li>减少了 HTML 导出/打印中表格和组标题之间的空间</li> 
 <li>改进了键盘快捷键检查</li> 
 <li>TrlUtil：改进了字符串表的处理</li> 
 <li>各种 CHM 文件的改进</li> 
 <li>升级了安装程序</li> 
 <li>UI 文本的改进</li> 
 <li>代码优化</li> 
 <li>其他小的改进</li> 
</ul> 
<h3><strong>错误修正：</strong></h3> 
<ul> 
 <li>条目对话框的保存确认对话框不会再创建不必要的历史条目；</li> 
 <li>条目/组对话框中的标签按钮下拉菜单现在使用正确的字体；</li> 
 <li>TrlUtil：修复了一个字符串表关联的错误</li> 
</ul>
