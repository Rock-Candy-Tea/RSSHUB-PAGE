                                                                                            <p>Qmmp 是一个免费开源的跨平台音频播放器，它是用 C++ 编写的，用户界面使用 Qt widget 工具包。Qmmp 以其小巧、可设置主题的用户界面和对系统资源的低占用率而闻名。</p> 
<p>Qmmp 2.0.3 发布，自 2.0.0 以来的变化包括：</p> 
<ul> 
 <li>修复了构建问题</li> 
 <li>修复了传输插件的加载</li> 
 <li>修复了双倍大小模式下的音量条绘制</li> 
 <li>修复了符号链接支持</li> 
 <li>修正了双面板文件对话框中的一些问题</li> 
 <li>删除了 KDE4 通知支持</li> 
 <li>修复了源代码外的构建</li> 
 <li>修复了 ffmpeg 插件的默认设置</li> 
 <li>修复了构建警告</li> 
 <li>修复了 http 轨道之间的过渡</li> 
 <li>改进了 qsui 插件： 
  <ul> 
   <li>修复了工具条重命名</li> 
   <li>修复了 Wayland 支持</li> 
   <li>修复了标签栏的上下文菜单</li> 
  </ul> </li> 
 <li>在 Qt 6.2 或更高版本下启用 QtMultimedia 支持</li> 
 <li>删除了 Qt 6.2 或更高版本的私有 API 使用</li> 
 <li>更新了多国语言翻译</li> 
</ul> 
<p>更多详情可查看：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fqmmp.ylsoftware.com%2Findex.php" target="_blank">https://qmmp.ylsoftware.com/index.php</a></p>
