                                                        <p><span style="color:#333333">NativeScript 8.2.2 现已发布。NativeScript 使用户能够直接从 JavaScript 访问 native API。目前，该框架为丰富的移动开发提供了 iOS 和 Android 运行时，并可用于多种不同的用例。</span></p> 
<h3><strong>Bug 修复</strong></h3> 
<ul> 
 <li><strong>ActionItem：</strong>修复有一个嵌套的子元素时，渲染会引发错误的问题。(<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FNativeScript%2FNativeScript%2Fissues%2F9821" target="_blank">#9821</a>)</li> 
 <li><strong>Android：</strong>手势事件修复 (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FNativeScript%2FNativeScript%2Fissues%2F9842" target="_blank">#9842</a>)</li> 
 <li><strong>css：</strong>修复对 borderColor 解析的 hsl(a) 颜色值处理 (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FNativeScript%2FNativeScript%2Fissues%2F9857" target="_blank">#9857</a>)</li> 
 <li><strong>ios：</strong>导航按钮允许使用自定义图标 (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FNativeScript%2FNativeScript%2Fissues%2F9835" target="_blank">#9835</a>)</li> 
 <li><strong>RootLayout：</strong>关闭实时同步的弹出视图 (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FNativeScript%2FNativeScript%2Fissues%2F9834" target="_blank">#983</a> )</li> 
 <li>通过重用不同的设置实现 Utils.queueGC 去抖动和节流  (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FNativeScript%2FNativeScript%2Fissues%2F9852" target="_blank">#9852</a>)</li> 
</ul> 
<h3>特性</h3> 
<ul> 
 <li><strong>RootLayout：</strong>添加 topmost 方法来检索顶部视图 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FNativeScript%2FNativeScript%2Fissues%2F9826" target="_blank">#9826</a> )</li> 
 <li><strong>TouchManager：</strong>嵌套点击控件的 touchDelay 属性 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FNativeScript%2FNativeScript%2Fcommit%2Fc05145bd9b62cb017349acf46827fec929b3289c" target="_blank">c05145b</a> )</li> 
</ul> 
<h3><strong>性能改进</strong></h3> 
<ul> 
 <li>改进了 RootLayout 的实时同步功能 (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FNativeScript%2FNativeScript%2Fissues%2F9836" target="_blank">#9836</a>)</li> 
</ul>
