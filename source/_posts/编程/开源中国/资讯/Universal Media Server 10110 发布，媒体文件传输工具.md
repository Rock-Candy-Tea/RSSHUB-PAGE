                                                                                            <p>Universal Media Server 10.11.0 现已发布。Universal Media Server 是一个兼容 DLNA，UPnP 和 HTTP/S 的媒体服务器，支持所有主要操作系统，包括 Windows，Linux 和 macOS 版本，几乎不需要配置就可以流式传输或转码许多不同的媒体格式。</p> 
<p><strong>主要更新内容</strong></p> 
<ul> 
 <li>General 
  <ul> 
   <li>在自动更新程序中添加了更改日志的链接</li> 
   <li>修复杜比视界视频被检测为数字视频的问题</li> 
   <li>修复了在某些渲染器上的浏览</li> 
   <li>修复了 tsMuxeR 在 macOS 上不工作的问题</li> 
  </ul> </li> 
 <li>Renderers 
  <ul> 
   <li>改进了对 LG OLED 电视字幕的支持</li> 
  </ul> </li> 
 <li>依赖性 
  <ul> 
   <li>将 checkstyle 从 8.45.1 提升到 9.0</li> 
   <li>将 JNA 从 5.8.0 升级到 5.9.0</li> 
   <li>将 oshi-core 从 5.8.0 升级到 5.8.1</li> 
   <li>将 tsMuxeR 调到 2021-09-02-01-59-46</li> 
  </ul> </li> 
</ul>
