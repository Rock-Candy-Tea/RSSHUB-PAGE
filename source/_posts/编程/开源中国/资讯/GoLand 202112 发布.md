                                                        <p>GoLand 2021.1.2  正式发布，该版本主要是一个 bug 修复版本。</p> 
<p><strong>主要更新内容</strong></p> 
<ul> 
 <li>修复了采用 M1 处理器的 macOS 用户在最新的 Big Sur 和 Xcode 更新后，无法启动调试会话的问题</li> 
 <li>修复了一个导致 "即时优化导入" 选项工作不正常的错误，它不断地删除和重新添加导入</li> 
 <li>在行末尾建议 “从 json 生成类型”</li> 
 <li>修复了 Go Tools 参数不解析宏字段的问题</li> 
 <li>运行目标：'vendor' 目录没有通过 SSH 进行复制</li> 
 <li>当存在一个私有方法时，查找实现的方法被破坏了</li> 
 <li>修复了 GoLand 自动导入对软件包 "fmt" 处理出错的问题</li> 
 <li>修复了 Tab 和缩进设置无法保存的问题</li> 
 <li>修复了编辑的配色方案一直被换成 Darcula 的问题</li> 
 <li>将 IPR 转换为基于目录的项目并不转换项目库</li> 
 <li>修复了临时添加模块时的，自动卸载新添加的模块不起作用的问题</li> 
 <li>项目共享索引元数据使用到存储库的相对路径</li> 
 <li>共享索引会提供有关索引诊断的更多信息</li> 
</ul>
