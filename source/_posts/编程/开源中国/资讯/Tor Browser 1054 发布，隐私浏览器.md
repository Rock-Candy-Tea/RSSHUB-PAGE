                                                                                            <p>Tor Browser 是一个基于 Firefox ESR (Firefox with extended support) 的 Web 浏览器，默认配置通过 Tor 和 Vidalia 实现了个人隐私保护和匿名。</p> 
<p>这个版本将 Firefox 更新到 78.13.0esr。该版本包括对 Firefox 的重要安全更新。</p> 
<p>自 Tor Browser 10.5.2 以来的变化包括：</p> 
<ul> 
 <li>Windows + OS X + Linux 
  <ul> 
   <li>更新 Firefox 到 78.13.0esr；</li> 
   <li>更新 NoScript 到 11.2.11；</li> 
   <li>Bug 40041：移除桌面版 about:tor 上的 V2 弃用 banner；</li> 
   <li>Bug 40506：在 10.5 中无法使用已保存的登录信息；</li> 
   <li>Bug 40524：更新 DuckDuckGo 在搜索首选项时的洋葱网站网址；</li> 
  </ul> </li> 
 <li>构建系统 
  <ul> 
   <li>Windows + OS X + Linux 
    <ul> 
     <li>更新 Go 至 1.15.14</li> 
    </ul> </li> 
  </ul> </li> 
</ul>
