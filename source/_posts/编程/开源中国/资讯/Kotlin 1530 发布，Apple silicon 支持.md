                                                                                            <p>Kotlin 1.5.30 现已发布。这是 Kotlin 1.6.0 之前的最后一个增量版本，包含了许多计划在 Kotlin 1.6.0 中发布的实验性语言和标准库功能。这些实验性功能包括 sealed when 语句、对 opt-in 要求的修改、注解类的实例化、对 Duration 和 Regex stdlib API 的改进等等。</p> 
<p>通过 Kotlin 1.5.30，你还可以享受对 Apple silicon 的原生支持、Kotlin/JS IR 后端晋升为 Beta 版、能够在多平台应用程序的共享原生代码中使用自定义 cinterop 库、支持 Kotlin Gradle 插件提供的 Java 工具链等等。</p> 
<p>此版本主要更新内容如下：</p> 
<ul> 
 <li>语言上的改进，包括 sealed<code>when</code>语句、对作为超类型的<code>suspend</code>函数的支持以及对 opt-in 要求的更改。</li> 
 <li>多平台改进，包括在共享本机代码中使用自定义的<code>cinterop</code>库的能力以及对 XCFrameworks 作为输出格式的支持。</li> 
 <li>Kotlin/JVM 改进，包括注释类的实例化和 nullability 注释支持的改进配置。</li> 
 <li>Kotlin/Native 改进，包括对 Apple silicon 的原生支持和改进的 CocoaPods Gradle 插件的 Kotlin DSL。</li> 
 <li>Kotlin/JS IR 改进，包括将 JS IR 后端提升到 Beta 以及对新后端更好的调试体验。</li> 
 <li>Gradle 改进，包括对 Java 工具链的支持和更简单的方法来明确设置 Kotlin 守护进程的 JVM 参数。</li> 
 <li>标准库中对<code>Duration</code>和<code>Regex</code>API 的改进。</li> 
</ul>
