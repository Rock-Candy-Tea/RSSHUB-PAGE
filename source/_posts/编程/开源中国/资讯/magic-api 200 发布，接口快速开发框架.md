                                                                                            <h3 style="margin-left:0; margin-right:0; text-align:left">本次更新内容：</h3> 
<div style="text-align:left"> 
 <ul style="list-style-type:disc; margin-left:0; margin-right:0"> 
  <li>新增<code>mybatis</code>语法<code><elseif></code>、<code><else></code>、<code><foreach></code>新增<code>index</code>参数</li> 
  <li>新增<code>RequestBody</code>参数类型<code>Date</code>、<code>Any</code></li> 
  <li>修复复制的接口无法删除的问题</li> 
  <li>修复定时任务插件中定时任务描述无法保存的问题</li> 
  <li>修复没有历史记录时，提示不正确的问题</li> 
  <li>修复编辑器字体、字体大小配置无效的问题</li> 
  <li>修复<code>import</code>其它接口时需要携带<code>magic-api.prefix</code>的问题</li> 
  <li>优化找不到模块、函数时的错误提示</li> 
  <li>优化<code>Swagger</code>插件，减少无影响的异常日志</li> 
  <li>优化<code>WebSocket</code>链接，缓解<code>linux</code>下链接可能超时的问题</li> 
  <li>优化单表<code>API</code>中<code>insert</code>接口，在驱动不支持返回主键时返回设定的主键值。</li> 
  <li>优化返回结构解析，保持原样输出</li> 
  <li>优化<code>UI</code>细节，切换tab时，记住滚动条位置、优化左右下三处面板的可拖动范围</li> 
 </ul> 
</div> 
<h4 style="margin-left:0; margin-right:0; text-align:start"><strong>项目介绍</strong></h4> 
<p data-darkreader-inline-color style="--darkreader-inline-color:#c8c3bc; color:#333333; margin-left:0px; margin-right:0px; text-align:left"><span data-darkreader-inline-bgcolor data-darkreader-inline-color style="--darkreader-inline-bgcolor:#181a1b; --darkreader-inline-color:#b7b1a7; background-color:#ffffff; color:#40485b">magic-api 是一个基于 Java 的接口快速开发框架，编写接口将通过 magic-api 提供的UI界面完成，自动映射为  HTTP 接口，无需定义 Controller、Service、Dao、Mapper、XML、VO 等 Java 对象即可完成常见的 HTTP API 接口开发</span></p> 
<h3 style="margin-left:0; margin-right:0; text-align:left"><span data-darkreader-inline-color style="--darkreader-inline-color:#b7b1a7; color:#40485b">特性</span></h3> 
<ul style="list-style-type:disc; margin-left:0; margin-right:0"> 
 <li>支持 MySQL、MariaDB、Oracle、DB2、PostgreSQL、SQLServer 等支持jdbc规范的数据库</li> 
 <li>支持非关系型数据库 Redis、Mongodb</li> 
 <li>支持集群部署、接口自动同步。</li> 
 <li>支持分页查询以及自定义分页查询</li> 
 <li>支持多数据源配置，支持在线配置数据源</li> 
 <li>支持 SQL 缓存，以及自定义 SQL 缓存</li> 
 <li>支持自定义 JSON 结果、自定义分页结果</li> 
 <li>支持对接口权限配置、拦截器等功能</li> 
 <li>支持运行时动态修改数据源</li> 
 <li>支持 Swagger 接口文档生成</li> 
 <li>基于 <a href="https://gitee.com/ssssssss-team/magic-script" target="_blank">magic-script</a> 脚本引擎，动态编译，无需重启，实时发布</li> 
 <li>支持 Linq 式查询，关联、转换更简单</li> 
 <li>支持数据库事务、SQL 支持拼接，占位符，判断等语法</li> 
 <li>支持文件上传、下载、输出图片</li> 
 <li>支持脚本历史版本对比与恢复</li> 
 <li>支持脚本代码自动提示、参数提示、悬浮提示、错误提示</li> 
 <li>支持导入 Spring 中的 Bean、Java 中的类</li> 
 <li>支持在线调试</li> 
 <li>支持自定义工具类、自定义模块包、自定义类型扩展、自定义方言、自定义列名转换等自定义操作</li> 
</ul>
