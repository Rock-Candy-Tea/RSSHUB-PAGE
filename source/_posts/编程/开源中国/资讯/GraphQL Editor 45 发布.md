                                                                                            <p>GraphQL Editor 4.5 已经版本。Graphql editor 是一款 Graphql 的可视化编辑器和 IDE，帮助用户理解 GraphQL 模式，并通过使用可视化块系统创建模式。</p> 
<p><strong>主要更新内容</strong></p> 
<ul> 
 <li><strong>关系视图</strong></li> 
</ul> 
<p>关系视图是显示项目的 GraphQL 模式一种新方式。其灵感来自于 PCB 板，它能帮助用户识别其模式元素之间的关系，并在它们之间进行导航。</p> 
<p><img alt height="339" src="https://static.oschina.net/uploads/space/2021/0325/071829_xY7F_4937141.png" width="590" referrerpolicy="no-referrer"></p> 
<ul> 
 <li><strong>UI 升级</strong></li> 
</ul> 
<p>本次更新在应用程序的外观上做了一些改动，以改善其美观和可用性，并且加入了主题切换系统的基础。目前只有一个深色主题，但很快会提供更多的主题。</p> 
<ul> 
 <li><strong>重新设计学习区</strong></li> 
</ul> 
<p>GraphQL Editor 的主要目标之一是成为使用 GraphQL 工作以及学习它的地方。因此本次更新重新设计了学习体验，使其更具互动性，以帮助用户能够更容易学习 GraphQL 的所有最重要的方面。</p> 
<p><img alt height="417" src="https://static.oschina.net/uploads/space/2021/0325/071854_yLIn_4937141.png" width="590" referrerpolicy="no-referrer"></p> 
<ul> 
 <li><strong>GitHub 集成</strong></li> 
</ul> 
<p>本次更新后的 GitHub 集成支持用户将其 GitHub 账户链接到 GraphQL Editor，导出其 GraphQL 模式并以类似于 GitHub 流程的方式对其进行工作。</p> 
<p>详情请查看<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fblog.graphqleditor.com%2Fgraphql-editor-v45%2F" target="_blank">更新公告</a>。</p>
