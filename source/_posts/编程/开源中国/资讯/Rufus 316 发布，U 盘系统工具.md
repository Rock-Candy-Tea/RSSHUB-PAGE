                                                        <p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><span style="color:#333333">Rufus 是一个可以帮助格式化和创建可引导 USB 闪存盘的工具，它可以快速把 ISO 格式的系统镜像文件快速制作成可引导的 USB 启动安装盘，支持 Windows 或 Linux 启动。3.16 版本更新内容如下： </span></p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>修复对 Red Hat 8.2+ 及其衍生产品的 ISO 模式支持</li> 
 <li>修复对 Arch 衍生产品的 BIOS 启动支持</li> 
 <li>修复对 Ubuntu 衍生品的一些 boot entries 的删除</li> 
 <li>修复退出时未保存日志的问题</li> 
 <li>添加 Windows 11“Extended”安装支持（禁用 TPM/Secure Boot/RAM requirements）</li> 
 <li>添加 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fpbatard%2FUEFI-Shell%2Freleases" target="_blank">UEFI Shell</a> ISO 下载</li> 
 <li>添加对英特尔 NUC 读卡器的支持</li> 
 <li>改进 Windows 11 支持</li> 
 <li>改进 Windows 版本报告</li> 
 <li>加速清除 MBR/GPT</li> 
</ul> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><span style="color:#333333">更新说明：</span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Frufus.ie%2Fzh%2F">https://rufus.ie/zh/</a> </p>
