                                                                                            <p>GNOME 团队在今天推出了 GNOME 42 的 Beta 版本，该版本比最初计划的发布日期延迟了十天。</p> 
<p>在 GNOME 42 Beta 中，变化包括：</p> 
<ul> 
 <li>GNOME Calls 现在允许 SIP 供应商处理 "tel" URI。在 Calls 中还支持从通话历史中添加联系人</li> 
 <li>GNOME Epiphany 网络浏览器已经更新了其绑定的 PDF.js 和 readability.js 代码</li> 
 <li>GJS 已经针对 SpiderMonkey 91 进行了更新，因此现在支持许多新的 JavaScript 功能</li> 
 <li>Glib 增加了一个异步文件移动 API</li> 
 <li>加入了新的默认的浅色和深色壁纸</li> 
 <li>重新设计的扩展应用程序</li> 
 <li>改进了对笔记本电脑和键盘背光的支持</li> 
 <li>GNOME 蓝牙现在为支持的蓝牙设备输出电池信息</li> 
 <li>GNOME 日历已被移植到 GTK4</li> 
 <li>GNOME Desktop 为缩略图的创建增加了异步方法</li> 
 <li>GNOME Software 改进了它的黑暗主题和各种 UI 更新</li> 
 <li>Nautilus 文件管理器现在支持黑暗模式设置</li> 
 <li>在这个测试版中被移植到 GTK4 的其他 GNOME 程序包括 GNOME Font Viewer、GNOME Music、GNOME Tour 和 GNOME Weather</li> 
 <li>许多 GNOME Shell 和 Mutter 的变化包括新的截图/截屏 UI、对黑暗壁纸的支持，改进的XWayland 和 DRM 隐私屏幕支持等</li> 
 <li>gtksourceview 4 和 libsoup 2.4 已被弃用，并将在未来的版本中被删除</li> 
</ul> 
<p>GNOME 42 候选版本将在 3 月中旬发布，GNOME 42.0 正式版将在 3 月底正式亮相。</p>
