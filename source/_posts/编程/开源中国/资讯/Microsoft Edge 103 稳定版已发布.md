                                                                                            <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">Microsoft Edge 103 稳定版已更新，该版本主要带来如下内容：</p> 
<h3 style="margin-left:.6em; margin-right:0; text-align:start">功能更新</h3> 
<ul style="margin-left:0; margin-right:0"> 
 <li><strong>能够控制自动配置文件切换：</strong>新的<strong><span> </span></strong>GuidedSwitchEnabled 策略允许 Microsoft Edge 在检测到个人链接或工作链接时，提示用户切换到适当的配置文件。</li> 
 <li><strong>客户端证书切换器：</strong>用户在访问需要 http 证书身份验证的站点时，可以清除记住的证书，并重新显示证书选择器。（无需退出 Microsoft Edge 即可完成证书的切换）</li> 
 <li><strong>更可靠的网络防御：</strong>重写的<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdocs.microsoft.com%2Fen-us%2Fdeployedge%2Fmicrosoft-edge-security-smartscreen" target="_blank">Microsoft Defender SmartScreen</a><span> </span>库在浏览 Web 时提供更可靠的保护。（旧版本的库将在 Microsoft Edge 版本 105 中被弃用）</li> 
</ul> 
<h3 style="margin-left:.6em; margin-right:0; text-align:start">新策略</h3> 
<ul style="margin-left:0; margin-right:0"> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdocs.microsoft.com%2Fen-us%2FDeployEdge%2Fmicrosoft-edge-policies%23guidedswitchenabled" target="_blank">GuidedSwitchEnabled</a><span> </span>：这就是上文提及的配置文件切换策略，已启用引导开关</li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdocs.microsoft.com%2Fen-us%2FDeployEdge%2Fmicrosoft-edge-policies%23internetexplorerzoomdisplay" target="_blank">InternetExplorerZoomDisplay</a><span> </span>：该策略可在 IE 模式选项卡中显示缩放，其中包含 DPI 比例，就像在 Internet Explorer 中一样</li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdocs.microsoft.com%2Fen-us%2FDeployEdge%2Fmicrosoft-edge-policies%23livecaptionsallowed" target="_blank">LiveCaptionsAllowed</a><span> </span>：该策略允许实时字幕</li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdocs.microsoft.com%2Fen-us%2FDeployEdge%2Fmicrosoft-edge-policies%23originagentclusterdefaultenabled" target="_blank">OriginAgentClusterDefaultEnabled</a><span> </span>：该策略默认启用 Origin-keyed 代理集群</li> 
</ul> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><strong>策略变更</strong></p> 
<ul style="margin-left:0; margin-right:0"> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdocs.microsoft.com%2Fen-us%2FDeployEdge%2Fmicrosoft-edge-policies%23sleepingtabstimeout" target="_blank">SleepingTabsTimeout</a><span> </span>：此策略中添加了后台标签 30 秒不活动就被视作超时，设置为睡眠标签。</li> 
</ul>
