                                                                                            <p>DBeaver 21.1.5 现已发布。DBeaver是一个免费开源的通用数据库工具，适用于开发人员和数据库管理员。</p> 
<p><strong>主要更新内容</strong></p> 
<ul> 
 <li>ERD 编辑器 
  <ul> 
   <li>为图表元素添加了属性面板</li> 
   <li>添加了图表拖动工具</li> 
   <li>搜索用户界面得到改进</li> 
   <li>图表刷新现在要求确认</li> 
   <li>表列导航已修复</li> 
   <li>改进了 GraphML 导出（注释、样式）</li> 
   <li>添加了以 XML (.erd) 格式导出 ERD</li> 
  </ul> </li> 
 <li>Data 编辑器 
  <ul> 
   <li>改进了默认值的行插入</li> 
   <li>数据过滤器弹出窗口现在可视化加载过程</li> 
   <li>固定列现在适用于多个选定的列</li> 
   <li>过滤器对话框现在支持列固定/取消固定</li> 
   <li>分组面板中的自动完成已修复</li> 
   <li>明文记录模式切换已修复</li> 
   <li>Excel 导入已修复（添加了 NULL 单元格支持）</li> 
  </ul> </li> 
 <li>SQL 编辑器：操作按钮工具提示已修复（键盘快捷键）</li> 
 <li>SQL Server：添加了用户管理</li> 
 <li>Oracle：查看触发器读取已修复</li> 
 <li>PostgreSQL 
  <ul> 
   <li>添加了额外的 PostGIS 数据类型支持</li> 
   <li>连接用户角色句柄已修复</li> 
  </ul> </li> 
 <li>程序体读取现在按需重新建立连接</li> 
 <li>Shell 命令现在支持带引号的参数（Linux、MacOS）</li> 
 <li>Debian/RPM 软件包安装程序现在为社区和商业版本使用不同的目录</li> 
 <li>简单的模式比较向导 UI 已修复</li> 
 <li>包含密码的 SQL 查询已从日志中删除</li> 
 <li>驱动程序编辑器对话框增强</li> 
</ul>
