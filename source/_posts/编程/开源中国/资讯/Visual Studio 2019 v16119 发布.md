   <h3 style="margin-left:.6em; margin-right:0">此版本中解决的问题</h3> 
   <ul style="margin-left:0; margin-right:0"> 
    <li>修正了当 Windows Terminal 被用作默认终端时，无法多次调试应用程序的问题。</li> 
    <li>设置修补程序以取消阻止受限制配置的客户。</li> 
    <li>修复了阻止客户端更新最新引导程序的问题。 一旦客户端使用 2022 年 1 月或更高版本附带的引导程序和安装程序，所有使用后续引导程序的更新都应在产品生命周期内正常工作。</li> 
    <li>解决了在检测具有易失性元数据的二进制文件导致检测分析失败时 VSInstr 不会退出的偶发实例。</li> 
    <li>修复了使用 /Og 或 #pragma optimize ("g") 编译具有极大型函数的 C++ 代码时，可能会生成无效代码的问题。</li> 
    <li>修复了 C++ Concurrency::parallel_for_each 中的一个 bug，该 bug 会因整数溢出而使调用进程崩溃。</li> 
   </ul>
