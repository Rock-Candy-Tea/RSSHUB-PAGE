<div class="content">
                                                                                            <p>微软<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fblogs.windows.com%2Fwindowsexperience%2F2021%2F08%2F31%2Fwindows-11-available-on-october-5%2F" target="_blank">发文宣布</a> Windows 11 将于 10 月 5 日开始提供，届时微软将开始让符合条件的 Windows 10 PC 免费升级到 Windows 11，并且新 PC 也将预装 Windows 11。</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-b5966a41490b34a5ad8f0366500f1bcbbb1.jpg" referrerpolicy="no-referrer"></p> 
<p>根据微软的计划，免费的 Windows 11 升级将以质量为重点分阶段提供，根据硬件资格、可靠性指标、设备年龄和其他影响升级体验的因素，新的符合条件的设备将先获得升级。预计到 2022 年中期，所有符合条件的设备都将免费升级到 Windows 11。此外，微软还在文中列出了 Windows 11 的亮点，主要包括：</p> 
<ul> 
 <li>新的桌面设计和声音</li> 
 <li>Start 将置于中心位置，并且其可利用云和 Microsoft 365 向用户展示最近在任何设备上查看的文件</li> 
 <li>Snap Layouts、Snap Groups 和 Desktops 提供了一种更强大的方式来处理多任务，并优化屏幕空间、</li> 
 <li>Microsoft Teams 聊天将集成到任务栏</li> 
 <li>由 AI 提供支持的全新个性化订阅源 Widgets</li> 
 <li>支持 DirectX12 Ultimate、DirectStorage 和 Auto HDR 等技术</li> 
 <li>全新的 Microsoft Store，并将继续引入 Android 应用程序，这会从之后几个月的 Windows Insiders 预览开始</li> 
 <li>专为残障人士打造的全新辅助功能改进</li> 
 <li>将开放商店，让更多的开发人员和独立软件供应商 (ISV) 将其应用程序发布到商店中</li> 
 <li>对触控、数字笔和语音输入体验进行了优化</li> 
</ul> 
<p>此外，微软正在迅速关闭在不合规硬件上安装新操作系统的途径。即使已经加入 Windows Insider 计划的 PC，如果硬件不合规也会被通知其应该降级到 Windows 10。</p>
