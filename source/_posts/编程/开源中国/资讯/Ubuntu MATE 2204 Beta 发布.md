                                                                                            <p>Ubuntu MATE 22.04 Beta 近日发布，通过该 Beta 版本，用户可以看到 Ubuntu MATE 的最新变化和正在尝试的内容。按照开发计划，稳定版本将于 4 月底发布。</p> 
<p>以下是 Ubuntu MATE 22.04 Beta 的主要变化和亮点：</p> 
<h3>MATE Desktop 1.26.1</h3> 
<p>Ubuntu MATE 22.04 具有 MATE Desktop 1.26.1。MATE Desktop 1.26.0 修复了大量 bug、优化性能，并解决了内存泄漏问题。还修复了 Plank 和 Brisk Menu 和屏幕阅读器在安装过程中的问题。</p> 
<h3>Yaru</h3> 
<p>Ubuntu MATE 21.04 是第一个带有 Yaru 主题的版本。一年后，团队为上游的 Yaru 带来完全的 MATE 兼容性。</p> 
<p><img alt height="420" src="https://static.oschina.net/uploads/space/2022/0406/072240_Z5Ar_4937141.png" width="700" referrerpolicy="no-referrer"></p> 
<p>Ubuntu MATE 22.04 中的 Yaru 主题</p> 
<p>在 Ubuntu MATE 22.04 中包含了所有的 Yaru 主题，传统的 Ambiant/Radiant 主题和 MATE Desktop 主题不再被默认安装。新版本增加了一个自动设置迁移功能，以便将升级的用户过渡到合适的 Yaru MATE 主题。</p> 
<h3>暗色/亮色</h3> 
<p>在与 Yaru 团队和 Ubuntu Desktop 团队的合作下，团队为 MATE Desktop 的 Yaru 和 Unity 增加了暗色/亮色面板和面板图标。</p> 
<p><img alt height="81" src="https://static.oschina.net/uploads/space/2022/0406/072257_Cm9v_4937141.png" width="700" referrerpolicy="no-referrer"></p> 
<p>新版本在外观控制中心添加了补丁，将主题变化应用于 Plank（Dock）、Pluma（文本编辑器）并正确切换 GNOME 42 应用程序的色彩方案偏好。当你选择一个黑暗的主题时，所有的东西都会统一变成暗色，反之亦然。</p> 
<h3>AI 生成的壁纸</h3> 
<p>新版本已经将 3 张最高票数的 "Jammy Jellyfish" 纳入了壁纸选择，因为这些壁纸生动而充满活力的风格与 Yaru 强调色主题选项相得益彰。</p> 
<p><img alt height="394" src="https://static.oschina.net/uploads/space/2022/0406/072308_xA1s_4937141.png" width="700" referrerpolicy="no-referrer"></p> 
<p>除默认包含的三张壁纸意外，如果想要获得完整的八张壁纸，可点击链接下载</p>

</ul>
