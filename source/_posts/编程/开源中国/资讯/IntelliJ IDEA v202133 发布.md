                                                                                            <p style="margin-left:0px">IntelliJ IDEA <span style="color:#27282c">v2021.3.3 发布了，</span>最新版本包含以下改进和修复：</p> 
<ul> 
 <li>IntelliJ IDEA 现在在使用 Maven 3.8.5 时可以正确构建和过滤资源。[ <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-290419" target="_blank">IDEA-290419</a> ]</li> 
 <li>修复了索引 WSL 项目时导致 IDE 冻结和挂起的问题。[ <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-286059" target="_blank">IDEA-286059</a> ]</li> 
 <li>修复了使用 Gradle 时出现的一些问题。[ <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-287618" target="_blank">IDEA-287618</a> ]；[<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-285663" target="_blank"> 285663</a> ]；[<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-284391" target="_blank">284391</a> ]；[ <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-284013" target="_blank">IDEA-284013</a> ]</li> 
 <li>修复了导致 IDE在推送到 Gerrit 时添加意外的<em>refs/heads前缀的问题。</em>[ <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-282180" target="_blank">IDEA-282180</a> ]</li> 
 <li>修复了阻碍通过 SSH 与远程主机正确连接的问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-284623" target="_blank">284623</a> ]；[ <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-281113" target="_blank">IDEA-281113</a> ]</li> 
 <li>修复了<em>快速文档</em>弹出窗口，使其正确显示包信息。[ <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-281424" target="_blank">IDEA-281424</a> ]</li> 
 <li>修复了创建文件模板时的自定义字段。[<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-259172" target="_blank">259172</a> ]；[<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-127570" target="_blank"> 127570</a> ]</li> 
 <li>修复了在重命名 Java 包时导致基于 Groovy 的测试类调整不正确的问题。[<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-240421" target="_blank">24042</a> ]</li> 
 <li>修复了在 macOS 上使用<em>Cmd+Shift+[ 和 Cmd+Shift+]</em>在 Big Sur 选项卡之间切换时的 IDE 行为。[ <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-261595" target="_blank">IDEA-261595</a> ]</li> 
 <li>修复了<em>资源包编辑器</em>插件，以便不再错误地覆盖条目。[ <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-285267" target="_blank">IDEA-285267</a> ]</li> 
 <li>修复了在 Windows 终端中使用时的<em>Ctrl+箭头快捷键。</em>插入符号现在按预期跳转到下一个单词。[ <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-283023" target="_blank">IDEA-283023</a> ]</li> 
 <li>修复了将不同位置的资源文件粘贴到源文件夹的问题。[ <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-204744" target="_blank">IDEA-204744</a> ]</li> 
</ul> 
<p style="margin-left:0px">有关更多已解决的问题，请参阅<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Farticles%2FIDEA-A-111%2FIntelliJ-IDEA-2021.3.3-%28213.7172.25-build%29-Release-Notes" target="_blank">发行说明</a>。</p>
