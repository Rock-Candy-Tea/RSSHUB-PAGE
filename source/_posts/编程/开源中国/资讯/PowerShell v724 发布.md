                                                                                            <p style="color:#333333; margin-left:0; margin-right:0; text-align:left">PowerShell Core 是一个跨平台（Windows、Linux 和 macOS）的自动化和配置工具 / 框架，可以很好地与你现有的工具配合，并为处理结构化数据（如 JSON、CSV、XML 等）、REST API 和对象模型而优化。它包括一个命令行 Shell、一种相关的脚本语言和一个处理 cmdlets 的框架。</p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">PowerShell v7.2.4 发布，更新内容如下：</p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:start"><strong>Build and Packaging Improvements</strong></p> 
<ul> 
 <li>为 Ubuntu22.04 Jammy 添加映射 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FPowerShell%2FPowerShell%2Fpull%2F17317" target="_blank">#17317</a> )</li> 
 <li>更新使用<code>mcr.microsoft.com</code>( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FPowerShell%2FPowerShell%2Fpull%2F17272" target="_blank">#17272</a> )</li> 
 <li>更新第三方通知</li> 
 <li>更新 global.json 和 wix</li> 
 <li>将安全的供应链分析放在正确的位置 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FPowerShell%2FPowerShell%2Fpull%2F17273" target="_blank">#17273</a> )</li> 
 <li>修复 web cmdlet，使空<code>Get</code>不包含<code>content-length</code>header ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FPowerShell%2FPowerShell%2Fpull%2F16587" target="_blank">#16587</a> )</li> 
 <li>为 Ubuntu 更新软件包 fallback list（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FPowerShell%2FPowerShell%2Fpull%2F17217" target="_blank">#17217</a>）</li> 
 <li>将<code>sha256</code>摘要添加到 RPM 包 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FPowerShell%2FPowerShell%2Fpull%2F17215" target="_blank">#17215</a> )</li> 
 <li>允许多次安装 dotnet。( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FPowerShell%2FPowerShell%2Fpull%2F17216" target="_blank">#17216</a> )</li> 
</ul>
