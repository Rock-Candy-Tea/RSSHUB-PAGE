<div class="content">
                                                                    
                                                        <p>JetBrains 日前发布了 IntelliJ IDEA 2022.1 系列的首个小幅更新版本，该版本主要修复了之前存在的各种错误，以下是这次更新中一些重要的内容：</p> 
<ul> 
 <li>JAX-WS 操作再次可以从 Tools 菜单中使用 [<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-293502%2F" target="_blank">IDEA-293502</a>]</li> 
 <li>缺失的 Swagger UI 预览已被修复 [<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-291069%2F" target="_blank">IDEA-291069</a>]</li> 
 <li>修复了对有许多工件的项目造成长时间索引和 UI 冻结的问题 [<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-279542%2F" target="_blank">IDEA-279542</a>]</li> 
 <li>修正了拖放标签到不同的编辑器窗口时，按字母顺序排序标签被禁用的问题 [<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-283610%2F" target="_blank">IDEA-283610</a>]</li> 
 <li>修正了在处理大型 JSON 文件时导致内存占用率高和性能下降的问题 [<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-291190%2F" target="_blank">IDEA-291190</a>]</li> 
 <li>修正了在「设置-首选项-工具-任务-服务器」中配置的服务器列表中的项目丢失的问题 [<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-292023%2F" target="_blank">IDEA-292023</a>]</li> 
 <li>修正了导致项目位置被错误地报告为已被占用的问题，这干扰了新 Spring 模块的创建 [<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-292362%2F" target="_blank">IDEA-292362</a>]</li> 
 <li>当在浅色和深色主题之间切换时，Find 操作文本字段的背景颜色会相应改变。 [<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-290491%2F" target="_blank">IDEA-290491</a>]</li> 
 <li>修正了提交对话框中的按操作分组的问题，以便保留以前使用的设置 [<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-289528%2F" target="_blank">IDEA-289528</a>]</li> 
 <li>修正了在使用 JDK 8 的项目中运行 JUnit5 单元测试时出现的问题 [<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-292483%2FUnsupportedClassVersionError-when-trying-to-run-JUnit5-unit-test" target="_blank">IDEA-292483</a>]</li> 
 <li>修正了从项目结构（Project Structure）对话框调用 Maven 项目时的导入问题 [<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-283563%2FImport-Maven-project-from-Project-Structure-dialog-fails-with-Cl" target="_blank">IDEA-283563</a>]</li> 
</ul> 
<p>更多详情可查看：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fblog.jetbrains.com%2Fidea%2F2022%2F05%2Fintellij-idea-2022-1-1%2F" target="_blank">https://blog.jetbrains.com/idea/2022/05/intellij-idea-2022-1-1/</a></p>
