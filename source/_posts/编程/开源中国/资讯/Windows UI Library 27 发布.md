 <p>WinUI 2.7 现已发布。WinUI 是适用于 UWP 应用程序（以及使用 XAML 的桌面应用程序）的 Windows UI 库。</p> 
 <h4>InfoBadge</h4> 
 <p>标记是一种非侵入性且直观的方式，用于指示通知、显示警报、突出显示新内容或将焦点吸引到应用程序内的某个区域。InfoBadge 是一小块 UI，可以添加到应用程序中并进行自定义以显示数字、图标或简单的点。</p> 
 <p><img alt height="160" src="https://oscimg.oschina.net/oscnet/up-dd5edbae8e789b41d10626bc38b900320de.png" width="596" referrerpolicy="no-referrer"></p> 
 <p>InfoBadge 内置于 NavigationView 控件中，它还可以指定为 XAML 树中的独立元素，让开发者可以将 InfoBadge 放入其选择的任何控件或 UI 部分。</p> 
 <p><img alt height="145" src="https://oscimg.oschina.net/oscnet/up-9257a03dacea951e2cb0cb27928a08a7430.png" width="269" referrerpolicy="no-referrer"></p> 
 <h4>Horizontal Orientation in ColorPicker</h4> 
 <p>使用 ColorPicker 控件的 Orientation 属性指定编辑控件相对于色谱是垂直对齐还是水平对齐。</p>
