                                                                                            <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">Q4OS 是基于 Debian 的桌面 Linux 发行版，使用 KDE 4 的分支 Trinity 和 KDE Plasma 作为桌面环境。Q4OS 提供了简单的附件，并提供稳定的应用程序编程接口以支持复杂的第三方应用软件，例如谷歌 Chrome、VirtualBox 以及其他开发工具等。该系统对于虚拟云环境而言很有用，这得益于其甚低的硬件要求。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">Q4OS 4.10 Gemini 稳定版近日正式发布，更新内容如下：</p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>获得了 Debian Bullseye 11.4 更新，带来了更加稳定的 Debian 内核更新以及重要的安全和错误修复</li> 
 <li>Q4OS 还将 Trinity 桌面环境的最新版本也纳入了 4.10 中，因此 Q4OS 现在具有最新的 Trinity 14.0.12 
  <ul style="margin-left:0; margin-right:0"> 
   <li>新的应用程序： 
    <ul style="margin-left:0; margin-right:0"> 
     <li>Polkit-agent-tde：一个 D-Bus 服务，提供一个基于 TDE 的 polkit 认证代理</li> 
     <li>Polkit-tqt：一个让开发者通过一个 TQt 样式的接口使用 PolicyKit API 的库</li> 
     <li>tdemarkdown part：一个可嵌入的轻量级 markdown 文档查看器</li> 
    </ul> </li> 
   <li>增强： 
    <ul style="margin-left:0; margin-right:0"> 
     <li>对 Konsole 终端模拟的一些改进</li> 
     <li>Quanta：改进了对 VPL 编辑器中复杂字符的支持</li> 
     <li>KSSL：增加了对 Let's Encrypt 证书的支持</li> 
     <li>Kxkb：系统托盘标签现在可以有一个透明的背景</li> 
     <li>Konsole：增加了真正的透明度复选框选项</li> 
     <li>Quanta：增加了对 HTML 5 的支持</li> 
     <li>Sip4-tqt: 开始支持 Python 3</li> 
     <li>Tdebase：增加了对 dispwin（Argyll）的支持，以设置 ICC 配置文件</li> 
     <li>一些软件包已迁移到 CMake 构建系统中</li> 
     <li>放弃了各种软件包的 automake 构建系统</li> 
     <li>继续打磨整体外观</li> 
    </ul> </li> 
  </ul> </li> 
</ul>
