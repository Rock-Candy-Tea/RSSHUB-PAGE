                                                                                            <p>React Native 0.70.1 现已发布，具体更新内容如下：</p> 
<h4><span><span><span><span><span style="color:#24292f"><span><span><span><span><span><span><span><span><span><span><span style="background-color:#ffffff"><span><span><span>Added</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></h4> 
<ul> 
 <li>为 HermesExecutorFactory 添加更多调试设置</li> 
 <li>为 turbo 模块支持 TypeScript 数组类型（仅限组件）</li> 
</ul> 
<h4><span><span><span><span><span style="color:#24292f"><span><span><span><span><span><span><span><span><span><span><span style="background-color:#ffffff"><span><span><span>Changed</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></h4> 
<ul> 
 <li>接受 TypeScript 类型<code>T | null | undefined</code>作为 turbo 模块中 T 的可能类型</li> 
 <li>将 react-native-gradle-plugin 升级到 0.70.3</li> 
 <li>将 react-native-codegen 升级到 0.70.5</li> 
 <li>Hermes 版本升级</li> 
</ul> 
<h4><span><span><span><span><span style="color:#24292f"><span><span><span><span><span><span><span><span><span><span><span style="background-color:#ffffff"><span><span><span>Fixed</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></h4> 
<ul> 
 <li>修复 hermes profiler</li> 
</ul> 
<p style="text-align:start"><strong><span><span><span><span><span style="color:#24292f"><span><span><span><span><span><span><span><span><span><span><span style="background-color:#ffffff"><span><span><span>Android specific</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></strong></p> 
<ul> 
 <li>在 borderColor 中支持 PlatformColor</li> 
 <li>如果 webview 被禁用，避免 ForwardingCookieHandler 崩溃</li> 
 <li>使用 FindClass(..) 正确解析类</li> 
</ul> 
<p style="text-align:start"><span><span><span><span><span style="color:#24292f"><span><span><span><span><span><span><span><span><span><span><span style="background-color:#ffffff"><span><span><span><strong>iOS specifi</strong>c</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></p> 
<ul> 
 <li>启用“Prefer Cross-Fade Transitions”时修复 KeyboardAvoidingView height</li> 
 <li>修复 React 模块在新架构模式下与 swift 集成的构建错误</li> 
 <li>修复 ios pod 安装错误</li> 
</ul>
