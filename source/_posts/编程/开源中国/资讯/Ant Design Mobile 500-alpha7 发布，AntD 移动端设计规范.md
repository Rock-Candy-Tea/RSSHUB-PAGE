                                                                                            <p>Ant Design Mobile 5.0.0-alpha.7 发布了。Ant Design Mobile 即 Ant Design 移动端设计规范，<code>antd-mobile</code> 是 Ant Design 的移动规范的 React 实现，服务于蚂蚁及口碑无线业务。</p> 
<p>更新内容如下：</p> 
<ul> 
 <li>修复 
  <ul> 
   <li>修复了高分屏适配计算逻辑有误的 bug</li> 
   <li>Badge 组件之前的 <code>offest</code> 属性拼写有误，修改为 <code>offset</code> </li> 
  </ul> </li> 
</ul> 
<p>迁移建议</p> 
<ul> 
 <li>如果有用到 Badge 的 <code>offest</code> 属性，需要同步调整一下拼写</li> 
</ul>
