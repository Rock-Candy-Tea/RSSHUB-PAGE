                                                                                            <p>AppWeb 8.2.4 现已发布，此版本被划分为“建议升级，但不是必须的”。Appweb是一个嵌入式HTTP Web服务器，主要的设计思路是安全。这是直接集成到客户的应用和设备，便于开发和部署基于Web的应用程序和设备。</p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">具体更新内容：</p> 
<ul> 
 <li>修复了代理处理程序对 Connection header 的处理</li> 
 <li>修复了在 Windows 上使用 ATOMIC APIs 的构建</li> 
 <li>修复了在 Windows 上不使用 fast/proxy 的构建</li> 
 <li>修复一些小的 Windows 编译器警告</li> 
</ul>
