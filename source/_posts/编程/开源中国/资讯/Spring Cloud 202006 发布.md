                                                                                            <p>Spring Cloud 2020.0.6 已发布。这主要是一个修复错误和升级依赖项的版本，包括从<code>2021.0.x</code>版本开始的各种问题修复的反向移植，还向后移植了一些增强功能。</p> 
<p>更新内容：</p> 
<p style="text-align:start"><strong>Spring Cloud Commons</strong></p> 
<ul> 
 <li><span style="color:#191e1e">向后移植来自</span><code>2021.0.x</code>的错误修复</li> 
 <li><span style="color:#191e1e">将请求数据功能反向移植到 LoadBalancerLifecycle bean 以实现阻塞 Spring Cloud LoadBalancer </span>(<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-cloud%2Fspring-cloud-commons%2Fissues%2F1090" target="_blank">#1090</a>)</li> 
</ul> 
<p style="text-align:start"><strong>Spring Cloud OpenFeign</strong></p> 
<ul> 
 <li>向后移植来自<code>2021.0.x</code>的错误修复</li> 
</ul> 
<p style="text-align:start"><strong>Spring Cloud Netflix</strong></p> 
<ul> 
 <li>向后移植来自<code>2021.0.x</code>的错误修复</li> 
</ul>
