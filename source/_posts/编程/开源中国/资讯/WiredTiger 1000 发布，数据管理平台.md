                                                        <p>WiredTiger 是一个高性能、可扩展、生产质量、NoSQL、开源可扩展的数据管理平台。</p> 
<p>WiredTiger 10.0.0 正式发布，此次更新内容如下：</p> 
<ul> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fjira.mongodb.org%2Fbrowse%2FWT-4008" target="_blank">WT-4008</a> 增加 ARM NEON 对行搜索操作的支持；</li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fjira.mongodb.org%2Fbrowse%2FWT-5083" target="_blank">WT-5083</a> 增加 mips64el 支持；</li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fjira.mongodb.org%2Fbrowse%2FWT-6598" target="_blank">WT-6598</a> 增加新的 API，允许改变 dhandle hash bucket 的大小；</li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fjira.mongodb.org%2Fbrowse%2FWT-4427" target="_blank">WT-4427</a> 使 WiredTiger 的时间戳始终为 8 字节；</li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fjira.mongodb.org%2Fbrowse%2FWT-4642" target="_blank">WT-4642</a> 持久存储事务 ID；</li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fjira.mongodb.org%2Fbrowse%2FWT-5225" target="_blank">WT-5225</a> 为历史存储创建持久性文件；</li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fjira.mongodb.org%2Fbrowse%2FWT-5500" target="_blank">WT-5500</a> 实施新的历史存储格式；</li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fjira.mongodb.org%2Fbrowse%2FWT-6677" target="_blank">WT-6677</a> 将已承诺/未承诺的读取隔离映射为只读事务；</li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fjira.mongodb.org%2Fbrowse%2FWT-6710" target="_blank">WT-6710</a> 将默认事务隔离改为快照；</li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fjira.mongodb.org%2Fbrowse%2FWT-4833" target="_blank">WT-4833</a> 取消对 Python2 的支持；</li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fjira.mongodb.org%2Fbrowse%2FWT-6410" target="_blank">WT-6410</a> 删除 WT_SESSION.rebalance；</li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fjira.mongodb.org%2Fbrowse%2FWT-6528" target="_blank">WT-6528</a> 删除 WiredTiger API 和源码中的攻击性术语；</li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fjira.mongodb.org%2Fbrowse%2FWT-6674" target="_blank">WT-6674</a> 删除 Async API 代码和文档；</li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fjira.mongodb.org%2Fbrowse%2FWT-6675" target="_blank">WT-6675</a> 删除 WiredTiger 的 Java 语言 API 和文档；</li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fjira.mongodb.org%2Fbrowse%2FWT-6678" target="_blank">WT-6678</a> 取消对密钥的 Huffman 编码支持；</li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fjira.mongodb.org%2Fbrowse%2FWT-6713" target="_blank">WT-6713</a> 取消对自定义数据源的事务支持；</li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fjira.mongodb.org%2Fbrowse%2FWT-7295" target="_blank">WT-7295</a> 与旧版 WT 的兼容性；</li> 
</ul>
