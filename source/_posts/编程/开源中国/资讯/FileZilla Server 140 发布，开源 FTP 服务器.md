                                                                                            <p>FileZilla Server 是一个免费开源的 FTP 和 FTPS 服务器，能够提供与服务器的安全加密连接。Filezilla Server 没有实现对 SFTP（SSH 文件传输协议）的支持。</p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">FileZilla Server 1.4.0 现已发布，更新内容如下：</p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><strong>Bugfixes and minor changes</strong></p> 
<ul> 
 <li>Debian：默认情况下，该服务现在配置为从日志行中排除标题，因为 journald 已经输出了自己的标题。</li> 
 <li>Mac：修复了导致已安装服务无法启动的回归</li> 
 <li>Admin UI：更改了日志记录设置中的措辞</li> 
 <li>Admin UI：使默认连接值正确，以防设置文件丢失。</li> 
 <li>MSW：修复了 ExecDos 插件中的各种错误，这些错误可能导致管理员密码设置不正确。</li> 
 <li>MSW：现在可以正确使用 non-ASCII admin 密码。</li> 
 <li>MSW：修复了 libfilezilla 中的一个 bug，该 bug 在创建访问受限的目录时导致意外失败。它对 ACME 帐户的创建产生了影响。</li> 
</ul> 
<p>更新说明：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Ffilezilla-project.org%2F" target="_blank">https://filezilla-project.org/</a></p>
