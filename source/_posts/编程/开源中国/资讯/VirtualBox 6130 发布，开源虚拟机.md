                                                        <p>VirtualBox 是一款功能强大的虚拟机软件，它不仅具有丰富的特色，而且性能也很优异。VirtualBox 6.1.30 是一个维护版本，修复和/或增加了以下内容：</p> 
<ul> 
 <li>VMM：修正了 6.1.28 版本的回归问题，即当在 Windows 10 上使用 Hyper-V 模式时无法启动虚拟机。</li> 
 <li>GUI：修正了浏览外部镜像后无法完成首次运行向导的问题。</li> 
 <li>GUI：修正了在 macOS Big Sur 上从首次运行向导浏览外部镜像时的崩溃问题。</li> 
 <li>GUI：修正了在 Windows 上无法将拍摄的屏幕截图保存在具有本地名称的文件夹下的问题。</li> 
 <li>GUI：修正了 X11 上的错误，即在虚拟机存储设置中单次点击鼠标时启动拖放功能。</li> 
 <li>GUI：修正了对不支持硬件虚拟化的机器的设置检查。</li> 
 <li>GUI：非关键媒体相关的错误不应导致模态弹出错误信息。</li> 
 <li>修正了解析 /etc/vbox/networks.conf 时的崩溃问题。</li> 
 <li>DVD：修复了虚拟机重置时的驱动器锁处理</li> 
 <li>VBoxHeadless：修正了在 macOS Monterey 上运行时的崩溃。</li> 
 <li>VBoxManage：修正了 "hostonlyif" 的不正确帮助文本</li> 
 <li>vboximg-mount：如果没有指定镜像，会出现错误信息。</li> 
 <li>macOS：修复了 macOS Monterey 在安装程序和启动内核扩展时的多个特定 bug。</li> 
 <li>macOS：在基于 M1 的 Mac 上显示不支持 CPU 的信息，并中止安装。</li> 
 <li>Linux：引入了对内核 5.16 的初步支持。</li> 
 <li>共享剪贴板：当 Guest 没有剪贴板数据需要报告时，改进了 Host 和 Guest 之间的通信。</li> 
</ul> 
<p>更多详情可查看：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.virtualbox.org%2Fwiki%2FChangelog" target="_blank">https://www.virtualbox.org/wiki/Changelog</a></p>
