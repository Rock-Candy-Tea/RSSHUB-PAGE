                                                                                            <p>Ant Design 4.20.6 现已发布，主要变化如下：</p> 
<ul> 
 <li>修复 DatePicker 初次打开时 placeholder 闪烁的问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F35620" target="_blank">#35620</a></li> 
 <li>移除 Grid 默认 <code>role</code> 标签，以使其更好的适配 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Faccessibilityinsights.io%2Finfo-examples%2Fweb%2Faria-required-parent%2F" target="_blank">aria-required-parent</a> 要求。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F35616" target="_blank">#35616</a></li> 
 <li>修复 Anchor 在某些游览器下会被切割内容的问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F35612" target="_blank">#35612</a></li> 
 <li>修复 Table 存在表头分组和垂直滚动条时表头边框异常的问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F35591" target="_blank">#35591</a></li> 
 <li>修复 Drawer 内按钮关闭速度过快问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F35339" target="_blank">#35339</a></li> 
</ul>
