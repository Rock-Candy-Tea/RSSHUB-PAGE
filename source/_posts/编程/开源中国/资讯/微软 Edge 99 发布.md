                                                                                            <p>微软 Edge 99 现已在所有支持的桌面平台上正式推出，这是该浏览器在 100 版本里程碑之前的最后一个主要稳定版本，该版本更新内容如下：</p> 
<ul> 
 <li>这个版本最大的特点是能够设置一个自定义的主密码来保护你保存的密码，有助于防止未经授权的用户使用保存的密码登录网站；</li> 
 <li>可以使用 PasswordManagerBlocklist 策略配置 Microsoft Edge 应禁用密码管理器的域列表，从而确保这些网站的密码无法保存或自动填写到 Web 表单中；</li> 
 <li>为浏览 PDF 文件新增了一个页面缩略图功能，现在可以使用代表页面的缩略图浏览 PDF 文档，这些缩略图将出现在 PDF 阅读器左侧的窗格中；</li> 
 <li>Edge 99 的最后一个功能是能够通过网站的配置文件偏好来实现多配置的个性化体验。浏览器现在可以让用户创建一个自定义的网站列表，以便在浏览对应网站时自动切换配置文件；</li> 
 <li>网站所有者可以通过在 <code>edge://flags</code> 中启用 <code>#force-major-version-to-100</code> 实验标志以确保 3 位数的 User-Agent 解析逻辑能按预期工作；</li> 
</ul> 
<p>用户可以访问 Edge 的设置菜单，点击帮助和反馈，选择关于 Microsoft Edge，就能完成下载和更新。</p>
