                                                        <p>RetroArch 1.9.8 现已发布。RetroArch 是款功能强大的跨平台模拟器，不但能够模拟许多不同的游戏主机，还能在 Windows、MacOS、Linux、Android、iOS 以及多种游戏主机上执行。</p> 
<p>主要更新内容如下：</p> 
<ul> 
 <li>AUDIO/WINDOWS/WASAPI：停止在快进时停用音频</li> 
 <li>CHEEVOS：重置时隐藏 challenge indicators</li> 
 <li>CHEEVOS：支持超过 64 个内存区域</li> 
 <li>CHEEVOS：自动重试“http error code -1”</li> 
 <li>CONTENT INFORMATION：始终显示内容信息标签+路径行</li> 
 <li>CORE OPTIONS：实施的核心选项类别</li> 
 <li>CORE OPTIONS：添加选项以禁用核心选项类别</li> 
 <li>D3D10/11/12：修复 gfx_display_draw_texture - 修复 OSK（屏幕键盘）问题</li> 
 <li>DATABASE：修复获取 CRC 值时的堆缓冲区溢出</li> 
 <li>DATABASE/EXPLORE：修复探索菜单中的 CRC32 读取问题</li> 
 <li>DATABASE/LIBRETRODB：修复数值的写入</li> 
 <li>DATABASE/LIBRETRODB：修复 big endian 上的 libretro-db 加载</li> 
 <li>INPUT/SDL2/WINDOWS：修复 keyboard event keycodes</li> 
 <li>LIBNX/SWITCH：修复布局未正确应用并在拆分 Joycon 时挂起</li> 
 <li>LIBRETRO：实现了核心选项类别 API</li> 
 <li>LIBRETRO：修复RETRO_ENVIRONMENT_SET_FASTFORWARDING_OVERRIDE回调，当runahead被启用时。</li> 
 <li>MENU：将“Manage Playlists”重新定位到顶部</li> 
 <li>OSX：修复一些内存泄漏</li> 
 <li>WIIU：修复 L3/R3 按钮</li> 
 <li>WIIU：添加 ICInvalidateRange（对于 JIT 是必需的）</li> 
 <li>WIIU：轻微的文件系统优化</li> 
 <li>WIIU：添加在没有 core info 的情况下运行的选项（emscripten-style）</li> 
 <li>......</li> 
</ul>
