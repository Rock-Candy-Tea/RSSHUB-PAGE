                                                                                            <p>CherryTree 0.99.39 现已发布。CherryTree 是一个支持无限层级分类的笔记软件，Python 编写，支持富文本编辑和代码高亮，支持 Linux 和 Windows 平台。数据采用 sqlite 或 XML 存储，支持密码保护，支持从 NoteCase、KeepNote、Knowit、Tomboy、TuxCards、Treepad、Leo 等笔记软件导入数据。</p> 
<p><img alt height="608" src="https://static.oschina.net/uploads/space/2021/0722/064237_TNzL_2744687.png" width="888" referrerpolicy="no-referrer"></p> 
<p><img alt height="608" src="https://static.oschina.net/uploads/space/2021/0722/064244_aMlQ_2744687.png" width="888" referrerpolicy="no-referrer"></p> 
<p>此版本更新内容如下：</p> 
<ul> 
 <li>修复了菜单元素图标不总是可见的问题</li> 
 <li>修复了导出 PDF 中的文件链接在路径不相关时不起作用的问题</li> 
 <li>修复了在表格单元格中不起作用的 TODO 列表</li> 
 <li>修复了 Windows 上受密码保护的档案中的错误，密码中的反斜杠字符被替换为斜杠字符</li> 
 <li>修复了 Windows 上 p7zip 和文档路径中 non ascii 字符的问题 - p7zip 现在使用 glib</li> 
 <li>修复了与 fmt 8.0.0 的构建</li> 
 <li>当系统托盘和隐藏在系统托盘上的启动被启用时，只有在用户单击系统托盘图标以显示 cherrytree 后才会加载文档</li> 
 <li>修复了代码框纯文本错误地使用代码字体而不是纯文本字体的问题</li> 
 <li>修复了 ru.po 翻译中的错误</li> 
</ul>
