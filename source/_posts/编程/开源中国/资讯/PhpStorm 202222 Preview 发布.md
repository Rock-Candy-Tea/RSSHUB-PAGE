                                                                                            <p>PhpStorm 2022.2.2 Preview 现已<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fblog.jetbrains.com%2Fphpstorm%2F2022%2F09%2Fphpstorm-2022-2-2-preview%2F" target="_blank">发布</a>，带来了一些错误修复和 quality-of-life 改进。一些亮点更新内容如下：</p> 
<ul> 
 <li>新增：对 Rector 远程解释器的支持 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FWI-66504" target="_blank">WI-66504</a> )</li> 
 <li>修复：ESLint 8.23 fatal error ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FWEB-57089" target="_blank">WEB-57089</a> )</li> 
 <li>修复：VSC 路径下拉菜单中的控制 + 单击 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-298409" target="_blank">IDEA-298409</a> )</li> 
 <li>修复：拖放以进行多项选择（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-298747" target="_blank">IDEA-298747</a>）</li> 
 <li>修复：在 wsl 路径上运行 Rector ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FWI-67468" target="_blank">WI-67468</a> )</li> 
 <li>修复：Markdown 文件中的自动格式化表格 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-262735" target="_blank">IDEA-262735</a> )</li> 
 <li>修复：缺少 Codeception “reporters” 选项 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FWI-65867" target="_blank">WI-65867</a> )</li> 
 <li>修复：粘贴时的错误缩进 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-299336" target="_blank">IDEA-299336</a> )</li> 
 <li>修复：通过 HTTP 客户端向子域发送 cookie ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-230876" target="_blank">IDEA-230876</a> )</li> 
 <li>修复：缺少行注释选项 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-283781" target="_blank">IDEA-283781</a> )</li> 
 <li>修复：commit 时被卡住 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FWI-62509" target="_blank">WI-62509</a> )  </li> 
 <li>修复：Angular 项目的 Jest 测试失败 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FWEB-56303" target="_blank">WEB-56303</a> ) </li> 
</ul>
