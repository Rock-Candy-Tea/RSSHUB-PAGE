<div class="content">
                                                                                            <p>Files v2.3 发布了重要版本<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Ffiles.community%2Fblog%2Fposts%2Fintroducing-files-v2-3" target="_blank"> 2.3</a>。</p> 
<blockquote> 
 <p>Files 是 Windows 平台的<strong>第三方文件管理器</strong>，功能强大、设计优美且直观，包含多标签、窗格、文件预览、shell 扩展、云文件集成等功能。Files 是 2022 年 Microsoft Store App Awards 竞赛的获胜者，人气颇高。</p> 
</blockquote> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-82ec2b7457f3ccaf6e3f5957ec2542a9238.png" referrerpolicy="no-referrer"></p> 
<p>Files v2.3 主要变化：</p> 
<ul> 
 <li><strong>提供更紧凑的间距</strong></li> 
</ul> 
<p>紧凑间距能够让用户查看更多项目：</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-460b646f50c8d2f2e325e0e32bdd2ca66a3.png" referrerpolicy="no-referrer"></p> 
<ul> 
 <li><strong>文件冲突对话框 (Conflicts Dialog) 中支持批量操作</strong></li> 
</ul> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-beda87b294db2d36adb118557a39c05dcb5.png" referrerpolicy="no-referrer"></p> 
<ul> 
 <li><strong>通过双击手势返回上一级目录</strong></li> 
 <li><strong>重新设计选项卡控件，以连接选项卡内容</strong></li> 
</ul> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-de8f693be98b451c0157212ff7048343729.png" referrerpolicy="no-referrer"></p> 
<ul> 
 <li><strong>添加在侧边栏显示标签的选项</strong></li> 
</ul> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-41bef988051a5fbe8192340882906aff53e.png" referrerpolicy="no-referrer"></p> 
<p><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Ffiles.community%2Fblog%2Fposts%2Fintroducing-files-v2-3" target="_blank">点此查看详细更新说明</a>。</p> 
<p>官方还提到了 Files 的未来规划，他们将改进 Files 代码库的架构，加速向 Windows App SDK 的过渡，将消除当前应用模型中长期存在的限制，改进网络性能和可靠性。</p>
