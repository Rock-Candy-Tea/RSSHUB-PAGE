                                                        <p>Sublime Text 4 首个稳定版终于来了！此版本除了引入不少改进外，对于 Sublime Text 一直以来的长处也在加以完善。开发团队称新增的部分功能可显著优化使用者的工作流。</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-d5c9b6e7ac2176da7d1437abb30f2bd0901.png" referrerpolicy="no-referrer"></p> 
<p>更新亮点：</p> 
<ul> 
 <li>变更许可证使用条款<br> <em>Sublime Text License 密钥（注册码）不再与独立的主要版本绑定，而是对购买后 3 年内的所有更新均有效。注册码到期后，使用者仍可以使用购买注册码 3 年内发布的 Sublime Text 的每个版本。如果要使用更新的版本则需要升级 License。</em></li> 
 <li>支持多选项卡</li> 
 <li>支持 Apple Silicon 和 Linux ARM64</li> 
 <li>全新 UI</li> 
 <li>感知上下文的自动补全</li> 
 <li>支持 TypeScript, JSX 和 TSX</li> 
 <li>GPU 渲染</li> 
 <li>升级 Python API</li> 
</ul>
