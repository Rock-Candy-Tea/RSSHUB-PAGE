<h2 style="text-align:left">三、MateCloud 3.3.8快照版同步更新</h2> 
<h3>3.1　关于MateCloud</h3> 
<p><span style="background-color:#ffffff; color:#24292e">MateCloud是一款基于Spring Cloud Alibaba的微服务架构。目前已经整合Spring Boot 2.5.0、 Spring Cloud 2020、Spring Cloud Alibaba 2021、Spring Security Oauth2、Feign、Dubbo、JetCache、RocketMQ等服务套件，为您的开发保驾护航。</span></p> 
<h3>3.2　更新内容</h3> 
<ul> 
 <li>Spring Boot 升级至2.5.0</li> 
 <li>Spring Cloud 2020.2</li> 
 <li>Spring Cloud Alibaba 2021.1</li> 
 <li>Nacos 2.0.1</li> 
 <li>Sentinel 1.8.1</li> 
 <li>消息队列集成rocketmq和kafka，并提供基于最新的函数式编程的消费者模式。</li> 
</ul> 
<p>基于最新的技术栈，搭建的微服务的脚手架，紧跟技术最前沿。</p>
