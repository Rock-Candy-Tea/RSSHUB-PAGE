                                                                                            <p>Microsoft PowerToys 是 Windows 系统实用程序，供高级用户调整和简化其 Windows 体验，可最大限度地提高生产力。</p> 
<p>该版本主要集中在稳定性和功能改进方面，具体更新内容如下：</p> 
<h3>更新亮点</h3> 
<ul> 
 <li> <p>FancyZones 的区域限制已经增加到 128 个区域，之前的限制是 40</p> </li> 
 <li> <p>PowerToys Run 的时区转换插件</p> </li> 
 <li> <p>FancyZones 支持子窗口和弹出窗口</p> </li> 
 <li> <p>通过改变设置，Find my mouse 现在可以通过摇动鼠标来激活了</p> <p><img alt height="551" src="https://oscimg.oschina.net/oscnet/up-beed0d8e738cc1ed2dea14ab40201df573e.png" width="700" referrerpolicy="no-referrer"></p> </li> 
</ul> 
<h3>Always on Top</h3> 
<ul> 
 <li>修复了启用时多余的 GPU / CPU 使用率</li> 
 <li>如果边框有焦点，则无法通过 F4 关闭</li> 
 <li>更改边框大小应为现有窗口正确调整大小</li> 
 <li>边框随 Outlook 模态窗口消失</li> 
</ul> 
<h3>ColorPicker</h3> 
<ul> 
 <li>在更换主题时不再崩溃</li> 
</ul> 
<h3>FancyZones</h3> 
<ul> 
 <li>将区域限制从 40 增加到 128</li> 
 <li>FancyZones 支持的子窗口和弹出窗口</li> 
</ul> 
<h3>File explorer</h3> 
<ul> 
 <li>修复了 Dev 文件预览的问题 
  <ul> 
   <li>修复了对象引用未设置的问题</li> 
   <li>修复 UTF-8 编码的问题</li> 
   <li>修复文件正在使用中的问题</li> 
   <li>修复了提示文件太大的问题</li> 
  </ul> </li> 
</ul> 
<h3>Image Resizer</h3> 
<ul> 
 <li>修复 <code>invalid operation</code> 的错误</li> 
</ul> 
<h3>Mouse utility</h3> 
<ul> 
 <li>通过更改设置，Find my mouse 现在可以通过摇动鼠标来激活</li> 
 <li>Find my mouse 现在可以排除应用程序了</li> 
 <li>修复了由于抗锯齿导致的十字线厚度看起来很奇怪的问题</li> 
</ul> 
<h3>PowerRename</h3> 
<ul> 
 <li>不会再对文件名进行拼写检查了</li> 
 <li>自动完成的结果框不会遮住应用按钮</li> 
 <li>Regex 修复</li> 
 <li>……</li> 
</ul>
