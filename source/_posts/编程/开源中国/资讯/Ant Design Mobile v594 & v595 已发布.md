                                                                                            <p><span style="color:#333333">Ant Design Mobile v5.9.4 和 </span><span style="color:#000000">v5.9.5 </span><span style="color:#333333">发布了。Ant Design Mobile 即 Ant Design 移动端设计规范， 是 Ant Design 的移动规范的 React 实现，服务于蚂蚁及口碑无线业务。</span></p> 
<h3><span style="color:#24292f"><strong>v5.9.4</strong></span></h3> 
<ul> 
 <li>特性 
  <ul> 
   <li>Modal 和 Dialog 增加了 <code>disableBodyScroll</code> 属性 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design-mobile%2Fpull%2F5019" target="_blank">#5019</a></li> 
  </ul> </li> 
 <li>优化 
  <ul> 
   <li>Form.Item 的 <code>validateTrigger</code> 属性现在会默认和 <code>trigger</code> 属性保持一致了 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design-mobile%2Fpull%2F5010" target="_blank">#5010</a></li> 
   <li>移除了 <code>package.json</code> 中的 <code>exports</code> 字段 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design-mobile%2Fpull%2F5013" target="_blank">#5013</a></li> 
  </ul> </li> 
 <li>修复 
  <ul> 
   <li>调整了 umd 的打包逻辑，适配 stackblitz <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design-mobile%2Fpull%2F5016" target="_blank">#5016</a> </li> 
  </ul> </li> 
</ul> 
<p>更新公告：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design-mobile%2Freleases%2Ftag%2Fv5.9.4" target="_blank">https://github.com/ant-design/ant-design-mobile/releases/tag/v5.9.4</a> </p> 
<h3><span style="color:#24292f"><strong>v5.9.5</strong></span></h3>

  </ul> </li> 
 <li>其他 
  <ul> 
   <li>文档站点视觉优化</li> 
  </ul> </li> 
</ul>
