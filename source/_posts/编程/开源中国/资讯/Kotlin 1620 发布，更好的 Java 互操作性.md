                                                        <p>Kotlin 1.6.20 已经正式发布。它提供了对未来语言功能的预览，使分层结构成为多平台项目的默认结构，并为 JVM、JS 和 Native 平台带来了性能改进。</p> 
<p>Kotlin 1.6.20 具体更新内容如下：</p> 
<ul> 
 <li>Kotlin/JVM 的上下文接收器原型为定义上下文相关声明提供了新的支持</li> 
 <li>由于绝对不可为空的类型，与通用 Java 类和接口有了更好的互操作性</li> 
 <li>由于在 JVM IR 后端对单个模块进行了并行编译，因此构建时间更快</li> 
 <li>通过 Kotlin/JS IR 中的增量编译模式，简化开发体验</li> 
 <li>Kotlin/Native 的性能改进</li> 
 <li>由于多平台项目的分层结构，更容易分享代码</li> 
</ul>
