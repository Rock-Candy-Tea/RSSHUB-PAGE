<div class="content">
                                                                                            <p style="text-align:left">electron-egg 是一个简单、快速、功能丰富的跨平台桌面软件开发框架，使用node.js编写。</p> 
<h4 style="text-align:left">特性</h4> 
<div style="text-align:left"> 
 <div>
  <span style="color:#0451a5">1.</span> 
  <span style="color:#000000"> 跨平台：一套代码，可以打包成windows版、Mac版、Linux版或者以web网站运行</span>
 </div> 
 <div>
  <span style="color:#0451a5">2.</span> 
  <span style="color:#000000"> 简单高效：支持vue、react、ejs等</span>
 </div> 
 <div>
  <span style="color:#0451a5">3.</span> 
  <span style="color:#000000"> 工程化：可以用服务端的开发思维，来编写桌面软件</span>
 </div> 
 <div>
  <span style="color:#0451a5">4.</span> 
  <span style="color:#000000"> 高性能：可启动多个工作进程协同工作</span>
 </div> 
 <div>
  <span style="color:#0451a5">5.</span> 
  <span style="color:#000000"> 功能丰富：服务端的技术场景都可以使用，如：路由、中间件、控制器、服务、定时任务、队列、插件等</span>
 </div> 
 <div>
  <span style="color:#0451a5">6.</span> 
  <span style="color:#000000"> 功能demo：桌面软件常见功能，后续逐步集成并完善或提供demo</span>
 </div> 
 <div>
  <span style="color:#0451a5">7.</span> 
  <span style="color:#000000"> 自动更新</span>
 </div> 
</div> 
<h4 style="text-align:left">本次更新：</h4> 
<div style="text-align:left"> 
 <div> 
  <div>
   <span style="color:#0451a5">1.</span>
   <span style="color:#000000"> 自动升级支持多平台更新</span>
  </div> 
  <div>
   <span style="color:#0451a5">2.</span>
   <span style="color:#000000"> 支持将任意网站打包成桌面软件</span>
  </div> 
  <div>
   <span style="color:#0451a5">3.</span>
   <span style="color:#000000"> 优化启动时白屏问题</span>
  </div> 
  <div>
   <span style="color:#0451a5">4.</span>
   <span style="color:#000000"> 安全检查，防止inspect debug</span>
  </div> 
  <div>
   <span style="color:#0451a5">5.</span>
   <span style="color:#000000"> 部分代码优化</span>
  </div>
