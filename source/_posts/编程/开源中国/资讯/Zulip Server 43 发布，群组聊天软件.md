                                                        <p>Zulip Server 4.3 发布了。Zulip 是一个强大的开源群组聊天软件，采用 Python 编写，使用 Django 框架，支持通过会话流的私人消息和群聊。Zulip 还支持快速搜索、拖放文件上传、图像预览、组私人消息、可听通知、错过电子邮件消息提醒与桌面应用等。</p> 
<p>具体更新内容包括：</p> 
<ul> 
 <li>修复了在升级旧服务器时，将<code>JITSI_SERVER_URL</code>设置为<code>None</code>以禁用 Jitsi 的异常。</li> 
 <li>修复了服务器未配置 GIPHY API 密钥时出现的 GIPHY 集成下拉列表。</li> 
 <li>不再为不积极使用 GIPHY 集成的用户加载 GIPHY API 库。</li> 
 <li>改进了 Grafana 集成的格式。</li> 
 <li>修复了 Dropbox 图像链接的预览。</li> 
 <li>修复了在 non-S3 上传后端存储头像/表情符号的支持。</li> 
 <li>修复了 code playgrounds 过于严格的数据库约束。</li> 
 <li>标记的用户状态字符串以供翻译。</li> 
 <li>更新了 Transifex 的翻译数据。</li> 
</ul>
