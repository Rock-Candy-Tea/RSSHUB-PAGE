                                                        <p>Spring HATEOAS 1.2.6 & 1.3.1 现已发布。Spring HATEOAS 是一个用于实现 REST Web 服务的开发库，它提供了一些 API，以便在使用 Spring，特别是 Spring MVC 时可以轻松创建遵循 HATEOAS 原则的 REST 表述，其试图解决的核心问题是链接的创建和表述组装。</p> 
<p><strong>主要更新内容</strong></p> 
<ul> 
 <li>1.2.6 
  <ul> 
   <li>为兼容 kotlin 1.5.0 添加构建配置文件</li> 
   <li>升级到 Spring 5.3.7</li> 
   <li>HAL 序列化异常时，对象映射器配置为按键对 map 进行排序</li> 
  </ul> </li> 
 <li>1.3.1 
  <ul> 
   <li>升级到 Spring 5.3.7</li> 
   <li>为兼容 kotlin 1.5.0 添加构建配置文件</li> 
   <li>公开  WebConverters </li> 
   <li>为复选框、单选和文件提供额外的 HtmlInputType</li> 
   <li>在没有启用默认媒体类型的情况下无法启动 HATEOAS</li> 
   <li>处理 PropertyNamingStrategyBase 的弃用问题</li> 
   <li>HAL 序列化异常时，对象映射器配置为按键对 map 进行排序</li> 
  </ul> </li> 
</ul>
