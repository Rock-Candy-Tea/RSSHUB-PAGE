                                                        <p>Git for Windows 2.33.0-rc0 现已发布，主要更新内容如下：</p> 
<p><strong>New Features</strong></p> 
<ul> 
 <li>随附 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fgit%2Fgit%2Fblob%2Fv2.33.0-rc0%2FDocumentation%2FRelNotes%2F2.33.0.txt" target="_blank">Git v2.33.0-rc0</a></li> 
 <li>随附 <a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fsearch.cpan.org%2Fdist%2Fperl-5.34.0%2Fpod%2Fperldelta.pod" target="_blank">Perl v5.34.0</a>（以及一些更新的 Perl 模块）</li> 
</ul> 
<p><strong>Bug 修复</strong></p> 
<ul> 
 <li>在选中 Windows 终端配置文件选项时，安装程序在升级时不再<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fgit-for-windows%2Fgit%2Fissues%2F3312" target="_blank">显示错误对话框</a>。</li> 
</ul>
