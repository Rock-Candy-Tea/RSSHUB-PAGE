                                                                                            <h3 style="margin-left:0em; margin-right:0em; text-align:left">橙单开源</h3> 
<p style="color:#40485b; margin-left:0; margin-right:0; text-align:left"><strong><em>橙单已完整开源全部脚手架代码，同时提供面向不同技术栈和架构场景的工程样例。与此同时，我们还会提供非常详尽、专业、完全免费的线上开发文档。希望通过我们共同的努力，让每一位对技术有兴趣的开发者，可以轻松迈过这个小小的门槛</em></strong>。 如果橙单对您确有帮助，Fork 的同时，也请 Star 一下。</p> 
<blockquote> 
 <p style="margin-left:0; margin-right:0">再次确认你没看错，极高质量和成熟度的脚手架代码全部开源，18万字的线上文档也完全开放，包括完整的工作流、在线表单和数据过滤等组件 (<span> </span><strong>同类竞品售价几近万元</strong><span> </span>)，橙单全部源码可见，没有一丝遮蔽。</p> 
</blockquote> 
<h3 style="margin-left:0; margin-right:0; text-align:start"><span>新功能</span></h3> 
<ul> 
 <li> <p style="margin-left:.5rem; margin-right:0"><span>在线表单，支持主表数据的批量删除。</span></p> </li> 
 <li> <p style="margin-left:.5rem; margin-right:0"><span>在线表单，支持一对多从表数据的批量删除。</span></p> </li> 
 <li> <p style="margin-left:.5rem; margin-right:0"><span>在线表单，支持批量导出，配置时可指定主表、一对一从表和一对多从表聚合计算等导出字段。</span></p> </li> 
 <li> <p style="margin-left:.5rem; margin-right:0"><span>在线表单，工作流数据添加接口支持一对一关联数据的级联插入。</span></p> </li> 
 <li> <p style="margin-left:.5rem; margin-right:0"><span>在线表单，工作流数据更新接口支持一对一关联数据的级联更新。</span></p> </li> 
 <li> <p style="margin-left:.5rem; margin-right:0"><span>基础功能，升级工作流、在线表单、静态表单等所有模块，提供 PostgreSQL 的支持。</span></p> </li> 
 <li> <p style="margin-left:.5rem; margin-right:0"><span>基础功能，升级 Spring Cloud 版本至 2020.0.4。</span></p> </li> 
 <li> <p style="margin-left:.5rem; margin-right:0"><span>基础功能，升级 Spring Boot 版本至 2.5.8。</span></p> </li> 
 <li> <p style="margin-left:.5rem; margin-right:0"><span>基础功能，基于远程过程调用的数据查询，支持多字段联合的模糊搜索过滤。</span></p> </li> 
 <li> <p style="margin-left:.5rem; margin-right:0"><span>生成器，支持表单组件配置的批量添加。</span></p> </li> 
 <li> <p style="margin-left:.5rem; margin-right:0"><span>生成器，支持批量指定，前端表单、后台接口、工作流和定时任务的独立生成，持续提升代码合并的效率。</span></p> </li> 
 <li> <p style="margin-left:.5rem; margin-right:0"><span style="background-color:#ffffff; color:#40485b">线上文档，根据用户的反馈，修复部分不同步的章节，同时新增生产部署、数据库迁移、Spring版本升级等实用部分</span></p> </li> 
</ul> 
<h3 style="margin-left:0; margin-right:0; text-align:start"><span>其他更新</span></h3> 
<ul> 
 <li> <p style="margin-left:.5rem; margin-right:0"><span>提供微服务版本 Spring Cloud 和 单体版本 Spring Boot 升级在线文档，便于用户升级。</span></p> </li> 
 <li> <p style="margin-left:.5rem; margin-right:0"><span>直接升级 common-core、common-online、common-flow、common-log、common-online-api </span>模块。</p> </li> 
</ul>
