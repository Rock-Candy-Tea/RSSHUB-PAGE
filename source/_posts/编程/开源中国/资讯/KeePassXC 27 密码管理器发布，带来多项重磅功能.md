                                                        <p><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fkeepassxc.org%2F" target="_blank">KeePassXC</a>  是由 KeePass 密码管理器的一个分支发展而来的，相比原版 KeePass，KeePassXC 除了适配 Windows，还适配了 Linux 和 macOS。</p> 
<p>通过 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fkeepassxc.org%2F" target="_blank">KeePassXC</a> 可以存储和管理个人的敏感信息。保存在 KeePassXC 上的信息全部都会进行离线加密，借助于云存储服务可以实现跨平台同步。</p> 
<p>KeePassXC 2.7.0 发布，本次更新内容如下：</p> 
<h3>主要新增内容</h3> 
<ul> 
 <li>实施 KDBX 4.1</li> 
 <li>为云存储和 GVFS 添加直接写入保存选项</li> 
 <li>可阻止在 Windows 和 macOS 上进行屏幕截图</li> 
 <li>支持使用 Windows Hello 快速解锁</li> 
 <li>支持使用 Touch ID 快速解锁</li> 
 <li>支持使用 Apple Watch 快速解锁</li> 
 <li>允许指定数据库备份路径</li> 
 <li>增加标签功能</li> 
 <li>在条目视图中添加密码等级栏</li> 
 <li>增加群组克隆操作</li> 
 <li>显示条目历史项目之间的修改</li> 
 <li>能够批量删除和清除未使用的自定义图标</li> 
 <li>通过无线 NFC 实现对硬件密钥的支持</li> 
 <li>SSH 代理：增加对 OpenSSH 8.2 FIDO/U2F 密钥的支持</li> 
 <li>浏览器：在 Linux 上增加对 Microsoft Edge 的支持</li> 
 <li>浏览器：支持扩展中的本地密码生成器</li> 
 <li>……</li> 
</ul> 
<h3>其他变化</h3> 
<ul> 
 <li>将加密后端移植到 Botan</li> 
 <li>改进附件处理和安全</li> 
 <li>允许在解锁对话框中选择任何打开的数据库</li> 
 <li>KeeShare：删除检查签名的容器和 QuaZip 的依赖</li> 
 <li>引入安全选项，使双击时能够复制（默认关闭）</li> 
 <li>增加 "无需确认即可删除条目" 功能</li> 
 <li>改进 macOS 和 Windows 平台的整合</li> 
 <li>默认只锁定当前数据库</li> 
 <li>在数据库解锁时显示过期条目</li> 
 <li>TOTP 预览新增倒计时进度条</li> 
 <li>修复下载 favicon 时产生的崩溃</li> 
 <li>……</li> 
</ul> 
<p>更多详情可查看：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fkeepassxreboot%2Fkeepassxc%2Freleases" target="_blank">https://github.com/keepassxreboot/keepassxc/releases</a></p>
