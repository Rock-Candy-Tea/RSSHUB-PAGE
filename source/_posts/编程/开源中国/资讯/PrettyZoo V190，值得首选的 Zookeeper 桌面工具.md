                                                        <p>Hello，距离上个版本 Release 已经过去了一个月，新版总算是支持了国际化（TODO 了好几个版本），也调整和优化了操作体验，强烈建议升级。</p> 
<p>简介：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fvran-dev%2FPrettyZoo">PrettyZoo</a> 是一款基于 JavaFX 实现的 Zookeeper 桌面图形化工具，不仅颜值极高，而且功能齐全，支持 Windows / Mac / Linux 等主流平台。</p> 
<p><img alt height="506" src="https://oscimg.oschina.net/oscnet/up-ba1dfc03de28800b87b707e008cecac3668.png" width="900" referrerpolicy="no-referrer"></p> 
<h2><strong>新版特性</strong></h2> 
<ol> 
 <li> <p>feature：国际化支持，目前支持中文、英文</p> </li> 
 <li> <p>feature：节点增删操作调整为右键菜单</p> </li> 
 <li> <p>feature：节点支持多选操作</p> </li> 
 <li> <p>feature：server 列表新增右键菜单</p> </li> 
 <li> <p>feature：创建 server 时焦点默认集中在输入框</p> </li> 
 <li> <p>feature：创建 node 时焦点默认集中在输入框</p> </li> 
</ol>
