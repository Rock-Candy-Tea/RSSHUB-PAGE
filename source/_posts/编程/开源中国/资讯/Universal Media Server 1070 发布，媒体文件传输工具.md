                                                        <p>Universal Media Server 10.7.0 现已发布。Universal Media Server 是一个兼容 DLNA，UPnP 和 HTTP/S 的媒体服务器，支持所有主要操作系统，包括 Windows，Linux 和 macOS 版本，几乎不需要配置就可以流式传输或转码许多不同的媒体格式。</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-e524d5b0e4d7ac198ff8e3f6f086e501282.png" referrerpolicy="no-referrer"></p> 
<p><strong>主要更新内容</strong></p> 
<ul> 
 <li>改进了对电影和动漫剧集的检测</li> 
 <li>增加了 8GB 以上内存的 Windows 机器的默认最大内存</li> 
 <li>从 TRANSCODE 文件夹内的条目中删除了文件名</li> 
 <li>修复了 UPnP 搜索结果中的 upnp:class</li> 
 <li>修复了 "重设缓存" 按钮也不能重设 API 元数据的问题</li> 
 <li>修复了完全播放状态下数据库升级失败的问题</li> 
 <li>修复了 FFmpeg 崩溃的问题</li> 
 <li>修复了将一个目录标记为完全播放后不改变缓存缩略图的问题</li> 
 <li>改进了对 iPhone 上 VLC 的支持</li> 
</ul>
