                                                        <p>Jmix 框架和 Studio 1.3 版本发布，这次发布带来了不少新功能以及框架和 Studio 的 Bug 修复：</p> 
<p>主要新功能：</p> 
<ol> 
 <li>Java 兼容版本提升至 Java 11，不再支持 Java 8.</li> 
 <li>FlowUI 预览，该版本提供了基于 Vaadin 23 的 FlowUI 预览版。</li> 
 <li>一键云部署预览，该版本提供了 AWS 一键部署预览版。</li> 
</ol> 
<p>主要解决的 Bug 有：</p> 
<ol> 
 <li>解决了 Liquibase 脚本在 MySQL 运行出错的问题。</li> 
 <li>多租户场景时，租户信息未保存的问题。</li> 
 <li>集合属性的懒加载出错问题。</li> 
</ol> 
<p>详细内容请参考中文官网发布的最新更新：https://docs.jmix.cn/jmix/whats-new/index.html</p> 
<div>
