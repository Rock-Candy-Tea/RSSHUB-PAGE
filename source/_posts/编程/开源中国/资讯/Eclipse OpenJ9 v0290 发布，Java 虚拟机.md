<div class="content">
                                                                    
                                                        <p><span style="background-color:#ffffff; color:#333333">Eclipse OpenJ9 v0.29.0 现已<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.eclipse.org%2Fopenj9%2Fdocs%2Fversion0.29%2F" target="_blank">发布</a>，Eclipse OpenJ9 是 OpenJDK 的 JVM，由 IBM 创建并捐赠给 Eclipse 基金会。此 Java 虚拟机经过优化，具备占用空间小、启动速度快以及吞吐量高等优势。</span></p> 
<p><span style="background-color:#ffffff; color:#333333">OpenJ9 0.29.0 版本支持 OpenJDK 8 和 11。所有版本都针对 OpenJ9 功能验证（FV）测试套件、OpenJDK 测试套件以及 Adoptium 提供的其他测试进行了测试。</span></p> 
<p><span style="background-color:#ffffff; color:#333333">主要更新内容有：</span></p> 
<ul style="list-style-type:disc"> 
 <li>新二进制文件和对受支持环境的更改：<span style="background-color:#ffffff; color:#121212">增加了对 AArch64 Linux 的全面支持</span></li> 
 <li><span><span><span><span><span><span style="color:rgba(0, 0, 0, 0.87)"><span><span><span><span><span><span><span><span><span><span style="background-color:#ffffff"><span><span><span><code>-Xsoftmx</code> gencon 的更新</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></li> 
 <li><span style="background-color:#ffffff; color:#24292f">JITServer 技术现在是 Linux on x86-64 和 Power 的支持功能</span></li> 
 <li><span style="color:#000000">-</span><code>-XX:[+|-]UTFCache</code><span style="color:#000000">选项添加</span></li> 
</ul>
