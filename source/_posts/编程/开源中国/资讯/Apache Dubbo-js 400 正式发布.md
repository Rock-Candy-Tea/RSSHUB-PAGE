                                                                                            <p>Apache Dubbo-js 4.0.0 已正式发布。</p> 
<p><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fapache%2Fdubbo-js%2Freleases%2Ftag%2F4.0.0" target="_blank">发布公告写道</a>，dubbo-js 4.0.0 将是一个新的里程碑。开发团队对软件架构进行了重大调整，并提供了全栈的 dubbo 服务能力。正在开发中的 dubbo/dj 将通过为 dubbo 构建跨语言的 DSL 来赋予框架代码零开发 (code zero development) 能力。</p> 
<p>Dubbo-js 为 Dubbo 项目的子项目，通过 Node.js 打通对 Dubbo RPC 服务的调用，实现了 Java 的接口代码自动化翻译成 JS 代码和完善 JavaScript 类型体系的能力，极大提升了项目运作效率和开发的体验。</p> 
<p>4.0 版本由来自 vivo互联网技术团队（<a href="https://my.oschina.net/vivotech" target="_blank">@vivo互联网技术</a>）的成员主导，推动 Node 和 Dubbo 生态的对接和融合。</p>
