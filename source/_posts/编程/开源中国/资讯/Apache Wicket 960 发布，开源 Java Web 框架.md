                                                                                            <p>Apache Wicket 是一个开源的面向 Java 组件的 Web 应用框架，为政府、商店、大学、城市、银行、电子邮件提供商等成千上万的 Web 应用和网站提供支持。</p> 
<p>Wicket 的开发中使用了语义版本，因此与 9.0.0 相比，9.6.0 版本没有出现 API 中断。</p> 
<h3>改进</h3> 
<ul> 
 <li>[WICKET-6920] - 改进示例以使用浏览器的明/暗模式</li> 
 <li>[WICKET-6924] - 在 SecurePackageResourceGuard 中允许 image/avif</li> 
 <li>[WICKET-6927] - 移除 java.security.AccessController</li> 
</ul> 
<h3>Bug</h3> 
<ul> 
 <li>[WICKET-6921] - MultipartFormComponentListener 在隐藏组件上失效</li> 
</ul> 
<h3>Task</h3> 
<ul> 
 <li>[WICKET-6918] - 在网站上添加最新的 wicket.xsd 的链接</li> 
 <li>[WICKET-6919] - 改进 EnclosureContainer 的 javadoc，以解释它不应该与其他应用程序一起使用</li> 
 <li>[WICKET-6925] - 弃用 AbstractWrapModel</li> 
</ul>
