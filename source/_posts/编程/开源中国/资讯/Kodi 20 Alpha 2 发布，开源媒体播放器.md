                                                                                            <p>Kodi 是由 XBMC 基金会开发的开源媒体播放器，原名 XBMC（最后一个以 XBMC 命名的版本是 13.2 Gotham，14.0 Helix 是第一个以 Kodi 命名的版本），Kodi 可以运行在多种操作系统和硬件平台。 它可以让用户播放本地或网络存储设备中的大多数视频、音乐、播客及各种常见数位媒体文件。</p> 
<p>该软件最初是为计划运行在 Xbox 上的，名称也由此而来。随后有了 Android、Linux、BSD、macOS、iOS 和 Windows 操作系统的原生版。</p> 
<p>Kodi "Nexus" Alpha 2（20.x 系列）发布，以下是本次更新值得关注的地方：</p> 
<h3>特定平台</h3> 
<ul> 
 <li>Android 
  <ul> 
   <li>修复了 Kodi Android 应用程序打开、关闭和再次打开时的卡顿。</li> 
   <li>另一个 TrueHD 修复：缓冲区减少到一个特定的水平，以优化数据吞吐量</li> 
   <li>更多的本地应用程序图标应该在 Kodi 中完全呈现</li> 
   <li>修复二进制 python 模块（pycryptodome）的打包错误</li> 
  </ul> </li> 
 <li>Xbox 
  <ul> 
   <li>修复了一些特定的 H.265 视频崩溃问题</li> 
  </ul> </li> 
</ul> 
<h3>Python</h3> 
<ul> 
 <li>随着 CPython 3.9 系列的发布，Android 和苹果系统的 Python 版本已经提升到 3.9.13。CPython 团队专门针对 Kodi 的一些崩溃的问题进行了修复。</li> 
 <li>请注意，这并不影响任何 Windows 或者 Linux 版本</li> 
</ul> 
<h3>文件系统</h3> 
<ul> 
 <li>修复了从 "远程" 文件系统（如 WebDAV）访问时导致缩略图不显示的问题</li> 
</ul> 
<h3>隐藏字幕</h3> 
<ul> 
 <li>修复了隐藏字幕（Closed Captions）不工作的一个实例。</li> 
</ul> 
<h3>字幕</h3> 
<ul> 
 <li>多个 WebVTT 的修复</li> 
</ul> 
<h3>蓝光</h3> 
<ul> 
 <li>修复蓝光恢复问题</li> 
</ul>
