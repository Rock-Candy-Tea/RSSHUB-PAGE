<div class="content">
                                                                    
                                                        <p><span style="background-color:#ffffff; color:#333333">DBeaver 21.2.2 现已发布。DBeaver是一个免费开源的通用数据库工具，适用于开发人员和数据库管理员。</span></p> 
<p><span style="background-color:#ffffff; color:#333333"><strong>主要更新内容</strong></span></p> 
<ul> 
 <li><span style="background-color:#ffffff; color:#333333">DBeaver for MacOS X for M1/Silicon/ARM 已发布</span></li> 
 <li>SQL 编辑器 
  <ul> 
   <li>客户端命令 @include 已修复</li> 
   <li>修复了在长查询上挂起 SQL 解析器的错误</li> 
   <li>修复了自动完成功能</li> 
   <li>SQL 控制台中的架构选择器已修复</li> 
   <li>语句文本提取已在“块选择”模式下修复</li> 
  </ul> </li> 
 <li>数据编辑器 
  <ul> 
   <li>表达式解析器已在分组面板中修复</li> 
  </ul> </li> 
 <li>导航器：数据库元数据刷新速度显着提高</li> 
 <li>通用驱动程序：错误的表模式识别问题已修复</li> 
 <li>Exasol：在 SQL 中添加了额外的关键字</li> 
 <li><span style="background-color:#ffffff; color:#555555">HANA</span> 
  <ul> 
   <li><span style="background-color:#ffffff; color:#555555">开始/结束块解析器已修复</span></li> 
   <li><span style="background-color:#ffffff; color:#555555">修改表语法已修复</span></li> 
  </ul> </li> 
 <li><span style="background-color:#ffffff; color:#555555">Hive：添加了反斜杠转义支持</span></li> 
 <li><span style="background-color:#ffffff; color:#555555">PostgreSQL</span> 
  <ul> 
   <li><span style="background-color:#ffffff; color:#555555">​​​​​​​复合值编辑器得到改进（用于复合数据类型）</span></li> 
   <li><span style="background-color:#ffffff; color:#555555">改进了版本 13 支持（修复了读取数据类型元数据的罕见问题）</span></li> 
  </ul> </li> 
</ul>
