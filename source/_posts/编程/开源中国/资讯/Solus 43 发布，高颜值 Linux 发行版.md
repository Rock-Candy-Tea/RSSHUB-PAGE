                                                        <p>Solus 4.3 已发布，此版本带来了桌面环境的更新、新的软件堆栈和硬件支持。</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-f831f3070fa840d6c0aae9dcbc2287e5f99.png" referrerpolicy="no-referrer"></p> 
<p>Solus 4.3 采用了 Linux 5.13.1 内核，可为更广泛的硬件提供支持，例如：</p> 
<ul> 
 <li>AMD Radeon RX 6700 XT、6800、6800 XT 和 6900 XT 显卡</li> 
 <li>用于 Elkhart Lake 平台和带有 DA7219 + MAX98373/MAX98360A I2S 音频编解码器的 ASoC 机器驱动程序</li> 
 <li>通过 X86_PLATFORM_DRIVERS_DELL 支持面向 Dell X86 平台特定的驱动程序，例如供应商特定的笔记本电脑扩展驱动程序</li> 
 <li>通过 SENSORS_NZXT_KRAKEN2 对 NZXT Kraken X42/X52/X62/X72 一体式 CPU 液体冷却器进行硬件监控</li> 
 <li>通过 NET_VENDOR_MICROSOFT 支持 Microsoft 网络设备</li> 
 <li>用于 Elkhart Lake、Jasper Lake、Tiger Lake 的英特尔 ASoC</li> 
 <li>通过 HID_PLAYSTATION 支持索尼 PS5 控制器</li> 
 <li>补充对 Microsoft Surface 的支持，例如 CHARGER_SURFACE、SURFACE_ACPI_NOTIFY（Surface ACPI 通知 (SAN) 驱动程序）、SURFACE_DTX 和 SURFACE_HOTPLUG（具有热插拔 PCIe 卡的 Microsoft Surface 设备上的带外热插拔事件信号，更多详细信息请<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fcateee.net%2Flkddb%2Fweb-lkddb%2FSURFACE_HOTPLUG.html" target="_blank">点击此处</a>）。</li> 
 <li>通过 HID_SEMITEK 支持 SemiTek 键盘</li> 
</ul> 
<p>此外，Solus 4.3 也继续完善其内核配置，此版本具有以下更改：</p> 
<ul> 
 <li>启用 CONFIG_NETFILTER_XT_TARGET_CHECKSUM 以改进 lxd 支持</li> 
 <li>启用 KVM guest 通过 X86_SGX_KVM 创建 SGX enclave</li> 
 <li>启用 VIRTIO SND</li> 
 <li>禁用 RT_GROUP_SCHED 以改进 JACK 支持</li> 
</ul> 
<p>常用的软件包升级包括：</p> 
<ul> 
 <li>Firefox 89.0.2</li> 
 <li>LibreOffice 7.1.4.2</li> 
 <li>Thunderbird 78.11.0</li> 
 <li>Mesa 21.1.3</li> 
 <li>ffmpeg 4.4</li> 
 <li>……</li> 
</ul> 
<p>桌面环境方面，Solus 4.3 提供了最新版本的 Budgie / GNOME / MATE / KDE Plasma，并针对这些桌面环境提供了许多更新，例如 GNOME 桌面版本的 GNOME 升级到了最新的 GNOME 40.2。</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-1faaab35a3da0a158885b8b196f963b5a54.png" referrerpolicy="no-referrer"></p> 
<p>详细更新内容和下载地址查看 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgetsol.us%2F2021%2F07%2F11%2Fsolus-4-3-released%2F" target="_blank">https://getsol.us/2021/07/11/solus-4-3-released/</a>。</p> 
<p>Solus 是从无到有创建的 Linux 发行版，可选择自产的 Budgie 桌面环境，或者 GNOME、MATE、KDE Plasma 作为桌面环境。它的包管理器 eopkg 基于来自 Pardus Linux 的 PiSi 包管理系统，并有着准滚动发行模式。</p>
