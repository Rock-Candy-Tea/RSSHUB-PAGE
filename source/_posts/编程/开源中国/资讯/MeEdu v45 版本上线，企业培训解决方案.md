 <h2 style="margin-left:0; margin-right:0"><span>PHP 最低版为 7.3</span></h2> 
 <p style="margin-left:0; margin-right:0"><span>新版本的 meedu 要求 PHP 的最低版本为 7.3 ，请在使用 PHP7.2 运行 meedu 的用户尽快升级。</span></p> 
 <h2 style="margin-left:0; margin-right:0"><span>Laravel 框架升级到 v8.x</span></h2> 
 <p style="margin-left:0; margin-right:0"><span>新版本已经将 meedu 的底层框架升级到了 Laravel v8.x 的版本。因为 meedu 的单元测试覆盖率达到了 70% ，因为此次的底层框架升级较为轻松。</span></p> 
 <h2 style="margin-left:0; margin-right:0"><span>支持全文搜索</span></h2> 
 <p style="margin-left:0; margin-right:0"><span>新版本支持全文搜索啦。以前只能搜索录播课的标题，现在新版本支持搜索标题，详情介绍，录播课程，录播视频等。可以更快，更方便的检索出用户想要的信息。</span></p> 
 <h2 style="margin-left:0; margin-right:0"><span>更完善的社交登录账号绑定体系</span></h2> 
 <p style="margin-left:0; margin-right:0"><span>新版本对社交登录的账号体系做了全新的优化，体验更便捷，交互更符合日常习惯。</span></p> 
 <h2 style="margin-left:0; margin-right:0"><span>修复用户注册 IP 解析 Bug</span></h2> 
 <p style="margin-left:0; margin-right:0"><span>新版修复了用户注册 IP 解析错误，无法解析等情况。</span></p> 
 <h2 style="margin-left:0; margin-right:0"><span>新增腾讯云 SMS 的支持</span></h2> 
 <p style="margin-left:0; margin-right:0"><span>新版本增加了腾讯云 SMS 的支持。至此， meedu 所使用到的云服务基本上是都支持阿里云/腾讯云，方便用户选择。</span><span style="color:#e8323c">云片短信将在未来的某个版本中移除，建议新用户请勿使用云片短信服务。</span></p> 
 <h2 style="margin-left:0; margin-right:0"><span>图片存储新增能腾讯云 COS</span></h2> 
 <p style="margin-left:0; margin-right:0"><span>新版本增加了腾讯云 COS 图片存储，对于想要使用全套腾讯云服务的用户来说，将是非常好的选择。</span><span style="color:#e8323c">七牛云将会在未来某个版本移除，建议新用户请勿使用七牛云服务。</span></p> 
 <h2 style="margin-left:0; margin-right:0"><span>优化微信公众号菜单编辑</span></h2> 
 <p style="margin-left:0; margin-right:0"><span>本次优化了部分微信公众号无法使用 meedu 自带的菜单编辑功能。</span></p> 
 <h2 style="margin-left:0; margin-right:0"><span>完善 PC 用户资料编辑</span></h2> 
 <p style="margin-left:0; margin-right:0"><span>PC 增加用户资料的编辑，包括昵称，头像，基础信息，实名认证等信息。</span></p> 
 <h2 style="margin-left:0; margin-right:0"><span>优化微信支付</span></h2> 
 <p style="margin-left:0; margin-right:0"><span>新版本增加了微信支付先关的 API 接口。</span></p> 
 <h2 style="margin-left:0; margin-right:0"><span>优化邀请码生成逻辑</span></h2> 
 <p style="margin-left:0; margin-right:0"><span>新版下邀请码在用户符合条件的情况下自动生成。</span></p> 
 <h2 style="margin-left:0; margin-right:0"><span>修复免费课程视频看Bug</span></h2> 
 <p style="margin-left:0; margin-right:0"><span>获取视频播放 URL 地址 API 在课程免费且配置了试看的情况下试看依旧存在的Bug。</span></p> 
 <h2 style="margin-left:0; margin-right:0"><span>优化手机号换绑流程</span></h2> 
 <p style="margin-left:0; margin-right:0"><span>手机号换绑新增原手机号校验。</span></p> 
 <h2 style="margin-left:0; margin-right:0"><span>API 文档重构</span></h2> 
 <p style="margin-left:0; margin-right:0"><span>使用 apidoc.js 生成新版本的 API 文档。</span></p>
