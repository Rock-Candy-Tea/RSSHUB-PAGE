                                                                                            <p>Spring Boot 2.5.0 首个 RC 版本<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fspring.io%2Fblog%2F2021%2F04%2F15%2Fspring-boot-2-5-0-rc1-available-now" target="_blank">已发布</a>，其开发团队表示将在5月20日正式 GA。</p> 
<p>2.5.0-RC1 更新亮点：</p> 
<ul> 
 <li>增加用于 Quartz 的新执行器端点</li> 
 <li>支持使用 R2DBC 进行数据库初始化</li> 
 <li>Micometer 指标支持 Spring Data repo 和 MongoDB</li> 
 <li>支持 Gradle 7.0</li> 
 <li>支持 HTTP/2 over TCP (h2c)</li> 
 <li>升级依赖</li> 
</ul> 
<p style="text-align:start"><span style="color:#333333">有关变更和升级说明的完整列表，查看 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fspring-boot%2Fwiki%2FSpring-Boot-2.5.0-RC1-Release-Notes" target="_blank">release note</a> 和<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdocs.spring.io%2Fspring-boot%2Fdocs%2F2.5.0-RC1%2Freference%2Fhtml%2F" target="_blank">升级参考文档</a>。</span></p> 
<p style="text-align:start"><span style="color:#333333">如果要开始使用 </span>Spring Boot 2.5.0 <span style="color:#333333">并尝试新功能，可以在 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fstart.spring.io%2F%23%21type%3Dmaven-project%26language%3Djava%26platformVersion%3D2.5.0.RC1%26packaging%3Djar%26jvmVersion%3D11%26groupId%3Dcom.example%26artifactId%3Ddemo%26name%3Ddemo%26description%3DDemo%2520project%2520for%2520Spring%2520Boot%26packageName%3Dcom.example.demo" target="_blank">start.spring.io</a> 上启动一个新项目。</span></p>
