 <p>Metabase 发布了 0.40.5 版本。Metabase 是一个简单的分析工具，通过给公司成员提问，从得到的数据中进行分析、学习。</p> 
 <p style="color:#333333; margin-left:0; margin-right:0; text-align:left">此版本更新内容如下：</p> 
 <p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><strong>Bug 修复</strong></p> 
 <ul> 
  <li>GeoJSON URL 验证修复 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmetabase%2Fmetabase%2Fpull%2F17991" target="_blank">#17991</a> )</li> 
  <li>如果没有 metric，则在将网格悬停时，网格地图会导致前端重新加载或空白屏幕 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmetabase%2Fmetabase%2Fissues%2F17940" target="_blank">#17940</a> )</li> 
  <li>当 UI 语言为土耳其语时，无法创建 2 个以上的 metric - 屏幕变为空白或显示“Something went wrong”（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmetabase%2Fmetabase%2Fissues%2F16323" target="_blank">#16323</a>）</li> 
  <li>超过 100 个系列的可视化只显示一个空白图表 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmetabase%2Fmetabase%2Fissues%2F16000" target="_blank">#16000</a> )</li> 
  <li>数据点值在所有系列上使用第一个系列的格式样式 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmetabase%2Fmetabase%2Fissues%2F13095" target="_blank">#13095</a> )</li> 
 </ul>
