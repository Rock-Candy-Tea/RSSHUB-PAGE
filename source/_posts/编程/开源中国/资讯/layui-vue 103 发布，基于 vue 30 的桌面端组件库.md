                                                                                            <p>更新内容：</p> 
<ul> 
 <li><span style="background-color:#ffffff; color:#40485b">[新增] ripple 组件, 提供 水波纹 效果。</span></li> 
 <li><span style="background-color:#ffffff; color:#40485b">[新增] empty 组件 image 插槽, 支持自定义 image。</span></li> 
 <li><span style="background-color:#ffffff; color:#40485b">[新增] badge 组件 ripple 属性, 开启水波纹动画, 仅适用于 type 为 dot 时。</span></li> 
 <li><span style="background-color:#ffffff; color:#40485b">[新增] table 组件 column 配置 align 属性, 支持 left center right 对齐方式。</span></li> 
 <li><span style="background-color:#ffffff; color:#40485b">[新增] upload 组件 disabled-preview 属性, 上传预览区域添加禁用状态。</span></li> 
 <li><span style="background-color:#ffffff; color:#40485b">[修复] upload 组件 cut 属性, 开启时丢失图片信息。</span></li> 
 <li><span style="background-color:#ffffff; color:#40485b">[修复] table 组件 column 存在 1px 宽度误差。</span></li> 
 <li><span style="background-color:#ffffff; color:#40485b">[优化] date-picker 组件 monent 为 day.js。</span></li> 
 <li><span style="background-color:#ffffff; color:#40485b">[优化] table 组件 html document 结构。</span></li> 
 <li><span style="background-color:#ffffff; color:#40485b">[移除] xlsx 依赖, 减少打包体积。</span></li> 
</ul>
