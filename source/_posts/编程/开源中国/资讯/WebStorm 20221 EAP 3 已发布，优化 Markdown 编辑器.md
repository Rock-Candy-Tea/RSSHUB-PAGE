                                                                                            <p>WebStorm 2022.1 EAP #3 已发布，第三个尝鲜版主要包括以下新内容：</p> 
<h2><em>在 Save Document </em>上运行 Prettier 和其他工具</h2> 
<p>WebStorm 2022.1 EAP 3 支持在按下<em>⌘S / Ctrl+S </em>时运行多个操作和工具。以前，此功能仅在将快捷方式映射到 <em>Save All </em>操作时才有效。从 v2022.1 开始，如果将快捷方式映射到类似的操作：<em>Save Document</em> ，它也起作用。</p> 
<p style="margin-left:0px"><img alt height="350" src="https://oscimg.oschina.net/oscnet/up-45a2ca1a63ce3a48862791e558ebf39c179.png" width="700" referrerpolicy="no-referrer"></p> 
<h2>更新了 Markdown 编辑器浮动工具栏</h2> 
<ul> 
 <li>为了更容易格式化 Markdown 文件，WebStorm 2022.1  重新设计了出现在文本选择上方的浮动工具栏。</li> 
 <li>除了新设计之外，工具栏现在还提供<strong>创建列表</strong>功能和一个允许<strong>选择标题样式</strong>的下拉菜单。</li> 
 <li><span style="background-color:#ffffff; color:#27282c">该浮动工具栏是可自定义的，可以使用所需的选项去填充它。</span></li> 
</ul> 
<p><img alt height="350" src="https://oscimg.oschina.net/oscnet/up-1047c7eee3944b79cb7c9acfc6863f4a934.gif" width="700" referrerpolicy="no-referrer"></p> 
<h2>其他显著变化</h2> 
<ul> 
 <li>调试器控制台支持表情符号 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FWEB-54763" target="_blank">WEB-54763</a> )。</li> 
 <li><span style="background-color:#ffffff; color:#2e3033">现在可以在 </span><span style="background-color:#ffffff; color:#27282c">Node.js 运行环境中</span><span style="background-color:#ffffff; color:#2e3033">指定自定义的 Docker Сompose 标志，比如 </span><code>--rm</code><span style="background-color:#ffffff; color:#27282c"><span> </span>或<span> </span></span><code>--service-ports</code>( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FWEB-54160" target="_blank">WEB-54160</a> )。</li> 
 <li>在 Big Sur 上，切换编辑器选项卡的快捷方式将恢复预期的效果 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-261595" target="_blank">IDEA-261595</a> )。</li> 
</ul>
