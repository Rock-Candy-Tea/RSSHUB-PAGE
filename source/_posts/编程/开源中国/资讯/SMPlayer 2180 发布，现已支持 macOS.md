                                                        <p>SMPlayer 是一个带有 YouTube 支持的跨平台媒体播放器，原本仅支持 Windows 和 Linux，随着 SMPlayer 21.8.0 版本的推出，现已支持 macOS。</p> 
<h3>更新：</h3> 
<ul> 
 <li>现在有了适用于 macOS 的版本；</li> 
 <li>更好地自动调整主窗口的大小，以防止出现黑边；</li> 
 <li>修复加载 YouTube 播放列表的问题；</li> 
 <li>增加一个选项，将视频旋转 180 度；</li> 
 <li>播放列表：防止在使用播放「上一个/下一个」时出现一秒钟的延迟；</li> 
 <li>增加一些预定义速度（0.25x、0.5x、1.25x、1.5x、1.75x）；</li> 
 <li>修正 mpv 的音频 CD 播放；</li> 
 <li>修复音频通道；</li> 
</ul> 
<h3><strong>Windows</strong></h3> 
<ul> 
 <li>安装 YouTube 支持现在是可选项；</li> 
 <li>使用 Qt 5.15.2；</li> 
</ul> 
<h3><strong>Linux</strong></h3> 
<ul> 
 <li>在播放过程中在 Wayland 上禁用省电功能；</li> 
 <li>将 KDE 从 *.desktop 文件的类别中移除；</li> 
 <li>SMPlayer 现在也有 appimage、flatpak 和 snap 的版本；</li> 
</ul> 
<p>更多详情可查看：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fsmplayer-dev%2Fsmplayer%2Freleases" target="_blank">https://github.com/smplayer-dev/smplayer/releases</a></p>
