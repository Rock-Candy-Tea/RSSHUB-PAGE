                                                        <p>Zorin OS 16 在今年 8 月正式发布，在短短 4 个月的时间里已被下载超过 100 万次。Zorin OS 16 Lite 是完整版（基于 GNOME Shell）Zorin OS 16 的精简版。它仍然基于 Ubuntu 20.04.3 LTS 和 Linux Kernel 5.11。</p> 
<p>Zorin OS 16 Lite 将得到软件更新和安全补丁的支持，直到 2025 年 4 月。</p> 
<h3>新外观</h3> 
<p>Zorin OS 16 Lite 从 Zorin OS 16 中引入了新的、更精炼的默认主题，为整个系统带来了细节润色和令人愉悦的动画，提升了使用体验。</p> 
<p><img alt height="525" src="https://static.oschina.net/uploads/space/2021/1210/092015_ZnF3_4937141.jpeg" width="700" referrerpolicy="no-referrer"></p> 
<p>新版本中还引入了新的艺术作品和桌面壁纸。</p> 
<p><img alt height="263" src="https://static.oschina.net/uploads/space/2021/1210/092025_vB6S_4937141.png" width="700" referrerpolicy="no-referrer"></p> 
<h3>带有窗口预览的新任务栏</h3> 
<p>Zorin OS 16 Lite 重写了任务栏，现在当把鼠标悬停在一个应用图标上时，它会显示窗口预览，让你对工作流程有一个更好的概述。</p> 
<p><img alt height="231" src="https://static.oschina.net/uploads/space/2021/1210/092040_1Gjg_4937141.jpeg" width="700" referrerpolicy="no-referrer"></p> 
<h3>桌面布局</h3> 
<p>在 Zorin OS 16 Lite 中引入两种新的桌面布局。类似 Chrome OS 和 Windows Classic。</p> 
<p>类似 Chrome OS 的桌面环境具有居中的图标任务栏设计和全新的全屏应用程序网格菜单。对于更熟悉智能手机和平板电脑等设备的用户而言，这是个不错的选择。</p> 
<p><img alt height="524" src="https://static.oschina.net/uploads/space/2021/1210/092052_fuiN_4937141.png" width="700" referrerpolicy="no-referrer"></p> 
<p>类似于 Windows Classic 的桌面布局采用了传统的列表式任务栏和简单的应用菜单，旨在让老牌电脑用户感到熟悉。</p> 
<p><img alt height="525" src="https://static.oschina.net/uploads/space/2021/1210/092103_0iKF_4937141.jpeg" width="700" referrerpolicy="no-referrer"></p> 
<h3>重新设计的 Zorin Appearance</h3> 
<p>Zorin OS 16 Lite 现在包括来自 Zorin OS 16 的新的 Zorin Appearance，它被设计成更容易使用和导航。Zorin Appearance 允许你选择不同的桌面布局，改变应用程序和图标主题，选择你的桌面字体，并调整你的桌面的其他部分。</p> 
<p><img alt height="496" src="https://static.oschina.net/uploads/space/2021/1210/092114_yfBI_4937141.png" width="700" referrerpolicy="no-referrer"></p> 
<h3>启用 Flathub</h3> 
<p>在 Zorin OS 16 Lite 的软件商店中，除了 Snap store 和 Ubuntu & Zorin OS APT 软件库之外，现在还预装了 Flathub。因此现在可以在 Zorin OS 16 Lite 中安装更多的应用程序。</p> 
<p>内置的软件商店也进行了用户界面的改进，使应用能够更容易被找到和安装不同来源的应用程序。</p> 
<p><img alt height="328" src="https://static.oschina.net/uploads/space/2021/1210/092124_RGsc_4937141.jpeg" width="700" referrerpolicy="no-referrer"></p> 
<h3>更安全、更容易地侧载 Windows 软件</h3> 
<p>在侧载应用程序和游戏时，Zorin OS 16 Lite使您更容易以最兼容和安全的方式安装您想要的软件。</p> 
<p>Zorin OS 现在带有一个内置的数据库，可以检测 Windows 安装文件。如果系统识别出你双击的 .exe 或 .msi 文件，它将自动建议你从推荐的来源安装该应用程序或游戏。</p> 
<p><img alt height="306" src="https://static.oschina.net/uploads/space/2021/1210/092134_5nVH_4937141.png" width="700" referrerpolicy="no-referrer"></p>
