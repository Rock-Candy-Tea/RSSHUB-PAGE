<div class="content">
                                                                    
                                                        <p>Firefox 89 在上个月正式发布了，在该版本中对设计进行了重大更新。Firefox 90 相比 89 虽然没有那么大的变化，但仍然有一些值得关注的新功能和改进。</p> 
<ul> 
 <li>在这个新版本中，Firefox 引入了一个新的 about:third-party 页面，以帮助用户识别由第三方模块和应用程序引起的兼容性问题。</li> 
 <li>Windows 版还引入了一个新的选项，开启后 Firefox 浏览器可以在没有运行的情况下，也能使用后台服务静默安装更新。其目的是通过更新提醒尽量减少干扰，可以通过 「设置 > 常规 > Firefox 更新」查看和配置该选项。</li> 
 <li>用户可以通过 「设置 > 隐私」添加和管理 HTTPS-only 模式下不使用该模式的网站 —— 此前用户只能通过地址栏中的锁定图标将网站排除在 HTTPS-only 模式之外。 
  <ul> 
   <li><img alt height="438" src="https://static.oschina.net/uploads/space/2021/0714/065906_p9KI_4937141.png" width="700" referrerpolicy="no-referrer"></li> 
  </ul> </li> 
 <li>支持 Fetch Metadata Request Headers，它允许 Web 应用程序保护自己及其用户免受各种跨源威胁。</li> 
 <li>SmartBlock 功能也已更新到 V2 版本，现在可以在隐私浏览模式下阻止第三方 Facebook 脚本。</li> 
 <li>Print to PDF 功能也得到了改进，可以从生成的页面中产生有效的超链接。</li> 
 <li>右键单击并选择 "在新标签页中打开图像"，默认将在背景标签页中打开图像和媒体。</li> 
 <li>FTP 支持已被删除</li> 
 <li>而大多数无法访问硬件加速 WebRender 的用户现在将默认切换到使用软件 WebRender。</li> 
</ul>
