                                                                                            <p>AlmaLinux 是一个自由和开源的 Linux 发行版，最初由 CloudLinux 创建，提供一个社区支持的生产级企业操作系统，与 Red Hat Enterprise Linux（RHEL）二进制兼容。AlmaLinux 的第一个稳定版本于 2021 年 3 月 30 日发布。</p> 
<p>2021 年 3 月 30 日，AlmaLinux OS Foundation（AlmaLinux 操作系统基金会）成立，从 CloudLinux 手中接管了 AlmaLinux 的开发和治理，CloudLinux 承诺每年为该项目提供 100 万美元的资金。</p> 
<p><strong>AlmaLinux 9.0 Beta 发布，更新内容如下：</strong></p> 
<ul> 
 <li>AlmaLinux 9 的软件包和软件库已用新的 RPM-GPG-KEY-AlmaLinux-9 签名。</li> 
 <li>Linux Kernel 5.14</li> 
 <li>更新了动态编程语言、Web 和数据库服务器： 
  <ul> 
   <li>Perl 5.32</li> 
   <li>PHP 8.0</li> 
   <li>Python 3.9</li> 
   <li>Git 2.31</li> 
   <li>Apache HTTP Server 2.4</li> 
   <li>Varnish Cache 6.5</li> 
   <li>Squid 5.2</li> 
   <li>MySQL 8.0</li> 
   <li>Redis 6.2</li> 
  </ul> </li> 
 <li>更新的组件： 
  <ul> 
   <li>GCC 11.2</li> 
   <li>glibc 2.34</li> 
   <li>binutils 2.35</li> 
  </ul> </li> 
 <li>编译器更新： 
  <ul> 
   <li>Go Toolset 1.17.7</li> 
  </ul> </li> 
 <li>性能工具和调试器的更新： 
  <ul> 
   <li>GDB 10.2</li> 
   <li>Valgrind 3.18.1</li> 
   <li>SystemTap 4.6</li> 
   <li>Dyninst 11.0.0</li> 
   <li>elfutils 0.186</li> 
  </ul> </li> 
 <li>Java 工具更新： 
  <ul> 
   <li>Maven 3.6</li> 
   <li>Ant 1.10</li> 
  </ul> </li> 
</ul> 
<p>更多详情可查看：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwiki.almalinux.org%2Frelease-notes%2F9.0-beta.html" target="_blank">https://wiki.almalinux.org/release-notes/9.0-beta.html</a></p>
