                                                        <p><span style="background-color:#ffffff; color:#333333">Apache Tomcat 的四个分支发布了更新，分别是 </span>8.5.73、9.0.55、10.0.13 和 10.1.0-M7 (alpha)。</p> 
<p><strong>8.5.73</strong></p> 
<ul> 
 <li>改进 native/APR，包括在 APR 未能正确初始化的情况下避免 JVM 崩溃，以及改进错误处理</li> 
 <li>提升 JNDIRealm 对获取连接时发生的异常的稳健性</li> 
</ul> 
<p><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Ftomcat.apache.org%2Ftomcat-8.0-doc%2Fchangelog.html%23Tomcat_8.5.73_%28schultz%29" target="_blank">详情查看 Changelog</a>。</p> 
<p><strong>9.0.55、10.0.13</strong></p> 
<ul> 
 <li>通过在 Java 17 中孵化的 Panama API 实验性支持 OpenSSL，以及支持 OpenSSL 1.1+</li> 
 <li>增加对网络应用资源的自定义缓存策略的支持，这个初始实现版本允许控制是否对资源进行缓存</li> 
 <li>提升 JNDIRealm 对获取连接时发生的异常的稳健性</li> 
</ul> 
<p><strong>10.1.0-M7 (alpha)</strong></p> 
<ul> 
 <li>针对 Servlet 6 的 Servlet API 更新，包括重构 HttpServlet.doHead()，支持 Cookies 的泛型属性，更一致的 URI 处理——包括拒绝“可疑”URI 的选项</li> 
 <li>EL 5.0 的 EL API 更新，包括对 ELResolver.getType() 的更改</li> 
 <li>通过在 Java 17 中孵化的 Panama API 实验性支持 OpenSSL，以及支持 OpenSSL 1.1+</li> 
</ul>
