<div class="content">
                                                                                            <p>Ubuntu 20.04.3 LTS 现已<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Flists.ubuntu.com%2Farchives%2Fubuntu-announce%2F2021-August%2F000271.html" target="_blank">发布</a>。和以前的 LTS 系列一样，20.04.3 包括用于较新硬件的硬件支持（HWE）堆栈；所有架构都提供此支持。Ubuntu Server 默认安装 GA 内核；但是，用户可以从安装程序引导加载程序中选择 HWE 内核。</p> 
<p><img alt height="283" src="https://oscimg.oschina.net/oscnet/up-cab300df3e77f3dfc92f1cb5134d7268917.JPEG" width="500" referrerpolicy="no-referrer"></p> 
<p><em>图源：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.omgubuntu.co.uk%2F2021%2F08%2Fubuntu-20-04-3-point-release" target="_blank">omgubuntu</a></em></p> 
<p>公告指出，像往常一样，此版本包含许多更新；并且提供了更新的安装介质，因此安装后需要下载的更新就会减少。这些更新包括安全更新和对其他高影响的错误的修正，重点是保持与 Ubuntu 20.04 LTS 的稳定性和兼容性。</p> 
<p>Kubuntu 20.04.3 LTS、Ubuntu Budgie 20.04.3 LTS、Ubuntu MATE 20.04.3 LTS、Lubuntu 20.04.3 LTS、Ubuntu Kylin 20.04.3 LTS、Ubuntu Studio 20.04.3 LTS 和 Xubuntu 20.04.3 LTS 现在也已可用。</p> 
<p>将为 Ubuntu Desktop、Ubuntu Server、Ubuntu Cloud 和 Ubuntu Core 提供为期 5 年的维护更新。其余所有类型的产品将被支持 3 年。额外的安全支持可通过 ESM（扩展安全维护）获得。</p> 
<p>详细的更改日志可查看：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwiki.ubuntu.com%2FFocalFossa%2FReleaseNotes%2FChangeSummary%2F20.04.3" target="_blank">https://wiki.ubuntu.com/FocalFossa/ReleaseNotes/ChangeSummary/20.04.3</a></p>
