                                                        <p style="text-align:left">Mybatis-Plus 是一款 MyBatis 的增强工具包，简化 CRUD 操作。启动加载 XML 配置时注入单表 SQL 操作 ，为简化开发工作、提高生产率而生。Mybatis-Plus 启动注入非拦截实现、性能更优，让你专注业务快速敏捷开发。</p> 
<h4 style="text-align:left"><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fbaomidou%2Fmybatis-plus-samples" target="_blank">演示例子 mybatis-plus-samples</a>    <br> <a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fbaomidou.com" target="_blank">帮助文档</a> </h4> 
<h2 style="text-align:left">更新日志：</h2> 
<ul> 
 <li>增加瀚高数据库支持</li> 
 <li>增加注解 Order By 支持默认排序</li> 
 <li>Wrapper exists notExists orderBy groupBy 支持参数绑定</li> 
 <li>Wrapper 支持 setParamAlias 其它优化</li> 
 <li>优化 KeyGenerator 支持多实现多数据源注入</li> 
 <li>增强 ServiceImpl 泛型推断，解决多继承与代理问题</li> 
 <li>新增 PageDto 用于微服务对象传输序列化</li> 
 <li>新增 Page 提供静态 of 构造方式</li> 
 <li>增加代理 MethodHandleProxies 对 lambda 调试支持</li> 
 <li>调整 ActiveRecord 日志对象初始化</li> 
 <li>调整 ActiveRecord 模式 Model 类开发 pkVal 方法外部可用</li> 
 <li>删除标记过时代码</li> 
 <li>优化枚举值获取方式</li> 
 <li>分页 count 安全处理</li> 
 <li>Sequence 方法支持重写支持</li> 
 <li>升级 Mybatis 3.5.7</li> 
 <li>修复自动配置 lazy-initialization 无属性提示</li> 
 <li>修复 mysql on duplicate key update 字段名判断为表名问题</li> 
 <li>修复 lambda 条件 npe 异常</li> 
 <li>重构 lambda 信息提取方法</li> 
 <li>获取 lambda 信息不在序列化</li> 
 <li>合并 gitee pulls/ 141</li> 
 <li>fixed github issues/3208 3016</li> 
 <li>fixed github issues/3482 数据权限处理器支持 union all</li> 
 <li>调整事务未启用打印提示信息</li> 
 <li>单元测试优化相关依赖升级</li> 
</ul>
