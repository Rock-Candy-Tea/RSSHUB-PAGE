 <p>Universal Media Server 是一个兼容 DLNA，UPnP 和 HTTP/S 的媒体服务器，支持所有主要操作系统，包括 Windows，Linux 和 macOS 版本，几乎不需要配置就可以流式传输或转码许多不同的媒体格式。</p> 
 <p>Universal Media Server 10.12.0 发布，该版本更新内容如下：</p> 
 <h3>常规：</h3> 
 <ul> 
  <li>提高了启动扫描的性能并减少了内存的使用</li> 
  <li>渲染器的配置现在默认启用了 MediaInfo</li> 
  <li>修正了文件在下载过程中或刚下载完时被锁定的问题</li> 
  <li>修复了关闭时进程挂起的问题</li> 
  <li>使得构建项目的速度更快</li> 
  <li>开始使用 DeepSource 进行静态代码分析</li> 
 </ul> 
 <h3>依赖性：</h3> 
 <ul> 
  <li>将 h2database 从 1.4.199 恢复到 1.4.197</li> 
  <li>在 macOS 上将 JRE 升级到 8u302+8</li> 
  <li>将 junit5.version 从 5.7.2 升级到 5.8.0</li> 
  <li>将 maven-javadoc-plugin 从 3.3.0 升级到 3.3.1</li> 
  <li>将 maven-pmd-plugin 从 3.14.0 升级到 3.15.0</li> 
  <li>将 oshi-core 从 5.8.1 升级到 5.8.2</li> 
  <li>修正了在 x86(i586) Linux 上使用 x64 JRE 的问题</li> 
 </ul> 
 <h3>通过 Crowdin 进行翻译更新：</h3> 
 <ul> 
  <li>法语（100%）</li> 
  <li>德语（100%）</li> 
  <li>意大利语（100%）</li> 
  <li>葡萄牙语（巴西）</li> 
  <li>俄语（96%）</li> 
 </ul>
