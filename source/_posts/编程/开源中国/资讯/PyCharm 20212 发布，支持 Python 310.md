                                                                                            <p>JetBrains 正式发布 PyCharm 2021.2，特点是支持 Python 3.10。从新版本开始，来自亚洲的用户可以享受完全本地化的中文、韩文和日文用户界面。在下一个版本中，它将结束对几个软件包的支持，如 mako、buildout、web2py。</p> 
<p>PyCharm 2021.2 正式发布，更新内容如下：</p> 
<ul> 
 <li>完整的 match 语句和它们相应的 case 块；</li> 
 <li>支持用新的语法类型对 isinstance 和 issubclass 参数进行类型推理；</li> 
 <li>现在在客户端和主机端都支持在 Python 控制台中工作；</li> 
 <li>MongoDB 控制台中的字段和运算符的代码补全（仅适用于专业版）；</li> 
 <li>从数据编辑器中的上下文实时模板与SQL脚本动作（仅专业版）；</li> 
 <li>自动重新加载浏览器 HTML 预览（专业版）；</li> 
 <li>require() 的自动导入（专业版）；</li> 
 <li>IDE Features Trainer 插件中的新 Onboarding Tour；</li> 
 <li>测试运行器自动检测；</li> 
 <li>测试管理插件包括 Python 支持，可用于 PyCharm 社区；</li> 
 <li>新的测试数据插件，生成随机数据；</li> 
</ul> 
<p>更多详情可查看：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.jetbrains.com%2Fpycharm%2Fwhatsnew%2F" target="_blank">https://www.jetbrains.com/pycharm/whatsnew/</a></p>
