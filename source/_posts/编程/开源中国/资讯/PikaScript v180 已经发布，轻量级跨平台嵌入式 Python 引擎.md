                                                                                            <p>PikaScript v1.8.0 已经发布，轻量级跨平台嵌入式 Python 引擎</p> 
<p>此版本更新内容包括：</p> 
<p>功能更新：</p> 
<ol> 
 <li>支持PC端将python文件编译为字节码打包进固件，支持多python文件（无需文件系统）。</li> 
 <li>内存管理使用引用计数，支持工厂方法。</li> 
 <li>使用 valgrind 对内存隐患进行了全面排查。</li> 
</ol> 
<p>功能解读视频： <a href="https://gitee.com/link?target=https%3A%2F%2Fwww.bilibili.com%2Fvideo%2FBV14t4y1x7nv" target="_blank">https://www.bilibili.com/video/BV14t4y1x7nv</a></p> 
<p>不兼容更新：</p> 
<ol> 
 <li>C 模块的python声明文件后缀由.py修改为.pyi，语法内容不变。</li> 
</ol> 
<p>迁移指南：</p> 
<ol> 
 <li>将 C 模块的.py文件后缀修改为.pyi。</li> 
</ol> 
<p>详情查看：<a href="https://gitee.com/Lyon1998/pikascript/releases/v1.8.0">https://gitee.com/Lyon1998/pikascript/releases/v1.8.0</a></p>
