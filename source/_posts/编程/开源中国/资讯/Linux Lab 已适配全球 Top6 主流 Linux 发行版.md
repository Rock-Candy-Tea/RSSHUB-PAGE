                                                                                            <p style="color:#121212; margin-left:0; margin-right:0; text-align:start">大家好，</p> 
<p style="color:#121212; margin-left:0; margin-right:0; text-align:start">为了提供更好的使用体验，国产 Linux Lab 开源项目与 Linux Lab Disk（随身 Linux Lab 实验盘）正在紧锣密鼓地开发，预计明后天发布 v0.9 的第 2 个候选版本。</p> 
<p style="color:#121212; margin-left:0; margin-right:0; text-align:start">截止今日，国产 Linux Lab 开源项目已经被同学们广泛使用在包括 Windows、MacOS 和各大 Linux 发行版在内的操作系统上，这里是同学们自己提交的一份列表：</p> 
<div style="text-align:start">
 <a href="https://gitee.com/tinylab/linux-lab/issues/I1FZBJ" target="_blank"><span><span style="color:#121212">成功运行过的操作系统和Docker版本列表 · Issue #I1FZBJ · 泰晓科技/Linux Lab - Gitee.com</span></span><span style="background-color:#ebebeb"><img alt height="60" src="https://pic1.zhimg.com/v2-d9ed8cc7800391932b5b55eb259d8744_ipico.jpg" width="60" referrerpolicy="no-referrer"></span></a>
</div> 
<div style="text-align:start">
  
</div> 
<p style="color:#121212; margin-left:0; margin-right:0; text-align:start">为了优化体验，社区也在做更深度的适配，包括运行细节上的优化，比如说对不同 terminal 的支持，对权限差异的兼容，对设备文件缺失的处理等等，与此同时，<strong><em>社区也专门制作了免安装、即插即用、方便随身携带的 Linux Lab Disk</em></strong>，Linux Lab Disk 同步支持这些深度适配过的系统，更多的系统正在陆续适配优化中。</p> 
<p><img src="https://pic1.zhimg.com/80/v2-89be61980675f70559c854561e6a0164_720w.jpg" width="2518" referrerpolicy="no-referrer"></p> 
<p>128G固态跑车款 Linux Lab Disk</p> 
<p style="color:#121212; margin-left:0; margin-right:0; text-align:start">当前适配情况如下：</p> 
<h3 style="margin-left:0; margin-right:0; text-align:start">1. Manjaro</h3> 
<p style="color:#121212; margin-left:0; margin-right:0; text-align:start">适合 ArchLinux 系用户，软件丰富，滚动更新，更新快，需要有一定的 Linux 使用基础。默认搭配为 Gnome 桌面。</p> 
<p><img src="https://pic2.zhimg.com/80/v2-36b7ec2198778cda9ef309d81fc37139_720w.jpg" width="1920" referrerpolicy="no-referrer"></p> 
<h3 style="margin-left:0; margin-right:0; text-align:start">2. Deepin 20.2+</h3> 
<p style="color:#121212; margin-left:0; margin-right:0; text-align:start">适合国产软件用户，能直接运行大量 Windows 和安卓应用，国产软件适配和支持友好，有专门的应用下载中心。</p> 
<p><img src="https://pic1.zhimg.com/80/v2-90fa45e944b3eab807e91019ec226834_720w.jpg" width="1920" referrerpolicy="no-referrer"></p> 
<h3 style="margin-left:0; margin-right:0; text-align:start">3. Ubuntu 18.04/20.04/21.04</h3> 
<p style="color:#121212; margin-left:0; margin-right:0; text-align:start">适合不爱折腾求稳健的开发者用户，Ubuntu 是各企业的主流开发用系统（<em><strong>社区专门优化了桌面</strong>）</em>。</p> 
<p><img src="https://pic4.zhimg.com/80/v2-71d950f42ba46b05f35fd471fc01ec2b_720w.jpg" width="1920" referrerpolicy="no-referrer"></p> 
<h3 style="margin-left:0; margin-right:0; text-align:start">4. Fedora 34</h3> 
<p style="color:#121212; margin-left:0; margin-right:0; text-align:start">适合 rpm 包管理系列发行版用户。</p> 
<p><img src="https://pic2.zhimg.com/80/v2-1a3221587b67494dbbb913289e7add2d_720w.jpg" width="1920" referrerpolicy="no-referrer"></p> 
<h3 style="margin-left:0; margin-right:0; text-align:start">5. Kali Linux</h3> 
<p style="color:#121212; margin-left:0; margin-right:0; text-align:start">基于 Debian 定制的面向 Security 领域的系统，适合 Ubuntu/Debian 系用户。</p> 
<p><img src="https://pic4.zhimg.com/80/v2-c2b313fe974c7956ef1146d088e75cb7_720w.jpg" width="1920" referrerpolicy="no-referrer"></p> 
<h3 style="margin-left:0; margin-right:0; text-align:start">6. Linux Mint 20.2</h3> 
<p style="color:#121212; margin-left:0; margin-right:0; text-align:start">Linux Mint 继承了 Ubuntu 众多优点，集成大量「开箱即用」的软件包，加上良好的人机交互体验，使其自 2006 年发行后人气持续狂飙，目前已稳居 Distrowatch 全球 Linux 发行版排行榜的第一名，势头相当凶猛。</p> 
<p><img src="https://pic4.zhimg.com/80/v2-6fd25b93f9bf929aeff3616833c3356b_720w.jpg" width="1920" referrerpolicy="no-referrer"></p> 
<p style="color:#121212; margin-left:0; margin-right:0; text-align:start">另外，社区还研发了 Linux Lab Disk 的智能启动管理软件 vmboot，已经适配上述所有 Linux 发行版，安装以后插入 Linux Lab Disk，可以自动并行启动 Linux Lab Disk 中的系统，<em><strong>1 分钟内即可开展 Linux 内核等实验与开发</strong></em>。</p> 
<p style="color:#121212; margin-left:0; margin-right:0; text-align:start">vmboot 也支持 Windows，可以在 Windows 上自动并行启动 Linux Lab Disk 中的系统，还支持免关机自动切换 Windows 和 Linux Lab Disk 中的系统。多个 Linux Lab Disk 之间也支持相互启动或者切换。</p> 
<p style="color:#121212; margin-left:0; margin-right:0; text-align:start"><strong>在某宝检索“Linux Lab真盘”，可选 32G-512G 之间各种容量的高速或固态盘，也可任选上述某个发行版。如果不需要预装 Linux Lab，可以直接检索“Pocket Linux系统”。</strong></p>
