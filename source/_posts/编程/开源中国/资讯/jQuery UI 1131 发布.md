                                                                                            <p>jQuery UI 1.13.1 更新内容如下：</p> 
<h3>Widget factory</h3> 
<ul> 
 <li>修复：优化 <code>_untrackClassesElement</code> 监听器的附件</li> 
 <li>修复：jQuery-UI 小部件在用作模块时会中断的问题</li> 
</ul> 
<h3>自动完成</h3> 
<ul> 
 <li>修复：用延迟重写 Live Regions，而不是对其追加，所以 Live Region 不会在每个鼠标移动事件或快速遍历时被更新。</li> 
</ul> 
<h3>工具提示</h3> 
<ul> 
 <li>修复：不要在空内容时崩溃</li> 
</ul> 
<h3>交互</h3> 
<ul> 
 <li>修复：修复了将 Draggable 项目移入 Sortable 时的定位问题</li> 
 <li>修复：允许 0-height 的容器像 1.12.1 版本中那样可排序</li> 
</ul> 
<h3>构建</h3> 
<ul> 
 <li>变化：将 bug 跟踪器从 <a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fbug.jqueryui.com" target="_blank">bug.jqueryui.com</a> 更新到 GitHub</li> 
 <li>变化：从 Travis CI 切换到 GitHub Action</li> 
</ul> 
<p>更多详情可查看：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fjqueryui.com%2Fchangelog%2F1.13.1%2F" target="_blank">https://jqueryui.com/changelog/1.13.1/</a></p>
