                                                                                            <p>GoPlus v1.0.0-beta5 现已发布。GoPlus 是一门静态类型语言，与 Go 语言完全兼容。其代码样式类似脚本，并且比 Go 更具可读性，专门用于数据科学。</p> 
<p>具体更新内容如下：</p> 
<p><strong>新特性：</strong></p> 
<ul> 
 <li>支持 <code>[expr for k, v <- container, init; cond]</code></li> 
 <li>支持 return &#123;x.score for x <- a, x.name == name&#125;</li> 
 <li>支持 .spx/.gmx 文件（见<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fgoplus%2Fgop%2Fissues%2F762" target="_blank">#762</a>）</li> 
 <li>TestErrImport</li> 
</ul> 
<p><strong>changes<strong>：</strong> </strong></p> 
<ul> 
 <li>在 install/test 之前保存缓存</li> 
 <li>gop install/test bugfix：跳过 unknown switches</li> 
 <li>gop run bugfix：保持当前目录不变</li> 
</ul> 
<p>更新说明：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fgoplus%2Fgop%2Freleases%2Ftag%2Fv1.0.0-beta5" target="_blank">https://github.com/goplus/gop/releases/tag/v1.0.0-beta5</a></p>
