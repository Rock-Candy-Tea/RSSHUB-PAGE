 <li>全面切换到 WAT 后端, 支持 Linux/macOS/Windows 平台</li> 
 <li>简化命令行字命令, 不依赖任何第三方工具</li> 
 <li>暂时去掉对字符串和浮点数的例子</li> 
</ul> 
<p style="color:#000000; text-align:start">安装和测试:</p> 
<ol> 
 <li><code>go install github.com/wa-lang/wa@v0.2.0</code></li> 
 <li><code>wa init -name=_examples/hi</code></li> 
 <li><code>wa run _examples/hi</code></li> 
</ol>
