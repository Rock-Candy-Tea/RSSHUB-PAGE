                                                        <p>Spring Tools 4.11.0 现已发布。Spring Tools 4 是由 Spring 团队打造的下一代 Spring 开发工具，基本上是从零开始构建，融合了现代技术和开发者工具架构。它在单独的进程中运行，从构建之初就考虑到了性能问题，并且熟悉最新的 Spring 技术，为开发基于 Spring 的企业应用提供世界级支持。同时，全新版本的 Spring Tools 与 IDE 无关，可在各种编码环境中使用，支持 Eclipse、Visual Studio Code 与 Theia。</p> 
<p><strong>主要更新内容</strong></p> 
<ul> 
 <li>更新到 Eclipse 2021-06 版本（包括对 Java 16 的支持）</li> 
 <li>提供 Apple Silicon 平台 (ARM M1) 的早期访问版本</li> 
 <li>在 VS Code 中开始使用 Spring Boot</li> 
 <li>将 LightWeight 模式更改为 Standard 时不起作用</li> 
 <li>修复了 VSCode 中的语言服务器进程无法正常关闭的问题</li> 
 <li>修复了 VSCode 扩展 JDK 版本</li> 
 <li>优化了 Boot Dash，灵活且可配置的过滤器</li> 
 <li>自动从引导仪表板隐藏没有主类的项目</li> 
 <li>修复了 VSCode 扩展无法理解组配置中的 glob 模式的问题</li> 
 <li>修复了无法识别步骤和属性的问题</li> 
</ul>
