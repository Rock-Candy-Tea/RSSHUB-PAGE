                                                        <p>自汉得宣布开源以来，Choerodon猪齿鱼已被上千个组织所使用，帮助企业完成软件的生命周期管理，从而更快、更频繁地交付更稳定的软件。经过一千一百多天的奋战，2021年06月30日，Choerodon猪齿鱼迎来了<strong>1.0先行版</strong>正式发布，标志着Choerodon猪齿鱼走向的成熟和稳定，欢迎各位升级体验。</p> 
<ul> 
 <li>发布版本：1.0</li> 
 <li>发布时间：2021年06月30日</li> 
 <li>更新范围：全局</li> 
 <li>版本范围：开源版、商业版</li> 
</ul>

<ul>

<h1>1.0版本的重大调整</h1> 
<h2>使用开源组件库Choerodon UI 1.4.0 新主题-风铃紫</h2> 
<p><strong>Choerodon UI****开源组件库</strong>（缩写 C7N UI），拥有开箱即用的高质量 React 组件，全链路开发和设计工具体系，帮助企业级中后台产品提升开发效率。自V 0.1.0就开始支撑Choerodon猪齿鱼的前端组件，并在2021年2月4日发布稳定开发正式版 V 1.0，支持平滑升级，目前除了Choerodon猪齿鱼还支撑着HZERO、飞搭等产品的前端组件。</p>

</blockquote>

<h1>Choerodon猪齿鱼1.0商业版更新说明</h1> 
<p>商业版除了对开源版功能模块做了升级，还对以下模块做了不同程度的新增、修改和优化。</p> 
<h2>需求管理</h2> 
<p>贯穿着产品的整个生命周期，包括项目内部及外部用户的需求收集、需求审核、分析、拆解及开发进度的跟进。</p> 
<h3>新增功能</h3> 
<ul> 
 <li>需求审核增加创建需求；</li> 
 <li>需求审核支持导出需求；</li> 
 <li>导出需求增加描述字段；</li> 
 <li>需求池支持自定义列表字段显示、顺序以及列宽度；</li> 
 <li>需求池支持导入需求；</li> 
</ul> 
<p><img alt="image" src="https://minio.choerodon.com.cn/knowledgebase-service/0/a9bd318ff5e8476199d452855bc971ff@blob.png" referrerpolicy="no-referrer"></p> 
<h2>缺陷修复</h2> 
<ul> 
 <li>修复项目成员不能星标需求的问题；</li> 
</ul> 
<h2>规模化敏捷</h2> 
<p>以企业级的大规模敏捷框架SAFe为基础，对多项目并行开发、多团队业务需求整理及产品开发路线图等进行管理，帮助团队提高协作性，降低团队管理的复杂性。</p> 
<h3>新增功能</h3> 
<ul> 
 <li>版本关联新增与devops联动；</li> 
 <li>路线图支持自定义列表字段显示、顺序以及列宽度；</li> 
</ul> 
<h3>功能优化</h3> 
<ul> 
 <li>优化PI目标关联特性可选择是否隐藏已关联特性；</li> 
</ul> 
<h3>缺陷修复</h3> 
<ul> 
 <li>修复迭代日历时间范围不能选择1年的问题；</li> 
 <li>修复ART设置跳转到子项目报错的问题；</li> 
</ul> 
<h2>自动化测试</h2> 
<p>包括接口测试、性能测试、流量回归测试、UI测试，贯穿项目管理、敏捷开发DevOps全流程，提供敏捷化的持续测试工具，提高团队测试效率，保证质量。</p> 
<h3>新增功能</h3> 
<ul> 
 <li>性能测试-执行性能测试，新增支持为每次执行添加执行备注；</li> 
 <li>性能测试-执行性能测试，新增支持为每次性能测试执行设置持续时长；</li> 
 <li>性能测试-执行性能测试，新增支持永远循环执行选中的测试任务；</li> 
 <li>性能测试模块新增支持手动停止正在执行中的性能测试；</li> 
</ul> 
<h3>功能优化</h3> 
<ul> 
 <li>API测试任务开启定时设置后，加上【定时】的标签；树结构中也加上筛选；</li> 
</ul> 
<h3>缺陷修复</h3> 
<ul> 
 <li>修复了用例库-创建/修改API测试用例时，点击【保存并执行】，跳转至记录界面后，弹框未收起的问题；</li> 
</ul> 
<h2>质量管理</h2> 
<p>通过报表以图形化的方式直观的展示项目下应用代码质量数据，便于直观展示当前项目的总体代码质量及每个应用的代码质量，以供团队管理参考。</p> 
<h3>新增功能</h3> 
<ul> 
 <li>单表预览脚本；</li> 
 <li>表设计字段新增unsigned、time、blob、longblob类型支持；</li> 
 <li>表设计添加数据库类型、字段类型大小写、团队邮箱配置；</li> 
 <li>表模块可变更、模块编码可修改、可迁移模块表到其它模块；</li> 
 <li>期初数据导入添加操作选项，支持新建、更新、删除、重置选项；</li> 
 <li>新增自动提交、升级表设计功能；</li> 
 <li>添加PostgreSQL数据库支持；</li> 
 <li>克隆表结构；</li> 
 <li>表设计，表结构字段顺序可修改、可插入字段；</li> 
 <li>批量提交表；</li> 
</ul> 
<h3>功能优化</h3> 
<ul> 
 <li>表结构页面，备注字段宽度调整优化；</li> 
 <li>生成DDD模型代码，过时注解替换；</li> 
 <li>发布操作改为提交；</li> 
 <li>字段默认值添加EMPTY_STRING（空字符串）支持；</li> 
 <li>不同模块可支持同名表；</li> 
 <li>索引列可在索引列表展示；</li> 
 <li>关联关系区分版本；</li> 
 <li>弹性域字段创建可定制化；</li> 
</ul> 
<h2>应用市场</h2> 
<p>Choerodon应用市场是平台内应用服务组件与常用中间件的管理中心，支持平台下所有项目直接部署便可使用。</p> 
<h3>新增功能</h3> 
<ul> 
 <li>应用市场中，默认预置MySQL的配置与信息；</li> 
 <li>基础组件部署中，支持部署人员对MySQL进行主机部署与容器部署；</li> 
</ul>
