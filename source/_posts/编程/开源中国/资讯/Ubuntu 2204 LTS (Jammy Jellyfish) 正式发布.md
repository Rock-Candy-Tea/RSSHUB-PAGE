                                                                                            <p>Ubuntu 22.04 LTS 已正式<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Flists.ubuntu.com%2Farchives%2Fubuntu-announce%2F2022-April%2F000279.html" target="_blank">发布</a>，代号“Jammy Jellyfish”。这是一个长期支持版本，其开发商会提供 5 年的技术支持。同时发布的还有其它“风味”版本，包括 Ubuntu Budgie、Kubuntu、Lubuntu、Ubuntu Kylin、Ubuntu MATE、Ubuntu Studio 和 Xubuntu，这些版本将提供三年的支持。</p> 
<ul> 
 <li><strong>下载地址：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fubuntu.com%2Fdownload" target="_blank">https://ubuntu.com/download</a></strong></li> 
</ul> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-cd01ec765357fe8b76cda799ffa50d7c43f.png" referrerpolicy="no-referrer"></p> 
<p><strong>Ubuntu 22.04 LTS 主要变化</strong></p> 
<ul> 
 <li>基于所使用的 GNOME 42 桌面环境，显著改进了可用性、优化性能以及电池续航</li> 
 <li>将英特尔和 Raspberry Pi 显示驱动的桌面帧率提升至原来的 2 倍</li> 
 <li>完整支持近期发布的 Raspberry Pi 设备，从新的 Raspberry Pi Zero 2W 到 Raspberry Pi 4</li> 
 <li>引入内存安全的系统级编程语言 Rust</li> 
 <li>采用 OpenSSL v3，以及新的加密算法，以提高安全性</li> 
 <li>Ubuntu 22.04 LTS 为每个产品提供了不同的优化内核： 
  <ul> 
   <li> <p>Ubuntu Desktop 将在最新一代的认证设备 (linux-oem-22.04) 上自动选择加入 v5.17 内核</p> </li> 
   <li> <p>Ubuntu Desktop 在所有其他几代硬件上使用滚动 HWE 内核 (linux-hwe-22.04)。滚动 HWE 内核基于 22.04.0 和 22.04.1 版本的 v5.15 内核。</p> </li> 
   <li> <p>Ubuntu Server 默认为非滚动 LTS 内核 v5.15 (linux-generic)。</p> </li> 
   <li> <p>Ubuntu Cloud 和设备与合作伙伴合作使用优化的内核 v5.15+。</p> </li> 
  </ul> </li> 
 <li>Firefox 现在仅作为 snap 软件包在 Ubuntu 中提供</li> 
</ul>
