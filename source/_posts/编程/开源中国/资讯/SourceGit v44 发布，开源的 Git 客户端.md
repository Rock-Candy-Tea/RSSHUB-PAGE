                                                        <p>SourceGit v4.4 已经发布，开源的 Git 客户端</p> 
<p>此版本更新内容包括：</p> 
<p>[CHANGELOG]</p> 
<ol> 
 <li>使用【空格】快速STAGE/UNSTAGE选中文件</li> 
 <li>修复启动恢复上次打开标签页时，如果该仓库已经库删除，Crash的BUG</li> 
 <li>修正自动更新使用的URL</li> 
 <li>修复快速切换提交，导致Crash的BUG</li> 
 <li>优化CommitGraph在大量分支情况下的渲染效率</li> 
 <li>错误提示的显示优化</li> 
</ol> 
<p>详情查看：<a href="https://gitee.com/sourcegit/sourcegit/releases/v4.4">https://gitee.com/sourcegit/sourcegit/releases/v4.4</a></p>
