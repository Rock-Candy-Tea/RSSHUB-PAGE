                                                                                            <p>Apache Tomcat Native 1.2.31 稳定版已发布，Tomcat Native 项目可以让 Tomcat 使用 Apache 的 apr 包来处理引用文件和网络 IO 操作，从而提升性能。</p> 
<p>Apache Tomcat Native 库为当代 JDK 中没有的功能提供可移植的 API。它使用 Apache Portable Runtime 作为操作系统抽象层，使用 OpenSSL 进行 SSL 联网，可在生产环境中实现最佳性能。</p> 
<p>主要更新内容</p> 
<ul> 
 <li>使用 OpenSSL 1.1.1l 构建 Windows 二进制文件</li> 
 <li>修复了使用 OpenSSl 3.0.0 构建时的问题</li> 
</ul>
