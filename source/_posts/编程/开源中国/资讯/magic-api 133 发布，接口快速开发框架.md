                                                        <h3 style="text-align:left">本次更新内容：</h3> 
<ul> 
 <li>新增复制相对路径功能</li> 
 <li>新增注释中的<code>TODO</code>、<code>FIXME</code>高亮</li> 
 <li>修复推送重命名后的接口未被正确同步的<code>BUG</code></li> 
 <li>修复<code>Swagger</code>文档不显示<code>ResponseBody</code>的问题</li> 
 <li>修复全局搜索中关键字高亮不正确的<code>BUG</code></li> 
 <li>修复不回显<code>RequestBody</code>、<code>ResponseBody</code>的注释、验证信息的<code>BUG</code></li> 
 <li>修复全局搜索中有时展示不出代码的<code>BUG</code></li> 
 <li>修复左侧树复制接口时不能弹出编辑框的<code>BUG</code></li> 
 <li>修复分组下没有接口时无法被搜索的<code>BUG</code></li> 
 <li>修复拖拽左侧菜单后右侧编辑器大小未自适应的问题</li> 
 <li>优化<code>UI</code>权限配置，细化到分组&接口&函数&数据源上</li> 
 <li>优化全量推送&上传模式的逻辑，改为强一致实现。</li> 
 <li>优化顶部<code>tab</code>页，使其切换时始终展示在视野中</li> 
 <li>优化粘贴<code>RequestBody</code>时，自动对<code>key</code>添加双引号</li> 
 <li>优化代码提示新增方法参数、返回值说明</li> 
 <li>优化全局搜索对话框，隐藏滚动条、修正显示范围</li> 
</ul> 
<h4 style="text-align:start"><strong>项目介绍</strong></h4> 
<p style="text-align:left"><span style="background-color:#ffffff; color:#40485b">magic-api 是一个基于Java的接口快速开发框架，编写接口将通过magic-api提供的UI界面完成，自动映射为HTTP接口，无需定义Controller、Service、Dao、Mapper、XML、VO等Java对象即可完成常见的HTTP API接口开发</span></p> 
<h3 style="text-align:left"><span style="color:#40485b">特性</span></h3> 
<ul> 
 <li>支持MySQL、MariaDB、Oracle、DB2、PostgreSQL、SQLServer 等支持jdbc规范的数据库</li> 
 <li>支持非关系型数据库Redis、Mongodb</li> 
 <li>支持集群部署、接口自动同步。</li> 
 <li>支持分页查询以及自定义分页查询</li> 
 <li>支持多数据源配置，支持在线配置数据源</li> 
 <li>支持SQL缓存，以及自定义SQL缓存</li> 
 <li>支持自定义JSON结果、自定义分页结果</li> 
 <li>支持对接口权限配置、拦截器等功能</li> 
 <li>支持运行时动态修改数据源</li> 
 <li>支持Swagger接口文档生成</li> 
 <li>基于<a href="https://gitee.com/ssssssss-team/magic-script" target="_blank">magic-script</a>脚本引擎，动态编译，无需重启，实时发布</li> 
 <li>支持Linq式查询，关联、转换更简单</li> 
 <li>支持数据库事务、SQL支持拼接，占位符，判断等语法</li> 
 <li>支持文件上传、下载、输出图片</li> 
 <li>支持脚本历史版本对比与恢复</li> 
 <li>支持脚本代码自动提示、参数提示、悬浮提示、错误提示</li> 
 <li>支持导入Spring中的Bean、Java中的类</li> 
 <li>支持在线调试</li> 
 <li>支持自定义工具类、自定义模块包、自定义类型扩展、自定义方言、自定义列名转换等自定义操作</li> 
</ul>
