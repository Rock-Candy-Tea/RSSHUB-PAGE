                                                                                            <p>今天，Canonical 正式发布了 Ubuntu 21.04，其中包括原生的微软 Active Directory 集成、用于 Flutter 应用开发的 SDK，以及默认启用 Wayland 替换了已有 36 年历史的 X Window System。</p> 
<p>虽然 Ubuntu 21.04 是一个稳定的版本，但是 21.04 版本并没有长期支持版本，因此该版本将会在 2022 年 1 月结束支持。如果想要获得长期支持，目前 Ubuntu 20.04 LTS 可能是一个更好的选择，按照计划它会持续获得支持直至 2025 年。</p> 
<p><img alt height="394" src="https://static.oschina.net/uploads/space/2021/0424/071910_J29x_4937141.png" width="700" referrerpolicy="no-referrer"></p> 
<p>Ubuntu 21.04 将 Linux 内核升级至了 5.11，这意味着支持较新的硬件，不过 Canonical 表示他们还需要更多时间打磨 GNOME，因此在此次更新中决定暂时不引入最新的 GNOME 40。</p> 
<p>通过引入 Active Directory（AD） 支持，Ubuntu 21.04 增加了从 AD 域控制器配置系统设置的功能。使用组策略客户端，系统管理员可以在所有连接的客户端上指定安全策略，如密码策略、用户访问控制、桌面环境设置、登录界面、背景和偏好的应用程序。</p> 
<p>此外，Ubuntu 21.04 默认使用 Wayland，这是安全方面的一个重大飞跃。Firefox、OBS Studio 和许多用 Electron 和 Flutter 构建的应用程序都能自动使用 Wayland，以获得更平滑的图形和更好的比例缩放。</p> 
<p>Flutter SDK 的集成还可以让开发者轻松地发布多平台 Flutter 应用，让数百万 Linux 桌面用户可以一键安装 Flutter 应用。</p> 
<p>Flutter 产品经理 Chris Sells 在官方博客中分享道："Canonical 一直是 Flutter 的杰出贡献者，在 Flutter 中实现了对 Linux 桌面的支持，并为将高质量的 Flutter 应用引入 Linux 提供了机会。我们期待着继续与之合作，使 Flutter 成为应用开发者的最佳选择，无论他们的目标是哪个平台"。</p> 
<p>此次更新中还包括一个重新设计的黑暗主题 —— Yaru，并且在导航方面做了无障碍改进，加入了新的文件图标以适配黑暗主题。Yaru 由社区维护，并得到 Canonical 的支持。</p> 
<h3>Ubuntu 21.04 的其他改进：</h3> 
<ul> 
 <li>增加了对智能卡（smartcard）认证的支持；</li> 
 <li>桌面视图正确处理拖放操作；</li> 
 <li>可以从设置中改变电源配置文件；</li> 
 <li>默认情况下启用了 Pipewire 支持；</li> 
 <li>对 Microsoft SQL Server 进行性能优化和联合支持；</li> 
 <li>安装程序包括新的选项，可以在密码丢失的情况下创建一个恢复密钥来解密加密磁盘；</li> 
 <li>一些应用程序被更新到它们的 GNOME 4.0 版本，例如 Firefox、LibreOffice 和 Thunderbird；</li> 
 <li>Ubuntu Server 得到了 Rails 6 的支持；</li> 
</ul>
