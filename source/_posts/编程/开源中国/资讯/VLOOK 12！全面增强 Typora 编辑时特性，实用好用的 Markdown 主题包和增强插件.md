 <p style="margin-left:0; margin-right:0">VLOOK™ 是针对<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Ftypora.io%2F" target="_blank">Typora</a>（跨平台 Markdown 编辑器）的 <strong>主题包</strong> 和 <strong>增强插件</strong>（针对导出的 HTML 文件)。</p>

 <p style="margin-left:0; margin-right:0">VLOOK™ 属于开源软件（遵从 <strong>MIT License</strong>），也是 ![OSChina]<strong><a href="https://www.oschina.net/p/vlook">开源中国</a></strong> 推荐的国产开源产品、Typora 的首个增强插件。</p>

<p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><strong>VLOOK™ 的所有特性清单，详见快速入坑 → <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fmadmaxchow.github.io%2FVLOOK%2Findex.html%23%E5%BF%AB%E9%80%9F%E5%85%A5%E5%9D%91" target="_blank"><span>一键了解</span></a> （<a href="https://madmaxchow.gitee.io/vlook/index.html#%E5%BF%AB%E9%80%9F%E5%85%A5%E5%9D%91">备用链接</a>）</strong></p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><strong>V12.0 版本为各位 Markdown 粉带以下重大惊喜！！！</strong></p> 
<h2 style="text-align:start">NEW! 全面增强 Typora 编辑时特性</h2> 
<ul> 
 <li>【大纲显示模式】不同显示模式（自动收起 / 自动全部展开 / 无大纲）的样式优化</li> 
 <li>【大纲封面、封底】大纲面板中标识中符合 VLOOK™ 标准的「封面、封底」条目</li> 
 <li>【侧边栏标题】大纲面板、文件面板标题栏支持显示当前 VLOOK™ 主题名称</li> 
 <li>【输入态增强】标识出段落、表格行的当前行，以及链接 / code / kbd 内图片 / YAML</li> 
 <li>【融合菜单、快速打开】融合菜单 ( Win 版 )、快速打开适配主题，并提供好友的样式</li> 
 <li>【表格行增强】鼠标在表格上悬停时，能标识出对应悬停的表格行</li> 
 <li>【图片资源异常】通过不同样式，明显地标识出无效、无法加载的图片资源，便于及时进行修正</li> 
 <li>【全局 UI 1】统一 Typora 编辑/导出提示等相关 UI 字体、图标颜色，与当前 VLOOK™ 主题保持一致</li> 
 <li>【全局 UI 2】统一了 Typora 的源代码模式强制显示同一等宽字体，并提供更宽阔的可视空间</li> 
</ul>
