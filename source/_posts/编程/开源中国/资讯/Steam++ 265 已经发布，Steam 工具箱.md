                                                                                            <p>Steam++ 2.6.5 已经发布，Steam 工具箱</p> 
<p>此版本更新内容包括：</p> 
<h3>版本亮点</h3> 
<ol> 
 <li>ASF 升级至 <strong>V5.2.2.4</strong></li> 
 <li>Android 12 适配</li> 
 <li>Android 上的 ASP.NET Core 升级至 2.2.5</li> 
 <li>Desktop 上的 Avalonia 升级至 0.10.12</li> 
</ol> 
<h3>修复问题</h3> 
<ol> 
 <li>修复 Steam Beta 版账号切换失效</li> 
 <li>修复 ASF 在线配置文件生成器链接错误</li> 
 <li>修复 Linux 上运行时闪退</li> 
 <li>修复 Windows 7 上点击任务栏图标没有最小化</li> 
 <li>改进 打开 Web 链接失败时候的错误提示</li> 
 <li>改进 JumpList 在 Windows 10.0.10586 上使用 UWP API</li> 
 <li>改进 Windows 安装包默认路径使用 $PROGRAMFILES64</li> 
 <li>改进 Windows 安装版 AppData 与 Cache 文件夹不再使用根目录</li> 
 <li>改进 Android Apk 包使用单一包，不再根据 ABI 拆分</li> 
 <li>改进 Android 本地加速与 ASF 使用前台服务运行</li> 
 <li>改进 本地加速 增加 重置 Hosts 文件 与 打开 Hosts 所在文件夹</li> 
 <li>改进 Hosts 文件存在多个 Mark 导致失败的错误提示</li> 
</ol>
