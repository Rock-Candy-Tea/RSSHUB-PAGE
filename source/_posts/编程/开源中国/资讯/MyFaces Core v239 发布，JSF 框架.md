                                                        <p>MyFaces Core v2.3.9 发布了，MyFaces 是 JSF (JavaServerFaces) Web框架（JSR 127）的一个实现。JavaServer（tm）Faces Web 框架是一个新的实现 MVC 模式的规范。</p> 
<p><strong>主要更新内容</strong></p> 
<ul> 
 <li>修复了当 facet 包含无效 ID 字符时会创建无效的 ID 的问题</li> 
 <li>修复由于 CDIManagedBeanHandlerImpl 中缺少检查，导致大量的 "@SessionScoped 不存在于当前线程中" 异常的问题</li> 
 <li>Disabled/ReadOnly HTML 属性不应使用布尔值</li> 
 <li>资源：设置 cache-control-header 而不是 expires-header</li> 
</ul>
