                                                        <p><span style="color:#333333">Next.js 是一个用于在服务端渲染 React 应用程序的简单框架，目前发布了 </span>v12.0.7 和 v12.0.8-canary.0 版本，分别有如下改动：</p> 
<h3>v12.0.7</h3> 
<h3><strong>核心变化</strong></h3> 
<ul> 
 <li>添加 TailwindCSS JIT 模式重新加载测试：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fvercel%2Fnext.js%2Fpull%2F32130" target="_blank"><u>#32130</u></a></li> 
 <li>确保 createServer 的 conf 类型未更改：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fvercel%2Fnext.js%2Fpull%2F32134" target="_blank">#32134</a></li> 
</ul> 
<p>更新公告：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fvercel%2Fnext.js%2Freleases%2Ftag%2Fv12.0.7" target="_blank">https://github.com/vercel/next.js/releases/tag/v12.0.7</a></p> 
<h3>v12.0.8-canary.0</h3> 
<h3><strong>核心变化</strong></h3> 
<ul> 
 <li>修复子文件夹中间件的 no-server-import-in-page eslint 规则：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fvercel%2Fnext.js%2Fpull%2F32139" target="_blank">#32139</a> </li> 
</ul> 
<h3><strong>文档更改</strong></h3> 
<ul> 
 <li>固定 Yarn 和 NPM<code>dev</code>交换参数：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fvercel%2Fnext.js%2Fpull%2F32135" target="_blank">#32135</a> </li> 
</ul> 
<p>发布公告：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fvercel%2Fnext.js%2Freleases%2Ftag%2Fv12.0.8-canary.0" target="_blank">https://github.com/vercel/next.js/releases/tag/v12.0.8-canary.0</a></p>
