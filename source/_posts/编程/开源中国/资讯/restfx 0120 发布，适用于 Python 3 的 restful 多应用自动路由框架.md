                                                                                            <p>restfx 0.12.0 已经发布，一个适用于 Python 3 的 restful 多应用自动路由框架。</p> 
<p>此版本更新内容包括：</p> 
<ul> 
 <li>优化 接口页面测试时，支持对响应类型为图片</li> 
 <li>优化 接口页面测试时，支持对JSON数据进行格式化显示</li> 
 <li>优化 接口页面测试时，支持将一些不支持的响应类型提供下载为文件</li> 
 <li>优化 请求时的参数类型或数据无效时的提示</li> 
 <li>优化 接口页面支持深色模式</li> 
 <li>添加 框架插件支持</li> 
</ul> 
<p>详情查看：<a href="https://gitee.com/hyjiacan/restfx/releases/0.12.0">https://gitee.com/hyjiacan/restfx/releases/0.12.0</a></p>
