                                                        <p>XQuartz 是一项开源项目，旨在开发一个可以在 macOS 上运行的 <a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fx.org%2F" target="_blank">X.Org</a> X Window System 版本。它与支持的库和应用程序一起构成了 X11.app，苹果在 10.5 至 10.7 版本的 OS X 中提供了这一应用。</p> 
<h3>操作系统要求：</h3> 
<p>安装该软件包需要 macOS 10.9 或更高版本；</p> 
<h3>默认的 X11 服务器：</h3> 
<p>如果这是你第一次安装 XQuartz，你需要注销并重新登录以更新你的 DISPLAY 环境变量；</p> 
<h3>2.8.1 版本中的变化：</h3> 
<ul> 
 <li>将软件包的名称调整为 XQuartz.pkg，而不是 Xquartz.pkg，以保持一致性 (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FXQuartz%2FXQuartz%2Fissues%2F137" target="_blank">#137</a>)；</li> 
 <li>重新添加了旧版本的 libpng，以便与 wine 的二进制兼容 (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FXQuartz%2FXQuartz%2Fissues%2F151" target="_blank">#151</a>)；</li> 
 <li>更新了 xorg-server 以解决 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fcve.mitre.org%2Fcgi-bin%2Fcvename.cgi%3Fname%3DCVE-2021-3472" target="_blank">CVE-2021-3472</a></li> 
 <li>更新了 mesa 和其他项目至更新的上游版本；</li> 
 <li>修正了字体目录设置 (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FXQuartz%2FXQuartz%2Fissues%2F152" target="_blank">#152</a>)；</li> 
</ul> 
<p>更多详情可查看：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.xquartz.org%2Freleases%2FXQuartz-2.8.1.html" target="_blank">https://www.xquartz.org/releases/XQuartz-2.8.1.html</a></p>
