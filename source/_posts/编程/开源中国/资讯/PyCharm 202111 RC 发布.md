                                                                                            <p>PyCharm 2021.1.1 RC 现已发布。该版本对之前发布的 2021.1 RC 版本进行了一些快速错误修复。</p> 
<p><strong>主要更新内容</strong></p> 
<ul> 
 <li>"在文件中查找" 现在可以正常使用</li> 
 <li>禁用了 Python Console 的自动导入功能。在 Python Console 中，代码补完现在可以正常工作</li> 
 <li>Jupyter Notebooks：在预览和编辑窗口中以滚动的方式自动同步</li> 
 <li>Jupyter Notebooks：预览窗格保持活跃和并实时更新</li> 
 <li>如果自定义的 virtualenv 目录被删除，PyCharm 将不再扫描主目录中的 virtualenvs。并且，PyCharm 现在会将 virtualenvs 保留在 projectvenv 路径中</li> 
 <li>“None” 类型现在不显示在函数注释/类型提示内的代码补完中</li> 
 <li>修复 Gradle 运行配置可能不使用 project dir 作为工作目录的问题</li> 
 <li>扩大评估表达式弹出窗口</li> 
 <li>只允许在 EAP 版本的 IDE 中使用 assertConsistency</li> 
 <li>修复更新后下拉菜单中缺少字体值的问题</li> 
</ul>
