<div class="content">
                                                                    
                                                        <p>FydeOS for PC 发布 v12.0，在这一次里程碑更新中，Chromium 浏览器升级至 89.0.4389.*，同步了上游所有关于浏览器的新特性：</p> 
<ul> 
 <li> <p>Chromium 浏览器添加「搜索标签页」按钮；日常浏览网页如果开启多个标签页，该功能可以帮助你轻松快速地切换到指定的标签页。</p> </li> 
</ul> 
<p><img alt height="667" src="https://fydeos.com/wp-content/uploads/2021/03/v12-searchtabs.jpg" width="1000" referrerpolicy="no-referrer"></p> 
<ul> 
 <li> <p>在桌面的右下角添加全新设计的「文件缓冲区」卡片以及「已固定的文件」卡片，帮助你快速打开刚才下载或者编辑过的文件。</p> </li> 
</ul> 
<p><img alt height="667" src="https://fydeos.com/wp-content/uploads/2021/03/v12-stickyfiles.jpg" width="1000" referrerpolicy="no-referrer"></p> 
<ul> 
 <li> <p>优化过的「虚拟桌面」功能帮助你更好地管理每个虚拟桌面的属性；每个应用程序可以分别配置在指定的或者所有桌面打开；虚拟桌面的相关设置在系统重启之后亦会保留。</p> </li> 
</ul> 
<p><img alt height="667" src="https://fydeos.com/wp-content/uploads/2021/03/v12-vdesktop.jpg" width="1000" referrerpolicy="no-referrer"></p> 
<ul> 
 <li> <p>当你需要粘贴内容时，按下 <code>❖</code>(win) + <code>v</code>（而不是平常的 <code>ctrl</code> + <code>v</code>）即可呼出全新增加的「超级剪贴板」功能，你可以查看、编辑并选择剪贴板内 5 项之前复制的内容粘贴。</p> </li> 
</ul> 
<p><img alt height="667" src="https://fydeos.com/wp-content/uploads/2021/03/v12-clipboard.jpg" width="1000" referrerpolicy="no-referrer"></p> 
<p>除了浏览器之外，其他值得一提的更新有：</p> 
<ul> 
 <li> <p>在右下角系统菜单中可以呼出优化过的系统内置的截屏工具；添加了内置的屏幕视频录制功能。</p> </li> 
 <li> <p>加入针对 Surface 系列硬件内核驱动的优化：缓解了休眠时电池损耗过快的问题；重启设备之前屏幕亮度的设定将在重启之后予以保留。</p> </li> 
 <li> <p>为某些不携带触屏的设备添加了「旋转屏幕」的功能按键。</p> </li> 
 <li> <p>优化了系统设置内「FydeOS 设置」的相关功能。</p> </li> 
 <li> <p>优化了「FydeOS 系统控制」插件。</p> </li> 
 <li> <p>优化了「FydeOS 远程桌面」服务。</p> </li> 
 <li> <p>优化了安卓子系统的兼容性和稳定性。</p> </li> 
</ul>
