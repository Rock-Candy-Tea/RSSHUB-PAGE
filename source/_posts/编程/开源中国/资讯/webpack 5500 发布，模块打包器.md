                                                                                            <p>webpack 5.50.0 现已发布。webpack 是一个模块打包器，主要目的是在浏览器上打包 JavaScript 文件。</p> 
<p>具体更新内容如下：</p> 
<p><strong>Features</strong></p> 
<ul> 
 <li>hashbangs ( <code>#! ...</code>) 现在由 webpack 处理 
  <ul> 
   <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ftc39%2Fproposal-hashbang" target="_blank">https://github.com/tc39/proposal-hashbang</a></li> 
  </ul> </li> 
</ul> 
<p><strong>Performance</strong></p> 
<ul> 
 <li>默认情况下禁用缓存压缩，因为它会使性能变差 
  <ul> 
   <li>仍然可以针对特定场景再次启用</li> 
  </ul> </li> 
 <li>减少缓存序列化期间的分配次数 
  <ul> 
   <li>这提高了性能和内存使用率</li> 
  </ul> </li> 
</ul>
