<div class="content">
                                                                                            <p style="text-align:left">ShopWind <span style="background-color:#ffffff; color:#333333">是一款基于Yii2的PHP</span><span style="background-color:#ffffff; color:#000000">开源的多商户电商系统。标准版代码全开源，可以免费用于商业或二次开发。ShopWind </span>v3.3.1 新增了<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.shopwind.net" target="_blank">社区团购</a>模块，可以搭建类似于橙心优选，美团优选等社区团购O2O类型的网站系统平台。</p> 
<p style="text-align:left"><img alt height="559" src="https://oscimg.oschina.net/oscnet/up-1b7dd28090e0ba71ff1a2f244290ae127e3.JPEG" width="776" referrerpolicy="no-referrer"></p> 
<ul> 
 <li>【新增】社区团购模块</li> 
 <li>【新增】后台社区团购门店管理和审核，团长返佣比率设置，可单独设置每个团长的佣金率</li> 
 <li>【新增】社区团购订单交易完成后，给团长分佣</li> 
 <li>【新增】APP/小程序端 团长入驻模块</li> 
 <li>【新增】APP/小程序端 团长处理自提订单，通知提货功能</li> 
 <li>【新增】APP/小程序端 收益显示 及 收益明细记录</li> 
 <li>【新增】APP/小程序端 生成自提门店推广海报、商品分享推广海报</li> 
 <li>【新增】APP/小程序端 选择附近自提门店功能，并显示距离，从近到远显示门店</li> 
 <li>【新增】批发价功能，可以同时支持零售与批发下单模式，支持批量下单</li> 
 <li>【新增】后台 微信-小程序设置</li> 
 <li>【修改】预存款设置表（depisit_setting）增加字段（guider_rate）</li> 
 <li>【修改】订单表（order）增加字段（guider_id）</li> 
 <li>【修改】微信设置表（weixin_setting） 修改字段“auto_login” 为 “autologin”，增加 code 字段</li> 
 <li>【修复】系统自动确认收货 或 定时计划任务 处理已发货订单 为交易完成时，没有排除退款中订单的问题</li> 
</ul> 
<p style="text-align:left">详情查看：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.shopwind.net" target="_blank">官方网站</a> / <a href="https://gitee.com/shopwind/yii2-shopwind">源码下载</a></p>
