                                                        <p>Apache AGE 是一个 PostgreSQL 扩展，提供图数据库功能。AGE 是 A Graph Extension 的首字母缩写。该项目的目标是创建可以处理关系和图模型数据的单一存储，以便用户可以使用标准 ANSI SQL 和图查询语言 openCypher。</p> 
<p>Apache AGE 0.5.0 发布，更细内容如下：</p> 
<p>该版本更新内容如下：</p> 
<ul> 
 <li>增加了对在 Hash Joins 中使用 Agtype 的支持;</li> 
 <li>增加了对 DELETE 作为 Cypher 语句中最后一个子句的支持；</li> 
 <li>修正了 DELETE、REMOVE 和 SET 不能更新 vertex/edge 的错误；</li> 
 <li>增加了对 SKIP 和 LIMIT 的支持，作为 WITH 和 RETURN 语句的子句；</li> 
 <li>增加了对 CASE 语句的支持；</li> 
 <li>增加了对 PostgreSQL EXPLAIN 工具命令的支持；</li> 
 <li>添加了 agtype 到数字的操作重载；</li> 
 <li>添加了 ANTLR4 语法文件，可与 JDBC 和 NodeJS 驱动程序一起使用；</li> 
 <li>添加了 NULL 回归测试；</li> 
 <li>添加了标签函数；</li> 
</ul>
