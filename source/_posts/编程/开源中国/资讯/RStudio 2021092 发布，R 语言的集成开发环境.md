                                                                                            <p>RStudio 2021.09.2 现已发布。RStudio 是 R 语言的集成开发环境，分为面向桌面用户 IDE 和 Linux R 服务器版编辑器两种编辑器，采用 AGPL v3 与 <a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fwww.rstudio.com%2Fabout%2Feula%2F">RStudio License Agreement</a> 双协议授权。该版本的更新内容如下：</p> 
<p style="text-align:left"><span style="color:#000000"><strong>Features</strong></span></p> 
<ul> 
 <li>支持带有 UCRT 构建的 R (>= 4.2.0) 的 UTF-8 代码页 (#9824)</li> 
 <li>在 Windows 上支持 R (>= 4.2.0) (#9792)</li> 
 <li>R 4.2.0 中对 R 图形引擎版本 15 的初步支持。(#10058)</li> 
</ul> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><strong>Bug 修复</strong></p> 
<ul> 
 <li>修复导致在可视化编辑器中的引用中添加反斜杠的问题 (#10075)</li> 
 <li>修复在多用户 macOS 环境中无法启动的问题 (#9945, #10267)</li> 
</ul> 
<p>更新说明：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.rstudio.com%2Fproducts%2Frstudio%2Frelease-notes%2F" target="_blank">https://www.rstudio.com/products/rstudio/release-notes/</a></p>
