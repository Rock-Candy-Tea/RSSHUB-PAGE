                                                        <p>Apache PDFBox 2.0.24 已经发布。Apache PDFBox 库是一个开源的用于处理 PDF 文档的 Java 工具。</p> 
<p><strong>主要更新内容</strong></p> 
<ul> 
 <li>对特定的 PDF 文件渲染缓慢</li> 
 <li>在 PageDrawer.shadingFill上 的渲染非常缓慢</li> 
 <li>渲染中丢失字形</li> 
 <li>对 JPEG 图像的错误分类导致在 pdf 文档中添加空白图像</li> 
 <li>嵌入文件不能从具有多级 EmbeddedFiles 树的 PDF 文件中提取出来</li> 
 <li>COSObjectKey::fixGeneration 问题</li> 
 <li>从 PDF 中提取文本时出错：无法读取嵌入的 Type1 字体 FDFBJU+NewsGothic</li> 
 <li>在识别 PDF 注释符号 % 作为 PDF 名称的标记分隔符时出错</li> 
 <li>读取损坏的字典时出现堆栈溢出</li> 
 <li>破折号图案 [0] 应该是不可见的</li> 
 <li>java.io.IOException: 页面树根必须是一个字典</li> 
 <li>快照部署不起作用</li> 
 <li>使用用户名 + 密码的 TSAClient</li> 
 <li>将 PDFRenderer 传递给 PDFPrintable 构造函数</li> 
 <li>CreateCheckBox 的例子画出了太大的、被剪切的复选标记</li> 
 <li>为 HelloWorld 示例创建测试</li> 
 <li>允许只制作时间戳签名 “LTV”</li> 
</ul>
