                                                        <p>Go 1.17 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgroups.google.com%2Fg%2Fgolang-announce%2Fc%2FgJE7OtHlRbM%2Fm%2F21x8zAR-AAAJ" target="_blank">发布</a>了首个 RC 版本，正式版将于下个月推出。</p> 
<p>以下内容摘录自 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Ftip.golang.org%2Fdoc%2Fgo1.17" target="_blank">Go 1.17 release notes</a>。</p> 
<p>Go 1.17 包含三个针对语言特性的改进：</p> 
<ul> 
 <li>支持<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Ftip.golang.org%2Fref%2Fspec%23Conversions_from_slice_to_array_pointer" target="_blank">从 slice （切片）到数组指针的转换</a></li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Ftip.golang.org%2Fpkg%2Funsafe%23Add" target="_blank"><code>unsafe.Add</code></a>：<code>unsafe.Add(ptr, len)</code>将<code>len</code>添加<code>ptr</code>并返回更新的指针<code>unsafe.Pointer(uintptr(ptr) + uintptr(len))</code></li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Ftip.golang.org%2Fpkg%2Funsafe%23Slice" target="_blank"><code>unsafe.Slice </code></a>: 对于<code>*T</code>类型的<code>ptr</code>表达式，<code>unsafe.Slice(ptr, len) </code>返回一个<code>[]T</code>类型的切片，其底层数组从<code>ptr</code>开始， 其长度和容量为<code>len</code></li> 
 <li>……</li> 
</ul> 
<p>此外，对于 Mac 平台，Go 1.17 要求在 macOS 10.13 High Sierra 或更高版本上运行，并已停止支持以前的系统版本。对于 Windows 平台，Go 1.17 添加了对 64 位 ARM 架构的支持。对于 OpenBSD，Go 1.17 添加了对 64 位 MIPS 架构的支持。</p> 
<p><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Ftip.golang.org%2Fdoc%2Fgo1.17" target="_blank">更多内容查看 release notes</a>。</p> 
<p>升级命令</p> 
<pre><code>$ go get golang.org/dl/go1.17rc1
$ go1.17rc1 download</code></pre> 
<p>下载源码或二进制文件：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgolang.org%2Fdl%2F%23go1.17rc1" target="_blank">https://golang.org/dl/#go1.17rc1</a></p>
