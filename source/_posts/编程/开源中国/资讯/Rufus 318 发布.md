                                                                                            <p><span style="background-color:#ffffff; color:#333333">Rufus 是一个可以帮助格式化和创建可引导 USB 闪存盘的工具，它可以快速把 ISO 格式的系统镜像文件快速制作成可引导的 USB 启动安装盘，支持 Windows 或 Linux 启动。3.18 版本更新内容如下： </span></p> 
<ul> 
 <li>修复 DLL sideloading 漏洞</li> 
 <li>修复在 Windows 11 上运行时的 SO → ESP 创建问题</li> 
 <li>修复了在禁用 DHCP 服务时检测网络连接时的问题</li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.freedos.org%2F" target="_blank">将 FreeDOS</a> 更新到 1.3 版</li> 
 <li>为就地升级添加绕过 Windows 11 限制的功能</li> 
 <li>将 Miracle Linux 8 添加到 Red Hat exceptions 列表中</li> 
 <li>其他内部修复和改进</li> 
</ul> 
<p>更新说明：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Frufus.ie%2Fzh%2F" target="_blank">https://rufus.ie/zh/</a></p>
