<div class="content">
                                                                                            <p><span style="color:#333333">Appsmith 是一个用于构建管理面板、CRUD 应用程序和工作流的框架，它允许拖放组件来构建仪表板、使用 JavaScript 对象编写逻辑并连接到任何 API、数据库或 GraphQL 源。</span></p> 
<p><span style="color:#333333">目前 Appsmith 发布了 1.6.9 版本，带来如下改动：</span></p> 
<p><span style="color:#333333"><strong>特性</strong></span></p> 
<ul> 
 <li>添加 Helm configMap 以更新应用程序配置 (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fappsmithorg%2Fappsmith%2Fissues%2F10157" target="_blank">#10157</a>)</li> 
 <li>实体资源管理器改造：现在实体资源管理器仅显示选定的页面实体，减少了具有多个页面的应用程序的混乱。（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fappsmithorg%2Fappsmith%2Fissues%2F9811" target="_blank">#9811</a>）</li> 
</ul> 
<p><strong>Bug 修复</strong></p> 
<ul> 
 <li><span style="color:#24292f">修复 Select 和 Treeselect WIdget 的验证 </span> (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fappsmithorg%2Fappsmith%2Fissues%2F10760" target="_blank">#10760</a>)</li> 
 <li><span style="color:#24292f">添加 iFrame ，用于在文档查看器中加载 Base64 PDF 数据 </span> (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fappsmithorg%2Fappsmith%2Fissues%2F10770" target="_blank">#10770</a>)</li> 
 <li><span style="color:#24292f">评估 PrivateWidgets ，</span><span style="color:#2e3033">以修复列表小部件子组件评估问题</span> (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fappsmithorg%2Fappsmith%2Fissues%2F10737" target="_blank">#10737</a>)</li> 
 <li>重新验证脚本不加载的问题 (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fappsmithorg%2Fappsmith%2Fissues%2F10764" target="_blank">#10764</a>)</li> 
 <li><span style="color:#24292f">当应用程序连接到 git 时，为重定向 url 添加分支参数，同时授权 Google Sheet </span>(<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fappsmithorg%2Fappsmith%2Fissues%2F10715" target="_blank">#10715</a>)</li> 
 <li><span style="color:#24292f">将查询参数添加到选项/表单</span> (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fappsmithorg%2Fappsmith%2Fissues%2F10681" target="_blank">#10681</a>)</li> 
 <li><span style="color:#24292f">将 meta 添加到 eval 循环，并在默认值更改时更新它</span> (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fappsmithorg%2Fappsmith%2Fissues%2F10401" target="_blank">#10401</a>)</li> 
 <li><span style="color:#24292f">当代码中存在未使用的变量时显示 lint</span> (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fappsmithorg%2Fappsmith%2Fissues%2F10599" target="_blank">#10599</a>)</li> 
 <li><span style="color:#24292f">从源应用程序复制克隆应用程序的评估版本，以修复动态动态绑定评估，进而支持没有双重转义的转义字符</span> (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fappsmithorg%2Fappsmith%2Fissues%2F10702" target="_blank">#10702</a>)</li> 
 <li>图标按钮移除默认的绿色 (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fappsmithorg%2Fappsmith%2Fissues%2F10573" target="_blank">#10573</a>)</li> 
 <li><span style="color:#2e3033">重置密码时清空会话 </span>(<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fappsmithorg%2Fappsmith%2Fissues%2F9955" target="_blank">#9955</a>)</li> 
</ul>
