                                                        <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">Porteus Kiosk 是一个轻量级的 Linux 操作系统，Porteus Kiosk 5.3.0 现已正式发布，这个版本中的主要升级包括：Linux Kernel 5.10.73、Chrome 浏览器 93.0.4577.82 和 Mozilla Firefox 91.2.0 ESR，其他更新内容包括：</p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>Libinput 已经成为我们的默认输入驱动，因为它为 Firefox 浏览器提供了原生的触控手势支持。在 Porteus Kiosk 5.2.0 或以下版本中校准的触摸屏将继续使用 "evdev "驱动，以保持与现有配置的兼容性。</li> 
 <li>可以为 Firefox 和 Chrome 启用虚拟键盘扩展。</li> 
 <li>增加了一个选项，可以为这两个浏览器启用实验性的硬件视频解码功能。</li> 
 <li>屏幕上的按钮可以重新定位到 kiosk 屏幕上的不同位置。</li> 
 <li>从系统中删除了 Adobe Flash 插件。通过我们的 "定制构建" 服务，仍然可以支持 Flash 独立应用程序。</li> 
 <li>更新了'dns_server='参数，以便对 DHCP 配置也起作用。</li> 
 <li>增加了'sound open firmware'软件包，允许在选定的硬件上使用替代的声音驱动。</li> 
 <li>更新了 Porteus Kiosk Server 变体 "Basic" 中的管理面板。</li> 
 <li>其他错误修正和改进。</li> 
</ul>
