                                                                                            <h3 style="margin-left:0px; margin-right:0px; text-align:left">开源轻量级应用交付平台</h3> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">渠成云原生应用交付平台是北京渠成软件有限公司自主设计、研发的开源轻量级应用交付平台。它底层基于容器和 Kubernetes 技术，通过 Helm 进行应用的封装，提供了渠成云原生应用市场、服务管理、服务监控与告警、日志管理与审计、集群管理等平台功能，是一款使用简单，功能完备的云原生应用交付平台。</p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">视频介绍请看：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.qucheng.com%2Fdynamic%2Fqucheng-v1.0-11.html" target="_blank">https://www.qucheng.com/dynamic/qucheng-v1.0-11.html</a></p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">我们秉承 Unix 哲学，坚持用简单的方法来解决问题，因此平台在设计之初就坚持简单的原则，将底层复杂的技术进行了抽象和屏蔽，让使用者更专注于平台应用本身的体验，同时提供了应用的一键安装、升级、备份与回滚功能。<strong>渠成平台，您可信赖的应用交付平台</strong></p> 
<p style="color:#222222"><strong>渠成平台 1.3 版本发布，新增数据库管理功能、服务导航页，优化服务备份管理，多款应用发布了新版本，为您更好地使用平台和应用、享受云原生应用的快捷与方便提供了更有力的支撑。</strong></p> 
<h3 style="margin-left:0; margin-right:0">平台主要新增功能如下：</h3> 
<blockquote> 
 <p>通过集成Adminer新增数据库管理功能，方便您管理平台上的所有数据库。</p> 
</blockquote>

<blockquote> 
 <p>新增服务导航页，通过把常用应用固定在导航页可以快速访问已安装的应用，还可以通过名称全局检索已安装的应用。</p>

<blockquote> 
 <p>新上架4款应用：<strong>2FAuth</strong> 、<strong>Gitea</strong>、<strong>Chartmuseum</strong>、<strong>YApi</strong>。</p> 
</blockquote>

<h3 style="margin-left:0; margin-right:0">V1.3 版详细更新列表</h3> 
<h4 style="margin-left:0; margin-right:0">一、新特性</h4> 
<p><strong>1.1 命令行工具升级</strong></p> 
<ul> 
 <li>新增全局数据库管理入口</li> 
 <li>新安装支持外部数据存储如MySQL</li> 
</ul> 
<p><strong>1.2 新增功能</strong></p> 
<ul> 
 <li>新增数据库管理功能Adminer，方便您管理平台上的所有数据库。</li> 
 <li>新增服务导航页，通过把常用应用固定在导航页可以快速访问已安装的应用，还可以通过名称检索已安装的应用。</li> 
</ul> 
<h3 style="margin-left:0; margin-right:0">二、应用市场</h3> 
<ul> 
 <li>新增<code>2FAuth</code>应用:：基于Web的动态口令管理工具，可替代Google Authenticator。</li> 
 <li>新增<code>Gitea</code>应用： 一个开源社区驱动的轻量级代码托管解决方案。</li> 
 <li>新增<code>Chartmuseum </code>应用：开源Chart仓库托管工具。</li> 
 <li>新增<code>YApi</code>应用：高效、易用、功能强大的 api 管理平台。</li> 
</ul> 
<h3 style="margin-left:0; margin-right:0">三、体验优化</h3> 
<ul> 
 <li>新版后台管理，以图标的形式展示后台管理，清晰、直观。</li> 
 <li>新版服务备份管理页面，增加了更多的备份信息，让您更详实的了解备份数据。</li> 
</ul> 
<h3 style="margin-left:0; margin-right:0">四、Bug修复</h3> 
<ul> 
 <li>修复禅道应用安装异常的问题</li> 
 <li>修复命令status状态显示异常</li> 
</ul>
