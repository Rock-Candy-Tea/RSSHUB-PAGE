                                                        <p>WebStorm 2022.1 的候选版本发布，该版本的改进内容包括：</p> 
<ul> 
 <li>更好的 Next.js 支持：WebStorm 2022.1 改进了使用 Next.js 的体验</li> 
 <li>支持 Volta 和 Corepack：WebStorm 现在与 Volta（一个 JavaScript 工具管理器）完全集成；它还支持 Corepack（实验性的 Node.js 工具）</li> 
 <li>ESM 对 webpack 配置的支持：从现在开始，webpack 编码辅助将与你 webpack.config.js 文件中的 ES 模块一起使用</li> 
 <li>对 Docker 的改进：重新设计了 Docker 在 Services 工具窗口中的 UI，并增加了对 Docker Registry HTTP API V2 的支持，以便与 Docker 1.6+ 一起使用</li> 
 <li>从 Markdown 文件中运行命令：你现在可以直接从你的 README 文件中运行命令 —— 只需点击命令左边的运行图标。</li> 
 <li>通知工具窗口：新版本用一个新的通知工具窗口取代了 WebStorm 的事件日志，以帮助你更好地保持对 IDE 中重要通知的概览</li> 
 <li>改进了使用 Git Blame 进行注释的功能，以使可以更轻松地调查引入的更改</li> 
</ul>
