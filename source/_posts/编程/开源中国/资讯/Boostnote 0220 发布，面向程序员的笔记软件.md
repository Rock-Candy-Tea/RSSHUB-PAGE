 <p>Boostnote 是一款面向程序员的笔记应用程序，适用于 Mac、Windows 和 Linux 平台。它着重于快速编写 Markdown 和代码片段，可以以更好的方式组织这些内容。</p> 
 <p>由于 Boostnote 是一个 Markdown 编辑器，所以主要用 markdown 来写内容。在编辑笔记时，内容会自动保存，并且可以一键查看预览。在代码片段中，可以高亮显示 Javascript、Python、HTML 等 100 多种语言的代码语法，并在一个笔记中保存多个代码片段。</p> 
 <p>自 Boost Note 0.20.0 版本以来的变化包含：</p> 
 <ul> 
  <li>增加了应用内搜索 (macOS : <code>Cmd + F</code> / Windows 和 Linux : <code>Ctrl + F</code>)</li> 
  <li>在菜单中增加了缩放控制的快捷键。(查看 -> 放大、缩小、实际大小) 
   <ul> 
    <li>放大（macOS：<code>Cmd + +</code> / Windows 和 Linux：<code>Ctrl + +</code>)</li> 
    <li>缩小 (macOS : <code>Cmd + -</code> / Windows 和 Linux : <code>Ctrl + -</code>)</li> 
    <li>实际大小（macOS : <code>Cmd + 0</code> / Windows 和 Linux : <code>Ctrl + 0</code>)</li> 
   </ul> </li> 
  <li>修正了渲染性能问题</li> 
  <li>将本地空间功能从 <a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fboostnote.next" target="_blank">BoostNote.next</a> 中分离出来（要继续使用本地空间，请下载 BoostNote.next-local <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FBoostIO%2FBoostNote.next-local%2Freleases" target="_blank">https://github.com/BoostIO/BoostNote.next-local/releases</a>)</li> 
  <li>修正了应用程序在闲置后崩溃的问题</li> 
  <li>修正了 macOS 上应用程序更新器的重启行为</li> 
  <li>改变了 macOS 的应用程序图标，以遵循苹果的设计准则</li> 
  <li>修正了删除废弃的本地空间后的稳定性问题</li> 
  <li>删除了登录页面中的创建本地空间按钮</li> 
  <li>删除了创建本地空间的菜单项</li> 
  <li>修正了首选项/新文件的键盘快捷方式</li> 
  <li>修正了从桌面应用创建文档后，打开团队切换下拉菜单时的应用崩溃问题</li> 
  <li>改善了应用程序的稳定性</li> 
  <li>改进了导航的用户界面</li> 
 </ul> 
 <p>更多详情可查看：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FBoostIO%2FBoostNote-App%2Freleases" target="_blank">https://github.com/BoostIO/BoostNote-App/releases</a></p>
