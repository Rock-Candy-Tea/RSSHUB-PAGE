                                                                                            <p><span style="background-color:#ffffff; color:#333333">Git for Windows 2.35.0 现已发布，主要更新内容如下：</span></p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><strong>Upcoming breaking changes</strong></p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">官方计划在 Git for Windows 2.35.0 发布后不久，将其中的 bash 更新到 5.1 版本（目前是 4.4）。建议用户先检查自己的 shell 脚本是否有潜在的兼容性问题。</p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><strong>New Features</strong></p> 
<ul> 
 <li>随附 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fgit%2Fgit%2Fblob%2Fv2.35.0%2FDocumentation%2FRelNotes%2F2.35.0.txt" target="_blank">Git v2.35.0</a></li> 
 <li>随附一个基于 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fcygwin.com%2Fpipermail%2Fcygwin-announce%2F2021-December%2F010338.html" target="_blank">Cygwin 3.3.3</a> 的 MSYS2 运行时版本（Git for Windows flavor）。</li> 
 <li>随附<span style="background-color:#ffffff; color:#24292f"><span> </span></span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.openssl.org%2Fnews%2Fopenssl-1.1.1-notes.html" target="_blank">OpenSSL v1.1.1m</a></li> 
 <li>随附<span style="background-color:#ffffff; color:#24292f"><span> </span></span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FGitCredentialManager%2Fgit-credential-manager%2Freleases%2Ftag%2Fv2.0.632" target="_blank">Git Credential Manager Core v2.0.632.34631</a></li> 
 <li>随附<span style="background-color:#ffffff; color:#24292f"><span> </span></span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fcurl.haxx.se%2Fchanges.html%237_81_0" target="_blank">cURL v7.81.0</a></li> 
 <li>随附 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fjonas%2Ftig%2Freleases%2Ftag%2Ftig-2.5.5" target="_blank">tig v2.5.5</a></li> 
 <li>随附基于 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fcygwin.com%2Fpipermail%2Fcygwin-announce%2F2021-December%2F010338.html" target="_blank">Cygwin 3.3.3</a> 的 MSYS2 运行时的<span style="background-color:#ffffff; color:#24292f"><span> </span></span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fgit-for-windows%2Fmsys2-runtime%2Fcommit%2Fb600e8ead500aef55e23810c2e630d9be46f3a4c" target="_blank">patch level 4</a>（Git for Windows flavor） 。</li> 
</ul> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:start"><strong><span><span><span><span><span style="color:#24292f"><span><span><span><span><span><span><span><span><span><span><span style="background-color:#ffffff"><span><span><span>Bug 修复</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></strong></p> 
<ul> 
 <li>修复了一个在 32 位构建中使用自定义日期格式运行<code>git log</code>时导致崩溃的<span style="background-color:#ffffff; color:#24292f"><span> </span></span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fgit-for-windows%2Fgit%2Fissues%2F3624" target="_blank">bug</a>。</li> 
</ul>
