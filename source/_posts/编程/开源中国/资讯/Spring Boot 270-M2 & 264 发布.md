                                                                                            <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">Spring Boot<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fspring.io%2Fblog%2F2022%2F02%2F24%2Fspring-boot-2-7-0-m2-available-now" target="_blank">2.7.0-M2</a><span> </span>&<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fspring.io%2Fblog%2F2022%2F02%2F24%2Fspring-boot-2-6-4-available-now" target="_blank">2.6.4</a> 已发布。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">2.7.0-M2 包含<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fspring-boot%2Freleases%2Ftag%2Fv2.7.0-M2" target="_blank">137 个</a>错误修复、文档改进和依赖项升级。值得关注的新功能：</p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>提供面向 Couchbase 和 Elasticsearch 的测试切片 (Test slices)</li> 
 <li>增强自动配置注册和排序功能</li> 
 <li>支持 OkHttp4</li> 
</ul> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">详情查看<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fspring-boot%2Fwiki%2FSpring-Boot-2.7.0-M2-Release-Notes" target="_blank">release notes</a>。</p> 
<hr> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">2.6.4 包含<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fspring-boot%2Freleases%2Ftag%2Fv2.6.4" target="_blank">61 个</a>错误修复、文档改进和依赖项升级。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">部分已修复的错误：</p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>读取日志更新事件时忽略无效的流类型 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fspring-boot%2Fissues%2F29691" target="_blank">#29691</a></li> 
 <li>Liquibase 的依赖管理不包括其 liquibase-cdi 模块 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fspring-boot%2Fissues%2F29741" target="_blank">#29741</a></li> 
 <li>遇到某些条件时，Health Web Endpoint Extension 无法初始化 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fspring-boot%2Fissues%2F29532" target="_blank">#29532</a></li> 
 <li>失败的应用程序上下文不会从 SpringApplicationShutdownHook 中注销 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fspring-boot%2Fissues%2F29905" target="_blank">#29905</a></li> 
 <li>ErrorPageSecurityFilter 不能在 Servlet 3.1 兼容的容器中被销毁 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fspring-boot%2Fissues%2F29558" target="_blank">#29558</a></li> 
</ul>
