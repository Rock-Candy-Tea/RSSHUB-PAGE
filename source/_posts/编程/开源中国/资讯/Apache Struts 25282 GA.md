                                                                                            <p>由于 <a href="https://www.oschina.net/news/174133/apache-struts-2-5-28-released" target="_blank">Apache Struts 2.5.28 </a>发布后被曝出了 log4shell 漏洞，因此 Struts 团队近日先后推出了针对 2.5.28 的补丁版本 2.5.28.1 和 2.5.28.2。</p> 
<ul> 
 <li>Struts 2.5.28.1</li> 
</ul> 
<p>此版本使用最新的 Log4j 2.12.2 版本（与 Java 1.7 兼容）解决了 Log4j 漏洞 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Flogging.apache.org%2Flog4j%2F2.x%2Fsecurity.html%23CVE-2021-45046" target="_blank">CVE-2021-45046</a>。</p> 
<p><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fcwiki.apache.org%2Fconfluence%2Fdisplay%2FWW%2FVersion%2BNotes%2B2.5.28.1" target="_blank">详细更新说明点此查看</a>。</p> 
<ul> 
 <li>Struts 2.5.28.2</li> 
</ul> 
<p>此版本使用最新的 Log4j 2.12.3 版本（与 Java 1.7 兼容）解决了 Log4j 漏洞 <span style="background-color:#ffffff; color:#17233b"><span> </span></span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Flogging.apache.org%2Flog4j%2F2.x%2Fsecurity.html%23CVE-2021-45105" target="_blank">CVE-2021-45105</a>。</p> 
<p><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fcwiki.apache.org%2Fconfluence%2Fdisplay%2FWW%2FVersion%2BNotes%2B2.5.28.2" target="_blank">详细更新说明点此查看</a>。</p> 
<p><span style="background-color:#ffffff; color:#333333">Apache Struts 2 是一个优雅、可扩展的框架，可用于创建企业级 Java Web 应用程序。Struts 2 旨在简化从构建到部署，再到维护应用程序整个开发周期。</span></p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">Apache Struts 2.5.x 系列对以下规范版本有最低要求：Servlet API 2.4, JSP API 2.0 和 Java 7。</p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">下载地址：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fstruts.apache.org%2Fdownload.cgi%23struts-ga" target="_blank">https://struts.apache.org/download.cgi#struts-ga</a></p>
