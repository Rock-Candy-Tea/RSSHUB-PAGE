                                                                                            <p>Firefox 91 正式发布，更新内容如下：</p> 
<h3>新增：</h3> 
<ul> 
 <li>在 Total Cookie Protection 的基础上，我们增加了一个更全面的清除 Cookie 的逻辑，可以防止隐藏的数据泄露，让用户更容易了解哪些网站在存储本地信息；</li> 
 <li>Firefox 现在支持使用 Windows 单点登录功能来登录微软、工作和学校账户；</li> 
 <li>打印时的简化页面功能回来了！打印时，在 "更多设置" > "格式" 下选择 "简化" 选项，以获得一个简洁的页面；</li> 
 <li>HTTPS-First Policy：Firefox 隐私浏览窗口现在尝试使用更加安全的方式连接所有网站，只有在网站不支持的情况下才会退回到不安全的连接；</li> 
 <li>在 macOS 上勾选 "增加对比度" 时，Firefox 现在自动启用高对比度模式；</li> 
 <li>用户交互的响应时间提高了 10-20%；</li> 
 <li>增加了一个新的本地化语言：苏格兰语；</li> 
</ul> 
<h3>修复：</h3> 
<ul> 
 <li>各种<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.mozilla.org%2Fsecurity%2Fadvisories%2Fmfsa2021-33%2F" target="_blank">安全问题</a>的修复；</li> 
</ul> 
<h3>Web 平台：</h3> 
<ul> 
 <li>Visual Viewport API 现在在桌面平台上得到支持；</li> 
</ul>
