                                                        <p>RedisDesktopManager 2021.7 现已发布。RedisDesktopManager 是一个快速、简单、支持跨平台的 Redis 桌面管理工具，基于 Qt 5 开发，支持通过 SSH Tunnel 连接。新版本主要变化如下：</p> 
<p><strong>New features</strong></p> 
<ul> 
 <li>允许从文件添加键值</li> 
</ul> 
<p><strong>修复和改进</strong></p> 
<ul> 
 <li>记住每个键的选定 formatters <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fuglide%2FRedisDesktopManager%2Fissues%2F4859" target="_blank">#4859</a></li> 
 <li>自动检测 JSON <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fuglide%2FRedisDesktopManager%2Fissues%2F5061" target="_blank">#5061</a></li> 
 <li>修复 JSON 格式化错误 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fuglide%2FRedisDesktopManager%2Fissues%2F4964" target="_blank">#4964</a></li> 
</ul>
