                                                                                            <p>System 76 的 Pop!_OS 是基于 Ubuntu 开发的，而 Ubuntu 22.04 LTS 也即将到来。按照发布时间表，Pop!_OS 22.04 的 Beta ISO 现在可以在 GitHub 上下载和测试。Beta 版本目前还不稳定，不建议在生产环境中使用，但用户可以提前了解新的特点。</p> 
<p><img alt height="394" src="https://static.oschina.net/uploads/space/2022/0407/070917_gNap_4937141.png" width="700" referrerpolicy="no-referrer"></p> 
<p>Ubuntu 22.04 LTS 的主要亮点是 GNOME 42，毫不意外，Pop!_OS 22.04 中的 Cosmic 桌面同样是以 GNOME 42 为核心，但它被大量地删减以符合 Cosmic 桌面的原则。</p> 
<p>Files 42 看起来与之前的版本几乎一样。但是 Files 42 拥有所有的核心功能，例如：Files 42 应用本身有了新的图标，小工具的紧凑间距和新的菜单。不过 Cosmic 桌面中的 Files 42 仍然保留了它自己的文件夹图标和颜色。</p> 
<p><img alt height="451" src="https://static.oschina.net/uploads/space/2022/0407/070928_lJ4e_4937141.png" width="700" referrerpolicy="no-referrer"></p> 
<p>除此之外，新设计的暗色/亮色主题支持与自动更换墙纸的功能也没有加入 Pop!_OS 22.04。因为 Pop!_OS 在 Cosmic 桌面诞生之初就已经有了暗色/亮色主题的支持。</p> 
<p>另一个关键的应用程序 GNOME Setting（gnome-control-center）的版本保持在 41.1，因为 GNOME Setting 42 在转换到 libadwaita 和 GTK4 的过程中十分复杂，并具有许多功能，但该版本的一些功能已经通过不同的方式在 Pop!_OS 中内置的。因此 Pop!_OS 团队并没有在该版本中升级它。</p> 
<p>应用程序及其版本的变化摘要如下：</p> 
<ul> 
 <li>基于 Ubuntu 22.04 LTS</li> 
 <li>Linux Kernel 5.16（Ubuntu 22.04 LTS 使用 Linux Kernel 5.15）</li> 
 <li>GNOME Shell 42</li> 
 <li>GNOME Setting 41.4</li> 
 <li>Firefox 98</li> 
 <li>GNOME Terminal 3.43.9</li> 
 <li>GEdit 41.4</li> 
</ul>
