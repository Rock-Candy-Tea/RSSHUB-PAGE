                                                                                            <p><span style="background-color:#ffffff; color:#333333"> <strong>小诺layui版本为小诺系列软件快速开发平台中单体版本，基于SpringBoot + Layui + easyweb的低代码多数据库后台权限管理系统， </strong></span><strong><span style="background-color:#ffffff; color:#40485b">此次版本大更新、大改在、大修复详细更新如下：</span></strong></p> 
<p><span style="background-color:#ffffff; color:#40485b">1、【新增】文件管理与文件选择、在线文档等新增文件下载功能</span><br> <span style="background-color:#ffffff; color:#40485b">2、【修复】修复页面刷新（或F5刷新）后默认回到首页的bug</span><br> <span style="background-color:#ffffff; color:#40485b">3、【新增】添加菜单表单页新增“菜单地址”、“权限标识”、“所属应用”等tips说明，优化使用体验</span><br> <span style="background-color:#ffffff; color:#40485b">4、【修复】修复定时任务编辑失败</span><br> <span style="background-color:#ffffff; color:#40485b">5、【优化】默认加载主页，取消默认展开第一个菜单，优化体验</span><br> <span style="background-color:#ffffff; color:#40485b">6、【优化】新增页签记忆相关注释说明</span><br> <span style="background-color:#ffffff; color:#40485b">7、【新增】新增展开式菜单授权样式</span><br> <span style="background-color:#ffffff; color:#40485b">8、【更新】去掉启动项目时数据库密码强校验，配置数据库信息错误提示细化</span><br> <span style="background-color:#ffffff; color:#40485b">9、【修复】在线文档编辑器title更正</span><br> <span style="background-color:#ffffff; color:#40485b">10、【新增】新增用户导出功能</span><br> <span style="background-color:#ffffff; color:#40485b">11、【升级】模板渲染引擎由thymeleaf全面升级beetl3.1.8，修复打成war包部署时路径404等问题</span><br> <span style="background-color:#ffffff; color:#40485b">12、【升级】application.yml增加beetl后缀配置为html</span><br> <span style="background-color:#ffffff; color:#40485b">13、【修复】修复在线文档后缀名称重复的问题</span><br> <span style="background-color:#ffffff; color:#40485b">14、【修复】修复新增用户添加附属机构页面打不开的bug</span><br> <span style="background-color:#ffffff; color:#40485b">15、【修复】修复表格重载之后路径问题</span><br> <span style="background-color:#ffffff; color:#40485b">16、【新增】新增sql相关说明，更新sql编码为utf8mb4</span><br> <span style="background-color:#ffffff; color:#40485b">17、【更新】更新oracle版本sql，增加oracle地区数据sql，适配mssql，新增mssql文件</span></p>
