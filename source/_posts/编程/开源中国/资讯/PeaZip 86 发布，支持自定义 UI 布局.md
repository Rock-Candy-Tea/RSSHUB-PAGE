                                                                                            <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">PeaZip 是一个适用于 Windows 和 Linux 的免费文件管理工具和 rar 提取器，可处理 200 多种存档类型（7z, ace, arc, bz2, cab, gz, iso, paq, pea, rar, tar, wim, zip, zipx...），处理跨区存档（001, r01, z01...）并支持多种存档加密标准。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">该项目旨在为多种开源技术（7-Zip、FreeArc、PAQ、PEA、UPX）提供一个跨平台、可移植的 GUI 前端，专注于文件和档案管理，以及安全（强加密、双因素认证、加密密码管理器、安全删除）。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">PeaZip 8.6.0 正式发布，该版本更新内容如下：</p> 
<h3 style="margin-left:.6em; margin-right:0; text-align:start"><strong>后端</strong></h3> 
<ul style="margin-left:0; margin-right:0"> 
 <li>后端版本更新为 Pea 1.07 和 (Linux) Zstd 1.5.2 。</li> 
</ul> 
<h3 style="margin-left:.6em; margin-right:0; text-align:start"><strong>代码</strong></h3> 
<ul style="margin-left:0; margin-right:0"> 
 <li>修复了使用 XZ 压缩的 ZIP 的字典大小</li> 
 <li>(Windows) 修复了与 Windows XP 的兼容性问题</li> 
 <li>(Windows) 修复了一些特定操作后，系统文件资源管理器的刷新问题</li> 
</ul> 
<h3 style="margin-left:.6em; margin-right:0; text-align:start"><strong>文件管理器</strong></h3> 
<ul style="margin-left:0; margin-right:0"> 
 <li>引入了一个名为“Main”的新默认主题，该主题的图标与大多数受支持操作系统的设计进行视觉集成</li> 
 <li><strong>(peazip)/res/share/icons 目录中现在提供了新的 .ico 和 .png 图标，可用于在 Linux、macOS 和 Windows 系统上自定义应用的 UI 布局</strong></li> 
 <li><span style="color:#333333"><strong>工具栏右侧的 New Style 弹出菜单统一了多个子菜单，可快速自定义UI </strong></span></li> 
 <li>上下文菜单中的导航子菜单更简单，专注于文件浏览，删除了与样式相关的重复控件</li> 
</ul> 
<h3 style="margin-left:.6em; margin-right:0; text-align:start">提取和归档</h3> 
<ul style="margin-left:0; margin-right:0"> 
 <li><span style="color:#404040">新的“立即执行”选项，无需确认即可开始压缩选定的文件。</span></li> 
 <li>两个新的命令行选项 (<code>-add2preset N</code>和<code>-add2custom</code>) ，可使用预设的压缩配置来压缩文件，并加载已保存的自定义压缩设置。</li> 
</ul>
