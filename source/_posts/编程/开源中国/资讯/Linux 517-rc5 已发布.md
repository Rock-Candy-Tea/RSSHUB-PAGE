                                                                                            <p>Linux 5.17-rc5 已<span style="color:#121212">作为 Linux 5.17 的第五个预览版本</span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Flore.kernel.org%2Flkml%2FCAHk-%3DwgsMMuMP9_dWps7f25e6G628Hf7-B3hvSDvjhRXqVQvpg%40mail.gmail.com%2F" target="_blank">发布</a>，该版本依旧是一些补丁修复，并未包含重大功能更新，Linux 5.17 稳定版应该会在 3 月推出。</p> 
<p>Linus 在 Linux 5.17-rc5 邮件中的说法：</p> 
<blockquote> 
 <p>一切看起来都很正常。到处都有补丁，这次发布的补丁也没有比平时多，统计数据看起来也很正常，大部分变化都发生在驱动身上。</p> 
 <p>diffstat 看起来有点不同寻常，英特尔 iwlwifi 驱动程序显示了很多修改，但这几乎完全是由于删除了已弃用的广播过滤，它甚至不适用于较新的硬件。在驱动子系统之外，主要是架构更新（kvm 再次出现）、工具和网络。</p> 
</blockquote> 
<p>Linux 5.17 稳定版将于 3 月中下旬发布，并将被 Fedora 36 和其他 Linux 发行版采用。但 Ubuntu 除外，据外媒 Phoronix <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.phoronix.com%2Fscan.php%3Fpage%3Dnews_item%26px%3DLinux-5.17-rc5-Released" target="_blank">介绍</a>： Ubuntu 22.04 LTS 由于其 LTS 状态而坚持使用 Linux 5.15，而不是已经稳定的 v5.16 或 v5.17 。</p>
