                                                                                            <p>DBeaver 21.2.0 现已发布。DBeaver是一个免费开源的通用数据库工具，适用于开发人员和数据库管理员。</p> 
<p><strong>主要更新内容</strong></p> 
<ul> 
 <li>数据传输： 
  <ul> 
   <li>添加了值绑定模式切换器（注意会影响性能）</li> 
   <li>修复了自动列映射</li> 
   <li>修复了从文件/关系表导入到面向文档的数据库</li> 
   <li>优化了列长检测</li> 
  </ul> </li> 
 <li>SSH tunnel：添加了跳转服务器支持</li> 
 <li>任务管理视图：添加了任务文件夹</li> 
 <li>导航器：在单个列表中显示来自不同模式的表（可选）</li> 
 <li>连接驱动程序设置：修复了不可编辑属性的问题</li> 
 <li>Data 编辑器： 
  <ul> 
   <li>修复了工具栏命令启用处理程序</li> 
   <li>修复了分离的文本/二进制编辑器</li> 
   <li>行着色：修复了 LIKE 和 REGEX 过滤器值编辑器</li> 
  </ul> </li> 
 <li>SQL 编辑器： 
  <ul> 
   <li>修复了从粘贴的源代码中提取查询的换行问题</li> 
   <li>优化了多字节 unicode 数字处理程序（应用程序挂起修复）</li> 
   <li>主窗口标题现在会显示有关当前编辑器的正确信息</li> 
  </ul> </li> 
 <li>Greenplum：修复了存储过程自动完成和导航</li> 
 <li>PostgreSQL： 
  <ul> 
   <li>修复了多列外键导航</li> 
   <li>优化了引用标识符的自动完成</li> 
   <li>修复了活动模式检测</li> 
   <li>修复了浮点数格式</li> 
  </ul> </li> 
 <li>红移： 
  <ul> 
   <li>数据传输性能显着提高（高达 x10000）</li> 
   <li>改进了表和程序 DDL（添加了安全定义器）</li> 
   <li>修复了存储过程搜索和自动完成</li> 
   <li>修复了时间戳值编辑器（在本机日期格式模式下）</li> 
  </ul> </li> 
 <li>Snowflake：修复了驱动程序配置（JNA 依赖项）</li> 
 <li>修复了许多小的 UI 和数据库的错误</li> 
</ul>
