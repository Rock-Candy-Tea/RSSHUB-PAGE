                                                        <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">Linux Kernel 5.18<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Flkml.org%2Flkml%2F2022%2F5%2F22%2F274" target="_blank">发布</a>了！该版本主要变化之一是将<a href="https://www.oschina.net/news/182489/intel-sdsi-merge-to-linux-5-18">英特尔软件定义芯片(SDSi) 驱动程序</a>包含到主线内核中，该功能引起了不小的争议，但英特尔尚未明确表示该驱动的具体用途。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">硬件方面， <span> </span><strong>Raspberry Pi Zero 2 W</strong> 在此版本中获得了完整的主线 Linux 内核支持；<strong>Tesla FSD<span> </span></strong>芯片进入上游；其他则是一些 AMD 新显卡的驱动和 Intel 处理器的相关文件。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">该版本的主要新特性包括：</p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>用于英特尔混合处理器的新硬件反馈接口 (HFI) 驱动程序</li> 
 <li>AMD EPYC 服务器的 NUMA 平衡调度程序更新</li> 
 <li>AMD HSMP 驱动程序</li> 
 <li>各种 AMD 嵌套虚拟化改进</li> 
 <li>英特尔至强处理器驱动程序支持</li> 
 <li>在 AMDGPU 驱动程序中默认启用FreeSync</li> 
 <li>Btrfs 支持编码 I/O 和更快的 fsync</li> 
 <li>带有触控栏的 MacBook Pro 的 FN 键映射</li> 
</ul> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">其他内容可参考社区关于 Linux 5.18 新功能的报道：</p> 
<ul style="margin-left:0; margin-right:0"> 
 <li><a href="https://www.oschina.net/news/182489/intel-sdsi-merge-to-linux-5-18">英特尔 CPU 内购功能 (SDSI)将合并到 Linux 5.18 中</a></li> 
 <li><a href="https://www.oschina.net/news/189659/linux-5-18-more-flexible-llvm">Linux 5.18 支持更灵活的 LLVM 配置方式</a></li> 
 <li><a href="https://www.oschina.net/news/189563/linux-5-18-exfat">Linux 5.18 对 exFAT 文件系统支持进行了两项关键改进</a></li> 
 <li><a href="https://www.oschina.net/news/185952/linux-5-18-c11-plan">Linux 5.18 内核 C 标准从 C89 升级到 C11/GNU11</a></li> 
 <li><a href="https://www.oschina.net/news/183520/apple-keyboards-in-linux-5-18">Linux 5.18 带来许多 Apple 键盘改进</a></li> 
</ul> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">随着 5.18 正式发布，5.19 的合并窗口已经打开，据 Linus 介绍：“目前已经有一些 Linux 5.19 的 PR 等待处理。”</p>
