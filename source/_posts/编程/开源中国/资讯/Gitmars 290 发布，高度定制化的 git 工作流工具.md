                                                        <p>Gitmars 2.9.0 已经发布，高度定制化的 git 工作流工具。</p> 
<p>此版本更新内容包括：</p> 
<ol> 
 <li><code>go</code>指令增加<code>command</code>参数，直接进入对应的指令</li> 
 <li><code>ui</code>指令增加<code>--port</code>参数，自定义端口号</li> 
 <li>完善每个指令的help提示信息</li> 
 <li>完善type类型</li> 
</ol> 
<p>详情查看：<a href="https://gitee.com/saqqdy/gitmars/releases/2.9.0">https://gitee.com/saqqdy/gitmars/releases/2.9.0</a></p>
