                                                        <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">WebStorm 2021.2.3 正式发布，该版本为 WebStorm 2021.2 的第三个错误修复更新，用户可以使用 Toolbox 应用程序、从 IDE 或者直接从网站中下载来更新该版本。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">WebStorm 2021.2.3 中值得注意的更新包括：</p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>解决了 CSS 高亮和颜色预览的几个问题 (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FWEB-40779" target="_blank">WEB-40779</a>, <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FWEB-49754" target="_blank">WEB-49754</a>).</li> 
 <li>修正了导致运行配置不能与 npm 7 配合的问题 (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FWEB-49835" target="_blank">WEB-49835</a>).</li> 
 <li>TypeScript 引擎不再在 Vue 项目中随机停止工作 (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FWEB-47248" target="_blank">WEB-47248</a>).</li> 
 <li>Dart：mixin 不会折叠（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FWEB-52669" target="_blank">WEB-52669</a>）</li> 
 <li>Core. Indexing：添加 warm-up 命令以准备索引和下载项目的共享索引（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-270854" target="_blank">IDEA-270854</a>)</li> 
 <li>Core. Licensing：支持 License Dialog 的 Login API (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-270216" target="_blank">IDEA-270216</a>)</li> 
</ul>
