                                                                                            <p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><span style="color:#000000">Microsoft PowerToys 是 Windows 系统实用程序，供高级用户调整和简化其 Windows 体验，可最大限度地提高生产力。</span></p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><span style="color:#000000">v0.60.0 版本</span>专注于稳定性和改进，带来大量改动：</p> 
<h3 style="margin-left:.6em; margin-right:0; text-align:start">亮点内容</h3> 
<ul style="margin-left:0; margin-right:0"> 
 <li>PowerRename 和 Image Resizer 会出现在新的 Windows 11 上下文菜单中。</li> 
 <li>OneNote 有一个新的 PowerToys Run 插件。</li> 
 <li>FancyZones 使用带有监视器 ID 的新区域标识，来提高稳定性并避免区域重置。</li> 
 <li>AlwaysOnTop 现在使用圆角突出显示 Windows 11 上的圆角窗口。</li> 
 <li>为 PowerToys Run 添加了设置，以更好地控制查询结果的顺序。</li> 
</ul> 
<h3 style="margin-left:.6em; margin-right:0; text-align:start">其他内容</h3> 
<ul style="margin-left:0; margin-right:0"> 
 <li><span style="color:#24292f">将 Windows App SDK 运行时升级到 1.1.1。</span></li> 
 <li>添加了对多样化的键盘快捷键的支持，并回退到低级键盘 Hook。</li> 
 <li>为 Windows 11 上的圆角窗口添加了圆角高光。</li> 
 <li>修复了桌面上显示的窗口无法与 FancyZones 一起使用的错误。</li> 
 <li>更改区域布局时，如果打开该选项，窗口将匹配新的大小/位置。</li> 
 <li>FancyZones 编辑器中的次要 UI 修复。</li> 
 <li>修复了 FancyZones Editor 中正确的画布布局渲染，现在显示了未缩放的监视器分辨率。</li> 
 <li>修复了某些窗口中的透明度导致窗口变为空白的问题。</li> 
</ul> 
<h3 style="margin-left:.6em; margin-right:0; text-align:start"><strong>文件浏览器插件</strong></h3> 
<ul style="margin-left:0; margin-right:0"> 
 <li>向 svg 文件缩略图添加 viewBox 属性，以便显示整个图像。</li> 
 <li>在 Markdown 预览测试中删除了对远程图像的访问。</li> 
 <li>使用适当的组件初始化超时，修复了 Markdown 预览测试套件中的脆弱性。</li> 
 <li>修复了 svg 缩略图导致的 WebView2 资源泄露问题。</li> 
</ul>
