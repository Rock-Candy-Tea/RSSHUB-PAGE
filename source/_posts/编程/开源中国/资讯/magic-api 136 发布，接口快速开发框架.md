                                                        <h3 style="text-align:left">本次更新内容：</h3> 
<ul> 
 <li>新增注释补全功能</li> 
 <li>增加<code>not_blank</code>、<code>not_null</code>、<code>is_null</code>、<code>is_blank</code>、<code>current_timestamp</code>、<code>print</code>等相关函数</li> 
 <li>修复未改动脚本时无法保存接口的<code>BUG</code></li> 
 <li>修复注销不应该验证需要登录的问题</li> 
 <li>修复文件参数必填验证失效的<code>BUG</code></li> 
 <li>修复在使用过程中<code>Token</code>失效后未显示登录框的问题</li> 
 <li>修复无法删除接口的<code>BUG</code></li> 
 <li>修复<code>Body</code>参数在编辑后丢失配置信息的问题</li> 
 <li>修复<code>Redis</code>模块部分场景无法注册模块的问题</li> 
 <li>修正删除接口的权限配置错误</li> 
 <li>修复定义<code>async</code>、<code>select</code> 字符串时被解析成语句的<code>BUG</code></li> 
 <li>修复<code>LINQ</code> <code>left join</code> 缓存不正确的问题 <a href="https://gitee.com/ssssssss-team/magic-api/issues/I42W1F" target="_blank">I42W1F</a></li> 
 <li>修复引用<code>actuator</code>组件可能造成的重复注册接口的问题 <a href="https://gitee.com/ssssssss-team/magic-api/issues/I42UYG" target="_blank">I42UYG</a></li> 
 <li>优化<code>http</code>模块注册，解决某些场景冲突导致无法启动的问题</li> 
 <li>优化状态条显示，解决一直显示<code>开始测试...</code>的问题</li> 
</ul>

<h4 style="text-align:start"><strong>项目介绍</strong></h4> 
<p style="text-align:left"><span style="background-color:#ffffff; color:#40485b">magic-api 是一个基于Java的接口快速开发框架，编写接口将通过magic-api提供的UI界面完成，自动映射为HTTP接口，无需定义Controller、Service、Dao、Mapper、XML、VO等Java对象即可完成常见的HTTP API接口开发</span></p> 
<h3 style="text-align:left"><span style="color:#40485b">特性</span></h3> 
<ul> 
 <li>支持MySQL、MariaDB、Oracle、DB2、PostgreSQL、SQLServer 等支持jdbc规范的数据库</li> 
 <li>支持非关系型数据库Redis、Mongodb</li> 
 <li>支持集群部署、接口自动同步。</li> 
 <li>支持分页查询以及自定义分页查询</li> 
 <li>支持多数据源配置，支持在线配置数据源</li> 
 <li>支持SQL缓存，以及自定义SQL缓存</li> 
 <li>支持自定义JSON结果、自定义分页结果</li> 
 <li>支持对接口权限配置、拦截器等功能</li> 
 <li>支持运行时动态修改数据源</li> 
 <li>支持Swagger接口文档生成</li> 
 <li>基于<a href="https://gitee.com/ssssssss-team/magic-script" target="_blank">magic-script</a>脚本引擎，动态编译，无需重启，实时发布</li> 
 <li>支持Linq式查询，关联、转换更简单</li> 
 <li>支持数据库事务、SQL支持拼接，占位符，判断等语法</li> 
 <li>支持文件上传、下载、输出图片</li> 
 <li>支持脚本历史版本对比与恢复</li> 
 <li>支持脚本代码自动提示、参数提示、悬浮提示、错误提示</li> 
 <li>支持导入Spring中的Bean、Java中的类</li> 
 <li>支持在线调试</li> 
 <li>支持自定义工具类、自定义模块包、自定义类型扩展、自定义方言、自定义列名转换等自定义操作</li> 
</ul>
