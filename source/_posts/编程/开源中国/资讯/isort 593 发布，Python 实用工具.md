                                                                                            <p>isort 是一个 Python 实用程序/库，用于按字母顺序对导入进行排序，并自动分成不同的部分和类型。它为各种编辑器提供命令行实用程序、Python 库和插件，以快速对所有导入进行排序。它需要 Python 3.6+ 才能运行，但也支持格式化 Python 2 代码。</p> 
<p>isort 5.9.3 正式发布，更新内容如下：</p> 
<ul> 
 <li>改进了跳过文件信息的文本，以提及 gitignore 功能；</li> 
 <li>使得所有的异常都变得容易处理；</li> 
 <li>修复：Pylama 集成忽略了 Pylama 特定的 isort 配置覆盖；</li> 
 <li>修复： <code>-from-first</code> CLI 标志不应该接受任何参数；</li> 
 <li>修复：排序文字放在文件的前几行时，有时会被忽略；</li> 
 <li>修复：当 skip_gitignore=true 时，extend_skip 在 git 子模块中不被遵循；</li> 
</ul>
