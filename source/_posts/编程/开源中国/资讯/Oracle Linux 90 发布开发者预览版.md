<div class="content">
                                                                                            <p>Oracle <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fblogs.oracle.com%2Flinux%2Fpost%2Foracle-linux-9-developer-preview-now-available-for-download" target="_blank">宣布</a> Oracle Linux 9 首个开发者预览版已可下载，提供了适用于 Intel & AMD (x86_64) 和 Arm (aarch64) 平台的 64 位版本。</p> 
<p><strong><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.oracle.com%2Flinux%2Fdownloads%2Flinux-beta9-downloads.html" target="_blank">下载地址</a></strong></p> 
<p>Oracle Linux 9 包含许多新特性、功能增强和其他变动，与 Red Hat Enterprise Linux 9 GA 版本完全二进制兼容。</p> 
<ul> 
 <li>安全特性更新</li> 
 <li>改进网络、高可用性和文件系统</li> 
 <li>增强开发者工具</li> 
 <li>增强对编译器和脚本语言的支持</li> 
 <li>内核相关的变动</li> 
 <li>改进虚拟化、容器功能</li> 
</ul> 
<p>Oracle Linux 9 Developer Preview 附带 <strong style="color:#161513"><code>kernel-uek-5.15.0-0.23.1.el9uek</code></strong> Unbreakable Enterprise Kernel 7 (UEK) 和 <strong style="color:#161513"><code>kernel-5.14.0-1.7.1.el9</code></strong> Red Hat Compatible Kernel (RHCK) 软件包。</p> 
<p><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdocs.oracle.com%2Fen%2Foperating-systems%2Foracle-linux%2F9%2Frelnotes9.0%2F" target="_blank">详细更新说明查看 release note</a>。</p> 
<p>Oracle Linux 是甲骨文公司 (Oracle) 发行的企业级 Linux，基于 Red Hat Enterprise Linux 构建。Oracle Linux 有两种内核：兼容 Red Hat Enterprise Linux 的内核（使用 RHEL 源代码编译）和 Oracle 自己的 Unbreakable Enterprise 内核。Oracle 表示 Unbreakable Enterprise 内核兼容 RHEL，Oracle 中间件和经过 RHEL 认证的第三方应用程序可以不经过修改地在 Unbreakable Enterprise 内核上运行。</p>
