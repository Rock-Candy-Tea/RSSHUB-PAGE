                                                                                            <p><span style="color:#000000"><span style="background-color:#ffffff">Zulip Server 5.1 现已发布。Zulip 是一个</span><span style="background-color:#ffffff">开源团队协作工具，一款专为实时和异步对话而设计的现代团队聊天应用程序，</span><span style="background-color:#ffffff">支持快速搜索、拖放文件上传、图像预览、组私人消息、可听通知、错过电子邮件消息提醒与桌面应用等。</span></span></p> 
<p><span style="color:#000000"><span style="background-color:#ffffff">具体更新内容如下：</span></span></p> 
<ul> 
 <li>修复了升级错误，其中预先存在的动画表情符号仍将始终在状态中进行动画处理。</li> 
 <li>改进了防止服务器意外降级的检查，以不阻止升级在 2017 年中之前最初安装 Zulip Server 的服务器。</li> 
 <li>修复了 Slack imports 中的电子邮件地址去重问题。</li> 
 <li>当通知横幅出现在顶部时，防止出现不相干的滚动条。</li> 
 <li>修复了 LXC 容器中的安装，由于<code>chrony</code>无法在那里运行而失败。</li> 
 <li>防止即使配置了推送通知，还会在新用户默认设置面板中出现"未配置推送通知"的警告。</li> 
 <li>修复了一个错误，该错误在不常见的配置中会阻止 Tornado 在升级期间重新启动；用户将能够登录，但会立即 log out。</li> 
 <li>更新了翻译。</li> 
</ul>
