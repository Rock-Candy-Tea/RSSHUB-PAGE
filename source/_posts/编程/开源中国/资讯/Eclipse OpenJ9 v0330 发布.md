                                                                                            <p><span style="background-color:#ffffff; color:#333333">Eclipse OpenJ9 v0.33.0 现已</span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.eclipse.org%2Fopenj9%2Fdocs%2Fversion0.33%2F" target="_blank">发布</a><span style="background-color:#ffffff; color:#333333">，Eclipse OpenJ9 是 OpenJDK 的 JVM，由 IBM 创建并捐赠给 Eclipse 基金会。此 Java 虚拟机经过优化，具备占用空间小、启动速度快以及吞吐量高等优势。</span></p> 
<p style="text-align:start"><span><span><span style="color:rgba(0, 0, 0, 0.87)"><span><span><span><span><span><span><span><span><span><span><span><span style="background-color:#ffffff"><span><span><span>自 0.30.0 版以来，此版本包含以下新功能和显着更改：</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></p> 
<ul> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.eclipse.org%2Fopenj9%2Fdocs%2Fversion0.33%2F%23binaries-and-supported-environments" target="_blank">新的二进制文件和对受支持环境的更改</a></li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.eclipse.org%2Fopenj9%2Fdocs%2Fversion0.33%2F%23jitserver-technology-feature-updated" target="_blank">JITServer technology feature 更新</a></li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.eclipse.org%2Fopenj9%2Fdocs%2Fversion0.33%2F%23xl-c-runtime-required-on-aix" target="_blank"><img alt height="16" src="https://oscimg.oschina.net/oscnet/up-42964df872d5b74350d2a978420ea6f1a45.png" width="54" referrerpolicy="no-referrer"> AIX 上所需的 XL C++ 运行时</a></li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.eclipse.org%2Fopenj9%2Fdocs%2Fversion0.33%2F%23linux-reference-compiler-updated-to-gcc-103" target="_blank"><img alt height="16" src="https://oscimg.oschina.net/oscnet/up-6257f2bfc08217476e3d7565448d0e520e4.png" width="54" referrerpolicy="no-referrer"> Linux 参考编译器更新到 gcc 10.3</a></li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.eclipse.org%2Fopenj9%2Fdocs%2Fversion0.33%2F%23the-maximum-number-of-jit-compilation-threads-is-increased" target="_blank">增加 JIT 编译线程的最大数量</a></li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.eclipse.org%2Fopenj9%2Fdocs%2Fversion0.33%2F%23linux-reference-compiler-updated-to-gcc-103" target="_blank"><img alt height="16" src="https://oscimg.oschina.net/oscnet/up-6257f2bfc08217476e3d7565448d0e520e4.png" width="54" referrerpolicy="no-referrer"></a> <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.eclipse.org%2Fopenj9%2Fdocs%2Fversion0.33%2F%23default-operating-system-stack-size-increased-on-x64-platforms" target="_blank">x64 平台上的默认操作系统堆栈大小增加</a></li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.eclipse.org%2Fopenj9%2Fdocs%2Fversion0.33%2F%23control-groups-v2-support" target="_blank">Control groups v2 支持</a></li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.eclipse.org%2Fopenj9%2Fdocs%2Fversion0.33%2F%23support-for-openssl-30x" target="_blank">支持 OpenSSL 3.0.x</a></li> 
</ul> 
<h4><strong><span><span><span><span><span><span style="color:rgba(0, 0, 0, 0.87)"><span><span><span><span><span><span><span><span><span><span style="background-color:#ffffff"><span><span><span>特性和变化</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></strong></h4> 
<p><strong><span><span><span><span><span><span style="color:rgba(0, 0, 0, 0.87)"><span><span><span><span><span><span><span><span><span><span style="background-color:#ffffff"><span><span><span>二进制文件和支持的环境</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></strong></p> 
<p style="text-align:start"><span><span><span style="color:rgba(0, 0, 0, 0.87)"><span><span><span><span><span><span><span><span><span><span><span><span style="background-color:#ffffff"><span><span><span>OpenJ9 版本 0.33.0 支持 OpenJDK 8、11、17 和 18。</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></p> 
<p style="text-align:start"><span><span><span style="color:rgba(0, 0, 0, 0.87)"><span><span><span><span><span><span><span><span><span><span><span><span style="background-color:#ffffff"><span><span><span>OpenJ9 Windows builds for OpenJDK 8 现在使用 Microsoft Visual Studio 2017 编译。构建中包含的 Visual Studio 可再发行文件已更新以匹配。</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></p> 
<p style="text-align:start"><span><span><span style="color:rgba(0, 0, 0, 0.87)"><span><span><span><span><span><span><span><span><span><span><span><span style="background-color:#ffffff"><span><span><span>要了解有关 OpenJ9 版本支持的更多信息，包括 OpenJDK 级别和平台支持，可参阅<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.eclipse.org%2Fopenj9%2Fdocs%2Fopenj9_support%2F" target="_blank">支持的环境</a>。</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></p> 
<p style="text-align:start"><strong><span><span><span><span><span><span style="color:rgba(0, 0, 0, 0.87)"><span><span><span><span><span><span><span><span><span><span style="background-color:#ffffff"><span><span><span>JITServer technology feature 更新</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></strong></p> 
<p style="text-align:start"><span><span><span style="color:rgba(0, 0, 0, 0.87)"><span><span><span><span><span><span><span><span><span><span><span><span style="background-color:#ffffff"><span><span><span>JITServer technology feature 已更新以提供以下新功能，默认情况下禁用：</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></p> 
<ul> 
 <li>在服务器上缓存 AOT 编译的方法。当客户端请求编译以前缓存的方法时，AOT 缓存会提高 CPU 使用率。使用<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.eclipse.org%2Fopenj9%2Fdocs%2Fxxjitserveruseaotcache%2F" target="_blank"><code>-XX:+JITServerUseAOTCache</code></a>命令行选项启用此功能。</li> 
 <li>向遵循 OpenMetrics 标准的监控工具提供指标。以下指标可用于 JITServer 服务器：CPU 使用率、可用内存、连接的客户端数和活动编译线程数。使用<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.eclipse.org%2Fopenj9%2Fdocs%2Fxxjitservermetrics%2F" target="_blank"><code>-XX:+JITServerMetrics</code></a>命令行选项启用此功能。</li> 
</ul> 
<p><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.eclipse.org%2Fopenj9%2Fdocs%2Fversion0.33%2F%23xl-c-runtime-required-on-aix" target="_blank"><img alt height="16" src="https://oscimg.oschina.net/oscnet/up-42964df872d5b74350d2a978420ea6f1a45.png" width="54" referrerpolicy="no-referrer"> </a><strong>​​​​<span><span><span><span><span><span style="color:rgba(0, 0, 0, 0.87)"><span><span><span><span><span><span><span><span><span><span style="background-color:#ffffff"><span><span><span>AIX 上所需的 XL C++ 运行时</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></strong></p> 
<p style="text-align:start"><span><span><span style="color:rgba(0, 0, 0, 0.87)"><span><span><span><span><span><span><span><span><span><span><span><span style="background-color:#ffffff"><span><span><span>AIX OpenJ9 构建现在需要 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.ibm.com%2Fsupport%2Fpages%2Ffix-list-xl-cc-runtime-aix%23161X" target="_blank">IBM XL C++ Runtime</a> 16.1 版。OpenJDK 17 已经需要这个版本，现在从 11.0.16 开始也需要此功能，以适应 HarfBuzz text shaping 库的安全更新。</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></p> 
<p style="text-align:start"><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.eclipse.org%2Fopenj9%2Fdocs%2Fversion0.33%2F%23linux-reference-compiler-updated-to-gcc-103" target="_blank"><img alt height="16" src="https://oscimg.oschina.net/oscnet/up-6257f2bfc08217476e3d7565448d0e520e4.png" width="54" referrerpolicy="no-referrer"></a> <strong><span><span><span><span><span><span style="color:rgba(0, 0, 0, 0.87)"><span><span><span><span><span><span><span><span><span><span style="background-color:#ffffff"><span><span><span>Linux 参考编译器更新到 gcc 10.3</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></strong></p> 
<p style="text-align:start"><span><span><span style="color:rgba(0, 0, 0, 0.87)"><span><span><span><span><span><span><span><span><span><span><span><span style="background-color:#ffffff"><span><span><span>适用于所有平台的 Linux 构建现在使用 gcc 10.3 而不是 gcc 7.5。可参见<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.eclipse.org%2Fopenj9%2Fdocs%2Fopenj9_support%2F%23build-environments" target="_blank">构建环境</a>列表。</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></p> 
<p style="text-align:start"><strong><span><span><span><span><span><span style="color:rgba(0, 0, 0, 0.87)"><span><span><span><span><span><span><span><span><span><span style="background-color:#ffffff"><span><span><span>增加 JIT 编译线程的最大数量</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></strong></p> 
<p style="text-align:start"><span><span><span style="color:rgba(0, 0, 0, 0.87)"><span><span><span><span><span><span><span><span><span><span><span><span style="background-color:#ffffff"><span><span><span>JIT 编译线程的最大数量从 7 增加到 15。如果启用 JITServer 功能，则服务器上的 JIT 编译线程的最大数量进一步增加，达到 999。你可以通过指定<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.eclipse.org%2Fopenj9%2Fdocs%2Fxcompilationthreads%2F" target="_blank"><code>-XcompilationThreads</code></a>命令行选项设置要使用的编译线程数。</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></p> 
<p style="text-align:start"><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.eclipse.org%2Fopenj9%2Fdocs%2Fversion0.33%2F%23linux-reference-compiler-updated-to-gcc-103" target="_blank"><img alt height="16" src="https://oscimg.oschina.net/oscnet/up-6257f2bfc08217476e3d7565448d0e520e4.png" width="54" referrerpolicy="no-referrer"> </a><strong><span><span><span><span><span><span style="color:rgba(0, 0, 0, 0.87)"><span><span><span><span><span><span><span><span><span><span style="background-color:#ffffff"><span><span><span>x64 平台上的默认操作系统堆栈大小增加</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></strong></p> 
<p style="text-align:start"><span><span><span style="color:rgba(0, 0, 0, 0.87)"><span><span><span><span><span><span><span><span><span><span><span><span style="background-color:#ffffff"><span><span><span>Windows x64、macOS x64 和 Linux x64 平台上的默认操作系统堆栈大小从 256 KB 增加到 512 KB，以适应矢量支持。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.eclipse.org%2Fopenj9%2Fdocs%2Fxmso%2F" target="_blank">你可以使用 -Xmso</a> 选项更改操作系统堆栈大小。</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></p> 
<p style="text-align:start"><strong><span><span><span><span><span><span style="color:rgba(0, 0, 0, 0.87)"><span><span><span><span><span><span><span><span><span><span style="background-color:#ffffff"><span><span><span>Control groups v2 支持</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></strong></p> 
<p style="text-align:start"><span><span><span style="color:rgba(0, 0, 0, 0.87)"><span><span><span><span><span><span><span><span><span><span><span><span style="background-color:#ffffff"><span><span><span>Linux 内核有两个 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fman7.org%2Flinux%2Fman-pages%2Fman7%2Fcgroups.7.html" target="_blank">control groups (cgroups) 变体： v1 和 v2</a>。许多 Linux 操作系统正逐渐从 cgroups v1 过渡到 v2 作为它们的默认选择。现在，OpenJ9 添加了对 cgroups v2 的支持，这与对 cgroups v1 的支持完全相同。</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></p> 
<p style="text-align:start"><strong><span><span><span><span><span><span style="color:rgba(0, 0, 0, 0.87)"><span><span><span><span><span><span><span><span><span><span style="background-color:#ffffff"><span><span><span>支持 OpenSSL 3.0.x</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></strong></p> 
<p style="text-align:start"><span><span><span style="color:rgba(0, 0, 0, 0.87)"><span><span><span><span><span><span><span><span><span><span><span><span style="background-color:#ffffff"><span><span><span>JITServer technology feature 现在支持 OpenSSL 3.0.x。有关 OpenSSL 支持的更多信息，请参阅<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.eclipse.org%2Fopenj9%2Fdocs%2Fintroduction%2F%23cryptographic-operations" target="_blank"><code>Cryptographic operations</code></a>。</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></p>
