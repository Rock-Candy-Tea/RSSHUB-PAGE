                                                        <p style="margin-left:0"><span style="color:#000000">Spring Batch 5.0 M2 现已</span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fspring.io%2Fblog%2F2022%2F03%2F23%2Fspring-batch-5-0-m2-available-now" target="_blank">发布</a><span style="color:#000000">，新版本可从 </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Frepo.spring.io%2Fmilestone">https://repo.spring.io/milestone</a><span style="color:#000000"> 获取。</span></p> 
<p style="margin-left:0"><span style="color:#000000">Spring Batch 是一个轻量级且功能全面的批处理框架，使用 Spring 和 Java 编写离线和批处理应用程序，旨在为开发对企业系统日常运行至关重要的批处理应用程序提供支持。</span></p> 
<p><span style="color:#000000">此版本有两个主要特点：</span></p> 
<ul> 
 <li><span style="color:#000000">从 Micrometer 2.0 引入新的 Observation API，为你的批处理作业提供指标和跟踪。</span></li> 
 <li><span style="color:#000000">添加 SAP HANA 作为作业存储库的支持数据库。</span></li> 
</ul> 
<p><span style="color:#000000">以及一些增强功能和依赖性的更新。 具体包括：</span></p> 
<ul> 
 <li><span style="color:#000000">将 schema-oracle10g 重命名为 schema-oracle</span> <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fspring-batch%2Fissues%2F1057" target="_blank">#1057</a></li> 
 <li><span style="color:#000000">带有 var args/Lists 的 Constructors </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fspring-batch%2Fpull%2F686" target="_blank">#686</a></li> 
 <li><span style="color:#000000">从批处理构建中删除 benign [WARNINGS] </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fspring-batch%2Fpull%2F4066" target="_blank">#4066</a></li> 
 <li><span style="color:#000000">MongoItemReader#setSort 检查其参数 </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fspring-batch%2Fpull%2F4014" target="_blank">#4014</a></li> 
 <li><span style="color:#000000">使用 CollectionUtils.isEmpty 检查 Collection 的空数据 </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fspring-batch%2Fissues%2F4021" target="_blank">#4021</a></li> 
 <li><span style="color:#000000">添加<code>@Nullable</code>到 StepExecution::endTime </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fspring-batch%2Fpull%2F4034" target="_blank">#4034</a></li> 
 <li><span style="color:#000000">在开始测试之前删除 SQLLite Batch 数据库表 </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fspring-batch%2Fpull%2F4063" target="_blank">#4063</a></li> 
 <li><span style="color:#000000">升级到 Spring Framework 6.0.0-M3</span></li> 
 <li><span style="color:#000000">升级到 Spring Data 2022.0.0-M3</span></li> 
 <li><span style="color:#000000">升级到 Spring Integration 6.0.0-M2</span></li> 
 <li><span style="color:#000000">升级到 Spring AMQP 3.0.0-M2</span></li> 
 <li><span style="color:#000000">升级到 Spring for Apache Kafka 3.0.0-M3</span></li> 
 <li><span style="color:#000000">升级到 Micrometer 2.0.0-M2</span></li> 
</ul> 
<p><span style="color:#000000">更多详情可查看 </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fspring-batch%2Freleases%2Ftag%2F5.0.0-M2" target="_blank">change log</a>。</p>
