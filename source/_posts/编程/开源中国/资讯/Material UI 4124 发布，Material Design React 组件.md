                                                                                            <p><span style="color:#000000">Material UI 4.12.4 现已发布，</span>此版本的动机是修复与 TypeScript 4.7 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmui%2Fmaterial-ui%2Fpull%2F32041" target="_blank">#32041</a> ) 的兼容性问题。<span style="color:#000000">Material UI 是一组实现 Google Material Design 规范的 React 组件，它是一个前端 JS 框架，主要用在 web 端。 </span></p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">具体更新内容如下：</p>

 <li>[core] 修复 TS 4.7 兼容问题（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmui%2Fmaterial-ui%2Fpull%2F32041" target="_blank">#32041</a>，<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmui%2Fmaterial-ui%2Fpull%2F32126" target="_blank">#32126</a>）</li> 
 <li>[TextField] 在已弃用的道具上使用 deprecatedPropType ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmui%2Fmaterial-ui%2Fpull%2F27503" target="_blank">#27503</a> )</li>
