<div class="content">
                                                                                            <p>Thunderbird 91.0 版只能从 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.thunderbird.net%2F" target="_blank">thunderbird.net</a> 下载并安装，不能从 Thunderbird 78 或更早的版本直接升级至 91 版本。</p> 
<h3><strong>系统要求</strong></h3> 
<ul> 
 <li>Windows：Windows 7 或更高版本；</li> 
 <li>macOS：macOS 10.12 或更高版本；</li> 
 <li>Linux：GTK+ 3.14 或更高版本；</li> 
</ul> 
<h3>新增</h3> 
<ul> 
 <li>原生支持使用了 Apple Silicon CPU 的 macOS 设备；</li> 
 <li>Thunderbird 现在默认在多进程（e10s）模式下运行；</li> 
 <li>拉脱维亚语支持；</li> 
 <li>添加附件的新用户界面；</li> 
 <li>启用邮件重定向功能；</li> 
 <li>能够改变界面中账户的顺序；</li> 
 <li>允许在撰写窗口中显示空的 CC/BCC 行；</li> 
 <li>当向一个可能不存在的电子邮件地址发送邮件时，弹出警告；</li> 
 <li>当邮件的公共收件人超过阈值时，弹出警告信息；</li> 
 <li>增加对收件人地址中非 ASCII 字符的支持；</li> 
 <li>PDF.js 查看器现在包含在 Thunderbird 中；</li> 
 <li>OpenPGP：增加了配置选项，可以不对保存的草稿进行加密；</li> 
 <li>为已停止支持/不兼容的附加组件提供替换建议；</li> 
 <li>聊天：对 Matrix 服务器的 Beta 级支持；</li> 
 <li>……</li> 
</ul> 
<h3>变化</h3> 
<ul> 
 <li>将 "附加组件" 更名为 "附加组件和主题"，将 "选项" 更名为 "首选项"；</li> 
 <li>帐户设置移至一个独立标签；</li> 
 <li>Sending backend、SMTP 协议和 LDAP 协议的实现用 Javascript 重写；</li> 
 <li>Sending backend、SMTP 和 IMAP 协议现在只在 UTF-8 模式下运行；</li> 
 <li>移除 Movemail 支持；</li> 
 <li>基于图片的表情符号被替换成 Unicode；</li> 
 <li>……</li> 
</ul> 
<h3>已知问题</h3> 
<ul> 
 <li>未解决的问题：在 32 位 Windows 版本上 OTR 聊天加密失败；</li> 
</ul>
