                                                                                            <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">IPython 是一个综合环境，可以帮助程序员或开发人员等高级计算机用户测试或探索各种功能。尽管 Python 附带了一个强大的交互式解释器，使用户无需在目标计算机上创建额外的文件即可运行测试，但它在用户与软件交互方面存在一些限制。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">IPython 的三个核心部分包括一个高度交互式的 Python shell，一个解耦的双进程通信模型和交互式并行计算的架构。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">IPython 8.2 是 8.0 之后的第二个小版本，更新内容如下：</p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>自动建议现在可以用<span> </span><code>end</code><span> </span>键来选择<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fipython%2Fipython%2Fpull%2F13566%2F" target="_blank">PR #13566</a></li> 
 <li>修复了<span> </span><code>assert etb is not None</code><span> </span>的一些回溯问题<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fipython%2Fipython%2Fpull%2F13588%2F" target="_blank">PR #13588</a></li> 
 <li>历史记录现在可以从 sqitel 数据库中提取，而不是从内存中提取。特别是当使用<span> </span><code>%paste</code>时，粘贴的文本内容将成为历史的一部分。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fipython%2Fipython%2Fpull%2F13592%2F" target="_blank">PR #13592</a></li> 
 <li>修复<span> </span><code>Ctrl-\\</code>退出清理<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fipython%2Fipython%2Fpull%2F13603%2F" target="_blank">PR #13603</a></li> 
 <li>修复了在 IPython 之外使用<span> </span><code>ultratb</code><span> </span>ipdb 的支持<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fipython%2Fipython%2Fpull%2F13498%2F" target="_blank">PR #13498</a></li> 
 <li>修复了 8.1 版本中引入的 virtualenv 和 Python 3.8 的问题</li> 
</ul> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">更多详情可查看：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fipython%2Fipython%2Freleases%2Ftag%2F8.2.0" target="_blank">https://github.com/ipython/ipython/releases/tag/8.2.0</a></p>
