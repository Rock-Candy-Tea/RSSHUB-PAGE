                                                        <h3 style="text-align:left">本次更新内容：</h3> 
<ul> 
 <li>新增支持自定义选择接口推送和导出 <a href="https://gitee.com/ssssssss-team/magic-api/issues/I3TRT4" target="_blank">#I3TRT4</a></li> 
 <li>新增国产化数据库人大金仓kingbase方言适配 <a href="https://gitee.com/ssssssss-team/magic-api/issues/I3YCN2" target="_blank">#I3YCN2</a></li> 
 <li>新增主动刷新功能，用于在未开启集群配置且使用同一个存储不能同步的问题。</li> 
 <li>修复<code>DEBUG</code>会造成多次验证的BUG</li> 
 <li>修复设置参数类型不生效的BUG</li> 
 <li>优化UI样式，将右上角不常用的图标移至右下角、以及左侧菜单优化</li> 
</ul> 
<h4 style="text-align:start"><strong>项目介绍</strong></h4> 
<p style="text-align:left"><span style="background-color:#ffffff; color:#40485b">magic-api 是一个基于Java的接口快速开发框架，编写接口将通过magic-api提供的UI界面完成，自动映射为HTTP接口，无需定义Controller、Service、Dao、Mapper、XML、VO等Java对象即可完成常见的HTTP API接口开发</span></p>
