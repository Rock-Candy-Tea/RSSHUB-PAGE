                                                                                            <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">PMD 是一个代码分析器，能够帮助发现常见的编程问题，比如未使用的变量、空的 catch 块、不必要的对象创建等等。最初仅支持 Java 代码，目前还可支持 JavaScript、<a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fsalesforce.com%2F">Salesforce.com</a> Apex 和 Visualforce、PLSQL、Apache Velocity、XML 和 XSL 。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">PMD 6.48 正式发布，本次更新内容如下：</p> 
<h3 style="margin-left:.6em; margin-right:0; text-align:start">新特性</h3> 
<ul style="margin-left:0; margin-right:0"> 
 <li>对 Java 19 的支持 
  <ul style="margin-left:0; margin-right:0"> 
   <li>注意：对 Java 17 预览语言功能的支持已被删除</li> 
  </ul> </li> 
 <li>Gherkin 支持</li> 
</ul> 
<h3 style="margin-left:.6em; margin-right:0; text-align:start">错误修复</h3> 
<ul style="margin-left:0; margin-right:0"> 
 <li>apex 
  <ul style="margin-left:0; margin-right:0"> 
   <li>ApexSOQLInjection：增加支持计数查询</li> 
  </ul> </li> 
 <li>core 
  <ul style="margin-left:0; margin-right:0"> 
   <li>CPD也应该提供一个<span> </span><code>-debug</code><span> </span>标志</li> 
   <li>CPD：在 XML 报告中增加 token 的总数</li> 
   <li>如果报告被写到 stdout，stdout 不应该被关闭</li> 
   <li>额外的规则集没有在文档中列出</li> 
   <li>允许在 Java 18+ 下构建 PMD</li> 
  </ul> </li> 
 <li>java 
  <ul style="margin-left:0; margin-right:0"> 
   <li>支持 JDK 19</li> 
  </ul> </li> 
 <li>lua 
  <ul style="margin-left:0; margin-right:0"> 
   <li>修复了使用 CPD 时发现的几个相关 Lua 解析问题</li> 
  </ul> </li> 
</ul> 
<h3 style="margin-left:.6em; margin-right:0; text-align:start">弃用的 API</h3> 
<ul style="margin-left:0; margin-right:0"> 
 <li>实验性的 Java AST 类<span> </span><code>ASTGuardedPattern</code><span> </span>已经被弃用，并且将被删除。它是为 Java 17 和 Java 18 预览版引入的，但它不再被 Java 19 Preview 支持。</li> 
 <li>接口<span> </span><code>CPDRenderer</code><span> </span>已被弃用，对于自定义的 CPD 渲染器应使用新的接口<span> </span><code>CPDReportRenderer</code> 
  <ul style="margin-left:0; margin-right:0"> 
   <li><code>TestDescriptor</code><span> </span>类已被弃用，由<span> </span><code>RuleTestDescriptor</code><span> </span>取代</li> 
  </ul> </li> 
 <li><code>RuleTst</code><span> </span>的许多方法已经作为内部 API 被弃用了</li> 
</ul> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">更多详情可查看：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fpmd%2Fpmd%2Freleases%2Ftag%2Fpmd_releases%252F6.48.0" target="_blank">https://github.com/pmd/pmd/releases/tag/pmd_releases%2F6.48.0</a></p>
