                                                                                            <p>Eurynome Cloud v2.6.7.0 已经发布，企业级技术中台微服务架构。</p> 
<p>此版本更新内容包括：</p> 
<ul> 
 <li>主要更新 
  <ul> 
   <li>Spring Boot 版本升级至 2.6.7</li> 
   <li>去除 Spring Integration 强制降级配置，恢复使用 Spring Boot 默认依赖版本</li> 
   <li>由于 Fastjson 2.0.1 版本存在破坏性升级，暂不做升级处理。</li> 
   <li>使用最新版本 Antisamy XSS 防护配置文件。</li> 
   <li>Debezium 相关 Docker 版本升级至 1.9</li> 
  </ul> </li> 
 <li>其它更新 
  <ul> 
   <li>Lombok 版本升级至 1.18.24</li> 
   <li>Springdoc版本升级至 1.6.8</li> 
   <li>WxJava 版本升级至 4.3.1.B</li> 
   <li>Bce-java-sdk 版本升级至 0.10.206</li> 
   <li>Alipay-sdk-java 版本升级至 4.22.110.ALL</li> 
   <li>Aliyun-sdk-oss 版本升级至 3.14.1</li> 
  </ul> </li> 
</ul> 
<p>详情查看：<a href="https://gitee.com/herodotus/eurynome-cloud/releases/v2.6.7.0">https://gitee.com/herodotus/eurynome-cloud/releases/v2.6.7.0</a></p>
