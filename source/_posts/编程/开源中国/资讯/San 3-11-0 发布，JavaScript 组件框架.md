                                                        <p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><span style="color:#333333">San 是一个由百度开源的快速、轻量、灵活的 JavaScript 组件框架。San 3.11.0 现已正式发布，更新内容如下：</span></p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>【变更】- 重写了 index.d.ts，提供更强大的类型支持，同时有一些与原先不兼容</li> 
 <li>【变更】- checkbox 在双向绑定下，选中和非选中的添加删除，使用原始 value 值。对 value 为 number 等类型时有影响</li> 
 <li>【变更】- 模板 parse 时，取消对 tagName 的小写处理，以允许子组件的 tagName 使用 pascal-case</li> 
 <li>【优化】- 对 data.set 方法对象遍历过程进行调整，以减少对顺序的影响</li> 
</ul>
