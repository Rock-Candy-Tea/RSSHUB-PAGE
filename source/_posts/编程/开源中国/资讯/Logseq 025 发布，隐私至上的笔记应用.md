                                                        <p>Logseq 是一个知识管理和协作的平台。它专注于隐私和用户控制。服务器永远不会存储或分析你的私人笔记。目前支持 Markdown 和 Emacs Org 模式（更多内容即将加入）。</p> 
<p>Logseq 0.2.5 发布，更新内容如下：</p> 
<ul> 
 <li>修正了由 0.2.4 引入的数据丢失问题；</li> 
 <li>查询可以以表格或列表的形式呈现： 
  <ul> 
   <li><img alt height="618" src="https://static.oschina.net/uploads/space/2021/0719/065941_JZLO_4937141.gif" width="700" referrerpolicy="no-referrer"></li> 
  </ul> </li> 
 <li>块状时间戳被带回来了，默认情况下它是禁用的，你可以在设置页面上启用它；</li> 
 <li>现在添加了页面时间戳，"所有页面" 都支持按页面名称、链接、创建时间和更新时间排序； 
  <ul> 
   <li><img alt height="261" src="https://static.oschina.net/uploads/space/2021/0719/070013_R1DP_4937141.png" width="700" referrerpolicy="no-referrer"></li> 
  </ul> </li> 
 <li>修正了重新索引后，在 alt-dragged 块上的块参考会丢失的问题；</li> 
 <li>保留了区块属性顺序；</li> 
</ul> 
<p>更多详情可查看：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Flogseq%2Flogseq%2Freleases%2Ftag%2F0.2.5" target="_blank">https://github.com/logseq/logseq/releases/tag/0.2.5</a></p>
