<p>当前版本：0.2.8</p> 
<p>更新内容：<br> [新增] tooltip 警告提示，展现需要关注的信息。<br> [新增] input-number 数字输入框, 通过鼠标或键盘，输入范围内的数值。<br> [新增] layer 组件 isHtmlFangement 属性，函数调用时，用于解析 html 片段。<br> [新增] layer 组件 resize 属性, 开启弹层尺寸拉伸, 常用于 页面层 与 Iframe 层。<br> [加强] layer 组件 area 属性, 支持 字符串 与 数组 类型, 默认 auto 宽高根据内容自适应。<br> [修复] layer 组件 body 禁用拖动, 仅支持标题拖动窗体。<br> [修复] icon-picker 组件 select 图标时, 自动隐藏选择内容。<br> [修复] dropdown 组件触发方式为 hover 时，移动不到菜单子项的问题<br> [集成] utteranc.es 插件, 基于 issues 提供为文档提供留言能力。<br> [升级] layer-vue 1.2.0, 更稳定的 layer 版本。<br> [升级] vue 3.2.26 版本。<br> <br> 其他新闻：<br> 1. layui-vue 入驻 Github，Gitee 仍是后续工作重心。</p>
