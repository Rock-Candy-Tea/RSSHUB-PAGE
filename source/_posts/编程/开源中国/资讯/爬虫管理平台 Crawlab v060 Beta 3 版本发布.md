                                                                                            <p>分布式爬虫管理平台 Crawlab v0.6.0 Beta (20211224) 版本发布</p> 
<h2>概览</h2> 
<p>这是下个重大版本 v0.6.0 的第三次 beta 发布。伴随着更多功能和优化，正式版本 v0.6.0 将会很快发布了。</p> 
<h2><span>升级优化</span></h2> 
<ul> 
 <li> <p>[x] <strong style="color:#3594f7"><span>「</span>国际化<span>」</span></strong>. 支持中文.</p> </li> 
 <li> <p>[x] <strong style="color:#3594f7"><span>「</span>命令行工具上传爬虫<span>」</span></strong>. <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fcrawlab-team%2Fcrawlab%2Fissues%2F1020" target="_blank">#1020</a></p> </li> 
 <li> <p>[x] <strong style="color:#3594f7"><span>「</span>官方插件<span>」</span></strong>. 允许用户在 Crawlab 用户界面上安装 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdocs-next.crawlab.cn%2Fzh%2Fguide%2Fuse-crawlab%2Fpublic-plugins%2F" target="_blank">官方插件</a>.</p> </li> 
 <li> <p>[x] <strong style="color:#3594f7"><span>「</span>更多文档<span>」</span></strong>. 加入了关于插件和命令行工具的文档.</p> </li> 
</ul>

<ul> 
 <li> <p>[x] <strong style="color:#3594f7"><span>「</span>上传文件问题<span>」</span></strong> <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fcrawlab-team%2Fcrawlab%2Fissues%2F990" target="_blank">#990</a> <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fcrawlab-team%2Fcrawlab%2Fissues%2F1014" target="_blank">#1014</a> <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fcrawlab-team%2Fcrawlab%2Fissues%2F1022" target="_blank">#1022</a></p> </li> 
 <li> <p>[x] <strong style="color:#3594f7"><span>「</span>无法重命名文件<span>」</span></strong> <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fcrawlab-team%2Fcrawlab%2Fissues%2F1008" target="_blank">#1008</a></p> </li> 
</ul>

<h2><span>未来计划</span></h2> 
<p style="color:#2b2b2b; margin-left:0; margin-right:0">下一个版本有可能是 v0.6.0 的正式版本，但是没有确定。本次发布版本将会进行更多的测试，以保证其健壮性以及生产可用。</p>
