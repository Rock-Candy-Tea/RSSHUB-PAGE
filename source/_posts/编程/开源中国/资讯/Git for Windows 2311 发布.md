                                                                                            <p>Git for Windows 2.31.1 现已发布，主要更新内容如下：</p> 
<p><strong>New Features</strong></p> 
<ul> 
 <li>随附 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fgit%2Fgit%2Fblob%2Fv2.31.1%2FDocumentation%2FRelNotes%2F2.31.1.txt" target="_blank">Git v2.31.1</a></li> 
 <li>随附 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Flists.gnupg.org%2Fpipermail%2Fgnupg-announce%2F2021q1%2F000452.html" target="_blank">GNU Privacy Guard v2.2.27</a></li> 
 <li>随附 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.openssl.org%2Fnews%2Fopenssl-1.1.1-notes.html" target="_blank">OpenSSL v1.1.1k</a></li> 
 <li>随附 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fgit-lfs%2Fgit-lfs%2Freleases%2Ftag%2Fv2.13.3" target="_blank">Git LFS v2.13.3</a></li> 
</ul> 
<p><strong>Bug Fixes</strong></p> 
<ul> 
 <li> <p>现在可以通过 Git Bash 执行 Windows Store 版本的<code>python3.exe</code>了。</p> </li> 
</ul>
