                                                                                            <p style="color:#333333; margin-left:0; margin-right:0; text-align:left">DBeaver 22.0.3 已发布。</p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">DBeaver 是免费的多平台数据库工具，适用于开发人员、数据库管理员、分析师和所有需要使用数据库的人，并且支持所有流行的数据库：MySQL、PostgreSQL、SQLite、Oracle、DB2、SQL Server、Sybase、MS Access、Teradata、Firebird、Apache Hive、Phoenix、Presto 等。</p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><img alt src="https://oscimg.oschina.net/oscnet/up-38c99e9030ab8f3b1f73848b29cd8184697.png" referrerpolicy="no-referrer"></p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">更新内容：</p> 
<ul style="margin-left:0; margin-right:1.5em"> 
 <li>数据传输： 
  <ul style="margin-left:0; margin-right:0"> 
   <li>改进了“附加到现有文件”功能（适用于 CSV、TXT 和 XLSX 格式）</li> 
   <li>导出向导现在按需加载保存的属性</li> 
   <li>修复了使用重新排序/隐藏列导出结果集</li> 
   <li>以 SQL 格式导出：“拆分多个文件”现在可以正常工作</li> 
   <li>任务执行通知得到改进</li> 
   <li>SQL 脚本任务支持已修复（NullPointer 错误已解决）</li> 
   <li>修复了将多个表导出到单个目标表中的问题</li> 
   <li>从 CSV 导入已修复（空处理问题已解决）</li> 
   <li>为 CSV 导入添加了修剪空白选项</li> 
   <li>表列映射 UI 在 MacOS 上已修复</li> 
  </ul> </li> 
 <li>SQL 编辑器： 
  <ul style="margin-left:0; margin-right:0"> 
   <li>错误消息中添加了错误位置信息</li> 
   <li>查询错误突出显示得到改进</li> 
  </ul> </li> 
 <li>数据库导航器： 
  <ul style="margin-left:0; margin-right:0"> 
   <li>改进了对具有错误模式数的数据库的支持</li> 
   <li>列分组现在显示分组元素的数量</li> 
   <li>在属性编辑器中改进了自动完成</li> 
  </ul> </li> 
 <li>一般更新 
  <ul style="margin-left:0; margin-right:0"> 
   <li>DBeaver 现在在开始时显示 EULA（一次）</li> 
   <li>网络路径上的工作区问题已解决</li> 
   <li>在 RHEL 上修复 Kerberos 配置负载</li> 
  </ul> </li> 
 <li>Apache Phoenix：通过设置正确的驱动程序版本，修复了连接问题</li> 
 <li>CockroachDB：添加了 ROWID 列支持</li> 
 <li>DB2 ： 
  <ul style="margin-left:0; margin-right:0"> 
   <li>添加了序列支持</li> 
   <li>查看定义读取已实现</li> 
   <li>添加了唯一和检查约束支持</li> 
   <li>添加了元数据读取的多项小改进</li> 
  </ul> </li> 
 <li>Exasol：在视图/程序源代码中实现搜索功能</li> 
 <li>MySQL：修复了数据库转储期间的表锁定问题</li> 
 <li>Oracle：已修复查看源代码编辑问题（查看注释问题已解决）</li> 
 <li>PostgreSQL： 
  <ul style="margin-left:0; margin-right:0"> 
   <li>pg_dump 和 pg_restore 版本升级（至 PG 14）</li> 
   <li>脚本执行任务会显示所有脚本错误</li> 
  </ul> </li> 
 <li>Snowflake：添加了对自定义 SQL 查询的 LIMIT 支持（影响性能）</li> 
 <li>意大利语本地化得到显着改善</li> 
</ul>
