                                                        <p>Spring Boot 3.0.0-M5 和 2.6.12 已发布。更新内容主要是功能增强、文档优化、依赖升级和错误修复。</p> 
<p><strong><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fspring.io%2Fblog%2F2022%2F09%2F22%2Fspring-boot-3-0-0-m5-available-now" target="_blank">Spring Boot 3.0.0-M5</a></strong></p> 
<ul> 
 <li>改进 Ahead-of-Time (AOT) 处理和原生镜像支持</li> 
 <li>优化 actuator 端点清理</li> 
 <li>Jersey 支持 re-instated</li> 
</ul> 
<p><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fspring-boot%2Freleases%2Ftag%2Fv3.0.0-M5" target="_blank">Release Note</a>。</p> 
<p><strong><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fspring.io%2Fblog%2F2022%2F09%2F22%2Fspring-boot-2-6-12-available-now" target="_blank">Spring Boot 2.6.12</a></strong></p> 
<ul> 
 <li>支持 Java 19</li> 
 <li>将 NINETEEN 添加到 JavaVersion enum<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fspring-boot%2Fissues%2F32226" target="_blank">#32226</a></li> 
 <li>修复 PeriodStyle.ISO8601 检测不支持小写输入的问题<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fspring-boot%2Fpull%2F32235" target="_blank">#32235</a></li> 
 <li>修复使用 SnakeYaml 1.31 未正确处理 YAML 时间戳的问题<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fspring-boot%2Fissues%2F32228" target="_blank">#32228</a></li> 
 <li>修复 DurationStyle.ISO8601 检测不支持小写输入的问题<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fspring-boot%2Fpull%2F32223" target="_blank">#32223</a></li> 
</ul>
