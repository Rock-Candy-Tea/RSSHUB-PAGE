                                                                                            <p><span style="color:#333333">Apache ECharts 是一个使用 JavaScript 实现的开源可视化库，可以流畅的运行在 PC 和移动设备上，兼容当前绝大部分浏览器（IE8/9/10/11，Chrome，Firefox，Safari等），底层依赖矢量图形库 ZRender，提供直观，交互丰富，可高度个性化定制的数据可视化图表。</span></p> 
<p><span style="color:#333333">Apache ECharts 5.3.1 现已发布，具体更新内容如下：</span></p> 
<ul> 
 <li>[Feature] [i18n] 添加韩语翻译 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fapache%2Fecharts%2Fissues%2F16582" target="_blank">#16582</a></li> 
 <li>[Feature] [heatmap] 为 heatmap 添加 borderRadius 选项。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fapache%2Fecharts%2Fissues%2F16493" target="_blank">#16493</a></li> 
 <li>[Fix] [lines] 修复 setOption 与 notMerge 时效果线中潜在的内存泄漏。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fapache%2Fecharts%2Fissues%2F16525" target="_blank">#16525</a></li> 
 <li>[Fix] [line] 修复空数据的阶梯折线图。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fapache%2Fecharts%2Fissues%2F16435" target="_blank">#16435</a></li> 
 <li>[Fix] [line] 修复 lttb 采样中的空数据。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fapache%2Fecharts%2Fissues%2F16431" target="_blank">#16431</a></li> 
 <li>[Fix] [tooltip] 将默认时间格式化程序更改为 24 小时格式。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fapache%2Fecharts%2Fissues%2F16421" target="_blank">#16421</a></li> 
 <li>[Fix] [graphic] 修复光标在图形组件中不起作用的问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fapache%2Fecharts%2Fissues%2F16513" target="_blank">#16513</a></li> 
 <li>[Fix] [pictorialBar] 修复 pictorialBar 数据值为 0 的问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fapache%2Fecharts%2Fissues%2F16469" target="_blank">#16469</a></li> 
 <li>[Fix] [svg] 修复使用 decal 时的渲染错误。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fecomfe%2Fzrender%2Fissues%2F889" target="_blank">#889</a></li> 
 <li>[Fix] [tooltip] 修复 legend.tooltip.enterable 和 tooltip.enterable。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fapache%2Fecharts%2Fissues%2F16463" target="_blank">#16463</a></li> 
 <li>[Feature] [event] 在 axis events 中提供标签索引<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fapache%2Fecharts%2Fissues%2F16187" target="_blank">#16187</a></li> 
 <li>[Fix] [sunburst] 第一层的 sunburst 标签居中。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fapache%2Fecharts%2Fissues%2F16425" target="_blank">#16425</a></li> 
 <li>[Fix] [tooltip] 打开数据视图时隐藏工具提示。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fapache%2Fecharts%2Fissues%2F16321" target="_blank">#16321</a></li> 
 <li>[Fix] 修复零宽度图表的 sampling。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fapache%2Fecharts%2Fissues%2F16372" target="_blank">#16372</a></li> 
</ul>
