                                                        <p><span style="background-color:#ffffff; color:#333333">Node.js v14.18.0 正式发布。Node.js 是能够在服务器端运行 JavaScript 的开放源代码、跨平台 JavaScript 运行环境。Node.js 由Node.js Foundation（已与 JS Foundation 合并为 OpenJS Foundation）持有和维护，亦为 Linux 基金会的项目。Node.js 采用 Google 开发的 V8 运行代码，使用事件驱动、非阻塞和异步输入输出模型等技术来提高性能，可优化应用程序的传输量和规模。这些技术通常用于资料密集的即时应用程序。</span></p> 
<p><strong>主要更新内容</strong></p> 
<ul> 
 <li>断言：更改遗留断言的状态</li> 
 <li>缓冲区：引入 Blob</li> 
 <li>缓冲区：添加 base64url 编码选项</li> 
 <li>子进程：允许 options.cwd 接收 URL</li> 
 <li>子进程: 添加超时以生成和 fork</li> 
 <li>子进程: 允许取消已提交的 exec</li> 
 <li>子进程：添加 "重叠" stdio 标志</li> 
 <li>cli: 为 --conditions 标志添加 -C 别名</li> 
 <li>cli: 添加 --node-memory-debug 选项</li> 
 <li>dns：将 "尝试" 选项添加到解析选项 </li> 
 <li>dns: 允许 --dns-result-order 逐字更改默认 dns</li> 
 <li>文档：重构 fs 文档结构</li> 
 <li>esm：弃用传统的模块主要查找</li> 
 <li>fs：允许临时目录前缀为空字符串</li> 
 <li>fs: 允许无参数 fsPromises fileHandle 读取</li> 
 <li>fs：将异步迭代器的支持添加到 fsPromises.writeFile</li> 
 <li>fs：提高 fsPromises readFile 性能</li> 
 <li>fs: 添加 fsPromises.watch() </li> 
 <li>fs: 允许位置参数在 read 和 readSync 中是 BigInt</li> 
 <li>http2：添加对敏感标头的支持 </li> 
 <li>http2: 允许设置会话的本地窗口大小 </li> 
 <li>模块：添加对 URL 的支持 import.meta.resolve</li> 
 <li>模块：添加对 node:-prefixed require(…) 调用的支持</li> 
 <li>网络：引入 net.BlockList</li> 
 <li>node-api: 允许检索附加文件名</li> 
</ul>
