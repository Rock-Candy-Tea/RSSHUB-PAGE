                                                                                            <h2>fastposter v2.7.0 大版本发布 电商海报编辑器</h2> 
<p>fastposter海报生成器，电商海报编辑器，电商海报设计器，fast快速生成海报 海报制作 海报开发。二维码海报，图片海报，分享海报，二维码推广海报，支持Java Python PHP Go JS 小程序。基于Vue 和Pillow</p> 
<p>fastposter v2.7.0 发布 电商海报编辑器</p> 
<p>大版本更新</p> 
<ul> 
 <li>大版本升级 
  <ul> 
   <li>统一使用token，与专业版保持一致</li> 
   <li>统一使用id，后续会弃用posterId</li> 
  </ul> </li> 
 <li>精简fast.py代码到200行</li> 
 <li>简化响应代码</li> 
 <li>去除静态QRcode生成</li> 
 <li>增加响应头Fastposter，返回对应版本号</li> 
 <li>修改登录界面</li> 
 <li>token支持启动参数，或者app.yml文件配置</li> 
 <li>修改代码生成，支持token</li> 
 <li>增加示例海报</li> 
 <li>调整组件设置</li> 
 <li>更新使用文档</li> 
</ul>
