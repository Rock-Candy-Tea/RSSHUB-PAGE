                                                                                            <p style="margin-left:0px"><span style="color:#000000">Material UI 是一组实现 Google Material Design 规范的 React 组件，它是一个前端 JS 框架，主要用在 web 端。 </span></p> 
<p style="margin-left:0px"><span style="color:#000000">Material UI </span>5.8.7<span style="color:#000000"> 现已发布，</span>具体更新内容如下：</p> 
<h3><code><strong>@mui/material@5.8.7</strong></code></h3> 
<ul> 
 <li>[Autocomplete] 在 <code>useAutocomplete</code> 中加入缺失的属性 (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmui%2Fmaterial-ui%2Fpull%2F33269" target="_blank">#33269</a>) </li> 
 <li>[Autocomplete] 扩展<code>componentsProps</code>到包含<code>popper</code>和<code>popupIndicator</code>插槽 (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmui%2Fmaterial-ui%2Fpull%2F33283" target="_blank">#33283</a>) </li> 
 <li>[Select] <span style="color:#24292f">将空字符串注解为有效值 prop</span> (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmui%2Fmaterial-ui%2Fpull%2F33088" target="_blank">#33088</a>) </li> 
 <li>[SnackbarContent] <span style="color:#24292f">使用 css var provider 修复消息文本颜色</span> (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmui%2Fmaterial-ui%2Fpull%2F33285" target="_blank">#33285</a>) </li> 
</ul> 
<h3><code><strong>@mui/styled-engine@5.8.7</strong></code></h3> 
<ul> 
 <li>[styled-engine] <span style="color:#24292f">添加缺失的对 csstype 的类型依赖</span> (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmui%2Fmaterial-ui%2Fpull%2F33310" target="_blank">#33310</a>)</li> 
</ul> 
<h3><code><strong>@mui/system@5.8.7</strong></code></h3> 
<ul> 
 <li>[system] 简化 <code>CssVarsProvider</code> 的<span style="color:#24292f">主题输入类型</span>(<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmui%2Fmaterial-ui%2Fpull%2F33381" target="_blank">#33381</a>) </li> 
 <li>[system] 导出所需类型 (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmui%2Fmaterial-ui%2Fpull%2F33324" target="_blank">#33324</a>) </li> 
</ul> 
<h3><code><strong>@mui/joy@5.0.0-alpha.35</strong></code></h3> 
<ul> 
 <li>[Joy] <span style="color:#24292f">添加单选按钮文档</span> (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmui%2Fmaterial-ui%2Fpull%2F33254" target="_blank">#33254</a>) </li> 
 <li>[Joy] <span style="color:#24292f">添加开关文档</span> (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmui%2Fmaterial-ui%2Fpull%2F33302" target="_blank">#33302</a>) </li>
