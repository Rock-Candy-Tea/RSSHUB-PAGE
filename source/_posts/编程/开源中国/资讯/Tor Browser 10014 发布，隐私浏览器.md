                                                                                            <p>Tor Browser 10.0.14 已正式发布，这个版本将 Desktop Firefox 更新到 78.9.0esr。此外，Tor Browser 10.0.14 还将 NoScript 更新到 11.2.3，将 Tor 更新到 0.4.5.7。这个版本包括 Firefox for Desktop 的重要安全更新。感兴趣的用户现在可以从<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.torproject.org%2Fdownload%2F" target="_blank">下载页面</a>中下载。</p> 
<p>注意：这个版本不包括 Android 版本 Tor Browser 的更新。</p> 
<p>自 Tor Browser 10.0.12 以来的完整变更日志：</p> 
<ul> 
 <li>Windows + OS X + Linux 
  <ul> 
   <li>更新 Firefox 到 78.9.0esr；</li> 
   <li>更新 NoScript 到 11.2.3；</li> 
   <li>更新 Tor 到 0.4.5.7；</li> 
   <li>移除 Tor Browser 稳定版上的调查横幅。</li> 
  </ul> </li> 
 <li>Linux 
  <ul> 
   <li>修复了升级到 glibc 2.33 后的不稳定性</li> 
  </ul> </li> 
 <li>构建系统 
  <ul> 
   <li>Windows 
    <ul> 
     <li>为 Firefox 78.9 更新 mingw-w64 和 clang。</li> 
    </ul> </li> 
  </ul> </li> 
</ul> 
<p>更多详情可查看：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fblog.torproject.org%2Fnew-release-tor-browser-10014" target="_blank">https://blog.torproject.org/new-release-tor-browser-10014</a></p>
