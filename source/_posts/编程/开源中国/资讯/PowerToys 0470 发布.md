                                                        <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">Microsoft PowerToys 是 Windows 系统实用程序，供高级用户调整和简化其 Windows 体验，可最大限度地提高生产力。0.47 版本的目标主要集中在稳定性更新和优化、安装程序更新、错误修复和可访问性改进，更新内容如下：</p> 
<h3 style="margin-left:.6em; margin-right:0; text-align:start"><strong>变更日志</strong></h3> 
<ul style="margin-left:0; margin-right:0"> 
 <li>将 WinUI 升级到 2.7.0</li> 
 <li>通过压缩 GIF 文件减少了安装程序的大小</li> 
 <li>用 MarkdownTextblock 替换了 ShortcutTextContorl</li> 
 <li>合并了 nuget 包</li> 
 <li>为 PowerToys Run 添加了额外的日志和错误处理</li> 
 <li>摆脱了传统的 windevbuildagents 库</li> 
 <li>在 PowerToys 中增加了对嵌入式 MSIX 应用程序的支持</li> 
 <li>用 ComboBox 替换了 Image Resizer 的 ListView</li> 
 <li>修复了构建错误</li> 
</ul> 
<h3 style="margin-left:.6em; margin-right:0; text-align:start">常规</h3> 
<ul style="margin-left:0; margin-right:0"> 
 <li>修正了新更新改变 PowerToys 安装位置的问题</li> 
 <li>修正了 NumberBox 元素与删除按钮重叠的设置问题</li> 
 <li>修正了错误报告工具不能生成 .zip 文件的问题</li> 
 <li>更新了设置中的快捷方式配置体验</li> 
 <li>修正了侧边栏图标的宽度不一致的问题</li> 
 <li>修正了侧边栏的 UI 在某些本地化中对较长的文本字符串不进行缩放的问题</li> 
 <li>修正了设置中不显示无效的按键分配的问题</li> 
</ul> 
<h3 style="margin-left:.6em; margin-right:0; text-align:start">颜色选择器</h3> 
<ul style="margin-left:0; margin-right:0"> 
 <li>解决了可访问性问题</li> 
 <li>增加了 CIELAB 和 CIEXYZ 颜色格式</li> 
 <li>修正了手动改变 RGB 值时不会自动更新显示的颜色的错误</li> 
</ul> 
<h3 style="margin-left:.6em; margin-right:0; text-align:start">快捷指南</h3> 
<ul style="margin-left:0; margin-right:0"> 
 <li>重新添加了长按 Win 键来激活实用程序的功能</li> 
</ul>
