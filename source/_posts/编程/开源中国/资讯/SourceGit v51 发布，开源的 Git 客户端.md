                                                                                            <p>SourceGit v5.1 已经发布，开源的 Git 客户端。</p> 
<p>此版本更新内容包括：</p> 
<p>[CHANGELOG]</p> 
<ol> 
 <li>优化Tracking逻辑</li> 
 <li>支持Windows Terminal</li> 
 <li>优化git.exe查找</li> 
 <li>新版Welcome页</li> 
 <li>支持Visual Studio Code中打开仓库</li> 
 <li>发布增加 .NET 5.0 win-x64 版本</li> 
</ol>
