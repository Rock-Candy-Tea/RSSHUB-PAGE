                                                                                            <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">Universal Media Server 是一个兼容 DLNA，UPnP 和 HTTP/S 的媒体服务器，支持所有主要操作系统，包括 Windows，Linux 和 macOS 版本，几乎不需要配置就可以流式传输或转码许多不同的媒体格式。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">在更新至 V11 之前，团队正在努力提高 V10 版本的稳定性和性能。这个版本改进了文件名预设/解析、浏览和扫描速度、视频元数据的准确性、API 查询速度和稳定性、转码的稳定性等等。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">自 10.18.0 以来的变化：</p> 
<h3 style="margin-left:.6em; margin-right:0; text-align:start">常规：</h3> 
<ul style="margin-left:0; margin-right:0"> 
 <li>提高了视频元数据的查询速度</li> 
 <li>改进了浏览/扫描文件夹时的速度和驱动器的使用</li> 
 <li>匹配文件名中的更多年份，使视频元数据查询更准确</li> 
 <li>修复了转码后视频末尾的垃圾数据</li> 
 <li>修复了查找表升级/创建失败的问题</li> 
 <li>修复了电视剧数据的版本问题</li> 
 <li>修复了多余的电视剧查找</li> 
 <li>修复了响应缓存</li> 
 <li>修复了视频元数据缓存失败的问题</li> 
 <li>修复了挂起的 API 请求</li> 
</ul> 
<h3 style="margin-left:.6em; margin-right:0; text-align:start">依赖项：</h3> 
<ul style="margin-left:0; margin-right:0"> 
 <li>将 logback-version 从 1.2.10 升级到 1.2.11</li> 
</ul>
