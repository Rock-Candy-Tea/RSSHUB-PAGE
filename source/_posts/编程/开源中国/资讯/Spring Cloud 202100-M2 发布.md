<h3 style="margin-left:.6em; margin-right:0; text-align:start">Spring Cloud Commons</h3> 
<p data-darkreader-inline-color style="--darkreader-inline-color:#d8d4cf; color:#e8e6e3; margin-left:0px; margin-right:0px; text-align:start">调整用于 Sleuth 的<code>NamedContextFactory</code>API</p> 
<h3 style="margin-left:.6em; margin-right:0; text-align:start">Spring Cloud Config</h3> 
<p data-darkreader-inline-color style="--darkreader-inline-color:#d8d4cf; color:#e8e6e3; margin-left:0px; margin-right:0px; text-align:start">JDBC Environment 仓库支持除 String 之外的其他类型 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-cloud%2Fspring-cloud-config%2Fissues%2F1952" target="_blank">issue 1952</a></p> 
<h3 style="margin-left:.6em; margin-right:0; text-align:start">Spring Cloud Contract</h3> 
<ul style="margin-left:0; margin-right:0"> 
 <li>支持 UUID v4，以及修复大小写敏感问题 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-cloud%2Fspring-cloud-contract%2Fpull%2F1688" target="_blank">issue 1688</a></li> 
 <li>使用独立的 docker 镜像时支持自定义<code>build.gradle</code><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-cloud%2Fspring-cloud-contract%2Fissues%2F1672" target="_blank">issue 1672</a></li> 
</ul> 
<h3 style="margin-left:.6em; margin-right:0; text-align:start">Spring Cloud Gateway</h3> 
<p data-darkreader-inline-color style="--darkreader-inline-color:#d8d4cf; color:#e8e6e3; margin-left:0px; margin-right:0px; text-align:start">支持 HTTP 2 协议 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-cloud%2Fspring-cloud-gateway%2Fissues%2F7" target="_blank">issue 7</a></p> 
<h3 style="margin-left:.6em; margin-right:0; text-align:start">Spring Cloud Kubernetes</h3> 
<p data-darkreader-inline-color style="--darkreader-inline-color:#d8d4cf; color:#e8e6e3; margin-left:0px; margin-right:0px; text-align:start">保证不同 java 客户端之间的一致性</p> 
<h3 style="margin-left:.6em; margin-right:0; text-align:start">Spring Cloud Openfeign</h3> 
<p data-darkreader-inline-color style="--darkreader-inline-color:#d8d4cf; color:#e8e6e3; margin-left:0px; margin-right:0px; text-align:start">支持<code>@CookieValue</code>注解 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-cloud%2Fspring-cloud-openfeign%2Fpull%2F604" target="_blank">issue 604</a></p>
