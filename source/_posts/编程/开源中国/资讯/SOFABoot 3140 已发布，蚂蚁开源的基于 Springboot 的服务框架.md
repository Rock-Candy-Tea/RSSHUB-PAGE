                                                                                            <p><span style="color:#333333">SOFABoot 是蚂蚁金服开源的基于 Spring Boot 的研发框架，它在 Spring Boot 的基础上，提供了诸如 Readiness Check，类隔离，日志空间隔离等等能力。在增强了 Spring Boot 的同时，SOFABoot 提供了让用户可以在 Spring Boot 中非常方便地使用 SOFA 中间件的能力。</span></p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><span style="color:#333333">SOFABoot<span> </span></span>3.14.0<span style="color:#333333"><span> </span>现已发布，具体更新内容如下：</span></p>

<ul> 
 <li>更新 SpringBoot 2.6.9 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fsofastack%2Fsofa-boot%2Fpull%2F991" target="_blank">#991</a></li> 
 <li>支持自定义 HealthCheckerComparator <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fsofastack%2Fsofa-boot%2Fpull%2F997" target="_blank">#997</a></li> 
 <li>支持模块应用 api <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fsofastack%2Fsofa-boot%2Fpull%2F1001" target="_blank">#1001</a></li> 
</ul>
