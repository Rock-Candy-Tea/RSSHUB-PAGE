                                                                                            <p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><span style="color:#000000">Zulip Server 5.5 现已发布。Zulip 是一个开源团队协作工具，一款专为实时和异步对话而设计的现代团队聊天应用程序，支持快速搜索、拖放文件上传、图像预览、组私人消息、可听通知、错过电子邮件消息提醒与桌面应用等。</span></p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><span style="color:#000000">具体更新内容如下：</span></p> 
<ul> 
 <li>CVE-2022-31168：修复更改机器人角色的授权检查。由于 Zulip Server 5.4 和所有先前版本中的授权检查不正确，组织的成员可以制作一个 API 调用，将组织管理员权限授予其机器人之一。</li> 
 <li>为该工具添加了新选项<code>restore-backup</code>以简化在具有不同配置的系统上还原备份。</li> 
 <li>更新了翻译，包括对蒙古语和塞尔维亚语翻译的重大更新。</li> 
</ul>
