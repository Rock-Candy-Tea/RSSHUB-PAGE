                                                                                            <p>Spyder 5.1.3 现已发布。Spyder 是一个强大的交互式 Python 语言开发环境，提供高级的代码编辑、变量监视、交互测试、调试等特性，支持包括 Windows、Linux 和 OS X 系统。</p> 
<p><strong>主要更新内容</strong></p> 
<ul> 
 <li>修复在 macOS 应用程序中启动内核时的错误</li> 
 <li>修复 RTreeError 相关的代码片段</li> 
 <li>修复类型错误相关的代码片段</li> 
 <li>修复 5.1.2 版本中 Mac 安装程序的 bug</li> 
 <li>为工作目录组合框设置最小和推荐大小</li> 
 <li>删除主工具栏中弹出菜单的指示器</li> 
 <li>捕捉由 Kite (Snippets) 产生的几个错误</li> 
 <li>修复 debugpy 与 zip 不兼容（macOS 平台上）的问题</li> 
</ul>
