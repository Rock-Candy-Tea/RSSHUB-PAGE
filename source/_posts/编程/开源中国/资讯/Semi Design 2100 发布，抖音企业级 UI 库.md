                                                        <p>Semi Design 是现代、全面、灵活的设计系统和 UI 库，由字节跳动抖音前端与 UED 团队设计、开发并维护，是一款包含设计语言、React 组件、主题等开箱即用的中后台解决方案，可用于快速搭建美观的 React 应用。</p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">目前 Semi Design 发布了 v2.10.0 版本，此版本带来如下更新内容：</p> 
<ul> 
 <li>【Fix】 
  <ul> 
   <li>修复Form组件在<code>Nextjs</code>中使用时，<code>x-form-id</code>在服务端和客户端不匹配的报错问题<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FDouyinFE%2Fsemi-design%2Fissues%2F808" target="_blank">#808</a><span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fxuerzong" target="_blank">@xuerzong</a></li> 
   <li>修复 InputNumber 在有 precision 配置时，输入非法字符不会置空的问题<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FDouyinFE%2Fsemi-design%2Fissues%2F786" target="_blank">#786</a><span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FMuxinFeng" target="_blank">@MuxinFeng</a></li> 
   <li>修复 Carousel 自动播放时点击箭头切换或指示器切换不能即时响应问题</li> 
   <li>修复 InputNumber 受控使用且设置最小值，格式化传入的值到范围内时未 notifyChange 问题<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FDouyinFE%2Fsemi-design%2Fissues%2F812" target="_blank">#812</a></li> 
   <li>修复 ButtonGroup 传入 多个children 时 ts 类型检查报错的问题<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FDouyinFE%2Fsemi-design%2Fissues%2F811" target="_blank">#811</a></li> 
  </ul> </li> 
 <li>【Docs】 
  <ul> 
   <li>A11y：Switch、Bannner 增加 键盘和焦点 行为说明</li> 
   <li>更新 Tabs 组件 FAQ</li> 
  </ul> </li> 
</ul>
