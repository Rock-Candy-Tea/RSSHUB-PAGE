                                                                                            <p>Traefik 2.4.11 现已发布，该版本更新内容如下：</p> 
<p><strong>Bug fixes：</strong></p> 
<ul> 
 <li><strong>[k8s,k8s/crd,k8s/ingress] </strong>默认在 Kubernetes providers 上禁用 ExternalName 服务</li> 
 <li><strong>[k8s,k8s/crd,k8s/ingress] </strong>修复：测试中错误的 Kubernetes 资源名称和引用</li> 
 <li><strong>[k8s,k8s/crd] </strong>默认情况下禁用 IngressRoute provider 的 Cross-Namespace</li> 
 <li><strong>[logs,middleware]</strong> Accesslog：支持给定 header 的多个值</li> 
 <li><strong>[logs] </strong>忽略 http 1.0 请求主机丢失错误</li> 
 <li><strong>[middleware] </strong> Headers Middleware：支持 http.CloseNotifier 接口</li> 
 <li><strong>[TLS] </strong>检测证书内容修改</li> 
</ul> 
<p><strong>Documentation</strong><strong>：</strong></p> 
<ul> 
 <li><strong>[middleware,k8s] </strong>修正无效的子域</li> 
 <li>增加可用的供应商名称列表</li> 
 <li>修复维护者指南的页面标题</li> 
 <li>贡献部分中的错别字</li> 
</ul>
