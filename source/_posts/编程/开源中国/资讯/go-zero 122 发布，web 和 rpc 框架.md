                                                        <p>go-zero 1.2.2 发布了。go-zero 是一个集成了各种工程实践的 web 和 rpc 框架。通过弹性设计保障了大并发服务端的稳定性，经受了充分的实战检验。go-zero 包含极简的 API 定义和生成工具 goctl，可以根据定义的 API 文件一键生成 Go, iOS, Android, Kotlin, Dart, TypeScript, JavaScript 代码，并可直接运行。</p> 
<p>本次更新内容包括：</p> 
<p>框架：</p> 
<ol> 
 <li>支持 OpenTelemetry，Trace ID 通过 HTTP header X-Trace-Id 返回</li> 
 <li>syncx.AtomicError 可以设置 nil error</li> 
 <li>废弃 redis.NewRedis，使用 redis.New 代替</li> 
</ol> 
<p>goctl：</p> 
<ol> 
 <li>支持 protoc 插件 protoc-gen-goctl</li> 
 <li>bug fixes</li> 
</ol>
