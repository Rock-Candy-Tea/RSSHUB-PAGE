                                                        <p>Solon v1.4.2 已经发布，这是一个插件式 Java 微型开发框架。</p> 
<p>此版本更新内容包括：</p> 
<ul> 
 <li>1.升级 Snack3 版本</li> 
 <li>2.升级 Jetty 版本</li> 
 <li>3.升级 Smart-Http 和 Smart-Socket</li> 
 <li>4.升级 Nami coder: FastJson 版本，调整某些场景下反序列化策略</li> 
 <li>5.调整 Nami coder: hessian,protostuff，异常处理策略</li> 
 <li>6.增加 CloudListService 快捷接口</li> 
</ul> 
<p>详情查看：<a href="https://gitee.com/noear/solon/releases/v1.4.2">https://gitee.com/noear/solon/releases/v1.4.2</a></p>
