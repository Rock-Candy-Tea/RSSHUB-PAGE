                                                        <p>Oracle 已面向 64 位英特尔和 AMD(x86_64) 以及 64 位 Arm(aarch64) 平台发布了<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fblogs.oracle.com%2Flinux%2Fannouncing-the-release-of-oracle-linux-8-update-4" target="_blank"> Oracle Linux 8.4</a>。</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-e58c0b994d1b14044dfc1d63442fa410763.png" referrerpolicy="no-referrer"></p> 
<p>Oracle Linux 8.4 在安装镜像中使用的内核版本是 UEK R6 Update 2 和 RHCK。对于全新的安装，默认使用 UEK R6，并且是首次启动时的默认内核。UEK R6 是由 Oracle 开发、构建和测试的内核，它基于 Linux Kernel 5.4 mainline，比其他商业 Linux 内核提供了更多的创新。</p> 
<p>新版本主要变化：</p> 
<ul> 
 <li>已升级的模块流包括：python39:3.0, swig:4.0, subversion:1.14 和 redis:6</li> 
 <li>mysql-selinux 软件包包含 SELinux 模块，此模块为 MySQL 数据库提供规则</li> 
 <li> <p><span style="color:#404040">python-PyMySQL 软件包已更新至 0.10.1 版本，此软件包包含在 python36，python38 和 python39 模块中</span></p> </li> 
 <li> <p><span style="color:#404040">python3-pyodbc 软件包提供对开放式数据库连接 </span>(ODBC) <span style="color:#404040">数据库的访问</span></p> </li> 
 <li> <p>Oracle Linux 8.4 提供了 GCC Toolset 10，这是一个应用程序流，在 AppStream 信息库中以软件集合的形式分发。GCC Toolset 类似于 Oracle Linux Developer Toolset</p> </li> 
 <li> <p><span style="color:#404040">Bind 已更新至 9.11.26，提供了多项 bug 修复和针对旧版本的功能增强</span></p> </li> 
 <li> <p><span style="color:#404040">ghostscript 已更新至 9.27，提供了针对多个漏洞的修复程序</span></p> </li> 
 <li> <p>RHCK 已包含 EDAC 模块</p> </li> 
 <li> <p>RHCK 添加空闲内存页面功能 (Free memory page)</p> </li> 
 <li> <p>……</p> </li> 
</ul>
