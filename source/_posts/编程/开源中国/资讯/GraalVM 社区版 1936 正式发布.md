                                                        <p style="color:#333333; margin-left:0; margin-right:0; text-align:left">GraalVM 社区版 19.3.6 已发布。GraalVM 是 Oracle 打造的高性能跨语言虚拟机，支持运行 JavaScript、Python 3、Ruby、R、基于 JVM 的语言（如 Java、Scala 和 Kotlin），以及基于 LLVM 的语言，如 C 和 C++。</p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">此外，GraalVM 在编程语言之间具有高效的互操作性，并可提前将 Java 应用程序编译为本地可执行文件，以加快启动时间，降低内存开销。</p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">GraalVM 社区发行版包括：</p> 
<ul style="list-style-type:disc; margin-left:0; margin-right:0"> 
 <li>带有 GraalVM 编译器的 Java runtime</li> 
 <li>带有 GraalVM JavaScript 解释器的 Node.js runtime</li> 
 <li>LLVM runtime</li> 
 <li>开发者监控和调试工具</li> 
</ul> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">GraalVM 环境可以通过 GraalVM Updater 工具使用可选的可用组件进行扩展，例如 Native Image、Ruby、R、Python、LLVM 工具链、WebAssembly 和 Espresso。</p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">该版本是一个重要补丁更新 (CPU) 版本，并且是 GraalVM 社区版 19.3.x 的最后一个版本。</p> 
<p><strong>主要更新内容</strong></p> 
<ul> 
 <li>基于 Java 8 的 GraalVM 社区更新到 8u292，请参阅 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwiki.openjdk.java.net%2Fdisplay%2Fjdk8u%2FMain" target="_blank">OpenJDK 8u292 更新</a></li> 
 <li>基于 Java 11 的 GraalVM 社区更新到 11.0.11，请参阅 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwiki.openjdk.java.net%2Fdisplay%2FJDKUpdates%2FJDK11u" target="_blank">OpenJDK 11.0.11 更新</a></li> 
 <li>Node.js 运行时更新至 12.22.1，其中包含 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fnodejs.org%2Fen%2Fblog%2Fvulnerability%2Fapril-2021-security-releases%2F" target="_blank">2021 年 4 月安全版本</a>中提到的所有安全修复程序</li> 
</ul> 
<p>更多详细内容，请查看<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fgraalvm%2Fgraalvm-ce-builds%2Freleases%2Ftag%2Fvm-19.3.6" target="_blank">更新公告</a>。</p>
