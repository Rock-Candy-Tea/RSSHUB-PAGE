                                                        <p>Spring Data 2021.0.3 和 2020.0.11，以及 2021.1.0-M1 已发布。</p>

<h2>2021.1.0-M1</h2> 
<ul> 
 <li>升级至 Querydsl 5</li> 
 <li>支持广泛的 Redis 6.2 命令</li> 
 <li>支持 Spring Data JDBC 中依赖方言的自定义转换。这用于改进<code>java.time</code>跨多个数据库的类型支持</li> 
 <li>支持在 Spring Data JDBC 中流式传输大型结果数据集</li> 
 <li>支持查询派生中的 AggregateReference</li> 
 <li>REF_CURSOR 支持 Spring Data JPA 中的存储过程</li> 
 <li>升级到 MongoDB Driver 4.3 并初步支持 MongoDB Server 5.0 时间序列</li> 
 <li> <p>Spring Data MongoDB 提供了用于<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdocs.spring.io%2Fspring-data%2Fmongodb%2Fdocs%2F3.3.0-M1%2Freference%2Fhtml%2F%23mapping-usage.document-references" target="_blank">链接文档的</a>扩展功能，并在现有的 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdocs.spring.io%2Fspring-data%2Fmongodb%2Fdocs%2F3.3.0-M1%2Freference%2Fhtml%2F%23mapping-usage-references" target="_blank">DBRef</a> 方法旁边提供了一种基于属性的解决方案，它使用简单的值，例如目标<code>_id</code>或字段组合来链接到另一个文档。<br> 下面的示例显示了<code>Account</code>通过Java 代码中的<code>_id</code>对文档集合的引用以及它在 MongoDB 中的表示方式：</p> </li>
