                                                                                            <p><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Flibrewolf.net%2F" target="_blank">LibreWolf</a> 是 Firefox 的一个独立分支，其主要目标是隐私、安全和用户自由。LibreWolf 在 Firefox 的基础之上增加了对跟踪和指纹技术的保护，同时也包括一些安全方面的改进。</p> 
<p>Librewolf 96 正式发布，更新内容如下：</p> 
<ul> 
 <li>现在默认强制执行 <code>Same-Site=lax</code> cookie 策略。它是 Same-Site 策略的三个可用值之一，对于未指定 Same-Site 指令的站点，将自动使用 lax 值。</li> 
 <li>使用了一个全新的构建过程，并已经过测试</li> 
 <li>添加了新的 「about」对话框，并改进了 LibreWolf 的设置，搜索引擎也进行了修改</li> 
 <li>对 librewolf.cfg 做了许多改进，已经更新至 5.1 版本</li> 
 <li>噪声抑制和自动增益控制方面进行了重大改进，并在回声消除方面略有改进，为用户提供更好的整体体验</li> 
 <li>显着减少了主线程负载</li> 
</ul> 
<h3>修复：</h3> 
<ul> 
 <li>最新版本修复了视频间歇性掉线 SSRC 的问题</li> 
 <li>修复了 WebRTC 降级屏幕共享分辨率的问题，以提供更清晰的浏览体验的问题</li> 
 <li>还修复了某些网站上的视频质量下降问题</li> 
 <li>macOS 上的全屏分离视频已被暂时禁用，以避免出现损坏、亮度变化、缺少字幕和 CPU 使用率高的问题</li> 
</ul> 
<h3>开发者：</h3> 
<ul> 
 <li>提供对 color-scheme 属性的支持</li> 
 <li>默认情况下启用 Web Locks API</li> 
 <li>为 WebP 图像格式添加了图像编码器支持</li> 
</ul> 
<p>更多详情可查看：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgitlab.com%2Flibrewolf-community%2Fbrowser%2Fwindows%2F-%2Freleases" target="_blank">https://gitlab.com/librewolf-community/browser/windows/-/releases</a></p>
