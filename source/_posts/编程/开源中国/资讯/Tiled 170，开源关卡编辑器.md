                                                        <p>Tiled 是一个通用的 tile 地图编辑器，适用于所有基于 tile 的游戏，如 RPG、平台游戏等。</p> 
<p>Tiled 具有高度灵活的特性。它可以用来创建任何大小的地图，对 tile 的大小、可使用的层数或数量没有限制。Tiled 的地图格式（TMX）很容易理解，允许在任何地图中使用多个 tileset。Tileset 可以随时修改。</p> 
<p>Tiled 1.7.0 正式发布，该版本更新内容如下：</p> 
<ul> 
 <li>增加了基本的 "主要网格" 选项，有更强的线条；</li> 
 <li>增加了在 tileset 中重新排列 tiles 的功能；</li> 
 <li>增加了选择背景淡化颜色的选项；</li> 
 <li>增加了便携模式，当检测到 Tiled 可执行文件旁有 "tiled.ini" 时启用；</li> 
 <li>出于安全考虑，默认禁用项目扩展；</li> 
 <li>渲染选择预览和标签上方悬停的项目高亮；</li> 
 <li>更改了捐赠提醒，使之成为非模式化的提醒；</li> 
 <li>docs: 使用 TypeDoc 生成脚本 API 文档；</li> 
 <li>QMake/Qbs: 增加了一个禁用 DBus 支持的方法；</li> 
 <li>脚本：修正了脚本工具的 "mouseLeft" 回调；</li> 
 <li>脚本：修正了加载带有明确 "ext:" 前缀的图标；</li> 
 <li>脚本：使得 TileMap 坐标转换函数始终工作；</li> 
 <li>tBIN 插件：保存时转换颜色、对象和文件属性；</li> 
 <li>Python 插件：增加了 Layer.offset 和 Layer.setOffset；</li> 
 <li>Python 插件：Linux AppImage 现在支持 Python 3.6 而不是 3.5；</li> 
 <li>Windows：根据 notepad.exe 添加默认的 "在文本编辑器中打开" 命令；</li> 
 <li>修正了在切换文件时，在 Properties 视图可能出现的软件崩溃问题；</li> 
 <li>修正了从项目中添加/删除时对文件夹的 watching/unwatching；</li> 
 <li>修正了在导出时通过扩展确定所需的文件格式；</li> 
 <li>修正了GCC 10的编译问题；</li> 
</ul>
