<div class="content">
                                                                    
                                                        <p>PureScript  v0.14.2 发布了。PureScript 是个小巧而强大的静态类型语言，可以编译成 JavaScript。PureScript 主要是由 Haskell 和 PureScript 编写的。</p> 
<p><strong>New features</strong></p> 
<ul> 
 <li>使 type class 实例名称可选（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fpurescript%2Fpurescript%2Fpull%2F4085" target="_blank">#4085</a>）</li> 
</ul> 
<p><strong>Bug 修复</strong></p> 
<ul> 
 <li>未使用的标识符警告现在可以报告更小、更相关的 source spans（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fpurescript%2Fpurescript%2Fpull%2F4088" target="_blank">#4088</a>）。还修正了涉及 let-pattern 绑定影射现有标识符的情况下的不正确警告。</li> 
</ul> 
<p><strong>Internal</strong></p> 
<ul> 
 <li>删除 libtinfo 依赖项（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fpurescript%2Fpurescript%2Fissues%2F3696" target="_blank">#3696</a>）</li> 
 <li>将 CI 从 Travis 迁移到 GitHub Actions（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fpurescript%2Fpurescript%2Fpull%2F4077" target="_blank">#4077</a>）</li> 
 <li>从测试套件中删除 tasty 并使用 hspec ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fpurescript%2Fpurescript%2Fissues%2F4056" target="_blank">#4056</a> )</li> 
 <li>避免在测试套件中编译两次带有诊断功能的测试（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fpurescript%2Fpurescript%2Fpull%2F4079" target="_blank">#4079</a>）</li> 
 <li>在测试初始化​​中做更少的工作（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fpurescript%2Fpurescript%2Fpull%2F4080" target="_blank">#4080</a>）</li> 
 <li>遵循更多 HLint 建议（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fpurescript%2Fpurescript%2Fpull%2F4090" target="_blank">#4090</a>）</li> 
 <li>导出<code>rebuildModule'</code>以加速 Try PureScript! slightly（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fpurescript%2Fpurescript%2Fpull%2F4095" target="_blank">＃4095</a>）</li> 
 <li>将<code>purescript-ast</code>并入<code>purescript-cst</code>（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fpurescript%2Fpurescript%2Fpull%2F4094" target="_blank">#4094</a>）</li> 
</ul>
