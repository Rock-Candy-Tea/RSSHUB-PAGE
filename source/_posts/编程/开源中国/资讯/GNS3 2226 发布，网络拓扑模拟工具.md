                                                        <p>GNS3 是一个图形化的网络拓扑逻辑模拟工具。GNS3 2.2.26 现已发布，具体更新内容如下：</p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><strong>GUI</strong></p> 
<ul> 
 <li>在 Windows 包中将嵌入式 Python 升级到 3.7 版</li> 
 <li>在 Windows 包中升级 Visual C++ Redistributable for Visual Studio 2019</li> 
 <li>修复 Windows 包中的 SSL 支持</li> 
 <li>双击“Preferences”中的模板名称，打开“template configuration”对话框。Fixes <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FGNS3%2Fgns3-gui%2Fissues%2F3239" target="_blank">#3239</a></li> 
 <li>仅在启用旧节点时显示“virtio”网络适配器。Fixes <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FGNS3%2Fgns3-gui%2Fissues%2F1969" target="_blank">#1969</a></li> 
 <li>双击模板打开“template configuration”对话框。Fixes <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FGNS3%2Fgns3-gui%2Fissues%2F3236" target="_blank">#3236</a></li> 
 <li>修复使用“Filter”字段后无法展开“Custom symbols”的问题。Fixes <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FGNS3%2Fgns3-gui%2Fissues%2F3231" target="_blank">#3231</a></li> 
</ul> 
<p style="text-align:start"><strong>Server</strong></p> 
<ul> 
 <li>发布 web UI 2.2.26</li> 
 <li>按主题对符号进行排序。Fixes <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FGNS3%2Fgns3-gui%2Fissues%2F3230" target="_blank">#3230</a></li> 
 <li>修复内存百分比剩余警告。Fixes <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FGNS3%2Fgns3-gui%2Fissues%2F1966" target="_blank">#1966</a></li> 
 <li>更新 affinity symbols。Fixes <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FGNS3%2Fgns3-gui%2Fissues%2F3232" target="_blank">#3232</a></li> 
</ul>
