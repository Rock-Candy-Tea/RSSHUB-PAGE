                                                        <p>更新日志：</p> 
<p># 2.6.8</p> 
<ul> 
 <li>[重写] sort 方法，以便对数字、非数字及混合类等所有内容的排序进行支持</li> 
 <li>[新增] dropdown 组件的 align 参数，用于控制下拉菜单水平对齐方式（支持 left、center、right）</li> 
 <li>[新增] table 组件的 escape 参数，用于是否开启 xss 字符过滤（默认 false）</li> 
 <li>[加强] table 组件的自定义模板功能，返回 LAY_COL 字段，可得到当前列的表头配置信息</li> 
 <li>[加强] form 组件对验证不通过的表单项自动定位到可视区域，不再是只对输入框自动获焦定位 <a href="https://gitee.com/sentsin/layui/issues/I3O5TC" target="_blank">#I3O5TC</a></li> 
 <li>[加强] form 组件对 url 的验证</li> 
 <li>[修复] form 相关 css 的 layui-checkbox-disabled 书写错误（之前为 disbaled） <a href="https://gitee.com/sentsin/layui/issues/I1NGDC" target="_blank">#I1NGDC</a></li> 
 <li>[修复] form 组件的 select option 内容出现换行时的样式异常问题 <a href="https://gitee.com/sentsin/layui/issues/I3T5ZC" target="_blank">#I3T5ZC</a></li> 
 <li>[修复] colorpicker 颜色选择组件在 Firefox 下选择颜色时的若干兼容问题 <a href="https://gitee.com/sentsin/layui/issues/I3POP2" target="_blank">#I3POP2</a></li> 
 <li>[加强] colorpicker 组件的坐标定位计算方式</li> 
 <li>[修复] 低版本 ie 若干报错问题</li> 
</ul> 
<blockquote> 
 <p>该版本是 2.6.x 的一个收尾版，基本很稳定，推荐官网下载升级（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.layui.com%2F" target="_blank">www.layui.com</a>）。<br> 那么下一个版本 2.7 也将会开启另外一个篇章</p> 
</blockquote>
