                                                                                            <p><span style="color:#333333">Taro 3.4.13 现已发布。Taro 是一个开放式跨端跨框架解决方案，支持使用 React/Vue/Nerv 等框架来开发微信 / 京东 / 百度 / 支付宝 / 字节跳动 / QQ 小程序 / H5 等应用。具体更新内容如下：</span></p> 
<h2>修复</h2> 
<h3>小程序</h3> 
<ul> 
 <li>修复 preact 使用 <code>custom-tab-bar</code> 时报错的问题，<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FNervJS%2Ftaro%2Fissues%2F11962" target="_blank">#11962</a></li> 
 <li>修复编译小程序插件时报错的问题，<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FNervJS%2Ftaro%2Fissues%2F11932" target="_blank">#11932</a></li> 
 <li>让微信小程序的 <code>editImage</code> API 支持 promise 式调用</li> 
 <li>让字节跳动小程序的 <code>openAwemeUserProfile</code> API 支持 promise 式调用</li> 
 <li>修复 preact 地图组件 <code>onRegionChange</code> 事件报错的问题，<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FNervJS%2Ftaro%2Fissues%2F11462" target="_blank">#11462</a></li> 
 <li>修复 Vue <code>event.self</code> 不能使用的问题，<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FNervJS%2Ftaro%2Fissues%2F11967" target="_blank">#11967</a></li> 
</ul> 
<h3>H5</h3> 
<ul> 
 <li>修复 <code>innerAudioContext</code> API 的问题</li> 
</ul> 
<h3>RN</h3> 
<ul> 
 <li>修复当项目路径中包含 <code>react-native</code> 时 ， rn 启动失败的问题，<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FNervJS%2Ftaro%2Fissues%2F11988" target="_blank">#11988</a></li> 
</ul> 
<h3>CLI</h3> 
<ul> 
 <li>修复 <code>inspect</code> 命令的参数缩写无效的问题，<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FNervJS%2Ftaro%2Fissues%2F11941" target="_blank">#11941</a></li> 
 <li>修复 <code>designWidth</code> 配置为函数时校验不通过的问题</li> 
</ul> 
<h2>Typings</h2> 
<ul> 
 <li>补充 <code>AppConfig</code> 的字段</li> 
</ul>
