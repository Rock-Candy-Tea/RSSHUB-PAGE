                                                        <p><strong>trzsz</strong> ( trz / tsz ) 是兼容 tmux 的 rz / sz ：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ftrzsz%2Ftrzsz" target="_blank">https://github.com/trzsz/trzsz</a></p> 
<p><strong>trzsz.js</strong> 是 trzsz ( trz / tsz ) 的 Node.js 版组件：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ftrzsz%2Ftrzsz.js" target="_blank">https://github.com/trzsz/trzsz.js</a></p> 
<p><strong>trzsz.js</strong> 现已发布 <strong>0.3.2</strong> 版本，在 tmux 分屏里显示进度条时，影响到其他分屏的问题已解决。</p>
