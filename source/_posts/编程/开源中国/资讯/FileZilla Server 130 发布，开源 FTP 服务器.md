                                                                                            <p>FileZilla Server 是一个免费开源的 FTP 和 FTPS 服务器，能够提供与服务器的安全加密连接。Filezilla Server 没有实现对 SFTP（SSH 文件传输协议）的支持。</p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">FileZilla Server 1.3.0 现已发布，更新内容如下：</p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><strong>New features</strong></p> 
<ul> 
 <li>用于设置被动模式的配置向导</li> 
 <li>Linux：如果 sysctl knob kernel.yama.trace_scope 为 0，则发出警告</li> 
 <li>Linux：如果 sysctl knob fs.protected_hardlinks 为 0，则拒绝运行</li> 
</ul> 
<p style="text-align:start"><strong>Bugfixes and minor changes</strong></p> 
<ul> 
 <li>Admin UI：拒绝证书指纹提示不再触发自动重新连接</li> 
</ul> 
<p>更新说明：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Ffilezilla-project.org%2F" target="_blank">https://filezilla-project.org/</a></p>
