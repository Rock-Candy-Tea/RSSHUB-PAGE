                                                        <p>draw.io 是一款以 HTML5 和 JavaScript 开发的免费开源跨平台图形绘制软件。可用于创建流程图、线框图、UML 图、组织结构图和网络图等图表。</p> 
<p>draw.io 可以作为在线的跨浏览器网络应用，以及 Linux、macOS 和 Windows 的离线桌面应用。其离线应用使用 Electron 框架构建。</p> 
<p>draw.io 19.0 正式发布，自 18.0 版本以来的主要变化包括：</p> 
<ul> 
 <li>删除了对 IE 11 的支持</li> 
 <li>更新 mermaid.js 至 9.1.1</li> 
 <li>更新 JSZip 到 3.10.0</li> 
 <li>在布局中只忽略了有隐藏状态的单元格</li> 
 <li>修复了查看器中文档溢出的重置</li> 
 <li>修复了 CSV 导入中更新现有单元格的问题</li> 
 <li>在 CSV 导入中增加了对 orgchart 布局的支持</li> 
 <li>更新 AWS</li> 
 <li>修复了 sketch 主题中不一致的菜单</li> 
 <li>修复了导入嵌入式 SVG 时忽略 XML 数据的问题</li> 
 <li>修复了忽略双箭头标记选择的问题</li> 
 <li>修复黑暗模式下的链接图标</li> 
 <li>为平行布局增加了间距对话框</li> 
 <li>增加布局构造器名称的允许列表</li> 
 <li>为 childLayout 样式添加 JSON 值</li> 
 <li>为获取的连接数据增加了大小检查</li> 
 <li>允许链接中的自定义协议</li> 
 <li>增加了 DOM_PURIFY_CONFIG 全局变量</li> 
 <li>检查 ExportProxy 中的环境变量是否设置</li> 
 <li>处理 OneDrive 认证流程中可能出现的 400 错误</li> 
 <li>修复 CSV 导入中单元格元数据的更新</li> 
 <li>在过滤前解决 Java 代理中的 URL 参数问题</li> 
 <li>增加了对 CSV 文件中断行的支持</li> 
 <li>在导入页面 ID 时，修复了图表名称中的尾部空格。</li> 
 <li>修复了复合单元格的光标处理</li> 
 <li>在加载插件后添加严格的 CSP</li> 
</ul>
