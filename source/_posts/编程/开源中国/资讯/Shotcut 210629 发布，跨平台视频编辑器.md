                                                        <p>Shotcut 21.06.29 已经发布。这是一个错误修复版本，主要用于解决发行后在 21.05.01 版中发现的问题。Shotcut 是一款免费、开源、跨平台的视频编辑器，适用于 Windows、Mac 和 Linux。主要功能包括支持多种格式、本地时间线编辑且无需导入、支持输入和预览监视的Blackmagic Design、支持 4k 分辨率。</p> 
<p>具体更新内容如下：</p> 
<p><strong>Minor Additions or Changes</strong></p> 
<ul> 
 <li>为 Timer video filter 增加了速度参数。</li> 
 <li>增加了键盘快捷键来切换库存布局：Alt+1 到 6。</li> 
 <li>在播放列表中增加了导入按钮。</li> 
 <li>在 Properties > Speed 旁边增加了应用按钮，以改善使用鼠标滚轮时的工作流程。</li> 
</ul> 
<p><strong>New Bugs (in version 21.05) </strong></p> 
<ul> 
 <li>修复了播放不同的播放列表项目时可能会挂起。</li> 
 <li>修复了 Export 中的alpha预设。</li> 
 <li>修复了改变 Settings > Preview Scaling 时可能出现的崩溃。</li> 
 <li>修复了在同一剪辑上使用 Crop: Source 和 Size, Position & Rotate 滤镜时可能出现的崩溃。</li> 
 <li>修复了 Size, Position & Rotate filters 不能重新加载的问题。</li> 
 <li>修复了在源播放器中改变 Properties > Color Range 的片段。</li> 
</ul> 
<p><strong>Recent Bugs</strong></p> 
<ul> 
 <li>修复了<strong> </strong>Crop: Source filter 的 Center bias。</li> 
 <li>修复了更改 Settings > Theme 时可能发生的崩溃。</li> 
 <li>修复了重置 Size, Position & Rotate > Zoom 时保持图像中心的问题。</li> 
 <li>改进了在 Size, Position & Rotate filter 中使用鼠标滚轮进行缩放的问题。</li> 
 <li>改进了当 keyframes 被改变或删除时，新的 Zoom keyframe values 的范围。</li> 
 <li>修复了自定义导出预设可能会以不正确的帧率加载。</li> 
 <li>修复了 Stabilize analysis (.stab) file 无法打开时的崩溃问题。</li> 
 <li>修复了当 Stabilize > Accuracy 为 9 或更高时的崩溃问题。</li> 
</ul>
