                                                        <p>MX Linux 是一个基于 Debian 稳定版的中量级 Linux 操作系统，使用 antiX 的核心组件，以及由 MX 社区创建或打包的额外软件。</p> 
<p>MX Linux 21.1 是 MX Linux 21 版本的第一次更新，其中包括错误修复、内核和应用程序更新。该版本是一个相对较小的版本更新呢，更新内容如下：</p> 
<ul> 
 <li>基于 Debian 11.3 "bullseye"</li> 
 <li>新的和更新的应用程序</li> 
 <li>磁盘管理器（Disk-manager）回到了官方 iso 中</li> 
 <li>mx-samba-config 用于配置 samba/cifs 用户共享</li> 
 <li>mx-installer 得到了一些改进</li> 
 <li>所有内核更新：AHS 现在使用 5.16 内核</li> 
</ul>
