                                                                                            <h1><span>更新日志 (v0.6.0-beta.20211120)</span></h1> 
<h2><span>概览</span></h2> 
<p style="color:black; margin-left:0; margin-right:0">这是下个重大版本 v0.6.0 的第二次 beta 发布（参考 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fcrawlab-team%2Fcrawlab%2Freleases%2Ftag%2Fv0.6.0-beta.20210803" target="_blank">第一次 beta 版本</a>)。伴随着更多功能和优化，正式版本 v0.6.0 将会很快发布了。</p> 
<h2><span>升级优化</span></h2> 
<h4><span>后端</span></h4> 
<ul style="list-style-type:disc"> 
 <li> <p>[x] <strong style="color:black">插件框架</strong>. <strong style="color:black">Crawlab 插件框架 (CPF)</strong> 已发布. 详情请参考 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdocs-next.crawlab.cn%2Fzh%2Fguide%2Fuse-crawlab%2Fbasic-concepts%2Fplugin.html" target="_blank">这里</a>.</p> </li> 
 <li> <p>[x] <strong style="color:black">Git 集成</strong>. Git 集成被作为内置功能.</p> </li> 
 <li> <p>[x] <strong style="color:black">Scrapy 集成</strong>. Scrapy 集成以插件形式存在，插件为 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fcrawlab-team%2Fplugin-spider-assistant" target="_blank">spider-assistant</a>.</p> </li> 
 <li> <p>[x] <strong style="color:black">依赖集成</strong>. Dependency 集成以插件形式存在，插件为 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fcrawlab-team%2Fplugin-dependency" target="_blank">dependency</a>.</p> </li> 
 <li> <p>[x] <strong style="color:black">消息通知</strong>. 消息通知功能以插件形式存在，插件为 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fcrawlab-team%2Fplugin-notification" target="_blank">notification</a>.</p> </li> 
 <li> <p>[x] <strong style="color:black">文档网站</strong>. 搭建 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdocs-next.crawlab.cn" target="_blank">文档网站</a>.</p> </li> 
</ul> 
<h4><span>前端</span></h4> 
<ul style="list-style-type:disc"> 
 <li> <p><strong style="color:black">Bug 修复</strong>.</p> </li> 
</ul> 
<h4><span>待完成</span></h4> 
<ul style="list-style-type:disc"> 
 <li> <p>[ ] <strong style="color:black">关联任务</strong>. 如果任务执行模式为 “所有节点” 或 “指定节点”，那么将会有主任务和子任务之分。</p> </li> 
 <li> <p>[ ] <strong style="color:black">Crontab 编辑器</strong>. 可视化 Crontab 编辑的前端组件。</p> </li> 
 <li> <p>[ ] <strong style="color:black">结果去重</strong>.</p> </li> 
 <li> <p>[ ] <strong style="color:black">环境变量</strong>.</p> </li> 
 <li> <p>[ ] <strong style="color:black">国际化</strong>. 支持中文.</p> </li> 
 <li> <p>[ ] <strong style="color:black">前端易用性优化</strong>. 更多高级功能，例如表格形式保存。</p> </li> 
 <li> <p>[ ] <strong style="color:black">日志自动清理</strong>.</p> </li> 
 <li> <p>[ ] <strong style="color:black">跟多文档</strong>.</p> </li> 
</ul> 
<h2><span>未来计划</span></h2> 
<p style="color:black; margin-left:0; margin-right:0">下一个版本有可能是 v0.6.0 的正式版本，但是没有确定。本次发布版本将会进行更多的测试，以保证其健壮性以及生产可用。</p>
