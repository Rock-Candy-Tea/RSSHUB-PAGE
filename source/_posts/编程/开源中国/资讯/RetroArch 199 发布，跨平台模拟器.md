                                                                                            <p>RetroArch 1.9.9 现已发布。RetroArch 是款功能强大的跨平台模拟器，不但能够模拟许多不同的游戏主机，还能在 Windows、MacOS、Linux、Android、iOS 以及多种游戏主机上执行。</p> 
<p>主要更新内容如下：</p> 
<ul> 
 <li>3DS：添加底部触摸屏菜单</li> 
 <li>3DS/SAVESTATES：在 RAM 中保存和加载保存状态</li> 
 <li>AUDIO/MIXER：确保在调用 CMD_EVENT_AUDIO_REINIT 时重新启用菜单声音</li> 
 <li>AUDIO/RESAMPLER/MIXER：修复使用质量低于“normal”的“sinc”重采样器时的菜单声音（音频混合）</li> 
 <li>CHEEVOS：升级到 rcheevos 10.2</li> 
 <li>CHEATS：在“Cheats”菜单中添加增强的搜索功能</li> 
 <li>CHEATS/RUNAHEAD：修复使用第二个实例 runahead 时的 cheats</li> 
 <li>配置：添加选项以（强制）将当前核心选项写入磁盘（快速菜单）</li> 
 <li>CORE INFO CACHE：从 core info cache 中删除核心路径。现在应该使 core info cache 可移植（例如：你可以将 RetroArch 移动到单独的目录，它们仍然可以工作）。</li> 
 <li>D3D11：添加 HDR 支持（目前对 UWP 禁用）</li> 
 <li>INPUT/WINRAW：修复启用 overlay 时的崩​​溃</li> 
 <li>MAC/METAL：添加 Discord RPC 支持</li> 
 <li>VIDEO：屏幕分辨率列表清理</li> 
 <li>VULKAN：修复一些 Vulkan 验证层错误</li> 
 <li>UWP：更新图标，由 Danp142 提供</li> 
 <li>UWP/XBOX：禁用 Xbox 上的 CPU 模型检查，因为它不起作用甚至可能崩溃</li> 
 <li>UWP/VFS/XBOX：UWP VFS 驱动的代码清理和简化</li> 
 <li>......</li> 
</ul>
