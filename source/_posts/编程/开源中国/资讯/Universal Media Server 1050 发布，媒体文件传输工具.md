                                                                                            <p>Universal Media Server 是一个兼容 DLNA，UPnP 和 HTTP/S 的媒体服务器，支持所有主要操作系统，包括 Windows，Linux 和 macOS 版本，几乎不需要配置就可以流式传输或转码许多不同的媒体格式。</p> 
<p>Universal Media Server 10.5.0 发布，该版本更新内容如下：</p> 
<p>通用：</p> 
<ul> 
 <li>增加了新的选项，在将视频移动到新的文件夹后，可以标记为完全播放的视频；</li> 
 <li>实施了外部 API，允许用户从授权的外部来源对他们的 UMS 实例进行操作；</li> 
 <li>修正了某些电视剧的元数据无法保存的问题；</li> 
 <li>改进了对剧集文件名的检测；</li> 
 <li>优化了某些渲染器的网络使用和对未实现的 UPnP 操作的处理；</li> 
 <li>修正了音频封面图的提取；</li> 
 <li>修正了某些用户的数据库升级问题；</li> 
 <li>修正了对某些渲染器的识别；</li> 
</ul> 
<p>渲染器：</p> 
<ul> 
 <li>改进了对松下 VT60 上 H.264 的支持；</li> 
</ul> 
<p>依赖项：</p> 
<ul> 
 <li>将 checkstyle 从 8.41 升级到 8.42；</li> 
 <li>将 com.sun.xml.bind-version 从 3.0.0 升级到 3.0.1；</li> 
 <li>将 commons-lang3 从 3.11 升级到 3.12.0；</li> 
 <li>将 icu4j 从 68.2 升级到 69.1；</li> 
 <li>将 JRE 从 15 升级到 8u292；</li> 
 <li>将 maven-project-info-reports-plugin 从 3.1.1 升级到 3.1.2；</li> 
 <li>将 MediaInfo 从 20.09 升级到 21.03；</li> 
 <li>将 metadata-extractor 从 2.15.0 升级到 2.16.0；</li> 
 <li>将 spotbugs-maven-plugin 从 4.2.2 升级到 4.2.3；</li> 
 <li>将 twelvemonkeys-imageio-version 从 3.6.4 提升到 3.7.0；</li> 
</ul>
