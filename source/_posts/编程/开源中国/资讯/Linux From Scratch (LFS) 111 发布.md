                                                                                            <p>Linux From Scratch 社区宣布 Linux From Scratch (LFS) 和 Beyond Linux From Scratch（BLFS）11.1 版本正式发布。<br> <br> 此版本是 LFS 和 BLFS 的重大更新，LFS 版本包括对 glibc-2.35 和 binutils-2.38 的更新，且将 Linux 内核更新到 5.16.9 版本 ，整个<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.linuxfromscratch.org%2Flfs%2Fview%2F11.1%2F" target="_blank">教程文档</a>都进行了更改。</p> 
<ul> 
 <li>在线阅读 LFS 11.1 教程：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.linuxfromscratch.org%2Flfs%2Fview%2F11.1%2F" target="_blank">https://www.linuxfromscratch.org/lfs/view/11.1/</a></li> 
 <li>下载 LFS 11.1：<a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fwww.linuxfromscratch.org%2Flfs%2Fdownloads%2F11.1%2F" target="_blank">http://www.linuxfromscratch.org/lfs/downloads/11.1/</a></li> 
 <li>LFS 11.1 完整变更列表：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.linuxfromscratch.org%2Flfs%2Fview%2F11.1%2Fchapter01%2Fchangelog.html" target="_blank">https://www.linuxfromscratch.org/lfs/view/11.1/chapter01/changelog.html</a> </li> 
</ul> 
<p>BLFS 是一个基于 LFS 的项目，除了拥有 LFS 的教程文档之外，还包含大量软件包。BLFS 11.1 版本包括大约 1000 个软件包，远超 10.0 版本。除了大量的文本和格式更改外，此版本还比以前的版本进行了 900 多项更新。</p> 
<ul> 
 <li>在线阅读 BLFS 11.1 教程：<a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fwww.linuxfromscratch.org%2Flfs%2Fdownloads%2F11.1%2F" target="_blank">http://www.linuxfromscratch.org/lfs/downloads/11.1/</a></li> 
 <li>下载 BLFS 11.1： <a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fwww.linuxfromscratch.org%2Fblfs%2Fdownloads%2F11.1%2F" target="_blank">http://www.linuxfromscratch.org/blfs/downloads/11.1/</a> </li> 
</ul> 
<p>LFS 介绍：</p> 
<blockquote> 
 <p style="margin-left:0px">Linux From Scratch 项目简称 LFS，它提供必要的步骤，帮你一步一步地创建自己定制的 Linux 系统。关于为什么有人希望安装一套 LFS 系统，我们可以给出很多的理由。人们问得最多的问题是：既然可以下载到一份成型的套件 —— 比如 Debian 或者 Red Hat，为什么还要去争论从头开始手工安装一份 Linux 系统呢？</p> 
 <p style="margin-left:0px">这是一个很合理的问题，我将回答你。LFS 存在的最重要的原因就是教给人们认识一个 Linux 系统其内部是如何工作的。自己创建一份 LFS 系统将让你了解到 Linux 工作的每个细节，以及它们是如何彼此依赖和协同工作的。最重要的在于，你可以按自己的喜好和需求定制它。</p> 
 <p style="margin-left:0px">Beyond Linux From Scratch(BLFS) 提供了在 LFS 的基础上，进一步完善 Linux 基本系统的指引。在 BLFS 中，读者可以学习到如何为系统增加网络、X桌面、声音、打印机和扫描仪支持。</p> 
</blockquote>
