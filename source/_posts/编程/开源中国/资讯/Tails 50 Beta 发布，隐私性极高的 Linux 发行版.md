                                                                                            <p>Tails 开发团队公开了其 Tails 5.0 的测试版，供公众测试。Tails 5.0 的正式版将在下个月发布，它基于最新的 Debian GNU/Linux 11 "Bullseye" 操作系统。</p> 
<p>Tails 5.0 Beta 发布，更新内容如下：</p> 
<h3>变化和升级：</h3> 
<ul> 
 <li>更新至 Debian 11 (Bullseye) 
  <ul> 
   <li>GNOME 3.38</li> 
   <li>OnionCircuits 0.7</li> 
   <li>Pidgin 2.14.1</li> 
   <li>Mozilla Thunderbird 91.7.0</li> 
   <li>LibreOffice 7.0.4</li> 
   <li>Inkscape 1.0.2</li> 
   <li>GIMP 2.20.22</li> 
   <li>Audacity 2.4.2</li> 
   <li><a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2FX.Org" target="_blank">X.Org</a> 1.20.11</li> 
  </ul> </li> 
 <li>移除 Python 2</li> 
 <li>Tor Browser 11.0.10</li> 
 <li>免驱动的打印和扫描</li> 
</ul> 
<h3>修复问题</h3> 
<ul> 
 <li>修复欢迎屏幕中的语言翻译</li> 
 <li>修复用长密码打开 Veracrypt 卷的问题</li> 
</ul> 
<h3>已知问题</h3> 
<ul> 
 <li>WhisperBack 无法发送错误报告</li> 
 <li>Tails 安装程序有时无法安装或升级 Tails</li> 
 <li>Additional Software 有时在配置后首次启动时无法安装软件包</li> 
 <li>OpenPGP Applet 无法使用：其图标无法显示</li> 
 <li>Tails 5.0 Beta 比以前的版本需要更多时间来启动</li> 
</ul> 
<p>Tails 5.0 尚处于测试阶段，因此不要将该版本用于任何生产环境中。</p>
