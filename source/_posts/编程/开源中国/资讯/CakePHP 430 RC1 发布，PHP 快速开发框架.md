 <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">CakePHP 是一个运用了诸如 ActiveRecord、Association Data Mapping、Front Controller 和 MVC（model–view–controller） 等著名设计模式的开源 Web 框架。CakePHP 用 PHP 编写，以 Ruby on Rails 的概念为模型，并在 MIT 许可下进行分发。</p> 
 <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">CakePHP 4.3.0 RC1 是 4.3.0 的第一个候选版本发布，更新内容如下：</p> 
 <h3 style="margin-left:.6em; margin-right:0; text-align:start">新功能亮点：</h3> 
 <ul style="margin-left:0; margin-right:0"> 
  <li>增加了<span> </span><code>Controller::middleware()</code>，允许你只为一个控制器定义中间件；</li> 
  <li>增加了<span> </span><code>Http\\Client::addMockResponse()</code>，使集成测试更容易编写，而不需要使用复杂的定义模拟;</li> 
  <li>增加了一个基于 JSON 的记录器;</li> 
  <li><code>FormHelper</code><span> </span>生成了 aria 属性，提高了开箱即用的可访问性;</li> 
  <li>增加了一个新的 fixture 子系统，利用你的迁移或 SQL 转储来生成测试数据库模式;</li> 
 </ul> 
 <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">更多详情可查看：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fcakephp%2Fcakephp%2Freleases%2Ftag%2F4.3.0-RC1" target="_blank">https://github.com/cakephp/cakephp/releases/tag/4.3.0-RC1</a></p>
