                                                        <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">Brave 是基于 Chromium 引擎开发的免费开源网络浏览器。Brave 是一个注重隐私的浏览器，它通过在默认设置中自动阻止在线广告和网站跟踪器来将自己与其他浏览器区分开来。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">Brave Browser 1.33 正式发布，该版本更新内容如下：</p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>增加了签署类型数据的能力（eth_signTypedData_v3 和 eth_signTypedData_v4）</li> 
 <li>将 Chromium 升级到 96.0.4664.110</li> 
 <li>增加了通过组件更新来更新新标签页背景图片的功能</li> 
 <li>在 Windows 下增加了卸载后的用户调查</li> 
 <li>增加了通过 brave://flags/#file-system-access-api 启用 FileSystemAccess API 的功能</li> 
 <li>根据 kkarfalcon 在 HackerOne 上的报告，删除了来自".onion"来源的跨来源请求中的 referrer 和 origin</li> 
 <li>实现了有时间限制的同步 QR 码</li> 
 <li>更新了新标签页上的 Brave Rewards 小工具的用户界面</li> 
 <li>更新了 Brave Rewards 的某些错误信息，以便在 Brave://rewards 下显示为模版</li> 
 <li>更新了新标签页下的默认小部件列表</li> 
 <li>更新了 Brave 以在 Linux 上使用 "etc/brave/policies"</li> 
 <li>从某些 URL 中删除了已知的用户跟踪参数 "wbraid "和 "gbraid"</li> 
 <li>修正了能够在 brave://settings/wallet/networks 下删除活动网络的问题</li> 
 <li>修正了在某些情况下，点击自适应验证码的 "Solve" 无法正常工作的问题</li> 
 <li>修复了在某些窗口尺寸上无法访问 IPFS 设置链接的问题</li> 
 <li>修正了在启用自动 DNSLink 重定向时无法回到 IPFS 页面</li> 
 <li>在 brave://settings/wallet 中添加了默认货币和默认加密货币设置</li> 
 <li>在 brave://settings/wallet/networks 下增加了编辑和删除网络的菜单选项</li> 
 <li>在 brave://settings/rewards 下增加了奖励设置部分</li> 
 <li>……</li> 
</ul> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">更多详情可查看：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fbrave.com%2Flatest%2F" target="_blank">https://brave.com/latest/</a></p>
