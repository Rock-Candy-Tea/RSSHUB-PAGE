<div class="content">
                                                                                            <p>PHP 8.0.9 现已<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.php.net%2Farchive%2F2021.php%232021-07-29-1" target="_blank">发布</a>，这是一个错误修复版本；官方鼓励所有 PHP 8.0 用户升级到此版本。具体 更新内容如下：</p> 
<ul> 
 <li>Core： 
  <ul> 
   <li>修复了 bug <a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fbugs.php.net%2F81145" target="_blank">#81145</a>（对于 +4GB 文件，copy() 和 stream_copy_to_stream() 失败）。</li> 
   <li>修复了 bug <a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fbugs.php.net%2F81163" target="_blank">#81163</a>（在 __sleep 中不正确处理间接变量）。</li> 
   <li>修复了 bug <a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fbugs.php.net%2F81159" target="_blank">#81159</a>（使用对象作为字符串偏移量时的对象到 int 警告）。</li> 
   <li>修复了 bug <a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fbugs.php.net%2F80728" target="_blank">#80728</a>（PHP 内置 Web 服务器在可以终止进程时重置超时）。</li> 
   <li>修复了 bug <a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fbugs.php.net%2F73630" target="_blank">#73630</a>（内置 Weberver - 覆盖 $_SERVER['request_uri']）。</li> 
   <li>修复了 bug <a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fbugs.php.net%2F80173" target="_blank">#80173</a>（使用 zend_assign_to_variable() 的返回值是不安全的）。</li> 
   <li>修复了 bug​​​​​​​ <a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fbugs.php.net%2F73226" target="_blank">#73226</a>（-- <a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fbugs.php.net%2F73226" target="_blank">r</a> [fcez] 总是返回零退出代码）。</li> 
  </ul> </li> 
 <li>Intl： 
  <ul> 
   <li>修复了 bug​​​​​​​ <a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fbugs.php.net%2F72809" target="_blank">#72809</a>（Locale::lookup() 在使用 canonicalize 选项时出现错误结果）。</li> 
   <li>修复了 bug​​​​​​​ <a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fbugs.php.net%2F68471" target="_blank">#68471</a>（IntlDateFormatter 对“GMT+00:00”时区失效）。</li> 
   <li>修复了 bug​​​​​​​ <a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fbugs.php.net%2F74264" target="_blank">#74264</a>（grapheme_strrpos () 因负偏移而损坏）。</li> 
  </ul> </li> 
 <li>OpenSSL： 
  <ul> 
   <li>修复了 bug​​​​​​​ <a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fbugs.php.net%2F52093" target="_blank">#52093</a>（openssl_csr_sign 截断 $serial）。</li> 
  </ul> </li> 
 <li>PCRE： 
  <ul> 
   <li>修复了 bug​​​​​​​ <a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fbugs.php.net%2F81101" target="_blank">#81101</a>（PCRE2 10.37 显示意外结果）。</li> 
   <li>修复了 bug​​​​​​​ <a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fbugs.php.net%2F81243" target="_blank">#81243</a>（为 preg_replace() 分配了太多内存）。</li> 
  </ul> </li> 
 <li>Reflection： 
  <ul> 
   <li>修复了 bug​​​​​​​ <a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fbugs.php.net%2F81208" target="_blank">#81208</a>（从属性创建 newInstance 时出现分段错误）。</li> 
  </ul> </li> 
 <li>Standard： 
  <ul> 
   <li>修复了 bug​​​​​​​ <a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fbugs.php.net%2F81223" target="_blank">#81223</a>（flock() 只锁定文件的第一个字节）。</li> 
  </ul> </li> 
</ul>
