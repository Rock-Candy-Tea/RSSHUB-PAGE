                                                                                            <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">11 月 5 日，Python 发布了 3.9.8 和 3.11.0a2 两个版本，主要是<span style="color:#2e3033">绑定了固定的 Tcl/Tk 版本，来修复 tkinter 相关内容在 masOS 12 Monterey 操作系统中显示不出来的问题。</span></p> 
<h2 style="margin-left:.6em; margin-right:0; text-align:start">Tcl/Tk 更新</h2> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><span style="color:#2e3033">tkinter 文件对话框在最近发布的 macOS 12 Monterey 中显示不出来，包括在 Python 内置的 IDLE 中。Python 3.9.8 和Python 3.11.0a2 绑定了固定的 Tcl/Tk 版本来修复这个问题。</span></p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><span style="color:#2e3033">在 3.9.8 中， Tcl/Tk  是一个打了补丁的 8.6.11 版本，而 3.11.0a2 则是在最新的 8.6.12rc1 版本。</span></p> 
<h3 style="margin-left:.6em; margin-right:0; text-align:start">Python 3.9.8</h3> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.python.org%2Fdownloads%2Frelease%2Fpython-398%2F" target="_blank">Python 3.9.8</a><span> </span>是 3.9 系列的第七个维护版本，3.10 才是 Python 3 的最新版本，建议<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fpython.org%2Fdownloads%2F" target="_blank">下载 Python 3.10</a><span> </span>。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><span style="color:#2e3033">从 3.9.7 以来，社区已经有 202 条提交，这说明用户对 Python 3.9 的改进仍然有相当大的兴趣。相比之下，在发布周期的同一阶段，Python 3.8 减少了 25% 以上的提交。由于使用了最新版本的 Tk， IDLE 和其他基于 Tk 的应用程序行为可能出现出现差异，可能需要将第三方组件(如 pip )升级到最新版本。</span></p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><span style="color:#2e3033">下一个 Python 3.9 维护版本将是 3.9.9，目前计划于 2022-01-03 发布。</span></p> 
<h3 style="margin-left:.6em; margin-right:0; text-align:start">Python 3.11.0a2 </h3> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">Python 3.11.0a2 是 Python 3.11 的第二个 Alpha 版本，可<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.python.org%2Fdownloads%2Frelease%2Fpython-3110a2%2F" target="_blank">点此下载</a>。注意：Alpha是预览版，目的是为了测试新特性和 bug 修复的状态，以及测试发布的过程，不建议将此版本投入生产环境。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start"> Alpha 阶段的特性可能会被添加到 Python 3.11 的 beta 版本 (预计 2022-05-06 发布)，也有可能会被修改或删除，直到发布 RC 版本 (预计 2022-08-01)才能确认。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">Python 3.11的许多新特性仍在规划和编写，目前为止，主要的新功能和变化如下：</p> 
<ul style="margin-left:0; margin-right:0"> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.python.org%2Fdev%2Fpeps%2Fpep-0657%2F" target="_blank">PEP 657</a><span> </span>：在错误回溯中包括详细的错误位置。</li> 
 <li>相比 3.10 版本， CPython 3.11 的 Py Performance 基准测试几何平均值快了约 12% 。</li> 
</ul> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">Python 3.11 的 下一个预览版是 3.11.0a3，目前定于2021-12-06 (周一)发布。</p>
