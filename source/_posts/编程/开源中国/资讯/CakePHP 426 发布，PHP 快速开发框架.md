                                                        <p>CakePHP 是一个运用了诸如 ActiveRecord、Association Data Mapping、Front Controller 和 MVC（model–view–controller） 等著名设计模式的开源 Web 框架。CakePHP 用 PHP 编写，以 Ruby on Rails 的概念为模型，并在 MIT 许可下进行分发。</p> 
<p>CakePHP 4.2.6 更新内容如下：</p> 
<p>错误修正</p> 
<ul> 
 <li>更新了 FrozenDate 的 API 文档；</li> 
 <li>改进了 Security.salt 警告信息；</li> 
 <li>改进了 Collection::groupBy() 中没有使用 key 时的异常信息；</li> 
 <li>cake 插件现在可以处理 Windows 下的符号链接；</li> 
 <li>修正了 ControllerFactory 中的变量字符串参数解析；</li> 
 <li>修正了在虚拟机上删除文件时可能出现的 "文本文件繁忙" 错误；</li> 
 <li>修正了 assertMailSentToAt(0) 不能正确工作的问题。</li> 
</ul>
