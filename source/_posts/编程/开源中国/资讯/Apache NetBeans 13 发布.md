<div class="content">
                                                                                            <p><span style="background-color:#ffffff; color:#121212">Apache NetBeans 13 现已<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fblogs.apache.org%2Fnetbeans%2Fentry%2Fannounce-apache-netbeans-13-released" target="_blank">发布</a>，包含了许多新功能和改进。Apache NetBeans 是一个针对 Java SE、Java EE、PHP、JavaScript、HTML5 等的完整 IDE，包括对 Groovy 和 C/C++ 的一些支持。</span></p> 
<p><span style="background-color:#ffffff; color:#121212">一些更新内容<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.phoronix.com%2Fscan.php%3Fpage%3Dnews_item%26px%3DApache-NetBeans-13" target="_blank">如下</a>：</span></p> 
<ul> 
 <li>通过"Light FlatLaf"外观提供了新的默认用户体验</li> 
</ul> 
<p><img alt height="312" src="https://oscimg.oschina.net/oscnet/up-afac0d3d3ed81cb3270b2bb1059f4cf08c1.jpg" width="500" referrerpolicy="no-referrer"></p> 
<ul> 
 <li>升级了对 Maven、Gradle 和 PHP 的支持。升级后的 Maven 还支持 Maven 守护进程 (mvnd)。</li> 
 <li>NetBeans 13 现在还捆绑了 nb-javac，以获得更简单、更流畅的启动体验。nb-javac 是 NetBean 的 Java 编辑器支持使用的 Java 编译器插件。</li> 
</ul> 
<p><img alt height="312" src="https://oscimg.oschina.net/oscnet/up-8243901335e2f13dde561c67cdababba310.jpg" width="500" referrerpolicy="no-referrer"> </p> 
<ul> 
 <li>PHP 支持方面有关 PHP 8.1 的许多升级</li> 
 <li>PHPStan 静态分析器集成改进</li> 
 <li><span style="background-color:#ffffff; color:#121212">CSS 解析器修复</span></li> 
 <li><span style="background-color:#ffffff; color:#121212">语言服务器协议 (LSP) 更新</span></li> 
</ul> 
<p><span style="background-color:#ffffff; color:#121212">更多详情可查看发布公告：</span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fnetbeans.apache.org%2Fdownload%2Fnb13%2Findex.html" target="_blank">https://netbeans.apache.org/download/nb13/index.html</a></p>
