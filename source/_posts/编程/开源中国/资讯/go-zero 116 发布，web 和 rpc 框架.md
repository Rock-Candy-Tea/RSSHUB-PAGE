                                                                                            <p style="text-align:left">go-zero 1.1.5 发布了。go-zero 是一个集成了各种工程实践的 web 和 rpc 框架。通过弹性设计保障了大并发服务端的稳定性，经受了充分的实战检验。go-zero 包含极简的 API 定义和生成工具 goctl，可以根据定义的 API 文件一键生成 Go, iOS, Android, Kotlin, Dart, TypeScript, JavaScript 代码，并可直接运行。</p> 
<p style="text-align:left">本次更新内容包括：</p> 
<p style="text-align:left">框架：</p> 
<ol> 
 <li>支持 websocket, 示例见 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fzeromicro%2Fzero-examples%2Ftree%2Fmain%2Fchat" target="_blank">https://github.com/zeromicro/zero-examples/tree/main/chat</a></li> 
 <li>logx支持自定义时间格式</li> 
 <li>支持 postgresql</li> 
</ol> 
<p style="text-align:left">goctl：<br> 1. 支持 mongodb 代码生成<br> 2. 一些小改进</p> 
<p style="text-align:left">更新详情查看：https://github.com/tal-tech/go-zero/releases/tag/v1.1.6</p>
