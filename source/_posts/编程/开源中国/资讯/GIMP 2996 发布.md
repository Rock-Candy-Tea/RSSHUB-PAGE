                                                        <p>GIMP 2.99.6 现已发布，这是迈向 GIMP 3.0 版本漫长曲折旅程中的又一步。官方表示，GIMP 2.99.6 包含了相当多可见的、有趣的改进，然而最大的变化却隐藏在相当稳定发展的 API（面向插件开发者的应用编程接口）中，不为公众所知。</p> 
<p>公告指出，“许多已经为 GIMP 2.99.2 或 2.99.4 移植的第三方插件最终会被破坏，而且在我们稳定 API 之前，它们很有可能在进一步的开发版本中再次被破坏。我们对此表示歉意，但这是为一个正在开发的程序制作插件的代价。我们认为，与其在未来几年内被一个糟糕的界面困住，不如现在就这样做（因为一旦 GIMP 3 出来，稳定性就会得到保证）。”</p> 
<p>GIMP 即 GNU Image Manipulation Program（GNU 图像处理程序）的首字母组成，是一个自由开源的位图图像编辑器，用于图像照片润饰及编辑、自由绘图、调整大小、裁剪、照片蒙太奇、装换图像格式以及其他专业任务。GIMP 几乎拥有所有图象处理所需的功能，号称 Linux 下的 Photoshop。 </p> 
<p><img alt src="https://attachments.tower.im/tower/15180ec23f5663d27e61be5826e69660?version=auto" referrerpolicy="no-referrer"><img alt src="https://oscimg.oschina.net/oscnet/up-dbda11dfb708f8ca5642af4f7996b5cc24e.png" referrerpolicy="no-referrer"></p> 
<p><strong>GIMP 2.99.6 更新亮点：</strong></p> 
<ul> 
 <li>支持 Off-canvas guides</li> 
 <li>Canvas Size 对话框中的 Template selector </li> 
 <li>在 canvas 上用捏的手势进行缩放</li> 
 <li>改进的 Paint Select 工具（实验性）</li> 
 <li>更好地处理 PNG 的 gAMA 和 cHRM chunks</li> 
 <li>尽管在 3.0 及其他与插件相关的工作之前尚未稳定下来，但有关 GIMP API 的工作仍在继续</li> 
</ul> 
<p>官方表示，在 GIMP 3.0 对 GTK3 工具包的使用方面，GtkAction 的移植仍然是最大的麻烦。此外，还有一些"严重的 Wayland 问题"仍亟待解决。GIMP 开发人员仍不确定 GIMP 3.0 何时可以稳定发布，也没有给出确定的更新时间表。</p>
