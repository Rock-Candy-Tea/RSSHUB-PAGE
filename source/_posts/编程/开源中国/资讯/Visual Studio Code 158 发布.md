                                                        <p>Visual Studio Code 1.58 现已发布， 其中一些主要<strong>亮点内容如下</strong>：</p> 
<ul> 
 <li>编辑器中的终端：创建或移动终端到编辑器区域</li> 
</ul> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-fc20363610160f4872ee18e3f2e95df64d1.png" referrerpolicy="no-referrer"></p> 
<ul> 
 <li>Workspace Trust：快速定制工作区信任设置，禁用受限模式横幅</li> 
</ul> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-3faf53fec46e2deb0a376b91d18a2860384.png" referrerpolicy="no-referrer"></p> 
<ul> 
 <li>编辑器滚动条自定义： 设置滚动条的可见性、宽度和单击导航行为</li> 
</ul> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-371d62ec5fee8027e21aa8454ab50166dd4.png" referrerpolicy="no-referrer"></p> 
<ul> 
 <li>Markdown 预览 KaTeX 支持：Markdown 预览中的数学支持</li> 
</ul> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-db1540b0bd5ce1a608f8207bae545bfe503.png" referrerpolicy="no-referrer"></p> 
<ul> 
 <li>编辑器设置的优化：支持多个复选框和枚举式下拉菜单</li> 
</ul> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-b4c9a126e5e07051b5f43c05b09c77a6ab0.gif" referrerpolicy="no-referrer"></p> 
<ul> 
 <li>粘性调试环境选择：调试器会记住用户之前的环境选择</li> 
</ul> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-60fbcdb4e94f138f812f453e97ece61665f.gif" referrerpolicy="no-referrer"></p> 
<ul> 
 <li>Jupyter 交互式窗口：在交互式窗口中直接运行 Jupyter 代码</li> 
</ul> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-f910eb5fe12256bda4ce974d2a8582e524a.gif" referrerpolicy="no-referrer"></p> 
<ul> 
 <li>Jupyter 笔记本调试：从 VS Code 中调试 Jupyter notebook 代码</li> 
</ul> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-703fa97d49bce9b0d95440d174bdf04d721.png" referrerpolicy="no-referrer"></p> 
<ul> 
 <li>“Open in VS Code” 标志：让访问者在 VS Code 中快速打开 GitHub repo</li> 
</ul> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-b1093a0c00b87e43433800c4728e2bfe023.png" referrerpolicy="no-referrer"></p> 
<ul> 
 <li>数据科学教程： 了解 Jupyter Notebooks、PyTorch 等</li> 
</ul> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-bd19d1edfef2a04e5faea65939dac856c40.png" referrerpolicy="no-referrer"></p>
