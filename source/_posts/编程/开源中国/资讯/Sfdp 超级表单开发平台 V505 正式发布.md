<h1 style="text-align:left">Sfdp V5.0.5正式版</h1> 
<p style="text-align:left"><strong>欢迎使用 Sfdp 超级表单开发平台</strong></p> 
<blockquote> 
 <p style="text-align:left"><strong><span style="color:#e74c3c">5.0.5 增加了事件管理，优化模型管理</span></strong></p> 
 <p style="text-align:left"><strong><span style="color:#e74c3c">我们已经在我们的商业版本Gadmin全面应用，如果觉得还不错，可以给我们点个 Star支持下我们呗，开源不易。</span></strong></p> 
</blockquote> 
<ul> 
 <li>Sfdp通过可视化的业务单据设计，可以完整的构建一套业务逻辑系统。</li> 
 <li>Sfdp通过脚本、函数的支持，在逻辑实现更加简洁、可用。</li> 
</ul> 
<h3 style="text-align:start"><strong>关于5.0.5版本更新内容</strong></h3> 
<ul> 
 <li>【新增】单据事件模型【支持新增前，新增后，修改前，修改后，删除前，审核后】</li> 
 <li>【新增】修复了子表为空的报错</li> 
 <li>【新增】优化列子表的操作，新增了事务逻辑</li> 
 <li>【新增】修复了未部署导致挂带工作流错误</li> 
</ul> 
<ul> 
 <li>【优化】优化SFDP权限过滤组</li> 
 <li>【优化】新增列表树形结构</li> 
</ul> 
<h3 style="text-align:start"><strong>关于5.0.4版本更新内容</strong></h3> 
<ul> 
 <li>【新增】热更新布局功能 ---可对业务进行无缝实时更新设计，让设计更加便捷简单</li> 
 <li>【新增】新增布局组件  --- 丰富页面设计功能</li> 
 <li>【新增】新增函数对接接口</li> 
 <li>【新增】新增流程对接接口</li> 
</ul> 
<ul> 
 <li>【优化】设计器底层代码</li> 
 <li>【优化】更新组件底层代码</li> 
</ul> 
<h3 style="text-align:start"><strong>关于5.0.3版本更新内容</strong></h3> 
<ul> 
 <li>【新增】新增图片上传组件</li> 
 <li>【新增】新增按钮、超级链接功能</li> 
 <li>【新增】新增时间区间范围选择</li> 
 <li>【新增】富文本编辑框</li> 
 <li>【新增】系统用户选择组件</li> 
 <li>【新增】系统角色选择组件</li> 
</ul> 
<ul> 
 <li>【优化】设计器部分细节功能，UI底层优化</li> 
 <li>【优化】优化上传组件部分功能</li> 
 <li>【优化】列表组件功能</li> 
</ul> 
<p style="text-align:left"><strong>历史版本更新5.0.2版本更新内容</strong></p> 
<ul> 
 <li>【新增】全面对系统UI调整，为栅格布局</li> 
 <li>【新增】全新子表构建系统</li> 
 <li>【新增】全新UI配置系统</li> 
</ul> 
<ul> 
 <li>【优化】设计器全部功能</li> 
 <li>【优化】设计器底层功能</li> 
</ul>
