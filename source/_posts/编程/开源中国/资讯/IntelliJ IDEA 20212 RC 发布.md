<div class="content">
                                                                    
                                                        <p>IntelliJ IDEA 2021.2 RC 现已发布，用户可以使用 Toolbox 应用程序或从<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fblog.jetbrains.com%2Fwebstorm%2F2021%2F07%2Fwebstorm-2021-2-beta%2F" target="_blank">官方网站</a>升级到最新版本。</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-797ae190db5040108843f2d4579f01c37e2.png" referrerpolicy="no-referrer"></p> 
<p><strong>主要更新内容</strong></p> 
<ul> 
 <li>性能测试插件：开放支持请求线索提供非附加链接</li> 
 <li>修复 Android SDK 管理器在 apple slicon 上失败的问题</li> 
 <li>添加 scala sdk 时，在模块 untitled814 中跳过没有 Scala SDK 的 Scala 文件</li> 
 <li>IntelliJ Gradle 集成在拥有子项目时选择了错误的 settings.gradle 和 build.gradle</li> 
 <li>带有 “允许并行运行” 选项的 Gradle 任务在执行时挂起</li> 
 <li>如果在 gradle 视图中执行包含构建任务并因异常而失败，则 Gradle 复合构建会调用错误的目标</li> 
 <li>Maven importer 找不到 parent 中声明的依赖版本，并报验证错误</li> 
 <li>显示弹出暂存列表和特定暂存的操作</li> 
 <li>关闭项目时删除打开的空暂存文件</li> 
 <li>从运行日志复制到暂存窗口</li> 
 <li>Spring：向 Spring Initializr 向导添加共享索引同意选项</li> 
 <li>在 Spring 5.2.5 中支持 @DynamicPropertySource 方法</li> 
 <li>Java EE：提供全局重构以迁移到 Jakarta</li> 
 <li>HTTP 客户端：为隐式 Web 引用提供生成 HTTP 请求参考</li> 
 <li>支持将命名空间标记为收藏</li> 
 <li>SSH 终端添加新连接</li> 
</ul> 
<p>详情请查看<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fblog.jetbrains.com%2Fidea%2F2021%2F07%2Fintellij-idea-2021-2-release-candidate%2F" target="_blank">更新公告</a>。</p>
