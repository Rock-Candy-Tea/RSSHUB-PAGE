<div class="content">
                                                                    
                                                        <ol> 
 <li>修改一言类型为诗词</li> 
 <li>添加文章默认封面图片</li> 
 <li>调整 web 端页面风格，参考：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.dancoder.cn%2F" target="_blank">蛋码农 - https://www.dancoder.cn</a></li> 
 <li>文章支持加密，加密后的文章需要验证密码后才能查看详情，默认只能看文章的介绍</li> 
 <li>修改 web 端分类菜单的 sql 查询条件，已禁用的分类不再显示</li> 
 <li>增加 admin 端系统配置页面部分配置项的说明，请按照说明进行配置</li> 
 <li>web 端所有图片容易死链的地方，添加了默认图片</li> 
 <li>留言板页面添加<strong>正能量</strong>提示</li> 
 <li>添加自定义“关于本站”、“留言板”、“友情链接”、“免责声明”、“Footer”、“鼠标点击时的气泡文字”、“热门搜索的待选项”等内容</li> 
 <li>去掉 web 端侧边栏中的 Gitee Fork Me 链接</li> 
 <li>使用 <a href="https://gitee.com/fujieid/jap" target="_blank">JustAuthPlus（JAP）</a> 替换 <a href="https://gitee.com/yadong.zhang/JustAuth" target="_blank">JustAuth</a></li> 
 <li>后台管理系统增加社会化登录的配置，可以在线添加、启用、禁用某个平台的第三方登录功能</li> 
 <li>调整部分依赖 JAR 的版本</li> 
 <li>其他 BUG 修复</li> 
</ol> 
<h2 style="text-align:start">升级指南</h2> 
<p style="text-align:start">执行 <code>docs/db/update.sql</code> 中的 SQL，然后 admin 端重新登录即可。</p>
