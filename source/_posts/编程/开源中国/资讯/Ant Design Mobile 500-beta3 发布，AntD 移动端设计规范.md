                                                                                            <p>Ant Design Mobile 5.0.0-beta.3 发布了。Ant Design Mobile 即 Ant Design 移动端设计规范，<code>antd-mobile</code> 是 Ant Design 的移动规范的 React 实现，服务于蚂蚁及口碑无线业务。</p> 
<p>更新内容如下：</p> 
<ul> 
 <li>特性 
  <ul> 
   <li>Swiper.Item 支持了 <code>onClick</code> 属性</li> 
   <li>Swiper 增加了 <code>--slide-width</code> 和 <code>--border-radius</code> CSS 变量</li> 
   <li>Swiper 增加了 <code>indicator</code> 和 <code>indicatorProps</code> 属性，用于自定义指示器</li> 
   <li>增加了 PageIndicator 组件</li> 
   <li>绝大部分的组件都支持了透传 <code>data-*</code> 和 <code>aria-*</code> 属性</li> 
   <li>Search 组件支持了 Ref</li> 
  </ul> </li> 
 <li>优化 
  <ul> 
   <li>Swiper 在拖拽时暂停自动播放</li> 
   <li>Input 和 TextArea 组件限制了属性的类型，只保留了一部分对原生属性的支持</li> 
  </ul> </li> 
</ul> 
<hr> 
<p>迁移建议</p> 
<ul> 
 <li>对比新的文档，检查项目中是否用到的 Input 和 TextArea 的属性是否都在支持范围内</li> 
</ul>
