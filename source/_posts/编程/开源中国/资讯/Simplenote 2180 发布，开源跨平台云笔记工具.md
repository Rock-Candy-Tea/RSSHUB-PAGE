 <p>Simplenote 是一个简单、轻量级的跨平台云笔记工具，可以用来保存文字、表格、代码等信息，用户记录的内容可以与所有设备保持同步。Simplenote 支持大多数主流的 WEB 浏览器，同时支持 iOS，Android，Mac，Windows 和 Linux 操作系统。</p> 
 <p>自 Simplenote 2.16.0 以来的更新内容包括：</p> 
 <h3><strong>改进</strong></h3> 
 <ul> 
  <li>增加了对处理因未验证的账户电子邮件而导致的登录失败的支持 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fautomattic%2Fsimplenote-electron%2Fpull%2F2976" target="_blank">#2976</a></li> 
  <li>增加了对处理已知数据泄露中发现的密码的支持 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fautomattic%2Fsimplenote-electron%2Fpull%2F2972" target="_blank">#2972</a></li> 
  <li>更新了 Simperium 库的依赖至最新版本 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fautomattic%2Fsimplenote-electron%2Fpull%2F2984" target="_blank">#2984</a></li> 
  <li>更新了 Note List、Note Cell 和 No Notes 组件，以使用 CSS 变量来显示颜色 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fautomattic%2Fsimplenote-electron%2Fpull%2F2969" target="_blank">#2969</a></li> 
  <li>更新了 Windows 的瓷贴图标，增加了新的尺寸和阴影 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fautomattic%2Fsimplenote-electron%2Fpull%2F2965" target="_blank">#2965</a></li> 
 </ul> 
 <p>更多详情可查看：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FAutomattic%2Fsimplenote-electron%2Freleases" target="_blank">https://github.com/Automattic/simplenote-electron/releases</a></p>
