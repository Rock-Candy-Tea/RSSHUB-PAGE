                                                                                            <p style="color:#333333; margin-left:0; margin-right:0; text-align:left">Seed7 version 2022-06-26 已发布，此版本值得关注的变化：</p> 
<ul style="margin-left:8px; margin-right:0"> 
 <li> <p style="margin-left:0; margin-right:0"><span><span>改进 <a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fseed7.sourceforge.net%2Ffaq.htm%23syntax_highlighting" target="_blank">Nano 和 vim 对 Seed7 语法高亮</a></span></span>的支持</p> </li> 
 <li> <p style="margin-left:0; margin-right:0"><span><span>改进 <a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fseed7.sourceforge.net%2Ffaq.htm" target="_blank">FAQ</a>、<a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fseed7.sourceforge.net%2Fmanual%2Findex.htm" target="_blank">手册</a>和 <a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fseed7.sourceforge.net%2Fbuild.htm" target="_blank">src/read_me.txt</a></span></span></p> </li> 
 <li> <p style="margin-left:0; margin-right:0"><span><span>在声明失败的情况下，改进<a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fseed7.sourceforge.net%2Ffaq.htm%23interpreter" target="_blank">解释器</a>(s7)，可以用正确的行编写错误消息</span></span></p> </li> 
</ul> 
<p>开发者可从 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FThomasMertes%2Fseed7" target="_blank">GitHub</a> 和 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fsourceforge.net%2Fprojects%2Fseed7%2Ffiles%2Flatest%2Fdownload" target="_blank">SF</a> 获取新版本，此外还有一个 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fsourceforge.net%2Fprojects%2Fseed7%2Ffiles%2Fbin%2Fseed7_05_20191117_win.exe%2Fdownload" target="_blank">Windows 的 Seed7 安装程序</a>。</p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.reddit.com%2Fr%2Fseed7%2Fcomments%2Fvl8a5f%2Fseed7_version_20220626_released_on_github_and_sf%2F" target="_blank">其他部分更新内容</a>：</p> 
<ul style="margin-left:8px; margin-right:0"> 
 <li> <p style="margin-left:0; margin-right:0">Syntax highlighting for the Nano editor has been added. Many thanks to Duke Normandin for creating<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FThomasMertes%2Fseed7%2Fblob%2Fmaster%2Fdoc%2Fseed7.nanorc" target="_blank">seed7.nanorc</a>.</p> </li> 
 <li> <p style="margin-left:0; margin-right:0">The syntax highlighting of vim (in<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FThomasMertes%2Fseed7%2Fblob%2Fmaster%2Fdoc%2Fseed7.vim" target="_blank">seed7.vim</a>) has been improved.</p> </li> 
 <li> <p style="margin-left:0; margin-right:0">The file<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FThomasMertes%2Fseed7%2Fblob%2Fmaster%2Fprg%2Fsyntaxhl.sd7" target="_blank">syntaxhl.sd7</a><span> </span>has been improved.</p> </li> 
 <li> <p style="margin-left:0; margin-right:0">Explanations about<span> </span><a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fseed7.sourceforge.net%2Ffaq.htm%23type_inference" target="_blank">type inference</a>,<span> </span><a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fseed7.sourceforge.net%2Ffaq.htm%23overload_result" target="_blank">return type overloading</a><span> </span>and<span> </span><a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fseed7.sourceforge.net%2Ffaq.htm%23integer_overflow" target="_blank">integer overflow</a><span> </span>have been improved in the<span> </span><a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fseed7.sourceforge.net%2Ffaq.htm" target="_blank">FAQ</a>.</p> </li> 
 <li> <p style="margin-left:0; margin-right:0">In the<span> </span><a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fseed7.sourceforge.net%2Fmanual%2Findex.htm" target="_blank">manual</a><span> </span>the tutorial chapter "<a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fseed7.sourceforge.net%2Fmanual%2Ftutorial.htm%23For_loop_and_containers" target="_blank">For loop and containers</a>" has been improved. The chapter about<span> </span><a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fseed7.sourceforge.net%2Fmanual%2Ferrors.htm" target="_blank">errors</a><span> </span>has been improved also.</p> </li> 
 <li> <p style="margin-left:0; margin-right:0">The documentation in<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FThomasMertes%2Fseed7%2Fblob%2Fmaster%2Fsrc%2Fread_me.txt" target="_blank">src/read_me.txt</a><span> </span>has been improved and the new file<span> </span><a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fseed7.sourceforge.net%2Fbuild.htm" target="_blank">src/read_me.htm</a><span> </span>has been introduced.</p> </li> 
</ul>
