                                                                                            <p>DBeaver 是一个免费开源的通用数据库工具，适用于开发人员和数据库管理员。</p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">DBeaver 22.2 现已发布，更新内容如下：</p> 
<ul style="margin-left:0; margin-right:1.5em"> 
 <li>Data editor： 
  <ul style="margin-left:0; margin-right:0"> 
   <li>添加了嵌入式浏览器（Edge vs IE）配置</li> 
   <li>改进了记录模式下的自定义数据类型呈现</li> 
   <li>控制台查看器切换已在 Linux/MACOS 上修复</li> 
   <li>改进了数组视图/编辑 UI</li> 
   <li>多维数组支持已修复</li> 
  </ul> </li> 
 <li>Data transfer： 
  <ul style="margin-left:0; margin-right:0"> 
   <li>修复了具有不同标识符情况的数据库的列自动映射</li> 
   <li>旧配置加载问题已修复</li> 
   <li>统计计算已修复</li> 
  </ul> </li> 
 <li>SQL editor： 
  <ul style="margin-left:0; margin-right:0"> 
   <li>查询解析器已修复（选择项中的单行注释问题）</li> 
   <li>按连接类型对屏幕着色的问题得到修复</li> 
  </ul> </li> 
 <li>SSH： 
  <ul style="margin-left:0; margin-right:0"> 
   <li>SSHJ 现在是默认的隧道实现</li> 
   <li>SSHJ 版本升级，增加 ASN1 解码器</li> 
  </ul> </li> 
 <li>Connection： 
  <ul style="margin-left:0; margin-right:0"> 
   <li>解决了 Web 浏览器中的双重身份验证问题（所有具有 MFA 身份验证的驱动程序）</li> 
  </ul> </li> 
 <li>Clickhouse： 
  <ul style="margin-left:0; margin-right:0"> 
   <li>添加了布尔数据类型支持</li> 
   <li>时间戳数据编辑问题已解决</li> 
  </ul> </li> 
 <li>Firebird：overloaded 程序加载问题已解决</li> 
 <li>MS Access：图表加载已修复</li> 
 <li>Oracle：添加了密码过期警告</li> 
 <li>PostgreSQL： 
  <ul style="margin-left:0; margin-right:0"> 
   <li>为解释计划对话框中增加了额外的选项</li> 
   <li>数据类型约束加载得到改进</li> 
   <li>数据类型 DDL 生成已修复</li> 
  </ul> </li> 
</ul>
