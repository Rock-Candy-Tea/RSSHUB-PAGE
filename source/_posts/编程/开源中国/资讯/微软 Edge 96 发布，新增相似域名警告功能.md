                                                                                            <p>近日微软发布了 Microsoft Edge 96，作为其常规四周更新周期的一部分。更新内容包括：</p> 
<h2>微软 Edge 96 中的新内容</h2> 
<ul> 
 <li>现在可以在 Edge 设置中找到 "效能模式"。它允许你通过减少浏览器对 CPU 和内存造成的负载来延长电池续航。<strong>（该功能在 Edge 95 中就有出现，但此前处于 Controlled Feature Rollout 阶段）</strong></li> 
 <li>如果你的密码被泄露了，在 Edge 的密码管理器中新增了一个「Go to Website」（跳转到网站）的按钮，可以快速修改密码。 
  <ul> 
   <li>Edge 将打开页面自动填写当前的密码，并且将生成一个新的密码。这个功能目前只适用于有限的几个网站。</li> 
  </ul> </li> 
</ul> 
<p>    <img alt height="392" src="https://static.oschina.net/uploads/space/2021/1121/082135_nxdH_4937141.png" width="700" referrerpolicy="no-referrer"></p> 
<ul> 
 <li>Edge 现在可以跟踪在线商店的产品价格，并在价格发生变化时提醒你。这意味着你不再需要手动检查你的购物清单上的每一件商品的折扣。 
  <ul> 
   <li>内置的价格比较和价格历史目前已支持在桌面端和 Android 端的 Microsoft Edge 上使用。</li> 
  </ul> </li> 
</ul> 
<p>    <img alt height="396" src="https://static.oschina.net/uploads/space/2021/1121/082202_ejK5_4937141.png" width="700" referrerpolicy="no-referrer"></p> 
<ul> 
 <li>Microsoft Rewards 菜单现在可以被钉在浏览器的工具栏上。微软必应已与 Good On You 合作。这个平台根据一系列可持续性和社会责任指标对时尚品牌进行评估。 
  <ul> 
   <li>使用 Edge 的英国用户可以看到这些分数，因此可以在微软必应上购物时过滤相应结果，帮助你放心地购物，使你做出符合你价值观的选择。</li> 
  </ul> </li> 
</ul> 
<p>    <img alt height="517" src="https://static.oschina.net/uploads/space/2021/1121/082218_Aubp_4937141.png" width="700" referrerpolicy="no-referrer"></p> 
<h3>新工具和改进</h3> 
<ul> 
 <li>微软 Edge 现在可以使用协议链接直接启动渐进式网络应用程序（PWA）。这为用户在使用浏览器和网络应用时提供了更好、更综合的体验，使它们更接近于原生应用。</li> 
 <li>改进了 PDF 文档中的滚动，使其更加顺畅，用户在滚动时将不再看到白色滚动条。</li> 
 <li><strong>对于某些网站，如果用户输入的 URL 与其他网站的地址非常相似，浏览器将显示警告，提醒用户注意所访问的网站可能是知名网站的虚假版本。</strong></li> 
 <li>对 IE 模式和 WSUS 也有一些改进。后者将允许重新分配 WebView2 组件的独立更新。</li> 
 <li>Edge 更新后，版本号为 96.0.1054.29</li> 
</ul>
