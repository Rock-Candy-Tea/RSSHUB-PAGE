                                                                                            <p>Subtitle Edit 是一款针对文本格式类字幕的编辑工具，支持多种字符编码和字幕格式（Sub Station Alpha、Advanced Sub Station Alpha、SubRip、MicroDVD、MPL2、MPsub (MPlayer subtitle)、SubViewer 2.0、Plain-Text、Adobe Encore DVD）。</p> 
<p>目前，Subtitle Edit 3.6.6 已发布，该版本更新内容如下：</p> 
<p><strong>New</strong></p> 
<ul> 
 <li>为 iTT 属性添加工具栏图标</li> 
 <li>添加新的子格式</li> 
 <li>在 UI 中关闭“word split list”的选项</li> 
 <li>自定义文本格式的自定义文件 ext</li> 
 <li>添加语音识别的快捷方式</li> 
 <li>添加 CPS 计算规则“No space/punctuation”</li> 
 <li>添加一些对阅读 Manzanita dvb 格式的支持</li> 
 <li>添加批量音频到文本（语音识别）</li> 
</ul> 
<p><strong>Improved ​​​​</strong></p> 
<ul> 
 <li>更新保加利亚语翻译</li> 
 <li>更新匈牙利语翻译</li> 
 <li>更新中文翻译</li> 
 <li>更新葡萄牙语翻译</li> 
 <li>更新法语翻译</li> 
 <li>更新韩语翻译</li> 
 <li>更新意大利语翻译</li> 
 <li>将 dotnet 框架更新到 4.8</li> 
 <li>将 Tesseract OCR 更新到 5.1.0</li> 
 <li>更新 Vosk 至 0.3.38</li> 
 <li>将 yt-dlp.exe 更新为 2022.05.18（不再重命名为 youtube-dl.exe）</li> 
 <li>添加一些新的英文名称</li> 
 <li>如果用户 OCR 修复替换列表无效，则添加错误消息</li> 
 <li>改进老化设置</li> 
 <li>添加 AMD 支持 (hevc_amf + h264_amf) 进行 burn-in</li> 
 <li>支持导出自定义文本中的大括号</li> 
 <li>BD edit - 始终打开 bdsup raw（不合并）</li> 
 <li>对 .itt 文件使用大写的“UTF-8”</li> 
 <li>BD sup 编辑器中的单独垂直边距</li> 
 <li>添加新的音频到文本模型</li> 
 <li>改进 ASSA 对“Remove Leading '...'”的支持</li> 
 <li>在颜色选择器中为最近的颜色添加四种额外颜色</li> 
 <li>在拼写检查中添加一点 vobsub 支持</li> 
 <li>为在线视频的音频到文本添加 error msg</li> 
 <li>翻译后尽量保留 assa 标签</li> 
 <li>记住最近文件的在线视频（自动重新加载）</li> 
 <li>允许在“cmd line convert”中使用大型 .mcc 文件</li> 
 <li>改进 File - Statistics 中的“gap info”</li> 
 <li>允许导入带空字幕的时间码</li> 
 <li>改进 jp/cn 的音频到文本后处理</li> 
 <li>添加更多 GT 语言</li> 
 <li>视频轨迹栏鼠标滚轮用作 waveform</li> 
 <li>允许在“Netflix quality check”中对问题进行分类</li> 
 <li>添加四个额外的自定义颜色快捷方式</li> 
</ul> 
<p><strong>Fixed </strong></p> 
<ul> 
 <li>修复加载新子程序后选定的行更新</li> 
 <li>删除了 vertical-scroll-bar-auto-show-hide，因为它可能会冻结 SE</li> 
 <li>删除错误的 OCR 替换规则</li> 
 <li>安装程序文件关联的小修复</li> 
 <li>修复 auto-backup origianl</li> 
 <li>使“Ignore formatting”在比较文本框着色中起作用</li> 
 <li>修复打开带有视频偏移的文件时的错误</li> 
 <li>不要在颜色选择器中重复颜色</li> 
 <li>修复 ebu stl headerfile 的 cmd 行转换问题</li> 
 <li>修复最后两行的图像导出丢失错误</li> 
 <li>修复“Batch convert”“Merge short lines”UI 设置</li> 
 <li>修复删除 HI 文本中的大小写问题</li> 
 <li>修复在 SMPTE 模式下转到 next/prev/bookmark/error/empty-line 的问题</li> 
 <li>修复转到上一个字幕（不播放）</li> 
 <li>修复 Netflix ttml 和 bold/color tags</li> 
</ul>
