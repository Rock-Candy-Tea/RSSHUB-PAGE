                                                                                            <p>Manjaro 21.0 "Ornara" 已正式发布。</p> 
<p><img height="281" src="https://static.oschina.net/uploads/space/2021/0324/184739_w0ve_2720166.jpeg" width="500" referrerpolicy="no-referrer"></p> 
<p>Manjaro Linux 是快速、用户友好、面向桌面、基于 Arch Linux 的发行版，其部分显著特性包括：一份直观的安装程序、自动硬件检测、稳定的滚动发布模式、对安装多个内核的支持、用于管理显卡的特别 bash 脚本以及高度的桌面可配置性。</p> 
<p><img height="313" src="https://static.oschina.net/uploads/space/2021/0322/073548_pTHQ_2720166.png" width="500" referrerpolicy="no-referrer"></p> 
<p>Manjaro 21.0 已将内核升级至 Linux Kernel 5.10 LTS，并使用了对应的最新驱动。得益于 Manjaro 支持多内核的特性，它通过应用 Linux Kernel 5.4 LTS 内核的最小化 ISO 为旧硬件提供了额外的支持。</p> 
<p>21.0 版本还改进了安装程序 Calamares，Calamares 现在拥有对每个国家/地区的“最佳猜测”语言表，当启用 GeoIP 后，它将自动选择该国家/地区的语言作为默认语言。当然，用户可以选择其他语言。</p> 
<p>其他的变化主要是所使用的 GNOME、KDE 和 Xfce 三个桌面环境升级而带来的更新。</p> 
<h2>GNOME</h2> 
<p>GNOME 已升级至 3.38 版本，支持重新排列应用程序并将它们组织到自定义文件夹中。GNOME Shell 可以用不同的刷新率來驱动多台显示器，以确保获得最佳的桌面体验。此外，GNOME Shell 中的屏幕录制基础架构也得到了改进——通过利用 PipeWire 和内核 API 来减少资源消耗并提升响应速度。</p> 
<h2>KDE</h2> 
<p>KDE Plasma 升级到了最新稳定版<a href="https://www.oschina.net/news/127709/kde-plasma-5-21-first-look"> 5.21</a>，带来了全新应用启动容器、全新的系统资源监控应用，改进键盘、鼠标以及触摸功能，并努力为 Wayland 提供一流的支持。</p> 
<h2>Xfce</h2> 
<p>Xfce 升级至 4.16 版本，窗口管理器在合成和 GLX 领域得到了升级和改进。其他改进包括面板改进、显示对话框改进和设置管理器改进等。</p>
