                                                        <p><strong>Redis Desktop Manager 项目在今年二月更名为 RESP.app</strong></p> 
<p>RESP.app 是基于 Qt 5 的 Redis 开源跨平台桌面管理器。RESP.app 2022.4.2 发布，更新内容如下：</p> 
<h3>新功能和改进</h3> 
<ul> 
 <li>增加了 Extension Server 支持，弃用了 Native Formatters</li> 
 <li>改进了 Server Actions 选项卡，有 <code>MONITOR</code> 命令的快捷方式</li> 
</ul> 
<p><img alt="https://user-images.githubusercontent.com/1655867/181498271-cdf39533-92e9-45f9-aed4-38d01a7ceafc.png" src="https://user-images.githubusercontent.com/1655867/181498271-cdf39533-92e9-45f9-aed4-38d01a7ceafc.png" referrerpolicy="no-referrer"></p> 
<ul> 
 <li>增加设置，以控制 Value 编辑器中的 <code>SCAN</code> 限制和每页项目</li> 
</ul> 
<h3>修复</h3> 
<ul> 
 <li>修复导入/导出 Connections 在 Linux 上不工作的问题</li> 
 <li>修复显示服务器信息时无法向下滚动的问题</li> 
 <li>修复 Windows 上的黑暗模式问题</li> 
 <li>修复 macOS 下 TreeView 中的快捷方式</li> 
</ul>
