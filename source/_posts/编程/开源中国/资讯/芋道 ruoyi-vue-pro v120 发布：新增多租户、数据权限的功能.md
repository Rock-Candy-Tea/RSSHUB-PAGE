<h1 style="margin-left:0px; margin-right:0px; text-align:left"><span style="color:#e74c3c">项目介绍</span></h1> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><strong style="color:#40485b">芋道</strong><span style="background-color:#ffffff; color:#40485b">，一套</span><strong style="color:#40485b">全部开源</strong><span style="background-color:#ffffff; color:#40485b">的</span><strong style="color:#40485b">企业级</strong><span style="background-color:#ffffff; color:#40485b">的快速开发平台，毫无保留给个人及企业免费使用。</span></p> 
<ul style="list-style-type:disc; margin-left:0; margin-right:0"> 
 <li><code>yudao-admin-server</code> + <code>yudao-admin-ui</code>面向 B 端的管理后台。</li> 
 <li><code>yudao-user-server</code> + <code>yudao-user-ui</code>面向 C 端的用户前台（微信小程序 + H5 页面）。</li> 
</ul> 
<h1 style="margin-left:0px; margin-right:0px; text-align:left"><span style="color:#e74c3c"><span style="background-color:#ffffff">更新说明</span></span></h1> 
<h3 style="margin-left:0em; margin-right:0em; text-align:start">⚠️ Warning</h3> 
<p style="color:#40485b; margin-left:0; margin-right:0; text-align:start">这个版本新增了多租户与数据权限两个重量级的功能，建议花点时间进行了解与学习。</p> 
<h3 style="margin-left:0; margin-right:0; text-align:start">⭐ New Features</h3> 
<ul> 
 <li> <p style="margin-left:0; margin-right:0">【新增】多租户，支持 Web、Security、Job、MQ、Async、DB、Redis 组件</p> </li> 
 <li> <p style="margin-left:0; margin-right:0">【新增】数据权限，内置基于部门过滤的规则</p> </li> 
 <li> <p style="margin-left:0; margin-right:0">【新增】用户前台的昵称、头像的修改</p> </li> 
 <li> <p style="margin-left:0; margin-right:0">【新增】用户前台的微信公众号、微信小程序的社交登陆的 API 接口</p> 
  <blockquote> 
   <p style="margin-left:0; margin-right:0">完整功能，需要等基于 Uniapp 实现的用户前台一起~ 努力 coding 中，胖友可以 star 持续关注一波！</p> 
  </blockquote> </li> 
 <li> <p style="margin-left:0; margin-right:0">【优化】管理后台的登陆成功后，LoginUser 使用统一方法补全信息</p> </li> 
</ul> 
<h3 style="margin-left:0; margin-right:0; text-align:start">🐞 Bug Fixes</h3> 
<ul> 
 <li>【修复】通知和字典查询接口的 @PreAuthorize 权限标识错误</li> 
 <li>【修复】代码生成的 Java 类路径缺少 modules 目录</li> 
 <li>【修复】代码生成的 Test 单元测试类的引入 Util 工具类的包路径不正确</li> 
</ul> 
<h3 style="margin-left:0; margin-right:0; text-align:start">🔨 Dependency Upgrades</h3> 
<ul> 
 <li>【引入】mockito-inline 3.6.28：Mockito 提供对 final、static 的支持</li> 
</ul>
