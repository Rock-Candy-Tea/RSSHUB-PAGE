                                                                                            <p>Jpom v2.8.10 已经发布，Java 项目在线管理</p> 
<p>此版本更新内容包括：</p> 
<h3>新增功能</h3> 
<ol> 
 <li>nginx 删除操作新增还原配置</li> 
 <li>【server】新增用户登录日志记录</li> 
 <li>【server】log view 新增正则参考</li> 
 <li>【server】docker 控制台新增 docker 信息查看</li> 
 <li>【server】新增 docker swarm 集群管理</li> 
 <li>【server】构建新增是否保存产物配置</li> 
 <li>【server】构建 Dockerfile 镜像 tag 支持仓库目录 <code>.env</code> 环境变量文件</li> 
 <li>【server】docker 控制台新增 docker 网络信息查看</li> 
 <li>【server】构建 Dockerfile 镜像支持发布 swarm service</li> 
</ol> 
<h3>解决BUG、优化功能</h3> 
<ol> 
 <li>【server】 ssh 授权目录不能为根目录</li> 
 <li>【server】 在线构建-构建列表中，选择构建命令时新增多环境下打包的命令</li> 
 <li>【server】 在线构建-构建列表中，选择构建命令时提供不同的跳过执行测试类的方式</li> 
 <li>【server】 编辑用户工作空间权限判断没有选择父级的空权限</li> 
 <li>【server】 修改后分发后 <strong>并发重启</strong> 描述改为 <strong>并发执行</strong></li> 
 <li>【server】 自动构建触无法执行（感谢@<a href="https://gitee.com/zhangfuhua" target="_blank">zhangfuhua</a> ）</li> 
</ol>
