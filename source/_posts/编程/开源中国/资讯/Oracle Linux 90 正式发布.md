                                                                                            <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">面向 Intel-64、AMD-64 (x86_64) 和 Arm (aarch64) 的 Oracle Linux 9 已正式发布，作为 Oracle 的 RHEL 9 的分支。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">Oracle Linux 9.0 在镜像中使用的内核版本包括 Unbreakable Enterprise Kernel Release 7 (UEK R7) 以及 Red Hat Compatible Kernel (RHCK)。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">带有 UEK R7 的 Oracle Linux 9 提供了内核、性能和安全性增强功能，包括：  </p> 
<ul style="margin-left:0; margin-right:0"> 
 <li><strong>UEK R7<span> </span></strong>基于 Linux 5.15 LTS 版本，提供最新的硬件支持，与 Linux 内核社区保持一致，更及时地交付关键修复和安全更新，UEK R7 也<strong>可用于 Oracle Linux 8</strong>。</li> 
 <li><strong>Oracle Linux 9 中的OpenSSL 3.0<span> </span></strong>包括许多新概念和结构增强功能。新 FIPS 模块可防止使用非 FIPS 算法，同时可以在内核中设置 FIPS 标志，而无需将 OpenSSL 切换到 FIPS 模式。</li> 
 <li><strong>UEK R7 的文件系统增强功能：</strong>包括 Btrfs、新的异步 SSD 修整、并行文件系统同步功能和更少的校验和树查找。XFS 支持直接访问 (DAX) 操作，允许直接访问字节可寻址的持久内存，有助于避免使用传统块 I/O 约定的延迟。NFS 引入了“eager write”挂载选项来帮助减少延迟。</li> 
 <li>UEK R7 原生支持<span> </span><strong>VirtualBox 共享文件夹(vboxsf)。</strong>有助于使用 Oracle VM VirtualBox 在 Oracle Linux 来宾和主机操作系统之间共享文件夹。</li> 
</ul>
