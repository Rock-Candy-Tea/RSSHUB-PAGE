                                                        <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">WebStorm 2021.3 正式发布，该版本是今年的最后一次重大更新。这次更新带来了开发者期待已久的增强功能，包括支持远程开发、改进的 HTML 补全和 Deno 集成。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">新的功能和改进包括：</p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>JavaScript 和 TypeScript：改进了对 monorepos 的支持、加快了 JavaScript 文件的索引时间、快速修复了下载远程 ES6 模块的问题、支持 ES2022 private class members。</li> 
 <li>框架和技术：更容易管理项目的依赖性、重新设计的 Deno 插件、支持 Angular 13、更好的 HTML 补全。</li> 
 <li>远程开发：用于远程开发的 JetBrains 网关、Node.js 远程解释器更新，以及 SSH 连接的代理支持。</li> 
 <li>Docker：改进的连接过程、新的 Clean up 操作、Minikube 集成、支持 Compose V2。</li> 
 <li>用户体验：能够拆分运行工具窗口、新的书签工具窗口，在 Search Everywhere 中基于 ML 的搜索操作。</li> 
 <li>版本控制：重新组织了 VCS 设置、改进了推送提交的工作流程、新的更改工具窗口。</li> 
</ul>
