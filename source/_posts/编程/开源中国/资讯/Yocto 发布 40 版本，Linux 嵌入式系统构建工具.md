                                                                                            <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">基于 Linux 基金会的 Yocto 项目<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Flore.kernel.org%2Fall%2FDM5PR1101MB236176E376BD554365050E72B9FA9%40DM5PR1101MB2361.namprd11.prod.outlook.com%2F" target="_blank">发布了</a><span> </span>4.0 版本。Yocto 是一个开源的协作软件，适合嵌入式 Linux 开发人员使用。Yocto 提供模板、工具和方法帮助开发者创建基于 Linux 的定制版物联网/嵌入式操作系统，而无需关心硬件体系。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">Yocto 4.0 提供了在构建中使用 Linux 5.15 LTS 内核以及更新的工具链组件（如 GNU C Library 2.35）的支持。此外还有大约 300 种软件/工具得到了版本升级。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">该版本的压缩套件从 Gzip 切换到 Zstd ，以提供更好的压缩/解压缩性能，同时还有各种 ARM 和 RISC-V 的特定改进，以及各种其他增强，比如对一些变量名称的包容性语言调整。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">有关该版本的更多内容可查看<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdownloads.yoctoproject.org%2Freleases%2Fyocto%2Fyocto-4.0%2FRELEASENOTES" target="_blank">更新公告</a>。</p>
