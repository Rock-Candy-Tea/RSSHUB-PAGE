                                                        <p>pgAdmin 是 PostgreSQL 领先的开源图形化管理工具。pgAdmin 4 旨在满足新手和有经验的 Postgres 用户的需求，提供强大的图形界面，简化了数据库对象的创建、维护和使用。</p> 
<p>pgAdmin 4 v6.1 版本正式发布。本次版本更新中值得注意的变化包括：</p> 
<h3>功能</h3> 
<ul> 
 <li>增加了对浏览器树中缩进指南的支持；</li> 
 <li>增加了对高级表字段的支持，如外键、ERD 工具中的主键；</li> 
 <li>增加了允许将表拖到 ERD 工具中的支持；</li> 
 <li>在 ERD 工具中生成 SQL 时增加了索引创建；</li> 
 <li>增加了对通过 webserver（REMOTE_USER）认证的支持；</li> 
 <li>增加了对启用/禁用规则的支持；</li> 
</ul> 
<h3>Bugs/Housekeeping</h3> 
<ul> 
 <li>修正了 OAuth2 集成的重定向问题；</li> 
 <li>确保查询工具中的查询高亮颜色应该不那么密集；</li> 
 <li>修正了浏览器树在改变分辨率时不显示所有内容的问题；</li> 
 <li>确保在浏览器树中，列应按创建顺序而不是按字母顺序显示；</li> 
 <li>修正了浏览器树中的背景颜色问题；</li> 
 <li>在 ERD 工具中增加了对复合外键的支持；</li> 
 <li>修正了用户无法加载 HTTP 反向代理后面的数据库的问题；</li> 
 <li>修正了每次点击刷新集合节点，对象的数量减少几十或更多的问题；</li> 
 <li>修正了浏览器树形排序顺序回归问题；</li> 
 <li>修正了一个问题，在 SQLite 数据库的服务器表中，空白字符串被存储，而不是 NULL；</li> 
</ul>
