                                                                                            <p>Sailfish OS 发布了今年的第三个版本，版本号为 4.2.0，代号为 Verla。<br> <img alt height="329" src="https://static.oschina.net/uploads/space/2021/0923/070121_dG7m_4937141.png" width="700" referrerpolicy="no-referrer"></p> 
<h3>版本更新亮点：</h3> 
<ul> 
 <li>账户 
  <ul> 
   <li>修复了与创建/删除基于 Telepathy 的账户有关的错误</li> 
   <li>由于配置缺陷，与 Office 365 Exchange 服务器的连接失败得到修复</li> 
   <li>Facebook 从可用的账户类型中删除</li> 
  </ul> </li> 
 <li>氛围 
  <ul> 
   <li>UI 组件得到了修复，以遵循新的氛围颜色</li> 
  </ul> </li> 
 <li>Android App Support 
  <ul> 
   <li>修复了 "设置>系统> Android App Support>显示许可证"</li> 
   <li>相机与 Android App Support 更优雅地衔接了</li> 
   <li>应用设置页面现在显示应用图标</li> 
   <li>如果 App Support 没有运行，会显示一个关于启动 App Support 以启动该应用的通知，该通知现在会显示应用程序名称和图标</li> 
   <li>改进了 Android 应用通过 WLAN 的互联网连接</li> 
   <li>更新至 10.0.0.54 安全补丁</li> 
  </ul> </li> 
 <li>备份和恢复 
  <ul> 
   <li>从云端账户恢复备份后，会通知用户重新登录</li> 
  </ul> </li> 
 <li>浏览器 
  <ul> 
   <li>关闭按钮从工具栏中移除</li> 
   <li>为混合网络应用开发者添加了 OAuth2 helper API</li> 
   <li>改进了设置和工具栏</li> 
   <li>现在可以选择下载目录</li> 
   <li>增加了历史项目</li> 
   <li>增加了检查网页的用户名/密码的功能</li> 
   <li>点击下载的文件现在可以打开查看器应用，而不是跳转到传输的地方</li> 
  </ul> </li> 
 <li>相机 
  <ul> 
   <li>支持 Xperia 10 II 上的多个摄像头</li> 
   <li>第三方的相机 API 错误修复</li> 
  </ul> </li> 
 <li>连接性 
  <ul> 
   <li>在 Xperia 10 II 上修复了 5GHz WiFi 接入点的信号强度</li> 
   <li>当加入一个新的 WiFi 网络时，IPv6 地址现在被分配到接口</li> 
   <li>尽管采用了双 Wi-Fi/STA+AP 模式，但确保互联网共享在 Xperia 10 II 上的正常使用</li> 
  </ul> </li> 
 <li>……</li> 
</ul>
