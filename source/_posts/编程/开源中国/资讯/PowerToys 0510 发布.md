                                                        <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">Microsoft PowerToys 是 Windows 系统实用程序，供高级用户调整和简化其 Windows 体验，可最大限度地提高生产力。0.51 版本的更新内容如下：</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><strong>需要注意的事项</strong></p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>转移了本地化的内部服务，并正在努力将自动集成重新加入。</li> 
</ul> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><strong>PowerToys Awake</strong></p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>系统托盘和设置使用相同的语言来打开东西。</li> 
</ul> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><strong>Color Picker</strong></p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>增加了新的格式，可以将颜色复制为浮点值或十进制值。</li> 
 <li>调整颜色窗口现在接受小写的 HEX 代码。</li> 
</ul> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><strong>FancyZones</strong></p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>新的窗口切换功能，现在用户可以为一个区域指定多个窗口，并在它们之间使用默认的 Win + PgDn/PgUp 命令在它们之间循环。</li> 
 <li>为区域添加了采用系统强调色和主题的功能。</li> 
 <li>在设置菜单中增加了区域外观的视觉预览。</li> 
 <li>修正了 FancyZones 启动时崩溃的问题。</li> 
</ul> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><strong>Image Resizer</strong></p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>修正了调整图片大小时产生空文件夹的问题。</li> 
 <li>增加了删除非必要的元数据的选项，有助于大大减少文件的大小。</li> 
 <li>修正了 Image Resizer 接收到意外的属性类型或值所引起的错误。</li> 
</ul> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><strong>Mouse utilities</strong></p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>查找我的鼠标（Find My Mouse）：改进了当用户双击时间配置设置为 100ms 以上时的激活功能。</li> 
 <li>查找我的鼠标：修正了在所有虚拟桌面上的显示，而不是只在创建它的虚拟桌面上显示。</li> 
 <li>查找我的鼠标：新的设置，根据你的反馈实现了更多的定制。</li> 
 <li>新的鼠标高亮器：启用后，默认情况下用 Win + Shift + H 激活鼠标高亮，当鼠标左键或右键被点击时也同样在显示屏上显示视觉提示。</li> 
</ul> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><img alt height="187" src="https://static.oschina.net/uploads/space/2021/1202/070939_kjUE_5430600.gif" width="600" referrerpolicy="no-referrer"></p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><strong>PowerToys Run</strong></p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>为设置插件添加了新条目。</li> 
 <li>增加了对应用 URI 处理的支持，如<span> </span><code>mailto:</code><span> </span>和<span> </span><code>ms-settings:</code></li> 
 <li>在 VSCode Workspaces 插件的搜索结果中添加了 DevContainer 工作空间。</li> 
 <li>修复了崩溃的问题。</li> 
</ul> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><strong>Development relevant</strong></p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>主要开发分支现在被命名为<span> </span><code>Main</code>。</li> 
 <li>调整 PT Run 的插件文件夹结构。</li> 
 <li>正在努力将我们的发布管道转移到 Windows Terminal 使用的同一系统上。</li> 
 <li>改进了 PT Run 中环境变量的使用/更新过程。</li> 
 <li>更新 .NET 至 3.1.20。</li> 
 <li>在 BugReportTool 中集中了进程列表。</li> 
 <li>MSI 和文件资源管理器附加组件的注册表处理改进。</li> 
</ul>
