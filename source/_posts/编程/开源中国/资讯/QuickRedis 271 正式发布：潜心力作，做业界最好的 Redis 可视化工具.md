                                                                                            <p style="color:#40485b; margin-left:0; margin-right:0; text-align:left">QuickRedis 2.7.1 正式发布，增强功能和修复一些问题：</p> 
<ol> 
 <li>支持显示编辑修改二进制数据；</li> 
 <li>key列表可以支持排序；</li> 
 <li>key的ttl支持设置为-1；</li> 
 <li>优化获取keys列表。</li> 
</ol> 
<p style="color:#40485b; margin-left:0; margin-right:0; text-align:left">QuickRedis 是一款<span> </span><strong>永久免费</strong><span> </span>的 Redis 可视化管理工具。它支持<strong>直连、哨兵、集群</strong>模式，支持亿万数量级的 key，还有令人兴奋的 UI。QuickRedis 支持 Windows 、 Mac OS X 和 Linux 下运行。</p> 
<p style="color:#40485b; margin-left:0; margin-right:0; text-align:left">QuickRedis 是一个效率工具，<strong>当别人在努力敲命令的时候，而你已经在喝茶。</strong></p>
