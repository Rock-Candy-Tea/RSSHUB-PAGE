                                                        <p>Ant Design 4.15.2 发布了。Ant Design 是一套企业级的 UI 设计语言和 React 实现，使用 TypeScript 构建，提供完整的类型定义文件，自带提炼自企业级中后台产品的交互语言和视觉风格、开箱即用的高质量 React 组件与全链路开发和设计工具体系。</p> 
<p>此版本更新内容如下：</p> 
<ul> 
 <li>修复 Tabs <code>centered</code> 失效问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F30106" target="_blank">#30106</a></li> 
 <li>修复 Badge <code>status="warning"</code> 时切换样式异常的问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F30090" target="_blank">#30090</a></li> 
 <li>修复 Button 为 <code>type="link"</code> 时禁用手型丢失的问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F30197" target="_blank">#30197</a></li> 
 <li>修复 TextArea <code>onChange</code> 事件返回 <code>target</code> 对象不是 HTMLTextArea 对象的问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F30124" target="_blank">#30124</a></li> 
 <li>Upload <code>fileList</code> 中的文件将总是有 uid。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F30087" target="_blank">#30087</a></li> 
 <li>补充遗漏的 TimePicker 丹麦语国际化。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F30128" target="_blank">#30128</a></li> 
 <li>增加 Image 组件 <code>preview</code> 繁體文案。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F30100" target="_blank">#30100</a></li> 
 <li>TypeScript 
  <ul> 
   <li>修改 LIST_IGNORE 字段为 string 类型。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F30188" target="_blank">#30188</a></li> 
  </ul> </li> 
</ul>
