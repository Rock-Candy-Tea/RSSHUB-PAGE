                                                        <p>ownCloud 2.9.2 现已发布，该版本更新内容如下：</p> 
<p><strong>Fixed</strong></p> 
<ul> 
 <li><span style="color:#000000">长文件路径的问题</span></li> 
 <li><span style="color:#000000">在 .owncloudsync.log 中添加请求时间和其他缺失的数据</span></li> 
 <li><span style="color:#000000"><span style="background-color:#ffffff">在用户注销时不要求在启动时提供凭据</span></span></li> 
 <li><span style="color:#000000"><span style="background-color:#ffffff">修复私有复制 url 位置链接</span></span></li> 
 <li><span style="color:#000000"><span style="background-color:#ffffff">立即将变化传播到 ignore list 中</span></span></li> 
 <li><span style="color:#000000">修复桌面文件中的图标名称与 ownBrander 主题</span></li> 
 <li><span style="color:#000000">处理锁定文件时发生崩溃</span></li> 
 <li><span style="color:#000000">对包含 :?*"<>| 的文件显示正确的错误信息</span></li> 
 <li><span style="color:#000000">当 unsyncedfolders 文件不能被读取时不要同步</span></li> 
</ul> 
<p><strong>Change</strong></p> 
<ul> 
 <li>使 sharedialog 预览更有弹性</li> 
</ul> 
<p><strong>Enhancement</strong></p> 
<ul> 
 <li><span style="background-color:#ffffff; color:#505a67">允许删除损坏的同步文件夹</span></li> 
</ul>
