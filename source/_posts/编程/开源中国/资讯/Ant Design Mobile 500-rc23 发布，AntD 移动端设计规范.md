                                                                                            <p>Ant Design Mobile 5.0.0-rc.23 发布了。Ant Design Mobile 即 Ant Design 移动端设计规范，<code>antd-mobile</code> 是 Ant Design 的移动规范的 React 实现，服务于蚂蚁及口碑无线业务。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><strong>更新内容如下：</strong></p> 
<ul> 
 <li>特性 
  <ul> 
   <li>Badge 增加了<span> </span><code>bordered</code><span> </span>属性，默认是关闭的，你可以通过它为 Badge 增加描边</li> 
   <li>ImageViewer.Multi 支持了通过 ref 进行控制<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design-mobile%2Fpull%2F4742" target="_blank">#4742</a></li> 
  </ul> </li> 
 <li>优化 
  <ul> 
   <li>调整了 Checkbox 和 Radio 的图标样式</li> 
   <li>优化了 SearchBar 的 placeholder 文字颜色</li> 
   <li>优化了 Switch、Modal 的样式</li> 
   <li>完善了 PasscodeInput 文档和类型定义中对 CSS 变量的支持<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design-mobile%2Fpull%2F4736" target="_blank">#4736</a></li> 
  </ul> </li> 
 <li>修复 
  <ul> 
   <li>修复了 Picker 系列组件在弹出时，可能在 iOS 设备下出现文字颜色闪烁的问题</li> 
  </ul> </li> 
</ul> 
<p>更新说明：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design-mobile%2Freleases%2Ftag%2Fv5.0.0-rc.23" target="_blank">https://github.com/ant-design/ant-design-mobile/releases/tag/v5.0.0-rc.23</a></p>
