                                                                                            <p>FileZilla 是一个免费和开源的跨平台 FTP 应用程序，由 FileZilla Client 和 FileZilla Server 组成。客户端可用于 Windows、Linux 和 macOS，服务器仅可用于 Windows。服务器和客户端都支持 FTP 和 FTPS（FTP over SSL/TLS），而客户端还可以连接到 SFTP 服务器。</p> 
<p>FileZilla Client 3.61.0 正式发布，更新内容包括：</p> 
<h3>新功能</h3> 
<ul> 
 <li>使用来自 libfilezilla 的改进的异步文件读/写器</li> 
</ul> 
<h3>错误修正和小改动</h3> 
<ul> 
 <li>更新 libfilezilla 至 0.39.1</li> 
 <li>修复了 Site Manager 中根项目的导出</li> 
 <li>处理更新器中不存在的下载目录的罕见情况</li> 
</ul> 
<p>更多详情可查看：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Ffilezilla-project.org%2Findex.php" target="_blank">https://filezilla-project.org/index.php</a></p>
