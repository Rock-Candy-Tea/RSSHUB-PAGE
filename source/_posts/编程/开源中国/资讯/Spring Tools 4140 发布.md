                                                        <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">Spring Tools 4.14.0 现已发布。Spring Tools 4 是由 Spring 团队打造的 Spring 开发工具，从零开始构建，融合了现代技术和开发者工具架构。它在单独的进程中运行，从构建之初就考虑到了性能问题，并且支持最新的 Spring 技术，为开发基于 Spring 的企业应用提供世界级支持。同时，全新版本的 Spring Tools 与 IDE 无关，可在各种编码环境中使用，支持 Eclipse、Visual Studio Code 与 Theia。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><strong>主要更新内容</strong></p> 
<ul> 
 <li>(VSCode) fixed：VSCode Spring boot tools 1.30.0 在试图查找 JVM 时出错 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fsts4%2Fissues%2F726" target="_blank">#726</a> )</li> 
 <li>(VSCode) fixed：vscode“Problems”诊断条目缺少“source”字段（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fsts4%2Fissues%2F725" target="_blank">#725</a>）</li> 
 <li>(VSCode) fixed：orphan vscode 扩展进程仍在运行 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fsts4%2Fissues%2F704" target="_blank">#704</a> )</li> 
 <li>(VSCode) fixed：语言服务器进程中的死锁，避免进程被关闭 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fsts4%2Fissues%2F741" target="_blank">#741</a> )</li> 
 <li>(VSCode) fixed：Outline View 在 Windows 上再次损坏 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fsts4%2Fissues%2F742" target="_blank">#742</a> )</li> 
 <li>(VSCode) fixed：boot-java.live-information.automatic-tracking.on 不适用于 vscode ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fsts4%2Fissues%2F733" target="_blank">#733</a> )</li> 
 <li>(VSCode) fixed：Java 17 被错误识别为 Java 8 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fsts4%2Fissues%2F713" target="_blank">#713</a> )</li> 
 <li>(Eclipse)fixed：Eclipse 4.23 中 Spring LSP 与 STS 4.23 nightlies 的死锁 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fsts4%2Fissues%2F729" target="_blank">#729</a> )</li> 
</ul> 
<p><span style="color:#333333">Spring Tools 4.14.1 计划于 2022 年 4 月下旬发布。</span></p>
