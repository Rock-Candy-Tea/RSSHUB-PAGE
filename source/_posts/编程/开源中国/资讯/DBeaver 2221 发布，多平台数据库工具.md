                                                        <p>DBeaver 22.2.1 已发布。</p> 
<p>DBeaver 是免费的多平台数据库工具，适用于开发人员、数据库管理员、分析师和所有需要使用数据库的人，并且支持所有流行的数据库：MySQL、PostgreSQL、SQLite、Oracle、DB2、SQL Server、Sybase、MS Access、Teradata、Firebird、Apache Hive、Phoenix、Presto 等。</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-38c99e9030ab8f3b1f73848b29cd8184697.png" referrerpolicy="no-referrer"></p> 
<p>更新内容：</p> 
<ul> 
 <li>数据编辑器 
  <ul> 
   <li>修复 URL 转换问题（编辑模式下）</li> 
   <li>修复分组面板行为</li> 
   <li>BLOB/CLOB 查看器显示智能警告</li> 
   <li>空间查看器中的 NullPointer 错误已修复</li> 
  </ul> </li> 
 <li>SQL 编辑器：“Open in new tab” 行为已修复</li> 
 <li>连接： 
  <ul> 
   <li>PostgreSQL、SQL Server 和通用驱动程序添加了基于 URL 的配置支持</li> 
   <li>SSH 用户支持可选</li> 
   <li>SSH：修复凭据提示问题</li> 
  </ul> </li> 
 <li>数据传输：改进了列自动映射（不区分大小写的搜索）</li> 
 <li>图表：修复关联创建</li> 
 <li>元数据编辑器：schema 刷新性能显著提升</li> 
 <li>Eclipse 插件：错误日志中许多与 DBeaver 相关的警告已解决</li> 
 <li>i18n: 
  <ul> 
   <li>改进中文、德语本文化</li> 
  </ul> </li> 
 <li>Clickhouse: 
  <ul> 
   <li>添加了 SSL 连接配置支持</li> 
   <li>添加了 UUID 数据类型支持</li> 
  </ul> </li> 
 <li>Greenplum：<span style="background-color:#ffffff; color:#555555">正确的客户端应用程序名称现在发送到服务器</span></li> 
 <li>MariaDB：修复数据传输问题</li> 
 <li>PostgreSQL：改进对 box 数据类型的支持</li> 
 <li>SQL Server：<span style="background-color:#ffffff; color:#555555">修复了序列值选择问题</span></li> 
 <li>Vertica：<span style="background-color:#ffffff; color:#555555">默认驱动程序升级到版本 12</span></li> 
 <li>UI 错误修复</li> 
</ul>
