                                                        <p>uView 2.0 2.0.10 已经发布，多平台快速开发 UI 框架。</p> 
<p>此版本更新内容包括：</p> 
<h3>修复</h3> 
<ul> 
 <li>修复button sendMessagePath属性不生效</li> 
 <li>修复DatetimePicker选择器title无效</li> 
 <li>修复u-toast设置loading=true不生效</li> 
 <li>修复u-text金额模式传0报错</li> 
 <li>修复u-toast组件的icon属性配置不生效</li> 
</ul> 
<h3>优化</h3> 
<ul> 
 <li>button 的icon在特殊场景下的颜色优化</li> 
 <li>Picker选择器 singleIndex 文档修正</li> 
 <li>IndexList优化，增加#</li> 
</ul> 
<p><a href="https://gitee.com/link?target=https%3A%2F%2Fuviewui.com%2Fcomponents%2Finstall.html" target="_blank">安装方式</a></p> 
<p>更多内容/文档详见 <a href="https://gitee.com/link?target=https%3A%2F%2Fuviewui.com%2F" target="_blank">uView UI 官网</a></p> 
<p>详情查看：<a href="https://gitee.com/umicro/uView2.0/releases/2.0.10">https://gitee.com/umicro/uView2.0/releases/2.0.10</a></p>
