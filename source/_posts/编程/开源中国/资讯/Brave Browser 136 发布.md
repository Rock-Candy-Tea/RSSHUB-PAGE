                                                                                            <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">Brave 是基于 Chromium 的一款免费开源网络浏览器，主打隐私保护，其默认设置就会自动阻止在线广告和网站跟踪。Brave Software（Brave 浏览器所属公司）于 2015 年由 Brendan Eich 和 Brian Bondy 联合创立，其中前者是 JavaScript 的创造者，同时也是 Mozilla 联合创始人。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">Brave 1.36 发布，更新内容如下：</p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>将 Chromium 升级到 99.0.4844.51</li> 
 <li>为 Brave News 增加了自定义 RSS 支持</li> 
 <li>增加了 Brave Talk 小部件作为新标签页下小部件栈的默认选项</li> 
 <li>为 Brave Wallet 增加了 signTypedData 支持</li> 
 <li>当 Brave Wallet 被重置时增加了确认模式</li> 
 <li>增加了为 Brave Wallet 交易设置自定义 nonce 值的功能</li> 
 <li>为 Brave Wallet 的添加网络面板添加了 "精度的小数"</li> 
 <li>在 Brave Wallet 的 "选择账户" 组件下，在当前选择的账户旁边添加了复选标记图标</li> 
 <li>在重新加载页面时添加了 window.ethereum 对象</li> 
 <li>启用 Shields 时增加了 WebSockets 连接池限制</li> 
 <li>更新了关于 Widevine 提示的描述</li> 
 <li>更新了 P3A pings 以使用 JSON 格式</li> 
 <li>更新了 "Open using IPFS" 按钮文本为 "IPFS"</li> 
 <li>更新了子资源重定向在 IPFS 页面上的限制</li> 
 <li>更新了 brave://settings/privacy 下的 "自动发送诊断报告" 的设置名称和描述，以提高清晰度</li> 
 <li>更新了 brave://settings/appearance 下的 "隐藏 Brave Rewards 按钮" 设置文本为 "在地址栏显示 Brave Rewards 图标"</li> 
 <li>在 Windows 上禁用了后台组件更新器</li> 
 <li>修复了 macOS 上的 PWA 崩溃问题</li> 
 <li>修正了在某些情况下，点击 "解决" 自适应验证码不工作的问题</li> 
 <li>修复 "IPFS" 按钮在某些情况下从 URL 栏丢失的问题</li> 
</ul> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">更多详情可查看：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fbrave.com%2Flatest%2F" target="_blank">https://brave.com/latest/</a></p>
