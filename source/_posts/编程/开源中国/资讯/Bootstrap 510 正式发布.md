                                                        <p>Bootstrap 5.1.0 正式发布，该版本更新内容如下：</p> 
<h3>亮点：</h3> 
<ul> 
 <li>增加了实验性的、可选择的 CSS 网格支持；</li> 
 <li>在导航栏中增加了对响应式 offcanvas 组件的支持；</li> 
 <li>为 skeleton 加载屏幕添加了新的占位符组件；</li> 
 <li>在折叠组件中增加了对水平折叠的支持；</li> 
 <li>增加了新的堆叠和垂直规则帮助器；</li> 
 <li>为主体样式、颜色、RGB 颜色等增加了大量新的 CSS 变量；</li> 
 <li>更新了 <code>.bg-*</code> 和 <code>.text-*</code> 实用程序，以使用 CSS 变量和新的 RGB CSS 变量进行实时自定义；</li> 
 <li>增加了四个新的组件实例，包括下拉菜单、页脚、列表组和模态；</li> 
 <li>更新了模态和 offcanvas 的背景，使其对每个组件都是独一无二的；</li> 
 <li>暂时恢复了 <code>.col-*</code> 类覆盖 <code>.row-col-*</code> 类的功能，后续还将修复一些关键的错误；</li> 
</ul>
