                                                        <p>pgAdmin 是 PostgreSQL 领先的开源图形化管理工具。pgAdmin 4 旨在满足新手和有经验的 Postgres 用户的需求，提供强大的图形界面，简化了数据库对象的创建、维护和使用。</p> 
<p>这个版本的 pgAdmin 4 包括错误修复和新功能，主要更新内容如下：</p> 
<h3>新特性</h3> 
<ul> 
 <li>增加了在查询工具中按下键触发自动完成的选项</li> 
 <li>允许用户从存储管理器中删除文件/文件夹</li> 
 <li>允许用户在文件/存储管理器中搜索</li> 
 <li>在查询工具中增加了对使用堆叠线图、条形图和堆叠条形图的可视化支持</li> 
 <li>在查询工具中增加了对使用饼状图的可视化图表的支持</li> 
</ul> 
<h3>Housekeeping</h3> 
<ul> 
 <li>将文件/存储管理器移植到 React</li> 
 <li>将更改密码对话框移植到React</li> 
 <li>将主密码对话框移植到 React</li> 
 <li>删除动态模块加载，改为静态加载</li> 
 <li>将名为还原点的对话框移植到 React</li> 
</ul> 
<h3>错误修复</h3> 
<ul> 
 <li>保留用户在导入/导出数据对话框中的设置</li> 
 <li>修正了索引上的注释不显示的问题</li> 
 <li>确保通知不应该从消息标签中消失</li> 
 <li>执行完成后启用开始调试按钮</li> 
 <li>确保面板关闭后不调用仪表盘 graph API</li> 
 <li>确保所有选中的单元格都能显示几何图形</li> 
 <li>修正了 JSON 编辑器中隐藏第一条记录的问题</li> 
 <li>增加了对服务器模式下 Azure PostgreSQL 部署的支持</li> 
</ul>
