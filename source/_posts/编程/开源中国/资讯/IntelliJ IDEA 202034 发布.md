                                                        <p>IntelliJ IDEA 2020.3.4 现已发布，这主要是一个错误修复版本。</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-2dd5eb7e34f2a05c6d8b704fa777cc4caf0.png" referrerpolicy="no-referrer"></p> 
<p><strong>主要更新内容</strong></p> 
<ul> 
 <li>IDE 现在会加载正确版本的 Maven 依赖项</li> 
 <li>自动完成速度恢复正常</li> 
 <li>修复了有关非法反射访问操作的警告</li> 
 <li>修复了使用 JAXB 从 XML 模式生成 Java 代码时的死锁</li> 
 <li>修复了在静态块中使用日志时会发生的 Lombok 错误</li> 
 <li>解决了 Maven 依赖项导入的问题</li> 
 <li>支持新的 Dart 语法：通用类型别名</li> 
 <li>Dart 文件移动：相对导入成为绝对导入（仅适用于 Windows）</li> 
 <li>修复了无法删除模块的问题</li> 
</ul>
