                                                        <p>Git Extensions 是一个独立的 UI 工具，用于管理 git 存储库。它还与 Windows Explorer 和 Microsoft Visual Studio 集成。Git Extensions 3.5 正式发布，该版本更新内容如下：</p> 
<p>变化：</p> 
<ul> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fgitextensions%2Fgitextensions%2Fpull%2F9103" target="_blank">9103</a> GitHub Token：添加一个链接，让用户生成其个人 Access Token；</li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fgitextensions%2Fgitextensions%2Fpull%2F9102" target="_blank">9102</a> 恢复分割器的持久性；</li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fgitextensions%2Fgitextensions%2Fpull%2F9094" target="_blank">9094</a> 将更多的异常处理作为失败的外部操作；</li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fgitextensions%2Fgitextensions%2Fpull%2F9083" target="_blank">9083</a> 在一个单独的进程中报告bug；</li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fgitextensions%2Fgitextensions%2Fpull%2F9018" target="_blank">9018</a> 检查剪贴板上的源 URL 时，忽略 git clone 前缀；</li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fgitextensions%2Fgitextensions%2Fpull%2F8971" target="_blank">8971</a> 支持名称中带有点的Bitbucket仓库；</li> 
</ul>
