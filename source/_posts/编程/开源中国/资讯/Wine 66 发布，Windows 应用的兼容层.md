                                                                                            <p>Wine（Wine Is Not an Emulator）是一个能够在多种兼容 POSIX 接口的操作系统（诸如 Linux、macOS 与 BSD 等）上运行 Windows 应用的兼容层。它不是像虚拟机或者模拟器一样模仿内部的 Windows 逻辑，而是将 Windows API 调用翻译成为动态的 POSIX 调用，免除了性能和其它一些行为的内存占用，让你能够干净地整合 Windows 应用到桌面。</p> 
<p>Wine 6.6 已经正式发布，该版本的新内容包括：</p> 
<ul> 
 <li>Mono 引擎更新至 6.1.1 版本，并进行了上游更新；</li> 
 <li>DWrite 和 DnsApi 库转换为 PE；</li> 
 <li>改进了即插即用驱动的支持；</li> 
 <li>各种 bug 修复； 
  <ul> 
   <li>Google Drive 登录前崩溃的问题；</li> 
   <li>半衰期软件的渲染错误；</li> 
   <li>Wine 的任务栏未在 KDE 5 的任务栏中显示的问题；</li> 
   <li>Microsoft Flight Simulator 崩溃的问题；</li> 
   <li>Wine 控制面板无法启动小程序（applets）；</li> 
   <li>......</li> 
  </ul> </li> 
</ul>
