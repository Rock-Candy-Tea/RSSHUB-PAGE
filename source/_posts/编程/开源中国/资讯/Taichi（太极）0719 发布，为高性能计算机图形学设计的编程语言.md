                                                        <p>Taichi（太极）0.7.19 已经发布，这是专为高性能计算机图形学设计的编程语言。</p> 
<p><img src="https://static.oschina.net/uploads/space/2021/0316/070528_Jp1b_2720166.gif" referrerpolicy="no-referrer"></p> 
<p>此版本具体更新内容如下：</p> 
<ul> 
 <li>[IR] 实验性 real function 支持（第 1 阶段）（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ftaichi-dev%2Ftaichi%2Fpull%2F2306" target="_blank">＃2306</a>）</li> 
 <li>[Opt] 简化 multiplying/dividing POT（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ftaichi-dev%2Ftaichi%2Fpull%2F2332" target="_blank">＃2332</a>）</li> 
 <li>[Lang] 为 2x2 矩阵添加 ti.sym_eig（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ftaichi-dev%2Ftaichi%2Fpull%2F2328" target="_blank">＃2328</a>）</li> 
 <li>[perf] Loop-invariant code motion（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ftaichi-dev%2Ftaichi%2Fpull%2F2323" target="_blank">＃2323</a>）</li> 
 <li>[bug] 修复 test_eig.py 中的潜在 bug（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ftaichi-dev%2Ftaichi%2Fpull%2F2329" target="_blank">＃2329</a>）</li> 
 <li>[example] 彩色三角形光栅器（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ftaichi-dev%2Ftaichi%2Fpull%2F2315" target="_blank">＃2315</a>）</li> 
 <li>[doc] Windows 使用 backslash 作为文件路径（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ftaichi-dev%2Ftaichi%2Fpull%2F2319" target="_blank">＃2319</a>）</li> 
 <li>[gui] 将 DPI 感知上下文设置为每台显示器感知，以便在没有系统缩放的情况下创建像素完美的窗口（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ftaichi-dev%2Ftaichi%2Fpull%2F2320" target="_blank">＃2320</a>）</li> 
 <li>[doc] 为避免混淆而进行 wrap（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ftaichi-dev%2Ftaichi%2Fpull%2F2322" target="_blank">＃2322</a>）</li> 
 <li>[ir] 使 BLSAnalyzer 可测试（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ftaichi-dev%2Ftaichi%2Fpull%2F2294" target="_blank">＃2294</a>）</li> 
 <li>[lang] 添加 rescale_index() 并测试（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ftaichi-dev%2Ftaichi%2Fpull%2F2313" target="_blank">＃2313</a>）</li> 
 <li>[bugfix]  防护带有空格的路径。（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ftaichi-dev%2Ftaichi%2Fpull%2F2318" target="_blank">＃2318</a>）</li> 
 <li>[LLVM] 添加随机 seed 支持（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ftaichi-dev%2Ftaichi%2Fpull%2F2297" target="_blank">＃2297</a>）</li> 
 <li>[Lang] 为 2x2 矩阵添加 ti.eig（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ftaichi-dev%2Ftaichi%2Fpull%2F2303" target="_blank">＃2303</a>）</li> 
 <li>支持 diffRange 乘法（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ftaichi-dev%2Ftaichi%2Fpull%2F2310" target="_blank">＃2310</a>）</li> 
</ul>
