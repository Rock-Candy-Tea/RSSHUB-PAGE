                                                                                            <p>egui 是一个简单、快速、高度可移植的即时模式 GUI 库。egui 可以在 Web 上运行，也可以在你喜欢的游戏引擎中运行。egui 的目标是成为最容易使用的 Rust GUI 库，以及用 Rust 制作 Web 应用的最简单方法。</p> 
<h3>egui 更新亮点</h3> 
<ul> 
 <li>使用任何大小的字体</li> 
 <li>定义自定义文本样式</li> 
 <li>添加 <code>Ui::text_style_height</code></li> 
 <li>添加 <code>TextStyle::resolve</code></li> 
 <li>添加 <code>Context::move_to_top</code>和 <code>Context::top_most_layer</code>用于管理顶部的图层</li> 
 <li>将<code>CtxRef</code> 重命名为 <code>Context</code></li> 
 <li>使用新 crate egui_extras 轻松加载图片</li> 
 <li>将 <code>Style::body_text_style</code> 替换为更通用的 <code>Style::text_styles</code></li> 
</ul> 
<h3>eframe 更新亮点：</h3> 
<ul> 
 <li>glow 现在是本地和 Web 上的默认渲染器</li> 
 <li>遵循操作系统的明/暗模式偏好</li> 
</ul>
