                                                        <p>Ant Design Mobile 5.0.0-rc.5 发布了。Ant Design Mobile 即 Ant Design 移动端设计规范，<code>antd-mobile</code> 是 Ant Design 的移动规范的 React 实现，服务于蚂蚁及口碑无线业务。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><strong>更新内容如下：</strong></p> 
<ul> 
 <li>特性 
  <ul> 
   <li>Picker 系列组件为选中项元素增加了<span> </span><code>data-selected</code><span> </span>属性，可以用来做样式的自定义<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design-mobile%2Fpull%2F4547" target="_blank">#4547</a></li> 
   <li>Collapse 的<span> </span><code>arrow</code><span> </span>属性支持了通过 render prop 来完全自定义动画<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design-mobile%2Fpull%2F4552" target="_blank">#4552</a></li> 
   <li>Toast 增加了<span> </span><code>config</code><span> </span>方法，可以用来做全局统一配置<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design-mobile%2Fpull%2F4553" target="_blank">#4553</a></li> 
   <li>新增了试验性的 PasscodeInput 组件<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design-mobile%2Fpull%2F4548" target="_blank">#4548</a></li> 
  </ul> </li> 
 <li>优化 
  <ul> 
   <li>优化了 List 在点击时内部边框的视觉效果</li> 
   <li>优化了 Picker 组件对屏幕阅读器的支持</li> 
  </ul> </li> 
 <li>修复 
  <ul> 
   <li>修复了 Picker 在 iOS 设备下会出现弹出时遮罩层闪烁的问题</li> 
   <li>修复了 Dialog 的<span> </span><code>getContainer</code><span> </span>属性效果不正确的问题</li> 
   <li>修复了 NumberKeyboard 小数点按钮背景颜色不正确的问题<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design-mobile%2Fpull%2F4556" target="_blank">#4556</a></li> 
  </ul> </li> 
</ul>
