  <p>Visual Studio Code 1.67.2 更新已发布，该小版本修复了以下问题：</p> 
  <ul> 
   <li>部分终端启动配置参数被双重转义 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmicrosoft%2Fvscode%2Fissues%2F149283" target="_blank"><span style="color:var(--color-fg-muted)!important">#149283</span></a></li> 
   <li>build[macos]: 上传用于 Bing 设置搜索的配置时超时 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmicrosoft%2Fvscode%2Fissues%2F149089" target="_blank">#149089</a> </li> 
   <li>1.67 版本中，禁用的工具栏图标不会变暗 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmicrosoft%2Fvscode%2Fissues%2F149075" target="_blank"><span style="color:var(--color-fg-muted)!important">#149075</span></a><span style="color:var(--color-fg-muted)!important"> </span></li> 
   <li>启动快捷方式的选择指示器不会自动清理 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmicrosoft%2Fvscode%2Fissues%2F148893" target="_blank">#148893</a></li> 
   <li>1.67 版本中，调试器的 STDIN/STDOUT 重定向不起作用 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmicrosoft%2Fvscode%2Fissues%2F148887" target="_blank">#148887</a></li> 
   <li>2022 年 4 月 - Recovery 2 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmicrosoft%2Fvscode%2Fissues%2F148935" target="_blank">#148935</a></li> 
  </ul>
