<p><span>大家好，禅道17.2版本发布！新版本更新了邮箱重置密码、文档高级搜索、测试类型自定义等功能，为您打造多场景、全方位的管理方式。本次还调整了地盘、执行模块的界面细节，丰富了看板自定义设置，力求带来更佳的使用体验。</span></p>

<h1><span>一、本次新增功能点</span></h1> 
<h3> 体验：</h3> 
<ul> 
 <li>多选下拉框支持全选和取消全选。</li> 
 <li>列表勾选后增加选中数量统计。</li> 
 <li>按钮禁用状态原因提示。 </li> 
</ul> 
<h3> 测试：</h3> 
<ul> 
 <li>测试单中测试类型字段支持自定义。</li> 
 <li>支持用例导入用例库。</li> 
 <li>批量导入用例时性能优化。 </li> 
</ul> 
<h3> 项目集：</h3> 
<ul> 
 <li>项目集的项目列表中增加状态检索标签。 </li> 
</ul> 
<h3> 文档：</h3> 
<ul> 
 <li>文档增加搜索、导出、排序功能。</li> 
</ul> 
<h3> 看板：</h3> 
<ul> 
 <li>丰富看板自定义设置。 </li> 
</ul> 
<h3> 后台：</h3> 
<ul> 
 <li>增加邮箱重置密码功能。</li> 
 <li>项目管理员权限分组增加对不同项目集、产品、执行的维护权限设置。</li> 
</ul>
