                                                        <p>Kodi 是由 XBMC 基金会开发的开源媒体播放器，原名 XBMC（最后一个以 XBMC 命名的版本是 13.2 Gotham，14.0 Helix 是第一个以 Kodi 命名的版本），Kodi 可以运行在多种操作系统和硬件平台。 它可以让用户播放本地或网络存储设备中的大多数视频、音乐、播客及各种常见数位媒体文件。</p> 
<p>该软件最初是为计划运行在 Xbox 上的，名称也由此而来。随后有了 Android、Linux、BSD、macOS、iOS 和 Windows 操作系统的原生版。</p> 
<p>Kodi 19.1 正式发布，以下是本次更新值得关注的地方：</p> 
<h3>视频：</h3> 
<p>修正：HDR 元数据现在可以在 VP9 配置文件 2 streams 中检测到，并且可以在支持 HDR 穿透或音调映射的平台上播放此类视频时使用；</p> 
<h3>光盘：</h3> 
<ul> 
 <li>修正：在 Linux 中播放 DVD 的问题；</li> 
 <li>修正：通过遥控器/键盘的 BD-J 蓝光跳章问题；</li> 
</ul> 
<h3><strong>PVR</strong></h3> 
<ul> 
 <li>修正：当使用非常基本的遥控器来控制 Kodi 时，在 PVR Guide window 中无法访问上下文菜单;</li> 
 <li>修正：客户端的频道名称和编号在改变时不能持续存在的问题；</li> 
 <li>修正：Kodi 重启后，录音的播放次数和恢复位置丢失；</li> 
 <li>修正：当使用 MySQL 作为 EPG 数据库时，浏览 EPG 时崩溃；</li> 
 <li>修正：在 Estuary Timer/Timer 规则窗口中，next recording on 标签时间未被定位；</li> 
 <li>修正：频道管理器不能重命名后端频道；</li> 
 <li>修正：打开 Guide window 时不能选择播放存档的节目；</li> 
 <li>修正：GUI 在删除/插入/隐藏/取消频道组时不更新；</li> 
 <li>更新：改善 Estuary 中 PVR 窗口的外观；</li> 
</ul> 
<h3>音乐库：</h3> 
<ul> 
 <li>修正：提示单中的音乐问题，在重新扫描时只有第一首曲目被添加到库中，其余的都被删除了；</li> 
</ul> 
<h3>字幕：</h3> 
<ul> 
 <li>修正：检测 ASS 字幕的系统字体；</li> 
 <li>修正：检测 ASS 字幕的用户字体；</li> 
 <li>修正：在 Wayland 上呈现半透明的 ASS 字幕；</li> 
</ul> 
<h3>GUI：</h3> 
<ul> 
 <li>修正：DVD/蓝光的媒体标记；</li> 
</ul> 
<h3>网页界面：</h3> 
<ul> 
 <li>更新：基于社区贡献的 Chorus2；</li> 
</ul> 
<h3>文件系统：</h3> 
<ul> 
 <li>更新：默认启用网络文件系统的文件缓存；</li> 
 <li>更新：改进文件缓存的错误处理</li> 
</ul> 
<h3>网络：</h3> 
<ul> 
 <li>更新：改善了 HTTP 和 NFS 网络文件系统的可靠性；</li> 
</ul>
