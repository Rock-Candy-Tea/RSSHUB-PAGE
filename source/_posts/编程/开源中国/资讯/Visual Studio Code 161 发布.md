<div class="content">
                                                                    
                                                        <p><span style="background-color:#ffffff; color:#333333">Visual Studio Code 1.61 现已发布，其中一些主要</span><strong>亮点内容如下：</strong></p> 
<ul> 
 <li><strong><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fcode.visualstudio.com%2Fupdates%2Fv1_61%23_split-an-editor-without-creating-a-new-group" target="_blank">在同一组内拆分编辑器</a>：</strong><span style="background-color:#ffffff; color:#444444">为同一文件快速创建 side-by-side 的编辑器。</span>添加了一个新命令 <strong>Split in Group</strong> ( <span><span><span><span style="color:#0072be"><span style="background-color:#f4f2f9"><span><span><span><span><span><span><span><span><span><span><span><span><span>Ctrl+K Ctrl+Shift+\</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span> ) 以将一个编辑器分成两部分，而无需第二个编辑器组。这允许你在同一个文件上并排使用两个编辑器。你还可以通过"切换布局"按钮在垂直或水平拆分之间切换布局：</li> 
</ul> 
<p> <img height="345" src="https://oscimg.oschina.net/oscnet/up-1bdf67071bcfe596d7a1741e873dfbc1e4f.png" width="500" referrerpolicy="no-referrer"></p> 
<ul> 
 <li><strong><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fcode.visualstudio.com%2Fupdates%2Fv1_61%23_locked-editor-groups" target="_blank">锁定编辑器组</a></strong>：选择编辑器类型以自动锁定编辑器组。该功能在新版本中的设置体验得到了改进，允许你轻松选择一个编辑器，在打开时自动锁定一个组。</li> 
</ul> 
<p><img alt height="250" src="https://oscimg.oschina.net/oscnet/up-ff8b376394eab6fe5e9299f2eb769e21a2e.png" width="500" referrerpolicy="no-referrer"></p> 
<ul> 
 <li><strong><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fcode.visualstudio.com%2Fupdates%2Fv1_61%23_indicate-deleted-and-readonly-editors-via-decorations" target="_blank">更好地显示已删除和只读文件</a></strong>：Tab decorations 突出显示已删除和只读文件。</li> 
</ul> 
<p><img alt height="130" src="https://oscimg.oschina.net/oscnet/up-a59ca4822483d1287f72e412cee8bbcb30d.png" width="500" referrerpolicy="no-referrer"></p> 
<ul> 
 <li><strong><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fcode.visualstudio.com%2Fupdates%2Fv1_61%23_bracket-pair-guides" target="_blank">Bracket pair guides</a>：</strong>显示用于链接匹配 bracket pairs 的 vertical guides。</li> 
</ul> 
<p><img alt height="300" src="https://oscimg.oschina.net/oscnet/up-8224cd7279eefdec730b54584debd863ce0.png" width="206" referrerpolicy="no-referrer"></p> 
<ul> 
 <li><strong><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fcode.visualstudio.com%2Fupdates%2Fv1_61%23_fixed-dimensions" target="_blank">修复的</a> </strong><strong><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fcode.visualstudio.com%2Fupdates%2Fv1_61%23_fixed-dimensions" target="_blank">terminal dimensions</a>：</strong>现在可以通过<strong> </strong><strong>Terminal: Set Fixed Dimensions </strong><span style="color:#444444">命令 ( </span><code>workbench.action.terminal.setDimensions</code><span style="color:#444444">)设置 terminal dimensions。这对于换行可能产生干扰或破坏​​性的情况很有用。</span>该命令将显示一个提示，你可以在其中输入首选宽度和/或高度。</li> 
</ul> 
<p><img alt height="74" src="https://oscimg.oschina.net/oscnet/up-b0c4466b4dafc82281fb782807e7f4886c1.png" width="500" referrerpolicy="no-referrer"></p> 
<ul> 
 <li><strong><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fcode.visualstudio.com%2Fupdates%2Fv1_61%23_jupyter" target="_blank">Jupyter Notebook 改进</a></strong>：目录视图，全面的调试支持。</li> 
</ul>
