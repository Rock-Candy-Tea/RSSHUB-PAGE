                                                                                            <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">GnuCash 4.9 现已发布，这是稳定版 4.x 系列的第十个版本。GnuCash 是一个适用于个人或小型企业的财务软件，采用 GPL 协议开源，支持 GNU / Linux、BSD、Solaris、Mac OS X 及 Microsoft Windows。GnuCash 易于使用，且功能强大，它可以追踪银行帐户、股票、收入与支出，也可以基于专业的会计原则确保帐目平衡并提供准确的报表。 </p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><strong>在 4.8 和 4.9 之间，完成了以下错误修复：</strong></p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>Bug 797502 - (RTL) - 下拉时报告的右边缘被锁定</li> 
 <li>Bug 797618 - Shares 和 Stock 的西班牙语翻译</li> 
 <li>Bug 797678 - OFX importer 应该补充而不是取代现有的注释（以及描述和备忘录）字段。</li> 
 <li>Bug 797772 - 允许对账报告按过账和对账日期进行双重过滤。</li> 
 <li>Bug 798222 - 账户搜索对话框：按回车键没有任何作用</li> 
 <li>Bug 798266 - 类别的下拉选取器菜单没有出现</li> 
 <li>Bug 798313 - "File/Import/Import Transactions From CSV "使 Gnucash 崩溃</li> 
 <li>Bug 798320 - 关闭应用程序时，错误信息表明 GNUcash 崩溃 - MacOS 10.15.7 (19H1419)</li> 
 <li>Bug 798325 - 委内瑞拉的新货币(VED)</li> 
 <li>Bug 798327 - 对于多账户的 OFX，如果一个账户没有新的交易，OFX import 会过早停止。</li> 
 <li>Bug 798335 - 不再可能改变交易账户</li> 
 <li>Bug 798341 - 希伯来语中的右对齐 - 帐户屏幕</li> 
 <li>Bug 798346 - 运行检查和修复时崩溃</li> 
 <li>Bug 798352 - 输入共同基金交易时的小数点精度</li> 
 <li>Bug 798382 - 评论中的错别字更正</li> 
</ul> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><strong>以下修复和改进与错误报告无关：</strong><span style="color:#333333"> </span></p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>更多的内存泄漏堵塞和 GList 效率改进</li> 
 <li>修复使用 3.18 版以上的 cmake 进行的构建</li> 
 <li>[account-piecharts] 不使用 gnc:make-internal-option</li> 
 <li>对实验性的 IFRS 成本基础报告做了一些改进。</li> 
 <li>修复在文档链接对话框中显示链接的两个小问题。</li> 
 <li>修复翻译中指向 bugzilla.gnome.org 的过时链接</li> 
 <li>......</li> 
</ul>
