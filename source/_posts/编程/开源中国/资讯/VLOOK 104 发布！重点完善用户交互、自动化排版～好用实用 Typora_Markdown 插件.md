                                                        <p>上月由于配合 Typora V0.10.x 的重大升级，同步发布了 VLOOK 10.3 版本。</p> 
<p>由于时间稍有仓促，在用户体验和自动化排版的细节还有些可打磨提升的空间，特别是【导航中心】，以及「演示辅助」实用吸睛的【聚光灯】【表格十字光标】，在这次 V10.4 中进行了更深入的优化，并修复了一些问题。</p> 
<p><strong>优化了针对 Typora 的导出模板的兼容性</strong></p> 
<p><strong>【导航中心】</strong></p> 
<ul> 
 <li>（+新增）导航中心的各类索引支持即输即过滤</li> 
 <li>（+新增）过滤有结果时可按回车进行逐个结果跳转</li> 
</ul> 
<p><img alt src="https://z3.ax1x.com/2021/05/14/grfvNj.png" referrerpolicy="no-referrer"></p> 
<p><strong>【演示辅助】</strong></p> 
<ul> 
 <li>（^完善）让聚光灯提供更真实的光影效果</li> 
</ul> 
<p><img alt src="https://z3.ax1x.com/2021/05/14/grfqu8.png" referrerpolicy="no-referrer"></p> 
<p>————————————</p> 
<p>完整的更新日志如下：</p> 
<p><strong>Typora 集成</strong></p> 
<ul> 
 <li>（^完善）提高对Typora导出机制的兼容性</li> 
</ul> 
<p><strong>导航中心</strong></p> 
<ul> 
 <li>（+新增）导航中心的各类索引支持即输即过滤</li> 
 <li>（+新增）过滤有结果时可按回车进行逐个结果跳转</li> 
 <li>（^完善）重构分类索引底层，以及用户界面与交互</li> 
 <li>（*修复）被隐藏的索引对象无法跳转的问题</li> 
</ul> 
<p><strong>表格、图片、多媒体、公式</strong></p> 
<ul> 
 <li>（^完善）在不同表格间显示十字光标的动画效果</li> 
 <li>（^完善）公式的样式及应用示例（后续版本规划进行重点升级）</li> 
 <li>（~调整）音频计数器区分标准模式、mini模式</li> 
 <li>（*修复）Typora 编辑模式下图片加载失败后的样式不兼容多媒体链接</li> 
 <li>（*修复）图片与文本段落间没有空行隔间会导致不能生成题注的问题</li> 
</ul> 
<p><strong>演示辅助</strong></p> 
<ul> 
 <li>（^完善）让聚光灯提供更真实的光影效果</li> 
 <li>（*修复）激光笔的部分样式失效问题</li> 
 <li>（*修复）聚光灯与段落导航操作提示冲突问题</li> 
</ul> 
<p><strong>其他</strong></p> 
<ul> 
 <li>（^完善）彩虹标签内排版样式</li> 
 <li>（^完善）遮罩增加关闭提示器，优化体验更好的样式</li> 
 <li>（*修复）解决Windows环境下滚动条样式冲突问题</li> 
</ul>
