                                                        <p><span style="color:#333333">Taro 3.4.12 现已发布。Taro 是一个开放式跨端跨框架解决方案，支持使用 React/Vue/Nerv 等框架来开发微信 / 京东 / 百度 / 支付宝 / 字节跳动 / QQ 小程序 / H5 等应用。具体更新内容如下：</span></p> 
<h2><strong>修复</strong></h2> 
<h3><strong>小程序</strong></h3> 
<ul> 
 <li>补充 <code>Input</code> 组件的 <code>auto-fill</code> 属性，<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FNervJS%2Ftaro%2Fissues%2F11766" target="_blank">#11766</a></li> 
 <li>兼容百度小程序的 <code>usingSwanComponents</code> 配置，<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FNervJS%2Ftaro%2Fissues%2F11794" target="_blank">#11794</a></li> 
 <li>优化智能提取分包依赖插件</li> 
 <li>修复 <code>Taro.request</code> API 未正确返回 <code>onHeadersRecieved</code> 等方法的问题，<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FNervJS%2Ftaro%2Fissues%2F11904" target="_blank">#11904</a></li> 
 <li>修复字节小程序独立分包报错的问题，<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FNervJS%2Ftaro%2Fissues%2F11915" target="_blank">#11915</a></li> 
</ul> 
<h3><strong>H5</strong></h3> 
<ul> 
 <li>修复 <code>Swiper</code> 组件随着 tab 页面切换，总是跳到最后一个元素开始轮播的问题，<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FNervJS%2Ftaro%2Fissues%2F11920" target="_blank">#11920</a>，by <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmoyafaxi" target="_blank"><strong>@moyafaxi</strong></a></li> 
</ul> 
<h3><strong>RN</strong></h3> 
<ul> 
 <li>修复 <code>!import</code> 样式失效，<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FNervJS%2Ftaro%2Fissues%2F11728" target="_blank">#11728</a></li> 
 <li>修复初始化模板后，运行白屏并报错的问题，<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FNervJS%2Ftaro%2Fissues%2F11760" target="_blank">#11760</a></li> 
 <li>修复 <code>border-color</code> 设置 <code>rgb(a)</code> 报错的问题，<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FNervJS%2Ftaro%2Fissues%2F11792" target="_blank">#11792</a></li> 
</ul> 
<p>更新公告：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FNervJS%2Ftaro%2Freleases%2Ftag%2Fv3.4.12" target="_blank">https://github.com/NervJS/taro/releases/tag/v3.4.12</a></p>
