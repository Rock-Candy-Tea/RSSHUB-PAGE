                                                        <p><span style="color:#000000">Spring Tools 4.15.0 现已发布。Spring Tools 4 是由 Spring 团队打造的 Spring 开发工具，从零开始构建，融合了现代技术和开发者工具架构。它在单独的进程中运行，从构建之初就考虑到了性能问题，并且支持最新的 Spring 技术，为开发基于 Spring 的企业应用提供世界级支持。同时，全新版本的 Spring Tools 与 IDE 无关，可在各种编码环境中使用，支持 Eclipse、Visual Studio Code 与 Theia。</span></p> 
<p><span style="color:#191e1e"><strong>Spring Tools 4 for Eclipse 发行版的主要变化</strong></span></p> 
<ul> 
 <li>更新到 Eclipse 2022-06 版本（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.eclipse.org%2Feclipseide%2F2022-06%2F" target="_blank">new and noteworthy</a>）</li> 
</ul> 
<p><span style="color:#000000"><strong>fixes and improvements</strong></span></p> 
<ul> 
 <li>(Spring Boot) fixed：vscode-sts：@PutMapping 的工作区符号的边缘情况（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fsts4%2Fissues%2F781" target="_blank">#781</a>）</li> 
 <li>(VSCode) fixed：retries: 10 后无法从进程 12704 - com.xxxx.xx.xxx.BillingServiceApp 刷新实时数据 (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fsts4%2Fissues%2F748" target="_blank">#748</a>)</li> 
 <li>(Eclipse) fixed：Spring Boot Language Server 在关闭最后打开的编辑器后不会立即关闭，以避免打开下一个文件时需要重新启动服务器（与<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fsts4%2Fissues%2F568" target="_blank">#568</a>相关）</li> 
 <li>(Eclipse) fixed：打开 symbols 时，从相关项目中打开编译单元（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fsts4%2Fissues%2F769" target="_blank">#769</a>）</li> 
 <li>(Eclipse) fixed：内部 Web 浏览器无法在 macOS 上打开 HTTP URL 的问题已得到修复。如果你更喜欢使用内部浏览器而不是外部浏览器，则可以在 preferences 中进行切换。</li> 
 <li>(Concourse) fixed：更新 Concourse registry-image schema (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fsts4%2Fissues%2F777" target="_blank">#777</a>)</li> 
 <li>(Concourse) fixed：添加 registry_mirror & cross 作为已知属性 (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fsts4%2Fissues%2F752" target="_blank">#752</a>)</li> 
</ul> 
<p><span style="color:#333333">Spring Tools 4.15.1 计划于 2022 年 8 月上旬发布。</span></p>
