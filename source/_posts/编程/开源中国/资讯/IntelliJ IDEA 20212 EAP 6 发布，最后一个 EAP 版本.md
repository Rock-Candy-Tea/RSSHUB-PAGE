                                                        <p>IntelliJ IDEA 2021.2 EAP 6 现已发布。这是在 Beta 版之前的最后一个 EAP 版本，开发团队把重点放在一些特定框架的更新上。其中包括一个新的重构，以简化和加快 Java EE 使用迁移到 Jakarta EE 的速度、为新的 Spring Boot 项目自动下载共享索引、微服务图等等。该版本还方便地将保存时的操作集中在一个地方，并为图表提供了一个重新设计的结构视图。</p> 
<p>具体更新亮点如下：</p> 
<ul> 
 <li>自动下载新 Spring Boot 项目的共享索引</li> 
</ul> 
<p><img alt height="200" src="https://oscimg.oschina.net/oscnet/up-933edd34c14b16073d46636fc7b195ad9b6.png" width="500" referrerpolicy="no-referrer"></p> 
<ul> 
 <li>从 Java EE 自动迁移到 Jakarta EE</li> 
</ul> 
<p><img alt height="200" src="https://oscimg.oschina.net/oscnet/up-07f70edd42474149702b17070e9a0d5110e.gif" width="500" referrerpolicy="no-referrer"></p> 
<ul> 
 <li>支持 JPA 实体图</li> 
</ul> 
<p><img alt height="200" src="https://oscimg.oschina.net/oscnet/up-3810e970b1f8329a2088f7994354244cc55.gif" width="500" referrerpolicy="no-referrer"></p> 
<ul> 
 <li>支持 Micronaut 和 Quarkus 中的缓存注释：现在可以在 Micronaut 中使用 @Cacheable、@CachePut 和 @CacheInvalidate。每个注解的值都指向一个特定的缓存。</li> 
</ul> 
<p><img alt height="200" src="https://oscimg.oschina.net/oscnet/up-9cca6ac7ed17190799aebfe0dc999910f9f.gif" width="500" referrerpolicy="no-referrer"></p> 
<ul> 
 <li>JavaScript 和 TypeScript 中更好的微服务导航支持：在客户端和服务器端都显着改进了 JavaScript 和 TypeScript 中的 URL 导航</li> 
 <li>微服务图：现在可以在图表中显示微服务之间的交互。可以通过单击<em> </em>Endpoints 工具窗口右上角的 Services 图标来调用此图。</li> 
</ul>
