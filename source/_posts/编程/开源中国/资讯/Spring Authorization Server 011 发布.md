                                                        <p>Spring Authorization Server 0.1.1 已经发布，Spring Authorization Server 是 Spring Security 团队领导的社区驱动项目，致力于解决 Spring 社区的 OAuth 2.0 Authorization Server 支持问题。</p> 
<p><strong>主要更新内容</strong></p> 
<ul> 
 <li>在 OAuth2ClientAuthenticationProvider 中使用 PasswordEncoder</li> 
 <li>使用 PasswordEncoder 验证客户端凭据</li> 
 <li>环回地址的重定向 URI 验证</li> 
 <li>实现 OpenID 客户端注册端点</li> 
 <li>实现 OAuth 2.0 服务器元数据（RFC 8414）</li> 
 <li>实现令牌自省端点</li> 
 <li>实现 OpenID Connect 1.0 客户端注册端点</li> 
 <li>实现 OAuth 2.0 授权服务器元数据</li> 
 <li>修复示例身份验证服务器无法正常工作的问题</li> 
 <li>不应在 OAuth2AuthorizationServerConfigurer 中配置登录页面</li> 
 <li>范围 “openid” 应在访问令牌范围内</li> 
</ul>
