                                                        <p><span style="background-color:#ffffff; color:#333333">Spring Tools 4.13.0 现已发布。</span><span style="background-color:#ffffff; color:#333333">Spring Tools 4 是由 Spring 团队打造的 Spring 开发工具，从零开始构建，融合了现代技术和开发者工具架构。它在单独的进程中运行，从构建之初就考虑到了性能问题，并且支持最新的 Spring 技术，为开发基于 Spring 的企业应用提供世界级支持。同时，全新版本的 Spring Tools 与 IDE 无关，可在各种编码环境中使用，支持 Eclipse、Visual Studio Code 与 Theia。</span></p> 
<p><strong><span style="background-color:#ffffff; color:#333333">主要更新内容</span></strong></p> 
<ul> 
 <li><span style="background-color:#ffffff; color:#333333">更新到 Eclipse 2021-12 版本（包括对 Java 17 的支持）</span></li> 
 <li><span style="background-color:#ffffff; color:#333333">现在可以从常规下载页面获得针对 Apple Silicon 平台 (ARM M1) 的构建</span></li> 
 <li><span style="background-color:#ffffff; color:#333333">(Spring Boot) 修复：STS 4.12.0 (for eclipse) *.yml 不能自动提示</span></li> 
 <li><span style="background-color:#ffffff; color:#333333">(Spring Boot) 修复：用于在 Eclipse IDE 中自动完成的可序列化 Kotlin 数据类的 StackOverFlow 错误</span></li> 
 <li><span style="background-color:#ffffff; color:#333333">(Eclipse) 修复：Typescript 比较：在 "LSP4E Linked Editing Highlight" 期间发生内部错误</span></li> 
 <li><span style="background-color:#ffffff; color:#333333">(Eclipse) 修复：STS4 禁用了 js 语法高亮显示</span></li> 
 <li><span style="background-color:#ffffff; color:#333333">(Eclipse) 修复：无法在没有转义序列的情况下从 ANSI 控制台复制/粘贴</span></li> 
</ul>
