                                                                                            <p>Qt Creator 5.0 RC 发布，该版本主要更新内容如下：</p> 
<h3>通用：</h3> 
<ul> 
 <li>修正了关机时可能出现的各种崩溃</li> 
</ul> 
<h3>编辑</h3> 
<ul> 
 <li>在文件属性信息中增加了行尾和缩进；</li> 
 <li>增加了编辑书签注释的菜单项和快捷方式；</li> 
 <li>修复了 Markdown 的折叠；</li> 
</ul> 
<h3>C++</h3> 
<ul> 
 <li>增加了对 <code>clangd</code> 的实验性支持；</li> 
 <li>增加了函数参数的高亮选项；</li> 
 <li>为 Locator 中的符号添加了模板参数；</li> 
 <li>修正了重命名符号时默认选择与项目无关的文件的问题；</li> 
 <li>修正了带有多字节字符的字符串字面的高亮显示；</li> 
 <li>修正了已修改但尚未构建的<code>.ui</code>和<code>.scxml</code>文件的代码模型；</li> 
 <li>修正了模板的 "switch" 语句补全；</li> 
 <li>……</li> 
</ul> 
<h3>QML</h3> 
<ul> 
 <li>更新了解析器至 Qt 6.2；</li> 
 <li>修正了对多行模板字符串的处理；</li> 
 <li>修正了对必填和只读属性的处理；</li> 
 <li>修正了内联组件的重新格式化；</li> 
 <li>修正了带有默认值的函数的重新格式化；</li> 
 <li>修正了同名但不同命名空间的类型的错误警告；</li> 
</ul> 
<h3>美化器</h3> 
<ul> 
 <li>修正了<code>clang-format</code>和多字节字符的问题；</li> 
</ul> 
<h3>项目</h3> 
<ul> 
 <li>增加了对 Docker 设备上构建和运行的实验性支持；</li> 
 <li>修正了在改变当前配置时关闭项目的崩溃问题；</li> 
 <li>修正了运行配置环境的更新缺失；</li> 
 <li>修正了用户文件不必要地以新的时间戳保存的问题；</li> 
 <li>减少了加载项目后的 UI 冻结；</li> 
</ul> 
<h3>CMake</h3> 
<ul> 
 <li>删除了 "自动创建构建目录" 选项，使其成为默认行为；</li> 
 <li>在 "项目" 模式的右侧添加了 CMake 输出；</li> 
 <li>修正了 "跳转到文件" 中带有特殊字符的文件名；</li> 
 <li>修正了 CMake 工具选项的持久性；</li> 
</ul> 
<h3>平台</h3> 
<ul> 
 <li>Windows 
  <ul> 
   <li>增加了对 MSVC ARM64 工具链的支持；</li> 
  </ul> </li> 
 <li>macOS 
  <ul> 
   <li>改进了基于 ARM 的 macOS 与基于 Intel 的 Qt Creator 构建的体验；</li> 
   <li>修正了加载项目后注册文件表的性能问题；</li> 
  </ul> </li> 
 <li>Android 
  <ul> 
   <li>增加了对 Android 12 的检测；</li> 
   <li>修正了对 CMake 项目的 <code>_prepare_apk_dir</code> 目标的检测；</li> 
  </ul> </li> 
</ul>
