                                                                                            <p><span style="background-color:#ffffff; color:#000000">Stats 是一款开源的 macOS 系统监控工具，可以驻留在菜单栏中对 CPU、GPU、内存、磁盘和网络等进行实时监测。目前 Stats </span><span style="background-color:#ffffff; color:#333333">v2.7.31</span><span style="background-color:#ffffff; color:#000000"> 发布了 ，此版本带来如下变更：</span></p> 
<h2><strong>关闭的问题</strong></h2> 
<ul> 
 <li>功能：睡眠时将风扇模式更改为自动。如果启用“保存风扇速度”（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fexelban%2Fstats%2Fissues%2F1002" target="_blank">#1002</a>），风扇控制将在睡眠后恢复</li> 
 <li>功能：在电池弹出窗口中添加了低功耗模式状态（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fexelban%2Fstats%2Fissues%2F1024" target="_blank">#1024</a>）</li> 
 <li>功能：为电池小部件添加了低功耗模式着色（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fexelban%2Fstats%2Fissues%2F1024" target="_blank">#1024</a>）</li> 
</ul> 
<h2><strong>错误修复和新功能</strong></h2> 
<ul> 
 <li>功能：将ip请求从ipify服务移动到自己的服务器</li> 
 <li>功能：增加总能耗 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fexelban%2Fstats%2Fissues%2F976" target="_blank">#976</a> ) ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fexelban%2Fstats%2Fpull%2F1037" target="_blank">#1037</a> )</li> 
</ul> 
<h2><strong>本土化</strong></h2> 
<ul> 
 <li>功能：更新了越南语翻译（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fexelban%2Fstats%2Fpull%2F1036" target="_blank">#1036</a>）</li> 
 <li>功能：更新繁体中文翻译（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fexelban%2Fstats%2Fpull%2F1043" target="_blank">#1043</a>）</li> 
</ul>
