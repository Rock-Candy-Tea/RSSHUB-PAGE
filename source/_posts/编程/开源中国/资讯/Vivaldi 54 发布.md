                                                                                            <p>最新版本的 Vivaldi 5.4 带来了关闭网页面板声音的功能，可以对 Rocker Gestures 进行编程，以及微调的 Vivaldi 邮件、日历和 Feed Reader。</p> 
<h3>Web Panels</h3> 
<p>Web Panels 可以让用户更加方便地访问最喜欢和最常用的网站，它可以在浏览器侧边栏创建一个分屏视图。但 Web Panels 可能有声音，就像正常标签页一样。</p> 
<p>现在在 Vivaldi 5.4 中，官方已经添加了使 Web Panels 静音的功能，你可以选择你想听的声音来源。当你不希望社网站发出声音时，只需右键单击并将 Web Panels 静音。</p> 
<p></p> 
<h3>可定制的鼠标、Rocker 手势</h3> 
<p>与大多数主流浏览器不同，Vivaldi 的鼠标手势可以执行你喜欢的任何操作。Rocker 手势是 Vivaldi 鼠标手势功能的一部分。</p> 
<p>到目前为止，执行的操作是硬编码的，即在历史中往前或往后。但现在你可以为这两个手势分配任何命令。</p> 
<p></p> 
<h3>更多的功能</h3> 
<ul> 
 <li>复制链接以高亮显示：这允许用户快速分享网页的特定部分。例如，高亮显示并分享维基百科页面的部分，右键单击并选择 "复制链接到高亮"，然后将该 URL 发送给任何使用基于 Chromium 的浏览器的人。当页面加载时，它将直接转到文本的这一部分，而不是转到维基百科文章的顶部。</li> 
</ul> 
<p></p> 
<ul> 
 <li>总是使用安全连接：这个选项自动将所有连接升级为 HTTPS，并在加载不支持 HTTPS 的网站前显示提示。</li> 
</ul> 
<p></p> 
<h3>Vivaldi 邮件、日历和 Feed Reader</h3> 
<p>6月，Vivaldi 推出了 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fvivaldi.com%2Fpress%2Freleases%2Fvivaldi-mail-1-0-calendar-feed-reader-launches%2F" target="_blank">Vivaldi Mail 1.0</a>，与日历和 Feed Reader 集成。自推出以来，官方一直在 完善 Vivaldi Mail。</p> 
<p></p> 
<p><strong>以下是内置的电子邮件客户端的一些突出特点：</strong></p> 
<ul> 
 <li>在 Vivaldi 中管理所有的电子邮件帐户，全部免费，支持 IMAP 和 POP3 电子邮件帐户</li> 
 <li>你可以在 Vivaldi Mail 中使用你现有的邮件和日历服务，也可以在 <a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fvivaldi.net" target="_blank">vivaldi.net</a> 上建立一个账户。甚至可以添加一个信任的第三方服务</li> 
 <li>账户的邮件都有索引，因此可以离线搜索。</li> 
 <li>用 Vivaldi Mail 的数据库搜索你所有账户中的任何邮件。</li> 
</ul> 
<h3>其他更新内容</h3> 
<ul> 
 <li>[Chromium] 升级到 104.0.5112.83</li> 
 <li>[搜索] 允许在搜索中隐藏搜索引擎关键字</li> 
 <li>[阻止程序] 被阻止内容的图标被一个突出的框包围</li> 
 <li>[主题] 下拉高亮显示在深色模式下不可见</li> 
 <li>[开发工具] 无法复制和粘贴</li> 
</ul>
