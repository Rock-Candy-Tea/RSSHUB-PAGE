                                                        <p>Flax Engine 1.2 现已发布， Flax Engine 是用 C ++ 和 C＃ 编写的高质量现代 3D 游戏引擎，专为快速工作流程而设计。</p> 
<p><strong>主要更新内容</strong></p> 
<ul> 
 <li>车辆</li> 
</ul> 
<p>此版本包括对车辆的支持。现在，用户可以创建具有自定义物理形状、车轮和引擎设置的汽车，使用 Nvidia PhysX 来驱动模拟并公开许多高度可配置的属性来调整车辆。它可用于制作简单的汽车甚至高级逼真的赛车游戏。</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-94e9936498d6b070602a10a7cec8f83b17a.gif" referrerpolicy="no-referrer"></p> 
<ul> 
 <li>支持 Nintendo Switch</li> 
</ul> 
<p>Flax 将可供 Switch 开发人员使用。一旦准备就绪，将通过开发博客通知用户。</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-b7798511b53b4f0fb118b533e3353f07c63.JPEG" referrerpolicy="no-referrer"></p> 
<ul> 
 <li>低级网络</li> 
</ul> 
<p>该版本实现了一个具有网络对等点的低级网络系统，用于创建具有网络消息传输层的多人会话。默认网络驱动程序使用 ENet，可以使用想要的任何库（如 ValveSockets、SteamRelay、SteamP2P、Photon Realtime 等）。此外还发布了一个开源网络示例项目，以帮助用户开始在 Flax 中制作多人游戏。它具有游戏服务器客户端和游戏会话、聊天和玩家大厅。</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-85c7248c87d22d31f266e22030999be646e.png" referrerpolicy="no-referrer"></p> 
<ul> 
 <li>本地化</li> 
</ul> 
<p>该版本发布了用于游戏内的新本地化工具。这包括 UI 语言本地化、价值观文化（数字、货币和日期格式）以及为不同语言创建翻译。添加了新的本地化设置，这是一个带有许多实用程序的仪表板，例如添加新区域设置、搜索本地化字符串是代码或内容。此外，编辑器具有当前语言环境的实时预览选项。</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-d6ea0374607cdd2aac5fc769171890553f3.gif" referrerpolicy="no-referrer"></p> 
<ul> 
 <li>作业系统和任务图</li> 
</ul> 
<p>该版本创建了一个作业系统，它可以为每个 CPU 内核运行一个工作线程，以异步运行大量计算。此外，对于游戏系统或引擎系统使用的更高级的计算，该版本实现了任务图，以帮助更好地并行化游戏，而且它支持异步系统之间的依赖关系。</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-ece5cf079065f7f6c61fc9221c7c5b705ec.png" referrerpolicy="no-referrer"></p> 
<ul> 
 <li>分析器</li> 
</ul> 
<p>现在，Flax 支持用于 CPU 检测和性能分析的开源 Tracy 分析器。它可以分析游戏和编辑器，支持远程分析，并可用于捕获分析会话以进行比较。此外，Flax 将自动为 C# 和 Visual Script 方法插入配置文件区域，因此现在可以更轻松地进行游戏性能分析。</p>
