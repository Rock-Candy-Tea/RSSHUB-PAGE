                                                                                            <p>LLVM 15.0.1 发布了，此版本包含 LLVM 15.0 的第一批修复。<br> <br> LLVM 15.0 引入了许多安全缓解措施，同时正在为最新规范开发新的 C 和 C++ 功能，其他更改包括：</p> 
<ul> 
 <li>实验性 Microsoft HLSL / DirectX 支持，</li> 
 <li>RISC-V 更新，</li> 
 <li>更多 OpenMP 5.1 功能，</li> 
 <li>用于 debuginfod 的 HTTP 服务器，</li> 
 <li>初始 SPIR-V 返回-end 集成、</li> 
 <li>AMDGPU GFX11 工作</li> 
 <li>许多随机修复，</li> 
 <li>clang 格式、测试问题和其他更改。</li> 
 <li>其他各种新编译器功能。</li> 
</ul> 
<p> LLVM 15.0.1 有 32 处更改，可以通过 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdiscourse.llvm.org%2Ft%2Fllvm-15-0-1-released%2F65380" target="_blank">LLVM.org</a> 找到 LLVM 15.0.1 更改的完整列表。LLVM 15.0.1 的源代码和二进制文件可以从<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fllvm%2Fllvm-project%2Freleases%2Ftag%2Fllvmorg-15.0.1" target="_blank">GitHub</a>下载。<br> <br> 根据 LLVM 的发布节奏， LLVM 15.0.2 将在两周内发布。下一个主要功能版本 LLVM 16.0 正在开发，将于明年 3 月左右首次亮相。</p>
