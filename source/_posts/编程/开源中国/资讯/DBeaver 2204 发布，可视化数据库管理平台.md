                                                                                            <p>DBeaver 是一个免费开源的通用数据库工具，适用于开发人员和数据库管理员。</p> 
<p>DBeaver 22.0.4 发布，更新内容如下：</p> 
<ul> 
 <li>SQL editor： 
  <ul> 
   <li>修复了在新行中出现分号的问题</li> 
   <li>修正了客户端命令解析的问题</li> 
   <li>改进了对错误高亮的支持</li> 
   <li>改进了标签关闭/确认的行为</li> 
   <li>修复了多个 @set 命令的脚本处理问题</li> 
   <li>增加了数据自动输出的命令</li> 
   <li>修复了切换注释命令</li> 
   <li>修复了 SQL 脚本数据源关联命令</li> 
  </ul> </li> 
 <li>Data editor： 
  <ul> 
   <li>值过滤器现在可以显示不同值的总数</li> 
   <li>搜索整个单词已被修复</li> 
   <li>自定义过滤器值保存被修复</li> 
  </ul> </li> 
 <li>Spatial viewer: 
  <ul> 
   <li>增加了在地图和数据网格上选择空间对象的功能</li> 
   <li>增加了默认缩放级别选项</li> 
  </ul> </li> 
 <li>Data transfer: 
  <ul> 
   <li>改进了默认的数据类型映射</li> 
   <li>多表传输现在可以打开所有目标表</li> 
   <li>数据库恢复任务现在需要额外的确认</li> 
   <li>输出文件夹参数现在支持额外的变量</li> 
   <li>导入任务的用户界面得到了修复</li> 
   <li>SQL 导出格式现在支持追加模式</li> 
  </ul> </li> 
 <li>Navigator: 数据库选择器弹出窗口已被修复</li> 
 <li>SSH: 解决了连接和损坏的 known_hosts 文件的问题</li> 
 <li>CLI: 对于使用 -con 参数打开的连接，修复了 "crate" 参数</li> 
 <li>Windows: DBeaver 现在支持网络路径上的工作空间 (e.g. \\network-share\workspace)</li> 
 <li>MacOS: 修复了数据编辑后 UI 冻结的问题</li> 
 <li>DB2 i: 键和外键元数据的读取得到了修复</li> 
 <li>MySQL： 
  <ul> 
   <li>修复了为几何数据类型生成 SQL 的问题</li> 
   <li>权限编辑器被修复（权限重复的问题）</li> 
   <li>修复了带有 zerofill 标志的小数数据类型的问题</li> 
   <li>增加了 UPDATE SET 替换方法</li> 
  </ul> </li> 
 <li>Netezza：表重命名 SQL 已被修复</li> 
 <li>Oracle：修复了表分区元数据的读取问题</li> 
 <li>PostgreSQL: 
  <ul> 
   <li>修复了 SQL 生成和 JSONB 列的问题</li> 
   <li>添加了视图列编辑支持</li> 
   <li>由于安全原因，默认驱动版本改为 42.2.25</li> 
  </ul> </li> 
 <li>SQLite：修复了连接设置对话框的用户界面</li> 
 <li>SQL Server: 
  <ul> 
   <li>修复了对货币数据类型的支持</li> 
   <li>重新设计了连接设置对话框</li> 
  </ul> </li> 
 <li>在应用程序的本地化中进行了许多小的修正</li> 
</ul>
