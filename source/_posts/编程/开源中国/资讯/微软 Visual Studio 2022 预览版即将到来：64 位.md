                                                        <p>微软方面表示，Visual Studio 2022 的第一个公开预览版将于今年夏天发布。并透露，Visual Studio 的下一个主要版本将更快、更易于使用、更轻量级，专为学习者和构建工业规模解决方案的人设计。同时，Visual Studio 将是 64 位的；用户体验也将会更完善。</p> 
<p>官方公告指出，今年夏天的公开预览版将只包括即将发布的部分功能，其他功能将在后续的更新中出现。预览版 1 将特别包括一些 UI 的完善和可访问性的改进。</p> 
<p>其解释称，Visual Studio 2022 转向 64 位后，将不再局限于主 devenv.exe 进程中的约 4gb 内存。在 Windows 上使用 64 位的 Visual Studio，用户可以打开、编辑、运行和调试最大和最复杂的解决方案，而不会耗尽内存。</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-038489d4bad6ae274da3357bb42296b0eb7.gif" referrerpolicy="no-referrer"></p> 
<p>此外，Visual Studio 2022 版本还将包括：</p> 
<ul> 
 <li>核心调试器中的性能改进</li> 
 <li>一个更新的 UI，旨在减少复杂性，这将增加与 Accessibility Insights 的集成。微软计划更新图标，并增加对 Cascadia Code 的支持，这是一种新的 fixed-width 字体，可提高可读性。</li> 
</ul> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-55c5a4c1665dad5ea4261501a6e8cba7661.png" referrerpolicy="no-referrer"></p> 
<ul> 
 <li>对 .NET 6 的支持，Windows 和 Mac 开发人员都可以使用它来构建 Web、客户端和移动应用程序，并为开发 Azure 应用程序提供了更好的支持。</li> 
</ul> 
<p><img alt height="464" src="https://oscimg.oschina.net/oscnet/up-81fbd9ee302a55ad5a91e242b596c1e5843.png" width="500" referrerpolicy="no-referrer"></p> 
<ul> 
 <li>Visual Studio 2022 将包括对 C++ 工作负载的强大支持，包括新的 productivity features、C++20 工具和 IntelliSense。还集成了对 CMake、Linux 和 WSL 的支持，使用户更容易创建、编辑、构建和调试跨平台应用程序。</li> 
 <li>将文本聊天集成到 Live Share 协作功能中。</li> 
 <li>增加对 Git 和 GitHub 支持。</li> 
 <li>改进的 Code search。</li> 
 <li>计划将 Visual Studio for Mac 移至原生 macOS UI，从而对其进行刷新。</li> 
</ul>
