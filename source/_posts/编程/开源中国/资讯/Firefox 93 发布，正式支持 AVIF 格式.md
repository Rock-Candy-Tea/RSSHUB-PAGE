                                                        <p>Firefox 93.0 正式发布，该版本是 Firefox 最新的稳定版本，包含功能更新，以及安全和稳定性修复。</p> 
<h3>新功能</h3> 
<ul> 
 <li> <p>默认启用 AVIF 格式支持</p> <p>Mozilla 原计划是准备在 Firefox 92 中启用对 AVIF 格式的支持，但在上个版本中 Mozilla 临时推迟了这个功能的发布时间。如今在 Firefox 93 中，对 AVIF 的支持已被默认启用。</p> <p>与现有的图像格式相比，AVIF 可以为网站显著节省带宽。</p> </li> 
 <li> <p>当内存严重不足时在 Windows 上卸载标签页</p> <p>此次更新，Firefox 针对 Windows 提供了一个新的标签卸载功能。如果 Windows 设备的可用系统内存严重不足，那么 Firefox 将会自动卸载标签页。</p> <p>Firefox 对标签页的卸载将基于最后的访问时间、内存使用和其他属性等多方面因素共同决定。Mozilla 希望该功能将减少 Firefox 用户遇到与内存有关的崩溃次数。</p> </li> 
 <li> <p>默认阻止不安全的下载</p> <p>Firefox 现在会默认阻止不安全的下载（即 HTTPS 网页上的 HTTP 下载）。</p> <p>有这方面需求的用户，可以通过将 <code>dom.block_download_insecure</code> 首选项设置为 <code>False</code>，再次允许不安全的下载。</p> </li> 
 <li> <p>Firefox PDF 查看器现在支持填写更多的表格</p> </li> 
 <li> <p>为了防止 macOS 用户从挂载的 .dmg 文件中运行 Firefox 的会话丢失，现在会被提示完成安装。这个提示只在用户第一次在他们的电脑上运行 Firefox 时出现</p> </li> 
 <li> <p>改进了 SmartBlock 3.0 的隐私保护的网页兼容性</p> </li> 
 <li> <p>在 Strict Tracking Protection 和 Private Browsing 中引入了新的引荐来源跟踪保护</p> </li> 
</ul> 
<h3>修复</h3> 
<ul> 
 <li>VoiceOver 屏幕阅读器现在可以正确地报告可访问的树形控件中的可选项目为选中或未选中</li> 
 <li>Orca 屏幕阅读器现在可以在 Firefox 中正常工作</li> 
 <li>各种安全问题的修复</li> 
</ul>
