                                                        <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">RetroArch 1.9.11 现已发布。RetroArch 是款功能强大的跨平台模拟器，不但能够模拟许多不同的游戏主机，还能在 Windows、MacOS、Linux、Android、iOS 以及多种游戏主机上执行。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">主要更新内容如下：</p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>ANDROID：新的 launcher logo - 将填充主屏幕上的角落</li> 
 <li>D3D9：驱动程序再次工作（仅 RGUI 和软件渲染核心工作，尚不支持着色器）</li> 
 <li>HID/MAC：WiiU GameCube 适配器现在可以使用</li> 
 <li>HID/WIIU：WiiU 和通用 HID subsystem 统一/合并了</li> 
 <li>INPUT：重构菜单切换组合按钮逻辑以允许退出组合按钮</li> 
 <li>INPUT/UDEV：添加鼠标相关检查并适当设置以解决问题</li> 
 <li>LIBNX/SWITCH：拆分的 Joycon 按钮映射</li> 
 <li>LIBRETRO：添加环境回调以使 cores 能够通知前端 core otion value 发生了变化</li> 
 <li>MIYOO：添加初始端口</li> 
 <li>OPENDINGUX/RG350：为 RG350 启用 tinyalsa</li> 
 <li>PS2：添加“Change Resolution”选项</li> 
 <li>PS2：添加更改视频窗口偏移的选项</li> 
 <li>STEAM/LINUX：移动到新的“soldier”运行时</li> 
 <li>WAYLAND：删除 xdg-shell-v6 协议</li> 
 <li>WINDOWS：修复窗口标题中的 non-ASCII 文本显示</li> 
 <li>WINDOWS 11：现在显示 Windows 11 版本名称（Information -> System Information）</li> 
 <li>UWP：对 WinRT VFS 层的进一步改进</li> 
</ul>
