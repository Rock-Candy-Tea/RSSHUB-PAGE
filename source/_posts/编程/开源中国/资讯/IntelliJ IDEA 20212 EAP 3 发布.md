                                                        <p>IntelliJ IDEA 2021.2 EAP 3 现已发布。此版本带来了对 Async profiler 2.0 的支持、来自 Toolbox App 的更新通知、以及调试器中的预览标签等内容。</p> 
<p><strong>来自 Toolbox App 的更新通知</strong></p> 
<p>现在，你不会错过 Toolbox App 的任何重要产品更新。你的 IDE 会通知你是否有新版本可供下载，并提供直接从 IntelliJ IDEA 升级到该版本的建议。为此，你则需要安装 Toolbox App 1.20.8804 或更高版本。</p> 
<p><img alt height="200" src="https://oscimg.oschina.net/oscnet/up-92072544625b43182834f254793784f4b00.png" width="500" referrerpolicy="no-referrer"></p> 
<p><strong>支持 Async profiler 2.0</strong></p> 
<p>此版本引入了对 Async Profiler 2.0 的支持。新的配置 Async Profiler 结合了 CPU 和 Allocation Profiler 的强大功能，并在与它们相同的位置工作。</p> 
<p><img alt height="199" src="https://oscimg.oschina.net/oscnet/up-edfcb12d52c6b22d8a92d338b86fb215fd9.JPEG" width="500" referrerpolicy="no-referrer"></p> 
<p>在 Flame Graph、Call Tree 和 Method List tabs 中，新的 Show 下拉列表让你可以选择是要显示 CPU samples 还是内存分配。</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-b3c13681d57049f9a56015879989d81063c.gif" width="500" referrerpolicy="no-referrer"></p> 
<p>Timeline 会同时显示这两种情况。你可以使用右上角的控制器过滤要显示的内容。</p> 
<p><strong>重新设计的 Gradle 配置</strong></p> 
<p>基本参数现在被方便地分组在一个屏幕上，并能根据你的需要修改选项。</p> 
<p><strong><img alt height="315" src="https://oscimg.oschina.net/oscnet/up-6f3bf71d6a01450eb825f68db140bcb5649.png" width="500" referrerpolicy="no-referrer"></strong></p> 
<p><strong>预览标签现在可以在调试器中使用</strong></p> 
<p>以前只适用于项目视图中的文件的预览标签，现在也适用于调试期间打开的文件。工作原理如下：</p> 
<p><img alt height="325" src="https://oscimg.oschina.net/oscnet/up-e41ac541cbc7cbc70432608842b3278a36f.gif" width="500" referrerpolicy="no-referrer"></p> 
<p><strong>同步的 code completion</strong></p> 
<p>当你在 Code With Me 会话中处于 Following 模式时，可以跟踪你所关注的人使用的 code completion 建议。</p>
