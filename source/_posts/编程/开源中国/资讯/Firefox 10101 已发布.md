                                                                                            <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">Firefox 101.0.1 已发布，新版本修复了三个主要问题和几个较小的问题，包括 Windows 上无法使用的问题。</p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>修复了 Firefox 在 macOS 上关闭时清除剪贴板的问题（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fbugzilla.mozilla.org%2Fshow_bug.cgi%3Fid%3D1771823" target="_blank">错误 1771823</a>）</li> 
</ul> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">在 Apple Macintosh 设备上运行 Firefox 的用户<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fbugzilla.mozilla.org%2Fshow_bug.cgi%3Fid%3D1771823" target="_blank">报告</a>说，浏览器在关闭时正在清除剪贴板。Firefox 不应该在退出时清除剪贴板，在 Windows 或 Linux 设备上也不应如此，该问题已解决。</p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>修复了在某些 Windows 系统上启用 win32k 锁定，导致功能严重受损的兼容性问题（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fbugzilla.mozilla.org%2Fshow_bug.cgi%3Fid%3D1769845" target="_blank">错误 1769845</a>）</li> 
</ul> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">报道称，Firefox 的大部分界面在 Windows 10 虚拟机中都无法使用。另一位用户报告说该问题在 Windows 11 设备上出现。Mozilla 退出了某些更改，已经解决了受影响用户的问题。</p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>修复了在某些 Linux 系统上右键单击画中画窗口时，不出现上下文菜单的问题（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fbugzilla.mozilla.org%2Fshow_bug.cgi%3Fid%3D1771914" target="_blank">错误 1771914</a>）</li> 
</ul> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">当用户右键单击画中画窗口时，Firefox 显示的右键菜单在某些 Linux 发行版上没有出现。画中画模式允许用户将媒体流移动到一个专用窗口，该窗口在很大程度上独立于主浏览器窗口。Mozilla 已<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fbugzilla.mozilla.org%2Fshow_bug.cgi%3Fid%3D1771914" target="_blank">修复</a>这个问题。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start"> </p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">该版本还包括几个未指定的稳定性修复，详情可查看<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.mozilla.org%2Fen-US%2Ffirefox%2F101.0.1%2Freleasenotes%2F" target="_blank">更新公告</a>。</p>
