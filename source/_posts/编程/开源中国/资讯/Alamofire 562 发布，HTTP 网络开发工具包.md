                                                        <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">Alamofire 5.6.2 已发布，Alamofire 是用一个功能强大的，用 Swift 语言编写的 HTTP 网络开发工具包，支持各种 HTTP Method、JSON、文件上传、文件下载和多种认证方法。</p> 
<h4 style="margin-left:.6em; margin-right:0; text-align:start"><strong>添加</strong></h4> 
<ul style="margin-left:0; margin-right:0"> 
 <li>当提供给 JSONEcoding 的对象包含无法通过 JSONSerialization 传递的值时报错。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FAlamofire%2FAlamofire%2Fpull%2F3624" target="_blank">#3624</a></li> 
</ul> 
<h4 style="margin-left:.6em; margin-right:0; text-align:start"><strong>更新</strong></h4> 
<ul style="margin-left:0; margin-right:0"> 
 <li>CI 支持所有受支持的 Xcode、macOS 和 Linux 版本。<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FAlamofire%2FAlamofire%2Fpull%2F3607" target="_blank">#3607</a><span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FAlamofire%2FAlamofire%2Fpull%2F3622" target="_blank">#3622</a></li> 
 <li>支持各种 Xcode 版本。<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FAlamofire%2FAlamofire%2Fpull%2F3616" target="_blank">#3616</a><span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FAlamofire%2FAlamofire%2Fpull%2F3632" target="_blank">#3632</a></li> 
 <li><code>Cache-Control</code><span> </span>标头测试不需要硬编码的延迟。<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FAlamofire%2FAlamofire%2Fpull%2F3601" target="_blank">#3601</a></li> 
</ul> 
<h4 style="margin-left:.6em; margin-right:0; text-align:start"><strong>修复</strong></h4>

 <li>Swift 版本检查<span> </span><code>SecTrustCopyCertificateChain</code>。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FAlamofire%2FAlamofire%2Fpull%2F3606" target="_blank">#3606 </a></li> 
 <li>各种文档问题。</li> 
</ul>
