                                                                                            <p>Kendo UI Admin & Site v0.14.0 已经发布，后台管理框架。</p> 
<p>此版本更新内容包括：</p> 
<h4>新增 [ New ]</h4> 
<p>左侧导航</p> 
<ul> 
 <li>布局 [ Layout ] 
  <ul> 
   <li>磁贴布局 [ TileLayout ]</li> 
   <li>表单 [ Form ]</li> 
  </ul> </li> 
 <li>导航 [ Navigation ] 
  <ul> 
   <li>导航栏 [ AppBar ]</li> 
   <li>步骤条 [ Stepper ]</li> 
   <li>向导 [ Wizard ]</li> 
  </ul> </li> 
 <li>表单 [ Forms ] 
  <ul> 
   <li>文本框 [ TextBox ]</li> 
   <li>文本域框 [ TextArea ]</li> 
   <li>图像编辑框 [ ImageEditor ]</li> 
  </ul> </li> 
 <li>交互 [ Interactivity ] 
  <ul> 
   <li>加载中 [ Loader ]</li> 
  </ul> </li> 
</ul> 
<h4>✨ 更新 [ Update ]</h4> 
<p>基础功能</p> 
<ul>

   <li>忘记密码 [ Forget Password ]</li> 
  </ul> </li>

  <ul> 
   <li>便签 [ Note ]</li> 
  </ul> </li> 
</ul>

 <li>导航折叠 [ Navigation Drawer ]</li> 
 <li>全局搜索 [ Global Search ]</li> 
 <li>主题 [ Theme ] 
  <ul> 
   <li>IKKI Amikoko</li> 
   <li>Ant Design</li> 
   <li>Kendo UI Less</li> 
   <li>Kendo UI Sass</li>

   <li>Material Design</li> 
  </ul> </li> 
 <li>信息 [ Message ]</li> 
 <li>提醒 [ Notice ]</li> 
 <li>用户名头像显示 [ User Name & Avatar ] 
  <ul> 
   <li>用户中心 [ User Center ]</li> 
  </ul> </li> 
</ul> 
<p>左侧导航</p> 
<ul> 
 <li>综合 [ Dashboard ]

   <li>表格 [ Grids ]
