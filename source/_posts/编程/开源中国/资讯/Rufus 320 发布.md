                                                                                            <p><span style="background-color:#ffffff; color:#333333">Rufus 是一个可以帮助格式化和创建可引导 USB 闪存盘的工具，它可以快速把 ISO 格式的系统镜像文件快速制作成可引导的 USB 启动安装盘，支持 Windows 或 Linux 启动。3.20 版本更新内容如下： </span></p> 
<ul> 
 <li>为 Windows 10 启用适用的 Windows 用户体验选项</li> 
 <li>在会话之间记住最后的 Windows 用户体验选择</li> 
 <li>增加自动创建本地账户和区域选项的复制<br> （注意：这仅限于创建一个与当前用户同名的帐户，并且密码为空，用户将在第一次重启后被提示更改）</li> 
 <li>尽可能为 TPM/SB/RAM bypass 恢复 registry keys 的 offline insertion</li> 
 <li>删除存储旁路，因为这是一个没有任何作用的虚假旁路</li> 
 <li>在显示“UEFI boot only”警告消息时提高 BIOS 兼容性</li> 
 <li>修复 Windows To Go 出现两次 Windows 用户体验对话框的问题</li> 
 <li>修复 Windows 用户体验选项不适用于 ARM64 的问题</li> 
 <li>修复 Microsoft Account bypass 不被应用的问题，除非选择 TPM/SB/RAM bypass</li> 
 <li>修复对具有非标准前缀的 GRUB2 bootloaders 的 overeager 检测</li> 
</ul>
