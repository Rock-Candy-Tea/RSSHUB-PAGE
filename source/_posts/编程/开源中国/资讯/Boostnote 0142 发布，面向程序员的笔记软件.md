                                                                                            <p>Boostnote 是一款面向程序员的笔记应用程序，适用于 Mac、Windows 和 Linux 平台。它着重于快速编写 Markdown 和代码片段，可以以更好的方式组织这些内容。</p> 
<p>由于 Boostnote 是一个 Markdown 编辑器，所以主要用 markdown 来写内容。在编辑笔记时，内容会自动保存，并且可以一键查看预览。在代码片段中，可以高亮显示 Javascript、Python、HTML 等100多种语言的代码语法，并在一个笔记中保存多个代码片段。</p> 
<p>Boost Note 0.14.1 版本以来的变化包含：</p> 
<ul> 
 <li>修复应用程式初始化</li> 
 <li>修复 SVG 附件</li> 
 <li>改善 PDF/HTML 导出行为</li> 
 <li>修复云空间打开时窗口的可拖动区域</li> 
 <li>不再推荐使用的旧版云存储，而推荐新版 Cloud Space</li> 
 <li>放弃采用 Note 应用程序的布局</li> 
 <li>增加了从本地空间到云空间的迁移</li> 
 <li>修正了代码围栏的 mime 类型处理</li> 
</ul>
