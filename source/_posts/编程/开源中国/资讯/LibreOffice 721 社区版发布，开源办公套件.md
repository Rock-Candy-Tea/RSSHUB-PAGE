 <p><span style="background-color:#ffffff; color:#333333">LibreOffice 7.2.1 社区版现已发布，这是 LibreOffice 7.2 系列的第 1 个小版本，面向技术爱好者和高级用户。该版本包含大约 87 个错误修复和文档兼容性改进</span>。</p> 
 <p><span style="background-color:#ffffff; color:#333333"><strong>主要更新内容</strong></span></p> 
 <ul> 
  <li><span style="background-color:#ffffff; color:#333333">自动过滤器打开时出现分页问题</span></li> 
  <li><span style="background-color:#ffffff; color:#333333">修复更正中间带有注释的拼写错误的单词会删除注释的问题</span></li> 
  <li><span style="background-color:#ffffff; color:#333333">修复法导入使用 CsvHelper 创建的 CSV 文件的问题</span></li> 
  <li><span style="background-color:#ffffff; color:#333333">修复放大或缩小锚定时改变文档的位置的问题</span></li> 
  <li><span style="background-color:#ffffff; color:#333333">定义数据库范围不应在使用虚假名称后关闭</span></li> 
  <li><span style="background-color:#ffffff; color:#333333">隐藏一列后，插入分栏和换行 </span></li> 
  <li><span style="background-color:#ffffff; color:#333333">删除工作表列时出现额外的过滤器图标</span></li> 
  <li><span style="background-color:#ffffff; color:#333333">修复无法链接到 macOS 地址簿，导致崩溃并重新启动的问题</span></li> 
  <li><span style="background-color:#ffffff; color:#333333">导航器的工具提示显示 URI 编码的 URL（带有 %%%）而不是文件的 </span><span style="background-color:#ffffff; color:#202122">cyrillic<span> </span></span><span style="background-color:#ffffff; color:#333333">路径</span></li> 
  <li><span style="background-color:#ffffff; color:#333333">图像工具栏中的区域样式/填充下拉菜单不起作用</span></li> 
  <li><span style="background-color:#ffffff; color:#333333">如果 Next Style 设置为 Heading 2，章节标题 1 的编号会在设置 Return 后发生变化</span></li> 
  <li><span style="background-color:#ffffff; color:#333333">表格在剪切/复制和粘贴后会失去其样式设置</span></li> 
  <li><span style="background-color:#ffffff; color:#333333">在单个标签中写入了多个样式属性时会导致表格边框消失</span></li> 
  <li><span style="background-color:#ffffff; color:#333333">LO Draw 的选项卡式界面中没有克隆按钮</span></li> 
  <li><span style="background-color:#ffffff; color:#333333">切换到动画侧边栏面板后取消选择自动布局的对象区域</span></li> 
  <li><span style="background-color:#ffffff; color:#333333">如果段落末尾有脚注或尾注锚点，REPLACE 不起作用</span></li> 
  <li>编辑自动行高或更改行高会删除自动筛选结果</li> 
  <li>改进 Calc 的自动完成功能</li> 
  <li>在大型电子表格中添加新行并保存导致崩溃</li> 
  <li>当宏尝试在 Base 中启动表单文档时，LO 崩溃（仅限 kf5</li> 
  <li>FilePicker 服务不显示打开和取消按钮（仅限 gtk3）</li> 
  <li>CALC Row 操作在 7.2 中比 7.1 慢 100 倍以上</li> 
  <li>Basic IDE 在出现几条错误消息后挂起</li> 
  <li>带有自定义字符串+通用数字格式的 "显示精度" 的结果变为 0</li> 
  <li>OpenGL 转换不适用于 GTK3 VCL 插件</li> 
  <li>无法使用 LibreOffice 7.2.0 从 ODT 加载 Formula-14，显示为 OLE</li> 
 </ul>
