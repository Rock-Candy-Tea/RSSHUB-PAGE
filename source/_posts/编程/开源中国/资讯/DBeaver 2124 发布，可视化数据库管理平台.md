                                                                                            <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">DBeaver是一个免费开源的通用数据库工具，适用于开发人员和数据库管理员。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">DBeaver 21.2.4 发布，更新内容如下：</p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>为数据传输、管理任务和 SQL 任务重新设计了任务管理界面；</li> 
 <li>Data transfer: 
  <ul style="margin-left:0; margin-right:0"> 
   <li>为PostgreSQL 增加了批量加载支持；</li> 
   <li>改进了 PostgreSQL、Oracle 和 Sybase 的跨数据库数据类型映射；</li> 
   <li>为文件名模式增加了额外的变量 （$&#123;year&#125;、$&#123;month&#125;、$&#123;day&#125; 等）；</li> 
   <li>在数据传输向导中进行了一些修复</li> 
  </ul> </li> 
 <li>Data editor: 
  <ul style="margin-left:0; margin-right:0"> 
   <li>修复了几何值的复制粘贴操作；</li> 
   <li>Gis viewer：对象工具提示已修复；</li> 
   <li>解决了无效的 SRID 的问题；</li> 
   <li>增加了从上面/下面的行复制的命令；</li> 
   <li>在虚拟列定义中增加了 JSON 函数</li> 
   <li>修复了图像查看器闪烁的问题；</li> 
   <li>修正了行数计算问题；</li> 
   <li>修复了对复合数据类型的支持；</li> 
  </ul> </li> 
 <li>SQL editor: 
  <ul style="margin-left:0; margin-right:0"> 
   <li>增加了新的脚本模板；</li> 
   <li>修复了执行计划解释标签的工具提示；</li> 
  </ul> </li> 
 <li>增加了共享 jar 库的配置；</li> 
 <li>Navigator: 
  <ul style="margin-left:0; margin-right:0"> 
   <li>修复了命令行处理程序；</li> 
   <li>修正了布尔编辑器的用户界面；</li> 
   <li>修复了元数据编辑器重新打开时的用户界面冻结问题；</li> 
  </ul> </li> 
 <li>Clickhouse: 
  <ul style="margin-left:0; margin-right:0"> 
   <li>表的统计信息读取被修复</li> 
   <li>改进了 SQL dialect 的定义；</li> 
  </ul> </li> 
 <li>CockroachDB：增加了查询结果限制；</li> 
 <li>CSV、Athena：数据库元数据现在是只读的；</li> 
 <li>Redshift：增加了对几何图形编辑的支持</li> 
 <li>Snowflake：驱动版本现在是硬编码的；</li> 
 <li>Sybase：增加了数据库统计信息；</li> 
</ul>
