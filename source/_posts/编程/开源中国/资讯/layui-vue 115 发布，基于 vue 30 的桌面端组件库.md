                                                                                            <p><strong>新特性</strong></p> 
<p>[新增] table 组件 childrenColumnName 属性, 配置 children 子节点为其他字段<br> [新增] table 组件 indent-size 属性, 用于 tree-table 模式控制每一层的缩进宽度<br> [新增] table 组件 children 字段解析, 当字段中存在 children 时会自动转化为树表<br> [新增] table 组件 expand 插槽, 内容较多不能一次性完全展示时使用, 参数 data 为当前行数据<br> [新增] input 组件 label-position 属性, 通过设置改变表单域标签的位置, top、left 为可选值<br> [新增] tree 组件 title 插槽, 参数 data 为当前行数据, 用于自定义节点标题</p> 
<p><strong>已修复</strong></p> 
<p>[修复] tab 组件 brief 风格 position 为 bottom 时 active-bar 的位置<br> [修复] tab 组件 brief 风格 background-color 为 transparent 透明色</p> 
<p><strong>优化项</strong></p> 
<p>[优化] tab 组件 tab-panel 宽度超出当前组件宽度时, 增加左右滑动的能力<br> [优化] input 组件 foucs 状态, 新增 global-checked-color 选中色<br> [优化] layer 组件 夜间模式 样式, 新增 border 边框样</p> 
<p>查看详情，请前往： <a href="https://gitee.com/layui/layui-vue/releases">https://gitee.com/layui/layui-vue/releases</a></p>
