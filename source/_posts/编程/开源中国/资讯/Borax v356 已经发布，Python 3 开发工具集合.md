                                                                                            <p>Borax v3.5.6 已经发布，Python 3 开发工具集合</p> 
<p>此版本更新内容包括：</p> 
<ul> 
 <li>新增Docs Test</li> 
 <li><code>borax.calendars.lunardate</code> 
  <ul> 
   <li>新增文本解析函数 <code>LunarDate.strptime</code> （<a href="https://gitee.com/link?target=https%3A%2F%2Fgithub.com%2Fkinegratii%2Fborax%2Fissues%2F44" target="_blank">#44</a>）</li> 
   <li>新增格式化修饰符 <code>%c</code></li> 
   <li>新增 <code>TextUtils.gz2offset</code> / <code>TextUtils.offset2gz</code> 干支转化方法</li> 
  </ul> </li> 
 <li><code>borax.calendars.festivals2</code> 
  <ul> 
   <li>星期型节日支持倒数序号 和 每月频率（<a href="https://gitee.com/link?target=https%3A%2F%2Fgithub.com%2Fkinegratii%2Fborax%2Fissues%2F43" target="_blank">#43</a>）</li> 
   <li>"除夕"节日修改为“农历年最后一天”而不是“十二月最后一天”</li> 
   <li>修正三伏九九天计算错误的BUG（<a href="https://gitee.com/link?target=https%3A%2F%2Fgithub.com%2Fkinegratii%2Fborax%2Fissues%2F45" target="_blank">#45</a>）</li> 
   <li><code>TermFestival</code> 支持干支日推算的节日 （<a href="https://gitee.com/link?target=https%3A%2F%2Fgithub.com%2Fkinegratii%2Fborax%2Fissues%2F46" target="_blank">#46</a>）</li> 
   <li><code>TermFestival</code> 新增 term 参数</li> 
   <li>节气名称支持拼音首字母形式</li> 
  </ul> </li> 
 <li><code>borax.structures.percentage</code> 
  <ul> 
   <li>优化底层百分数格式化显示</li> 
  </ul> </li> 
 <li><code>borax.devtools</code> 
  <ul> 
   <li>新增 <code>RuntimeMeasurer.print_</code> 方法</li> 
  </ul> </li> 
</ul> 
<p>详情查看：<a href="https://gitee.com/kinegratii/borax/releases/v3.5.6">https://gitee.com/kinegratii/borax/releases/v3.5.6</a></p>
