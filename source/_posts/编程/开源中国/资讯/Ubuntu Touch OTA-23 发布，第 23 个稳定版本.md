                                                                                            <p>Canonical 曾经也是移动领域的参与者并推出了 Ubuntu Touch，它的目标是实现桌面、平板和手机的真正融合。后来 Canonical 搁置了这个项目，并将其移动操作系统 Ubuntu Touch 出售给了 UBports 基金会，Ubuntu Touch 也是一款尊重隐私和自由的移动操作系统。</p> 
<p>近日 Ubuntu Touch 发布了 OTA-23，即 Ubuntu Touch 的第 23 个稳定版本。</p> 
<h3>新内容：</h3> 
<ul> 
 <li>这个版本的 Ubuntu Touch 仍然是基于 Ubuntu 16.04（Xenial Xerus）</li> 
 <li>对 FM 收音机的初步支持</li> 
 <li>消息应用：对 MMS 处理较大附件的改进</li> 
 <li>支持媒体播放器应用中的硬件视频解码</li> 
 <li>无线显示支持：成功地修复了 Aethercast 协议，允许为你的设备设置一个外部显示器，并像在桌面上工作。</li> 
 <li>背光调光更加灵敏：现在所有的设备都可以在更短的的时间内点亮屏幕或熄灭屏幕，无论环境光线如何</li> 
</ul> 
<h3>错误修复</h3> 
<ul> 
 <li>外部显示的改进：指示器缩放现在在外部屏幕上是正确的，用鼠标操作时，启动器和应用程序抽屉不再消失。</li> 
 <li>某些设备在进入和退出睡眠状态时，声音播放不稳定</li> 
 <li>修复随机创建新的 Wi-Fi 连接的问题</li> 
</ul>
