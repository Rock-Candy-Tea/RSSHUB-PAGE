                                                        <p>Apache Ant 1.9.16 和 1.10.11 已发布，这也是目前正在维护的两个分支——1.9.x 和 1.10.x。前者要求 Java 5 以上，后者要求 Java 8 以上。1.9.x 系列主要是修复 bug，1.10.x 会增加新功能，两个分支都以 Ant 1.9.7 为基础。除非在构建过程中需要使用 Java 8 之前的版本，否则最好还是使用官方推荐的 1.10.x 分支。</p> 
<p>最新发布的两个版本都修复了潜在的拒绝服务漏洞，详情查看 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fant.apache.org%2Fsecurity.html" target="_blank">https://ant.apache.org/security.html</a>。</p> 
<p><strong>1.10.11 主要变化</strong></p> 
<ul> 
 <li>修复当并行运行任务时，一个竞争条件可能导致 NullPointerExceptions 的问题</li> 
 <li>修复在使用 tar 或 zip 格式或从 zip 派生的格式读取受破损的档案时可能出现的 OutOfMemory 错误</li> 
 <li>将 AntUnit 升级至 1.4.1 版本</li> 
</ul> 
<p><strong>1.9.16 主要变化</strong></p> 
<ul> 
 <li>修复发布到 Maven 中央仓库的 ant-testutil-sources.jar 不包含任何源代码文件的错误</li> 
 <li>修复在使用 tar 或 zip 格式或从 zip 派生的格式读取受破损的档案时可能出现的 OutOfMemory 错误</li> 
 <li>将 AntUnit 升级至 1.4.1 版本</li> 
</ul>
