                                                                                            <h3 style="text-align:start">重要更新</h3> 
<p><span style="color:null"><span style="background-color:#ffffff">橙单的工作流集成，并非仅仅是对组件的引用和 API 的封装，而是提供了很多非常实用的工作流前后端业务能力。在上一个版本中，我们提供了Activiti 7.x 的支持，根据用户反馈，我们在该版本同时支持了 Flowable 6.6，</span></span><span style="color:null"><span style="background-color:#ffffff">业务功能方面两者完全等价，</span></span><span style="color:null"><span style="background-color:#ffffff">开发者用户可根据自身需求二选一。</span></span></p> 
<h3>工作流</h3> 
<ul> 
 <li>同时集成 Flowable 6.6 和 Activiti 7.x 作为工作流引擎，创建工程时可二选一。</li> 
 <li>支持部门岗位和用户岗位，配置工程时可随时选择是否支持。</li> 
 <li>支持部门、岗位、本部门领导岗位和上级领导岗位等工作流任务候选组。</li> 
 <li>根据配置选项，可自动生成用户任务监听器代码，以便于业务扩展和二次开发。</li> 
 <li>持续同步完善与工作流相关的线上文档。</li> 
</ul> 
<h3>其他更新</h3> 
<ul> 
 <li>根据用户的反馈，新增橙单低代码工具快速上手配置的线上文档。</li> 
</ul>
