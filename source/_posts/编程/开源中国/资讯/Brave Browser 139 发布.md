                                                        <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">Brave 是基于 Chromium 的一款免费开源网络浏览器，主打隐私保护，其默认设置就会自动阻止在线广告和网站跟踪。Brave Software（Brave 浏览器所属公司）于 2015 年由 Brendan Eich 和 Brian Bondy 联合创立，其中前者是 JavaScript 的创造者，同时也是 Mozilla 联合创始人。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">Brave 1.39 发布，更新内容如下：</p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>增加了对 Solana 账户创建的支持</li> 
 <li>为 Brave Wallet 的 API 帮助请求添加了 JSONSanitizer</li> 
 <li>为 Brave News 增加了允许用户搜索网站 RSS 源的功能</li> 
 <li>增加了对 blob 分区的支持</li> 
 <li>为 Sparkle 更新过程增加了最小 macOS 版本要求</li> 
 <li>[安全] 在第三方 iframes 中完全阻止了 "window.ethereum"</li> 
 <li>[安全] 修正了在Brave Wallet中显示的不正确的起源，当消费批准正在等待。</li> 
 <li>更新了 Windows Omaha 安装程序版本至 v1.3.36.113</li> 
 <li>更新了默认的 IPFS 配置值</li> 
 <li>通过优化未使用的勒索规则，减少了广告屏蔽过滤器的内存使用</li> 
 <li>从 URL 中删除了已知的 Dialog Insight 用户跟踪参数</li> 
 <li>修正了在使用 Google Meet 时打开 Brave Shields 时发生的崩溃</li> 
 <li>修正了由 OpenSSL 3.0 引起的 webpack 构建中的中断</li> 
 <li>修正了在 Windows 上安装没有管理员权限的浏览器时，启动时打开两个窗口的问题</li> 
 <li>将 Chromium 升级到 102.0.5005.61</li> 
</ul> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">更多详情可查看：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fbrave.com%2Flatest%2F" target="_blank">https://brave.com/latest/</a></p>
