                                                        <p style="text-align:left">go-zero 1.1.7 发布了。go-zero 是一个集成了各种工程实践的 web 和 rpc 框架。通过弹性设计保障了大并发服务端的稳定性，经受了充分的实战检验。go-zero 包含极简的 API 定义和生成工具 goctl，可以根据定义的 API 文件一键生成 Go, iOS, Android, Kotlin, Dart, TypeScript, JavaScript 代码，并可直接运行。</p> 
<p style="text-align:left">本次更新内容包括：</p> 
<p style="text-align:left">框架：</p> 
<ol> 
 <li>支持redis tls</li> 
 <li>prometheus按需收集，跟着配置走</li> 
 <li>移除contextx.ShrinkDeadline，跟context.WithTimeout功能重叠</li> 
</ol> 
<p style="text-align:left">goctl：<br> 1. 解决 antlr 包冲突问题<br> 2. 一些小改进</p>
