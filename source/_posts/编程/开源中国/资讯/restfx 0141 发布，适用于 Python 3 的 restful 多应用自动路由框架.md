                                                        <p>restfx 0.14.1 已经发布，这是一个适用于 Python 3 的 restful 多应用自动路由框架。</p> 
<p>此版本更新内容包括：</p> 
<ul> 
 <li>修复 session 模块中存在的问题</li> 
 <li>修复 上下文在使用时存在的问题</li> 
 <li>增强中间件功能，移除事件支持</li> 
 <li>优化 session 管理</li> 
 <li>优化 MySQLSessionProvider 使用 bytes 直接存储 session 数据</li> 
 <li>添加 应用和请求上下文支持，以在任意位置获取当前的请求信息</li> 
</ul> 
<p>详情查看：<a href="https://gitee.com/hyjiacan/restfx/releases/0.14.1">https://gitee.com/hyjiacan/restfx/releases/0.14.1</a></p>
