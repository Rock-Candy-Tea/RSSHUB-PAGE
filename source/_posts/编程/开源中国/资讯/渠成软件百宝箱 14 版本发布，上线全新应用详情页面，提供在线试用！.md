                                                                                            <p>渠成企业软件百宝箱 1.4 版本发布，上线<strong>全新应用详情</strong>页面，提供应用截图、源代码地址、Dockerfile源码地址、论坛地址和应用动态信息，让您用的放心。后续我们会针对每一款应用提供使用教程与版本更新日志。</p> 
<p>我们的官网提供<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.qucheng.com%2Fmarket-browse.html" target="_blank"><strong>渠成软件百宝箱的试用</strong></a>，您可以在应用市场选取喜欢的应用体验安装与试用，同时，我们还提供每一款应用的演示服务，让您快速体验不同的应用。</p> 
<h2>一、新增功能</h2> 
<h3>1.1 全新应用详情页</h3> 
<p><img alt="应用详情页" src="https://img.qucheng.com/www/news/release/v1.4/1.png?ver=20220831" referrerpolicy="no-referrer"></p> 
<ul> 
 <li>新增应用截图</li> 
 <li>新增应用源码地址、Dockerfile地址、论坛地址</li> 
 <li>新增应用动态信息，包括应用更新、应用使用手册等该应用相关文章</li> 
</ul> 
<h3>1.2 导航页支持自定义服务</h3> 
<p>本次更新，导航页支持更换默认背景图片、添加非渠成安装的服务，支持隐藏未启用的应用。</p> 
<p><img alt="导航页" src="https://img.qucheng.com/www/news/release/v1.4/2.png" referrerpolicy="no-referrer"></p> 
<h3>1.3 官网提供应用试用与演示</h3> 
<p>本次更新将应用市场集成到了官网，这样，您不需要安装渠成软件百宝箱，即可在线体验应用的<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.qucheng.com%2Fmarket-browse.html" target="_blank">安装与试用</a>。</p> 
<p><img alt="官网应用试用与演示" src="https://img.qucheng.com/www/news/release/v1.4/3.png" referrerpolicy="no-referrer"></p>

<h4>全新应用详情页</h4> 
<ul> 
 <li> <p>新增应用截图</p> </li>

<h4>应用市场试用与演示</h4> 
<ul>

</ul>

<h3>2.3 变更</h3> 
<ul> 
 <li> <p>应用Helm版本号改为 yyyy.mm.ddnn 的形式，方便标记Helm版本号。</p> </li> 
 <li> <p>应用升级判断条件改为应用版本，而非Helm版本。</p> </li> 
 <li> <p>支持安装历史版本的功能(测试环境)。</p> </li> 
 <li> <p>禅道应用变更</p> 
  <ul> 
   <li> <p>新增旗舰版3.3 Kubernetes镜像构建命令,解决kubernetes环境下授权失效问题.</p> </li> 
   <li> <p>新增旗舰版3.3 Kubernetes arm64架构 镜像构建命令.</p> </li> 
   <li> <p>调整安装向导检查脚本,保证安装完成后再删除 install.php 和 upgrade.php文件</p> </li> 
   <li> <p>替换MySQL服务检查命令,提升检查效率</p> </li> 
   <li> <p>更新MySQL-Client包,支持MySQL 8.0</p> </li> 
   <li> <p>Dockerfile设置默认bash,提升极端情况下的兼容性.</p> </li> 
   <li> <p>设置make命令的默认指令,当未加参数时,显示make help指令.</p> </li> 
   <li> <p>提升make help命令的兼容性,支持指令中包含数字的情况.</p> </li> 
   <li> <p>解决设置Redis保存Session时无法进行初始化安装的问题 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fquicklyon%2Fzentao-docker%2Fissues%2F1" target="_blank">#1</a></p> </li> 
  </ul> </li> 
</ul> 
<h3>2.4 Bug修复</h3> 
<ul> 
 <li>修复已安装的服务切换应用市场后，关闭、启动、卸载按钮无法正常使用的问题。</li> 
 <li>修复应用市场搜索框无法搜索包含空格的应用名称的问题。</li> 
 <li>修复应用市场搜索包含大写字母的应用时无法找到的问题。</li> 
</ul>
