                                                                                            <p>更新内容：</p> 
<p>[新增] select 组件 多选模式下 的搜索功能。<br> [新增] table 组件 even 属性, 用于开启斑马条纹背景样式。<br> [新增] dropdown 组件 placement 属性 right left right-bottom right-top left-bottom left-top 值。<br> [新增] affix 组件, 使用锚点，可以将内容固定在容器内，常用于侧边菜单导航等。<br> [新增] affix 组件 position 属性, 用于设置固定的位置, 可选值为 top 与 bottom。<br> [新增] affix 组件 offset 属性, 定位偏移量, 默认为 0。<br> [新增] affix 组件 target 属性, 定位时的参考容器, 默认为 document.body。<br> [新增] tab 组件 activeBarTransition 属性, 启用 activeBar 过渡动画。<br> [新增] cascader 组件 onlyLastLevel 属性 控制 displayValue 回显数据层级。<br> [修复] cascader 组件 children 不能为空的问题。<br> [修复] date-picker 组件 prefix-icon 前置图标无边距的问题。<br> [修复] input 组件 foucs 状态时 border 颜色为 #d2d2d2。<br> [修复] cascader 组件 children 不能为空的问题。<br> [修复] scroll 组件 按需加载时 index.css 不存在的问题。<br> [优化] icon-picker 组件 lay-input 引用使用类型错误<br> [优化] row 与 col 组件的所有属性同时兼容 number 与 string 类型。<br> [优化] table 组件 childrenColumnName 属性为非必填。<br> [优化] notice-bar 组件 color 与 background 属性的类型警告。<br> [优化] upload 组件 cutOptions 属性非必填, 在 cut 为 false 时。<br> [优化] radio 组件 name 属性非必填, 在一些应用场景下 name 没有存在的必要。<br> [优化] checkbox 组件 isIndeterminate 属性类型非必填。</p> 
<p>更多详情：http://www.layui-vue.com</p>
