                                                                                            <p>BS Redis Desktop Client v1.0.4 已经发布，Redis 桌面客户端。</p> 
<p>此版本更新内容包括：</p> 
<ol> 
 <li>把keys * 替换使用scan,</li> 
 <li>hash使用hscan,</li> 
 <li>新增zset数据支持，使用zscan，</li> 
 <li>新增按钮添加zset数据类型</li> 
 <li>list类型使用异步，</li> 
 <li>set类型使用异步和sscan <a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fbs.echosocket.com" target="_blank">软件下载地址</a></li> 
</ol>
