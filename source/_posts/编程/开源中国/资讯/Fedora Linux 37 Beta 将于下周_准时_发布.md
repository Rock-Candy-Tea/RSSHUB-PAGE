                                                                                            <p>按照 Fedora 一贯的调性，延期发布对它而言不是新闻，能“准时发布”才是闻所未闻。毕竟 Fedora 的目标是提供一个前沿而又可靠的 Linux 发行版，经常会因为阻塞性错误而跳票，所以比计划推迟一两个星期发布并不是稀罕事。</p> 
<p>当然，Fedora 也一直在提供比过去更及时的发布，而对于 Fedora Linux 37 Beta，他们似乎已经实现了<strong>准时发布</strong>的“壮举”。</p> 
<p>在本周 Fedora 37 Beta 的 Go/No-Go 会议上，团队宣布 Fedora Linux 37 Beta 在准时发布方面是 "GO" 状态，意思就是能够实现准时发布。</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-4d01568d7e5fffe8121c4d5897271d4e7dd.png" referrerpolicy="no-referrer"></p> 
<p>Fedora 37 系统级别范围的变更包括：弃用 OpenSSL 1.1 软件包、删除 i686 OpenJDK、升级至 Glibc 2.36 和 Binutils 2.38 以更新 GNU 工具链支持、Golang 1.19、停用 ARMv7 支持，以及添加 ELN-Extras 作为在 EPEL N+1 工作的新构建目标。</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-d4ce1ac47766855ed9a938e4c14f426a457.png" referrerpolicy="no-referrer"></p> 
<p>部分独立的变更包括许多软件包更新，例如 LLVM 15 编译器和 LXQt 1.1、用于新安装程序的 Anaconda Web UI 的公共预览版、对 Raspberry Pi 4 的官方支持，以及提供参考 KVM VM 磁盘镜像。</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-160e88c1f2527f9d87ab489dd3fa206f4e4.png" referrerpolicy="no-referrer"></p> 
<p>Fedora 37 Beta 将于下周二（9月13日）正式发布。或者如果你急于在这个周末尝试，已经有宣布准备好的 ISO 镜像：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdl.fedoraproject.org%2Fpub%2Falt%2Fstage%2F37_Beta-1.5%2F" target="_blank">https://dl.fedoraproject.org/pub/alt/stage/37_Beta-1.5/</a>。</p>
