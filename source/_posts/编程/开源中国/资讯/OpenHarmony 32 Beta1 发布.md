                                                                                            <p><span style="background-color:#ffffff; color:#333333">OpenHarmony 3.2 Beta1 已正式发布。</span></p> 
<p><img alt height="1423" src="https://oscimg.oschina.net/oscnet/up-686c1594c61f3f6ddc1747d11e8b73ed7bb.jpg" width="500" referrerpolicy="no-referrer"></p> 
<p><em>图源：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fmp.weixin.qq.com%2Fs%2FKW3DYwx3V34x9bdDbSxKsQ" target="_blank">华为开源</a></em></p> 
<p><span style="background-color:#ffffff; color:#40485b">当前版本在 OpenHarmony 3.1 Release 的基础上，更新支持以下能力：</span></p> 
<p style="color:#40485b; margin-left:0; margin-right:0; text-align:left"><strong>轻量系统基础能力增强</strong></p> 
<ul> 
 <li style="text-align:left">轻量LiteOS系统和Linux系统IPC归一，构建和完善了进程间通信能力，包括分布式进程间通信，为分布式业务及APP提供本地和分布式的进程间通信能力。</li> 
</ul> 
<p style="color:#40485b; margin-left:0; margin-right:0; text-align:left"><strong>标准系统基础能力增强</strong></p> 
<ul> 
 <li style="text-align:left">新增支持针对Arm 64位的系统版本编译构建能力。</li> 
 <li style="text-align:left">图形支持安全截图、安全录屏；支持嵌入其他应用界面显示能力。</li> 
 <li style="text-align:left">窗口支持快速切换桌面和恢复应用窗口、窗口唤醒屏幕、保持屏幕常亮/窗口调整屏幕亮度、安全图层等能力。</li> 
 <li style="text-align:left">方舟虚拟机支持多实例调试，以及条件断点和watch表达式能力。</li> 
 <li style="text-align:left">多模输入支持遥控器、鼠标、键盘输入设备热插拔监听能力。</li> 
 <li style="text-align:left">安全支持OpenHarmony跨设备之间的同帐号互信认证能力。</li> 
</ul> 
<p style="color:#40485b; margin-left:0; margin-right:0; text-align:left"><strong>标准系统分布式能力增强</strong></p> 
<ul> 
 <li style="text-align:left">系统服务管理支持群组校验机制能力。</li> 
</ul> 
<p style="color:#40485b; margin-left:0; margin-right:0; text-align:left"><strong>标准系统应用程序框架能力增强</strong></p> 
<ul> 
 <li style="text-align:left">ArkUI支持AbilityComponent组件将应用界面（Ability）作为控件嵌入其他应用显示；增强分栏与侧边栏组件能力，支持拖拽自动隐藏等能力。</li> 
 <li style="text-align:left">包管理支持沙箱应用的安装、卸载、查询，拆包工具支持Stage模型能力。</li> 
 <li style="text-align:left">元能力支持Stage模型支持Worker机制，新增ANR、应用主线程卡死检测等DFX能力。</li> 
</ul> 
<p style="color:#40485b; margin-left:0; margin-right:0; text-align:left"><strong>标准系统应用能力增强</strong></p> 
<ul> 
 <li style="text-align:left">图库支持基础动效，支持图片编辑保存能力，提供OEM定制开发能力。</li> 
</ul> 
<p style="color:#40485b; margin-left:0; margin-right:0; text-align:left"><strong>标准系统应用开发样例</strong></p> 
<ul> 
 <li style="text-align:left"><span style="color:#000000">新增图形GPU资源绘制、扫雷游戏、2048数字游戏等Demo样例，为开发者提供SDK使用的样例程序，方便开发者快速上手使用OpenHarmony系统基础能力。</span></li> 
</ul>
