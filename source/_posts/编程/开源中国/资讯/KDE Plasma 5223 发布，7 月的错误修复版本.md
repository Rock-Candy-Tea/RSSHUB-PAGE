                                                        <p>KDE Plasma 5.22.3 现已发布，此版本更新了来自 KDE 贡献者两周的新翻译和修复，是 7 月份的错误修复版本。</p> 
<p><img alt height="477" src="https://oscimg.oschina.net/oscnet/up-497480cf7416d228283d696ed671d2130b7.png" width="800" referrerpolicy="no-referrer"></p> 
<p><strong>主要更新内容</strong></p> 
<ul> 
 <li>Info Center 
  <ul> 
   <li>删除单元格尺寸设置</li> 
   <li>解除刷新按钮</li> 
   <li>禁用 NIC kcm 中的布局孪生功能</li> 
  </ul> </li> 
 <li>kwayland-server 
  <ul> 
   <li>在按下按钮时不再移动键盘焦点</li> 
  </ul> </li> 
 <li>KWin 
  <ul> 
   <li>Platforms/drm：抛弃 gbm_format_get_name()</li> 
   <li>Xdgshellclient：修复触摸和 CSD 的 moveresize 问题</li> 
   <li>Input：修复触摸输入在快速磁贴上的故障</li> 
   <li>Platforms/drm：drmModeGetConnector 失败时不再崩溃</li> 
   <li>Wayland：实施活动窗口规则</li> 
  </ul> </li> 
 <li>Plasma Desktop 
  <ul> 
   <li>Task Manager：异步加载专辑艺术背景</li> 
   <li>kimpanel：在 Wayland 上使用窗口标志 ToolTip 以避免从输入法中获取焦点</li> 
   <li>keyboard layout applet： 修复启动时自动打开fullRepresentation的问题</li> 
  </ul> </li> 
 <li>Plasma Systemmonitor 
  <ul> 
   <li>在获取选择数据时传递父索引</li> 
  </ul> </li> 
 <li>xdg-desktop-portal-kde 
  <ul> 
   <li>Background portal：避免允许应用程序在后台运行时崩溃</li> 
  </ul> </li> 
</ul> 
<p>详情请查看<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fkde.org%2Fannouncements%2Fchangelogs%2Fplasma%2F5%2F5.22.2-5.22.3%2F" target="_blank">更新公告</a>。</p>
