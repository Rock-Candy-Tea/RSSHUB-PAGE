                                                        <p>Red Hat Enterprise Linux (RHEL) 8.4 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.redhat.com%2Fen%2Fblog%2Fred-hat-enterprise-linux-84-beta-now-available" target="_blank">已发布</a>。根据发布公告的介绍，此版本主要是增强了在开放式混合云方面的功能。</p> 
<ul> 
 <li>通过改进跨云和本地部署的订阅可移植性，以及改进跨规模混合云区域订阅部署的可视性，确保混合云的采用更加简单、灵活。</li> 
 <li>通过 Red Hat Insights 服务提供的新功能扩展了主动式管理，包括威胁情报和安全合规性报告以及订阅管理，在部署 RHEL 8.4 Beta 的任何地方都可使用。</li> 
 <li>通过 RHEL Web 控制台、Grafana 仪表板和 Red Hat Performance Co-Pilot 带来更强大的数据可视化功能，可以创建更简单的复杂系统性能数据视图。</li> 
 <li>通过新的 python39 模块支持 Python 3.9，Redis 6 和 PostgreSQL 13 模块也已可用</li> 
 <li>支持英特尔 Gen12 Tiger Lake 显卡</li> 
 <li>支持 SMB Direct</li> 
 <li>……</li> 
</ul> 
<p>除了上述的变化，RHEL 8.4 通过 Application Streams 添加了新的精选编程语言和开发工具，与 Red Hat Ansible 自动化平台进行了更深层次的集成，并增强了 Red Hat Performance Co-Pilot 功能，以简化实时管理和可视化。更多内容查看 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Faccess.redhat.com%2Fdocumentation%2Fen-us%2Fred_hat_enterprise_linux%2F8-beta%2Fhtml%2F8.4_release_notes%2Findex" target="_blank">release notes</a>。</p> 
<p>下载地址：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Faccess.redhat.com%2Fproducts%2Fred-hat-enterprise-linux%2Fbeta" target="_blank">https://access.redhat.com/products/red-hat-enterprise-linux/beta</a></p>
