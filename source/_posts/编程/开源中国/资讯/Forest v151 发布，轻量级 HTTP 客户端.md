                                                        <p>Forest v1.5.1 已经发布，这是一个轻量级 HTTP 客户端。</p> 
<p>此版本更新内容包括：</p> 
<h4>新增的特性：</h4> 
<ul> 
 <li>feat: #I3E9T4</li> 
 <li>feat: #I3SR0L</li> 
 <li>feat: #I3P1R0</li> 
 <li>feat: #I3SUSL</li> 
 <li>feat: #I3E63M</li> 
</ul> 
<h4>解决的问题：</h4> 
<ul> 
 <li>fix: #I3IUKC</li> 
 <li>fix: #I3RBR2</li> 
 <li>fix：去掉protobuf decoder 根据返回类型序列化</li> 
 <li>fix: #I3E1MF</li> 
 <li>fix: #I3AJSD</li> 
 <li>fix: #I3IUKD</li> 
 <li>fix: #I3A54I</li> 
 <li>fix: 单向SSL不填protocol时报空指针</li> 
 <li>fix: statusCode空指针</li> 
 <li>fix: status在非网络错误情况下为404</li> 
 <li>fix: 修改方法名拼写错误 getDateType -> getDataType</li> 
</ul> 
<h4>其他改动：</h4> 
<ul> 
 <li>update: 修改gson版本 -> 2.8.6</li> 
 <li>refactor: 指定统一的spring-boot版本</li> 
 <li>update: 去掉QLExpress</li> 
 <li>add: @URLEncode注解</li> 
 <li>add: URLEncodeLifeCycle类</li> 
</ul>
