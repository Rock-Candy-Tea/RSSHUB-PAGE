                                                        <p>Tails 4.20 正式发布，更新内容如下：</p> 
<h3>更改和更新：</h3> 
<ul> 
 <li>将 OnionShare 从 1.3.2 更新到 2.2：这个更新增加了一个功能，即托管一个可以从 Tor 洋葱服务访问的网站；</li> 
 <li>将 KeePassXC 从 2.5.4 更新到 2.6.2：这一更新带来了界面的重新设计；</li> 
 <li>将 Tor Browser 更新到 10.5.2；</li> 
 <li>更新 Thunderbird 到 78.11.0；</li> 
 <li>更新 Tor 到 0.4.5.9；</li> 
 <li>将 Linux 内核更新到 5.10.46，这会改善对较新硬件（显卡、Wi-Fi 等）的支持；</li> 
</ul> 
<h3>修复的问题：</h3> 
<p><strong>自动升级</strong></p> 
<ul> 
 <li>使升级的下载和错误的处理更加强大；</li> 
 <li>在检查可用升级失败时显示错误信息；</li> 
</ul> 
<p><strong>Tails 安装程序</strong></p> 
<ul> 
 <li>使得重新安装按钮的显示更加显眼；</li> 
 <li>使安装和升级在 U 盘被移除后不可用；</li> 
</ul>
