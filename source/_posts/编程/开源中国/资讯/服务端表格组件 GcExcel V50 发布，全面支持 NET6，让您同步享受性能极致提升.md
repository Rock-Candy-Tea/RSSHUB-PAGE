                                                                                            <p style="color:#555555; margin-left:0; margin-right:0; text-align:start">2020年，随着 .NET5 和 .NET Core的相继推出，.NET这款主流编程语言再次崭露头角，吸引了众多互联网巨头的关注。今年，.NET6继往开来，凭借着Blazor升级、跨平台UI方案MAUI、CLR性能优化等利器，为当下微服务&云原生的新互联网时代描绘出更加宏伟的蓝图！</p> 
<p style="color:#555555; margin-left:0; margin-right:0; text-align:start">继2020年 V3.0 发布之日起，葡萄城服务端表格组件 GcExcel 便紧随时代的步伐，不但与微软保持紧密的技术合作，还在第一时间对产品版本更新迭代，让开发者不再为新技术而困扰，不再为系统兼容性而担忧，可以同步享受到性能的极致提升。</p> 
<p style="color:#555555; margin-left:0; margin-right:0; text-align:start">近日，GcExcel正式迎来其V5.0 的发布更新，从该版本开始GcExcel的组件功能将完全支持 .NET 6 平台，开发者在该平台下使用数据导入、动态数组公式和数据绑定等API时不再需要额外的步骤或任何依赖项。</p> 
<p style="color:#555555; margin-left:0; margin-right:0; text-align:start">下面我们将详细介绍 GcExcel V5.0 的新特性内容，在此之前，如果您已经在使用我们的GcExcel产品，想进一步了解新功能，请移步前往葡萄城官网<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.grapecity.com.cn%2Fdeveloper%2Fgrapecitydocuments%2Fexcel-java" target="_blank">下载最新版产品安装包</a>，以便同步体验。</p> 
<p style="color:#555555; margin-left:0; margin-right:0; text-align:start">以下是本次发布的新特性内容：</p> 
<p style="color:#555555; margin-left:0; margin-right:0; text-align:start"> </p> 
<h3 style="margin-left:0; margin-right:0; text-align:start">1. 数据导入（Import Data）</h3> 
<p style="color:#555555; margin-left:0; margin-right:0; text-align:start">使用数据导入（ImportData）接口，当您需要从 Excel 文件中提取数据时，不再需要加载整个 Excel 文件。借助此功能可以更快、更有效地同时从多个Excel文件中导入数据。</p> 
<p><img alt src="https://www.grapecity.com.cn/images/developer/grapecitydocuments/whatsnew/v5.0/%E5%9B%BE%E7%89%872.jpg" referrerpolicy="no-referrer"></p> 
<h3 style="margin-left:0; margin-right:0; text-align:start">2. 动态数组公式接口（Dynamic Array Formula）</h3> 
<p style="color:#555555; margin-left:0; margin-right:0; text-align:start">Microsoft Excel在2018年推出了动态数组公式的概念：可基于一个公式将多个结果返回到一个单元格区域中，为此动态数组公式也被称为溢出范围功能。动态数组公式通过删除重复项、对列表进行排序、输出筛选的数据范围等来创建唯一值的列表，以确保现有函数可以利用相同的溢出范围功能。</p> 
<p style="color:#555555; margin-left:0; margin-right:0; text-align:start">GcExcel 在本次更新中也增加了动态数组公式接口，如下图所示。</p> 
<p><img alt src="https://www.grapecity.com.cn/images/developer/grapecitydocuments/whatsnew/v5.0/%E5%9B%BE%E7%89%873.jpg" referrerpolicy="no-referrer"></p> 
<p> </p> 
<h3 style="margin-left:0; margin-right:0; text-align:start">3. 上下文批注（Threaded Comments）</h3> 
<p style="color:#555555; margin-left:0; margin-right:0; text-align:start">在Excel中，上下文批注被广泛应用于组内讨论和响应，该功能支持回复内联、嵌套注释，以及将多个批注串联在一起的对话样式，该功能在GcExcel V5.0中被正式引入。</p> 
<p><br> <img alt src="https://www.grapecity.com.cn/images/developer/grapecitydocuments/whatsnew/v5.0/%E5%9B%BE%E7%89%874.jpg" referrerpolicy="no-referrer"></p> 
<h3 style="margin-left:0; margin-right:0; text-align:start">4. 区域快照（Linked Picture）</h3> 
<p style="color:#555555; margin-left:0; margin-right:0; text-align:start">借助区域快照，可将不同大小的图表、数据表、条件格式等全部组合在一个工作表中，不仅可以适应大小，而且当数据更改时，区域快照中的数据也会更改。</p> 
<p><br> <img alt src="https://www.grapecity.com.cn/images/developer/grapecitydocuments/whatsnew/v5.0/%E5%9B%BE%E7%89%876.gif" referrerpolicy="no-referrer"></p> 
<h3 style="margin-left:0; margin-right:0; text-align:start">5. 工作簿视图（Workbook Views）</h3> 
<p style="color:#555555; margin-left:0; margin-right:0; text-align:start">标准的Microsoft Excel视图可以根据要分析的内容进行更改。例如，"页面布局"视图可帮助您可视化文档在打印时的外观，而"分页符"视图则显示分页符的显示位置。</p> 
<p style="color:#555555; margin-left:0; margin-right:0; text-align:start">如果打开多个文档时需要显示公共视图，GcExcel允许以编程方式对其进行控制。</p> 
<p style="color:#555555; margin-left:0px; margin-right:0px; text-align:center !important"><img src="https://www.grapecity.com.cn/images/developer/grapecitydocuments/whatsnew/v5.0/%E5%9B%BE%E7%89%878.jpg" referrerpolicy="no-referrer"></p> 
<h3 style="margin-left:0; margin-right:0; text-align:start">6. GETPIVOTDATA 函数</h3> 
<p style="color:#555555; margin-left:0; margin-right:0; text-align:start">虽然数据透视表可以显示大量数据，但用户可能只想根据数据透视表检索特定的数据结构，而非单元格引用，这就可以利用GETPIVOTDATA 函数来完成。</p> 
<h3 style="margin-left:0; margin-right:0; text-align:start">7. 与 SpreadJS 兼容相关：支持表格expandBoundRows 接口</h3> 
<p style="color:#555555; margin-left:0; margin-right:0; text-align:start">GcExcel支持 SpreadJS 的API 来扩展表中的绑定行。</p> 
<p style="color:#555555; margin-left:0; margin-right:0; text-align:start">如果将该属性设置为 true，则在更改数据时将添加或删除整行以调整行计数。如果设置为 false，则应用 Microsoft Excel 策略，即不允许在数据源发生更改时添加或删除行计数。</p> 
<h3 style="margin-left:0; margin-right:0; text-align:start">8. 直连打印机打印（仅针对 .NET）</h3> 
<p style="color:#555555; margin-left:0; margin-right:0; text-align:start">到目前为止，在 .NET 标准应用中工作的跨平台 API 在使用"打印"命令时都面临着挑战。这一挑战主要是由于打印 API 可用性带来的限制。在当前版本中，GcExcel 添加了一个新的打印 API，可以实现将 Excel 文档直接打印到 Windows 平台的物理打印机上。</p> 
<p style="color:#555555; margin-left:0; margin-right:0; text-align:start">GcExcel的新接口可以调用物理打印机来打印工作簿或工作表，不仅支持打印，还可以提前设置其他选项。</p>
