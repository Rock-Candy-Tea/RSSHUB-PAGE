 <h2 style="text-align:start">v0.6.2<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Faf.newbe.pro%2Fzh-Hans%2Fdocs%2F99-1-Whats-New%23062" target="_blank">#</a></h2> 
 <h3 style="text-align:start">🌟 支持 GoogleDrive 和 OneDrive 来同步您的数据<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Faf.newbe.pro%2Fzh-Hans%2Fdocs%2F99-1-Whats-New%23-%25E6%2594%25AF%25E6%258C%2581-googledrive-%25E5%2592%258C-onedrive-%25E6%259D%25A5%25E5%2590%258C%25E6%25AD%25A5%25E6%2582%25A8%25E7%259A%2584%25E6%2595%25B0%25E6%258D%25AE" target="_blank">#</a></h3> 
 <p style="text-align:start">现在，除了使用 Newbe API 之外，您还可以通过 GoogleDrive 或者 OneDrive 在多个设备中同步您的数据。</p> 
 <p style="text-align:start">您可以在控制面板中开启对应的存储方式。</p> 
 <p style="text-align:start"><img alt="Cloud Sync in control panel" height="1012" src="https://oscimg.oschina.net/oscnet/up-0786ef59ffbafdfbd8b1c614633886ff222.png" width="1880" referrerpolicy="no-referrer"></p> 
 <p style="text-align:start">开启之后，您便可以通过 Manager UI 右上角的按钮来同步您的数据。</p> 
 <p style="text-align:start"><img alt="Login to sync" height="285" src="https://oscimg.oschina.net/oscnet/up-07070cb916e45a968ec0e37b900e77af95f.png" width="1247" referrerpolicy="no-referrer"></p> 
 <p style="text-align:start">每当同步成功时，您就可以通过 Manager UI 中得到提示。</p> 
 <p style="text-align:start"><img alt="sync success" height="234" src="https://oscimg.oschina.net/oscnet/up-2e0abae71513d80c72ad81e3c21697428d8.png" width="698" referrerpolicy="no-referrer"></p> 
 <h3 style="text-align:start">⭐ 搜索历史自动提示<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Faf.newbe.pro%2Fzh-Hans%2Fdocs%2F99-1-Whats-New%23-%25E6%2590%259C%25E7%25B4%25A2%25E5%258E%2586%25E5%258F%25B2%25E8%2587%25AA%25E5%258A%25A8%25E6%258F%2590%25E7%25A4%25BA" target="_blank">#</a></h3> 
 <p style="text-align:start">现在，当您成功搜索并点击搜索结果时，当前搜索文本将会被记录下来，以便您下次搜索时进行提示。加强您对常用内容的可访问性。</p> 
 <h3 style="text-align:start">🛠 控制面板 UI 改进<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Faf.newbe.pro%2Fzh-Hans%2Fdocs%2F99-1-Whats-New%23-%25E6%258E%25A7%25E5%2588%25B6%25E9%259D%25A2%25E6%259D%25BF-ui-%25E6%2594%25B9%25E8%25BF%259B" target="_blank">#</a></h3> 
 <p style="text-align:start">控制面板 UI 迎来了一次小幅度的改版，以便您可以轻松的理解每个功能。</p> 
 <p style="text-align:start">但功能没有被启用时，将会展示该功能相关的帮助信息，以便您可以了解该功能的作用。</p> 
 <p style="text-align:start"><img alt="control panel all disable" height="1196" src="https://oscimg.oschina.net/oscnet/up-f1fa013b9603822807829d012f54d7faaab.png" width="1838" referrerpolicy="no-referrer"></p> 
 <p style="text-align:start">当您启用特定的功能时，面板将会被默认折叠以减少空间的占用。当然，您还是可以点击标题以展开面板，查看其中的详细设置。</p> 
 <p style="text-align:start"><img alt="feature enabled" height="1004" src="https://oscimg.oschina.net/oscnet/up-0e1032c23f48aa18baa9486b382ccd04be8.png" width="1870" referrerpolicy="no-referrer"></p> 
 <h3 style="text-align:start">🐞 Bug 修复<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Faf.newbe.pro%2Fzh-Hans%2Fdocs%2F99-1-Whats-New%23-bug-%25E4%25BF%25AE%25E5%25A4%258D" target="_blank">#</a></h3> 
 <ul> 
  <li>修复了 application insight 没有启用时，可能出现程序错误的问题。</li> 
 </ul> 
 <h3>软件简介</h3>

  <p>Amazing Favorites 是一款采用 Blazor 技术开发的 Chromium 内核的浏览器插件，因此，其可以被安装到 Chrome / Microsoft Edge / 360 极速浏览器 / Vivaldi / 搜狗高速浏览器 等浏览器之上。<br> 它可以帮助使用者高效的管理和搜索自己的收藏夹，为此增加了一些增强功能：</p> 
  <ul> 
   <li>支持为链接标记 tag，从而为通过 tag 为链接进行交叉分组</li> 
   <li>支持快捷键操作</li> 
  </ul>
