                                                        <p>DBeaver 社区版 21.1 已发布，主要更新内容：</p> 
<ul> 
 <li>优化连接恢复 (Connection recovery) 功能，修复 SSH tunnel 无限重新开启的问题</li> 
 <li>数据传输 (Data transfer) 优化： 
  <ul> 
   <li>修复 Insert/replace 方法的配置问题</li> 
   <li>优化跨数据库列数据类型映射</li> 
  </ul> </li> 
 <li>项目设置：添加修复损坏的文件夹功能</li> 
 <li>更新旧的状态图标</li> 
 <li>减少应用启动时间</li> 
 <li>SQL Editor： 
  <ul> 
   <li>修复结果选项卡折叠的问题</li> 
   <li>修复 Dummy NullPointerException 和 StackOverflowError 错误</li> 
  </ul> </li> 
 <li>Data editor：LOB 查看器现在显示有关数据加载错误的对话框</li> 
 <li>CLI：修复 reuseWorkspace 参数处理程序</li> 
 <li>SQL Server： 
  <ul> 
   <li>添加扩展属性编辑器</li> 
  </ul> </li> 
 <li>Oracle：支持配置同义词搜索</li> 
 <li>PostgreSQL： 
  <ul> 
   <li>添加修改用户密码功能</li> 
   <li>修复 Domain arrays 支持问题</li> 
   <li>Role DDL 现已包含权限信息</li> 
   <li>修复 Sequences DDL 问题</li> 
   <li>修复 VARBIT 数据类型支持问题</li> 
   <li>修复默认数据库切换的问题</li> 
  </ul> </li> 
 <li>MySQL：添加重命名用户功能</li> 
 <li>Snowflake： 针对区分大小写的 schema 名称修复了元数据读取问题</li> 
 <li>BigQuery：升级驱动程序</li> 
 <li>捆绑的 Java 版本已升级至 11.0.11</li> 
 <li>修复许多小的 UI 错误</li> 
</ul>

<p>DBeaver 是一款自由开源的多平台通用数据库工具，支持所有流行数据库，如 MySQL, PostgreSQL, SQLite, Oracle, DB2, SQL Server, Sybase, MS Access, Teradata, Firebird, Apache Hive, Phoenix 和 Presto 等。</p>
