                                                                                            <p>更新内容：</p> 
<p>[修复] uuid 在 typescript 中的兼容性问题。<br> [修复] tab 组件数据刷新后不显示标题的问题。<br> [修复] countUp 组件 useGrouping 属性异常。<br> [修复] scroll 组件无法监听 subtree 的节点变化。<br> [优化] table 组件 window.resize 动态计算 table 滚动条占位宽度。<br> [优化] table 组件 page 参数 total 值为 0 时, 隐藏分页操作。<br> [优化] select 组件多选模式下边框被遮挡的问题。<br> [优化] table 组件 tool 布局, 使用 flex 撑满。</p> 
<p>其他内容：</p> 
<p>本次更新，伴随 Admin 1.0.0 的正式开源。</p> 
<p>项目仓库：<a href="https://gitee.com/layui-vue/layui-vue-admin">https://gitee.com/layui-vue/layui-vue-admin</a></p>
