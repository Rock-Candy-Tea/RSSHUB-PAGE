<div class="content">
                                                                                            <p style="color:#333333; margin-left:0; margin-right:0; text-align:left">bsFormBuilder v1.0.5 发布了，bsFormBuilder 是一个基于 Bootstrap  v4.x 表单构建工具。bsFormBuilder 最大的特点是组件和组件的属性配置自由扩展，可以导出 html 和 json。</p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><strong>bsFormBuilder 的特点：</strong></p> 
<ul style="list-style-type:disc; margin-left:0; margin-right:0"> 
 <li>1、基于 Bootstrap (v4.x) + JQuery，简单易用</li> 
 <li>2、拖动的 html 组件，支持通过 Json 自定义扩展</li> 
 <li>3、组件的属性面板，支持通过 Json 自定义扩展</li> 
 <li>4、支持导出 html 和 json，自己通过 json 来构建 UI 页面</li> 
 <li>5、支持导入 json 到 bsFormBuilder，进行二次编辑</li> 
 <li>6、丰富的 API，方便二次开发和扩展</li> 
 <li>7、支持 “模板” 功能，可以选择已有模板进行二次开发</li> 
 <li>8、内置轻量的 html 渲染引擎，速度极快，极好用～～</li> 
</ul> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><img alt="bsFormBuilder" src="https://gitee.com/fuhai/bs-form-builder/raw/master/assets/images/bsformbuilder.png" referrerpolicy="no-referrer"></p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left"> </p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><strong>bsFormBuilder v1.0.4 更新内容如下：</strong></p> 
<ul style="list-style-type:disc; margin-left:0; margin-right:0"> 
 <li>新增：添加自定义 "不允许拖动" 组件的配置支持</li> 
 <li>新增：自定义 UI 组件的支持，自己来自定义 bsFormBuilder 整体布局</li> 
 <li>新增：options 初始化配置新增 onDataUpdated 的监听，用于监听每个组件的数据变化</li> 
 <li>新增：自定义拖动组件分类分组的支持</li> 
 <li>新增：自定义拖动组件分类分组支持自定义 placeholder 的配置</li> 
 <li>新增：自定义 container 的 sortable group 名称配置</li> 
 <li>新增：component 新增 disableTools 配置的支持，用于禁止删除和复制</li> 
 <li>新增：component 支持定义 "不允许在属性面板设置" 的属性 "none"</li> 
 <li>新增：component 支持自定义属性私有的面板渲染方法 onRenderPropsPanel()</li> 
 <li>新增：component 允许不被从左侧 "拖拽" 到容器来设计的特性，但可以通过 datas 来进行初始化</li> 
 <li>新增：options 配置 components 可以通过配置 url 来读取后台数据</li> 
 <li>新增：options 配置 components 可以配置为异步方法</li> 
 <li>新增：options 配置 datas 可以通过配置 url 来读取后台数据</li> 
 <li>新增：options 配置 datas 可以配置为异步方法</li> 
 <li>新增：options.customRender 来配置自己的渲染方法，此项也可以配置为 url 地址，让后台来渲染</li> 
 <li>优化：删除导出的 json 内容的 elementId</li> 
 <li>优化：重构 bsFormBuilder 布局组件的 class 名称</li> 
 <li>修复：自定义组件时，在某些极端情况下，父级的 dataIndex 获取不正确的问题</li> 
 <li>修复：deepCopy() 方法的 withNewElementIdAndId 参数不起作用的问题</li> 
 <li>修复：_mergeProps() 方法合并的结果不正确的问题</li> 
</ul>
