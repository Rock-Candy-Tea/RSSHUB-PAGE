                                                                                            <p>Deluge 是一个 BitTorrent 客户端，它有各种可用的用户界面，例如 GTK-UI、Web-UI 和 Console-UI。它以 libtorrent 为核心来处理 BitTorrent 协议。</p> 
<p>近日 Deluge 连发 2.0.4 和 2.0.5 版本，距离其 2.0.3 版本更新已时隔两年半（2019-06-12） 。</p> 
<h3>Gtk UI</h3> 
<ul> 
 <li>增加对 GTK UI 的 torrent URL 的检测</li> 
 <li>修复启用时 piecesbar 崩溃的问题</li> 
 <li>删除统计中的 num_blocks_cache_hits</li> 
 <li>修复空剪贴板时未处理的错误</li> 
 <li>添加 torrentdetails 标签的位置菜单</li> 
 <li>隐藏控制台中的 pygame 社区横幅</li> 
 <li>修复 None 类型的 cmp 函数</li> 
 <li>修复加载字符串中带有双引号的配置</li> 
 <li>修复状态选项卡的下载速度和上传速度</li> 
</ul> 
<h3><strong>Web UI</strong></h3> 
<ul> 
 <li>修复了 js minifying 错误导致的 WebUI 屏幕空白</li> 
 <li>处理 torrent 添加失败的问题</li> 
 <li>添加菜单选项来复制 magnet URI</li> 
 <li>修复 torrent 文件中的 md5sums 破坏文件列表的问题</li> 
</ul> 
<h3>Console UI</h3> 
<ul> 
 <li>修复允许在 Windows 上使用 windows-curses 的问题</li> 
 <li>修复主机列表状态查询错误</li> 
 <li>修复设置 "Skip" 优先级</li> 
</ul> 
<h3><strong>Core</strong></h3> 
<ul> 
 <li>修复允许启用任何 Python 版本插件的问题</li> 
 <li>导出 torrent get_magnet_uri 方法</li> 
 <li>修复 Python 3.8 的兼容性</li> 
 <li>修复加载字符串中带有双引号的配置问题</li> 
 <li>……</li> 
</ul> 
<h3>通知插件</h3> 
<ul> 
 <li>修复带有状态名称的电子邮件 KeyError</li> 
 <li>修复 Python 3 上未处理的 TypeErrors</li> 
</ul> 
<h3>执行插件</h3> 
<ul> 
 <li>修复在 Windows 上运行失败的问题</li> 
</ul>
