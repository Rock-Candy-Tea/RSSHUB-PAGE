                                                                                            <p>GNOME 43 首个 Alpha<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fmail.gnome.org%2Farchives%2Fdevel-announce-list%2F2022-July%2Fmsg00002.html" target="_blank"> 已发布</a>，正式版计划于 9 月推出。</p> 
<p><strong>重要变化</strong></p> 
<ul> 
 <li>对 VoIP 应用程序 GNOME Calls 的 UI 进行优化，以及其他功能方面的改进</li> 
 <li>GNOME Epiphany 作为 GNOME 的 Web 浏览器，新版本重构了 WebExtensions，修复了下载功能，改进了阅读器模式样式表，改进了 Web 应用程序支持，以及其他多项改进</li> 
 <li>用于虚拟化的 GNOME Boxes 现在支持系统配色方案，将其开发分支名称更改为"main"，以及其他一些变</li> 
 <li>GNOME Builder 已移植到 GTK 4</li> 
 <li>GNOME Calendar 在主窗口中增加了一个侧边栏，改进了事件的样式，并对日历 UI 进行了其他改进</li> 
 <li>系统日志查看器 GNOME Logs 现已支持直接打开日志文件，以及其他小的功能增强</li> 
 <li>GNOME Music 恢复了对音频文件的"shuffle"支持</li> 
 <li>GNOME Online Accounts 放弃支持 Foursquare、Facebook 和 Flickr 作为登录帐号</li> 
 <li>GNOME Software 改进了通知功能，优化样式，支持 Web App</li> 
 <li>GNOME 文本编辑器现在使用来自 libadwaita 的新对话框，支持打开本地和远程 STDIN 流，改进拼写检查。如果旧 gnome-text-editor 实例崩溃了，还支持检测，以及其他改进</li> 
 <li>GNOME Weather 优化每日视图中的温度计部件</li> 
 <li>Sysprof 已移植到 GTK4</li> 
 <li>完善 libadwaita</li> 
 <li>改进 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.phoronix.com%2Fscan.php%3Fpage%3Dnews_item%26px%3DGNOME-Shell-Mutter-43-Alpha" target="_blank">GNOME Shell 和 Mutter</a></li> 
</ul>
