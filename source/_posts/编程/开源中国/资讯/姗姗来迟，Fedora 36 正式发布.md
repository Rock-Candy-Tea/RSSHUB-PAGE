                                                        <p>Fedora 36 原定于 4 月发布，如今推迟了半个多月时间，该版本终于正式发布了。</p> 
<p><img alt height="296" src="https://oscimg.oschina.net/oscnet/up-6efca1d877a02b6f846a34a027f5ca35f28.jpg" width="700" referrerpolicy="no-referrer"></p> 
<h3>桌面改进</h3> 
<p>Fedora Workstation 采用了最新的 GNOME 版本 —— GNOME 42。GNOME 42 带来了很多改进，许多应用程序都被移植到了 GTK 4，以改善样式和性能；包括一个全局黑暗风格的 UI 设置；一个重新设计的屏幕截图工具；GNOME 42 中还有两个新的应用程序：Text Editor 和 Console。Text Editor 是新的默认文本编辑器，而 Console 则在软件仓库中可用。</p> 
<p>如果你使用 NVIDIA 的专有图形驱动，你的桌面会话现在将默认使用 Wayland。</p> 
<h3>Sysadmin 改进</h3> 
<p>Fedora Linux 36 包含最新版本的 Ansible。Ansible 5 将 "engine" 分割成 ansible-core 包和 collections 包。这使得维护更容易，并允许你只下载你需要的集合。</p> 
<p>从 Fedora Server 36 开始，Cockpit 提供了一个模块用于配置和持续管理 NFS 和 Samba 共享。这允许管理员通过用于配置其他服务器属性的 Cockpit 网页界面来管理网络文件共享。</p> 
<h3>其他更新</h3> 
<p>Fedora Silverblue 和 Kinoite 现在将 /var 放在一个单独的子卷上，用于新的安装，这使得处理动态数据的快照更容易独立于系统快照来管理。</p> 
<p>Podman 4.0 在 Fedora Linux 36 中首次全面发布，Podman 4.0 有大量的变化和一个全新的网络栈。它也带来了向后兼容的 API 变化。</p>
