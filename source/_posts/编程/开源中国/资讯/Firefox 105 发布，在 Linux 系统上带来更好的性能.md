                                                                                            <p>Mozilla 于今天正式发布了 Firefox 浏览器的 105 版本，此次更新内容如下：</p> 
<h3>新功能</h3> 
<ul> 
 <li>增加了一个在打印预览对话框中只打印当前页面的选项</li> 
 <li>Firefox 现在支持第三方上下文中的 Partitioned Service Workers</li> 
 <li>在 Windows 上的滑动导航现在已经启用（在触摸板上的两个手指向左或向右滑动来执行）</li> 
 <li>Firefox 现在符合 User Timing L3 规范，它为 <code>performance.mark</code> 和 <code>performance.measure</code> 方法增加了额外的可选参数，以提供自定义的开始时间、结束时间、持续时间和附加细节。</li> 
 <li>在大型列表中搜索单个项目的速度现在提高了 2 倍。</li> 
</ul> 
<h3>修复</h3> 
<ul> 
 <li>在 Windows 上的稳定性得到了明显的改善，因为 Firefox 可以更好地处理低内存情况</li> 
 <li>在 macOS 上的触摸板滚动更易于使用</li> 
 <li>在 Linux 上，Firefox 不太可能出现内存耗尽的情况，当内存不足时，对系统其他部分的执行效率更高</li> 
 <li>各种安全问题的修复</li> 
</ul> 
<h3>Web 平台</h3> 
<ul> 
 <li>支持 Offscreen Canvas DOM API，具有完整的上下文和字体支持</li> 
</ul>
