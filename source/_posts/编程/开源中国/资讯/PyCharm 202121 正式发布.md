                                                                                            <p>PyCharm 2021.2 的第一个小版本更新包含了不同功能方面的改进和错误修复。</p> 
<h3>利用 Feature Trainer 学习 PyCharm：新的 VCS 课程</h3> 
<p>通过 IDE Feature Trainer 插件，你可以进一步学习有关 PyCharm 中的 Git 集成。新的课程教你如何克隆仓库、提交修改、预览和浏览项目历史、管理分支和变更列表、执行交互式变基，以及用 Git Blame 进行注释。要开始学习相关课程，从主菜单中选择 “帮助|学习 IDE Features“，在可用课程列表中展开 Git 模块。</p> 
<h3>更快地为 Python 3 创建环境</h3> 
<p>从 2021.2 版开始，PyCharm 使用 virtualenv 而不是 venv 来创建 Python 3 环境。根据 virtualenv 的文档，在大多数情况下都能更快速地创建环境。</p> 
<h3>对 Vue 的改进</h3> 
<p>PyCharm 2021.2.1 包括对 Vue 支持的若干改进。IDE 现在支持最终版本的 script setup <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fvuejs%2Frfcs%2Fblob%2Fmaster%2Factive-rfcs%2F0040-script-setup.md" target="_blank">RFC</a>。</p> 
<h3>PyCharm 2021.2.1 的其他改进：</h3> 
<ul> 
 <li>Docker Compose：当启用 Docker Compose V2 实验性功能时，现在可以创建一个 Python 解释器 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FPY-49305" target="_blank">[PY-49305]</a></li> 
 <li>Flask：调试器现在可以在 Flask 项目中正确工作 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FPY-49984" target="_blank">[PY-49984]</a></li> 
 <li>解析器：不再出现 "Statement expected.found Py:DEDENT" 误报提示 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FPY-49970" target="_blank">[PY-49970]</a></li> 
 <li>HTML 预览：你可以在内置服务器和 HTML 预览的“保存时重载”和“更改时重载”模式之间切换 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FWEB-51695" target="_blank">[web-51695]</a></li> 
 <li>调试器：可以对 Windows 10 x64 的 Python 3.9 进行调试 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FPY-45084" target="_blank">[PY-45084]</a></li> 
 <li>调试工具窗口：Update Running Application 按钮可用 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-274540" target="_blank">[IDEA-274540]</a></li> 
 <li>运行/调试配置：当你点击运行按钮时，IDE 会运行在组合框中选择的配置 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-244454" target="_blank">[idea-244454]</a></li> 
 <li>终端：终端中的几个快捷方式问题得到了修复</li> 
 <li>快捷键：Ctrl+Shift+←/→ 快捷键现在可以正确地在光标之前或之后扩展文本选择 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-267294" target="_blank">[idea-267294]</a></li> 
</ul>
