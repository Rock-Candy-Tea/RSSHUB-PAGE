                                                        <p>Chrome OS 96 近日正式推出，该版本在专注于增强相机和其他常规应用的升级。作为过渡到每四周更新一次的一部分，Chrome OS 95 版被跳过。</p> 
<h3>相机应用</h3> 
<p>在这个版本中，Google 特别关注对 Chrome OS 相机应用的改进。拥有后置摄像头的 Chrome OS 设备将受益于一个新增的内置文件扫描仪，它可以将捕获的文件转换为 PDF 和 JPEG 格式。</p> 
<p>相机应用中将有一个新的 "扫描" 选项，该功能将自动检测文档边缘，用户也可以手动调整。这项功能此前已经在 Chrome OS 94 出现，如今正式大规模推送。</p> 
<p>明年，Chrome OS 相机应用还将推出一个 GIF 制作器功能，可以让你录制长达 5 秒钟的视频并转换为 GIF 图片。</p> 
<p><img alt="https://i1.wp.com/9to5google.com/wp-content/uploads/sites/4/2021/11/chrome-os-96-camera-scan-1.png?ssl=1" height="461" src="https://i1.wp.com/9to5google.com/wp-content/uploads/sites/4/2021/11/chrome-os-96-camera-scan-1.png?ssl=1" width="700" referrerpolicy="no-referrer"></p> 
<h3>Nearby Share</h3> 
<p>Nearby Share 现在可以在 Android 应用和它们的共享菜单内访问。此前，该功能仅限于文件应用、PWA 应用和系统应用。</p> 
<p><img alt="https://i2.wp.com/9to5google.com/wp-content/uploads/sites/4/2021/11/chrome-os-96-nearby-share.png?ssl=1" height="394" src="https://i2.wp.com/9to5google.com/wp-content/uploads/sites/4/2021/11/chrome-os-96-nearby-share.png?ssl=1" width="700" referrerpolicy="no-referrer"></p> 
<h3>Chrome OS 应用程序设置</h3> 
<p>在 Chrome OS 96 中，"设置" > "应用程序" 中有一个新的 "通知" 页面，而之前只有在 "快速设置" 中才能立即切换开/关。</p> 
<p><img alt="https://i2.wp.com/9to5google.com/wp-content/uploads/sites/4/2021/11/chrome-os-96-notifications-2.png?ssl=1" height="394" src="https://i2.wp.com/9to5google.com/wp-content/uploads/sites/4/2021/11/chrome-os-96-notifications-2.png?ssl=1" width="700" referrerpolicy="no-referrer"></p> 
<p>在 "管理你的应用程序" 列表中的应用程序现在有一个 "打开支持的链接" 选项，以确定当点击 URL 时，是通过安装的客户端还是在 Chrome 浏览器中开启该 URL。</p> 
<p><img alt="https://i1.wp.com/9to5google.com/wp-content/uploads/sites/4/2021/11/chrome-os-96-apps-open-links.png?ssl=1" height="394" src="https://i1.wp.com/9to5google.com/wp-content/uploads/sites/4/2021/11/chrome-os-96-apps-open-links.png?ssl=1" width="700" referrerpolicy="no-referrer"></p> 
<hr> 
<p>最后，壁纸应用在 Chrome OS 96 版本中被重新设计，有了更大的网格 UI，还可以从应用启动器图标访问，而不仅限直接在桌面上右键点击。</p>
