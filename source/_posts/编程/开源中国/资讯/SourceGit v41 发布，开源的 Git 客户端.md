                                                        <p>SourceGit v4.1 已经发布，开源的 Git 客户端</p> 
<p>此版本更新内容包括：</p> 
<ol> 
 <li>删除PageTabItem，将功能合并到PageTabBar中</li> 
 <li>添加SubTree时，如果当前目录中已存在同名子目录，直接连接信息</li> 
 <li>增加index.html为后面Gitlab Pages做准备</li> 
 <li>修复编辑远程URL时，判断重名的BUG</li> 
 <li>修改仓库地址为：<a href="https://gitee.com/sourcegit/sourcegit.git" target="_blank">https://gitee.com/sourcegit/sourcegit.git</a></li> 
 <li>删除Controls.WindowBorder，增加Controls.Window替换System.Windows.Window</li> 
 <li>其他一些主题上的调整</li> 
</ol> 
<p>详情查看：<a href="https://gitee.com/sourcegit/sourcegit/releases/v4.1">https://gitee.com/sourcegit/sourcegit/releases/v4.1</a></p>
