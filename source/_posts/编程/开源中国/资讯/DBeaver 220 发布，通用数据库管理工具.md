                                                                                            <p><span style="color:#333333">DBeaver 是一个免费开源的通用数据库工具，目前更新了 22.0 版本，带来如下变更：</span></p> 
<h3>数据传输：</h3> 
<ul> 
 <li>添加了表重新创建映射类型</li> 
 <li>表选择器对话框现在支持过滤器配置</li> 
 <li>数据传输任务的变量配置已修复</li> 
 <li>变量自动完成问题已在输出文件名中修复</li> 
 <li>自定义数据格式现在适用于 TXT 导出格式</li> 
 <li>Insert 语句现在支持空列列表的 DEFAULT 子句</li> 
</ul> 
<h3>数据编辑器：</h3> 
<ul> 
 <li>行数计算查询已修复</li> 
 <li>数据修改查询禁用行滚动</li> 
 <li>修复添加新行后的编辑器焦点问题</li> 
 <li>列值过滤器已修复（错误的 SQL 查询）</li> 
</ul> 
<p><strong>SQL 编辑器：</strong></p> 
<ul> 
 <li>改进了参数绑定对话框（添加了多行编辑器，修复了键盘快捷键）</li> 
 <li>按具有别名的列进行过滤和排序已修复</li> 
 <li>光标跳转到脚本结尾的问题已修复</li> 
 <li>脚本选择器弹出已修复（数据源关联问题）</li> 
 <li>脚本任务执行已修复（对于没有默认数据库的连接）</li> 
</ul> 
<h3>H2：</h3> 
<ul> 
 <li>添加了 2.x 版的驱动程序配置</li> 
 <li>添加了检查约束支持</li> 
 <li>添加了唯一键支持</li> 
</ul> 
<h3>甲骨文：</h3> 
<ul> 
 <li>元数据读取性能显着提升</li> 
 <li>区分大小写的导航过滤器支持已修复</li> 
</ul> 
<p> </p> 
<ul> 
 <li>MySQL：新视图创建已修复 (DDL)</li> 
 <li>DDL 生成器：添加了 <em>将 FK 作为单独语句生成 </em>的选项</li> 
 <li>元数据搜索：按 LIKE 模式搜索已修复</li> 
 <li>会话管理器 UI 已修复（Linux 上的多行单元格，连接类型的背景颜色）</li> 
 <li>现在可以在首选项中更改界面语言（适用于 MacOS 和 Linux 安装程序）</li> 
 <li>SSH：添加了禁用主机名验证的选项</li> 
 <li>本地客户端配置负载已修复（MySQL 和 PostgreSQL 派生驱动程序）</li> 
 <li>Azure SQL 服务器：视图定义读取已修复</li> 
 <li>Clickhouse：只读连接支持得到改进</li> 
 <li>雪花：自定义身份验证配置选项已修复</li> 
 <li>SQLite：序列读取已修复</li> 
 <li>SQL Server：打开数据库对象对话框现在尊重选定的数据库</li> 
 <li>意大利语本地化得到改进</li> 
 <li>修复了许多小的 UI 错误</li> 
</ul>
