<div class="content">
                                                                    
                                                        <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">DBeaver 是一个免费开源的通用数据库工具，适用于开发人员和数据库管理员。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">DBeaver 22.0.5 发布，更新内容如下：</p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>Data editor: 
  <ul style="margin-left:0; margin-right:0"> 
   <li>数字格式化：增加了数字组大小选项</li> 
   <li>修正了时间戳值意外变化的问题</li> 
   <li>改进了 macOS 上的平滑滚动</li> 
  </ul> </li> 
 <li>Data transfer: 改进了 XML 格式导出</li> 
 <li>SQL editor: 
  <ul style="margin-left:0; margin-right:0"> 
   <li>结果选项卡的重复使用行为得到了修复+增加了新的配置选项</li> 
   <li>为脚本执行任务增加了结果记录</li> 
   <li>删除了多余的 "问题" 视图弹出窗口</li> 
   <li>修复了关键词突出显示的问题</li> 
  </ul> </li> 
 <li>Diagrams: 
  <ul style="margin-left:0; margin-right:0"> 
   <li>增加了连接实体的高亮显示</li> 
   <li>改进了实体和列的搜索</li> 
   <li>增加了新的图表路径</li> 
   <li>修复了无法阅读注释的问题（黑暗主题）</li> 
   <li>改进了 SVG 和 Graphml 格式的导出</li> 
   <li>删除了图表中的伪列</li> 
   <li>修复了图表的重新排列</li> 
   <li>修复了许多小错误</li> 
  </ul> </li> 
 <li>Connections: 
  <ul style="margin-left:0; margin-right:0"> 
   <li>改进了从 JDBC URL 创建连接</li> 
   <li>改进了使用 CLI 打开连接</li> 
  </ul> </li> 
 <li>Preference pages: 解决了小字体的问题</li> 
 <li>Linux: 
  <ul style="margin-left:0; margin-right:0"> 
   <li>工作区加载的问题在某些情况下得到了解决</li> 
   <li>修复了连接类型编辑器中 UI 冻结的问题</li> 
  </ul> </li> 
 <li>PostgreSQL: 
  <ul style="margin-left:0; margin-right:0"> 
   <li>增加了 SSPI 认证支持</li> 
   <li>改进了对只读连接的支持</li> 
   <li>增加了计划任务管理</li> 
  </ul> </li> 
 <li>SQL Server: 
  <ul style="margin-left:0; margin-right:0"> 
   <li>修复了外部表元数据的读取</li> 
   <li>修正了对 NTLM 认证的支持</li> 
  </ul> </li> 
 <li>改进了意大利语和中文本地化</li> 
</ul>
