                                                                                            <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">Matomo 是一套基于 PHP + MySQL 技术构建的开源网站访问统计系统，能够提供详细的统计信息，比如网页浏览人数、访问最多的页面、搜索引擎关键词等等流量分析功能。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">Matomo 4.11 正式发布，这是一个维护版本，改善了 Matomo 的可靠性和稳定性，同时也包括一些重要的功能和改进。在 Matomo 4.11 中增加了包括可用于跟踪目标的全新指标和报告，开发者可以很容易地看到每个页面的目标转换和收入。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">Matomo 4.11 具体更新内容如下：</p> 
<h3 style="margin-left:.6em; margin-right:0; text-align:start">平台变化</h3> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">在 4.11 版本中，有一个重要的 API 变化，用户管理 UI 不再允许直接创建一个新的用户，一个新的配置设置<code>force_matomo_http_request</code>和新的 PHP 事件用于通过新的邀请系统添加用户。</p> 
<h3 style="margin-left:.6em; margin-right:0; text-align:start">新的和更新的 SDK</h3> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">Matomo 团队提供官方 SDK（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdeveloper.matomo.org%2Fguides%2Ftracking-api-clients%23mobile-sdks" target="_blank">Tracking API Clients</a>），用于监测你的移动应用程序和任何其他类型的应用程序。</p> 
<ul style="margin-left:0; margin-right:0"> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmatomo-org%2Fmatomo-sdk-ios%2Freleases" target="_blank">iOS SDK</a></li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmatomo-org%2Fmatomo-sdk-android%2F" target="_blank">Android SDK</a></li> 
</ul> 
<h3 style="margin-left:.6em; margin-right:0; text-align:start">其他变化</h3> 
<ul style="margin-left:0; margin-right:0"> 
 <li>matomo.org 的请求将很快默认使用 HTTPS</li> 
 <li>为行为类别中的页面报告添加目标可视化</li> 
 <li>为可视化（如表格、饼图、条形图）添加刷新/重新加载的键盘快捷键</li> 
 <li>修复 PHP 8.1 运行控制台 config:get 命令时的错误</li> 
 <li>如果 PHP 版本已过期，显示通知</li> 
 <li>确保 Safari 浏览器显示信息通知</li> 
 <li>修复 Matomo 内部的锚链接问题</li> 
 <li>修复在某些情况下无法将仪表盘复制给用户的问题</li> 
 <li>改进 DataTable/CSV 的内存管理</li> 
</ul> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">更多详情可查看：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fmatomo.org%2Fchangelog%2Fmatomo-4-11-0%2F" target="_blank">https://matomo.org/changelog/matomo-4-11-0/</a></p>
