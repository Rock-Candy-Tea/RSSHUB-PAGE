                                                                                            <p>Tails 开发团队发布了 Tails 5.0 的正式版本，它基于最新的 Debian GNU/Linux 11 "Bullseye" 操作系统，带来了很多 Tails 中包含的软件的新版本以及新的 OpenPGP 工具。</p> 
<p>Tails 5.0 正式发布，更新内容如下：</p> 
<h3>新功能</h3> 
<p>新版本增加了 Kleopatra，以取代 OpenPGP Applet 和 Password and Keys 工具。OpenPGP Applet 已经不再被积极开发，因此在 Tails 中继续保留它是很复杂的。Password and Keys 工具也没有得到很好的维护，Tails 用户还面临着它所带来的太多问题。</p> 
<p>Kleopatra 在一个单一的工具中提供了同等的功能，而且开发得更加积极。</p> 
<p><img alt height="537" src="https://static.oschina.net/uploads/space/2022/0505/072851_GKi9_4937141.png" width="700" referrerpolicy="no-referrer"></p> 
<h3>变化和更新</h3> 
<ul> 
 <li>持久性存储的附加软件功能被默认启用，以使其更快、更好地配置你的第一个附加软件包。</li> 
 <li>你现在可以使用活动概览来访问你的窗口和应用程序。</li> 
</ul> 
<p><img alt height="525" src="https://static.oschina.net/uploads/space/2022/0505/072910_nL8l_4937141.png" width="700" referrerpolicy="no-referrer"></p> 
<h3>包含的软件</h3> 
<p>大多数包含的软件在 Debian 11 中已经升级了，例如：</p> 
<ul> 
 <li>将 Tor Browser 升级到 11.0.11。</li> 
 <li>将 GNOME 从 3.30 升级到 3.38，对桌面、GNOME 核心工具和锁定屏幕做了很多小的改进。</li> 
 <li>将 MAT 从 0.8 更新到 0.12，增加了对清理 SVG、WAV、EPUB、PPM 和 Microsoft Office 文件的元数据的支持。</li> 
 <li>将 Audacity 从 2.2.2 更新到 2.4.2。</li> 
 <li>将 Disk Utility 从 3.30 更新到 3.38。</li> 
 <li>将 GIMP 从 2.10.8 更新到 2.10.22。</li> 
 <li>将 Inkscape 从 0.92 更新到 1.0。</li> 
 <li>将 LibreOffice 从 6.1 升级到 7.0。</li> 
</ul> 
<h3>硬件支持</h3> 
<ul> 
 <li>对 Linux 中的无驱动打印和扫描的支持使最近的打印机和扫描仪更容易在 Tails 中工作。</li> 
</ul> 
<h3>修复的问题</h3> 
<ul> 
 <li>修复了解锁具有很长密码的 VeraCrypt 卷的问题</li> 
</ul>
