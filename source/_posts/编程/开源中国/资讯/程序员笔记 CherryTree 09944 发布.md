                                                                                            <p>CherryTree 0.99.44 现已发布。CherryTree 是一个支持无限层级分类的笔记软件，Python 编写，支持富文本编辑和代码高亮，支持 Linux 和 Windows 平台。数据采用 sqlite 或 XML 存储，支持密码保护，支持从 NoteCase、KeepNote、Knowit、Tomboy、TuxCards、Treepad、Leo 等笔记软件导入数据。</p> 
<p data-darkreader-inline-color style="--darkreader-inline-color:#c8c3bc; color:#333333; margin-left:0px; margin-right:0px; text-align:left"><img alt height="342" src="https://static.oschina.net/uploads/space/2021/0722/064237_TNzL_2744687.png" width="500" referrerpolicy="no-referrer"></p> 
<p data-darkreader-inline-color style="--darkreader-inline-color:#c8c3bc; color:#333333; margin-left:0px; margin-right:0px; text-align:left"><img alt height="342" src="https://static.oschina.net/uploads/space/2021/0722/064244_aMlQ_2744687.png" width="500" referrerpolicy="no-referrer"></p> 
<p data-darkreader-inline-color style="--darkreader-inline-color:#c8c3bc; color:#333333; margin-left:0px; margin-right:0px; text-align:left">此版本更新内容如下：</p> 
<ul> 
 <li>将文件从文件管理器中复制和粘贴到富文本中会导致该文件被嵌入/捆绑在文档中；在 Linux 和 Mac OS 上，也支持 multiple copy</li> 
 <li>在 Linux 中，从文件管理器中拖放文件到富文本中，也会导致文件被嵌入/捆绑在文档中</li> 
 <li><span data-darkreader-inline-bgcolor data-darkreader-inline-color style="--darkreader-inline-bgcolor:#181a1b; --darkreader-inline-color:#cecac3; background-color:#ffffff; color:#24292f">修复</span>当模式包含 ampersand 时，信息对话框显示未找到模式</li> 
 <li><span data-darkreader-inline-bgcolor data-darkreader-inline-color style="--darkreader-inline-bgcolor:#181a1b; --darkreader-inline-color:#cecac3; background-color:#ffffff; color:#24292f">修复将多个节点导出为 HTML，在节点名称/页面标题中使用撇号时，节点/页面的链接不起作用的问题</span></li> 
 <li><span data-darkreader-inline-bgcolor data-darkreader-inline-color style="--darkreader-inline-bgcolor:#181a1b; --darkreader-inline-color:#cecac3; background-color:#ffffff; color:#24292f">修复将带有表格和/或代码框的多个节点导出为纯文本</span></li> 
 <li><span data-darkreader-inline-bgcolor data-darkreader-inline-color style="--darkreader-inline-bgcolor:#181a1b; --darkreader-inline-color:#cecac3; background-color:#ffffff; color:#24292f">剥离尾部空格的动作，使之在代码框内也能工作</span></li> 
 <li><span data-darkreader-inline-bgcolor data-darkreader-inline-color style="--darkreader-inline-bgcolor:#181a1b; --darkreader-inline-color:#cecac3; background-color:#ffffff; color:#24292f">增加了新的中文繁体语言</span></li> 
 <li><span data-darkreader-inline-bgcolor data-darkreader-inline-color style="--darkreader-inline-bgcolor:#181a1b; --darkreader-inline-color:#cecac3; background-color:#ffffff; color:#24292f">更新了语言：韩语、土耳其语、芬兰语、西班牙语、俄语、哈萨克语、德语</span></li> 
</ul>
