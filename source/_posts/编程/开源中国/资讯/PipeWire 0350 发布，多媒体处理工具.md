                                                                                            <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">PipeWire 是用于处理多媒体管道的服务器和用户空间 API 的多媒体处理工具，包括提供视频源（例如来自捕获设备或应用程序提供的流）并将其与客户端复用、访问视频源进行消费、生成用于音频和视频处理的图形。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">PipeWire 0.3.50 正式发布，该版本是一个修复版本，与之前版本的 API 和 ABI 兼容，主要更新内容如下：</p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>pw-stream 现在可以报告更多的时间信息，并且可以建议排队等待播放的最佳样本数</li> 
 <li>pw-dot 现在又可以正常使用了</li> 
 <li>module-pulse-tunnel 的延迟得到了改善</li> 
 <li>使用 JACK 后端的 WINE 应用程序不应该再崩溃</li> 
 <li>channelmixer 的默认值得到了改进，并且在播放 5.1 和 7.1 素材时的消音现象也得到了改善</li> 
 <li>其他错误修复和改进</li> 
</ul>
