                                                                                            <p style="color:#333333; margin-left:0; margin-right:0; text-align:left">GraalVM 社区版 22.0 已发布。GraalVM 是 Oracle 打造的高性能跨语言虚拟机，支持运行 JavaScript、Python 3、Ruby、R、基于 JVM 的语言（如 Java、Scala 和 Kotlin），以及基于 LLVM 的语言，如 C 和 C++。</p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">此外，GraalVM 在编程语言之间具有高效的互操作性，并可提前将 Java 应用程序编译为本地可执行文件，以加快启动时间，降低内存开销。</p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">GraalVM 社区发行版包括：</p> 
<ul style="list-style-type:disc; margin-left:0; margin-right:0"> 
 <li>带有 GraalVM 编译器的 Java runtime</li> 
 <li>带有 GraalVM JavaScript 解释器的 Node.js runtime</li> 
 <li>LLVM runtime</li> 
 <li>开发者监控和调试工具</li> 
</ul> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">GraalVM 环境可以通过 GraalVM Updater 工具使用可选的可用组件进行扩展，例如 Native Image、Ruby、R、Python、LLVM 工具链、WebAssembly 和 Espresso。</p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">此版本的<span style="background-color:#ffffff; color:#121212">一些<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.phoronix.com%2Fscan.php%3Fpage%3Dnews_item%26px%3DGraalVM-22.0-Released" target="_blank">亮点</a>包括：</span></p> 
<ul> 
 <li style="text-align:left">Java 8 支持已被删除。GraalVM 22.0 仅针对 JDK 11 和 JDK 17，同时删除了对 JDK 12/13/14/15/16 的支持。</li> 
 <li style="text-align:left">一项新的优化，以提高类型切换的 Native Image 的性能。还有 Native Image 更新以减小图像大小。</li> 
 <li style="text-align:left">Native Image 改进了对 Java 平台模块系统的支持。</li> 
 <li style="text-align:left">各种新的 Java 编译器优化，尽管仅限于 GraalVM Enterprise。</li> 
 <li style="text-align:left">现在默认为 GraalVM 的 JavaScript 支持启用 ECMAScript 2022 features。</li> 
 <li style="text-align:left">GraalVM 的 LLVM 运行时已切换到 Truffle Fame API，进行了新的优化和其他修复。</li> 
 <li style="text-align:left">GraalVM 的 WebAssembly 实现的各种改进。</li> 
</ul>
