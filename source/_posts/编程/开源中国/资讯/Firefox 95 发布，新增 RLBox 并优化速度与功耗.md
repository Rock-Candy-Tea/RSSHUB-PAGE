                                                        <p>Firefox 95.0 今天正式发布，这是 Firefox 在 2021 年的最后一个主要稳定版本。Firefox 95 包含多项新功能和改进：</p> 
<h3>新功能</h3> 
<ul> 
 <li> <p>RLBox：新的沙盒技术，可以加固 Firefox 浏览器以防止第三方库中的潜在安全漏洞。现在已在所有平台上启用。（技术详情可查看这篇报道：<a href="https://www.oschina.net/news/172316/mozilla-firefox-rlbox">链接</a>）</p> </li> 
 <li> <p>现在可以在 Windows 10 和 Windows 11 平台上从微软商店下载 Firefox。</p> <p><img alt="https://www.mozilla.org/media/img/firefox/releasenotes/note-images/95_windows_store.png" src="https://static.oschina.net/uploads/img/202112/08072029_m00j.png" referrerpolicy="no-referrer"></p> </li> 
 <li> <p>减少了事件处理过程中 Firefox 和 WindowServer 在 macOS 上的 CPU 使用率。</p> </li> 
 <li> <p>减少了 macOS 上软件解码视频的功耗，特别是在全屏状态下。这包括如 Netflix 和 Amazon Prime Video 这样的流媒体网站。</p> </li> 
 <li> <p>你现在可以把画中画切换按钮移到视频的另一侧。</p> </li> 
 <li> <p>为了更好地保护 Firefox 用户免受 Spectre 等攻击，现在所有 Firefox 95 都启用了站点隔离。</p> </li> 
</ul> 
<h3>修复</h3> 
<ul> 
 <li>启动 Firefox 后，使用 JAWS 屏幕阅读器和 ZoomText 放大镜的用户将不再需要切换应用程序以访问 Firefox。</li> 
 <li>macOS 上的内容进程启动更快。</li> 
 <li>对内存分配器进行了改进。</li> 
 <li>通过推测性地提前编译 JavaScript 来提高页面加载性能。</li> 
 <li>安全问题的修复</li> 
</ul>
