                                                        <p>Simplenote 2.9.0 发布了。Simplenote 是一个简单、轻量级的跨平台云笔记工具，可以用来保存文字、表格、代码等信息，用户记录的内容可以与所有设备保持同步。Simplenote 支持大多数主流的 WEB 浏览器，同时支持 iOS，Android，Mac，Windows 和 Linux 操作系统。 </p> 
<p>此版本的更新内容包括：</p> 
<p><strong>Enhancements</strong></p> 
<ul> 
 <li>更新了 Focus Mode/Toggle Sidebar 命名，使其与 Focus Mode 保持一致</li> 
 <li>没有标签时，从导航菜单中删除了标签标题</li> 
 <li>将注释预览中的行数更新为扩展显示中的三行</li> 
 <li>更新以在打开确认对话框时自动对焦删除标签按钮</li> 
 <li>改进了对多个粘贴标签的解析</li> 
 <li>改进了应用程序对键盘和屏幕阅读器用户的可访问性</li> 
 <li>改进了应用程序内的错误处理和消息传递</li> 
 <li>改进的导航侧边栏和 revision selector 对键盘和屏幕阅读器的可访问性</li> 
 <li>将导入更新为单个对话框，在该对话框中可以同时导入任何受支持的文件类型</li> 
 <li>改进内联代码元素的可识别性</li> 
</ul> 
<p><strong>Fixes</strong></p> 
<ul> 
 <li> <p>修复了重命名所选标签时 menubar 标题已更新以反映新标签名称的问题</p> </li> 
 <li>修复了 Electron builds 中 offline badge indicator 的位置</li> 
 <li>修复了非同步音符警告对话框中的一些样式问题</li> 
 <li>修复了 pending tag input 的编辑</li> 
 <li>修复为仅在需要时在标签列表上显示滚动条</li> 
 <li>修复了关闭禁用按钮的工具提示的问题</li> 
 <li>修复了预览模式下的复选框</li> 
</ul>
