                                                                                            <p><span style="background-color:#ffffff; color:#333333">rtty 8.0.0 已发布。</span></p> 
<p>性能改善</p> 
<ul> 
 <li>使用流量控制</li> 
 <li>重构设登录流程，提升稳定性（需要rttys 4.x）</li> 
 <li>改善从设备下载文件到本地：采样传输过程中保存文件的策略</li> 
</ul> 
<p>新特性</p> 
<ul> 
 <li>上传下载文件支持针对每个终端会话</li> 
</ul>

<h1 style="margin-left:0; margin-right:0; text-align:start">特性</h1> 
<ul style="list-style-type:disc; margin-left:0; margin-right:0"> 
 <li>客户端C语言实现，非常小，适合嵌入式Linux</li> 
 <li>远程批量执行命令</li> 
 <li>支持SSL: openssl, mbedtls, CyaSSl(wolfssl)</li> 
 <li>非常方便的上传和下载文件</li> 
 <li>根据设备ID访问不同的设备</li> 
 <li>基于<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fxtermjs%2Fxterm.js" target="_blank">Xterm.js</a>的全功能终端</li> 
 <li>部署简单，使用方便</li> 
 <li>支持 HTTP 代理，访问您的设备的 Web</li> 
</ul>
