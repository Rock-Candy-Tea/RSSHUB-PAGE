<p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><span style="background-color:#ffffff; color:#40485b">MyCms 是一款基于Laravel开发的开源免费的自媒体博客CMS系统，助力开发者知识技能变现。</span></p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><span style="background-color:#ffffff; color:#40485b">MyCms 基于Apache2.0开源协议发布，免费且不限制商业使用，欢迎持续关注我们。</span></p> 
<p style="color:#40485b; margin-left:0; margin-right:0; text-align:start"><strong>v3.2.0 更新内容</strong></p> 
<p><span style="background-color:#ffffff; color:#40485b">新增：阿里云短信插件</span><br> <span style="background-color:#ffffff; color:#40485b">新增：阿里云OSS插件</span><br> <span style="background-color:#ffffff; color:#40485b">新增：后台筛选通用方法</span><br> <span style="background-color:#ffffff; color:#40485b">优化：后台支付记录筛选优化</span><br> <span style="background-color:#ffffff; color:#40485b">优化：后台评论列表筛选优化</span><br> <span style="background-color:#ffffff; color:#40485b">优化：后台文章标签列表筛选优化</span><br> <span style="background-color:#ffffff; color:#40485b">优化：后台文章列表筛选优化</span><br> <span style="background-color:#ffffff; color:#40485b">优化：后台商品分类列表筛选优化</span><br> <span style="background-color:#ffffff; color:#40485b">优化：后台商品列表筛选优化</span><br> <span style="background-color:#ffffff; color:#40485b">优化：后台会员列表筛选优化</span><br> <span style="background-color:#ffffff; color:#40485b">优化：后台积分明细增加筛选</span><br> <span style="background-color:#ffffff; color:#40485b">优化：后台余额明细增加筛选</span></p> 
<p><strong>更新重点</strong></p> 
<p>一、阿里云OSS插件</p> 
<p>安装好插件后依次填写好配置，然后进入系统模块 > 系统配置 > <span style="background-color:#ffffff; color:#000000">文件上传，选择 阿里云OSS，保存即可。</span></p> 
<p><img alt height="502" src="https://oscimg.oschina.net/oscnet/up-9465aa79f3d42185941f82fff9b7dddbecb.png" width="1078" referrerpolicy="no-referrer"></p> 
<p>二、阿里云短信插件</p> 
<p>安装好插件后，进入插件工作区，添加配置。</p> 
<p>使用方法：ali_sms($mobile, 0, ['code' => $number])</p> 
<p>参数说明：参数1 = 手机号码 ，参数2 = 短信模板 ( 0 为默认模板)，参数3：验证码</p>
