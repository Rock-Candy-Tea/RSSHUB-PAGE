                                                                                            <p>zlt-microservices-platform v5.0.0 已经发布，企业级微服务多租户系统架构。</p> 
<p><strong>更新说明</strong>：</p> 
<h3>特性/增强</h3> 
<ul> 
 <li>授权中心支持 <strong>多用户体系</strong> 扩展</li> 
 <li>升级 <code>spring-boot</code> 到 <code>2.5.4</code></li> 
 <li>升级 <code>spring-cloud</code> 到 <code>2020.0.3</code></li> 
 <li>升级 <code>spring-cloud-alibaba</code> 到 <code>2021.1</code></li> 
 <li>升级 <code>升级druid-spring-boot-starter</code> 到 <code>1.2.6</code></li> 
 <li>升级 <code>elasticsearch</code> 到 <code>7.14.0</code></li> 
 <li>升级 <code>spring-boot-admin</code> 到 <code>2.5.1</code></li> 
</ul> 
<h3>新增工程/分支</h3> 
<ul> 
 <li>新增 <code>zlt-loadbalancer-spring-boot-starter</code> 工程</li> 
 <li>增加分支 <code>4.x</code>（该分支不会更新） 
  <ul> 
   <li>Spring Boot 2.3.12.RELEASE</li> 
   <li>Spring Cloud Hoxton.SR12</li> 
   <li>Spring Cloud Alibaba 2.2.6.RELEASE</li> 
  </ul> </li> 
</ul> 
<h3>移除工程</h3> 
<ul> 
 <li>移除 <code>zlt-ribbon-spring-boot-starter</code> 工程</li> 
 <li>移除 <code>zuul-gateway</code> 工程</li> 
</ul> 
<p>详情查看：<a href="https://gitee.com/zlt2000/microservices-platform/releases/v5.0.0">https://gitee.com/zlt2000/microservices-platform/releases/v5.0.0</a></p>
