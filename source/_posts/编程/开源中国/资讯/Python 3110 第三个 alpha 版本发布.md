                                                                                            <p>Python 3.11.0a3，即 Python 3.11 第三个 alpha 版本目前<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fpythoninsider.blogspot.com%2F2021%2F12%2Fpython-3110a3-is-available.html" target="_blank">已发布</a>。</p> 
<p><span style="background-color:#ffffff; color:#333333">在 alpha 阶段，每个版本可能都会添加功能，直到进入 beta 阶段 (2022-05-06)。如有必要，也可以修改或删除特性，直到进入候选阶段 (2022-08-01)。请记住，这是一个预览版本，不建议在生产环境中使用。</span></p> 
<p><strong>Python 3.11.0 主要变化</strong></p> 
<ul style="margin-left:0; margin-right:0"> 
 <li><span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.python.org%2Fdev%2Fpeps%2Fpep-0657%2F" target="_blank">PEP 657</a> – </span>在错误信息报告中引入细粒度的 Error Locations</li> 
 <li><span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.python.org%2Fdev%2Fpeps%2Fpep-0654%2F" target="_blank">PEP 654</a> – PEP 654 – Exception Groups and except*</span></li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffaster-cpython" target="_blank">Faster Cpython Project</a> 已经出现了一些令人振奋的结果：与 3.10.0 相比，这个版本的 CPython 3.11 在 <a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fspeed.python.org%2F" target="_blank">PyPerformance 基准</a>的几何平均值上快了大约 19%</li> 
 <li><span style="color:#222222"><span>(Hey, </span></span><span style="color:#222222">fellow core developer,</span><span style="color:#222222"> if a feature you find important is missing from this list, </span><a href="https://www.oschina.net/action/GoToLink?url=mailto%3Apablogsal%40python.org" target="_blank">let Pablo know</a><span style="color:#222222">.)</span></li> 
</ul>
