                                                        <p>Ant Design 4.15.6 发布了。Ant Design 是一套企业级的 UI 设计语言和 React 实现，使用 TypeScript 构建，提供完整的类型定义文件，自带提炼自企业级中后台产品的交互语言和视觉风格、开箱即用的高质量 React 组件与全链路开发和设计工具体系。</p> 
<p>此版本更新内容如下：</p> 
<ul> 
 <li>修复 Upload 上传文件被 <code>accept</code> 过滤后 <code>beforeUpload</code> 无法接收到的问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F30549" target="_blank">#30549</a></li> 
 <li>移除全局样式中对 <code>th</code> 的 <code>text-align</code> 属性的重置。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F30399" target="_blank">#30399</a></li> 
 <li>国际化 
  <ul> 
   <li>补充印地语国际化文案。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F30541" target="_blank">#30541</a></li> 
   <li>补充葡萄牙语（巴西）国际化文案。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F30532" target="_blank">#30532</a></li> 
  </ul> </li> 
</ul>
