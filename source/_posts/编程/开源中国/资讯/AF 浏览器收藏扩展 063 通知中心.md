   <h3 style="text-align:start">🛠 部分改进和修复<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fblack-stone-0c40f9100-34.eastasia.azurestaticapps.net%2Fzh-Hans%2Fdocs%2F99-1-Whats-New%23-%25E9%2583%25A8%25E5%2588%2586%25E6%2594%25B9%25E8%25BF%259B%25E5%2592%258C%25E4%25BF%25AE%25E5%25A4%258D" target="_blank">#</a></h3> 
   <ul> 
    <li>修复了编辑标签时占用 CPU 过高的问题</li> 
    <li>现在，首次打开 ManagerUI 的传参方式发生了改变，由查询字符串改为了 AfEventHub。这使得用户不会在出现意外弹出编辑框的情况。</li> 
    <li>开发阶段支持使用 Seq 来输出日志，看起来似乎更容易排查错了呢</li> 
   </ul>

  <h3 style="margin-left:0; margin-right:0">软件简介</h3> 
 </div> 
 <div style="text-align:start"> 
  <div style="text-align:left"> 
   <p style="margin-left:0; margin-right:0">Amazing Favorites 是一款采用 Blazor 技术开发的 Chromium 内核的浏览器插件，因此，其可以被安装到 Chrome / Microsoft Edge / 360 极速浏览器 / Vivaldi / 搜狗高速浏览器 等浏览器之上。<br> 它可以帮助使用者高效的管理和搜索自己的收藏夹，为此增加了一些增强功能：</p> 
   <ul style="list-style-type:disc; margin-left:0; margin-right:0"> 
    <li>支持为链接标记 tag，从而为通过 tag 为链接进行交叉分组</li> 
    <li>支持快捷键操作</li> 
   </ul>
