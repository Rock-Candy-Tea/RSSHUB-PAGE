                                                                                            <p>Material UI 4.12.3 现已发布。Material UI 是一组实现 Google Material Design 规范的 React 组件，它是一个前端 JS 框架，主要用在 Web 领域。</p> 
<p>主要更新内容：</p> 
<ul> 
 <li>[AccordionSummary] 修复`disableGeneration`的假阳性 propType 警告 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmui-org%2Fmaterial-ui%2Fpull%2F27385" target="_blank">#27385</a> ) </li> 
 <li>[ImageList] 修复弃用警告 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmui-org%2Fmaterial-ui%2Fpull%2F27502" target="_blank">#27502</a> )</li> 
 <li>[TablePagination] 将弃用的 `onChangePage` 重新引入 `ActionsComponent` ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmui-org%2Fmaterial-ui%2Fpull%2F27407" target="_blank">#27407</a> )</li> 
 <li>[TextareaAutosize] 更新了弃用警告以建议使用 minRows 而不是 rowsMin ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmui-org%2Fmaterial-ui%2Fpull%2F27398" target="_blank">#27398</a> )</li> 
</ul> 
<p><strong>Docs</strong></p> 
<ul> 
 <li>[docs] 修复 ImageList 的 404 链接</li> 
 <li>[docs] 在 v4 中修复 DataTable.tsx 演示 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmui-org%2Fmaterial-ui%2Fpull%2F27196" target="_blank">#27196</a> )</li> 
</ul> 
<p><strong>Core</strong></p> 
<ul> 
 <li> <p>[core] 添加发布脚本 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmui-org%2Fmaterial-ui%2Fpull%2F27399" target="_blank">#27399</a> ) </p> </li> 
</ul>
