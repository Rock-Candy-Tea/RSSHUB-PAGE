                                                                                            <p>经过 8 个 RC 版本的更新， Linux Kernel 5.17 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Flore.kernel.org%2Flkml%2FCAHk-%3Dwj4fFjx2pgbGNBM4wJs3%3DeReZ05EQyprzgT2Jv8qJ2vJg%40mail.gmail.com%2FT%2F%23u" target="_blank">正式发布</a>了，从 Linux 5.17 rc-8 版本到稳定版本仅有一些关于硬件驱动的恢复和修复。</p> 
<p>Linux 5.17 具有许多新功能，包括替代 ACPI CPUFreq 的新 AMD P-State 驱动程序、对 KVM 虚拟化的 Intel AMX 支持、早期的 Intel Raptor Lake 支持、Intel 平台固件运行时更新、通用触控笔支持、各种性能优化等等。</p> 
<p>下面整理一些 Linux 5.17 的新功能介绍：</p> 
<ul> 
 <li><a href="https://www.oschina.net/news/178989/linux-kernel-5-17-merge-rlta">Linux 5.17 引入“RTLA”：实时 Linux 性能分析与追踪工具</a></li> 
 <li><a href="https://www.oschina.net/news/176193/intel-pfrut-linux-5.17">Linux 5.17 将正式迎来免重启更新系统固件</a></li> 
 <li><a href="https://www.oschina.net/news/175094/linux-5.17-p-state">Linux 5.17 将为 Alder Lake 提供更好的睿频性能</a></li> 
 <li><a href="https://www.oschina.net/news/173155/more-apple-m1-for-linux-5-17">Linux 5.17 内核将更好地兼容 Apple M1</a></li> 
 <li><a href="https://www.oschina.net/news/171978/linux-5.17-floppy">Linux 5.17 将修复损坏的软盘导致的内核挂起问题</a></li> 
 <li><a href="https://www.oschina.net/news/171175/linux-5-17-af-unix-socket-improvment">Linux 5.17 将为 AF_UNIX 套接字提供延迟优化</a></li> 
</ul> 
<p>5.17 的发布，意味着 Linux 5.18 的合并窗口将打开， 5.18 有更多有意思的新功能，比如<a href="https://www.oschina.net/news/182489/intel-sdsi-merge-to-linux-5-18">英特尔 CPU 内购功能 (SDSI)</a>、<a href="https://www.oschina.net/news/185952/linux-5-18-c11-plan">内核 C 标准从 C89 升级到 C11/GNU11</a> ...</p>
