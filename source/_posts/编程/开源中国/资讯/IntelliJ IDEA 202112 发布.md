                                                        <p>IntelliJ IDEA 2021.1.1 现已发布，这是第二个针对 IntelliJ IDEA 2021.1 的 Bug 修复版本。主要更新内容如下：</p> 
<ul> 
 <li>修复了导致 color scheme 意外更改的问题。[ <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-265169" target="_blank">IDEA-265169</a> ]</li> 
 <li>当 distributionSha256Sum 出现在 Gradle 项目中时，IDE 不再返回错误。[ <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-266355" target="_blank">IDEA-266355</a> ]</li> 
 <li>当调用 Quick Documentation action 时，中文字符现在可以正确显示。[ <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-268158" target="_blank">IDEA-268158</a> ]</li> 
 <li>修复了几个影响 MacOS 上包含 Option 键的快捷方式的错误。[ <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FJBR-3409" target="_blank">JBR-3409</a> ]</li> 
 <li>修复了使用旧 Maven 分发版（3.3.0 及更早版本）重新导入项目时的异常。[ <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-268495" target="_blank">IDEA-268495</a> ]</li> 
 <li>重新启动 IDE 时，Tab 和缩进设置会重置。[ <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-267368" target="_blank">IDEA-267368</a> ]</li> 
 <li>工具窗口的大小在 Undocked 模式下不会波动。[ <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-266173" target="_blank">IDEA-266173</a> ]</li> 
</ul> 
<p><strong>IntelliJ IDEA 2021.1 中的质量改进</strong></p> 
<p>开发团队表示， 在开发 IntelliJ IDEA 2021.1 的过程中，其花费了比平时更多的时间来提高产品质量。主要集中在三个关键领域：性能、消除 freezes 和异常，以及回归处理。目前共解决了超过 1500 个 YouTrack 问题。</p> 
<ul> 
 <li>修复了<strong>IntelliJ IDEA 核心功能</strong>中的 600 多个问题，以及 400 多个小的 <strong>UI 改进</strong>。</li> 
 <li>我们实施了大约 400 个其他修复，包括旨在消除 freezes 和提高 <strong>Java </strong>重构和代码完成性能的修复。</li> 
 <li>对于 <strong>JetBrains Runtime (JBR)，</strong>解决了 JCEF 中的功能问题和崩溃问题。此外，还在 OpenJDK 11.0.10 之上重新构建了 JBR。</li> 
 <li>修复了 <strong>IntelliJ IDEA Ultimate 中的 </strong>150 多个问题，包括针对特定于框架的功能和图表的性能优化。一些优先解决的问题与 Spring、JPA、Java EE 和其他框架中的回归有关。</li> 
 <li>通过加速错误高亮和代码完成显着提高了 <strong>Kotlin 插件的</strong>性能。至少有十几个导致 IDE freezes 的不同问题也得到了修复。</li> 
 <li>在 <strong>Version Control</strong> <strong>中，</strong>解决了 Staging area 支持和 Commit message 处理中的许多小问题。还重新设计了用于创建 GitHub 拉取请求的 UI，并修复了 GitHub 评论的各种小问题。</li> 
</ul>
