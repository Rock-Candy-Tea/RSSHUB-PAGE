                                                        <p>Slackware Linux 15.0 发布了第二个 RC 版本，意味着正式即将到来。创始人 Patrick Volkerding 表示这个时候需要更谨慎地保证一切稳定，由于进行了一次测试的大规模构建，因此不会再出现“从源码构建失败”的情况。</p> 
<p><a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fwww.slackware.com%2Fchangelog%2Fcurrent.php%3Fcpu%3Dx86_64" target="_blank">最新的更新日志显示</a>，此版本还包括一个关于重新构建 Python Markdown 以修复 Samba 构建的说明：“较新版本的 Markdown 需要 importlib_metadata 和 zipp，添加的这些变化也修复了构建，但奇怪的是，安装的 PKG-INFO 显示版本为 0.0.0，因此可能是 setuptools 发生了故障。Markdown 3.3.4 似乎是目前最安全的选择。”</p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">Slackware Linux 15.0 的主要变化是升级内核、桌面环境，以及常用软件包：</p> 
<ul style="list-style-type:disc; margin-left:0; margin-right:0"> 
 <li>内核版本升级为 Linux Kernel 5.13</li> 
 <li>KDE Plasma 5.21</li> 
 <li>Xfce 4.16</li> 
 <li>GNOME 3.36/3.38</li> 
 <li>Python 3.9</li> 
 <li>Mozilla Firefox 91.0.1</li> 
 <li>Thunderbird 91.0.1</li> 
</ul> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">Slackware 没有提供任何官方的测试镜像，但其开发者之一 Eric Hameleers 定期从 Slackware 的"Current"分支构建了非官方的安装 DVD：</p> 
<ul> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fslackware.nl%2Fslackware%2Fslackware64-current-iso%2Fslackware64-current-install-dvd.iso" target="_blank">slackware64-current-install-dvd.iso</a></li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fslackware.nl%2Fslackware%2Fslackware64-current-iso%2Fslackware64-current-mini-install.iso" target="_blank">slackware64-current-mini-install.iso</a><span style="background-color:#fdffe3; color:#000000"><span> </span></span></li> 
</ul> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">Slackware Linux 是一款最古老同时也是最活跃的发行版，由 Patrick Volkerding 制作，从 1993 年发布至今也一直在 Patrick 带领下进行维护。它为着易用性和高稳定性双重目标而设计。Slackware 包含最新的流行软件，并按照传统提供简单易用性、灵活性和强大的功能，可装备使用在从桌面工作站到机房服务器的任何场合。</p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><img src="https://static.oschina.net/uploads/space/2021/0223/065325_NOQU_2720166.png" referrerpolicy="no-referrer"></p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">Slackware 走了一条与其他的发行版（Red Hat、Debian、Gentoo、SuSE、Mandriva、Ubuntu 等）不同的道路，它力图成为“UNIX 风格”的 Linux 发行版。它的方针是只吸收稳定版本的应用程序，并且缺少其他 Linux 版本中那些为发行版定制的配置工具。它支持按需使用各种 Web、FTP 和 Email 服务器，正如可以在各种流行的桌面环境中作出选择。大量的开发工具和编辑器、库文件被包纳进来，以方便那些希望开发或编译额外软件的用户们。</p>
