                                                                                            <p>JetBrains 日前发布了 IntelliJ IDEA 2022.2 系列的第二个小幅更新版本，该版本主要修复了之前存在的各种错误，以下是这次更新中一些重要的内容：</p> 
<ul> 
 <li>Markdown 自动格式化操作现在可以正确地格式化带有表格的 Markdown 文件</li> 
 <li>IDE 现在会在后台对以失败结果结束的预提交检查显示一个气球通知</li> 
 <li>性能改进：当启用 Show Method Return Values 选项时，调试性能不再变慢</li> 
 <li>修复：再次可以将只读调试器附加到 macOS 上的 Java 进程</li> 
 <li>修复：Line Comment 再次可用</li> 
 <li>修复：Ctrl + 鼠标点击的快捷键再次适用于文件路径</li> 
 <li>修复：粘贴操作不再包括一个额外的前导缩进</li> 
 <li>修复：拖放动作在多行选择中正确工作</li> 
 <li>修复：运行远程 SSH 目标不再返回 "Failed to prepare environment（准备环境失败）" 错误</li> 
</ul>
