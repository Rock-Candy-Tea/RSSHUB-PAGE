                                                        <p>DBeaver 21.1.2 现已发布。DBeaver是一个免费开源的通用数据库工具，适用于开发人员和数据库管理员。</p> 
<p><strong>主要更新内容</strong></p> 
<ul> 
 <li>Data viewer 
  <ul> 
   <li>Record mode 现在支持多记录视图</li> 
   <li>过滤器弹出窗口中的双击行为已修复</li> 
   <li>Spatial viewer 现在允许复制点坐标</li> 
   <li>为 CTE 查询和子选择修复了源表解析</li> 
  </ul> </li> 
 <li>Navigator 
  <ul> 
   <li>快速过滤器已修复（仅显示已连接）</li> 
   <li>修复了“Set connection from navigator”的行为</li> 
  </ul> </li> 
 <li>SQL editor 
  <ul> 
   <li>修复了带有特殊字符的标识符的自动补全问题</li> 
   <li>SQL 控制台现在为选定的架构打开</li> 
   <li>表列按自动完成的原始顺序显示</li> 
   <li>带有嵌入式注释的查询格式已修复</li> 
   <li>添加了 Local variables coloring（Sybase、SQL Server、MySQL）</li> 
  </ul> </li> 
 <li>Metadata editor 
  <ul> 
   <li>元数据搜索现在支持在过程/视图定义中搜索</li> 
   <li>Object info 刷新已修复</li> 
   <li>大多数数据库的 table triggers 加载性能显着提高</li> 
  </ul> </li> 
 <li>Clickhouse 
  <ul> 
   <li>Table DDL 已修复</li> 
   <li>添加了嵌套评论支持</li> 
   <li>数据编辑（更新/删除）查询生成器已修复</li> 
  </ul> </li> 
 <li>DB2 iSeries：修复了加载长名称的模式</li> 
 <li>Hive：constraints definition 已修复</li> 
 <li>MySQL： 
  <ul> 
   <li>CTE 查询的执行计划解释已修复</li> 
   <li>修复了使用密码转义的问题</li> 
  </ul> </li> 
 <li>PrestoDB/Trino：改进了数据和元数据编辑器</li> 
 <li>PostgreSQL： 
  <ul> 
   <li>现在使用数据类型完全限定名称而不是比例/精度</li> 
   <li>自定义数据类型（包括几何）分辨率得到改进</li> 
  </ul> </li> 
 <li>SAP HANA：改进了存储过程结果加载器（多个游标 + atomic values）</li> 
 <li>Snowflake：改进了 $$ 解析器</li> 
 <li>SQL Server 
  <ul> 
   <li>Temporary table 支持得到改进（元数据加载、SQL 自动建议）</li> 
   <li>日期时间数据类型 DDL 已修复</li> 
   <li>Active schema selector 已修复</li> 
   <li>添加了 BEGIN/END 块支持</li> 
  </ul> </li> 
 <li>Vertica 
  <ul> 
   <li>Flex 表元数据已修复</li> 
   <li>Function body parser 已修复</li> 
  </ul> </li> 
</ul>
