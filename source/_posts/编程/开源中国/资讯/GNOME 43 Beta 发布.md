                                                                                            <p>GNOME 43 Beta 发布了，该版本带有最新的桌面增强功能和更多 GTK 4 / Adwaita 移植。</p> 
<p><strong>一些亮点</strong>包括：</p> 
<ul> 
 <li>GNOME Calls 应用程序的启动时间和滚动性能更快。</li> 
 <li>使用 DConf 编辑器支持深色方案。</li> 
 <li>Epiphany Web 浏览器删除了 PDF.js，添加了各种 WebExtensions 功能和其他更新。</li> 
 <li>GJS 正在发布来自 SpiderMonkey 102 的各种新 JavaScript 功能。</li> 
 <li>添加新的 GNOME 43 桌面背景。</li> 
 <li>GNOME 控制台已移植到 GTK 4 和 Adwaita 以及对这个新的 GNOME 终端替代品的其他改进。</li> 
 <li>GNOME 控制中心的蜂窝网络区域支持 WWAN 5G 连接，还支持来自 GNOME 控制中心的隐私屏幕配置。</li> 
 <li>GNOME 初始设置已移植到 GTK 4 和 Adwaita。</li> 
 <li>GNOME 日志已移植到 GTK 4。</li> 
 <li>对 GNOME 软件的 UI 更新和其他增强。</li> 
 <li>围绕 GNOME 文本编辑器中的文档加载进行了许多性能改进，还有桌面集成改进、UI 重新着色改进等等。</li> 
 <li>Nautilus 文件管理器在较小的屏幕尺寸、插件更改、属性窗口现代化等方面对主窗口元素进行了改进。</li> 
 <li>GNOME Mutter 改进，包括直接扫描改进、高分辨率滚轮支持、Wayland 上的单像素缓冲区支持等等。</li> 
 <li>GNOME Shell 改进，包括更好的概览动画性能、改进的应用程序网格导航、改进的屏幕键盘等等。 </li> 
</ul> 
<p>GNOME 43 处于功能、UI 和 API 冻结状态，稳定版预计在 9 月 21 日发布。</p> 
<p>通过<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fmail.gnome.org%2Farchives%2Fdevel-announce-list%2F2022-August%2Fmsg00002.html" target="_blank">发布公告</a>了解 GNOME 43 Beta 的更多详细信息。</p>
