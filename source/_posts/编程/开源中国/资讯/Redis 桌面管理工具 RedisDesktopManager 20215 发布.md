                                                        <p>RedisDesktopManager 2021.5 现已发布。RedisDesktopManager 是一个快速、简单、支持跨平台的 Redis 桌面管理工具，基于 Qt 5 开发，支持通过 SSH Tunnel 连接。新版本主要变化如下：</p> 
<p><strong>New features</strong></p> 
<ul> 
 <li><strong>Beta：</strong>支持 Apple Silicon（M1 芯片）ARM <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fuglide%2FRedisDesktopManager%2Fissues%2F4935" target="_blank">#4935</a></li> 
</ul> 
<p><strong>修复和改进</strong></p> 
<ul> 
 <li>为 Linux 和 Windows 添加 dark 模式开关</li> 
 <li>修复：关闭 ValueTab 时正确关闭连接</li> 
 <li>尽可能重用 ValueTab 的连接</li> 
 <li>修复 Server Stats 中的客户端列表解析</li> 
</ul>
