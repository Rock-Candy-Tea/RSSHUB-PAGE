                                                        <p>CherryTree 0.99.36 现已发布。CherryTree 是一个支持无限层级分类的笔记软件，Python 编写，支持富文本编辑和代码高亮，支持 Linux 和 Windows 平台。数据采用 sqlite 或 XML 存储，支持密码保护，支持从 NoteCase、KeepNote、Knowit、Tomboy、TuxCards、Treepad、Leo 等笔记软件导入数据。</p> 
<p><img height="342" src="https://static.oschina.net/uploads/space/2021/0127/070748_SLU0_2720166.png" width="500" referrerpolicy="no-referrer"></p> 
<p><img height="342" src="https://static.oschina.net/uploads/space/2021/0127/070803_tv03_2720166.png" width="500" referrerpolicy="no-referrer"></p> 
<p>此版本更新内容如下：</p> 
<ul> 
 <li>当配置为在 systray 上启动时，修复启动时的 segfault（in 0.99.34,0.99.35 - #1610, #1619）</li> 
 <li>更新了日语</li> 
</ul>
