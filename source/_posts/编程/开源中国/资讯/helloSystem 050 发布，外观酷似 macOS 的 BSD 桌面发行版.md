                                                        <p>helloSystem 是近期热度颇高的 BSD 桌面发行版之一，它提供了精良的桌面体验，希望成为 BSD 中的 macOS。</p> 
<p>helloSystem 项目具有与 macOS 相似的设计目标，即“能够正常工作”，不需要太多配置就可以带来良好的开箱即用的桌面体验。helloSystem 基于 FreeBSD，除了设计理念，helloSystem 桌面环境也已被配置为看起来像早期 MacOS X 的桌面。</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-dfbc41a41b6051eb5ffed69db628375a141.png" referrerpolicy="no-referrer"></p> 
<p>helloSystem 的开发人员不是要创建一个 macOS 的 1:1 副本，而是要与 macOS 的用户体验理念保持一致。因此 helloSystem 不能直接重新打包现有软件，为此他们还在开发更多专门用于 helloSystem 的桌面应用程序，其中许多应用程序都是用 PyQt 编写的。</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-59e1f65b22a6ee9af06055a6bd9662eeec5.png" referrerpolicy="no-referrer"></p> 
<p>近日，helloSystem 发布了 0.5 版本，此版本基于 FreeBSD 12.2，最大的变化是显著减少了 ISO 镜像文件的体积。除此之外，该系统的 macOS 风格桌面也有了许多改进，例如增加 MTP Android 文件传输工具、改进主题等。helloDesktop 堆栈还支持使用 KWin 而不是 Openbox，不过后者仍然是默认设置项。</p> 
<p>helloSystem 0.5 还修复了不少错误，如<code>sudo su</code> 现在可以正常使用、通知样式也已修复、错误字体大小已修复等变化。</p> 
<p>更多细节以及新版本的 ISO<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FhelloSystem%2FISO%2Freleases%2Ftag%2Fr0.5.0" target="_blank"> 查看 release note</a>。</p>
