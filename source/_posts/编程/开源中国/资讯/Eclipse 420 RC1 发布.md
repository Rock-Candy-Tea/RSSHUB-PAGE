<p>Eclipse 4.20 更新亮点：</p> 
<ul> 
 <li>支持 Mac AArch64(Arm64) 架构</li> 
 <li>将 Ant 升级至 1.10.10 版本</li> 
 <li>提供在 Linux 上禁用“插入表情符号”菜单项的选项</li> 
 <li>搜索视图区分正常导入和静态导入</li> 
</ul> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-2c7b6b611f66cd564c273b16e85cf66f97e.png" referrerpolicy="no-referrer"></p> 
<ul> 
 <li>在创建新 Java 项目期间增加 module-info.java 选项</li> 
</ul> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-e78f22db2b20014f01a940ed669ebdb5948.png" referrerpolicy="no-referrer"></p> 
<ul> 
 <li>Eclipse 的默认 Java 清理配置文件已更新</li> 
</ul>

 <li>“快速搜索”功能新增最大出现次数选项</li> 
</ul>
