                                                                                            <p>Apache Wicket 是一个开源的面向 Java 组件的 Web 应用框架，为政府、商店、大学、城市、银行、电子邮件提供商等成千上万的 Web 应用和网站提供支持。</p> 
<p>Wicket 的开发中使用了语义版本，因此与 9.0.0 相比，9.8.0 版本没有出现 API 中断。</p> 
<p>9.8.0 的更新内容包括：</p> 
<h3>错误修复</h3> 
<ul> 
 <li>[WICKET-6944] - WicketEndpoint 的内存泄漏</li> 
 <li>[WICKET-6945] - MultipartFormComponentListener 修改了不可见表单的 enctype，导致 JavaScript 错误</li> 
 <li>[WICKET-6947] - IndicatingAjaxButton 在 Bootstrap 4 上无法正常使用</li> 
 <li>[WICKET-6953] - JavaScriptDeferHeaderResponse 对 AJAX 请求不能正常响应</li> 
</ul> 
<h3>改进</h3> 
<ul> 
 <li>[WICKET-6943] - 能够检查当前会话的本地化设置是否为 RTL</li> 
 <li>[WICKET-6946] - isVisibleInHierarchy/isEnabledInHierarchy 被 isVisible/isEnabled 覆盖所破坏</li> 
 <li>[WICKET-6948] - 将 Spring 升级到最新的 5.3.x</li> 
 <li>[WICKET-6949] - 让 AjaxDownloadBehavior 更有动态性</li> 
 <li>[WICKET-6950] - wicket web-sockets 实现不允许在 onOpen 上配置/访问 web socket 会话</li> 
 <li>[WICKET-6952] - Strings.isEmpty 的性能改进</li> 
</ul> 
<h3>任务</h3> 
<ul> 
 <li>[WICKET-6942] - 用 slf4j-simple 取代 wicket-examples 和测试中 log4j 1.x 的使用</li> 
</ul>
