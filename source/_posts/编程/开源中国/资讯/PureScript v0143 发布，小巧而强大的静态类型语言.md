                                                        <p>PureScript  v0.14.3 发布了。PureScript 是个小巧而强大的静态类型语言，可以编译成 JavaScript。PureScript 主要是由 Haskell 和 PureScript 编写的。</p> 
<p><strong>New features：</strong></p> 
<ul> 
 <li>在文档中显示 kind signatures 及其 comments（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fpurescript%2Fpurescript%2Fpull%2F4100" target="_blank">#4100</a> 和 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fpurescript%2Fpurescript%2Fpull%2F4119" target="_blank">#4119</a>）</li> 
</ul> 
<p><strong>Bug 修复：</strong></p> 
<ul> 
 <li>确保未命名的实例出现在文档中（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fpurescript%2Fpurescript%2Fissues%2F4109" target="_blank">＃4109</a>）</li> 
 <li>在 REPL 中允许 fixity、kind、角色声明（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fpurescript%2Fpurescript%2Fpull%2F4046" target="_blank">#4046</a>）</li> 
 <li>CI 中使用的 Pin OS 版本（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fpurescript%2Fpurescript%2Fissues%2F4107" target="_blank">#4107</a>）</li> 
 <li>修复多个非递归 let 绑定的 UnusedName 警告（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fpurescript%2Fpurescript%2Fpull%2F4114" target="_blank">#4114</a>）</li> 
 <li>从关于实例的错误中删除生成的名称（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fpurescript%2Fpurescript%2Fpull%2F4118" target="_blank">＃4118</a>）</li> 
</ul> 
<p><strong>Internal：</strong></p> 
<ul> 
 <li>修正了 Haddock（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fpurescript%2Fpurescript%2Fissues%2F4072" target="_blank">＃4072</a>、<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fpurescript%2Fpurescript%2Fpull%2F4139" target="_blank">＃4139</a>）</li> 
 <li>更新 RELEASE_GUIDE.md，提供更多的细节（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fpurescript%2Fpurescript%2Fpull%2F4104" target="_blank">＃4104</a>）</li> 
 <li>将 GenIdent 用于匿名实例（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fpurescript%2Fpurescript%2Fpull%2F4096" target="_blank">#4096</a>）</li> 
 <li>type class desugaring pass 中的 desugar type class 实例名称（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fpurescript%2Fpurescript%2Fpull%2F4099" target="_blank">#4099</a>）</li> 
</ul>
