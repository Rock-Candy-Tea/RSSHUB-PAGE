                                                                                            <p>eSearch 是一款跨平台的识屏搜索软件，可通过开源的离线 Paddle OCR 或在线 OCR 对屏幕上的文字进行识别，并利用可自定义的搜索引擎对文字进行搜索、翻译。或者直接对屏幕上的图片进行以图搜图。它还拥有二维码扫描、滚动截屏、截屏标注、取色器、贴图、录屏等额外功能。</p> 
<p>目前 eSearch 1.7.3 版本已发布，带来如下内容：</p> 
<h2><strong>主要更新</strong></h2> 
<ul> 
 <li>修复部分系统下 ocr 错误</li> 
 <li>截屏界面插件</li> 
 <li>自定义用户数据路径</li> 
</ul> 
<h2><strong>新增</strong></h2> 
<ul> 
 <li>主要 
  <ul> 
   <li>关联 png、jpg、svg 打开方式</li> 
   <li>命令行支持打开文件参数</li> 
  </ul> </li> 
 <li>截屏 
  <ul> 
   <li>编辑 序号</li> 
   <li>编辑 箭头</li> 
   <li>导入 css 或 js 插件 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fesearch.vercel.app%2Fplugin.html" target="_blank">插件下载</a></li> 
  </ul> </li> 
 <li>设置 
  <ul> 
   <li>自定义用户数据路径</li> 
  </ul> </li> 
</ul> 
<h2><strong>修复</strong></h2> 
<ul> 
 <li>主页面 
  <ul> 
   <li>浏览器从最大化恢复后标签页窗口大小未适应</li> 
  </ul> </li> 
 <li>截屏 
  <ul> 
   <li>从命令行开启时截屏无效</li> 
   <li>部分情况下工具栏竖直方向超出屏幕</li> 
   <li>屏幕缩放时导出错位</li> 
   <li>部分系统下截屏窗口大小不应被更改</li> 
   <li>修复只点击不拖动时不正确框选</li> 
   <li>编辑 新建形状时有时会跳跃</li> 
  </ul> </li> 
 <li>OCR 
  <ul> 
   <li>离线 ocr 在部分 Windows 系统下报错找不到模型</li> 
  </ul> </li> 
 <li>以图搜图 
  <ul> 
   <li>使用谷歌识图时报错</li> 
  </ul> </li> 
</ul> 
<h2><strong>优化</strong></h2> 
<ul> 
 <li>所有 
  <ul> 
   <li>禁止选中图标</li> 
  </ul> </li> 
 <li>截屏 
  <ul> 
   <li>提高编辑边框的最大宽度</li> 
  </ul> </li> 
 <li>设置 
  <ul> 
   <li>ocr 自定义路径使用文件选择窗口选择</li> 
   <li>ffmpeg 路径使用文件选择窗口选择</li> 
  </ul> </li> 
</ul>
