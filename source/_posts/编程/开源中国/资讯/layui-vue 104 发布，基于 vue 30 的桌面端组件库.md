                                                                                            <p>更新内容:</p> 
<p><strong>1.0.4</strong></p> 
<p><span style="background-color:#ffffff; color:#40485b">[新增] line 组件 direction 属性, 支持垂直分割。</span><br> <span style="background-color:#ffffff; color:#40485b">[新增] line 组件 offset 属性, 设置内容的偏移量。</span><br> <span style="background-color:#ffffff; color:#40485b">[新增] notice-bar 组件 scrollable 属性, 支持垂直滚动。</span><br> <span style="background-color:#ffffff; color:#40485b">[新增] line 组件 border-style 属性, 设置分割线的样式。</span><br> <span style="background-color:#ffffff; color:#40485b">[新增] line 组件 border-wdith 属性, 设置分割线的宽度。</span><br> <span style="background-color:#ffffff; color:#40485b">[新增] line 组件 content-position 属性, 设置内容的对齐方式。</span><br> <span style="background-color:#ffffff; color:#40485b">[修复] table 组件 columns 配置 width 属性不生效。</span><br> <span style="background-color:#ffffff; color:#40485b">[修复] checkbox 组件夜间模式下边框颜色不兼容。</span><br> <span style="background-color:#ffffff; color:#40485b">[修复] carousel 组件 index 索引下标越界。</span><br> <span style="background-color:#ffffff; color:#40485b">[优化] layer 方法 option 参数为可选。</span><br> <span style="background-color:#ffffff; color:#40485b">[升级] vueuse/core 8.3.0 版本。</span><br> <span style="background-color:#ffffff; color:#40485b">[升级] vue 3.2.33 版本。</span><br> <span style="background-color:#ffffff; color:#40485b">[升级] vite 2.9.2 版本。</span></p> 
<p><span style="color:#40485b">往期更新：</span></p> 
<p><strong><span style="color:#40485b">1.0.3</span></strong></p> 
<p><span style="background-color:#ffffff; color:#40485b">[新增] ripple 组件, 提供 水波纹 效果。</span><br> <span style="background-color:#ffffff; color:#40485b">[新增] empty 组件 image 插槽, 支持自定义 image。</span><br> <span style="background-color:#ffffff; color:#40485b">[新增] badge 组件 ripple 属性, 开启水波纹动画, 仅适用于 type 为 dot 时。</span><br> <span style="background-color:#ffffff; color:#40485b">[新增] table 组件 column 配置 align 属性, 支持 left center right 对齐方式。</span><br> <span style="background-color:#ffffff; color:#40485b">[新增] upload 组件 disabled-preview 属性, 上传预览区域添加禁用状态。</span><br> <span style="background-color:#ffffff; color:#40485b">[修复] upload 组件 cut 属性, 开启时丢失图片信息。</span><br> <span style="background-color:#ffffff; color:#40485b">[修复] table 组件 column 存在 1px 宽度误差。</span><br> <span style="background-color:#ffffff; color:#40485b">[优化] date-picker 组件 monent 为 day.js。</span><br> <span style="background-color:#ffffff; color:#40485b">[优化] table 组件 html document 结构。</span><br> <span style="background-color:#ffffff; color:#40485b">[移除] xlsx 依赖, 减少打包体积</span></p> 
<p><span style="color:#40485b">更多详情：http://www.layui-vue.com</span></p>
