                                                                                            <p>Zotero 是一个参考文献管理软件，也是一个免费 Firefox 和 Google 浏览器扩展。Zotero 可以协助用户收集、管理及引用研究资源，包括期刊、书籍等各类文献和网页、图片等。</p> 
<p>目前，Zotero 6.0.9 已发布， 该版本更新内容如下：</p> 
<p><strong>功能更改或添加</strong></p> 
<ul> 
 <li>从 PDF 阅读器侧边栏的缩略图选项卡添加 PDF 页面旋转和删除</li> 
</ul> 
<p><strong>已修复的错误</strong></p> 
<ul> 
 <li> <p>修复了 ScienceDirect PDF 的下载</p> </li> 
 <li> <p>修复了 library tab tooltip 在切换应用程序后保持可见</p> </li> 
 <li> <p>修复了对一些设置为彩色标签的表情符号的检测</p> </li> 
 <li> <p>修复了中文和其他 non-ASCII 月份名称的解析</p> </li> 
 <li> <p>[Mac] 修复了拖拽过程中的强制点击有时会导致意外的 collection switching</p> </li> 
 <li> <p>[Mac] 修复了在 macOS 上编辑集合名称时选项箭头不起作用的问题</p> </li> 
 <li> <p>修复了按附件列排序的问题</p> </li> 
 <li> <p>在笔记编辑器中聚焦搜索栏时选择搜索输入文本</p> </li> 
 <li> <p>修复了在包含重音字符的注释中突出显示搜索结果的位置</p> </li> 
 <li> <p>修复了 Preprint item type 中 Archive ID 到 CSL 编号字段的映射问题</p> </li> 
 <li> <p>修复了纯文本输出中某些数字样式（IEEE、Nature）的引文编号后不出现空格的问题</p> </li> 
</ul> 
<p>更新说明：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.zotero.org%2Fsupport%2Fchangelog" target="_blank">https://www.zotero.org/support/changelog</a></p>
