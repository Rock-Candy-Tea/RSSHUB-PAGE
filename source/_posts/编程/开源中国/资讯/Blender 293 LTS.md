                                                        <p>Blender 基金会已经发布了 Blender 2.93，这是这个开源 3D 软件的最新更新。作为该软件最新的长期支持（LTS）版本，其功能列表并不像最近的一些版本那样丰富，现在的开发重点是今年 8 月将推出的 Blender 3.0。</p> 
<p>Blender 2.93 LTS 更新亮点包含：</p> 
<ul> 
 <li><strong>几何节点的新工作区和电子表格编辑器</strong></li> 
</ul> 
<p>在 Blender 2.92 中引入，最初主要局限于物体散射，在最新的版本中，它向功能更全面的程序化建模工具集又迈进了一步。</p> 
<p>除了增加了 20 多个新节点——包括用于生成 2D 和 3D 基元的节点，现在还可以访问网格属性，如顶点颜色数据、一些 UV 属性和材料分配。</p> 
<p>这次更新还为几何节点增加了一个默认的工作区，包括一个新的电子表格编辑器，用于显示网格、point clouds 和实例的属性值，以及更好的错误报告。</p> 
<ul> 
 <li><strong>雕刻时遮蔽模型的新方法</strong></li> 
</ul> 
<p>新的雕刻功能是 Blender 最近版本的一个亮点。在 2.93 中，主要的变化是新的 "扩展"（Expand）操作符，旨在作为掩盖雕塑表面的部分的主要工具。</p> 
<p>与之前的版本不同，新的操作符能够一次性创建多个遮罩形状，而且现在能够检测到网格边界，使得创建边缘与几何体平行的遮罩成为可能。</p> 
<p>除了创建 Face Sets 组来控制几何体的可见性外，新的运算器还可以修改现有的 Face Sets 组，并可以用来控制顶点颜色，包括应用表面颜色梯度的选项。</p> 
<ul> 
 <li><strong>在 Cycles 和 Eevee 中更快、更高质量的渲染</strong></li> 
</ul> 
<p>Blender 的两个主要渲染引擎都在 Blender 2.93 中得到了更新，实时渲染引擎 Eevee 在景深和环境遮挡方面得到了完全的重写，并且在体积内有更好的阴影。</p> 
<p>然而，对于生产来说，更重要的变化可能是 Cycles，它得到了一个新的选项可以保留一个场景的全部渲染数据，加快了重新渲染的速度，代价是增加了内存的使用。</p> 
<p>在管线集成方面，Cycles 成为支持 OCIO 2 的最新渲染引擎，这是 OpenColorIO 色彩管理标准的最新版本，也被 Arnold 和 OctaneRender 所采用。</p> 
<p>在 Blender 2.93 中，Grease Pencil 工具集中的填充工具被重构，获得了一次性填充多个类似框架的能力，包括目前位于 viewport 之外的框架部分。</p> 
<ul> 
 <li><strong>对 Grease Pencil 2D 动画工具集的重大更新</strong></li> 
</ul> 
<p>Grease Pencil 是 Blender 的 2D 动画工具集，本次更新得到了重大更新，包括一个新的 LineArt 修改器，它可以从 3D 几何体的轮廓或内部产生 Grease Pencil 笔触。</p> 
<p>它不仅可以用于静态图像和动画，现在还可以导入 SVG 格式的矢量图形，并将其转换为 Grease Pencil" 笔触，并以 SVG 或 PDF 格式导出 "Grease Pencil" 笔触。</p> 
<p>此外，插值工具现在可以在不同点数的笔画之间进行插值，而填充工具可以一次性填充多个类似的框架，包括 Viewport 外的区域。</p> 
<ul> 
 <li><strong>新的库覆盖系统的进一步进展</strong></li> 
</ul> 
<p>Blender 的库覆盖系统，旨在为从事复杂生产场景的工作室提供一个更强大的替代旧的代理系统，又得到了更新。</p> 
<p>Blender 2.93 中的变化包括当链接文件更新时自动重新同步覆盖，在 Outliner 中显示覆盖，以及系统的 "最终文档"。</p> 
<ul> 
 <li><strong>其他</strong></li> 
</ul> 
<p>Blender 的内置合成器得到了一个新的抗锯齿节点和更好的 Cryptomatte 工作流程，使得它可以直接从 Cryptomatte 节点选择图像或渲染层。</p> 
<p>glTF 导入器/导出器以及 BlenderKit 和 Collection Manager 扩展也得到了更新。</p> 
<ul> 
 <li><strong>系统要求</strong></li> 
</ul> 
<p>Blender 2.93 可用于 Windows 8.1+、macOS 10.13+ 和 Linux。Blender 基金会现在已经放弃了对 Windows 7 操作系统的支持。</p>
