                                                                                            <p><strong>Java通用代码生成器光文明Beta版，vscode编译运行PostgreSQL代码生成物视频</strong></p> 
<p>Java通用代码生成器光2.3.0 文明版本发布第一个Beta版，文档更新，支持数据库脚本中文注释。可以使用vscode编译运行代码生成物。并已发布介绍视频。请见：<br> <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.bilibili.com%2Fvideo%2FBV1z34y1Y77Q%2F" target="_blank">https://www.bilibili.com/video/BV1z34y1Y77Q/</a></p> 
<p>数据库脚本的中文注释取域对象的标签文本，由一个复选框开关，在所有四种数据库和两个生成界面和两个模式中均可使用。基于用户的真实需求。另，Beta更新了文档。<br> Java通用代码生成器光2.3.0 文明版本尝鲜版十九，支持生成Vue和Element UI前端的复杂版面功能。现在，已经支持父子表，树表和树父子表三种复杂版面。<br> 您只需要编写一个符合规范的Excel模板文件，系统会把Excel模板编译为一个后端项目和一个前端项目。两个项目都会自动保留原来的Excel模板文件。<br> 请把光的war包部署在Tomcat9或8.5的webapps目录下。<br> 现在文明版本支持Echarts图形报表，可配置，一键生成。 支持三种报表，图形报表，带数据网格图形报表和带双数据网格的对比图形报表。支持折线图，柱状图和饼图三种图形报表，支持原始数据和累加数据两种数据格式。<br> 本版本三个技术栈的均可跨域，经过详细测试，是比较完善的版本。修复了尝鲜版8以来所有已知的错误。独立前端经过仔细的开发和测试，可用性大大增强。您可以使用此Vue + ElementUI的前端。<br> 最新版本最大特色是一键生成前端和后端，即一键生成双系统，共享一套登录权限系统，session,token等信息不需要人工设置，默认生成，前端是基于Vue的，您可以使用此独立Vue前端管理系统。等前端项目生成完成复杂版面功能后，即可进入Beta阶段。<br> Java通用代码生成器光文明版本尝鲜版十八已公布。光乃是原型神器。可以瞬间将符合光的模板规范的Excel模板秒变为一套可以运行的双系统。非常神奇。它的基本理念是利用一种拥有超越高级语言抽象程度的超级语言SGS2来描绘双系统，这种语言很大程度是域对象驱动的。通过域对象和动词算子的结合描绘万事万物。光2.3.0文明新增了弹性登录模块，复杂版面和图形报表。由此，代码生成物的完成度有了显著提升。对超级语言而言，模板和系统是硬币的两面，正象对高级语言而言，系统和源代码的关系那样。<br> 完善了模板向导界面，现在，模板向导界面已完善支持新功能。您可以使用模板向导界面生成和操作Excel模板，非常方便。<br> 完善了对Oracle数据库的支持，修复了一系列错误。<br> 新增了PPT格式数据导出。<br> 现在，光支持Excel,PDF,Word和PPT共四种数据导出格式。未来的几个尝鲜版将有精彩重要的功能增强。敬请期待。请参考B站视频，完整演示了使用一套Excel模板生成和运行双系统。</p>
