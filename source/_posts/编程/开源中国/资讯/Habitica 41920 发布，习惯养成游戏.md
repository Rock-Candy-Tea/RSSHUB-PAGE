                                                        <p>Habitica 4.192.0 发布了。Habitica 是一个习惯养成及生产力应用，让你“游戏人生”。游戏里的奖惩措施能激励用户完成任务，还有一个强大的互动社区为完成任务提供好建议，帮助达成目标，变得健康、勤奋与快乐。</p> 
<p>此版本更新内容如下：</p> 
<p><strong>API</strong></p> 
<ul> 
 <li> <p>Feature：添加路线和处理以对小组计划和挑战任务进行排序</p> </li> 
 <li>Fix：允许用户发起任务以接收 Webhook<code>questInvited</code></li> 
 <li>Fix：在 production 时，强制用户 URL 到 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fhabitica.com%2F" target="_blank">https://habitica.com</a></li> 
 <li>Fix：对 public user profiles 的更新进行违禁词检查</li> 
 <li>Fix：为 Challenge prizes 强制执行 non-null（可以为零）</li> 
 <li>Chore：Locale files 已更新</li> 
</ul> 
<p><strong>Client</strong></p> 
<ul> 
 <li> <p>Feature：在 Vue 文件中通过 Babel 启用<code>?</code>属性链接</p> </li> 
 <li> <p>Feature：允许拖放编辑小组计划和挑战赛的任务列表</p> </li> 
 <li>Fix：大修与任务相关的对话框以更好地符合设计</li> 
 <li>Fix：来自挑战的标签可以在标签编辑弹出窗口中排序</li> 
 <li>Fix：如果不允许用户在该 Party/Guild 中创建挑战，则不要显示“Create Challenge”按钮</li> 
 <li>Fix：允许在聊天文本中的任意位置提及@，而不仅仅是结尾，并在从下拉列表中进行选择后恢复 focus</li> 
 <li>......</li> 
</ul> 
<p><strong>Repo</strong></p> 
<ul> 
 <li> <p>Chore：每个 Dependabot 的软件包更新</p> </li> 
</ul>
