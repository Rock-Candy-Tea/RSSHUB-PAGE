<div class="content">
                                                                                            <p><span style="background-color:#ffffff"><span style="color:#000000">DBeaver 21.3.0 现已发布。DBeaver 是一个免费开源的通用数据库工具，适用于开发人员和数据库管理员。</span></span></p> 
<p><strong><span style="color:#000000">主要更新内容</span></strong></p> 
<ul> 
 <li><span style="color:#000000">数据查看器</span> 
  <ul> 
   <li><span style="color:#000000">过滤器配置对话框 UI 已修复（MacOS）</span></li> 
   <li><span style="color:#000000">发现事件突出显示已修复（查找/替换）</span></li> 
   <li><span style="color:#000000">长列调整大小的问题已修复</span></li> 
   <li><span style="color:#000000">重叠列图标的问题已修复</span></li> 
  </ul> </li> 
 <li><span style="color:#000000">图表查看器</span> 
  <ul> 
   <li><span style="color:#000000">​​​​​​​连接自动布局已修复</span></li> 
  </ul> </li> 
 <li><span style="color:#000000">在 MacOS 上修复了 SQL 脚本预览对话框的问题</span></li> 
 <li><span style="color:#000000">在 Windows 上修复了错误 "PKIX 路径构建失败"（操作系统证书存储现在是默认值）</span></li> 
 <li><span style="color:#000000">深色主题：MacOS 上修复了颜色不匹配的错误</span></li> 
 <li><span style="color:#000000">缺少文件扩展名的问题已修复（Windows）</span></li> 
 <li><span style="color:#000000">元数据编辑器</span> 
  <ul> 
   <li><span style="color:#000000">修复了模式更改后编辑器闪烁的问题</span></li> 
  </ul> </li> 
 <li><span style="color:#000000">全文搜索现在保留原始表格过滤器</span></li> 
 <li><span style="color:#000000">添加了中断项目自动恢复功能</span></li> 
 <li><span style="color:#000000">旧错误/警告图标已替换为新图标</span></li> 
 <li><span style="color:#000000">版本自动升级</span> 
  <ul> 
   <li><span style="color:#000000">​​​​​​​错误的包类型检测已修复</span></li> 
   <li><span style="color:#000000">错误的架构 (M1/ARM/x86) 检测已修复</span></li> 
  </ul> </li> 
 <li><span style="color:#000000">驱动编辑器</span> 
  <ul> 
   <li><span style="color:#000000">​​​​​​​为所有驱动程序添加了 URL 模板编辑支持</span></li> 
   <li><span style="color:#000000">Maven 工件导入/编辑 UI 得到改进</span></li> 
  </ul> </li> 
 <li><span style="color:#000000">SSH 隧道测试：添加了系统变量支持</span></li> 
 <li><span style="color:#000000">PostgreSQL</span> 
  <ul> 
   <li><span style="color:#000000">​​​​​​​重新设计了数据库结构（所有集群级别的实体都移到了顶层）</span></li> 
   <li><span style="color:#000000">添加了多范围数据类型支持</span></li> 
   <li><span style="color:#000000">架构过滤器支持已修复</span></li> 
   <li><span style="color:#000000">添加了对唯一键中的 JSON 列的支持</span></li> 
  </ul> </li> 
</ul>
