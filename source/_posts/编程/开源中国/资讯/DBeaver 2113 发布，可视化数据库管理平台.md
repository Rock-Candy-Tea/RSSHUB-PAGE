                                                        <p>DBeaver 21.1.3 发布，该版本更新内容如下：</p> 
<ul> 
 <li>SQL 编辑器： 
  <ul> 
   <li>修改了输出/执行日志面板布局；</li> 
   <li>增加了变量/参数管理面板；</li> 
   <li>修复了 SQL 模板变量的补全；</li> 
   <li>增加了将查询结果分配给变量的命令；</li> 
  </ul> </li> 
 <li>数据传输： 
  <ul> 
   <li>为表列映射增加了自定义转换功能；</li> 
   <li>为 MySQL、SQL Server 和 Exasol 增加了参考完整性禁用功能；</li> 
   <li>列映射页面现在支持重新排序；</li> 
   <li>修复了数据导出任务编辑器；</li> 
   <li>基于查询的数据导出任务现在支持查询编辑；</li> 
   <li>修复了 HTML 格式的导出；</li> 
   <li>修正了列自动分配按钮的行为；</li> 
   <li>数据传输后的执行过程现在支持多表导出；</li> 
   <li>修复了混合大小写列的自动映射问题；</li> 
   <li>修复了编辑器打开时出现 NullPointerException 的问题；</li> 
  </ul> </li> 
 <li>导航器： 
  <ul> 
   <li>打开文件：最后打开的文件夹被保存；</li> 
   <li>对象过滤器：增加了特殊字符的转义；</li> 
  </ul> </li> 
 <li>MySQL： 
  <ul> 
   <li>数据导入性能得到显著改善；</li> 
  </ul> </li> 
 <li>Oracle： 
  <ul> 
   <li>视图索引的读取被修复；</li> 
   <li>改进了 TNS 别名的解析；</li> 
   <li>修复了以 SQL 格式导出数据的问题；</li> 
  </ul> </li> 
 <li>PostgreSQL： 
  <ul> 
   <li>修复了自定义格式的数据库备份/恢复问题；</li> 
   <li>数据库备份验证了输出文件的存在；</li> 
  </ul> </li> 
 <li>Firebird：修正了查询执行计划的解释；</li> 
 <li>修复了几个小的用户界面错误；</li> 
</ul>
