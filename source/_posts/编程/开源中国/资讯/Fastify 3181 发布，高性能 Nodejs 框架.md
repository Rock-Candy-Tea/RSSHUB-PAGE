                                                        <p>Fastify 3.18.1 发布了，Fastify 是一个高度专注于以最少开销和强大的插件架构为开发者提供最佳体验的 Node.js Web 框架，速度极快，它的灵感来源于 Hapi 和 Express。</p> 
<p>此版本更新内容包括：</p> 
<p><strong>PR</strong></p> 
<ul> 
 <li>docs：在 hooks 中添加早期响应通知（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffastify%2Ffastify%2Fissues%2F3135" target="_blank">#3135</a>）（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffastify%2Ffastify%2Fpull%2F3138" target="_blank">#3138</a>）</li> 
 <li>chore：添加基准工作流以按标签运行（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffastify%2Ffastify%2Fpull%2F3139" target="_blank">#3139</a>）</li> 
 <li>docs：更新 onReady hooks 的描述（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffastify%2Ffastify%2Fpull%2F3136" target="_blank">#3136</a>）</li> 
 <li>更新 FST_ERR_CTP_BODY_TOO_LARGE 描述（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffastify%2Ffastify%2Fpull%2F3142" target="_blank">#3142</a>）</li> 
 <li>更新 TypeScript 定义：http2SessionTimeout ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffastify%2Ffastify%2Fpull%2F3144" target="_blank">#3144</a> )</li> 
 <li>fix(types)：在封装的 hooks 中为<code>this</code>添加缺失的类型（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffastify%2Ffastify%2Fpull%2F3143" target="_blank">#3143</a>）</li> 
 <li>更新 Server.md ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffastify%2Ffastify%2Fpull%2F3147" target="_blank">#3147</a> )</li>

 <li>更新 Ecosystem.md ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffastify%2Ffastify%2Fpull%2F3148" target="_blank">#3148</a> )</li> 
 <li>修复格式化错误（以及随后的网站构建失败）（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffastify%2Ffastify%2Fpull%2F3152" target="_blank">#3152</a>）</li> 
 <li>更新 ContentTypeParser.md ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffastify%2Ffastify%2Fpull%2F3159" target="_blank">#3159</a> )</li>
