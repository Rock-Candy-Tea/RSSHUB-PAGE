                                                        <p>NativeScript 8.1.4 现已发布。NativeScript 使用户能够直接从 JavaScript 访问 native API。目前，该框架为丰富的移动开发提供了 iOS 和 Android 运行时，并可用于多种不同的用例。</p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">主要更新内容如下：</p>

<ul>

 <li>背景解析颜色<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FNativeScript%2FNativeScript%2Fissues%2F9559" target="_blank">#9559</a> ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FNativeScript%2FNativeScript%2Fissues%2F9560" target="_blank">#9560</a> ) ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FNativeScript%2FNativeScript%2Fcommit%2F3e21748af4dcf76fec3822b4d5e9c844794eda1d" target="_blank">3e21748</a> )</li>
