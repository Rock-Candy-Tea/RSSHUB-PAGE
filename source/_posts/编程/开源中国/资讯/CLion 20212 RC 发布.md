                                                        <p>CLion 2021.2 RC 现已发布。该本现在可从官方网站、工具箱应用程序或作为 Ubuntu 的 snap 包获得，并且需要有有效订阅或试用许可证。</p> 
<p><strong>主要更新内容</strong></p> 
<ul> 
 <li>修复多光标模式下的奇怪行为</li> 
 <li>修复 LLVM 合并后的 clang 格式方案规则</li> 
 <li>修复语法高亮在一个小项目中需要很长时间才能显示出来的问题</li> 
 <li>无法在预配置命令脚本中插入新行</li> 
 <li>yasm/nasm 调用的对象 -f 和调试 -g 格式参数被错误地视为输入文件</li> 
 <li>添加对 GNU Autotools（Autoconf、Automake）项目的支持</li> 
 <li>从 Python 迁移 macOS 环境阅读器</li> 
 <li>当 diff 已经被打开时，单独窗口中的 diff 不会在显示 diff 的调用中得到关注</li> 
 <li>修复 Scratch Files 弹出窗口中的排序问题</li> 
 <li>创建临时文件时忽略自定义文件类型</li> 
 <li>在某些情况下，从 Find Results 调用的 “New Scratch File” 会创建空文件</li> 
 <li>无法在 Docke 中禁用 Run Built image 选项</li> 
 <li>所有文档都应该在 Docker Compose 操作之前保存</li> 
 <li>弹出窗口中的项目未显示在触摸栏上</li> 
 <li>单击后 “文件” 菜单有时无法打开</li> 
 <li>添加注册表项以将 Code With Me 切换到 OSR 模式</li> 
 <li>GPG 密钥无法正确显示所有者名称</li> 
</ul> 
<p>详情请查看<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fblog.jetbrains.com%2Fclion%2F2021%2F07%2Fclion-2021-2-rc%2F" target="_blank">更新公告</a>。</p>
