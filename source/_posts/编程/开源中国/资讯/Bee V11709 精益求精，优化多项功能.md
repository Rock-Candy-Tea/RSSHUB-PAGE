                                                                                            <p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><strong style="color:#333333">ORM Bee</strong><span style="background-color:#ffffff; color:#40485b"><span> </span><strong>简单易用，文件小，性能好</strong>；</span><strong style="color:#333333">支持 JDBC，</strong><span style="background-color:#ffffff; color:#40485b">同时支持</span><strong style="color:#333333"><span> </span>Android 和 Harmony</strong><strong style="color:#333333">。</strong></p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><strong style="color:#3498db">Bee，互联网新时代的 Java ORM 工具，更快、更简单、更自动，开发速度快，运行快，更智能！</strong></p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><strong style="color:#40485b">V1.17.0.9(精益求精·七夕)</strong><br> <span style="background-color:#ffffff; color:#40485b">1)字段名称引用类(默认格式:实体名_F(自动生成))增加ALL_NAMES属性,可一次获取实体的所有字段值</span><br> <span style="background-color:#ffffff; color:#40485b">2)Ddl.java支持创建<strong>索引</strong>(normal,unique),<strong>联合主键</strong></span><br> <strong><span style="background-color:#ffffff; color:#40485b">3)condition.op(fieldName, Op.in, Value)增加支持List,Set,Number Array,单个Number元素</span></strong><br> <span style="background-color:#ffffff; color:#40485b">4)condition.opOn(fieldName, Op.in, Value) Value限定只支持Number和String</span><br> <span style="background-color:#ffffff; color:#40485b">5)增强:ExcelReader数据列数目动态计算</span><br> <span style="background-color:#ffffff; color:#40485b">6)增强:like;Op添加likeLeft,likeRight,likeLeftRight(参数值由框架负责转义);打印SQL日志作相应转义</span><br> <span style="background-color:#ffffff; color:#40485b">7)增强:GenBean生成Javabean,当id是BigDecimal时,重置为Long型</span><br> <span style="background-color:#ffffff; color:#40485b">8)动态获取JdbcToJavaType</span><br> <span style="background-color:#ffffff; color:#40485b">9)命名转换增加种类4(DbUpperAndJavaLower):数据库使用大写字母，Java使用小写字母;忽略大小写,使用的字符是一样的</span><br> <span style="background-color:#ffffff; color:#40485b">10)多种命名时,缓存添加TranslateType部分</span><br> <span style="background-color:#ffffff; color:#40485b">11)事务注解@Tran,支持在类级别使用</span><br> <span style="background-color:#ffffff; color:#40485b">12)@Ignore(@Transient)可以兼容JPA相应注解(在AnnoAdapter接口定义)</span><br> <span style="background-color:#ffffff; color:#40485b">13)fixed bug:level 2缓存判断;TypeHandler返回值类型转换</span><br> <span style="background-color:#ffffff; color:#40485b">14)增强:SQLite日期类型(date)匹配转换支持</span></p>

<p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><strong style="color:#333333">Bee</strong><span style="background-color:#ffffff; color:#333333"> 是一个简单，易用，功能强大，开发速度快，编码少的 JAVA ORM 框架。连接，事务都可以由 Bee 框架负责管理.<span> </span></span><strong style="color:#333333">Bee 简化了与 DB 交互的编码工作量，是 </strong><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fbaike.baidu.com%2Fitem%2F%25E7%25BC%2596%25E7%25A0%2581%25E5%25A4%258D%25E6%259D%2582%25E5%25BA%25A6%2F23229411%3Ffr%3Daladdin" target="_blank">编码复杂度</a> 为<span> </span><strong><span style="color:#c0392b">O(1)</span><span> </span></strong>的 Java 框架！</p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><span style="background-color:#ffffff; color:#333333">Bee 简单易用：单表操作、多表关联操作，可以不用写 sql, 极少语句就可以完成 SQL 操作；</span><strong style="color:#333333">概念简单</strong><span style="background-color:#ffffff; color:#333333"><span> </span>,10 分钟即可入门。</span><br> <span style="background-color:#ffffff; color:#333333">Bee 功能强大：<strong>复杂查询也支持向对象方式，分页查询性能更高</strong>，一级缓存即可支持个性化优化；具有分布式特性。</span><strong><span style="background-color:#ffffff">高级要求，还可以方便自定义 SQL 语句</span></strong><span style="background-color:#ffffff">。</span></p>
