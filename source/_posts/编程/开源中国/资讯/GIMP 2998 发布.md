                                                        <p><span><span><span style="color:#555555"><span><span><span><span><span><span><span><span><span><span><span><span style="background-color:#ffffff"><span><span><span>GIMP</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span> 2.99.8 现已发布，此版本再次带来了大量改进。</p> 
<p>GIMP 即 GNU Image Manipulation Program（GNU 图像处理程序）的首字母组成，是一个自由开源的位图图像编辑器，用于图像照片润饰及编辑、自由绘图、调整大小、裁剪、照片蒙太奇、装换图像格式以及其他专业任务。GIMP 几乎拥有所有图象处理所需的功能，号称 Linux 下的 Photoshop。 </p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><img alt src="https://attachments.tower.im/tower/15180ec23f5663d27e61be5826e69660?version=auto" referrerpolicy="no-referrer"><img alt height="281" src="https://oscimg.oschina.net/oscnet/up-54c3d7d008e3b57a4ae7dedf1b30be52abf.jpg" width="500" referrerpolicy="no-referrer"></p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><strong>新版本更新亮点有：</strong></p> 
<ul> 
 <li> 在 multiple layers 上的 Clone-type 工具</li> 
 <li>修复了 Wayland 和 macOS 上的 Selection cue</li> 
 <li>由于 Windows Ink 支持，输入设备的覆盖范围更广</li> 
</ul> 
<p><img alt height="366" src="https://oscimg.oschina.net/oscnet/up-5b5386d3a83a91c7ff15e34a5a7f1dd95d2.png" width="500" referrerpolicy="no-referrer"></p> 
<ul> 
 <li>Canvas-focus by toolbox clicking</li> 
 <li>删除 thumbnail icon</li> 
 <li>改进的文件格式支持：<span>JPEG</span> - <span>XL</span>、<span>PSD</span>/<span>PSB </span>等</li> 
</ul> 
<p><img alt height="219" src="https://oscimg.oschina.net/oscnet/up-5200f18f313f377f31d72f033c6db3a945b.jpg" width="500" referrerpolicy="no-referrer"></p> 
<ul> 
 <li>插件开发</li> 
 <li>内存泄漏修复</li> 
 <li>持续集成变化 
  <ul> 
   <li>Windows 
    <ul> 
     <li>开发安装程序“nightlies”</li> 
     <li>自动发布安装程序</li> 
     <li>安装程序脚本的一致性</li> 
    </ul> </li> 
   <li>Linux</li> 
   <li>MacOS</li> 
   <li>merge requests 的自动构建</li> 
  </ul> </li> 
 <li>更新的编码风格指南</li> 
 <li><span>GEGL </span>和 babl</li> 
</ul> 
<p><span style="color:#000000"><span style="background-color:#ffffff">官方表示，v2.99.8 是一个大版本更新；自 2.99.6 以来共关闭了 50 多个 reports，接受了 25 个合并请求和超过 500 次 commits。目前，开发团队仍在致力于完成 GTK3 移植以及新的插件 API。不过最近发生的一些技术变化（尤其是 Linux 和 macOS 上的 Wayland）也对团队的开发效率造成了相当大的影响，导致其花费了很多时间来修复那些因为底层系统变化而被破坏的东西。</span></span></p>
