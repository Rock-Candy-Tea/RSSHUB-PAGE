                                                        <p>Tethys IM Server V1.0.0-B1 已经发布，响应式 IM 服务端。</p> 
<p>此版本更新内容包括：</p> 
<h2>更新功能</h2> 
<ul> 
 <li>新增 HTTP 发送消息接口</li> 
 <li>新增 gRPC 发送消息接口</li> 
 <li>完善 Server to Server 对接 API 
  <ul> 
   <li>完善同步用户 ID 接口</li> 
   <li>完善同步群组 ID 接口</li> 
   <li>完善群组用户绑定接口</li> 
   <li>完善访问令牌获取接口</li> 
  </ul> </li> 
 <li>优化消息存储表结构设计</li> 
 <li>删除消息包中无用的字段设计</li> 
 <li>规范化私聊、群聊消息包设计</li> 
 <li>修复大量 sonarlint 规范性代码错误</li> 
 <li>补充大量单元测试</li> 
</ul> 
<h2>文档更新</h2> 
<ul> 
 <li><a href="https://gitee.com/kk70/tethys/blob/main/docs/dev/index.md" target="_blank">新增开发手册</a></li> 
 <li><a href="https://gitee.com/kk70/tethys/blob/main/docs/design/message_id.md" target="_blank">完善分布式消息 ID 设计</a></li> 
 <li><a href="https://kk70.gitee.io/tethys/tethys-rest.html" target="_blank">完善 OpenAPI 接口文档</a></li> 
</ul>
