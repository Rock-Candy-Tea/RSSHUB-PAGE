                                                        <p>Apache Tomcat 10.1.0-M1 已发布。</p> 
<p>Apache Tomcat 10.1.0-M1 是 10.1.x 的第一个里程碑版本分支，已为用户提供早期访问 Apache Tomcat 10.1.x 中新的功能。与 10.0.x 相比，此版本的主要变化包括：</p> 
<ul> 
 <li>删除此前在 10.1.x 中标记为“移除”的代码（APR/Native Connector 除外）。因为 APR/Native Connector 几乎肯定会在未来的里程碑版本中被移除</li> 
 <li>使 Servlet API 的实现与当前 Servlet API 的开发分支保持一致</li> 
 <li>使 EL API 的实现与当前 El API 的开发分支保持一致</li> 
</ul> 
<p><a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Ftomcat.apache.org%2Ftomcat-10.1-doc%2Fchangelog.html" target="_blank">点此查看完整变更日志</a>。</p> 
<p>下载地址：<a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Ftomcat.apache.org%2Fdownload-10.cgi" target="_blank">http://tomcat.apache.org/download-10.cgi</a></p> 
<p>从 Apache Tomcat 7.0.x, 8.5.x 和 9.0.x 进行升级的迁移手册：<a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Ftomcat.apache.org%2Fmigration.html" target="_blank">http://tomcat.apache.org/migration.html</a></p>
