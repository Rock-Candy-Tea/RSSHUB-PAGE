<p><strong>添加</strong></p> 
<ul> 
 <li>加入 <code>reduceMany</code> 到集合中 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Flaravel%2Fframework%2Fpull%2F39078" target="_blank">#39078</a> )</li> 
 <li>加入 <code>Illuminate/Support/Stringable::stripTags()</code>（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Flaravel%2Fframework%2Fpull%2F39098" target="_blank">#39098</a><span style="color:#24292f">）</span></li> 
 <li>加入 <code>Illuminate/Console/OutputStyle::getOutput()</code>(<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Flaravel%2Fframework%2Fpull%2F39099" target="_blank">#39099</a><span style="color:#24292f">）</span></li> 
 <li>加入 <code>lang_path</code> 辅助函数 （<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Flaravel%2Fframework%2Fpull%2F39099" target="_blank">#39099</a>）</li> 
 <li>加入 <code>@aware</code> blade 指令  ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Flaravel%2Fframework%2Fpull%2F39100" target="_blank">#39100</a> )</li> 
 <li>加入新事件 <code>JobRetrying</code>( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Flaravel%2Fframework%2Fpull%2F39097" target="_blank">#39097</a> )</li> 
 <li><span style="background-color:#ffffff; color:#24292f">在客户端响应中加入 <code>throwIf</code> 方法 </span>( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Flaravel%2Fframework%2Fpull%2F39148" target="_blank">#39148</a> )</li> 
 <li>加入了 <code>hasAny()</code> 到集合中 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Flaravel%2Fframework%2Fpull%2F39155" target="_blank">#39155</a> )</li> 
</ul> 
<p><strong>修复</strong></p> 
<ul> 
 <li>修复了<span style="background-color:#ffffff; color:#24292f"><span> </span>PHP 8.1 上没有前缀名的分组路由</span></li> 
 <li>修复了在 InteractsWithInput 中查找 token 令牌的代码（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Flaravel%2Fframework%2Fpull%2F39150" target="_blank">#39150</a>）</li> 
</ul>

<ul> 
 <li>重构 <code>Illuminate/Log/LogManager::prepareHandler()</code>( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Flaravel%2Fframework%2Fpull%2F39093" target="_blank">#39093</a> )</li> 
 <li>渲染完成后刷新组件状态 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Flaravel%2Fframework%2Fcommit%2F04fc7c2f87372511b0f77e539bc0e2e3357ec200" target="_blank">04fc7c2</a> )</li>

 <li>通过刷新 SchemaState 进程来排除超时现象  ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Flaravel%2Fframework%2Fpull%2F39139" target="_blank">#39139</a> ) </li> 
</ul>
