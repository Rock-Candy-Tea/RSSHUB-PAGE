                                                        <p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><span style="background-color:#ffffff; color:#333333">Apache Syncope 2.1.9 已发布，Apache Syncope 是一种用于管理企业环境中的数字身份的开源系统，采用 Java EE 技术实现。</span></p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><strong>主要更新内容</strong></p> 
<ul> 
 <li style="text-align:left">修复当连接器初始化失败时，控制台变为 NPE 的问题</li> 
 <li style="text-align:left">JPA JSON：修复包含斜线的日期转换模式会导致不正确的搜索结果的问题</li> 
 <li style="text-align:left">修复组所有者更新/删除操作不会触发传播操作的问题</li> 
 <li style="text-align:left">使用每个域的配置创建规则，使创建线程安全</li> 
 <li style="text-align:left">修复创建枚举模式时未捕获的异常</li> 
 <li style="text-align:left">修复 Realm 更新不会触发用户配置的问题</li> 
 <li style="text-align:left">修复多节点部署的任务运行失败的问题</li> 
 <li style="text-align:left">修复不区分大小写的 Elasticsearch 扩展搜索返回错误结果的问题</li> 
 <li style="text-align:left">使用组所有者扩展委派管理</li> 
 <li style="text-align:left">将 ConnId ObjectClass 传递给 ReconFilterBuilder</li> 
 <li style="text-align:left">可以在确认对话框中添加自定义消息</li> 
 <li style="text-align:left">在历史管理的差异视图中提供属性排序</li> 
 <li style="text-align:left">允许清除传播任务</li> 
</ul>
