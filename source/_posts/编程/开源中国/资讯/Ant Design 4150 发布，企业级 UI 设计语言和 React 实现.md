                                                        <p>Ant Design 4.15.0 发布了。Ant Design 是一套企业级的 UI 设计语言和 React 实现，使用 TypeScript 构建，提供完整的类型定义文件，自带提炼自企业级中后台产品的交互语言和视觉风格、开箱即用的高质量 React 组件与全链路开发和设计工具体系。</p> 
<p>此版本更新内容如下：</p> 
<ul> 
 <li>Table 新增 <code>rowSelection.defaultSelectedRowKeys</code> 支持。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F29879" target="_blank">#29879</a></li> 
 <li>修复 TextArea 受控时展示值会被 <code>maxLength</code> 截取的问题；修复 <code>maxLength</code> 下超出部分无法触发 <code>onCompositionEnd</code> 事件的问题；修复使用 emoji 时 <code>maxLength</code> 和 <code>showCount</code> 计数逻辑不一致的问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F29867" target="_blank">#29867</a></li> 
 <li>修复 Button 点击效果在 Shadow Root 下不生效的问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F29897" target="_blank">#29897</a></li> 
 <li>Pagination 
  <ul> 
   <li>优化 Pagination 切换页码后 focus 样式遗留的问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F29891" target="_blank">#29891</a></li> 
   <li>修复 Pagination 使用 <code>modifyVars</code> 自定义主题时，页面跳转输入框高度没有按照预期改变。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F29886" target="_blank">#29886</a></li> 
  </ul> </li> 
 <li>修复 Steps 进度在边缘会被截断的问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F29893" target="_blank">#29893</a></li> 
 <li>Upload 默认阻止拖拽冒泡事件以支持嵌套结构。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F29832" target="_blank">#29832</a></li> 
 <li>ConfigProvider 的 <code>componentSize</code> 现在也会作用到 Tabs 上。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F29844" target="_blank">#29844</a></li> 
 <li>移除 Tag 鼠标悬浮样式以避免其误以为是可点击交互组件。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F29874" target="_blank">#29874</a></li> 
 <li>补充 <code>fr_FR</code> 语言缺失内容。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F29839" target="_blank">#29839</a></li> 
 <li>补充 <code>sv_SE</code> 语言缺失内容。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F29896" target="_blank">#29896</a></li> 
</ul>
