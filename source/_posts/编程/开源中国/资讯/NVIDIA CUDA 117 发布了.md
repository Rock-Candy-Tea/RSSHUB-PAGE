                                                        <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">CUDA（Compute Unified Device Architecture），是显卡厂商 NVIDIA 推出的运算平台。作为一种通用并行计算架构，CUDA 使 GPU 能够解决复杂的计算问题。 它包含了 CUDA 指令集架构（ISA）以及 GPU 内部的并行计算引擎。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">目前，NVIDIA CUDA 11.7 发布了，CUDA 11.7 为新的 NVIDIA Open GPU Kernel Modules 带来了兼容性支持，另一个重要的亮点是延迟加载支持。 而 CUDA 11.7 Toolkit 已作为 NVIDIA 专有计算堆栈的最新功能更新提供。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">CUDA 11.7 主要功能更新如下：</p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>支持将 NVIDIA Open GPU 内核模块与 CUDA 11.7 和 R515 beta 驱动程序一起使用。</li> 
 <li>延迟加载计算内核，将程序开始时的加载推迟到首次调用计算内核时，可节省设备端内存。“CUDA_MODULE_LOADING=LAZY”环境变量控制这个 CUDA 模块延迟加载行为。</li> 
 <li>NVCC 主机编译器支持 Clang 13。</li> 
 <li>Nsight Compute、CUPTI、NVPROF / Visual Profiler 和其他 CUDA 开发工具的各种新功能。</li> 
 <li>现在，Android 和 Tegra 的 L4T 上的 Vulkan-CUDA 互操作支持所有颜色格式。</li> 
 <li>弃用 NVCC 对所有 GPU 的 32 位编译。</li> 
 <li>由于上游 CentOS 8 EOL，CentOS Linux 8 也已从 CUDA 工具包中删除，并由 Rocky Linux 8 支持取代。</li> 
</ul>
