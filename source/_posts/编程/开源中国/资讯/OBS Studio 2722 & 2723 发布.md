                                                                                            <p>OBS Studio 是一款免费且开源的用于视频录制以及直播串流的软件，提供实时高性能的视频/音频捕捉与混合，以及无限的场景模式帮助用户通过自定义实现无缝转换。</p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">v27.2.2 版本更新内容包括：</p> 
<ul> 
 <li>修复了预览会 freeze 的错误，并且 stream 可能会断开连接</li> 
 <li>修复了使用脚本时可能在关机时发生的随机崩溃</li> 
 <li>修复了在 Windows 上使用 virtual camera 时可能在 Firefox/Chrome 中发生的崩溃</li> 
 <li>修复了与 Windows 上的音频捕获相关的启动崩溃</li> 
 <li>修复了 FreeBSD 上的编译问题</li> 
 <li>修复了编辑菜单中的菜单选项（如“Reset Transform”）会被错误地置灰的问题，这也会导致某些快捷方式失效</li> 
 <li>修复了 .cube LUT 文件在某些​​非英语语言的 macOS/Linux 上不起作用的问题</li> 
 <li>修复了 FFmpeg 依赖项仍然会破坏 macOS 上的第三方插件的问题</li> 
</ul> 
<p>v27.2.3 版本更新内容包括：</p> 
<ul> 
 <li><span style="background-color:#ffffff; color:#24292f">修复了 Windows 上的 </span>virtual camera <span style="background-color:#ffffff; color:#24292f">无法正确显示的错误</span></li> 
</ul>
