                                                                                            <p>IntelliJ IDEA 2021.3 系列的第一个更新正式推出。IntelliJ IDEA 在 2020.3.3 版本中引入了受信任项目的概念，以减轻从未知和不信任的来源打开项目所带来的风险。</p> 
<p>在 2021.3.1 版本中，IntelliJ IDEA 改变了受信任项目对话框的行为和实现。现在，它们会在 IntelliJ IDEA 中首次打开项目之前显示。</p> 
<p><strong>此外，这个版本带来了一系列重要的修复：</strong></p> 
<ul> 
 <li>修复了在 macOS Big Sur 上导致对话框出现在错误窗口的问题: <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-263628" target="_blank">IDEA-263628</a>.</li> 
 <li>修复了在尝试调整工具窗口大小时，意外激活窗口拖放功能的问题: <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-274904" target="_blank">IDEA-274904</a>.</li> 
 <li>文本搜索现在可以在快速文档（Ctrl + Q）中使用: <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-152644" target="_blank">IDEA-152644</a>.</li> 
 <li>当使用 JDK 9 或更高版本时，现在可以使用 JAXB 对话框从 Java 生成 XML 模式: <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-179691" target="_blank">IDEA-17969</a>.</li> 
 <li>修复了导致停止 Wildfly Run Configuration 时的意外行为的问题: <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-273497" target="_blank">IDEA-273497</a>.</li> 
</ul> 
<p><strong>还修复了一些与远程开发功能有关的问题：</strong></p> 
<ul> 
 <li>修正了空闲时 CPU 使用率高的问题: <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FGTW-685" target="_blank">GTW-685</a>.</li> 
 <li>现在可以使用 Gateway 通过 symlink 打开一个项目: <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FGTW-720" target="_blank">GTW-720</a>.</li> 
 <li>打开的类和标签的位置现在在会话之间被保留了: <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FCWM-4048" target="_blank">CWM-4048</a>.</li> 
 <li>修正了在工具窗口中滚动浏览长列表时出现的故障: <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FCWM-3679" target="_blank">CWM-3679</a>.</li> 
 <li>修正了不可点击的弹出式窗口: <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FCWM-4422" target="_blank">CWM-4422</a>.</li> 
 <li>为 Gradle 任务添加了一个上下文菜单: <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FCWM-4312" target="_blank">CWM-4312</a>.</li> 
</ul>
