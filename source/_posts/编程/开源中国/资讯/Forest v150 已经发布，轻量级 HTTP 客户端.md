                                                                                            <p>Forest v1.5.0 已经发布，这是一个轻量级 HTTP 客户端。</p> 
<p>此版本更新内容包括：</p> 
<p>此版本为正式版本，新增了以下特性和变动</p> 
<h4>新增特性:</h4> 
<ul> 
 <li>feat: #I3B3UZ</li> 
 <li>feat: #I3DOAP</li> 
 <li>feat: #I3E0FD</li> 
</ul> 
<h4>修复问题:</h4> 
<ul> 
 <li>fix: #I3DAEQ</li> 
</ul> 
<h4>代码变更:</h4> 
<ul> 
 <li>add: ForestConfiguration#isVariableDefined方法</li> 
 <li>add: @Var注解</li> 
 <li>add: ResourceRequestBody类</li> 
 <li>add: ResourceRequestBodyBuilder类</li> 
 <li>add: MultipartRequestBody类</li> 
 <li>add: MultipartRequestBodyBuilder类</li> 
 <li>add: SupportFormUrlEncoded类</li> 
 <li>refactor: 将DataVariableLifeCycle改名为VariableLifeCycle</li> 
 <li>refactor: Body的解析方式改为构建器模式</li> 
 <li>refactor: ForestVariableUndefined.java 改名为 ForestVariableUndefinedException</li> 
</ul> 
<h4>Maven依赖变更:</h4> 
<ul> 
 <li>update: 更新Spring-Core和Spring-Webmvc版本</li>
