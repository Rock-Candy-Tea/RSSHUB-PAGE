                                                        <p style="color:#333333; margin-left:0; margin-right:0; text-align:left">OBS Studio 是一款免费且开源的用于视频录制以及直播串流的软件，提供实时高性能的视频/音频捕捉与混合，以及无限的场景模式帮助用户通过自定义实现无缝转换。</p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">OBS Studio 27.1.1 已发布，更新内容如下。</p> 
<h3 style="text-align:start">27.1.1 Hotfix</h3> 
<ul> 
 <li style="color: rgb(51, 51, 51); margin-left: 0px; margin-right: 0px; text-align: left;"><span style="background-color:#ffffff; color:#24292f">修复了在 Windows 上部署 27.1 出现错误，并导致显示捕获源看起来褪色的问题</span></li> 
</ul> 
<h3 style="text-align:start">新特性</h3> 
<ul> 
 <li>集成 YouTube 
  <ul> 
   <li>支持在不使用流密钥的情况下连接到 YouTube 帐户</li> 
   <li>支持在设置流时添加各种功能，例如：流标题、描述、隐私设置等等</li> 
   <li>YouTube 流通过“Start Streaming”旁边的新“Manage Broadcast”按钮进行创建和管理</li> 
   <li>还具有用于公共和未列出广播的聊天平台（当前为只读）</li> 
   <li>登录后，自动配置向导现支持带宽测试</li> 
  </ul> </li> 
 <li>添加 18-scene 多视图选项</li> 
 <li>为 stinger 轨道遮罩添加 “Mask Only” 选项</li> 
 <li>向浏览器源添加了一项功能，用于对 OBS 进行权限控制，此权限必须由用户明确授予</li> 
 <li>添加了在预览中绘制安全区域的选项（与多视图中的安全区域相同）</li> 
</ul> 
<h2 style="text-align:start">调整</h2> 
<ul> 
 <li>针对<span style="background-color:#ffffff; color:#24292f"><span> </span>Windows 11 和 Windows 10 Insider 版本添加性能修复</span></li> 
 <li>Auto-Remux 现在可与重播缓冲区 (Replay Buffer) 搭配使用</li> 
 <li>优化媒体源，减少不必要的 CPU 占用</li> 
 <li>在“Scene Collections”菜单添加了“Missing Files Check”菜单项</li> 
 <li><span style="background-color:#ffffff; color:#24292f">在源工具栏中添加源图标</span></li> 
 <li><span style="background-color:#ffffff; color:#24292f">各个区域的 UI 小改进，包括源工具栏、变换对话框等</span></li> 
 <li><span style="background-color:#ffffff; color:#24292f">改进了游戏捕获源中的内部行为，避免不必要的重新连接</span></li> 
 <li><span style="background-color:#ffffff; color:#24292f">支持使用 Wayland 捕获源，无需从命令行将自定义参数传递给 OBS</span></li> 
 <li>……</li> 
</ul>
