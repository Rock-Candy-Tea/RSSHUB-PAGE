                                                                                            <p><span style="color:#333333">Taro 3.3.18 发布了。Taro 是一个开放式跨端跨框架解决方案，支持使用 React/Vue/Nerv 等框架来开发微信/京东/百度/支付宝/字节跳动/ QQ 小程序/H5 等应用。</span></p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><span style="color:#333333">此版本更新内容包括：</span></p> 
<h4><strong>特性</strong></h4> 
<p style="text-align:start"><strong>H5</strong></p> 
<ul> 
 <li>新增路由动画配置能力</li> 
 <li>新增自定义配置多路由能力</li> 
 <li>新增懒加载 Shaking 方案（for PWA）</li> 
</ul> 
<h4><strong>修复</strong></h4> 
<p style="text-align:start"><strong>小程序</strong></p> 
<ul> 
 <li>修复阿里小程序 2.0 构建报错，fix<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FNervJS%2Ftaro%2Fissues%2F10162" target="_blank">#10162</a></li> 
</ul> 
<p style="text-align:start"><strong>H5</strong></p> 
<ul> 
 <li>优化 Swiper 组件非 circular 模式状态更新问题</li> 
 <li>优化动画部分 API 实现与参数错误</li> 
 <li>修复 downloadFile 、uploadFile API 导出问题，fix<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FNervJS%2Ftaro%2Fissues%2F10951" target="_blank">#10951</a></li> 
 <li>修复弹窗 API 入参校验问题，fix<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FNervJS%2Ftaro%2Fissues%2F10949" target="_blank">#10949</a></li> 
 <li>修复 innerAudioContext 问题，fix<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FNervJS%2Ftaro%2Fissues%2F10946" target="_blank">#10946</a></li> 
 <li>修复图片组件默认宽高问题</li> 
 <li>修复页面滚动元素获取错误</li> 
</ul> 
<p style="text-align:start"><strong>RN</strong></p> 
<ul> 
 <li>修复 swiper 组件 selectIndex 更新问题，<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FNervJS%2Ftaro%2Fpull%2F10962" target="_blank">#10962</a></li> 
 <li>修复 swiper 组件 circular 模式左滑卡顿问题</li> 
</ul> 
<h4><strong>Typings</strong></h4> 
<ul> 
 <li>部分 API、参数平台支持标签补充</li> 
 <li>优化 Animation 类型</li> 
 <li>调整 Button openType 参数类型</li> 
 <li>补充挂载 Events 对象 fix<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FNervJS%2Ftaro%2Fissues%2F10880" target="_blank">#10880</a></li> 
</ul>
