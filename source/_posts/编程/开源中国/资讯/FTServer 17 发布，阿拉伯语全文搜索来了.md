                                                        <p>FTServer 是一个实用的多国语言文字搜索服务器，支持古今文字的各种用法组合。<br> 使用 CRUD + JSP 构建，非常容易整合到各类应用程序中。<br> 搜索速度快，内存使用低，同时支持设置大内存进行加速。</p> 
<p>支持： 中文 日文 英文 俄文 德文 等混合搜索。</p> 
<p><strong>更新内容：</strong><br> 增加阿拉伯语支持，希伯来文应该也能支持，<br> 这是一个有20亿人使用的非常多地区变体的语言，不确定所有都支持。<br> 另外，要支持 dir="auto" 的浏览器才能正常阅读。<br> 这个版本主要方便一带一路地区的相关用户，<br> 中英文用户没必要升级，因为作者只学习了三天阿拉伯语。</p> 
<p><br> <strong>安装使用：</strong><br> 首先准备 30GB 硬盘空间。<br> <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fiboxdb%2Fftserver%2F" target="_blank">FTServer Java 源码下载</a><br> Java 版本使用 Netbeans 编译，放入JSP服务器中。<br> 或者在目录下执行<br> mvn package cargo:run</p>
