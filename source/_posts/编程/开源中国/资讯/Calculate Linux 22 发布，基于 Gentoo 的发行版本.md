                                                        <p>Calculate Linux 是一个为在组织环境中快速部署而优化的 Linux 发行版。它以 Gentoo Linux 项目为基础，包括许多预配置的功能。</p> 
<p>Calculate Linux 以五种方式发布。Calculate Linux Desktop（CLD）、Calculate Directory Server（CDS）、Calculate Linux Scratch（CLS）、Calculate Scratch Server（CSS）和 Calculate Linux Container（CLC）。</p> 
<p>Calculate Linux 22 正式发布，更新内容如下：</p> 
<h3>变化</h3> 
<ul> 
 <li>增加了对系统更新之间长间隔的支持。现在你可以随时升级，即使你已经很久没有更新你的系统了。</li> 
 <li>最新的 Calculate Utils 3.7 完全过渡到 Python 3，Python 2.7 从发行镜像中删除。</li> 
 <li>PulseAudio 被 PipeWire 取代，用户仍然可以选择 ALSA。</li> 
 <li>增加了对 ALSA 的蓝牙支持。</li> 
 <li>增强了对 Hyper-V 硬件虚拟化的支持。</li> 
 <li>更好的系统性能。</li> 
 <li>Clementine 被其 fork 的 Strawberry 音乐播放器取代。</li> 
 <li>设备管理现在由 udev 而不是 eudev 处理。</li> 
</ul>
