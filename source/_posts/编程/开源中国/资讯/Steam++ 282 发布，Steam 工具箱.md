                                                                                            <p>Steam++ 2.8.2 已经发布，Steam 工具箱。</p> 
<p>此版本更新内容包括：</p> 
<h3>修复问题</h3> 
<ol> 
 <li>修复 Windows FDE 版在 Windows 8.1/10 上错误识别成 Windows 8</li> 
 <li>修复 网络加速 DNS 设置无效</li> 
 <li>修复 网络加速 PAC 代理模式不能正常工作</li> 
 <li>修复 Windows 7 上的一些兼容性问题导致加速失效</li> 
 <li>改进 Windows Desktop Bridge 版本中不再弹出防火墙规则授权弹窗</li> 
 <li>修复 Windows Desktop Bridge 版本开机自启静默失效</li> 
 <li>改进 DNS 驱动拦截模式停止后不再占用 sys 文件</li> 
 <li>改进 通知 UI 列表项中仅显示两行文字</li> 
 <li>修复 第三方快速登录失败</li> 
 <li>改进 关闭代理时仅根据当前代理模式进行清理</li> 
</ol>
