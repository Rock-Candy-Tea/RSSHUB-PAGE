                                                                                            <p> 作为 Ubuntu 20.04 LTS 的第五个维护版本，Ubuntu 20.04.5 LTS 现已<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Freleases.ubuntu.com%2Ffocal%2F" target="_blank">可下载</a>。这个版本集合了自 2 月发布的 Ubuntu 20.04.4 LTS 以来的所有安全补丁、错误修复和应用程序更新，但是没有新功能。</p> 
<p>没有新功能就意味着，如果你正在运行的就是 Ubuntu 20.04 LTS ，且在 Ubuntu 20.04.5 发布之前就已经安装了所有可用的更新，则无需重新安装 20.04.5 LTS。此版本适用于从其他 LTS 版本升级过来的用户。</p> 
<p>Ubuntu 20.04 LTS 采用的是 Linux 5.15 内核，与新 ISO 捆绑在一起的其他软件更新包括 GNOME Shell 3.36.8、LibreOffice 6.4.7.2 和 Mozilla Firefox 104，但不是 Snap 版本。</p> 
<p>可以<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Freleases.ubuntu.com%2Ffocal%2F" target="_blank">从 Ubuntu 网站</a>下载 Ubuntu 20.04.5 LTS 。它以 3.1GB ISO 的形式提供。</p>
