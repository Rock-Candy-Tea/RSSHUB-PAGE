<div class="content">
                                                                    
                                                        <p>Tailwind CSS v3.0 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Ftailwindcss.com%2Fblog%2Ftailwindcss-v3" target="_blank">已正式发布</a>，新版本在性能和工作流优化方面带来了巨大的提升，同时还增加了大量新特性。</p> 
<p><img src="https://static.oschina.net/uploads/space/2021/1213/074650_2wJ7_2720166.png" referrerpolicy="no-referrer"></p> 
<p><span style="background-color:#ffffff; color:#333333">Tailwind CSS v2.0 主要更新内容：</span></p> 
<ul style="margin-left:0; margin-right:0"> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Ftailwindcss.com%2Fblog%2Ftailwindcss-v3%23just-in-time-all-the-time" target="_blank">默认启用 Just-in-Time</a><span> </span>— 闪电般的快速构建时间、可堆叠的变体、支持生成任意样式、更好的浏览器性能</li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Ftailwindcss.com%2Fblog%2Ftailwindcss-v3%23every-color-out-of-the-box" target="_blank">开箱即用的各种颜色</a><span> </span>— 包括所有扩展调色板颜色，如青色、玫瑰色、紫红色和青柠色，以及五十度的灰白色</li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Ftailwindcss.com%2Fblog%2Ftailwindcss-v3%23colored-box-shadows" target="_blank">彩色的边框阴影</a><span> </span>— <span style="background-color:#ffffff; color:#334155">有趣的发光和反射效果，以及为彩色背景添加更自然的阴影</span><br> <img src="https://static.oschina.net/uploads/space/2021/1213/080742_WXO3_2720166.png" referrerpolicy="no-referrer"></li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Ftailwindcss.com%2Fblog%2Ftailwindcss-v3%23scroll-snap-api" target="_blank">Scroll snap API</a><span> </span>— <span style="background-color:#ffffff; color:#334155">一套全面且可组合的实用程序，用于 </span>CSS-only 的<span style="background-color:#ffffff; color:#334155">滚动捕捉</span></li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Ftailwindcss.com%2Fblog%2Ftailwindcss-v3%23multi-column-layout" target="_blank">多栏布局</a> — 报纸布局类型<br> <img src="https://static.oschina.net/uploads/space/2021/1213/080827_4CWX_2720166.png" referrerpolicy="no-referrer"></li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Ftailwindcss.com%2Fblog%2Ftailwindcss-v3%23native-form-control-styling" target="_blank">原生表单控件样式</a><span> </span>— 使复选框、单选按钮和文本输入与品牌相匹配，无需重新发明轮子</li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Ftailwindcss.com%2Fblog%2Ftailwindcss-v3%23print-modifier" target="_blank">Print modifier</a><span> </span>— 直接从 HTML 控制网站被打印时的外观</li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Ftailwindcss.com%2Fblog%2Ftailwindcss-v3%23modern-aspect-ratio-api" target="_blank">引入现代宽高比 API</a></li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Ftailwindcss.com%2Fblog%2Ftailwindcss-v3%23fancy-underline-styles" target="_blank">丰富的下划线样式</a><br> <img src="https://static.oschina.net/uploads/space/2021/1213/080856_r4HK_2720166.png" referrerpolicy="no-referrer"></li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Ftailwindcss.com%2Fblog%2Ftailwindcss-v3%23rtl-and-ltr-modifiers" target="_blank">新增 RTL 和 LTR 修饰符</a><img src="https://static.oschina.net/uploads/space/2021/1213/080940_4njD_2720166.png" referrerpolicy="no-referrer"></li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Ftailwindcss.com%2Fblog%2Ftailwindcss-v3%23portrait-and-landscape-modifiers" target="_blank">新增 Portrait 和 landscape 修饰符</a></li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Ftailwindcss.com%2Fblog%2Ftailwindcss-v3%23arbitrary-properties" target="_blank">支持任意属性</a><span> </span>— <span style="background-color:#ffffff; color:#334155">Tailwind 现在支持从未听说过的 CSS 属性</span></li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Ftailwindcss.com%2Fblog%2Ftailwindcss-v3%23play-cdn" target="_blank">Play CDN</a><span> </span>— 新的 Just-in-Time 引擎被压缩到一个直接在浏览器中运行的 CDN 脚本中</li> 
 <li><strong style="color:#0f172a">其他的实用程序</strong><span> </span>— <span style="background-color:#ffffff; color:#334155">包括对触摸动作、意志改变、弹性基础、文本缩进、滚动行为等的支持</span></li> 
</ul>
