                                                                                            <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">MX Linux 是一个基于 Debian 稳定版的中量级 Linux 操作系统，使用 antiX 的核心组件，以及由 MX 社区创建或打包的额外软件。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">MX Linux 21.2 是 MX Linux 21 版本的第一次更新，其中包括错误修复、内核和应用程序更新。该版本是一个相对较小的版本更新，更新内容如下：</p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>mx-installer 收到了几个 buxfixes 和改进。 </li> 
 <li>mx-tweak 具有禁用蓝牙适配器和将 Xfce/gtk 文件对话框按钮移动到底部而不是对话框顶部的新选项。</li> 
 <li>fluxbox 获得了一个新的 mxfb-look 工具，可保存和恢复主题组合。 </li> 
 <li>mx-updater/apt-notifier 可以选择使用 nala 作为后端（参见 /usr/share/apt-notifier 中的 .conf 文件）</li> 
 <li>将内核清理工具添加到 mx-cleanup </li> 
 <li>磁盘空间检查 /boot 分区，以确保磁盘有足够的空间用于内核更新。</li> 
 <li>将 uefi 管理工具添加到 mx-boot-options</li> 
 <li>用于 mx-snapshot 的新自动关机选项</li> 
 <li>用于快速系统信息的新 gui 包装器</li> 
</ul> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">内核更新： Xfce、fluxbox 和 KDE iso 都更新到最新的 5.10 内核，AHS 使用 5.18 内核（附带更新的 linux-firmware 软件包）</p>
