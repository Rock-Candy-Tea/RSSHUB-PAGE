                                                                                            <p>Tor Browser 10.0.15 现已发布。Tor Browser 是一个基于 Firefox ESR (Firefox with extended support) 的 Web 浏览器，默认配置通过 <a href="http://www.oschina.net/p/tor">Tor</a> 和 Vidalia 实现了个人隐私保护和匿名。 </p> 
<p><img height="308" src="https://oscimg.oschina.net/oscnet/up-f12978d244c86d868889f485487f7615291.png" width="700" referrerpolicy="no-referrer"></p> 
<p>此版本将 Openssl 更新到了 1.1.1k，还包括了一个针对网站上 Javascript 被禁用时的错误修复。官方表示，强烈建议使用 Windows Expert Bundle 的 Relay operators 升级他们的 relay。</p> 
<p>注意：这个版本不包括 Android 版本 Tor Browser 的更新。</p> 
<p>自 Tor Browser 10.0.14 以来的完整变更日志：</p> 
<ul> 
 <li>Windows + OS X + Linux 
  <ul> 
   <li>将 Openssl 更新到 1.1.1k</li> 
   <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fbugs.torproject.org%2Ftpo%2Fapplications%2Ftorbutton%2F40030" target="_blank">Bug 40030</a>： 向 NoScript 添加了 noscript 功能</li> 
  </ul> </li> 
</ul>
