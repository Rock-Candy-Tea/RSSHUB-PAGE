                                                        <p>Calibre 开源项目是 Calibre 官方出的电子书管理工具。它可以查看，转换，编辑和分类所有主流格式的电子书。Calibre 是个跨平台软件，可以在 Linux、Windows 和 macOS 上运行。</p> 
<p>Calibre 6.2 正式发布，此次更新内容如下：</p> 
<h3>新功能</h3> 
<ul> 
 <li>全文搜索：从书籍中提取可搜索文本时，忽略软连字符。</li> 
 <li>内容服务器：让阅读按钮在浏览器内置的 PDF 查看器中打开 PDF 文件，而不是 Calibre 查看器，因为 Calibre 查看器不是很适合 PDF。</li> 
 <li>Kobo 驱动程序：增加对新固件的支持</li> 
 <li>创建图标主题对话框：增加一个复选框，将新创建的主题安装到 calibre 中。</li> 
 <li>允许通过右键点击图书细节面板为 FTS 重新索引图书</li> 
</ul> 
<h3>错误修正</h3> 
<ul> 
 <li>内容服务器：解决 Chrome 浏览器在重启后连接到服务器时造成多次登录提示的问题</li> 
 <li>修复 6.0 的回归，即当 EPUB 没有实际的封面时，损坏了 EPUB 的第一页作为封面的渲染。</li> 
 <li>内容服务器：修复 6.0 中的一个回归，导致在黑暗模式下书籍细节面板中评论元数据的不正确渲染</li> 
 <li>FB2 输入：修复 6.0 中的一个回归，该回归破坏了 macOS 上的 FB2 转换</li> 
 <li>从 Google 下载元数据：改为通过 Google 搜索进行查询，因为这样可以找到一些通过 API 搜索不到的书</li> 
 <li>电子书查看器：修正当 ToC 窗口浮动时，点击 ToC 条目将键盘焦点移至 ToC 搜索框的问题</li> 
 <li>……</li> 
</ul>
