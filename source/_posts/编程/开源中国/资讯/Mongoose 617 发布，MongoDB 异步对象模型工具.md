                                                                                            <p><span style="color:#333333">Mongoose 是设计用于异步环境的 MongoDB 对象模型工具，支持 promises 和 callbacks。Mongoose 6.1.7 正式发布，本次更新内容如下：</span></p> 
<ul> 
 <li>fix(model): <span style="color:#2e3033">正确处理 $push 嵌套数组</span> <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FAutomattic%2Fmongoose%2Fissues%2F11108" target="_blank">#11108</a></li> 
 <li>fix(update): <span style="color:#2e3033">正确地应用时间戳更新管道 </span> <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FAutomattic%2Fmongoose%2Fissues%2F11151" target="_blank">#11151</a></li> 
 <li>fix(document): <span style="color:#24292f">将数组子文档设置为自身后，正确处理数组子文档的修改</span> <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FAutomattic%2Fmongoose%2Fissues%2F11172" target="_blank">#11172</a></li> 
 <li>fix(index.d.ts): 允许将选项传递给 model() 代替已移除的 <code>skipInit</code> <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FAutomattic%2Fmongoose%2Fissues%2F11137" target="_blank">#11137</a></li> 
 <li>fix(aggregate): <span style="color:#2e3033">允许将详细信息传递给 Aggregate.prototype.explain()</span> <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FAutomattic%2Fmongoose%2Fissues%2F11144" target="_blank">#11144</a></li> 
 <li>fix(index.d.ts): <span style="color:#24292f">避免 TypeScript 将 _id 推断为</span>任何<span style="color:#24292f">带有 HydratedDocument</span> 的类型 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FAutomattic%2Fmongoose%2Fissues%2F11085" target="_blank">#11085</a></li> 
 <li>docs: <span style="color:#24292f">修复 Node.js 驱动程序的兼容链接</span> <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FAutomattic%2Fmongoose%2Fpull%2F11214" target="_blank">#11214</a> <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fwesbos" target="_blank">wesbos</a></li> 
 <li>docs: <span style="color:#24292f">删除无关的反引号</span> <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FAutomattic%2Fmongoose%2Fpull%2F11204" target="_blank">#11204</a> <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fjoebowbeer" target="_blank">joebowbeer</a></li>
