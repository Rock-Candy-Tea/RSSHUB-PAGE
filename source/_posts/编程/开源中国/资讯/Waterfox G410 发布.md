                                                                                            <p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><span style="color:#000000">Waterfox 是一个基于 Firefox 的开源浏览器，由 Alex Kontos 于 2011 年首次发布。目前 Waterfox 分为两个版本 —— Classic 和 Current（G4）。Waterfox 在 Firefox 的基础上，将影响用户隐私的数据收集、遥测和启动分析等功能进行了移除。</span></p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><span style="color:#000000">Waterfox G4.1.0 现已发布，更新内容如下：</span></p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><span style="color:#000000"><strong>New</strong></span></p> 
<ul style="margin-left:0; margin-right:0"> 
 <li><span style="color:#000000">在 about:addons 中，</span><span style="color:#24292f">插件</span><span style="color:#000000">的版本号再次显示在其名称旁边。</span></li> 
 <li><span style="color:#000000">现在可以在 about:preferences → General → Waterfox Updates 中禁用自动更新。</span></li> 
 <li><span style="color:#000000">在 Private Tab 和 Private Window 中进行搜索时，Startpage 现在是默认搜索。</span></li> 
 <li><span style="color:#000000">在 Linux 上希望使用 HTML GeoLocation API 的用户现在可以在 about:preferences → Languages → Geolocation API 中启用它。</span></li> 
 <li>当你的系统内存不足时，Tabs 现在会自动卸载。</li> 
 <li>Tabs 现在可以在 about:preferences → General → Tabs 中启用该偏好后手动卸载。然后你可以右键点击一个 tab 并卸载它。</li> 
</ul> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><span style="color:#000000"><strong>Fixed</strong></span></p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>ChromeWeb Store 扩展程序现在应该再次正确安装。</li> 
 <li>各种鼠标上的后退和前进按钮现在可以在 Waterfox 中正常工作，而不是充当 middle-click。</li> 
 <li>有时 non-private tabs 会恢复为 private tabs。这将不再发生。</li> 
 <li>对默认 Lepton 主题的各种修复<span style="color:#24292f"> [</span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fblack7375%2FFirefox-UI-Fix%2Freleases%2Ftag%2Fv4.6.0" target="_blank">1</a><span style="color:#24292f">][</span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fblack7375%2FFirefox-UI-Fix%2Freleases%2Ftag%2Fv4.6.1" target="_blank">2</a><span style="color:#24292f">][</span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fblack7375%2FFirefox-UI-Fix%2Freleases%2Ftag%2Fv4.6.2" target="_blank">3</a><span style="color:#24292f">]</span>。</li> 
 <li>各种日语翻译改进</li> 
</ul> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><span style="color:#000000"><strong>Changed</strong></span></p> 
<ul style="margin-left:0; margin-right:0"> 
 <li><span style="color:#000000">最低要求已经从 SSE4.2 略微降低到 SSE4.1。</span></li> 
 <li><span style="color:#000000">取消了 -moz-fit-content 的前缀。-fit-content show 现在可以在没有 -moz 前缀的情况下工作。</span></li> 
 <li><span style="color:#000000"><dialog> HTML element 现在已被启用。</span></li> 
 <li><span style="color:#000000">对 his 和 her 的引用在相关的地方被替换为 their。</span></li> 
</ul> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><span style="color:#000000">更新说明：</span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FWaterfoxCo%2FWaterfox%2Freleases%2Ftag%2FG4.1.0" target="_blank">https://github.com/WaterfoxCo/Waterfox/releases/tag/G4.1.0</a></p>
