                                                                                            <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">DBeaver 21.2.5 现已发布。DBeaver 是一个免费开源的通用数据库工具，适用于开发人员和数据库管理员。</p> 
<p><strong>主要更新内容</strong></p> 
<ul> 
 <li><span style="background-color:#ffffff; color:#555555">Data 编辑器</span> 
  <ul> 
   <li><span style="background-color:#ffffff; color:#555555">空间查看器现在为表和列保存正确的 SRID</span></li> 
   <li><span style="background-color:#ffffff; color:#555555">查找/替换 BLOB/CLOB 列已修复</span></li> 
  </ul> </li> 
 <li><span style="background-color:#ffffff; color:#555555">Metadata 编辑器</span> 
  <ul> 
   <li><span style="background-color:#ffffff; color:#555555">​​​​​​​导航仪刷新已修复</span></li> 
   <li><span style="background-color:#ffffff; color:#555555">模式级索引创建已修复</span></li> 
  </ul> </li> 
 <li><span style="background-color:#ffffff; color:#555555">ERD 编辑器</span> 
  <ul> 
   <li><span style="background-color:#ffffff; color:#555555">​​​​​​​列搜索已经实现</span></li> 
  </ul> </li> 
 <li>SQL 编辑器 
  <ul> 
   <li>文件复制到新项目已修复</li> 
  </ul> </li> 
 <li>备份/恢复向导 UI 已修复（MySQL 和 PostgreSQL）</li> 
 <li>任务向导按钮启用/禁用已修复</li> 
 <li>SQLite 
  <ul> 
   <li>驱动程序更新至 3.36 版</li> 
  </ul> </li> 
 <li>Greenplum/Redshift 
  <ul> 
   <li>添加了会话和锁管理器</li> 
  </ul> </li> 
 <li>MariaDB/MySQL 
  <ul> 
   <li>改进了表元数据加载性能</li> 
  </ul> </li> 
 <li><span style="background-color:#ffffff; color:#555555">Oracle</span> 
  <ul> 
   <li><span style="background-color:#ffffff; color:#555555">​​​​​​​​​​​​​​实现了功能和程序的全文搜索</span></li> 
   <li><span style="background-color:#ffffff; color:#555555">选项 "仅显示当前模式对象" 已实现</span></li> 
  </ul> </li> 
 <li><span style="background-color:#ffffff; color:#555555">PostgreSQL</span> 
  <ul> 
   <li><span style="background-color:#ffffff; color:#555555">​​​​​​​外键设置编辑已实现</span></li> 
   <li><span style="background-color:#ffffff; color:#555555">数组值解析器已修复</span></li> 
   <li><span style="background-color:#ffffff; color:#555555">角色名称引用已在授权/撤销命令中修复</span></li> 
  </ul> </li> 
</ul>
