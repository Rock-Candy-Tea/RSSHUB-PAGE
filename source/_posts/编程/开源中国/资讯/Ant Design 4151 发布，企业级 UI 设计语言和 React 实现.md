                                                                                            <p>Ant Design 4.15.1 发布了。Ant Design 是一套企业级的 UI 设计语言和 React 实现，使用 TypeScript 构建，提供完整的类型定义文件，自带提炼自企业级中后台产品的交互语言和视觉风格、开箱即用的高质量 React 组件与全链路开发和设计工具体系。</p> 
<p>此版本更新内容如下：</p> 
<ul> 
 <li>修复 Descriptions <code>labelStyle</code> <code>contentStyle</code> 当设置 <code>layout="vertical"</code> 和 <code>bordered=&#123;true&#125;</code> 时不生效的问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F29942" target="_blank">#29942</a></li> 
 <li>Upload 的 <code>fileList</code> 为冻结对象时不再崩溃。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F29944" target="_blank">#29944</a></li> 
 <li>修复 Tabs 在 Safari 13.1 垂直模式下不能滚动问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F29983" target="_blank">#29983</a></li> 
 <li>修复点击 Button 时有时会报错 <code>Cannot read property 'removeChild' of null</code> 的问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F29961" target="_blank">#29961</a></li> 
 <li>修复 Transfer 按钮区域 box-shadow 被截断的问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F30030" target="_blank">#30030</a></li> 
 <li>修复 CheckBox.Group 和 <code>@ant-design/compatible</code> 的 Form 无法一起工作的问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F30039" target="_blank">#30039</a></li> 
 <li>修复 PageHeader 下卡片类型 <code>tabs</code> 样式问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F29819" target="_blank">#29819</a></li> 
 <li>Table 
  <ul> 
   <li>优化 Table 分页器在狭窄空间的样式问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F30065" target="_blank">#30065</a></li> 
   <li>修复 Table <code>@table-border-radius-base</code> 没有被应用到 <code>.ant-table</code> 节点上。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F29966" target="_blank">#29966</a></li> 
  </ul> </li> 
 <li>修复 Input.TextArea <code>maxLength</code> 在受控时无效的问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F30070" target="_blank">#30070</a></li> 
 <li>TypeScript 
  <ul> 
   <li>修复 Select 类型不能指定为 <code>undefined</code> 的问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F30068" target="_blank">#30068</a></li> 
   <li>修复 Popconfirm <code>okButtonProps</code> 和 <code>cancelButtonProps</code> 定义。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F29928" target="_blank">#29928</a></li> 
  </ul> </li> 
 <li>国际化 
  <ul> 
   <li>添加西班牙语 <code>Table.SELECTION_NONE</code> 文案。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F29962" target="_blank">#29962</a></li> 
  </ul> </li> 
</ul>
