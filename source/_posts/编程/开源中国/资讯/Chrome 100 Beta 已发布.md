                                                                                            <p>Chrome 99 发布之后， Chrome 100 已<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fblog.chromium.org%2F2022%2F03%2Fchrome-100-beta-reduced-user-agent.html" target="_blank">升级为测试版</a>，该版本附带两个新 API ，包括多屏幕窗口放置 API 和数字商品 API，还包含一些功能更新。</p> 
<p>多屏幕窗口放置：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fchromestatus.com%2Ffeature%2F5252960583942144" target="_blank">Multi-Screen Window Placement API</a> 支持是为了能够更好地处理跨多个显示器的多窗口应用，其<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fweb.dev%2Fmulti-screen-window-placement%2F" target="_blank">最新的示例用例</a>包括跨屏幕幻灯片放映、跨显示器窗口仪表板的财务应用程序、医学成像应用程序、多屏幕上的创意应用程序等。</p> 
<p>数字商品：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fchromestatus.com%2Ffeature%2F5339955595313152" target="_blank">Digital Goods API</a> 则是用于查询和管理数字产品的 API，以方便从 Web 应用程序进行应用内购。新 API 与 Payment Request API 配合使用，后者用于实物商品购买，点此查看该 API 的详细介绍。</p> 
<p>其他更新项不一一介绍，感兴趣的兄弟们可以在<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fblog.chromium.org%2F2022%2F03%2Fchrome-100-beta-reduced-user-agent.html" target="_blank">博客中</a>查看，Chrome 100 正式版预计在 3 月底正式发布。</p>
