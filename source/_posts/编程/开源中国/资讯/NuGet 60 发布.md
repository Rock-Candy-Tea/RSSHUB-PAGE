                                                                                            <p>NuGet 6.0 已经包含在此前发布的 Visual Studio 2022 和 .NET 6.0 中，你也可以将 NuGet 6.0 作为独立的可执行文件下载到 Windows、macOS 和 Linux 上。</p> 
<p>NuGet 工具可以帮助开发者发现新的 .NET 包，以用于他们的 .NET 应用程序，同时在日常开发中使包的管理变得更容易。</p> 
<p>NuGet 6.0 值得关注的更新内容如下：</p> 
<ul> 
 <li>创建和使用具有 .NET 6 支持的 NuGet 包，NuGet 6.0 是第一个为面向 .NET 6.0 的 NuGet 包提供完整创建和恢复支持的版本</li> 
 <li>通过源码映射知道你的包来自哪里，加强软件供应链抵御攻击的安全性</li> 
 <li>在 Visual Studio 中查看包的漏洞，包括漏洞数量和严重性等详细信息</li> 
 <li>在 NuGet 客户端中配置 retry & backoff 行为，以便在遇到较弱的网络连接时获得更有弹性的体验</li> 
 <li>可以在构建输出中排除默认文件扩展，更好地控制构建输出文件夹中包含的扩展</li> 
 <li>在 Visual Studio 中利用改进的弃用信息查找可替代的包，已弃用的包现在包含指向建议的备用包的链接</li> 
 <li>在 Visual Studio 中添加包的 README，有助于传达有关包的重要信息</li> 
 <li>在 Visual Studio 中更快地加载解决方案和分支切换，通过仅调用一次而不是多次来提高性能，显着缩短了后台进程完成所需的时间</li> 
</ul>
