 <p>pgAdmin 是 PostgreSQL 领先的开源图形化管理工具。pgAdmin 4 旨在满足新手和有经验的 Postgres 用户的需求，提供强大的图形界面，简化了数据库对象的创建、维护和使用。</p> 
 <p>pgAdmin 4 5.7 版本正式发布。本次版本更新中值得注意的变化包括：</p> 
 <h3>功能：</h3> 
 <ul> 
  <li>增加了对具有重启标识的截断表的支持；</li> 
  <li>在 <strong>Maintenance process watcher</strong> 对话框中增加了数据库和服务器信息；</li> 
  <li>允许被引用的表与 ERD 工具的一对多关系中的本地表相同；</li> 
  <li>当用户关闭一个标签时，通过聚焦于适当的标签，使关闭标签变得更加智能；</li> 
 </ul> 
 <h3><strong>Bugs/Housekeeping:</strong></h3> 
 <ul> 
  <li>修正了 Windows 上的空白屏幕问题，并做了修改以使用 NWjs manifest 来记忆窗口大小</li> 
  <li>确保触发器函数 SQL 应该有 "创建或替换函数" 而不是只有 "创建函数"</li> 
  <li>修正了关系线在节点上的导出图像问题</li> 
  <li>修正了 PSQL 工具窗口的宽度限制问题</li> 
  <li>修正了使用 diff 工具时，带有序列的列被不必要地更改的问题</li> 
  <li>修正了一个在使用 Schema Diff 生成脚本时，函数体中的空格没有被应用的问题</li> 
  <li>为 Oauth2 范围配置引入了 OAUTH2_SCOPE 变量</li> 
  <li>确保 pgAdmin 不应由于主机名中的特殊字符而在登录时失败</li> 
 </ul>
