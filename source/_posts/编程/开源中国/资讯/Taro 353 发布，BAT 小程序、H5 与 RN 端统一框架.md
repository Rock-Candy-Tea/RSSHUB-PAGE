                                                                                            <p><span style="color:#333333">Taro 3.5.3 现已发布。Taro 是一个开放式跨端跨框架解决方案，支持使用 React/Vue/Nerv 等框架来开发微信 / 京东 / 百度 / 支付宝 / 字节跳动 / QQ 小程序 / H5 等应用。具体更新内容如下：</span></p> 
<h2><strong>特性</strong></h2> 
<h3><strong>小程序</strong></h3> 
<ul> 
 <li>补充 Map 组件的属性和事件</li> 
</ul> 
<h3><strong>@tarojs/plugin-mini-ci</strong></h3> 
<ul> 
 <li>暴露支付宝 minidev 的 <code>project</code> 参数</li> 
</ul> 
<h2><strong>修复</strong></h2> 
<h3><strong>小程序</strong></h3> 
<ul> 
 <li>优化主入口引用样式文件编译问题</li> 
</ul> 
<h3><strong>H5</strong></h3> 
<ul> 
 <li><strong>prebundle:</strong> 删除继承 webpack 冗余配置，<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FNervJS%2Ftaro%2Fissues%2F12240" target="_blank">#12240</a></li> 
 <li><strong>components:</strong> 修复 Safari 不支持的正则，<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FNervJS%2Ftaro%2Fissues%2F12249" target="_blank">#12249</a></li> 
 <li><strong>runtime:</strong> 修复 Vue3 开发环境 warn 提示</li> 
</ul> 
<h3><strong>RN</strong></h3> 
<ul> 
 <li><strong>taro-runtime-rn:</strong> 修复钩子注册逻辑</li> 
</ul> 
<h3><strong>CLI</strong></h3> 
<ul> 
 <li>修复 <code>taro update</code> 命令，<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FNervJS%2Ftaro%2Fissues%2F12234" target="_blank">#12234</a></li> 
</ul>
