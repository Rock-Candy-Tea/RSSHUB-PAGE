                                                        <p>Pidgin 2.14.6 现已发布。Pidgin 原名 Gaim，是一款采用 <a href="http://www.oschina.net/p/gtk">GTK</a> 开发的跨平台即时聊天客户端软件，支持包括 Aim、ICQ （基于 Oscar 协议）、MSN Messenger、Yahoo、IRC、Jabber、Gadu-Gadu、SILC、Groupwise Messenger 和 Zephyr 等即时通信软件。你可以同时用不同的账号在多个即时通信网络中登录。</p> 
<p>此版本的完整变更日志如下：</p> 
<p><strong>General：</strong></p> 
<ul> 
 <li>更新引用以指向当前的网站。(<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Freviews.imfreedom.org%2Fr%2F766" target="_blank">Review 766</a>)</li> 
 <li>在帮助菜单中添加 donate 链接。(<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Freviews.imfreedom.org%2Fr%2F749" target="_blank">Review 749</a>)</li> 
</ul> 
<p><strong>Finch：</strong></p> 
<ul> 
 <li>在手动查找 ncurses 之前检查 pkg-config。(<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Freviews.imfreedom.org%2Fr%2F729" target="_blank">Review 729</a>)</li> 
</ul> 
<p><strong>Pidgin：</strong></p> 
<ul> 
 <li>用空格替换主题中的换行符。(<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fissues.imfreedom.org%2Fissue%2FPIDGIN-16704" target="_blank">PIDGIN-16704</a>) (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Freviews.imfreedom.org%2Fr%2F730" target="_blank">Review 730</a>)</li> 
</ul> 
<p><strong>libpurple：</strong></p> 
<ul> 
 <li>添加了对 <code>no_proxy</code>环境变量的支持。(<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fissues.imfreedom.org%2Fissue%2FPIDGIN-17518" target="_blank">PIDGIN-17518</a>) (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Freviews.imfreedom.org%2Fr%2F667" target="_blank">Review 667</a>)</li> 
 <li>添加了用于模糊测试的基础设施以及一些初始模糊处理程序。(<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Freviews.imfreedom.org%2Fr%2F760" target="_blank">Review 760</a>）</li> 
 <li>修复了<code>purple_markup_linkify</code>中的一个越界写入。(<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Freviews.imfreedom.org%2Fr%2F781" target="_blank">Review 781</a>)</li> 
</ul> 
<p><strong>XMPP：</strong></p> 
<ul> 
 <li>绑定资源后启用会话管理。(<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fissues.imfreedom.org%2Fissue%2FPIDGIN-17520" target="_blank">PIDGIN-17520</a>) (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Freviews.imfreedom.org%2Fr%2F759" target="_blank">Review 759</a>)</li> 
</ul> 
<p><strong>Zephyr：</strong></p> 
<ul> 
 <li>修复 clang ogical-not-parentheses 警告。(<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fissues.imfreedom.org%2Fissue%2FPIDGIN-17528" target="_blank">PIDGIN-17528</a>) ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Freviews.imfreedom.org%2Fr%2F731" target="_blank">Review 731</a> )</li> 
</ul>
