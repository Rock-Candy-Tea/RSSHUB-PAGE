                                                                                            <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">时隔半年，7-Zip 22.00 版本<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fsourceforge.net%2Fp%2Fsevenzip%2Fdiscussion%2F45797%2Fthread%2F9c2d9061ce%2F" target="_blank">发布</a>了，上一个版本可以追溯到 2021 年 12 月的 7-Zip 21.07 。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">新版本的 7-Zip 支持提取提取用于 DMG 文件的 APFS 图像。APFS 是Apple 在 Mac OS  和 iOS 中引入的文件系统，该文件系统针对闪存和固态驱动器存储解决方案进行了优化。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">对于 Linux ，新版本的 7-Zip 可以以 1 ns 精度存储额外的文件时间戳，此外还对 TAR 文件的支持进行了优化，可以使用开关 -ttar -mm=pax 或 -ttar -mm=posix 创建 POSIX tar 格式的 TAR 文件。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">对于 Windows ，新的 -snz 开关可以将 Zone.Identifier 流传播到提取的文件，Zone.Identifier 流一般用于确定文件是本地创建或是从 Internet 下载。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><strong>适用系统</strong></p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">此版本 7-Zip 适用于 Windows 10 / 8 / 7 / Vista / XP / 2019 / 2016 / 2012 / 2008 / 2003 / 2000。（ Windows 11 也可以使用）</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">Linux/Posix 移植 ：命令行版本 7-Zip<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fsourceforge.net%2Fprojects%2Fp7zip%2F" target="_blank">p7zip</a>  。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><strong>支持的格式</strong></p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>压缩/解压缩：7z、XZ、BZIP2、GZIP、TAR、ZIP 和 WIM</li> 
 <li>仅解压：APFS、AR、ARJ、CAB、CHM、CPIO、CramFS、DMG、EXT、FAT、GPT、HFS、IHEX、ISO、LZH、LZMA、MBR、MSI、NSIS、NTFS、QCOW2、RAR、RPM、SquashFS 、UDF、UEFI、VDI、VHD、VHDX、VMDK、WIM、XAR 和 Z</li> 
</ul>
