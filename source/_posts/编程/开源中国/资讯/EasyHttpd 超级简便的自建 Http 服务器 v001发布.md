                                                        <h1 style="margin-left:0px; margin-right:0px; text-align:left">EasyHttpd</h1> 
<ul> 
 <li>EasyHttpd 超级简便的自建Http服务器</li> 
 <li>基于Go语言，极少依赖，一个文件，快速开启</li> 
</ul> 
<h2 style="margin-left:0; margin-right:0; text-align:left">下载快速开始</h2> 
<ul> 
 <li>从bin目录下载对应操作系统的可执行文件</li> 
 <li>windows平台，easyhttpd.exe 双击执行</li> 
 <li>从github下载<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fbitepeng%2Feasyhttpd%2Freleases%2Ftag%2Fv0.0.1" target="_blank"> https://github.com/bitepeng/easyhttpd/releases/tag/v0.0.1</a></li> 
 <li>从gitee下载 <a href="https://gitee.com/b0cloud/easyhttpd/tree/main/bin">https://gitee.com/b0cloud/easyhttpd/tree/main/bin</a> <p> </p> </li> 
</ul> 
<h2 style="margin-left:0; margin-right:0; text-align:left">命令行参数</h2> 
<div style="text-align:left"> 
 <div> 
  <pre><span>easyhttpd.exe -r ./ -p :8888</span></pre> 
  <div style="text-align:center">
    
  </div> 
 </div> 
</div> 
<ul> 
 <li>服务根文件目录 
  <ul> 
   <li><code>-r 根路径</code></li> 
  </ul> </li> 
 <li>服务端口(可指定IP) 
  <ul> 
   <li><code>-p 0.0.0.0:8888</code></li> 
   <li><code>-p :8888</code></li> 
  </ul> </li> 
</ul> 
<h2 style="margin-left:0; margin-right:0; text-align:left">使用Go语言功能点</h2> 
<ul> 
 <li><span>TCP网络服务 </span>net/http server</li> 
 <li><span> Go语言内嵌资源 </span>go embed fs</li> 
 <li><span> HTTP网页模板 </span>http template</li> 
 <li><span> HTTP 文件上传 </span>http file upload</li> 
 <li><span> Go语言执行命令行 </span>run cmd open</li> 
 <li><span> Go语言文件操作 </span>file read and write</li> 
 <li><span> 获取主机IP地址 </span>host ip address</li> 
 <li><span> 多平台交叉编译 </span>multi platform cross compilation</li> 
</ul>
