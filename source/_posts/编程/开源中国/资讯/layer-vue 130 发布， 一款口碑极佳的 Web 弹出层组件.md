                                                                                            <p><strong>新特性</strong></p> 
<ul> 
 <li><span style="background-color:#ffffff; color:#40485b">[新增] layer.msg 函数，消息框</span></li> 
 <li><span style="background-color:#ffffff; color:#40485b">[新增] layer.load 函数，加载层</span></li> 
 <li><span style="background-color:#ffffff; color:#40485b">[新增] layer.confirm 函数，询问框</span></li> 
 <li><span style="background-color:#ffffff; color:#40485b">[新增] layer.open 函数，模态窗</span></li> 
 <li><span style="background-color:#ffffff; color:#40485b">[新增] layer.drawer 函数，抽屉</span></li> 
 <li><span style="background-color:#ffffff; color:#40485b">[新增] layer.close 与 layer.closeAll 销毁函数</span></li> 
 <li><span style="background-color:#ffffff; color:#40485b">[新增] isHtmlFangement 属性支持 html 片段解析</span></li> 
 <li><span style="background-color:#ffffff; color:#40485b">[新增] resize 属性，允许窗体拉伸</span></li> 
 <li><span style="background-color:#ffffff; color:#40485b">[新增] reset 方法，提供 area 与 offset 复位</span></li> 
 <li><span style="background-color:#ffffff; color:#40485b">[新增] min 方法，最小化时底边固定</span></li> 
 <li><span style="background-color:#ffffff; color:#40485b">[新增] options 新增 success 与 end 回调函数</span></li> 
 <li><span style="background-color:#ffffff; color:#40485b">[新增] content 属性 VNode 类型</span></li> 
 <li><span style="background-color:#ffffff; color:#40485b">[新增] type 属性 drawer 值，支持抽屉模式</span></li> 
</ul> 
<p><strong>修复</strong></p> 
<ul> 
 <li><span style="background-color:#ffffff; color:#40485b">[修复] offset 为 px 时, 不计算偏移量</span></li> 
 <li><span style="background-color:#ffffff; color:#40485b">[修复] 仅 header 拖动，删除 body 拖动监听</span></li> 
 <li><span style="background-color:#ffffff; color:#40485b">[修复] v-model 隐藏前与隐藏后保持当前维持与尺寸</span></li> 
 <li><span style="background-color:#ffffff; color:#40485b">[修复] layer.close 与 layer.closeAll 不触发 OutAnim 过度动画</span></li> 
</ul> 
<p><strong>依赖</strong></p> 
<ul> 
 <li>[升级] <span style="color:#40485b">vue 3.2.27 版本</span></li> 
</ul>
