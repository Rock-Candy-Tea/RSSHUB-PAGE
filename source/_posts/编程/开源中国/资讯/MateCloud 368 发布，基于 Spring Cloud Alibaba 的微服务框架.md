                                                        <p>基于 Spring Cloud Alibaba 的微服务框架 MateCloud 3.6.8 已经发布，此版本更新内容包括：</p> 
<h2>功能升级</h2> 
<ul> 
 <li>更新README说明及样式</li> 
 <li>全局异常拦截器删除掉AccessDeniedException拦截</li> 
 <li>删除一些无用的类、文件和包</li> 
</ul> 
<h2>依赖升级</h2> 
<ul> 
 <li>升级至Spring Boot 2.5.1</li> 
 <li>升级至Nacos 2.0.2</li> 
 <li>升级至Dubbo 2.7.2</li> 
</ul> 
<p>详情查看：<a href="https://gitee.com/matevip/matecloud/releases/3.6.8">https://gitee.com/matevip/matecloud/releases/3.6.8</a></p>
