                                                                                            <p><span style="background-color:#ffffff"><span style="color:#333333">Android Studio 2021.1.1（代号 "Bumblebee"）第二个 Beta 版本</span></span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fandroidstudio.googleblog.com%2F2021%2F10%2Fandroid-studio-bumblebee-beta-2-now.html" target="_blank">已发布</a><span style="background-color:#ffffff"><span style="color:#333333">。</span></span></p> 
<p><strong>主要更新内容</strong></p> 
<ul> 
 <li>Android Gradle 插件 
  <ul> 
   <li>修复<span style="background-color:rgba(255, 255, 255, 0.95)"><span style="color:#202124">使用经过测试的代码中的内嵌函数时，ASM API 会使测试失败的问题</span></span></li> 
  </ul> </li> 
 <li><span style="background-color:rgba(255, 255, 255, 0.95)"><span style="color:#202124">应用更改</span></span> 
  <ul> 
   <li><span style="background-color:rgba(255, 255, 255, 0.95)"><span style="color:#202124">修复在 API 级别 30 模拟器上进行增量构建后，应用会崩溃并出现 KotlinReflectionNotSupportedError 的问题</span></span></li> 
  </ul> </li> 
 <li><span style="background-color:rgba(255, 255, 255, 0.95)"><span style="color:#202124">Databinding</span></span> 
  <ul> 
   <li><span style="background-color:rgba(255, 255, 255, 0.95)"><span style="color:#202124">Databinding 无法与平台工具 31.0.3 配合使用；无法再使用 platform-tools/api/api-versions.xml</span></span></li> 
  </ul> </li> 
 <li><span style="background-color:rgba(255, 255, 255, 0.95)"><span style="color:#202124">lint</span></span> 
  <ul> 
   <li><span style="background-color:rgba(255, 255, 255, 0.95)"><span style="color:#202124">更新到 AGP 7.1.0-alpha08 后，lint 的 SourceTransformationTestMode.performEdits 中出现 StringIndexOutOfBoundsException</span></span></li> 
   <li><span style="background-color:rgba(255, 255, 255, 0.95)"><span style="color:#202124">在 AGP 7.0 中，lint 会失败并显示 "Can't initialize detector"</span></span></li> 
   <li><span style="background-color:rgba(255, 255, 255, 0.95)"><span style="color:#202124">"针对 Lint 检查的 lint 检查" 未运行</span></span></li> 
  </ul> </li> 
 <li>运行测试 
  <ul> 
   <li><span style="background-color:rgba(255, 255, 255, 0.95)"><span style="color:#202124">通过 Gradle 运行时，Studio 中的测试失败不会显示测试输出</span></span></li> 
  </ul> </li> 
</ul> 
<p><span style="color:#202124"><span style="background-color:rgba(255, 255, 255, 0.95)">详情请查看<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fandroidstudio.googleblog.com%2F2021%2F10%2Fandroid-studio-bumblebee-beta-2-now.html" target="_blank">更新公告</a>。</span></span></p>
