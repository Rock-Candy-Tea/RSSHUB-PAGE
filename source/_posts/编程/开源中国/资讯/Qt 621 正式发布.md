                                                                                            <p>Qt 6.2.1 现已发布，这是 Qt 6.2 系列的第一个补丁版本，包含对 Qt 6.2.0 版本顶部的 220 多个错误修复，并且预计将于 11 月底发布 Qt 6.2.2。</p> 
<p><strong>主要更新内容</strong></p> 
<ul> 
 <li>qtbase 
  <ul> 
   <li>将 SQLite 更新至 v3.36.0</li> 
   <li>OpenSSL：支持使用 TLS 1.3 PSK 回调</li> 
   <li>修复查询与其他字体共享名称的字体别名</li> 
   <li>使 Yu Gothic UI 作为日语的主要后备字体</li> 
   <li>将 PCRE2 更新至 10.38</li> 
   <li>freetype/no-fc：消除不同宽度的字体的歧义</li> 
  </ul> </li> 
 <li>qtdeclarative 
  <ul> 
   <li>将按钮参数添加到 TapHandler.[single|double|] 信号</li> 
   <li>QQmlEngine：细粒度的翻译绑定跟踪</li> 
  </ul> </li> 
</ul> 
<p>其余皆是 bug 修复，请查看<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.qt.io%2Fblog%2Fqt-6.2.1-released" target="_blank">更新公告</a>。</p>
