                                                        <p>DBeaver 21.1.1 现已发布。DBeaver是一个免费开源的通用数据库工具，适用于开发人员和数据库管理员。</p> 
<p><strong>主要更新内容</strong></p> 
<ul> 
 <li>Navigator 
  <ul> 
   <li>快速搜索现在支持多个对象名称</li> 
   <li>改进了活动选项卡配置的实体编辑器首选项</li> 
   <li>对象重命名对话框已修复</li> 
   <li>元数据刷新 UI 已修复</li> 
  </ul> </li> 
 <li>Data editor 
  <ul> 
   <li>行着色现在支持 LIKE 和其他运算符</li> 
   <li>只读列标记已重新设计</li> 
   <li>为列过滤器添加了确认对话框</li> 
   <li>参考面板 UI 已修复（行突出显示）</li> 
   <li>空间数据查看器现在可以记住特定列的选定 SRID</li> 
  </ul> </li> 
 <li>Data export 
  <ul> 
   <li>特殊字符转义已修复为 HTML 和 JSON 格式</li> 
  </ul> </li> 
 <li>SQL editor 
  <ul> 
   <li>连接无效后活动模式重置的问题已修复</li> 
   <li>焦点在查询执行上丢失的问题已修复</li> 
   <li>引用标识符的自动完成是固定的</li> 
   <li>嵌入式 SQL 编辑器的性能大大提高</li> 
   <li>@set 命令的问题已修复</li> 
   <li>修复了查找/替换窗口中 NullPointerException 的问题</li> 
   <li>修复了复杂的 SQL 执行后挂起 UI 的几个问题</li> 
  </ul> </li> 
 <li>Greenplu 
  <ul> 
   <li>​​​​​​​物化视图 DDL 已修复</li> 
  </ul> </li> 
 <li>MySQL 
  <ul> 
   <li>添加了外键编辑支持</li> 
  </ul> </li> 
 <li>Oracle ​​​​​​​ 
  <ul> 
   <li>表 DDL 已修复（缺少分号）</li> 
   <li>时间戳列编辑器已修复</li> 
   <li>基于函数的索引列读取已修复（删除了重复的行）</li> 
   <li>包加载性能大大提高</li> 
  </ul> </li> 
 <li>PostgreSQL ​​​​​​​ 
  <ul> 
   <li>重新设计了列数据类型编辑器</li> 
   <li>无限时间戳处理程序已修复</li> 
   <li>数组和区间数据类型编辑器已修复</li> 
   <li>外部表 DDL 已修复（缺少 OPTIONS）</li> 
   <li>跨架构引用渲染已修复</li> 
  </ul> </li> 
</ul>
