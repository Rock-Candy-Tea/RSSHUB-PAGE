<h2 style="text-align:start">2021.x vs. 2022.x</h2> 
<p><a href="https://www.oschina.net/news/119565/spring-data-2020-0-0-goes-ga">按照版本命名方案</a>，2022.0.0 即原计划的 Spring Data 3.x，在开发新版本的过程中，旧版本 Spring Data 2.x 仍保持继续维护。因此，对于依赖 Sprig Boot 2 的项目，可以继续使用 2021.x 系列（基于 Spring Framework 5.3 和 Java 8），Spring Data 3.x 的底层则基于 Java 17 和 Spring Framework 6，并且兼容 Spring Boot 3。</p> 
<h2>2022.0.0-M1</h2> 
<p>2022.0.0-M1 是 Spring Data 3.x 系列的首个版本，此版本更新内容包括升级底层组件——主要是 JDK 17+ 和迁移到 Jakarta EE 9 API。同时，它还移除了很多弃用许久的工具类，包括几个对过时第三方基础设施的支持包。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fspring-data-commons%2Fwiki%2FRelease-Train-2022.0-%2528Turing%2529-Release-Notes" target="_blank">点此查看完整的发布说明</a>。</p>
