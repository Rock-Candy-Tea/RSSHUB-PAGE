                                                                                            <p>Microsoft PowerToys 是 Windows 系统实用程序，供高级用户调整和简化其 Windows 体验，可最大限度地提高生产力。</p> 
<p>以下是 PowerToys v0.62 版本中的一些亮点：</p> 
<h3>亮点</h3> 
<ul> 
 <li>新工具：Screen Ruler 是一个快速和简单的工具来测量屏幕上的像素</li> 
 <li>新工具：Quick Accent 是一个简单的方法来写带有重音的字母</li> 
 <li>新工具：Text Extractor 的工作原理与 Snipping Tool 类似，但它使用 OCR 将选定区域的文本复制出来，并将其放进剪贴板</li> 
 <li>PowerToy Run 有一个新的插件，可以让你在过去的查询结果中搜索</li> 
</ul> 
<h3>已知问题</h3> 
<ul> 
 <li>Text Extractor 工具在某些情况下无法识别运行 Windows 10 的 ARM64 设备上的文本</li> 
 <li>安装 PowerToys 后，PowerRename 和 Image Resizer 的 Windows 11 上的上下文菜单项在系统重启前可能不会出现。</li> 
 <li>有报告称，用户无法打开 "设置" 窗口。这是与一些应用程序不兼容造成的</li> 
</ul>
