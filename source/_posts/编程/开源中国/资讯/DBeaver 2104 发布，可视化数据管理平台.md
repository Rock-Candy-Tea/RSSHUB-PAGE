                                                                                            <p>DBeaver 21.0.4 现已发布，DBeaver是一个免费开源的通用数据库工具，适用于开发人员和数据库管理员。</p> 
<p><strong>Navigator </strong></p> 
<ul> 
 <li>修复了简单模式下的数据库统计</li> 
 <li>更新了传统的搜索图标</li> 
 <li>删除了原有的编辑工具条和主工具条项目</li> 
 <li>列的重新排序现在支持多列</li> 
 <li>对象查询功能现在支持在注释中搜索</li> 
 <li>DBeaver 主菜单仅在 DBeaver 视角下可见</li> 
</ul> 
<p><strong>SQL 编辑器</strong></p> 
<ul> 
 <li>折叠后的结果面板扩展已修复</li> 
</ul> 
<p><strong>Data 编辑器</strong></p> 
<ul> 
 <li>改进了 "查找替换" 对话框</li> 
 <li>重新设计了创建虚拟唯一键的对话框</li> 
 <li>修复了用于自定义 SQL 查询的引用面板</li> 
</ul> 
<p><strong>SSH </strong></p> 
<ul> 
 <li>修复了对 SSH 代理的支持</li> 
 <li>修复了 SSHJ 扩展安装问题</li> 
</ul> 
<p><strong>MySQL</strong></p> 
<ul> 
 <li><strong>​​​​​​​</strong>修复了列编辑器（删除了 DEFAULT_GENERATED 条款）</li> 
 <li>增加了索引编辑器</li> 
</ul> 
<p><strong>PostgreSQL </strong></p> 
<ul> 
 <li>修复了模式数据类型的读取</li> 
 <li>实现了序列编辑器</li> 
 <li>增加了对数据类型重命名的支持</li> 
</ul>
