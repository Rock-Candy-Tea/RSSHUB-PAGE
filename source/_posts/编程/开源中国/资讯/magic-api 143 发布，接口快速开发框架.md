 <p style="box-sizing: inherit; margin: 0px 0px 20px; line-height: inherit; color: rgb(51, 51, 51); font-family: -apple-system, BlinkMacSystemFont, "Apple Color Emoji", "Segoe UI Emoji", "Segoe UI Symbol", "Segoe UI", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei", "Helvetica Neue", Helvetica, Arial, sans-serif; font-size: 16px; font-style: normal; font-variant-ligatures: normal; font-variant-caps: normal; font-weight: 400; letter-spacing: normal; orphans: 2; text-align: left; text-indent: 0px; text-transform: none; white-space: normal; widows: 2; word-spacing: 0px; -webkit-text-stroke-width: 0px; background-color: rgb(255, 255, 255); text-decoration-thickness: initial; text-decoration-style: initial; text-decoration-color: initial;"><span style="box-sizing: inherit; background-color: rgb(255, 255, 255); color: rgb(64, 72, 91);">magic-api 是一个基于Java的接口快速开发框架，编写接口将通过magic-api提供的UI界面完成，自动映射为HTTP接口，无需定义Controller、Service、Dao、Mapper、XML、VO等Java对象即可完成常见的HTTP API接口开发</span></p> 
 <h3 style="box-sizing: inherit; font-family: "PingFang SC", "Helvetica Neue", "Microsoft YaHei UI", "Microsoft YaHei", "Noto Sans CJK SC", Sathu, EucrosiaUPC, Arial, Helvetica, sans-serif; line-height: 1.8; margin: 22px 0px 16px; font-weight: 600; padding: 0px; font-size: 22px; border: none; color: rgb(51, 51, 51); font-style: normal; font-variant-ligatures: normal; font-variant-caps: normal; letter-spacing: normal; orphans: 2; text-align: left; text-indent: 0px; text-transform: none; white-space: normal; widows: 2; word-spacing: 0px; -webkit-text-stroke-width: 0px; background-color: rgb(255, 255, 255); text-decoration-thickness: initial; text-decoration-style: initial; text-decoration-color: initial;"><span style="box-sizing: inherit; color: rgb(64, 72, 91);">特性</span></h3> 
 <ul style="box-sizing: inherit; margin: 0px 0px 20px; list-style-type: disc; color: rgb(51, 51, 51); font-family: -apple-system, BlinkMacSystemFont, "Apple Color Emoji", "Segoe UI Emoji", "Segoe UI Symbol", "Segoe UI", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei", "Helvetica Neue", Helvetica, Arial, sans-serif; font-size: 16px; font-style: normal; font-variant-ligatures: normal; font-variant-caps: normal; font-weight: 400; letter-spacing: normal; orphans: 2; text-align: left; text-indent: 0px; text-transform: none; white-space: normal; widows: 2; word-spacing: 0px; -webkit-text-stroke-width: 0px; background-color: rgb(255, 255, 255); text-decoration-thickness: initial; text-decoration-style: initial; text-decoration-color: initial;"> 
  <li style="box-sizing: inherit; line-height: 1.875em; margin-top: 0px;">支持MySQL、MariaDB、Oracle、DB2、PostgreSQL、SQLServer 等支持jdbc规范的数据库</li> 
  <li style="box-sizing: inherit; line-height: 1.875em;">支持非关系型数据库Redis、Mongodb</li> 
  <li style="box-sizing: inherit; line-height: 1.875em;">支持集群部署、接口自动同步。</li> 
  <li style="box-sizing: inherit; line-height: 1.875em;">支持分页查询以及自定义分页查询</li> 
  <li style="box-sizing: inherit; line-height: 1.875em;">支持多数据源配置，支持在线配置数据源</li> 
  <li style="box-sizing: inherit; line-height: 1.875em;">支持SQL缓存，以及自定义SQL缓存</li> 
  <li style="box-sizing: inherit; line-height: 1.875em;">支持自定义JSON结果、自定义分页结果</li> 
  <li style="box-sizing: inherit; line-height: 1.875em;">支持对接口权限配置、拦截器等功能</li> 
  <li style="box-sizing: inherit; line-height: 1.875em;">支持运行时动态修改数据源</li> 
  <li style="box-sizing: inherit; line-height: 1.875em;">支持Swagger接口文档生成</li> 
  <li style="box-sizing: inherit; line-height: 1.875em;">基于<a href="https://gitee.com/ssssssss-team/magic-script" style="box-sizing: inherit; background-color: transparent; color: rgb(52, 111, 182); text-decoration: none;" target="_blank">magic-script</a>脚本引擎，动态编译，无需重启，实时发布</li> 
  <li style="box-sizing: inherit; line-height: 1.875em;">支持Linq式查询，关联、转换更简单</li> 
  <li style="box-sizing: inherit; line-height: 1.875em;">支持数据库事务、SQL支持拼接，占位符，判断等语法</li> 
  <li style="box-sizing: inherit; line-height: 1.875em;">支持文件上传、下载、输出图片</li> 
  <li style="box-sizing: inherit; line-height: 1.875em;">支持脚本历史版本对比与恢复</li> 
  <li style="box-sizing: inherit; line-height: 1.875em;">支持脚本代码自动提示、参数提示、悬浮提示、错误提示</li> 
  <li style="box-sizing: inherit; line-height: 1.875em;">支持导入Spring中的Bean、Java中的类</li> 
  <li style="box-sizing: inherit; line-height: 1.875em;">支持在线调试</li> 
  <li style="box-sizing: inherit; line-height: 1.875em; margin-bottom: 0px;">支持自定义工具类、自定义模块包、自定义类型扩展、自定义方言、自定义列名转换等自定义操作</li> 
 </ul>
