                                                                                            <p>code-server v3.11.1 现已发布，具体更新内容如下：</p> 
<p><strong>VS Code v1.57.1 </strong></p> 
<p>升级就像在旧版本上安装新版本一样容易。 code-server 在 ~/.local/share/code-server 中保留了所有的用户数据，以便在两次安装之间保留这些数据。</p> 
<p><strong>Bug Fixes</strong></p> 
<p>修复服务器工作路径并修复 webviews 未加载资源的问题。</p> 
<ul> 
 <li>修复“serviceWorker.js”路径 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fcdr%2Fcode-server%2Fpull%2F3829" target="_blank">#3829</a></li> 
 <li>fix(lib/vscode)：asWebviewUri 中的补丁权限 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fcdr%2Fcode-server%2Fpull%2F3895" target="_blank">#3895</a></li> 
</ul>
