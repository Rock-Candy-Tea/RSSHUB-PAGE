                                                                                            <p>Flatpak 1.14 已发布。Flatpak 是 Linux 应用沙盒和分发框架，其特点包括：</p> 
<ul style="list-style-type:disc; margin-left:0; margin-right:0"> 
 <li>为每个发行版构建：创建一个应用程序并将其分发到整个 Linux 桌面市场</li> 
 <li>稳定的平台：运行时提供了可以依赖的通用库平台</li> 
 <li>一致的环境：在与用户所拥有的环境相同的环境中开发和测试应用程序</li> 
 <li>完全控制依赖关系：Flatpak 可以轻松地将您自己的库捆绑为您的应用程序的一部分</li> 
 <li>简单的构建工具：Flatpak 的构建工具简单易用，并带有全套文档</li> 
 <li>面向未来的构建：Flatpak 应用程序继续与新版本的 Linux 发行版兼容</li> 
 <li>分发变得容易：让快速增长的 Flatpak 用户群可以使用你的应用程序</li> 
 <li>独立项目：Flatpak 是由一个独立的社区开发的</li> 
</ul> 
<p><strong>新版本主要变化</strong></p> 
<ul> 
 <li>支持“--user”选项的 Flatpak 命令现在也允许使用“-u”别名。</li> 
 <li>命令行界面现在可以正确地通知用户正在使用终止运行时扩展的应用程序。</li> 
 <li>Flatpak 的卸载命令现在要求在删除使用中的运行时或运行时扩展之前进行确认。</li> 
 <li>允许子沙盒在会话总线上拥有 MPRIS 名称。</li> 
 <li>修复可能的内存损坏问题。</li> 
 <li>更新 SELinux 政策以涵盖 /var/lib/flatpak 中的符号链接。</li> 
 <li>其他错误修复和改进。</li> 
</ul>
