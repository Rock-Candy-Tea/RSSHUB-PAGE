                                                                                            <p>Snapcraft 是一个用于 Linux 系统上的打包、分发与更新工具，由于绑定了依赖项，所以不需要修改就在所有主要 Linux 系统上运行。</p> 
<p>Snapcraft 5.0 正式发布，更新内容如下：</p> 
<ul> 
 <li>storeapi：改进 candid 的交互错误；</li> 
 <li>cli：引入 metrics 命令；</li> 
 <li>store：为 metrics 添加 API 支持；</li> 
 <li>storeapi：为 metrics 引入数据模型模块；</li> 
 <li>更新 Crystal v1 插件；</li> 
 <li>store: 添加 ReviewQueued 状态；</li> 
 <li>cli：引入 <code>echo.echo_with_pager_if_needed()</code> ；</li> 
 <li>cli：增加对并行远程构建的支持；</li> 
 <li>project：验证元数据链接；</li> 
 <li>cli：让远程构建更方便用户使用；</li> 
 <li>dependencies：更新 click 到 8.0.1；</li> 
 <li>meta：支持元数据链接；</li> 
 <li>更新 kde neon 扩展以使用 qt-5-15-3 的较新内容快照；</li> 
 <li>storeapi：记录响应和注册错误代码；</li> 
 <li>project：将核心警告转换为错误；</li> 
 <li>ROS 插件 V2 的一些修复；</li> 
 <li>snap：移除对传统 re-exec 的支持；</li> 
</ul> 
<p>更多详情可查看：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fsnapcore%2Fsnapcraft%2Freleases%2Ftag%2F5.0" target="_blank">https://github.com/snapcore/snapcraft/releases/tag/5.0</a></p>
