                                                                                            <p>Visual Studio 2022 Preview 1 现已发布，这是第一个 64 位的 Visual Studio 版本。</p> 
<p><strong>下载地址：</strong><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Faka.ms%2Fvs2022preview" target="_blank">https://aka.ms/vs2022preview</a></p> 
<p><img src="https://oscimg.oschina.net/oscnet/up-e79caed0bfae5455390d484b2e7b87ff42d.png" referrerpolicy="no-referrer"></p> 
<p>根据介绍，此预览版的主要目标是测试和调整新的 64 位平台的可扩展性。有了新的 64 位平台，Visual Studio 现在将能够扩展利用所有的系统资源，以提高 Visual Studio 的可靠性，特别是在处理复杂的解决方案或长期使用 Visual Studio 时。</p> 
<p>微软方面表示，由于将 VS 转换为 64 位会影响到套件的每一部分，所以第一次预览的范围要比该团队通常的预览大得多。Visual Studio 2022 预览版可以与早期版本的 Visual Studio 并行安装，在所有三个版本（社区版、专业版和企业版）中均可使用，并且可以免费使用。</p> 
<p>且在 Visual Studio 2022 的早期测试中，客户能够连续几天运行 VS IDE，即使是包含 700 或更多项目的解决方案。</p> 
<p>由于预览版 1 的大部分升级都与 64 位支持有关，因此将从预览版 2 开始发布一系列令人兴奋的新功能和性能改进。可在 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdocs.microsoft.com%2Fen-us%2Fvisualstudio%2Fproductinfo%2Fvs-roadmap" target="_blank">Visual Studio 路线图</a>上阅读有关这些即将推出的功能的所有信息。</p> 
<p>此外，微软方面表示，其在将 Visual Studio 转移到 64 位方面还有一些工作没有完成；Visual Studio 2019 中的一小部分功能也尚不包括在 Visual Studio 2022 预览版 1 中。接下来，他们即将推出 Visua Studio 2022 for Mac 的第一个预览。</p> 
<p>更多详情可查看<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdevblogs.microsoft.com%2Fvisualstudio%2Fvisual-studio-2022-preview-1-now-available%2F" target="_blank">官方博客</a>。</p>
