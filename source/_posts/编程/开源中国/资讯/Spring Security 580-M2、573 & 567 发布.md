                                                                                            <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">Spring Security 的三个分支发布了更新：5.8.0-M2、5.7.3 和 5.6.7。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fspring.io%2Fblog%2F2022%2F08%2F16%2Fspring-security-5-8-0-m2-is-released" target="_blank"><strong>5.8.0-M2</strong></a></p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">此版本更新内容包括升级依赖项、修复错误和增强功能。下面是部分值得注意的变化：</p> 
<ul style="margin-left:0; margin-right:0"> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fspring-security%2Fissues%2F11631" target="_blank">为 SAML post pages 提供基于哈希的内容安全策略</a></li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fspring-security%2Fpull%2F11387" target="_blank">支持自定义重定向策略</a></li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fspring-security%2Fpull%2F11367" target="_blank">为 GitHub Actions 设置权限</a></li> 
</ul> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fspring-security%2Freleases%2Ftag%2F5.8.0-M2" target="_blank">详情查看 release note</a>。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fspring.io%2Fblog%2F2022%2F08%2F15%2Fspring-security-5-7-3-and-5-6-7-available-now" target="_blank"><strong>5.7.3 & 5.6.7</strong></a></p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">这两个版本主要是引入新特性、升级依赖项，以及修复小错误。</p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>添加展示与 WebTestClient 集成的 Kotlin 示例</li> 
 <li>为 GitHub Action 设置权限</li> 
 <li>升级 EnableWebSecurity 的 javadoc，说明 WebSecurityConfigurerAdapter 处于被弃用的状态</li> 
</ul> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><span style="color:#333333">详情查看 release note（</span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fspring-security%2Freleases%2Ftag%2F5.7.3" target="_blank">5.7.3</a>、<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fspring-security%2Freleases%2Ftag%2F5.6.7" target="_blank">5.6.7</a>）。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">Spring Security 是一个能够为基于 Spring 的企业应用系统提供声明式的安全访问控制解决方案的安全框架。它提供了一组可以在 Spring 应用上下文中配置的 Bean，充分利用了 Spring IoC，DI（控制反转 Inversion of Control ,DI:Dependency Injection 依赖注入）和 AOP（面向切面编程）功能，为应用系统提供声明式的安全访问控制功能，减少了为企业系统安全控制编写大量重复代码的工作。</p>
