                                                                                            <p>Mozilla 于今天正式发布了 Firefox 97，该版本更新内容如下：</p> 
<h3>主题</h3> 
<p>在 Firefox 97 中，Mozilla 为 Firefox 带来了 6 个全新的色彩主题方案，这 6 个新的主题还是<strong>限时提供</strong>，未来某个时间点将会下架。</p> 
<p><img alt height="528" src="https://static.oschina.net/uploads/space/2022/0209/070948_9zmm_4937141.png" width="700" referrerpolicy="no-referrer"></p> 
<p>与此同时，此前曾在 Firefox 94 版本中引入的 18 种主题将会下架，除非用户曾使用过这 18 种主题，否则在 Firefox 97 及以上版本中将不再可用。</p> 
<p><img alt height="401" src="https://static.oschina.net/uploads/space/2022/0209/070939_l1Md_4937141.png" width="700" referrerpolicy="no-referrer"></p> 
<h3><strong>其他变化</strong></h3> 
<ul> 
 <li>Firefox 在 Windows 11 设备上支持新的滚动条样式</li> 
 <li>改进了 macOS 设备上的系统字体加载，可以改善 "在某些情况下" 打开和切换标签页的速度</li> 
 <li>Linux 用户无法再直接生成 PostScript 用于在 Linux 中打印，但使用 PostScript 打印机的选项仍然可用。</li> 
</ul> 
<h3>开发者</h3> 
<ul> 
 <li>改进了针对扩展开发者的 cookieStoreId，包括注册特定于容器的内容脚本的功能</li> 
 <li>改进 WebDriver:NewSession 和 WebDriver:NewWindow 稳定性</li> 
 <li>默认情况下，多个 SVGPathSeg API 已被禁用，并将在将来的版本中被删除</li> 
 <li>SVG d 属性现在可以用作 CSS 中的属性</li> 
 <li>CSS 属性 color-adjust 已重命名为 print-color-adjust</li> 
</ul> 
<h3>推迟</h3> 
<ul> 
 <li>原计划于 Firefox 97 中改进的下载功能，将推迟至 Firefox 98 中实施</li> 
</ul>
