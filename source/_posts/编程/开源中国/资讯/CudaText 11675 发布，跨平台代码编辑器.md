                                                                                            <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">CudaText 是一个跨平台的文本编辑器，用 Object Pascal 编写。它是开源项目，启动速度相当快，它可以通过 Python 插件进行扩展，借助 EControl 引擎还带来了功能丰富的语法分析器。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">CudaText 1.167.5 现已发布，更新内容如下：</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><strong>Add：</strong></p> 
<ul> 
 <li>在消息框中使用翻译的 button captions，包括 internal 和 API</li> 
 <li>wiki 页面 https://wiki.freepascal.org/CudaText_VS_other_editors</li> 
 <li>选项 "renderer_tweaks "有新的子选项 "l"</li> 
</ul> 
<p style="color:#000000; margin-left:0px; margin-right:0px; text-align:start"><strong>Fix：</strong></p> 
<ul> 
 <li>历史文件中的书签部分有时会被错误地清除</li> 
 <li>单击 folded-block-rectangle 会隐藏 caret</li> 
 <li>在 Python lexer 中，新输入的文本可能会被 upper _folded_ block "捕获"</li> 
 <li>在某些 Linux 系统上，当 dots_per_inch=91 时，minimap 是空的</li> 
 <li>lexers XML/XML^：仅在“=”后突出显示 quoted string</li> 
 <li>lexer Markdown：斜体规则</li> 
</ul> 
<p>详情可查看：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fcudatext.github.io%2Fhistory.txt" target="_blank">https://cudatext.github.io/history.txt</a></p>
