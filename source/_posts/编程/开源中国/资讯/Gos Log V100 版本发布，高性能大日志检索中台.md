                                                        <p><strong>V1.0.0</strong> 已支持功能</p> 
<ul> 
 <li>✅ 1、基于bufio实现高性能的日志检索算法</li> 
</ul> 
<ul> 
 <li>✅ 2、基于beego实现服务端客户端分离设计</li> 
</ul> 
<ul> 
 <li>✅ 3、基于vue2.0实现服务端前后端分离设计</li> 
</ul> 
<ul> 
 <li>✅ 4、基于DBMan实现数据库表关系设计</li> 
</ul> 
<ul> 
 <li>✅ 5、实现多种数据库适配，支持MySQL、PostgreSQL</li> 
</ul> 
<ul> 
 <li>✅ 6、实现服务端的登陆登出功能</li> 
</ul> 
<ul> 
 <li>✅ 7、实现服务端的客户端增删改查、分页、注册接口</li> 
</ul> 
<ul> 
 <li>✅ 8、实现服务端的项目日志的增删改查、分页接口</li> 
</ul> 
<ul> 
 <li>✅ 9、实现服务端的项目日志单台和全部客户端的查询</li> 
</ul> 
<ul> 
 <li>✅ 10、实现客户端的日志检索功能</li> 
</ul> 
<ul> 
 <li>✅ 11、实现客户端的服务注册功能</li> 
</ul> 
<p> </p> 
<p>V2.0.0 待实现功能展望</p> 
<ul> 
 <li>🔲 1、优化压缩算法、解决压缩包路径过长问题</li> 
</ul> 
<ul> 
 <li>🔲 2、优化并行查询速度，查询全部服务器理论耗时于单台服务相当</li> 
</ul> 
<ul> 
 <li>🔲 3、实现客户端启动即注册，及服务端健康检查功能</li> 
</ul> 
<ul> 
 <li>🔲 4、优化向下截取行可输入，解决默认1000行有时不够用的问题</li> 
</ul> 
<ul> 
 <li>🔲 5、增加安全性，服务端所有接口实现token验证，客户端仅支持注册的服务端进行调用</li> 
</ul> 
<ul> 
 <li>🔲 6、有机会的话做一下权限管理，不同的用户（即不同的部门成员）仅可看到和查询该部门客户端下的服务器日志</li> 
</ul>
