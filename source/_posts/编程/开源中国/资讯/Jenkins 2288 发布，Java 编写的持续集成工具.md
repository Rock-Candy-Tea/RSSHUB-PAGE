                                                        <p>Jenkins 是一款由 Java 编写的开源的持续集成工具。Jenkins 提供了软件开发的持续集成服务。它运行在 Servlet 容器中（例如 Apache Tomcat）。它支持软件配置管理（SCM）工具（包括 AccuRev SCM、CVS、Subversion、Git、Perforce、Clearcase 和 RTC），可以执行基于 Apache Ant 和 Apache Maven 的项目，以及任意的 Shell 脚本和 Windows 批处理命令。Jenkins 是在 MIT 许可证下发布的自由软件。</p> 
<p>Jenkins 2.288 正式发布，本次更新内容如下：</p> 
<ul> 
 <li>减少创建或删除节点时的任务队列锁争用；</li> 
 <li>简化不稳定构建版本的图标（使用感叹号）；</li> 
 <li>修复负载统计图链接，使其包含正确的图表持续时间；</li> 
 <li>用 "新视图 "链接创建新的视图时，要遵循当前文件夹；</li> 
 <li>为新的登录表单添加土耳其语翻译；</li> 
 <li>开发者：移除不必要的捆绑 handlebars 资产。对依赖 handlebars 的插件没有预期的影响；</li> 
 <li>开发者：使用 stapler.jelly.trace 恢复对 Jelly 调试的支持；</li> 
 <li>内部：将 Stapler 从 1.262.1 更新到 1.263，以使用最新的 Apache commons-beanutils。将 Apache commons-beanutils 从 1.9.3 更新到 1.9.4；</li> 
</ul>
