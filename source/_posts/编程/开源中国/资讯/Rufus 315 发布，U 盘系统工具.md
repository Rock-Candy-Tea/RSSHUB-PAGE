                                                        <p>Rufus 是一个可以帮助格式化和创建可引导 USB 闪存盘的工具，它可以快速把 ISO 格式的系统镜像文件快速制作成可引导的 USB 启动安装盘，支持 Windows 或 Linux 启动。3.15 版本更新内容如下： </p> 
<ul> 
 <li>将 GRUB 更新到 2.06 版</li> 
 <li>添加对<code>.vtsi</code>文件的支持</li> 
 <li>添加 openSUSE Live ISO 的解决方法</li> 
 <li>将默认应用程序目录移动到<code>%LocalAppData%\Rufus\</code>并在退出时始终在那里保存日志</li> 
 <li>修复 AppStore 版本的 Rufus 无法存储下载文件的问题</li> 
 <li>修复当 Rufus 位于驱动器根目录时无法打开 Syslinux/GRUB 文件的问题</li> 
 <li>防止在以 DD 模式编写的 ESP 上创建<code>System Volume Information</code></li> 
 <li>防止为 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fpbatard%2Fuefi-ntfs" target="_blank">UEFI:NTFS</a> 分区分配 drive letter</li> 
 <li>防止由于大小导致的持久分区创建错误</li> 
 <li>在运行 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fpbatard%2FFido" target="_blank">Fido ISO 下载脚本</a>之前进行安全检查</li> 
 <li>其他内部修复和改进</li> 
</ul> 
<p>更新说明：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Frufus.ie%2Fzh%2F" target="_blank">https://rufus.ie/zh/</a></p>
