 <p style="margin-left:0">CherryTree 0.99.41 现已发布。CherryTree 是一个支持无限层级分类的笔记软件，Python 编写，支持富文本编辑和代码高亮，支持 Linux 和 Windows 平台。数据采用 sqlite 或 XML 存储，支持密码保护，支持从 NoteCase、KeepNote、Knowit、Tomboy、TuxCards、Treepad、Leo 等笔记软件导入数据。</p> 
 <div>
  <img alt src="https://oscimg.oschina.net/oscnet/up-567a52bd8a28acd66c1970e07c82497720a.png" referrerpolicy="no-referrer">
 </div> 
 <div>
   
 </div> 
 <p style="margin-left:0">此版本更新内容如下：</p> 
 <ul> 
  <li>在图像属性对话框中增加了翻转图像按钮</li> 
  <li>富文本标签的 monospace：增加了前景颜色的可配置性</li> 
  <li>修复了从纯文本剪贴板中插入带有 pipe char 的链接的错误</li> 
  <li>修复通过工具栏按钮而不是右键点击将链接应用到图片上</li> 
  <li>修复了主窗口取消最大化后，关闭，重新打开--仍然最大化的问题</li> 
  <li>修复全屏扩展 tree explorer 的宽度，而不是 tree 在右边时的编辑框</li> 
  <li>修复了导出带有表格的 imported markdown 文件时的故障/崩溃</li> 
  <li>修复 ubuntu 18.04 中的 todo 列表没有循环选择可用的标记</li> 
  <li>print/export 到 pdf 的可扩展标签（h1...h6,small）和 Monospace 使用配置的属性</li> 
  <li>改进了表格排序，不仅使用第一列</li> 
  <li>改进了从 markdown 导入带或不带前/后文本的表格</li> 
  <li>更新了 es.po</li> 
 </ul>
