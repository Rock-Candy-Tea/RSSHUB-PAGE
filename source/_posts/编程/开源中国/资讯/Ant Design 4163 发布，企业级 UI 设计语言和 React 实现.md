                                                        <p>Ant Design 4.16.3 发布了。Ant Design 是一套企业级的 UI 设计语言和 React 实现，使用 TypeScript 构建，提供完整的类型定义文件，自带提炼自企业级中后台产品的交互语言和视觉风格、开箱即用的高质量 React 组件与全链路开发和设计工具体系。</p> 
<p>此版本更新内容如下：</p> 
<ul> 
 <li>修复 Tooltip 的 getPopupContainer 属性为 undefined 时，Tooltip 只能挂载到 body 上的问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F30963" target="_blank">#30963</a></li> 
 <li>修复 Button 加载中图标间距丢失的问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F30962" target="_blank">#30962</a></li> 
 <li>Table 
  <ul> 
   <li>Table 增加内外部 <code>filteredKeys</code> 状态不一致的提示。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F30882" target="_blank">#30882</a></li> 
   <li>修复 Table 固定列上有排序按钮时背景透明的问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F30972" target="_blank">#30972</a></li> 
  </ul> </li> 
 <li>调整 Dropdown 条目宽度，使其展示时尽可能多的利用空间。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F30911" target="_blank">#30911</a></li> 
 <li>修复 Menu 暗色模式下，部分样式被亮色模式污染的问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F30907" target="_blank">#30907</a></li> 
 <li>修复 Dropdown/DatePicker/Select/Cascader 组件自适应位置后动画方向错误的问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F30892" target="_blank">#30892</a></li> 
 <li>国际化 
  <ul> 
   <li>调整 Table 波兰语国际化文案。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F30899" target="_blank">#30899</a></li> 
  </ul> </li> 
</ul>
