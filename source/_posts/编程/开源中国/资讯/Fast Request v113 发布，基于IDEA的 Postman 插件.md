                                                        <p style="text-align:left"><span style="background-color:#ffffff; color:#333333">Fast Request v1.1.3 正式发布，该版本更新内容如下：</span></p> 
<h3 style="text-align:left">新功能</h3> 
<ul> 
 <li>multipartFile support(文件上传支持)</li> 
 <li>controller url fixed variable can replace by config(控制器上的url支持变量替换)</li> 
</ul>

<ul> 
 <li>show tool window when click generate icon(点击图标弹出工具窗口)</li> 
 <li>Beautify the UI(改进了深色和浅色下面一些ui展现)</li> 
</ul>

</ul>
