                                                        <p>Spring Boot 为 4 个不同的分支发布了更新，分别是 3.0.0-M2、2.7.0-M3、2.6.5 和 2.5.11。</p> 
<p><strong>3.0.0-M2</strong></p> 
<p>此版本包含 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fspring-boot%2Freleases%2Ftag%2Fv3.0.0-M2" target="_blank">50 个错误修复、文档改进和依赖项升级</a>。值得关注的新功能：</p> 
<ul> 
 <li>恢复对 EhCache 3 的支持</li> 
 <li>恢复对 H2 控制台的支持</li> 
 <li>改进 @ConstructorBinding Detection</li> 
</ul> 
<p><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fspring-boot%2Fwiki%2FSpring-Boot-3.0.0-M2-Release-Notes" target="_blank">详情查看 release note</a>。</p> 
<p><strong>2.7.0-M3</strong></p> 
<p>此版本包含 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fspring-boot%2Freleases%2Ftag%2Fv2.7.0-M3" target="_blank">90 多个错误修复、文档改进和依赖项升级</a>。值得关注的新功能：</p> 
<ul> 
 <li>支持使用 Podman 生成 Docker 镜像</li> 
 <li>改进的<code>SanitizingFunction</code>支持</li> 
 <li>支持 Spring GraphQL 1.0.0-M6，包括新的 GraphQL 客户端</li> 
</ul> 
<p><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fspring-boot%2Fwiki%2FSpring-Boot-2.7.0-M3-Release-Notes" target="_blank">详情查看 release note</a>。</p> 
<ul> 
 <li>2.6.5：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fspring-boot%2Freleases%2Ftag%2Fv2.6.5" target="_blank">https://github.com/spring-projects/spring-boot/releases/tag/v2.6.5</a></li> 
 <li>2.5.11：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fspring-boot%2Freleases%2Ftag%2Fv2.5.11" target="_blank">https://github.com/spring-projects/spring-boot/releases/tag/v2.5.11</a></li>
