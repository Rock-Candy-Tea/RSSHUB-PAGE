                                                                                            <p style="margin-left:0">DataCleaner 5.8.1 现已发布，DataCleaner 是一个对数据质量进行分析、比较、验证和监督的工具。</p> 
<p style="margin-left:0">此版本更新内容：</p> 
<ul> 
 <li>添加了对使用 Java 9-17 版本运行 DataCleaner 的支持。</li> 
 <li>恢复了 RegexSwap 功能，它基于 GitHub 中开源社区托管的数据，而不是 Human Inference 的（decommissioned）API 端点。</li> 
 <li>更新的依赖项： 
  <ul> 
   <li>Jackson</li> 
   <li>Commons Beanutils</li> 
   <li>Guava</li> 
   <li>Guice</li> 
   <li>Commons IO</li> 
   <li>HTTP client</li> 
   <li>Log4j</li> 
   <li>Logback</li> 
   <li>Junit</li> 
   <li>Commons compress</li> 
   <li>MetaModel</li> 
   <li>OpenCSV</li> 
   <li>PostgreSQL</li> 
   <li>H2 database</li> 
   <li>Freemarker</li> 
   <li>SLF4j</li> 
   <li>Hadoop</li> 
   <li>Cassandra</li> 
   <li>Netty</li> 
   <li>ASM</li> 
   <li>Spring</li> 
   <li>Activation framework</li> 
   <li>JAXB</li> 
  </ul> </li> 
</ul>
