                                                                                            <p>BS Redis Desktop Client 是一个redis的桌面可视化客户端。使用rust和sciter库进行编写。由于市面上很多个开源产品基于electron进行编写。太大太重了。而rdm界面不符合审美所以该软件就诞生了。</p> 
<h2>请帮忙 Star 一下</h2> 
<ul> 
 <li><a href="https://gitee.com/fuyoo/bs">https://gitee.com/fuyoo/bs</a></li> 
</ul> 
<h2>基础功能</h2> 
<ul> 
 <li>mac,windows,linux 等系统支持。</li> 
 <li>支持密码和非密码连接。</li> 
 <li>支持基本的string，list, set, hash的增删改查，key过期时间设置,重命名等。</li> 
 <li>key支持分割符为 <strong>: </strong>的命名空间。</li> 
</ul> 
<h2>后期计划功能</h2> 
<ul> 
 <li>在线pub/sub调试。</li> 
 <li>ssh支持。</li> 
 <li>集群支持。</li> 
</ul>
