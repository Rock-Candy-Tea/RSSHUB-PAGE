                                                                                            <p>digiKam 是 KDE 桌面环境的影像管理和编辑程序，支持所有主要图像格式，并可以组织目录为基础的照片收藏，或按日期、时间、或标签的动态相册。用户还可以对图像添加标题和注释，搜索他们和透过智能文件夹保存搜索。添加插件还可以输出到 Flickr 的相册、Gallery2、谷歌地球的 KML 文件、Simpleviewer、刻录成光盘或创建 Web 画廊。</p> 
<p>近日，digiKam 7.4.0 正式发布，本次更新的部分内容如下：</p> 
<ul> 
 <li>在 bugzilla 上关闭了 1300 多个用户报告。</li> 
 <li>基于 OpenCV 深度学习模块的新神经网络引擎，用于检测和识别人脸。</li> 
 <li>更新幻灯片插件，加入新的模式。</li> 
 <li>为 HtmlGallery 插件添加新主题 Html5Responsive。</li> 
 <li>新增插件 ImageMosaicWall，基于一堆其他照片创建一个图像。</li> 
 <li>脸部扫描对话框的内容现在被简化并嵌入到左边的侧边栏。</li> 
 <li>增加对 Gimp XCF files >= 2.10 的支持。</li> 
 <li>增加对 Linux 平台的 Flatpak bundle 支持。</li> 
 <li>表视图现在可以显示 digiKam Tag-Paths 属性。</li> 
 <li>增加在文件元数据中写入地理位置信息的选项。</li> 
 <li>为高级元数据设置增加颜色标签支持。</li> 
 <li>增加对 HEIF 图像的 Exif、Iptc 和 Xmp 读取支持。</li> 
 <li>新的在线版本检查器和下载器，在 Windows 和 macOS 下实现自动升级。</li> 
 <li>查找重复的项目现在可以使用多核并行处理。</li> 
 <li>新工具可将项目导出至 iNaturatist 网络服务。</li> 
 <li>基于 ExifTool 的新元数据查看器。</li> 
 <li>……</li> 
</ul>
