                                                                                            <p>Electron 团队近日正式发布了 Electron 20.0，你可以使用 <code>npm install electron@latest</code> 用 npm 安装它，或者从官方网站下载它。它包括对 Chromium 104、V8 10.4 和 Node.js 16.15.0 的升级。</p> 
<h3>技术栈变化</h3> 
<ul> 
 <li>Chromium 104.0.5112.65</li> 
 <li>Node v16.15.0</li> 
 <li>V8 v10.4</li> 
</ul> 
<h3>新功能</h3> 
<ul> 
 <li>在 Windows 上增加了沉浸式的黑暗模式</li> 
 <li>增加了对类似面板行为的支持，窗口可以悬浮在全屏的应用程序上</li> 
 <li>更新了 Windows Control Overlay 按钮，使其在 Windows 11 上看起来更像原生的</li> 
 <li>渲染器现在默认是沙盒化的，除非指定 <code>nodeIntegration: true</code> 或 <code>sandbox: false</code></li> 
 <li>在用 nan 构建本地模块时增加了保障措施</li> 
</ul> 
<h3>结束对 17.x.y 的支持</h3> 
<p>根据项目的支持政策，Electron 17.x.y 已经达到了支持的终点，因此官方鼓励开发者和应用程序升级到更新的 Electron 版本。</p>
