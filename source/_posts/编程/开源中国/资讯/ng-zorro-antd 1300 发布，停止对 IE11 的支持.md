                                                                                            <p style="color:#333333; margin-left:0; margin-right:0; text-align:left">ng-zorro-antd 是 Ant Design 的 Angular 实现，主要用于研发企业级中后台产品。全部代码开源并遵循 MIT 协议，任何企业、组织及个人均可免费使用。</p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">ng-zorro-antd 13.0.0 正式发布，更新内容如下：</p> 
<h4 style="text-align:start">停止对 View Engine 的支持</h4> 
<p style="color:#24292f; text-align:start"><code>ng-zorro-antd@13</code><span> </span>不再支持 View Engine 并使用 Ivy 包进行分发，你将获得更快的打包速度和更小的包体积。</p> 
<p style="color:#24292f; text-align:start">获取更多帮助请前往<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fangular.io%2Fguide%2Fivy" target="_blank">Angular Ivy</a><span> </span>查看。</p> 
<h4 style="text-align:start">停止对 IE11 的支持</h4> 
<ul> 
 <li>Angular 可以通过原生的 Web API 使用现代浏览器功能，如 CSS 变量 和 Web 动画等特性</li> 
 <li>移除 IE 相关的 polyfills 和代码使得应用程序更小、加载速度更快</li> 
</ul> 
<p style="color:#24292f; text-align:start">获取更多帮助请前往<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fangular%2Fangular%2Fissues%2F41840" target="_blank">Issue #41840</a><span> </span>查看。</p> 
<h4 style="text-align:start">性能增强</h4> 
<p style="color:#24292f; text-align:start">修复了多处可能造成内存泄漏的问题。</p> 
<h3 style="text-align:start">BREAKING CHANGES</h3> 
<p style="color:#24292f; text-align:start"><strong>dropdown</strong></p> 
<ul> 
 <li><code>[nzHasBackdrop]</code><span> </span>已被移除，请使用<span> </span><code>[nzBackdrop]</code><span> </span>代替。</li> 
</ul>
