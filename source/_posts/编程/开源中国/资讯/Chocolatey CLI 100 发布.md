                                                        <p>Chocolatey 是一个基于 CLI 的 Windows 软件包管理器，有点像 apt-get。Chocolatey CLI 1.0.0 发布，更新内容如下：</p> 
<h3>重要变化</h3> 
<ul> 
 <li>移除了弃用的 Chocolatey 命令和 Shims <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fchocolatey%2Fchoco%2Fissues%2F2468" target="_blank">#2468</a></li> 
 <li>移除了弃用的 Chocolatey 辅助函数 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fchocolatey%2Fchoco%2Fissues%2F2469" target="_blank">#2469</a></li> 
 <li>删除 cpack 命令的别名 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fchocolatey%2Fchoco%2Fissues%2F89" target="_blank">#89</a></li> 
 <li>删除在推送软件包时使用 Chocolatey 社区仓库的旧 URL 的回退功能 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fchocolatey%2Fchoco%2Fissues%2F2628" target="_blank">#2628</a></li> 
 <li>删除推送命令中已废弃的 <code>-t</code> 参数 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fchocolatey%2Fchoco%2Fissues%2F2627" target="_blank">#2627</a></li> 
 <li>确保 Chocolatey 在运行 choco feature 命令时能正确处理已删除的功能 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fchocolatey%2Fchoco%2Fissues%2F2538" target="_blank">#2538</a></li> 
</ul> 
<h3>改进</h3> 
<ul> 
 <li>更新为使用 v1.0.0 版本的 shimgen <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fchocolatey%2Fchoco%2Fissues%2F2602" target="_blank">#2602</a></li> 
 <li>安装 Chocolatey 时，在配置文件中添加 Chocolatey tab-completion 模块时，应添加更多的描述性注释 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fchocolatey%2Fchoco%2Fissues%2F2443" target="_blank">#2443</a></li> 
</ul> 
<h3>文档</h3> 
<ul> 
 <li>更新推送命令的弃用通知，默认源将在 v2.0.0 中删除 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fchocolatey%2Fchoco%2Fissues%2F2634" target="_blank">#2634</a></li> 
 <li>将 Chocolatey shims 标记为已弃用 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fchocolatey%2Fchoco%2Fissues%2F2641" target="_blank">#2641</a></li> 
 <li>将 Get-BinRoot Chocolatey 辅助函数标记为已弃用 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fchocolatey%2Fchoco%2Fissues%2F2470" target="_blank">#2470</a></li> 
 <li>为所有将在 v2.0.0 版本中改变的东西添加弃用通知 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fchocolatey%2Fchoco%2Fissues%2F2462" target="_blank">#2462</a></li> 
 <li>为使用 Chocolatey 和 shimgen 的许可条款提供更清晰的说明 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fchocolatey%2Fchoco%2Fissues%2F2549" target="_blank">#2549</a></li> 
 <li>更新关于使用 Chocolatey 试用许可证的警告 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fchocolatey%2Fchoco%2Fissues%2F2407" target="_blank">#2407</a></li> 
</ul> 
<p>更多详情可查看：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdocs.chocolatey.org%2Fen-us%2Fchoco%2Frelease-notes" target="_blank">https://docs.chocolatey.org/en-us/choco/release-notes</a></p>
