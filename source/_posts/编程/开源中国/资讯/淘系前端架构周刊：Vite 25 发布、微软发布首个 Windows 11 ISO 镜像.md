                                                        <h2><strong>🗞 News</strong></h2> 
<h3><strong>Vite 2.5 发布</strong></h3> 
<p>本月 16 号，Vite 发布了 2.5 版本，该版本修复了一些 Bug，也带来了一些新特性：</p> 
<ul> 
 <li> <p>基于 esbuild 将 CSS 压缩速度提升了 6 倍</p> </li> 
 <li> <p>提供了非 Chromium 内核的 module preload polyfill</p> </li> 
 <li> <p>&etc.</p> </li> 
</ul> 
<p>Changelog：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fvitejs%2Fvite%2Fblob%2Fv2.5.0%2Fpackages%2Fvite%2FCHANGELOG.md" target="_blank">vite/CHANGELOG.md at v2.5.0 · vitejs/vite</a></p> 
<h3><strong>微软发布首个 Windows 11 ISO 镜像</strong></h3> 
<p>本月 22 号，微软发布了 Windows 11 Insider Preview Build 22000.132 的 ISO 镜像，目前已经可以通过下载 ISO 来体验 Windows 11。</p> 
<p>ISO 镜像下载地址：https://www.microsoft.com/en-us/software-download/windowsinsiderpreviewiso</p>
