                                                                                            <p>距离上一次 KMRE 版本更新已经有一段时间了，大家的使用体验如何呢？在上期 KMRE 更新新闻的结尾，我们发起了一轮投票，让大家选出各自最期待的 KMRE 新增功能。从统计数据可以看到，<strong>支持横竖屏切换</strong>功能是呼声最高的，其次是<strong>适配游戏手柄</strong>功能，然后就是<strong>适配 Andriod Studio</strong> 功能和<strong>支持预设手机型号</strong>功能。</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-b2e01594b20ebe8cb03e40f203166895e86.png" referrerpolicy="no-referrer"></p> 
<p>秉承一切以用户为主的理念，本次 KMRE 版本更新，除了新增以上四大功能 ，还增加了<strong>虚拟定位、鼠标灵敏度设置</strong>等其他功能，并对已有功能也进行了优化升级。让我们一起来看看吧~</p> 
<p><strong>1、新增横竖屏切换功能</strong></p> 
<p>投票排名 NO.1，当然得给大家安排上！</p> 
<p>在 KMRE 最新版本中，大家在使用安卓应用时，可以点击右上角“横竖屏切换”。这时，一些支持切换的应用就可以自如地进行横竖屏切换啦，如 WPS 、印象笔记等。</p> 
<p><strong>2、支持游戏手柄</strong></p> 
<p>有很多小伙伴非常期待 KMRE 支持使用游戏手柄。所以，在此次更新版本中，大家连接手柄即可进行单击、连击、左右摇杆等设置，可畅玩你喜爱的那些热门手游啦，快来试试吧！</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-50919092d4ed6608d258995ce5b90f066b7.png" referrerpolicy="no-referrer"></p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-51ebcb3ddbffff78325d56d805a5d4e95ab.png" referrerpolicy="no-referrer"></p> 
<p><strong>3、可开启开发者模式，支持与 Android Studio 联调模式</strong></p> 
<p>在“菜单-设置-通用设置”中，多次点击“当前版本”后，会在设置页面左侧显示“开发者模式”，点击进入即可勾选开启开发者模式。大家在操作过程中如果有不太明白的地方，那就根据提示打开帮助文档看看吧~</p> 
<p><strong>4、支持预设手机型号</strong></p> 
<p>打开 KMRE 属性设置，大家可以在这里预设手机型号，也可以自定义设置自己喜欢的手机品牌和型号~</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-e9409187812cb5189ba9026ba96c20b48ac.jpg" referrerpolicy="no-referrer"></p> 
<p><strong>5、支持鼠标灵敏度设置</strong></p> 
<p>在安卓应用内使用鼠标滚轮进行滑动时，大家是否会感觉滑动速度与自己的使用习惯不符？为优化使用体验，KMRE 增加了鼠标滚轮灵敏度设置，可根据各自的使用习惯设置滑动速度。需要注意的是，当前设置仅对当前应用生效，所以大家在使用不同的安卓应用时也可以体验不同的滚动速度~</p> 
<p><strong>6、支持虚拟定位</strong></p> 
<p>打开虚拟定位，既可定位到当前位置，也可定位到搜索出的位置，快来试试吧！</p> 
<p><strong>7、平行界面适配优化</strong></p> 
<p>开启平行界面后，在微信通讯录页面点击联系人，右侧会并排显示与该联系人的对话界面哦！</p> 
<p><strong>8、摄像头显示优化</strong></p> 
<p>大家在使用腾讯会议时是否有使用过摄像头呢？新版本对腾讯会议横屏时显示的画幅进行了增宽优化，并优化解决了头条系应用录像时的绿屏现象和原生相机使用中的花屏现象，让大家视频时不再烦恼~</p> 
<p>除了以上更新升级，新版本还适配了 Intel 11代CPU核显，电脑比较新的小伙伴也快来安装体验吧！</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-9005fecd8dcd11c8ee30e80850a0e8e6e98.png" referrerpolicy="no-referrer"></p> 
<h3><strong>更 新 方 法</strong></h3> 
<p><strong>1、20.04 版本</strong></p> 
<p><strong>（1）终端升级</strong></p> 
<p>打开终端输入以下命令升级：</p> 
<pre><code>sudo apt update
sudo apt install --only-upgrade kmre
</code></pre> 
<p><strong>（2）软件更新器一键更新</strong></p> 
<p>在开始菜单栏找到软件更新器，一键更新，等待完成即可</p> 
<p><strong>2、22.04 版本</strong></p> 
<p>已经安装 22.04 版本的，也可以通过在终端输入以下命令进行升级，修复一些已知问题：</p> 
<pre><code>sudo apt update
sudo apt install --only-upgrade kmre
</code></pre> 
<p>以上就是我们本次 KMRE 更新内容介绍啦！大家在使用过程中若有任何问题或建议都可前往优麒麟论坛留言哦！</p>
