 <p>Ant Design Mobile 5.0.0-beta.10 发布了。Ant Design Mobile 即 Ant Design 移动端设计规范，<code>antd-mobile</code> 是 Ant Design 的移动规范的 React 实现，服务于蚂蚁及口碑无线业务。</p> 
 <p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><strong>更新内容如下：</strong></p> 
 <ul> 
  <li>特性 
   <ul> 
    <li>Picker 增加了<span> </span><code>title</code><span> </span>属性<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design-mobile%2Fpull%2F3994" target="_blank">#3994</a></li> 
    <li>增加了<span> </span><code>ConfigProvider</code>，提供了最基础的国际化能力</li> 
    <li>Search 增加了<span> </span><code>cancelText</code><span> </span>属性<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design-mobile%2Fpull%2F3998" target="_blank">#3998</a></li> 
    <li>增加了 NavBar 组件<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design-mobile%2Fpull%2F3997" target="_blank">#3997</a></li> 
   </ul> </li> 
  <li>优化 
   <ul> 
    <li>Checkbox 增加了一些用法校验的提示<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design-mobile%2Fpull%2F3990" target="_blank">#3990</a></li> 
   </ul> </li> 
  <li>修复 
   <ul> 
    <li>修复了<span> </span><code>DatePicker.prompt</code><span> </span>中<span> </span><code>onConfirm</code><span> </span>属性没有被触发的问题<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design-mobile%2Fpull%2F3988" target="_blank">#3988</a></li> 
    <li>修复了 ErrorBlock 的样式问题</li> 
   </ul> </li> 
 </ul>
