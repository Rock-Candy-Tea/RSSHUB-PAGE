                                                                                            <p><span style="background-color:#ffffff; color:#27282c"><span> </span>PhpStorm 2021.3 的小更新发布了，该版本带来一些修复和少量改进：</span></p> 
<p><strong>显著更新项</strong></p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>修复：索引 WSL 项目时 IDE 冻结并挂起 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-286059" target="_blank">[IDEA-286059]</a></li> 
 <li>补充：能够生成文件模板的文件名 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-127570" target="_blank">[IDEA-127570]</a></li> 
</ul> 
<ul style="margin-left:0; margin-right:0"> 
 <li>添加：切换编辑器选项卡的快捷方式切换 Big Sur 上的项目选项卡<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-261595" target="_blank">[IDEA-261595]</a></li> 
 <li>已修复：PuTTY SSH 密钥 (.ppk) 支持 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-284623" target="_blank">[IDEA-284623]</a></li> 
 <li>已修复：分配给变量时，php“链式方法调用”->“在新行上放置‘;’”的代码样式格式无法正常工作 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FWI-62622" target="_blank">[WI-62622]</a></li> 
 <li>已修复：Xdebug：“无法获取属性”（自 PHP 8.1 起）<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FWI-64532" target="_blank">[WI-64532]</a></li> 
 <li>已修复：instanceof === false 上没有类型推断 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FWI-64663" target="_blank">[WI-64663]</a></li> 
 <li>已修复：提升属性的重命名不会更改另一个文件中的参数名称 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FWI-64676" target="_blank">[WI-64676]</a></li> 
</ul> 
<p>更完整的更改列表可在<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Farticles%2FWI-A-15%2FPhpStorm-2021.3.3-%28213.7172.28-build%29-Release-Notes" target="_blank">发行说明</a>中找到。</p>
