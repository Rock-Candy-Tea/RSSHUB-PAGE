                                                                                            <p>AppCode 2022.1.2 错误修复更新和 v2022.2 EAP2 现已发布，两个版本都支持 Xcode 13.4；其共同包含的一些更新内容有：</p> 
<ul> 
 <li>修复了导致具有完整构造函数参数的项目从 autocompletion list 中丢失的问题 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FOC-22819" target="_blank">OC-22819</a> )。</li> 
 <li>修复导致 freezes 的性能回归问题 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FOC-22821" target="_blank">OC-22821</a> )。</li> 
</ul> 
<p>此外，在 AppCode 2022.2 EAP 构建中还进行了以下改进：</p> 
<ul> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fapple%2Fswift-evolution%2Fblob%2Fmain%2Fproposals%2F0297-concurrency-objc.md%23additional-objective-c-attributes" target="_blank">添加了对 Swift 5.5</a> 中<code>_Nullable_result</code>Objective-C 属性的支持。</li> 
 <li>PackageDescription 模块包含在预建缓存中。</li> 
 <li>添加了对<code>extern</code>模块引用的支持( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FOC-22218" target="_blank">OC-22218</a> )。</li> 
</ul> 
<p>以及修复了 Kotlin 多平台移动插件中调试应用程序时出现的一些异常（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FKMA-437" target="_blank">KMA-437</a><span style="color:#27282c">, </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FKMA-440" target="_blank">KMA-440</a>）。 </p> 
<p>更多详情可<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fblog.jetbrains.com%2Fappcode%2F2022%2F06%2Fappcode-2022-1-2-bug-fix-update-and-2022-2-eap2-build%2F" target="_blank">查看官方博客</a>。</p>
