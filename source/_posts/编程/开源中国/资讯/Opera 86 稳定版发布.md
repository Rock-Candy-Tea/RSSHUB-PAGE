                                                                                            <p>Opera 是一个多平台网络浏览器，由其同名公司 Opera 开发。Opera 如今是一个基于 Chromium 的浏览器。它通过其用户界面和其他功能将自己与其他浏览器区分开来。</p> 
<p>Opera 86 正式发布，该版本基于 Chromium 100，具体更新内容如下：</p> 
<h3>地址栏下拉菜单中的类别</h3> 
<p>地址栏的下拉菜单现在可以按类别显示结果，如历史、书签和搜索。这使得你可以快速、直观地找到你要找的结果。</p> 
<p><img alt height="394" src="https://static.oschina.net/uploads/space/2022/0424/072855_AbcP_4937141.jpg" width="700" referrerpolicy="no-referrer"></p> 
<h3>Snapshot</h3> 
<p>在复制文本时，增加了一个 Snapshot 快捷键。现在，当选择三个或更多的单词时，弹出窗口中会出现一个 Snapshot 选项。点击它，就可以把选定的文本和周围的区域抓取下来。</p> 
<p><img alt height="394" src="https://static.oschina.net/uploads/space/2022/0424/072907_4mZc_4937141.jpg" width="700" referrerpolicy="no-referrer"></p> 
<p>此外，在 Snapshot 功能本身中，还增加了一个 <strong>Crop Tool</strong> 裁剪工具。这使得用户可以更简单地进一步裁剪和完善已经截取下来的图像。这个新工具位于文本工具的右边。</p> 
<p><img alt height="394" src="https://static.oschina.net/uploads/space/2022/0424/072918_pBVh_4937141.jpg" width="700" referrerpolicy="no-referrer"></p> 
<h3>侧边栏的数字钱包</h3> 
<p>Opera Crypto Wallet 以前只在 Crypto 浏览器中可用，现在还可以在桌面端的 Opera 浏览器中启用，方法是进入设置并搜索 Crypto Wallet，之后就会在侧边栏的功能部分添加一个数字钱包的图标。</p> 
<p><img alt height="394" src="https://static.oschina.net/uploads/space/2022/0424/072936_vR5d_4937141.jpg" width="700" referrerpolicy="no-referrer"></p> 
<h3>其他变化</h3> 
<ul> 
 <li>更新 Chromium 至 100.0.4896.127</li> 
 <li>将 cURL 升级到 7.81</li> 
 <li>[WIN] 在安装程序中对签名验证错误进行更好的诊断</li> 
 <li>改进显示长 URL 或网页标题的建议</li> 
 <li>升级 FFmpeg 编解码器</li> 
 <li>更新 opera_components/avro 中的脚本以使用 Python 3</li> 
 <li>修复无法删除最近搜索条目的问题</li> 
</ul>
