                                                                                            <p>微软近日发布了 <strong>Visual Studio 2022 v17.0.6</strong> 版本，除了 Visual Studio 2022，微软也一同发布了 <strong>Visual Studio 2019 v16.11.10</strong>。各个版本的更新内容如下：</p> 
<h3><strong>Visual Studio 2022 v17.0.6：</strong></h3> 
<ul> 
 <li>更新了 Visual Studio 安装程序以正确记录通道元数据</li> 
 <li>向 17.0 LTSC 通道添加了到期日期信息</li> 
 <li>在 Visual Studio 安装程序中发布了安装或更新的公告</li> 
 <li>更新了 Visual Studio 安装程序，以改善远程企业部署体验。</li> 
 <li>更新了 GitHub 添加帐户的体验，以提高其成功率</li> 
</ul> 
<h3><strong>Visual Studio 2019 v16.11.10</strong></h3> 
<ul> 
 <li>修复了导致 C++ 链接器偶然崩溃的问题</li> 
 <li>修复 x64 的静默错误代码生成问题</li> 
 <li>在后台 C++ 静态分析处理文件时阻止文件被删除的问题</li> 
 <li>解决了 C++20 模式下 C++ ATL CString 相等运算符的问题</li> 
 <li>修复了可能阻止初始化程序在负载测试场景中运行的问题</li> 
</ul> 
<p>更多详情可查看：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdocs.microsoft.com%2Fen-us%2Fvisualstudio%2Freleases%2F2022%2Frelease-notes" target="_blank">https://docs.microsoft.com/en-us/visualstudio/releases/2022/release-notes</a></p>
