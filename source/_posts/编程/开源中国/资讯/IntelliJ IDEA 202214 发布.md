                                                                                            <p>IntelliJ IDEA 2022.1.4 已发布，更新内容主要是修复错误：</p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>修复导致 Commit details 窗格中文本闪烁的问题<span style="background-color:#ffffff; color:#27282c"><span> </span>[</span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-284246" target="_blank">IDEA-284246</a><span style="background-color:#ffffff; color:#27282c">]</span></li> 
 <li>修复 Terraform 插件中本地模块识别的问题<span style="background-color:#ffffff; color:#27282c"><span> </span>[</span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-287371%2FTerraform-cannot-locate-module-locally-unknown-reason" target="_blank">IDEA-287371</a><span style="background-color:#ffffff; color:#27282c">]</span></li> 
 <li>修复 Commit 工具窗口 Before Commit 部分 Run Git hooks 的复选框问题 <span style="background-color:#ffffff; color:#27282c">[</span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-264817%2FRun-Git-hooks-checkbox-becomes-disabled-after-the-first-Commit-action-no-Git-pre-commit-hooks-are-executed" target="_blank">IDEA-264817</a><span style="background-color:#ffffff; color:#27282c">]</span></li> 
 <li>更改 .iml 文件不再删除关联的 Git 映射<span style="background-color:#ffffff; color:#27282c"><span> </span>[</span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-295303%2FChanges-in-root-iml-file-may-break-git-mappings" target="_blank">IDEA-295303</a><span style="background-color:#ffffff; color:#27282c">]</span></li> 
 <li>修复导致目录的外部差异工具 (external diff tool) 发生故障的问题<span style="background-color:#ffffff; color:#27282c"><span> </span>[</span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-295565%2FSuddenly-the-registered-external-diff-tool-doesnt-work-againsometimes" target="_blank">IDEA-295565</a><span style="background-color:#ffffff; color:#27282c">]</span></li> 
 <li>修复导致 IntelliJ IDEA 在 macOS High Sierra 上崩溃的错误<span style="background-color:#ffffff; color:#27282c"><span> </span>[</span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-295565%2FSuddenly-the-registered-external-diff-tool-doesnt-work-againsometimes" target="_blank">IDEA-295565</a><span style="background-color:#ffffff; color:#27282c">]</span></li> 
 <li>修复导致 IDE 在启动时卡顿的问题<span style="background-color:#ffffff; color:#27282c"><span> </span>[</span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-296486" target="_blank">IDEA-296486</a><span style="background-color:#ffffff; color:#27282c">]</span></li> 
</ul>
