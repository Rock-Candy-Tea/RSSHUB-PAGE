                                                                                            <p>Manjaro ARM 是用于 ARM 设备的 Manjaro 发行版，基于 Arch Linux ARM 版，该发行版打包了 Manjaro 工具、主题和为 ARM 设备制作安装镜像的基础工具。</p> 
<p><img alt height="394" src="https://oscimg.oschina.net/oscnet/up-f5a6fc5cc0f454ac697b9383bda182baac7.jpg" width="700" referrerpolicy="no-referrer"></p> 
<p>Manjaro ARM 团队近日发布了 Manjaro ARM 22.08 ，Manjaro ARM 22.08 是在 Manjaro ARM 22.06 之后的两个月发布的，该版本带来了一下更新内容：</p> 
<ul> 
 <li>该版本默认为所有版本引入 PipeWire</li> 
 <li>新版本带来了对 Orange Pi 3 LTS 和 Orange Pi 4 LTS 设备的支持</li> 
 <li>支持 RK356x 设备的官方镜像</li> 
 <li>KDE Plasma 5.25 桌面环境</li> 
 <li>Linux 内核 5.19</li> 
 <li>Pacman 6.0 作为默认软件包管理器</li> 
 <li>以及来自上游和 Manjaro ARM 的各种新的和更新的软件包</li> 
</ul> 
<p>Manjaro ARM 目前支持的设备包括：</p> 
<ul> 
 <li>Raspberry Pi 4 Model B</li> 
 <li>Raspberry Pi 3</li> 
 <li>Raspberry Pi 400</li> 
 <li>Odroid C4</li> 
 <li>Odroid HC4</li> 
 <li>Odroid M1</li> 
 <li>Odroid N2(+)</li> 
 <li>Orange Pi 3 LTS</li> 
 <li>Orange Pi 4 LTS</li> 
 <li>Khadas Vim 2</li> 
 <li>Khadas Vim 3</li> 
 <li>Pine64 LTS</li> 
 <li>Pinebook Pro</li> 
 <li>Radxa Zero</li> 
 <li>Rock 3A</li> 
 <li>Ugoos AM6 Plus</li> 
 <li>Quartz64 Model A</li> 
 <li>Quartz64 Model B</li> 
</ul> 
<p>用户现在可以从官方网站下载预装 KDE Plasma 5.25、GNOME 42.4、Xfce 4.16、MATE 1.26 和 Sway 1.7 环境的 Manjaro ARM 22.08 ，也可以下载没有预装桌面环境的 Minimal 版。</p>
