                                                        <p>Kodi 是由 XBMC 基金会开发的开源媒体播放器，原名 XBMC（最后一个以 XBMC 命名的版本是 13.2 Gotham，14.0 Helix 是第一个以 Kodi 命名的版本），Kodi 可以运行在多种操作系统和硬件平台。 它可以让用户播放本地或网络存储设备中的大多数视频、音乐、播客及各种常见的媒体文件。</p> 
<p>该软件最初是为计划运行在 Xbox 上的，名称也由此而来。随后有了 Android、Linux、BSD、macOS、iOS 和 Windows 操作系统的原生版。</p> 
<p>Kodi 19.2 正式发布，以下是本次更新值得关注的地方：</p> 
<h3><strong>PVR</strong></h3> 
<ul> 
 <li>修正：从收藏夹访问无效的 PVR 频道时崩溃</li> 
 <li>修正：在启动时搜索丢失的频道图标时崩溃</li> 
 <li>修正：启动时因缺少定时器类型而导致的崩溃</li> 
 <li>修复了：从垃圾箱中 "永久删除" 录制内容的问题</li> 
 <li>其他小的错误修复</li> 
</ul> 
<h3>视频库</h3> 
<ul> 
 <li>修正了一个错误，即在搜刮库时删除 UniqueID，这对依赖这些 ID 的插件造成了意外的行为</li> 
</ul> 
<h3>游戏</h3> 
<ul> 
 <li>增加了仿真器和游戏控制器插件的翻译</li> 
</ul> 
<h3><strong>JSON-RPC</strong></h3> 
<ul> 
 <li>将 EPG 标签的图标路径作为 "thumbnail" 属性公开</li> 
 <li>将 EPG 标签的序列号显示为 “seasonnum“ 属性</li> 
</ul> 
<h3>网络</h3> 
<ul> 
 <li>Websocket：处理部分信息</li> 
</ul> 
<h3>插件</h3> 
<ul> 
 <li>修正：官方和第三方模块的显示逻辑</li> 
</ul> 
<h3>Windows</h3> 
<ul> 
 <li>修正：由于添加/删除硬件，在切换显示器时可能会崩溃</li> 
 <li>修正：启用显示刷新率匹配时，音频直通在某些系统上不运行</li> 
 <li>改进：当检测到 HDR 显示时，Windows 10 系统上默认使用 10-bit 交换链</li> 
 <li>修正：当非计算机设备连接到同一网络时，可能发生与 WS-Discovery 有关的崩溃</li> 
 <li>……</li> 
</ul> 
<h3><strong>Android</strong></h3> 
<ul> 
 <li>修正：在运行旧的 Android 版本的设备上，外部存储卷不可见的问题</li> 
</ul> 
<h3><strong>Xbox</strong></h3> 
<ul> 
 <li>修正：由于内存不足的情况，在播放 4K HEVC 视频时崩溃</li> 
 <li>修正：GUI 以 1080P 渲染并升格为 4K，而不是原生 4K</li> 
 <li>修正：4K 分辨率在日志中被错误地报告为 1080P</li> 
 <li>修正：播放 HDR 视频时，默认不使用色调映射</li> 
 <li>修正：Video Debug Info OSD 上缺少显示刷新率信息</li> 
 <li>……</li> 
</ul>
