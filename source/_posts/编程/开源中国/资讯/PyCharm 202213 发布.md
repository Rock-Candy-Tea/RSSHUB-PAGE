<div class="content">
                                                                                            <p><span style="background-color:#ffffff; color:#27282c">PyCharm 2022.1.3 现已<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fblog.jetbrains.com%2Fpycharm%2F2022%2F06%2F2022-1-3%2F" target="_blank">发布</a>，此版本修复了 2022.1 和 2022.1.1 中出现的一些回归问题。主要更新内容如下：</span></p> 
<ul> 
 <li>修复了使用调试控制台截断输出的回归问题 [ <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FPY-53983" target="_blank">PY-53983</a> ]。</li> 
 <li>修复了导致 .pth 文件在 venv 站点包中被忽略的回归问题 [ <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FPY-54321" target="_blank">PY-54321</a> ]。</li> 
 <li>Git 日志面板中的 Cmd+Click / Ctrl+Click已修复，现在可以按预期工作 [ <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-292405" target="_blank">IDEA-292405</a> ]。</li> 
 <li>Welcome screen 上的新建项目按钮再次按预期工作 [ <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-276553%2F20212-Welcome-Screen-New-Project-button-not-working" target="_blank">IDEA-276553</a> ]。</li> 
 <li>修复了在 SCSS 文件的 calc() 函数中使用 $var 时导致误导性错误消息的问题 [ <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FWEB-54056" target="_blank">WEB-54056</a> ]。</li> 
 <li>修复了在 min() 和 max() 参数列表中使用变量时导致 Unexpected term 错误的问题 [ <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FWEB-52057" target="_blank">WEB-52057</a> ]。</li> 
</ul> 
<p>完整的错误修复列表，可查看<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Farticles%2FPY-A-233537973%2FPyCharm-202213-221592127-build-Release-Notes" target="_blank">发行说明</a>。</p>
