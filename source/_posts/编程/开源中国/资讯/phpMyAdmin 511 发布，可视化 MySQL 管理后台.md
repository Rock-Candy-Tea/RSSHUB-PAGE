                                                        <p>phpMyAdmin 5.1.1 现已发布，这是一个错误修复版本。phpMyAdmin 是一个非常受欢迎的 web MySQL 数据库管理工具。它能够创建和删除数据库，创建/删除/修改表格，删除/编辑/新增字段，执行 SQL 脚本等。</p> 
<p>此版本具体更新内容如下：</p> 
<ul> 
 <li>修复了几个 PHP 错误</li> 
 <li>修复了“$cfg['DefaultTabDatabase']”和其他相关配置指令无法正常工作的问题</li> 
 <li>修复 Yaml 导出以引用字符串，即使是数字也要加引号的问题</li> 
 <li>修复由于来自 TCPDF 的 internal guessing code 导致的 TCPDF open_basedir 问题</li> 
 <li>修复快速搜索在使用多个配置的服务器时不工作的问题</li> 
 <li>修复编辑后显示的日期时间的小数点 (.00000)</li> 
 <li>修复文本字段中的新行加倍</li> 
 <li>通过删除<code>&amp;</code>& 字符不需要的转义来修复 URL 生成</li> 
 <li>使用 PHP 8.1 的改进</li> 
 <li>改进了在 Percona 数据库服务器上添加新用户的处理方法</li> 
</ul>
