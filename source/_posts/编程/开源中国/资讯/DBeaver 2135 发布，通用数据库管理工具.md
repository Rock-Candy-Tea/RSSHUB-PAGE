                                                                                            <p style="margin-left:0px">DBeaver 是一个免费开源的通用数据库工具，适用于开发人员和数据库管理员。</p> 
<p style="margin-left:0px">DBeaver 21.3.5 发布，更新内容如下：</p> 
<ul> 
 <li>SQL 编辑器： 
  <ul> 
   <li>修复了执行非常复杂的查询时应用程序冻结的问题</li> 
   <li>参数绑定对话框现在支持多行编辑器</li> 
   <li>脚本选择器面板 UI 已修复</li> 
   <li>新脚本现在将光标定位到脚本末尾</li> 
  </ul> </li> 
 <li><span style="color:#555555">Data editor</span>： 
  <ul> 
   <li>列拖放支持已修复 (Linux)</li> 
   <li>字典表编辑器现在支持默认列名自定义</li> 
   <li>连接特定的数据格式设置保存已修复</li> 
  </ul> </li> 
 <li><span style="color:#555555">MetaData editor</span>： 
  <ul> 
   <li>表 DDL 现在在新表编辑后自动更新</li> 
  </ul> </li> 
 <li><span style="color:#555555">Navigator</span>： 
  <ul> 
   <li>连接期间连接元素扩展的错误已修复</li> 
   <li>表统计刷新已修复</li> 
   <li>列配置（可见性、大小、顺序）保存/加载得到改进</li> 
   <li>“连接时扩展”现在扩展数据库和模式</li> 
   <li>列重新排序现在支持多列</li> 
  </ul> </li> 
 <li>SSH：添加了 known_hosts 支持并在连接之前验证目标主机</li> 
 <li>各种数据库工具：修复了任务向导中的 NullPointer 错误</li> 
 <li>数据传输：添加了表重新创建选项</li> 
 <li>Clickhouse：修复了访问只读连接的问题</li> 
 <li>Firebird：添加了列自动增量选项编辑器</li> 
 <li>通用驱动程序：连接实例化期间的 ClassCast 错误已修复</li> 
 <li>PostgreSQL： 
  <ul> 
   <li>添加了事件触发器支持</li> 
   <li>Greenplum，EDB：添加了本地客户端配置</li> 
   <li>Google Cloud 驱动程序支持已修复（驱动程序依赖项）</li> 
   <li>函数签名现在包括输出参数</li> 
   <li>修复了复杂数据类型中列的 GIS 查看器</li> 
   <li>添加了 CIDR 数据类型支持</li> 
   <li>可延迟外键创建已修复</li> 
  </ul> </li> 
 <li>改进了 SQL Server Synapse 支持（元数据读取）</li> 
 <li>Trino：修复了长时间连接后导航树损坏的问题</li> 
 <li>新驱动程序：添加了 Denodo 驱动程序配置</li> 
</ul>
