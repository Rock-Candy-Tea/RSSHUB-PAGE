                                                                                            <p><span style="color:#000000">WinSCP 是一个 Windows 环境下使用的 SSH 的开源图形化 SFTP 客户端，同时支持 SCP 协议，它的主要功能是在本地与远程计算机间安全地复制文件，并且可以直接编辑文件。</span></p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">目前，WinSCP 5.21.1 已经发布。该版本具体更新内容如下：</p> 
<ul> 
 <li>翻译完成：波兰语。</li> 
 <li>TLS / SSL core 升级到 OpenSSL 1.1.1p。</li> 
 <li>不提供创建到父目录的链接，当它碰巧被选中时，开始创建链接时。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwinscp.net%2Ftracker%2F2091" target="_blank">2091</a></li> 
 <li>错误修复：扩展选项对话框布局在具有不同文本缩放的多个监视器的系统上损坏。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwinscp.net%2Ftracker%2F2090" target="_blank">2090</a></li> 
 <li>错误修复：启动并行 FTP 后台传输时失败。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwinscp.net%2Ftracker%2F2093" target="_blank">2093</a></li> 
 <li>错误修复：远程面板上的目录着色规则不起作用。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwinscp.net%2Ftracker%2F2094" target="_blank">2094</a></li> 
</ul>
