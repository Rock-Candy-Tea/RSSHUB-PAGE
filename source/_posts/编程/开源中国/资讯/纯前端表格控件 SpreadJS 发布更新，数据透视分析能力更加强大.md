                                                        <p>SpreadJS 是一款基于 HTML5 的纯前端表格控件，兼容 450 种以上的 Excel 公式，具备“高性能、跨平台、与 Excel 高度兼容”的产品特性。近日，SpreadJS V14.0 Update1版本发布更新，其数据透视表插件功能再度大幅增强，为用户提供更加稳定、强大的数据分析和数据可视化效果。</p> 
<h2>数据透视表功能增强</h2> 
<p>SpreadJS的<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdemo.grapecity.com.cn%2Fspreadjs%2FSpreadJSTutorial%2Ffeatures%2Fpivot-table%2Foverview%2Fpurejs" target="_blank">数据透视表插件</a>已经覆盖了现有Excel 中90% 以上的功能，并且做到了与 Excel 完全兼容，您可以将设计好的透视表在 SpreadJS 与 Excel 之前进行切换，而不用担心数据丢失。</p> 
<p>与此同时，SpreadJS也发布了多个与透视表有关的 API 接口，方便开发者通过代码控制透视表相关的行为和表现。</p> 
<p>如下是SpreadJS数据透视表的功能更新列表：</p> 
<p>1. 数据源刷新：当数据源发生变更后，可以通过右键菜单完成透视表的刷新。</p> 
<p><img height="365" src="https://oscimg.oschina.net/oscnet/up-87530597fe371c397fec392cd00d4968902.png" width="719" referrerpolicy="no-referrer"></p> 
<p>2. 详细数据的显示和获取：用于查看某一个汇总数据的详细数据。</p> 
<p><img height="596" src="https://oscimg.oschina.net/oscnet/up-5592a29e9d609d71b3024fca3699c428087.png" width="1371" referrerpolicy="no-referrer"></p> 
<p>3. 其他功能增强：</p> 
<ul> 
 <li>筛选与排序</li> 
 <li>条件格式化（透视表）</li> 
 <li>字段的组合和折叠</li> 
 <li>GETPIVOTDATA 函数</li> 
 <li>值显示方式（Show Value As）</li> 
 <li>视图管理</li> 
</ul> 
<h2>公式函数相关</h2> 
<p>SpreadJS 在持续不断地加入最新的公式与函数，方便最终用户使用这些公式和函数快速的构造各类数据表格及表单。</p> 
<p>1. 增强LET 函数功能，确保与 Microsoft 365 中的函数功能完全一致。</p> 
<p>2. 新增 TIMEAGO 函数，用于计算起点到当前时间点的时间。</p> 
<p>3. 新增 WEBSERVICE + FilterJSON 函数。</p> 
<h2>形状相关</h2> 
<p>1. 支持区域快照</p> 
<p>SpreadJS 提供区域快照的形状类型，可在工作表中创建一种视觉引用的功能，区域快照创建一个引用区域的镜像图片。在数据引用区域的内容发生变化的同时，快照图片也会发生改变。</p> 
<p><img height="388" src="https://oscimg.oschina.net/oscnet/up-3b1fd13487c6428af15a4516b8df7197f8d.png" width="725" referrerpolicy="no-referrer"></p> 
<p>2. 其他功能支持：超链接、填充效果</p> 
<h2>迷你图相关</h2> 
<p>SpreadJS的迷你图（Sparkline）作为很重要的数据可视化手段，可以以灵活的方式多角度展示数据规律及特性，此次新版本中也加入了多个迷你图方面的更新。</p> 
<p>1. 直方函数迷你图</p> 
<p>2. KPI 函数（仪表盘）迷你图</p> 
<p>3. IMAGE函数迷你图</p> 
<h2>其他功能增强</h2> 
<p>1. 图表：坐标轴增强、自定义系列</p> 
<p>2. 界面&布局：标签页位置、底部冻结行跟随最后一行、Undo/Redo 栈、复制粘贴跳过隐藏区域</p> 
<p>3. 行为控制：撤销后，选中位置更新到上一次操作的位置</p>
