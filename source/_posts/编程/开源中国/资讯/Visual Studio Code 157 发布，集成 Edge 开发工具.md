                                                        <p>Visual Studio Code 1.57 稳定版已发布。</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-bdb4d8cf3b092fec5a89f23233d7685b44e.png" referrerpolicy="no-referrer"></p> 
<h1>主要更新亮点：</h1> 
<ul> 
 <li> <h2><strong><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fcode.visualstudio.com%2Fupdates%2Fv1_57%23_workspace-trust" target="_blank">Workspace Trust</a></strong></h2> </li> 
</ul> 
<p>Workspace Trust 是为 Workspace 添加的安全加固功能，此功能可让用户决定自己的项目文件夹是否应该允许或限制自动代码执行。</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-fec071478e7d8ad8596c08996ef4ac76caa.png" referrerpolicy="no-referrer"></p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-ec55c681044e6821911792ca4d8ee260336.png" referrerpolicy="no-referrer"></p> 
<ul> 
 <li> <h2><strong><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fcode.visualstudio.com%2Fupdates%2Fv1_57%23_new-getting-started-experience" target="_blank">全新的入门指南体验</a></strong></h2> </li> 
</ul> 
<p>全新的欢迎页面默认面向所有用户开启，帮助他们快速设置和学习 VS Code。</p> 
<p><img src="https://static.oschina.net/uploads/space/2021/0612/082537_5gEq_2720166.gif" referrerpolicy="no-referrer"></p> 
<ul> 
 <li> <h2><strong><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fcode.visualstudio.com%2Fupdates%2Fv1_57%23_remote-repositories" target="_blank">远程仓库 (Remote Repositories)</a></strong></h2> </li> 
</ul> 
<p><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fmarketplace.visualstudio.com%2Fitems%3FitemName%3Dgithub.remotehub" target="_blank">远程仓库扩展</a>允许开发者直接从 VS Code 中即时浏览、搜索、编辑和提交代码到任何 GitHub 仓库，而无需在本地克隆或拥有仓库。</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-c5fadf31aa55e38af718311f5c8ecf3ce39.png" referrerpolicy="no-referrer"></p> 
<ul> 
 <li> <h2><strong><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fcode.visualstudio.com%2Fupdates%2Fv1_57%23_terminal-tabs" target="_blank">Terminal tabs</a></strong></h2> </li> 
</ul> 
<p>Terminal tabs 让开发者轻松地创建、管理和分组多个已打开的终端，此功能目前已默认启用。</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-c0892a044a8491538484b5ccc800952b40c.png" referrerpolicy="no-referrer"></p> 
<p>此功能在 1.57 中的部分更新亮点：</p> 
<ul> 
 <li>支持更改选项卡的颜色以实现它们之间的快速导航和区分</li> 
 <li>支持拖放选项卡以进行重新排列<br> <img height="238" src="https://static.oschina.net/uploads/space/2021/0612/083441_DvJV_2720166.gif" width="205" referrerpolicy="no-referrer"></li> 
 <li>将终端组中的选项卡拖入空白区域会将其从组中删除<br> <img height="237" src="https://static.oschina.net/uploads/space/2021/0612/083506_1XtN_2720166.gif" width="204" referrerpolicy="no-referrer"></li> 
</ul> 
<p><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fcode.visualstudio.com%2Fupdates%2Fv1_57%23_terminal-tabs" target="_blank">详细内容点此查看</a>。</p> 
<ul> 
 <li> <h2><strong><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fcode.visualstudio.com%2Fupdates%2Fv1_57%23_javascript-debugging" target="_blank">集成 Edge 浏览器调试工具</a></strong></h2> </li> 
</ul> 
<p>将 Microsoft Edge 开发工具直接集成到 VS Code 中</p> 
<p><img src="https://static.oschina.net/uploads/space/2021/0612/083847_IVa8_2720166.gif" referrerpolicy="no-referrer"></p> 
<ul> 
 <li> <h2><strong><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fcode.visualstudio.com%2Fupdates%2Fv1_57%23_jsdoc-link-support" target="_blank">支持 JSDoc @link</a></strong></h2> </li> 
</ul> 
<p>在注释中添加@link 标签，以实现快速导航。</p> 
<p><img src="https://static.oschina.net/uploads/space/2021/0612/084013_nje4_2720166.gif" referrerpolicy="no-referrer"></p> 
<ul> 
 <li> <h2><strong><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fcode.visualstudio.com%2Fupdates%2Fv1_57%23_go-to-definition-for-non-jsts-files" target="_blank">针对非代码文件的“Go to Definition”功能</a></strong></h2> </li> 
</ul> 
<p>快速跳转到图像资源和样式表。</p> 
<p><img src="https://static.oschina.net/uploads/space/2021/0612/084145_MgrZ_2720166.gif" referrerpolicy="no-referrer"></p>
