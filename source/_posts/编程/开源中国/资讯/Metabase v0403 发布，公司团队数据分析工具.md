                                                        <p>Metabase 发布了 0.40.3 版本。Metabase 是一个简单的分析工具，通过给公司成员提问，从得到的数据中进行分析、学习。</p> 
<p>此版本更新内容如下：</p> 
<p><strong>Bug 修复</strong></p> 
<ul> 
 <li>Run-overlay 在 GUI 上没有消失的问题 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmetabase%2Fmetabase%2Fissues%2F17514" target="_blank">#17514</a> )</li> 
 <li>保持仪表板订阅的<code>collection_id</code>与仪表板上的同一字段保持同步 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmetabase%2Fmetabase%2Fpull%2F17586" target="_blank">#17586</a> )</li> 
 <li>仪表板导致滚动条根据视口大小不断打开/关闭 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmetabase%2Fmetabase%2Fissues%2F17229" target="_blank">#17229</a> )</li> 
 <li>序列化 --mode skip 不正确地更新某些对象 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmetabase%2Fmetabase%2Fissues%2F16954" target="_blank">#16954</a> )</li> 
 <li>如果没有集合，序列化在转储时崩溃 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmetabase%2Fmetabase%2Fissues%2F16931" target="_blank">#16931</a> )</li> 
 <li>序列化：无法加载到 empty/blank target ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmetabase%2Fmetabase%2Fissues%2F16639" target="_blank">#16639</a> )</li> 
 <li>当侧边栏已打开时单击列格式按钮应正确打开该列的格式侧边栏 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmetabase%2Fmetabase%2Fissues%2F16043" target="_blank">#16043</a> )</li> 
 <li>仪表板订阅不遵循仪表板上卡片的顺序 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmetabase%2Fmetabase%2Fissues%2F15030" target="_blank">#15030</a> )</li> 
 <li>单击离开 Sandbox 模式会中断 perms 页面 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmetabase%2Fmetabase%2Fissues%2F14671" target="_blank">#14671</a> )</li> 
</ul>
