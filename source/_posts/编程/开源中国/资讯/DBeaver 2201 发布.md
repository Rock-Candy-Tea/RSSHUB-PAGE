                                                                                            <p>DBeaver 是一个免费开源的通用数据库工具，适用于开发人员和数据库管理员。</p> 
<p>DBeaver 22.0.1 发布，更新内容如下：</p> 
<ul> 
 <li>Navigator: 
  <ul> 
   <li>在主工具栏/菜单中增加了连接只读状态切换器</li> 
  </ul> </li> 
 <li>Data editor: 
  <ul> 
   <li>修复了自动刷新的用户界面</li> 
   <li>改进了数组数据类型的处理（PostgreSQL、Clickhouse）</li> 
   <li>修复了无效的小数秒渲染问题</li> 
   <li>过滤器面板在不适用的演示中被移除</li> 
   <li>修复了所选行数的计算</li> 
   <li>增加了行的颜色重置选项</li> 
   <li>修正了纯文本视图中编辑控件的启用状态</li> 
   <li>在线日期/时间编辑器中修复了键盘快捷键</li> 
   <li>修复了特定查询中的下一分段自动获取功能</li> 
  </ul> </li> 
 <li>Data transfer: 
  <ul> 
   <li>增加了 "忽略重复的行" 选项</li> 
   <li>改进了对表映射 "re-create" 的支持</li> 
   <li>修复了跳过的列配置保存</li> 
  </ul> </li> 
 <li>SQL editor: 
  <ul> 
   <li>改进了对自定义 SQL 查询的数据过滤器注入</li> 
   <li>修复了事务监视器</li> 
  </ul> </li> 
 <li>Tasks: 修复了任务排序（文件夹在顶部）</li> 
 <li>Command line: 增加了指定外部变量文件的可能性</li> 
 <li>MySQL: 增加了索引编辑支持</li> 
 <li>Oracle: 
  <ul> 
   <li>增加了序列 DDL 标签</li> 
   <li>调度器工作的元数据读取得到了修复</li> 
   <li>改进了数组的显示方式（数字格式得到了修复）</li> 
   <li>修复了物化视图的保存</li> 
  </ul> </li> 
 <li>PostgreSQL: 
  <ul> 
   <li>修复了视图源编辑器</li> 
   <li>在连接设置中指定的模板数据库现在是可见的</li> 
   <li>改进了对服务器版本 < 8 的支持</li> 
   <li>修复了保存的备份/恢复任务的用户界面</li> 
  </ul> </li> 
 <li>Redshift: 程序定义中的搜索已修复</li> 
 <li>Vertica: 
  <ul> 
   <li>元数据性能得到显著提高</li> 
   <li>表/列注释现在可在 Lazy 模式下读取</li> 
  </ul> </li> 
</ul>
