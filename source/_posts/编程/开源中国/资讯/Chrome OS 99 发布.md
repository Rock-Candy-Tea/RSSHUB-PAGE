                                                        <p>间隔四周时间，Google 正式发布 Chrome OS 99，该版本增加了一系列方便的功能，各地用户将陆续收到更新。</p> 
<p><img alt height="394" src="https://static.oschina.net/uploads/space/2022/0310/073509_4xqI_4937141.png" width="700" referrerpolicy="no-referrer"></p> 
<ul> 
 <li>Chrome OS 99 改善了手掌误触的检测，显然这适用于具备触摸屏的 Chromebook 设备。新版本中降低了用户意外点击屏幕并激活应用程序和功能的概率。</li> 
 <li>当处于 Overview（概览）模式时，你现在可以将一个窗口拖到屏幕的顶部来创建一个新的桌面。同时，上个月的 Chrome OS 98 中引入了 Shift + Search + 1 到 8 的快捷方式，让你快速切换到不同桌面。最多可以用这个键盘快捷键创建和浏览八个桌面。</li> 
</ul> 
<p><img alt height="394" src="https://static.oschina.net/uploads/space/2022/0310/073453_HZM1_4937141.png" width="700" referrerpolicy="no-referrer"></p> 
<ul> 
 <li>在 Chrome OS 99 中，Nearby Share 增加了后台扫描功能，当周围的人与你共享文件时，Chromebook 会主动发出提示，这意味着用户不必先进入「可见模式」。</li> 
 <li>Google 已经将 Files 应用重新实施为系统 Web 应用（SWA），因为它多年来一直是一个传统的 Chrome 应用，这是应用的一个底层变化，对功能上没有产生影响。</li> 
 <li>相机应用增加了一种模式（在视频选项中），可以将五秒钟的视频变成可分享的 GIF。</li> 
</ul> 
<p>更多详情可查看：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fchromereleases.googleblog.com%2F" target="_blank">https://chromereleases.googleblog.com/</a></p>
