                                                        <p>Cphalcon 是一个开源的 Web 框架，作为 PHP 语言 C 扩展，它提供了更高的性能与更低的资源消耗。Cphalcon 4.1.1 正式发布，该版本更新内容如下：</p> 
<p>变化：</p> 
<ul> 
 <li>纠正了 PECL 的最大 PHP 版本；</li> 
 <li>支持的 PHP 版本：7.2、 7.3、 7.4；</li> 
</ul> 
<p>修复：</p> 
<ul> 
 <li>修正了 <code>Logger\\Log::log()</code> 日志，以识别所有日志级别；</li> 
 <li>修改了 <code>setClaims</code> 为保护状态，以便 <code>Phalcon\\Security\\JWT\\Builder</code> 类可以被适当扩展；</li> 
</ul> 
<p>更多详情可查看：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fphalcon%2Fcphalcon%2Freleases%2Ftag%2Fv4.1.1" target="_blank">https://github.com/phalcon/cphalcon/releases/tag/v4.1.1</a></p>
