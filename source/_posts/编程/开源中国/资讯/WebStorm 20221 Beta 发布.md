                                                        <p>WebStorm 2022.1 已经达到了 Beta 里程碑，WebStorm 2022.1 Beta 中值得注意的改进包括：</p> 
<h3>改进的结构视图</h3> 
<p>通过结构视图，你可以检查你的文件并更快地跳到它们的特定部分。在这个版本中，WebStorm 让检查 JavaScript 和 TypeScript 文件变得更加简单。文件中的符号现在以更合理的顺序呈现。此外，WebStorm 现在将在导出的符号旁边显示导出图标，以使它们更容易区分。</p> 
<p><img alt height="350" src="https://static.oschina.net/uploads/space/2022/0327/071230_xzAy_4937141.png" width="700" referrerpolicy="no-referrer"></p> 
<h3>更新了文件选择器对话框</h3> 
<p>对于 Windows 和 Linux 用户，WebStorm 使用 NIO APIs 重新实现了文件选择器对话框，改善了浏览文件时的响应速度和 IDE 的整体性能，还更新了该对话框的用户界面。</p> 
<p><img alt height="350" src="https://static.oschina.net/uploads/space/2022/0327/071241_p5az_4937141.png" width="700" referrerpolicy="no-referrer"></p> 
<h3>Inlay Hint 设置的新 UI</h3> 
<p>Inlay Hint 设置现在有了新的用户界面和更新的工作流程。你现在可以根据你希望 IDE 为你提供的提示类型来配置你的偏好设置。</p> 
<p><img alt height="350" src="https://static.oschina.net/uploads/space/2022/0327/071252_LpFE_4937141.png" width="700" referrerpolicy="no-referrer"></p> 
<h3>更新了 Structural Search 和 Replace 对话框</h3> 
<p>Structural Search 和 Replace 对话框现在有一个所有模板的列表，以方便在这些模板之间进行导航。WebStorm 还在对话框的右上角添加了一个 Pin Dialog 图标，并将 Injected code 和 Match case 复选框移至搜索模板窗格的底部。</p> 
<p><img alt height="350" src="https://static.oschina.net/uploads/space/2022/0327/071303_sBfZ_4937141.png" width="700" referrerpolicy="no-referrer"></p> 
<h3>拉取请求注释中的建议修改</h3> 
<p>现在，在 WebStorm 中使用建议修改更容易了，增加了在 IDE 中本地应用或提交修改的功能。</p>
