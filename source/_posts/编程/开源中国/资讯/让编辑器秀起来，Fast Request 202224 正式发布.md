                                                                                            <h1 style="margin-left:0; margin-right:0; text-align:left">简介</h1> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">Restful Fast Request 是一个类似于 Postman 的 IDEA 插件。它是一个强大的 restful api 工具包插件，可以根据已有的方法帮助您快速生成 url 和 params。 Restful Fast Request = API 调试工具 + API 管理工具 + API 搜索工具。 它有一个漂亮的界面来完成请求、检查服务器响应、存储你的 api 请求和导出 api 请求。插件帮助你在 IDEA 界面内更快更高效得调试你的 API。</p> 
<h1 style="margin-left:0; margin-right:0; text-align:left">更新功能</h1> 
<ul style="list-style-type:disc; margin-left:0; margin-right:0"> 
 <li><span style="background-color:var(--badge-tip-color); color:var(--white)">全新的Logo</span></li> 
 <li><span style="background-color:var(--badge-tip-color); color:var(--white)">支持通过注释解析日期</span></li> 
 <li><span style="background-color:var(--badge-tip-color); color:var(--white)">支持解析BaseController</span></li> 
 <li><span style="background-color:var(--badge-info-color); color:var(--white)">自动更新可配置化</span></li> 
 <li><span style="background-color:var(--badge-info-color); color:var(--white)">参数编码传参优化</span></li> 
 <li><span style="background-color:var(--badge-danger-color); color:var(--white)">Map不带泛型解析错误</span></li> 
 <li><span style="background-color:var(--badge-danger-color); color:var(--white)">后置脚本删除添加头信息失败</span></li> 
</ul>
