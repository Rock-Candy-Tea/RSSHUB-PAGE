                                                                                            <p>Spring Boot 为多个分支发布了更新：3.0.0-M3, 2.6.8 & 2.5.14。</p> 
<p><strong><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fspring.io%2Fblog%2F2022%2F05%2F19%2Fspring-boot-3-0-0-m3-available-now" target="_blank">3.0.0-M3</a></strong></p> 
<ul> 
 <li>为 Micrometer Observation, Tracing 和 OtlpMeterRegistry 提供自动配置 (Auto-configuration) 功能</li> 
 <li>恢复对 REST Assured 和 Pooled JMS 的支持</li> 
 <li>在 Spring Boot Gradle 插件中集成 AOT 生成</li> 
 <li>在 Spring Boot Maven 插件中集成 AOT 生成</li> 
 <li><span style="background-color:#ffffff; color:#24292f">提供回调机制，用于自定义 LocalValidatorFactoryBean 的配置</span></li> 
 <li><span style="background-color:#ffffff; color:#24292f">移除在 Maven 进程中运行应用程序的能力</span></li> 
</ul> 
<p><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fspring-boot%2Freleases%2Ftag%2Fv3.0.0-M3" target="_blank">详情</a></p> 
<p><strong><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fspring.io%2Fblog%2F2022%2F05%2F19%2Fspring-boot-2-6-8-available-now" target="_blank">2.6.8</a></strong></p> 
<ul> 
 <li>在 SpringApplication 上配置的默认属性比用<code>@PropertySource</code>配置的属性具有更高的优先权</li> 
 <li>修复当一个工具化的 WebClient 记录指标时，导致请求失败的问题</li> 
 <li>修复 Artemis 的依赖性管理不完整问题</li> 
 <li>修复 Statsd 的缓冲和步骤属性的配置属性缺失</li> 
</ul> 
<p><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fspring-boot%2Freleases%2Ftag%2Fv2.6.8" target="_blank">详情</a></p> 
<p><strong><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fspring.io%2Fblog%2F2022%2F05%2F19%2Fspring-boot-2-5-14-available-now" target="_blank">2.5.14</a></strong></p> 
<p style="text-align:left">结束支持 OSS</p> 
<p style="color:#262626; text-align:left">官方宣布，<code>2.5.x</code>系列的 OSS 支持已结束，将不再发布常规版本。如果开发者还在使用此版本，请考虑在方便时尽早升级。如果此时无法升级，可以<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Ftanzu.vmware.com%2Fspring-runtime" target="_blank">使用商业支持</a>。</p>
