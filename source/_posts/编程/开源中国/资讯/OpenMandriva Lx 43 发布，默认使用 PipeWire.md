                                                                                            <p>OpenMandriva Lx 是从 Mandriva Linux 衍生出来的 Linux 发行版，近日 OpenMandriva Lx 4.3 正式发布，该版本继续使用 LLVM/Clang 来构建软件包，而不是像大多数 Linux 发行版那样使用 GCC。尽可能多地使用 LLVM/Clang 仍然是 OpenMandriva 与其他 Linux 发行版的显着区别之一。</p> 
<p>OpenMandriva Lx 4.3 已经完成了完整的 64 位 Arm 移植，可用于 PinebookPro、Raspberry Pi 4B 和 3B+、Rock Pi 4A、4B 和 4C、Synquacer、Cubox Pulse 等兼容设备。</p> 
<p>在当前版本中，PipeWire 已经成为了默认的声音服务器，从而取代了 PulseAudio。不过，PulseAudio 仍然在 OpenMandriva Lx 的资源库中，你可以在任何时候重新使用它。</p> 
<p><strong>OpenMandriva Lx 4.3 更新内容如下：</strong></p> 
<ul> 
 <li>kernel 5.16.7</li> 
 <li>KDE Frameworks 5.90</li> 
 <li>Plasma Desktop 5.23.5</li> 
 <li>Applications 21.12.2</li> 
 <li>Qt 5.15.3</li> 
 <li>LLVM/Clang 13.0.0</li> 
 <li>systemd 249</li> 
 <li>Java 17（和 18 beta, 19 dev）</li> 
 <li>Calamares 3.2.39</li> 
 <li>binutils 2.37</li> 
 <li>GCC 11.2</li> 
</ul> 
<p><strong>更新的应用程序包括：</strong></p> 
<ul> 
 <li>LibreOffice 7.3.0.3</li> 
 <li>Krita 5.0.2</li> 
 <li>Digikam 7.5</li> 
 <li>SMPlayer 21.10.0</li> 
 <li>VLC 3.0.16</li> 
 <li>Falkon 3.2</li> 
 <li>SimpleScreenRecorder 0.4.3</li> 
 <li>Firefox 96.0.3</li> 
 <li>Thunderbird 91.5.1</li> 
 <li>Virtualbox 6.1.32</li> 
 <li>OBS Studio 27.1.3</li> 
 <li>GIMP 2.10.30</li> 
 <li>Calligra Suite 3.2.1</li> 
 <li>FFMPEG 5.0</li> 
</ul> 
<p>移植到 RISC-V：正在开发中，但还没有包括在 4.3 版本中。</p>
