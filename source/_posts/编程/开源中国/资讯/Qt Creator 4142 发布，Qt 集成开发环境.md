                                                                                            <p>Qt Creator 4.14.2 已经发布，此版本修复了 Qt Creator 各个部分中的一些问题。 </p> 
<p><strong>主要更新内容</strong></p> 
<ul> 
 <li>General 
  <ul> 
   <li>修复了桌面和 AppStream 文件的安装位置</li> 
  </ul> </li> 
 <li>Help 
  <ul> 
   <li>修复了 "Previous/Next Open Document in History" 的崩溃问题</li> 
  </ul> </li> 
 <li>Editing 
  <ul> 
   <li>重新添加了 Autoconf 文件的通用高亮显示</li> 
  </ul> </li> 
 <li>Debugging 
  <ul> 
   <li>修复了 LLDB 的性能问题</li> 
  </ul> </li> 
 <li>macOS 
  <ul> 
   <li>修复了黑暗模式下控件消失的问题</li> 
  </ul> </li> 
</ul> 
<p>详情请查看<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.qt.io%2Fblog%2Fqt-creator-4.14.2-released" target="_blank">更新公告</a>。</p>
