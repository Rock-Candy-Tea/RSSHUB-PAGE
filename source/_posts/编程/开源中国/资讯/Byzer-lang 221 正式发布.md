                                                                                            <p>Byzer 是一门面向 Data 和 AI 的低代码、云原生的开源编程语言。作为一门结合了声明式编程和命令式编程的混合编程语言，Byzer 低代码且类 SQL 的编程逻辑配合内置算法及插件的加持，能帮助数据工作者们高效打通数据链路，完成数据的清洗转换，并快速地进行机器学习相关的训练及预测。</p> 
<p>近日 Byzer 2.2.1 正式发布，提供了更加灵活且稳定的 Byzer 引擎。除此之外，还一同升级发布了 Byzer Notebook 1.0.1，</p> 
<h2>Byzer-lang 2.2.1 更新如下：</h2> 
<h3>优化改进</h3> 
<ul> 
 <li>#1640 优化了 Discretizer ET 的用法和示例</li> 
 <li>#1636 向 CustomFS 添加 saveMode</li> 
</ul> 
<h3>缺陷修复</h3> 
<ul> 
 <li>#1660 修复了使用SendMessage ET 的问题</li> 
 <li>#1654 修复了python 守护进程激增的问题</li> 
 <li>#1650 修复无法识别跨单元变量的问题</li> 
 <li>#1644 修复了 PSExecutor 无法解析 Spark 中的新选项的问题</li> 
 <li>#1642 修复了 !show et; 等分布模式错误</li> 
</ul> 
<h2>Byzer Notebook 1.0.1 <strong>更新如下：</strong></h2> 
<h3><strong>优化改进</strong></h3> 
<ul> 
 <li>#34  当cell 执行失败时，默认收起详细报错信息</li> 
 <li>#61  完成使用方向键切换页面时的展示优化</li> 
</ul> 
<h3><strong>缺陷修复</strong></h3> 
<ul> 
 <li>#35  修复了不能将表保存至 hive 的问题</li> 
 <li>#36  修复了不能加载 delta 的 table 的问题</li> 
 <li>#43  修复了语法提示会触发 notebook 的 hint/annotation 的问题</li> 
 <li>#47  修复了首页创建 notebook 时，页面会自动刷新的问题</li> 
 <li>#48  修复了创建文件夹按钮显示“克隆”的文案错误问题</li> 
 <li>#51   修复了 markdown 预览模式下展示内容不正确的问题</li> 
 <li>#53  修复了工作区页面‘创建’按钮不生效的问题</li> 
 <li>#59  修复了工作区文件列表中新增或重命名文件夹后，移动文件时文件列表不更新的问题</li> 
 <li>#63  修复了移动文件至目标文件夹时存在重名文件时无提示的问题</li> 
 <li>#64  修复了拖动文件时，目录中的文件偶发性消失的问题</li> 
</ul>
