  <strong>软件简介</strong>
 </div> 
 <div>
  dboxShare 是一款简便易用的免费开源企业网盘，基于 .NET 技术开发，用于构建安全高效的文件云存储及云管理平台。 用户无需改变工作习惯，文件双向同步将会根据相应的权限自动进行上传、下载及版本更替，为共享协作提供便捷高效的解决方案。 系统具有安装简单、部署灵活和维护量小的特点，适用于企业组织及团队搭建安全高效的私有云网盘。
 </div> 
 <div>
   
 </div> 
 <div>
  v4.0 主要更新内容
 </div> 
 <div> 
  <ul> 
   <li>新增文件区域划分、文件收藏、文件关注、链接分享等功能模块</li> 
   <li>增加用户限制独立设置 (上传、下载文件大小)</li> 
   <li>改进单个文件上传最大支持 10GB</li> 
   <li>改进部分程序设计</li> 
   <li>修复 bug</li> 
  </ul> 
 </div>

 <div>
  <strong>主要功能特点</strong>
 </div> 
 <div> 
  <ul> 
   <li>支持 SQL Server 和 MySQL 两大主流数据库</li> 
   <li>用户账号禁止多点登录</li> 
   <li>站点独立安全密匙</li> 
   <li>多语言用户界面切换 (支持扩展语言包)</li> 
   <li>支持 5 级部门结构分类 (如果未设置则不显示分类)</li> 
   <li>支持用户角色分类 (如果未设置则不显示分类)</li> 
   <li>用户账号管理 (支持批量创建、更改、归类)</li> 
   <li>AES 银行级算法加密存储文件 (支持启用或禁用)</li> 
   <li>共享文件夹权限管控 (支持根据部门、角色、用户进行多维度快速授权)</li> 
   <li>强大上传模块 (支持大文件分片上传、拖放上传、文件夹上传)</li> 
   <li>单个文件上传最大支持 10GB</li> 
   <li>支持限制单个用户上传、下载文件大小</li> 
   <li>文件历史版本管理 (支持新版本上传、替换、移除)</li> 
   <li>在线预览多种文件格式 (支持自定义扩展)</li> 
   <li>在线提取及解压缩 (支持7z、rar、zip)</li> 
   <li>文件双向同步 (支持文件自动上传、下载、版本更替)</li> 
   <li>文件链接分享 (支持有效时间、提取密码、邮件通知)</li> 
   <li>文件操作日志记录</li> 
  </ul> 
 </div>
