                                                        <p>FileZilla Server 是一个免费开源的 FTP 和 FTPS 服务器，能够提供与服务器的安全加密连接。Filezilla Server 没有实现对 SFTP（SSH 文件传输协议）的支持。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">FileZilla Server 1.1.0-beta1 现已发布，更新内容如下：</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><strong>New features：</strong></p> 
<ul> 
 <li>用户模拟，FileZilla Server 现在可以选择让用户使用系统帐户的凭据登录并使用他们的文件系统权限</li> 
 <li>向用户和组添加了描述字段</li> 
 <li>为用户添加了启用复选框</li> 
 <li>管理 UI 中的状态栏现在显示连接状态</li> 
</ul> 
<p style="text-align:start"><strong><span style="color:#222222">错误修正和小改动：</span></strong></p> 
<ul> 
 <li>Logging fixes</li> 
 <li>修复自动禁止逻辑</li> 
 <li>如果在正在进行的目录列表期间更改用户，则修复了崩溃</li> 
</ul> 
<p>详情可查看：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Ffilezilla-project.org%2F" target="_blank">https://filezilla-project.org/</a></p>
