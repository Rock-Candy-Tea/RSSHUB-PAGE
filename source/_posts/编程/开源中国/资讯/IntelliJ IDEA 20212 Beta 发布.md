                                                        <p>IntelliJ IDEA 2021.2 Beta 版本现已发布。该版本包含了先前在 EAP 版本中介绍的所有变化；还提供了一个备受期待的更新：即，Async Profiler 现在适用于配备 M1 芯片的 Mac 用户。</p> 
<p><img alt height="540" src="https://static.oschina.net/uploads/space/2021/0710/063622_Gip6_2744687.png" width="960" referrerpolicy="no-referrer"></p> 
<p>以下是 v2021.2 的主要功能：</p> 
<ul> 
 <li>Java 项目的 project-wide analysis</li> 
 <li>保存时触发的操作</li> 
 <li>用于管理 Maven 和 Gradle 依赖项的新 UI</li> 
 <li>对 pre-commit checks 的更改</li> 
 <li>GPG signature for commits</li> 
 <li>快速访问 Eclipse 项目</li> 
 <li>自动下载新 Spring Boot 项目的共享索引</li> 
 <li>从 Java EE 自动迁移到 Jakarta EE</li> 
 <li>高级 IDE 设置</li> 
 <li>public、protected 和 private Java members（方法、字段和类）的基于可见性的颜色设置</li> 
</ul>
