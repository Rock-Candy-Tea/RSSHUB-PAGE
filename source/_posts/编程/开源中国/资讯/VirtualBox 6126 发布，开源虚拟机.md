                                                                                            <p>VirtualBox 6.1.26 现已发布。VirtualBox 是一款功能强大的 x86 虚拟机软件，它不仅具有丰富的特色，而且性能也很优异。</p> 
<p>该版本是一个维护版本，修复和/或添加了以下项目：</p> 
<ul> 
 <li>VMSVGA：修复了从保存状态恢复后的 VM screen artifacts（错误 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.virtualbox.org%2Fticket%2F20067" target="_blank">#20067</a>）</li> 
 <li>Storage：修复了某些 CUE 表 CD/DVD 图像的音频字节序。</li> 
 <li>VBoxHeadless：运行中的 VM 将在主机关闭时保存其状态</li> 
 <li>VBoxManage：通过 unattended install 修复 Ubuntu 20.10 ISO 的操作系统检测</li> 
 <li>Linux Additions：修复了多显示器 VM 设置中 VMSVGA 图形适配器的鼠标指针偏移问题（6.1.24 回归）</li> 
</ul> 
<p>更新说明：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.virtualbox.org%2Fwiki%2FChangelog" target="_blank">https://www.virtualbox.org/wiki/Changelog</a> </p>
