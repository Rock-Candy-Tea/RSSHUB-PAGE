<div class="content">
                                                                                            <p><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Ffastcopy.jp%2F" target="_blank">FastCopy 4.0.5 已发布</a>，这是一款 Windows 平台的免费文件复制工具，特点是速度快、不占用资源，尤其是在复制海量文件的场景下比 Windows 默认的文件复制功能快许多倍。如果有复制大量文件或备份文件的场景，这是一款非常实用的工具。</p> 
<blockquote> 
 <p>※ FastCopy 最初是遵循 GPLv3 的开源软件，不过官方最后公开源代码的版本为 3.63，此后再没公开过源代码。</p> 
</blockquote> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-84a8b579a9a132da57f0d7b4579c045c73e.png" referrerpolicy="no-referrer"></p> 
<p>4.x 增加了两个命令行工具，支持通过命令行完成复杂的操作：</p> 
<ol> 
 <li>FcHash.exe（基于 xxHash3/SHA 的快速哈希计算命令行）</li> 
 <li>fcp.exe（FastCopy 的命令行版本）</li> 
</ol> 
<p>FastCopy 在昨天发布了 4.0.5 版本，<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Ffastcopy.jp%2Fhelp%2Ffastcopy_eng.htm%23history" target="_blank">更新内容</a>主要是 bug 修复：</p> 
<ul> 
 <li>修复无效输入会导致异常的问题（v3.0 或更高版本）</li> 
 <li>修复在中断期间导致异常的问题（v3.0 或更高版本）</li> 
</ul> 
<p>其他的更新是支持运行在最新的 Windows 11 上。</p>
