                                                        <p>MPC-HC，全称 Media Player Classic Home Cinema，是 Windows 平台上一个非常轻量级的开源媒体播放器。它支持所有常见的视频和音频文件格式的播放。</p> 
<p>MPC-HC 1.9.12 正式发布，该版本更新内容如下：</p> 
<p><strong>改进：</strong></p> 
<ul> 
 <li>改变了字幕纹理分辨率的默认设置。这一改变只影响到在 4K 分辨率屏幕上的播放，在这种情况下，字幕现在是以 1080p 的分辨率渲染。这提高了在这类系统上的性能。当然，如果你喜欢稍微清晰的字幕，你可以把设置改为 4K；</li> 
 <li>一些与 youtube-dl 支持相关的改进；</li> 
 <li>增加了对下载由 youtube-dl 提取的视频流的字幕支持。你可以在选项 > 高级中指定所需的字幕语言；</li> 
 <li>当在 SRT 字幕中发现 html 标签时，将其删除；</li> 
 <li>增加了启用 OpenSubtitle 选项时的提示，提到需要在他们的网站上创建一个账户；</li> 
 <li>删除了 SubDB 字幕提供者，该网站已经关闭很久了；</li> 
 <li>删除了过时的 RealMedia 和 QuickTime 解码框架。这些框架只在 32 位构建中可用，而且默认情况下不使用它们；</li> 
</ul> 
<p><strong>修复：</strong></p> 
<ul> 
 <li>修正了当打开一个具有很长路径的文件时，最近的文件列表可能被清除的问题；</li> 
 <li>其他一些小的修复和改进；</li> 
</ul>
