                                                        <p>知了-轻量化知识管理 v2.0.0 已经发布。</p> 
<p>本次大版本更新的主要目标是全面支持附件内容、评论内容的全文检索，并完善基于Open API 实现的第三方应用的Demo的通用性</p> 
<ol> 
 <li>新增功能</li> 
</ol> 
<ul> 
 <li> <p>全文检索支持附件正文内容、评论内容（基于Elasticsearch 的嵌套文档实现）</p> </li> 
 <li> <p>搜索结果页新增搜索来源的标签图形（附件/评论）</p> </li> 
 <li> <p>资源详情页展示附件解析状态</p> </li> 
 <li> <p>用户管理新增分配及更新用户Open API 的AppKey</p> </li> 
 <li> <p>系统管理新增Open API Public Key 、默认密码等配置</p> </li> 
</ul> 
<ol start="2"> 
 <li>部分权限调整</li> 
</ol> 
<ul> 
 <li> <p>限制群组外部成员（除超级管理员、内容管理员）访问群组内部资源</p> </li> 
 <li> <p>实现基于AppKey 的Open API 校验方案</p> </li> 
</ul> 
<ol start="3"> 
 <li>基于Open API 实现的第三方应用Demo（数据集发现门户）</li> 
</ol> 
<ul> 
 <li> <p>高级搜索支持显示和实际搜索内容分别设置</p> </li> 
 <li> <p>美化前端样式，支持响应式布局</p> </li> 
 <li> <p>梳理配置项并提取到配置文件中</p> </li> 
</ul> 
<ol start="4"> 
 <li>修复若干BUG</li> 
</ol> 
<ul> 
 <li>评论折叠不能保留换行的问题</li> 
 <li>“重新搜索”按钮无法正确同步搜索条件的问题</li> 
 <li>部分场景下未登录状态搜索结果为空</li> 
</ul> 
<p>详情查看：<a href="https://gitee.com/free4inno-team/zhi/releases/v2.0.0">https://gitee.com/free4inno-team/zhi/releases/v2.0.0</a></p>
