                                                                                            <p>Microsoft PowerToys 是 Windows 系统实用程序，供高级用户调整和简化其 Windows 体验，可最大限度地提高生产力。0.49 版本的目标主要集中在稳定性更新和优化、安装程序更新、常规的错误修复和可访问性改进，更新内容如下：</p> 
<h3>常规</h3> 
<ul> 
 <li>增加了寻找鼠标（Find My Mouse）的功能，利用该功能可以快速定位你显示器上的光标；</li> 
 <li>设置页面的可访问性和小的 UI 改进；</li> 
 <li>添加了各种实用程序在各自编辑器中的设置菜单的深度链接；</li> 
 <li>设置改进，提高了各种选项的清晰度；</li> 
 <li>改进了设置窗口，以便在多显示器条件改变时根据需要调整大小和位置</li> 
</ul> 
<h3>PowerToys Awake</h3> 
<ul> 
 <li>改进了屏幕阅读器的可访问性；</li> 
</ul> 
<h3><strong>Color Picker</strong></h3> 
<ul> 
 <li>改变了颜色选择器的 HEX 格式，删除了 <code>#</code> 字符；</li> 
 <li>对屏幕阅读器和用户界面的可访问性进行了改进，以便在匹配时将颜色与边界区分开来；</li> 
</ul> 
<h3><strong>FancyZones</strong></h3> 
<ul> 
 <li>修正了颜色选择器和 OOBE 窗口被 FancyZones 抢走的问题；</li> 
 <li>修正了不能通过快捷键改变布局的问题；</li> 
 <li>修正了 FancyZones 编辑器的崩溃问题；</li> 
 <li>修正了屏幕锁定后重新设置区域布局的问题；</li> 
 <li>改进了编辑器中屏幕阅读器的可访问性；</li> 
</ul> 
<h3><strong>Keyboard Manager</strong></h3> 
<ul> 
 <li>修正了编辑器在 4K 显示器上以高缩放打开时的崩溃问题；</li> 
</ul> 
<h3><strong>PowerRename</strong></h3> 
<ul> 
 <li>新的 UI 更新；</li> 
</ul> 
<h3>PowerToys Run</h3> 
<ul> 
 <li>增加了 Windows Terminal 插件，默认使用 <code>_</code> 激活命令通过 Windows Terminal 打开 shells；</li> 
 <li>添加了环境变量到 Folder plugin 搜索；</li> 
 <li>修正了在某些文件路径被递归搜索时，程序插件陷入无限循环的问题；</li> 
</ul> 
<h3><strong>Video Conference Mute</strong></h3> 
<ul> 
 <li>在 PowerToys 的稳定版中添加了视频会议静音功能；</li> 
</ul>
