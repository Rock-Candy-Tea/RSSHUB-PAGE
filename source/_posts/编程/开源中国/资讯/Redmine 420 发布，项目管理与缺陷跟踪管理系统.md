                                                        <p>Redmine 是一个网页界面的项目管理与缺陷跟踪管理系统的自由及开放源代码软件工具。它集成了项目管理所需的各项功能：日历、燃尽图和甘特图以协助可视化表现项目与时间限制，问题跟踪和版本控制。此外，Redmine也可以同时处理多个项目。</p> 
<p>Redmine 是以 Ruby on Rails 撰写的架构，它横跨多个平台与数据库。此外，它也是Bitnami 应用库的一部分。</p> 
<p>Redmine 4.2.0 正式发布，以下是本次更新中的亮点：</p> 
<h3>认证/用户账户：</h3> 
<ul> 
 <li>现在可以启用两步认证作为账户的额外安全层；</li> 
 <li>管理员现在可以配置允许或禁止用户账户使用哪些电子邮件域；</li> 
 <li>现在可以从 CSV 中导入用户账户。</li> 
</ul> 
<h3>通知：</h3> 
<ul> 
 <li>通知用户高优先级问题：My Account（我的账户）中的这个新选项允许用户接收高优先级问题的电子邮件通知，即使他们没有被指定。</li> 
</ul> 
<h3>问题：</h3> 
<ul> 
 <li>批量添加相关问题：现在你可以通过提供一个用逗号分隔的问题 id 列表或从自动完成中选择它们来添加多个相关问题；</li> 
 <li>问题页面上的子任务查询链接：问题页面的子任务列表现在包含了带有指向问题页面链接的子任务总数；</li> 
 <li>当问题因开放的子任务或阻止开放的问题而无法关闭时，显示警告和原因：通过显示原因，用户将减少困惑。</li> 
</ul> 
<h3><strong>Watchers:</strong></h3> 
<ul> 
 <li>组可以被添加为问题的观察者；</li> 
 <li>现在可以查看论坛主题了；</li> 
 <li>观察者如果因为观察了一个不可见的对象而不会收到通知，现在在用户界面中被标记为无效；</li> 
</ul> 
<h3>文本格式化：</h3> 
<ul> 
 <li>新的插入表格的工具栏按钮；</li> 
 <li>Wiki 表列排序；</li> 
 <li>工具栏中高亮显示的代码按钮中的语言现在可以由每个用户自定义；</li> 
</ul> 
<h3>其他不错的功能：</h3> 
<ul> 
 <li>一次下载所有附件；</li> 
 <li>自动完成 wiki 页面链接：使用 <code>[[</code> 触发内联自动完成；</li> 
 <li>导入中的自动选择字段映射；</li> 
 <li>有验证错误的字段现在会高亮显示。</li> 
</ul>
