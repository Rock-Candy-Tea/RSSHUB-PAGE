                                                                                            <p style="color:#333333; margin-left:0; margin-right:0; text-align:left">墨干编辑器是一个基于 GNU TeXmacs 的结构化编辑器。</p> 
<p style="color:#40485b; margin-left:0em; margin-right:0em; text-align:start">墨干编辑器衍生自 GNU TeXmacs（作者是 Joris van der Hoeven）。与 TeXmacs 的主要区别是：</p> 
<ul> 
 <li>界面：从 Qt 4.x 升级到了 Qt 5.x</li> 
 <li>脚本引擎：从 Guile 1.8 切换到了 S7 (作者是 Massimiliano Gubinelli)</li> 
</ul> 
<p style="color:#40485b; margin-left:0; margin-right:0; text-align:start">墨干编辑器 v1.0.0 是基于 GNU TeXmacs 2.1.1 定制的基础版本，功能上和 2.1.1 完全一致。墨干编辑器 v1.0.x 将专注于错误修复、易用性改善和性能提升。</p>

</ul> 
<h2 style="margin-left:0; margin-right:0; text-align:start">特性一览</h2> 
<h3 style="margin-left:0; margin-right:0; text-align:start">默认配置优化</h3> 
<ul> 
 <li><code>编辑→首选项→通用→快捷键风格</code>分别从<code>macOS风格</code>和<code>Windows风格</code>切换成了<code>Emacs风格</code></li> 
 <li><code>编辑→首选项→通用→显示询问</code>从<code>在状态栏</code>切换到<code>在弹出窗口</code></li> 
 <li><code>编辑→首选项→通用→缓冲区管理</code>从<code>一个文档一个窗口</code>切换成<code>多个文档共享一个窗口</code></li> 
 <li><code>编辑→首选项→其它→执行文档更新</code>从<code>一次</code>切换成<code>三次</code></li> 
</ul> 
<h3 style="margin-left:0; margin-right:0; text-align:start">快捷键优化</h3> 
<p style="color:#40485b; margin-left:0; margin-right:0; text-align:start">Windows 平台的 Meta 快捷键默认是 Win，多数和系统快捷键冲突，目前已经将 Windows 平台的 Meta 快捷键从 Win 切换为 Alt，解决了大部分 Windows 平台快捷键失效的问题。</p> 
<h3 style="margin-left:0; margin-right:0; text-align:start">新增插件</h3> 
<ul> 
 <li>quiver 插件能够帮助你快速绘制交换图（感谢<span> </span><a href="https://gitee.com/xuyiqi1">@Yiqi Xu<span> </span></a>）</li> 
 <li>pyenv 插件能够帮助你切换 Python 会话所使用的 Python 虚拟环境</li> 
</ul>
