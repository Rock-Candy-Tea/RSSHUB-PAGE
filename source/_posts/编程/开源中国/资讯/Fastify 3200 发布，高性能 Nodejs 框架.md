                                                        <p>Fastify 3.20.0 发布了，Fastify 是一个高度专注于以最少开销和强大的插件架构为开发者提供最佳体验的 Node.js Web 框架，速度极快，它的灵感来源于 Hapi 和 Express。</p> 
<p>此版本更新内容包括：</p> 
<ul> 
 <li>将 this types 添加到 decorator functions 中 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffastify%2Ffastify%2Fpull%2F3203" target="_blank">#3203</a></li> 
 <li>修复在 setErrorHandler 中同步返回值时请求挂起的问题 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffastify%2Ffastify%2Fpull%2F3211" target="_blank">#3211</a></li> 
 <li>docs：添加 fastify-recaptcha <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffastify%2Ffastify%2Fpull%2F3212" target="_blank">＃3212</a></li> 
 <li>feat：更新 Getting-Started.md <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffastify%2Ffastify%2Fpull%2F3213" target="_blank">#3213</a></li> 
 <li>fix：在 FastifyLoggerOptions 接口中添加文件选项<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffastify%2Ffastify%2Fpull%2F3214" target="_blank">#3214</a></li> 
 <li>扩展内容类型解析器 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffastify%2Ffastify%2Fpull%2F3210" target="_blank">#3210</a></li> 
 <li>修复通用约束类型 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffastify%2Ffastify%2Fpull%2F3221" target="_blank">#3221</a></li> 
 <li>docs：修复路由通用示例中的接口 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffastify%2Ffastify%2Fpull%2F3222" target="_blank">#3222</a></li> 
</ul>
