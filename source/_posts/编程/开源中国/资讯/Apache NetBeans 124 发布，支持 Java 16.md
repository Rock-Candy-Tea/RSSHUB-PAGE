<blockquote> 
 <p>10.14.4 之前的 macOS 版本需要安装 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fsupport.apple.com%2Fkb%2Fdl1998%3Flocale%3Den_US" target="_blank">Swift 5 Runtime</a> 才能启动 Apache NetBeans 12.4</p> 
</blockquote> 
<p>Apache NetBeans 12.4 支持在 JDK 8 LTS、11 以及 JDK 16（即此 NetBeans 版本发布时的当前 JDK 版本）上运行。</p> 
<p>主要更新内容：</p> 
<ul> 
 <li>支持 Java 16</li> 
 <li>内置安装 OpenJDK</li> 
 <li>支持 Jakarta EE 9</li> 
 <li>支持 Micronaut 框架 
  <ul> 
   <li>创建 Micronaut 项目添加新的向导</li> 
   <li> <p>实现了 Micronaut yaml 配置文件中的代码补全</p> </li> 
   <li> <p>实现了 Micronaut yaml 配置文件的 HyperlinkProvider</p> </li> 
   <li>实现了 Micronaut yaml 配置文件的 WhereUsed RefactoringPlugin</li> 
  </ul> </li> 
 <li>增强 PHP 支持</li> 
 <li>升级 Gradle 至 7.0 版本</li> 
</ul>
