                                                                                            <p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><span style="color:#333333">Serverless 架构开发框架 Serverless Framework 发布了 3.0.1 版本，该框架使用 AWS Lambda、Azure Functions、Google CloudFunctions 等技术，可以构建 Serverless 架构的 Web、移动和 IoT 应用。</span></p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><span style="color:#333333">此版本更新内容包括：</span></p> 
<h3 style="margin-left:.6em; margin-right:0; text-align:start"><strong>Bug修复</strong></h3> 
<ul style="margin-left:0; margin-right:0"> 
 <li>报告更有意义的严格模式错误 (<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fserverless%2Fserverless%2Fpull%2F10574" target="_blank">#10574</a><span> </span>)</li> 
</ul> 
<h3 style="margin-left:.6em; margin-right:0; text-align:start"><strong>改进</strong></h3> 
<ul style="margin-left:0; margin-right:0"> 
 <li><strong>CLI Onboarding：</strong>细微的措辞调整 (<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fserverless%2Fserverless%2Fpull%2F10575" target="_blank">#10575</a><span> </span>)</li> 
</ul> 
<h3 style="margin-left:.6em; margin-right:0; text-align:start"><strong>模板</strong></h3> 
<ul style="margin-left:0; margin-right:0"> 
 <li>将依赖项<code>aws-nodejs-typescript</code><span> </span>更新到 v3 (<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fserverless%2Fserverless%2Fpull%2F10582" target="_blank">#10582</a><span> </span>)</li> 
</ul>
