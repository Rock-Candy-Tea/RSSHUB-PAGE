                                                                                            <p>Jenkins 是一款由 Java 编写的开源的持续集成工具。Jenkins 提供了软件开发的持续集成服务。它运行在 Servlet 容器中（例如 Apache Tomcat）。它支持软件配置管理（SCM）工具（包括 AccuRev SCM、CVS、Subversion、Git、Perforce、Clearcase 和 RTC），可以执行基于 Apache Ant 和 Apache Maven 的项目，以及任意的 Shell 脚本和 Windows 批处理命令。Jenkins 是在 MIT 许可证下发布的自由软件。</p> 
<p>自 2.283 版本以来的变化：</p> 
<ul> 
 <li>默认情况下，切换为发送POST请求以获取表单验证URL；</li> 
 <li>将 matrix-auth 从 2.6.5 升级到 2.6.6；</li> 
 <li>xstream 从 1.4.15 升级到 1.4.16；</li> 
 <li>将 bouncycastle-api 插件从 2.16 升级到 2.20。将 Bouncy Castle 库升级到 1.64；</li> 
 <li>修复了在执行 <code>ProcessTree.get()</code> 时， <code>NoClassDefFoundError</code> 异常;</li> 
 <li>当流行程度相等时，按名称对可用插件进行排序；</li> 
 <li>剪贴板、Gear 和用户图标再次换成透明样式；</li> 
 <li>将 Jenkins CLI 从 1.7.0 更新到 SSHD Core 的 2.6.0；</li> 
 <li>为 Dashboard、Views、Rename 添加日文翻译。</li> 
</ul>
