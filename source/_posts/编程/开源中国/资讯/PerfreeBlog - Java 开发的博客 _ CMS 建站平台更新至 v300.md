                                                                                            <p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><span style="background-color:#ffffff; color:#40485b">PerfreeBlog 是一款基于 Java SpringBoot 开发的博客建站平台，支持多主题及扩展插件功能，给您带来全新的创作体验</span></p> 
<h2 style="margin-left:0; margin-right:0; text-align:left">功能简介</h2> 
<ul style="list-style-type:disc; margin-left:0; margin-right:0"> 
 <li>设计简洁，界面美观</li> 
 <li>采用<span> </span><a href="https://gitee.com/link?target=https%3A%2F%2Fwww.markdownguide.org%2F">Markdown</a>/ 富文本双编辑器，支持一键插入视频、图片、附件等</li> 
 <li>支持多主题自由切换</li> 
 <li>强大的主题在线编辑功能，支持多种文件操作，实时生效，支持在网页直接创建新主题，快速开发页面</li> 
 <li>支持自定义页面</li> 
 <li>支持友情链接</li> 
 <li>支持附件管理，附件在线预览</li> 
 <li>支持扩展插件，编写 java 代码打包为 jar, 直接后台安装生效，丰富程序多样性</li> 
 <li>主题开发简单快速，多种自定义指令及 api 接口支持</li> 
 <li>支持邮件服务，评论信息邮件提醒</li> 
 <li>安装部署简单</li> 
 <li>支持阿里云 OSS / 腾讯 COS / 七牛 OSS</li> 
 <li>支持<span> </span><a href="https://gitee.com/link?target=https%3A%2F%2Fwww.mysql.com">mysql</a>/<a href="https://gitee.com/link?target=https%3A%2F%2Fwww.sqlite.org">sqlite</a><span> </span>数据库</li> 
</ul> 
<h2 style="margin-left:0; margin-right:0; text-align:left">更新内容</h2> 
<ul> 
 <li style="text-align:start">新增: 主题支持vue/angular/react进行开发</li> 
 <li style="text-align:start">新增: 标签新增颜色/别名/封面图</li> 
 <li style="text-align:start">新增: 分类新增别名/封面图</li> 
 <li style="text-align:start">新增: 配置文件支持配置登录超时时间</li> 
 <li style="text-align:start">新增: 文章新增点赞数量字段及相关接口</li> 
 <li style="text-align:start">新增: 增加个别接口访问限制拦截(如评论/点赞等)</li> 
 <li style="text-align:start">新增: 动态管理</li> 
 <li style="text-align:start">新增: 如未配置邮箱服务的情况下,重置密码时,可在后台日志查看验证码</li> 
 <li style="text-align:start">新增: 文章编写/页面编写->右侧面板支持收起展开</li> 
 <li style="text-align:start">新增: 文章接口支持按类型搜索</li> 
 <li style="text-align:start">新增: 默认主题对标签颜色,动态的支持</li> 
 <li style="text-align:start">优化: 后台UI整体调整</li> 
 <li style="text-align:start">优化: 附件管理操作逻辑及样式</li> 
 <li style="text-align:start">优化: 标签管理/分类管理操作逻辑及样式</li> 
 <li style="text-align:start">优化: 安装界面及操作逻辑优化</li> 
 <li style="text-align:start">修复: markdown编辑器滚动异常问题</li> 
 <li style="text-align:start">修复: 删除标签时未清除文章与标签关联的问题</li> 
 <li style="text-align:start">修复: 文章列表分类筛选框不展示子分类的问题</li> 
 <li style="text-align:start">调整: 升级Spring/Shiro/Mybatis等依赖库版本</li> 
</ul>
