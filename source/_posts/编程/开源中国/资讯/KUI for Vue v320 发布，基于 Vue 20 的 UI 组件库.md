                                                        <p>KUI for Vue v3.2.0 已经发布，基于 Vue 2.0 的 UI 组件库。</p> 
<p>此版本更新内容包括：</p> 
<p>Modal</p> 
<p>属性命名规范化<br> is-max => maximized<br> is-center => centered<br> can-move => draggable<br> Tree</p> 
<p>新增 TreeNode 组件，移除 rende-content 函数<br> 新增 show-line 属性，控制直线显示<br> 新增 show-icon 属性，控制图标显示<br> 新增 draggable 属性，是否可以拖拽<br> 新增 directory 属性，是否以目录形式展示<br> 新增 show-extra 属性，默认是否展示扩展属性<br> 新增 selectedKeys 属性，默认选中值<br> 新增 expandedKeys 属性，默认展开值<br> 新增 checkedKeys 属性，默认勾选值<br> 新增 dragstart，dragend， dragenter，dragleave，drop等函数<br> Select</p> 
<p>修复值为0，活值为空 时无法选中的问题<br> 优化展开卡顿的问题<br> 新增 bordered 属性，控制边框显示<br> 新增 show-arrow 属性，控制下拉箭头显示<br> 多选模式可以显示清除图标和下拉箭头<br> 完善动画细节， 完善多选tag节点问题<br> 完善禁用状态聚焦问题<br> DatePicker</p> 
<p>完善禁用状态聚焦问题<br> 新增 bordered 属性，控制边框显示<br> Form</p> 
<p>表单使用Grid 布局，使用labelCol,wrapperCol 代替 label-width<br> 增加原生函数：submit,reset 可以使用 Button 形态为 submit 和 reset 来提交和重置表单<br> 新增size 属性，统一控制表单内组件大小<br> 表单验证时，子组件无需 v-model 绑定，只指定相对应的prop 即可<br> Button</p> 
<p>属性buttontype 更名为 htmltype<br> Checkbox</p> 
<p>完善文字居中，完善默认选中的问题<br> Radio</p> 
<p>完善文字居中，完善默认选中的问题<br> Switch</p> 
<p>完善文字居中，完善默认选中的问题<br> Menu</p> 
<p>修复在特定场景展开延迟的问题<br> Tabs</p> 
<p>修复子组件变化无法更新的bug<br> 修复在 Modal 组件内的一些问题<br> Tooltip</p> 
<p>新增多彩标签，完善下拉箭头显示问题<br> Progress</p> 
<p>新增小尺寸和自定义宽度<br> Input</p> 
<p>新增 suffix 扩展</p> 
<p>详情查看：<a href="https://gitee.com/chuchur/kui-vue/releases/v3.2.0">https://gitee.com/chuchur/kui-vue/releases/v3.2.0</a></p>
