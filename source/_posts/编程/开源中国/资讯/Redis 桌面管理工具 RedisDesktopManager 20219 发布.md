                                                        <p>RedisDesktopManager 2021.9 现已发布。RedisDesktopManager 是一个快速、简单、支持跨平台的 Redis 桌面管理工具，基于 Qt 5 开发，支持通过 SSH Tunnel 连接。新版本主要变化如下：</p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><strong>New features</strong></p> 
<ul> 
 <li>添加对新压缩格式的支持： 
  <ul> 
   <li>ZSTD <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fuglide%2FRedisDesktopManager%2Fissues%2F5114" target="_blank">#5114</a></li> 
   <li><span style="background-color:#ffffff; color:#24292f">Snappy</span> <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fuglide%2FRedisDesktopManager%2Fissues%2F5117" target="_blank">#5117</a></li> 
   <li>Brotli</li> 
   <li>Magento 缓存和会话缓存 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fuglide%2FRedisDesktopManager%2Fissues%2F5028" target="_blank">#5028</a></li> 
   <li>PHP 压缩包</li> 
  </ul> </li> 
 <li>要求提供 SSH 密码 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fuglide%2FRedisDesktopManager%2Fissues%2F3955" target="_blank">#3955</a></li> 
 <li>在控制台中添加对<code>MONITOR</code>命令的支持</li> 
</ul> 
<p><strong>修复和改进</strong></p> 
<ul> 
 <li>加速初始 UI 渲染</li> 
 <li>加速加载大型二进制值</li> 
 <li>改进自动 JSON 检测</li> 
 <li>Streams：正确地将嵌入式对象和列表保存为字段值</li> 
 <li>Streams：从字段解析 JSON</li> 
 <li>Streams：修复过滤器布局</li> 
</ul>
