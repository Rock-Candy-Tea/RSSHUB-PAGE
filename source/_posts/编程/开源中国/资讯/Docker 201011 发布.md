                                                        <p>Docker 20.10.11 现已发布，具体更新内容如下：</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><strong>IMPORTANT</strong></p> 
<ul> 
 <li>由于<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgolang.org%2Fdoc%2Fgo1.16%23net%2Fhttp" target="_blank">Go 1.16</a><span> </span>中<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fgolang%2Fgo%2Fissues%2F40909" target="_blank">net/http</a><span> </span>的变化，通过 $HTTP_PROXY 环境变<span style="color:#000000">量配置的 HTTP 代理不再用于 TLS（https://）连接。用户需确保也设置一个 $HTTPS_PROXY 环境变量来处理对 https:// URL 的请求。可参阅</span> <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmoby%2Fmoby%2Fconfig%2Fdaemon%2Fsystemd.md%23httphttps-proxy" target="_blank">HTTP/HTTPS 代理部分</a>，以了解如何配置 Docker Daemon 以使用代理服务器。</li> 
</ul> 
<p><strong>Distribution</strong></p> 
<ul> 
 <li>处理模棱两可的 OCI 清单解析以缓解 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fcve.mitre.org%2Fcgi-bin%2Fcvename.cgi%3Fname%3DCVE-2021-41190" target="_blank">CVE-2021-41190</a>/<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fopencontainers%2Fdistribution-spec%2Fsecurity%2Fadvisories%2FGHSA-mc8v-mgrf-8f4m" target="_blank">GHSA-mc8v-mgrf-8f4m</a>。详见 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmoby%2Fmoby%2Fsecurity%2Fadvisories%2FGHSA-xmmx-7jpf-fx42" target="_blank">GHSA-xmmx-7jpf-fx42</a>。</li> 
</ul> 
<p><strong>Windows</strong></p> 
<ul> 
 <li>修复 panic.log 文件的只读属性设置问题 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmoby%2Fmoby%2Fpull%2F42987" target="_blank">moby/moby#42987</a></li> 
</ul> 
<p><strong>Packaging </strong></p> 
<ul> 
 <li>将 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fcontainerd%2Fcontainerd%2Freleases%2Ftag%2Fv1.4.12" target="_blank">containerd</a> 更新到 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fcontainerd%2Fcontainerd%2Freleases%2Ftag%2Fv1.4.12" target="_blank">v1.4.12</a> 以缓解 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fcve.mitre.org%2Fcgi-bin%2Fcvename.cgi%3Fname%3DCVE-2021-41190" target="_blank">CVE-2021-41190</a>。</li> 
 <li>将 Golang <span style="background-color:#ffffff; color:#24292f">runtime </span>更新到 Go 1.16.10。</li> 
</ul> 
<p>更新说明：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmoby%2Fmoby%2Freleases%2Ftag%2Fv20.10.11" target="_blank">https://github.com/moby/moby/releases/tag/v20.10.11</a> </p>
