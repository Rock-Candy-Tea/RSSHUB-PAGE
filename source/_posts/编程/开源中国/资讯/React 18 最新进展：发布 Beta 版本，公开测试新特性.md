                                                                                            <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">11月16日，<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Freactwg%2Freact-18" target="_blank">React 18 工作组</a><span> </span>宣布 React 18 从<span> </span><span style="color:#24292f">Alpha<span> </span></span>进入 Beta 阶段，Beta 是测试版本，<span style="color:#24292f">大部分工作都是对 Alpha 版本发布的新特性进行文档优化、功能测试和改进，在最终版本发布之前<strong>不会有任何额外的新特性或 API</strong></span>。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">关于 React 18<span> </span><span style="color:#24292f">Alpha 版本已发布</span>的新特性可以查看此新闻：<a href="https://www.oschina.net/news/145753/the-plan-for-react-18"><strong>React 18 最新进展：发布 alpha 版本、全新 SSR 架构</strong></a><strong><span> </span>。</strong></p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><span style="color:#24292f">Beta 版预计测试 1-2 个月，届时将推出 React 18 RC（候选）版本。</span>目前<span> </span><span style="color:#24292f">React  18</span><span> </span>正在<span style="color:#24292f">与 Redux、Next.js 和 React 测试库等关联库密切合作，以提供顺畅的升级路径，已知兼容 React 18 的库如下：</span></p> 
<h4 style="margin-left:.6em; margin-right:0; text-align:start"><strong>Next.js </strong></h4> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">Next.js 的最新版本支持 React 18 和并发特性：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fnextjs.org%2Fdocs%2Fadvanced-features%2Freact-18" target="_blank">https ://nextjs.org/docs/advanced-features/react-18</a></p> 
<h4 style="margin-left:.6em; margin-right:0; text-align:start">Gatsby </h4> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">Gatsby 从 v3.7 开始支持 React 18 和并发特性。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.gatsbyjs.com%2Fblog%2Fhow-to-try-react-18-in-gatsby%2F" target="_blank">https://www.gatsbyjs.com/blog/how-to-try-react-18-in-gatsby/</a></p> 
<h4 style="margin-left:.6em; margin-right:0; text-align:start"><span style="color:#24292f">React Redux </span></h4> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">React Redux v8 将支持 React 18。在内部使用新的<span> </span><code>useSyncExternalStore</code><span> </span>API 来确保与 React 18  并发特性的兼容性。</p> 
<h4 style="margin-left:.6em; margin-right:0; text-align:start"><strong>React Testing Library</strong></h4> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">React 测试库 v13 支持 React 18，会自动将测试切换到<span> </span><code>createRoot</code>，不需要作过多升级。</p>
