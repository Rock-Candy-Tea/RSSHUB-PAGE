<div class="content">
                                                                                            <p style="margin-left:0; margin-right:0">OpenCV 是 Intel 开源计算机视觉库，它实现了图像处理和计算机视觉方面的很多通用算法。OpenCV 4.6 版本的主要更新内容如下：</p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>OpenCV 项目基础设施为 CI 和发布目的迁移到 GitHub Actions 工作流</li> 
 <li>增加了对 GCC 12、Clang 15 的支持</li> 
 <li>增加了对 FFmpeg 5.0 的支持</li> 
 <li>DNN 模块的补丁： 
  <ul style="margin-left:0; margin-right:0"> 
   <li>修复了 Reduce、Slice、Expand 中的问题</li> 
   <li>更改了 ONNX 导入器中的层名，以正确支持 "output" 实体</li> 
   <li>增加了 TIM-VX NPU 后端支持</li> 
   <li>为 ClassificationModel 添加了 Softmax 参数</li> 
   <li>添加了音频语音识别样本（C++）</li> 
   <li>英特尔推理引擎后端（OpenVINO）： 
    <ul style="margin-left:0; margin-right:0"> 
     <li>增加了对 OpenVINO 2022.1 版本的初步支持</li> 
     <li>删除了对传统 API 的支持（自 2020.3 起放弃）</li> 
    </ul> </li> 
  </ul> </li> 
</ul> 
<p style="margin-left:0; margin-right:0">更多详情可查看：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fopencv%2Fopencv%2Fwiki%2FChangeLog" target="_blank">https://github.com/opencv/opencv/wiki/ChangeLog</a></p>
