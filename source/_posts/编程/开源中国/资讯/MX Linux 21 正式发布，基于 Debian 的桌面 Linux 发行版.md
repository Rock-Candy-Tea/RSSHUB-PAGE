                                                        <p>MX Linux 21 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fmxlinux.org%2Fblog%2Fmx-21-wildflower-released%2F" target="_blank">已正式发布</a>。</p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">主要变化如下：</p> 
<ul style="list-style-type:disc; margin-left:0; margin-right:0"> 
 <li>引入新的应用程序，以及升级已有应用程序</li> 
 <li>引入新的安装器分区选择区域，如果 lvm 卷已经存在，则提供 lvm 支持，以及在安装时设置挂载现有数据分区的功能</li> 
 <li>引入新的 UEFI live 系统启动菜单。现在，用户可以从引导菜单和子菜单中选择实时引导选项（持久状态等），而不是使用以前的控制台菜单</li> 
 <li>Xfce 4.16、Plasma 5.20、fluxbox 1.3.7 with mx-fluxbox 3.0 configs</li> 
 <li>执行 admin 任务默认要求使用用户密码 (sudo)，可以在 mx-tweak->Other 选项卡中进行切换</li> 
 <li>MX-Tour 支持显示每个桌面环境</li> 
 <li>默认情况下安装 Mesa vulkan 驱动程序</li> 
 <li>提供更好的 realtek wifi 支持</li> 
 <li>……</li> 
</ul> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">另外值得注意的是，官方希望主要版本号可遵循基于年份的规则，因此原本的 MX Linux 20 已被跳过，直接从 MX Linux 19 跳到了 MX Linux 21。</p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fmxlinux.org%2Fblog%2Fmx-21-wildflower-released%2F" target="_blank">下载地址</a></p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">MX Linux 是基于 Debian 稳定分支的面向桌面的 Linux 发行，它是 antiX 及早先的 MEPIS Linux 社区合作的产物。它采用 Xfce 作为默认桌面环境，是一份中量级操作系统，并被设计为优雅而高效的桌面与如下特性的结合：配置简单、高度稳定、性能可靠、占用空间不大也不小。</p>
