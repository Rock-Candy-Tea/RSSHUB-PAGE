                                                                                            <p>FreeRDP 2.4.0 版本现已<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.freerdp.com%2F2020%2F05%2F08%2F2_1_0-released" target="_blank">发布</a>。FreeRDP 是一个远程桌面协议（RDP，Remote Desktop Protocol）工具，用于从 Linux 下远程连接到 Windows 桌面。</p> 
<p>新版本更新内容具体如下：</p> 
<p>值得注意的变化：</p> 
<ul> 
 <li>向后移植的多线程渐进式解码器 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FFreeRDP%2FFreeRDP%2Fpull%2F7036" target="_blank">#7036</a> )</li> 
 <li>向后移植剪贴板修复 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FFreeRDP%2FFreeRDP%2Fpull%2F6924" target="_blank">#6924</a> )</li> 
 <li>修复了远程文件读取（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FFreeRDP%2FFreeRDP%2Fpull%2F7185" target="_blank">#7185</a>）</li> 
</ul> 
<p>Fixed issues：</p> 
<ul> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FFreeRDP%2FFreeRDP%2Fissues%2F6938" target="_blank">#6938</a>：RAILS clipboard remote -> local</li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FFreeRDP%2FFreeRDP%2Fissues%2F6985" target="_blank">#6985</a>：支持较新的 FFMPEG builds</li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FFreeRDP%2FFreeRDP%2Fissues%2F6989" target="_blank">#6989</a>：使用 OpenSSL 默认证书存储设置</li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FFreeRDP%2FFreeRDP%2Fpull%2F7073" target="_blank">#7073</a>：Planar alignment 修复</li> 
</ul>
