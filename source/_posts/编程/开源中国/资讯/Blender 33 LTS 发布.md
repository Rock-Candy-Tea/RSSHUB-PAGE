                                                                                            <p>Blender 是一个免费和开源的 3D 计算机图形软件工具集，用于创建动画电影、视觉效果、艺术、3D 打印模型、交互式 3D 应用、VR 和计算机游戏。</p> 
<p><strong>近日，Blender 3.3 正式发布，值得注意的是，Blender 3.3 是一个长期支持（LTS）版本，提供两年时间的修复和更新。</strong>该版本的更新内容如下：</p> 
<ul> 
 <li>对英特尔 oneAPI 后端的初步支持</li> 
 <li>AMD HIP 后端已经扩展到现在可以在 Windows 和 Linux 上使用 GFX9/Vega 硬件。从 Blender 3.2 开始，RDNA1 和 GFX9/Vega 的问题已经解决，以便让 Blender 获得更广泛的 AMD Radeon GPU 支持</li> 
 <li>改进了 macOS 上 Apple Metal 渲染 GPU 的性能</li> 
 <li>OpenVDB 卷的渲染精度为半浮点，而不是全浮点，从而大大降低了内存占用</li> 
 <li>在使用多个 NVIDIA GPU 进行渲染时，NVIDIA 的 OptiX 去噪器现在具有更好的性能</li> 
 <li>在启用 EEVEE 的情况下进行雕刻时，绘图性能得到了 "极大的改善"。</li> 
 <li>使用油性铅笔的 LineArt 的性能大大提升</li> 
 <li>改进了 Library Overrides UI/UX 的可用性</li> 
 <li>JPEG 预览缩略图创建得更快，同时也使用更少的内存。EXR 预览缩略图图像也应消耗更少的RAM</li> 
 <li>几何图形节点的各种性能改进</li> 
 <li>改进了导入具有大量对象的 USD/Alembic/OBJ 文件的性能</li> 
 <li>一个新的实验性 STL 导入器是用 C++ 编写的，比以前用 Python 编写的 STL 导入器快 8 倍左右</li> 
 <li>glTF 2.0 对导入/导出的处理改进</li> 
</ul>
