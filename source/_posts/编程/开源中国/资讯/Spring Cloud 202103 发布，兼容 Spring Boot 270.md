                                                        <p>Spring Cloud 2021.0.3<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fspring.io%2Fblog%2F2022%2F05%2F27%2Fspring-cloud-2021-0-3-is-available-compatible-with-spring-boot-2-7-0" target="_blank"> 已发布</a>，此版本主要是为了兼容<span style="color:#333333"><span> </span>Spring Boot 2.7.0 和 Spring Boot 2.6.x。</span></p> 
<p><strong>主要变化</strong></p> 
<p style="text-align:start"><strong>Spring Cloud Sleuth</strong></p> 
<ul> 
 <li>修复 bug 和功能增强</li> 
</ul> 
<p style="text-align:start"><strong>Spring Cloud Contract</strong></p> 
<ul> 
 <li>修复 bug 和升级 JsonAssert</li> 
</ul> 
<p style="text-align:start"><strong>Spring Cloud Gateway</strong></p> 
<ul> 
 <li>修复 bug，以及为<code>PrefixPath</code><span> </span>过滤器添加路径变量支持</li> 
</ul> 
<p><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-cloud%2Fspring-cloud-release%2Fwiki%2FSpring-Cloud-2021.0-Release-Notes" target="_blank">详情</a>。</p>
