                                                        <p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><span style="background-color:#ffffff; color:#27282c">PhpStorm 2021.3 Beta 现已发布，</span><span style="background-color:#ffffff; color:#333333">用户可以从 IDE 内部、通过 Toolbox 应用程序或使用 Ubuntu 的 snaps 进行升级，也可以从</span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fblog.jetbrains.com%2Fphpstorm%2F2021%2F11%2Fphpstorm-2021-3-beta%2F" target="_blank">官方网站</a><span style="background-color:#ffffff; color:#333333">下载。</span></p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><span style="background-color:#ffffff; color:#333333"><img alt height="800" src="https://oscimg.oschina.net/oscnet/up-ad89bb4674c17bc5cf0a6d0a26e900492de.png" width="1600" referrerpolicy="no-referrer"></span></p> 
<p><strong>主要更新内容</strong></p> 
<ul> 
 <li>完全支持 PHP 8.1 
  <ul> 
   <li>只读属性</li> 
   <li>纯交集类型</li> 
   <li>新的返回类型："never"</li> 
   <li>Final 常数</li> 
   <li>初始值设定项中的 "new" 运算符</li> 
   <li>一流的可调用对象</li> 
  </ul> </li> 
 <li>PHP 中的泛型 
  <ul> 
   <li>改进了对 Doctrine Collections 的支持</li> 
   <li>支持未来的 Laravel 9 集合</li> 
   <li>从构造函数推断泛型类型</li> 
   <li>支持模板的 phpstan-* 注释</li> 
  </ul> </li> 
 <li>改进重构 
  <ul> 
   <li>内联函数或方法</li> 
   <li>内联变量</li> 
   <li>引入（提取）变量</li> 
   <li>对移动类重构的拖放支持</li> 
  </ul> </li> 
 <li>更快更好的部署 
  <ul> 
   <li>支持 "rsync"</li> 
   <li>修复重试上传失败</li> 
   <li>使用多线程连接加速 SFTP/FTP/FTPS</li> 
   <li>上传所有打开的文件</li> 
   <li>快速切换项目默认服务器</li> 
   <li>SSH 连接的代理支持</li> 
  </ul> </li> 
 <li>HTTP 客户端 
  <ul> 
   <li>支持文本流和 JSON 流</li> 
   <li>二进制响应</li> 
   <li>输出重定向到自定义文件或目录</li> 
   <li>支持 HTTP 请求标识符</li> 
  </ul> </li> 
 <li>用户界面更新 
  <ul> 
   <li>新书签工具窗口</li> 
   <li>Show Usages 的源代码预览</li> 
   <li>能够拆分运行工具窗口</li> 
  </ul> </li> 
</ul> 
<p>详情请查看<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fblog.jetbrains.com%2Fphpstorm%2F2021%2F11%2Fphpstorm-2021-3-beta%2F" target="_blank">更新公告</a>。</p>
