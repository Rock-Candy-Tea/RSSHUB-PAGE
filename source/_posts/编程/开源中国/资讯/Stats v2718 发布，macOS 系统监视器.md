<div class="content">
                                                                                            <p><span style="color:#000000">Stats 是一款开源的 macOS 系统监控工具，可以驻留在菜单栏中对 CPU、GPU、内存、磁盘和网络等进行实时监测。目前 Stats </span>v2.7.18<span style="color:#000000"> 发布了 ，此版本带来如下变更：</span></p> 
<h2><strong>已关闭的问题</strong></h2> 
<ul> 
 <li>功能：省略超过 100A 的电流传感器（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fexelban%2Fstats%2Fissues%2F943" target="_blank">#943</a>）</li> 
 <li>功能：PieChartView 中的固定值舍入 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fexelban%2Fstats%2Fissues%2F933" target="_blank">#933</a> )</li> 
 <li>功能：为折线图工具提示添加了自定义后缀（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fexelban%2Fstats%2Fissues%2F946" target="_blank">#946</a>）</li> 
 <li>修复：从频率值中删除 GHz ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fexelban%2Fstats%2Fissues%2F945" target="_blank">#945</a> )</li> 
 <li>功能：在网络模块中添加了额外的 wifi 信息（标准、安全、通道）（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fexelban%2Fstats%2Fissues%2F941" target="_blank">#941</a>）</li> 
 <li>功能：在 GPU 图表工具提示中显示当前温度单位（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fexelban%2Fstats%2Fissues%2F947" target="_blank">#947</a>）</li> 
</ul> 
<h2><strong>错误修复和新功能</strong></h2> 
<ul> 
 <li>功能：点击折线图时使用阴影点数组，它可以防止在通过单击预览图表时丢失数据</li> 
 <li>功能：将 swiftlint file_length 规则从 600 增加到 800</li> 
 <li>修复：返回温度传感器的当前温度单位</li> 
</ul> 
<h2>本地化</h2> 
<ul> 
 <li>功能：更新了荷兰语翻译（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fexelban%2Fstats%2Fpull%2F934" target="_blank">#934</a>）</li> 
 <li>功能：更新了匈牙利语翻译（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fexelban%2Fstats%2Fpull%2F935" target="_blank">#935</a>）</li> 
 <li>功能：添加斯洛文尼亚语翻译（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fexelban%2Fstats%2Fpull%2F940" target="_blank">#940</a>）</li> 
</ul>
