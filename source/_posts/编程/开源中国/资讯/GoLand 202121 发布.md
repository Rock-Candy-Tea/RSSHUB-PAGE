                                                                                            <p>GoLand 2021.2.1 现已发布，该版本主要是一个 bug 修复版本。</p> 
<p><strong>主要更新内容</strong></p> 
<ul> 
 <li>在需要 “go mod tidy” 的情况下，同步依赖项快速修复可能会运行 “go mod vendor”</li> 
 <li>为模糊测试添加代码完成</li> 
 <li>Export/Import Settings 不适用于 vgo 设置</li> 
 <li>文件|管理 IDE 设置 |导出设置弹出窗口将 Go SDK List 显示为 GoSdkList</li> 
 <li>将 Homebrew 符号链接 (/usr/local/opt/go) 添加到 Go SDK 的预定义搜索目录</li> 
 <li>Support 针对 Go 的红绿视觉缺陷调整颜色设置</li> 
 <li>asm：U+00B7（中点）应被视为有效的标识符字符</li> 
 <li>'button.install' 在 IdeBundle 中找不到</li> 
 <li>Update Running Application 按钮/菜单项不见了</li> 
 <li>扩展数组时内联评估不会增加大小</li> 
 <li>Freeze 从 WSL2 打开项目</li> 
 <li>重新分配的文件类型通配符在 IDE 重启时丢失</li> 
 <li>“IntelliJ IDEA 加载环境失败” 实际上并未出现在日志中</li> 
 <li>当用户单击登录对话框中的 “复制链接” 时复制 “空”</li> 
 <li>无法在 EAP 中获取授权令牌</li> 
 <li>导航到 NEXT 方法的快捷方式消失了。</li> 
 <li>插件管理器不显示插件图标并抛出 org.apache.batik.bridge.BridgeException</li> 
 <li>将动态插件更新到明确标记为非动态的版本会导致插件被删除</li> 
 <li>选择目录或包并按 ⌃⇧R  运行所有单元测试而不是目录/包中的测试</li> 
</ul>
