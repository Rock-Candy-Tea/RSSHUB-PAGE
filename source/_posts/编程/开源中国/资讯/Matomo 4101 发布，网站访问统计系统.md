                                                        <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">Matomo 是一套基于 PHP + MySQL 技术构建的开源网站访问统计系统，能够提供详细的统计信息，比如网页浏览人数、访问最多的页面、搜索引擎关键词等等流量分析功能。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">Matomo 4.10.1 已正式发布，该版本对 Matomo 4.10.0 版本中发现的一些回归问题进行修复，主要与正在进行的 VueJS 迁移工作和由于在 apache 中提供静态文件的安全性加强导致的回归有关。具体更新内容如下：</p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>使用分段时在 KPI 小部件中正确显示火花线<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmatomo-org%2Fmatomo%2Fissues%2F19209" target="_blank">#19209</a></li> 
 <li>修复 manifest.json 发送 HTTP 状态 403<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmatomo-org%2Fmatomo%2Fissues%2F19251" target="_blank">#19251</a></li> 
 <li>将活动构建器组件中的默认 hasExtraPlugin 设置为 true<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmatomo-org%2Fmatomo%2Fpull%2F19241" target="_blank">#19241</a></li> 
 <li>在段编辑器中提供 CustomDimensions<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmatomo-org%2Fmatomo%2Fissues%2F19237" target="_blank">#19237</a></li> 
 <li>将 FieldNumber 中的默认 null/falsey 值设置为 0 ，以匹配以前的行为<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmatomo-org%2Fmatomo%2Fpull%2F19262" target="_blank">#19262</a></li> 
</ul>
