                                                                                            <p>Firefox 103 浏览器已于今天正式发布，这是一个新的稳定版本，带来了许多性能上的改进和其他方面的提升。</p> 
<h3>新功能：</h3> 
<ul> 
 <li>通过切换到现代的 lock API，改进了在 CPU 高负荷下 macOS 的响应速度</li> 
 <li>必填字段现在在 PDF 表格中被突出显示</li> 
 <li>改进了在高刷新率显示器（120Hz 或更高）上的性能</li> 
 <li>画中画功能变得更好了：用户现在可以在画中画窗口直接改变字幕字体大小，此外画中画字幕现在可以在 Funimation、Dailymotion、Tubi、Hotstar 和 SonyLIV 上使用。</li> 
 <li>Tabs 工具栏中的按钮现在可以用 Tab、Shift+Tab 和方向键选择</li> 
 <li>Windows 的 "使文本变大" 可访问性设置现在影响到所有的用户界面和内容页，而不是只适用于系统字体大小</li> 
 <li>你现在可以方便地使用 Firefox，它现在将在 Windows 10 和 11 的安装过程中被固定在 Windows 任务栏上</li> 
</ul> 
<h3>修复</h3> 
<ul> 
 <li>当从表单控件中复制文本时，现在保留了不间断空格，防止自动换行</li> 
 <li>修复了 Linux 上通过 DMA-Buf 使用 NVIDIA 二进制驱动程序时的 WebGL 性能问题</li> 
 <li>修复了 Firefox 启动可能会因处理 Web 内容本地存储而大幅减慢的问题</li> 
 <li>各种安全修复</li> 
</ul> 
<h3>变化</h3> 
<ul> 
 <li>删除了允许在证书中使用 SHA-1 签名的配置选项：证书中的 SHA-1 签名早已确定不再安全</li> 
</ul> 
<h3>Web 平台</h3> 
<ul> 
 <li>您的信息现在通过默认启用的 Total Cookie Protection 增强了对在线跟踪的保护。所有第三方 cookie 现在都被隔离到分区存储中。</li> 
</ul>
