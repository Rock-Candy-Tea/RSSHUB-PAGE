                                                        <p>DBeaver 21.2.3 现已发布。<span style="background-color:#ffffff; color:#333333">DBeaver是一个免费开源的通用数据库工具，适用于开发人员和数据库管理员。</span></p> 
<p><strong>主要更新内容</strong></p> 
<ul> 
 <li>SQL editor 
  <ul> 
   <li>修复了自动完成的模式检测</li> 
   <li>对带有变量的查询进行了 Intelli-sense 建议检测</li> 
  </ul> </li> 
 <li>Data editor 
  <ul> 
   <li>改进了对深色主题的支持</li> 
   <li>修复了布尔运算编辑器的渲染</li> 
  </ul> </li> 
 <li>Data transfer 
  <ul> 
   <li>数据导出现在按照列的顺序和可见性</li> 
   <li>导出的文件在操作系统文件资源管理器中被突出显示</li> 
   <li>增加了 Oracle 和 PostgreSQL 之间的 XML 导出/导入支持</li> 
   <li>修复了 CSV 解析器（意外的输入文件读取终止）</li> 
   <li>CSV 导出格式：添加了列头格式配置</li> 
   <li>CSV 导出：修复了NULL值的引用问题</li> 
   <li>从 XML 导入：改进了列的映射</li> 
   <li>为目标文件名模式引入了额外的变量</li> 
   <li>修复了多余的表名引号</li> 
   <li>修复了输出文件分割选项（有额外的空文件问题）</li> 
   <li>XLSX 格式：列值现在被修剪为 32kb</li> 
  </ul> </li> 
 <li>Navigator 
  <ul> 
   <li>添加了从 JDBC URL 命令创建新连接</li> 
   <li>数据库元数据刷新已修复</li> 
  </ul> </li> 
 <li><span style="background-color:#ffffff; color:#555555">ERD</span> 
  <ul> 
   <li><span style="background-color:#ffffff; color:#555555">调色板图标已更新</span></li> 
   <li><span style="background-color:#ffffff; color:#555555">改进了深色主题支持</span></li> 
  </ul> </li> 
 <li><span style="background-color:#ffffff; color:#555555">PostgreSQL</span> 
  <ul> 
   <li><span style="background-color:#ffffff; color:#555555">​​​​​​​复合数据类型支持得到改进</span></li> 
   <li><span style="background-color:#ffffff; color:#555555">数据库备份向导 UI 已修复</span></li> 
  </ul> </li> 
</ul>
