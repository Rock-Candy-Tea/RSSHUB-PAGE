                                                                                            <p><span style="color:#000000">Spring Data 2021.2.1 和 2021.1.5 现已<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fspring.io%2Fblog%2F2022%2F06%2F20%2Fspring-data-2021-2-1-and-2021-1-5-released" target="_blank">发布</a>，这两个版本都附带了针对大多数 bug fixes 和依赖项升级的修复。</span></p>

<ul> 
 <li><strong><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fspring.io%2Fblog%2F2022%2F06%2F20%2Fspring-data-mongodb-spel-expression-injection-vulnerability-cve-2022-22980" target="_blank">CVE-2022-22980</a></strong><br> <span style="color:#000000">“Spring Data MongoDB SpEL Expression Injection Vulnerability”<br> MongoDB 应用程序中的 SpEL 注入攻击，通过使用带有未过滤输入的参数化 SpEL 语句使用 @Query 或 @Aggregation 注释的存储库查询方法。<br> 严重性：</span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.first.org%2Fcvss%2Fcalculator%2F3.1%23CVSS%3A3.1%2FAV%3AN%2FAC%3AH%2FPR%3AN%2FUI%3AN%2FS%3AC%2FC%3AH%2FI%3AH%2FA%3AH%2FE%3AP%2FRL%3AO%2FRC%3AC%2FCR%3AH%2FIR%3AH%2FAR%3AH%2FMAV%3AN%2FMAC%3AH%2FMPR%3AN%2FMUI%3AN%2FMS%3AC%2FMC%3AH%2FMI%3AH%2FMA%3AH" target="_blank">High</a></li> 
</ul>
