                                                        <p>SourceGit v4.0 已经发布，开源的 Git 客户端</p> 
<p>此版本更新内容包括：</p> 
<p>[CHANGELOG]</p> 
<ol> 
 <li>优化LFS文件判断，如果仓库未启用LFS，不再检测</li> 
 <li>修改OutputType为WinExe</li> 
 <li>修复版本切换分页导致版本文件列表清空的BUG</li> 
 <li>修复外部合并工具不能自动找到时，无法保存自定义可执行文件路径的BUG</li> 
 <li>修复无本地分支时，Pull面板打开Crash的BUG</li> 
</ol> 
<p>详情查看：<a href="https://gitee.com/sourcegit/SourceGit/releases/v4.0">https://gitee.com/sourcegit/SourceGit/releases/v4.0</a></p>
