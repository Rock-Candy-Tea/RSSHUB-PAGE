                                                                                            <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">Universal Media Server 是一个兼容 DLNA，UPnP 和 HTTP/S 的媒体服务器，支持所有主要操作系统，包括 Windows，Linux 和 macOS 版本，几乎不需要配置就可以流式传输或转码许多不同的媒体格式。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">在更新至 V11 之前，团队正在努力提高 V10 版本的稳定性和性能。这个版本改进了文件名预设/解析、浏览和扫描速度、视频元数据的准确性、API 查询速度和稳定性、转码的稳定性等等。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">自 10.19.0 以来的变化：</p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>常规： 
  <ul style="margin-left:0; margin-right:0"> 
   <li>增加了通过 MusicBrainz 更新音频 ID3 标签的能力，默认情况下是禁用的</li> 
   <li>增加了通过 API 评价音乐的功能</li> 
   <li>在媒体库中增加了 "My Albums" 文件夹</li> 
   <li>使用 UPnP 搜索时，增加了对 UPnP 排序请求的支持</li> 
   <li>增加了 dc:date UPnP 属性，供渲染器选择使用</li> 
   <li>允许翻译 API 字符串</li> 
   <li>更新了默认的播客和 YouTube 频道</li> 
   <li>修复了根目录和网络文件夹浏览缓慢的问题</li> 
   <li>提高了启动扫描时的响应速度</li> 
  </ul> </li> 
 <li>通过 Crowdin 的翻译更新： 
  <ul style="margin-left:0; margin-right:0"> 
   <li>更新了包括繁体中文、日语、韩语、法语、西班牙语等多个国家和地区的语言翻译；</li> 
  </ul> </li> 
 <li>依赖项： 
  <ul style="margin-left:0; margin-right:0"> 
   <li>将 jna 从 5.10.0 升级到 5.11.0</li> 
   <li>将 junrar 从 7.4.1 升级到 7.5.0</li> 
   <li>将 metadata-extractor 从 2.16.0 升级到 2.17.0</li> 
   <li>将 oshi-core 从 6.1.4 升级到 6.1.5</li> 
   <li>将 spotbugs-maven-plugin 从 4.5.3.0 升级到 4.6.0.0</li> 
  </ul> </li> 
</ul> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">更多详情可查看：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FUniversalMediaServer%2FUniversalMediaServer%2Freleases" target="_blank">https://github.com/UniversalMediaServer/UniversalMediaServer/releases</a></p>
