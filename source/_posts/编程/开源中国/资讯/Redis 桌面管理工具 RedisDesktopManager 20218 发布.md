                                                                                            <p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><span style="color:#333333">RedisDesktopManager 2021.8 现已发布。RedisDesktopManager 是一个快速、简单、支持跨平台的 Redis 桌面管理工具，基于 Qt 5 开发，支持通过 SSH Tunnel 连接。新版本主要变化如下：</span></p> 
<ul style="margin-left:0; margin-right:0"> 
 <li><span style="color:#2e3033">修复服务器信息的图表布局。</span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fuglide%2FRedisDesktopManager%2Fissues%2F5099" target="_blank">#5099</a></li> 
 <li><span style="color:#24292f">在<span> </span></span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fflathub.org%2Fapps%2Fdetails%2Fdev.rdm.RDM" target="_blank">Flathub</a>上发布了 MDN 。</li> 
 <li>限制树视图中子项的最大数量，以支持更大的开箱即用的键空间。</li> 
 <li>改进树视图中的实时更新。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fuglide%2FRedisDesktopManager%2Fissues%2F3994" target="_blank">#3994</a></li> 
 <li>删除 SET 键中错误的值。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fuglide%2FRedisDesktopManager%2Fissues%2F5108" target="_blank">#5108</a> </li> 
 <li>修复 JSON 格式错误。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fuglide%2FRedisDesktopManager%2Fissues%2F5102" target="_blank">#5102</a></li> 
 <li>在控制台中为服务器响应添加文本换行。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fuglide%2FRedisDesktopManager%2Fissues%2F5100" target="_blank">#5100</a></li> 
 <li>使用<span style="color:#2e3033">最后选定的值格式化程序</span>作为后备。</li> 
 <li>允许在树视图中单击展开折叠的项目。</li> 
 <li>允许创建 ReJSON 密钥。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fuglide%2FRedisDesktopManager%2Fissues%2F5010" target="_blank">#5010</a></li> 
 <li>修复 ReJSON 编码问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fuglide%2FRedisDesktopManager%2Fissues%2F5106" target="_blank">#5106</a></li> 
 <li>修复连接树的调色板使用。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fuglide%2FRedisDesktopManager%2Fissues%2F5025" target="_blank">#5025</a></li> 
 <li>修复 ibus 无法输入中文文本的 snap 错误。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fuglide%2FRedisDesktopManager%2Fissues%2F5023" target="_blank">#5023</a></li> 
 <li>用可配置的范围替换硬编码的字体大小</li> 
</ul> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">更新公告：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fuglide%2FRedisDesktopManager%2Freleases" target="_blank">https://github.com/uglide/RedisDesktopManager/releases</a></p>
