                                                                                            <p>Dlink 0.5.0 已经发布，交互式的 FlinkSQL Studio .</p> 
<p><strong>此版本更新内容包括：</strong></p> 
<ol> 
 <li>支持 Kubernetes Session 和 Application 模式提交任务</li> 
 <li>新增 UDF Java方言的Local模式的在线编写、调试、动态加载</li> 
 <li>新增 FlinkSQL 执行环境方言及其应用功能</li> 
 <li>新增 BI选项卡的折线图、条形图、饼图</li> 
 <li>新增 元数据查看表和字段信息</li> 
 <li>新增 ChangLog 和 Table 的查询及自动停止实现</li> 
 <li>新增 Mysql,Oracle,PostGreSql,ClickHouse,Doris,Java 方言</li> 
 <li>新增 OpenAPI 的执行sql、校验sql、获取计划图、获取StreamGraph、获取预览数据、执行Jar、停止、SavePoint接口</li> 
 <li>新增 数据源的 Sql 作业语法校验和语句执行</li> 
 <li>新增 快捷键保存、校验、美化等</li> 
 <li>新增 引导页</li> 
 <li>新增 JobPlanGraph 展示</li> 
 <li>新增 SQLServer Jdbc Connector 的实现</li> 
 <li>新增 Local 的运行模式选择与分类</li> 
 <li>新增 SavePoint 的 restAPI 实现</li> 
 <li>新增 编辑器选项卡右键关闭其他和关闭所有</li> 
 <li>新增 FlinkSQL 及 SQL 导出</li> 
 <li>新增 集群与数据源的 Studio 管理交互</li> 
 <li>新增 Yarn 的 Kerboros 验证</li> 
 <li>建立 官网文档</li> 
 <li>修改 项目名为 Dinky 以及图标</li> 
 <li>优化 所有模式的所有功能的执行逻辑</li> 
 <li>升级 各版本 Flink 依赖至最新版本以解决核弹问题</li> 
 <li>修复 编辑集群配置测试后保存会新建的bug</li> 
 <li>修复 登录页报错弹框</li> 
 <li>修复 Yarn Application 解析数组异常问题</li> 
 <li>修复 自定义Jar配置为空会导致异常的bug</li> 
 <li>修复 任务提交失败时注册集群报错的bug</li> 
 <li>修复 set在perjob和application模式不生效的问题</li> 
 <li>修复 perjob和application模式的任务名无法自定义的问题</li> 
 <li>修复 set 语法在1.11和1.12的兼容问题</li> 
 <li>修复 血缘分析图由于前端依赖无法正常加载的问题</li> 
</ol> 
<p>详情查看：<a href="https://gitee.com/DataLinkDC/dlink/releases/0.5.0">https://gitee.com/DataLinkDC/dlink/releases/0.5.0</a></p>
