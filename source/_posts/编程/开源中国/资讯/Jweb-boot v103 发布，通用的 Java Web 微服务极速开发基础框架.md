                                                        <p style="text-align:start">Jweb-boot 是基于<a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fjfinal.com%2F" target="_blank">JFinal</a>+<a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fjboot.io%2F" target="_blank">Jboot</a>+undertow进行开发增强而来的极速开发基础框架，无它！</p> 
<h2 style="text-align:start">愿景</h2> 
<ol> 
 <li><strong>瑞士军刀</strong><br> 成为web开发的一把瑞士军刀。统一框架自由组合，以适应不同的系统结构。让不管是单体系统还是分布式微服务平台，配置调一调就可以快速组建完成！</li> 
 <li><strong>看的懂，改的动</strong><br> 必竟程序员都有洁癖，普遍认为别人写的代码都是垃圾，不改一改太难用了，至少我是这么认为的。希望本人写的代码大家能看的懂，改的动！</li> 
</ol> 
<h2 style="text-align:start">特点</h2> 
<p style="text-align:start">在<a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fjboot.io%2F" target="_blank">Jboot</a>的基础上开发，所有继承了<a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fjboot.io%2F" target="_blank">Jboot</a>+<a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fjfinal.com%2F" target="_blank">JFinal</a>的全部特点，并在此基础上进行增强。</p> 
<h3 style="text-align:start"><a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fjboot.io%2F" target="_blank">Jboot </a>特点</h3> 
<ol> 
 <li>基于 <a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fjfinal.com%2F" target="_blank">JFinal</a> 的 MVC + ORM 快速开发。</li> 
 <li>基于 ShardingSphere + Seata 分布式事务 和 分库分表。</li> 
 <li>基于 Dubbo 或 Motan 的 RPC 实现</li> 
 <li>基于 Sentinel 的分布式限流和降级</li> 
 <li>基于 Apollo 和 Nacos 的分布式配置中心</li> 
 <li>基于 EhCache 和 Redis 的分布式二级缓存</li> 
</ol> 
<h3 style="text-align:start">Jweb-boot特点</h3> 
<ol> 
 <li>提供jweb-discovery模块，支撑网关服务代理，提供各服务间直接发现与调用</li> 
 <li>提供jweb-security安全框架，借鉴于shiro，简化用户认证权限授权系统开发。</li> 
 <li>增强<a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fjboot.io%2F" target="_blank">Jboot </a>网关功能，适配jweb-discovery服务发现功能</li> 
 <li>增强Nacos分布式配置功能</li> 
 <li>增强jboot资源监控，资源查找功能</li> 
</ol>
