                                                        <p>Taro 3.2.8 发布了。Taro 是一个开放式跨端跨框架解决方案，支持使用 React/Vue/Nerv 等框架来开发微信/京东/百度/支付宝/字节跳动/ QQ 小程序/H5 等应用。此版本更新内容包括：</p> 
<h4>特性</h4> 
<p><strong>小程序</strong></p> 
<ul> 
 <li>为 Vue/Vue3 增加 <code>custom-wrapper</code> 组件的支持，<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FNervJS%2Ftaro%2Fissues%2F8989" target="_blank">#8989</a></li> 
</ul> 
<h4>修复</h4> 
<p><strong>小程序</strong></p> 
<ul> 
 <li>修复百度小程序 <code>PickerView</code> 不显示的问题，<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FNervJS%2Ftaro%2Fissues%2F9239" target="_blank">#9239</a></li> 
 <li>修复获取不到 <code>Taro.preload</code> 预加载数据的问题，<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FNervJS%2Ftaro%2Fissues%2F9175" target="_blank">#9175</a></li> 
</ul> 
<p><strong>H5</strong></p> 
<ul> 
 <li>修复 <code>createSelectorQuery</code> API 获取 <code>style</code> 错误的问题</li> 
 <li><code>previewImage</code> 改为异步加载图片，优化体验</li> 
</ul> 
<p><strong>Typings</strong></p> 
<ul> 
 <li>修复 QQ 小程序平台的类型问题，<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FNervJS%2Ftaro%2Fissues%2F9248" target="_blank">#9248</a></li> 
</ul> 
<p>更新说明：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FNervJS%2Ftaro%2Freleases%2Ftag%2Fv3.2.8" target="_blank">https://github.com/NervJS/taro/releases/tag/v3.2.8</a></p>
