                                                        <p>Ant Design 4.16.1 发布了。Ant Design 是一套企业级的 UI 设计语言和 React 实现，使用 TypeScript 构建，提供完整的类型定义文件，自带提炼自企业级中后台产品的交互语言和视觉风格、开箱即用的高质量 React 组件与全链路开发和设计工具体系。</p> 
<p>此版本更新内容如下：</p> 
<ul> 
 <li>Menu 
  <ul> 
   <li>修复超长省略不能按照预期工作的问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F30672" target="_blank">#30672</a></li> 
   <li>修复在卸载时报 <code>Can't perform a React state update on an unmounted component</code> 的警告信息。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F30678" target="_blank">#30678</a></li> 
   <li>修复 <code>inline</code> 模式下，没有 <code>icon</code> 的条目不能正确处理省略样式的问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F30682" target="_blank">#30682</a></li> 
  </ul> </li> 
 <li>Typography 
  <ul> 
   <li>修复 Typography.Text 在 css 省略可用时监听 ResizeObserver 的无用操作。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F30664" target="_blank">#30664</a></li> 
   <li>修复 <code>ellipsis</code> 在开发环境 HMR 生效后不再工作的问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F30754" target="_blank">#30754</a></li> 
  </ul> </li> 
 <li>Table 
  <ul> 
   <li>修复筛选/排序列的 <code>align</code> 属性。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F30744" target="_blank">#30744</a></li> 
   <li>修复 Table.Summary <code>sticky</code> 模式下，配置 <code>bordered</code> 边框样式丢失的问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F30666" target="_blank">#30666</a></li> 
  </ul> </li> 
 <li>修复 Collapse <code>expandIconPosition="right"</code> 未生效的问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F30680" target="_blank">#30680</a></li> 
 <li>TypeScript 
  <ul> 
   <li>导出 TableColumnsType 类型。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F30747" target="_blank">#30747</a></li> 
   <li>导出 TreeDataNode 类型。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F30745" target="_blank">#30745</a></li> 
  </ul> </li> 
</ul>
