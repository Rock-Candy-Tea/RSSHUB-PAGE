                                                                                            <p><span style="color:#000000">Wekan 5.95 发布了，Wekan 是模仿 Trello 且与它最接近的开源 kanban，它的前后端都是使用 JavaScript 编写，基于 Meteor 框架。</span></p> 
<p><span style="color:#000000">Wekan 简化界面、支持中文、拒绝其它花哨的功能，支持项目管理所需的标签、注释、添加成员等功能，而且它还有一个免费托管的 demo 站点，除了可以下载安装使用，还可以将 Wekan 通过 Dockerfile 镜像在一个容器里安全运行。</span></p> 
<p><span style="color:#000000">Wekan 5.95 </span><span style="color:#24292f">版本<strong>增加了以下新功能</strong>：</span></p> 
<ul> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fwekan%2Fwekan%2Fpull%2F4296" target="_blank">移动复制卡会记住最后确认的字段值</a></li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fwekan%2Fwekan%2Fpull%2F4297" target="_blank">添加了“移动清单”并创建了“清单操作菜单”</a></li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fwekan%2Fwekan%2Fpull%2F4298" target="_blank">添加清单项目 - 每行都成为新项目</a></li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fwekan%2Fwekan%2Fpull%2F4299" target="_blank">将进度条添加到清单</a></li> 
</ul> 
<p><strong>添加以下更新：</strong></p> 
<ul> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fwekan%2Fwekan%2Fpull%2F4295" target="_blank">更新 ISSUE_TEMPLATE.md：使用评论和改写</a>。</li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fwekan%2Fwekan%2Fcommit%2Fd4c3f323830e4692aac0f1d0faa413302087303b" target="_blank">更新了 WeKan s390x 捆绑构建脚本</a></li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fwekan%2Fwekan%2Fcommit%2Fa176a58a728ffc2bb410b3e8a0036fee25f0b0bd" target="_blank">使用不同的发行版构建 WeKan 版本</a></li> 
</ul> 
<p><strong>修复了以下错误：</strong></p> 
<ul> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fwekan%2Fwekan%2Fpull%2F4300" target="_blank">自定义字段字符串模板在按下 Ctrl+Enter 后未在桌面视图中保存最后一个输入值</a></li> 
</ul>
