<h1 style="margin-left:8px; margin-right:8px; text-align:center"><span style="color:#ffa900"><strong><strong><span style="color:#135200"><strong style="color:#24292e"><span style="color:#11510c"><span> </span>引言</span></strong></span></strong></strong></span></h1> 
<p>龙蜥社区的 Anolis OS 8 系列增加新成员啦！龙蜥操作系统（Anolis OS）8 是龙蜥社区（ OpenAnolis ）发行的开源 Linux 发行版，支持多计算架构，提供稳定、高性能、安全、可靠的操作系统支持。</p> 
<p style="margin-left:0; margin-right:0">Anolis OS 8.6 是 Anolis OS 8 系列发布的的第三个小版本！新版本的特性、开发节奏和发布内容是如何的呢？本文会给大家做一个相应的介绍。</p> 
<h1 style="margin-left:8px; margin-right:8px; text-align:center"><span style="color:#ffa900"><strong><strong><span style="color:#135200"><strong style="color:#24292e"><span style="color:#11510c"> </span></strong></span></strong></strong></span><strong style="color:#24292e"><span style="color:#11510c">开发时间</span></strong></h1> 
<h1 style="margin-left:8px; margin-right:8px; text-align:center"><img alt src="https://oscimg.oschina.net/oscnet/up-b14161117bb3dee88ba5410e4cb2df8cc74.png" referrerpolicy="no-referrer"></h1> 
<p style="color:#24292e; margin-left:0; margin-right:0; text-align:center">（图/发布节奏排期）</p> 
<p style="color:#24292e; margin-left:0; margin-right:0; text-align:justify">Anolis OS 8.6 经历了需求冻结、BETA 版本、RC 版本、GA 版本及正式发布阶段。在 5 月 23 日完成  BETA  版本的首发后，RC、GA 版本还进行了产业级别的功能、性能、稳定性、国产化平台等验证覆盖。</p> 
<h1 style="margin-left:8px; margin-right:8px; text-align:center"><span style="color:#ffa900"><strong><strong><span style="color:#135200"><strong style="color:#24292e"><span style="color:#11510c"> </span></strong></span></strong></strong></span><strong style="color:#24292e"><span style="color:#11510c">发布内容</span></strong></h1> 
<p style="color:#24292e; margin-left:0; margin-right:0">龙蜥操作系统(Anolis OS)  8.6 发布内容包括<strong> </strong><span style="color:#11510c"><strong>ISO、虚拟机镜像和容器镜像</strong></span><span style="color:#888888">。</span></p>

<h2><strong><span style="color:#11510c">6、</span><span style="color:#11510c">自研变更日志（ChangeLog）</span></strong></h2> 
<h3 style="margin-left:0px; margin-right:0px; text-align:justify"><span style="color:#888888">6.1 内核</span></h3> 
<ul style="list-style-type:disc; margin-left:0; margin-right:0"> 
 <li> <p>Linux Kernel 4.19 LTS 版本升级到 4.19.91-26 版本，特性更新如下：</p> </li> 
</ul> 
<ul style="list-style-type:circle; margin-left:16px; margin-right:16px"> 
 <li> <p>提供更丰富的新型 RAID 驱动支撑</p> </li> 
 <li> <p>龙蜥自研混部资源隔离特性 Group Identity 支持</p> </li> 
 <li> <p>龙蜥自研多租户容器网络 QoS 隔离特性增强</p> </li> 
 <li> <p>龙蜥自研 Slab 内存安全回收特性支持</p> </li> 
 <li> <p>龙蜥自研 UKFEF 特性支持</p> </li> 
 <li> <p>Megaraid_sas设备驱动更新，并支持 Aero 系列 Controllers</p> </li> 
 <li> <p>virtio-net 支持 XDP Socket</p> </li> 
 <li> <p>基于 EROFS 实现 RAFS v6 格式支持</p> </li> 
 <li> <p>livepatch 增加 static key 支持</p> </li> 
 <li> <p>SMR zoned 设备支持</p> </li> 
 <li> <p>ext4 delalloc buffer写性能优化</p> </li> 
 <li> <p>Kunit 测试框架支持</p> </li> 
 <li> <p>Arm 架构支持 kdump crashkernel 分配 4G 以上内存</p> </li> 
 <li> <p>Arm 架构 CMN PMU 特性支持</p> </li> 
 <li> <p>perf c2c 功能支持</p> </li> 
</ul> 
<h3><span style="color:#888888"><strong style="color:#888888">6.2 软件包</strong></span></h3> 
<ul style="list-style-type:disc; margin-left:8px; margin-right:8px"> 
 <li> <p style="margin-left:0; margin-right:0">更新 Dragonwell 8 到 1.8.0.332 版本，更新 Dragonwell 11 到 11.0.15.11.9 版本，和系统更无缝集成，无需指定 JAVA_HOME 即可运行，需开启 Plus 仓库安装。</p> </li> 
 <li> <p style="margin-left:0; margin-right:0">更新智能调优平台 Keentune 到 1.2.1 版本。</p> </li> 
 <li> <p style="margin-left:0; margin-right:0">来自高性能存储 SIG 的 nydus-rs 及 erofs-utils 工具结束孵化，正式进入 Plus 仓库。</p> </li> 
 <li> <p style="margin-left:0; margin-right:0">用户态软件包增加平头哥平台的补丁。</p> </li> 
 <li> <p style="margin-left:0; margin-right:0">安装过程增加对国产平台的动态内核选择适配。</p> </li> 
 <li> <p style="margin-left:0; margin-right:0">用户态软件包集成下游发行版回馈软件包集成 100+。</p> </li> 
 <li> <p style="margin-left:0; margin-right:0">用户态软件包增加 LoongArch64 平台的补丁。</p> </li> 
 <li> <p style="margin-left:0; margin-right:0">升级 Perl 5.32 到 AppStream、升级 PHP 8.0 到 AppStream 、升级 container-tools 4.0 到 AppStream。</p> </li> 
 <li> <p style="margin-left:0; margin-right:0">升级 AppStream 的 LLVM 工具集、升级 AppStream 中的 Rust 工具集、升级 AppStream 中的 Go 工具集。</p> </li> 
</ul> 
<h3 style="margin-left:0px; margin-right:0px; text-align:justify"><span style="color:#888888"><strong style="color:#888888">6.</strong><strong style="color:#888888">3</strong><strong style="color:#888888"> 镜像</strong></span></h3> 
<ul style="list-style-type:disc; margin-left:8px; margin-right:8px"> 
 <li> <p style="margin-left:0; margin-right:0">发布镜像丰富化，提供网络启动 ISO 和最小化 ISO。</p> </li> 
 <li> <p style="margin-left:0; margin-right:0">新增 RISC-V 试用预览版镜像。</p> </li> 
 <li> <p style="margin-left:0; margin-right:0">DVD 集成了 Plus 仓库。</p> </li> 
</ul> 
<h2 style="margin-left:0px; margin-right:0px; text-align:justify"><span style="color:#11510c">7、软件包改动</span></h2> 
<p style="margin-left:0px; margin-right:0px; text-align:justify">升级同步到 8.6 版本，主要语言和工具更新包括：</p> 
<ul style="margin-left:8px; margin-right:8px"> 
 <li> <p>升级 Perl 5.32 到 AppStream</p> </li> 
 <li> <p style="margin-left:0; margin-right:0">升级 PHP 8.0 到 AppStream</p> </li> 
 <li> <p>升级 AppStream 的 LLVM 工具集</p> </li> 
 <li> <p>升级 AppStream 中的 Rust 工具集</p> </li> 
 <li> <p>升级 AppStream 中的 Go 工具集</p> </li> 
 <li> <p style="margin-left:0; margin-right:0">新增 bind 9.16 和 dotnet 6.0</p> </li> 
</ul>
