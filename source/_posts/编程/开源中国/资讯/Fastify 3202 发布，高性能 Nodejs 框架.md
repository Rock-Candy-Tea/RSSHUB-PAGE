                                                                                            <p>Fastify 3.20.2 发布了，Fastify 是一个高度专注于以最少开销和强大的插件架构为开发者提供最佳体验的 Node.js Web 框架，速度极快，它的灵感来源于 Hapi 和 Express。</p> 
<p>此版本更新内容包括：</p> 
<ul> 
 <li>docs：添加有关 Reply 通用接口的信息 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffastify%2Ffastify%2Fpull%2F3223" target="_blank">#3223</a></li> 
 <li>docs：更新会话状态 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffastify%2Ffastify%2Fpull%2F3239" target="_blank">#3239</a></li> 
 <li>阐明 getting started <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffastify%2Ffastify%2Fpull%2F3241" target="_blank">#3241</a></li> 
 <li>build(deps): 将 fastify/github-action-merge-dependabot 从 2.2.0 升级到 2.3.0 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffastify%2Ffastify%2Fpull%2F3243" target="_blank">#3243</a></li> 
 <li>修复：请求设置 headers <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffastify%2Ffastify%2Fpull%2F2817" target="_blank">#2817</a></li> 
 <li>docs：修复导入错误声明 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffastify%2Ffastify%2Fpull%2F3248" target="_blank">＃3248</a></li> 
 <li>修复了 @fastify/session 的死链接 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffastify%2Ffastify%2Fpull%2F3246" target="_blank">#3246</a></li> 
 <li>build(deps): 将 fastify/github-action-merge-dependabot 从 2.3.0 升级到 2.4.0 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffastify%2Ffastify%2Fpull%2F3249" target="_blank">#3249</a></li> 
 <li>添加 Fastify-allow 插件参考 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffastify%2Ffastify%2Fpull%2F3242" target="_blank">#3242</a></li> 
 <li>修复了在 TLS 连接中的套接字泄漏<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffastify%2Ffastify%2Fpull%2F3254" target="_blank">#3254</a></li> 
</ul>
