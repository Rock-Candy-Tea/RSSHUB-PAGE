                                                        <p>虽然 <a href="https://www.oschina.net/news/148866/openzfs-2-1-0-released" target="_blank">OpenZFS 2.1</a> 已在 7 月正式发布，<span style="background-color:#ffffff; color:#121212">但有不少用户仍在使用 OpenZFS 2.0.x 系列并且尚未准备好使用 dRAID 和其他较大的变更。因此，官方针对这部分用户发布了 </span>OpenZFS 2.0.6 更新。</p> 
<p>OpenZFS 2.0.6 属于维护版本，适用于尚未迁移到 v2.1 系列的用户。OpenZFS 2.0.6 最显著的更新是支持较新版本的 Linux 内核：OpenZFS 2.0.5 一直支持到 Linux 5.12，而 OpenZFS 2.0.6 现在支持 Linux 5.13/5.14 以及一些早期的 5.15 兼容性补丁。</p> 
<p>除此之外，<span style="background-color:#ffffff; color:#121212">OpenZFS 2.0.6 还包含部分特定于 FreeBSD 的修复和其他一些核心修复。</span></p> 
<p>完整变更说明和下载地址<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fopenzfs%2Fzfs%2Freleases%2Ftag%2Fzfs-2.0.6" target="_blank">查看 release notes</a>。</p>
