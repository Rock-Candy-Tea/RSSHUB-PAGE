                                                                                            <p>uView 2.0 2.0.18 已经发布，多平台快速开发 UI 框架</p> 
<p>此版本更新内容包括：</p> 
<h3>修复</h3> 
<ol> 
 <li>修复app端编译报错问题</li> 
 <li>重新处理微信小程序端setData过大的性能问题</li> 
 <li>修复边框问题</li> 
 <li>修复最大最小月份不大于0则没有数据出现的问题</li> 
 <li>修复SwipeAction微信小程序端无法上下滑动问题</li> 
 <li>修复input的placeholder在小程序端默认显示为true问题</li> 
 <li>修复divider组件click事件无效问题</li> 
 <li>修复u-code-input maxlength 属性值为 String 类型时显示异常</li> 
 <li>修复当 grid只有 1到2时 在小程序端algin设置无效的问题</li> 
 <li>处理form-item的label为top时，取消错误提示的左边距</li> 
 <li>其他修复</li> 
</ol>
