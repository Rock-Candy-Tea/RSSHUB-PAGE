                                                        <h3><strong>更新日志</strong></h3> 
<ul> 
 <li>添加当前用户 (提权) 模式支持 (由 xspeed1989 建议)</li> 
 <li>修复在 Windows 服务上下文使用 NSudo 遇到的阻碍性问题 (感谢 xspeed1989)</li> 
 <li>改善数处实现</li> 
 <li>修复在 NSudo 不支持的语言设置下界面显示中文的问题 (感谢 rlesch)(#56)</li> 
 <li>升级 Mile.Cpp 到最新版本 
  <ul> 
   <li>从 Mile.Project 升级到 Mile.Project.VisualStudio 最新版本 </li> 
   <li>将 Mile.Windows.TrustedLibraryLoader 与 Mile.Platform.Windows 升级到 Mile.Library 最新版本 </li> 
   <li>升级 VC-LTL 到最新版本 </li> 
  </ul> </li> 
 <li>升级 Windows Template Library (WTL) 到 10.0.10320</li> 
 <li>移除 ARM32 支持 
  <ul> 
   <li>理由: <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fmourinaruto.github.io%2F2021%2F05%2F17%2FDrop-ARM32-support-on-the-Windows-platform%2F" target="_blank">https://mourinaruto.github.io/2021/05/17/Drop-ARM32-support-on-the-Windows-platform/</a></li> 
  </ul> </li> 
 <li>改善数处文档相关的实现 
  <ul> 
   <li>改善项目网站体验 (由 青春永不落幕 贡献) </li> 
   <li>改善码云体验</li> 
   <li>使用 GitHub Actions 部署项目网站</li> 
  </ul> </li> 
 <li>Add German Language. (由 Steve 贡献)</li> 
 <li>移除一些试验性实现, 包括 NSudo Sweeper</li> 
 <li>添加日志支持</li> 
</ul>

<h3><strong>后记</strong></h3> 
<p>该版本主要是引入了日志支持，还有命令行和 SDK 模式下的 当前用户（提权） 模式<br> 倒是 NSudo 8.1 被跳过了，最近发布的会是 8.2<br> 主要原因是之前自己改版本号没有去掉 Update 1 字样，于是 8.1 Update 1 先于 8.1 出现，可能会导致混淆，毕竟无忧论坛有人抢先体验了当时的那些版本</p>
