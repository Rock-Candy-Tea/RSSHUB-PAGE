                                                                                            <p>MyBatis 3.5.8<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fblog.mybatis.org%2F2021%2F12%2Fmybatis-358-released.html" target="_blank"> 已发布</a>，MyBatis 的前身为 iBatis，是一个数据持久层（ORM）框架，它提供的持久层能力包括 SQL Maps 和 Data Access Objects（DAO）。</p> 
<p>主要更新内容：</p> 
<ul style="margin-left:.5em; margin-right:.5em"> 
 <li>将空字符串映射到 java.lang.Character 时避免 `NullPointerException`<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmybatis%2Fmybatis-3%2Fissues%2F2368" target="_blank">#2368</a></li> 
 <li>修复初始化静态对象时的一个错误参数，这解决了与 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fquarkiverse%2Fquarkus-mybatis" target="_blank">quarkus-mybatis</a> 的兼容性问题<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmybatis%2Fmybatis-3%2Fissues%2F2284" target="_blank">#2284</a></li> 
 <li>改进性能<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmybatis%2Fmybatis-3%2Fissues%2F2297" target="_blank">#2297</a><span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmybatis%2Fmybatis-3%2Fissues%2F2335" target="_blank">#2335</a><span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmybatis%2Fmybatis-3%2Fissues%2F2340" target="_blank">#2340</a></li> 
 <li>大量文档更新</li> 
</ul> 
<p>发布公告还写道，自 3.5.7 以来，此版本没有任何已知的向后不兼容变化，完整变更内容查看 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmybatis%2Fmybatis-3%2Fissues%3Fq%3Dis%253Aclosed%2Bmilestone%253A3.5.8" target="_blank">3.5.8 milestone 页面</a>。</p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">此外还有代码和文档方面的改进，<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmybatis%2Fmybatis-3%2Fissues%3Fq%3Dis%253Aclosed%2Bmilestone%253A3.5.7" target="_blank">详情点此查看</a>。</p>
