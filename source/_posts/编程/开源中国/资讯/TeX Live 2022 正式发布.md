<ul style="margin-left:0; margin-right:0"> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.tug.org%2Ftexlive%2Facquire-netinstall.html" target="_blank">在线安装 TeX Live</a>（官方推荐方式，通过网络进行安装）</li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.tug.org%2Ftexlive%2Facquire-dvd.html" target="_blank">通过 DVD 安装 TeX Live</a>（官方推荐方式，使用媒介进行安装）</li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.tug.org%2Ftexlive%2Facquire-iso.html" target="_blank">下载体积巨大的 ISO 文件</a>（通常用于刻录 DVD）</li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.tug.org%2Ftexlive%2Facquire-tar.html" target="_blank">下载多个体积巨大的 tarball 包</a></li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.tug.org%2Ftexlive%2Facquire-mirror.html" target="_blank">下载 TeX Live 镜像</a></li> 
</ul> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">详情查看：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.tug.org%2Ftexlive%2Facquire.html" target="_blank">https://www.tug.org/texlive/acquire.html</a></p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><span><img alt src="https://attachments.tower.im/tower/b6f97f1cd98fdc725675c39cd5e1e652?version=auto" referrerpolicy="no-referrer"></span></p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">TEX Live 包括了 TEX, LATEX2ε, ConTEXt,METAFONT, MetaPost, BibTEX 等许多可执行程序；种类繁多的宏包、字体和文档，并支持世界上许多不同的语言。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">TEX Live 为多种基于 Unix 的平台提供了可执行文件，包括 GNU/Linux、Mac OS X、和 Cygwin。它还包含了源代码，可供在没有提供可执行文件的平台上编译安装。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">至于 Windows，TEX Live 仅支持 Windows 7 或后续版本。Windows Vista 可能可以继续工作，但 TEX Live 不能在 Windows XP 或者更早的版本上运行。TeX Live 没有包含 64 位的 Windows 可执行文件， 不过 32 位的可执行文件也能 64 位的系统上正常运行。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><strong>TeX Live 2022 更新亮点</strong></p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>新增全新排版引擎——<strong>HiTeX</strong>，专为阅读移动设备上的技术文档而设计，插图也比较方便。使用<code>texdoc hitex</code>命令即可查看 HiTeX 文档，HiTeX 输出格式是<code>.hnt</code>。</li> 
 <li>tangle , weave：支持使用可选的第三个参数来指定输出文件。</li> 
 <li>面向 epTeX, eupTeX 的新原语：\lastnodefont,\suppresslongerror,\suppressoutererror,\suppressmathparerror</li> 
 <li><code>tabularray</code> 宏包。<code>tabularray</code> 宏包是国内的开发者写的宏包。可使用<code>texdoc tabularray</code> 命令查看宏包文档。</li> 
</ul>
