                                                                                            <p>Spring Boot 3.0.0-M4 已发布，更新内容主要是功能增强、文档优化、依赖升级和错误修复。</p> 
<p><strong>值得关注的变化</strong></p> 
<ul> 
 <li>支持 Elasticsearch 的新版 Java 客户端</li> 
 <li>支持 Flyway 9</li> 
 <li>支持 Hibernate 6.1</li> 
 <li>AOT 支持 Actuator</li> 
 <li>移除 Actuator</li> 
 <li>升级 Kotlin 最低支持版本为 1.7</li> 
 <li>升级 Gradle 最低支持版本为 7.5</li> 
 <li><span style="background-color:#ffffff; color:#24292f">删除对运行多个 Spring Batch 作业的支持</span></li> 
 <li>支持<span style="background-color:#ffffff; color:#24292f">在非 Web 应用程序的原生镜像中自动配置 HTTP 消息转换器</span></li> 
 <li><span style="background-color:#ffffff; color:#24292f">自动配置 JvmCompilationMetrics</span></li> 
 <li><span style="background-color:#ffffff; color:#24292f">添加资源提示以提供静态 Web 内容</span></li> 
</ul>
