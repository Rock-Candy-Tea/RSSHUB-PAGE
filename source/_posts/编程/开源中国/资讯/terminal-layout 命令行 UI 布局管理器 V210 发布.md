                                                        <p>terminal_layout 是一个命令行布局管理器，支持Windows，Linux，OSX。</p> 
<ul> 
 <li>Github： <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fgojuukaze%2Fterminal_layout" target="_blank">https://github.com/gojuukaze/terminal_layout</a></li> 
 <li>文档：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdoc.ikaze.cn%2Fterminal_layout%2F" target="_blank">https://doc.ikaze.cn/terminal_layout/</a></li> 
</ul> 
<p> </p> 
<h1>V2.1更新内容</h1> 
<ul> 
 <li> <p>增加input扩展，可以获取文字输入了（不支持windows）</p> </li> 
 <li> <p>TextView 增加 <code>overflow</code> 属性，用户文本过长时隐藏左边还是右边</p> </li> 
 <li> <p>view 增加 parent 属性</p> </li> 
 <li> <p>ctl自动重绘可通过设置 <code>refresh_thread_stop</code> 停止重绘</p> </li> 
</ul>
