                                                                                            <p>Gitmars 3.0.0 已经发布，高度定制化的 git 工作流工具。</p> 
<p>此版本更新内容包括：</p> 
<ol> 
 <li>monorepo改造，拆分成<code>gitmars</code> <code>@gitmars/core</code> <code>@gitmars/server</code> <code>@gitmars/ui</code> <code>docs</code>五个子项目</li> 
 <li>放开<code>alias</code> 指令只能在git项目下面执行的限制</li> 
 <li>升级依赖包版本</li> 
 <li>优化部分性能</li> 
</ol> 
<p>详情查看：<a href="https://gitee.com/saqqdy/gitmars/releases/3.0.0">https://gitee.com/saqqdy/gitmars/releases/3.0.0</a></p>
