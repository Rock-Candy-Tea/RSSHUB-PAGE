                                                                                            <p>经过短暂的延迟，Chrome OS 98 稳定版正式推出。新版本带来了一些变化，其中包括新的表情符号、新增的虚拟桌面快捷键，以及全面修改的语言设置菜单。当然也包括一些与最近发布的 Chrome 98 相同的功能。</p> 
<p>Chrome OS 98 目前仍在使用 2014 年以来的旧版 Logo，并没有使用前几天刚刚推出的全新 Logo。</p> 
<p>以下是 Chrome OS 98 的更新内容：</p> 
<ul> 
 <li> <p>Files 应用获得黑暗主题支持</p> <p><img alt height="394" src="https://static.oschina.net/uploads/space/2022/0209/072224_PsSE_4937141.png" width="700" referrerpolicy="no-referrer"></p> </li> 
 <li> <p>可以选择屏幕截图的保存位置（此前所有截图都会保存到 "下载" 文件中）</p> <p><img alt height="264" src="https://static.oschina.net/uploads/space/2022/0209/072237_VUCv_4937141.png" width="700" referrerpolicy="no-referrer"></p> </li> 
 <li> <p>Emoji 选择器更新至 emoji 14.0，新增 37 个表情符号</p> <p><img alt height="394" src="https://static.oschina.net/uploads/space/2022/0209/072253_l2fu_4937141.png" width="700" referrerpolicy="no-referrer"></p> </li> 
 <li> <p>新增虚拟桌面快捷键</p> <p><img alt height="476" src="https://static.oschina.net/uploads/space/2022/0209/072310_pAiZ_4937141.png" width="700" referrerpolicy="no-referrer"></p> </li> 
 <li> <p>大幅改进语言设置菜单</p> </li> 
 <li> <p>推出基于网络的恢复功能</p> </li> 
 <li> <p>虚拟键盘新增黑暗模式标志</p> </li> 
 <li> <p>使用鼠标时，ARC 应用程序的滚动性能得到改善</p> </li> 
 <li> <p>在启动器中禁用 Google Play Store 搜索</p> </li> 
 <li> <p>修复了从 Linux 中进行文件拖放时，随机产生的崩溃问题</p> </li> 
 <li> <p>修复通过 WebUSB 自动授予 USB 访问权的问题</p> </li> 
</ul>
