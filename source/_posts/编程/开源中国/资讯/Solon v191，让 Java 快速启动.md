                                                                                            <h4 style="text-align:start">相对于 Spring Boot 和 Spring Cloud 的项目：</h4> 
<ul> 
 <li>启动快 5 ～ 10 倍。<span> </span><strong>（更快）</strong></li> 
 <li>qps 高 2～ 3 倍。<span> </span><strong>（更高）</strong></li> 
 <li>运行时内存节省 1/3 ~ 1/2。<span> </span><strong>（更少）</strong></li> 
 <li>打包可以缩小到 1/2 ~ 1/10；比如，90Mb 的变成了 9Mb。<span> </span><strong>（更小）</strong></li> 
 <li>基于 app.name 进行注册发现 与 k8s svc 相互对应</li> 
 <li>支持 Service Mesh 架构部署方案</li> 
</ul> 
<h4 style="text-align:start">本次更新：</h4> 
<ul> 
 <li>新增 dubbo3-solon-plugin 插件</li> 
 <li>简化 dubbo-solon-plugin 插件的适配</li> 
 <li>增加 solon.app.namespace 配置支持</li> 
 <li>增加 solon.data.tran 手动控制能力</li> 
 <li>插件 activerecord-solon-plugin 升级 activerecord 为 5.0.0</li> 
 <li>插件 solon.view.enjoy 升级 enjoy 为 5.0.0</li> 
 <li>插件 beetlsql-solon-plugin 升级 beetlsql 为 3.14.5-RELEASE</li> 
 <li>插件 dbvisitor-solon-plugin 升级 dbvisitor 为 5.0.1</li> 
 <li>插件 sqltoy-solon-plugin 升级 sqltoy 为 5.2.2</li> 
 <li>插件 solon.boot.smarthttp 升级 smart-http 为 1.1.5</li> 
 <li>插件 nacos2-solon-plugin 升级 nacos-client 为 2.1.0</li> 
 <li>插件 water-solon-plugin 升级 water 为 2.8.0</li> 
</ul>
