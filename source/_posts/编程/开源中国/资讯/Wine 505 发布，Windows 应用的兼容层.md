                                                                                            <p>Wine 5.0.5 已经发布。Wine（Wine Is Not an Emulator）是一个能够在多种兼容 POSIX 接口的操作系统（诸如 Linux、macOS 与 BSD 等）上运行 Windows 应用的兼容层。它不是像虚拟机或者模拟器一样模仿内部的 Windows 逻辑，而是将 Windows API 调用翻译成为动态的 POSIX 调用，免除了性能和其它一些行为的内存占用，让你能够干净地整合 Windows 应用到桌面。 </p> 
<p><strong>部分更新内容</strong></p> 
<ul> 
 <li>修复 QIP Infium 2.0 build 9030 在退出时崩溃的问题</li> 
 <li>修复 Memorex exPressit Label Design Studio 4.x 在创建新项目时崩溃的问题</li> 
 <li>修复从 Finddupe 生成的批处理文件创建硬链接失败的问题</li> 
 <li>修复 Gnomoria 演示在启动时崩溃的问题</li> 
 <li>修复将 WBEM_MASK_CONDITION_ORIGIN 错误地传递给 IWbemClassObject::GetNames 方法导致一些 WMI 应用程序崩溃的问题</li> 
 <li>修复 typelib writer 中的空指针检查不一致的问题</li> 
 <li>修复当 .bat 脚本变量有超过 256 个字符时堆栈溢出的问题</li> 
 <li>修复命运 2 启动时崩溃的问题</li> 
 <li>修复 GOG GALAXY 在虚拟桌面上无法运行的问题</li> 
 <li>完善内置的 Split 功能</li> 
</ul>
