                                                        <p>Electron 13.1.0 现已发布，主要更新内容如下： </p> 
<p><strong>Features</strong></p> 
<ul> 
 <li> <p>在 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.electronjs.org%2Fdocs%2Ftutorial%2Ffuses" target="_blank">Electron Fuse</a> 后面添加了实验性 cookie 加密支持。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Felectron%2Felectron%2Fpull%2F29493" target="_blank">#29493</a></p> </li> 
 <li>添加了对调试 URL 的支持，例如<code>chrome://gpucrash</code>. <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Felectron%2Felectron%2Fpull%2F29491" target="_blank">#29491</a></li> 
</ul> 
<p><strong>Fixes</strong></p> 
<ul> 
 <li> <p>对于使用注册协议的请求，修复了 Service Worker 中丢失的'fetch'事件。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Felectron%2Felectron%2Fpull%2F29446" target="_blank">＃29446 </a></p> </li> 
 <li>修复了用鼠标点击按下 Alt 会导致切换菜单栏的问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Felectron%2Felectron%2Fpull%2F29450" target="_blank">＃29450 </a></li> 
 <li>修复了在 KDE 上按下<code>Alt+Shift</code>和<code>Alt+Ctrl</code>切换菜单栏的问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Felectron%2Felectron%2Fpull%2F29443" target="_blank">＃29443 </a></li> 
</ul> 
<p><strong>Other Changes</strong></p> 
<ul> 
 <li> <p>对 1090441、1173656、1181434、1188117 的反向移植修复。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Felectron%2Felectron%2Fpull%2F29494" target="_blank">#29494</a></p> </li> 
 <li>修复了在 Apple Silicon 上使用自定义 v8 快照的问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Felectron%2Felectron%2Fpull%2F29339" target="_blank">#29339</a></li> 
 <li>将 Chromium 更新为 91.0.4472.77。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Felectron%2Felectron%2Fpull%2F29321" target="_blank">#29321</a></li> 
</ul> 
<p><strong>Documentation</strong></p> 
<ul> 
 <li> <p>Documentation changes：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Felectron%2Felectron%2Fpull%2F29430" target="_blank">#29430</a></p> </li> 
</ul>
