                                                        <p>youtube-dl 是一个小型的命令行工具，用于下载视频。虽然它最初仅支持 <a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fyoutube.com%2F">YouTube.com</a>，但它目前已支持许多其他平台，如 Anitube、<a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Farchive.org%2F">Archive.org</a>、Bloomberg、CBS、Discovery、dropbox、eHow、flickr、Google+、MTV、MyVideo、NBC、SoundCloud、Southpark、Steam 和 Vimeo 等。它可以保存视频 MP4 和其他提供的格式，也可以只提取音轨。</p> 
<p>youtube-dl 2021.04.17 正式发布，本次更新内容如下：</p> 
<p>核心：</p> 
<ul> 
 <li>[utils]增加对实验性 HTTP 响应状态码 308 永久重定向的支持；</li> 
</ul> 
<p>提取器：</p> 
<ul> 
 <li>[lbry] 增加对 HLS 视频的支持；</li> 
 <li>[youtube] 修复拉伸比率计算；</li> 
 <li>[youtube] 改善拉伸提取；</li> 
 <li>[youtube:tab] 改进网格提取；</li> 
 <li>[youtube:tab] 在播放列表页面检测剧集播放列表；</li> 
 <li>[youtube] 增加更多 invidious 的实例；</li> 
 <li>[pluralsight] 延长抗节流超时时间；</li> 
 <li>[youtube] 改进 URL 到提取器的路由；</li> 
 <li>[maoritv]增加对 <a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fmaoritelevision.com" target="_blank">maoritelevision.com</a> 的支持；</li> 
 <li>[youtube:tab] 传递 innertube 上下文和 x-goog-visitor-id header 以及延续请求；</li> 
 <li>[MTV]修复 Viacom A/B 测试视频播放器提取；</li> 
 <li>[pornhub]从 get_media 端点提取 DASH 和 HLS 格式；</li> 
 <li>[cbssports] 修复提取；</li> 
 <li>[jamendo]修复轨道提取；</li> 
 <li>[curiositystream] 修正格式提取；</li> 
</ul> 
<p>更多详情可查看：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fytdl-org%2Fyoutube-dl%2Freleases" target="_blank">https://github.com/ytdl-org/youtube-dl/releases</a></p>
