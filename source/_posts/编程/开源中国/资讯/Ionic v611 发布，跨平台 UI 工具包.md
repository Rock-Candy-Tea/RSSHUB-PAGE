                                                                                            <p>Ionic 是一个强大的跨平台 UI 工具包，通过使用 HTML、CSS 和 JavaScript 构建原生质量的 iOS、Android 和渐进式 Web 应用程序。Ionic 基于 Web 组件，可显著提升性能、可用性和功能，同时支持 Angular、React 和 Vue 等流行 Web 框架。</p> 
<p>Ionic 6 是目前最新的版本，于去年 12 月正式推出，值得关注的新特性包括：改进桌面支持、彻底优化组件，以及更改在 iOS 和 Android 上的设计。最新发布的 6.1.1 是一个 Bugfix 版本：</p> 
<ul> 
 <li>在模板应用程序中使用 ionic 时，使用正确的导入路径 <span style="color:#24292f">(</span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fionic-team%2Fionic-framework%2Fissues%2F25123" target="_blank">#25123</a><span style="color:#24292f">)</span></li> 
 <li>datetime: 获取当前日期时，考虑 30 和 45 分钟的时区  <span style="color:#24292f">(</span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fionic-team%2Fionic-framework%2Fissues%2F25120" target="_blank">#25120</a><span style="color:#24292f">)</span></li> 
 <li>modal, popover：不关闭从 select 发出时 ionDismiss <span style="color:#24292f">(</span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fionic-team%2Fionic-framework%2Fissues%2F25125" target="_blank">#25125</a><span style="color:#24292f">) </span></li> 
</ul>
