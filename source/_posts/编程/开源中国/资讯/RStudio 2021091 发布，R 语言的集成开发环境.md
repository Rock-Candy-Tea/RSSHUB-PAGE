                                                                                            <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">RStudio 2021.09.1 现已发布。RStudio 是 R 语言的集成开发环境，分为面向桌面用户 IDE 和 Linux R 服务器版编辑器两种编辑器，采用 AGPL v3 与 <a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fwww.rstudio.com%2Fabout%2Feula%2F">RStudio License Agreement</a> 双协议授权。该版本的更新内容如下：</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><strong>Bug 修复</strong> </p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>修复了导致验证安装在负载平衡启动器环境中挂起的问题（Pro #3036）</li> 
 <li>修复了无法从预览窗口发布到 Connect 的错误 (#9901)</li> 
 <li>修复了在运行“Check for Updates”时阻止 RStudio 正确报告新版本的错误（Pro #9911）</li> 
 <li>修复了无法禁用“自动激活项目本地 Python 环境”设置的错误 (#9952)</li> 
 <li>修复了当 conda 安装在典型位置但不在路径上时，无法检测 Python 环境的问题 (#9975)</li> 
 <li>使 Python 检测不太可能覆盖 RETICULATE_PYTHON (#9999)</li> 
</ul> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">更新说明：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.rstudio.com%2Fproducts%2Frstudio%2Frelease-notes%2F" target="_blank">https://www.rstudio.com/products/rstudio/release-notes/</a> </p>
