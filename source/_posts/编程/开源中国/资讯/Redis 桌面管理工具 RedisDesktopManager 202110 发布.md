                                                                                            <p>Redis Desktop Manager 是基于 Qt 5 的 Redis 开源跨平台桌面管理器。Redis Desktop Manager 2021.10 发布，更新内容如下：</p> 
<h3>修复和改进</h3> 
<ul> 
 <li>修复服务器在打开密钥的情况下关闭连接时的崩溃</li> 
 <li>修复 qml 运行时在 Windows 上的崩溃</li> 
 <li>修复初始 UI 渲染时的冻结问题</li> 
 <li>从 Python 部署中删除 numpy 和 pandas 以防止 Python 运行时崩溃</li> 
 <li>改进从 RDB 的导入：优化密钥扫描、验证和错误处理</li> 
 <li>修复：RDM 无法删除键名中带有空字节的 NS</li> 
 <li>修复实时更新中无效的单一命名空间的密钥渲染</li> 
 <li>修复 Windows/Linux 下 AddKeyDialog 的大小问题</li> 
</ul>
