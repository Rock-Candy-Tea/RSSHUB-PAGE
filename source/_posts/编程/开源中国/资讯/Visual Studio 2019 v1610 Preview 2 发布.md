                                                        <p>Visual Studio 2019 v16.10 Preview 2 现已发布。这个版本延续了开发者生产力和便利性的主题，并增加了 C++20 支持、IntelliSense 补全、用于检查继承链的边距图标、Docker 工具增强和 Git 集成的新功能。</p> 
<p><strong>主要更新内容</strong></p> 
<ul> 
 <li>完全实现 C++20 Ranges 功能。如果要尝试该功能，请使用 std:c++latest 编译</li> 
 <li>现在 Visual Studio 支持 CMakePresets.json 文件以替代 CMakeSetting.json 文件</li> 
</ul> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-41a39c73871d4af97f47274cf91eb8abfdc.png" referrerpolicy="no-referrer"></p> 
<ul> 
 <li>现在，外部头文件可以有自己的编译器警告级别、代码分析和模板诊断设置。用户可以为项目代码选择更严格的设置，以确保代码质量</li> 
</ul> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-d8ce16c0d1cc8fb3128361794dd00fdd46e.png" referrerpolicy="no-referrer"></p> 
<ul> 
 <li>通过添加一个提示来接受或拒绝服务器提供的主机密钥指纹，使创建远程连接更加安全</li> 
 <li>IntelliSense 现在提供了 cast、indexer 和 operator 的补完</li> 
</ul> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-b2c3c2292721118a4258987ff85d762534c.gif" referrerpolicy="no-referrer"></p> 
<ul> 
 <li>现在有一个完成选项：在编写方法调用时自动插入参数</li> 
</ul> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-2e16e25a5ea6b38aa83b999b9813799b472.gif" referrerpolicy="no-referrer"></p> 
<ul> 
 <li>EditorConfig 现在具有用户界面</li> 
</ul> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-f15a4fb970e5f3e849d5eb6a959557b84b0.png" referrerpolicy="no-referrer"></p> 
<ul> 
 <li>导航和检查继承链已经可视化</li> 
</ul> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-33f4ad9e593c27e42a5274b1e07d884dcf9.png" referrerpolicy="no-referrer"></p> 
<ul> 
 <li>现在在 Visual Studio 中的 Docker Compose 工具提供了运行 Compose 文件中定义的任何服务组合的能力</li> 
</ul> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-206474fe31b521ee39522615add9a59bb0a.png" referrerpolicy="no-referrer"></p>
