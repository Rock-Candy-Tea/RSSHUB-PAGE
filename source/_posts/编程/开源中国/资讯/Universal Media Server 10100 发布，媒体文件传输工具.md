                                                        <p>Universal Media Server 10.9.0 现已发布。Universal Media Server 是一个兼容 DLNA，UPnP 和 HTTP/S 的媒体服务器，支持所有主要操作系统，包括 Windows，Linux 和 macOS 版本，几乎不需要配置就可以流式传输或转码许多不同的媒体格式。</p> 
<p><strong>主要更新内容</strong></p> 
<ul> 
 <li>如果播放了所有剧集，则在媒体库中的电视剧缩略图中添加 “完全播放” 的叠加显示</li> 
</ul> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-f95618607c12742e8865bfafc0418183ea7.png" referrerpolicy="no-referrer"></p> 
<ul> 
 <li>修复了一些不反映内容更新的渲染器，例如完全播放的缩略图更改</li> 
 <li>修复了系列名称中带有年份的文件中剧集的 API 匹配问题</li> 
 <li>修复了文件在下载期间或刚下载后锁定的问题</li> 
 <li>忽略不完整的文件</li> 
</ul> 
<p>详情请查看<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.universalmediaserver.com%2Fforum%2Fviewtopic.php%3Ff%3D8%26t%3D14818" target="_blank">更新公告</a>。</p>
