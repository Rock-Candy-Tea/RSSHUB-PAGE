<div class="content">
                                                                                            <p><span style="background-color:#ffffff; color:#333333">Android Studio 2021.1.1（代号"Bumblebee"）稳定版<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fandroid-developers.googleblog.com%2F2022%2F01%2Fandroid-studio-bumblebee-202111-stable.html" target="_blank">已发布</a>。</span></p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-d7dfc68f5a83d124ade4952cefd7d51fe8a.png" referrerpolicy="no-referrer"></p> 
<p>新功能主要集中在三个方面：构建和部署、性能分析工具和审查，以及设计。</p> 
<p><strong>构建和部署</strong></p> 
<ul> 
 <li>在 Android Studio 中使用 Gradle 运行 Instrumented Tests</li> 
</ul> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-9a059a30bc91bd305f821e3419307159b46.png" referrerpolicy="no-referrer"></p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-a45e12cd4c87b3a7c0a118bf019c5b4f0d2.png" referrerpolicy="no-referrer"></p> 
<ul> 
 <li>Android Gradle 插件升级助手现已支持更新 API 用法</li> 
 <li>为新项目开启 Non-Transitive R classes</li> 
 <li>引入新的设备管理器</li> 
 <li>通过 Wi-Fi 连接 ADB</li> 
 <li>默认启用模拟器工具窗口</li> 
 <li>支持 Apple Silicon</li> 
</ul> 
<p><strong>性能分析工具和审查</strong></p> 
<ul style="margin-left:.5em; margin-right:.5em"> 
 <li>性能分析工具新增 Jank 检测通道</li> 
</ul> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-8959cf247a21f47176f358d7e1e920b7c47.png" referrerpolicy="no-referrer"></p> 
<ul> 
 <li>在 Studio Profilers 中为可分析性能的应用程序提供分析</li> 
 <li>在后台任务检测器中对 Jobs, Alarms 和 Wakelocks 进行审查</li> 
 <li>支持捕获布局检测器 (Layout Inspector) 的快照</li> 
 <li>为布局检测器提供组合语义支持</li> 
</ul> 
<p><strong>设计</strong></p> 
<ul> 
 <li>提供预览版的交互模式 (Interactive Preview)</li> 
</ul> 
<p><img src="https://static.oschina.net/uploads/space/2022/0127/172704_LgJ4_2720166.gif" referrerpolicy="no-referrer"></p> 
<ul> 
 <li>提供预览阶段的 Animated Vector Drawables</li> 
</ul> 
<p><img src="https://static.oschina.net/uploads/space/2022/0127/172849_pw3E_2720166.gif" referrerpolicy="no-referrer"></p> 
<ul> 
 <li>升级了设计工具的设备选择器</li> 
</ul> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-bad2c49660559960401ce9615efb8482fef.png" referrerpolicy="no-referrer"></p>
