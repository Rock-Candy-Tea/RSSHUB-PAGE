 <p style="color:#2c3e50; text-align:start">另外<code>2.x</code>版本尚未稳定，请勿在生产环境中使用。正式版本预计在春节后发布。</p>

<h4 style="margin-left:0; margin-right:0; text-align:start"><strong>项目介绍</strong></h4> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><span style="background-color:#ffffff; color:#40485b">magic-api 是一个基于 Java 的接口快速开发框架，编写接口将通过 magic-api 提供的UI界面完成，自动映射为  HTTP 接口，无需定义 Controller、Service、Dao、Mapper、XML、VO 等 Java 对象即可完成常见的 HTTP API 接口开发</span></p> 
<h3 style="margin-left:0; margin-right:0; text-align:left"><span style="color:#40485b">特性</span></h3> 
<ul style="list-style-type:disc; margin-left:0; margin-right:0"> 
 <li>支持 MySQL、MariaDB、Oracle、DB2、PostgreSQL、SQLServer 等支持jdbc规范的数据库</li> 
 <li>支持非关系型数据库 Redis、Mongodb</li> 
 <li>支持集群部署、接口自动同步。</li> 
 <li>支持分页查询以及自定义分页查询</li> 
 <li>支持多数据源配置，支持在线配置数据源</li> 
 <li>支持 SQL 缓存，以及自定义 SQL 缓存</li> 
 <li>支持自定义 JSON 结果、自定义分页结果</li> 
 <li>支持对接口权限配置、拦截器等功能</li> 
 <li>支持运行时动态修改数据源</li> 
 <li>支持 Swagger 接口文档生成</li> 
 <li>基于 <a href="https://gitee.com/ssssssss-team/magic-script" target="_blank">magic-script</a> 脚本引擎，动态编译，无需重启，实时发布</li> 
 <li>支持 Linq 式查询，关联、转换更简单</li> 
 <li>支持数据库事务、SQL 支持拼接，占位符，判断等语法</li> 
 <li>支持文件上传、下载、输出图片</li> 
 <li>支持脚本历史版本对比与恢复</li> 
 <li>支持脚本代码自动提示、参数提示、悬浮提示、错误提示</li> 
 <li>支持导入 Spring 中的 Bean、Java 中的类</li> 
 <li>支持在线调试</li> 
 <li>支持自定义工具类、自定义模块包、自定义类型扩展、自定义方言、自定义列名转换等自定义操作</li> 
</ul>
