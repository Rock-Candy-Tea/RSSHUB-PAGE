                                                        <p><span style="background-color:#ffffff; color:#333333">Apache Tomcat 的四个分支发布了更新，分别是 </span>8.5.72、9.0.54、10.0.12 & 10.1.0-M6 (alpha)。</p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">Apache Tomcat 10.1.0-M6 是 10.1.x 分支的第六个里程碑版本，已为用户提供早期访问 Apache Tomcat 10.1.x 中新的功能。</p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">官方表示，Tomcat 10 及更高版本的用户应注意，作为从 Java EE 迁移到 Eclipse Foundation 的的一部分，从 Java EE 迁移到 Jakarta EE 的结果是所有已实现 API 的主要软件包已从<code>javax.*</code>改为<code>jakarta.*</code>，因此部分项目会需要更改代码，以使应用程序能够从 Tomcat 9 及更低版本迁移到 Tomcat 10 及更高版本。官方表示正在开发一种迁移工具来辅助此过程。</p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">此版本的主要变化包括：</p> 
<ul> 
 <li>更新 Servlet 6 的 Servlet API，包括删除所有已被标记为弃用状态的代码、升级 schema 以及引入用于连接和请求 ID 的新 API</li> 
 <li>更新 EL 5.0 的 EL API，包括弃用 FeatureDescriptor、改进 BeanELResolver 和添加 MethodReference</li> 
 <li>对 HTTP/2 流量控制窗口管理的进一步健壮性改进</li> 
</ul> 
<p><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Ftomcat.apache.org%2Ftomcat-10.0-doc%2Fchangelog.html%23Tomcat_10.0.12_%28markt%29" target="_blank">详情查看 Changelog</a>。</p> 
<p>下载地址：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Ftomcat.apache.org%2Fdownload-10.cgi" target="_blank">https://tomcat.apache.org/download-10.cgi</a></p> 
<p>发布公告显示，8.5.72、9.0.54 和 10.0.12 这三个版本的更新内容一致：</p> 
<ul> 
 <li>对 HTTP/2 流量控制窗口管理的进一步健壮性改进</li> 
 <li>改进 DataSourceUserDatabase</li> 
 <li>修复导致部分 Servlet 非阻塞 API 读取 HTTP 请求正文错误使用阻塞 IO 的问题</li> 
</ul>
