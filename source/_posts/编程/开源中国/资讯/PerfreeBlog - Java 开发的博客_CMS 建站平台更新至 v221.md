                                                                                            <p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><span style="background-color:#ffffff; color:#40485b">PerfreeBlog是一款基于Java SpringBoot开发的博客建站平台,支持多主题及扩展插件功能,给您带来全新的创作体验</span></p> 
<ul style="list-style-type:disc; margin-left:0; margin-right:0"> 
 <li>官网地址:<span> </span><a href="https://gitee.com/link?target=http%3A%2F%2Fperfree.org.cn">http://perfree.org.cn</a></li> 
 <li>文档地址:<span> </span><a href="http://perfree.gitee.io/">http://perfree.gitee.io</a></li> 
</ul> 
<h2 style="margin-left:0; margin-right:0; text-align:left">功能简介</h2> 
<ul style="list-style-type:disc; margin-left:0; margin-right:0"> 
 <li>设计简洁，界面美观</li> 
 <li>采用<a href="https://gitee.com/link?target=https%3A%2F%2Fwww.markdownguide.org%2F">Markdown</a>/富文本双编辑器,支持一键插入视频、图片、附件等</li> 
 <li>支持多主题自由切换</li> 
 <li>强大的主题在线编辑功能,支持多种文件操作,实时生效,支持在网页直接创建新主题,快速开发页面</li> 
 <li>支持自定义页面</li> 
 <li>支持友情链接</li> 
 <li>支持附件管理,附件在线预览</li> 
 <li>支持扩展插件,编写java代码打包为jar,直接后台安装生效,丰富程序多样性</li> 
 <li>主题开发简单快速,多种自定义指令及api接口支持</li> 
 <li>支持邮件服务,评论信息邮件提醒</li> 
 <li>安装部署简单</li> 
 <li>支持<a href="https://gitee.com/link?target=https%3A%2F%2Fwww.mysql.com">mysql</a>/<a href="https://gitee.com/link?target=https%3A%2F%2Fwww.sqlite.org">sqlite</a>数据库</li> 
</ul> 
<h2 style="margin-left:0; margin-right:0; text-align:left">更新内容</h2> 
<ul> 
 <li style="text-align:start">新增: 插件可限定支持的数据库类型,如用户使用的数据库类型与插件限定不一致,则不允许安装插件</li> 
 <li style="text-align:start">新增: 插件相关API接口及获取所有插件的指令和判断插件是否启动的共享方法</li> 
 <li style="text-align:start">新增: 退出登录支持自定义重定向的页面</li> 
 <li style="text-align:start">新增: 插件websocket支持</li> 
 <li style="text-align:start">新增: 登录限制,同一账户在短时间内多次请求登录将会被锁定账户10分钟</li> 
 <li style="text-align:start">新增: 分页指令支持定义渲染html的class名</li> 
 <li style="text-align:start">修复: 网站设置页部分内容不会回显的问题</li> 
 <li style="text-align:start">修复: 当插件定义一些枚举类等不需要注入容器的类时,报错的问题</li> 
 <li style="text-align:start">修复: 当用户输入一次正确的验证码后,密码错误的情况下,验证码未刷新可能造成密码爆破的漏洞(感谢@treuhkgsdf)</li> 
 <li style="text-align:start">修复: 分类禁用后前台依然能获取到的问题</li> 
 <li style="text-align:start">优化: 主题编辑相关优化</li> 
</ul>
