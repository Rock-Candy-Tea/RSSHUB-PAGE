                                                                                            <p>Universal Media Server 是一个兼容 DLNA，UPnP 和 HTTP/S 的媒体服务器，支持所有主要操作系统，包括 Windows，Linux 和 macOS 版本，几乎不需要配置就可以流式传输或转码许多不同的媒体格式。</p> 
<p>Universal Media Server 10.9.0 发布，该版本更新内容如下：</p> 
<h3>通用：</h3> 
<ul> 
 <li> <p>增加了在网页界面下载文件夹作为播放列表的选项；</p> </li> 
 <li> <p>增加了在网页界面上将媒体作为播放列表下载的选项；</p> </li> 
 <li> <p>改变了 macOS 图标，以符合苹果公司对 Big Sur 的新设计准则；</p> <p><img alt height="199" src="https://static.oschina.net/uploads/space/2021/0812/071804_jExZ_4937141.png" width="700" referrerpolicy="no-referrer"></p> </li> 
 <li> <p>在网页界面上为媒体标题提供了更多空间；</p> <p><img alt="https://s3-us-west-2.amazonaws.com/secure.notion-static.com/35830102-7923-4089-9beb-66a63810af5e/Untitled.png" height="276" src="https://static.oschina.net/uploads/space/2021/0812/071816_pmFX_4937141.png" width="700" referrerpolicy="no-referrer"></p> </li> 
 <li> <p>增加了对 MP1 的识别；</p> </li> 
 <li> <p>改进了对蓝光、HDMV、DVB、WebVTT 和 EIA-608 字幕的支持；</p> </li> 
 <li> <p>修复了 SUSE Linux 上的崩溃问题；</p> </li> 
 <li> <p>修复了章节文件夹显示空白名称的问题；</p> </li> 
 <li> <p>多国语言的翻译更新；</p> </li> 
</ul> 
<h3>渲染器：</h3> 
<ul> 
 <li>改进了对 Google Chromecast Ultra 的支持；</li> 
 <li>改进了对 Android 上 MXPlayer 的支持；</li> 
 <li>改进了对 Windows Media Player 的支持；</li> 
 <li>修正了对 Naim-Mu-So、Roku 和三星设备上 OGG 格式的支持；</li> 
</ul> 
<h3>依赖项更新：</h3> 
<ul> 
 <li>将 checkstyle 8.44 升级至 8.45.1；</li> 
 <li>将 com.sun.xml.bind-version 3.0.1 升级至 3.0.2；</li> 
 <li>将 git-commit-id-plugin 4.0.5 升级至 4.9.10；</li> 
</ul>
