                                                        <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">DBeaver 是一个免费开源的通用数据库工具，适用于开发人员和数据库管理员。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">DBeaver 22.1 发布，更新内容如下：</p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>常规用户界面： 
  <ul style="margin-left:0; margin-right:0"> 
   <li>在偏好设置中增加了本地时区配置</li> 
   <li>连接类型的颜色现在会影响 SQL 编辑器面板</li> 
   <li>标签选择器（CTRL+E）现在支持快速/模糊搜索</li> 
   <li>项目重命名现在可以更新所有链接视图中的项目名称</li> 
   <li>解决了导航树刷新的问题</li> 
   <li>首选项页面现在遵循字体配置</li> 
   <li>针对低分辨率显示器的配置弹出窗口定位问题得到了解决</li> 
   <li>解决了嵌入式下拉式编辑器的问题（macOS）</li> 
   <li>修复了在某些 macOS 和 Linux 安装中的应用程序启动问题</li> 
   <li>重新设计了只读连接的切换键</li> 
   <li>在标签上下文菜单中添加了 "复制对象名称" 的操作</li> 
  </ul> </li> 
 <li>数据编辑器： 
  <ul style="margin-left:0; margin-right:0"> 
   <li>按范围着色现在支持前景色的定制</li> 
   <li>解决了 clob/json 弹出编辑器的问题</li> 
   <li>Epoch 时间转换器现在支持 4 种新模式</li> 
   <li>许多分析型数据库的 GROUP BY 查询生成问题已被修复</li> 
   <li>改进了日期/时间内联编辑器的用户界面</li> 
  </ul> </li> 
 <li>SQL 编辑器： 
  <ul style="margin-left:0; margin-right:0"> 
   <li>右键点击脚本文本现在可以改变光标位置</li> 
   <li>改进了部分分隔符元素的视觉效果</li> 
   <li>为 "删除此脚本" 动作添加了确认信息</li> 
   <li>修复了危险的查询执行确认配置</li> 
  </ul> </li> 
 <li>DB2：程序名称自动补全已被修复</li> 
 <li>HANA：修复了程序输出参数的获取</li> 
 <li>Netezza：增加了对区分大小写的用户名称的支持</li> 
 <li>Snowflake：数据库元数据读取性能得到明显改善</li> 
 <li>Teradata：增加了 SQL 查询限制（TOP）支持</li> 
 <li>许多小的 UI 错误修复</li> 
 <li>新的数据库驱动程序：Apache Kyuubi</li> 
</ul>
