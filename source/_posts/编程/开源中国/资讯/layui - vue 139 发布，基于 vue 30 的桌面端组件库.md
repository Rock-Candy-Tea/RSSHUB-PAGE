<p>[新增] switch 组件 name 原生属性。<br> [新增] input 组件 maxlength 属性, 原生属性限制输入长度。<br> [修复] textarea 组件 maxlength 属性, 限制内容长度不可用。<br> [修复] form 组件 position 属性 top 值的布局。<br> [修复] date-picker 构建之后 time 选择器错误。<br> [修复] table 组件 totalRow 属性, 计算 string 类型数字拼接的问题。<br> [修复] table 组件 columns 配置 fixed 属性开启时, 固定列无阴影的问题。<br> [修复] table 组件 filterColumns 布局。<br> [优化] select 组件 下拉 icon 图标。<br> [优化] cascader 组件 下拉 icon 图标。<br> [优化] select 组件 allow-clear 属性非必填。<br> [优化] switch 组件 disabled 状态效果。<br> [优化] input 组件 password 属性, 当 length 大于 0 时启用。<br> [优化] input 组件 input 事件, 参数由 event 调整为 value。<br> [优化] textarea 组件 input 事件, 参数由 event 调整为 value。<br> [优化] select 组件 multiple 属性为 true 时的 Search 样式。<br> [优化] panel 组件 css 样式。</p>
