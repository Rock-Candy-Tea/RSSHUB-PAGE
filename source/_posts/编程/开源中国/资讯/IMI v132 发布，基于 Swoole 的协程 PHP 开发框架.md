                                                        <p>基于 Swoole 的协程 PHP 开发框架 IMI v1.3.2 已经发布。</p> 
<h2>随便说说</h2> 
<p>目前 imi v2 版本已基本完成开发，正在进行文档编写工作。</p> 
<p>最近把官网、文档模版、所有组件都挪到了 imi 主仓库进行维护。</p> 
<p>以后所有的 issue、Pull Request 都在主仓库进行。</p> 
<hr> 
<p>imi v2 版本最新规划及动态: <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FYurunsoft%2Fimi%2Fissues%2F66" target="_blank">https://github.com/Yurunsoft/imi/issues/66</a></p> 
<p>如果有任何问题，欢迎联系我们！</p> 
<hr> 
<h2>更新日志</h2> 
<h3>优化</h3> 
<ul> 
 <li>优化 ConnectionBinder 使用 ServerBean</li> 
</ul> 
<h3>修复</h3> 
<ul> 
 <li> <p>修复模型保存更新问题</p> </li> 
 <li> <p>修复 Entity 注解 camel 设为 false 失效问题</p> </li> 
 <li> <p>修复关联模型问题</p> </li> 
 <li> <p>修复模型查询后的序列化问题</p> </li> 
 <li> <p>修复因心跳导致的，偶现连接跨协程使用问题</p> </li> 
 <li> <p>修复在用户进程中，向客户端推送数据有警告的问题</p> </li> 
 <li> <p>修复生成用注释表示的方法时，默认值中有数组时的格式问题</p> </li> 
 <li> <p>Fix typo (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FYurunsoft%2Fimi%2Fpull%2F95" target="_blank">#95</a>)</p> </li>
