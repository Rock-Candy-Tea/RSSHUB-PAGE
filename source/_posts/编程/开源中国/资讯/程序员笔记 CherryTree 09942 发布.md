                                                        <p>CherryTree 0.99.42 现已发布。CherryTree 是一个支持无限层级分类的笔记软件，Python 编写，支持富文本编辑和代码高亮，支持 Linux 和 Windows 平台。数据采用 sqlite 或 XML 存储，支持密码保护，支持从 NoteCase、KeepNote、Knowit、Tomboy、TuxCards、Treepad、Leo 等笔记软件导入数据。</p> 
<div>
 <img alt height="342" src="https://oscimg.oschina.net/oscnet/up-567a52bd8a28acd66c1970e07c82497720a.png" width="500" referrerpolicy="no-referrer">
</div> 
<div>
  
</div> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">此版本更新内容如下：</p> 
<ul> 
 <li>统一了节点名称+标签的搜索和节点内容的搜索 </li> 
 <li>修复了在多个节点中的搜索，可选择从选择中的第一个/所有范围中的第一个</li> 
 <li>修复了在 Windows 下受密码保护的数据提取和 non ascii 路径的问题</li> 
 <li>链接到节点插入/编辑对话框，增加了树中 anchor name 的搜索</li> 
 <li>实现了节点属性，从搜索中排除了节点和/或子节点</li> 
 <li>对文本的前景/背景应用颜色，使用默认的 gtk3 调色板</li> 
 <li>在首选对话框、标签主题中增加了选择 2 个新图标主题的可能性</li> 
 <li>增加了隐藏状态栏的选项</li> 
 <li>增加了对树中选定节点的背景/前景颜色的可配置性</li> 
 <li>树形资源管理器增加了显示连接 nodes parent-children 的视图选项</li> 
 <li>在首选对话框中增加了 tab fonts，按钮可以将字体重置为默认值</li> 
 <li>在 F8/当前日期节点，如果已经存在，则不访问年/月，只访问日</li> 
 <li>支持在非 Windows 操作系统中的便携模式，在 cherrytree 二进制可执行文件旁寻找 config/config.cfg</li> 
 <li>tree sorting menu items：移动 siblings 只排序第一和分离器后排序整个树</li> 
 <li>增加了实验性的选项 menubar_in_titlebar，可以在 config.cfg 中激活，但默认是关闭的</li> 
</ul>
