                                                                                            <p>Traefik 2.5.2 现已发布，该版本更新内容如下：</p> 
<p><strong>Bug fixes：</strong></p> 
<ul> 
 <li><strong>[http3] </strong>将 github.com/lucas-clemente/quic-go 升级到 v0.23.0 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ftraefik%2Ftraefik%2Fpull%2F8413" target="_blank">#8413</a>)</li> 
 <li><strong>[middleware] </strong>修复镜像中间件的 empty body 错误 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ftraefik%2Ftraefik%2Fpull%2F8381" target="_blank">#8381</a>)</li> 
 <li><strong>[tracing] </strong>将 go.elastic.co/apm 版本升级到 v1.13.1（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ftraefik%2Ftraefik%2Fpull%2F8399" target="_blank">#8399</a>）</li> 
 <li>更新 x/sys 以支持 go 1.17 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ftraefik%2Ftraefik%2Fpull%2F8368" target="_blank">#8368</a> )</li> 
 <li>将官方 Traefik 镜像的 Alpine docker 版本从 3.11 提升到 3.14</li> 
</ul> 
<p><strong>Documentation：</strong></p> 
<ul> 
 <li><strong>[k8s/ingress,k8s] </strong>为 v1 ingresses examples 添加 pathType（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ftraefik%2Ftraefik%2Fpull%2F8392" target="_blank">#8392</a>）</li> 
 <li>修复文档中的 http scheme URL（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ftraefik%2Ftraefik%2Fpull%2F8395" target="_blank">#8395</a>）</li> 
</ul>
