                                                                                            <p style="color:#333333; margin-left:0; margin-right:0; text-align:left">Ant Design Mobile 5.0.0-rc.18 发布了。Ant Design Mobile 即 Ant Design 移动端设计规范，<code>antd-mobile</code> 是 Ant Design 的移动规范的 React 实现，服务于蚂蚁及口碑无线业务。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><strong>更新内容如下：</strong></p> 
<ul> 
 <li>特性 
  <ul> 
   <li>Badge 增加了<span> </span><code>--color</code><span> </span>CSS 变量 (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design-mobile%2Fpull%2F4673" target="_blank">#4673</a>)</li> 
   <li>Cascader 和 CascaderView 增加了试验性的<span> </span><code>optionSkeleton</code><span> </span>常量，可以使用它在数据加载时显示骨架屏</li> 
  </ul> </li> 
 <li>优化 
  <ul> 
   <li>移除了之前为了兼容旧版本而存在的<span> </span><code>Search</code><span> </span>别名，现在请明确的使用<span> </span><code>SearchBar</code></li> 
   <li>调整了 Dropdown 的箭头样式</li> 
  </ul> </li> 
 <li>修复 
  <ul> 
   <li>修复了 Input 的<span> </span><code>inputMode</code><span> </span>属性没有生效的 bug</li> 
   <li>修复了 Popover 当没有设置<span> </span><code>trigger</code><span> </span>时，默认会变为 hover 触发的问题，现在如果你不传<span> </span><code>trigger</code><span> </span>属性，那么 Popover 不会自动在点击的时候弹出了</li> 
   <li>调整了编译逻辑，修复了部分项目中会出现<span> </span><code>regeneratorRuntime is not defined</code><span> </span>报错的问题</li> 
   <li>修复了 Ellipsis 在内容或其他属性变化时，显示内容可能不会更新的问题</li> 
  </ul> </li> 
</ul> 
<p>更新说明：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design-mobile%2Freleases%2Ftag%2Fv5.0.0-rc.18" target="_blank">https://github.com/ant-design/ant-design-mobile/releases/tag/v5.0.0-rc.18</a></p>
