                                                                                            <p>Zettlr 是一个基于 Electron 开发的开源跨平台 Markdown 编辑器，支持 macOS、Windows 和大多数 Linux 发行版。日前，Zettlr 2.1.2 正式发布，具体更新内容如下：</p> 
<h3>GUI 和功能</h3> 
<ul> 
 <li>新功能：现在可以更精细地控制文件的显示方式，可以选择是否总是使用文件名、或使用标题、或第一个 H1</li> 
 <li>新功能：现在可以折叠嵌套列表</li> 
 <li>新功能：现在可以选择显示 Markdown 的文件扩展名</li> 
 <li>新功能：可以选择总是只用文件名来链接</li> 
 <li>编辑器的 Vim 输入模式又可以使用了，在这次更新中重新启用了它。</li> 
 <li>修正了一个错误，该错误会导致全局搜索失灵</li> 
 <li>删除了一个未使用的偏好设置</li> 
 <li>重新排列了一些偏好设置</li> 
 <li>在 Windows 上，如果窗口变得太窄，标签式窗口会自动隐藏其标签</li> 
 <li>恢复了关于用户可以在 Zettelkasten generator 中使用哪些变量的信息</li> 
 <li>Zettlr 重新在一些偏好设置下面显示信息文本</li> 
 <li>Citations 现在具有自己的偏好设置标签</li> 
 <li>修复了重命名文件时会关闭其他文件的错误</li> 
 <li>修复了在 macOS 上进行全文搜索期间不显示上下文菜单的问题</li> 
 <li>如果在打开新的工作区或根文件期间出现问题，错误消息现在将给出更有意义的提示</li> 
 <li>修复了改变主题或通过自定义 CSS 修改编辑器的字体后，换行缩进会看起来不一样的问题</li> 
 <li>修复了 vim 模式光标在黑暗模式下几乎不可见的问题</li> 
 <li>已完成的任务列表项目现在会更快地被划掉</li> 
 <li>……</li> 
</ul>
