                                                                                            <p>Opera 是一个多平台网络浏览器，由其同名公司 Opera 开发。Opera 如今是一个基于 Chromium 的浏览器。它通过其用户界面和其他功能将自己与其他浏览器区分开来。</p> 
<p>Opera 90 正式发布，该版本基于 Chromium 104，具体更新内容如下：</p> 
<p><img alt height="394" src="https://static.oschina.net/uploads/space/2022/0823/071015_JpHg_4937141.jpg" width="700" referrerpolicy="no-referrer"></p> 
<p>该版本是一个小幅度的版本更新，主要更新内容包括优化调整、功能改进、更新翻译等。</p> 
<ul> 
 <li>将 Opera 90 升级到稳定版</li> 
 <li>在中国，将 360 设置为默认搜索引擎</li> 
 <li>更新 Chromium 至 104.0.5112.81</li> 
 <li>移除 flow client_capabilities 中的空字符串</li> 
 <li>[同步设置] 确认您的身份以启用加密信息的 flickers</li> 
 <li>让 webpack 编译的错误出现在日志中</li> 
 <li>禁用的 Pinboards 应该在账户弹出窗口有另一种颜色</li> 
 <li>同步 —— 在 <a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fauth.opera.com%2Faccount%2Fv3%2Fdesktop%2Flogin%2Fconfirm-password" target="_blank">auth.opera.com/account/v3/desktop/login/confirm-password</a> 上点击下一步并没有重定向到任何地方</li> 
 <li>在 Opera 账户弹出窗口中添加同步状态</li> 
 <li>创建私人 API 来打开账户弹出窗口</li> 
 <li>使用基于类别的建议列表对搜索框建议进行排序</li> 
 <li>无法从 JavaScript 中关闭一个窗口</li> 
 <li>chrome.operaAccountPrivate 是未定义的</li> 
 <li>"强制黑暗页面" 快捷键不起作用</li> 
 <li>地址栏的输入内容在标签之间共享</li> 
 <li>[AdBlock] 为 URL 过滤规则实现框架层次结构遍历</li> 
 <li>[AdBlock] 为元素隐藏规则实施框架层次结构遍历</li> 
 <li>Opera 在附加到标签条滚动按钮时崩溃了</li> 
 <li>移除 DNS-over-HTTPS UI 元素</li> 
 <li>修复翻译字符串中的 & 转义问题</li> 
</ul> 
<p>更多详情可查看：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fblogs.opera.com%2Fdesktop%2Fchangelog-for-90%2F" target="_blank">https://blogs.opera.com/desktop/changelog-for-90/</a></p>
