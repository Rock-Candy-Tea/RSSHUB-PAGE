                                                        <p>Freespire 是一个社区驱动的 Linux 发行版，它源于 Linspire。Freespire 1.0 基于 Debian，而 Freespire 2.0 及后续版本基于 Ubuntu。</p> 
<p>近日，Freespire 8.0 正式发布，该版本重点引入了多项来自 Google 的服务。Freespire 的核心没有任何改变：它仍然是一个全功能的桌面操作系统，所有的应用程序和 Ubuntu 软件库的资源都一如既往地可用。</p> 
<p>Freespire 8.0 更新内容包括：</p> 
<ul> 
 <li>XFCE Desktop 4.16</li> 
 <li>Linux Kernel 5.4.0-91 LTS</li> 
 <li>Google Chrome 96</li> 
 <li>Ice SSB</li> 
 <li>Kpatience</li> 
 <li>Dreamchess</li> 
 <li>Gmail</li> 
 <li>Google Docs</li> 
 <li>Google Drive</li> 
 <li>Google Calendar</li> 
 <li>Google Translate</li> 
 <li>Google News</li> 
 <li>Kolourpaint</li> 
 <li>Video Player</li> 
 <li>Rhythmbox</li> 
</ul> 
<p>除了上述内容，Freespire 8.0 还包括其他一些安全和功能更新。要创建一个 USB 密钥，可以使用以下程序：</p> 
<ul> 
 <li>Ubuntu Startup Disk Creator (Ubuntu、Linux Mint、Freespire、ElementaryOS)</li> 
 <li>Unetbootin (Linux、Windows)</li> 
 <li>Balena Etcher (Linux、Mac、Windows)</li> 
 <li>Rufus (Windows)</li> 
</ul>
