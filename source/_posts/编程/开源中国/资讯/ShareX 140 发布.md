                                                                                            <p>ShareX 是一个开源截图工具，可捕获或记录屏幕的任何区域，并一键共享。 它还允许将图像、文本或其他类型的文件上传到超过 80 个支持的存储服务上。</p> 
<p>ShareX 14.0 正式发布，更新内容如下：</p> 
<ul> 
 <li>支持带有更大指针尺寸的屏幕捕捉光标</li> 
 <li>增加图片编辑器的缩放支持</li> 
 <li>在图像编辑器画布尺寸窗口中增加了画布颜色按钮</li> 
 <li>增加了 "限制区域捕捉和光标在活动显示器内 "选项</li> 
 <li>为区域捕捉/图像编辑器增加了 FPS 限制选项，默认值为 100，您可以通过设置值为 0 来禁用限制。</li> 
 <li>允许步骤注释工具的值为 0</li> 
 <li>OCR 改进： 
  <ul> 
   <li>增加了离线 OCR 支持（需要 Windows 10 或更新的版本才能使用）</li> 
   <li>如果当前操作系统不支持 OCR，ShareX 将显示消息框</li> 
   <li>将 OCR 按钮从捕获菜单移至工具菜单</li> 
   <li>增加选项，以在应用 OCR 之前调整图像输入的大小，从而改善 OCR 结果</li> 
   <li>增加了可定制的服务链接，默认情况下这些服务是可用的：Google 翻译、Google 搜索、必应</li> 
   <li>在 OCR 窗口中增加了 "选择区域" 按钮，使你可以从屏幕上进行 OCR，而不必重新打开工具。</li> 
  </ul> </li> 
 <li>全屏图像预览的改进 
  <ul> 
   <li>支持使用左/右方向键、鼠标滚轮或点击两侧的导航按钮进行导航</li> 
   <li>在顶部显示文件名、图像大小和图像索引</li> 
   <li>增加了 "图像查看器" 工具</li> 
   <li>增加了 <code>ImageViewer "filePath"</code> CLI 命令</li> 
  </ul> </li> 
 <li>历史窗口的改进： 
  <ul> 
   <li>打字时自动搜索</li> 
   <li>当数值改变时自动应用过滤器</li> 
   <li>过滤后选择第一个项目</li> 
   <li>删除了历史窗口中的最大项目数设置</li> 
   <li>将高级搜索面板移至右侧并增加了关闭按钮</li> 
  </ul> </li> 
 <li>通知窗口的改进 
  <ul> 
   <li>增加了在屏幕捕捉/记录时自动隐藏通知窗口的选项，这样当快速连续拍摄多张屏幕截图时，通知窗口就不会在任何屏幕截图中出现。</li> 
   <li>只显示最后一个通知，而不是显示相互之间的多个通知</li> 
  </ul> </li> 
 <li>在所有的 FFmpeg 参数中增加了 <code>-hide_banner</code></li> 
 <li>删除了对 PortableApps 的支持，改用 GitHub Release 页上的 Portable 构建。</li> 
 <li>在更新信息框和下载器窗口中使用当前主题</li> 
 <li>为 x264 视频编码器增加了比特率选项</li> 
 <li>刷新设备列表时支持 FFmpeg 5.0</li> 
 <li>删除了 Streamable 匿名上传选项，因为 Streamable 不再支持</li> 
 <li>删除了 VideoBin，因为他们的 API 不再可用</li> 
</ul> 
<p>更多详情可查看：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgetsharex.com%2Fchangelog%2F" target="_blank">https://getsharex.com/changelog/</a></p>
