<div class="content">
                                                                                            <p><span style="background-color:#ffffff; color:#000000">DBeaver 21.3.1 现已发布。DBeaver 是一个免费开源的通用数据库工具，适用于开发人员和数据库管理员。</span></p> 
<p><span style="background-color:#ffffff; color:#000000"><strong>主要更新内容</strong></span></p> 
<ul> 
 <li><span style="background-color:#ffffff; color:#000000">SQL编辑器</span> 
  <ul> 
   <li><span style="background-color:#ffffff; color:#000000">没有别名的表的自动完成是固定的</span></li> 
   <li><span style="background-color:#ffffff; color:#000000">添加了脚本文件名模板支持</span></li> 
  </ul> </li> 
 <li><span style="background-color:#ffffff; color:#000000">数据编辑器</span> 
  <ul> 
   <li><span style="background-color:#ffffff; color:#000000">改进了列过滤器编辑对话框 UI</span></li> 
  </ul> </li> 
 <li><span style="background-color:#ffffff; color:#000000">资源管理器</span> 
  <ul> 
   <li><span style="background-color:#ffffff; color:#000000">跨项目连接拖放已修复</span></li> 
  </ul> </li> 
 <li><span style="background-color:#ffffff; color:#000000">任务管理器</span> 
  <ul> 
   <li><span style="background-color:#ffffff; color:#000000">已修复读取的任务统计信息</span></li> 
  </ul> </li> 
 <li><span style="background-color:#ffffff; color:#000000">SQL 查询生成器</span> 
  <ul> 
   <li><span style="background-color:#ffffff; color:#000000">字符串值引用已修复</span></li> 
  </ul> </li> 
 <li><span style="background-color:#ffffff; color:#000000">驱动程序管理器</span> 
  <ul> 
   <li><span style="background-color:#ffffff; color:#000000">改进了 Maven 工件导入/编辑 UI</span></li> 
  </ul> </li> 
 <li><span style="background-color:#ffffff; color:#000000">连接锁密码支持已修复</span></li> 
 <li><span style="background-color:#ffffff; color:#000000">导航</span> 
  <ul> 
   <li><span style="background-color:#ffffff; color:#000000">对象过滤器现在支持反斜杠转义</span></li> 
   <li><span style="background-color:#ffffff; color:#000000">修复了多个小的 UI 错误</span></li> 
  </ul> </li> 
 <li><span style="background-color:#ffffff; color:#000000">DB2</span> 
  <ul> 
   <li><span style="background-color:#ffffff; color:#000000">添加了表组织属性</span></li> 
  </ul> </li> 
 <li><span style="background-color:#ffffff; color:#000000">PostgreSQL、Greenplum、Redshift：</span> 
  <ul> 
   <li><span style="background-color:#ffffff; color:#000000">权限编辑器 UI 已修复（冗余 GRANT/REVOKE 命令合并）</span></li> 
   <li><span style="background-color:#ffffff; color:#000000">复合数据类型支持已修复</span></li> 
   <li><span style="background-color:#ffffff; color:#000000">执行计划渲染已修复（"并行" 扫描）</span></li> 
   <li><span style="background-color:#ffffff; color:#000000">分区表统计读取已修复</span></li> 
  </ul> </li> 
</ul>
