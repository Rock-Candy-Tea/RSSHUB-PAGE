                                                                                            <p>FileZilla 是一个免费和开源的跨平台 FTP 应用程序，由 FileZilla Client 和 FileZilla Server 组成。客户端可用于 Windows、Linux 和 macOS，服务器仅可用于 Windows。服务器和客户端都支持 FTP 和 FTPS（FTP over SSL/TLS），而客户端还可以连接到 SFTP 服务器。</p> 
<p>FileZilla Client 3.58.0 正式发布，更新内容包括：</p> 
<h3>新功能：</h3> 
<ul> 
 <li>在 TLS 握手过程中，与 FileZilla Server >= 1.3.0 协商自定义 ALPN，以在连接建立过程中减少一些往返次数。</li> 
 <li>准备让 FileZilla 在未来的 FileZilla Server 版本上支持基于 TOTP 的两步认证。</li> 
</ul> 
<h3>错误修复和小改动：</h3> 
<ul> 
 <li>如果一个已知存在的 XML 文件无法被打开，现在有更详细的错误信息</li> 
 <li>在网络连接不可靠的情况下，使下载更新更稳定</li> 
 <li>更新至 libfilezilla 0.36.0</li> 
</ul> 
<p>更多详情可查看：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Ffilezilla-project.org%2F" target="_blank">https://filezilla-project.org/</a></p>
