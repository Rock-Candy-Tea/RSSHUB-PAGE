                                                                                            <p>PowerShell Core 是一个跨平台（Windows、Linux 和 macOS）的自动化和配置工具/框架，可以很好地与你现有的工具配合，并为处理结构化数据（如 JSON、CSV、XML 等）、REST API 和对象模型而优化。它包括一个命令行 Shell、一种相关的脚本语言和一个处理 cmdlets 的框架。</p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">PowerShell v7.2.2 发布，更新内容如下：</p> 
<p style="text-align:start"><strong>Build and Packaging Improvements</strong></p> 
<ul> 
 <li>更新 WiX 文件（Internal 19460）</li> 
 <li>将 .NET SDK 版本更新到 6.0.201（Internal 19457）</li> 
 <li>更新实验性功能的 JSON 文件 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FPowerShell%2FPowerShell%2Fpull%2F16838" target="_blank">#16838</a> )</li> 
 <li>确保 Alpine 和 ARM SKU 的<code>powershell.config.json</code>文件已启用了实验性功能 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FPowerShell%2FPowerShell%2Fpull%2F16823" target="_blank">#16823</a> )</li> 
 <li>更新 macOS 构建的<code>vmImage</code>和 PowerShell 根目录 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FPowerShell%2FPowerShell%2Fpull%2F16611" target="_blank">#16611</a> )</li> 
 <li>更新 macOS 构建镜像和构建根文件夹 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FPowerShell%2FPowerShell%2Fpull%2F16609" target="_blank">#16609</a> )</li> 
 <li>删除 WiX install ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FPowerShell%2FPowerShell%2Fpull%2F16834" target="_blank">#16834</a> )</li> 
 <li>选择构建安全监控 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FPowerShell%2FPowerShell%2Fpull%2F16911" target="_blank">#16911</a> )</li> 
 <li>为发布包添加 SBOM 清单（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FPowerShell%2FPowerShell%2Fpull%2F16641" target="_blank">#16641</a>，<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FPowerShell%2FPowerShell%2Fpull%2F16711" target="_blank">#16711</a>）</li> 
 <li>添加用于打包的 Linux 包依赖项 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FPowerShell%2FPowerShell%2Fpull%2F16807" target="_blank">#16807</a> )</li> 
 <li>切换到自定义镜像进行构建和发布（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FPowerShell%2FPowerShell%2Fpull%2F16801" target="_blank">#16801</a>，<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FPowerShell%2FPowerShell%2Fpull%2F16580" target="_blank">#16580</a>）</li> 
 <li>删除此 repo 中构建的对<code>cmake</code>的所有引用（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FPowerShell%2FPowerShell%2Fpull%2F16578" target="_blank">#16578</a>）</li>
