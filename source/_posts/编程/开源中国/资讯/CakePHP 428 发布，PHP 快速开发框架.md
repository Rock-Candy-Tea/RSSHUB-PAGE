                                                                                            <p>CakePHP 是一个运用了诸如 ActiveRecord、Association Data Mapping、Front Controller 和 MVC（model–view–controller） 等著名设计模式的开源 Web 框架。CakePHP 用 PHP 编写，以 Ruby on Rails 的概念为模型，并在 MIT 许可下进行分发。</p> 
<p>CakePHP 4.2.8 现已发布，这是 4.2 分支的一个维护版本，修正了几个社区报告的问题。</p> 
<p>更新内容如下：</p> 
<p><strong>Bugfixes</strong></p> 
<ul> 
 <li>当涉及传递的参数时，Paginator scopes 现在可以正确生成 URL。</li> 
 <li>现在对于插件中的元素，<code>MissingTemplateException</code>中的搜索路径列表是正确的。</li> 
 <li>改进了<code>Query::applyOptions()</code>的文档。</li> 
</ul>
