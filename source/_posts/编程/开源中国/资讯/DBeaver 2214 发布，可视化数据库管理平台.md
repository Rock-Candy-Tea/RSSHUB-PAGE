                                                                                            <p>DBeaver 是一个免费开源的通用数据库工具，适用于开发人员和数据库管理员。</p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">DBeaver 22.1.4 现已发布，更新内容如下：</p> 
<ul style="margin-left:0; margin-right:1.5em"> 
 <li>Data editor： 
  <ul style="margin-left:0; margin-right:0"> 
   <li>改进了危险操作检测器（删除...返回）</li> 
   <li>已修复 sorted rows 的导出问题</li> 
   <li>改进了状态消息格式和 UI</li> 
   <li>按枚举值过滤已修复</li> 
   <li>Collection editor 得到改进（值面板中的 UI）</li> 
   <li>修复了统计标签打开时的问题</li> 
  </ul> </li> 
 <li>Console view： 
  <ul style="margin-left:0; margin-right:0"> 
   <li>添加了非 SELECT 查询支持</li> 
   <li>错误消息现在打印在控制台中</li> 
  </ul> </li> 
 <li>Driver editor：添加了将属性值设置为 NULL 的可能性</li> 
 <li>SQL 格式设置保存和预览已修复</li> 
 <li>数据传输：任务设置加载已修复</li> 
 <li>Azure SQL Server、Babelfish：重新设计了多数据库支持</li> 
 <li>Clickhouse： 
  <ul style="margin-left:0; margin-right:0"> 
   <li>修复了旧服务器版本的 Table list load</li> 
   <li>改进了跨数据库数据传输的数据类型映射</li> 
   <li>添加了多行插入支持</li> 
   <li>64 位整数支持已修复</li> 
  </ul> </li> 
 <li>H2：表元数据和 DDL 读取已修复</li> 
 <li>PostgreSQL、CocroachDB： 
  <ul style="margin-left:0; margin-right:0"> 
   <li>添加了行安全策略支持</li> 
   <li>PgPass 身份验证现在支持 SSH 主机和主机重定义</li> 
   <li>例程权限 DDL 已修复</li> 
  </ul> </li> 
 <li>General UI： 
  <ul style="margin-left:0; margin-right:0"> 
   <li>默认嵌入式浏览器已更改为 Microsoft Edge</li> 
   <li>Properties editor：焦点更改时保存的值已修复</li> 
   <li>修复了每日提示对话框布局</li> 
   <li>修复了许多小的 UI 错误</li> 
  </ul> </li> 
</ul>
