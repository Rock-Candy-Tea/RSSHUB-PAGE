                                                        <p>iTerm2 是一个开源的 macOS 终端模拟器。它源于早期的 iTerm 应用程序。iTerm2 支持操作系统功能，如窗口透明度、全屏模式、分割窗格、Exposé 标签、Growl 通知和标准键盘快捷键。其他功能包括可定制的配置文件和终端输入/输出的即时回放。</p> 
<p>iTerm2 v3.5.0 Beta 3 正式发布，该版本更新内容如下：</p> 
<h3>新功能</h3> 
<ul> 
 <li>增加对 120Hz 的支持。</li> 
 <li>增加对刘海屏的支持。</li> 
 <li>只要 TERM 以<code>xterm</code>开头，功能键（Function keys）现在可以发送修改标志。</li> 
 <li>对 "modifyOtherKeys" 模式的兼容性改进。</li> 
 <li>HTML 日志现在支持双下划线。</li> 
 <li>增加了一个 "compose"的操作（prefs > shortcuts > actions）。</li> 
 <li>现在你可以在密码管理器的设置菜单中禁用探测密码的提示了。</li> 
 <li>垂直线和水平线光标现在有一个细微的阴影，以提高其可见度。</li> 
 <li>在标签上下文菜单中添加 "在右边添加标签"选项。</li> 
 <li>有一个新的实用工具 —— it2tip，它将每日提示作为一个命令行应用来实现。安装 shell 集成和实用程序来获得它。</li> 
 <li>你现在可以将 "选择菜单项" 绑定到指针事件</li> 
 <li>使菜单项可以在键绑定和指针绑定中搜索。</li> 
 <li>……</li> 
</ul> 
<h3>其他改进</h3> 
<ul> 
 <li>在没有打开搜索栏的情况下执行查找下一个/上一个，现在将突出显示所有搜索结果。</li> 
 <li>Emacs 和 Python 现在得到了正确的标签图标。</li> 
 <li>使 Monterey 中的标签栏看起来更漂亮。</li> 
 <li>调整了命令历史的排序。</li> 
 <li>对 Toolbelt 的视觉改进。</li> 
 <li>启用安全键盘输入时，对 macOS 12 的变化发出警告。</li> 
 <li>启用安全键盘输入时，在当前会话中显示一个指示器。</li> 
 <li>……</li> 
</ul> 
<h3>错误修正：</h3> 
<ul> 
 <li>修复了在启用原生全屏的情况下，cmd-enter 不会为窗口切换全屏的错误。</li> 
 <li>修复调整大小后被破坏的滚动区域。</li> 
 <li>解决旧版 tmux 的一个 bug，即不像其他键那样接受 control-space。</li> 
 <li>为 tmux 1.8 修复 tmux 集成。</li> 
 <li>当在另一个应用的全屏窗口上打开时，在菜单栏下面打开热键窗口。</li> 
 <li>修复拖动标签的外观。</li> 
 <li>修复黑暗模式下全屏时，屏幕顶部闪烁的白线。</li> 
 <li>修正 GPU 和非 GPU 渲染器在混合情况下的差异。</li> 
 <li>修正了一个错误，即每个窗格和每个窗口的 gbackground iamge 可能同时可见。</li> 
 <li>修正在 GPU 渲染器与传统渲染器中对组成字符的渲染差异。</li> 
 <li>当 "启用全键盘访问" 在系统预设中被打开时，iTerm2 现在接受空格键作为输入。</li> 
 <li>热键窗口现在出现在通知中心信息的下方。</li> 
 <li>……</li> 
</ul>
