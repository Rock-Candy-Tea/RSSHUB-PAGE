                                                                                            <h3>新功能</h3> 
<ul> 
 <li>更安全的连接：Firefox 现在可以使用 HTTPS RR 作为 Alt-Svc headers 自动升级到 HTTPS</li> 
</ul> 
<p>注：Alt-Svc 全称为 “Alternative-Service”，直译为“备选服务”。该 headers 列举了当前站点备选的访问方式列表。一般用于在提供 “QUIC” 等新兴协议支持的同时，实现向下兼容。</p> 
<ul> 
 <li>在许多系统上的视频播放现在支持全范围的色彩水平（0-255）</li> 
 <li>Mac 用户现在可以从 Firefox 文件菜单中访问 macOS 共享选项</li> 
 <li>在 macOS 上启用了对包含 ICC v4 配置文件的图像的支持</li> 
 <li>默认支持 AVIF 图像格式（推迟到 Firefox 93）</li> 
</ul> 
<h3>修复</h3> 
<ul> 
 <li>如果 Thunderbird 在 Firefox 之后被安装或更新，那么 Firefox 的屏幕阅读器和其他无障碍工具的性能不再严重下降</li> 
 <li>macOS VoiceOver 现在可以正确报告使用 aria-expanded 属性标记为 "expanded" 的按钮和链接</li> 
 <li>一个标签中的 open alert 不再导致使用同一进程的其他标签出现性能问题</li> 
 <li>各种安全问题的修复</li> 
</ul> 
<h3>变化：</h3> 
<ul> 
 <li>macOS 上的书签工具栏菜单现在遵循 Firefox 的视觉风格</li> 
 <li>证书错误页面已被重新设计，以获得更好的用户体验</li> 
 <li>重构 Firefox 的 JavaScript 内存管理，以提高性能和使用更少的内存</li> 
</ul>
