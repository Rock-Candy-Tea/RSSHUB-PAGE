                                                                                            <p>VASSAL Engine 3.6.0-beta1 版本现已发布。VASSAL 是一个游戏引擎，用于在线构建棋盘游戏和纸牌游戏，构建的游戏可在 Internet 上或通过电子邮件实时运行。VASSAL Engine 可在所有平台上运行，并且是免费的开源软件。</p> 
<p><strong>主要更新内容</strong></p> 
<ul> 
 <li>新特性 
  <ul> 
   <li>如果当前正在绘制计数器详细信息查看器，则公开属性</li> 
   <li>图层和非矩形的缩放系数</li> 
   <li>为 Mass Piece Loader 添加了选项，以便在没有匹配图像时不在图层中创建空级别</li> 
   <li>在模块、地图、甲板、初始堆栈、原型和全局属性文件夹下添加了子文件夹以对组件进行分组</li> 
   <li>添加热键以隐藏和显示地图窗口。现有的显示/隐藏热键字段现在始终可见。允许更改隐藏/显示地图按钮设置而无需重新启动编辑器</li> 
   <li>库存窗口允许刷新热键</li> 
   <li>编辑器现在支持拖放</li> 
   <li>可以在编辑器中复制和插入组件，而不仅仅是添加到底部</li> 
   <li>CountStack() 添加到 beanshell 以计算当前堆栈中的件数</li> 
   <li>ReportState 复选框覆盖对单个报告的全局键命令抑制</li> 
   <li>添加 Deck Sort Key Command 组件以允许对 Deck 中的卡片进行排序</li> 
  </ul> </li> 
 <li>bug 修复 
  <ul> 
   <li>Mac 上添加禁用 OpenGL FBO 的选项</li> 
   <li>双倍最大堆并在图像拼贴器内存不足时重试</li> 
   <li>在编辑 Piece Slot 时，Can Rotate 特性不再丢失任何预定义的起始旋转</li> 
   <li>DeckPosition 报告的位置比实际位置高 1</li> 
   <li>修复和改进模块有效性检查</li> 
   <li>当网格大小非常大时，网格编号对话框可能会显示为空白</li> 
  </ul> </li> 
 <li>其它改进 
  <ul> 
   <li>添加删除/弃用检查器并删除一年前弃用的任何内容</li> 
   <li>编辑 IrregularGrid 时，在网格或其父区域附近的某处启动地图视图</li> 
   <li>不会将新区域自动命名为长字符串</li> 
   <li>适用于 Windows 和 Mac 的 64 位 ARM 版本：将捆绑的 Java 切换到 Zulu</li> 
   <li>升级到 Launch4j 3.14</li> 
   <li>新添加的图层特性现在将默认为“始终处于活动状态”</li> 
   <li>从复选框首选项字符串中删除 ?s</li> 
   <li>在需要时将滚动条添加到首选项对话框</li> 
  </ul> </li> 
</ul>
