                                                        <p>Apache Jackrabbit Oak 1.22.7 现已发布。Jackrabbit Oak 是一种可扩展的高性能分层内容存储库，旨在用作现代世界级网站和其它要求苛刻的内容应用程序的基础。</p> 
<p><strong>主要更新内容</strong></p> 
<ul> 
 <li>修复 BasicDocumentStoreTest.testLongId 在 MongoDB 4.2.0 中测试失败的问题</li> 
 <li>修复 1.8 分支的提交不会为  parents/root 设置 "_bc" 的问题</li> 
 <li>修复 DocumentNodeStore 可能会累积拆分候选项的问题</li> 
 <li>从 nt:frozenNode 定义中删除 mix:referenceable</li> 
 <li>在升级和迁移时遵循 nt:frozenNode的定义</li> 
 <li>重构检查可参考 nt:frozenNode 的定义</li> 
 <li>将 SystemPropertySupplier 从文档中移至 commons</li> 
 <li>更新到 MongoDB Java 驱动程序 3.12</li> 
 <li>使用 SystemPropertySupplier 简化向后移植</li> 
</ul>
