  <p><span style="color:#000000">Ionic Framework 是一个开源的移动应用程序开发框架，拥有强大的跨平台 UI 工具包，通过使用 HTML、CSS 和 JavaScript 就能够构建高质量原生的 iOS、Android 和 Progressive Web App。</span></p> 
  <p><span style="color:#000000">Ionic Framework 6.0.3 正式发布，本次更新内容如下：</span></p> 
  <h3><strong>Bug 修复</strong></h3> 
  <ul> 
   <li><strong>angular-server：</strong>使用正确的 @ionic/angular 依赖版本 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fionic-team%2Fionic-framework%2Fissues%2F24593" target="_blank">#24593</a> )</li> 
   <li><strong>angular:</strong>应用 <span style="color:#24292f">touch, dirty 和 pristine </span>的表单控制类 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fionic-team%2Fionic-framework%2Fissues%2F24558" target="_blank">#24558</a> ) </li> 
   <li><strong>datetime：</strong><span style="color:#2e3033">在 ios 15上显示日历网格，不再导致月份切换</span> ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fionic-team%2Fionic-framework%2Fissues%2F24554" target="_blank">#24554</a> ) </li> 
   <li><strong>item:</strong> 修复 Safari 中可见的错误槽 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fionic-team%2Fionic-framework%2Fissues%2F24579" target="_blank">#24579</a> ) </li> 
   <li><strong>menu: </strong>彻底移除本应该在 v5 版本中删除的主要属性<strong> ( </strong><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fionic-team%2Fionic-framework%2Fissues%2F24565" target="_blank">#24565</a> ) </li> 
   <li><strong>modal：</strong>修复控制器模式的生命周期事件 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fionic-team%2Fionic-framework%2Fissues%2F24508" target="_blank">#24508</a> ) </li> 
   <li><strong>overlays：</strong>getTop 现在会返回最顶部呈现的叠加层 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fionic-team%2Fionic-framework%2Fissues%2F24547" target="_blank">#24547</a> ) </li> 
   <li><strong>react：</strong>将 useRef 包装器添加到 useIonOverlay 状态，以避免过时引用 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fionic-team%2Fionic-framework%2Fissues%2F24553" target="_blank">#24553</a> )</li> 
   <li><strong>react：</strong>在删除包含元素（containing element）后，防止内联弹出框的错误 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fionic-team%2Fionic-framework%2Fissues%2F24569" target="_blank">#24569</a> ) </li> 
  </ul>
