                                                                                            <p>PikaScript v1.7.0 已经发布，轻量级跨平台嵌入式 Python 引擎。</p> 
<p>此版本更新内容包括：</p> 
<ol> 
 <li>在 C 模块的 py 方法声明中使用 self, 更好地支持 pylance 智能提示。</li> 
 <li>在 REPL 中支持定义函数。</li> 
 <li>在 REPL 中支持定义类。</li> 
 <li>增强了 REPL 的稳定性，增强了错误提示。</li> 
 <li>支持 list 和 dict 的 built-in 模式（需要配置）。</li> 
 <li>增加了 ctypes 模块，提供类似 cpython 的 ctypes 兼容。</li> 
 <li>脚本中支持了 import as, from import, from import as 语法。</li> 
 <li>增强了预编译器的稳定性，修复了一些 bug。</li> 
 <li>支持 bytes 基本数据类型。</li> 
 <li>支持 s = '\x33\x44' 字面值转义。</li> 
 <li>支持 b = b'\0x00\x03\x04' 创建 bytes 变量。</li> 
</ol> 
<p>详情查看：<a href="https://gitee.com/Lyon1998/pikascript/releases/v1.7.0">https://gitee.com/Lyon1998/pikascript/releases/v1.7.0</a></p>
