                                                                                            <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">VS Code Go 是 Golang 团队官方推出的一个 VS Code 扩展，为 Go 编程语言提供了丰富的语言支持。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">VS Code Go v0.35 近日正式发布，更新内容如下：</p> 
<h3 style="margin-left:.6em; margin-right:0; text-align:start">特性</h3> 
<ul style="margin-left:0; margin-right:0"> 
 <li>该版本增加了对 inlay 提示的支持</li> 
 <li>增加在扩展设置中对<span> </span><code>$&#123;workspaceFolderBasename&#125;</code><span> </span>替换的逻辑支持</li> 
 <li>增加对多文件测试套件的支持</li> 
 <li>增加对自定义格式器的支持</li> 
</ul> 
<h3 style="margin-left:.6em; margin-right:0; text-align:start">错误修复</h3> 
<ul style="margin-left:0; margin-right:0"> 
 <li>修复了（Issue<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fgolang%2Fvscode-go%2Fissues%2F%232339" target="_blank">2339</a>），该问题在 arm64 Mac 上无法更新 delve</li> 
 <li>禁用了实验性的 gopls 功能（如语义高亮）</li> 
</ul> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">更多详情可查看：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fgolang%2Fvscode-go%2Freleases" target="_blank">https://github.com/golang/vscode-go/releases</a></p>
