<div class="content">
                                                                                            <p>PHP 8.1.1 现已<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.php.net%2Farchive%2F2021.php%232021-12-17-1" target="_blank">发布</a>，这是一个<span style="color:#333333">错误修复版本</span>；<span style="color:#333333">官方鼓励所有 PHP 8.1 用户升级到此版本。具体更新内容如下：</span></p> 
<ul> 
 <li>IMAP： 
  <ul> 
   <li>修复了<span style="background-color:#ffffff; color:#333333"> bug </span><a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fbugs.php.net%2F81649" target="_blank">#81649</a>（imap_(un)delete 接受序列，而不是单一数字）。</li> 
  </ul> </li> 
 <li>PCRE： 
  <ul> 
   <li>将捆绑的 PCRE2 更新到 10.39。</li> 
   <li>修复了<span style="background-color:#ffffff; color:#333333"> bug </span><a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fbugs.php.net%2F74604" target="_blank">#74604</a>（php_pcre_replace_impl 中的越界）。</li> 
  </ul> </li> 
 <li>Standard： 
  <ul> 
   <li>修复了<span style="background-color:#ffffff; color:#333333"> bug </span><a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fbugs.php.net%2F81659" target="_blank">#81659</a>（stream_get_contents() 可能会不必要地过度分配）。</li> 
  </ul> </li> 
</ul> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">有关 PHP 8.1.1 的源码下载，可访问<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.php.net%2Fdownloads.php" target="_blank">下载页面</a>，Windows 源码和二进制文件可在 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwindows.php.net%2Fdownload%2F" target="_blank">windows.php.net/download/</a> 找到。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">更新说明：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.php.net%2FChangeLog-8.php%238.1.1" target="_blank">https://www.php.net/ChangeLog-8.php#8.1.1</a></p>
