                                                                                            <p>Ionic 是一个强大的跨平台 UI 工具包，通过使用 HTML、CSS 和 JavaScript 构建原生质量的 iOS、Android 和渐进式 Web 应用程序。Ionic 基于 Web 组件，可显著提升性能、可用性和功能，同时支持 Angular、React 和 Vue 等流行 Web 框架。</p> 
<p>Ionic 6 是目前最新的版本，于去年 12 月正式<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fionicframework.com%2Fblog%2Fannouncing-ionic-6%2F" target="_blank">推出</a>，值得关注的新特性包括：改进桌面支持、彻底优化组件，以及更改在 iOS 和 Android 上的设计。</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-54995e00fa28d63237bf5bb0def38e66478.png" referrerpolicy="no-referrer"></p>
