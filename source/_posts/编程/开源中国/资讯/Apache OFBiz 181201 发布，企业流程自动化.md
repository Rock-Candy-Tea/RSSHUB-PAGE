                                                                                            <p>Apache OFBiz 18.12.01 现已发布。<span style="background-color:#ffffff; color:#333333">Apache OFBiz 是用于企业流程自动化的开源产品，包括框架组件和业务应用程序。</span></p> 
<p><strong><span style="background-color:#ffffff; color:#333333">主要更新内容</span></strong></p> 
<ul> 
 <li>Subtask 
  <ul> 
   <li>任何电子商务用户都可以通过 "忘记密码" 重置他人的密码（包括管理员）</li> 
   <li>创建补丁以删除框架中的 IDEAL 代码</li> 
   <li>让电子商务组件反映 iDEAL 的新情况</li> 
   <li>添加会话跟踪模式并使 cookie 安全</li> 
   <li>删除对电子商务的 ftl 依赖</li> 
   <li>按字母顺序排列 UI 标签</li> 
  </ul> </li> 
 <li>Bug 
  <ul> 
   <li>电子商务电子邮件中的双斜杠</li> 
   <li>从采购发票创建会计交易时未正确处理税费</li> 
   <li>JSON 响应未在出错时设置 http 状态</li> 
   <li>createAcctgTransForPurchaseInvoice 服务中缺少 invoiceTaxTotal 值</li> 
   <li>添加组订单会产生错误</li> 
   <li>在 3.0 版文件中找不到元素 "web-app" 的声明</li> 
   <li>删除 Scrum 成员时出错</li> 
  </ul> </li> 
 <li>新功能 
  <ul> 
   <li>多部分输入参数在 Groovy 事件中不可用</li> 
   <li>存款价格支持</li> 
   <li>引入一种快速方法，用于为目录中的任何特定产品添加与客户的销售价格协议</li> 
   <li>管理 ModelService 上已弃用的服务</li> 
   <li>基于令牌的身份验证</li> 
   <li>实施和应用返回查找</li> 
   <li>为 zip 管理添加新的 FileUtil 方法</li> 
   <li>使用代发货流程协议</li> 
   <li>用自动登录身份验证从一个域导航到另一个域</li> 
  </ul> </li> 
 <li>改进 
  <ul> 
   <li>Product Promo Worker 描述补丁</li> 
   <li>minilang 中的“始终”日志级别被记录为 FATAL</li> 
   <li>用小部件替换 webtools/control/view/ModelInduceFromDb</li> 
   <li>让 QRCodeServices.java 使用 EntityUtilProperties</li> 
   <li>Minilang 代码可读性和一致性改进</li> 
   <li>如果要在文件中呈现的列表为空，则不应呈现空标题行</li> 
   <li>工作量协议应用程序重定向到错误的页面并给出错误</li> 
   <li>添加选项以在编辑设施方角色屏幕上更新设施方角色</li> 
  </ul> </li> 
</ul>
