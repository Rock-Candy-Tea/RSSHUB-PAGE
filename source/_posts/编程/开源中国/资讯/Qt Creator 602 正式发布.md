                                                                                            <p>Qt Creator 6.0.2 正式发布，本次更新带来以下变化和修复：</p> 
<h3>通用：</h3> 
<ul> 
 <li>修复了进程启动器的崩溃（QTCREATORBUG-26726）</li> 
</ul> 
<h3>编辑</h3> 
<ul> 
 <li>修正了 "选择全部" 滚动到底部的问题(QTCREATORBUG-26736)</li> 
 <li>修正了带有块选择的复制问题 (QTCREATORBUG-26761)</li> 
</ul> 
<h3>C++</h3> 
<ul> 
 <li>ClangCodeModel 
  <ul> 
   <li>修正了 Windows 和 macOS 上代码补全的性能退步问题 (QTCREATORBUG-26754)</li> 
  </ul> </li> 
</ul> 
<h3>Python</h3> 
<ul> 
 <li>修复了 "REPL" 的工作目录</li> 
</ul> 
<h3>Modeling</h3> 
<ul> 
 <li>修正了属性编辑器中丢失的选项 (QTCREATORBUG-26760)</li> 
</ul> 
<h3>项目</h3> 
<ul> 
 <li>修正了在 "应用程序输出" 窗格中关闭应用程序会杀进程的问题，即使选择了 "保持运行"，也会杀进程。</li> 
 <li>修正了目标设置页面的过滤问题 (QTCREATORBUG-26779)</li> 
</ul> 
<h3>CMake</h3> 
<ul> 
 <li>修正了 GUI 项目向导在 Windows 上不能创建 GUI 应用程序，在 macOS 上不能创建应用程序包的问题</li> 
</ul> 
<h3>Linux</h3> 
<ul> 
 <li>修正了商业插件链接到 libGLX 和 libOpenGL 的问题 (qtcreatorbug-26744)</li> 
</ul> 
<h3>macOS</h3> 
<ul> 
 <li>修正了切换屏幕配置时的崩溃问题 (QTCREATORBUG-26019)</li> 
</ul>
