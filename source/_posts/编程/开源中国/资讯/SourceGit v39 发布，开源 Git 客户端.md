                                                        <p>开源的 Git 客户端 SourceGit v3.9 已经发布，此版本更新内容包括：</p> 
<p>[CHANGELOG]</p> 
<ol> 
 <li>支持使用配置的第三方工具查看Diff</li> 
 <li>增加WinMerge工具支持</li> 
 <li>优化内存使用</li> 
 <li>增加快捷键<br> 4.1. <code>CTRL + TAB</code> 标签页轮换<br> 4.2. <code>CTRL + T</code> 打开新标签页<br> 4.3. <code>CTRL + W</code> 关闭当前标签页<br> 4.4. <code>CTRL + [1-9]</code> 切换到指定标签页<br> 4.5. <code>CTRL + F</code> 如果当前页为仓库页，打开搜索<br> 4.6. <code>F5</code> 如果当前页为仓库页，刷新</li> 
 <li>增加<code>git subtree</code>第一版实现</li> 
</ol> 
<h1>SourceGit</h1> 
<p>开源的Git客户端，仅用于Windows。</p> 
<h2>下载</h2> 
<p><a href="https://gitee.com/sourcegit/SourceGit/releases/">发行版</a></p> 
<ul> 
 <li><code>SourceGit.exe</code>为不带.NET 5.0运行时的可执行文件，需要先安装.NET 5</li> 
 <li><code>SourceGit_48.exe</code>为.NET 4.8编译的可执行文件，Window 10 已内置该运行时</li> 
</ul>
