                                                        <p>restfx 0.16.0 已经发布，这是一个适用于 Python 3 的 restful 多应用自动路由框架。</p> 
<p>此版本更新内容包括：</p> 
<ul> 
 <li>添加 路由参数添加枚举类型支持</li> 
 <li>优化 FileResponse 支持按范围返回数据 (216)</li> 
 <li>优化 在 FileResponse 指定 ranges 时，判断 fp 的类型是否为 IO</li> 
 <li>优化 装饰器中参数的解析</li> 
 <li>优化 错误消息提示</li> 
 <li>优化代码</li> 
 <li>优化 简化一些内置的 response 对象名称</li> 
 <li>优化 将所有内置的 response 对象都使用 Response 结尾</li> 
 <li>优化 校验添加为空判断支持</li> 
 <li>修复 FileResponse 中存在的问题</li> 
 <li>修复 HttpFile 解析错误</li> 
 <li>修复 执行参数校验后结果处理错误，导致路由调用失败的问题</li> 
 <li>修复 未指定校验器时会报错的问题</li> 
</ul> 
<p>详情查看：<a href="https://gitee.com/hyjiacan/restfx/releases/0.16.0">https://gitee.com/hyjiacan/restfx/releases/0.16.0</a></p>
