 <p>VLOOK™ 是针对 ![Typora] <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.typora.io%2F" target="_blank">Typora</a>[^Typora] （跨平台 Markdown 编辑器）的 <strong>主题包</strong> 和 <strong>增强插件</strong>（针对导出的 HTML 文件)。</p>

<blockquote> 
 <p>VLOOK™ 属于开源软件（遵从 <strong>MIT License</strong>），也是 ![OSChina]<strong><a href="https://www.oschina.net/p/vlook">开源中国</a></strong> 推荐的国产开源产品、Typora 的首个增强插件。</p>

<p style="text-align:left"><strong>VLOOK™ 的所有特性清单，详见快速入坑 → 一键进入 ( <a href="https://madmaxchow.gitee.io/vlook/index.html#%E5%BF%AB%E9%80%9F%E5%85%A5%E5%9D%91">Gitee</a> / <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fmadmaxchow.github.io%2FVLOOK%2Findex.html%23%25E5%25BF%25AB%25E9%2580%259F%25E5%2585%25A5%25E5%259D%2591" target="_blank">GitHub</a> )</strong></p> 
<p style="text-align:left"><strong>V10.7 版本为各位 Markdown 粉带以下新的惊喜！！！</strong></p> 
<h2 style="text-align:left">★ NEW! 插图/表格/代码块 自动开启「画中画」特性</h2> 
<ul> 
 <li>在文档右下角浮动显示，支持缩放和原比例显示</li> 
</ul> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-ac80a5465ee54be5f639b3771b9c8145722.png" referrerpolicy="no-referrer"></p> 
<h2 style="text-align:left">★ NEW! 全新的内容助手</h2> 
<ul> 
 <li>鼠标在插图、表格、代码块悬停激活，集成内容相关操作，更便捷</li> 
</ul> 
<h2 style="text-align:left">★ NEW! 支持「无封面」模式</h2> 
<ul> 
 <li>适用于简单文档发布，或集成到网站内等场景</li> 
</ul> 
<h2 style="text-align:left">★ NEW! 支持文档字数及阅读时长统计</h2> 
<ul> 
 <li>鼠标悬停后扩展显示CJK字数、非CJK字数</li> 
</ul> 
<hr> 
<h3 style="text-align:left">— What’s IMPROVED —</h3> 
<ul> 
 <li><code>章节导航</code> 增加文档标题，支持点击回到封面</li> 
 <li><code>导航中心</code> 分类索引搜索增加对表格、代码块、SVG内文本的匹配</li> 
 <li>完善「文艺范」字体主题配置，交互动画体验细节</li> 
 <li>完善演示辅助工具分组样式</li> 
 <li>完善页内跳转的微调处理</li> 
</ul> 
<h3 style="text-align:left">— What’s CHANGED —</h3> 
<ul> 
 <li><code>表格</code> 十字光标的开/关操作调整到内容助手中</li> 
 <li><code>插图</code> 点击D 插图浏览器中打开的操作调整到内容助手上</li> 
</ul> 
<h3 style="text-align:left">— What’s FIXED —</h3> 
<ul> 
 <li><code>逐章导航</code> 向前一章为封面时跳转失败</li> 
 <li><code>彩虹引用</code> 引用内 H6 指定为 theme1 的样式错误</li> 
</ul>
