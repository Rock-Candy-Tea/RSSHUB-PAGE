                                                        <p>Taichi（太极）0.7.25 已经发布，这是专为高性能计算机图形学设计的编程语言。<img alt src="https://attachments.tower.im/tower/43963cc20256e177dcb41b2ef5370a8f?version=auto" referrerpolicy="no-referrer"></p> 
<p><img src="https://static.oschina.net/uploads/space/2021/0611/070135_jaaS_2744687.gif" referrerpolicy="no-referrer"></p> 
<p>此版本具体更新内容如下：</p> 
<ul> 
 <li>[ci] 修复 docker prebuilt binary link ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ftaichi-dev%2Ftaichi%2Fpull%2F2483" target="_blank">#2483</a> )</li> 
 <li>[wasm] 为 wasm 后端实现 materialize() 并清理未使用的函数（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ftaichi-dev%2Ftaichi%2Fpull%2F2480" target="_blank">#2480</a>）</li> 
 <li>[misc] 统一 std::filesystem header ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ftaichi-dev%2Ftaichi%2Fpull%2F2478" target="_blank">#2478</a> ) </li> 
 <li>[opengl] 当 print_kernel_llvm_ir 为 true 时转储计算着色器源代码 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ftaichi-dev%2Ftaichi%2Fpull%2F2479" target="_blank">#2479</a> ))</li> 
 <li>[metal] 修复 randseedoffset_in_runtime_buffer ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ftaichi-dev%2Ftaichi%2Fpull%2F2477" target="_blank">#2477</a> ) </li> 
 <li>[Metal] 在 Metal 上支持指针 SNode ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ftaichi-dev%2Ftaichi%2Fpull%2F2441" target="_blank">#2441</a> )</li> 
 <li>[wasm] Recover code（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ftaichi-dev%2Ftaichi%2Fpull%2F2476" target="_blank">#2476</a>）</li> 
 <li>[AutoDiff] 自动判断 AdStack 的大小 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ftaichi-dev%2Ftaichi%2Fpull%2F2438" target="_blank">#2438</a> )</li> 
 <li>[misc] 在 cmake 中添加对 gitpython 的检查（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ftaichi-dev%2Ftaichi%2Fpull%2F2473" target="_blank">#2473</a>）</li> 
 <li>[ci] 将 slash-command-dispatch 限制为仅 PR ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ftaichi-dev%2Ftaichi%2Fpull%2F2472" target="_blank">#2472</a> )</li> 
 <li>[ir] 使用 SNodeTree 实现 root ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ftaichi-dev%2Ftaichi%2Fpull%2F2449" target="_blank">#2449</a> ) </li> 
 <li>[Lang] 修复 ti.maybe_transform_ti_func_call_to_stmt() 和 ti.external_func_call() 之间的参数冲突 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ftaichi-dev%2Ftaichi%2Fpull%2F2470" target="_blank">#2470</a> )</li> 
</ul>
