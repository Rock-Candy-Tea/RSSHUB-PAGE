                                                                                            <p>Ubuntu Budgie 是一个基于 Budgie 桌面环境的 Ubuntu 官方派生发行版，首个版本于 2016 年 4 月以 budgie-remix 的名称发布。从 16.10 版开始，Ubuntu Budgie 正式加入 Ubuntu 官方派生版之列。Budgie 桌面环境用 C 和 Vala 编写。</p> 
<p><img alt height="396" src="https://oscimg.oschina.net/oscnet/up-3eb022f3177057c2f4e2656ba137ec8d3cc.png" width="700" referrerpolicy="no-referrer"></p> 
<h3>更新内容如下：</h3> 
<ul> 
 <li> <p>Linux 5.15 LTS</p> </li> 
 <li> <p>Budgie 10.6.1</p> </li> 
 <li> <p>Budgie 控制中心取代了 GNOME 控制中心</p> </li> 
 <li> <p>新的 Chrome OS 布局</p> <p><img alt height="407" src="https://oscimg.oschina.net/oscnet/up-26cd3f34d22c22f3923b0fa6d1eb973725a.png" width="700" referrerpolicy="no-referrer"></p> </li> 
 <li> <p>更新了 WhiteSur 图标和 GTK 主题以匹配其上游版本</p> </li> 
 <li> <p>Budgie 欢迎工具现在同时具有 Brave 浏览器和微软 Edge 的安装选项</p> </li> 
 <li> <p>支持与 Evolution 和 Thunderbird 电子邮件客户端的桌面通知集成</p> </li> 
 <li> <p>顶部面板的外观变化</p> </li> 
 <li> <p>Firefox Snap 取代了 Firefox deb</p> </li> 
 <li> <p>Mesa 22</p> </li> 
 <li> <p>Firefox 99.x</p> </li> 
 <li> <p>LibreOffice 7.3.x</p> </li> 
 <li> <p>Nemo 文件管理器 5.2.4</p> </li> 
 <li> <p>Tilix 终端 1.2.4</p> </li> 
 <li> <p>Gedit 文本编辑器 41.0</p> </li> 
 <li> <p>从这个版本开始支持 RISC-V 64 架构</p> </li> 
</ul> 
<p><strong>这可能是最后一个带有 Budgie 桌面的 Ubuntu Budgie LTS 版本，在未来几年该团队将转向 EFL 桌面。</strong></p>
