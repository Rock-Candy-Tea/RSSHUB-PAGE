                                                                                            <p style="margin-left:0">Microsoft PowerToys 是 Windows 系统实用程序，供高级用户调整和简化其 Windows 体验，可最大限度地提高生产力。</p> 
<p style="margin-left:0">PowerToys 0.56.2 是一个补丁版本，用于修复 v0.56.1 中的问题，以修复一些官方认为对基于传入速率的稳定性很重要的错误。具体更新内容如下：</p> 
<ul> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmicrosoft%2FPowerToys%2Fissues%2F16252" target="_blank">#16252</a>：现在为安装程序传递 /norestart flag 给 .NET 运行时安装程序</li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmicrosoft%2FPowerToys%2Fissues%2F16155" target="_blank">#16155</a> : FancyZones 模板布局重置错误已修复</li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmicrosoft%2FPowerToys%2Fissues%2F16699" target="_blank">#16699</a>：使用中的开发文件预览处理文件的错误已被修复</li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmicrosoft%2FPowerToys%2Fissues%2F16669" target="_blank">#16669</a>：为 shake distance 增加了“Find my mouse”设置，以便于调整</li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmicrosoft%2FPowerToys%2Fissues%2F16712" target="_blank">#16712</a>：PowerToys Run 的 Windows 终端插件的本地化设置不正确，现在是正确的。</li> 
</ul>
