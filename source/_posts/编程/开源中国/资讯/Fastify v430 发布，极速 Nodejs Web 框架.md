                                                                                            <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">Fastify 是一个高度专注于以最少开销和强大的插件架构为开发者提供最佳体验的 Node.js Web 框架，速度极快，它的灵感来源于 Hapi 和 Express。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">Fastify 4.3.0 发布了，本此更新带来以下内容：</p> 
<ul> 
 <li>不缓存不必要的内容类型 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffastify%2Ffastify%2Fpull%2F4134" target="_blank">#4134</a></li> 
 <li>修复：默认 clientError 回复重用连接（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffastify%2Ffastify%2Fissues%2F4101" target="_blank">#4101</a>）<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffastify%2Ffastify%2Fpull%2F4133" target="_blank">#4133</a></li> 
 <li>docs(ecosystem): 添加电子服务器 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffastify%2Ffastify%2Fpull%2F4136" target="_blank">#4136</a></li> 
 <li>功能：通过请求和回复公开验证/序列化功能 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffastify%2Ffastify%2Fpull%2F3970" target="_blank">#3970</a></li> 
 <li><code>FastifyListenOptions</code>类型：在顶级类型中重新导出 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffastify%2Ffastify%2Fpull%2F4135" target="_blank">#4135</a></li> 
 <li>文档：删除 http2 实验状态（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffastify%2Ffastify%2Fissues%2F4142" target="_blank">#4142</a>） <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffastify%2Ffastify%2Fpull%2F4144" target="_blank">#4144</a></li> 
 <li>重构：重命名<code>request.validate</code>为<code>request.validateInput </code><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffastify%2Ffastify%2Fpull%2F4139" target="_blank">#4139</a></li> 
 <li>修复 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffastify%2Ffastify%2Fissues%2F4120" target="_blank">#4120</a>：将 FastifyRequestType 的解析推迟到 FastifyRequest <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffastify%2Ffastify%2Fpull%2F4123" target="_blank">#4123</a></li> 
</ul>
