                                                                                            <p>qBittorrent 4.3.5 现已正式发布。qBittorrent 是一款轻量高速的免费 BT 下载客户端软件，旨在打造成开源版的 μTorrent 。它支持所有主要平台（Linux、Mac OS X、Windows、OS/2 与 FreeBSD），并提供相同的功能。</p> 
<p><strong>更新日志：</strong></p> 
<ul> 
 <li>BUGFIX：在“Download from URLs”对话框中自动填充 URL/hash 值时，将光标移到末尾</li> 
 <li>BUGFIX：将无效的 QDateTime 值排序在有效值之后</li> 
 <li>BUGFIX：修复“Edit trackers”对话框中的 tabChangesFocus 属性</li> 
 <li>BUGFIX：更新 DynDNS 注册网址</li> 
 <li>BUGFIX：更轻松地处理“磁盘空间不足”错误</li> 
 <li>BUGFIX：使用样式表正确绘制进度背景</li> 
 <li>WEBUI：修复搜索工具中的 magnet url</li> 
 <li>WEBUI：修订文件夹监视功能</li> 
 <li>WEBUI：修复浏览器中的 magnet url</li> 
 <li>WEBUI：允许在 torrents/files API 中指定文件索引</li> 
 <li>WINDOWS: NSIS: 允许翻译更多字符串</li> 
 <li>WINDOWS: NSIS: 更新意大利语、德语、爱沙尼亚语、俄语、葡萄牙语的翻译</li> 
 <li>LINUX：修复 D-Bus Notification desktop-entry 字段 </li> 
 <li>MACOS：不要将可执行文件名称用作 CFBundleName 值</li> 
 <li>OTHER：将 Qt 要求降低到 5.11</li> 
 <li>OTHER：明确许可为 GPLv2 +</li> 
</ul> 
<p>更新说明：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.qbittorrent.org%2Fnews.php" target="_blank">https://www.qbittorrent.org/news.php</a></p>
