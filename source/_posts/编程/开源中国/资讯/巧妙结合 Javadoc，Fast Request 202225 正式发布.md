                                                                                            <p><strong>简介</strong></p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">Restful Fast Request 是一个类似于 Postman 的 IDEA 插件。它是一个强大的 restful api 工具包插件，可以根据已有的方法帮助您快速生成 url 和 params。 Restful Fast Request = API 调试工具 + API 管理工具 + API 搜索工具。 它有一个漂亮的界面来完成请求、检查服务器响应、存储你的 api 请求和导出 api 请求。插件帮助你在 IDEA 界面内更快更高效得调试你的 API。</p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><strong>Fast Request 为简化API调试而</strong></p> 
<p style="margin-left:0px; margin-right:0px; text-align:left"><span style="color:#1abc9c"><strong>倾听用户的声音，不断提升自我</strong></span>，本次Fast Request更新主要内容如下：</p> 
<ul> 
 <li><span style="color:#3498db"><strong><span style="background-color:var(--badge-tip-color)">添加Console控制台</span></strong></span></li> 
 <li><span style="color:#3498db"><strong><span style="background-color:var(--badge-tip-color)">从Javadoc读取注释</span></strong></span></li> 
 <li><span style="background-color:var(--badge-tip-color); color:var(--white)">添加打印语法支持</span></li> 
 <li><span style="background-color:var(--badge-tip-color); color:var(--white)">内置变量rfr添加currentProjectName、currentEnvName属性</span></li> 
 <li><span style="background-color:var(--badge-tip-color); color:var(--white)">@RequestPart支持</span></li> 
 <li><span style="background-color:var(--badge-tip-color); color:var(--white)">新增推特</span></li> 
 <li><span style="background-color:var(--badge-danger-color); color:var(--white)">后置脚本在请求成功但响应是401不执行</span></li> 
</ul>
