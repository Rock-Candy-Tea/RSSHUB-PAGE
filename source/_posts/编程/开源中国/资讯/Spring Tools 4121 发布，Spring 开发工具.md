                                                                                            <p>Spring Tools 4.12.1 现已发布。<span style="background-color:#ffffff; color:#333333">Spring Tools 4 是由 Spring 团队打造的 Spring 开发工具，从零开始构建，融合了现代技术和开发者工具架构。它在单独的进程中运行，从构建之初就考虑到了性能问题，并且支持最新的 Spring 技术，为开发基于 Spring 的企业应用提供世界级支持。同时，全新版本的 Spring Tools 与 IDE 无关，可在各种编码环境中使用，支持 Eclipse、Visual Studio Code 与 Theia。</span></p> 
<p><strong>主要更新内容</strong></p> 
<ul> 
 <li>嵌入式运行时升级到 JDK17（跨所有发行版，包括 Apple Silicon）</li> 
 <li>提供 Java 17 支持（通过单独的附加组件）</li> 
 <li>为即将发布的 Eclipse 2021-12 版本提供早期访问构建</li> 
 <li>提供 Apple Silicon 平台 (ARM M1) 的早期访问版本</li> 
 <li><span style="background-color:#ffffff; color:#182026">修复打开 sts 时总是下载源代码和 javadoc 的问题</span></li> 
 <li><span style="background-color:#ffffff; color:#182026">修复 docker 集成不适用于 Apple Silicon 的问题</span></li> 
 <li><span style="background-color:#ffffff; color:#182026">修复无法访问 CF 目标时，每次 Eclipse 启动都会出现错误消息的问题</span></li> 
</ul> 
<p><span style="color:#182026">此外，</span>内部 Web 浏览器在 macOS 上打开 HTTP URL 存在问题，其会显示来自操作系统的错误弹出窗口，而不是打开 HTTP URL。解决此问题的最佳方法是通过首选项 -> 常规 -> Web 浏览器在首选项中切换到外部浏览器，然后选择使用外部 Web 浏览器。</p>
