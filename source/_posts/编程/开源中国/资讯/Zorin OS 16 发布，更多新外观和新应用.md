                                                                                            <p>Zorin OS 16 现已发布。Zorin OS 16 基于 Ubuntu 20.04.2 LTS ，使用定制版本的 GNOME Shell 3.38 作为其桌面界面，并配备了 Linux 5.11 内核，预先填充了大量应用程序、工具。该 Zorin OS 16 在 2025 年 4 月之前都将支持软件更新和安全补丁。</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-aff32cca3c4cd80057d963a24b9ebad2387.JPEG" referrerpolicy="no-referrer"></p> 
<h4>全新的 GTK 主题</h4> 
<p>新的浅色主题（也有深色版本）辅以大胆的蓝色强调色，使用阴影和圆角，默认带有动态壁纸，且用户可以通过从六种强调色中选择一种来个性化主题。</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-edfc3da7a1ff5a3ad75221238e26ae8e36c.JPEG" referrerpolicy="no-referrer"></p> 
<h4>新的桌面布局</h4> 
<p>Zorin OS 16 Pro 提供了新的桌面布局供用户选择，包括受即将推出的 Windows 11 启发的桌面布局，用户可通过 Zorin Appearance 工具一键实时更改桌面布局。</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-bc57da250f675c9a1a1b191440603d33290.JPEG" referrerpolicy="no-referrer"></p> 
<h4>果冻模式</h4> 
<p>首次运行的新用户会通过运行 Tour 来引导进入 Zorin OS 体验，加上新的触摸板手势，使多任务处理变得更容易。此外，用户可以在 Zorin Appearance 中打开“果冻模式”。</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-113f0dae5776cc60dae4c70125a11262da3.gif" referrerpolicy="no-referrer"></p> 
<h4>更快的运行速度</h4> 
<p>尽管使用了许多相同的组件，但在 Zorin OS 16 中与 Ubuntu 20.04 LTS 相比联系更加紧密。此外，速度一直是 Zorin OS 16 的重中之重，因此桌面在各种新旧硬件上的运行速度都非常快。从内核到桌面环境，操作系统的各个层面都进行了性能优化。应用程序打开速度更快，动画更流畅，加载时间减少。</p> 
<p>而 Ubuntu 20.04 LTS 基础则意味着 Zorin OS 继承了最新 LTS 的所有优点，例如无闪烁启动、实验性分数缩放支持、QR 代码 Wi-Fi 热点、Active Directory 集成以及创建恢复密钥和加密安装。</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-122ccff8681b9a0f0b331d92b854e1f07b8.JPEG" referrerpolicy="no-referrer"></p> 
<h4>软件</h4> 
<p>Zorin OS 16 提供了对比 Ubuntu、elementary 甚至 Linux Mint 更广泛的“开箱即用”软件的访问。用户可以从以下位置安装应用程序：</p> 
<ul> 
 <li>Ubuntu 存储库</li> 
 <li>Snap 商店</li> 
 <li>Flathub</li> 
</ul> 
<p>此外，用户可以安装 .deb 和 .appimage 类型的软件，并使用 PPA 和其他第三方存储库，而无需先安装其他软件包。并且当用户尝试在 Zorin OS 16 中运行 .exe 软件时，会显示一个提示：如果本机 Linux 版本可用，该提示会让用户了解 WINE。</p>
