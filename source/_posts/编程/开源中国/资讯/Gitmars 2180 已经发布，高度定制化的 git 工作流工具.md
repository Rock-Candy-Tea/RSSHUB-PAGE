                                                                                            <p>Gitmars 2.18.0 已经发布，高度定制化的 git 工作流工具</p> 
<p>此版本更新内容包括：</p> 
<ol> 
 <li>新增<code>gitm alias</code>指令，初始化：<code>git alias init</code>，移除：<code>git alias remove</code></li> 
 <li>新增别名指令使用gitmars，可使用<code>git mars xxx</code> 或 <code>git flow xxx</code>，均等价于<code>gitm xxx</code></li> 
 <li>config配置增加<code>apis</code>，原<code>config.api</code>迁移到<code>config.apis.userInfo</code></li> 
 <li>新增<code>config.apis.buildConfig</code>接口配置，用于读取构建配置，优先级高于<code>config.apolloConfig</code></li> 
 <li>升级依赖包</li> 
</ol> 
<p>详情查看：<a href="https://gitee.com/saqqdy/gitmars/releases/2.18.0">https://gitee.com/saqqdy/gitmars/releases/2.18.0</a></p>
