                                                                                            <p><span style="background-color:#ffffff; color:#40485b">PerfreeBlog是一款基于Java SpringBoot开发的博客建站平台,支持多主题及扩展插件功能,给您带来全新的创作体验</span></p> 
<ul> 
 <li>官网地址:<span> </span><a href="https://gitee.com/link?target=http%3A%2F%2Fperfree.org.cn">http://perfree.org.cn</a></li> 
 <li>文档地址:<span> </span><a href="http://perfree.gitee.io/">http://perfree.gitee.io</a></li> 
</ul> 
<h2 style="margin-left:0; margin-right:0; text-align:left">功能简介</h2> 
<ul> 
 <li>设计简洁，界面美观</li> 
 <li>采用<a href="https://gitee.com/link?target=https%3A%2F%2Fwww.markdownguide.org%2F">Markdown</a>/富文本双编辑器,支持一键插入视频、图片、附件等</li> 
 <li>支持多主题自由切换</li> 
 <li>强大的主题在线编辑功能,支持多种文件操作,实时生效,支持在网页直接创建新主题,快速开发页面</li> 
 <li>支持自定义页面</li> 
 <li>支持友情链接</li> 
 <li>支持附件管理,附件在线预览</li> 
 <li>支持扩展插件,编写java代码打包为jar,直接后台安装生效,丰富程序多样性</li> 
 <li>主题开发简单快速,多种自定义指令及api接口支持</li> 
 <li>支持邮件服务,评论信息邮件提醒</li> 
 <li>安装部署简单</li> 
 <li>支持<a href="https://gitee.com/link?target=https%3A%2F%2Fwww.mysql.com">mysql</a>/<a href="https://gitee.com/link?target=https%3A%2F%2Fwww.sqlite.org">sqlite</a>数据库</li> 
</ul> 
<h2 style="margin-left:0; margin-right:0; text-align:left">更新内容</h2> 
<ul> 
 <li>新增: markdown/富文本编辑器支持粘贴图片自动上传</li> 
 <li>新增: 网站可设置自动将网页提交至百度</li> 
 <li>新增: 插件支持禁用和启用操作</li> 
 <li>新增: 文章/页面编辑器ctrl+s保存发布</li> 
 <li>新增: 可自定义全局Head代码(方便配置百度统计,谷歌广告等)</li> 
 <li>新增: 优化当插件版本相同时不能再次安装,当需要安装的插件版本小于已经安装的版本不能安装,当版本号大于已安装的插件执行更新方法</li> 
 <li>新增: 支持定义最低需要的PerfreeBlog版本,如达不到最低要求,则不允许安装</li> 
 <li>新增: 插件新增一系列代理事件类</li> 
 <li>新增: 支持categories分类页面,新增categoriesPage指令</li> 
 <li>新增: 支持tags标签页面,新增tagsPage指令</li> 
 <li>新增: 设置邮箱服务时支持发送测试邮件</li> 
 <li>新增: 评论邮件/找回密码邮件支持自定义模板</li> 
 <li>新增: 登录时验证码可设置开启或关闭</li> 
 <li>新增: 评论频率限制可在后台进行设置</li> 
 <li>修复: 主题在线预览时,如果有空的a标签绑定了事件,点击会自动跳转首页的问题</li> 
 <li>修复: 最新评论获取到其他自定义文章类型导致500问题</li> 
 <li>修复: 找回密码->重置密码失败的问题</li> 
 <li>调整: 将cdn资源移至项目内部</li> 
</ul>
