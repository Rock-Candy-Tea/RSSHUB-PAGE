                                                        <p>Kangaroo v1.7.1.210331 已经发布，此版本更新内容包括：</p> 
<h2>新特性或修复的缺陷列表</h2> 
<ol> 
 <li>实现十六进制编辑器支持 (已完成)</li> 
 <li>实现工作空间页样式支持 (已完成)</li> 
 <li>增加可独立查看日志支持 (已完成)</li> 
 <li>增加界面快捷键定制支持 (已完成)</li> 
 <li>增加数据图表支持 (前期开发中)</li> 
 <li>数据库模型化支持 (前期开发中) 
  <ul> 
   <li>支持模型文件加载和保存功能</li> 
   <li>支持模型节点复制与粘贴功能</li> 
   <li>支持模型节点多选功能</li> 
  </ul> </li> 
 <li>支持拖放 SQLite 数据库到连接首页并自动打开 #76</li> 
 <li>重构应用首页，方便用户操作</li> 
 <li>支持复制数据为 LUA 表结构</li> 
 <li>改进对象导航树操作体验</li> 
 <li>支持保存和恢复工作空间界面布局功能，缓存正在编辑的内容</li> 
 <li>增加更多语言支持</li> 
 <li>改进日志支持（日志消息中增加数据库名和连接名）</li> 
 <li>增加SQL脚本文件监测支持</li> 
 <li>重构数据表视图支持显示特性工具栏</li> 
 <li>打开连接时默认展示全部数据表</li> 
 <li>重构对象树视图改进对象树显示效果</li> 
 <li>增加快捷键对话框，支持查找和配置快捷键</li> 
 <li>改进标题栏样式支持，支持颜色和图片背景快捷配置</li> 
 <li>增加弹出桌面气泡消息支持</li> 
 <li>增加关闭页面确认对话框支持</li> 
 <li>重构数据表、设计器、模型页工具栏控件，支持图标和图标文字两种显示方式</li> 
 <li>修复: Access denied: you need the SUPER privilege(s) for this operation</li> 
 <li>修复: 删除对象后同步关闭其打开的视图</li> 
 <li>修复: 重命名对象后同步更新打开的视图</li> 
 <li>修复：苹果系统无法获得系统名称的错误</li> 
</ol> 
<p>详情查看：<a href="https://gitee.com/dbkangaroo/kangaroo/releases/v1.7.1.210331">https://gitee.com/dbkangaroo/kangaroo/releases/v1.7.1.210331</a></p> 
<p>袋鼠数据库工具，为热门数据库系统打造的 SQL 客户端和管理工具(SQLite / MySQL / PostgreSQL / ...) ，支持 Windows / Linux / MacOS</p>
