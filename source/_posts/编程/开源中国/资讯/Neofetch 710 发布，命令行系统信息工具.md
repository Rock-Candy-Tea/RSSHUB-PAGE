                                                                                            <p>Neofetch 是一个用 bash 3.2+ 编写的命令行系统信息工具。Neofetch 以令人愉悦的方式显示关于你的操作系统、软件和硬件的信息。</p> 
<p>Neofetch 支持近 150 种不同的操作系统。从 Linux 到 Windows，再到诸如 Minix、AIX 和 Haiku 等更晦涩的操作系统。</p> 
<p>Neofetch 7.1.0 版本更新内容包括：</p> 
<ul> 
 <li>增加了对最新 macOS 的支持</li> 
 <li>增加了对 Ubuntu flavours 的自动检测</li> 
 <li>修正了 Linux 上的秒数</li> 
 <li>默认启用了版本输出</li> 
 <li>现在检测到 OpenBSD 时会显示 -current 或 -release</li> 
 <li>增加了对 OpenStage Linux 的支持</li> 
 <li>更改 gtk3 配置顺序</li> 
 <li>将 AOSC 分成 OS 和 OS/Retro</li> 
 <li>增加了对 openmamba 的支持</li> 
 <li>增加了对 OIL shell 的支持</li> 
 <li>增加了对 Alter Linux 的支持</li> 
 <li>增加了对 Serene Linux 的支持</li> 
 <li>增加了对 xmonad 的支持</li> 
 <li>加快了 guix 的速度</li> 
 <li>增加了对 es shell 的支持</li> 
 <li>增加了对 OS Elbrus 的支持</li> 
 <li>增加了对 sambabox 的支持</li> 
 <li>修正了磁盘条目和文件名中的空格的错误</li> 
 <li>Windows 多显示器支持</li> 
 <li>增加了对 Venom Linux 的支持</li> 
 <li>增加了对 playerctl 的支持</li> 
 <li>更新了 iOS 设备信息</li> 
 <li>增加了对 Hash Linux 的支持</li> 
 <li>增加了对 Garuda Linux 的支持</li> 
 <li>增加了对 x11fs 的支持</li> 
 <li>增加了对 Tauon 音乐盒的支持</li> 
 <li>增加了对 DarkOS 的支持</li> 
 <li>增加了对 Olivia 播放器的支持</li> 
 <li>增加了对 ITX 的支持</li> 
 <li>添加了 Ubuntu Cinnamon 的标识</li> 
 <li>增加了对 OpenEuler 的支持</li> 
 <li>增加了对 Windows terminal 的支持</li> 
 <li>增加了对改变输出单位（kb、mb、gb）的支持</li> 
 <li>增加了对 LXQT 的支持</li> 
</ul>
