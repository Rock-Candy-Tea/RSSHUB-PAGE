<div class="content">
                                                                                            <p>IntelliJ IDEA 2022.2 EAP 2 现已<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fblog.jetbrains.com%2Fidea%2F2022%2F05%2Fintellij-idea-2022-2-eap-2%2F" target="_blank">发布</a>。此<span style="background-color:#ffffff; color:#27282c">版本包括改进了 JSON、YAML 和 .properties 文件的使用、支持 Jakarta Persistence 3.1 中的新功能，更好地支持 Bean Validation 注释等等。</span><span style="background-color:#ffffff; color:#333333">开发者可以从</span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.jetbrains.com%2Fidea%2Fnextversion%2F" target="_blank">网站</a><span style="background-color:#ffffff; color:#333333">、</span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.jetbrains.com%2Ftoolbox-app%2F" target="_blank">Toolbox App</a><span style="background-color:#ffffff; color:#333333">，或通过使用 Ubuntu 的 snaps 下载最新版本。</span></p> 
<p>具体更新内容如下：</p> 
<p style="margin-left:0px; margin-right:0px; text-align:start"><strong><span><span><span><span><span><span><span style="color:#27282c"><span><span><span><span><span><span><span><span><span><span><span style="background-color:#ffffff"><span><span><span>JSON、YAML 和 .properties 字符串值中的可点击 URL</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></strong></p> 
<p style="margin-left:0; margin-right:0; text-align:start"><span><span><span><span><span style="color:#27282c"><span><span><span><span><span><span><span><span><span><span><span><span style="background-color:#ffffff"><span><span><span>JSON、YAML 和 .properties 文件现在具有在以 http:// 和 https:// 开头的值内自动插入 Web 引用的功能。 用户</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span><span><span><span><span><span style="color:#27282c"><span><span><span><span><span><span><span><span><span><span><span><span style="background-color:#ffffff"><span><span><span>可以轻松地在 Web 浏览器中一键打开这些链接，也可以在 HTTP 客户端中从 Context Actions 菜单（Alt + Enter / Option + ⏎）生成请求。</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></p> 
<p style="margin-left:0px; margin-right:0px; text-align:start"><strong><span><span><span><span><span><span><span style="color:#27282c"><span><span><span><span><span><span><span><span><span><span><span style="background-color:#ffffff"><span><span><span><img alt height="200" src="https://oscimg.oschina.net/oscnet/up-bad4d94bb081369da23afc9562052028fd7.gif" width="500" referrerpolicy="no-referrer"></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></strong></p> 
<p style="margin-left:0px; margin-right:0px; text-align:start"><strong><span><span><span><span><span><span><span style="color:#27282c"><span><span><span><span><span><span><span><span><span><span><span style="background-color:#ffffff"><span><span><span>支持 Jakarta Persistence 3.1</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></strong></p> 
<p>IntelliJ IDEA 现在支持 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fprojects.eclipse.org%2Fprojects%2Fee4j.jpa%2Freleases%2F3.1" target="_blank">Jakarta Persistence 3.1</a> 中的所有功能，包括新的 JPQL 语法、函数和类型，并为它们提供语言高亮和代码完成。 </p> 
<p style="margin-left:0px; margin-right:0px; text-align:start"><strong><span><span><span><span><span><span><span style="color:#27282c"><span><span><span><span><span><span><span><span><span><span><span style="background-color:#ffffff"><span><span><span>改进了对 Bean Validation 注释的支持 </span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></strong></p> 
<p style="margin-left:0; margin-right:0; text-align:start"><span><span><span><span><span style="color:#27282c"><span><span><span><span><span><span><span><span><span><span><span><span style="background-color:#ffffff"><span><span><span>IntelliJ IDEA 2022.2 现在为 Java 和 Kotlin 的 Bean Validation 注释中的消息属性提供 references 并支持 folding。</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></p> 
<p style="margin-left:0; margin-right:0; text-align:start"><span><span><span><span><span style="color:#27282c"><span><span><span><span><span><span><span><span><span><span><span><span style="background-color:#ffffff"><span><span><span><img alt height="200" src="https://oscimg.oschina.net/oscnet/up-1dd53662e39f679f24ee581aa1976657472.png" width="500" referrerpolicy="no-referrer"></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></p> 
<p style="margin-left:0px; margin-right:0px; text-align:start"><strong><span><span><span><span><span><span><span style="color:#27282c"><span><span><span><span><span><span><span><span><span><span><span style="background-color:#ffffff"><span><span><span>使用 Testcontainers 在测试中完成 Docker 镜像</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></strong></p> 
<p style="margin-left:0; margin-right:0; text-align:start"><span><span><span><span><span style="color:#27282c"><span><span><span><span><span><span><span><span><span><span><span><span style="background-color:#ffffff"><span><span><span>这个新版本为 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.testcontainers.org%2F" target="_blank">Testcontainers</a> API 引入了 image completion，你可以在其中找到所有可用的 Docker 镜像及其版本。点击 reference 以在你的 Web 浏览器中打开相应的 Docker Hub URL。</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></p> 
<p style="margin-left:0; margin-right:0; text-align:start"><span><span><span><span><span style="color:#27282c"><span><span><span><span><span><span><span><span><span><span><span><span style="background-color:#ffffff"><span><span><span><img alt height="200" src="https://oscimg.oschina.net/oscnet/up-cd259bba103b5d6f063d899abce95705592.gif" width="500" referrerpolicy="no-referrer"></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></p> 
<p style="margin-left:0; margin-right:0; text-align:start"> </p> 
<p style="margin-left:0px; margin-right:0px; text-align:start"><strong><span><span><span><span><span><span><span style="color:#27282c"><span><span><span><span><span><span><span><span><span><span><span style="background-color:#ffffff"><span><span><span>改进的 AWS CloudFormation 插件</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></strong></p> 
<p style="margin-left:0; margin-right:0; text-align:start"><span><span><span><span><span style="color:#27282c"><span><span><span><span><span><span><span><span><span><span><span><span style="background-color:#ffffff"><span><span><span>通过更新 metadata schemas 和改进 property completion 来重新设计 AWS CloudFormation 插件。你可以从 Marketplace 在 IntelliJ IDEA 2022.2 EAP 版本上安装<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fplugins.jetbrains.com%2Fplugin%2F7371-aws-cloudformation" target="_blank">插件的更新版本。</a></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></p>
