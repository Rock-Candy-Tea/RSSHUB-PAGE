 <p><span style="background-color:#ffffff; color:#333333">DBeaver 21.2.1 现已发布。DBeaver是一个免费开源的通用数据库工具，适用于开发人员和数据库管理员。</span></p> 
 <p><span style="background-color:#ffffff; color:#333333"><strong>主要更新内容</strong></span></p> 
 <ul> 
  <li>Data 浏览器 
   <ul> 
    <li>重新设计了布尔值渲染配置</li> 
    <li>添加了 "带时区的时间/时间戳" 类型的特殊格式</li> 
    <li>高级粘贴：添加了配置对话框</li> 
    <li>基数转换器已修复</li> 
    <li>添加了数字转换器</li> 
    <li>几何数据类型编辑器已修复（Exasol、Oracle）</li> 
    <li>值面板现在保存上次使用的内容查看器设置</li> 
   </ul> </li> 
  <li>数据传输 
   <ul> 
    <li>输出设置配置器已修复</li> 
    <li>外部进程执行已修复（反斜杠转义）</li> 
    <li>从 XLSX 导入：已修复列映射</li> 
    <li>数据库转储配置页面已修复（Mysql、PostgreSQL）</li> 
    <li>数据加载性能得到改善</li> 
   </ul> </li> 
  <li>SSH <span style="background-color:#ffffff; color:#182026">通道：跳转服务器配置器已修复（端口号）</span></li> 
  <li><span style="background-color:#ffffff; color:#182026">添加了 Windows 快捷方式 (lnk) 分辨率</span></li> 
  <li><span style="background-color:#ffffff; color:#182026">ERD：添加了列排序支持</span></li> 
  <li><span style="background-color:#ffffff; color:#182026">重新设计了确认对话框 UI</span></li> 
  <li><span style="background-color:#ffffff; color:#182026">元数据搜索</span> 
   <ul> 
    <li><span style="background-color:#ffffff; color:#182026">在对象注释配置器中搜索已修复</span></li> 
    <li><span style="background-color:#ffffff; color:#182026">修复了跨数据库搜索 PostgreSQL 的搜索页面</span></li> 
   </ul> </li> 
  <li><span style="background-color:#ffffff; color:#182026">数据库会话管理器刷新已修复</span></li> 
  <li><span style="background-color:#ffffff; color:#182026">DB2：添加了列、过程和函数搜索支持</span></li> 
  <li><span style="background-color:#ffffff; color:#182026">Firebird：添加了计算字段支持</span></li> 
  <li>Greenplum：表 DDL 已修复（唯一键）</li> 
  <li><span style="background-color:#ffffff; color:#182026">H2：添加了过程别名支持</span></li> 
  <li><span style="background-color:#ffffff; color:#182026">PostgreSQL</span> 
   <ul> 
    <li><span style="background-color:#ffffff; color:#182026">​​​​​​​复合数据类型支持已修复</span></li> 
    <li><span style="background-color:#ffffff; color:#182026">UDF 参数支持已修复</span></li> 
    <li><span style="background-color:#ffffff; color:#182026">添加了外部表注释支持</span></li> 
   </ul> </li> 
 </ul>
