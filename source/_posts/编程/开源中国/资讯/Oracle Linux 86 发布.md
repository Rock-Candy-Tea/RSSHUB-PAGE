                                                                                            <p>Oracle Linux 8.6 已面向面向 64 位 Intel、64 位 AMD (x86_64) 和 64 位 Arm (aarch64) 平台<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fblogs.oracle.com%2Flinux%2Fpost%2Fannouncing-the-release-of-oracle-linux-8-update-6" target="_blank">发布</a>。</p> 
<p>该版本引入了对 OpenSSH 服务器的插入式配置文件支持、用于自动解锁 LUKS 加密驱动器的基于策略的解密 (PBD) ，并提供了许多增强功能，包括：</p> 
<p><strong>软件管理</strong></p> 
<ul> 
 <li>模块同步（modulesync）：<strong> </strong>该命令用于确保模块化元数据的存在，并在模块化元数据不可用时促进模块的安装。</li> 
</ul> 
<h4>编译器和开发工具集</h4> 
<ul> 
 <li>GCC 工具集更新到版本 11.2</li> 
 <li>Rust 工具集更新到版本 1.58.1</li> 
 <li>LLVM 工具集更新到版本 13.0.1</li> 
 <li>PCP 更新至版本 5.3.5</li> 
 <li>Grafana 更新到版本 7.5.11</li> 
</ul> 
<h4>动态编程语言</h4> 
<ul> 
 <li>添加了 php:8.0 模块流。这个新的模块在 7.4 版本上提供了几个错误修复和增强功能。</li> 
 <li>添加了 Perl 5.32 模块流。这个新的模块包括对 Perl 版本 5.30 的许多错误修复和增强</li> 
</ul> 
<h4>网络</h4> 
<ul> 
 <li>NetworkManager 更新到版本 1.36.0。</li> 
 <li>改进 nftables 框架。</li> 
</ul> 
<p>其他改进请查看<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fblogs.oracle.com%2Flinux%2Fpost%2Fannouncing-the-release-of-oracle-linux-8-update-6" target="_blank">更新公告</a>。</p>
