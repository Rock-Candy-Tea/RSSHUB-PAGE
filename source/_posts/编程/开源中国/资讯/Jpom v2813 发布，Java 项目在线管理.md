                                                                                            <p>Jpom v2.8.13 已经发布，Java 项目在线管理。</p> 
<p>此版本更新内容包括：</p> 
<h3>新增功能</h3> 
<ol> 
 <li>【agent】项目 DSL 支持配置项目路径下的脚本</li> 
</ol> 
<h3>解决BUG、优化功能</h3> 
<ol> 
 <li>【server】新增个性化配置,导航菜单打开方式（感谢@以为）</li> 
 <li>【server】工作空间没有变量不能使用容器构建问题（感谢@杨杰）</li> 
 <li>【server】在线构建 ssh 发布选择授权目录切换不生效问题（感谢@天天）</li> 
 <li>【server】在线构建本地构建命令不能换行问题（感谢@华仔）</li> 
 <li>【server】日志弹窗新增行号</li> 
 <li>【server】在线升级执行脚本、脚本模版等兼容 <code>debian</code> （感谢<a href="https://www.oschina.net/wxyShine">@wxyShine </a> <a href="https://gitee.com/dromara/Jpom/issues/I4UQBD" target="_blank">Gitee issues I4UQBD</a> ）</li> 
 <li>【agent】修复 windows 环境，节点首页进程列表切换不生效问题（感谢<a href="https://www.oschina.net/neoch">@Neo_Chen </a> <a href="https://gitee.com/dromara/Jpom/issues/I4UZA7" target="_blank">Gitee issues I4UZA7</a> ）</li> 
 <li>【server】脚本模版补充权限</li> 
</ol>
