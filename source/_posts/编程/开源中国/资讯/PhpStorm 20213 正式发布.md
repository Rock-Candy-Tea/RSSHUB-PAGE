                                                        <p>PhpStorm 2021.3 现已发布。这个版本主要包含引入对 PHP 8.1 的全面支持、更好地处理 PHP 中的泛型、远程开发、部署改进、HTTP 客户端、重构等内容。</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-abbcce308bb742c85b5cef4bfc6e5e9c8bd.png" referrerpolicy="no-referrer"></p> 
<h4><strong>完全支持 PHP 8.1</strong></h4> 
<p>PhpStorm 2021.3 支持所有最新的语言更改，包括枚举、只读属性、纯交集类型、新的返回类型 'never'、final 常数等内容。用户可以通过将项目语言级别设置为 PHP 8.1 来尝试新功能，并可以从工具栏或通过在 composer.json 中来切换。</p> 
<p><img alt height="1080" src="https://oscimg.oschina.net/oscnet/up-9bfe9867e0e571c3ac153beb8e2f0c78e6a.png" width="1248" referrerpolicy="no-referrer"></p> 
<h4>更好的泛型支持</h4> 
<p>该版本中泛型增强相关的功能包括改进对 Doctrine Collections 的支持、支持未来的 Laravel 集合、从构造函数推断泛型类型、支持模板的 phpstan-* 注释等等。</p> 
<p><img alt height="624" src="https://oscimg.oschina.net/oscnet/up-a098d5c50d1688da20600a1c8232628a216.gif" width="1248" referrerpolicy="no-referrer"></p> 
<h4>改进重构</h4> 
<p> PhpStorm 2021.3 中，用户可以直接将文件甚至整个文件夹拖到新的目的地，PhpStorm 将负责更新类的命名空间以及所有类的用法。此外，现有的重构方法Inline method、Inline variable、Extract variable 和 Extract method 也得到了更新和修复，变得更加可靠和可预测。</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-b20d35272599abb388864f80ed4bb49596f.gif" referrerpolicy="no-referrer"></p> 
<h4>远程开发</h4> 
<p>在该版本中，所有部署设置均在 Settings/Preferences 中，在使用 SFTP 服务器进行部署时，用户将在 "高级" 部分下看到一个新的 rsync 选项。此外，重新上传失败的文件变得更加简单，而且在连接 <span style="background-color:#ffffff; color:#182026">SFTP/FTP/FTPS </span>时会使用多线程加速。</p> 
<p><img alt height="1080" src="https://oscimg.oschina.net/oscnet/up-a73fe215a73701582ad2dcea426ee7a48f7.png" width="1774" referrerpolicy="no-referrer"></p> 
<h4>HTTP 客户端增强</h4> 
<p>在该版本中，HTTP 客户端得到了一系列增强，包括支持 HTTP 客户端中的文本和 JSON 流、控制台中显示图像预览、支持输出重定向到自定义文件或目录、支持 HTTP 请求标识符等等。</p> 
<p><img alt height="600" src="https://oscimg.oschina.net/oscnet/up-dd9f65016725a300b9b3cd61794054fc76b.png" width="1500" referrerpolicy="no-referrer"></p>
