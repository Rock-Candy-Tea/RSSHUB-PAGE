<div class="content">
                                                                    
                                                        <blockquote> 
 <p style="text-align:left">Jboot 一个更简单的分布式、微服务框架。</p> 
 <p style="text-align:left">Jboot是一个基于 JFinal、JFinal-Undertow、Dubbo、Seata、Sentinel、ShardingSphere、Nacos 等开发的微服务框架，帮助开发者降低微服务、分布式开发门槛。爽爽开发，快乐生活。</p> 
</blockquote> 
<p style="text-align:left"><span style="background-color:#ffffff; color:#333333">到目前为止，Jboot 已经开源超过了 5 年的时间，迭代了 200+ 个版本，已经被超过 1000+ 公司在使用，其中包含了多个知名的上市公司，我们了解到的多个使用 Jboot 开发的产品，用户量超过 1亿 以上。</span></p> 
<p style="text-align:left"><span style="background-color:#ffffff; color:#333333">Jboot 从 v3.10.0 的开始，带来了自研的 API 文档生成工具。</span></p> 
<p style="text-align:left"><img src="https://oscimg.oschina.net/oscnet/up-cc86548553fa8dfc9c4c0f88bf3029fc690.JPEG" referrerpolicy="no-referrer"></p> 
<p style="text-align:left">比如，以上的文档是使用了 Jboot 文档工具自动生成的，其里面包含了<span style="color:#ff0000">接口信息</span>、<span style="color:#ff0000">参数信息</span>、<span style="color:#ff0000">是否进行非空验证</span>、<span style="color:#ff0000">提交的方式</span>、<span style="color:#ff0000">render 的 json 数据</span>，<span style="color:#ff0000">json 里的字段内容 </span>等等.....</p> 
<p style="text-align:left"><span style="background-color:#ffffff; color:#333333">查看更多的示例，大家可以参考网址：</span><a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fdoc.jpress.io%2Fdevelopment%2Fapi%2Fapi_user.html" target="_blank">http://doc.jpress.io/development/api/api_user.html</a><span style="background-color:#ffffff; color:#333333"> ，其内容全部都是 Jboot 自动生成的。</span></p> 
<p style="text-align:left">同时，在自动生成文档的基础上，JbootAdmin 为 Jboot 持续做了增强。如下图所示。</p> 
<p style="text-align:left"><img height="1304" src="https://oscimg.oschina.net/oscnet/up-ebd97de1a35983a53b69b0f23bd369ed5dd.JPEG" width="1891" referrerpolicy="no-referrer"></p> 
<p style="text-align:left">在使用 JbootAdmin 开发的过程中，APIDoc 文档左边的接口菜单、文档接口、接口测试、接口Mock 都是跟随我们编写的代码实时生成，不需要任何的手动维护、不需要 Maven 编译推送... 在 Controller 中每增加一个 API 方法，左边菜单自动新增一个相关菜单.... 完全解放我们的双手。</p> 
<p style="text-align:left"><strong>Jboot v3.10.2 更新内容如下：</strong></p> 
<ul> 
 <li>新增：注解 @ApiPara() 增加 require 参数的配置，感谢 <a href="https://gitee.com/lijiahong">@ljh581120</a></li> 
 <li>优化：Json 增加递归深度配置，修复多层级数据返回不完全问题，感谢 <a href="https://gitee.com/lijiahong">@ljh581120</a></li> 
 <li>优化：升级 JFinal、jackson、Dubbo 等到最新版本</li> 
 <li>修复：JbootPaginateDirective 在默认情况下无法获取 page 的问题</li> 
 <li>修复：Redis 连接有密码时，Redis 集群时未设置 maxAttempts 连接权限不上问题，感谢 <a href="https://gitee.com/lijiahong">@ljh581120</a></li> 
</ul>

<p style="text-align:left"><span style="background-color:#ffffff; color:#333333">同时，Jboot 官方也推出了收费的、企业级快速开发框架 JbootAdmin （如下图所示），真诚的为各位开发者提供一站式、保姆式服务。请咨询海哥。</span></p>
