                                                                                            <p>KDE Plasma 5.22.5 现已发布，此版本增加了来自 KDE 贡献者一个月的新翻译和修复，是 8 月份的错误修复版本。</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-8ff21048476ec0871af5f815e220d2b315f.png" referrerpolicy="no-referrer"></p> 
<p><strong>主要更新内容</strong></p> 
<ul> 
 <li>Breeze 
  <ul> 
   <li>Kstyle：在决定箭头颜色时考虑 activeSubControls</li> 
  </ul> </li> 
 <li>Discover 
  <ul> 
   <li>显示刷新操作工具提示的实际快捷方式</li> 
  </ul> </li> 
 <li>KDE GTK Config 
  <ul> 
   <li>不将 window_decorations.css 应用于非 Breeze 主题</li> 
   <li>确保真正提交 GSettings 更改</li> 
  </ul> </li> 
 <li>ksystemstats 
  <ul> 
   <li>修复对 IPV6 地址的处理</li> 
  </ul> </li> 
 <li>KWin 
  <ul> 
   <li>使用 Xwayland 窗口修复拖放升高</li> 
   <li>Platforms/drm：在连接之前检查 wl_eglstream 缓冲区</li> 
   <li>Platforms/drm：修复 gpu 移除</li> 
   <li>Kcm/kwinddecoration：修复窗口缩略图大小</li> 
   <li>Platforms/drm：考虑 KWIN_DRM_DEVICES 用于热插拔 GPU</li> 
   <li>Platforms/drm：修复崩溃的调试操作符</li> 
   <li>将形状区域复制到已删除</li> 
   <li>避免丢弃以前的像素图</li> 
  </ul> </li> 
 <li>Plasma Desktop 
  <ul> 
   <li>[emojier] 设置 KLocalizedContext</li> 
   <li>[emojier] 将 QAbstractItemModel 注册到 QML</li> 
   <li>[containments/desktop] 停止硬编码小程序句柄工具提示位置</li> 
   <li>[containments/panel] 通过触摸修复面板小程序配置</li> 
   <li>[desktoppackage/alternatives] 不要让长文本溢出</li> 
   <li>[applet/taskmanager] 不要让固定的应用工具提示具有交互性</li> 
   <li>[containments/desktop] 通过触摸交互修复小程序覆盖图标大小</li> 
   <li>使用主题中正确的面板前缀</li> 
  </ul> </li> 
</ul> 
<p>详情请查看<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fkde.org%2Fannouncements%2Fplasma%2F5%2F5.22.5%2F" target="_blank">更新公告</a>。</p>
