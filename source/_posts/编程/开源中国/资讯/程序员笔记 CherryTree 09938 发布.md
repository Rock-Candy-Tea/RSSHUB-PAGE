                                                        <p>CherryTree 0.99.38 现已发布。CherryTree 是一个支持无限层级分类的笔记软件，Python 编写，支持富文本编辑和代码高亮，支持 Linux 和 Windows 平台。数据采用 sqlite 或 XML 存储，支持密码保护，支持从 NoteCase、KeepNote、Knowit、Tomboy、TuxCards、Treepad、Leo 等笔记软件导入数据。</p> 
<p><img height="342" src="https://static.oschina.net/uploads/space/2021/0127/070748_SLU0_2720166.png" width="500" referrerpolicy="no-referrer"></p> 
<p><img height="342" src="https://static.oschina.net/uploads/space/2021/0127/070803_tv03_2720166.png" width="500" referrerpolicy="no-referrer"></p> 
<p>此版本更新内容如下：</p> 
<ul> 
 <li>修复访问具有损坏/无效 utf-8 字符的节点的崩溃</li> 
 <li>修复当试图从 Glib 不支持的格式的文件中插入图片时的崩溃，如 .webp</li> 
 <li>修复复制/粘贴后发现的不需要的字符</li> 
 <li>修复托盘中状态图标丢失/错误渲染的问题</li> 
 <li>修复了在系统托盘上最小化启动时的树节点可见性问题</li> 
 <li>修复链接 Markdown 粘贴损坏的链</li> 
 <li>修复启用拼写检查但没有可用语言的问题</li> 
 <li>改进的网页链接识别</li> 
 <li>更新语言芬兰语、韩语、巴西葡萄牙语</li> 
 <li>此（gtkmm3）版本现在可以在较早的发行版上构建，如 debian 10/ubuntu 18.04</li> 
</ul>
