                                                        <p style="text-align:left">开源一年超万星的 go-zero 发布了 1.1.9 版本。go-zero 是一个集成了各种工程实践的 web 和 rpc 框架。通过弹性设计保障了大并发服务端的稳定性，经受了充分的实战检验。go-zero 包含极简的 API 定义和生成工具 goctl，可以根据定义的 API 文件一键生成 Go、iOS、Android、Kotlin、Dart、TypeScript、JavaScript 代码，并可直接运行。</p> 
<p style="text-align:left">本次更新内容包括：</p> 
<p>框架：<br> 1. `httpx.Parse(...)` 支持 `http header` 解析<br> 2. 修复了端口占用时启动只报错不退出的问题<br> 3. 一些 bug 修复和改进</p> 
<p>goctl:<br> 1. 支持 `gRPC stream` 的代码生成<br> 2. 支持 `postgresql` 代码生成<br> 3. 支持 `mysql` 的 `blob` 类型<br> 4. `goctl proto` 子命令增加了 `--go_opt` 选项兼容 `protoc-gen-go@v1.4.0`<br> 5. 增加 `--home` 参数指定目录保存 `goctl` 模板<br> 2. 优化 `sql` 文件解析</p>
