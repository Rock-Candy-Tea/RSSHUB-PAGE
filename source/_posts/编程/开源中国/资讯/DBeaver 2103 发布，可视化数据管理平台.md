                                                        <p>DBeaver 21.0.3 现已发布。DBeaver是一个免费开源的通用数据库工具，适用于开发人员和数据库管理员。</p> 
<p><strong>主要更新内容</strong></p> 
<ul> 
 <li>改进了对夜间主题的支持</li> 
 <li>Data viewer 
  <ul> 
   <li>增加了格式配置编辑器</li> 
   <li>过滤器对话框增加了额外配置</li> 
   <li>按固定的列排序（适用于较小的提取尺寸）</li> 
   <li>增加了不区分大小写的过滤器支持</li> 
   <li>纯文本视图现在支持顶部/底部分隔线</li> 
   <li>数据编辑器已修复（当列名与别名冲突时）</li> 
   <li>修复了多个选定行的重复行命令</li> 
   <li>增加了列自动大小配置</li> 
   <li>字典查看器已修复（只读连接）</li> 
   <li>增加了对当前选择行的高亮支持（可配置）</li> 
  </ul> </li> 
 <li>元数据搜索现在支持在注释中搜索</li> 
 <li>GIS/Spatial 
  <ul> 
   <li>增加了对带有 Z 和 M 坐标的几何图形的支持</li> 
   <li>Postgis：修复了 3D 几何列的 DDL</li> 
   <li>增加了对Presto + MySQL几何类型的支持</li> 
   <li>BigQuery 现在支持空间数据查看器</li> 
   <li>改进了对二进制 geo json 的支持</li> 
  </ul> </li> 
 <li>SQL editor 
  <ul> 
   <li>修复了对象名称中带空格时的自动补全功能</li> 
   <li>修复了数据库对象超链接的渲染问题</li> 
  </ul> </li> 
 <li>SQL Server：添加了MFA（多因素身份验证）支持</li> 
</ul>
