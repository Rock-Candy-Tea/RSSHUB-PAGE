<div class="content">
                                                                                            <p><span style="background-color:#ffffff; color:#333333">Docker 20.10.18 现已发布。</span>此版本的 Docker Engine 修复了一个低严重性的安全问题、一些小错误修复以及更新了 Docker Compose、Docker Buildx<code>containerd</code>和<code>runc</code>。</p> 
<h4><span><span><span><span><span style="color:#24292f"><span><span><span><span><span><span><span><span><span><span><span style="background-color:#ffffff"><span><span><span>Client</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></h4> 
<ul> 
 <li>为 Docker Compose 添加 Bash completion <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fdocker%2Fcli%2Fpull%2F3752" target="_blank">docker/cli#3752</a>。</li> 
</ul> 
<h4><span><span><span><span><span style="color:#24292f"><span><span><span><span><span><span><span><span><span><span><span style="background-color:#ffffff"><span><span><span>Builder</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></h4> 
<ul> 
 <li>修复了在构建期间未保留文件功能的问题 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmoby%2Fmoby%2Fpull%2F43876" target="_blank">moby/moby#43876</a>。</li> 
 <li>修复了可能导致并发 map 读取和 map 写入导致 panic 的问题 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmoby%2Fmoby%2Fpull%2F44067" target="_blank">moby/moby#44067</a>。</li> 
</ul> 
<h4><span><span><span><span><span style="color:#24292f"><span><span><span><span><span><span><span><span><span><span><span style="background-color:#ffffff"><span><span><span>Daemon</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></h4> 
<ul> 
 <li>修复与补充组权限相关的安全漏洞，该漏洞可能允许容器进程绕过容器内的主要组限制 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fcve.mitre.org%2Fcgi-bin%2Fcvename.cgi%3Fname%3DCVE-2022-36109" target="_blank">CVE-2022-36109</a>、 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmoby%2Fmoby%2Fsecurity%2Fadvisories%2FGHSA-rc4r-wh2q-q6c4" target="_blank">GHSA-rc4r-wh2q-q6c4</a>。</li> 
 <li>seccomp：在默认策略中添加对 Landlock 系统调用的支持 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmoby%2Fmoby%2Fpull%2F43991" target="_blank">moby/moby#43991</a>。</li> 
 <li>seccomp：更新默认策略以支持内核 5.12 - 5.16 中引入的新系统调用 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmoby%2Fmoby%2Fpull%2F43991" target="_blank">moby/moby#43991</a>。</li> 
 <li>修复了图像清单的缓存查找失败的问题，从而导致对图像注册表的冗余往返 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmoby%2Fmoby%2Fpull%2F44109" target="_blank">moby/moby#44109</a>。</li> 
 <li>修复了 exec processes 和 healthchecks 超时未终止的问题 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmoby%2Fmoby%2Fpull%2F44018" target="_blank">moby/moby#44018</a>。</li> 
</ul> 
<h4><span><span><span><span><span style="color:#24292f"><span><span><span><span><span><span><span><span><span><span><span style="background-color:#ffffff"><span><span><span>Packaging</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></h4> 
<ul> 
 <li>将 Docker Buildx 更新到 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fdocker%2Fbuildx%2Freleases%2Ftag%2Fv0.9.1" target="_blank">v0.9.1</a>。</li> 
 <li>将 Docker Compose 更新到 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fdocker%2Fcompose%2Freleases%2Ftag%2Fv2.10.2" target="_blank">v2.10.2</a>。</li> 
 <li>将 containerd (<code>containerd.io</code>包) 更新到 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fcontainerd%2Fcontainerd%2Freleases%2Ftag%2Fv1.6.8" target="_blank">v1.6.8</a>。</li> 
 <li>将 runc 更新到 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fopencontainers%2Frunc%2Freleases%2Ftag%2Fv1.1.4" target="_blank">v1.1.4</a>。</li> 
 <li>将 Go 运行时更新到<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgo.dev%2Fdoc%2Fdevel%2Frelease%23go1.18.minor" target="_blank">1.18.6</a>，其中包含针对 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fcve.mitre.org%2Fcgi-bin%2Fcvename.cgi%3Fname%3DCVE-2022-27664" target="_blank">CVE-2022-27664</a> 和 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fcve.mitre.org%2Fcgi-bin%2Fcvename.cgi%3Fname%3DCVE-2022-32190" target="_blank">CVE-2022-32190 的修复</a>。</li> 
</ul> 
<p>更新说明：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmoby%2Fmoby%2Freleases%2Ftag%2Fv20.10.18" target="_blank">https://github.com/moby/moby/releases/tag/v20.10.18</a></p>
