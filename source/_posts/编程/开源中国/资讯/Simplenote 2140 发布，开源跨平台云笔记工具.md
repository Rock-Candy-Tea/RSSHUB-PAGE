                                                        <p>Simplenote 是一个简单、轻量级的跨平台云笔记工具，可以用来保存文字、表格、代码等信息，用户记录的内容可以与所有设备保持同步。Simplenote 支持大多数主流的 WEB 浏览器，同时支持 iOS，Android，Mac，Windows 和 Linux 操作系统。</p> 
<p>自 Simplenote 2.12.0 以来的更新内容包括：</p> 
<h3><strong>增强</strong></h3> 
<ul> 
 <li>为笔记修订选择器添加了一个工具提示，以更好地解释新的恢复已删除标签的操作；</li> 
 <li>根据新的设计更新了新的笔记图标；</li> 
 <li>更新了黑暗模式下的颜色以使用 CSS 变量；</li> 
 <li>升级了 Electron 和 Electron Builder 的依赖性到新的版本；</li> 
 <li>为 Mac Electron 应用添加了一个默认的窗口标题栏；</li> 
</ul> 
<h3><strong>修复</strong></h3> 
<ul> 
 <li>更新了钉在 Windows 开始菜单上时使用的标志；</li> 
 <li>修正了搜索滚动条的高亮显示，使其在编辑器中始终显示搜索匹配；</li> 
 <li>修正了 Mac 应用程序在第一次请求时不会退出的错误；</li> 
 <li>修正了设置为按字母排序标签时，垃圾箱标签按钮的间距问题；</li> 
</ul>
