                                                                                            <p>pgAdmin 是 PostgreSQL 领先的开源图形化管理工具。pgAdmin 4 旨在满足新手和有经验的 Postgres 用户的需求，提供强大的图形界面，简化了数据库对象的创建、维护和使用。</p> 
<p>这个版本的 pgAdmin 4 包括错误修复和新功能，主要更新内容如下：</p> 
<h3>功能</h3> 
<ul> 
 <li>增加了对在一个单独的面板中显示所有后台进程的支持 
  <ul> 
   <li>通过使用这个功能，可以在新的进程面板中看到后台进程。你可以查看有关备份、恢复、维护、导入/导出和创建云实例的细节。</li> 
  </ul> </li> 
 <li>增加了对从 EPAS 中现有的触发器函数创建触发器的支持</li> 
</ul> 
<h3>Bugs/Housekeeping:</h3> 
<ul> 
 <li>移植 Role Reassign、User Management、Change Ownership 和 URL Dialogs 到 React</li> 
 <li>修正了 Pallets Werkzeug v2.1.0 及以下版本中 HTTP 请求的不当解析</li> 
 <li>Pin Flask-SocketIO <= v5.2.0，最新版本不支持生产环境中的 Werkzeug</li> 
 <li>确保在不提供剪贴板访问时抛出错误，并更改复制行的快捷方式</li> 
 <li>确保在使用方向键导航代码时不会出现自动补全</li> 
 <li>修正了从 pgAdmin 注销时加密键丢失的问题</li> 
 <li>确保重命名面板应该在视图/编辑面板上工作</li> 
 <li>修正了一个问题，即 schema diff 没有拾取 RLS 策略的变化</li> 
 <li>修正了当选择一个完整的列时渲染几何的问题</li> 
 <li>修正了一个问题，在结果网格中复制和粘贴行不会设置布尔的默认值</li> 
</ul>
