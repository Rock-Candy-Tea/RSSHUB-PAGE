                                                                                            <h3>变化：</h3> 
<ul> 
 <li>改进的 Fast Refresh：快速迭代对于本地开发工作流程至关重要。Next.js 使用 Fast Refresh 来即时反馈对 React 组件所做的编辑。<code>.env</code>、 <code>jsconfig.json</code> 和 <code>tsconfig.json</code> 文件现在可以热重载</li> 
 <li>TypeScript 自动安装：Next.js 内置了对自动配置 TypeScript 的支持。以前，将 TypeScript 添加到现有项目需要手动安装必要的依赖项。如今添加一个 <code>.ts</code> 文件来自动配置 TypeScript 和安装部署</li> 
 <li>Image 组件：在 Next.js 12.2 中，有一个<strong>新的 Image</strong> 组件的实验预览，该组件通过使用本机浏览器延迟加载来简化图像样式并提高性能。新的 Image 组件现在很稳定，不再需要实验性标志。</li> 
 <li>SWC Minifier：作为 Next.js 编译器实验性功能的一部分，Next.js 在 12.0 版本中引入了 SWC 进行代码压缩。早期的结果显示，使用 SWC 进行代码压缩的速度是之前使用 Terser 版本的 7 倍。在这个版本中，该功能已达到稳定程度</li> 
 <li>新的 Router + Layouts 更新：实施工作已经开始，正在努力在下一个主要版本中进行公开测试。</li> 
</ul>
