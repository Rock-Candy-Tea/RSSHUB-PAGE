                                                        <p>Visual Studio 2022 v17.2.6 正式发布，更新内容如下：</p> 
<ul> 
 <li>删除了在某些情况下导致 Visual Studio 在启动期间显示毫无意义的 OS 登录对话框的授权遥测</li> 
 <li>更新了 LibraryManager 以适应 cdnjs API 的更改</li> 
 <li>修复了在同一编辑会话中重新打开 F# 项目时，Intelli 感知停止工作的问题。</li> 
 <li>修复了 VC++ 运行时中的 bug，其中包含 MFC OLE 链接/嵌入容器的链接中断</li> 
 <li>修复了 VS2022 和 VS2015 的 SxS 安装 (或较旧的) 导致 VS 无法扫描扩展的问题</li> 
 <li>改进了预配 Azure 容器应用时的容器名称验证</li> 
</ul> 
<p>更多详情可查看：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdocs.microsoft.com%2Fen-us%2Fvisualstudio%2Freleases%2F2022%2Frelease-notes" target="_blank">https://docs.microsoft.com/en-us/visualstudio/releases/2022/release-notes</a></p>
