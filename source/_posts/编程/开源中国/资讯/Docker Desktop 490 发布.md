                                                                                            <p>Docker Desktop 4.9.0 正式发布，更新内容如下：</p> 
<h3>新内容</h3> 
<ul> 
 <li>在主页上增加了以下内容的指南：Elasticsearch、MariaDB、Memcached、MySQL、RabbitMQ 和 Ubuntu</li> 
 <li>在 Docker Dashboard 上添加了一个页脚，包含了关于 Docker 桌面更新状态和 Docker 引擎统计的常规信息。</li> 
 <li>重新设计了容器表，增加了： 
  <ul> 
   <li>一个将容器 ID 复制到剪贴板的按钮</li> 
   <li>每个容器的暂停按钮</li> 
   <li>容器表的列的大小调整</li> 
   <li>容器表的排序和大小调整的持久性</li> 
   <li>容器表的批量删除</li> 
  </ul> </li> 
</ul> 
<h3>升级</h3> 
<ul> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fdocker%2Fcompose%2Freleases%2Ftag%2Fv2.6.0" target="_blank">Compose v2.6.0</a></li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdocs.docker.com%2Fengine%2Frelease-notes%2F%23201016" target="_blank">Docker Engine v20.10.16</a></li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fcontainerd%2Fcontainerd%2Freleases%2Ftag%2Fv1.6.4" target="_blank">containerd v1.6.4</a></li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fopencontainers%2Frunc%2Freleases%2Ftag%2Fv1.1.1" target="_blank">runc v1.1.1</a></li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fgolang%2Fgo%2Freleases%2Ftag%2Fgo1.18.2" target="_blank">Go 1.18.2</a></li> 
</ul> 
<h3>适用于所有平台</h3> 
<ul> 
 <li>修正了一个问题，如果你在 Docker Desktop 暂停时退出应用，会导致 Docker Desktop 挂起。</li> 
 <li>修正了 Kubernetes 集群在 PKI 过期后不能正确重置的问题。</li> 
 <li>修正了一个问题，即 Extensions Marketplace 没有使用定义的 http 代理。</li> 
 <li>改进了 Docker Dashboard 中的日志搜索功能，允许空格。</li> 
 <li>鼠标中键点击 Dashboard 中的按钮现在表现为左键点击，而不是打开一个空白窗口。</li> 
</ul> 
<h3>适用于 Mac</h3> 
<ul> 
 <li>修正了一个问题，即如果 <code>/opt</code> 已被添加到文件共享目录列表中，则可避免在主机上创建 <code>/opt/containerd/bin</code> 和 <code>/opt/containerd/lib</code>。</li> 
</ul> 
<h3>适用于 Windows</h3> 
<ul> 
 <li>修正了 WSL 2 集成中的一个错误</li> 
 <li>修正了一个错误，即托盘图标和 Dashboard UI 没有显示，Docker Desktop 没有完全启动。</li> 
</ul>
