                                                                                            <p>Brave 是基于 Chromium 的一款免费开源网络浏览器，主打隐私保护，其默认设置就会自动阻止在线广告和网站跟踪。Brave Software（Brave 浏览器所属公司）于 2015 年由 Brendan Eich 和 Brian Bondy 联合创立，其中前者是 JavaScript 的创造者，同时也是 Mozilla 联合创始人。</p> 
<p>Brave 1.35 发布，更新内容如下：</p> 
<ul> 
 <li>将 Chromium 升级到 98.0.4758.87</li> 
 <li>当为 OpenSea 提供多个账户时，修复了最后连接的账户被选中的问题</li> 
 <li>在某些情况下禁止查看 Brave Wallet 面板的 "网站权限"</li> 
 <li>禁用了 HaTS</li> 
 <li>删除了 "navigator.connection"</li> 
 <li>从某些 URL 中删除了已知的用户跟踪参数 "igshid"</li> 
 <li>删除了 Brave Wallet 下 "添加/切换网络" 面板中的 "网络" 和 "地址"</li> 
 <li>从 U2F 权限对话框中删除了权限有效期选项</li> 
 <li>修正了网络改变时输入值不被重置的问题</li> 
 <li>增加了 "wallet_requestPermissions" 和 "wallet_getPermissions"，以便与 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fmarket.x.immutable.com" target="_blank">https://market.x.immutable.com</a> 和其他网站兼容。</li> 
 <li>添加了 web3 的 "currentProvider" shim（垫片），以便与 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fsnowtrace.io" target="_blank">https://snowtrace.io</a> 和 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fstaking.trava.finance%2Fstaking" target="_blank">https://staking.trava.finance/staking</a> 互相兼容。</li> 
 <li>添加了 "personal_ecRecover" 签名，以兼容 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Floopring.io" target="_blank">https://loopring.io</a></li> 
 <li>在 Brave Wallet 中添加了 "添加建议 tokens" 面板</li> 
 <li>在 Brave Wallet 中添加了 "Ethereum Site Permissions" 用户界面</li> 
 <li>增加了在 Brave Wallet 下修改当前网络的功能</li> 
 <li>增加了在 Brave://settings/wallet/networks 下将钱包网络设置为活动的功能</li> 
 <li>在 Brave Wallet 下为交易历史增加了显示确切日期和时间的工具提示</li> 
 <li>为 "选择网络" 下拉菜单添加了加号图标，可在新标签中导航到 Brave://settings/wallet/networks</li> 
 <li>在发送小工具中增加了 EIP-55 校验地址检查</li> 
 <li>在面板中增加了活动交易和代币余额的显示</li> 
 <li>[安全] 在 Brave 中添加了 .torrc 文件</li> 
 <li>[安全] 修正了在某些情况下 Shields 下的 "允许脚本一次" 不工作的问题</li> 
 <li>改进了 Brave Wallet 的键盘可访问性</li> 
 <li>更新了 Brave Rewards 支付的状态信息</li> 
 <li>……</li> 
</ul> 
<p>更多详情可查看：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fbrave.com%2Flatest%2F" target="_blank">https://brave.com/latest/</a></p>
