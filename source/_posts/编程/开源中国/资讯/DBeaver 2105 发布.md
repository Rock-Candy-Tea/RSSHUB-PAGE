                                                        <p>DBeaver 21.0.5 现已发布，DBeaver是一个免费开源的通用数据库工具，适用于开发人员和数据库管理员。</p> 
<p><strong>主要更新内容</strong></p> 
<ul> 
 <li><strong>Navigator </strong> 
  <ul> 
   <li>重新设计了布尔属性编辑器的行为</li> 
   <li>修复了对象编辑器中破损的快捷方式（F2、F4等）</li> 
   <li>从标准对话框中删除了多余的帮助图标</li> 
   <li>增加了语言变更验证和文档</li> 
   <li>快速过滤器现在支持触发器</li> 
  </ul> </li> 
 <li><strong>SQL 编辑器</strong> 
  <ul> 
   <li>对表的别名、连接和星号的自动完成进行了修复</li> 
   <li>智能提交现在忽略 EXPLAIN 语句</li> 
   <li>增加了对同时更新大量行数（>2GB）的支持</li> 
   <li>增加了输出日志启用 / 禁用按钮</li> 
  </ul> </li> 
 <li><strong>Data 编辑器</strong> 
  <ul> 
   <li>修复了选定行的高亮显示</li> 
   <li>修复了时间戳和 FK 值重叠的问题</li> 
   <li>查找 / 替换对话框现在支持数字格式化</li> 
  </ul> </li> 
 <li><strong>ERD查看器</strong> 
  <ul> 
   <li>修复了属性查看支持，以及实体 / 属性的上下文菜单</li> 
  </ul> </li> 
 <li><strong>PostgreSQL</strong> 
  <ul> 
   <li>增加了对限制条件和索引的支持</li> 
   <li>修复了搜索路径不断增加的问题</li> 
   <li>修复了外键 DDL 中的全限定表名问题</li> 
   <li>PGPASS 认证现在可以使用空用户名了</li> 
   <li>修复了 NaN 和 Infinity 数字的格式化问题</li> 
   <li>修复了 ON DUPLICATE 查询的生成</li> 
   <li>增加了数据库大小统计（可配置，默认为关闭）</li> 
   <li>修复了 CREATE DOMAIN DD</li> 
  </ul> </li> 
</ul>
