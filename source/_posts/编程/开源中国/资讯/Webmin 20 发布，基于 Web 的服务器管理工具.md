                                                                                            <p>Webmin 是一个基于网页的类 Unix 服务器系统管理工具，有了它，用户可以配置操作系统内部，比如用户、磁盘配额、服务或者配置文件，以及修改和控制开源应用，如 BIND DNS Server、Apache HTTP Server、PHP、MySQL 等。</p> 
<ul> 
 <li>在启用 SSL 的模式下增加执行 HTTP Strict Transport Security（HSTS）策略</li> 
 <li>启用 SSL 时，增加更好的 <code>http</code> 到 <code>https</code> 的重定向功能</li> 
 <li>增加在 <code>systemd</code> 系统上安装多个版本 Webmin 的支持</li> 
 <li>增加对 AMD CPU 温度计的支持</li> 
 <li>增加对 Webmin 次要（发布）版本升级的更好支持</li> 
 <li>增加 Webmin 和 Usermin 配置模块显示次要（发布）版本</li> 
 <li>增加对 Linux Mint 的支持</li> 
 <li>添加最新的 Authentic 20.00 主题更新，并修复了一些错误</li> 
 <li>修复了在 <code>firewalld</code> 重启时也会重启依赖服务（如 <code>fail2ban</code>）的问题</li> 
 <li>修复在软件包升级时保留 Webmin 和 Usermin 的服务状态</li> 
 <li>修复了 CentOS 7 上 Webmin 升级时 Bind 模块配置不正确的问题</li> 
</ul>
