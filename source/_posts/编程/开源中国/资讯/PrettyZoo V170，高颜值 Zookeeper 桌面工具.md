                                                                                            <p><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fvran-dev%2FPrettyZoo">PrettyZoo</a> 是一款基于 JavaFX 实现的 Zookeeper 桌面图形化工具，不仅颜值极高，而且功能齐全，支持 Windows / Mac / Linux 等主流平台。</p> 
<p>这次更新增加了期待已久的节点数据高亮功能。</p> 
<p><img alt height="506" src="https://oscimg.oschina.net/oscnet/up-8bacc1f0c90820b95f3e9b013bde05d21bc.png" width="900" referrerpolicy="no-referrer"></p> 
<h3>v1.7.0 发布记录</h3> 
<ol> 
 <li> <p>feature: 自动检测节点数据语法并高亮显示（支持 xml / Json / properties 等语法）</p> </li> 
 <li> <p>feature: 支持放大、缩小节点数据区域（通过双击菜单栏）</p> </li> 
 <li> <p>bug fix: 创建新的 server 以后已连接的 server 会丢失连接状态</p> </li> 
 <li> <p>bug fix: 调整系统字体大小时有可能导致 server 列表项背景色透明</p> </li> 
</ol>
