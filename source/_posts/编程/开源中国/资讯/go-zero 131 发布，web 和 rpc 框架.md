                                                                                            <p style="color:#333333; margin-left:0; margin-right:0; text-align:left">go-zero v1.3.1 发布了。go-zero 是一个集成了各种工程实践的 web 和 rpc 框架。通过弹性设计保障了大并发服务端的稳定性，经受了充分的实战检验。go-zero 包含极简的 API 定义和生成工具 goctl，可以根据定义的 API 文件一键生成 Go, iOS, Android, Kotlin, Dart, TypeScript, JavaScript 代码，并可直接运行。</p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">本次更新内容包括：</p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">框架：</p> 
<ol style="list-style-type:decimal; margin-left:0; margin-right:0"> 
 <li>更新go-redis到v8版本，增加了ctx，并支持了OpenTelemetry</li> 
 <li>更新了sql driver，增加了ctx，并支持了OpenTelemetry</li> 
 <li>在 Cache 包中支持了ctx，并支持了OpenTelemetry</li> 
 <li>记录HTTP 404日志并支持了OpenTelemetry</li> 
 <li>bug fixes和一些优化</li> 
</ol>
