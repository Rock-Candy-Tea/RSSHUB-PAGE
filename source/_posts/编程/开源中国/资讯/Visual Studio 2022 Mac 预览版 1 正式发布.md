                                                        <p>Visual Studio 2022 for Mac 预览版 1 现已发布。这是该 .NET IDE 的第一个公开版本，具有全新的、完全原生的 macOS UI。</p> 
<h4>提供原生的 macOS 体验</h4> 
<p>该版本用原生的 macOS UI 重写了用户界面，以提供舒适的 Mac 用户体验，并为辅助技术提供更好的支持。</p> 
<p><img alt height="372" src="https://oscimg.oschina.net/oscnet/up-3a51fa4ec814279acbb4415ced797a91598.png" width="552" referrerpolicy="no-referrer"></p> 
<h4>提高IDE的速度和稳定性</h4> 
<p>随着迁移到原生 mac OS，该版本还提高了 IDE 性能，解决了长期存在的 UI 错误，并减少了崩溃。</p> 
<h4><span style="background-color:#ffffff; color:#182026">具有新的 Git 变更窗口</span></h4> 
<p>该 Git 变更窗口与 Visual Studio <span style="background-color:#ffffff; color:#182026">集成开发环境之间更加一致。</span></p> 
<p><span style="background-color:#ffffff; color:#182026"><img alt height="412" src="https://oscimg.oschina.net/oscnet/up-85d07fbd5fc7be6feb837ead4964afa9a0c.png" width="250" referrerpolicy="no-referrer"></span></p> 
<h4><span style="background-color:#ffffff; color:#182026">使用 .NET 6 和 C# 10 构建现代应用程序</span></h4> 
<p><span style="background-color:#ffffff; color:#182026">该版本增加了对 .NET 6 和 C# 10 开发的支持，并使用 .NET 6 RC1 版本。</span></p>
