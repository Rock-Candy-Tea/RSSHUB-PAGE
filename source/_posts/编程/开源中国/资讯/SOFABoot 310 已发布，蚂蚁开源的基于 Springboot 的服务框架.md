                                                                                            <p><span style="color:#333333">SOFABoot 是蚂蚁金服开源的基于 Spring Boot 的研发框架，它在 Spring Boot 的基础上，提供了诸如 Readiness Check，类隔离，日志空间隔离等等能力。在增强了 Spring Boot 的同时，SOFABoot 提供了让用户可以在 Spring Boot 中非常方便地使用 SOFA 中间件的能力。</span></p> 
<p><span style="color:#333333">SOFABoot 3.10.0 版本带来以下改动：</span></p> 
<p><span style="color:#24292f"><strong>新特性</strong></span></p> 
<ul> 
 <li>更新 SpringBoot 版本到 2.3.12 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fsofastack%2Fsofa-boot%2Fpull%2F894" target="_blank">#894</a></li> 
</ul> 
<p><span style="color:#24292f"><strong>优化</strong></span></p> 
<ul> 
 <li>发现重复模块时，抛出异常 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fsofastack%2Fsofa-boot%2Fpull%2F888" target="_blank">#888</a></li> 
 <li>某些模块安装失败时，抛出异常 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fsofastack%2Fsofa-boot%2Fpull%2F902" target="_blank">#902</a></li> 
</ul> 
<p><strong>Bug修复</strong></p> 
<ul> 
 <li>修复代码和注释中的一些拼写错误<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fsofastack%2Fsofa-boot%2Fpull%2F895" target="_blank">#895</a></li> 
</ul> 
<p>更新公告：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fsofastack%2Fsofa-boot%2Freleases%2Ftag%2Fv3.10.0" target="_blank">https://github.com/sofastack/sofa-boot/releases/tag/v3.10.0</a></p>
