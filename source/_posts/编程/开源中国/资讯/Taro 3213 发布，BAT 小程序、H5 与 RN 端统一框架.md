                                                        <p>Taro 3.2.13 发布了。Taro 是一个开放式跨端跨框架解决方案，支持使用 React/Vue/Nerv 等框架来开发微信/京东/百度/支付宝/字节跳动/ QQ 小程序/H5 等应用。此版本更新内容包括：</p> 
<h3>特性</h3> 
<h4>小程序</h4> 
<ul> 
 <li>支持支付宝小程序项目配置文件 <code>project.alipay.json</code>，close <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FNervJS%2Ftaro%2Fissues%2F9331" target="_blank">#9331</a></li> 
</ul> 
<h4>React Native</h4> 
<ul> 
 <li>更新 <strong>metro</strong> 版本，统一为 <code>0.64</code></li> 
 <li>支持 <code>sourcemap</code> 相关参数，用于打包时控制输出 <code>sourcemap</code> 文件</li> 
</ul> 
<h3>修复</h3> 
<h4>小程序</h4> 
<ul> 
 <li>修复微信小程序 <code>sourcemap</code> 映射失败的问题，fix <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FNervJS%2Ftaro%2Fissues%2F9412" target="_blank">#9412</a></li> 
 <li>修复使用 <code>jwt-decode</code> 解析 jwt token 时报错的问题，fix <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FNervJS%2Ftaro%2Fissues%2F8360" target="_blank">#8360</a></li> 
 <li>补全字节跳动小程序 <code>Button</code> 组件的 <code>data-channel</code> 属性，fix <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FNervJS%2Ftaro%2Fissues%2F9429" target="_blank">#9429</a></li> 
 <li>补全字节跳动小程序 <code>openEcGood</code> API，fix <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FNervJS%2Ftaro%2Fissues%2F9330" target="_blank">#9330</a></li> 
 <li>补全微信小程序 <code>choosePoi</code> API，fix <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FNervJS%2Ftaro%2Fissues%2F9420" target="_blank">#9420</a></li> 
 <li>补全微信小程序 <code>KeyboardAccessory</code> 组件，fix <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FNervJS%2Ftaro%2Fissues%2F9548" target="_blank">#9548</a></li> 
 <li>补全微信小程序 <code>Video</code> 组件的属性</li> 
</ul> 
<h4>H5</h4> 
<ul> 
 <li>修复 <code>Video</code> 组件在为获取到视频时长时显示 <strong>NaN:NaN</strong> 的问题</li> 
</ul>
