                                                        <p>Apache CloudStack 4.16.0.0 正式发布，该版本是一个 LTS 版本，包含 22 个主要的新功能，以及自 4.15 版本以来的 244 个改进和修复。</p> 
<p>本次更新的一些重点内容包括：</p> 
<ul> 
 <li>对管理服务器和管理主机的额外操作系统支持： 
  <ul> 
   <li>支持 OpenSuse</li> 
   <li>支持 Rocky Linux</li> 
  </ul> </li> 
 <li>新的集成，使 CloudStack 能够利用先进的存储功能： 
  <ul> 
   <li>Dell EMC PowerFlex 存储插件集成</li> 
   <li>LINSTOR 存储插件集成</li> 
  </ul> </li> 
 <li>UI 的改进： 
  <ul> 
   <li>在用户界面中对对象进行结构化注释</li> 
   <li>自定义资源图标</li> 
   <li>能够通过用户界面执行批量操作</li> 
  </ul> </li> 
 <li>增强 CloudStack Kubernetes 服务： 
  <ul> 
   <li>通过 CloudStack Kubernetes 提供商支持 Kubernetes 自动扩展</li> 
   <li>Kubernetes Clusters 现在以系统虚拟机为基础，提供简化的持续管理</li> 
  </ul> </li> 
 <li>改进工具和管理： 
  <ul> 
   <li>对虚拟机的 CPU/RAM 的动态扩展进行细化控制</li> 
   <li>通过用户界面导入和导出 VMware 的虚拟机</li> 
   <li>CloudStack-setup-databases 脚本改进，使升级更稳定</li> 
   <li>HA 的改进允许主机被标记为降级</li> 
   <li>新的 API 端点可以更新 Pod 管理网络 IP 范围</li> 
   <li>新的 API 端点可以更新 VLAN 的 IP 范围</li> 
  </ul> </li> 
</ul>
