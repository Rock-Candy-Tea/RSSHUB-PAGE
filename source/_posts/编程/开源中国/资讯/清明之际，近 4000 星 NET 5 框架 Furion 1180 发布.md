<h2>本期更新</h2> 
<blockquote> 
 <ul> 
  <li> <p><strong>新特性</strong></p> 
   <ul> 
    <li>[新增] <code>Oracle</code> 11 版本支持 <a href="https://gitee.com/dotnetchina/Furion/issues/I3EVL5">#I3EVL5</a></li> 
    <li>[新增] <code>Mysql</code> 官方包 <code>MySql.EntityFrameworkCore</code> 支持 <a href="https://gitee.com/dotnetchina/Furion/issues/I3E6J1">#I3E6J1</a></li> 
    <li>[新增] 全局配置 <code>WebApi</code> 参数 <code>[FromQury]</code> 化 <a href="https://gitee.com/dotnetchina/Furion/issues/I3EFYJ">#I3EFYJ</a></li> 
    <li>[新增] 公开框架底层依赖注入扫描注册拓展 <code>services.AddRisterTypes(types)</code> <a href="https://gitee.com/dotnetchina/Furion/issues/I3EIV3">#I3EIV3</a></li> 
    <li>[新增] SqlSugar 工作单元特性 <a href="https://gitee.com/dotnetchina/Furion/issues/I3EJO5">#I3EJO5</a></li> 
   </ul> </li> 
  <li> <p><strong>突破性变化</strong></p> </li> 
  <li> <p><strong>问题修复</strong></p> 
   <ul> 
    <li>[修复] 数据库上下文池一旦有上下文操作失败还数据库上下文出现二次提交数据库的问题 <a href="https://gitee.com/dotnetchina/Furion/issues/I3EIJJ">#I3EIJJ</a></li> 
    <li>[修复] 不同数据库命令参数前缀都添加了 <code>@</code> 处理 <a href="https://gitee.com/dotnetchina/Furion/issues/I3EBJP">#I3EBJP</a></li> 
    <li>[修复] 尝试修复事件总线线程安全问题 <a href="https://gitee.com/dotnetchina/Furion/issues/I3EGSB">#I3EGSB</a> <a href="https://gitee.com/dotnetchina/Furion/pulls/236">#PR236</a></li> 
    <li>[修复] <code>HttpContextExtensions</code> 的 <code>SignoutToSwagger</code> 方法无效 <a href="https://gitee.com/dotnetchina/Furion/issues/I3EHNQ">#I3EHNQ</a></li> 
    <li>[修复] 如果动态 WebApi 贴了 <code>[ApiController]</code> 特性后，导致路由参数重复生成 <a href="https://gitee.com/dotnetchina/Furion/issues/I3EOQQ">#I3EOQQ</a></li> 
    <li>[修复] 如果没有任何 webapi 控制器时，文档报错 <a href="https://gitee.com/dotnetchina/Furion/issues/I3EVLB">#I3EVLB</a></li> 
    <li>[修复] 依赖注入泛型类型注册失败 <a href="https://gitee.com/dotnetchina/Furion/issues/I3EX66">#I3EX66</a></li> 
   </ul> </li> 
  <li> <p><strong>其他更改</strong></p> 
   <ul> 
    <li>[调整] SqlSugar 拓展库仓储 <code>Context</code> 属性类型未 <code>SqlSugarClient</code> <a href="https://gitee.com/dotnetchina/Furion/issues/I3EHXA">#I3EHXA</a></li> 
    <li>[改进] 刷新 Token 黑名单 Redis 中分组 <a href="https://gitee.com/dotnetchina/Furion/issues/I3EQWO">#I3EQWO</a></li> 
    <li>[改进] 远程请求在请求拦截次发起二次请求导致异常问题 <a href="https://gitee.com/dotnetchina/Furion/issues/I3ER71">#I3ER71</a></li> 
    <li>[改进] 多租户默认缓存改为分布式缓存 <a href="https://gitee.com/dotnetchina/Furion/issues/I3EXEU">#I3EXEU</a></li> 
   </ul> </li> 
  <li> <p><strong>文档变化</strong></p> 
   <ul> 
    <li>[更新] 数据库操作文档 <a href="https://gitee.com/dotnetchina/Furion/issues/I3E84X">#I3E84X</a></li> 
   </ul> </li> 
  <li> <p><strong>问答答疑</strong></p> 
   <ul> 
    <li>[答疑] 如何方便的获取 <code>IDynamicApiController</code> API 产生的 url 和 谓词 <a href="https://gitee.com/dotnetchina/Furion/issues/I3ED17">#I3ED17</a></li> 
    <li>[答疑] Code First -执行命令 <code>Add-Migration</code> 遇到了问题 <a href="https://gitee.com/dotnetchina/Furion/issues/I3EHD0">#I3EHD0</a></li> 
    <li>[答疑] tools v1.16.0 无法生成实体，一直提示 Missing required argument <code><PROVIDER></code>. <a href="https://gitee.com/dotnetchina/Furion/issues/I3ENZ8">#I3ENZ8</a></li> 
    <li>[答疑] Authorize 的 Logout 按钮，无法实时请空 token<a href="https://gitee.com/dotnetchina/Furion/issues/I3EOF9">#I3EOF9</a></li> 
   </ul> </li> 
  <li> <p><strong>不做实现</strong></p> </li> 
 </ul> 
</blockquote>
