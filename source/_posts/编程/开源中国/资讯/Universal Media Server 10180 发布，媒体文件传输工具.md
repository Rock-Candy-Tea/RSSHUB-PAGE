                                                                                            <p>Universal Media Server 是一个兼容 DLNA，UPnP 和 HTTP/S 的媒体服务器，支持所有主要操作系统，包括 Windows，Linux 和 macOS 版本，几乎不需要配置就可以流式传输或转码许多不同的媒体格式。</p> 
<p>自 10.17.1 以来的变化：</p> 
<h3>常规</h3> 
<ul> 
 <li>修正了视频元数据不能写入数据库的问题</li> 
 <li>修正了通过向导添加的共享文件夹没有被保存的问题</li> 
 <li>修复如果 UMS.conf 丢失，用默认值覆盖 WEB.conf 的问题</li> 
 <li>修复了初始共享文件夹不被监控的问题</li> 
 <li>提高了数据库写入的性能</li> 
</ul> 
<h3>渲染器</h3> 
<ul> 
 <li>增加了对更多 LG 蓝光播放器的识别</li> 
</ul> 
<h3>依赖项</h3> 
<ul> 
 <li>将 oshi-core 从 6.1.3 升级到 6.1.4</li> 
</ul> 
<h3>其他</h3> 
<ul> 
 <li>通过 Crowdin 更新翻译</li> 
</ul>
