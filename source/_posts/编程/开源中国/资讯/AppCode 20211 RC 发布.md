                                                        <p>AppCode 2020.1.1 RC 现已发布，并且正式版将在不久之后发布。</p> 
<p><strong>主要更新内容</strong></p> 
<ul> 
 <li>修复 SourceKit 检查在 Xcode 项目 + SPM Firebase 依赖关系中被破坏的问题</li> 
 <li>如果 key 包含 “.&#123;DIGIT&#125;” 、“-&#123;DIGIT&#125;” 或以数字开头，则无法在 Localizable.strings 中找到字符串的使用</li> 
 <li>修复 Swift 软件包中对于指定了 “excluded” 和 “sources” 的目标，解析无法正常工作的问题</li> 
 <li>对于 Swift 软件包，添加文件以正确解析 "sources "或 "exclude" 选项的目标</li> 
 <li>在 Swift 包中支持 "Run With Coverage"</li> 
 <li>修复 AppCode 在编辑器中没有任何错误显示的问题</li> 
 <li>修复 git 分支切换导致 "Failed to update resolve configuration - not found for SwiftPackageMan-agerTargetId" 的问题</li> 
 <li>SwiftPM 增加 “Target.exclude” 属性</li> 
</ul>
