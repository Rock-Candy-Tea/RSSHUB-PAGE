                                                                                            <p><span style="color:#000000">MX Linux 是一个基于 Debian 稳定版的中量级 Linux 操作系统，使用 antiX 的核心组件，以及由 MX 社区创建或打包的额外软件。</span></p> 
<p><span style="color:#000000">目前 </span>MX Linux 21.2.1 发布了， 这是 MX 21.2 版本的特殊更新。因为很多用户对 debian 的 grub-pc 更新有疑问，官方做出了更新，让新的 MX Linux 安装不会遇到开箱即用的 grub 更新。</p> 
<p>其他的更新内亮点包括：</p> 
<ul> 
 <li>Debian 11.5 “bullseye” 基础，包括最新的 grub 更新。</li> 
 <li>内核更新：主要的 Xfce、fluxbox 和 KDE iso 都更新到最新的 5.10 内核。AHS 使用 5.18 内核（带有更新的 linux-firmware 软件包）</li> 
</ul>
