                                                                                            <p># v4.0</p> 
<p>* 彻底重写内核，全部基于 baigo 自主开发的框架 —— ginkgo 进行开发<br> * 做了兼容 PHP 7 的调整<br> * 改用 svg 图标，弃用字体图标<br> * 增加同步组功能，让应用之间的同步更加灵活<br> * 重写了几个常用 jQuery 插件<br> * 改用 PDO 连接数据库<br> * 增加注册界面<br> * 优化界面<br> * 改善安装、升级程序中，变更数据库结构的流程<br> * 修复后台管理中，插件会自动卸载的 bug<br> * 修复 api 通信时，IP 验证的错误<br> * 修复应用无法限制 IP 访问的问题<br> * 修复注册设置中，无法设定允许注册的邮箱、禁止注册的邮箱、禁止注册的用户名问题<br> * 修复 api 接口中，验证邮箱时的逻辑错误<br> * 修复 api 接口中，无法修改邮箱的错误<br> * 修复 api 接口中，同步登录中用户合法性校验的错误</p>

<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">baigo SSO 是一款基于 HTTP 协议的单点登录系统，baigo SSO 以简单为设计、开发的宗旨，安装部署简单、使用简单。baigo SSO 没有复杂的菜单，没有深奥的概念，没有晦涩难懂的名词，一切崇尚简单。</p> 
<h3 style="margin-left:0; margin-right:0; text-align:left">一个账号、全网通行</h3> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">利用 baigo SSO，可以便捷的实现多站点的用户整合，用户使用一个账号，便可以全网通行，无需在多个应用之间重复注册、登录。</p> 
<h3 style="margin-left:0; margin-right:0; text-align:left">基于最流行的语言开发</h3> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">baigo SSO 是采用 PHP + MySQL 开发的单点登录系统。PHP 与 MySQL 可以免费使用，流行广泛，可以运行在 Linux、Windows 或者 Mac 平台，具有良好的通用性。</p> 
<h3 style="margin-left:0; margin-right:0; text-align:left">丰富的 API 接口</h3> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">baigo SSO 采用 HTTP 方式调用 API 接口，可以完全消除兼容问题。部署灵活，可以是单台服务器，也可以是局域网、广域网。</p>
