<ul> 
 <li><span style="color:#24292f">公开所有 Yoga 标题并添加 #ifdef __cplusplus </span> (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffacebook%2Freact-native%2Fcommit%2F43f831b23caf22e59af5c6d3fdd62fed3d20d4ec" target="_blank">43f831b23c</a>)</li> 
</ul>

<ul> 
 <li>对 performance.now() 使用单调时钟 (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffacebook%2Freact-native%2Fcommit%2F114d31feeeb47f5a57419e5088c3cbe9340f757a" target="_blank">114d31feee</a>)</li> 
</ul> 
<h4>针对 iOS</h4> 
<ul> 
 <li><span style="color:#24292f">修复 React-RCTText 的构建（</span> (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffacebook%2Freact-native%2Fcommit%2F4ea38e16bf533955557057656cba5346d2372acd" target="_blank">4ea38e16bf</a> by <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fph4r05" target="_blank">@ph4r05</a>)</li>

 <li>修复 0.69 版本的 use_frameworks! (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffacebook%2Freact-native%2Fcommit%2Ff97c6a5b498eec95e99a02c7842cb2ae160cd6cd" target="_blank">f97c6a5b49</a> )</li> 
</ul>
