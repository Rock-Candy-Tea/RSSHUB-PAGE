                                                        <p>Robocode 1.9.4.2 现已发布。Robocode 是 2001 年 7 月在美国 IBM 的 Web alphaWorks 上发布的坦克机器人战斗仿真引擎。与通常玩的游戏不同的是：参赛者必须利用对机器人进行编程，给机器人设计智能来自动指挥它，而不是由键盘、鼠标简单地直接控制。</p> 
<p>Robocode 是一种有趣的竞赛性编程，使用几行简单的代码，就能够让你创建一个活生生的机器人，一个真正的在屏幕上与其 他机器人互相对抗的机器人。你可以看到它在屏幕上四处疾驰，碾碎一切挡道的东西。机器人配有雷达与火炮，选手在躲避对手进攻的同时攻击对手，以此来较量得分的多少。<strong>Robocode 可以让你在娱乐的同时学习与提高 Java 技术。</strong></p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-8c6aa070621f3406abf40017c88ac769ce2.png" referrerpolicy="no-referrer"></p> 
<p>此版本具体更新内容如下：</p> 
<p><strong>Changes</strong></p> 
<ul> 
 <li>所有机器人都增加了一个 robot index，格式为 name + (index)。这一变化是为了解决团队机器人等方面的各种问题，使内部逻辑更加简单。</li> 
 <li>改进了 TeamRobot 和 Droid 团队机器人的 API 文档。</li> 
</ul> 
<p><strong>Bugfix</strong></p> 
<ul> 
 <li>修复了返回错误名称的 getTeammates() 和对队友返回 false 的 isTeammate() 的问题。</li> 
</ul>
