                                                                                            <p>Android Studio "Arctic Fox" 2020.3.1 Patch 2 现已发布。从 2020.3.1 起，Android Studio 开始启用新的版本号命名方案。事实上，2020.3.1 就是原计划的 4.3。<a href="https://www.oschina.net/news/142143/android-studio-arctic-fox-beta" target="_blank">谷歌称</a>采用新的方案主要是为了与 IntelliJ 的版本对齐。此次更新是该系列的第二个补丁，主要用于修复 bug。</p> 
<p><strong>主要更新内容</strong></p> 
<ul> 
 <li>Android Gradle Plugin 
  <ul> 
   <li>修复从 AS Arctic Fox Canary 7 升级到 Canary 8 时，Gradle 同步启动了 10 次的问题</li> 
   <li>解码和可重现的构建</li> 
  </ul> </li> 
 <li>C++ build 
  <ul> 
   <li>如果使用 tasks.whenTaskAdded 闭包，Android Gradle Plugin 7.0.0 不会在 APK 中包含 jniLibs</li> 
  </ul> </li> 
 <li>Lint 
  <ul> 
   <li>修复 “lint check for lint 检查” 未运行的问题</li> 
  </ul> </li> 
 <li>Shrinker (R8) 
  <ul> 
   <li>修复在 AGP7 中使用 R8 运行构建时出现 ClassNotFoundException 的问题</li> 
  </ul> </li> 
</ul> 
<p>详情请查看<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fandroidstudio.googleblog.com%2F2021%2F09%2Fandroid-studio-arctic-fox-202031-patch.html" target="_blank">更新公告</a>。</p>
