<div class="content">
                                                                    
                                                        <p><span style="background-color:#ffffff; color:#333333">Docker 20.10.16 现已发布。</span><span><span><span style="color:#24292f"><span><span><span><span><span><span><span><span><span><span><span><span style="background-color:#ffffff"><span><span><span>此版本的 Docker Engine 修复了macOS 的 Docker CLI 构建中的一个回归，修复了使用 containerd 1.5 及更高版本时<code>docker stats</code>的一个问题，并更新了 Go 运行时以包含对 <a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fcve.mitre.org%2Fcgi-bin%2Fcvename.cgi%3Fname%3DCVE-2022-29526" target="_blank">CVE-2022-29526</a> 的修复。</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></p> 
<p style="text-align:start"><strong><span><span><span><span><span style="color:#24292f"><span><span><span><span><span><span><span><span><span><span><span style="background-color:#ffffff"><span><span><span>Client</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></strong></p> 
<ul> 
 <li>修复 20.10.15 中引入的 macOS 二进制文件中的回归<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmoby%2Fmoby%2Freleases%2Ftag%2Fv20.10.16%23201015" target="_blank">问题</a>，该问题会导致 panic <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fdocker%2Fcli%2Fpull%2F3592" target="_blank">docker/cli#43426</a>。</li> 
 <li>更新 golang.org/x/sys 依赖项，其中包含对 <a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fcve.mitre.org%2Fcgi-bin%2Fcvename.cgi%3Fname%3DCVE-2022-29526" target="_blank">CVE-2022-29526</a> 的修复。</li> 
</ul> 
<p style="text-align:start"><strong><span><span><span><span><span style="color:#24292f"><span><span><span><span><span><span><span><span><span><span><span style="background-color:#ffffff"><span><span><span>Daemon</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></strong></p> 
<ul> 
 <li>修复了一个问题，即当使用 containerd 1.5.0 或以上版本时，docker stats 会显示空的统计信息 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmoby%2Fmoby%2Fpull%2F43567" target="_blank">moby/moby#43567</a>。</li> 
 <li>更新<code>golang.org/x/sys</code> build-time dependency，其中包含对 <a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fcve.mitre.org%2Fcgi-bin%2Fcvename.cgi%3Fname%3DCVE-2022-29526" target="_blank">CVE-2022-29526</a> 的修复。</li> 
</ul> 
<p style="text-align:start"><strong><span><span><span><span><span style="color:#24292f"><span><span><span><span><span><span><span><span><span><span><span style="background-color:#ffffff"><span><span><span>Packaging</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></strong></p> 
<ul> 
 <li>将 Go 运行时更新到 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgo.dev%2Fdoc%2Fdevel%2Frelease%23go1.17.minor" target="_blank">1.17.10</a>，其中包含对 <a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fcve.mitre.org%2Fcgi-bin%2Fcvename.cgi%3Fname%3DCVE-2022-29526" target="_blank">CVE-2022-29526</a> 的修复。</li> 
 <li>对 docker scan CLI 插件使用“weak”dependencies，以防止当用户从下载的 RPM 包执行离线安装时出现“conflicting requests”错误 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fdocker%2Fdocker-ce-packaging%2Fpull%2F659" target="_blank">docker/docker-ce-packaging#659</a>。</li> 
</ul> 
<p>更新说明：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmoby%2Fmoby%2Freleases%2Ftag%2Fv20.10.16" target="_blank">https://github.com/moby/moby/releases/tag/v20.10.16</a></p>
