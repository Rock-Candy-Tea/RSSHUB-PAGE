                                                                                            <p>Ubuntu 团队宣布为其桌面和服务器发行版<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Flists.ubuntu.com%2Farchives%2Fubuntu-announce%2F2021-September%2F000272.html" target="_blank">发布</a>了 Ubuntu 18.04.6 LTS 更新，更新内容主要是安全修复，包括对 BootHole 安全问题的修复，解决了 Ubuntu 18.04.5 出现的无法启动安装介质的问题。</p> 
<p>发布公告写道，这次的点更新版本与之前的不同，因为在与 BootHole 漏洞相关的密钥撤销后，导致安装介质被损坏，所以 Ubuntu 18.04.6 对 amd64 和 arm64 安装程序介质进行了更新，并在启用 Secure Boot 的系统上重新使用。详细内容查看 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fubuntu.com%2Fblog%2Fgrub2-secure-boot-bypass-2021" target="_blank">https://ubuntu.com/blog/grub2-secure-boot-bypass-2021</a>。</p> 
<p>除此之外，新版本还包括许多其他的安全更新，用于修复高影响错误，同时保证 Ubuntu 18.04 LTS 的稳定性和兼容性。</p> 
<p>下载地址：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fubuntu.com%2Fdownload%2Falternative-downloads" target="_blank">https://ubuntu.com/download/alternative-downloads</a></p> 
<p>Ubuntu 16.04 LTS 的用户将获得自动升级到 18.04.6 通过更新管理器。有关升级的更多信息查看：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fhelp.ubuntu.com%2Fcommunity%2FBionicUpgrades" target="_blank">https://help.ubuntu.com/community/BionicUpgrades</a></p> 
<p>Release Note：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwiki.ubuntu.com%2FBionicBeaver%2FReleaseNotes" target="_blank">https://wiki.ubuntu.com/BionicBeaver/ReleaseNotes</a></p>
