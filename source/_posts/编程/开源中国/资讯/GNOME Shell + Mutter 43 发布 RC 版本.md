                                                                                            <p>GNOME Shell + Mutter 43 发布了候选版本， RC 版本意味着<span style="color:#121212">代码已冻结，不会有新功能更新。</span></p> 
<p>Mutter 是 GNOME 使用的窗口和组合管理器，它通过 OpenGL 显示和管理用户桌面。Mutter 43 RC 版本带来如下更改：</p> 
<ul> 
 <li>切换配置时记住监视器缩放</li> 
 <li>将 Wayland 输出名称嵌入到截屏流中</li> 
 <li>支持最大 BPC 连接器属性，以解决监视器问题</li> 
 <li>改进了后备监视器模式的 heuristics</li> 
 <li><strong>颜色管理</strong>现在由 Mutter 而不是 GNOME-Settings-Daemon 处理</li> 
 <li>支持屏幕外旋转视图的扫描输出</li> 
 <li>避免在屏幕上交换多余的缓冲区部分</li> 
 <li>现在使用原子模式设置带有 GBM 的 NVIDIA 路径</li> 
 <li>崩溃和内存泄漏修复</li> 
</ul> 
<p>通过<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FGNOME%2Fmutter%2Fcommit%2F383c32e7565da5f1ddd0d10ed397e5797c070366" target="_blank">此提交</a>可查看 Mutter 43.rc 的完整更改列表。</p> 
<p>GNOME Shell 43 RC 版本包括以下更改：</p> 
<ul> 
 <li>改进了对重复事件的处理。</li> 
 <li>解决截屏视频的 PipeWire 回归问题。</li> 
 <li>快速设置新增截图项</li> 
 <li>防止 SignalTracker 泄漏对象</li> 
 <li>其他错误修复，包括内存泄漏修复。</li> 
</ul> 
<p>有关 GNOME Shell 43.rc 更改的更多详细信息，请参阅<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FGNOME%2Fgnome-shell%2Fcommit%2F316dc289f2ba2906330d134a7f1a5d574405a682" target="_blank">此提交</a>。</p>
