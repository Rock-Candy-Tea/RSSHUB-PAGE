                                                                                            <p>Calibre 开源项目是 Calibre 官方出的电子书管理工具。它可以查看，转换，编辑和分类所有主流格式的电子书。Calibre 是个跨平台软件，可以在 Linux、Windows 和 macOS 上运行。</p> 
<p>Calibre 5.25.0 正式发布，此次更新内容如下：</p> 
<h3>新功能：</h3> 
<ul> 
 <li>编辑/修饰书籍：移除未使用的 CSS 现在也会移除未引用的样式表；</li> 
 <li>电子书查看器：添加了一些 CSS 变量和类，允许在电子书中编写 Calibre 特定的 CSS；</li> 
 <li>一个新的框架插件，可以用来通知 Calibre 库的变化；</li> 
 <li>编辑元数据对话框：当粘贴到标识符字段时，如果剪贴板包含一个 URL，则直接粘贴为 URL 标识符；</li> 
</ul> 
<h3>错误修复：</h3> 
<ul> 
 <li>电子书查看器：当显示弹出式脚注时，对脚注弹出的大小和标题使用与正文相同的书写方向；</li> 
 <li>电子书查看器：改进在 Google 中查找单词时的文本布局；</li> 
 <li>内容服务器查看器：修复朗读功能在移动端浏览器上无法运行的问题；</li> 
</ul> 
<p>更多详情可查看：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fcalibre-ebook.com%2Fwhats-new" target="_blank">https://calibre-ebook.com/whats-new</a></p>
