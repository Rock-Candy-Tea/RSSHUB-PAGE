                                                                                            <p>距离 GNOME 42 发布 6 个月后，GNOME 于近日正式推出了代号为 "Guadalajara" 的 GNOME 43。最新的 GNOME 版本带来了全面的改进，包括新的快速设置菜单、重新设计的 Files 应用和设备安全设置。GNOME 43 延续了 GNOME 应用程序从 GTK 3 迁移到 GTK 4 的趋势，除此之外还包括许多其他较小的改进。</p> 
<h3><strong>快速设置</strong></h3> 
<p>GNOME 43 带有一个重新设计的系统状态菜单，它允许快速改变常用的设置。以前需要深入菜单的设置，现在只需点击按钮就可以改变设置。新的设计也可以让你一目了然地看到设置状态。</p> 
<p><img alt height="515" src="https://static.oschina.net/uploads/space/2022/0923/080900_GGdF_4937141.png" width="700" referrerpolicy="no-referrer"></p> 
<p>除了使现有的设置更容易使用外，新的设置菜单还包括一些值得注意的新功能：</p> 
<ul> 
 <li>菜单中首次包含了用户界面风格的设置，允许在浅色和深色样式之间切换。此前这一设置只能从 "设置" 应用程序中修改</li> 
 <li>包括一个新的屏幕截图按钮，这是对 GNOME 42 中引入的内置屏幕截图功能的补充</li> 
 <li>当有多个音频设备时，现在可以从菜单中切换它们，无需深入设置应用程序进行调整</li> 
 <li>当 VPN 被关闭时，按下 VPN 按钮将连接到最后使用的网络。</li> 
</ul> 
<h3>更多工具迁移至 GTK 4</h3> 
<p>GTK 4 是 GNOME 的用户界面工具包的主要新版本，于 2020 年 12 月发布。有了新版本，应用程序可以获得更快、更流畅的图形，新的界面部件，以及全新的外观。</p> 
<p>在上一个 GNOME 版本，即 GNOME 42 中，许多 GNOME 应用程序采用了 GTK 4。在 GNOME 43 中，这一改变仍在继续，如今有更多的应用程序从 GTK 3 切换到 GTK 4，这包括：</p> 
<p><img alt height="161" src="https://static.oschina.net/uploads/space/2022/0923/080910_gPb7_4937141.png" width="700" referrerpolicy="no-referrer"></p> 
<h3>Files 应用更新</h3> 
<p>作为向 GTK 4 过渡的一部分，Files 应用程序也有了许多改进：</p> 
<ul> 
 <li>文件和文件夹属性窗口被赋予了一个新的、现代的设计，这使每个项目有了更好的概览</li> 
 <li>该应用程序现在是自适应的，这意味着它可以在不同的宽度下使用，当窗口被调整时，它也将自动调整其布局</li> 
 <li>菜单已经被重新组织，所以它们更有逻辑性、更容易使用</li> 
 <li>搜索结果列表、最近的文件列表和标星文件列表有一个新的布局，更好地显示了每个文件的位置</li> 
 <li>新的打开方式（Open With）对话框可以轻松控制应使用哪些应用程序打开不同类型的文件</li> 
</ul> 
<p><img alt height="478" src="https://static.oschina.net/uploads/space/2022/0923/080921_0FUu_4937141.png" width="700" referrerpolicy="no-referrer"></p> 
<h3>日历、联系人、电话</h3> 
<p>日历、联系人和电话应用程序都针对 GNOME 43 进行了增强：</p> 
<ul> 
 <li>日历有一个更新的界面，新的侧边栏包括一个导航日历和一个即将发生的事件的列表。日历的网格也被赋予了新的色彩，并采用了新的调色板</li> 
 <li>GNOME 43 中的联系人现在允许以 vCard 文件形式导入和导出联系人</li> 
 <li>电话有一系列的改进，包括更快的启动时间，支持加密的 VOIP 通话，以及从通话记录中发送短信的功能</li> 
</ul> 
<h3>设备安全设置</h3> 
<p>GNOME 的隐私设置在第 43 版中包括一个新的设备安全页面。这提供了关于你的物理硬件、其配置和固件的安全信息。</p> 
<p>设备安全设置可以用来检测各种硬件安全问题，包括制造错误和硬件错误配置。新的设置还可以在潜在的安全问题发生时发出警告，如物理设备被篡改等，这可能表明存在恶意软件。</p> 
<h3>Web 应用</h3> 
<p>GNOME 43 包含了将网站作为应用程序安装到桌面的功能。这个功能是 GNOME 以前版本中的 Web 应用程序功能的一个新的改进版本。</p> 
<p>在 Web 中，使用 "将网站安装为 Web 应用程序" 菜单项，网站可以被安装为应用程序。Web 应用可以以通常的方式从 Software 中删除。</p> 
<h3>其他</h3> 
<p>GNOME 43 还包括许多其他的小改进，包括：</p> 
<ul> 
 <li>屏幕键盘现在会在你输入时显示建议，当在终端输入时，它也会显示 Ctrl、Alt 和 Tab 键。</li> 
 <li>Web 的截图功能现在更容易使用了，它现在可以在网页的上下文菜单中找到，或者用 Shift+Ctrl+S 键盘快捷键来触发。</li> 
 <li>同样在 Web 中，网页中的界面元素的风格也被更新了，以符合现代 GNOME 应用程序。</li> 
 <li>Characters 应用现在包括了更多的表情符号，包括不同肤色、性别和发型的人，以及更多的地区旗帜。</li> 
 <li>Activities Overview 中的一些动画已经被优化，使其更加流畅。</li> 
 <li>GNOME 应用程序的 "关于窗口"（显示每个应用程序的详细信息） 已经经过改进</li> 
 <li>在 Software 中，应用程序页面有一个改进的切换器来选择源和格式</li> 
 <li>GTK 4 应用程序所使用的深色 UI 样式已经过优化，因此外观更加和谐</li> 
 <li>当用远程桌面应用程序连接到 GNOME 时，现在可以接收来自主机的音频</li> 
 <li>……</li> 
</ul>
