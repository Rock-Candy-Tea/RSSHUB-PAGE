                                                                                            <blockquote> 
 <p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><strong>VLOOK™</strong><span> </span>是针对<span><span> </span></span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Ftypora.io%2F" target="_blank">Typora</a>（跨平台 Markdown 编辑器）的<span> </span><strong>主题包</strong><span> </span>和<span> </span><strong>增强插件</strong>（针对导出的 HTML 文件)，<strong>旨在与众 Markdown 粉共创 Markdown 的自动化排版 2.0，在保持 Markdown 简洁性的基础上，让编辑、阅读 Markdown 文档更实用，也更愉悦。</strong></p> 
 <p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><strong>VLOOK™</strong><span> </span>属于开源软件（遵从<span> </span><strong>MIT License</strong>），也是<span><span> </span></span><strong><a href="https://www.oschina.net/p/vlook">OSCHINA 开源中国</a></strong><span> </span>推荐的国产开源产品、Typora 的首个增强插件。</p> 
</blockquote> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><strong style="color:#333333">VLOOK™ 的所有特性清单，详见快速入坑 →<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fmadmaxchow.github.io%2FVLOOK%2Findex.html%23%E5%BF%AB%E9%80%9F%E5%85%A5%E5%9D%91" target="_blank"><span>一键了解</span></a><span> </span>（<a href="https://madmaxchow.gitee.io/vlook/index.html#%E5%BF%AB%E9%80%9F%E5%85%A5%E5%9D%91">备用链接</a>）</strong></p> 
<hr> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">间隔 3 个多月后，在实践反馈中继续持续应用和改进，<strong>VLOOK™</strong><span> </span>作为最受欢迎的 Markdown 编辑器 Typora 的主题包和增强插件，释出了最新的 15.0 版本。</p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">在上个版本中对原有的大量特性进行了完善和增强，在这个版本重点对文档编辑体验，以及导出 HTML 后的自动化排版特性进行提升，让 Markdown 粉们继续刷新对 Markdown 文档编辑和阅读的理解和体验。</p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">以下是本次迭代涉及的关键的的新特性、改善点：</p> 
<h1 style="margin-left:0; margin-right:0; text-align:start">🎉<span> </span>NEW 新特性<span> </span>🎉</h1> 
<h2 style="text-align:start">🆕<span> </span>章节自动编号格式支持自定义了</h2> 
<p style="color:#24292f; text-align:start">支持为编号添加：前缀、后缀<br> 指定编号格式：数字格式（阿拉伯数字/英文字母/罗马/中文）、单级、多级、左侧补0、不编号</p> 
<p style="color:#24292f; text-align:start"><strong>效果示例：</strong><br> <code>第1章 xxx</code><br> <code>捌 xxx</code><br> <code>第一百零一回 xxx</code><br> <code>Chapter 3.1 xxx</code><br> <code>Step A.01 - xxx</code></p> 
<p><img height="996" src="https://oscimg.oschina.net/oscnet/up-7f2cb318071f0fb12c731c33d38cea83b5b.png" width="2532" referrerpolicy="no-referrer"></p> 
<p style="color:#24292f; text-align:start">▸ <a href="https://madmaxchow.gitee.io/vlook/guide2.html#%E7%AB%A0%E8%8A%82%E8%87%AA%E5%8A%A8%E7%BC%96%E5%8F%B7">详见这里</a></p> 
<h2 style="text-align:start">🆕<span> </span>新的题注语法，支持编辑时预览</h2> 
<p style="color:#24292f; text-align:start">支持范围：插图/表格/代码块/音频/视频</p> 
<p><img height="556" src="https://oscimg.oschina.net/oscnet/up-8de7dbd8229fecd73d9cef8c84242db7522.png" width="1294" referrerpolicy="no-referrer"></p> 
<p style="color:#24292f; text-align:start">▸ <a href="https://madmaxchow.gitee.io/vlook/guide.html#%E8%A1%A8%E6%A0%BC%E8%87%AA%E5%8A%A8%E7%BC%96%E5%8F%B7%E4%B8%8E%E9%A2%98%E6%B3%A8">详见这里</a></p> 
<h2 style="text-align:start">🆕<span> </span>字体主题增加「系统默认」</h2> 
<p style="color:#24292f; text-align:start">文档字体主题增加「系统默认」选项，自动适配为当前操作系统的默认字体</p> 
<p><img height="798" src="https://oscimg.oschina.net/oscnet/up-1f09b3e6c997e6e431eeeb4578a1372c6f2.png" width="952" referrerpolicy="no-referrer"></p> 
<hr> 
<h1 style="margin-left:0; margin-right:0; text-align:start">▼▼▼ 更多更新说明 ▼▼▼</h1> 
<h2 style="text-align:start">♻️<span> </span>— What's IMPROVED —<span> </span>♻️</h2> 
<p style="color:#24292f; text-align:start"><strong>样式</strong></p> 
<ul> 
 <li>有序/无序列表符号、表格行号、代码行号适配主题色或彩虹引用颜色</li> 
 <li>改善 Typora 编辑时样式（插图缩进、YAML、代码的光标等）</li> 
 <li>适配 Mermaid 样式及优化（flowchart流程图、时序图、状态图、类图等）</li> 
</ul> 
<p style="color:#24292f; text-align:start"><strong>自动排版</strong></p> 
<ul> 
 <li>徽章内容支持更多变量标识：#&#123;变量&#125;，变量变量，var(变量)</li> 
 <li>欢迎页默认内容没有通过YAML指定作者时的版式</li> 
</ul> 
<p style="color:#24292f; text-align:start"><strong>文档交互</strong></p> 
<ul> 
 <li>目录大纲搜索结果、访问历史、章节导航支持显示章节编号</li> 
 <li>目录大纲搜索范围增加章节自动编号自定义内容</li> 
 <li>按下option后显示复制为md图标或提示，包括内容助手</li> 
 <li>标签、徽章鼠标hover时显示为复制图标，带链接的标签仍为手形</li> 
 <li>激活「段落导航」后，可再次三击文档进行关闭</li> 
</ul> 
<p style="color:#24292f; text-align:start"><strong>基础</strong></p> 
<ul> 
 <li>模板主题、插件容量进一步深度瘦身</li> 
 <li>分栏引用在不同分辨率的效果与处理性能</li> 
 <li>文档初始加载时对分栏引用统一高度处理</li> 
 <li>文档滚动时的插件功能适配处理的性能</li> 
 <li>系统默认字体主题，不自动加载webfont</li> 
 <li>插件体积持续优化</li> 
</ul> 
<h2 style="text-align:start">⚠️<span> </span>— What's CHANGED —<span> </span>⚠️</h2> 
<ul> 
 <li>紧跟插图、表格、代码块的引用，样式微调整，在视觉上表示为关联上方内容</li> 
 <li>恢复通过调校参数强制指定字体主题（wf=<code>local</code><span> </span>|<span> </span><code>sans</code><span> </span>|<span> </span><code>serif</code>）</li> 
</ul> 
<h2 style="text-align:start">🧯— What's FIXED —<span> </span>🧯</h2> 
<ul> 
 <li>Solaris 主题的代码块行号样式问题</li> 
 <li>插件体积优化过度带来的bug（如激光笔失效、代码块Dark Mode样式）</li> 
 <li>分栏引用内的二级引用折叠，展示/收起后无法自动适配高度</li> 
 <li>搜索输入内容后，清空按钮错位</li> 
 <li>音频、视频样式的一些问题</li> 
 <li>引用折叠内的svg图片无法正常显示及替换颜色的问题</li> 
</ul>
