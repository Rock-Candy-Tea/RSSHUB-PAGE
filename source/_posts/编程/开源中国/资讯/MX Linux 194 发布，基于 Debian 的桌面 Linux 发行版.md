                                                                                            <p>MX Linux 19 的第四个维护版本 19.4 已发布，此版本的更新内容包括修复 bug 和更新应用程序。</p> 
<p>标准的 MX 19.4（32 位和 64 位版本）采用了最新的 debian 4.19 内核，与过去不同的是，现在的内核将与 Debian 源码一起自动更新。AHS（高级硬件支持） 版本的 ISO 镜像采用 debian 5.10.24 内核、mesa 20.3，以及新升级的固件包。KDE 版本 ISO 也已经更新，它基于 ASH 版本 ISO，和往常一样，这个版本包含了来自 Debian 10.6 (buster) 和 MX 仓库中的最新更新。</p> 
<p>已更新的应用程序：</p> 
<ul> 
 <li>Xfce 4.14 or KDE/plasma 5.15</li> 
 <li>GIMP 2.10.12</li> 
 <li>MESA 18.3.6 (20.3.4 ahs)</li> 
 <li>Latest debian 4.19 kernel (5.10 ahs)</li> 
 <li>Browser: Firefox 87</li> 
 <li>Video Player: VLC 3.0.12</li> 
 <li>Music Manager/Player: Clementine 1.3.1</li> 
 <li>Email client: Thunderbird 68.12.0 (78.9 available in test repo)</li> 
 <li>Office suite: LibreOffice 6.1.5 (plus security fixes) (LO 7 is available in MX-Packageinstaller->Popular Apps)</li> 
 <li>……</li> 
</ul> 
<p>自 19.3 以来 mx-apps 的更新亮点：</p> 
<ul> 
 <li>mx-installer (基于 gazelle-installer) 密码强度指示器和其他改进</li> 
 <li>mx-snapshot – 其他增强功能，包括适用于脚本的新 cli 模式</li> 
 <li>mx-packageinstaller – 更新内核条目和其他改进</li> 
 <li>mx-tweak – 修复自定义主题集功能</li> 
 <li>system-keyboard-qt – 新的工具，用于设置系统范围内的键盘（与 Xfce 键盘工具不一样，它是面向会话的）</li> 
 <li>job-scheduler – 适用用于大多数 Linux 发行版的标准cron 作业系统的 GUI 前端</li> 
</ul> 
<p>此外，许多 mx-apps 进行了翻译方面的更新。</p>

 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fsourceforge.net%2Fprojects%2Fmx-linux%2Ffiles%2FFinal%2FMX-19.4_x64.iso" target="_blank">64 bit Xfce/fluxbox debian with standard 4.19 kernel</a></li>

 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fsourceforge.net%2Fprojects%2Fmx-linux%2Ffiles%2FFinal%2FMX-19.4_kde.iso" target="_blank">64 bit KDE/plasma with AHS 5.10 kernel</a></li> 
</ul> 
<p>MX Linux 是基于 Debian 稳定分支的面向桌面的 Linux 发行，它是 antiX 及早先的 MEPIS Linux 社区合作的产物。它采用 Xfce 作为默认桌面环境，是一份中量级操作系统，并被设计为优雅而高效的桌面与如下特性的结合：配置简单、高度稳定、性能可靠、占用空间不大也不小。</p>
