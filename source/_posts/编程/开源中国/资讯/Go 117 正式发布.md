                                                                                            <p>Go 1.17 已正式<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fblog.golang.org%2Fgo1.17" target="_blank">发布</a>。</p> 
<p><img src="https://static.oschina.net/uploads/space/2021/0818/070126_NqZZ_2720166.png" referrerpolicy="no-referrer"></p> 
<p>此版本改进了编译器，具体来说是采用了<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgolang.org%2Fdoc%2Fgo1.17%23compiler" target="_blank">一种新的函数参数和结果传递方式</a>。官方称此项变化将 Go 程序的性能提升了大约 5%，并将 amd64 平台的二进制包大小减少了大约 2%，未来还计划支持更多平台。</p> 
<p>Go 1.17 还增加了对 Windows 上 64 位 ARM 架构的支持，让 Go 开发者能够在更多设备上原生运行 Go。</p> 
<p>新版本还增加了 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgolang.org%2Fdoc%2Fgo1.17%23go-command" target="_blank">pruned module graphs</a> 功能。官方对此功能的描述为，当 Modules 在其<code>go.mod</code>文件中指定了<code>go 1.17</code>或更高版本，其 module graph 只包括其他 Go 1.17 模块的直接依赖，而不是其全部的横向依赖。这将有助于避免<code>go.mod</code>为其他不相关的依赖下载或读取文件，从而在日常开发中节省时间。</p> 
<p>语言方面，Go 1.17 包含三个针对语言特性的改进：</p> 
<ul> 
 <li>支持<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Ftip.golang.org%2Fref%2Fspec%23Conversions_from_slice_to_array_pointer" target="_blank">从 slice （切片）到数组指针的转换</a></li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Ftip.golang.org%2Fpkg%2Funsafe%23Add" target="_blank"><code>unsafe.Add</code></a>：<code>unsafe.Add(ptr, len)</code>将<code>len</code>添加<code>ptr</code>并返回更新的指针<code>unsafe.Pointer(uintptr(ptr) + uintptr(len))</code></li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Ftip.golang.org%2Fpkg%2Funsafe%23Slice" target="_blank"><code>unsafe.Slice </code></a>: 对于<code>*T</code>类型的<code>ptr</code>表达式，<code>unsafe.Slice(ptr, len) </code>返回一个<code>[]T</code>类型的切片，其底层数组从<code>ptr</code>开始， 其长度和容量为<code>len</code></li> 
 <li>……</li> 
</ul> 
<p>其他的更新内容包括一些功能改进和错误修复，例如针对 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgolang.org%2Fdoc%2Fgo1.17%23crypto%2Fx509" target="_blank">crypto/x509 </a>的验证改进，以及对 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgolang.org%2Fdoc%2Fgo1.17%23semicolons" target="_blank">URL 查询解析的</a>更改。详情查看<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgolang.org%2Fdoc%2Fgo1.17" target="_blank">完整 release note</a>。</p>
