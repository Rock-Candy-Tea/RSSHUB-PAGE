                                                                                            <p><span style="background-color:#ffffff; color:#333333">PHP 8.1.3 现已</span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.php.net%2Farchive%2F2022.php%232022-02-17-3" target="_blank">发布</a><span style="background-color:#ffffff; color:#333333">，这是一个</span><span style="background-color:#ffffff; color:#333333">安全版本</span><span style="background-color:#ffffff; color:#333333">；官方鼓励所有 PHP 8.1 用户升级到此版本。具体更新内容如下：</span></p> 
<ul> 
 <li>Core： 
  <ul> 
   <li>修复了<span style="background-color:#ffffff; color:#333333"> bug </span><a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fbugs.php.net%2F81430" target="_blank">#81430</a>（属性实例化留下悬空指针）。</li> 
   <li>修复了<span style="background-color:#ffffff; color:#333333"> bug </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fphp%2Fphp-src%2Fissues%2F7896" target="_blank">#7896</a>（环境变量可能在 Windows 上被破坏）。</li> 
   <li>修复了<span style="background-color:#ffffff; color:#333333"> bug </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fphp%2Fphp-src%2Fissues%2F7883" target="_blank">#7883</a>（INI 文件不可读时的 Segfault）。</li> 
  </ul> </li> 
 <li>FFI： 
  <ul> 
   <li>修复了<span style="background-color:#ffffff; color:#333333"> bug </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fphp%2Fphp-src%2Fissues%2F7867" target="_blank">#7867</a>（FFI::cast() 从指针到数组的操作被破坏）。</li> 
  </ul> </li> 
 <li>Filter： 
  <ul> 
   <li>修复 #81708：由于 php_filter_float() 导致整数失败的 UAF。(CVE-2021-21708)</li> 
  </ul> </li> 
 <li>FPM： 
  <ul> 
   <li>修复了 invalid port 上的内存泄漏。</li> 
   <li>修复了<span style="background-color:#ffffff; color:#333333"> bug </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fphp%2Fphp-src%2Fissues%2F7842" target="_blank">#7842</a>（FPM 状态页面返回的 OpenMetrics 响应格式无效。</li> 
  </ul> </li> 
 <li>MBString： 
  <ul> 
   <li>修复了<span style="background-color:#ffffff; color:#333333"> bug </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fphp%2Fphp-src%2Fissues%2F7902" target="_blank">#7902</a>（mb_send_mail 可能仅使用 LF 来 delimit headers）。</li> 
  </ul> </li> 
 <li>MySQLnd: 
  <ul> 
   <li>修复了<span style="background-color:#ffffff; color:#333333"> bug </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fphp%2Fphp-src%2Fissues%2F7972" target="_blank">#7972</a>（MariaDB 版本前缀 5.5.5- 未去除）。</li> 
  </ul> </li> 
 <li>pcntl： 
  <ul> 
   <li>修复了 DragonFlyBSD 的 pcntl_rfork 构建。</li> 
  </ul> </li> 
 <li>Sockets： 
  <ul> 
   <li>修复了<span style="background-color:#ffffff; color:#333333"> bug </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fphp%2Fphp-src%2Fissues%2F7978" target="_blank">#7978</a>（sockets extension 编译错误）。</li> 
  </ul> </li> 
 <li>Standard： 
  <ul> 
   <li>修复了<span style="background-color:#ffffff; color:#333333"> bug </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fphp%2Fphp-src%2Fissues%2F7899" target="_blank">#7899</a>（解包中负 int 值的回归）。</li> 
   <li>修复了<span style="background-color:#ffffff; color:#333333"> bug </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fphp%2Fphp-src%2Fissues%2F7875" target="_blank">#7875</a>（即使 log throws exception 也会发送邮件）。</li> 
  </ul> </li> 
</ul> 
<p>更新说明：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.php.net%2FChangeLog-8.php%238.1.3" target="_blank">https://www.php.net/ChangeLog-8.php#8.1.3</a></p>
