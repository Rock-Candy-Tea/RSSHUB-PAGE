                                                        <p>项目介绍：</p> 
<p><span style="background-color:#ffffff; color:#40485b">layui - vue（谐音：类 UI) 是 一 套 Vue 3.0 的 桌 面 端 组 件 库。</span></p> 
<p>更新内容：</p> 
<p><strong>1.0.2</strong></p> 
<p><span style="background-color:#ffffff; color:#40485b">[新增] button 组件 border-style 属性, 自定义边框样式。</span><br> <span style="background-color:#ffffff; color:#40485b">[新增] carousel 组件 interval 属性, 用于控制轮播间隔。</span><br> <span style="background-color:#ffffff; color:#40485b">[新增] config-provider 组件 dark-partial 属性, 夜间主题偏好配置。</span><br> <span style="background-color:#ffffff; color:#40485b">[新增] carousel 组件 autoplay 属性, 开启自动轮播, 默认为 true。</span><br> <span style="background-color:#ffffff; color:#40485b">[修复] page 组件 pages 属性为单数时, 页码计算错误。</span><br> <span style="background-color:#ffffff; color:#40485b">[修复] panel 组件 background-color 为透明的问题。</span><br> <span style="background-color:#ffffff; color:#40485b">[修复] tab 组件 layui-tab-head 默认存在背景色。</span><br> <span style="background-color:#ffffff; color:#40485b">[修复] button-group 组件 first-child 样式。</span><br> <span style="background-color:#ffffff; color:#40485b">[修复] dark 模式 step 组件线条不显示。</span><br> <span style="background-color:#ffffff; color:#40485b">[修复] scroll 组件屏幕缩放时样式异常</span></p> 
<p><strong>1.0.1</strong></p> 
<p><span style="background-color:#ffffff; color:#40485b">[新增] select 组件 search 事件, 搜索回调。</span><br> <span style="background-color:#ffffff; color:#40485b">[新增] select 组件 keyword 属性, 提供关键词检索。</span><br> <span style="background-color:#ffffff; color:#40485b">[新增] upload 组件 cut cutOptions 属性, 支持上传裁剪。</span><br> <span style="background-color:#ffffff; color:#40485b">[新增] timeline 组件 direction 属性, 支持垂直与水平布局。</span><br> <span style="background-color:#ffffff; color:#40485b">[新增] dropdown 组件 open 与 hide 事件,于打开与关闭后的逻辑处理。</span><br> <span style="background-color:#ffffff; color:#40485b">[新增] dropdown 组件 disabled 属性, 禁止下拉框展示。</span><br> <span style="background-color:#ffffff; color:#40485b">[新增] date-picker 组件 disabled 属性, 禁止修改。</span><br> <span style="background-color:#ffffff; color:#40485b">[修复] date-picker 组件右下角实时数据无法联动。</span><br> <span style="background-color:#ffffff; color:#40485b">[修复] table 组件 datasource 属性丢失响应。</span><br> <span style="background-color:#ffffff; color:#40485b">[修复] input 组件 readonly 属性不生效。</span></p> 
<p><span style="color:#40485b">layui-vue 已投递至 layui 组织：</span><a href="https://gitee.com/layui/layui-vue">前往</a></p>
