                                                                                            <p>pgAdmin 是 PostgreSQL 领先的开源图形化管理工具。pgAdmin 4 旨在满足新手和有经验的 Postgres 用户的需求，提供强大的图形界面，简化了数据库对象的创建、维护和使用。</p> 
<p>这个版本的 pgAdmin 4 包括错误修复和新功能，主要更新内容如下：</p> 
<h3>新特性</h3> 
<ul> 
 <li>在调试器中为断点区域增加了鼠标悬停指示</li> 
 <li>在调试器面板上增加了搜索文本选项</li> 
 <li>增加了在 Microsoft Azure 上部署 PostgreSQL 服务器的功能</li> 
 <li>增加支持使用 Docker Secret 向 Docker 镜像传递密码</li> 
 <li>增加了 "Show template databases" 选项，以显示模板数据库，而不考虑 "Show system objects" 的设置。</li> 
 <li>在查询工具中增加了对使用折线图可视化图表的支持</li> 
</ul> 
<h3>Housekeeping</h3> 
<ul> 
 <li>将调试器移植到 React</li> 
 <li>更新 Traefik v2 容器部署的文档</li> 
 <li>将 pgcli 更新至最新版本 3.4.1</li> 
 <li>将 Chartjs 升级到最新的 3.8.0</li> 
</ul> 
<h3>错误修复</h3> 
<ul> 
 <li>修正了在查询工具中没有设置可以关闭通知的问题</li> 
 <li>修正了在 "连接到服务器" 对话框中输入的密码被截断的问题</li> 
 <li>确保在查询工具标签之间切换时，查询编辑器应被聚焦</li> 
 <li>修正了在查询工具中 "使用空格" 不起作用的问题</li> 
 <li>修正了数据库限制不起作用的问题</li> 
 <li>修正了 pgAdmin 在创建新索引时卡住的问题</li> 
 <li>修正了当用户切换到一个新的连接并关闭查询工具时，连接没有被关闭的问题</li> 
 <li>如果 JSON 信息不能被解析，则跳过历史记录，而不是显示 "无历史记录"</li> 
 <li>修正了创建新数据库时显示错误信息的问题</li> 
 <li>修正了使用 Docker 在 Azure 部署 PostgreSQL 时的权限拒绝错误</li> 
</ul>
