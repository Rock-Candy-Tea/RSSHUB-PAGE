                                                                                            <p>PhpStorm 2021.2.1 RC 现已发布。</p> 
<p><strong>主要更新内容</strong></p> 
<ul> 
 <li>为 PSR-12 禁用 “函数/构造函数调用参数|多行时对齐”</li> 
 <li>重新分配的文件类型通配符在 IDE 重启时丢失</li> 
 <li>无法创建 groovy 脚本</li> 
 <li>“将选项用作元键” 选项状态在 IDE 重新启动后未保存</li> 
 <li>如果设置从 2021.1.x 迁移，则Splash 不会在 2021.2 中显示</li> 
 <li>无法打开没有扩展名的空文件</li> 
 <li>如果 IDEA 处于全屏模式，则窗口视图模式下的工具窗口和非模态窗口将以全屏模式打开</li> 
 <li>全局菜单在 KDE 中不再响应（2021.2.1 - 212.5080.8 版本）</li> 
 <li>运行配置：单击“运行”按钮执行与组合框中显示的配置不同的配置，组合框停止工作</li> 
 <li>Vue：在脚本设置中标记为未使用的间接组件导入</li> 
 <li>更新 MDN 文档 2021.2.1</li> 
 <li>修复无限索引</li> 
 <li>Inspections：代码中内联的 HTML 属性的误报</li> 
 <li>修复 jsdoc 中的“无效类型语法”</li> 
 <li>如果代码中有 `*/`，JSDoc 存根生成在嵌入式 Javascript 中失败</li> 
 <li>Vue3：如果组件中有多个脚本标签，则无法解析 “脚本设置” 部分中的导入</li> 
 <li>Vue：不应报告绑定数据属性的“无法识别的属性或属性名称”</li> 
 <li>PhpStorm 将一些 sass 映射函数突出显示为未知的 css 函数。</li> 
 <li>pnpm 工作区：2021.2 RC 升级中 Typescript 和 ES 模块的符号解析中断</li> 
</ul>
