<div class="content">
                                                                    
                                                        <p>Anolis OS 首个社区发行版本 Anolis OS 8.2 RC1<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fopenanolis.org%2Fnews%2F263950449681794731" target="_blank"> 已发布</a>。</p> 
<p>Anolis OS 8 是 OpenAnolis 社区发行的开源 Linux 发行版，与 RHEL 8 100% 兼容， 支持多计算架构，提供稳定、高性能、安全、可靠的操作系统。8.2 RC1 是Anolis OS 8 首个发布版本。</p> 
<p><img src="https://oscimg.oschina.net/oscnet/up-200f025385f3def700d0d88525c612a5f58.JPEG" referrerpolicy="no-referrer"></p> 
<h1>发布内容</h1> 
<p>Anolis OS 8 RC1 发布内容包括 <strong>ISO、虚拟机镜像和 repo 源</strong>。</p> 
<ul> 
 <li><strong>ISO 列表</strong></li> 
</ul> 
<p>AnolisOS-8.2-RC1-x86_64-dvd.iso - x86_64 架构的基础安装 ISO，包含了运行最小系统的核心组件</p> 
<p><a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fmirrors.openanolis.org%2Fanolis%2F8%2Fiso%2Fx86_64%2FAnolisOS-8.2-RC1-x86_64-dvd.iso" target="_blank">http://mirrors.openanolis.org/anolis/8/iso/x86_64/AnolisOS-8.2-RC1-x86_64-dvd.iso</a></p> 
<p>AnolisOS-8.2-RC1-aarch64-dvd.iso - aarch64 架构的基础安装 ISO，包含了运行最小系统的核心组件</p> 
<p><a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fmirrors.openanolis.org%2Fanolis%2F8%2Fiso%2Faarch64%2FAnolisOS-8.2-RC1-aarch64-dvd.iso" target="_blank">http://mirrors.openanolis.org/anolis/8/iso/aarch64/AnolisOS-8.2-RC1-aarch64-dvd.iso</a></p> 
<p>AnolisOS-8.2-RC1-src-dvd.iso - Anolis OS 8.2 源码 ISO</p> 
<p><a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fmirrors.openanolis.org%2Fanolis%2F8%2Fiso%2Fsource%2FAnolisOS-8.2-RC1-src-dvd.iso" target="_blank">http://mirrors.openanolis.org/anolis/8/iso/source/AnolisOS-8.2-RC1-src-dvd.iso</a></p> 
<ul> 
 <li><strong>虚拟机镜像列表</strong></li> 
</ul> 
<p>AnolisOS-8.2-RC1-x86_64.qcow2 - x86_64 架构下 AnolisOS 8.2 虚拟机镜像</p> 
<p><a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fmirrors.openanolis.org%2Fanolis%2F8%2Fiso%2Fx86_64%2FAnolisOS-8.2-RC1-x86_64.qcow2" target="_blank">http://mirrors.openanolis.org/anolis/8/iso/x86_64/AnolisOS-8.2-RC1-x86_64.qcow2</a></p> 
<p>AnolisOS-8.2-RC1-aarch64.qcow2 - aarch64 架构下 AnolisOS 8.2 虚拟机镜像</p> 
<p><a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fmirrors.openanolis.org%2Fanolis%2F8%2Fiso%2Faarch64%2FAnolisOS-8.2-RC1-aarch64.qcow2" target="_blank">http://mirrors.openanolis.org/anolis/8/iso/aarch64/AnolisOS-8.2-RC1-aarch64.qcow2</a></p> 
<ul> 
 <li><strong>REPO 源列表</strong></li> 
</ul> 
<p>BaseOS 软件包源，该源目的是提供安装基础的所有核心包</p> 
<p><a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fmirrors.openanolis.org%2Fanolis%2F8%2FBaseOS%2F" target="_blank">http://mirrors.openanolis.org/anolis/8/BaseOS/</a></p> 
<ul> 
 <li><strong>AppStream</strong></li> 
</ul> 
<p>AppStream 软件包源，该源提供额外的多场景，多用途的用户态程序，数据库等。该部分引入了额外的 RPM Module 形态。</p> 
<p><a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fmirrors.openanolis.org%2Fanolis%2F8%2FAppStream%2F" target="_blank">http://mirrors.openanolis.org/anolis/8/AppStream/</a></p> 
<h1>硬件支持</h1> 
<ul> 
 <li> <p>最小硬件要求：4GB 内存、20GB 可用空间、64 位 x86 或者 ARM 系统</p> </li> 
 <li> <p>支持 AMD 架构、Intel 64 位架构，以及 64 位 ARM 架构</p> </li> 
</ul>
