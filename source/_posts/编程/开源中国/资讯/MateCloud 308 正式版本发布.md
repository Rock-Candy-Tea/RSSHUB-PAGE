                                                        <h1>关于MateCloud</h1> 
<p><span style="background-color:#ffffff; color:#40485b">MateCloud是一款基于Spring Cloud Alibaba的微服务架构。目前已经整合Spring Cloud Gateway、Spring Security Oauth2、Feign、Dubbo、JetCache、RocketMQ等服务套件，让您只专注于业务！</span></p> 
<h1><span style="color:#40485b">版本更新</span></h1> 
<p>此次版本紧跟最新的技术栈，其中Spring Boot 2.4.5、Spring Cloud 2020.0.2、Spring Cloud Alibaba 2021.1，适合尝鲜者关注。</p> 
<h1>功能升级</h1> 
<ul> 
 <li>大版本升级：Spring Boot、Spring Cloud、Spring Cloud Alibaba均升级到最新版本</li> 
 <li>感谢@madi1113提交的灰度部分的代码</li> 
 <li>引入hutool-core</li> 
 <li>感谢@xiang提供的docker的部署脚本</li> 
 <li>mate-starter-security增加资源管理功能</li> 
 <li>感谢@yunfei08提供的多租户优化功能</li> 
</ul> 
<h1>依赖升级</h1> 
<ul> 
 <li>升级至Spring Boot 2.4.5</li> 
 <li>升级至Spring Cloud 2020.0.2</li> 
 <li>升级至Spring Cloud Alibaba 2021.1</li> 
</ul>
