                                                                                            <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">AppWeb 8.2.5 现已发布，此版本被划分为“建议升级，但不是必须的”。Appweb 是一个嵌入式 HTTP Web 服务器，主要的设计思路是安全，它直接集成到客户的应用和设备，便于开发和部署基于 Web 的应用程序和设备。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">具体更新内容：</p> 
<ul style="margin-left:0; margin-right:0"> 
 <li><span style="color:#2e3033">修复范围请求错误</span></li> 
 <li><span style="color:#24292f">使用非简单管道修复中断的大文件传输。</span></li> 
 <li>修复删除的队列检测（针对错误的范围请求进行了更改）</li> 
 <li><span style="color:#2e3033">完成单元测试的修复</span></li> 
 <li><span style="color:#2e3033">增加每个 HTTP/2 连接的默认最大流</span></li> 
 <li><span style="color:#2e3033">添加许可证版权</span></li> 
</ul>
