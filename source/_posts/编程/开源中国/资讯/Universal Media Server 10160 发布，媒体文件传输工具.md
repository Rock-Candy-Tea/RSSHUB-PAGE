                                                                                            <p>Universal Media Server 是一个兼容 DLNA，UPnP 和 HTTP/S 的媒体服务器，支持所有主要操作系统，包括 Windows，Linux 和 macOS 版本，几乎不需要配置就可以流式传输或转码许多不同的媒体格式。</p> 
<p>自 10.15.0 以来的变化：</p> 
<ul> 
 <li>通用： 
  <ul> 
   <li>改进了对电视剧和电影的检测</li> 
   <li>改进了缓存性能和资源使用</li> 
   <li>使关闭功能变得更可靠</li> 
   <li>使得恢复文件出现在原始文件条目之后而不是之前</li> 
   <li>修复了过滤文件夹名称的问题</li> 
   <li>修正了当文件不能被解析时冻结的扫描</li> 
   <li>调整了音轨排序</li> 
   <li>增加了配置标志 sort_audio_tracks_by_album_position</li> 
   <li>显著的代码改进</li> 
  </ul> </li> 
 <li>依赖项 
  <ul> 
   <li>将 assertj-core 从 3.21.0 升级到 3.22.0</li> 
   <li>将 h2 从 2.0.204 升级到 2.0.206</li> 
   <li>将 maven-compiler-plugin 从 3.8.1 升级至 3.9.0</li> 
   <li>将 spotbugs-maven-plugin 从 4.5.2.0 升级到 4.5.3.0</li> 
   <li>将 thumbnailator 从 0.4.15 升级到 0.4.16</li> 
   <li>将 tsMuxeR 从 2021-11-14 升级到 2022-01-13</li> 
  </ul> </li> 
 <li>通过 Crowdin 更新了多个国家/地区的语言翻译</li> 
</ul> 
<p>更多详情可查看：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FUniversalMediaServer%2FUniversalMediaServer%2Freleases" target="_blank">https://github.com/UniversalMediaServer/UniversalMediaServer/releases</a></p>
