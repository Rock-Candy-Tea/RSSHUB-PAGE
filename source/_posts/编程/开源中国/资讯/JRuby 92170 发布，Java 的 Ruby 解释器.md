                                                        <p>JRuby 9.2.17.0 已经发布。JRuby 9.2.x 与 Ruby 2.5.x 兼容，并与 C Ruby 保持同步。和以往一样，该版本用于修复一些最近报告的问题，并鼓励所有用户升级。</p> 
<p><strong>主要更新内容</strong></p> 
<ul> 
 <li>加强对 JRuby 上使用 Sorbet 的支持</li> 
 <li>改进启动 JRuby 时对 Java 模块支持的检测</li> 
 <li>RubyGems 更新至3.1.6</li> 
 <li>修改了 zlib，使其能够正确处理 inflate 完成后流中的额外数据</li> 
 <li>进行较小的更新，以使标准库与 Ruby 2.5.8 保持一致</li> 
 <li>优化 dig 函数</li> 
 <li>修复 zlib 解压缩时产生错误输出的问题</li> 
 <li>更新 Windows jruby.exe 以进行模块检测</li> 
 <li>解除 CachingCallSite.isBuiltin</li> 
</ul>
