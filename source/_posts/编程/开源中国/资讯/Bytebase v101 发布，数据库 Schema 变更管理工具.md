                                                                                            <p style="margin-left:0px">Bytebase 是一个基于网络、零配置、无依赖的数据库 Schema 变更和版本控制管理工具，适用于开发人员和 DBA。</p> 
<p style="margin-left:0px">Bytebase 1.0.1 发布，更新内容如下：</p> 
<h3 style="margin-left:0px">特性</h3> 
<ul> 
 <li>支持实例只读连接。配置完成后，Bytebase 将使用该用户在 SQL Editor 中查询数据库，并将在未来的所有只读场景中使用它。</li> 
</ul> 
<h2><strong>增强功能</strong></h2> 
<ul> 
 <li>[租户模式] 为租户模式项目启用数据更改 (DML)。</li> 
 <li>[SQL Schema Review] 允许用户编辑 SQL 语句。</li> 
 <li>[SQL Schema Review] 允许使用无效的 SQL 语句重试失败的迁移。</li> 
 <li>[SQL 编辑器] 更改保存表逻辑。单击保存按钮或使用快捷方式 (CMD + S) 时保存工作表。</li> 
 <li>[安装改进] 改进了 Linux 平台的安装。</li> 
</ul> 
<h2><strong>值得注意的 Bug 修复</strong></h2> 
<ul> 
 <li>多项安全增强功能。</li> 
</ul>
