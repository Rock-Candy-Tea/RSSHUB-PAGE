                                                        <p>Keka 是一个 macOS 端的应用程序，可以快速创建具有高压缩率的压缩文件，还支持从多种类型的文件中解压缩文件，其中包括 7z、ISO、DMG、TAR、ZIP 和 Bzip2 等类型的文件格式。</p> 
<p>近日，Keka 更新至 1.2.50 版本，本次更新的内容如下：</p> 
<ul> 
 <li>适配于 macOS 新设计规范的全新圆角矩形应用程序图标</li> 
 <li>首选项中的新选项可以调整 Dock 中的任务计数器</li> 
 <li>更新 Sparkle 到最新的 2.0.0-beta.4</li> 
 <li>格式 
  <ul> 
   <li>添加了对 macOS 10.13 以上版本的 Apple Archive 的初始支持</li> 
   <li>添加了对 WordPress Backup（.wpress）提取支持</li> 
   <li>添加了对 InstallShield 3 (.Z) 提取支持</li> 
   <li>不要注册为 XIP 的默认提取工具</li> 
  </ul> </li> 
 <li>修复 
  <ul> 
   <li>完善了首选项和高级窗口的尺寸</li> 
   <li>修正了当剪切板为空时使用名称替换的问题</li> 
   <li>修正了 Finder Etension Preferences 表的操作</li> 
   <li>修正了 ZSTD 默认处理的问题</li> 
  </ul> </li> 
 <li>翻译 
  <ul> 
   <li>更新了包括简体中文和繁体中文在内的多国语言的翻译</li> 
  </ul> </li> 
</ul> 
<p>更多详情可查看：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Faonez%2FKeka%2Freleases" target="_blank">https://github.com/aonez/Keka/releases</a></p>
