<div class="content">
                                                                    
                                                        <p>微软宣布推出 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fblogs.windows.com%2Fwindowsdeveloper%2F2021%2F03%2F29%2Fannouncing-project-reunion-0-5%2F" target="_blank">Project Reunion 0.5</a> 版本。这是第一个生产就绪（production-ready）版本，也是第一个包含 WinUI 3 的完整版本。Project Reunion 仍处于初期阶段，但 0.5 版本中包含了对<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdocs.microsoft.com%2Fwindows%2Fmsix%2Foverview" target="_blank"><strong>打包的 MSIX 桌面应用程序的</strong></a>生产就绪支持，以供使用：</p> 
<ul> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdocs.microsoft.com%2Fwindows%2Fapps%2Fwinui%2Fwinui3%2F" target="_blank">WinUI 3</a></li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdocs.microsoft.com%2Fwindows%2Fapps%2Fproject-reunion%2Fdwritecore" target="_blank">文本渲染</a></li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdocs.microsoft.com%2Fwindows%2Fapps%2Fproject-reunion%2Fdwritecore" target="_blank">资源管理</a></li> 
</ul> 
<p><strong>Project Reunion</strong><strong> 0.5 与 0.5 preview 相比有什么新变化？</strong></p> 
<p>此前在预览版中引入的大部分功能现在已经稳定下来，可以在生产应用程序中使用，包括（但绝对不限于）：</p> 
<ul> 
 <li>能够使用 WinUI 创建桌面应用程序，包括 .NET 5 for Win32 apps。</li> 
 <li>基于 Chromium 的 WebView2 控件</li> 
 <li>自定义标题栏支持</li> 
 <li>ARM64 支持</li> 
 <li>SwapChainPanel</li> 
</ul> 
<p>值得注意的是，由于这是一个稳定的版本，预览功能已经从这个版本的 WinUI 3 中移除，其中包括对构建 UWP应用程序的支持。用户仍然可以通过使用之前的 Project Reunion 0.5 preview 来访问这些功能。</p> 
<p>详情可查看<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmicrosoft%2FProjectReunion%2Freleases%2Ftag%2F0.5.0" target="_blank">更新说明</a>。</p>
