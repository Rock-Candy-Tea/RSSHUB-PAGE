                                                        <p>KeePass 2.6.6 现已发布。KeePassXC  是由 KeePass 密码管理器的一个分支发展而来的，相比原版 KeePass，KeePassXC 除了适配 Windows，还适配了 Linux 和 macOS。通过 KeePassXC 可以存储和管理个人的敏感信息。保存在 KeePassXC 上的信息全部都会进行离线加密，借助于云存储服务可以实现跨平台同步。</p> 
<p>本次具体更新内容如下：</p> 
<p><strong>Fixed</strong></p> 
<ul> 
 <li>修复了按 hotkey 时的 focusing search <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fkeepassxreboot%2Fkeepassxc%2Fpull%2F6603" target="_blank">#6603</a></li> 
 <li>在处理之前从 TOTP 键输入中修剪空格 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fkeepassxreboot%2Fkeepassxc%2Fpull%2F6604" target="_blank">#6604</a></li> 
 <li>修复在 macOS 上的构建 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fkeepassxreboot%2Fkeepassxc%2Fpull%2F6598" target="_blank">#6598</a></li> 
 <li>解决未使用的返回值的编译器警告问题 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fkeepassxreboot%2Fkeepassxc%2Fpull%2F6607" target="_blank">#6607</a></li> 
</ul>
