                                                                                            <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">Universal Media Server 是一个兼容 DLNA，UPnP 和 HTTP/S 的媒体服务器，支持所有主要操作系统，包括 Windows，Linux 和 macOS 版本，几乎不需要配置就可以流式传输或转码许多不同的媒体格式。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">自 10.16.0 以来的变化：</p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>通用： 
  <ul style="margin-left:0; margin-right:0"> 
   <li>增加了一个新的、实验性的 HTTP 服务器</li> 
   <li>修复通过 Tvheadend 播放电视台和其他网络内容时偶尔出现的问题</li> 
   <li>网络界面使用 SSE 而不是轮询，以提高效率</li> 
   <li>增加了对 MusicBrainz 专辑进行 UPnP 搜索的支持</li> 
  </ul> </li> 
 <li>渲染器： 
  <ul style="margin-left:0; margin-right:0"> 
   <li>改进了对三星 5300 系列的支持</li> 
  </ul> </li> 
 <li>改进多国语言翻译</li> 
 <li>依赖项： 
  <ul style="margin-left:0; margin-right:0"> 
   <li>将 checkstyle 从 9.2.1 升级到 9.3</li> 
   <li>将 h2 从 2.0.206 升级到 2.1.210</li> 
   <li>将 jaudiotagger 从 2.2.5 升级到 3.0.1</li> 
   <li>将 JRE 从 8u312 升级到 8u322</li> 
   <li>将 junrar 从 7.4.0 升级到 7.4.1</li> 
   <li>将 maven-project-info-reports-plugin 从 3.1.2 升级到 3.2.1</li> 
  </ul> </li> 
</ul> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">更多详情可查看：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Ffreshfoss.com%2Fprojects%2Funimediaserver%2F" target="_blank">https://freshfoss.com/projects/unimediaserver/</a></p>
