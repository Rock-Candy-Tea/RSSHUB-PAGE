                                                        <p>San 是一个由百度开源的快速、轻量、灵活的 JavaScript 组件框架。San 3.10.6 正式发布，更新内容如下：</p> 
<ul> 
 <li>【优化】- s-is 支持 html 内置标签以及 fragment</li> 
 <li>【优化】- 模板预热时不再创建 HTMLElement 源节点</li> 
 <li>【优化】- 优化模板预热的结构与性能</li> 
 <li>【优化】- nextTick 实现增加对 setImmediate 是否原生方法的判断</li> 
 <li>【bug修复】- 忽略模板解析时尾部多余的空文本节点。该问题为 3.10.3 引入</li> 
</ul>
