                                                                                            <p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><span style="color:#333333">CakePHP 是一个运用了诸如 ActiveRecord、Association Data Mapping、Front Controller 和 MVC（model–view–controller） 等著名设计模式的开源 Web 框架。CakePHP 用 PHP 编写，以 Ruby on Rails 的概念为模型，并在 MIT 许可下进行分发。</span></p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">CakePHP 4.4.5 已发布，这是 4.4 分支的维护版本，主要有如下改动</p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>改进了对 SMTP 电子邮件传输中身份验证错误的处理。</li> 
 <li>重新启用：在开发错误页面上的堆栈帧中显示参数。</li> 
 <li>改进 API 文档。</li> 
 <li>修复了<span> </span><code>diffForHumans()</code><span> </span>不根据当前语言环境翻译文本的问题。</li> 
 <li><code>ValidatableInterface</code><span> </span>已弃用。</li> 
</ul>
