                                                                                            <p>Microsoft PowerToys 是 Windows 系统实用程序，供高级用户调整和简化其 Windows 体验，可最大限度地提高生产力。</p> 
<p>距离之前的 0.51.1 版本仅一个多月时间，微软近日发布了 0.53.1。该版本主要带来了三个重要功能，其中包括文件资源管理器预览窗格和缩略图中的 G 代码支持，PowerToys Run 通过 <code>??</code> 触发词提供的全新网络搜索功能，以及通过 <code>Win+Ctrl+T</code> 快捷键提供的 Always on Top 工具。</p> 
<h3>Always on Top</h3> 
<ul> 
 <li> <p><strong>通过 Win+Ctrl+T 快捷键可激活 Always on Top 功能，此时聚焦的窗口会被切换并始终保持在桌面顶部。再次触发快捷键就会恢复到正常状态。</strong></p> <p><img alt height="447" src="https://static.oschina.net/uploads/space/2022/0110/071748_z3tO_4937141.png" width="700" referrerpolicy="no-referrer"></p> </li> 
</ul> 
<h3>ColorPicker</h3> 
<ul> 
 <li>对调整颜色菜单的 HEX 输入进行了改进，包括支持无 hashtag 的十六进制代码和短的十六进制代码，如 #CF0</li> 
</ul> 
<h3>FancyZones</h3> 
<ul> 
 <li>修复了不抓取子窗口的问题</li> 
 <li>修复了编辑器启动时清除键盘焦点的问题</li> 
</ul> 
<h3>File Explorer</h3> 
<ul> 
 <li><strong>增加了对缩略图和预览窗格的 G 代码支持</strong></li> 
</ul> 
<h3>PowerRename</h3> 
<ul> 
 <li>现在实现了行高亮+预览支持</li> 
 <li>修正了 AltGR 输入问题</li> 
 <li>改进了文件夹重命名支持</li> 
</ul> 
<h3>PowerToys Run</h3> 
<ul> 
 <li><strong>增加了网络搜索功能，使用触发词 「<code>??</code>+需要搜索的内容」将直接通过默认浏览器和默认搜索引擎来查找对应内容。</strong></li> 
 <li>VS Code 工作区的改进</li> 
 <li>支持二进制和十六进制数字</li> 
 <li>在计算中使用阶乘的功能</li> 
 <li>PT Run 将不再显示在 Window Walker 结果中</li> 
 <li>修复 log / ln 的计算</li> 
 <li>修复检测符号链接和防止递归循环的问题</li> 
 <li>修复触控板滚动过快的问题</li> 
 <li>删除了不需要的 nuget 包</li> 
 <li>更好地检测打包的应用程序是否可以被提升</li> 
 <li>改进程序插件的崩溃恢复能力</li> 
 <li>改进了 Windows 设置结果</li> 
 <li>修正了一些类似的激活短语没有达到预期效果的错误</li> 
</ul> 
<h3>Runner</h3> 
<ul> 
 <li>改进了互斥支持，以防止多个 PT Run 实例的运行</li> 
</ul> 
<h3>视频会议静音</h3> 
<ul> 
 <li>将摄像头和麦克风静音的（默认）快捷键从 Win+N 改为 Win+Shift+Q，以避免与 Windows 11 的快捷键冲突</li> 
 <li>……</li> 
</ul>
