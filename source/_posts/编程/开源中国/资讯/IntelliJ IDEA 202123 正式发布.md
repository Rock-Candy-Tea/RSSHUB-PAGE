                                                        <p>IntelliJ IDEA 2021.2.3 现已发布，用户可以从 IDE 内部、通过 Toolbox 应用程序或使用 Ubuntu 的 snaps 进行升级，也可以从<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.jetbrains.com%2Fidea%2Fdownload%2F%3F_gl%3D1*1q899ua*_ga*MTI2NTc5ODM1Mi4xNjExOTM3NjE1*_ga_0WQ2ZF5VGT*MTYzNDQyNjExOC43OS4xLjE2MzQ0MjY1MTguMA..%26_ga%3D2.210866706.68559054.1634426118-1265798352.1611937615%23section%3Dmac" target="_blank">官方网站</a>下载。</p> 
<p><strong>主要更新内容</strong></p> 
<ul> 
 <li>修复了接口方法上的调用层次结构以正确返回结果</li> 
 <li>如果 IDE 由于操作系统关闭而关闭，则修复了每次启动时重复索引的问题</li> 
 <li>修复了在 WSL 2 中重新打开项目时导致 IDE 冻结的问题</li> 
 <li>修复了停止按钮的行为。当您停止 WildFly 运行配置时，不再出现终止批处理作业提示</li> 
 <li>修复了添加新 Arquillian 容器的对话框</li> 
 <li>修复了导致 IDE 冻结的问题</li> 
 <li>修复了 xslt 文件关联</li> 
 <li>修复了 jsp 文件中未使用导入的优化导入操作</li> 
 <li>修复了有关配置 Spring Boot 注释处理器的文档的链接</li> 
 <li>修复了 Maven 项目重新导入时，依赖项视图没有更新的问题</li> 
 <li>在依赖项工具窗口中输入时出现超级延迟</li> 
 <li>在 macOS IntelliJ IDEA 似乎用 rc 文件（.bashrc 或 .zshrc）中设置的环境变量覆盖了 shell 中设置的环境变量</li> 
</ul> 
<p>详情请查看<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fblog.jetbrains.com%2Fidea%2F2021%2F10%2Fintellij-idea-2021-2-3%2F" target="_blank">更新公告</a>。</p>
