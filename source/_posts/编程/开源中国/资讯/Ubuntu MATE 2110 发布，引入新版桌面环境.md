                                                        <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">Ubuntu MATE 21.10 的重大变化是引入了 MATE Desktop 1.26.0，该版本已经酝酿了 18 个月。由于 MATE Desktop 1.26 的优化，Ubuntu MATE 21.10 速度更快也更精简。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">以下是 MATE Desktop 1.26 中改进：</p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>控制中心的功能： 
  <ul style="margin-left:0; margin-right:0"> 
   <li>改进了窗口首选项对话框，提供了更全面的窗口行为和位置选项</li> 
   <li>显示首选项现在有一个离散显示缩放的选项</li> 
   <li>电源管理器有一个新的选项来启用键盘调光</li> 
   <li>通知现在支持超链接</li> 
  </ul> </li> 
 <li>Caja 可以格式化驱动器，并有一个新的书签侧边栏</li> 
 <li>Caja Actions 允许你添加任意程序，通过上下文菜单启动，现在是 Desktop 的一部分</li> 
 <li>计算器现在使用 GNU MPFR/MPC 来实现高精度、更快的计算和附加功能</li> 
 <li>Atril 在滚动浏览大型文件时速度更快，内存占用也减少了</li> 
 <li>归档管理器 Engrampa 现在支持 EPUB、ARC 和加密的 RAR 文件</li> 
 <li>Marco 窗口管理器： 
  <ul style="margin-left:0; margin-right:0"> 
   <li>正确地将最小化的窗口恢复到它们原来的位置</li> 
   <li>缩略图窗口预览支持 HiDPI</li> 
  </ul> </li> 
 <li>Netspeed 小程序显示更多关于网络接口的信息</li> 
</ul> 
<h3 style="margin-left:.6em; margin-right:0; text-align:start">Ubuntu MATE 的改进</h3> 
<ul style="margin-left:0; margin-right:0"> 
 <li>图像 Extrapolation 和 Interpolation 被默认禁用，以使图像浏览更快，图像质量更清晰</li> 
 <li>Alt-Tab 弹出窗口现在被扩展以适应长的窗口标题</li> 
 <li>如果你使用 Mutiny 布局，会话加载现在更快</li> 
</ul> 
<h3 style="margin-left:.6em; margin-right:0; text-align:start">主要应用</h3> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">伴随 MATE Desktop 1.26.0 和 Linux 5.13 一同更新的主要应用还包括：Firefox 93.0、Celluloid 0.20 和 LibreOffice 7.2.1.2</p>
