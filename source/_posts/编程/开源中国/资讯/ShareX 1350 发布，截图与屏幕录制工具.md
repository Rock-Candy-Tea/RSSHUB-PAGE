<div class="content">
                                                                    
                                                        <p>ShareX 13.5.0 发布了。ShareX 是一个开源截图工具，可捕获或记录屏幕的任何区域，并一键共享。 它还允许将图像、文本或其他类型的文件上传到超过 80 个支持的存储服务上。</p> 
<ul> 
 <li>在任务设置窗口，将 toast 通知和声音设置从高级选项卡移至新增加的通知选项卡；</li> 
 <li>为 toast 通知添加了拖放文件的支持；</li> 
 <li>增加了 "复制文件" 和 "复制文件路径" 的通知点击操作；</li> 
 <li>改进 hotkey 设置窗口的 UI；</li> 
 <li>增加了自动清理旧备份和日志文件的选项；</li> 
 <li>删除了自定义上传器设置窗口中的 "测试" 标签。它现在是一个单独的窗口，可以在不同的地方使用；</li> 
 <li>删除了自定义上传器设置窗口中的 "响应" 标签。请求和响应的内容现在都在同一个面板上；</li> 
 <li>增加了检查窗口工具；</li> 
 <li>在图像组合器工具中增加了 "自动填充背景" 选项；</li> 
 <li>为 “图片” 图像效果增加了翻转选项；</li> 
 <li>为 "边框" 图像效果添加了 dash 样式选项；</li> 
 <li>更新了 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fzws.im%2F" target="_blank">Zero Width Shortener</a> API 到 V2 版，现在支持自定义托管；</li> 
 <li>操作现在使用 $ 前缀，而不是 % 用于输入和输出。新的前缀不会在文件路径周围添加引号，以给用户更多的控制；</li> 
 <li>创建了操作范例网页：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgetsharex.com%2Factions" target="_blank">https://getsharex.com/actions</a> ；</li> 
 <li>在应用程序设置高级选项卡中增加了 "UseAlternativeClipboardGetImage" 选项，该选项允许从剪贴板中获取带有阿尔法通道的图像；</li> 
 <li>增加了在步骤注释工具中使用罗马数字的功能；</li> 
 <li>支持小写字母和罗马数字；</li> 
 <li>增加了 Vault.ooo 文件上传器；</li> 
 <li>当 hotkey 被禁用时，在 hotkey 设置窗口显示启用 hotkey 按钮；</li> 
 <li>ShareX 卸载程序现在会提示用户关闭 ShareX，如果它目前正在运行的话；</li> 
</ul> 
<p>更多详情可查看：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgetsharex.com%2Fchangelog%2F%23sharex-1350---2021-05-13" target="_blank">https://getsharex.com/changelog/#sharex-1350---2021-05-13</a></p>
