                                                                                            <p><span style="color:#000000">Kodi 是由 XBMC 基金会开发的开源媒体播放器，原名 XBMC（最后一个以 XBMC 命名的版本是 13.2 Gotham，14.0 Helix 是第一个以 Kodi 命名的版本），Kodi 可以运行在多种操作系统和硬件平台。 它可以让用户播放本地或网络存储设备中的大多数视频、音乐、播客及各种常见数位媒体文件。</span></p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><span style="color:#000000">该软件最初是为计划运行在 Xbox 上的，名称也由此而来。随后有了 Android、Linux、BSD、macOS、iOS 和 Windows 操作系统的原生版。</span></p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><span style="color:#000000">Kodi 19.4 正式发布，具体更新内容如下：</span></p> 
<p style="margin-left:0; margin-right:0; text-align:start"><span style="color:#000000"><strong>Estuary</strong></span></p> 
<ul> 
 <li><span style="color:#000000">修复：当菜单中有超过 10 个项目时，主屏幕上下文菜单修复。</span></li> 
 <li><span style="color:#000000">修复：使用远程应用程序（例如 Kore 或 JSON）暂停 Kodi 时，Seekbar 不会在 5 秒后消失。</span></li> 
</ul> 
<p style="margin-left:0; margin-right:0; text-align:start"><span style="color:#000000"><strong>Input</strong></span></p> 
<ul> 
 <li><span style="color:#000000">修复：英文键盘缺少分号。</span></li> 
 <li><span style="color:#000000">修复：中文键盘字符现在正确显示。</span></li> 
</ul> 
<p style="margin-left:0; margin-right:0; text-align:start"><span style="color:#000000"><strong>Music</strong></span></p> 
<ul> 
 <li><span style="color:#000000">修复：艺术家唱片导出到 NFO。</span></li> 
</ul> 
<p style="margin-left:0; margin-right:0; text-align:start"><span style="color:#000000"><strong>PVR</strong></span></p> 
<ul> 
 <li><span style="color:#000000">New：根据季节和剧集对录音进行排序。</span></li> 
 <li><span style="color:#000000">修复：浏览空指南时崩溃。</span></li> 
 <li><span style="color:#000000">修复：隐藏/删除频道组现在可以正常工作。</span></li> 
</ul> 
<p style="margin-left:0; margin-right:0; text-align:start"><span style="color:#000000"><strong>UI</strong></span></p> 
<ul> 
 <li><span style="color:#000000">修复：现在隐藏了受限/锁定媒体源的所有收藏夹。</span></li> 
 <li><span style="color:#000000">修复：Blu-ray ISO 标题和观看状态现在正确显示。</span></li> 
 <li><span style="color:#000000">修复：从字幕名称中识别的错误语言已更正。</span></li> 
</ul> 
<p style="margin-left:0; margin-right:0; text-align:start"><span style="color:#000000"><strong>Video</strong></span></p> 
<ul> 
 <li><span style="color:#000000">修复：EDL mute 现在可以工作。</span></li> 
 <li><span style="color:#000000">修复：ASS 字幕显示现在可以正常工作。</span></li> 
</ul> 
<p style="margin-left:0; margin-right:0; text-align:start"><span style="color:#000000"><strong>Android specifics</strong></span></p> 
<ul> 
 <li><span style="color:#000000">修复：调整了音频缓冲区。</span></li> 
 <li><span style="color:#000000">修复：纠正了各种 TrueHD 卡顿/丢失问题。</span></li> 
</ul> 
<p style="margin-left:0; margin-right:0; text-align:start"><span style="color:#000000"><strong>Linux specifics</strong></span></p> 
<ul> 
 <li><span style="color:#000000">修复：自动播放 DVD 的问题。</span></li> 
</ul> 
<p style="margin-left:0; margin-right:0; text-align:start"><span style="color:#000000"><strong>MacOS specifics</strong></span></p> 
<ul> 
 <li><span style="color:#000000">修复：全屏下的鼠标光标定位。</span></li> 
</ul> 
<p style="margin-left:0; margin-right:0; text-align:start"><span style="color:#000000"><strong>Windows specifics</strong></span></p> 
<ul> 
 <li><span style="color:#000000">修复：在 AMD GPU 上使用隔行扫描 H.264 SD 闪烁。</span></li> 
 <li><span style="color:#000000">修复：意外的网络配置崩溃（例如，没有获得 MAC 地址）。</span></li> 
</ul> 
<p style="margin-left:0; margin-right:0; text-align:start"><span style="color:#000000"><strong>Xbox specifics</strong></span></p> 
<ul> 
 <li><span style="color:#000000">New：通过 WASAPI 传递音频。</span></li> 
 <li><span style="color:#000000">修复：阻止安装 Python 附加组件的权限错误。</span></li> 
</ul>
