                                                        <p>dbMigration .NET 是一个简单、方便、直观的多数据库迁移和同步工具，你可以轻松地将数据迁移和同步到 PostgreSQL，反之亦然，无需复杂的程序。</p> 
<p><strong>从 13.5 版到 14.4 版的主要新功能和改进包括：</strong></p> 
<ul> 
 <li>增加了 Bulk Insert 模式的支持；</li> 
 <li>增加了 Asynchronous Processing 支持；</li> 
 <li>增加了多实例支持；</li> 
 <li>增加了 Table Size Information 支持；</li> 
 <li>增加了对 PostgreSQL 14 beta 3 或更高版本的支持；</li> 
 <li>增加了对 PostgreSQL 的 hstore、ghstore 和 multirange 类型的支持；</li> 
 <li>为所有数据库添加了对 biginteger 类型的支持；</li> 
 <li>增加了对数组数据类型到文本数据类型的支持；</li> 
 <li>更新至 Npgsql.dll 4.1.9；</li> 
 <li>与微软 Windows 11 兼容；</li> 
 <li>改进了用户体验和用户界面；</li> 
 <li>改进了自动映射类型；</li> 
 <li>性能改进；</li> 
</ul>
