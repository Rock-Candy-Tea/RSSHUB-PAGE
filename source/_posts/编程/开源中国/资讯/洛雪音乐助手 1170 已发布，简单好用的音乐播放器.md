                                                                                            <p>洛雪音乐助手是一个功能强大、音源齐全，操作简单的音乐播放器，支持导入其他音乐播放器的歌单（如 kugou、wyy音乐等）、支持同步功能。</p> 
<p>目前洛雪音乐更新到 1.17.0 版本，带来以下改动：</p> 
<h3><strong>新增</strong></h3> 
<ul> 
 <li>新增“便携”功能，在 Windows 平台下，若程序目录下存在 portable 目录，则自动使用此目录作为数据存储目录</li> 
 <li>新增 Scheme URL 支持，同时发布 lx-music-script 项目配合使用（一个油猴脚本，可以在浏览器中的官方平台网页直接调用LX Music），Scheme URL 的调用说明可查看 Readme.md 文档的 Scheme URL 支持部分</li> 
 <li>新增启动参数<code>-proxy-server</code>与<code>-proxy-bypass-list</code>，详细介绍看 Readme.md 文档的启动参数部分</li> 
 <li>新增桌面歌词是否延迟滚动设置，默认开启，若你不想要桌面歌词延迟滚动，可以找到设置-桌面歌词，设置关掉</li> 
</ul> 
<h3><strong>优化</strong></h3> 
<ul> 
 <li>为可视化音频的频谱整体添加频谱均值加成，使频谱显示更有节奏感</li> 
 <li>优化程序初始化逻辑，修复无网络的情况下的初始化问题</li> 
 <li>我的列表-列表名的右击菜单更新已收藏的在线列表时，将始终重新加载，不再使用缓存，解决在原平台更新歌单后，在LX点击更新可能看到的还是在原平台更新前的歌单的问题</li> 
</ul> 
<h3><strong>修复</strong></h3> 
<ul> 
 <li>修复代理不生效的问题</li> 
 <li>修复 <code>openDevTools</code> 选项无效的问题</li> 
 <li>修复播放状态的提示问题</li> 
 <li>修复 tx 源无搜索结果的问题</li> 
</ul> 
<h3><strong>其他</strong></h3> 
<ul> 
 <li>更新 Electron 到 v13.6.7</li> 
</ul>
