                                                                                            <p>SourceGit v4.9 已经发布，开源的 Git 客户端</p> 
<p>此版本更新内容包括：</p> 
<p>[CHANGELOG]</p> 
<ol> 
 <li>修改工程管理方式</li> 
 <li><a href="https://gitee.com/sourcegit/sourcegit/issues/I44GJO" target="_blank">https://gitee.com/sourcegit/sourcegit/issues/I44GJO</a> BUG修复</li> 
 <li>第三方Merge工具支持TortoiesGitMerge</li> 
 <li>支持自定义界面字体</li> 
 <li>崩溃日志上报可开关</li> 
 <li>修复DiffViewer连续选择失灵BUG</li> 
 <li>关闭所有标签页后，下次启动不再恢复最后打开的仓库</li> 
 <li>修复<code>rebase --continue</code>操作有时Crash的BUG</li> 
 <li>修复DiffViewer乱序选择复制时，文件顺序出错的BUG</li> 
 <li>修复多应用占用剪切板时，复制操作Crash的BUG</li> 
 <li>修复分支、标签名验证时，传入为空引起的Crash</li> 
 <li>界面主题调整</li> 
</ol> 
<p>详情查看：<a href="https://gitee.com/sourcegit/sourcegit/releases/v4.9">https://gitee.com/sourcegit/sourcegit/releases/v4.9</a></p>
