                                                                                            <p>smart-doc 2.2.7 已经发布，Java Restful API 文档生成工具。</p> 
<p>此版本更新内容包括：</p> 
<h2>Enhancement</h2> 
<ul> 
 <li>JSR303分组优化，标记Null的分组字段将不再显示在文档中 。</li> 
</ul> 
<h2>Bug Fixes</h2> 
<ul> 
 <li>修复dubbo文档css链接错误。</li> 
 <li>修复2.2.6分组归错误。</li> 
 <li>修复路径常量相似度时替换错误的bug。</li> 
</ul> 
<p>详情查看：<a blank="_target" href="https://gitee.com/smart-doc-team/smart-doc/releases/2.2.7">https://gitee.com/smart-doc-team/smart-doc/releases/2.2.7</a></p>
