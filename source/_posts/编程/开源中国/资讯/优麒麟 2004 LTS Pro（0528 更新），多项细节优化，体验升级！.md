<p><span style="color:#35ade5"><strong>（2021-05-28）</strong></span><span style="color:#35ade5"><strong>功能改进与BUG修复:</strong></span></p> 
<p> </p> 
<p style="text-align:left"><span style="color:#888888"><strong><span style="color:#333333">安装程序</span></strong></span></p> 
<ul> 
 <li> <p>更新部分翻译；</p> </li> 
 <li> <p>滚动条样式修改；</p> </li> 
 <li> <p>修复自定义分区页面分区图块显示问题</p> </li> 
 <li> <p>更换电源关闭图标；</p> </li> 
 <li> <p>修复系统安装的自定义分区的‘删除’/'添加’文本与边框几乎重合问题</p> </li> 
 <li> <p>修复 home 分区与其它分区挂载到不同的硬盘中，无法取消勾选问题</p> </li> 
 <li> <p>修复系统安装，用户密码输错之后无提示问题</p> </li> 
</ul> 
<p style="text-align:left"><span style="color:#888888"><strong><span style="color:#333333">控制面板</span></strong></span></p> 
<ul> 
 <li> <p>修复台式机亮度判断错误</p> </li> 
 <li> <p>修复黑名单文件夹路径显示不完整的问题</p> </li> 
 <li> <p>修复蓝牙设置界面“我的设备”连续点击多次连接按键，设备可以正常连接，但显示状态一直为加载的问题</p> </li> 
 <li> <p>修复连接JBLcharge4,点击控制面板声音后控制面板闪退问题</p> </li> 
 <li> <p>标准用户的新用户，安装软件及更改系统配置，需要授权</p> </li> 
 <li> <p>修复已经关闭屏幕坐标设置错误</p> </li> 
 <li> <p>修复更改用户头像时，侧边栏不显示外部设备的问题</p> </li> 
 <li> <p>修复双屏关闭副屏显示器后预览窗口重叠的问题</p> </li> 
 <li> <p>修复镜像模式下无法设置刷新率</p> </li> 
 <li> <p>设置图标跟随主题图标变化</p> </li> 
 <li> <p>修复开机启动中程序图标没有跟随图标主题实时变化的问题</p> </li> 
 <li> <p>统一输出后保留旧配置</p> </li> 
 <li> <p>修复快速移动导致二级导航菜单显示不全，hover和选中长度不一致的问题</p> </li> 
 <li> <p>修复删除某个文件夹后黑名单列表未更新的问题</p> </li> 
 <li> <p>增大用户账户中改名操作响应点击事件的区域</p> </li> 
 <li> <p>在当次添加程序中，以如用户手动输入程序名和程序描述为准，未输入相关字段时，自动更改</p> </li> 
 <li> <p>修复锁屏背景本地图片不能用于登陆背景的问题</p> </li> 
 <li> <p>修复插入4段式耳机，控制面板"输入-反馈"无波动问题</p> </li> 
 <li> <p>修复同步时间gif加载慢的问题</p> </li> 
 <li> <p>修复窗口置顶问题</p> </li> 
</ul> 
<p style="text-align:left"><span style="color:#888888"><strong><span style="color:#333333">其他</span></strong></span></p> 
<ul> 
 <li> <p>修复软件商店安装微信（windows版）失败问题</p> </li> 
 <li> <p>修复网络列表刷新异常问题</p> </li> 
 <li> <p>修复打开云账户卡顿问题</p> </li> 
 <li> <p>修复开始菜单弹出失败问题</p> </li> 
</ul>
