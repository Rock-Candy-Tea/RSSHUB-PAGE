                                                        <p><span style="background-color:#ffffff; color:#333333">Rufus 是一个可以帮助格式化和创建可引导 USB 闪存盘的工具，它可以快速把 ISO 格式的系统镜像文件快速制作成可引导的 USB 启动安装盘，支持 Windows 或 Linux 启动。3.17 版本更新内容如下： </span></p> 
<ul> 
 <li><span style="background-color:#ffffff; color:#333333">修复 MBR 没有被正确清除的问题</span></li> 
 <li><span style="background-color:#ffffff; color:#333333">修复退出时没有删除 commandline hogger 的问题</span></li> 
 <li><span style="background-color:#ffffff; color:#333333">改进对支持 ReFS 的平台的处理</span></li> 
 <li><span style="background-color:#ffffff; color:#333333">将 UEFI:NTFS 更新为最新版本，并删除安全 Secure Boot，因为该版本已签署 Secure Boot。</span></li> 
 <li><span style="background-color:#ffffff; color:#333333">更新 Grub4DOS 到最新版本</span></li> 
</ul> 
<p>更新说明：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Frufus.ie%2Fzh%2F">https://rufus.ie/zh/</a></p>
