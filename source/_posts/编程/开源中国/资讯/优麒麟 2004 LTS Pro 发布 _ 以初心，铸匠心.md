                                                        <p>北京时间 2021 年 4 月 22 日，优麒麟团队很高兴地宣布优麒麟（Ubuntu Kylin）开源操作系统 20.04 LTS Pro 版本正式发布。优麒麟 20.04 Pro 是基于 20.04 长期支持版本的更新，官方将提供 5 年的技术支持。</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-cdb0fa1e8569ea4f7ac1325d2650c161d6a.png" referrerpolicy="no-referrer"></p> 
<p><span style="color:#000000">优麒麟 20.04 Pro 版本默认搭载最新 <strong>Linux 5.10 LTS </strong>内核，新增自主研发的</span><strong><span style="color:#b2a2c7">麒麟音乐、麒麟录音、麒麟摄像头、麒麟桌面安装程序、麒麟蓝牙、麒麟传书、麒麟U盘启动器、麒麟计算器、麒麟扫描</span></strong><span style="color:#000000">等</span><span style="color:#d92142"><strong> 9</strong></span><span style="color:#000000"> 款应用软件。通过进一步优化动画效果、重构主题框架控件样式，带来更酷炫更统一的视觉体验。同时，默认集成 </span><span style="color:#d92142"><strong>26</strong></span><span style="color:#000000"> 个 CVE 漏洞修复补丁，修复大量桌面环境和应用软件方面等</span><span style="color:#000000"><span style="color:#000000">已知</span>问题，大幅提升系统稳定性和安全性，从而为用户提供更加高效稳定的使用体验。欢迎小伙伴们下载使用，并反馈您的使用体验和优化意见。</span></p> 
<p><strong>全新内核升级</strong></p> 
<p>集成最新 Linux 5.10 LTS 版本内核，带来了许多功能增强和更多的设备支持。同时，提供 Linux 5.4 LTS 版本内核，用户可以在 Grub 界面选择任一版本内核进行体验，满足更多场景下的用户使用需求，进一步提升系统的兼容性。</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-bf06d679551fef8910721d020da534fa027.png" referrerpolicy="no-referrer"></p> 
<p>Linux 5.10 LTS 重要更新：</p> 
<ul> 
 <li> <p>支持 SoundBlaster AE-7 ；</p> </li> 
</ul> 
<ul> 
 <li> <p>支持 Nintendo Switch Joy Cons 和 Pro 控制器；</p> </li> 
 <li> <p>支持 Sega Saturn 控制器；</p> </li> 
</ul> 
<ul> 
 <li> <p>提供对 AMD Zen3 处理器的硬件监视支持；</p> </li> 
 <li> <p>提供对联想 X1 平板电脑轨迹点和按钮的多点触摸支持；</p> </li> 
 <li> <p>提供对即将推出的 USB 4 规范的成熟支持；</p> </li> 
 <li> <p>Tiger Lake GPU 改进；</p> </li> 
</ul> 
<p><span style="color:#d92142"><strong><span style="color:#d92142">系统性能提升</span></strong></span></p> 
<p>通过优化桌面组件和应用的资源调度策略，充分利用硬件的调节接口，降低系统功耗。同时，通过深度优化桌面组件和应用的内存占用，提升系统运行流畅度，增强系统整体性能，为更多入门级设备提供澎湃动力，带来更高效的使用体验。</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-5eac94db4ac1dfad95a5f247e51cd755d7f.png" referrerpolicy="no-referrer"></p> 
<p><span style="color:#d92142"><strong>麒麟桌面安装程序</strong></span></p> 
<p>全新的桌面安装程序对 UI 界面进行改版，同时大幅提升安装速度，优化安装体验，不仅让萌新小白们轻松上手优麒麟，同时也让一直陪伴优麒麟的小伙伴们眼前一亮。</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-fe76a0e0a70ca5b4024ac52197bf9754d94.png" referrerpolicy="no-referrer"></p> 
<p><span style="color:#d92142"><strong>麒麟软件商店</strong></span></p> 
<p>经过UI升级和功能重构后，麒麟软件商店在用户可视性方面做到了化繁为简，提高了页面利用率。并且在新版麒麟软件商店中应用的下载更加容易、应用的管理也更加轻松便捷。</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-7a02fa2836e66e97adbe83790fee9874182.png" referrerpolicy="no-referrer"></p> 
<p><span style="color:#d92142"><strong><span style="color:#d92142">麒麟音乐</span></strong></span></p> 
<p><span style="color:#0c0c0c">基于 QT5 开发的一款简单音乐播放器，它涵盖了多种音频格式的播放工具，界面简单，操作快捷。支持自定义歌单，小窗口模式等功能。</span></p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-b698461d707490a38560b55712806cf6d35.png" referrerpolicy="no-referrer"></p> 
<p><span style="color:#d92142"><strong><span style="color:#d92142">麒麟录音</span></strong></span></p> 
<p><span style="color:#000000">基于 QT 开发的一款简单易用的录音软件，支持 MP3 和 WAV 等音频格式，支持文件列表的播放和删除。相较于同类录音软件，具有 mini 模式和实时显示声音频率大小等功能</span><span style="color:#000000">，多方位满足您的录音需求。</span></p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-35f36952ded4b8b713abfbbe5c1a45a9221.png" referrerpolicy="no-referrer"></p> 
<p><span style="color:#d92142"><strong><span style="color:#d92142">麒麟摄像头</span></strong></span></p> 
<p><span style="color:#0c0c0c">基于 V4l2 开发的相机应用，<span style="background-color:#ffffff; color:#0c0c0c">操作简单，界面友好</span>，支持绝大多数摄像头设备，具有拍照和录像以及在相册中显示照片的功能。</span></p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-a9e1d56e3f42cbda906a25e3a100852402f.png" referrerpolicy="no-referrer"></p> 
<p><span style="color:#d92142"><strong><span style="color:#d92142">更多自研应用</span></strong></span></p> 
<p><span style="color:#0c0c0c">优麒麟团队不断加大应用生态的投入力度，已自主研发 20 余款麒麟系列应用，此次新版本中除以上 3 款媒体应用外，还带来了 6 款用户需求较高的自研应用，将为大家的工作和生活提供便利和帮助：</span></p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-9a9595b7aacce59cec369d16369426a194e.png" referrerpolicy="no-referrer"></p> 
<p><span style="color:#d92142"><strong>丰富应用生态</strong></span></p> 
<p><span style="color:#0c0c0c">为积极解决用户生态问题，在优麒麟 20.04 Pro 版本中，软件商店上架了QQ音乐、腾讯视频、百度网盘、百度输入法、迅雷下载、360 浏览器等近 <span style="color:#d92142"><strong>30 </strong></span>款三方主流软件，并提供 Crossover 解决方案，使优麒麟能够兼容运行 Windows 应用，进一步丰富优麒麟的生态软件。</span></p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-958e6056f79995d248d42f09ab596d122aa.png" referrerpolicy="no-referrer"></p> 
<p><strong><span style="color:#2568fe">更新方式</span></strong></p> 
<p>目前，优麒麟用户和其他 Linux 爱好者均可以通过优麒麟官网或镜像站在线下载安装优麒麟 20.04 <span style="background-color:#ffffff">LTS </span>Pro 版本。</p> 
<p>1. 通过官网下载安装：</p> 
<p><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.ubuntukylin.com%2Fdownloads%2F" target="_blank">https://www.ubuntukylin.com/downloads/</a></p> 
<p><span style="color:#0052ff">麒麟桌面安装程序注意事项：</span></p> 
<p><span style="color:#888888">1）多系统安装请注意备份好重要数据；</span></p> 
<p><span style="color:#888888">2）Windows用户多系统安装时，需关闭 BitLocker 功能；</span></p> 
<p><span style="color:#888888">3）不支持加密和逻辑卷；</span></p> 
<p><span style="color:#888888">4）不支持设置为系统保留用户名（如root）；</span></p> 
<p><span style="color:#888888">5）如安装失败，可进入试用模式，选择开始菜单中ubiquity程序进行安装。</span></p> 
<p>2. 已装 20.04 和 20.04.1 正式版的用户直接升级：</p> 
<pre><code>$ sudo apt update

$ sudo apt full-upgrade</code></pre>
