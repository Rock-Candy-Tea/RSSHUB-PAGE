                                                        <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">Bitwarden 是一款开源的密码管理服务，使用者可在加密的保管库中储存敏感信息。Bitwarden 平台提供有多种客户端应用程式，包括 Web 版本、桌面应用，浏览器扩展、移动端应用和 CLI 版本等。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">日前 Bitwarden 更新了 Desktop 1.330、Browser Extension 1.58.0、Mobile 2.18.0、CLI 1.22.1、Directory Connector 2.10.1、Server 1.48.0 和 Web 2.28.0 版本，带来了以下多项更新内容：</p> 
<h3 style="margin-left:.6em; margin-right:0; text-align:start">支持性公告</h3> 
<blockquote> 
 <p style="margin-left:0; margin-right:0">从这个版本开始，从 App Store 下载的 macOS 桌面应用程序将需要 macOS Mojave（10.14）或更高版本。从<span> </span><a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fbitwarden.com%2Fdownload" target="_blank">bitwarden.com/download</a><span> </span>和 Github 获得的 ".dmg" 安装程序不受同样的限制。</p> 
</blockquote> 
<h3 style="margin-left:.6em; margin-right:0; text-align:start">更新内容：</h3> 
<ul style="margin-left:0; margin-right:0"> 
 <li>用于浏览器和桌面的用户名生成器：使用基于电子邮件的惯例（如加地址或使用随机词）为新凭证生成用户名</li> 
 <li>网络保险库的用户名生成器：使用基于电子邮件的惯例（如加地址或使用随机词）为新凭据生成用户名</li> 
 <li>CLI - 新的<code>serve</code>选项: 使用<span> </span><code>-hostname</code><span> </span>选项将你的 API 网络服务器安全地绑定到一个主机上</li> 
 <li>移动端增加了账户切换功能：在 Android 和 iOS 上使用 Bitwarden 时，最多可登录 5 个账户并在它们之间无缝切换</li> 
 <li>支持 Firefox 隐私模式：该版本包括对 Firefox 隐私窗口更强大的支持</li> 
 <li>网络保险库-报告页：更新了报告页面的位置，使其更容易找到并对报告结果采取行动</li> 
 <li>对 macOS 和 Safari 导入器的改进：修复了一些导致 macOS 和 Safari 导入器不能正确导入 URL 和注释的问题</li> 
 <li>可访问性的改进：这个版本还包括了一些可访问性的变化，这些变化将提高 Bitwarden 在辅助设备（如：屏幕阅读器）上的性能。</li> 
</ul> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">更多详情可查看：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fbitwarden%2Fdesktop%2Freleases" target="_blank"><code>https://github.com/bitwarden/desktop/releases</code></a></p>
