                                                        <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">QuickRedis 2.6.0 发布，增强功能和修复一些问题：</p> 
<ol> 
 <li>命令行点击 tab 支持命令提示；</li> 
 <li>修复ssh无法登录的问题；</li> 
 <li>升级electron至11.5.0；</li> 
 <li>优先使用 config get databases 获取db数量。</li> 
</ol> 
<p style="color:#24292f; margin-left:0; margin-right:0; text-align:start">QuickRedis 是一款<span> </span><strong>永久免费</strong><span> </span>的 Redis 可视化管理工具。它支持<strong>直连、哨兵、集群</strong>模式，支持亿万数量级的 key，还有令人兴奋的 UI。QuickRedis 支持 Windows 、 Mac OS X 和 Linux 下运行。</p> 
<p style="color:#24292f; margin-left:0; margin-right:0; text-align:start">QuickRedis 是一个效率工具，<strong>当别人在努力敲命令的时候，而你已经在喝茶。</strong></p>
