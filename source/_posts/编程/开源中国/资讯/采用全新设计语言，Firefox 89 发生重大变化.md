                                                        <p>6 月 1 日，Firefox 89 稳定版本正式发布。该版本的重点是界面的重新设计，新设计影响到主要的界面元素，如主工具栏、地址栏、菜单、浏览器显示的提示和标签。该版本也是自 2017 年 Firefox Quantum 发布以来做出的最大改变。</p> 
<p>所有的 Firefox 频道都在差不多的时间内被提升了一个版本。Firefox 88 稳定版被更新到 89 版，测试版和开发者版被更新到 90 版，Firefox Nightly 被更新到 91 版。</p> 
<h3>新功能和改进</h3> 
<p>Mozilla 表示，Firefox 89 采用了新的设计语言 Proton，产生的结果就是该界面更容易使用，以及更现代化。Firefox 89 有一个新的标签栏和地址栏设计，新的菜单和提示。一些项目已被删除或重新命名。在新版本中，用户可以在浏览时减少干扰。</p> 
<p>以下是主要的变化：</p> 
<h3>标签栏：</h3> 
<p><img alt height="394" src="https://static.oschina.net/uploads/space/2021/0602/072212_OZ4r_4937141.png" width="700" referrerpolicy="no-referrer"></p> 
<p>用户当前正在查看的网页标签会以「浮动」标签的形式进行展示。如果页面有音频正在播放，小喇叭的图标会直接替换标签栏中的网页图标，减少标签栏的空间占用（如上图）。</p> 
<p><img alt height="342" src="https://static.oschina.net/uploads/space/2021/0602/072807_VMAZ_4937141.png" width="700" referrerpolicy="no-referrer"></p> 
<p>Firefox 89 取消了广受欢迎的紧凑布局设计，新版本中标签栏只对新用户支持普通和触控密度。已经使用紧凑布局设计的现有用户在更新后可以继续保留这一设计。密度决定了标签页和标签栏的大小，新版本中占用的空间会更大。</p> 
<h3><strong>菜单：</strong></h3> 
<p><strong><img alt height="394" src="https://static.oschina.net/uploads/space/2021/0602/072235_pkXF_4937141.png" width="700" referrerpolicy="no-referrer"></strong></p> 
<p> </p> 
<p>Firefox 89 具有更加清晰和精简的菜单，Mozilla 重新组织了菜单内容并确定了优先次序，并删除了此前每个选项之前的图标。</p> 
<ul> 
 <li>登录 Firefox 浏览器被重新命名为同步和保存数据；</li> 
 <li>保护信息面板被删除，更改为直接点击地址左侧的盾牌图标；</li> 
 <li>增加了新建标签页；</li> 
 <li>缩放在菜单中被下移；</li> 
 <li>编辑功能已被删除；</li> 
 <li>我的足迹已被删除，书签、历史和下载现在直接在菜单中列出；</li> 
 <li>附加组件现在被称为扩展和主题；</li> 
 <li>定制工具栏被移除；</li> 
</ul> 
<h3>更新提示：</h3> 
<p><img alt height="394" src="https://static.oschina.net/uploads/space/2021/0602/072253_40uZ_4937141.png" width="700" referrerpolicy="no-referrer"></p> 
<p>当用户安装扩展或当一个网站要求访问麦克风或其他设备的权限时，Firefox 浏览器会向用户显示各种提示。新版本中更新了提示，如今信息栏、面板和模版的设计更简洁，语言更清晰。</p> 
<h3>其他变化：</h3> 
<ul> 
 <li>在隐私浏览模式中启用了 Total Cookie Protection，以进一步限制基于 Cookie 的跟踪；</li> 
 <li>在 Firefox 中查看 PDF 文件时，现在显示签名；</li> 
 <li>增加了 AllowedDomainsForApps 策略，定义了允许访问 Google Workspace 的域；</li> 
 <li>更新了 ExtensionSetting 策略，以禁用个别扩展的更新；</li> 
 <li>偏好设置策略支持以 gfx. 或 layer. 开头的首选项；</li> 
 <li>macOS 上的 Firefox 支持系统的 elastic overscroll 效果；</li> 
 <li>支持智能缩放；</li> 
</ul> 
<h3>开发者变化：</h3> 
<ul> 
 <li>重新设计的界面对主题 API 的改变：受影响的是 tab_background_separator 和 toolbar_field_separator，它们不再被支持，以及 tab_line 和 toolbar_vertical_separator，它们的行为在新版本中有所不同；</li> 
 <li>支持 Event Timing API；</li> 
 <li>支持 CSS forced-colors 媒体查询；</li> 
</ul>
