                                                                                            <p><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fstruts.apache.org%2Fannounce-2021.html%23a20211212" target="_blank">Apache Struts 2.5.28 已发布</a>，Apache Struts 2 是一个优雅、可扩展的框架，可用于创建企业级 Java Web 应用程序。Struts 2 旨在简化从构建到部署，再到维护应用程序整个开发周期。</p> 
<p>主要变更内容：</p> 
<ul> 
 <li>[WW-5149] - 修复 Struts 2.5.27 中的 labelposition 属性损坏</li> 
</ul> 
<p><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fcwiki.apache.org%2Fconfluence%2Fdisplay%2FWW%2FVersion%2BNotes%2B2.5.28" target="_blank">详情查看 release notes</a>，官方强烈建议所有开发者升级至此版本。</p> 
<p>Apache Struts 2.5.x 系列对以下规范版本有最低要求：Servlet API 2.4, JSP API 2.0 和 Java 7。</p> 
<p>下载地址：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fstruts.apache.org%2Fdownload.cgi%23struts-ga" target="_blank">https://struts.apache.org/download.cgi#struts-ga</a></p> 
<p>此外，Apache Struts 安全团队还宣布，所有使用最新 Struts 2.5.x 系列的用户应将 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Flogging.apache.org%2Flog4j%2F2.x%2F" target="_blank">Log4j</a> 库升级到最新版本，以解决远程代码执行漏洞。</p>
