<h2>概述说明</h2> 
<p>yicode 是一个以 yicode-cli 脚手架为核心驱动的快速、便捷、强大的项目开发生态和整体解决方案。</p> 
<h2>项目定位</h2> 
<p>yicode 致力于提高中小企业、外包公司、软件工作室/团队、个人开发者的开发效率，稳定性和可维护性。</p> 
<h2>使用声明</h2> 
<blockquote> 
 <p>yicode 致力于解决中小型项目的快速开发以及维护等问题，大型项目请谨慎调研再做决定！！！</p> 
</blockquote> 
<h2>功能特性</h2> 
<ul> 
 <li> <p>[x] 目录结构即路由，约定大于配置，路由自动导入</p> </li> 
 <li> <p>[x] 一键生成全局组件/指令/过滤器/混入/插件，且自动导入</p> </li> 
 <li> <p>[x] 默认提供 JavaScript 可选链，双问号语法</p> </li> 
 <li> <p>[x] 提供 yicode.config.js 配置文件进行自定义配置</p> </li> 
 <li> <p>[x] 方便简单支持产品、测试、本地等多环境变量开发方案</p> </li> 
 <li> <p>[x] 统一开发模板，自带样式重置和兼容处理，提供浏览器存储方案</p> </li> 
 <li> <p>[x] 一键切换 NPM 仓库镜像地址命令</p> </li> 
 <li> <p>[x] 统一的 js/css/html/vue/git 格式化和提交规范</p> </li> 
 <li> <p>[x] 提供页面级组件，让项目管理维护更合理，更方便</p> </li> 
 <li> <p>[x] 提供接口封装和多接口请求区分处理方案</p> </li> 
 <li> <p>[x] 默认支持国际化多语言方案</p> </li> 
 <li> <p>[ ] 主题功能</p> </li> 
 <li> <p>[x] 精研每一个配置参数，提供最大程序的项目性能打包优化和输出资源美化</p> </li> 
 <li> <p>[ ] 一键生成项目说明文档</p> </li> 
 <li> <p>[ ] yicode ui 命令，通过界面管理项目</p> </li> 
 <li> <p>[ ] 集成错误监控和上报</p> </li> 
 <li> <p>[ ] 元素一键重命名，避免手动多处改动</p> </li> 
</ul>
