                                                                                            <p>SourceGit v4.7 已经发布，开源的 Git 客户端</p> 
<p>此版本更新内容包括：</p> 
<p>[CHANGELOG]</p> 
<ol> 
 <li>优化提交路线图的生成算法</li> 
 <li>重写并简化DiffPlex算法，改为基于指定分隔符的分词比较方式</li> 
 <li>更加清晰的字体显示</li> 
 <li>修复二进制文件大小比对算法显示错误的BUG</li> 
 <li>程序崩溃时，自动生成Gitee的ISSUE，方便定位问题</li> 
</ol> 
<p>详情查看：<a href="https://gitee.com/sourcegit/sourcegit/releases/v4.7">https://gitee.com/sourcegit/sourcegit/releases/v4.7</a></p>
