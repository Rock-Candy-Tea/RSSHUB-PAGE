                                                        <p>Visual Studio 2022 Preview 2 现已推出，该版本专注于提供以个人和团队生产力、现代开发和不断创新为主题的新功能。</p> 
<p><strong>下载地址：</strong><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Faka.ms%2Fvs2022preview" target="_blank">https://aka.ms/vs2022preview</a></p> 
<p><a href="https://www.oschina.net/news/146486/visual-studio-2022-released" target="_blank">Visual Studio 2022 Preview 1</a> 中曾引入了新的 Cascadia Code 字体，旨在更易于阅读。在本次预览版本中，团队则更新了图标，使其更清晰、更易于区分，旨在 为每个人改进 Visual Studio 的可用性。 </p> 
<p><img src="https://oscimg.oschina.net/oscnet/up-7f7e0ba1288c35c672846ead454c02e86ff.png" referrerpolicy="no-referrer"></p> 
<p>同时，Preview 2 已完全本地化，包括了十多个语言包。可供选择的语言包括：英语、中文（简体）、中文（繁体）、捷克语、法语、德语、意大利语、日语、韩语、波兰语、葡萄牙语（巴西）、俄语、西班牙语和土耳其语。</p> 
<p>Preview 2 附带最新版本 (v143) 的 C++ 构建工具，这些工具与其他 14x 工具二进制兼容，可用于 Visual Studio 和<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Faka.ms%2Fvs%2F17%2Fpre%2Fvs_BuildTools.exe" target="_blank">独立的 Build Tools installer</a>。</p> 
<p>官方表示，其还添加了一些功能，以使得 Visual Studio 成为构建跨平台应用程序的最佳场所。借助 Preview 2，他们扩展了跨平台功能，添加了新的 CMake 集成和 WSL2 的无缝定位，而无需手动配置。“对于 .NET 开发人员来说，Visual Studio 2022 将是你使用 .NET MAUI 和 Blazor 构建下一波跨平台应用程序的地方。”</p> 
<p>生产力改进方面则包括 XAML 和 Web 应用程序的实时预览体验、实时显示应用程序的更改、以及新的调试命令 Force Run，其允许开发人员将应用程序运行到特定点，而忽略其他断点和异常。 </p> 
<p><img alt height="282" src="https://static.oschina.net/uploads/space/2021/0716/075134_cg7s_4252687.gif" width="500" referrerpolicy="no-referrer"></p> 
<p>此外，该版本还包括了对 Hot Reload 的更新、增加了对 C++ 的支持。开发人员现在可以在应用程序运行时使用 Hot Reload 来编辑 C++ 或 .NET 应用程序。</p>
