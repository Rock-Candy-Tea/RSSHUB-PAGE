                                                                                            <p>BS Redis Desktop Client v1.1.0   现已发布。</p> 
<p>BS RDC 是一个用rust+tauri编写的（前期用rust+sciter编写、现已重构），现代化可视化管理工具。</p> 
<ul> 
 <li>Pub/Sub在线调试功能上线。</li> 
 <li>修改了诸多bug</li> 
 <li>增加key列表的自动刷新</li> 
 <li>增加数据的格式化功能 string、json、blob</li> 
 <li> <p>set、zset、hash、采用scan方式</p> </li> 
 <li> <p>所有key通过scan获取。并用虚拟列表展示</p> </li> 
 <li> <p>界面采用vue3+element-plus编写</p> </li> 
 <li> <p>更多功能欢迎尝试使用</p> </li> 
</ul> 
<p>软件下载地址请查看 <a href="https://gitee.com/fuyoo/bs/releases">https://gitee.com/fuyoo/bs/releases</a> 或者软件官网 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fbs.echosocket.com" target="_blank">https://bs.echosocket.com</a></p>
