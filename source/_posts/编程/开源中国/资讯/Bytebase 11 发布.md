                                                                                            <p>Bytebase 是一个基于网络、零配置、无依赖的数据库 Schema 变更和版本控制管理工具，适用于开发人员和 DBA。</p> 
<p>Bytebase 1.1 发布，更新内容如下：</p> 
<h3><strong>更新了 CLI 文档</strong></h3> 
<p>bb CLI 是 Bytebase 的命令行工具。不仅为开发者带来了熟悉的操作感，还可以把针对 MySQL 和 PostgreSQL 的数据库管理操作集成到各种 CI/CD 系统 (例如 GitLab CI，GitHub Actions，Jenkins) 的工作流中。</p> 
<p><img alt height="325" src="https://oscimg.oschina.net/oscnet/up-1c71b618d0369448a1077488ee834b85bd1.png" width="700" referrerpolicy="no-referrer"></p> 
<h3><strong>新功能</strong></h3> 
<ul> 
 <li>从 SQL Editor，用户可以从关联的 VCS 代码仓库同步 SQL 脚本到 Bytebase。</li> 
</ul> 
<p><img alt height="447" src="https://oscimg.oschina.net/oscnet/up-484fcd0b82e455a797fd69633e4bf6300f1.png" width="700" referrerpolicy="no-referrer"></p> 
<ul> 
 <li>新增工作表的收藏功能以及列表页面</li> 
</ul> 
<p><img alt height="426" src="https://oscimg.oschina.net/oscnet/up-e210bf43ca12e02aa452b89c2d4a9497d05.png" width="700" referrerpolicy="no-referrer"></p> 
<ul> 
 <li>显示 PostgreSQL 数据库安装的插件</li> 
</ul> 
<p><img alt height="401" src="https://oscimg.oschina.net/oscnet/up-66469270433272d443578cb64284bca4710.png" width="700" referrerpolicy="no-referrer"></p> 
<ul> 
 <li>支持在创建 Postgres 数据库时设置所有者 (Owner)</li> 
</ul> 
<p><img alt height="727" src="https://oscimg.oschina.net/oscnet/up-5fe5770bd6f9c46320f7f14b42aa131bcd9.png" width="700" referrerpolicy="no-referrer"></p> 
<h3><strong>改进</strong></h3> 
<ul> 
 <li>在采用多租户模式的项目中，支持修改单个数据库中的数据</li> 
</ul> 
<h3><strong>Bug 修复</strong></h3> 
<ul> 
 <li>修复项目 VCS 配置时最多只显示 20 个代码仓库的问题</li> 
 <li>修复 SQL 编辑器格式化功能会意外清空编辑器的问题</li> 
 <li>修复了项目列表中出现重复的 bug</li> 
</ul>
