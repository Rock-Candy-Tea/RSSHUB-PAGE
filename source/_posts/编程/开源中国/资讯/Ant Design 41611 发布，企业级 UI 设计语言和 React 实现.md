 <li>修复 ConfigProvider 切换国际化时子组件 rerender 的问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F31630" target="_blank">#31630</a> <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FMap1en" target="_blank">@Map1en</a></li> 
 <li>简化 ghost 按钮的样式代码，去掉 <code>!important</code> 样式规则。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F31659" target="_blank">#31659</a></li> 
 <li>优化 RangePicker 的范围样式的 transition 效果。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F31645" target="_blank">#31645</a></li> 
 <li>修复 Dropdown <code>destroyPopupOnHide</code> TypeScript 定义丢失的问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F31700" target="_blank">#31700</a> <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Flinxianxi" target="_blank">@linxianxi</a></li> 
 <li>移除多余的 Omit 类型定义。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F31661" target="_blank">#31661</a> <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FDreamerryao" target="_blank">@Dreamerryao</a></li> 
</ul> 
<p>Ant Design 是一套企业级 UI 设计语言和 React 组件库。</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-3772d839c0da54232bf3aac6e582aa8d05d.png" referrerpolicy="no-referrer"></p> 
<p>特性</p> 
<ul> 
 <li>提炼自企业级中后台产品的交互语言和视觉风格。</li> 
 <li>开箱即用的高质量 React 组件。</li> 
 <li>使用 TypeScript 开发，提供完整的类型定义文件。</li> 
 <li>全链路开发和设计工具体系。</li> 
 <li>数十个国际化语言支持。</li> 
 <li>深入每个细节的主题定制能力。</li> 
</ul> 
<p>兼容环境</p> 
<ul> 
 <li>现代浏览器和 IE11（需要 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fant.design%2Fdocs%2Freact%2Fgetting-started-cn%23%25E5%2585%25BC%25E5%25AE%25B9%25E6%2580%25A7" target="_blank">polyfills</a>）。</li> 
 <li>支持服务端渲染。</li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.electronjs.org%2F" target="_blank">Electron</a></li> 
</ul>
