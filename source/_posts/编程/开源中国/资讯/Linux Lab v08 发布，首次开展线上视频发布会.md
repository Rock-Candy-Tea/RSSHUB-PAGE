<p style="color:#404040; margin-left:.67em; margin-right:0; text-align:start"><strong>版本更新</strong></p> 
<p style="color:#404040; margin-left:.67em; margin-right:0; text-align:start">Linux Lab 于 6 月初发布了<span> </span><a href="https://gitee.com/tinylab/linux-lab/tree/v0.7/">v0.7</a><span> </span>正式版本，同步发布了 Cloud Lab v0.5。</p> 
<p style="color:#404040; margin-left:.67em; margin-right:0; text-align:start">经过 4 个月的开发，本次发布<span> </span><a href="https://gitee.com/tinylab/linux-lab/tree/v0.8">v0.8</a>，新增 LLVM/Clang, Rust 和 openEuler 支持，同步发布 Cloud Lab v0.6，并升级 Docker 镜像。</p> 
<p style="color:#404040; margin-left:.67em; margin-right:0; text-align:start">期间，Linux Lab 实验盘（Linux Lab Disk）进一步新增了 Deepin 20.2, Fedora 34 和 Manjaro 系统支持。</p> 
<p style="color:#404040; margin-left:.67em; margin-right:0; text-align:start">本次相比 v0.7 版本，合计 155 笔变更，本次主要更新如下：</p> 
<ul style="margin-left:2.5em; margin-right:0"> 
 <li>v0.8 
  <ul style="margin-left:2.5em; margin-right:0"> 
   <li>镜像中新增 mipsel 4.3/4.7 编译器</li> 
   <li>90% 的开发板内核版本升级到 v5.13</li> 
   <li>新增 linux 和 buildroot targets，未来直接允许使用 pkg 的名字作为任务目标</li> 
  </ul> </li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Ftinylab.org%2Flinux-lab-v08-rc3%2F" target="_blank">v0.8 rc3</a> 
  <ul style="margin-left:2.5em; margin-right:0"> 
   <li>新增 Rust for Kernel 开发支持</li> 
   <li>新增 openEuler Kernel 开发支持</li> 
   <li>新增 LLVM/Clang 编译支持，make kernel LLVM=1</li> 
   <li>新增 rust 环境安装脚本</li> 
   <li>Pocket Linux Disk 和 Linux Lab Disk 相继支持 Fedora</li> 
  </ul> </li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Ftinylab.org%2Fmanjaro2go%2F" target="_blank">v0.8 rc2</a> 
  <ul style="margin-left:2.5em; margin-right:0"> 
   <li>Pocket Linux Disk 和 Linux Lab Disk 相继支持 Manjaro</li> 
   <li>早期文档中描述的更新步骤较重，替换为更为轻量级的更新步骤</li> 
   <li>修复 MacOS 上的 i386/pc 支持</li> 
   <li>进一步清理 rootfs 各种格式的依赖关系</li> 
   <li>进一步优化 make debug，确保 debug 基于最新的改动</li> 
   <li>清理不必要的 1234 端口映射，该部分可以让用户按需开启</li> 
  </ul> </li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Ftinylab.org%2Fpocket-linux-disk-ubuntu%2F" target="_blank">v0.8 rc1</a> 
  <ul style="margin-left:2.5em; margin-right:0"> 
   <li>发布了首支 Pocket Linux Disk</li> 
   <li>Pocket Linux Disk 和 Linux Lab Disk 相继支持 Deepin</li> 
  </ul> </li> 
</ul>

<p style="color:#404040; margin-left:.67em; margin-right:0; text-align:start"><strong>环境准备</strong></p> 
<p style="color:#404040; margin-left:.67em; margin-right:0; text-align:start">在非 Ubuntu 平台，请提前自行安装好 docker，可参考<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdocs.docker.com%2Fdocker-for-mac%2F" target="_blank">Docker for Mac</a>、<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdocs.docker.com%2Fdocker-for-windows%2F" target="_blank">Docker for Windows</a>。</p> 
<p style="color:#404040; margin-left:.67em; margin-right:0; text-align:start">如果是老版本的 Windows，可以用<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdocs.docker.com%2Ftoolbox%2Foverview%2F" target="_blank">Docker Toolbox</a>，也可以通过 Virtualbox 或 Vmware 自行安装 Ubuntu。</p> 
<p style="color:#404040; margin-left:.67em; margin-right:0; text-align:start">国内的同学请<strong>务必</strong>使用国内的 Docker 镜像服务，否则无法正常下载镜像，推荐参考<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fhelp.aliyun.com%2Fdocument_detail%2F60750.html" target="_blank">阿里云镜像配置文档</a>。</p> 
<p style="color:#404040; margin-left:.67em; margin-right:0; text-align:start">下面的链接是大家成功运行过 Linux Lab 的主机环境，欢迎参考，安装完以后也欢迎登记上来。</p> 
<ul style="margin-left:2.5em; margin-right:0">

<p style="color:#404040; margin-left:.67em; margin-right:0; text-align:start"><strong>极速体验</strong></p> 
<p style="color:#404040; margin-left:.67em; margin-right:0; text-align:start">该版本依赖最新的 Cloud Lab 和 docker 镜像：</p> 
<div style="margin-left:0; margin-right:0; text-align:start"> 
 <div style="margin-left:0; margin-right:0"> 
  <pre style="margin-left:.67em; margin-right:auto">$ git clone https://gitee.com/tinylab/cloud-lab.git
$ cd cloud-lab
$ tools/docker/run linux-lab
</pre> 
  <ol style="list-style-type:decimal; margin-left:2.5em; margin-right:0"> 
  </ol> 
 </div> 
</div> 
<p style="color:#404040; margin-left:.67em; margin-right:0; text-align:start">已经下载过的，请更新到最新版本并重启 Linux Lab：</p> 
<div style="margin-left:0; margin-right:0; text-align:start"> 
 <div style="margin-left:0; margin-right:0"> 
  <pre style="margin-left:.67em; margin-right:auto">$ cd cloud-lab && git pull
$ pushd labs/linux-lab && git pull && popd
$ tools/docker/rerun linux-lab
</pre> 
 </div> 
</div> 
<p style="color:#404040; margin-left:.67em; margin-right:0; text-align:start">进去以后，打开控制台，敲入如下命令即可启动一个虚拟开发板（自动下载预编译的版本）：</p> 
<pre><code>$ make boot</code></pre> 
<p style="color:#404040; margin-left:.67em; margin-right:0; text-align:start">一键编译和启动（自动下载源码、检出版本、打补丁、配置、编译）：</p> 
<div style="margin-left:0; margin-right:0; text-align:start"> 
 <div style="margin-left:0; margin-right:0"> 
  <pre style="margin-left:.67em; margin-right:auto">$ make boot BUILD=kernel</pre> 
 </div> 
</div> 
<p style="color:#404040; margin-left:.67em; margin-right:0; text-align:start">默认使用的是<span> </span><code>arm/vexpress-a9</code>，如果要使用本次新增的真实开发板，在<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fshop155917374.taobao.com%2F" target="_blank">购买</a>并连接串口和网口到开发主机后，可以简单切换：</p> 
<div style="margin-left:0; margin-right:0; text-align:start"> 
 <div style="margin-left:0; margin-right:0"> 
  <pre style="margin-left:.67em; margin-right:auto">$ make BOARD=arm/ebf-imx6ull
$ make login
</pre> 
 </div> 
</div> 
<p style="color:#404040; margin-left:.67em; margin-right:0; text-align:start">真实开发板与虚拟开发板用法有细微差异，其详细用法请参考：<a href="https://gitee.com/tinylab/linux-lab/tree/master/boards/arm/ebf-imx6ull">boards/arm/ebf-imx6ull/README.md</a>。</p> 
<p style="color:#404040; margin-left:.67em; margin-right:0; text-align:start"><strong>关键特性</strong></p> 
<p style="color:#404040; margin-left:.67em; margin-right:0; text-align:start">Linux Lab 具备如下特性：</p> 
<ol style="list-style-type:decimal; margin-left:2.5em; margin-right:0"> 
 <li>支持 3 大操作系统（Windows、MacOS、Linux），可以轻松在这三大操作系统下使用。</li> 
 <li>支持 7+ 大处理器架构（X86、ARM、MIPS、PPC、CSKY，RISC-V, LOONGSON），其中 LOONGSON 和 CSKY 为国产处理器。</li> 
 <li>支持 ~20 款开发板（i386/pc, x86_64/pc, arm/versatilepb, arm/vexpress-a9, ppc/g3beige, mips/malta, aarch64/virt, aarch64/raspi3, riscv32/virt, riscv64/virt, csky/virt, loongson/ls1b, loongson/ls2k, loongson/ls232, loongson/ls3a7a, arm/ebf-imx6ull）。</li> 
 <li>支持 5 种登陆方式（docker, ssh, vnc，webssh, webvnc），可以本地访问，也可以远程访问。</li> 
 <li>集成了 5 大组件（Qemu、U-boot、Buildroot、Linux、Toolchain），都有预编译版本。</li> 
 <li>内置了 5 大平台，32 位和 64 位共 10 个 Hello World 汇编语言例程，见<span> </span><code>examples/assembly</code>。</li> 
 <li>可以学习处理器指令集、Qemu、Shell、汇编、C、Linux 内核、嵌入式 Linux。</li> 
 <li>支持 Debugging 和 Testing。</li> 
 <li>host & guest 双侧免 root 使用。</li> 
 <li>可选购即插即用的 Linux Lab 实验盘。</li> 
</ol>
