                                                        <p>在 6 月我们第一次发布 KMRE 时，凭借其突破性的能让桌面 PC 和移动设备应用兼容的功能，获得大家的一致喜爱。不过在使用中 KMRE 还是存在一些不足，但是在经过我们研发小哥哥们半年的不懈奋斗之后，<strong><strong>全新的 KMRE 平行界面版本重磅上线！</strong></strong></p> 
<p><strong><strong>8大新增功能</strong></strong></p> 
<p><strong><strong>大幅性能提升</strong></strong></p> 
<p><strong><strong>全新升级玩法</strong></strong></p> 
<p><strong><strong>更多适配应用</strong></strong></p> 
<p><strong><strong>满足你全方位的需求！</strong></strong></p> 
<p>具体升级部分如下（下载方式附在文末~）：</p> 
<p><strong>一、版本新增功能</strong></p> 
<p><strong>1.KMRE上方菜单分类和功能优化</strong></p> 
<p>帮助、配置、关于，统一放置在设置页面中；打开移动数据、打开图库、打开微信下载目录统一到一个弹窗中。</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-e45acde9bbc400e0ec07e02d9ca4f5fcc9d.png" referrerpolicy="no-referrer"></p> 
<p><strong>2.平行界面</strong></p> 
<p>开启平行界面，应用内容会在屏幕上双屏显示，同时展示应用首页和内容页，方便大家操作和查阅，与华为平板平行视界相同体验。</p>

<p><strong>3.游戏按键</strong></p> 
<ul> 
 <li> <p>游戏按键支持设置为快捷键：可使用“Shift、Ctrl、Alt+字母”组成组合键设置为游戏按键。</p> </li> 
 <li> <p>游戏按键支持透明度调节。</p> </li> 
 <li> <p>游戏按键方向键支持8方向控制：同时按住两个临近方向键的时候，向斜方向生效，方向键由4方向增加到8方向。</p> </li> 
 <li> <p>游戏按键优化，可根据坐标自适应：修改游戏窗口大小、分辨率，游戏按键可自适应。</p> </li> 
</ul>

<p><strong>4.支持移动应用访问 u 盘数据</strong></p> 
<ul> 
 <li> <p>微信可访问 u 盘文件数据；</p> </li> 
 <li> <p>使用微信可直接发送 u 盘文件给好友；</p> </li> 
 <li> <p>可保存微信文件至 u 盘中。</p> </li> 
</ul>

<p><strong>5.锁屏功能升级</strong></p> 
<p>当安卓应用播放视频时，桌面不会自动锁屏。</p> 
<p><strong>6.截屏录屏</strong></p> 
<p><strong>截屏</strong>：一键截图，可选是否隐藏当前安卓应用窗口。</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-ffe468ed975a92deb475ce8d3b6ead9dfe0.png" referrerpolicy="no-referrer"></p> 
<p>截图后，可分享至微信好友。</p> 
<p><strong>录屏</strong>：一键录屏，可选择是否录制音频、是否录制全屏、分享录屏视频给微信好友。</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-956e5c50573b1cf23bc1f9373db99b09061.png" referrerpolicy="no-referrer"></p> 
<p><strong>7.安卓应用收纳至托盘区</strong></p> 
<p>在配置-托盘选项中勾选应用，被选中的应用可收纳至托盘区，退出应用后不会使应用关闭，在托盘区点击应用图标即可回到上次退出应用的界面。</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-1f39f9bbff5a1c8948e45e825284245dc36.png" referrerpolicy="no-referrer"></p>

<p><strong>8.KMRE 应用通知与 Linux 系统通知合并</strong></p> 
<p>当微信有新消息时，系统会在桌面右上角显示通知提示，打开侧边栏也会显示通知提醒。</p>

<p>目前版本支持此功能的有：微信、QQ 极速版、手机 QQ。</p> 
<p><strong>二、新增兼容应用</strong></p> 
<p><strong>1.娱乐</strong></p> 
<p><strong>音乐</strong>：已上架 QQ 音乐、酷狗音乐等大家常用的听歌应用。</p> 
<p><strong>视频</strong>：已上架抖音、快手等短视频应用，以及爱奇艺、腾讯视频、影音先锋等看剧软件。</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-165b8627850688bffb4e8b90c7506f66444.png" referrerpolicy="no-referrer"></p> 
<p><strong>游戏</strong>：已上架王者荣耀、狼人杀、全民突击等大家常玩的手游。</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-14a2eb92bc86094c3f0051d790ed63f3b45.png" referrerpolicy="no-referrer"></p> 
<p><strong>2.办公</strong></p> 
<p><strong>微信</strong>：支持截图、转发、引用、编辑图片、接龙、拍一拍等安卓微信超多常用功能，并可以快速定位到微信下载目录，拖拽文件即可轻松发送给好友；更多办公软件如企业微信等都可以正常使用，完全可以满足办公需求。</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-77ab94f5db8962f1f0af0ee7995c5e0ae74.png" referrerpolicy="no-referrer"></p> 
<p><strong>3.订餐</strong></p> 
<p>目前软件商店已经更新饿了么、大众点评等订餐软件，轻松可享与手机订餐无异的体验。</p> 
<p><strong>小提示：上文中提到的某部分应用若在软件商店没有上架，可通过下载 apk 安装包自主安装使用。</strong></p> 
<p> </p> 
<p><strong>三、版本性能提升</strong></p> 
<ul> 
 <li> <p>在 X86 架构下，支持 DRM 模式，提升显示性能；</p> </li> 
 <li> <p>常用 X86 架构硬件显卡支持视频硬解码；</p> </li> 
 <li> <p>在 X86 处理器+AMD/Intel/NVIDIA GPU 硬件环境下，支持视频播放硬解码技术，降低 CPU 占用率；</p> </li> 
</ul> 
<p> </p> 
<p><strong>四、安装方式</strong></p> 
<p><strong>1.命令行更新</strong></p> 
<p>打开终端输入以下命令进行更新：</p> 
<p><strong>sudo apt update</strong></p> 
<p><strong>sudo apt upgrade</strong></p> 
<p><strong>2.仅升级 KMRE</strong></p> 
<p>打开终端输入以下命令升级：</p> 
<p><strong>sudo apt update</strong></p> 
<p><strong>sudo apt install --only-upgrade kmre</strong></p> 
<p><strong>3.软件更新器一键更新</strong></p> 
<p>在开始菜单栏找到软件更新器，一键更新，等待完成即可。</p> 
<p>KMRE 平行界面版本已经上传到源，欢迎大家更新体验！若在使用过程中遇到问题 ，可在文章下方留言，也可通过论坛、交流群进行反馈~</p>
