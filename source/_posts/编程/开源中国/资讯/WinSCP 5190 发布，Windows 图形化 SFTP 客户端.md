                                                        <p>WinSCP 是一个 Windows 环境下使用的 SSH 的开源图形化 SFTP 客户端，同时支持 SCP 协议，它的主要功能是在本地与远程计算机间安全地复制文件，并且可以直接编辑文件。</p> 
<p>目前，WinSCP 5.19 已经发布。该版本具体更新内容如下：</p> 
<ul> 
 <li>从 URL 安装扩展时，不要使用 BOM 保存扩展。</li> 
 <li>在删除文件或寻找差异时失去连接，会话重新连接时不中止"Keep remote directory up to date"功能。</li> 
 <li>错误修复：在现有 WinSCP 实例中处理 URL 时，不使用来自以 URL 主机名命名的存储站点的会话设置。</li> 
 <li>错误修复：可以在进度窗口上移动工具栏。</li> 
 <li>错误修复：从文件安装扩展时，它总是以 UTF-8 与 BOM 的形式保存，而不考虑原始编码。</li> 
 <li>错误修复：从命令行启动“Keep remote directory up to date”功能时，未检测到断开连接。</li> 
 <li>错误修复：当连接丢失且未重新连接时，“Keep remote directory up to date”窗口未关闭。</li> 
</ul>
