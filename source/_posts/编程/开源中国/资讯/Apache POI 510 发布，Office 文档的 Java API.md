                                                                                            <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">Apache POI 是 Apache 软件基金会的开放源码库，POI 提供 API 给 Java 程序对 Microsoft Office 格式文件读和写的功能。它同时支持旧的（OLE2）和新的（OOXML - Office Open XML）格式。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">Apache POI 5.1.0 正式发布，更新内容如下：</p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>XDDF - 错误修复</li> 
 <li>将 Batik 的依赖性升级到 1.14</li> 
 <li>将 BouncyCastle 的依赖性升级到 1.69</li> 
 <li>将 Commons-Compress 依赖性升级到 1.21</li> 
 <li>将 XMLSec 依赖性升级到 2.2.3</li> 
 <li>将 PDFBox Graphics2d 的依赖关系升级到 0.33</li> 
 <li>添加 commons-io 2.11.0 作为依赖项</li> 
 <li>将 XMLBeans 升级到 5.0.2</li> 
 <li>POI 的内部日志现在使用 Apache Log4J 2</li> 
 <li>对 XSSFReader 进行了小规模的重构，使其更具扩展性</li> 
 <li>默认情况下，XML 文件中不接受 DTD。这可以通过设置<span> </span><code>POIXMLTypeLoader.DEFAULT_XML_OPTIONS.setDisallowDocTypeDeclaration(false)</code><span> </span>来放宽限制</li> 
</ul> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">更多详情可查看：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fpoi.apache.org%2Fchanges.html" target="_blank">https://poi.apache.org/changes.html</a></p>
