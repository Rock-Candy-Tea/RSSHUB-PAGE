                                                                                            <p style="margin-left:0px">Semi Design 是现代、全面、灵活的设计系统和 UI 库，由字节跳动抖音前端与 UED 团队设计、开发并维护，是一款包含设计语言、React 组件、主题等开箱即用的中后台解决方案，可用于快速搭建美观的 React 应用。</p> 
<p style="margin-left:0px">目前 Semi Design 发布了 v2.5.0 版本，此版本带来如下更新内容：</p> 
<ul> 
 <li>【Fix】 
  <ul> 
   <li>修复 resizable Table 动态删除列时列宽计算错误问题 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FDouyinFE%2Fsemi-design%2Fissues%2F650" target="_blank">#650</a></li> 
   <li>Select 组件当用键盘上下键操作时，增加自动滚动交互，调整聚焦 option 的相对位置 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FDouyinFE%2Fsemi-design%2Fissues%2F607" target="_blank">#607</a> </li> 
   <li>修复 next.js 项目使用 @douyinfe/semi-next 插件后，webpack.resolve.alias配置失效的问题 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FDouyinFE%2Fsemi-design%2Fissues%2F630" target="_blank">#630</a></li> 
   <li>修复 DatePicker open 受控时关闭面板后输入框聚焦态没有清空问题 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FDouyinFE%2Fsemi-design%2Fissues%2F528" target="_blank">#528</a></li> 
   <li>修复 Tooltip 在 React17 里如果父级有阻止点击事件冒泡弹出层收起会失效 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FDouyinFE%2Fsemi-design%2Fissues%2F593" target="_blank">#593</a> </li> 
  </ul> </li> 
</ul>
