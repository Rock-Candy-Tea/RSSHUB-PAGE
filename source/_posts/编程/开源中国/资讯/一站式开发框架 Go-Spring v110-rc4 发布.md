                                                                                            <p>距离上次发版已经三个月了，本次发布的版本应该是 v1.1.0 正式版之前的最后一个 RC 版本。</p> 
<blockquote> 
 <p style="color:#24292f; text-align:start">Go-Spring 的愿景是让 Go 程序员也能用上如 Java Spring 那般威力强大的编程框架。</p> 
 <ol> 
  <li>提供了完善的 IoC 容器，支持依赖注入、属性绑定；</li> 
  <li>提供了强大的启动器框架，支持自动装配、开箱即用；</li> 
  <li>提供了常见组件的抽象层，支持灵活地替换底层实现；<br> 3.1 抽象 web 框架，echo 和 gin 可以灵活替换。<br> 3.2 抽象 redis 框架，redigo 和 go-redis 可以灵活替换。</li> 
  <li>遵循最小依赖原则，部分组件零依赖，避免依赖地狱；<br> 4.1 提供 assert 包，满足日常测试断言的需求。<br> 4.2 提供 cast 包，满足日常数据转换的需求。<br> 4.3 提供 atomic 包，方便并发安全的存取数据。</li> 
  <li>首创基于框架的流量录制和回放技术，让测试更简单；</li> 
  <li>实现 Log4J 的日志架构，让日志管理更简单、更强大；</li> 
 </ol> 
</blockquote>
