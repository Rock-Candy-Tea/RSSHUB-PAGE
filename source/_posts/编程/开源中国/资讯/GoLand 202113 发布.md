                                                        <p>GoLand 2021.1.3  正式发布，该版本主要是一个 bug 修复版本。</p> 
<p><strong>主要更新内容</strong></p> 
<ul> 
 <li>修复了 gocheck 不显示在包宽测试菜单下的问题</li> 
 <li>如果调试器在更新前正在运行，则 Delve 版本不会更新</li> 
 <li>更新期间无法从 M1 上的旧 GoLand 版本更新 Delve</li> 
 <li>将 Delve 版本添加到 Troubleshooting.txt</li> 
 <li>如果大小写不匹配，则完成在修改表中不起作用</li> 
 <li>DataGrip SQL Server - 从生成 DDL 中排除扩展属性</li> 
 <li>修复了自上一版 IntelliJ 以来无法连接到远程 MySQL 的问题</li> 
 <li>Oracle DB - 升级到 2020.3.1 后自省失败（通过 XML 检索默认值和检查谓词时为 ORA-00907）</li> 
</ul>
