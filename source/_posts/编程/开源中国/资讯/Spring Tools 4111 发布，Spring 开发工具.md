                                                                                            <p>Spring Tools 4.11.1 现已发布。Spring Tools 4 是由 Spring 团队打造的下一代 Spring 开发工具，基本上是从零开始构建，融合了现代技术和开发者工具架构。它在单独的进程中运行，从构建之初就考虑到了性能问题，并且熟悉最新的 Spring 技术，为开发基于 Spring 的企业应用提供世界级支持。同时，全新版本的 Spring Tools 与 IDE 无关，可在各种编码环境中使用，支持 Eclipse、Visual Studio Code 与 Theia。</p> 
<p><strong>主要更新内容</strong></p> 
<ul> 
 <li>(VS Code, Spring Boot) 修复：[codespaces] spring boot 扩展在 Codespaces 设置中多次启动</li> 
 <li>(VS Code, Spring Boot) 修复：[codespaces] spring boot 扩展在连接到代码空间时找不到正确的 JDK</li> 
 <li>(VS Code, Spring Boot) 修复：JAVA_Home 是正确的，但仍然显示 “注意 Java 8 仍然可以在你自己的项目中使用”</li> 
 <li>(Eclipse) 改进：进度视图显示许多 AsyncLiveExpression 刷新消息并且不再响应</li> 
 <li>(Eclipse) 改进：最新的 CI 版本不再显示正在运行的应用程序的端口</li> 
 <li>(Concourse) 改进：semver 资源无法识别属性 “commit_message”</li> 
 <li>(Concourse) 改进：GitResource 架构，添加缺少的属性</li> 
 <li>(Concourse) 改进：未知属性 ‘fetch_tags’ 类型 ‘GitSource’(YamlSchemaProblem)</li> 
</ul>
