                                                                                            <p><strong>Eoapi</strong> 是一个<strong>开源的 api 管理工具</strong>，轻量化且可拓展。</p> 
<p><img alt height="363" src="https://oscimg.oschina.net/oscnet/up-9f7c2618a41b77e90b8d2e275946abf89bb.png" width="640" referrerpolicy="no-referrer"></p> 
<p>如今， Eoapi 已经更新发布到 v 1.6.0 版本，此次版本更新有几大亮点：</p> 
<ul> 
 <li><strong>Tab 支持</strong>，现在即使你切换到其他 tab，再回来内容也会保留，你关闭应用，Tab 还是会帮你保留，<br> 如果你没有保存就关闭 Tab， 页面也会有相应的提示；</li> 
</ul> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-9cfa1c66e9dee7f0de414a4da195643fc65.gif" referrerpolicy="no-referrer"></p> 
<ul> 
 <li><strong>测试支持 binary，预览图片响应</strong>，支持 Http 请求体格式为 Binary 文件的形式，支持直接预览图片的响应体；</li> 
</ul> 
<p><img alt height="368" src="https://oscimg.oschina.net/oscnet/up-5085153d1beb314c0acef9634ce66f16bf9.png" width="640" referrerpolicy="no-referrer"></p> 
<p><img alt height="355" src="https://oscimg.oschina.net/oscnet/up-28bd5b515adf37bf238a484384a2214cb05.png" width="640" referrerpolicy="no-referrer"></p> 
<p> </p> 
<ul> 
 <li><strong>上线导入 Postman 插件</strong>，支持将 Postman 的 Collection 导入到 Eoapi 平台；</li> 
</ul> 
<p><img alt height="365" src="https://oscimg.oschina.net/oscnet/up-7d65afe33ced79114cadfab0f74ed10a49e.png" width="640" referrerpolicy="no-referrer"></p> 
<ul> 
 <li><strong>优化环境管理的位置，</strong>现在入口在页面的右边，用户可以更加便捷的操作环境；</li> 
</ul> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-d38ce18d1ff9b127cc515a702bf43bbe1ba.gif" referrerpolicy="no-referrer"></p> 
<ul> 
 <li>全新的代码编辑器，针对桌面端还进行了优化，很适合 Electron 项目；</li> 
</ul> 
<p><img alt height="303" src="https://oscimg.oschina.net/oscnet/up-a7f4d1b5eb4745b3568d54ec13a956080f7.png" width="560" referrerpolicy="no-referrer"></p> 
<ul> 
 <li>布局支持动态拖动，可以随意拖动；</li> 
</ul> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-b5439e0fc02a19158d98b471c19b6030cef.gif" referrerpolicy="no-referrer"></p> 
<ul> 
 <li>效率工具</li> 
</ul> 
<p><img alt height="502" src="https://oscimg.oschina.net/oscnet/up-542d0e985a3c277ca31232b00725902c602.png" width="640" referrerpolicy="no-referrer"></p> 
<h2>功能预告：</h2> 
<p><span style="background-color:#ffffff; color:#000000">我们的迭代计划里面很重要的一环是支持 API 多协议，Websocket、GraphQL、gRPC 等等，可以让我们后续可以更好地对接各个平台或者应用。</span></p> 
<p><span style="background-color:#ffffff; color:#000000"><img alt height="315" src="https://oscimg.oschina.net/oscnet/up-d6f2eb6a63bf6117ba40bcd8a195409e565.png" width="640" referrerpolicy="no-referrer"></span></p> 
<p><span style="color:#000000">多协议的设计和开发已经开始了，预计在 9 月中旬能够发布 1.7.0 版本～</span></p> 
<p> </p> 
<p><span style="color:#000000">优先实现：</span></p> 
<ul style="list-style-type:square; margin-left:0; margin-right:0"> 
 <li> <p><span style="color:#000000">Websocket</span></p> </li> 
 <li> <p><span style="color:#000000">GraphQL</span></p> </li> 
 <li> <p><span style="color:#000000">gRPC</span></p> </li> 
</ul>
