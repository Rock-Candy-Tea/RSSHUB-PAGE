                                                        <p>RHEL 8.4 已于 5 月 19 日向客户全面推出。8.4 版本为混合云、边缘部署带来了重要功能，并为运营商、开发商和 RHEL 标准化组织带来了增强功能。</p> 
<p><strong>边缘计算</strong></p> 
<p>在 RHEL 8.3 中，红帽宣布了一个 RHEL 的部署选项，以帮助解决边缘计算环境中常见的挑战。在 8.4 版本中，在原基础上改进了 RHEL 的边缘计算功能的用户体验。</p> 
<p>RHEL 8.4 简化了更新，提供了额外的选项来安装到断开连接的系统（如许多边缘系统），并增加了使用 OSTree 存储库和 OCI 容器镜像的能力。</p> 
<p><strong>Red Hat Insights</strong></p> 
<p>Red Hat Insights 服务中的扩展功能（漏洞、合规性、资源优化和订阅）可帮助组织更有效地跨开放式混合云（包括公共云中的部署）管理其 RHEL 资产。</p> 
<p><strong>混合云</strong></p> 
<p>结合 8.4 版本所做的工作之一是简化云访问计划，使其在部署基于云的系统时更容易消费内容，并使用 Insights 生态系统管理它们。</p> 
<p><strong>8.4 中的更多内容</strong></p> 
<ul> 
 <li>RHEL 8.4 带有 Container Tools 3.0 AppStream，其中包含了 Podman、Buildah、Skopeo 和 runc 的新版本</li> 
 <li>Podman 现在在 RHEL 中支持自动更新容器镜像，这使得 Podman 可以轮询查看容器是否被更新，并拉取新的镜像，如果有新的镜像，则使用 systemd 来重启它们。</li> 
 <li>这个版本还增加了 Red Hat Universal Base Image（UBI）的微型镜像。这是最小的 UBI 产品，省略了镜像本身的软件包管理，但可以依靠底层 RHEL 主机上的软件包管理器来安装软件包。</li> 
 <li>除此之外，8.4 版本还引入了包括 Python 3.9、Redis 6、PostgreSQL 13、MariaDB 10.5 以及 GCC 10、LLVM 11、Rust 1.49 和 Go 1.15.7。</li> 
 <li>RHEL 8.4 现在支持英特尔 Tiger Lake GPU，包括英特尔 UHD Graphics 和英特尔 Xe 集成 GPU。</li> 
 <li>RHEL 8.4 引入了两个新的系统角色。首先，一个新的可靠事件日志协议（RELP）系统角色用于转发或接收带有 RELP 的 rsyslog 日志信息。其次还引入了 SSH 客户端系统角色，用于以一致和稳定的方式为 SSH 服务器和客户端配置 RHEL SSH。</li> 
</ul>
