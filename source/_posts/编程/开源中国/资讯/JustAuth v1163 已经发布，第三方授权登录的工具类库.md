                                                                                            <p>JustAuth v1.16.3 已经发布，第三方授权登录的工具类库</p>
<p>此版本更新内容包括：</p>
<ul> 
 <li>发布 v1.16.3</li> 
 <li>新增 
  <ul> 
   <li>集成“企业微信的第三方应用”平台登录</li> 
  </ul> </li> 
 <li>PR 
  <ul> 
   <li> <code>AuthRequst</code> 增加 <code>Builder</code> 构建方式，使用起来更简单。 ( <a href="https://gitee.com/yadong.zhang/JustAuth/pulls/27" rel="nofollow noreferrer noopener" target="_blank">gitee *27</a>)</li> 
   <li>使用 Github Action 添加发布快照的 workflow。 (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fjustauth%2FJustAuth%2Fpull%2F126" rel="nofollow noreferrer noopener" target="_blank">#126</a>)</li> 
   <li>新增了企业微信的第三方应用登录，<code>AuthWeChatEnterpriseThirdQrcodeRequest</code>。 (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fjustauth%2FJustAuth%2Fpull%2F127" rel="nofollow noreferrer noopener" target="_blank">#127</a>)</li> 
   <li>添加快照版本对应更详细的文档。 (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fjustauth%2FJustAuth%2Fpull%2F128" rel="nofollow noreferrer noopener" target="_blank">#128</a>)</li> 
  </ul> </li> 
 <li>修改 
  <ul> 
   <li>在 Gitee PR (<a href="https://gitee.com/yadong.zhang/JustAuth/pulls/27" rel="nofollow noreferrer noopener" target="_blank">*27</a>) 的基础上重构代码，增加 Builder 方式创建 AuthRequest</li> 
   <li>解决 Line 登录的错误。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fjustauth%2FJustAuth%2Fissues%2F122" rel="nofollow noreferrer noopener" target="_blank">#122</a> </li> 
  </ul> </li> 
</ul>
<p>详情查看：<a href="https://gitee.com/yadong.zhang/JustAuth/releases/v1.16.3" blank="_target">https://gitee.com/yadong.zhang/JustAuth/releases/v1.16.3</a></p>
