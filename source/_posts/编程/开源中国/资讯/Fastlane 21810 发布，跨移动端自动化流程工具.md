                                                        <p>Fastlane 2.181.0 发布了，Fastlane 是一个针对 iOS 和 Android 全方位开发自动化流程的工具。利用目前支持的工具可以做包含自动化和可持续化构建的每个环节，比如单元测试、截图、分发渠道、上传元数据和 ipa 包提交审核等等。</p> 
<p>新版更新内容：</p> 
<ul> 
 <li>[scan] 保留初始 xcresult</li> 
 <li>[spaceauth] 在 spaceauth 中添加 --copy_to_clipboard 选项</li> 
 <li>[Ruby 3.0] 改进 specs for Slack action</li> 
 <li>[spaceship] 修复 Spaceship::Members.find 的电子邮件大小写敏感问题</li> 
 <li>[supply] 上传用于 crash symbolication 的 native symbols</li> 
 <li>[fastlane][lanemanager] running using bundler 时，fastlane command suggestion 的改进 </li> 
 <li>[gym] gym/runner 中的修复。当移动 mac'app' 时，如果文件已经存在于该位置，则默示失败。</li> 
 <li>[snapshot] 在 snapshot reports generator 中添加了 iPhone 12 模型 </li> 
 <li>[fastlane_core][pilot][deliver] 当轮询构建处理时，修复了对 X.Y 和 X.Y.0 的轮询</li> 
 <li>[action][ensure_env_vars] 提高效率</li> 
 <li>[fastlane][action] 修复不在 git repo 中时的 git_branch</li> 
</ul>
