                                                                                            <p>Apache Wicket 是一个开源的面向 Java 组件的 Web 应用框架，为政府、商店、大学、城市、银行、电子邮件提供商等成千上万的 Web 应用和网站提供支持。</p> 
<p>Wicket 的开发中使用了语义版本，因此与 9.0.0 相比，9.10 版本没有出现 API 中断。</p> 
<p>9.10 的更新内容包括：</p> 
<h3>错误修复</h3> 
<ul> 
 <li>[WICKET-6970] - 在 AssociatedMarkupSourcingStrategy 中不必要的字符串构建</li> 
 <li>[WICKET-6971] - ModificationWatcher 中的 NullPointerException</li> 
 <li>[WICKET-6974] - JavaxUpgradeHttpRequest 返回一个空的 textPath</li> 
 <li>[WICKET-6975] - Behavior.renderHead 可能被多次调用</li> 
</ul> 
<h3>改进</h3> 
<ul> 
 <li>[WICKET-6963] - 对 PanelMarkupSourcingStrategy 使用单例</li> 
 <li>[WICKET-6967] - 允许发送非阻塞的 Web socket 消息</li> 
 <li>[WICKET-6969] - 允许以异步方式处理 web socket 推送消息</li> 
 <li>[WICKET-6972] - 在 Localizer.java 中添加资源键来记录警告</li> 
 <li>[WICKET-6977] - hashCode 计算产生过多的垃圾对象</li> 
</ul>
