                                                        <p>MyBatis 3.5.7 已发布，MyBatis 的前身为 iBatis，是一个数据持久层（ORM）框架，它提供的持久层能力包括 SQL Maps 和 Data Access Objects（DAO）。</p> 
<p>主要更新内容：</p> 
<ul> 
 <li>优化 JDK 8 下的性能 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmybatis%2Fmybatis-3%2Fissues%2F2223" target="_blank">#2223</a></li> 
 <li>修复并发访问缓存时可能出现错误的问题 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmybatis%2Fmybatis-3%2Fpull%2F2179" target="_blank">#2179</a></li> 
 <li>将已被弃用的 Mockito.verifyZeroInteractions 替换为 verifyNoInteractions <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmybatis%2Fmybatis-3%2Fpull%2F2077" target="_blank">#2077</a></li> 
</ul> 
<p>此外还有代码和文档方面的改进，<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmybatis%2Fmybatis-3%2Fissues%3Fq%3Dis%253Aclosed%2Bmilestone%253A3.5.7" target="_blank">详情点此查看</a>。</p>
