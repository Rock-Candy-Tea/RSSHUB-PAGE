<p><em>作者 / 开发者关系工程师 Nick Butcher</em></p> 
<p>上个月，<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdeveloper.android.google.cn%2Fevents%2Fdev-summit" target="_blank">Android 开发者峰会</a> 为 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdeveloper.android.google.cn%2Fjetpack%2Fcompose" target="_blank">Jetpack Compose</a> 带来了许多 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fandroid-developers.googleblog.com%2F2021%2F10%2Fads21-keynote-wrap.html" target="_blank">激动人心的更新</a>，包括 Google 的新设计语言 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fm3.material.io%2F" target="_blank">Material You</a>，Compose 现已对其提供支持。若您错过了某些内容，可以通过本文简要了解精彩内容，也可以 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdevrel.andfun.cn%2Fdevrel%2Fposts%2F2021%2F11%2FsaiW0l.mp4" target="_blank">点击这里</a> 查看精彩内容视频。</p> 
<h2><strong>新版本: Jetpack Compose 1.1 Beta 版和 compose-material 3</strong></h2> 
<p>我们发布了 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdeveloper.android.google.cn%2Fjetpack%2Fandroidx%2Freleases%2Fcompose%23versions" target="_blank">Jetpack Compose 1.1 Beta 版</a>。这意味着 1.1 中的新 API 现已稳定，可以为您提供新的功能并带来性能提升。1.1 新增了一些功能，比如经过优化的焦点处理、触摸目标值、"ImageVector" 缓存和对 Android 12 拉伸滚动的支持。Compose 1.1 还让许多之前的实验性 API 过渡到了稳定阶段，并支持新版 Kotlin。我们已经更新了 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fandroid%2Fcompose-samples" target="_blank">示例</a>、<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdeveloper.android.google.cn%2Fcourses%2Fpathways%2Fcompose" target="_blank">Codelab</a> 和 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fgoogle%2Faccompanist" target="_blank">Accompanist 开发库</a>，以便与 Compose 1.1 搭配使用。</p> 
<p>我们发布了 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdeveloper.android.google.cn%2Fjetpack%2Fandroidx%2Freleases%2Fcompose-material3%231.0.0-alpha01" target="_blank">compose-material 3</a>。这是使用 Jetpack Compose 构建 Material You UI 的全新工件，可提供更新的 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdeveloper.android.google.cn%2Freference%2Fkotlin%2Fandroidx%2Fcompose%2Fmaterial3%2Fpackage-summary" target="_blank">组件</a> 和 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdeveloper.android.google.cn%2Fjetpack%2Fcompose%2Fthemes%2Fmaterial%23m3-colorscheme" target="_blank">颜色系统</a>，其中包括对 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdeveloper.android.google.cn%2Fjetpack%2Fcompose%2Fthemes%2Fmaterial%23m3-dynamic-color" target="_blank">动态配色</a> 的支持，因此开发者可以根据用户的壁纸创建个性化的调色板。这是我们的首个 Alpha 版，欢迎您与我们 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fissuetracker.google.com%2Fissues%2Fnew%3Fcomponent%3D612128%26template%3D1253476" target="_blank">分享反馈</a>，我们会继续添加新功能并迭代 API。欢迎您前往全新的 <a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fm3.material.io" target="_blank">m3.material.io</a> 网站，了解关于 Material Design 3 的更多信息，找到有助您使用动态配色进行设计和构建的工具，比如 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fmaterial-foundation.github.io%2Fmaterial-theme-builder%2F" target="_blank">Material Theme Builder</a>。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdevrel.andfun.cn%2Fdevrel%2Fposts%2F2021%2F11%2FBBd9zk.mp4" target="_blank">点击这里</a> 观看视频了解更多详细内容。</p> 
<h2><strong>更多 Jetpack Compose 的指南和文档</strong></h2> 
<p>我们发布了大量关于 Jetpack Compose 的技术分享，深入探讨了布局、动画和状态，展示了如何在 Wear OS、主屏幕微件 (widget) 和大屏设备中使用 Compose，并举办了 3 场 <em>Code-Alongs 活动</em>；让您可以实时编写您的首个 Compose 应用、迁移现有应用或在 Wear OS 上使用 Compose。最后我们举行了专题讨论会，并在会上回答了与 Jetpack Compose 和 Material 相关的 <strong>大家</strong> 最关心的问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdevrel.andfun.cn%2Fdevrel%2Fposts%2F2021%2F11%2F0ttzvJ.mp4" target="_blank">点击这里</a> 观看视频了解更多详细内容。</p> 
<p>我们还扩展了 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdeveloper.android.google.cn%2Fjetpack%2Fcompose%2Fdocumentation" target="_blank">Compose 文档</a>，增加了针对 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdeveloper.android.google.cn%2Fjetpack%2Fcompose%2Fphases" target="_blank">Jetpack Compose 不同阶段</a> 的新指南、<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdeveloper.android.google.cn%2Fjetpack%2Fcompose%2Flayouts%2Fadaptive" target="_blank">构建自适应布局</a>，还扩展了 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdeveloper.android.google.cn%2Fjetpack%2Fcompose%2Fthemes" target="_blank">主题设置指南</a>，包括对 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdeveloper.android.google.cn%2Fjetpack%2Fcompose%2Fthemes%2Fmaterial%23material3" target="_blank">Material 3</a> 的指南。</p> 
<h2><strong>Android Studio Bumblebee 的工具更新</strong></h2> 
<p>在 Android 开发者峰会上，<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdeveloper.android.google.cn%2Fstudio%2Fpreview" target="_blank">Android Studio Bumblebee</a> 已宣布进入 Beta 版，可为 Jetpack Compose 提供更丰富的支持，其中包括:</p> 
<ul> 
 <li>默认启用 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdeveloper.android.google.cn%2Fjetpack%2Fcompose%2Ftooling%23preview-interactive" target="_blank">互动式预览</a>。</li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdeveloper.android.google.cn%2Fjetpack%2Fcompose%2Ftooling%23layout-inspector" target="_blank">布局检查器</a> 支持 Jetpack Compose 或混合 UI，还可提供 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdeveloper.android.google.cn%2Fstudio%2Fpreview%2Ffeatures%23support_for_inspecting_compose_semantics" target="_blank">检查语义</a> 功能。</li> 
</ul> 
<p>Android Studio Chipmunk Canary 版还为 Compose 和 (基于视图的) Material 3 应用引入了新模板。</p>

<h2><strong>Handoff</strong></h2> 
<p>最后，我们 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fmaterial.io%2Fblog%2Fdesigntocode" target="_blank">抢先介绍</a> 了一些用于设计接力 (Handoff) 的新工具，使您能够导出在 Figma 中设计的组件，以生成通用的 Jetpack Compose 代码。您可以迭代设计并引入新的更改，安全地编辑生成的代码。我们正在寻找可以直接合作的小团队，如果您感兴趣就 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fservices.google.com%2Ffb%2Fforms%2Fdesigntocode%2F" target="_blank">快来注册</a> 吧。</p> 
<p>Jetpack Compose 已推出稳定版本，供大家在生产环境中使用。很高兴看到越来越多的应用开始在生产环境中使用 Jetpack Compose，我们会继续构建我们的功能路线图，让您可以使用 Compose 构建跨设备的卓越应用。欢迎您持续关注 "Android 开发者" 微信公众号了解更多最新内容。</p> 
<p>欢迎您 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fservices.google.cn%2Ffb%2Fforms%2Fandroiddevswechat2%2F" target="_blank">点击这里</a> 向我们提交反馈，或分享您喜欢的内容、发现的问题。您的反馈对我们非常重要，感谢您的支持！</p>
