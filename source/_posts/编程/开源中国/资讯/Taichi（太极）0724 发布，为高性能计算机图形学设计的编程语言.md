<ul> 
 <li>[ci] 修复 docker 缓存问题 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ftaichi-dev%2Ftaichi%2Fpull%2F2468" target="_blank">#2468</a> ) </li> 
 <li>[ci] 分离 Win 构建和测试步骤 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ftaichi-dev%2Ftaichi%2Fpull%2F2465" target="_blank">#2465</a> )</li> 
 <li>[misc] 修复参数的类型。（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ftaichi-dev%2Ftaichi%2Fpull%2F2464" target="_blank">#2464</a>）</li> 
 <li>[wasm] [aot] 在 webassembly 后端支持 AOT。（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ftaichi-dev%2Ftaichi%2Fpull%2F2461" target="_blank">#2461</a>）</li> 
</ul>
