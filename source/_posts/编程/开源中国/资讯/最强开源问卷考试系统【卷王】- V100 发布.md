                                                                                            <p><span style="background-color:#ffffff; color:#333333">卷王问卷考试系统是目前 gitee 里面功能最强大的问卷和考试系统，经过 1 个月多的迭代，本次发布 v1.0.0 版本。</span></p> 
<p>经过一年多的发展，卷王系统在<strong>问卷编辑、逻辑设置、数据校验、数据导出</strong>等很多方面的功能体验超过了问卷星这些主流的商业调查问卷系统。</p> 
<p>并且卷王系统提供了多种途径的部署方式，整个安装包只有 40M 大小：</p> 
<ul> 
 <li>根据 <a href="https://gitee.com/surveyking/surveyking">SurveyKing / SurveyKing</a> 源码自行编译，同时支持 maven 和 gradle 进行编译</li> 
 <li>支持 docker 一键部署，`docker run -d -p 1991:1991 surveyking/surveyking `</li> 
 <li>也支持 windows 版本下面的一键启动</li> 
</ul> 
<p>本次更新内容如下：</p> 
<ul> 
 <li>添加富文本题</li> 
 <li>横向填空题同时支持级联、下拉、普通文本</li> 
 <li>单行文本题支持唯一答案设置</li> 
 <li>支持题干说明</li> 
 <li>考试添加随机问题</li> 
 <li>考试结束允许查看成绩单和排名</li> 
 <li>白名单添加导出</li> 
 <li>数据表格支持打印，图片和文件在线预览，表格支持选中数据求和、平均值</li> 
 <li>数据导出支持时间段</li> 
 <li>题库拆分为问卷题库和考试题库</li> 
 <li>全新的 UI 界面，支持设置皮肤</li> 
 <li>编辑器支持快捷键</li> 
 <li>其他bug修复和功能优化</li> 
</ul>
