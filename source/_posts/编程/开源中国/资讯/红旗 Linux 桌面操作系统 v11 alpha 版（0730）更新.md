<div class="content">
                                                                                            <p>红旗 Linux 桌面操作系统 v11 alpha 版<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fmp.weixin.qq.com%2Fs%2F2DQbKoVoiGNFcFbVLn88OQ" target="_blank">发布</a>了（0730）更新。 较上一版本，此版本主要更新如下：</p> 
<ul> 
 <li>完善系统开始菜单体验；</li> 
 <li>修复AMD64架构LIVE环境网络源地址报错的问题；</li> 
 <li>修复AMD64架构部分环境安装失败的问题；</li> 
 <li>增加两款益智游戏2048,纸牌游戏；</li> 
 <li>增加建军94周年主题壁纸； </li> 
 <li>修复部分已知问题；</li> 
</ul> 
<p><strong>更新及下载方法</strong>：</p> 
<p>1、已安装红旗Linux桌面操作系统v11 alpha前序版本的用户，可参考下列方法进行系统更新。</p> 
<p>2、未安装红旗Linux桌面操作系统v11 alpha版的用户，需按下方提供的下载链接，访问红旗Linux官方社区下载版块，下载安装体验。</p> 
<p><strong>方式1、软件商店自定义更新</strong></p> 
<p>点击桌面左下角“R”图标，点击“系统”-“软件商店”-“检查更新”，根据更新提示可自定义选取需要更新的软件包或更新全部升级的软件包。</p> 
<p><img height="381" src="https://oscimg.oschina.net/oscnet/up-6960e8bf48ed0d3536da80129216a8b5bce.png" width="500" referrerpolicy="no-referrer"></p> 
<p><strong>方式2、终端命令更新</strong></p> 
<p>更新网络源</p> 
<ul> 
 <li> <pre><code>sudo apt update -y</code></pre> </li> 
</ul> 
<p>查看可更新软件</p> 
<p> </p> 
<ul> 
 <li> <pre><code>apt list --upgradable</code></pre> </li> 
</ul> 
<p>可根据需求全部或选择性的手动安装相应更新包。</p> 
<p><strong>下载链接：</strong></p> 
<p>访问红旗 Linux 官方社区下载版块 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.linuxsir.cn%2F%3Fdownload.htm" target="_blank">https://www.linuxsir.cn/?download.htm</a> 跳转，获取镜像下载链接。</p>
