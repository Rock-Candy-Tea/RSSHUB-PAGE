<div class="content">
                                                                                            <p><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Femilk%2Fegui%2Freleases%2Ftag%2F0.18.0" target="_blank">egui 0.18 已发布</a>。egui 是易于使用、可移植的即时模式 (immediate mode) GUI 库，采用 Rust 编写，可运行在 Web 和原生应用中。</p> 
<p><strong>更新亮点</strong></p> 
<ul> 
 <li>在 egui 中嵌入 3D 渲染</li> 
 <li>在明亮模式下提供更好的文本对比度</li> 
 <li>新增 egui_extras::Table</li> 
 <li>添加<code>Plot::allow_scroll</code>,<span> </span><code>Plot::allow_zoom</code><span> 不再影响滚动</span> (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Femilk%2Fegui%2Fpull%2F1382" target="_blank">#1382</a>).</li> 
 <li>添加<code>Ui::push_id</code><span> 以解决 id 冲突</span> </li> 
 <li>添加<code>ComboBox::icon</code><span> </span></li> 
 <li>添加<code>Ui::scroll_with_delta</code>.</li> 
 <li>添加<code>Frame::outer_margin</code>.</li> 
 <li>添加<code>Painter::hline</code><span> 和</span><span> </span><code>Painter::vline</code>.</li> 
 <li>添加<code>Link</code><span> </span>和<span> </span><code>ui.link</code></li> 
</ul>
