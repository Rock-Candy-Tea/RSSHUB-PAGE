                                                                                            <p style="margin-left:0px">Electron 是一个使用 JavaScript、HTML 和 CSS 构建跨平台的桌面应用程序。它基于 Node.js 和 Chromium，被 Atom 编辑器和许多其他应用程序使用。Electron 兼容 Mac、Windows 和 Linux，可以构建出三个平台的应用程序。</p> 
<p style="margin-left:0px"><span style="color:#333333">Electron 20.2.0 现已发布，具体更新内容如下：</span></p> 
<p><strong>Features</strong></p> 
<ul> 
 <li>添加了<code>webFrameMain.origin</code>。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Felectron%2Felectron%2Fpull%2F35535" target="_blank">#35535</a></li> 
</ul> 
<p style="margin-left:0px"><span style="color:#24292f"><strong>Fixes</strong></span></p> 
<ul> 
 <li>修复<code>uv_os_gethostname</code>了在 Windows 7 上失败的问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Felectron%2Felectron%2Fpull%2F35705" target="_blank">#35705 </a></li> 
 <li>修复了<code>app.isInApplicationsFolder()</code>在某些情况下会错误返回 false 的 edge case。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Felectron%2Felectron%2Fpull%2F35731" target="_blank">#35731</a></li> 
</ul> 
<p style="margin-left:0px"><strong>Other Changes</strong></p> 
<ul> 
 <li>修复了使用 WCO 的窗口的子窗口没有 navigator defined 的问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Felectron%2Felectron%2Fpull%2F35530" target="_blank">#35530 </a></li> 
 <li>将 Chromium 更新为 104.0.5112.124。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Felectron%2Felectron%2Fpull%2F35674" target="_blank">#35674</a></li> 
</ul> 
<p><strong>Documentation</strong></p> 
<ul> 
 <li>文档更改：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Felectron%2Felectron%2Fpull%2F35666" target="_blank">#35666</a></li> 
</ul>
