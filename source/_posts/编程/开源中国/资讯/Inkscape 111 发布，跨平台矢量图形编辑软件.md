                                                        <p><span style="background-color:#ffffff; color:#333333">Inkscape 1.1.1 现已发布。Inkscape 是一个矢量图形编辑软件，与 Illustrator、Freehand、CorelDraw、Xara X 等软件很相似，它使用 W3C 标准的 Scalable Vector Graphics (SVG) 文件格式，支持包括形状、路径、文本、标记、克隆、alpha 混合、变换、渐变、图案、组合等 SVG 特性。它也支持创作共用的元数据、节点编辑、图层、复杂的路径运算、位图描摹、文本绕路径、流动文本、直接编辑 XML 等。它可以导入 JPEG、PNG、TIFF 等格式，并输出为 PNG 和多种矢量格式。 </span></p> 
<p><strong>主要更新内容</strong></p> 
<ul> 
 <li>新功能 
  <ul> 
   <li>在帮助 → 捐赠中添加了一个指向捐赠页面的链接，以便用户更轻松地为项目捐款</li> 
  </ul> </li> 
 <li>Bug 修复 
  <ul> 
   <li>压力敏感度在激活 Windows Ink 时再次起作用</li> 
   <li>描边到路径再次适用于文本</li> 
   <li>撤消对路径的描边不再丢失对象的 id</li> 
   <li>现在可以在不撤消先前操作的情况下撤消具有实时路径效果的对象的对象到路径</li> 
   <li>保存不再泄漏内存</li> 
   <li>启动时间缩短，在 Windows 平台上从 3.3 秒缩短到 3 秒</li> 
   <li>在终端中隐藏有关未使用的可选字体目录的不必要警告</li> 
   <li>画布不会在将窗口从 HiDPI 移动到常规显示器时变模糊</li> 
   <li>在 wmii 和其他一些窗口管理器上再次单击会选择对象</li> 
   <li>Caps lock 不再阻止 Delete、Page Down 和其他一些快捷键在 Windows 上工作</li> 
   <li>快速裁剪：现在在应用过滤器 → 填充和透明度时会考虑透明度</li> 
   <li>EPS：安装了 snap 包格式的 Inkscape 的用户可以再次打开 EPS 文件</li> 
   <li>优化的 SVG：另存为 → 优化的 SVG 可以在 Linux AppImage 中使用</li> 
   <li>帮助菜单：现在使用一组不同的网络链接从“帮助”菜单链接到文档</li> 
   <li>从“欢迎”对话框生成的所有模板现在都使用 1 的文档比例，从而可以更轻松地手动编辑使用它们创建的文档</li> 
  </ul> </li> 
</ul>
