                                                                                            <p><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fblog.getbootstrap.com%2F2021%2F03%2F23%2Fbootstrap-5-beta-3%2F" target="_blank">Bootstrap 5 Beta 3</a> 现已发布，这是最后一个 Beta 版本。官方表示，该版本包括了一个新的组件、文档更新；同时还修复了一些自上个版本以来的重要 bug。 本次更新的一些重点内容如下：</p> 
<p><strong>新的 offcanvas 组件</strong></p> 
<p><strong><img alt height="278" src="https://oscimg.oschina.net/oscnet/up-e5ba22141438802af38d0da152fbbcdc540.png" width="500" referrerpolicy="no-referrer"></strong></p> 
<p>Beta 3 中推出了一个全新的组件 offcanvas，它具有可配置的背景、body scroll 和 placement。Offcanvas 组件可以放置在视口的左侧、右侧和底部。可使用<code>data</code>属性或通过 JavaScript API 来配置这些选项。</p> 
<p><strong>New and refreshed examples</strong></p> 
<p><strong><img alt height="313" src="https://oscimg.oschina.net/oscnet/up-6a43cbf8f840fc86761fcac0b071e48e051.png" width="500" referrerpolicy="no-referrer"></strong></p> 
<p>Beta 3 中还进行了 examples 更新。添加了四个全新的 snippet-heavy examples，以及一些其他的更新。还新增一些用于 headers、heroes、features 和 sidebars 的 snippets。官方表示，其计划继续增加新的 snippets。</p> 
<p><img alt height="313" src="https://oscimg.oschina.net/oscnet/up-38d536ff26a370d5b9dae1ee1dafb07974c.png" width="500" referrerpolicy="no-referrer"></p> 
<p>同时，开发团队还通过更新、简化的设计和更多的资源链接更新了 starter template。并将这个新的外观和感觉融入到 npm 启动项目项目中，最终添加一个 Parcel 启动项目。以及更新了 pricing、checkout 和 sign-in examples。添加了一个新的 jumbotron example，向用户展示如何创建自己的 jumbotron。</p> 
<p><strong>改进的 Sass 文档</strong></p> 
<p><strong><img alt height="313" src="https://oscimg.oschina.net/oscnet/up-a20176bff69b84a10491b248ed77c94ced8.png" width="500" referrerpolicy="no-referrer"></strong></p> 
<p>现在在适当的地方列出了 Sass 变量、映射、循环和动画关键帧。这些都是直接从源文件中链接过来的，所以每当官方发布新的代码时，它们会自动更新。</p>
