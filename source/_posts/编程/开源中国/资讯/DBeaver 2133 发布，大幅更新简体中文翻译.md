                                                                                            <p>DBeaver 是一个免费开源的通用数据库工具，适用于开发人员和数据库管理员。</p> 
<p>DBeaver 21.3.3 发布，更新内容如下：</p> 
<ul> 
 <li>Data editor: 
  <ul> 
   <li>日期/时间编辑器现在支持类似日历的用户界面</li> 
   <li>列名引用得到了修复</li> 
   <li>修复了对数组列的过滤</li> 
   <li>Gis viewer: 修复了 SRID 转换行为</li> 
   <li>保存/取消按钮现在在数据保存过程中被禁用</li> 
   <li>列过滤器现在对数字值使用精确匹配</li> 
   <li>状态信息现在显示执行时间戳</li> 
  </ul> </li> 
 <li>SQL Editor: 
  <ul> 
   <li>修复了多个结果标签的刷新问题</li> 
   <li>导入文件现在支持多文件选择</li> 
   <li>客户端命令（@set、@include）解析器已修复</li> 
   <li>SELECT ... INTO 查询的自动补全修复</li> 
   <li>自动补全现在默认使用 "hippie" 算法</li> 
   <li>修复了 Delphi 格式</li> 
   <li>增加了 "Copy selected query（复制选定的查询）" 命令</li> 
   <li>改进了变量和参数的删除</li> 
   <li>修复了 SQL 格式（消除了多余的空行）</li> 
   <li>改进了标签上下文菜单（增加了其他项目）</li> 
   <li>修复了代码折叠恢复</li> 
   <li>增加了多光标编辑模式</li> 
  </ul> </li> 
 <li>Data transfer: 
  <ul> 
   <li>改进了 XLSX 的导入</li> 
   <li>从 CSV 导入现在使用近似的列长度计算</li> 
   <li>修正了追加到文件（Append to file）的模式</li> 
  </ul> </li> 
 <li>Navigator: 修复了树形状态的加载</li> 
 <li>SSH tunnel: 
  <ul> 
   <li>改进了高级隧道配置用户界面</li> 
   <li>隧道测试对话框现在支持测试中断</li> 
   <li>对 Jump servers 的私钥配置进行了修复</li> 
  </ul> </li> 
 <li>任务执行状态现在可以显示更详细的信息</li> 
 <li>Eclipse 插件：修复了历史导航快捷键支持</li> 
 <li>macOS：修复了分离窗口关闭时的应用程序终止问题</li> 
 <li>本地化： 
  <ul> 
   <li>简体中文本地化已大幅更新</li> 
  </ul> </li> 
 <li>……</li> 
</ul>
