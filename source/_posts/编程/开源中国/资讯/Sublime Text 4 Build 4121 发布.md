                                                                                            <p>Sublime Text 是一个商业代码编辑器。它原生支持许多编程语言和标记语言，用户可以通过插件来扩展它的功能，这些插件通常是由社区建立的，并以自由软件许可证的形式维护。为了方便插件，Sublime Text 有一个 Python API。</p> 
<p>Sublime Text 4 Build 4121 发布，部分更新内容如下：</p> 
<ul> 
 <li>新的更新对话框会显示版本、许可证状态和更新日志的链接；</li> 
 <li>各种语法高亮的改进；</li> 
 <li>使用 goto-symbol 在查找结果中跳转到特定文件；</li> 
 <li>改进了配色方案/主题；</li> 
 <li>当 "hot_exit" 被禁用时，Sublime 不再记住布局，你可以使用 "remember_layout" 来改变这一状态；</li> 
 <li>改进了 OpenGL 下的 mini-diff 和 white space 的渲染性能；</li> 
 <li>改进了从 stdin 读取大文件时的性能；</li> 
 <li>修正了 "save_on_focus_lost" 在重载对话框中不能如期工作的问题；</li> 
 <li>修正了当 "hot_exit" 被禁用时，打开的文件不被添加到最近文件列表中的问题；</li> 
 <li>修复了片段补全不尊重单词边界的问题；</li> 
 <li>修复了在某些情况下启动 Sublime 时，打开额外的空白窗口的问题；</li> 
 <li>修正了当 "hot_exit" 被禁用时，提示不能正确退出的问题；</li> 
 <li>修正了当项目没有添加文件夹且 "close_windows_when_empty" 被启用时，窗口意外关闭的问题；</li> 
 <li>修正了下划线宽度不能正确缩放的问题；</li> 
 <li>修正了在禁用 gutter 时，高亮代码行不起作用的问题；</li> 
 <li>修正了在一些情况下，文件夹被添加到现有窗口而不是在新窗口中打开；</li> 
 <li>当 "sublime_merge_path" 设置为 null 时，Sublime Merge 菜单项被隐藏；</li> 
 <li>修正了在 Sublime Merge 中查看文件历史时有时会使用错误的路径；</li> 
 <li>特定于文件的缩进设置现在可以在重启时保持不变；</li> 
 <li>……</li> 
</ul>
