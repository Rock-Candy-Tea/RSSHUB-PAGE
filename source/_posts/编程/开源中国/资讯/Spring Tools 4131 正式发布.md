                                                                                            <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">Spring Tools 4.13.1 现已发布。Spring Tools 4 是由 Spring 团队打造的 Spring 开发工具，从零开始构建，融合了现代技术和开发者工具架构。它在单独的进程中运行，从构建之初就考虑到了性能问题，并且支持最新的 Spring 技术，为开发基于 Spring 的企业应用提供世界级支持。同时，全新版本的 Spring Tools 与 IDE 无关，可在各种编码环境中使用，支持 Eclipse、Visual Studio Code 与 Theia。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><strong>主要更新内容</strong></p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>(Spring Boot）改进：考虑是否可以为 spring.config.import 属性键提供内容辅助（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fsts4%2Fissues%2F536" target="_blank">#536</a>）</li> 
 <li>(Spring Boot)修复：YAML 编辑器在 Windows 中生成错误的 EOL 字符 (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fsts4%2Fissues%2F709" target="_blank">#709</a>)</li> 
 <li>(VSCode)修复：总是弹出提示框：indexing spring boot properties (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fsts4%2Fissues%2F697" target="_blank">#697</a>)</li> 
 <li>(VSCode)修复：用于 VS Code 的 Spring Boot Tools 错误地将用于 Java 语言支持的内部 JDK 作为 JRE 支持(<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fsts4%2Fissues%2F715" target="_blank">#715</a>)</li> 
 <li>(Eclipse)修复：创建或修改新项目 - maven 更新问题 (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fsts4%2Fissues%2F703" target="_blank">#703</a>)</li> 
</ul>
