<div class="content">
                                                                                            <p><span style="color:#333333">Armbian Linux 是一款适用于 ARM 开发板的、基于 Debian 的 Linux 发行版，具有强大的图形化配置工具和安装程序，简单易上手。</span>Armbian 每三个月提供一次稳定版本，由最近的 LTS 内核驱动，喜欢滚动发布的用户可以检查 EDGE 版本，这些版本使用来自 sid、hirsute 或 impish userland 的最新每日内核构建。</p> 
<p><img alt height="394" src="https://static.oschina.net/uploads/space/2022/0901/073759_tF4a_5430600.png" width="700" referrerpolicy="no-referrer"></p> 
<p><span style="color:#333333">目前 </span>Armbian Linux 22.08 已发布，新版本越来越接近 Linux 桌面版，带来了大量桌面环境以供选择：</p> 
<p>本次更新的内容：</p> 
<ul> 
 <li>添加了服务器或 XFCE、Cinnamon 和 Budgie 桌面构建</li> 
 <li>首次运行时，提供快速有效的自动语言选择</li> 
 <li>定期稳定和每日 beta 和 EDGE 更新</li> 
 <li>CLI 由 ZSH 或 BASH 提供支持</li> 
 <li>在 EDGE 5.13.y 内核上添加了自动内核升级</li> 
 <li>为 Odroid HC4 添加了基于主线的 SPI 启动支持</li> 
 <li>添加了 Qemu 虚拟 Armbian 构建</li> 
 <li>为 Tinkerboard 2、Rockpi N10 添加了 CSC 图像</li> 
 <li>ZFS 升级到 v2.1</li> 
 <li>改进的 Github Actions CI 和 CDN 网络</li> 
 <li>尽可能启用 3D 支持，并且运行良好</li> 
 <li>添加 Khadas VIM1-3 和边缘板，Avnet Microzed</li> 
 <li>为瑞芯微启用 VPU 支持</li> 
 <li>添加了对 OrangepiZero2、Nvidia Jetson 的旧内核支持</li> 
 <li>将 Ubuntu Hirsute 和 Debian Bullseye 软件包声明为稳定的</li> 
 <li>添加了 Ubuntu Impish 和 Debian Sid 作为 beta 构建目标</li> 
 <li>添加了 KDE 等 DE 作为 beta 构建目标</li> 
</ul>
