                                                                                            <p>phpMyAdmin 近期发布了两个版本更新：5.2.0 和 5.1.4，后者是 5.1 系列的最后一次更新。团队表示 5.1.4 主要是为了帮助下游的打包团队，并建议用户迁移到 5.2.0。</p> 
<p><strong>5.2.0 更新亮点</strong></p> 
<ul> 
 <li>删除对 Microsoft Internet Explorer 的支持</li> 
 <li>要求 PHP 7.2 或更高版本</li> 
 <li>要求安装 openssl PHP 扩展</li> 
 <li>改进了对系统 CA bundle 和 cacert.pem 的处理，如果需要可以回退到 Mozilla CA</li> 
 <li>将 "master/slave" 术语替换为 "primary/replica"</li> 
 <li>将 "NOT LIKE %...%" 运算符添加到 Table Search</li> 
 <li>添加对 Mroonga 引擎的支持</li> 
 <li>支持锁定帐号</li> 
 <li>对 SQL 解析器库的修复和改进</li> 
</ul> 
<p>此外，本次更新的两个版本都修复了导出文件时的<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fphpmyadmin%2Fphpmyadmin%2Fissues%2F17445" target="_blank">网络错误问题</a>。</p>
