                                                        <p><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Flampepfl%2Fdotty%2Freleases%2Ftag%2F3.0.0" target="_blank">Scala 3 </a>稳定版来了！<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.scala-lang.org%2Fblog%2F2021%2F05%2F14%2Fscala3-is-here.html" target="_blank">发布公告写道</a>：“经过 8 年的努力开发，在此期间共产生了 28000 多次 commit、7400 多个 PR，以及关闭了 4100 多个 issue。从2012年12月6日 Scala 3 的首个 commit 算起，共计超过 100 人为项目做出了贡献。现在，Scala 3 通过结合类型理论的最新研究以及 Scala 2 的行业经验，Scala 的第三次迭代版本变得更易于使用、学习和扩展。”</p> 
<p><img alt height="174" src="https://oscimg.oschina.net/oscnet/up-7ad9ff71a05bf989b56f117fbbf3a6b2746.png" width="500" referrerpolicy="no-referrer"></p> 
<p>按照规划，3.x 期间的更新频率是每隔 6 周发布一次常规维护更新。</p> 
<p>下边简单介绍一下 Scala 3 值得关注的新特性。</p> 
<p>新类型</p> 
<ul> 
 <li>交集类型</li> 
 <li>并集类型</li> 
 <li>类型 Lambda</li> 
 <li>匹配类型</li> 
 <li>依赖函数类型</li> 
 <li>多态函数类型</li> 
</ul> 
<p>上下文抽象</p> 
<ul> 
 <li>Using 子句</li> 
 <li>扩展方法</li> 
 <li>实现 Type Class</li> 
 <li>Type Class 推导</li> 
 <li>Multiversal Equality</li> 
 <li>上下文函数</li> 
 <li>隐式转换</li> 
</ul> 
<p>其他新特性</p> 
<ul> 
 <li>Trait 参数</li> 
 <li>透明 Trait</li> 
 <li>统一应用函数</li> 
 <li>Export 子句</li> 
 <li>不透明类型别名</li> 
 <li>开放类</li> 
 <li>参数解元组</li> 
 <li>Kind 多态</li> 
 <li>Matchable Trait</li> 
 <li>@threadUnsafe 注解</li> 
 <li>@targetName 注解</li> 
 <li>新控制语法</li> 
 <li>可选括号</li> 
 <li>显式 null</li> 
 <li>安全初始化</li> 
 <li>TypeTest</li> 
 <li>……</li> 
</ul>
