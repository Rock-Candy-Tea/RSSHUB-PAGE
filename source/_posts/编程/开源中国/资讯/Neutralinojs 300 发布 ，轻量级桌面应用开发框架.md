                                                        <p><span style="background-color:#ffffff; color:#24292f">Neutralinojs 3.0.0 发布了！</span><span style="background-color:#ffffff; color:rgba(0, 0, 0, 0.87)">Neutralinojs 是一个轻量级、便携的桌面应用程序开发框架，可以使用 JavaScript、HTML 和 CSS 开发轻量级跨平台桌面应用程序。</span></p> 
<h3 style="margin-left:.6em; margin-right:0; text-align:start"><span style="color:rgba(0, 0, 0, 0.87)">主要更新内容：</span></h3> 
<ul style="margin-left:0; margin-right:0"> 
 <li>新的 JavaScript API 规范实现。</li> 
 <li>使用 WebSocket 消息进行本地调用。</li> 
 <li>增加了用于识别”本地调用失败“的错误代码。</li> 
 <li>静态服务器改进（404，支持任何<span> </span><span style="color:#24292f">mime-type，等</span>等）</li> 
 <li>添加新方法：加载远程 URL 时，通过<span> </span><code>exitProcessOnClose</code><span> </span>配置来关闭窗口。</li> 
 <li>通过<span> </span><code>os.getPath(name)</code><span> </span>获取已知平台专用的文件夹。</li> 
 <li>支持从模式中覆盖根级配置的属性（例如：一个 URL 用于浏览器模式，另一个用于窗口模式）。</li> 
 <li>对话框有新的库。</li> 
 <li>新的配置属性(<code>logging</code>) 用来启用/禁用日志记录功能。</li> 
 <li>新事件：<code>ready</code>，<code>clientConnect</code>和<code>clientDisconnect</code></li> 
 <li>支持客户端库 v 2.0.0</li> 
 <li>一些 bug 的修复和增强。</li> 
</ul>
