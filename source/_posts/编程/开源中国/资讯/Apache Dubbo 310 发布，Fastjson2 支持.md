                                                                                            <p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><span style="background-color:#ffffff; color:#000000">Apache Dubbo 3.1.0 已发布，</span><span style="background-color:#ffffff; color:#24292f">这是 Dubbo 3 的全新版本，包括 xDS 支持、kubernetes 原生支持、Fastjson2 支持……</span></p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><span style="background-color:#ffffff; color:#000000">Apache Dubbo 是一款高性能、轻量级的开源 Java RPC 框架，它提供了三大核心能力：面向接口的远程方法调用、智能容错和负载均衡，以及服务自动注册和发现。</span></p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><span style="background-color:#ffffff; color:#000000">具体更新内容包括：</span></p> 
<ul> 
 <li>Sidecar mesh。有关说明，可参见 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdubbo.apache.org%2Fzh%2Foverview%2Ftasks%2Fmesh%2Fdubbo-mesh%2F" target="_blank">https://dubbo.apache.org/zh/overview/tasks/mesh/dubbo-mesh/</a>。</li> 
 <li>xDS proxyless mesh。有关说明，可参见 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdubbo.apache.org%2Fzh%2Foverview%2Ftasks%2Fmesh%2Fproxyless%2F" target="_blank">https://dubbo.apache.org/zh/overview/tasks/mesh/proxyless/</a>。</li> 
 <li>错误代码故障排除文档指南。有关说明，可参见 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdubbo.apache.org%2Fzh%2Fdocs3-v2%2Fjava-sdk%2Ffaq%2Fintro%2F" target="_blank">https://dubbo.apache.org/zh/docs3-v2/java-sdk/faq/intro/</a>。</li> 
 <li>Fastjson2 支持。通过指定序列化到 fastjson2 手动启用。有关说明，可参阅 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Falibaba%2Ffastjson2" target="_blank">https://github.com/alibaba/fastjson2</a>。</li> 
 <li>端口统一支持。支持将各种协议绑定到一个信号端口。</li> 
 <li>带有 project reactor 的 Dubbo Reactive stream。</li> 
</ul>
