                                                                                            <p>qBittorrent 一个跨平台的 BitTorrent 客户端，具有美观的 Qt 用户界面以及用于远程控制的 Web UI 和集成搜索引擎。qBittorrent 旨在满足大多数用户的需求，同时使用尽可能少的 CPU 和内存。</p> 
<h3>Windows:</h3> 
<ul> 
 <li>Windows 10 及更高版本的用户也可以使用 Qt6 版本的安装程序。Qt6 提供了更好的 HiDPI 兼容性。</li> 
 <li>已知问题：进度条上的文字可能会显示错误的颜色或完全不显示的问题。</li> 
</ul> 
<h3>Linux:</h3> 
<ul> 
 <li>现在提供了一个 AppImage，它使用最新版本的 Qt6、libtorrent、boost 和 openssl。</li> 
</ul> 
<h3>其他更新：</h3> 
<ul> 
 <li>Windows：增加 windows-clang 支持</li> 
 <li>Windows：更新 Windows 下的 Python 安装程序 URL</li> 
 <li>Linux：延长 qbittorrent-nox 关机的等待时间</li> 
 <li>Linux：安装矢量程序图标</li> 
 <li>Linux：在配置脚本中增加对 OpenBSD、Haiku 的检测</li> 
 <li>macOS：为 Big Sur 更新 Mac 图标</li> 
 <li>功能：支持 v2 版本的 torrent，同时支持 libtorrent 2.0.x</li> 
 <li>功能：支持 Qt6</li> 
 <li>功能：增加 "通知超时" 选项</li> 
 <li>功能：在高级设置中添加 connection_speed</li> 
 <li>功能：为各栏目添加工具提示</li> 
 <li>功能：在选项对话框中记住最后浏览的页面</li> 
 <li>功能：为监视文件夹添加 “跳过哈希检查” 选项</li> 
 <li>功能：允许为每个种子/类别设置临时文件夹</li> 
 <li>功能：支持基于文件夹的 UI 主题</li> 
 <li>BUGFIX：文件优先级改变后，保存 "恢复数据"</li> 
 <li>BUGFIX：如果在 "Add New Torrent" 对话框中没有其他菜单，则在顶层显示优先级菜单</li> 
 <li>BUGFIX：大写 "peer flags" 描述</li> 
 <li>BUGFIX：使用不同的配置文件修复启动</li> 
 <li>BUGFIX：将一些 torrent 上下文菜单操作移到 "Torrent 选项" 对话框中</li> 
 <li>BUGFIX：正确应用文件优先级更改</li> 
 <li>BUGFIX：当应用程序退出时禁用系统托盘图标菜单</li> 
 <li>BUGFIX：系统托盘图标初始化后显示 GUI 锁定图标</li> 
 <li>WEBUI：添加反向代理源 IP 解析</li> 
 <li>WEBUI：支持使用箭头键导航 UI 表</li> 
 <li>WEBUI：支持使用箭头键展开/折叠 UI 文件夹 (Thomas Piccirello)</li> 
 <li>WEBUI：设置图标尺寸属性</li> 
 <li>WEBUI：对 WebUI 的语言选择值进行排序</li> 
 <li>WEBUI：当 HTTPS 被启用时，在用户提示中使用正确的 URL 方案</li> 
 <li>其他：对内部代码进行了重构并修复了错误</li> 
 <li>……</li> 
</ul>
