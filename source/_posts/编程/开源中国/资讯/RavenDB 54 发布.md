                                                        <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">RavenDB 是由 Hibernating Rhinos Ltd. 开发的跨平台开源 ACID 文档数据库，用 C# 语言编写。它支持 Windows、Linux 和 macOS。RavenDB 将数据存储为 JSON 文档，可以部署在分布式集群中。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">RavenDB 5.4 近日正式发布，更新内容如下：</p> 
<h3 style="margin-left:.6em; margin-right:0; text-align:start">特性</h3> 
<ul style="margin-left:0; margin-right:0"> 
 <li>[ETL] 为 ETL 添加了 Kafka 支持</li> 
 <li>[ETL] 在 ETL 中添加了 RabbitMQ 支持</li> 
 <li>[Corax] 新的实验性搜索和索引引擎</li> 
</ul> 
<h3 style="margin-left:.6em; margin-right:0; text-align:start">Client API</h3> 
<ul style="margin-left:0; margin-right:0"> 
 <li>[Graph API] 将所有的 API 标记为过时</li> 
</ul>
