                                                                                            <p>在 KaOS Linux 2022.04 发布两个月后，开发团队近日发布了 KaOS Linux 2022.06，该版本带有最新的 KDE Frameworks 5.95 和 KDE Gear 22.04.2 软件套件，所有这些都是基于 Qt 5.15.5 框架构建的。</p> 
<p>在该版本中，KaOS 的 Plasma 主题 Midna 已经针对 KDE Plasma 5.25 桌面环境的变化进行了更新，视觉上最大的变化就是在登录和锁定屏幕中增加了一个虚拟键盘。</p> 
<p><img alt height="400" src="https://oscimg.oschina.net/oscnet/up-37e6d53958da2c53268a71cfef0bd979eb7.png" width="700" referrerpolicy="no-referrer"></p> 
<p>KaOS Linux 2022.06 版本还搭载了尚未发布的 Calamares 3.3 系列，这个强大而通用的 GNU/Linux 发行版通用图形安装程序。</p> 
<p>KaOS Linux 开发者因其对 Calamares 安装程序的贡献而闻名，即将推出的 Calamares 3.3 系列将带来改进的 KPMCore 集成，更好的 LUKS 支持和一个新的选项，可以用未加密的启动分区安装加密的系统，以及对图形用户界面的各种改进。</p> 
<p>新的 KaOS Linux 版本由 Linux 5.17 内核系列驱动，该内核现在在 <a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fkernel.org" target="_blank">kernel.org</a> 网站上被标记为 EOL（End-of-Life），这意味着它将不再接受维护更新。</p> 
<p>其他更新内容包括：</p> 
<ul> 
 <li>GNU C Library 2.35</li> 
 <li>GCC 11.3.0</li> 
 <li>GNU Binutils 2.38</li> 
 <li>systemd 250.7</li> 
 <li>DBus 1.14.0</li> 
 <li>Nettle 3.8</li> 
</ul>
