                                                                                            <p>Ant Design Mobile 5.0.0-beta.1 发布了。Ant Design Mobile 即 Ant Design 移动端设计规范，<code>antd-mobile</code> 是 Ant Design 的移动规范的 React 实现，服务于蚂蚁及口碑无线业务。</p> 
<p>更新内容如下：</p> 
<ul> 
 <li>优化 
  <ul> 
   <li>CSS 类名中的 <code>am-</code> 前缀调整为 <code>adm-</code>，从而避免 v2 和 v5 的 CSS 相互干扰</li> 
  </ul> </li> 
</ul> 
<p>迁移建议</p> 
<ul> 
 <li>检查项目中是否有用到了 antd-mobile 组件的 CSS 类名，如果有的话需要同步将 <code>am-</code> 前缀替换为 <code>adm-</code></li> 
 <li>由于 v2 和 v5 现在几乎不会出现样式的相互干扰了，所以一定程度上更好地进行渐进式的升级了</li> 
</ul>
