                                                                                            <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">JetBrains 日前发布了 IntelliJ IDEA 2022.1 系列的第三个小幅更新版本，该版本主要修复了之前存在的各种错误，以下是这次更新中一些重要的内容：</p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>修正了在运行有覆盖率的 Gradle 项目时出现的问题 [<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-235250%2FRun-with-coverage-in-a-Kotlin-Gradle-project-a-dialog-pops-up-saying-my-project-is-out-of-date-and-triggers-a-rebuild" target="_blank">IDEA-235250</a>]; [<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-194073%2FGradle-Run-with-Coverage-triggers-IDEA-JPS-build-Delegate-IDE-buildrun-actions-to-Gradle-setting-is-ignored" target="_blank">IDEA-194073</a>]</li> 
 <li>Git Log 面板中的 Cmd+Click/Ctrl+Click 已被修复，现在可按预期工作 [<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-292405%2FCtrlClick-on-file-paths-no-longer-works" target="_blank">IDEA-292405</a>]</li> 
 <li>欢迎屏幕上的 New Project 按钮现在又能按预期工作了 [<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-276553%2F20212-Welcome-Screen-New-Project-button-not-working" target="_blank">IDEA-276553</a>]</li> 
 <li>修复了阻止从 Java 类生成 WSDL 的问题 [<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-286585%2FGenerate-WSDL-from-Java-class-not-working" target="_blank">IDEA-286585</a>]</li> 
 <li>解决了 VCS 编辑器内菜单被截断的 UI 问题 [<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Fissue%2FIDEA-294928%2FVCS-marker-popup-is-cut" target="_blank">IDEA-294928</a>]</li> 
</ul>
