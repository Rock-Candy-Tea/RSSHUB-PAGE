                                                                                            <p>xm-select v1.2.4 已经发布，多选解决方案。</p> 
<p>此版本更新内容包括：</p> 
<h4>新增</h4> 
<ul> 
 <li>新增渲染完成的回调<code>done</code></li> 
 <li><code>create</code>支持一次性创建多个, 返回一个数组即可 <a href="https://gitee.com/maplemei/xm-select/issues/I40MU0" target="_blank">#I40MU0</a></li> 
 <li>增加预选配置<code>enableHoverFirst</code>, 默认选中第一项可选中数据 <a href="https://gitee.com/maplemei/xm-select/issues/I3ZCRX" target="_blank">#I3ZCRX</a></li> 
 <li>新增focus后的样式提醒</li> 
 <li>新增自定义设置选中键盘KeyCode配置<code>selectedKeyCode</code>, 可选 <code>xmSelect.KeyCode.Enter</code>, <code>xmSelect.KeyCode.Space</code>, 也可自行定义数值 <a href="https://gitee.com/maplemei/xm-select/issues/I41NNI" target="_blank">#I41NNI</a></li> 
 <li>表单验证失败后 滚动到可视范围内 ##灰度##</li> 
</ul> 
<h4>Bug fixes</h4> 
<ul> 
 <li>修复<code>tree</code>非严格模式下, 无法选中父节点</li> 
 <li>修复<code>tree</code>模式下搜索后无法折叠的问题 <a href="https://gitee.com/maplemei/xm-select/issues/I3XBF1" target="_blank">#I3XBF1</a></li> 
</ul> 
<p>详情查看：<a href="https://gitee.com/maplemei/xm-select/releases/v1.2.4">https://gitee.com/maplemei/xm-select/releases/v1.2.4</a></p>
