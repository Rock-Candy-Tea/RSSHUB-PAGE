                                                                                            <p>Bytebase 是一个基于网络、零配置、无依赖的数据库 Schema 变更和版本控制管理工具，适用于开发人员和 DBA。</p> 
<p>Bytebase 1.0 发布，更新内容如下：</p> 
<h3>增强</h3> 
<ul> 
 <li>(SQL Editor) 在运行按钮旁边增加一个 EXPLAIN 按钮</li> 
 <li>(SQL Editor) 重新设计数据库连接导航和连接状态</li> 
 <li>(SQL Editor) 运行 non-SELECT SQL 导航到不同的工作流程</li> 
 <li>改进租户项目 pipeline UI</li> 
 <li>转换到使用 PostgreSQL 来存储 Bytebase 元数据，这使 Bytebase 能够在未来处理更复杂的场景</li> 
 <li>增加准备环境的快速入门，包括 Bytebase、GitLab（macOS Apple Silicon）和 Clickhouse Cluster</li> 
 <li>从配置的 VCS（GitLab）同步项目成员</li> 
</ul> 
<h3>错误修复</h3> 
<ul> 
 <li>(SQL Editor) 查询结果表覆盖显示</li> 
 <li>(SQL Editor) 查询结果表属性顺序</li> 
</ul> 
<h3>停用</h3> 
<ul> 
 <li>以前，Bytebase 允许用户为回滚指定回滚 SQL，此功能已停用</li> 
</ul> 
<h3>团队计划（$29/月）</h3> 
<ul> 
 <li>基于角色的访问控制（RBAC），包括所有者、DBA、开发者等角色，以满足数据库开发过程中的合作</li> 
 <li>GitLab 登录</li> 
 <li>SQL 向后兼容检查和 Schema 漂移检测</li> 
 <li>……</li> 
</ul>
