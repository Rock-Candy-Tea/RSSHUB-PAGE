                                                                                            <p>PhpStorm 2021.2 RC 现已发布，这是 PhpStorm 重大更新之前的最终版本。与之前的 EAP 版本不同，此 RC 要求拥有有效的 PhpStorm 许可证。</p> 
<p><strong>主要更新内容</strong></p> 
<ul> 
 <li>缺少 PHPDoc 注释，选项在生成的文档块为空时不报告或至少删除快速修复</li> 
 <li>从 Python 迁移 macOS 环境阅读器</li> 
 <li>与本地比较打开 diff 窗口而不是修订和本地更改对话框之间的更改</li> 
 <li>当 diff 已经被打开时，单独窗口中的 diff 不会在显示 diff 的调用中得到关注</li> 
 <li>创建临时文件时忽略自定义文件类型</li> 
 <li>在某些情况下，从 Find Results 调用的 “New Scratch File” 会创建空文件</li> 
 <li>Scratch Files 弹出窗口中的排序问题</li> 
 <li>在某些情况下，智能删除工作会出错</li> 
 <li>无法在 Docker 中禁用 Run Built image 选项</li> 
 <li>所有文档都应该在 Docker Compose 操作之前保存</li> 
 <li>弹出窗口中的项目未显示在触摸栏上</li> 
 <li>单击 “文件” 菜单有时无法打开</li> 
 <li>添加注册表项以将 Code With Me 切换到 OSR 模式</li> 
</ul>
