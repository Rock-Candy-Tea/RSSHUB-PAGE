                                                                                            <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">Chrome 100 发布之后， Chrome 101 已<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fblog.chromium.org%2F2022%2F03%2Fchrome-101-federated-credential.html" target="_blank">升级为测试版</a>。测试版包含以下变化：</p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>引入试用版的 Federated Credential Management API (FedCM) ，该 API 旨在创建保护隐私的联合身份，而无需使用第三方 cookie 等跨站点跟踪机制。</li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fweb.dev%2Fpriority-hints%2F" target="_blank">优先提示功能</a>转为正式功能，优先提示功能可指示资源对浏览器的相对重要性，从而更好地控制加载资源的顺序。</li> 
 <li><code>font-palette</code><span> </span>CSS 属性允许开发人员从颜色字体中选择调色板。</li> 
 <li>新的<span> </span><code><em>hwb()</em><span> </span></code>CSS 函数，用于为色相白度和黑度 (HWB) 值指定 sRGB 颜色。</li> 
 <li>支持 WebRTC 流的 MediaCapabilities API </li> 
 <li>支持安全支付确认（Secure Payment Confirmation） API v3 版本的<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.chromestatus.com%2Ffeature%2F5675682238562304" target="_blank">新功能</a>。</li> 
 <li>新的<span> </span><code>USBDevice forget()</code><span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fweb.dev%2Fusb%2F%23revoke-access" target="_blank">方法</a>允许 Web 开发人员自愿撤销用户授予的 USBDevice 权限。</li> 
 <li>弃用并删除第三方上下文的 WebSQL 。</li> 
</ul> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">更多内容可在<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fblog.chromium.org%2F2022%2F03%2Fchrome-101-federated-credential.html" target="_blank">Chromium 博客<span> </span></a>中查看。</p>
