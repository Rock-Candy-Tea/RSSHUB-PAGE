                                                                                            <p>Canonical 近日发布了 Ubuntu 22.04 LTS (Jammy Jellyfish) 操作系统的 Beta 版本，让社区尽早体验即将到来的特性和变化。请注意，Beta 版主要是为了测试，不应该在你的生产系统中使用它。</p> 
<p>Ubuntu 22.04 LTS Beta 的更新内容包括：</p> 
<h3>重点色彩</h3> 
<p>虽然大多数其他基于 Ubuntu 的 Linux 发行版都提供了设置重点颜色的功能，但 Ubuntu 此前一直没有提供这个功能。</p> 
<p><img alt height="394" src="https://oscimg.oschina.net/oscnet/up-3b59a95c0a1ca556edcef375540f1a335bb.png" width="700" referrerpolicy="no-referrer"></p> 
<p>现在，在 Ubuntu 22.04 中，你可以轻松地设置你所选择的重点颜色，该设置会影响文件夹、通知区域和其他桌面元素。</p> 
<h3>GNOME 42</h3> 
<p><img alt height="469" src="https://oscimg.oschina.net/oscnet/up-f465313793b5e6909ddd119f3e3ded545ba.png" width="700" referrerpolicy="no-referrer"></p> 
<p>GNOME 42 带来了视觉上的变化、GNOME 应用程序的完善，以及新的屏幕截图应用程序。</p> 
<p>开启或关闭黑暗模式的体验会有所改善，全系统的黑暗模式实现也会随着 GNOME 42 而得到升级。</p> 
<p><img alt height="529" src="https://oscimg.oschina.net/oscnet/up-c5495c89211f75da8aad1d768c512a6b99d.png" width="700" referrerpolicy="no-referrer"></p> 
<h3>Snap 版本 Firefox</h3> 
<p><img alt height="251" src="https://oscimg.oschina.net/oscnet/up-9075ab7c2406e8c9975a675caad0a7e9cb6.png" width="700" referrerpolicy="no-referrer"></p> 
<p>Firefox 从 deb 包到 Snap 的过渡已经进行了一段时间了。在 Ubuntu 22.04 中，Firefox 现在将被默认为 Snap 包，Mozilla 旨在推送更快的安全更新并确保跨发行版的支持，而不需要维护众多的东西。</p> 
<h3>改进对 Raspberry Pi 的支持</h3> 
<p>随着新系统引入 zswap，Raspberry Pi 的性能在 Ubuntu 22.04 中应该会有明显的提升。它甚至可以在 2GB 的 Raspberry Pi 4 上运行。</p> 
<h3>Linux Kernel 5.15 LTS</h3> 
<p>Linux Kernel 5.15 LTS 是一个重要的升级，它改进了 NTFS 驱动程序，并对即将推出的英特尔独立显卡带来了初步支持，并有很多性能方面的改进。</p> 
<h3>用于 RISC-V 的 Live Image</h3> 
<p>从 Ubuntu 22.04 开始，Canonical 将会提供单独的 RISC-V 架构的 Live Image。</p> 
<h3>其他变化</h3> 
<p>除了上面提到的主要变化外，升级后的其他一些重要变化包括：</p> 
<ul> 
 <li>改进后的安装程序，具有更好的用户体验</li> 
 <li>更新的软件包，如 OpenSSL 3.0、Ruby 3.0、Python 3.10、PHP 8.1 等</li> 
</ul>
