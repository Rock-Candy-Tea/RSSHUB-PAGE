<p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><span style="background-color:#ffffff; color:#333333">V1.11主要功能更新</span><br> <span style="background-color:#ffffff; color:#333333">1)拦截器、多租户</span><br> <span style="background-color:#ffffff; color:#333333">2)二级缓存扩展支持,</span><strong style="color:#333333">Redis缓存</strong><span style="background-color:#ffffff; color:#333333">支持</span><br> <span style="background-color:#ffffff; color:#333333">3)</span><strong style="color:#333333">支持自定义TypeHandler,处理查询的ResultSet结果</strong><br> <span style="background-color:#ffffff; color:#333333">4)添加转换器SetParaTypeConvert转换PreparedStatement参数的类型</span><br> <span style="background-color:#ffffff; color:#333333">5)支持在Suid等对象设置</span><strong style="color:#333333">命名转换器</strong><span style="background-color:#ffffff; color:#333333">.</span><br> <span style="background-color:#ffffff; color:#333333">6)支持</span><strong style="color:#333333">Cassandra</strong><span style="background-color:#ffffff; color:#333333">.</span><br> <span style="background-color:#ffffff; color:#333333">7)添加</span><strong style="color:#333333">Jndi</strong><span style="background-color:#ffffff; color:#333333">数据源支持</span><br> <span style="background-color:#ffffff; color:#333333">8)PreparedSql</span><strong style="color:#333333">自定义sql支持批量插入</strong><span style="background-color:#ffffff; color:#333333">.</span><br> <span style="background-color:#ffffff; color:#333333">PreparedSql自定义sql支持多表查询,返回多表关联Javabean结构数据</span><br> <span style="background-color:#ffffff; color:#333333">9)</span><strong style="color:#333333">自定义动态SQL标签,@in,</strong><span style="background-color:#ffffff; color:#333333">@toIsNULL1,@toIsNULL2,<if isNotNull>,<if isNotBlank>.</span><br> <span style="background-color:#ffffff; color:#333333">动态sql,将list转为像in (1,2,3)的语句,不需要foreach,批量插入也</span><strong style="color:#333333">不需要foreach</strong><span style="background-color:#ffffff; color:#333333">.</span></p>
