                                                        <p>时隔三年，Mithril.js 发布了 v2.2.0。Mithril 是一个现代化的 JavaScript MVC 框架，用于构建单页面应用，<span style="background-color:#ffffff; color:#333333">可将应用分成数据层、UI 层和控制层。</span>它非常小巧（< 8kb gzip），且内置了路由和 XHR 工具。</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-fd28c718312310bc0987c26457f04c36f2b.png" referrerpolicy="no-referrer"></p> 
<p>本次更新包含一些小改动和补丁变更：</p> 
<ul> 
 <li>升级众多组件</li> 
 <li>改进文档</li> 
 <li>为 pr-release 启用 --minimize-semver-change</li> 
 <li>捕获格式错误的 URI 组件</li> 
 <li>从 removeAttr 中删除额外的 isLifecycleMethod 调用</li> 
 <li>删除无法访问的 keyed node 逻辑</li> 
 <li>……</li> 
</ul>
