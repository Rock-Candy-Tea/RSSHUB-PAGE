                                                                                            <p>CCleaner v5.77.8448 现已发布，此版本包括对 cleaning 自动化、可访问性的改进，CCleaner 的新用户还会看到一个新的初学者指南。</p> 
<p><img src="https://static.oschina.net/uploads/space/2021/0221/071749_zJN7_2720166.png" referrerpolicy="no-referrer"></p> 
<p>Cleaner 是一款系统优化和隐私保护工具，主要用来清除 Windows 系统的垃圾文件与使用者的上网记录。CCleaner 的体积小，运行速度极快，可以对临时文件夹、历史记录、回收站、注册表等进行垃圾清理。</p> 
<p>新版本更新内容如下：</p> 
<p><strong>Cleaning Settings</strong></p> 
<ul> 
 <li>恢复对本地域名的支持，将其加入"Cookies To Keep"白名单（如 localhost、desktop2）。</li> 
 <li>禁用了"Wipe Free Space"功能，同时正在调查 erroneous data erasure 的报告。</li> 
</ul> 
<p><strong>Cleaning Automation</strong></p> 
<ul> 
 <li>用户现在可以选择使用 standard cleaning rules 或 Custom Clean 来进行计划性 cleans</li> 
 <li>修复了取消选中“Smart Cleaning”并没有取消选中其子选项的错误</li> 
</ul> 
<p><strong>New Onboarding Flow</strong></p> 
<ul> 
 <li>添加了一个名为“ Help Hub  ”的新快速入门指南，该指南在某些用户的试用期间有效（目前只有英文）</li> 
</ul> 
<p><strong>Accessibility</strong></p> 
<ul> 
 <li>新的 CCleaner onboarding experience 完全支持 NVDA 屏幕阅读器并满足 WCAG AA 要求</li> 
 <li>应用窗口不同区域的标签导航顺序现在更加一致了</li> 
 <li>改进了 NVDA 的搜索字段标签</li> 
 <li>修复了如果运行 NVDA 时磁盘分析器在分析结束时挂起的问题</li> 
</ul>
