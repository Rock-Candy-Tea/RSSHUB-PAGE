                                                        <h3 style="margin-left:0; margin-right:0; text-align:left"><strong>一、更新内容简介</strong></h3> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><span style="background-color:#ffffff; color:#333333">本次更新为次要版本更新，进行了若干优化（</span><span style="background-color:#ffffff; color:#999999">更新历史详见：</span><a href="https://gitee.com/jackjiang/MobileIMSDK/tree/master/release_notes">码云 Release Nodes</a><span style="background-color:#ffffff; color:#333333">）</span>。可能是市面上唯一同时支持<span style="color:#4e5f70"><strong><span> </span>UDP+TCP+WebSocket</strong></span><span> </span>三种协议的同类开源 IM 框架。</p> 
<h3 style="margin-left:0; margin-right:0; text-align:left"><strong>二、MobileIMSDK 简介</strong></h3> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><img alt height="53" src="https://oscimg.oschina.net/oscnet/d4acb10eea465b539ca98a1a1b229aa6ab9.jpg" width="338" referrerpolicy="no-referrer"></p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:start"><strong><a href="https://gitee.com/jackjiang/MobileIMSDK">MobileIMSDK</a> 是一套专为移动端开发的原创 IM 通信层框架：</strong></p> 
<ul style="list-style-type:disc; margin-left:0; margin-right:0"> 
 <li>历经 8 年、久经考验；</li> 
 <li>超轻量级、高度提炼，lib 包 50KB 以内；</li> 
 <li>精心封装，一套 API 同时支持 UDP、TCP、WebSocket 三种协议（可能是全网唯一开源的）；</li> 
 <li>客户端支持 <strong>iOS</strong>、<strong>Android</strong>、<strong>标准 Java</strong>、<strong>H5</strong>、<strong>小程序</strong><span> </span>(<span style="color:#999999">开发中..</span>)、<strong>Uniapp</strong>(<span style="color:#999999">开发中..</span>)；</li> 
 <li>服务端基于 Netty，性能卓越、易于扩展；<img alt="👈" height="14" src="https://gitee.com/assets/emoji/point_left-4570b6929b8880ab2e8b8031007fef18.png" width="14" referrerpolicy="no-referrer"></li> 
 <li>可与姊妹工程 <a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fwww.52im.net%2Fthread-959-1-1.html" target="_blank">MobileIMSDK-Web</a> 无缝互通实现网页端聊天或推送等；<img alt="👈" height="14" src="https://gitee.com/assets/emoji/point_left-4570b6929b8880ab2e8b8031007fef18.png" width="14" referrerpolicy="no-referrer"></li> 
 <li>可应用于跨设备、跨网络的聊天 APP、企业 OA、消息推送等各种场景。</li> 
</ul> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">MobileIMSDK 工程始于 2013 年 10 月，起初用作某产品的即时通讯底层实现，<span style="color:#e74c3c">完全从零开发，技术自主可控！</span></p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><strong>您可能需要：</strong><a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fwww.52im.net%2Fthread-60-1-1.html" target="_blank">查看关于 MobileIMSDK 的</a><a href="https://gitee.com/jackjiang/MobileIMSDK/blob/master/README.md">详细介绍</a>。</p>

<h3 style="margin-left:0; margin-right:0; text-align:left"><strong>四、MobileIMSDK 设计目标</strong></h3> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">让开发者专注于应用逻辑的开发，底层复杂的即时通讯算法交由 SDK 开发人员，从而解偶即时通讯应用开发的复杂性。</p> 
<h3 style="margin-left:0; margin-right:0; text-align:left"><strong>五、MobileIMSDK 框架组成</strong></h3> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><strong>整套 MobileIMSDK 框架由以下 5 部分组成：</strong></p> 
<ol style="list-style-type:decimal; margin-left:0; margin-right:0"> 
 <li><strong>Android 客户端 SDK：</strong>用于 Android 版即时通讯客户端，支持 Android 2.3 及以上，<a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fdocs.52im.net%2Fextend%2Fdocs%2Fapi%2Fmobileimsdk%2Fandroid%2F" target="_blank">查看 API 文档</a>；</li> 
 <li><strong>iOS 客户端 SDK：</strong>用于开发 iOS 版即时通讯客户端，支持 iOS 9.0 及以上，<a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fdocs.52im.net%2Fextend%2Fdocs%2Fapi%2Fmobileimsdk%2Fios%2F" target="_blank">查看 API 文档</a>；</li> 
 <li><strong>Java 客户端 SDK：</strong>用于开发跨平台的 PC 端即时通讯客户端，支持 Java 1.7 及以上，<a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fdocs.52im.net%2Fextend%2Fdocs%2Fapi%2Fmobileimsdk%2Fjava%2F" target="_blank">查看 API 文档</a>；</li> 
 <li><strong>H5 客户端 SDK：</strong>资料整理中，不日正式发布；</li> 
 <li><strong>服务端 SDK：</strong>用于开发即时通讯服务端，支持 Java 1.7 及以上版本，<a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fdocs.52im.net%2Fextend%2Fdocs%2Fapi%2Fmobileimsdk%2Fserver%2F" target="_blank">查看 API 文档</a>。</li> 
</ol>

<h3 style="margin-left:0; margin-right:0; text-align:left"><strong>六、MobileIMSDK v6.2 更新内容 </strong></h3> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><strong>【重要说明】：</strong></p> 
<blockquote> 
 <p style="margin-left:0; margin-right:0"><span style="color:#40485b">MobileIMSDK v6.2 为次要版本，进行了若干优化！ </span><a href="https://gitee.com/jackjiang/MobileIMSDK/releases/6.2" target="_blank">查看详情</a></p> 
</blockquote> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><strong>【新增的特性】</strong></p> 
<blockquote> 
 <ol style="list-style-type:decimal; margin-left:0; margin-right:0"> 
  <li>[服务端]新增两个聊天消息前置处理回调，方便开发者进行内容鉴黄、过滤、修改等运营管理；</li> 
  <li>[服务端]新增新增了一个与Web互通情况下的C2C模式回调，用于开发者在互通模式下实现离线消息Push逻辑；</li> 
 </ol> 
</blockquote> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><strong>【其它优化和提升】</strong></p> 
<blockquote> 
 <ol style="list-style-type:decimal; margin-left:0; margin-right:0"> 
  <li>[Andriod]支持最新的Andriod 12，解决了Demo工程中的Andriod12兼容问题；</li> 
  <li>[Andriod]解决了Demo工程在最新Android Studio编译时报方法数超过65535的经典问题；</li> 
  <li>[服务端] 升级log4j2至2.17.0，解决Log4j2远程代码执行高危漏洞；</li> 
  <li>[服务端] 为ServerEventListener类中的onUserLogout回调增加beKickoutCode参数；</li> 
  <li>[服务端] [优化]尝试解决与Web互通情况下，MQProvider中的work方法会因异步消息导致的AlreadCloseException问题；</li> 
 </ol> 
</blockquote>
