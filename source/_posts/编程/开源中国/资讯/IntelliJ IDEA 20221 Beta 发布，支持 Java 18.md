                                                        <p>IntelliJ IDEA 2022.1 的测试版现在已经推出，开发者可以从网站、Toolbox 应用程序，或通过使用 Ubuntu 的 snaps 下载最新版本。</p> 
<p><img alt height="350" src="https://static.oschina.net/uploads/space/2022/0326/071648_FRz1_4937141.png" width="700" referrerpolicy="no-referrer"></p> 
<p>IntelliJ IDEA 2022.1 Beta 包含了一些功能和改进，以下是一些最值得注意的更新：</p> 
<ul> 
 <li>支持 Java 18</li> 
 <li>新的通知工具窗口</li> 
 <li>依赖性分析器</li> 
 <li>重新设计的新项目向导</li> 
 <li>Java 反编译器更新</li> 
 <li>支持 Kotlin 1.6.20-M1</li> 
 <li>增强的 Markdown 编辑器</li> 
 <li>与 Docker 和 Kubernetes 合作的新功能</li> 
 <li>支持 Go 微服务</li> 
 <li>可访问性改进</li> 
</ul> 
<p>解决的问题和交付的功能的完整列表可查看官方<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fyoutrack.jetbrains.com%2Farticles%2FIDEA-A-143%2FIntelliJ-IDEA-2022.1-Beta-%28221.5080.56-build%29-Release-Notes" target="_blank">发布说明</a>。</p>
