                                                        <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">Docker 20.10.10 现已发布，具体更新内容如下：</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><strong>IMPORTANT</strong></p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>由于<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgolang.org%2Fdoc%2Fgo1.16%23net%2Fhttp" target="_blank">Go 1.16</a><span> </span>中<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fgolang%2Fgo%2Fissues%2F40909" target="_blank">net/http</a><span> </span>的变化，通过 $HTTP_PROXY 环境变<span style="color:#000000">量配置的 HTTP 代理不再用于 TLS（https://）连接。用户需确保也设置一个 $HTTPS_PROXY 环境变量来处理对 https:// URL 的请求。可参阅</span> <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmoby%2Fmoby%2Fconfig%2Fdaemon%2Fsystemd.md%23httphttps-proxy" target="_blank">HTTP/HTTPS 代理部分</a>，以了解如何配置 Docker Daemon 以使用代理服务器。</li> 
</ul> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><strong>Builder </strong></p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>修复平台匹配逻辑，以解决使用 BuildKit 时，docker build 在 Arm 机器的本地图像缓存中找不到图像的问题 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmoby%2Fmoby%2Fpull%2F42954" target="_blank">moby/moby#42954</a></li> 
</ul> 
<h4 style="margin-left:.6em; margin-right:0; text-align:start"><strong>Runtime </strong></h4> 
<ul style="margin-left:0; margin-right:0"> 
 <li>在默认的 seccomp 策略中添加对 clone3 系统调用的支持，以支持基于最新版本的 Fedora 和 Ubuntu 运行容器。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmoby%2Fmoby%2Fpull%2F42836" target="_blank">moby/moby/#42836</a></li> 
 <li>Windows：更新 hcsshim 库，以修复容器层中稀疏文件处理的错误。这个错误由 Windows 中最近的更改暴露。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmoby%2Fmoby%2Fpull%2F42944" target="_blank">moby/moby#42944</a></li> 
 <li>修复了 docker stop 可能 hang forever 的一些情况。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmoby%2Fmoby%2Fpull%2F42956" target="_blank">moby/moby#42956</a></li> 
</ul> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><strong>Swarm</strong></p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>修复更新服务失败时不回滚的问题 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmoby%2Fmoby%2Fpull%2F42875" target="_blank">moby/moby#42875</a></li> 
</ul> 
<h4 style="margin-left:.6em; margin-right:0; text-align:start">Packaging</h4> 
<ul style="margin-left:0; margin-right:0"> 
 <li>添加适用于 Ubuntu 21.10 "Impish Indri" 和 Fedora 35 的软件包。</li> 
 <li>更新<code>docker scan</code>到 v0.9.0</li> 
 <li>更新 Golang runtime 到 Go 1.16.9。</li> 
</ul> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">更新说明：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmoby%2Fmoby%2Freleases%2Ftag%2Fv20.10.10" target="_blank">https://github.com/moby/moby/releases/tag/v20.10.10</a></p>
