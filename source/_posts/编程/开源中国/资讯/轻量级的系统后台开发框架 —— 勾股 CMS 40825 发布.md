                                                                                            <p style="color:#40485b; margin-left:0; margin-right:0; text-align:start"><span style="background-color:#ffffff; color:#40485b">勾股CMS是一套基于ThinkPHP6 + Layui + MySql打造的轻量级极速后台开发框架。系统设计思想很简单，以简洁为原则，结构清晰，易于功能扩展，代码维护，<span> </span></span><strong style="color:#40485b">支持【一键CRUD】生成代码功能</strong><span style="background-color:#ffffff; color:#40485b"><span> </span>，非常容易定制和二次开发，帮助开发者提高开发的效率极大降低了二次开发成本。</span><br> <span style="background-color:#ffffff; color:#40485b">系统集成了主流的通用功能：登录验证、系统配置、操作日志管理、角色权限管理、模块管理、一键CRUD、导航设置、网站地图、轮播广告、TAG关键字管理、文件上传、数据备份/还原、文章功能、图集功能、商品功能、单页面管理、用户管理、用户操作日志、用户注册/登录、 API接口等。更多的个性化功能可以基于当前系统便捷做二次开发。</span></p> 
<ul style="list-style-type:disc; margin-left:0; margin-right:0"> 
</ul> 
<h3 style="margin-left:0; margin-right:0; text-align:start"><strong>🔴更新日志</strong></h3> 
<p><span style="background-color:#ffffff; color:#40485b">1、更新前端UI框架layui版本为2.7.6最新版本；</span><br> <span style="background-color:#ffffff; color:#40485b">2、权限操作相关界面优化修改；</span><br> <span style="background-color:#ffffff; color:#40485b">3、新增主题切换，默认为简约白和经典黑，新增打开tab个数限制；</span><br> <span style="background-color:#ffffff; color:#40485b">4、新增一键crud功能，帮你完成60%以上的开发工作；</span><br> <span style="background-color:#ffffff; color:#40485b">5、新增图集模块；</span><br> <span style="background-color:#ffffff; color:#40485b">6、新增部门模块；</span><br> <span style="background-color:#ffffff; color:#40485b">7、新增职位模块；</span><br> <span style="background-color:#ffffff; color:#40485b">8、其他现有的其他功能模块代码重构；</span><br> <span style="background-color:#ffffff; color:#40485b">9、已知的其他问题修改。</span></p> 
<h3 style="margin-left:0; margin-right:0; text-align:left">⭕系统特点</h3> 
<ul> 
 <li>基于ThinkPHP6 + MySql + Layui 开发的轻量级、高性能极速后台开发框架。</li> 
 <li>通用型的后台权限管理框架，各管理模块操作简单，开箱即用。支持前后台用户的操作记录覆盖跟踪。</li> 
 <li>系统整合了易用、且功能强大的<strong>富文本编辑器（TinyMCE）</strong>与现今流行的<strong>Mardown编辑器（Editor.md）</strong>于自身，可在后台配置根据习惯切换不同的编辑器。封装和优化后，<strong>支持截屏复制粘贴上传图片，支持拖拽上传图片。</strong></li> 
 <li><strong style="color:#40485b">支持【一键CRUD】生成代码功能</strong><span style="background-color:#ffffff; color:#40485b"><span> </span>，非常容易定制和二次开发，帮助开发者提高开发的效率极大降低了二次开发成本。</span></li> 
 <li>可以快速基于此系统进行ThinkPHP6的快速开发，免去每次都写一次后台基础的痛苦，让开发更多关注业务逻辑。既能快速提高开发效率，帮助公司节省人力成本，同时又不失灵活性。</li> 
</ul>
