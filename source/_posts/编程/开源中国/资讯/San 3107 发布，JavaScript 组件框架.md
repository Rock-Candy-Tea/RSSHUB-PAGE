                                                        <p>San 是一个由百度开源的快速、轻量、灵活的 JavaScript 组件框架。San 3.10.7 现已正式发布，更新内容如下：</p> 
<ul> 
</ul> 
<ul> 
 <li>【新特性】- 组件增加 error hook 的支持</li> 
 <li>【新特性】- 组件的 watch 方法，handler 支持第二个参数 &#123;newValue, oldValue, change&#125;</li> 
 <li>【新特性】- main 入口根据 process.env.NODE_ENV 判断引用的版本</li> 
 <li>【bug修复】- 元素的 id 属性，当绑定表达式的值为 falsy(false / 0 /'') 时，属性没有正确设置</li> 
 <li>【bug修复】- 在 IE8-，模板的属性值不支持单引号包裹</li> 
</ul>
