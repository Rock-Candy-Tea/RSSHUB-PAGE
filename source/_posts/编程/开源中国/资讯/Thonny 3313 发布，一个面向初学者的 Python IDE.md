                                                                                            <p>Thonny 3.3.13 现已发布。3.3.12 版是一个错误修复版本（未在 GitHub 上发布），3.3.13 版增加了一个小的元数据更新相关（仅与 Linux 打包程序有关）。Thonny 是一个面向初学者的 Python IDE。Thonny 由爱沙尼亚的 Tartu 大学开发，它采用了不同的方法，因为它的调试器是专为学习和教学编程而设计的。 具体更新内容如下：</p> 
<p><strong>Changes since 3.3.11：</strong></p> 
<ul> 
 <li>为 RPi Pico 恢复同步到本地时间（修复了 3.3.11 中引入的回归）<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fthonny%2Fthonny%2Fissues%2F1886" target="_blank">#1886</a></li> 
 <li>修复在 MicroPython 中从 requirements.txt 安装的问题 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fthonny%2Fthonny%2Fissues%2F1879" target="_blank">#1879</a></li> 
 <li>允许为方法和函数调用指定不同的颜色 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fthonny%2Fthonny%2Fpull%2F1884" target="_blank">#1884</a></li> 
 <li>为插件扩展编辑器 API（关闭事件和内容 load/save hooks）<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fthonny%2Fthonny%2Fpull%2F1891" target="_blank">#1891</a></li> 
 <li>修复某些翻译中的技术错误 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fthonny%2Fthonny%2Fissues%2F1895" target="_blank">#1895</a></li> 
 <li>在 MicroPython 中输入 non-ascii 字符时发出警告 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fthonny%2Fthonny%2Fissues%2F1911" target="_blank">#1911</a></li> 
 <li>修复在 Windows 安装程序中为所有用户安装 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fthonny%2Fthonny%2Fissues%2F1119" target="_blank">#1119</a> 和 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fthonny%2Fthonny%2Fissues%2F1909" target="_blank">#1909</a></li> 
 <li>为 Flatpak 准备 Thonny <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fthonny%2Fthonny%2Fpull%2F1900" target="_blank">#1900</a> 和 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fthonny%2Fthonny%2Fpull%2F1912" target="_blank">#1912</a></li> 
 <li>更新 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Faivarannamaa%2Fminipip" target="_blank">minipip</a> 到 0.1b5 以便在安装 CircuitPython 的库时跳过安装 Blinka</li> 
 <li>更新波兰语翻译</li> 
 <li>更新 org.thonny.Thonny.appdata.xml</li> 
</ul> 
<p>更新说明：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fthonny%2Fthonny%2Freleases%2Ftag%2Fv3.3.13" target="_blank">https://github.com/thonny/thonny/releases/tag/v3.3.13</a></p>
