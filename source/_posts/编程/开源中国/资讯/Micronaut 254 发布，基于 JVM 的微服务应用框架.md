                                                        <p>Micronaut 是 Grails 框架作者打造的开源项目，也是新一代基于 JVM 的全栈微服务框架，用于构建模块化的、易于测试的微服务应用。有关 Micronaut 的特性介绍<a href="https://www.oschina.net/news/96381/micronaut-open-sourced">点此查看</a>。</p> 
<p>近日，Micronaut 2.5.4 正式发布，本次更新内容如下：</p> 
<p><strong>变化：</strong></p> 
<ul> 
 <li>通过更新 Netty 版本修正了一个内存泄露问题；</li> 
 <li>修正了关于 websocket 连接的回归问题；</li> 
 <li>修正了返回 <code>Any</code> 的 suspend 方法的编译错误；</li> 
 <li>修正了关于无效 cookie 的 NPE 问题；</li> 
</ul> 
<p><strong>依赖性升级：</strong></p> 
<ul> 
 <li>更新 Lombok 至 1.18.20；</li> 
 <li>更新 Micronaut Oracle Cloud 至 1.3.5；</li> 
 <li>更新 Micronaut Test 至 2.3.4；</li> 
 <li>更新 Micronaut Data 至 2.4.3；</li> 
 <li>升级至 Netty 4.1.64；</li> 
</ul> 
<p>更多详情可查看：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fmicronaut-projects%2Fmicronaut-core%2Freleases%2Ftag%2Fv2.5.4" target="_blank">https://github.com/micronaut-projects/micronaut-core/releases/tag/v2.5.4</a></p>
