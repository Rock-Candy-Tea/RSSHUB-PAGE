<h3>更新内容</h3> 
<ul> 
 <li>增加用户批量导入</li> 
 <li>增加微信公众号扫码登录</li> 
 <li>增加开始学习功能</li> 
 <li>增加系统安全配置</li> 
 <li>增加电子执照，ISP许可配置</li> 
 <li>增加腾讯云视频ID支持</li> 
 <li>优化验证码</li> 
 <li>优化拼团详情封面样式</li> 
 <li>优化课程以及套餐发货</li> 
 <li>优化点播上传转码流程</li> 
 <li>修正二级分类查询条件</li> 
 <li>修正电子兑换卡导出问题</li> 
 <li>完善积分兑换会员</li> 
 <li>完善标签查询规则</li> 
</ul> 
<h3>友情提示</h3> 
<p>（１）本次更新需要重新打包PHP镜像，然后再执行upgrade.sh脚本</p> 
<pre><code>sudo docker-compose build --pull php
sudo docker-compose up -d --force-recreate --build php
sudo bash upgrade.sh
</code></pre> 
<p>（２）系统后台 -> 系统管理 -> 注册登录 -> 基本设置，重新提交配置</p> 
<h3>系统介绍</h3> 
<p>酷瓜云课堂，依托腾讯云基础服务架构，采用C扩展框架Phalcon开发，致力在线教育系统。</p> 
<h3>系统功能</h3> 
<p>实现了点播、直播、专栏、面授、问答、会员、群组、微聊、积分、秒杀、拼团、分销等。</p> 
<p>官方网站：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fkoogua.com" target="_blank">酷瓜云课堂</a></p>
