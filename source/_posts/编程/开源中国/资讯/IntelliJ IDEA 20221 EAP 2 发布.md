                                                                                            <p><span data-darkreader-inline-bgcolor data-darkreader-inline-color style="--darkreader-inline-bgcolor:#181a1b; --darkreader-inline-color:#cecac3; background-color:#ffffff; color:#27282c">IntelliJ IDEA 2022.1 EAP 2 现已推出，提供了对 Java 反编译器和 QA 工具的一系列改进，以及对 VCS 工具的 UI 增强、Groovy 模板支持和捆绑的 TOML 插件。</span></p> 
<p><span data-darkreader-inline-bgcolor data-darkreader-inline-color style="--darkreader-inline-bgcolor:#181a1b; --darkreader-inline-color:#cecac3; background-color:#ffffff; color:#27282c">具体更新内容包括：</span></p> 
<h4><strong><span><span><span><span><span><span><span data-darkreader-inline-color style="--darkreader-inline-color:#cecac3; color:#27282c"><span><span><span><span><span><span><span><span><span><span><span data-darkreader-inline-bgcolor style="--darkreader-inline-bgcolor:#181a1b; background-color:#ffffff"><span><span><span>Java 反编译器</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></strong></h4> 
<p><span data-darkreader-inline-bgcolor data-darkreader-inline-color style="--darkreader-inline-bgcolor:#181a1b; --darkreader-inline-color:#cecac3; background-color:#ffffff; color:#27282c">对 IntelliJ IDEA 中的内置 Java 反编译器实施了许多值得注意的更新，以使其与最新的 Java 17 版本更加兼容。</span>反编译器现在支持现代语言构造器，例如密封类型（sealed types）和模式匹配。此外，它还具有更好的字符串反编译切换功能，提供类型注释，并使用其 names 而不是 literal values 来检测 <span data-darkreader-inline-bgcolor data-darkreader-inline-color style="--darkreader-inline-bgcolor:#181a1b; --darkreader-inline-color:#cecac3; background-color:#ffffff; color:#27282c">common constants</span>。 </p>

<h4><strong><span><span><span><span><span><span><span data-darkreader-inline-color style="--darkreader-inline-color:#cecac3; color:#27282c"><span><span><span><span><span><span><span><span><span><span><span data-darkreader-inline-bgcolor style="--darkreader-inline-bgcolor:#181a1b; background-color:#ffffff"><span><span><span>质量检查工具</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></strong></h4> 
<p style="margin-left:0px; margin-right:0px; text-align:start"><strong><span><span><span><span><span><span><span data-darkreader-inline-color style="--darkreader-inline-color:#cecac3; color:#27282c"><span><span><span><span><span><span><span><span><span><span><span data-darkreader-inline-bgcolor style="--darkreader-inline-bgcolor:#181a1b; background-color:#ffffff"><span><span><span>Page Object Editor 改进</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></strong></p> 
<p style="margin-left:0px; margin-right:0px; text-align:start"><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fplugins.jetbrains.com%2Fplugin%2F13691-selenium-ui-testing" target="_blank">Selenium UI 测试插件</a>的 Page Object Editor 得到了巨大的改进。优化了工作流程并重新设计了 UI，以确保出现更少的干扰元素。与以前的版本不同，你可以即时将网页中的页面元素添加到代码中，然后继续编辑，而无需离开 Page Object Editor。</p> 
<p style="margin-left:0px; margin-right:0px; text-align:start"><img alt height="200" src="https://oscimg.oschina.net/oscnet/up-55368678f0c0aadfe31eb28d7234fdb8a7a.gif" width="500" referrerpolicy="no-referrer"></p> 
<p style="margin-left:0px; margin-right:0px; text-align:start"><span data-darkreader-inline-bgcolor data-darkreader-inline-color style="--darkreader-inline-bgcolor:#181a1b; --darkreader-inline-color:#cecac3; background-color:#ffffff; color:#27282c">在更新版本的代码中添加新的页面元素时，你可以选择使用哪种选择器类型（即 CSS、XPath、Aria-label 等）。</span></p> 
<p style="margin-left:0px; margin-right:0px; text-align:start"><span data-darkreader-inline-bgcolor data-darkreader-inline-color style="--darkreader-inline-bgcolor:#181a1b; --darkreader-inline-color:#cecac3; background-color:#ffffff; color:#27282c"><img alt height="200" src="https://oscimg.oschina.net/oscnet/up-02969613ddf2d9bc0ea13471c7bf3dee8ad.gif" width="500" referrerpolicy="no-referrer"></span></p> 
<p style="margin-left:0px; margin-right:0px; text-align:start"><strong><span><span><span><span><span><span><span data-darkreader-inline-color style="--darkreader-inline-color:#cecac3; color:#27282c"><span><span><span><span><span><span><span><span><span><span><span data-darkreader-inline-bgcolor style="--darkreader-inline-bgcolor:#181a1b; background-color:#ffffff"><span><span><span>新 Page Object wizard</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></strong></p> 
<p style="margin-left:0px; margin-right:0px; text-align:start">新 Page Object wizard 允许你在创建新页面对象类时选择语言和测试自动化框架。且该向导在你的代码中所有与 Selenium 库有依赖关系的地方都可用。</p>

<p>通过添加大多数类似 java 的后缀模板来改进 Groovy 的 <span><span><span><span><span><span><span data-darkreader-inline-color style="--darkreader-inline-color:#cecac3; color:#27282c"><span><span><span><span><span><span><span><span><span><span><span data-darkreader-inline-bgcolor style="--darkreader-inline-bgcolor:#181a1b; background-color:#ffffff"><span><span><span>code completion</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span>，例如 arg、cast、for、iter、nn、notnull、null、not、throw、try、var、par、sout、serr、while、new、reqnonnull 和 return。</p>
