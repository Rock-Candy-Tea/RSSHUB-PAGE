                                                                                            <p>FreeBSD 13.1 首个 RC 版本已发布。主要更新内容包括：修复 VSF、更新 LLVM 以修复在 POWER 上构建 Rust 移植的问题、Linux ABI 相关的更新和修复，从上游引入 OpenZFS 2.1，针对 802.11 WiFi/无线的多项更新和错误修复，以及大量关于 ARM64 (AArch64) 的修复。</p> 
<p>FreeBSD 13.1 对于 CPU 架构，如 POWER 和 RISC-V，已经取得了巨大进展。在此版本中，更广泛的整体改进包括：</p> 
<ul> 
 <li>支持 FreeBSD PowerPC 架构上的 LLVM 调试器 (LLDB)</li> 
 <li>为 PowerPC / PowerPC64 / PowerPC64LE 上的 OpenSSL 启用优化的汇编代码</li> 
 <li><span style="background-color:#ffffff; color:#121212">支持在 RISC-V 64 位上构建 AddressSanitizer (ASAN) 和 UndefinedBehaviorSanitizer (UBSAN)</span></li> 
 <li><span style="background-color:#ffffff; color:#121212">启用在 RISC-V 64 位上构建 OpenMP 库</span></li> 
 <li>多项 PowerPC 64 位修复程序</li> 
 <li>为<span style="background-color:#ffffff; color:#121212">更多 FreeBSD 内核设备驱动程序提供 big endian 支持</span></li> 
</ul>
