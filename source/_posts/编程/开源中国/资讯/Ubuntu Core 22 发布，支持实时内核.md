                                                        <p>Canonical 正式发布了 Ubuntu Core 22，Ubuntu Core 是 Canonical 针对嵌入式设备、物联网和其他工业硬件的精简版 Ubuntu。它是普通 Ubuntu 的一个容器化版本，与完整的 Ubuntu 相比，Ubuntu Core 非常轻量、具有更加安全的设计，并支持使用 Snaps 的事务性软件更新。</p> 
<p><img alt height="394" src="https://static.oschina.net/uploads/space/2022/0617/072754_MsBv_4937141.jpg" width="700" referrerpolicy="no-referrer"></p> 
<p>Ubuntu Core 22 是最新的版本，从它的版本号可以看出它是基于 Ubuntu 22.04 LTS 的。它得到了Canonical 公司对内核、操作系统和应用程序级代码长达 10 年的安全维护支持。</p> 
<p>在 Ubuntu Core 22 的新功能中，有一个实时内核的测试版、为对时间敏感的工业、电信、汽车和机器人用例提供高性能、超低延迟和工作负载可预测性。</p> 
<p>其他特性：</p> 
<ul> 
 <li>迁移和向后兼容：Ubuntu Core 22 保持了与 Ubuntu Core 20 相同的分区布局，以便更顺利地升级和向后兼容；</li> 
 <li>与 LTS 接轨：该操作系统基于 Ubuntu 22.04 LTS，并将在 22.04 LTS 的整个生命周期内得到全面支持（直到 2032年）；</li> 
 <li>性能改进：占用空间和内存使用都得到了优化和减少，Ubuntu Core 22 还提供了更快的启动时间和快照执行时间；</li> 
 <li>出厂重置：将 Ubuntu Core 设备恢复到一个已知的原始状态，同时保留初始化过程中创建的基本数据。这在 Ubuntu Core 20 中也可以做到，但新版本的操作系统使这项任务变得更加容易；</li> 
 <li>对其包含的 snaps 内的服务设置资源限制，目前支持最大内存和 CPU 使用率；</li> 
 <li>支持 MAAS，即 Metal As A Service；</li> 
</ul>
