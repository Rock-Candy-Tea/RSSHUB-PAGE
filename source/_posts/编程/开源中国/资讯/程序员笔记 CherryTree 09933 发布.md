                                                                                            <p>CherryTree 0.99.33 现已发布。CherryTree 是一个支持无限层级分类的笔记软件，Python 编写，支持富文本编辑和代码高亮，支持 Linux 和 Windows 平台。数据采用 sqlite 或 XML 存储，支持密码保护，支持从 NoteCase、KeepNote、Knowit、Tomboy、TuxCards、Treepad、Leo 等笔记软件导入数据。</p> 
<p><img height="342" src="https://static.oschina.net/uploads/space/2021/0127/070748_SLU0_2720166.png" width="500" referrerpolicy="no-referrer"></p> 
<p><img height="342" src="https://static.oschina.net/uploads/space/2021/0127/070803_tv03_2720166.png" width="500" referrerpolicy="no-referrer"></p> 
<p>此版本更新内容如下：</p> 
<ul> 
 <li>修复了在 Windows 上编辑键盘快捷键时崩溃的问题</li> 
 <li>修复了更改以百分比设置宽度的代码框宽度的问题</li> 
 <li>return key presses 和数字键盘的 enter key presses 一视同仁</li> 
 <li>修复了从 Cherrytree 文件导入时缺少询问 parent node 的对话框的问题</li> 
 <li>user-* themes 代码是不可选择，现在不包括在列表中</li> 
 <li>为 Dart 语言添加了自动语法高亮显示</li> 
 <li>更新了捷克语</li> 
 <li>更新了芬兰语</li> 
</ul>
