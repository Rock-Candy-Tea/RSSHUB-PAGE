                                                                                            <p>Redis Desktop Manager 是基于 Qt 5 的 Redis 开源跨平台桌面管理器。Redis Desktop Manager 2022.0 发布，更新内容如下：</p> 
<h3>常规</h3> 
<p><strong>Redis Desktop Manager 项目被重新命名为 RESP.app，不再使用 Redis Desktop Manager 名称</strong></p> 
<h3>新功能和改进</h3> 
<ul> 
 <li> <p>新的适合黑暗模式的图标和布局控制：</p> 
  <ul> 
   <li>连接树可以隐藏起来，以便专注于键值、控制台或服务器图表</li> 
   <li>全屏模式用于编辑键值</li> 
  </ul> <p><img alt height="491" src="https://static.oschina.net/uploads/space/2022/0206/071845_SzNc_4937141.png" width="700" referrerpolicy="no-referrer"></p> </li> 
 <li> <p>现在可以为每个连接调整格式化行为。默认情况下，RESP.app 会尝试自动检测最常见的格式，即 JSON、HEX 和纯文本。你可以改变这一行为，加载最后选定的格式，或为特定 redis-server 中的所有键选择特定的格式。</p> <p><img alt height="689" src="https://static.oschina.net/uploads/space/2022/0206/071825_16Qg_4937141.png" width="700" referrerpolicy="no-referrer"></p> </li> 
</ul> 
<h3>修复</h3> 
<ul> 
 <li>SSH：增加对 RFC8332 的支持</li> 
 <li>修复命令自动补全和更新 commands.json</li> 
 <li>修复 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fuglide%2FRedisDesktopManager%2Fissues%2F5166" target="_blank">#5166</a></li> 
</ul>
