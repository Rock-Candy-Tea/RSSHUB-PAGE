<div class="content">
                                                                    
                                                        <p style="margin-left:0px; margin-right:0px; text-align:start">Windows Terminal Preview 1.12 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdevblogs.microsoft.com%2Fcommandline%2Fwindows-terminal-preview-1-12-release%2F" target="_blank">已发布</a>。</p> 
<p>更新亮点：</p> 
<ul> 
 <li>支持将 Windows Terminal 设置为默认终端</li> 
 <li>匹配配置文件</li> 
 <li>支持恢复窗口</li> 
 <li>增加完全透明模式</li> 
</ul> 
<h3>将 Windows Terminal 设置为默认终端</h3> 
<p>在 Windows 11 或 <span><span><span><span style="color:#333333"><span><span><span><span><span><span><span><span><span><span><span><span style="background-color:#ffffff"><span><span><span>Windows Insider Program Dev Channel 中，用户可以将 Windows Terminal 设置为默认终端。设置默认终端后，当我们启动任何命令行应用程序时，它将自动在 Windows Terminal 内打开。</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-2c1f2d4799ecc6b706c19c3a578260fd1d5.png" referrerpolicy="no-referrer"></p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-6dd695f30c190f916e13b65272161738be6.png" referrerpolicy="no-referrer"></p> 
<h3>匹配配置文件 (Profile matching)</h3> 
<p>Windows Terminal 现在支持正确处理终端配置文件的可执行启动匹配，以便在符合条件的情况下启用所有自定义设置。比如点击“开始”菜单中的命令提示符，将打开对应的 CMD 配置文件，而不是调用 cmd.exe 的默认配置文件。目前该功能仅可在 Widnows Terminal 预览版中使用。</p> 
<p><img src="https://static.oschina.net/uploads/space/2021/1021/175933_GylQ_2720166.gif" referrerpolicy="no-referrer"></p> 
<h3>窗口恢复</h3> 
<p>Windows Terminal 增加了在重新启动后恢复上次会话的选项。</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-c5371e7c90e442e0752684dd3507e30204f.png" referrerpolicy="no-referrer"></p> 
<h3>增加完全透明模式</h3> 
<p>Windows Terminal 在 Windows 11 上现已支持完全透明模式，<span style="background-color:rgba(255, 255, 255, 0.65); color:#000000">可通过 Ctrl + Shift + Scroll 组合键来调节窗口的透明度。当背景变为透明时，控制台中的文本不会透明，确保阅读效果正常。</span></p> 
<p><img src="https://static.oschina.net/uploads/space/2021/1021/182505_io5U_2720166.gif" referrerpolicy="no-referrer"></p> 
<h3>其他改进</h3> 
<ul> 
 <li><span style="background-color:#ffffff; color:#333333">支持使用键盘选择已打印在缓冲区中的文本</span></li> 
 <li><span style="background-color:#ffffff; color:#333333">支持自动生成 Visual Studio Developer Command Prompt (VS2017+)</span> 和 <span style="background-color:#ffffff; color:#333333">Visual Studio Developer PowerShell (VS2019.2+)</span> 的配置文件</li> 
 <li>支持使用 Alt + Space 组合键打开系统菜单，此快捷键会使用新的<code><span>openSystemMenu</span></code><span style="background-color:#ffffff; color:#333333"><span> </span>action</span></li> 
 <li><span style="background-color:#ffffff; color:#333333">……</span></li> 
</ul>
