                                                                                            <p>smart-doc 2.2.6 已经发布，Java Restful API 文档生成工具</p> 
<p>此版本更新内容包括：</p> 
<h2>New Features</h2> 
<ul> 
 <li>新增接口分组支持。</li> 
 <li>html文档静态资源支持离线模式。</li> 
 <li>支持在html文档中不显示参数列表。</li> 
</ul> 
<h2>Bug Fixes</h2> 
<ul> 
 <li>修复Boolean类型字段命名为is前缀时，is前缀被去除的bug。</li> 
 <li>修复配置了字典码列表后枚举字段注释显示错误#139 。</li> 
 <li>修复分析过程中出现的数组越界异常</li> 
</ul> 
<p>详情查看：<a href="https://gitee.com/smart-doc-team/smart-doc/releases/2.2.6">https://gitee.com/smart-doc-team/smart-doc/releases/2.2.6</a></p>
