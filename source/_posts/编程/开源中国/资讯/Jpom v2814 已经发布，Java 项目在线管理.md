<p>此版本更新内容包括：</p> 
<h3>新增功能</h3> 
<ol> 
 <li>【server】构建触发器新增获取当前状态接口 （感谢@wxyShine）</li> 
 <li>【server】构建确认流程支持修改分支（感谢@）</li>

</ol> 
<h3>解决BUG、优化功能</h3> 
<ol> 
 <li>【server】修复编辑构建 svn 类型仓库无法提交问题（感谢<a href="https://www.oschina.net/gjfjie">@杰 </a> ）</li> 
 <li>【server】修复无法使用 git ssh 问题</li> 
</ol>
