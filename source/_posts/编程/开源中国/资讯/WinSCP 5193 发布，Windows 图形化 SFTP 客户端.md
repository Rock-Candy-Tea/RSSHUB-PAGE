                                                        <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">WinSCP 是一个 Windows 环境下使用的 SSH 的开源图形化 SFTP 客户端，同时支持 SCP 协议，它的主要功能是在本地与远程计算机间安全地复制文件，并且可以直接编辑文件。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">目前，WinSCP 5.19.3 已经发布。该版本具体更新内容如下：</p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>翻译更新：法语。</li> 
 <li>TLS/SSL core 升级到 OpenSSL 1.1.1l。</li> 
 <li>当资源管理器界面中上次使用的本地目录不再存在时，使用 Documents 文件夹。</li> 
 <li>错误修复：当服务器需要重用先前传输的会话时，TLS 会话恢复不适用于使用 TLS 1.3 的后续 FTP 传输。</li> 
 <li>错误修复：当访问策略检查空前缀时无法访问 S3 bucket root。</li> 
 <li>错误修复：无法识别来自 ProFTPD FTP 校验命令的响应。</li> 
 <li>错误修复：在选择“Never ask me again”的情况下提交提示时失败。</li> 
 <li>错误修复：切换 Full row select 后面板绘制不正确。</li> 
 <li>错误修复：使用 TLS 1.3 将文件上传到某些 FTP 服务器时超时。</li> 
 <li>错误修复：S3 服务器的不完整列表，在内容之后指示截断列表，并且其分页是 8 的倍数（e.g. Backblaze）。</li> 
</ul>
