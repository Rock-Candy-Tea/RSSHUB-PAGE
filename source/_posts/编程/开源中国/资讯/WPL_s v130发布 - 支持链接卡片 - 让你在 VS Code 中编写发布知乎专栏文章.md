  WPL/s 是一个 VS Code 插件，允许你在 VS Code 中用 Markdown 写知乎文章。最新版支持知乎链接卡片。
 </div> 
 <div>
   
 </div> 
 <p>链接卡片格式如下</p> 
 <div class="highlight"> 
  <pre><code class="language-text">[![zhihu-link-card:本项目 GitHub 主页](./pics/vs-code-extension-search-zhihu.png)](https://github.com/jks-liu/WPL-s)
</code></pre> 
 </div> 
 <p>语法上和一个图片链接一样，但图片的文字需要以<code>zhihu-link-card:</code>开头。知乎链接卡片的外观如下：</p>
