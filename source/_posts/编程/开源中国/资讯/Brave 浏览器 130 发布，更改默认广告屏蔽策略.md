                                                        <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">Brave 是基于 Chromium 引擎开发的免费开源网络浏览器。Brave 是一个注重隐私的浏览器，它通过在默认设置中自动阻止在线广告和网站跟踪器来将自己与其他浏览器区分开来。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">Brave Browser 1.30 正式发布，该版本更新内容如下：</p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>为 Brave 新增了处理来自 Windows Shell 和 Cortana 搜索的功能 (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fbrave%2Fbrave-browser%2Fissues%2F13875" target="_blank">#13875</a>)</li> 
 <li>在<span> </span><code>Brave://settings/search</code><span> </span>下增加了 "索引其他搜索引擎" 的设置 (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fbrave%2Fbrave-browser%2Fissues%2F14496" target="_blank">#14496</a>)</li> 
 <li>增加了在 macOS 和 Linux 上通过管理策略（Admin Policy）禁用 Tor 的功能 (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fbrave%2Fbrave-browser%2Fissues%2F17530" target="_blank">#17530</a>)</li> 
 <li>增加了在<span> </span><code>brave://rewards-internals</code><span> </span>日志中右击的功能 (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fbrave%2Fbrave-browser%2Fissues%2F16915" target="_blank">#16915</a>)</li> 
 <li>增加了 IPFS 网关 URL 验证功能 (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fbrave%2Fbrave-browser%2Fissues%2F16999" target="_blank">#16999</a>)</li> 
 <li>更新了默认（标准）广告屏蔽策略，以始终允许第一方请求 (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fbrave%2Fbrave-browser%2Fissues%2F17366" target="_blank">#17366</a>)</li> 
 <li>从 URL 中删除了已知的用户跟踪参数 (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fbrave%2Fbrave-browser%2Fissues%2F17507" target="_blank">#17507</a>, <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fbrave%2Fbrave-browser%2Fissues%2F17452" target="_blank">#17452</a>, <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fbrave%2Fbrave-browser%2Fissues%2F17451" target="_blank">#17451</a>)</li> 
 <li>修正了下载信息栏中的重叠文本 (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fbrave%2Fbrave-browser%2Fissues%2F17313" target="_blank">#17313</a>)</li> 
 <li>修正了 Brave 特定的 URL 没有被列在<span> </span><code>brave://about</code><span> </span>下的问题 (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fbrave%2Fbrave-browser%2Fissues%2F17299" target="_blank">#17299</a>)</li> 
 <li>修正了在<span> </span><code>brave://settings/ipfs/keys</code>下无法删除 IPNS 键 (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fbrave%2Fbrave-browser%2Fissues%2F17484" target="_blank">#17484</a>)</li> 
 <li>修正了在某些情况下 IPFS 关机和重启的问题 (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fbrave%2Fbrave-browser%2Fissues%2F16978" target="_blank">#16978</a>)</li> 
 <li>将 Chromium 升级到 94.0.4606.61 (<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fbrave%2Fbrave-browser%2Fissues%2F18333" target="_blank">#18333</a>)</li> 
</ul>
