                                                                                            <p>Keka 是一个 macOS 端的应用程序，可以快速创建具有高压缩率的压缩文件，还支持从多种类型的文件中解压缩文件，其中包括 7z、ISO、DMG、TAR、ZIP 和 Bzip2 等类型的文件格式。</p> 
<p>Keka 1.2.53 具体更新内容如下：</p> 
<ul> 
 <li>增加了配置压缩和提取默认密码的功能</li> 
 <li>删除了密码重试 8 次的限制</li> 
 <li>增加了任务失败时的额外信息</li> 
 <li>在弹出窗口中建议使用 UTF-8 编码</li> 
 <li>完善了任务视图中的 show in Finder 操作</li> 
 <li>在 12.0+ 系统中正确显示 DMG 和 ISO 任务图标</li> 
 <li>多个行为和性能的调整</li> 
</ul> 
<p>格式</p> 
<ul> 
 <li>增加了对 Compact Pro (.cpt) 提取支持</li> 
 <li>添加 Synology 软件包（.spk）提取支持</li> 
</ul> 
<p>修复</p> 
<ul> 
 <li>修正了 ZIP 和 7Z 与旧版本和 Windows 编码的兼容性</li> 
 <li>修复了提取文件名包含表情符号的 ZIP 文件的问题</li> 
</ul> 
<p>翻译</p> 
<ul> 
 <li>更新了简体中文、繁体中文、日文等多个国家和地区的语言翻译</li> 
</ul>
