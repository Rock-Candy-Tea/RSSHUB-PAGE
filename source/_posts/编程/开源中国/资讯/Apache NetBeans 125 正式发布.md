                                                                                            <p>Apache NetBeans 12.5 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fnetbeans.apache.org%2Fdownload%2Fnb125%2Findex.html" target="_blank">已正式发布</a>，属于季度性的功能更新版本（非 LTS），因此没有像 LTS 版本那样经过严格的测试。目前的 LTS 版本是 Apache NetBeans 12.0。</p> 
<p>12.5 增强了对 Java、C/C++、PHP 和其他常用的工具的支持。</p> 
<p>主要更新内容包括：</p> 
<ul> 
 <li>增强对 Java 的支持 
  <ul> 
   <li>暂时支持配置 <span style="background-color:#ffffff; color:#0a0a0a">SecurityManager</span></li> 
   <li>新增正则表达式窗口和提示</li> 
   <li>新增面向 JDK 18 的 javadoc</li> 
   <li>向 LSP 服务器添加声明式的 Java 提示</li> 
  </ul> </li> 
 <li>增强对 PHP 的支持 
  <ul> 
   <li>支持使用来自<span style="background-color:#ffffff; color:#0a0a0a"><span> </span>Skeleton Generator 的分支的 URL</span></li> 
   <li>当属性用作构造函数参数时，显示未使用的属性提示</li> 
   <li>添加一个选项（系统属性），用于禁用折叠 PHP 标签的功能</li> 
  </ul> </li> 
 <li>增强对 C/C++ 的支持 
  <ul> 
   <li>修复新创建的 cpplite 项目，避免在配置不可读时启动 C/C++ LSP 服务器</li> 
   <li>在 cpplite 项目中，当读取配置时，空字符串不会被认为是有效配置</li> 
  </ul> </li> 
</ul>
