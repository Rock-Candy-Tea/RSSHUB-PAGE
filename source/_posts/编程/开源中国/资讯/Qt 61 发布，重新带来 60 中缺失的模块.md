                                                        <p>Qt 6.1 已正式发布，Qt 6.1 是 Qt 6 系列中的第一个小版本更新。Qt 6.1 最大的亮点是把 Qt 5.15 中支持的许多附加模块带到 Qt 6 中。</p> 
<h3>新模块：</h3> 
<ul> 
 <li>Active Qt：支持 Windows 上的 COM 和 ActiveX 控件的 Qt 模块；</li> 
 <li>Qt Charts：在应用程序中添加图表；</li> 
 <li>Qt Data Visualization：实现静态和动态数据的可视化；</li> 
 <li>Qt Device Utilities：Qt 的一部分，仅用于 Device Creation；</li> 
 <li>Qt Graphical Effects：作为兼容模块添加，以帮助用户移植到 Qt 6；</li> 
 <li>Qt Lottie：在 Adobe After Effects 中创建的图形和动画渲染；</li> 
 <li>Qt State Machine: 包含了 Qt 5 中 Qt Core 的 State Machine API 以及基于 SCXML 的 State Machine 代码；</li> 
 <li>Qt Virtual Keyboard: 在嵌入式设备上添加一个虚拟键盘；</li> 
</ul> 
<p>这些模块带回了很多在 Qt 6.0 中缺失的 Qt 5.15 的功能。目前还有一些模块正在努力改善中，准备随着 Qt 6.2 的发布而带到 Qt 6 中。</p>
