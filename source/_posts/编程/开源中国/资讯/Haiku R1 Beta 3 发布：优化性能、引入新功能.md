                                                                                            <p>Haiku OS 于 7 月 25 日<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.haiku-os.org%2Fnews%2F2021-07-26_media_release_the_haiku_project_celebrates_the_release_of_beta_3%2F" target="_blank">发布</a>了第三个 Beta 测试版，即 R1/Beta3。此版本引入了不少新功能和性能改进，这也意味着离 R1 正式版的到来更近了一步。</p> 
<p>主要变化：</p> 
<ul> 
 <li>浏览器内核 WebKit 获得多项重大改进，这为浏览器渲染体验提供了良好的基础</li> 
 <li>更便捷的安装过程，对磁盘分区程序的用户界面 (DriveSetup GUI) 进行了多项错误修复，带来更轻松、更顺畅的安装过程</li> 
 <li>改进硬件驱动，包括 
  <ul> 
   <li>改进 Wi-FI 网络以匹配 FreeBSD 13</li> 
   <li>改进音视频驱动、大容量存储、USB</li> 
   <li>优化在 NVIDIA 显卡 (GeForce 6200-GeForce Go 6400) 上的性能</li> 
  </ul> </li> 
 <li>提升 WebPositive 的稳定性和兼容性</li> 
 <li>出现网络问题时可恢复软件更新</li> 
 <li>弃用 Python 2，将 Python 3 作为默认安装的版本</li> 
 <li>更高的 POSIX 兼容性</li> 
 <li>修复多项错误</li> 
</ul> 
<p><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.haiku-os.org%2Fget-haiku%2Fr1beta3%2Frelease-notes%2F" target="_blank">完整变更内容点此查看</a>。</p> 
<p>Haiku 前身是 BeOS 操作系统，在 BeOS 被收购后更名为 Haiku。Haiku 延续了 BeOS 的用户体验和个性化，它是一个快速、简洁优雅、易学易用，而且非常强大的操作系统。“Haiku”一词来源于日语“俳句”，因 BeOS 里大量输出信息使用俳句的形式而得名。</p>
