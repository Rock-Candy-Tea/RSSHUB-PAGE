                                                                                            <p style="text-align:left">electron-egg 是一个简单、快速、功能丰富的跨平台桌面软件开发框架。</p> 
<h4 style="text-align:left">特性</h4> 
<ol> 
 <li>直接打包成 windows 版、Mac 版、Linux 版、或 web 网站运行</li> 
 <li>可以用服务端的开发思维，来编写桌面软件</li> 
 <li>也可以用前端来开发，数据服务请求外部 api 即可</li> 
 <li>服务端的技术场景几乎都可以使用，如：路由、中间件、控制器、服务、定时任务、队列、插件等</li> 
 <li>桌面软件常见功能，后续逐步集成并完善或提供 demo</li> 
 <li>自动更新</li> 
</ol> 
<h4 style="text-align:left">本次更新：</h4> 
<ol> 
 <li><span style="color:#404040">实现渲染进程与主进程通信功能及demo。</span></li> 
 <li>修复资源迁移时错误。</li> 
 <li><span style="color:#404040">修复自动更新时，闪退的问题。</span></li> 
 <li><span style="color:#404040">增加工具库文件。</span></li> 
</ol>
