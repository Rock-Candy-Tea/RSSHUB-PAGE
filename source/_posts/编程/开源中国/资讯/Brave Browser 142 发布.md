                                                                                            <p>Brave 是基于 Chromium 的一款免费开源网络浏览器，主打隐私保护，其默认设置就会自动阻止在线广告和网站跟踪。Brave Software（Brave 浏览器所属公司）于 2015 年由 Brendan Eich 和 Brian Bondy 联合创立，其中前者是 JavaScript 的创造者，同时也是 Mozilla 联合创始人。</p> 
<p>Brave 1.42 发布，更新内容如下：</p> 
<ul> 
 <li>将 Chromium 升级到 104.0.5112.81</li> 
 <li>增加了从 Chromium 版本 Microsoft Edge 导入的功能</li> 
 <li>在 Brave://settings/search 下增加了隐私窗口地址栏中使用的搜索引擎的设置</li> 
 <li>在 brave://settings/appearance 下增加了细化地址栏中自动完成建议的设置</li> 
 <li>增加了根据应用在路径上的正则表达式来调整 URL 的能力</li> 
 <li>[安全] 添加了一个警告，表明本地 IPFS 节点将绕过浏览器代理</li> 
 <li>[安全] 修正了 IPFS 崩溃的问题</li> 
 <li>[安全] 修正了 Brave Wallet API 冻结问题</li> 
 <li>在某些地区的新安装中，将默认搜索引擎更新为 Brave Search</li> 
 <li>在 Sparkle 中更新了 macOS 最低支持版本至 10.13</li> 
 <li>更新了私人窗口和带 Tor 的私人窗口的新标签页设计</li> 
 <li>更新了 brave://settings/cookies，以更好地与 brave://shields 对齐</li> 
 <li>删除了 P3A protobuf 报告</li> 
 <li>删除了 "从 Brave 获取图片描述" 的上下文菜单选项</li> 
 <li>从 URL 中删除了已知的 Marketo 电子邮件跟踪参数</li> 
 <li>删除了未使用的 Brave Shields 代码</li> 
 <li>修复了无法点击固定标签页的音频图标来取消该标签的静音</li> 
 <li>修复了打开新标签页时的画面闪烁问题</li> 
 <li>修复了在子框架内不应用外观过滤器的问题</li> 
</ul> 
<p>更多详情可查看：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fbrave.com%2Flatest%2F" target="_blank">https://brave.com/latest/</a></p>
