                                                        <p>Apache OpenOffice 4.1.11 现已发布。这是一个包含错误修复和少量改进的安全版本，官方强烈建议所有 Apache OpenOffice 4.1.10 或更早版本的用户进行升级。</p> 
<p>公告指出，其希望将此版本的 Apache OpenOffice 献给最近去世的 Patricia Shanahan。Patricia 是一位活跃的开发人员、Committer 和 PMC，“我们将非常怀念她”。Pat 于 2015 年加入该项目。</p> 
<h4>Security</h4> 
<ul> 
 <li> <p><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fcve.mitre.org%2Fcgi-bin%2Fcvename.cgi%3Fname%3D2021-33035" target="_blank">CVE-2021-33035</a> - 来自 crafted DBF file 的缓冲区溢出</p> </li> 
 <li> <p><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fcve.mitre.org%2Fcgi-bin%2Fcvename.cgi%3Fname%3D2021-40439" target="_blank">CVE-2021-40439</a> - Billion Laughs</p> </li> 
 <li> <p><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fcve.mitre.org%2Fcgi-bin%2Fcvename.cgi%3Fname%3D2021-28129" target="_blank">CVE-2021-28129</a> - 使用 non-root userid 和 groupid 安装的 Apache OpenOffice 4.1.8 的 DEB packaging</p> </li> 
</ul> 
<h4>Improvements/Enhancements</h4> 
<p><strong>Other Improvements/Enhancements</strong></p> 
<ul> 
 <li>帮助文本字体较小（难以阅读）且无法持续调整</li> 
 <li>将字体添加到插入菜单</li> 
 <li>文件菜单中缺少 PDF 导出图标</li> 
</ul> 
<h4>Bug 修复</h4> 
<p><strong>Highlighted Improvements/Enhancements</strong></p> 
<ul> 
 <li> <p>AOO 4.1.10 引入了新的安全警告阻止了有用的功能</p> </li> 
 <li> <p>Windows 安装文件的属性缺失，并且缺少代码签名</p> </li> 
 <li> <p>Windows 安装：切换到 NSIS 3.x 后丢失的 adapted texts</p> </li> 
 <li> <p>以 .ODS 格式保存时图表丢失</p> </li> 
</ul>
