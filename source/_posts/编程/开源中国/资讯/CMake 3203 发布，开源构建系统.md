                                                        <p>CMake 3.20.3 已经发布，CMake 是一个跨平台的自动化构建系统，它使用一个名为 CMakeLists.txt 的文件来描述构建过程，可以产生标准的构建文件，如 Unix 的 Makefile 或Windows Visual C++ 的 projects/workspaces 。文件 CMakeLists.txt 需要手工编写，也可以通过编写脚本进行半自动的生成。</p> 
<p><strong>主要更新内容</strong></p> 
<ul> 
 <li>在 IDE 集成指南中使用相对路径链接到预设模式</li> 
 <li>使用更私密的临时变量名</li> 
 <li>增加对 Boost 1.76 的支持</li> 
 <li>在符号链接的构建树中恢复对 Fortran 的支持</li> 
 <li>更新 Sphinx 4 的手册页面配置</li> 
 <li>如果更新到还没有的哈希值，确保 git fetch</li> 
 <li>只在 git 1.7.7 或更高版本中添加 git 配置设置</li> 
 <li>通过配置分割长命令行</li> 
 <li>在生成时检查 EXCLUDE_FROM_ALL 属性</li> 
 <li>改进 nvcc 详细输出中的 CUDA Toolkit root 的正则表达式</li> 
 <li>正确地记录解析失败的情况</li> 
 <li>提供更多语法错误信息</li> 
</ul>
