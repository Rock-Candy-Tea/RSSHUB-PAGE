<div class="content">
                                                                                            <p>MX Linux 21 Beta 2 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fmxlinux.org%2Fblog%2Fmx-21-beta-2-now-available-for-testing-purposes%2F" target="_blank">已发布</a>。</p> 
<p>更新亮点：</p> 
<ul> 
 <li>引入新的应用、升级应用</li> 
 <li>引入新的安装程序分区选择区域，以及对 lvm 的部分支持（如果存在 lvm 卷）</li> 
 <li>引入新的 UEFI live 系统引导菜单，支持从启动菜单和子菜单中选择 live 启动选项（例如 persistence），而不是使用以前的控制台菜单</li> 
 <li>Xfce 4.16, Plasma 5.20, fluxbox 1.3.7 with mx-fluxbox 3.0 configs</li> 
 <li>管理员任务 (admin tasks) 支持使用默认密码，并可以在 mx-tweak-> Other标签中切换</li> 
 <li>部分翻译的更新</li> 
 <li>将 mesa Vulkan 驱动程序添加到默认软件包集合中</li> 
 <li>……</li> 
</ul> 
<p>另外值得注意的是，官方希望主要版本号可遵循基于年份的规则，因此原本的 MX Linux 20 已被跳过，直接从 MX Linux 19 跳到了 MX Linux 21。</p> 
<p><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fmxlinux.org%2Fblog%2Fmx-21-beta-2-now-available-for-testing-purposes%2F" target="_blank">下载地址</a></p> 
<p>MX Linux 是基于 Debian 稳定分支的面向桌面的 Linux 发行，它是 antiX 及早先的 MEPIS Linux 社区合作的产物。它采用 Xfce 作为默认桌面环境，是一份中量级操作系统，并被设计为优雅而高效的桌面与如下特性的结合：配置简单、高度稳定、性能可靠、占用空间不大也不小。</p>
