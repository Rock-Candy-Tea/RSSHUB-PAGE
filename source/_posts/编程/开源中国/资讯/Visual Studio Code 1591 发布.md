                                                        <p>Visual Studio Code 1.59.1 发布，该版本属于小型修复版本，解决了以下这些问题：</p> 
<ul> 
 <li>修复了笔记本无法在 1.59 上打开的问题；</li> 
 <li>认证解密错误；</li> 
 <li>在 VS Code v1.59 中， <code>tasks.onDidEndTaskProcess</code> 被调用的次数过多；</li> 
 <li>通用构建版本在 M1 MBP 上出现永久挂起/崩溃的情况；</li> 
 <li>Stdout 生成的原始数据文件不符合预期格式；</li> 
 <li>在 js 中输入重复的 <code>.</code> 会自动展开第一个点的属性建议；</li> 
</ul>
