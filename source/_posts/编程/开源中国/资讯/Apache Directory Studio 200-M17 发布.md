                                                                                            <p>Apache Directory Studio 2.0.0-M17 现已发布。Apache Directory Studio 是一个完整的目录工具平台旨在与任何 LDAP 服务器一起使用，尤其是设计用于 ApacheDS。本身是一个 Eclipse RCP 应用程序，由几个 Eclipse（OSGi）插件组成，可以轻松升级，这些插件甚至可以在 Eclipse 本身中运行。</p> 
<p><strong>主要更新内容</strong></p> 
<ul> 
 <li>bug 修复 
  <ul> 
   <li>使用 ADS LDAP 客户端 API 时 GSSAPI 身份验证失败</li> 
   <li>Directory Studio 在认证前不启动 TLS</li> 
   <li>Directory Studio 在协商使用后不使用 SASL 保密层</li> 
   <li>对象被删除时不会从缓存中删除</li> 
   <li>无法创建具有强制二进制属性的条目</li> 
   <li>SOCKS 代理不工作</li> 
   <li>Oracle OUD 11.1.2.3 没有在 M16 的 ldapbrowser 中列出任何命名上下文，以前的版本是可以的</li> 
   <li>搜索日志/修改日志未显示在 MacOS 中</li> 
   <li>只有代理服务器显示目录树</li> 
   <li>org.apache.directory.studio.connection.core.io.api.LdifSearchLogger 的非法反射访问</li> 
  </ul> </li> 
 <li>优化 
  <ul> 
   <li>显示 SSL/TLS 连接信息和证书</li> 
   <li>删除网络连接首选项页面（socks 代理设置）</li> 
  </ul> </li> 
</ul>
