                                                        <p>Universal Media Server 是一个兼容 DLNA，UPnP 和 HTTP/S 的媒体服务器，支持所有主要操作系统，包括 Windows，Linux 和 macOS 版本，几乎不需要配置就可以流式传输或转码许多不同的媒体格式。</p> 
<p>Universal Media Server 10.6.0 发布，该版本更新内容如下：</p> 
<p>通用：</p> 
<ul> 
 <li>增加了对 UPnP 搜索的支持；</li> 
 <li>删除了损坏的新媒体文件夹，因为该功能存在于媒体库中；</li> 
 <li>改进了剧集检测和预置功能；</li> 
 <li>修正了数据库连接失败和挂起的进程；</li> 
 <li>修正了媒体库/缓存设置的描述</li> 
</ul> 
<p>渲染器：</p> 
<ul> 
 <li>改进了对索尼 XBR OLED 电视的支持；</li> 
</ul> 
<p>依赖性：</p> 
<ul> 
 <li>将 checkstyle 从 8.42 升级至 8.43；</li> 
 <li>将 common-io 从 2.8.0 升级至 2.9.0；</li> 
 <li>将 macOS 上的 FLAC 升级至 1.3.3；</li> 
 <li>将 macOS 上的 tsMuxeR 升级至 2021-04-30-02-13-20；</li> 
 <li>将 gson 从 2.8.6 升级至 2.8.7；</li> 
 <li>将 jna-version 从 5.7.0 升级至 5.8.0；</li> 
 <li>将 junit5.version 从 5.7.1 升级至 5.7.2；</li> 
 <li>将 maven-javadoc-plugin 从 3.2.0 升级至 3.3.0；</li> 
 <li>将 oshi-core 从 5.5.0 升级至 5.7.4；</li> 
</ul>
