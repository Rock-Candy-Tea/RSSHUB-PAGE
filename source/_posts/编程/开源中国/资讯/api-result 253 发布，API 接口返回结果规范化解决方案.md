                                                                                            <p>api-result 2.5.3 已经发布，API 接口返回结果规范化解决方案。</p> 
<p>此版本更新内容包括：</p> 
<ul> 
 <li>【优化】返回的属性为空时不再自动隐藏，交由业务系统自行配置</li> 
 <li>【修复】ResponseTemplate 无参数构造方法 private > public，解决无法被继承的问题</li> 
 <li>【删除】删除依赖 jackson-annotations</li> 
</ul> 
<p>详情查看：<a href="https://gitee.com/fengwenyi/api-result/releases/2.5.3">https://gitee.com/fengwenyi/api-result/releases/2.5.3</a></p>
