                                                        <p style="text-align:start">Nebula Graph Studio v2.2.0 在原有图探索可视化看板基础上增强了相应的绘图操作，改善了图探索的交互，新增快捷键、看板数据导出等场景功能。</p> 
<div style="text-align:start">
 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdiscuss-cdn.nebula-graph.com.cn%2Fuploads%2Fdefault%2Foriginal%2F2X%2Fd%2Fd257580d31f2d9264460c2f1a608e158b35bf9c5.jpeg" target="_blank"><img alt="image" height="322" src="https://discuss-cdn.nebula-graph.com.cn/uploads/default/optimized/2X/d/d257580d31f2d9264460c2f1a608e158b35bf9c5_2_690x322.jpeg" width="690" referrerpolicy="no-referrer"></a>
</div> 
<h2 style="text-align:start">New Features</h2> 
<ul> 
 <li>增强操作面板，新增颜色、锁定、画板搜索等功能</li> 
 <li>支持快捷键，例如：拓展、放大、缩小、撤销、删除等操作</li> 
 <li>完善可视化拓展功能 
  <ul> 
   <li>支持自定义步数</li> 
   <li>支持自定义颜色</li> 
   <li>新增 FIND PATH 图路径算法 
    <ul> 
     <li>最短路</li> 
     <li>全路径</li> 
     <li>非循环路径</li> 
    </ul> </li> 
  </ul> </li> 
 <li>新增鼠标右键便捷操作</li> 
 <li>完善画板数据导出功能 
  <ul> 
   <li>csv 数据导出，支持全量数据和选中数据导出两种模式</li> 
   <li>查询结果直接生成图片</li> 
  </ul> </li> 
</ul> 
<h2 style="text-align:start">Bugfix</h2> 
<ul> 
 <li>修复控制台 match 查询结果无法导入图探索问题</li> 
</ul>
