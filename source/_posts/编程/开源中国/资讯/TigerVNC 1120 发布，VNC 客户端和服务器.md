                                                                                            <p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><span style="color:#24292f">TigerVNC 1.12.0 现已推出。</span><span style="color:#333333">TigerVNC 是一个高级的 VNC 远程访问的实现，</span><span style="color:#24292f">允许你查看运行在网络上另一台计算机上的虚拟桌面环境并与之交互；它是基于 RealVNC 4 和 X.org 代码库的高速 VNC 版本。</span></p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><span style="color:#24292f">重点更新内容如下：</span></p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>Native viewer 现在支持在一部分监视器上全屏显示（例如 3 个中的 2 个），并对添加或删除的监视器做出正确反应</li> 
 <li>Native viewer 中的最近服务器历史记录</li> 
 <li>如果连接断开，native viewer 现在可以选择重新连接</li> 
 <li>现在在 Windows 和 macOS 上启用了 native viewer 的翻译功能</li> 
 <li>Native viewer 现在尊重系统安全策略¹</li> 
 <li>在 Java viewer 中更好地处理 accented keys</li> 
 <li>Unix 服务器现在可以同时监听 Unix 套接字和 TCP 端口</li> 
 <li>服务器和 native viewer 中的网络代码都经过重组，以提供更灵敏的体验</li> 
 <li>vncserver 服务现在可以正确处理设置为“0”的设置</li> 
 <li>修复了 native viewer 和服务器中剪贴板 Unicode 处理的问题</li> 
 <li>在 Xvnc 和 native viewer 中支持 pointer "warping"，支持例如 FPS 游戏</li> 
</ul> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><span style="color:#24292f">¹ 除非 GnuTLS 是静态链接的。</span></p>
