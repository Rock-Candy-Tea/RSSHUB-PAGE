                                                                                            <p>Visual Studio Code 是一个开源的代码编辑器，支持 IntelliSense、调试、Git 和代码片断。可在 Windows、Linux 和 macOS 上下载使用。支持常见的脚本和编程语言，还可以通过安装扩展来获得更多语言和功能的支持。</p> 
<p>近日微软发布了 1.64 版本，更新内容如下：</p> 
<ul> 
 <li> <p><strong><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fcode.visualstudio.com%2Fupdates%2Fv1_64%23_new-side-panel" target="_blank">新的侧面板</a></strong> —— 使用新的侧面板一次显示更多视图。</p> <p><img alt height="573" src="https://static.oschina.net/uploads/space/2022/0207/070342_YbGo_4937141.gif" width="700" referrerpolicy="no-referrer"></p> </li> 
 <li> <p><strong><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fcode.visualstudio.com%2Fupdates%2Fv1_64%23_settings-editor" target="_blank">设置编辑器搜索</a></strong> —— 搜索现在优先考虑整个单词的匹配<img alt height="220" src="https://static.oschina.net/uploads/space/2022/0207/073553_cvWz_4937141.png" width="700" referrerpolicy="no-referrer"></p> </li> 
 <li> <p><strong><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fcode.visualstudio.com%2Fupdates%2Fv1_64%23_audio-cues" target="_blank">音频提示</a></strong> —— 当光标移动到折叠区域、错误和断点时能听到</p> </li> 
 <li> <p><strong><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fcode.visualstudio.com%2Fupdates%2Fv1_64%23_unicode-highlighting-improvements" target="_blank">Unicode 高亮显示</a></strong> —— 避免高亮显示受支持语言中的字符</p> </li> 
 <li> <p><strong><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fcode.visualstudio.com%2Fupdates%2Fv1_64%23_automatic-replies" target="_blank">自动终端回复</a></strong> —— 对常见的终端提示创建自动回复<img alt height="354" src="https://static.oschina.net/uploads/space/2022/0207/073608_nx44_4937141.gif" width="700" referrerpolicy="no-referrer"></p> </li> 
 <li> <p><strong><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fcode.visualstudio.com%2Fupdates%2Fv1_64%23_notebooks" target="_blank">Notebook UI 改进</a></strong> —— 在 Markdown 和输出单元中搜索文本<img alt height="394" src="https://static.oschina.net/uploads/space/2022/0207/073632_LMZr_4937141.gif" width="700" referrerpolicy="no-referrer"></p> </li> 
 <li> <p><strong><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fcode.visualstudio.com%2Fupdates%2Fv1_64%23_viewing-and-editing-binary-data" target="_blank">调试二进制数据视图</a></strong> —— 调试时查看和编辑二进制数据</p> </li> 
 <li> <p><strong><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fcode.visualstudio.com%2Fupdates%2Fv1_64%23_markdown-path-intellisense" target="_blank">Markdown 路径建议</a></strong> —— 快速插入相对文件路径和 header 链接<img alt height="319" src="https://static.oschina.net/uploads/space/2022/0207/073644_k6XK_4937141.png" width="700" referrerpolicy="no-referrer"></p> </li> 
 <li> <p><strong><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fcode.visualstudio.com%2Fupdates%2Fv1_64%23_surround-with-snippets-for-jsts" target="_blank">JS/TS 用代码段括起来</a></strong> —— 在代码段中插入选定的代码</p> </li> 
 <li> <p><strong><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fcode.visualstudio.com%2Fupdates%2Fv1_64%23_remote-repositories" target="_blank">VS Code for the Web</a></strong> —— 支持在 vscode.dev 和 github.dev 中签名的 GitHub 提交</p> </li> 
</ul>
