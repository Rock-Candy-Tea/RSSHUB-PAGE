<div class="content">
                                                                    
                                                        <p>Spring Framework 6.0.0 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fspring.io%2Fblog%2F2022%2F05%2F12%2Fspring-framework-6-0-0-m4-available-now" target="_blank">发布</a>了第 4 个里程碑版本，此版本包含所有针对 5.3.20 的修复补丁，以及特定于 6.0 分支的 39 项修复和改进。</p> 
<p>主要变化</p> 
<ul> 
 <li>为核心直接添加原生提示 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fspring-framework%2Fissues%2F28442" target="_blank">#28442</a></li> 
 <li>将 byte[] 添加到 SimpleJmsHeaderMapper 中支持的类型 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fspring-framework%2Fissues%2F28421" target="_blank">#28421</a></li> 
 <li>添加接受 ResolvableType 的 RootBeanDefinition 构造函数 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fspring-framework%2Fissues%2F28418" target="_blank">#28418</a></li> 
 <li>添加 Throwable 函数式接口 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fspring-framework%2Fissues%2F28417" target="_blank">#28417</a></li> 
 <li>支持多个 SpringFactoriesLoader 文件 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fspring-framework%2Fissues%2F28416" target="_blank">#28416</a></li> 
 <li>添加 ObjectUtils.addObjectToArray 的位置变量 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fspring-framework%2Fissues%2F28415" target="_blank">#28415</a></li> 
 <li>支持 Jakarta Concurrency 3.0 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fspring-framework%2Fissues%2F28400" target="_blank">#28400</a></li> 
 <li>为 HttpExchange 方法添加 header 解析器 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fspring-framework%2Fpull%2F28395" target="_blank">#28395</a></li> 
 <li>FormHttpMessageConverter 不应依赖于 Jakarta Mail API <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fspring-framework%2Fissues%2F28392" target="_blank">#28392</a></li> 
 <li>支持从带注释的 HTTP 服务接口创建代理 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fspring-projects%2Fspring-framework%2Fissues%2F28386" target="_blank">#28386</a></li> 
</ul>
