                                                                                            <p>IPython 7.26 现已发布，这是一个次要版本，修复了一些问题、更新了 API，以及围绕代码库各部分的版权/许可证问题。IPython 提供了丰富的工具包，可帮助你以交互方式充分利用 Python。 </p> 
<p>官方强调了 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fipython%2Fipython%2Fissues%2F13039" target="_blank">https://github.com/ipython/ipython/issues/13039</a> 这个问题，指出其包括并引用了 Stack Overflow 的代码；而这些代码是 CC-BY-SA 的，因此与 IPython 的 BSD 许可不兼容。这导致开发团队重写了相应的逻辑，在其案例中以一种更有效的方式完成（搜索的是字符串前缀而不是完整的字符串）。</p> 
<p>此外，新版本还还包括了许多文档改进和清理。以及：</p> 
<ul> 
 <li>IPython 指令现在使用 Sphinx logging 来处理警告。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fipython%2Fipython%2Fpull%2F13030%2F" target="_blank">PR#13030</a>。</li> 
 <li>为 pastebin magic 添加到期日选项并将 http 协议更改为 https。 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fipython%2Fipython%2Fpull%2F13056%2F" target="_blank">PR#13056</a></li> 
 <li>使 Ipython.utils.timing 与 jupyterlite 一起工作。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fipython%2Fipython%2Fpull%2F13050%2F" target="_blank">PR #13050</a></li> 
</ul> 
<p>更多详情可查阅<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fipython.readthedocs.io%2Fen%2Fstable%2Fwhatsnew%2Fversion7.html%23ipython-7-26" target="_blank">发行说明</a>。</p>
