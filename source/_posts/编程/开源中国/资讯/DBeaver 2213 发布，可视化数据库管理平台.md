                                                                                            <p>DBeaver 是一个免费开源的通用数据库工具，适用于开发人员和数据库管理员。</p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">DBeaver 22.1.3 现已发布，更新内容如下：</p> 
<ul style="margin-left:0; margin-right:1.5em"> 
 <li>SQL editor： 
  <ul style="margin-left:0; margin-right:0"> 
   <li>修复了带有前缀 @ 的变量的 Autocomplete 问题</li> 
   <li>分组面板中的列别名问题已解决</li> 
   <li>SQL 控制台文本现在可以保存为项目脚本</li> 
  </ul> </li> 
 <li>Data editor： 
  <ul style="margin-left:0; margin-right:0"> 
   <li>集合字符串表示得到了改进</li> 
   <li>数据读取性能问题已修复</li> 
  </ul> </li> 
 <li>Data transfer： 
  <ul style="margin-left:0; margin-right:0"> 
   <li>在导入向导中添加了目标表属性（例如引擎、字符集等）</li> 
   <li>CSV 导入已针对具有无效值报价的文件修复</li> 
   <li>修复了从字符串到几何列的导入问题</li> 
  </ul> </li> 
 <li>全文搜索向导 (NullPointer) 中的问题已解决</li> 
 <li>General UI：改进了 tab 快速搜索（按子字符串查找、模糊搜索）</li> 
 <li>Connections：对非关系数据库禁用了多连接模式</li> 
 <li>Azure SQL：驱动程序版本已升级，auth token 刷新问题已解决</li> 
 <li>Clickhouse： 
  <ul style="margin-left:0; margin-right:0"> 
   <li>表引擎和引擎参数（可编辑）已添加到表属性中</li> 
   <li>View code editor 已修复</li> 
  </ul> </li> 
 <li>DB2：dependent views 可以从表 DDL 中排除</li> 
 <li>Oracle： 
  <ul style="margin-left:0; margin-right:0"> 
   <li>添加了 DDL 选项卡到 scheduler jobs 中</li> 
   <li>新表列属性已修复</li> 
  </ul> </li> 
 <li>Debian 安装程序的问题已解决（配置文件标记已删除）</li> 
</ul>
