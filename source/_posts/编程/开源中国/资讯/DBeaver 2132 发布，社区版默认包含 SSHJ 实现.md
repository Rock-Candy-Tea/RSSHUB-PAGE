                                                                                            <p>DBeaver 是一个免费开源的通用数据库工具，适用于开发人员和数据库管理员。</p> 
<p>DBeaver 21.3.2 发布，更新内容如下：</p> 
<ul> 
 <li>数据编辑器： 
  <ul> 
   <li>过滤器面板中的自动补全功能得到了修复</li> 
   <li>修复了 XML 的自动格式化</li> 
   <li>修复了用别名过滤列的问题</li> 
   <li>修复了数字类型转换器</li> 
   <li>增加了额外的标签关闭动作</li> 
   <li>修复了记录视图中的数组显示</li> 
  </ul> </li> 
 <li>在项目资源管理器中增加了对文件拖放的支持</li> 
 <li>SQL 编辑器：新增 "hippie" 自动补全支持</li> 
 <li>SSHJ 实现现在默认包含在 DBeaver 社区版中</li> 
 <li>修复了 DBeaver 崩溃的问题</li> 
 <li>驱动管理： 
  <ul> 
   <li>增加了新的 Java11 模块以支持新的 JDBC 驱动</li> 
   <li>驱动程序复制问题得到修复</li> 
   <li>修复了 Maven 构件编辑对话框</li> 
  </ul> </li> 
 <li>Data transfer:</li> 
 <li>数据传输： 
  <ul> 
   <li>CSV 解析器性能显著提高</li> 
   <li>在 SQL 导出格式中增加了标识符大小写配置</li> 
   <li>增加了将结果附加到现有文件的选项</li> 
   <li>修复了 JSON 格式的日期格式化问题</li> 
   <li>修复了保存任务中的变量支持</li> 
   <li>增加了对从 CSV 导入数组的支持</li> 
   <li>修正了导出向导按钮启用的问题</li> 
  </ul> </li> 
 <li>会话管理器：增加了列分组功能</li> 
 <li>支持命令行：现在为打开的 SQL 脚本设置默认连接</li> 
 <li>添加 SalesForce 驱动</li> 
 <li>AWS Athena：更新了驱动版本，解决了列过滤器的问题</li> 
 <li>……</li> 
</ul>
