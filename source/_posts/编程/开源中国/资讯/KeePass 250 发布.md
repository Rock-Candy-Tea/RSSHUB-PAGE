                                                                                            <p>KeePass 是一个免费开源的密码管理器，它帮助你以安全的方式管理密码。你可以把你所有的密码放在一个数据库中，用一个主密码或一个密钥文件锁定。因此，你只需要记住一个主密码或选择密钥文件来解锁整个数据库。数据库使用非常安全的加密算法（AES/Rijndael, Twofish）进行加密。</p> 
<p>KeePass 2.50 正式发布，这是一个稳定的版本，该版本更新内容如下：</p> 
<h3><strong>新功能</strong></h3> 
<ul> 
 <li>在大多数 Windows 系统上，AES-KDF 现在的速度是以前的两倍</li> 
 <li>在大多数 Linux 系统上，如果安装了 "libgcrypt" 库，AES-KDF 的速度现在大约是以前的四倍</li> 
 <li>在大多数 Windows 系统上，Argon2d 和 Argon2id 现在的速度大约是以前的两倍</li> 
 <li>在大多数 Linux 系统上，如果安装了 "libargon2" 库，Argon2d 和 Argon2id 现在的速度是以前的三倍</li> 
 <li>当在 .NET 4.7.2 或更高版本上运行时，现在 GZip 解压速度更快</li> 
 <li>“在安全桌面上输入主密钥" 选项现在也被导入、确认和触发动作中显示的主密钥提示对话框所支持</li> 
 <li>增加了对用 Pale Moon、Epiphany 和 Midori 在隐私模式下打开 URL 的支持</li> 
 <li>增强了对类 Unix 系统的应用检测</li> 
 <li>增加了对类 Unix 系统上 Chromium 最新版本的检测支持</li> 
 <li>在 "URL" 菜单中，如果 Chrome 和 Chromium 都已安装，现在有单独的命令</li> 
 <li>增强了对检测 Vivaldi、Brave、Pale Moon 和 Epiphany 的支持</li> 
 <li>增加了对导入卡巴斯基密码管理器 9.0.2 TXT 文件的支持</li> 
 <li>Bitwarden 导入模块：增加了对导入子文件夹的支持，集合名称现在可以作为标签导入</li> 
 <li>……</li> 
</ul> 
<h3>改进</h3> 
<ul> 
 <li>内置的密码生成模式 <code>Hex Key - *-Bit</code> 现在使用大写的十六进制符号</li> 
 <li>改进了主密钥创建/更改对话框中的密钥文件存在性检查</li> 
 <li>改进了主密钥的构建</li> 
 <li>改进了某些对话框与可以取消关闭对话框的插件的兼容性</li> 
 <li>在导入/导出对话框中，导入/导出模块的图标现在与文件/程序类型匹配</li> 
 <li>DataVault CSV 导入模块：改进了笔记的导入</li> 
 <li>改进了本地缓冲区的处理</li> 
 <li>当打开/复制一个条目的URL时，该条目的最后访问时间将被更新</li> 
 <li>各种 UI 文本改进</li> 
 <li>代码优化</li> 
 <li>……</li> 
</ul>
