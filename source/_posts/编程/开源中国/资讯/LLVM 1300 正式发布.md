                                                        <p><span style="background-color:#ffffff; color:#333333">LLVM 13.0.0 现已发布。本次更新内容主要包括 bug 修复和一些小型功能优化。</span></p> 
<p><strong><span style="background-color:#ffffff; color:#333333">主要更新内容</span></strong></p> 
<ul> 
 <li><span style="background-color:#ffffff; color:#333333">Flang 作为 Fortran 前端现在包含在官方 LLVM 二进制包中</span></li> 
 <li><span style="background-color:#ffffff; color:#333333">LLDB 可执行文件现在包含在官方预构建的 LLVM 二进制文件包中</span></li> 
 <li><span style="background-color:#ffffff; color:#333333">对 Armv9-A 领域管理扩展 (RME) 和可扩展矩阵扩展 (SME) 的初始组装支持</span></li> 
 <li><span style="background-color:#ffffff; color:#333333">在 Clang 中对 OpenCL 内核语言支持进行了许多改进，包括如果未指定其他版本，则默认使用 OpenCL C 版本 1.2。 “.clcpp”文件扩展名现在也支持用于 OpenCL 文件的 C++。还支持许多新的 OpenCL 扩展以及 OpenCL C 3.0</span></li> 
 <li><span style="background-color:#ffffff; color:#333333">Clang 现在支持来自 OpenMP 5.1 的循环转换指令</span></li> 
 <li><span style="background-color:#ffffff; color:#333333">改进 clang-format 工具</span></li> 
 <li><span style="background-color:#ffffff; color:#333333">改进 Clang 静态分析器</span></li> 
 <li><span style="background-color:#ffffff; color:#333333">LLVM-MCA 机器分析器现在支持有序处理器</span></li> 
 <li><span style="background-color:#ffffff; color:#333333">LLDB 现在支持 AArch64 SVE 寄存器访问、AArch64 指针身份验证，并支持使用 MTE 进行调试</span></li> 
 <li><span style="background-color:#ffffff; color:#333333">Libcxx 现在在使用 MinGW 的 Windows 上包含 std::filesystem 功能完整支持，并具有可用的 C++20 概念库支持和其他实现的 C++20 功能</span></li> 
 <li><span style="background-color:#ffffff; color:#333333">AMDGPU 后端支持 GFX1013 RDNA2 APU</span></li> 
 <li><span style="background-color:#ffffff; color:#333333">添加了 AMD Zen 3 调度程序模型</span></li> 
</ul>
