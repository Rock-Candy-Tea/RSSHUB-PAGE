                                                        <p>restfx 0.16.6 已经发布，适用于 Python 3 的 restful 多应用自动路由框架</p> 
<p>此版本更新内容包括：</p> 
<ul> 
 <li>一些优化</li> 
 <li>fix #I3SBXR</li> 
 <li>fix #I3QGI9 响应头中的编码使用 utf-8 代替 utf8 ，以解决 IE 不识别的问题</li> 
 <li>修复 异常处理中的错误导致异常消息异常的问题</li> 
 <li>修复 参数为 list 时接收到的数据类型不正确的问题</li> 
</ul> 
<p>详情查看：<a href="https://gitee.com/hyjiacan/restfx/releases/0.16.6">https://gitee.com/hyjiacan/restfx/releases/0.16.6</a></p>
