                                                                                            <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">GNOME 发布了 GNOME 40.7 版本，作为 GNOME 40 桌面环境系列的第七次维护更新，其中包含更多改进和错误修复。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">GNOME 40.7 将 Gnome shell 升级到 40.8 版本，改进了 GNOME Shell 中的窗口跟踪，调整最小化和取消最小化动画，简化滚动渐变着色器以便在旧硬件上工作，改进了日历小程序中全天和零长度事件的处理，并改进放大镜以避免离屏渲染（offscreen rendering）。此外，GNOME Shell 进行了改进，在<em>“请勿打扰”</em>模式下不再唤醒屏幕，并在删除消息后保持键盘的焦点在通知列表中。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">另一方面，Mutter 窗口和合成管理器更新到 40.8 版本，获得了更多修复，针对未更新的未重定向 Xwayland 窗口、映射为全屏的窗口的非全屏显示，以及默认插件中的工作区切换动画。此外，窗口组小程序和损坏处理也得到了修复。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">GNOME 40.7 版本中也更新了几个默认的 GNOME Shell 扩展。其中包括 Window List，它获得了更好的屏幕键盘支持，以及 Native Window Placement，它修复了应用网格中的扭曲布局。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">除此之外，Simple Scan 应用程序已更新至 40.7 版本，该版本在创建新文档后会删除自动保存的记录，在有多个相同型号时将设备名称添加到标签中，自动保存 Page Side 属性，并正确用空格替换扫描仪名称中的下划线。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">完整的应用程序版本更新：</p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>cantarell-fonts (0.301 => 0.303.1)</li> 
 <li>epiphany (40.3 => 40.6)</li> 
 <li>gnome-desktop (40.6 => 40.7)</li> 
 <li>gnome-session (40.1.1 => 40.8)</li> 
 <li>gnome-shell (40.6 => 40.8)</li> 
 <li>gnome-shell-extensions (40.4 => 40.5)</li> 
 <li>gtk+ (3.24.30 => 3.24.31)</li> 
 <li>libgsf (1.14.47 => 1.14.48)</li> 
 <li>mutter (40.6 => 40.8)</li> 
 <li>orca (40.0 => 40.1)</li> 
 <li>pango (1.48.10 => 1.48.11)</li> 
 <li>pyatspi (2.38.1 => 2.38.2)</li> 
 <li>rygel (0.40.2 => 0.40.3)</li> 
 <li>simple-scan (40.6 => 40.7)</li> 
 <li>vala (0.52.8 => 0.52.10)</li> 
</ul> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">当然，GNOME 40.7 还包括各种清理、错误、崩溃和泄漏修复、Wayland 改进、对非系统会话的更好支持，以及更新的翻译。</p>
