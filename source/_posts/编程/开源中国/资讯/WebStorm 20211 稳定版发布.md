<div class="content">
                                                                                            <p>WebStorm 2021.1 稳定版<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fblog.jetbrains.com%2Fwebstorm%2F2021%2F04%2Fwebstorm-2021-1%2F" target="_blank">已发布</a>，这是今年首个大更新版本，主要变化包括：更智能的 JavaScript 和 TypeScript 代码补全、改进对 Stylelint 的支持、内置 HTML 预览、编辑器字体粗细选择的新设置，以及 Pull Request 相关操作的增强。</p> 
<p><img height="350" src="https://static.oschina.net/uploads/space/2021/0408/142055_odXE_2720166.png" width="700" referrerpolicy="no-referrer"></p> 
<h2>更智能的代码补全</h2> 
<p>JavaScript 和 TypeScript 文件中的代码补全建议现在采用了机器学习算法进行排序。</p> 
<p><img height="350" src="https://static.oschina.net/uploads/space/2021/0408/151620_6lF6_2720166.png" width="700" referrerpolicy="no-referrer"></p> 
<h2>更好的符号名称补全建议</h2> 
<p>WebStorm 现在会在显示顶层变量以及函数和类名的补全建议时考虑文件名。</p> 
<p><img height="200" src="https://static.oschina.net/uploads/space/2021/0408/151957_xrxY_2720166.png" width="700" referrerpolicy="no-referrer"></p> 
<h2>内置远程协作开发服务</h2> 
<p>WebStorm 已整合 Code With Me，这是用于远程协作开发和结对编程的新服务。开发者可以使用它进行实时合作编码，并用 IDE 与其他人交流。</p> 
<p><img height="368" src="https://static.oschina.net/uploads/space/2021/0408/163512_0Cc5_2720166.png" width="700" referrerpolicy="no-referrer"></p> 
<h2>支持 Vue 中的 CSS 模块</h2> 
<p>从 v2021.1 开始，WebStorm 将支持 <em>.</em>vue 文件中使用的 CSS 模块。IDE 可正确解析 $style 属性，为其提供代码补全、导航和基本的重构功能。</p> 
<p><img height="350" src="https://static.oschina.net/uploads/space/2021/0408/164050_ue6Y_2720166.png" width="700" referrerpolicy="no-referrer"></p> 
<h2>改进对 Stylelint 的支持</h2> 
<p>将鼠标悬停在文件中的问题上或将文本光标放在其上，然后按 Alt+Enter，选择 <em>Stylelint: Fix current file</em>，即可使用 Stylelint 修复 CSS 代码。此外，开发者现在可以在 <em>Preferences / Settings | Languages & Frameworks | Style Sheets | Stylelint</em> 的相应字段下指定配置文件的路径。</p> 
<p><img height="313" src="https://static.oschina.net/uploads/space/2021/0408/165203_ZXfo_2720166.png" width="700" referrerpolicy="no-referrer"></p> 
<h2>内置 HTML 预览</h2> 
<p>支持直接在 WebStorm 中预览静态 HTML 文件。开发者对 HTML 文件或链接的 CSS 和 JavaScript 文件所做的变更将被保存，并自动重新加载预览，以便开发者查看变更。</p> 
<p><img height="350" src="https://static.oschina.net/uploads/space/2021/0408/172216_2NoJ_2720166.gif" width="700" referrerpolicy="no-referrer"></p> 
<h2>支持调节字体粗细</h2> 
<p>开发者可选择不同的编辑器字体，包括浅色、常规和粗体，在 <em>Typography Settings in Preferences / Settings | Editor | Fonts</em> 下调整这些新选项。</p> 
<p><img height="450" src="https://static.oschina.net/uploads/space/2021/0408/172459_A3we_2720166.gif" width="700" referrerpolicy="no-referrer"></p>
