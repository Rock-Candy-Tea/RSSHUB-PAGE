                                                        <p>KDE Frameworks 5.84.0 已正式发布。KDE Frameworks 是 Qt 的 83 个附加库，它在成熟的、经过同行评审和测试的库中提供了各种常见的功能，并且具有友好的许可条款。</p> 
<p><strong>主要更新内容</strong></p> 
<ul> 
 <li>Attica 
  <ul> 
   <li>在 yaml 文件中说明库的许可证</li> 
   <li>在 Metadata 中添加 http 头信息（并从 BaseJob 中设置）</li> 
  </ul> </li> 
 <li>Breeze Icons 
  <ul> 
   <li>贴纸的图标</li> 
   <li>添加网络图标</li> 
   <li>添加 kdenlive 中高级修剪工具的图标</li> 
   <li>添加 Qt 资源母题图标</li> 
  </ul> </li> 
 <li>Extra CMake Modules 
  <ul> 
   <li>使预提交钩子可执行</li> 
   <li>[fastlane] 添加塞尔维亚语的映射</li> 
   <li>从 5.84 开始设置 C++17 标准</li> 
   <li>将编译 C++ 标准提升至17</li> 
   <li>在 tar PaxHeaders 中重写 atime 和 ctime</li> 
   <li>为 fish shell 添加 prefix.sh.fish 文件</li> 
  </ul> </li> 
 <li>KDE Doxygen Tools 
  <ul> 
   <li>改变前台产品的排序，使其总是包括框架</li> 
   <li>在标题中添加到 hig 的链接，以便与 develop.kde.org 的标题一致</li> 
  </ul> </li> 
 <li>KCalendarCore 
  <ul> 
   <li>分配和比较更多的 Incidence::Private 成员</li> 
   <li>在对 to-dos 进行排序时，将未定义的 dtDues 视为比定义的 dtDues 晚</li> 
   <li>生成 pkgconfig 文件</li> 
   <li>为非周期性事件返回开始日期</li> 
  </ul> </li> 
 <li>KCMUtils 
  <ul> 
   <li>使 KPluginMetaData 可以从 KCModuleProxy 类中访问</li> 
   <li>允许使用 KPluginMetaData 加载 KCModules</li> 
  </ul> </li> 
 <li>KCodecs 
  <ul> 
   <li>增加 Base45 解码</li> 
  </ul> </li> 
 <li>KConfigWidgets 
  <ul> 
   <li>暴露 KColorSchemeManager 的模型</li> 
   <li>更新与 KXMLGuiFactor::showConfigureShortcutsDilaog() 有关的 API 文档。</li> 
   <li>废弃 KStatefulBrush::brush(const QWidget*)</li> 
   <li>修复带有圆角的微小黑色矩形</li> 
   <li>修复 KCommandBar 中的过滤功能</li> 
   <li>KRecentFilesAction：记录临时文件夹中的本地文件 URL 被忽略</li> 
  </ul> </li> 
</ul> 
<p>更多详细内容，请查看<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fkde.org%2Fannouncements%2Fframeworks%2F5%2F5.84.0%2F" target="_blank">更新公告</a>。</p>
