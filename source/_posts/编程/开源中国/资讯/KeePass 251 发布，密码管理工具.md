                                                                                            <p>KeePass 是一个免费开源的密码管理器，它帮助你以安全的方式管理密码。你可以把你所有的密码放在一个数据库中，用一个主密码或一个密钥文件锁定。因此，你只需要记住一个主密码或选择密钥文件来解锁整个数据库。数据库使用非常安全的加密算法（AES/Rijndael, Twofish）进行加密。</p> 
<p>KeePass 2.51 正式发布，这是一个稳定的版本，该版本更新内容如下：</p> 
<h3>新功能</h3> 
<ul> 
 <li>大多数具有固定尺寸的对话框现在可以检测它们是否适合当前屏幕，当一个对话框不适合时，它的尺寸会缩小，并显示滚动条。</li> 
 <li>在主窗口中增加了复数条目命令名称（例如，编辑当前选定的条目/条目的命令现在被称为 'Edit Entry' 或 'Edit Entries'，这取决于选定条目的数量）。</li> 
 <li>为主窗口的状态栏的主要部分增加了工具提示。</li> 
 <li>增强了条目对话框、数据库设置对话框和选项对话框中的颜色按钮</li> 
 <li>在选项对话框中增加了 "Interface (2)" 标签页。</li> 
 <li>增强了选项对话框中的字体选择控制</li> 
 <li>在选项对话框中增加了 "黑暗主题" 和 "主要字体" 帮助链接。</li> 
 <li>增加了对 Waterfox 在隐私模式下打开 URL 的支持。</li> 
 <li>在 HTML 导出/打印对话框中增加了 "密码字符着色" 选项，颜色可以自定义。</li> 
 <li>在 HTML 导出/打印对话框中增加了 "自定义主字体" 和 "自定义密码字体" 选项。</li> 
 <li>在表格式的 HTML 导出/打印中增加了水平条目分隔线。</li> 
 <li>插件现在可以更容易地改变条目的到期日期。</li> 
 <li>增加了 Wine bug 52457 的解决方法。</li> 
 <li>……</li> 
</ul> 
<h3>改进</h3> 
<ul> 
 <li>改进了主窗口的初始化性能。</li> 
 <li>改进了最小化或最大化主窗口的初始出现（减少闪烁，提高性能，...）。</li> 
 <li>改进了主窗口中的数据库工具栏按钮的名称/工具提示。</li> 
 <li>改进了对粗体/大写列表字体的处理。</li> 
 <li>改进了某些情况下条目列表的更新性能。</li> 
 <li>改进了动态菜单的解构性能。</li> 
 <li>以 <code>HmacOtp-</code> 或 <code>TimeOtp-</code> 开头的字段不再显示在条目串复制菜单中。</li> 
 <li>改进了工具提示和密码重复文本框的可访问性。</li> 
 <li>当深色主题被激活时，文本框的错误背景颜色现在更深。</li> 
 <li>主密钥创建/更改对话框的标题现在根据上下文调整。</li> 
 <li>增加了 Authenticode 证书密钥长度。</li> 
 <li>升级了安装程序。</li> 
 <li>各种 UI 文本的改进。</li> 
 <li>各种代码优化。</li> 
 <li>……</li> 
</ul>
