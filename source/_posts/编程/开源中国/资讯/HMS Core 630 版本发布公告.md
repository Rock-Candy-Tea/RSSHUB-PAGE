 <li> <p>新增曲线变速功能，可自定义调整视频播放速度，实现非线性变速；</p> </li> 
 <li> <p>新增裁剪功能，支持修改图片、视频尺寸。</p> </li>

 <li> <p>新增单麦降噪功能，满足用户个性化使用需求；</p> </li> 
 <li> <p>新增原子能力SDK的集成方式，可自定义UI，开发者能根据App需求自定义使用界面。</p> </li> 
</ul>

 <li> <p>提供将拍摄图像转换为PBR材质的贴图功能，目前可以支持diffuse map、normal map、specular map、roughness map、height map五种贴图；</p> </li> 
 <li> <p>更新了种类丰富的材质库，包括混凝土、砖、石膏、黏土、金属、木材等16个种类的常见材质，支持查询、预览和下载等功能。</p> </li> 
</ul>

 <li>新增Maker旋转功能。开发者可根据App应用场景需求，顺时针或逆时针旋转Maker角度，比如打车时展示汽车头朝向，机场大屏展示飞机机头朝向。</li> 
</ul>

<ul> 
 <li> <p>提供独立的IP定位接口，支持通过用户IP获取其所属国家、省份、城市以及城市中心的经纬度信息；</p> </li> 
 <li> <p>新增离线GNSS定位功能，支持离线地图和离线导航，实现无网络环境下的卫星定位。</p> </li> 
</ul> 
<p><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdeveloper.huawei.com%2Fconsumer%2Fcn%2Fdoc%2Fdevelopment%2FHMSCore-Guides%2Fintroduction-0000001050706106%3Fha_source%3Dhms1" target="_blank">查看详情>></a></p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-1f4d367b8a1180e7a2a45c9777985089803.png" referrerpolicy="no-referrer"></p> 
<ul> 
 <li> <p>新增JNI模块，可支持渲染流程控制，并提供Java接口；</p> </li> 
 <li> <p>新增延迟渲染管线特性，实现从默认的前向渲染切换到延迟渲染；</p> </li> 
 <li> <p>新增阴影系统特性，支持方向光的阴影功能；</p> </li> 
 <li> <p>新增异步加载特性，实现资源加载和渲染流程隔离，资源的动态异步加载可以消除耗时的资源加载对渲染流程的影响；</p> </li> 
 <li> <p>新增自定义网格特性，可将3D世界中任何需要绘制的面通过三角形绘制出来，实现形状的动态显示效果；</p> </li> 
 <li> <p>新增骨骼动画和帧动画特性，实现骨骼动画和帧动画的解析功能；</p> </li> 
 <li> <p>新增动态材质特性，实现材质的颜色和效果动态变化；</p> </li> 
 <li> <p>新增关键帧动画特性，实现渲染绘制过程中的动画效果；</p> </li> 
 <li> <p>新增定时器、自定义渲染顺序、MSAA特性，帮助您在开发过程中更轻松实现更高级的功能；</p> </li> 
 <li> <p>新增V2版本cgmat配置文件，可支持同一个cgmat中配置多个材质实现方法和多个渲染过程，实现物体描边等功能。</p> </li> 
</ul>
