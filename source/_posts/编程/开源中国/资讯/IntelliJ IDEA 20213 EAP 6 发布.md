                                                                                            <p><span style="background-color:#ffffff; color:#333333">IntelliJ IDEA 2021.3 EAP 6 现已发布。此版本包含了增强的 HTTP client、对 Qute 模板语言的支持、</span><span style="background-color:#ffffff; color:#27282c">对 CDI 的 Kotlin 支持</span><span style="background-color:#ffffff; color:#333333">等内容。</span></p> 
<p><span style="background-color:#ffffff; color:#333333">一些亮点更新内容如下：</span></p> 
<h4><strong><span style="background-color:#ffffff; color:#333333">HTTP client 增强</span></strong></h4> 
<p>新版本显着改进了 IntelliJ IDEA HTTP <span style="background-color:#ffffff; color:#333333">client</span>。添加了对 gRPC requests 的支持、对自定义文件或目录的输出重定向、对文本和 JSON streams 的支持、对 HTTP 请求标识符的支持，以及对二进制响应的支持。</p>

<h4><strong><span><span><span><span><span><span style="color:#27282c"><span><span><span><span><span><span><span><span><span><span><span style="background-color:#ffffff"><span><span><span>Qute 模板语言支持</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></strong></h4> 
<p>IntelliJ IDEA 2021.3 将 Qute 添加到支持的模板语言列表中。IDE 将提供代码高亮显示、自动完成、检查、上下文操作和其他有用的功能，以帮助用户在处理 Quarkus 项目时开发模板。</p> 
<p><img alt height="200" src="https://oscimg.oschina.net/oscnet/up-208fca3d562b3b8faec7304a39b5625388d.png" width="500" referrerpolicy="no-referrer"></p> 
<h4><strong><span><span><span><span><span><span style="color:#27282c"><span><span><span><span><span><span><span><span><span><span><span style="background-color:#ffffff"><span><span><span>Kotlin 对 Java EE CDI 的支持</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></strong></h4> 
<p style="margin-left:0; margin-right:0; text-align:start"><span><span><span><span style="color:#27282c"><span><span><span><span><span><span><span><span><span><span><span><span style="background-color:#ffffff"><span><span><span>从 IntelliJ IDEA 2021.3 开始，以前仅适用于 Java 文件的所有 CDI 检查也适用于 Kotlin 文件。</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span><span><span><span><span style="color:#27282c"><span><span><span><span><span><span><span><span><span><span><span><span style="background-color:#ffffff"><span><span><span>此外，CDI gutter icons 现在可在 Kotlin 文件中使用。</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></p> 
<p style="margin-left:0; margin-right:0; text-align:start"><span><span><span><span style="color:#27282c"><span><span><span><span><span><span><span><span><span><span><span><span style="background-color:#ffffff"><span><span><span><img alt height="200" src="https://oscimg.oschina.net/oscnet/up-cc5cfecbce581a5b7aac131393a4e4aa17d.png" width="500" referrerpolicy="no-referrer"></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></p>

<h4><strong><span><span><span><span><span><span style="color:#27282c"><span><span><span><span><span><span><span><span><span><span><span style="background-color:#ffffff"><span><span><span>改进的 UML 类图 </span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></strong></h4> 
<p style="margin-left:0; margin-right:0; text-align:start"><span><span><span><span style="color:#27282c"><span><span><span><span><span><span><span><span><span><span><span><span style="background-color:#ffffff"><span><span><span>显着改善了 Java 和 Kotlin 的 UML 类图的用户体验。首先改进了 UML 类图的性能，以便提升图构建、类别切换、缩放和撤消/重做功能。其他显着变化包括新的 fancy grid、member highlighting 和快速文档预览弹窗。</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></p>
