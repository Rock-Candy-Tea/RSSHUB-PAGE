                                                                                            <p>Apache Tomcat 9.0.52 已<a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fmail-archives.apache.org%2Fmod_mbox%2Fwww-announce%2F202108.mbox%2F%253CCANwj8ZqefodBSHA4y04gRYZxuKEvUZ_Ws-UHDA7yF%3DpVO6x7dg%40mail.gmail.com%253E" target="_blank">正式发布</a>，Apache Tomcat 9 是一个开源软件，它实现了 Java Servlet、JavaServer Pages、Java Unified Expression Language、Java WebSocket 和 JASPIC 技术。本次更新是一个错误修复版本。</p> 
<p>值得关注的变化：</p> 
<ul> 
 <li>修复前一版本中 HTTP/2 流量控制窗口管理的回归问题</li> 
 <li>修复在使用 NIO 时可能导致某些 TLS 连接挂起的回归</li> 
 <li>使用 GraalVM 本地镜像不再自动禁用 JMX 支持</li> 
</ul> 
<p>详情查看 <a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Ftomcat.apache.org%2Ftomcat-9.0-doc%2Fchangelog.html" target="_blank">Changelog</a>。</p>
