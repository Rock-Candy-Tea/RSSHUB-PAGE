                                                                                            <p style="margin-left:0px">uBlock 是一款支持多浏览器的高效过滤工具，快速、有效且简洁，它不仅过滤广告，屏蔽广告的功能是通过支持 Adblock Plus 过滤规则语法实现的，因此它支持自定义过滤规则。</p> 
<p style="margin-left:0px">uBlock 1.44.2 正式发布，此次更新内容如下：</p> 
<ul> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fgorhill%2FuBlock%2Fcommit%2F14d4340edd375b861129ec3519799fcf9a8ae1f5" target="_blank">在有明确的说明时，强制进行程序外观过滤</a></li> 
</ul> 
<p style="margin-left:0px">使用 #?# 代替 ## ，防止 uBlock 在某些情况下使用过滤器的声明形式。</p> 
<ul> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fgorhill%2FuBlock%2Fcommit%2F8f39a457b869c6a3bc32e2190e7a11a33a74695f" target="_blank">当参数不是有效的纯 CSS 选择器时，回退到程序 :has() 运算符</a></li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fgorhill%2FuBlock%2Fcommit%2F04114942aaaa371683930bbf799e439e33f8aaea" target="_blank">向 shim 脚本添加缺少的方法</a></li> 
</ul> 
<p><strong>安装稳定版本：</strong></p> 
<ul> 
 <li><strong>Firefox（</strong>在 Firefox 上效果最好<strong>）</strong>：点击 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Faddons.mozilla.org%2Ffirefox%2Fdownloads%2Ffile%2F3998742%2Fublock_origin-1.44.2.xpi" target="_blank">uBlock0_1.44.2.firefox.signed.xpi</a></li> 
 <li><strong>Chromium</strong>：从 Chrome 商店安装</li> 
 <li><strong>Edge</strong>：从 Microsoft Store 安装（ Microsoft Store 版本由 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fnicole-ashley%2FuBlock-Edge" target="_blank">Nicole Rolls </a>发布）</li> 
 <li><strong>Opera</strong>：从 Opera 插件安装</li> 
</ul> 
<p>更新公告：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fgorhill%2FuBlock%2Freleases%2Ftag%2F1.44.2" target="_blank">https://github.com/gorhill/uBlock/releases/tag/1.44.2</a></p>
