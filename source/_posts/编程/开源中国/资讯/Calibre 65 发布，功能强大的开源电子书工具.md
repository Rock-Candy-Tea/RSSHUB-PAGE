                                                                                            <p>Calibre 开源项目是 Calibre 官方出的电子书管理工具。它可以查看，转换，编辑和分类所有主流格式的电子书。Calibre 是个跨平台软件，可以在 Linux、Windows 和 macOS 上运行。</p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">Calibre 6.5 正式发布，此次更新内容如下：</p> 
<p style="margin-left:0px; margin-right:0px; text-align:left"><strong>新功能</strong></p> 
<ul> 
 <li>Kobo driver：支持新的 Kobo Clara 2e</li> 
 <li>当使用 --with-library 命令行 flag 运行 calibre 并且 calibre 已经在运行时，切换到 calibre 中的指定库</li> 
 <li>E-book viewer：删除自定义高亮样式时要求确认</li> 
 <li>不存在 virtual libraries 时不显示 virtual libraries tag 栏</li> 
 <li>Tag mapper：指定替换 tag 时，允许从当前打开的库中的 tag 完成</li> 
 <li>Template language：新函数 strcasecmp() 和 to_hex()</li> 
</ul> 
<p style="margin-left:0px; margin-right:0px; text-align:start"><strong><span style="color:#39322b">Bug 修复</span></strong></p> 
<ul> 
 <li>Comments editor：Qt 在将段落转换为标题时插入无效 font-weight 的解决方法</li> 
 <li>E-book viewer：当章节标题相同时，显示 highlights 时不要将不同章节的 highlights 分组</li> 
 <li>Tag browser：如果用户类别包含对已删除自定义列的引用，则修复错误</li> 
 <li>通过电子邮件向 Kindle 和 PocketBook 电子邮件递送服务发送书籍时，使用由简单英文字母组成的随机主题和文件名，来尝试解决其电子邮件 processing code 中的各种错误</li> 
</ul>
