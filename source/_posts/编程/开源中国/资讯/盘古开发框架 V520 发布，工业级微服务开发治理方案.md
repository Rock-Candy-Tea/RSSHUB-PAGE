                                                                                            <p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><a href="https://pulanos.gitee.io/pangu-framework/" target="_blank">盘古开发框架</a> v5.2.0 已经发布，工业级微服务开发和治理框架</p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><strong>此版本更新内容包括：</strong></p> 
<ul> 
 <li>为配合 github pages 服务部署，修改了项目官网 react 编译构建输出路径。</li> 
 <li>移除 Apache ShenYu 依赖，将缺省网关开发模式调整为pangu-webapi模式。</li> 
 <li>封装 pangu-web-spring-boot-starter，提供 WebMVC 相关框架缺省内置拦截器的自动装配功能。</li> 
 <li>修正 pang-parent 模块的依赖库版本冲突导致 package 打包报错的问题。</li> 
 <li>集成 TLog 通过对日志打标签完成微服务应用的分布式日志追踪。</li> 
 <li>集成 Seata 提供了完善的分布式事务管理方案。编撰了详尽的文档说明和范例。</li> 
 <li>集成了分布式锁 Lock4j。支持注解声明式和 API 编程式对业务代码在分布式环境下加并发锁。</li> 
 <li>封装了 RabbitMQ ，提供了分布式系统在异步通信、服务解耦、消峰填谷、缓存同步等应用场景的使用。</li> 
</ul> 
<hr> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fpangu.pulanit.com%2F" target="_blank">盘古开发框架</a> 是完全独立于 Spring Cloud 生态的一套轻量灵活、成熟可靠的工业级分布式微服务开发和治理框架（兼容垂直单体分层架构)。它基于 Apache-2.0 协议开源发布，且是免费的。我们希望不仅是开源的受益者，也能成为开源的贡献者，与开源社区一起「共建共享开源生态」。</p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><strong>开箱即用、轻量灵活</strong><span> </span><br> 开箱即用的技术组件库、搭配自定义 SpringBoot 启动器和轻巧的代码生成 Maven 插件，可随意组合、自动装配、灵活插拔。 既能构建支持大并发高可用的互联网分布式架构也能快速搭建垂直单体分层架构。</p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><strong>成熟架构、稳定可靠</strong><span> </span><br> 盘古技术栈融合云原生思想，在微服务、容器化、DevOps 等方面完美支持和适配。可在云原生时代构建以「服务」为中心的现代应用架构，帮助用户在云环境中更好的构建、交付和管理自己的微服务平台。</p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left"><strong>全栈百搭适配各类业务场景</strong><span> </span><br> 聚焦面向（移动）互联网的电商、金融、云端 SAAS 服务等大并发高可用可扩展的业务场景。 同时也能满足诸如 ERP 、CRM 等各种企业数字化和各级电子政务系统的中小规模应用需求。<br> <img alt="输入图片说明" src="https://images.gitee.com/uploads/images/2022/0406/142521_40b5cd77_431745.gif" referrerpolicy="no-referrer"></p>
