                                                        <p>GNU 调试器 GDB 10.2 发布。GDB 是 Ada、C、C++、Go 与 Rust 等源代码级调试器语言，它可以在最流行的 GNU/Linux、Unix 和 Windows 变体上运行，并且调试十几种不同的处理器架构。</p> 
<p><strong>主要更新内容</strong></p> 
<ul> 
 <li>AddressSanitizer: remote_async_inferior_event_handler 中 extended_remote_target 的堆在使用后释放</li> 
 <li>修复内部错误: void target_mourn_inferior(ptid_t): assert `ptid == inferior_ptid' 失败。操作系统：Mac OSX Catalina；编译器：GCC；语言：C</li> 
 <li>修复 gdb 内部错误: 用 qemu 调试 linux 内核时，未知的 CFA 规则</li> 
 <li>修复数组下标由于没有大小的灵活数组成员而报错的问题</li> 
 <li>修复当不包括新布局中的状态窗口时，gdb 崩溃的问题。</li> 
 <li>修复静态成员参数的 Value.format_string 文档错误</li> 
 <li>将 rust_demangler 添加到 10.x 分支</li> 
 <li>修复无法使用 gcc-4.8 编译 arc.c的问题</li> 
 <li>修复局部变量在 sparc64 上具有错误地址和值的问题</li> 
</ul> 
<p>详细内容请查看<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.gnu.org%2Fsoftware%2Fgdb%2F" target="_blank">更新公告</a>。</p>
