                                                        <p>Matomo 4.6.0 发布了。Matomo 是一套基于 PHP + MySQL 技术构建的开源网站访问统计系统，能够提供详细的统计信息，比如网页浏览人数、访问最多的页面、搜索引擎关键词等等流量分析功能。</p> 
<p>此版本包含以下更新内容</p> 
<h3>新功能和 UI 改进</h3> 
<p><strong>1、现在可以用图表显示“未完成”和“完成”时期的数据差异，未完成时期现在用虚线表示。</strong></p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-93c00b2f1c6dff1644501c40bf36127c3ed.png" referrerpolicy="no-referrer"></p> 
<p><strong>2、当行演化从以下位置打开时，将显示对应的目标指标</strong></p> 
<ul> 
 <li>目标报告页面上的报告（具体或概述）</li> 
 <li>电子商务销售页面</li> 
 <li>其他目标可视化的报告</li> 
</ul> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-468975f3f11cd76b5536ca691ace9fdc9d3.png" referrerpolicy="no-referrer"></p> 
<p><strong>3、可以使用导出按钮（右下角）以 XML 或 JSON 格式导出转换报告。</strong></p> 
<p><img alt height="262" src="https://oscimg.oschina.net/oscnet/up-971e8e727e823b6e286728d1fbdf0faf3fc.png" width="700" referrerpolicy="no-referrer"></p> 
<p><strong>4、删除了“推荐我们”功能和评论弹出窗口，用更简单、不引人注目的横幅替换</strong></p> 
<p><img alt height="31" src="https://oscimg.oschina.net/oscnet/up-bd288c272421a3444ba38dba3b5fe28a09c.png" width="700" referrerpolicy="no-referrer"></p> 
<p><strong>5、Matomo 标签管理器更新</strong></p> 
<p><span style="background-color:#ffffff; color:#606060">以前使用跟踪代码管理调试器时，必须将参数附加到 URL，现在只需在表单中输入 URL 并单击调试即可。</span></p> 
<p><img alt height="141" src="https://oscimg.oschina.net/oscnet/up-cbf160d1c42bb85ba9c0279ffb4a49b2cbf.png" width="700" referrerpolicy="no-referrer"></p> 
<h3>支持和依赖变更</h3> 
<ul> 
 <li>在 Matomo 4.6 中放弃对 IE11 的支持（注意：跟踪代码仍然支持 IE11，这只影响 Matomo 用户界面）</li> 
 <li>随着 AngularJS 计划于今年年底结束，4.6 版本中添加了对 Vue.JS 的 支持，为将来从 AngularJS 过渡做准备</li> 
 <li>随着最近发布的新版本的 PHP，对 PHP 8.1 的支持也得到了改进。</li> 
</ul> 
<p>4.6.0 版本还包含一个新的 javascript 框架、新 API、破坏性 API 更改、新命令、新配置的设置，具体可在面向开发人员的<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdeveloper.matomo.org%2Fchangelog" target="_blank">变更日志</a>中查看</p> 
<p> </p>
