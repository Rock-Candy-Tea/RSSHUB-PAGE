                                                                                            <p>RetroArch 1.10.1 现已发布。RetroArch 是款功能强大的跨平台模拟器，不但能够模拟许多不同的游戏主机，还能在 Windows、MacOS、Linux、Android、iOS 以及多种游戏主机上执行。</p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><strong>更新内容如下：</strong></p> 
<ul> 
 <li>ANDROID：将<code>HAVE_LANGEXTRA</code>重新添加到 makefile 中</li> 
 <li>ANDROID：包括自适应启动器图标</li> 
 <li>ANDROID：在 Android 11+ 设备上的文件浏览器内填充外部存储设备</li> 
 <li>CHEEVOS：添加 mastery placard</li> 
 <li>CHEEVOS：为缺失的 RetroAchievements credentials 提供更多描述信息</li> 
 <li>CHEEVOS：如果无法检索成就数据，则停止加载过程</li> 
 <li>CHEEVOS：支持识别 Dreamcast CHD</li> 
 <li>CHEEVOS：更新到 10.3.2</li> 
 <li>CONTENTLESS CORES：启用 config overrides</li> 
 <li>CONTENTLESS CORES：启用 runtime logging</li> 
 <li>CORE INFORMATION：显示核心版本</li> 
 <li>CORE UPDATER：添加“Core System Files Downloader”</li> 
 <li>CORES/SETTINGS：添加“Standalone Cores”菜单</li> 
 <li>D3D10/D3D11：添加 Vsync swap interval</li> 
 <li>EMSCRIPTEN：启用 7zip</li> 
 <li>GBA：新的 GBA filter Upscale_240x160-320x240</li> 
 <li>GONG：已移除，因为它是一个独立的 core</li> 
 <li>HISTORY/FAVORITES：修复“Add to Favorites”的 default core</li> 
 <li>......</li> 
</ul>
