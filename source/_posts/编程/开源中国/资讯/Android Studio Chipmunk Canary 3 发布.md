                                                                                            <p>Android Studio Canary 3 现已发布。</p> 
<p><strong>主要更新内容</strong></p> 
<ul> 
 <li>AVD 管理器 
  <ul> 
   <li><span style="background-color:rgba(255, 255, 255, 0.95)"><span style="color:#202124">CLI 输出被解析日志污染</span></span></li> 
  </ul> </li> 
 <li><span style="background-color:rgba(255, 255, 255, 0.95)"><span style="color:#202124">Android Gradle 插件</span></span> 
  <ul> 
   <li><span style="background-color:rgba(255, 255, 255, 0.95)"><span style="color:#202124">AGP 7.0.0-alpha10 回归：databinding 处于启用状态时，无法应用插件 "com.android.internal.library"</span></span></li> 
   <li><span style="background-color:rgba(255, 255, 255, 0.95)"><span style="color:#202124">lintDebug 破坏配置缓存</span></span></li> 
   <li><span style="background-color:rgba(255, 255, 255, 0.95)"><span style="color:#202124">确保从 v2 开始，IDE 模型支持 "[consumer]proguardFiles"</span></span></li> 
  </ul> </li> 
 <li><span style="background-color:rgba(255, 255, 255, 0.95)"><span style="color:#202124">Android Studio</span></span> 
  <ul> 
   <li><span style="background-color:rgba(255, 255, 255, 0.95)"><span style="color:#202124">Jetpack Compose 1.0 模板更新</span></span></li> 
  </ul> </li> 
 <li><span style="background-color:rgba(255, 255, 255, 0.95)"><span style="color:#202124">Compose 编辑</span></span> 
  <ul> 
   <li><span style="background-color:rgba(255, 255, 255, 0.95)"><span style="color:#202124">​​​​​​​由于 Kotlin 语言版本较旧，无法构建 Studio</span></span></li> 
  </ul> </li> 
 <li><span style="background-color:rgba(255, 255, 255, 0.95)"><span style="color:#202124">Compose 预览</span></span> 
  <ul> 
   <li><span style="background-color:rgba(255, 255, 255, 0.95)"><span style="color:#202124">​​​​​​​当我拖动滚动视图后，互动式预览停止运行</span></span></li> 
  </ul> </li> 
 <li><span style="background-color:rgba(255, 255, 255, 0.95)"><span style="color:#202124">Databinding</span></span> 
  <ul> 
   <li><span style="background-color:rgba(255, 255, 255, 0.95)"><span style="color:#202124">​​​​​​​Databinding 7.0.0 使用已废弃的 lifecycle-extensions</span></span></li> 
  </ul> </li> 
 <li><span style="background-color:rgba(255, 255, 255, 0.95)"><span style="color:#202124">布局编辑器</span></span> 
  <ul> 
   <li><span style="background-color:rgba(255, 255, 255, 0.95)"><span style="color:#202124">​​​​​​​加载 Compose 预览字体时，会尝试启动线程（导致渲染错误）</span></span></li> 
  </ul> </li> 
</ul>
