                                                                                            <p>PeaZip 是一款优秀的压缩管理工具，可以运行在 Linux 系统和 Windows 系统。目前， PeaZip 7.8.0 已正式发布；该版本引入了交互式提取、改善了预览效果、改善了 Windows 上的拖放功能，并在 Linux 上提供了新的 Qt5 版本。</p> 
<p>具体更新内容如下：</p> 
<p><strong>BACKEND </strong></p> 
<ul> 
 <li>PEA 0.77</li> 
</ul> 
<p><strong>CODE </strong></p> 
<ul> 
 <li>在 Linux 上提供了新的 Qt5 版本</li> 
 <li>如果设置要求重启PeaZip（本地化、系统集成、更改工作文件夹），则在提示文本中告知该项目</li> 
 <li>各种修复和改进</li> 
</ul> 
<p><strong>FILE MANAGER </strong></p> 
<ul> 
 <li>(Windows)Drag&drop 现在允许直接拖拽文件到其他应用程序，以前只能拖拽到系统。</li> 
 <li>右键单击面包屑上的某个元素，现在可以在使用上下文菜单功能。例如显示属性，在系统的文件资源管理器中打开路径，在此处打开命令提示符等。</li> 
</ul> 
<p><strong>EXTRACTION and ARCHIVING </strong></p> 
<ul> 
 <li>交互式提取选项（在 Windows 中默认为勾选）可以为 7z/p7zip 支持的所有归档类型实现两步提取：内容被提取到一个新的文件夹，然后通过标准系统的调用移动到实际输出目标位置。</li> 
 <li>现在可以在 Options > Settings > Archive manager，Extract group 中为 Extract everything for 设置自定义扩展。</li> 
 <li>现在可以在归档界面的"Advanced"选项卡中用自定义扩展名替换归档扩展名。</li> 
 <li>现在可以通过 Options > Settings > General，Tasks priority 永久设置存档和提取任务的优先级。</li> 
 <li>（Windows 7+）全局进度显示在任务栏中的应用程序图标中</li> 
</ul> 
<p>详情可查看：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fpeazip.github.io%2Fchangelog.html" target="_blank">https://peazip.github.io/changelog.html</a></p>
