                                                                                            <p><strong>更新日志：</strong></p> 
<p>较上一版本，红旗 Linux 桌面操作系统 v11 alpha 版（国庆献礼版）主要更新如下：</p> 
<ol> 
 <li>新增献礼中华人民共和国成立 72 周年主题；</li> 
 <li>修复 rtl8192e，rtl8723b，cxgb4 等型号无线网卡无法驱动的问题；</li> 
 <li>新增针对 i915 主板芯片组部分集成显卡的支持；</li> 
 <li>新增针对 WCN6750，WCN3991，RTL8822C，RTL8723D 等蓝牙设备的支持；</li> 
 <li>新增针对 AMD 低功耗 CPU 的支持；</li> 
 <li>升级针对 AMD 显卡的 Navi14，Navi12，Navi10 核心相关设备的支持；</li> 
 <li>升级针对 AMD 的 vega20GPU，vega12GPU，vega10GPU 设备的支持；</li> 
 <li>更新固件版本增加部分硬件支持；</li> 
 <li>新增磁盘分区工具，方便用户进行磁盘管理；</li> 
 <li>修复部分已知问题；</li> 
</ol> 
<p><strong>更新方法：</strong></p> 
<ol> 
 <li>已安装红旗 Linux 桌面操作系统 v11 alpha 前序版本的用户，可参考下列方法进行系统更新。</li> 
 <li>未安装红旗 Linux 桌面操作系统 v11 alpha 版的用户，需按下方提供的下载链接，访问红旗 Linux 官方社区下载版块，下载安装体验。</li> 
</ol> 
<p><strong>方式 1. 软件商店自定义更新</strong></p> 
<p>点击桌面左下角 “R” 图标，点击“系统”-“软件商店”-“检查更新”，根据更新提示可自定义选取需要更新的软件包或更新全部升级的软件包。</p> 
<p><br> <img alt height="524" src="https://static.oschina.net/uploads/space/2021/0925/072756_TwZB_4937141.png" width="700" referrerpolicy="no-referrer"></p> 
<p><strong>方式 2. 终端命令更新</strong></p> 
<p>更新网络源</p> 
<pre><code>sudo apt update -y

</code></pre> 
<p>查看可更新软件</p> 
<pre><code>apt list --upgradable

</code></pre> 
<p>可根据需求全部或选择性的手动安装相应更新包。</p> 
<p><strong>下载链接：</strong></p> 
<p>访问红旗 Linux 官方社区下载版块（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.linuxsir.cn%2F%3Fdownload.htm" target="_blank">https://www.linuxsir.cn/?download.htm</a>），获取镜像下载链接。</p>
