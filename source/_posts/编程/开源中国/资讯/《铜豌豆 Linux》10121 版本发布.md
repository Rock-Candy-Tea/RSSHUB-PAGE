                                                        <p style="text-align:center"><strong>《铜豌豆 Linux》10.12.1 版本发布</strong></p> 
<p>https://www.atzlinux.com/News/2022/20220408.htm</p> 
<p>2022-03-26, Debian 官方发布 10.12 版本：<br> <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.debian.org%2FNews%2F2022%2F2022032602" target="_blank">https://www.debian.org/News/2022/2022032602 </a></p> 
<p>铜豌豆 Linux 跟进发布 10.12.1。<br> 此次发布主要为更新升级软件包、调整软件包。</p> 
<h3>软件包更新</h3> 
<h3>安全更新</h3> 
<ul> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.debian.org%2Fsecurity%2F" target="_blank">Debian 官方软件包安全更新</a> 同步到 2022-04-08 日。</li> 
</ul> 
<h3>iso 文件调整及优化</h3> 
<ul> 
 <li>完善对 BIOS 启动的老旧机器支持</li> 
 <li>可以在安装过程中，自动安装好 grub 启动管理器。对超过 10 年以上的老旧机器，都能够自动化完成安装。</li> 
 <li>默认安装使用 5.10 内核启动</li> 
 <li>内核版本号为：5.10.103-1~bpo10+1 (2022-03-08)<br> 4.19 版内核也继续安装，可以在机器启动菜单中选择使用。</li> 
</ul> 
<h3>主要软件包版本</h3> 
<ul> 
 <li>liblunar-calendar-gtk3-module 农历 3.0.1-2</li> 
 <li>可以在 MATE 等桌面环境的默认日历程序中，直接显示农历和中国节假日信息</li> 
 <li>wps 11.1.0.10920</li> 
 <li>钉钉 1.4.0.47</li> 
 <li>该软件包也可以在 Debian 11 上运行，吸收了铜豌豆提供的补丁。</li> 
 <li>google-chrome-stable 100.0.4896.75-1</li> 
 <li>软件包主版本号进入三位数时代</li> 
 <li>opengnb 1.2.8.5-1</li> 
 <li>方便快捷搭建 VPN 网络，内网穿透，国人自己开发，特别适合国内网络环境，速度极快。</li> 
</ul> 
<h3>主要软件包调整：</h3> 
<ul> 
 <li>铜豌豆补丁包</li> 
 <li>升级到 10.12.1 版，后续铜豌豆补丁包的版本号，和铜豌豆 iso 操作系统版本号，会保持一致。<br> /etc/atzlinux_version 文件, 版本号设置为 10.12.1</li> 
</ul> 
<h3>各版本软件包增加：</h3> 
<ul> 
 <li>纯文字版</li> 
 <li>strace 系统调用跟踪器<br> vim-scripts vim 插件<br> blhc<br> reprotest<br> apt-file 支持操作系统文件名路径全局检索<br> nvme-cli NVMe 硬盘命令行工具<br> edid-decode<br> opengnb<br> squashfuse squash 文件系统工具<br> links 纯文本浏览器<br> lunar-date 命令行农历<br> diffoscope 操作系统级别的 deb 包差异性对比<br> memtester 内存测试软件<br> libncurses-dev 编译内核需要用到的开发包<br> libssl-dev<br> libelf-dev</li> 
 <li>xfce 精简版</li> 
 <li>net-tools ifconfig 等命令行<br> network-manager-gnome 图形化配置网络</li> 
 <li>xfce 完整版</li> 
 <li>firmware-b43-atzlinux b43 博通无线网卡固件<br> fonts-lxgw-wenkai 霞鹜文楷字体，铅笔字形<br> gnome-sound-recorder 录音机<br> lunar-date<br> nvme-cli<br> opengnb<br> google-chrome-stable 浏览器</li> 
 <li>三合一版</li> 
 <li>firmware-b43-atzlinux b43 博通无线网卡固件<br> fonts-lxgw-wenkai 霞鹜文楷字体，铅笔字形<br> gnome-sound-recorder 录音机<br> lunar-date<br> nvme-cli<br> opengnb<br> google-chrome-stable 浏览器</li> 
 <li>KDE 版</li> 
 <li>firmware-b43-atzlinux b43 博通无线网卡固件<br> fonts-lxgw-wenkai 霞鹜文楷字体，铅笔字形<br> gnome-sound-recorder 录音机<br> lunar-date<br> nvme-cli<br> opengnb<br> google-chrome-stable 浏览器<br> plasma-discover KDE 软件中心<br> dists-buster-files-dep11-atzlinux dep11 apt 原数据包</li> 
</ul> 
<h3>软件包删除：</h3> 
<ul> 
 <li>纯文字版</li> 
 <li>无</li> 
 <li>xfce 精简版</li> 
 <li>无</li> 
 <li>xfce 完整版</li> 
 <li>无</li> 
 <li>三合一版</li> 
 <li>xorg-server-source 无需用到的源代码包</li> 
 <li>KDE 版</li> 
 <li>无</li> 
</ul>
