                                                                                            <p>Python 发布了 3.10.2 和 3.9.10 版本，3.10.2 作为包含所有功能的最新版本，3.9.10 则是 3.9 的第十个维护版本，带来一些改进和优化。</p> 
<h3>Python 3.10.2</h3> 
<p style="margin-left:0px">Python 3.10.2 是 Python 编程语言的最新主要版本，包含许多新功能和优化。</p> 
<p style="margin-left:0px"><strong>这是一个特殊的错误修复版本，用于解决在使用 </strong><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fcython%2Fcython" target="_blank"><strong>Cython </strong></a><strong>时某些函数调用上发生的内存泄漏</strong>，这些内存泄漏由来自 Cython 代码的某些函数调用中的少量恒定的字节组成。尽管在大多数情况下影响不是很明显，但它对长时间运行的应用程序和某些使用模式会带来非常大的影响，通过查阅 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fbugs.python.org%2Fissue46347" target="_blank">bpo-46347 </a>了解更多信息。</p> 
<p style="margin-left:0px">更新公告：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.python.org%2Fdownloads%2Frelease%2Fpython-3102%2F" target="_blank">https://www.python.org/downloads/release/python-3102/</a></p> 
<h3 style="margin-left:0px">Python 3.9.10</h3> 
<p style="margin-left:0px">Python 3.9.10 是旧版 3.9 系列的第九个维护版本。注意： Python 3.10 现在是 Python 3 的最新功能版本，建议升级到 Python 3.10.2。</p> 
<p style="margin-left:0px">此版本带来一些修复和性能优化，如：</p> 
<ul> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fbugs.python.org%2Fissue46085" target="_blank">bpo-46085</a>：修复 <code>OrderedDict</code></li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fbugs.python.org%2Fissue46110" target="_blank">bpo-46110</a>：向 PEG 解析器添加最大递归检查以避免堆栈溢</li> 
 <li><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fbugs.python.org%2Fissue46000" target="_blank">bpo-46000</a>：提高<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdocs.python.org%2Frelease%2F3.9.10%2Flibrary%2Fcurses.html%23module-curses" target="_blank"><code>curses</code></a>模块与 NetBSD 的兼容性</li> 
</ul> 
<p>完整的优化项可以<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdocs.python.org%2Frelease%2F3.9.10%2Fwhatsnew%2Fchangelog.html" target="_blank">在 Python 3.9.10 更新日志</a>中查看。</p>
