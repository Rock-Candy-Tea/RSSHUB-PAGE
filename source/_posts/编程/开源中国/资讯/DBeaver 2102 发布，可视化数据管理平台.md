                                                                                            <p>DBeaver 21.0.2 现已发布。DBeaver是一个免费开源的通用数据库工具，适用于开发人员和数据库管理员。</p> 
<p><strong>主要更新内容</strong></p> 
<ul> 
 <li>SQL Editor 
  <ul> 
   <li>添加了打开编辑器操作的配置</li> 
   <li>改进了列值的智能自动完成功能</li> 
   <li>解决了列名称解析不明确的问题</li> 
   <li>修复了超链接检测</li> 
   <li>SQL 模板视图冻结的问题已修复</li> 
  </ul> </li> 
 <li>Database navigator 
  <ul> 
   <li>修复了快速模式过滤器</li> 
   <li>元数据修改保存：单笔交易处理得到改善。</li> 
  </ul> </li> 
 <li>Data transfer 
  <ul> 
   <li>导出到平面文件现在支持列配置</li> 
   <li>BOM 支持已添加到 CSV 导入</li> 
  </ul> </li> 
 <li>Data viewer 
  <ul> 
   <li>BLOB 查看器现在遵守重单元格值的内容类型。</li> 
   <li>分组面板：现在正确处理带有尾部注释的查询</li> 
  </ul> </li> 
</ul>
