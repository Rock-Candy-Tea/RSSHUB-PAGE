 <h3>软件简介</h3> 
</div> 
<div style="text-align:start"> 
 <div style="text-align:left"> 
  <p style="text-align:start">Jweb-adai 是基于jweb-boot开发的一套易于开发与使用的极速开发后台管理系统。</p> 
  <p style="text-align:start"><img alt src="https://oscimg.oschina.net/oscnet/up-e0bf0de526c3c1b06ac8334eb059553acef.JPEG" width="1627" referrerpolicy="no-referrer"></p> 
  <h2 style="text-align:start">特点</h2> 
  <p style="text-align:start">系统提供常见的功能模块，减少重复功能模块开发，自动生成常见功能代码，结合官方模板中心，可下载更多功能模块模板。从而大大提高工作效率。</p> 
  <ol> 
   <li><strong>文件管理</strong>（代码生成）： 管理，查看，预览上传的文件资源</li> 
   <li><strong>数据字典</strong>（代码生成）：通用系统数据字典，为后面功能模块提供有限数据选择</li> 
   <li><strong>组织机构</strong>（代码生成）：管理系统机构信息与人员信息</li> 
   <li><strong>角色人员权限</strong>（代码生成）：管理系统角色，角色人员关系，角色权限关系等</li> 
   <li><strong>操作权限</strong>（代码生成）：管理系统操作权限，包含菜单、权限等类型</li> 
   <li><strong>系统日志</strong>（代码生成）：显示系统操作日志</li> 
   <li><strong>代码生成</strong>（代码生成）： 提供根据模板快速生成前后端代码，提供编译部署测试功能 
    <ul> 
     <li><strong>模板中心</strong>（代码生成）：提供官方模板与自定义模板管理</li> 
     <li><strong>表模型管理</strong>（代码生成）：管理代码生成器数据库表模型数据与关系配置</li> 
     <li><strong>代码生成管理</strong>（代码生成）：根据配置实现代码生成、编译、部署、测试</li> 
    </ul> </li> 
  </ol> 
  <p><strong>更新日志：</strong></p> 
  <p><strong>v1.0.6 </strong></p> 
  <ol> 
   <li>添加模板文件可被渲染的文件后缀类型配置</li> 
   <li>添加模板包创建表_create_tables.sql文件支持</li> 
   <li>添加代码生成器模板assets目录与_config.json文件解析部署支持</li> 
   <li>改进jwebSecurity安全框架拦截逻辑，支持jfinal中urlPara的访问控制</li> 
   <li>改进路径为/,避免linux不识别window导入的模板路径数据</li> 
   <li>改进模板导入版本判断逻辑</li> 
   <li>解决 java -cp 执行时判断isRunInJar失败的问题。</li> 
   <li>解决数据字典列表查询dic_pcodef过滤不正确的问题</li> 
   <li>解决注销在无session时报错异常</li> 
   <li>解决xce组件无法预览svg文件的问题。优化图片背景</li> 
   <li>解决工程中引用jweb-adai库时，代码编译找到jweb-adai相关类的问题</li> 
   <li>解决数据字典列表总数统计错误问题</li>
