                                                                                            <p>OBS Studio 是一款免费且开源的用于视频录制以及直播串流的软件，提供实时高性能的视频 / 音频捕捉与混合，以及无限的场景模式帮助用户通过自定义实现无缝转换。</p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">OBS Studio 28.0.1 现已发布，该版本更新内容包括：</p> 
<ul> 
 <li>修复了导致 MP4/MOV 文件有时以损坏状态保存的错误</li> 
 <li>修复了导致 DeckLink 设备源有时无法捕获的错误</li> 
 <li>修复 Windows 自动更新程序仅在 Windows 10 及更高版本上更新的问题</li> 
 <li>恢复了导致视频捕获设备设置在使用时被重置的更改 
  <ul> 
   <li>注意：这将恢复视频捕获设备配置设置的自动保存。这将在未来的版本中重新引入。</li> 
  </ul> </li> 
 <li>将“Resize output (source size)”添加回 source context menu</li> 
</ul>
