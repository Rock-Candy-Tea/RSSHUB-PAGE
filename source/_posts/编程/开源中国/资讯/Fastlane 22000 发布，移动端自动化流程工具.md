                                                                                            <p><span style="color:#000000">Fastlane 2.200.0 发布了，</span><span style="color:#333333">Fastlane 是一个针对 iOS 和 Android 全方位开发自动化流程的工具。利用目前支持的工具可以做包含自动化和可持续化构建的每个环节，比如单元测试、截图、分发渠道、上传元数据和 ipa 包提交审核等等。</span></p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><strong>Fastlane 2.<span style="color:#000000">200.0</span> 版本主要更新内容：</strong></p> 
<ul> 
 <li>[spaceship] 初步支持更新的审查提交流程  ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffastlane%2Ffastlane%2Fpull%2F19751" target="_blank">#19751</a> )</li> 
 <li>[fastlane_core] 修复分析 ipa 时的 non-UTF-8 字符问题 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffastlane%2Ffastlane%2Fpull%2F19697" target="_blank">#19697</a> )</li> 
 <li>[supply] 添加一个选项以更改 promoted track 的发布状态  ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffastlane%2Ffastlane%2Fpull%2F19701" target="_blank">#19701</a> )</li> 
 <li>[fastlane_core] 将二进制文件上传到 Testflight/App Store 时，使用随机生成的文件名作为 -assetFile ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffastlane%2Ffastlane%2Fpull%2F19716" target="_blank">#19716</a> )</li> 
 <li>[action][get_version_number] xcode 13 项目中的 get_version_number 无法读取版本号 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffastlane%2Ffastlane%2Fpull%2F19773" target="_blank">#19773</a> )</li> 
 <li>[deliver] 添加 Watch Series 7 屏幕尺寸 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffastlane%2Ffastlane%2Fpull%2F19782" target="_blank">#19782</a> )</li> 
 <li>[action][download_dsyms] 提供<code>after_uploaded_date</code>时，修复 download-dsym 不必要的构建扫描 ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffastlane%2Ffastlane%2Fpull%2F19765" target="_blank">#19765</a> )</li> 
 <li>[gym] 使用选项“analyze_build_time”时，正确继承 OTHER_SWIFT_FLAGS ( <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffastlane%2Ffastlane%2Fpull%2F19737" target="_blank">#19737</a> )</li> 
 <li>[scan] 修复：测试重试不成功的问题（<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Ffastlane%2Ffastlane%2Fpull%2F19749" target="_blank">#19749</a>）</li> 
</ul>
