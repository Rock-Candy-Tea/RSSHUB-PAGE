                                                        <p>后台管理系统 Django-Vue-Admin v1.1.2 已经发布。</p> 
<p><img src="https://static.oschina.net/uploads/img/202107/11155519_5kYm.png" referrerpolicy="no-referrer"></p> 
<p>此版本更新内容包括：</p> 
<ul> 
 <li>新功能 
  <ul> 
   <li>所有包引入都加apps</li> 
   <li>python manage.py createapp 一级文件名/app01 ... # 支持多级目录建app</li> 
   <li>新增弹出框设置参数，在点击弹窗外不关闭弹窗功能</li> 
   <li>接入drf-yasg，进行swagger形式的api展示</li> 
   <li>可配置取消redis及Windows文件上传问题修复</li> 
  </ul> </li> 
 <li>修复bug 
  <ul> 
   <li>CURD表单点击重置分页器失灵修复</li> 
   <li>修复系统接口菜单无法显示问题</li> 
   <li>部门动态加载,根据角色分配</li> 
   <li>init初始数据中的普通用户角色，默认添加首页权限</li> 
   <li>取消在线用户功能</li> 
   <li>修复定时任务只显示10条问题</li> 
  </ul> </li> 
</ul> 
<p>具体更新内容：<a href="https://gitee.com/liqianglog/django-vue-admin/pulls/37" target="_blank">https://gitee.com/liqianglog/django-vue-admin/pulls/37</a></p> 
<p>详情查看：<a href="https://gitee.com/liqianglog/django-vue-admin/releases/v1.1.2">https://gitee.com/liqianglog/django-vue-admin/releases/v1.1.2</a></p>
