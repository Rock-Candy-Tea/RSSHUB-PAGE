                                                        <p>Qt 6.0.3 已经发布，这是 Qt 6.0 系列的第三个补丁版本。作为补丁版本，Qt 6.0.3 不会添加任何新功能，但会提供 bug 修复和其他优化。 与 Qt 6.0.2 相比，新的 Qt 6.0.3 包含 40 多个 bug 修复。</p> 
<p><img alt height="291" src="https://oscimg.oschina.net/oscnet/up-f246319b198417d159d8572e91a69a22b4a.png" width="802" referrerpolicy="no-referrer"></p> 
<p><strong>主要更新内容</strong></p> 
<ul> 
 <li>qtbase 
  <ul> 
   <li>修复了与以姓名结尾或以空格开头的字体的匹配</li> 
   <li>libjpeg-turbo 已更新至版本 2.0.6</li> 
   <li>将 SQLite 更新为 v3.35.2</li> 
   <li>修复没有 QWGLContext 文件的问题</li> 
   <li>修复 QFuture 文档链接错误</li> 
  </ul> </li> 
 <li>qtsvg 
  <ul> 
   <li>修复当输入工艺 svg 文件时，在函数 QRadialFetchSimd<QSimdSse2>::fetch 中读取到越界信息的问题</li> 
  </ul> </li> 
 <li>qtwayland 
  <ul> 
   <li>修复了一个问题，如果两个客户端几乎同时启动，虚拟键盘将无法正确更新</li> 
  </ul> </li> 
 <li>qttools 
  <ul> 
   <li>在QDoc文档中，关于 "实例 "的陈述不完整</li> 
  </ul> </li> 
</ul> 
<p>详情请查看<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.qt.io%2Fblog%2Fqt-6.0.3-released" target="_blank">更新公告</a>。</p>
