                                                                                            <p>Logseq 是一个知识管理和协作的平台。它专注于隐私和用户控制。服务器永远不会存储或分析你的私人笔记。目前支持 Markdown 和 Emacs Org 模式（更多内容即将加入）。</p> 
<p>Logseq 0.8 发布，更新内容如下：</p> 
<h3>功能</h3> 
<ul> 
 <li>新增设置选项 <code>:feature/enable-search-remove-accents</code>，以获得更好的搜索体验</li> 
 <li>将宏提取为块；为慢速查询增加一个刷新按钮（这些查询不会自动重新执行）</li> 
</ul> 
<h3>修复</h3> 
<ul> 
 <li>在全局图表视图中删除每个级别的命名空间层次结构页面的链接</li> 
 <li>扩展模板时间变量时时区错误</li> 
 <li>侧边栏中的 Recent 不记得其状态</li> 
 <li>隐藏属性和属性排序的属性补全</li> 
 <li>重启文本的光标样式</li> 
 <li>从 zotero 的链接文件路径中删除绝对文件路径</li> 
 <li>链接引用和查询的错误修正；查询的性能提升</li> 
 <li>只在必要时显示属性搜索模式</li> 
 <li>用多个选定的块进行导航</li> 
 <li>处理损坏的搜索数据库，导致应用程序崩溃</li> 
</ul> 
<h3>增强</h3> 
<ul> 
 <li>多语言高亮 PDF 文本的友好换行符</li> 
 <li>增加了更多的西班牙语字符串</li> 
 <li>荷兰语翻译</li> 
 <li>更多中文翻译</li> 
 <li>点击插件标题可打开 README</li> 
 <li>一些插件的 API 增强</li> 
</ul>
