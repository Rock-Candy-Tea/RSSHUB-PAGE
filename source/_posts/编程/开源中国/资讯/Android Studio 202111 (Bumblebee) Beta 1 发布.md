<div class="content">
                                                                    
                                                        <p>Android Studio 2021.1.1（代号"Bumblebee"）首个 Beta 版本<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fandroidstudio.googleblog.com%2F2021%2F10%2Fandroid-studio-bumblebee-beta-1.html" target="_blank">已发布</a>。</p> 
<p>主要新特性：</p> 
<ul> 
 <li style="margin-right: -40px; text-align: start;"><span>Lint analysis task 现已支持缓存</span></li> 
 <li style="margin-right: -40px; text-align: start;">C/C++模块现已支持引用同一项目中的其他 C/C++ 模块</li> 
 <li style="margin-right: -40px; text-align: start;"><span>支持执行 KMP 测试</span></li> 
 <li style="margin-right: -40px; text-align: start;"><span>在</span><span><code>settings.gradle</code></span>文件增加仓库和版本设置项</li> 
 <li style="margin-right: -40px; text-align: start;"><span>引入新的 Device Manager</span></li> 
</ul> 
<p style="margin-right:-40px; text-align:start"><img alt src="https://oscimg.oschina.net/oscnet/up-f56b3c2ed5af5f3af28ec64c28141cdff91.png" referrerpolicy="no-referrer"></p> 
<ul> 
 <li style="margin-right: -40px; text-align: start;"><span>支持在 </span>Layout Inspector <span>中捕获 </span>layout hierarchy 快照</li> 
</ul> 
<p style="margin-right:-40px; text-align:start"><img alt src="https://oscimg.oschina.net/oscnet/up-1a005cc3f399f2377a306a553ec52b19efa.png" referrerpolicy="no-referrer"></p> 
<ul> 
 <li style="margin-right: -40px; text-align: start;">引入统一的 Gradle test runner</li> 
</ul> 
<p style="margin-right:-40px; text-align:start"><img alt src="https://oscimg.oschina.net/oscnet/up-b3bbb9910ef6d33d757f47030e413fff673.png" referrerpolicy="no-referrer"></p> 
<ul> 
 <li style="margin-right: -40px; text-align: start;"> <p>Android 模拟器现在默认直接在 Android Studio 中运行</p> </li> 
 <li style="margin-right: -40px; text-align: start;"> <p>支持具有动态功能的应用程序</p> </li> 
 <li style="margin-right: -40px; text-align: start;"> <p>实验性进一步缩小应用程序体积</p> </li> 
</ul> 
<p>详情查看 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdeveloper.android.com%2Fstudio%2Fpreview%2Ffeatures%232021.1.1" target="_blank">https://developer.android.com/studio/preview/features#2021.1.1</a>。</p>
