                                                                                            <p>Keka 是一个 macOS 端的应用程序，可以快速创建具有高压缩率的压缩文件，还支持从多种类型的文件中解压缩文件，其中包括 7z、ISO、DMG、TAR、ZIP 和 Bzip2 等类型的文件格式。</p> 
<p>Keka 1.2.55 具体更新内容如下：</p> 
<ul> 
 <li>增加了从 Finder 扩展中压缩文件夹内容的选项 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Faonez%2FKeka%2Fissues%2F1037" target="_blank">#1037</a></li> 
 <li>小小的 UI 调整</li> 
</ul> 
<h3>格式</h3> 
<ul> 
 <li>将 7ZZ 从 21.07 更新到 22.00</li> 
 <li>这个 7ZZ 的更新增加了对 macOS 10.13+ 上 APFS DMG 提取的支持</li> 
 <li>将 UNRAR 从 6.02 更新到 6.12</li> 
</ul> 
<h3>修复</h3> 
<ul> 
 <li>增强了 RAR 的格式检测 <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Faonez%2FKeka%2Fissues%2F1123" target="_blank">#1123</a></li> 
 <li>正确提取一些含有文件链接的 7Z 文件</li> 
</ul> 
<h3>翻译</h3> 
<p>更新了包括简体中文、繁体中文在内的多个国家和地区的语言翻译</p> 
<p>更多详情可查看：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Faonez%2FKeka%2Freleases" target="_blank">https://github.com/aonez/Keka/releases</a></p>
