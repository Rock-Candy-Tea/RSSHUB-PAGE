                                                        <p style="margin-left:0px">洛雪音乐助手是一个功能强大、音源齐全，操作简单的音乐播放器，支持导入其他音乐播放器的歌单（如 kugou、wyy 音乐等）、支持同步功能。</p> 
<p style="margin-left:0px">目前洛雪音乐更新到 1.18.0 版本，带来大量功能改进，建议大伙速速升级，体验新功能：</p> 
<h3><strong>新增</strong></h3> 
<ul> 
 <li><strong>新增“双击列表里的歌曲时自动切换到当前列表播放”设置，</strong>此功能仅对歌单、排行榜有效，默认关闭</li> 
 <li>新增打开收藏的在线列表的对应平台详情页功能，可以在我的列表-列表右键菜单中使用</li> 
 <li>新增定时暂停播放功能，由于此功能大多数人可能不常用，所以将其放在设置-基本设置中</li> 
 <li>新增任务栏缩略图工具栏控制按钮（此功能仅在 Windows 平台可用），按钮分别为收藏/取消收藏（将歌曲添加到“我的收藏”列表）、上一曲、播放/暂停、下一曲</li> 
 <li>新增设置-基本设置-软件字体设置，此设置可用于设置主界面的字体（已知的问题：Windows 7 下可能会出现字体列表为空的情况，这是当前系统的 Powershell 版本小于 5.1 导致的，请自行尝试看常见解决）</li> 
 <li>新增 Scheme URL 对音乐搜索的调用支持，详情看常见问题 -Scheme URL支持</li> 
 <li>新增 Scheme URL 以 url 传参的方式调用，详情看常见问题 -Scheme URL支持</li> 
 <li>自定义源新增更新弹窗方法，同时自定义源管理新增是否允许源显示更新弹窗设置（出于防止滥用考虑），当源作者想要通知用户源已更新时，可以调用此方法弹窗告诉用户，调用说明看常见问题-自定义源部分</li> 
</ul> 
<h3><strong>优化</strong></h3> 
<ul> 
 <li>过滤 tx 源某些不支持播放的歌曲，解决播放此类内容会导致意外的问题</li> 
 <li>把歌曲的热门评论与最新评论拆分成两个列表显示</li> 
</ul> 
<h3><strong>修复</strong></h3> 
<ul> 
 <li>修复排行榜名字右击菜单的播放功能在播放非激活的列表时的列表获取问题</li> 
 <li>修复修改列表名时无法使用 <code>Ctrl</code> 键的问题</li> 
 <li>修复 wy 源某些歌曲获取歌词翻译的问题处理</li> 
 <li>修复下载功能的歌词换源时会进入死循环的问题</li> 
 <li>修复某些歌曲无法下载的问题</li> 
 <li>修复 windows 平台下软件目录存在<code>portable</code>文件夹时，仍会创建 <code>C:\Users\<user>\AppData\Roaming\lx-music-desktop\Dictionaries\en-US-9-0.bdic</code> 文件的问题，现在不会再创建文件，但仍会创建空目录（Electron 的问题，目前暂无解决方法）</li> 
 <li>修复播放器的停止逻辑问题</li> 
</ul> 
<h3><strong>其他</strong></h3> 
<ul> 
 <li>更新 electron 到 v13.6.9 </li> 
</ul>
