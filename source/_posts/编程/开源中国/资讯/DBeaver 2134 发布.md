                                                                                            <p>DBeaver 是一个免费开源的通用数据库工具，适用于开发人员和数据库管理员。</p> 
<p>DBeaver 21.3.4 发布，更新内容如下：</p> 
<ul> 
 <li>Data editor: 
  <ul> 
   <li>行的 fetch 大小现在可以设置为一个较小的值</li> 
   <li>启用数字格式的数据网格中的搜索现在可以正常使用了</li> 
   <li>日期/时间编辑器的日历演示现在可以显示正确的错误信息</li> 
   <li>增加了突出所选单元格值的选项</li> 
   <li>值视图面板现在支持在数据格式之间以快捷键进行切换（可配置）</li> 
  </ul> </li> 
 <li>SQL editor: 
  <ul> 
   <li>修复了丢失结果集标签焦点的问题</li> 
   <li>脚本选择器面板现在支持 "All connections" 模式</li> 
  </ul> </li> 
 <li>Data transfer: 
  <ul> 
   <li>增加了对复合数据类型的列的导出支持</li> 
   <li>弹出式消息框被替换为任务栏通知</li> 
  </ul> </li> 
 <li>Navigator tree: 增加了数据库对象注释的渲染（可配置）</li> 
 <li>BLOB/binary editor: 修复了在 hex 编辑器中保存变化的问题</li> 
 <li>Dashboard view: 解决了内存泄漏的问题</li> 
 <li>增加了自动关闭空闲连接的选项（可配置）</li> 
 <li>Driver editor: 修复了 Maven 工件设置的保存/加载问题</li> 
 <li>修复了重启后编辑器标签为空的问题</li> 
 <li>修复了打开连接上下文菜单时的冗余日志记录问题</li> 
 <li>新的数据库驱动： 
  <ul> 
   <li>Apache Calcite Avatica</li> 
   <li>Apache Druid</li> 
   <li>Apache Kylin</li> 
  </ul> </li> 
 <li>Clickhouse: 
  <ul> 
   <li>增加了新的驱动程序，旧的驱动程序被重新命名为 Legacy</li> 
   <li>为数据读取查询添加了 LIMIT 子句支持</li> 
  </ul> </li> 
 <li>Exasol: 实现了执行计划的保存/加载</li> 
 <li>MySQL/MariaDB: 重新设计了权限编辑器，修正了授权选项支持</li> 
 <li>MariaDB: 
  <ul> 
   <li>增加了对新驱动程序（3.x）的支持</li> 
   <li>增加了对包和序列的支持</li> 
  </ul> </li> 
 <li>Oracle: 
  <ul> 
   <li>表约束负载性能明显提高</li> 
   <li>改进了 PL/SQL 语法支持</li> 
   <li>改进了 WITH 子句解析器</li> 
  </ul> </li> 
 <li>PostgreSQL: 
  <ul> 
   <li>为表的 DDL 增加了标识符的引用</li> 
   <li>数据库备份现在支持 "drop if exists" 选项</li> 
   <li>行替换方法 "ON CONFLICT DO UPDATE" 支持得到了修复</li> 
  </ul> </li> 
 <li>Redshift: 
  <ul> 
   <li>修复了新视图的创建</li> 
   <li>驱动程序版本更新为 2.1</li> 
  </ul> </li> 
 <li>通用驱动现在支持序列和同义词编辑/重命名</li> 
 <li>改进了日语本地化</li> 
 <li>修复了几个小的 UI 错误</li> 
</ul>
