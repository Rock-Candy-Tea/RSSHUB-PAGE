                                                                                            <p>Hunt Framework 是完全使用 D语言(DLang)开发的 Web 服务框架，可以非常快速的开发出 RESTful 以及 MVC 服务端应用，马上 2022 年元旦了把版本的稳定性进一步提升！Hunt Framework 3.4.6 累计了上个版本以来的问题进行集中修复！</p> 
<h3>主要变更如下：</h3> 
<ol> 
 <li>Logger 整合，降低 GC 使用，优化编译时控制日志输出</li> 
 <li>完善 ObjectPool 模块</li> 
 <li>基于 ObjectPool 对 Redis 连接池进行重构</li> 
 <li>基于 ObjectPool 对 Database 连接池进行重构</li> 
 <li>升级兼容依赖容器 Poodinis 8.1.x版本</li> 
 <li>修复 HashMap 模块在处理超4万个元素时容易出现的断言异常问题</li> 
 <li>修复 ArrayList 模块存在内存泄漏问题</li> 
 <li>修复静态文件缓存时间参数失效的问题</li> 
 <li>修复 config 配置频繁加载问题</li> 
 <li>其他细节完善</li> 
</ol>
