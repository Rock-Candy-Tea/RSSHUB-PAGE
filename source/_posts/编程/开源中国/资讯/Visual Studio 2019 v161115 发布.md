                                                                                            <p>Visual Studio 2019 v16.11.15 正式发布，更新内容如下：</p> 
<ul> 
 <li>修复了 SQL Server Data Tools 中 Azure SQL 托管实例的连接，包括 Schema 比较和 SQL Server 资源管理器。注意：对 Azure Arc 启用的托管实例的支持将在后续版本中发布</li> 
 <li>为剩余的 C++20 缺陷报告添加了实现，所有 C++20 功能现在都在 /std：c++20 开关下提供。</li> 
 <li>已将 Visual Studio 和可安装可选组件使用的 Windows 版本的 Git 更新为 2.36.0.1</li> 
 <li>修复了 git 集成的问题，其中，如果拉取/同步分支已偏离，则输出窗口不会显示有关如何解决它的本地化提示。</li> 
</ul> 
<p>更多详情可查看：<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fdocs.microsoft.com%2Fen-us%2Fvisualstudio%2Freleases%2F2019%2Frelease-notes" target="_blank">https://docs.microsoft.com/en-us/visualstudio/releases/2019/release-notes</a></p>
