                                                                                            <p><span style="color:#333333">NsCDE 是一个复古但功能强大的轻量级 UNIX 桌面环境，类似于 CDE 的外观，但具有更强大和更灵活的底层框架，更适合现代类 unix 和 Linux 系统和用户需求。</span></p> 
<p>NsCDE 结合了一些自由软件组件和自定义 FVWM 应用程序和大量配置，可视作一个依赖于 FVWM 的轻量级混合桌面环境。</p> 
<p><img height="438" src="https://static.oschina.net/uploads/space/2022/0726/111150_N1dz_5430600.png" width="700" referrerpolicy="no-referrer"></p> 
<p>目前 NsCDE 发布了 2.2 版本，添加了一些功能、第三方集成、Bug 修复等内容。</p> 
<ul> 
 <li>修复 rofi 和 dunst 主题</li> 
 <li>重命名 NsCDE .desktop 文件以符合标准</li> 
 <li>图标主题更新</li> 
 <li>CSS：支持火狐100+</li> 
 <li>合并 GTK2 和 GTK3 引擎 css 文件，添加一些修复</li> 
 <li>添加了通用用户访问 (CUA) 键绑定，作为 NsCDE 设置的默认键绑定。</li> 
 <li>修复键盘样式管理器插件和鼠标样式管理器</li> 
 <li>构建：在配置中添加 --with-python-shebang="STRING"</li> 
 <li>重组 f_PolkitAgent ，使其更加便携和稳定</li> 
 <li>Kvantum：重塑更多 Motif 风格的 Qt5 组合框</li> 
 <li>Firefox CSS 更新</li> 
</ul>
