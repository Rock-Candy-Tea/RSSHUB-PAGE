<div class="content">
                                                                                            <p>经过 6 个多月的开发后，Godot 3.4 现已正式<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgodotengine.org%2Farticle%2Fgodot-3-4-is-released" target="_blank">发布</a>。官方表示，虽然开发团队大部分的开发重点都在即将发布的 Godot 4.0 上；但鉴于许多贡献者和用户都希望有一个强大而成熟的 3.x 分支来开发他们的游戏，所以继续为 Godot 3 用户提供更好的游戏开发体验非常重要。因此，该版本大部分的重点是实现缺失的功能或错误修复，这对用 Godot 3 发布 2D 和 3D 游戏至关重要，并使现有功能更加优化和可靠。</p> 
<p><span style="background-color:#ffffff; color:#333333">Godot 3.4 与 Godot 3.3.x 兼容，开发团队推荐所有使用 3.3.x 的用户进行升级。</span></p> 
<p><span style="background-color:#ffffff; color:#333333">下载地址：</span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgodotengine.org%2Fdownload" target="_blank">https://godotengine.org/download</a></p> 
<p><img alt height="281" src="https://oscimg.oschina.net/oscnet/up-f3f230d938ab7e1358cdf4e5a13c8f69980.jpg" width="500" referrerpolicy="no-referrer"></p> 
<p>更新亮点如下：</p> 
<p style="text-align:start"><span style="color:#000000"><strong>Core：</strong></span></p> 
<ul> 
 <li><span style="color:#000000">促进 object validity 检查以发布版本</span></li> 
 <li><span style="color:#000000">大文件支持 (> 2.0 GiB)</span></li> 
 <li><span style="color:#000000">Frame delta smoothing</span></li> 
 <li><span style="color:#000000">改进的输入处理</span></li> 
 <li><span style="color:#000000">Crypto：AES 和 HMAC contexts</span></li> 
</ul> 
<p style="text-align:start"><span style="color:#000000"><strong>渲染：</strong></span></p> 
<ul> 
 <li><span style="color:#000000">Portal occlusion culling</span></li> 
 <li><span style="color:#000000">ACES Fitted tonemapper</span></li> 
 <li><span style="color:#000000">用于 3D 粒子的环形发射器</span></li> 
 <li><span style="color:#000000">着色器语言特性</span></li> 
 <li><span style="color:#000000">更多渲染改进</span></li> 
</ul> 
<p style="text-align:start"><span style="color:#000000"><strong>Platforms：</strong></span></p> 
<ul> 
 <li><span style="color:#000000">Android：范围存储、播放资产交付、输入响应</span></li> 
 <li><span style="color:#000000">HTML5：PWA、Godot/JavaScript 接口、AudioWorklet</span></li> 
 <li><span style="color:#000000">macOS：Mono 通用构建、GDNative 框架、notarization</span></li> 
</ul> 
<p style="text-align:start"><span style="color:#000000"><strong>Physics：</strong></span></p> 
<ul> 
 <li><span style="color:#000000">2D 和 3D 中的 KinematicBody 改进</span></li> 
 <li><span style="color:#000000">更快更可靠的 convex hull 生成</span></li> 
 <li><span style="color:#000000">检查器中改进的碰撞层网格</span></li> 
 <li><span style="color:#000000">用于 Godot Physics 2D 的动态 BVH</span></li> 
 <li><span style="color:#000000">Godot Physics 3D 的改进和新功能</span></li> 
</ul> 
<p style="text-align:start"><span style="color:#000000"><strong>Assets pipeline：</strong></span></p> 
<ul> 
 <li><span style="color:#000000">将 3D 场景导出为 glTF</span></li> 
 <li><span style="color:#000000">无损 WebP 编码</span></li> 
</ul> 
<p style="text-align:start"><span style="color:#000000"><strong>Editor：</strong></span></p> 
<ul> 
 <li><span style="color:#000000">改进的 UI 主题编辑器</span></li> 
 <li><span style="color:#000000">本地化类参考</span></li> 
 <li><span style="color:#000000">一般可用性改进</span></li> 
</ul> 
<p style="text-align:start"><span style="color:#000000"><strong>其它地方：</strong></span></p> 
<ul> 
 <li><span style="color:#000000">动画“reset”轨道</span></li> 
 <li><span style="color:#000000">2D 视口比例因子</span></li> 
 <li><span style="color:#000000">对 CSGPolygon 路径模式的改进和修复</span></li> 
</ul>
