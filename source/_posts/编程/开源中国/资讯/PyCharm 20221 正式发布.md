<div class="content">
                                                                                            <p><span style="background-color:#ffffff; color:#000000">v2022.1 版本是 PyCharm 在 2022 年的第一个版本，该版本专注于完善现有功能和工作流程，而不是添加新功能，尤其是改善在之前的版本中所引入的多个期待已久的功能之后，例如对 Jupyter 的支持和远程开发等。</span></p> 
<p><img alt height="394" src="https://static.oschina.net/uploads/space/2022/0415/072701_mugH_5430600.png" width="700" referrerpolicy="no-referrer"></p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start">PyCharm 2022.1 的更新内容包括：</p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>IDE 
  <ul style="margin-left:0; margin-right:0"> 
   <li>对自定义软件包库的认证支持 
    <ul style="margin-left:0; margin-right:0"> 
     <li>可以配置基本的 HTTP 认证来访问自定义软件包库，并通过 PyCharm 管理依赖项，而无需切换到终端进行手动安装。</li> 
    </ul> </li> 
  </ul> </li> 
 <li>代码洞察 
  <ul style="margin-left:0; margin-right:0"> 
   <li>TypedDict 的增强代码补全</li> 
   <li>改进的 TypedDict 警告</li> 
  </ul> </li> 
 <li>改进的 Markdown 支持 
  <ul style="margin-left:0; margin-right:0"> 
   <li>从 Markdown 文件运行命令</li> 
   <li>复制 Markdown 的代码片段</li> 
  </ul> </li> 
 <li>Jupyter 支持[Pro] 
  <ul style="margin-left:0; margin-right:0"> 
   <li>代码单元在执行后仍处于编辑模式</li> 
   <li>优化的单元格复制粘贴</li> 
  </ul> </li> 
</ul>
