                                                        <p>Vivaldi 是基于 Chromium 的网络浏览器，近日 Vivaldi 4.3 正式发布，新版本改进了 Vivaldi 的几个关键工具和功能，同时进一步改善了用户的隐私。该版本更新内容如下：</p> 
<h3>空闲检测 API</h3> 
<p><img alt="https://www.ghacks.net/wp-content/uploads/2021/10/vivaldi-4.3-idle-detection-api.png" height="443" src="https://www.ghacks.net/wp-content/uploads/2021/10/vivaldi-4.3-idle-detection-api.png" width="700" referrerpolicy="no-referrer"></p> 
<p>Google 近日在 Chromium 中引入了空闲监测 API，采用 Chromium 的 Vivaldi 在升级至新版 Chromium 之后自然也引入了该 API。不过，Vivaldi 已默认在浏览器中禁用了空闲检测 API（如上图），这意味着网站不能要求获得访问该 API 的许可。</p> 
<p>Vivaldi 的用户可以通过访问 "设置">"隐私" 手动修改这个选项。</p> 
<h3><strong>Vivaldi 翻译</strong></h3> 
<p><img alt="https://www.ghacks.net/wp-content/uploads/2021/10/vivaldi-4.3-translate.png" height="417" src="https://www.ghacks.net/wp-content/uploads/2021/10/vivaldi-4.3-translate.png" width="700" referrerpolicy="no-referrer"></p> 
<p>Vivaldi 在 4.0 版本中引入了翻译功能，但当时支持的语言数量相对较少。随着 Vivaldi 4.3 的发布，这种情况有所改变，目前支持的语言数量增加了 68 种，总共有 108 种语言。</p> 
<p>Vivaldi Translate 提供两个主要功能：</p> 
<ul> 
 <li>在选定的文本上点击右键，将选定的内容翻译成另一种语言；</li> 
 <li>通过点击翻译图标来翻译整个网页。</li> 
</ul> 
<h3>其他变化</h3> 
<ul> 
 <li>改进，并在视觉上更新了屏幕捕捉工具，能够在浏览器中直接调整屏幕截图的大小</li> 
 <li>同步设置得到了改进</li> 
 <li>下载面板现在显示一个图表，直观地显示下载速度</li> 
 <li>Vivaldi Feed Reader 支持订阅 YouTube 频道，无需登录 YouTube 或 Google 账户</li> 
 <li>Vivaldi 4.3 支持安装 PWA</li> 
</ul>
