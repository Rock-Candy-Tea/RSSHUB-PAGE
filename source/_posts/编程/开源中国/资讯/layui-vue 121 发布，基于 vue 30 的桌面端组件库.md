                                                                                            <p>更新内容：</p> 
<p><span style="background-color:#ffffff; color:#40485b">[新增] dropdown 组件 placement 属性, 设置下拉面板位置。</span><br> <span style="background-color:#ffffff; color:#40485b">[新增] dropdown 组件 autoFitPosition 属性, 超出浏览器边界自动调整下拉面板位置, 默认为 true。</span><br> <span style="background-color:#ffffff; color:#40485b">[新增] dropdown 组件 autoFitWidth 与 autoFitMinWidth 属性, 是否将下拉面板宽度设置为触发器宽度。</span><br> <span style="background-color:#ffffff; color:#40485b">[新增] dropdown 组件 updateAtScroll 属性, 是否在容器滚动时更新下拉面板的位置,默认 false。</span><br> <span style="background-color:#ffffff; color:#40485b">[新增] dropdown 组件 autoFixPosition 属性, 是否在触发器或下拉面板尺寸变化时更新下拉面板位置。</span><br> <span style="background-color:#ffffff; color:#40485b">[新增] dropdown 组件 clickOutsideToClose 属性, 点击面板外部关闭下拉, 默认 true。</span><br> <span style="background-color:#ffffff; color:#40485b">[新增] dropdown 组件 contentOffset 属性, 下拉面板距离触发器的偏移距离，默认 2。</span><br> <span style="background-color:#ffffff; color:#40485b">[新增] progress 组件 circleSize 和 circleWidth 属性, 用于控制环形进度条尺寸与线条宽度。</span><br> <span style="background-color:#ffffff; color:#40485b">[修复] progress 组件 circle 模式夜间主题不兼容的问题。</span><br> <span style="background-color:#ffffff; color:#40485b">[新增] carousel 组件 过渡动画，默认为滑动。</span><br> <span style="background-color:#ffffff; color:#40485b">[修复] carousel 组件 autoplay 属性为false时仍自动播放问题。</span><br> <span style="background-color:#ffffff; color:#40485b">[修复] cascader 组件 按需加载无法正常引入使用</span></p> 
<p><span style="color:#40485b">更多详情：<a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fwww.layui-vue.com" target="_blank">http://www.layui-vue.com</a></span></p>
