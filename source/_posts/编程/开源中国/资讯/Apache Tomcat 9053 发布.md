 <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">Apache Tomcat 9.0.53 现已<a href="https://www.oschina.net/action/GoToLink?url=http%3A%2F%2Fmail-archives.apache.org%2Fmod_mbox%2Fwww-announce%2F202109.mbox%2F%253cCANwj8Zp3bkLRL9Pj1fZJpVX-UyNS%3DKo4C2ojfYxdhW1XRjKx%2Bw%40mail.gmail.com%253e" target="_blank">发布</a>，Apache Tomcat 9 是一个开源软件，它实现了 Java Servlet、JavaServer Pages、Java Unified Expression Language、Java WebSocket 和 JASPIC 技术。本次更新是一个错误修复版本。</p> 
 <p style="color:#000000; margin-left:0; margin-right:0; text-align:start">Apache Tomcat 9.0.53 是一个错误修复和功能发布，值得关注的变化有：</p> 
 <ul style="margin-left:0; margin-right:0"> 
  <li>增加一个 UserDatabase 的实现，作为 DataSourceRealm 功能的超集。</li> 
  <li>将 Apache Commons DBCP 的内部分支更新到 2.9.0，将 Apache Commons Pool 更新到 2.11.1。</li> 
  <li>将 Tomcat  Native Library 的打包版本更新为 1.2.31，以获取用 OpenSSL 1.1.1l 构建的 Windows 二进制文件。</li> 
 </ul>
