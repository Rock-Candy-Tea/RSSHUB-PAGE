                                                                                            <p>VimR 项目是一个用于 macOS 的 Neovim GUI。目标是建立一个在内部使用 Neovim 的编辑器，并具有许多类似于现代编辑器的便利 GUI 功能，VimR 主要使用 Swift。</p> 
<p>VimR 0.37 正式发布，更新内容如下：</p> 
<ul> 
 <li>错误修正：vimr 不能正确处理文件名中的空格，请重新安装 CLI vimr 工具</li> 
 <li>在进入 Normal 模式时自动切换到与 ASCII 兼容的输入法，在重新进入 Insert 模式时切换回最后使用的输入法</li> 
 <li>CLI 工具 vimr 现在使用 Python 3</li> 
 <li>依赖项更新： 
  <ul> 
   <li>ReactiveX/RxSwift 更新至 6.5.0</li> 
  </ul> </li> 
</ul>
