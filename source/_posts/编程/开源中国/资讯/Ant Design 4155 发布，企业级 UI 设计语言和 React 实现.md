                                                        <p>Ant Design 4.15.5 发布了。Ant Design 是一套企业级的 UI 设计语言和 React 实现，使用 TypeScript 构建，提供完整的类型定义文件，自带提炼自企业级中后台产品的交互语言和视觉风格、开箱即用的高质量 React 组件与全链路开发和设计工具体系。</p> 
<p>此版本更新内容如下：</p> 
<ul> 
 <li>修复 Upload 在 IE 下，<code>beforeUpload</code> 返回 <code>false</code> 的报错问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F30391" target="_blank">#30391</a> </li> 
 <li>Calendar 
  <ul> 
   <li>修复 Calendar 在 Form.Item 下样式错乱的问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F30442" target="_blank">#30442</a></li> 
   <li>修复 Calendar <code>value</code> 设置 <code>null</code> 和 <code>undefined</code> 时报错的问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F30442" target="_blank">#30442</a>。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F30442" target="_blank">#30442</a></li> 
  </ul> </li> 
 <li>Table 
  <ul> 
   <li>修复 Table <code>filterDropdown</code> 调用 <code>confirm(&#123; closeDropdown: true &#125;)</code> 时也会触发 <code>onFilterDropdownVisibleChange</code> 的问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F30457" target="_blank">#30457</a></li> 
   <li>修复 Table 展开图标在 Windows Firefox 下显示异常的问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F30410" target="_blank">#30410</a></li> 
  </ul> </li> 
 <li>修复 Dropdown.Button 不支持 <code>mouseEnterDelay</code> 和 <code>mouseLeaveDelay</code> 的问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F30452" target="_blank">#30452</a></li> 
 <li>优化 Rate 聚焦行为从 <code>:focus</code> 到 <code>:focus-visible</code>。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F30451" target="_blank">#30451</a></li> 
 <li>修复 Steps 进度条圆圈在底部被切掉的问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F30373" target="_blank">#30373</a></li> 
 <li>修复在 DatePicker 里设置 <code>locale</code> 不能覆盖 ConfigProvider <code>locale</code> 的问题。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F30380" target="_blank">#30380</a></li> 
 <li>国际化 
  <ul> 
   <li>改进 DatePicker 芬兰语国际化。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F30433" target="_blank">#30433</a></li> 
   <li>更新法语国际化。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F30436" target="_blank">#30436</a></li> 
   <li>更新塞尔维亚语国际化。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F30401" target="_blank">#30401</a></li> 
  </ul> </li> 
 <li>TypeScript 
  <ul> 
   <li>Tooltip 导出 <code>zIndex</code> 类型定义。<a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2Fant-design%2Fant-design%2Fpull%2F30432" target="_blank">#30432</a></li>
