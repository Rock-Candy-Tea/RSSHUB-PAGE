                                                                                            <p>Calibre 开源项目是 Calibre 官方出的电子书管理工具。它可以查看，转换，编辑和分类所有主流格式的电子书。Calibre 是个跨平台软件，可以在 Linux、Windows 和 macOS 上运行。</p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">Calibre 5.34 正式发布，此次更新内容如下：</p> 
<p style="margin-left:0px; margin-right:0px; text-align:left"><strong>新功能</strong></p> 
<ul> 
 <li>新 Nook Glowlight 4 的驱动程序</li> 
 <li>Edit book：拼写检查工具：添加排除文件按钮以排除某些文件被检查</li> 
 <li>EPUB/MOBI Catalogs：将最大缩略图大小从 2 英寸增加到 3 英寸</li> 
 <li>允许在 Preferences->Shortcuts->Edit metadata 中创建快捷方式以粘贴元数据，而忽略 exclude_fields 调整的值</li> 
 <li>E-book viewer：将标记为词汇表和书目链接的链接显示为弹出窗口</li> 
 <li>在 Preferences->Tweaks 中添加一个调整以提供未定义数字的排序值</li> 
</ul> 
<p style="margin-left:0px; margin-right:0px; text-align:start"><strong><span style="color:#000000">Bug 修复</span></strong></p> 
<ul> 
 <li>Edit book：修复按 F8 跳转到下一个拼错的单词在当前文件中的最后一个单词之后不起作用</li> 
 <li>修复 calibre 5 中损坏的 PDB 电子阅读器输出</li> 
 <li>Edit book：Reports：包括在计算类使用情况时使用类的后代选择器</li> 
 <li>E-book viewer：修复了打开某些具有跨行文本格式的突出显示的书籍时的错误</li> 
 <li>MTP driver：请勿将 calibre 设备 db 文件发送到 Supernote A5 x 上的根文件夹，因为它会失败</li> 
 <li>ToC Editor：如果正在编辑的文件位于 DropBox/防病毒文件夹中，则解决在关闭 Windows 时偶尔出现的错误</li> 
 <li>修复了先前版本中导致创建新键盘快捷键失败的回归</li> 
 <li>Comments editor：将工具栏移动到多行时不要拆分按钮组</li> 
 <li>一些 Linux 发行版使用的 Python 3.10 的各种兼容性修复</li> 
</ul>
