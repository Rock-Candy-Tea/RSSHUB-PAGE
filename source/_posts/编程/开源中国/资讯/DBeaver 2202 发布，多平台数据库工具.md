                                                                                            <p>DBeaver 22.0.2 已发布。</p> 
<p>DBeaver 是免费的多平台数据库工具，适用于开发人员、数据库管理员、分析师和所有需要使用数据库的人，并且支持所有流行的数据库：MySQL、PostgreSQL、SQLite、Oracle、DB2、SQL Server、Sybase、MS Access、Teradata、Firebird、Apache Hive、Phoenix、Presto 等。</p> 
<p><img alt src="https://oscimg.oschina.net/oscnet/up-38c99e9030ab8f3b1f73848b29cd8184697.png" referrerpolicy="no-referrer"></p> 
<p>更新内容：</p> 
<ul style="margin-left:0; margin-right:1.5em"> 
 <li>数据编辑器： 
  <ul style="margin-left:0; margin-right:0"> 
   <li>添加“显示 whitespaces”选项</li> 
   <li>修复数据过滤器对话框中的列重新排序功能</li> 
   <li>添加行号到纯文本视图</li> 
  </ul> </li> 
 <li>SQL 编辑器： 
  <ul style="margin-left:0; margin-right:0"> 
   <li>脚本错误可视化不可配置</li> 
   <li>改进块语句 (BEGIN, IF, CASE) 的自动关闭</li> 
   <li>修复 INSERT 查询的自动完成功能</li> 
   <li>添加“Cancel active query”命令</li> 
   <li>修复别名列的过滤</li> 
   <li>修复在连接丢失时重新执行 DML 查询的问题</li> 
   <li>将 Python 源代码格式添加到 Copy As 命令</li> 
   <li>默认情况下禁用“Hippie”自动完成功能（由于性能问题）</li> 
   <li>修复 SQL 命令解析器/导航</li> 
   <li>长字符串（如 $$）的参数绑定对话框被禁用</li> 
   <li>修复 CASE/WHEN 格式化问题</li> 
  </ul> </li> 
 <li>驱动编辑器： 
  <ul style="margin-left:0; margin-right:0"> 
   <li>修复驱动程序描述编辑器</li> 
   <li>优化 Maven 版本方案</li> 
  </ul> </li> 
 <li>数据传输：修复重新创建表映射支持功能（保存的配置和任务）</li> 
 <li>SSH 隧道：修复连接超时/keep-alive 参数变更的问题</li> 
 <li>MacOS：修复应用程序启动问题（plist 文件修改问题）</li> 
 <li>深色主题：按钮 UI 现在与操作系统主题匹配</li> 
 <li>Apache Druid, Apache Kylin： 
  <ul style="margin-left:0; margin-right:0"> 
   <li>修复表数据读取问题</li> 
   <li>修复数据类型属性问题</li> 
   <li>修复索引读取问题</li> 
  </ul> </li> 
 <li>添加 Azure Databricks 驱动程序</li> 
 <li>Clickhouse：实现数组数据类型编辑器</li> 
 <li>DB2：添加了查询终止符重新定义支持</li> 
 <li>H2： 
  <ul style="margin-left:0; margin-right:0"> 
   <li>修复系统视图 DDL 生成功能</li> 
   <li>修复新 H2 驱动程序中的几何数据类型渲染</li> 
  </ul> </li> 
 <li>MySQL：修复函数执行</li> 
 <li>PostgreSQL： 
  <ul style="margin-left:0; margin-right:0"> 
   <li>实现嵌套数组编辑器</li> 
   <li>修复函数源代码生成（函数签名）</li> 
  </ul> </li> 
 <li>修复葡萄牙语本地化问题</li> 
</ul>
