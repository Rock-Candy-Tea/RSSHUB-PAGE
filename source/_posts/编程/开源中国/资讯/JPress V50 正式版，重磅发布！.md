                                                                                            <p style="color:#595959; margin-left:0; margin-right:0; text-align:left">JPress 是一个使用 Java 开发的、功能极其强大的 CMS 网站管理系统，始于 2015 年，经过七年的发展，目前已有超过 10w+ 网站使用 JPress 搭建，其中包括多个市级政府门户、200+上市公司、中科院、红+字会等等。</p> 
<p style="color:#595959; margin-left:0; margin-right:0; text-align:left">JPress 的发展，离不开诸多优秀的代码贡献用户，没有你们就没有 JPress 的今天。感谢！</p> 
<p><img height="1069" src="https://oscimg.oschina.net/oscnet/up-a9fd663d9912ab7f95e63e6353689c92eb8.png" width="1071" referrerpolicy="no-referrer"></p> 
<h2 style="margin-left:0; margin-right:0; text-align:left"><span>JPress v5 更新了哪些？</span></h2> 
<p style="color:#595959; margin-left:0; margin-right:0; text-align:left">JPress V5 的更新，主要是围绕着 “政企门户” 的功能点去更新的，因此，在新增功能的同时，相比 JPress V4.x ，我们也移除了许多的功能。</p> 
<h3 style="margin-left:0; margin-right:0; text-align:left">1、移除了电商的功能</h3> 
<p style="color:#595959; margin-left:0; margin-right:0; text-align:left">移除电商功能，主要是因为以下 3 点：</p> 
<ul style="list-style-type:disc; margin-left:0; margin-right:0"> 
 <li> <p>1、在目前中国的市场中，我们很难再见到有企业通过自己的官网来售卖产品；</p> </li> 
 <li> <p>2、由于国家或者地方的一些政策，企业在建立官网时有电商的功能，就需要更多的相关资质，电商功能反而给企业带来累赘。</p> </li> 
 <li> <p>3、电商功能逻辑复杂，若想完善支持，必然会带来更多的编码、测试和维护，因而增加了 JPress 产品的技术债务。</p> </li> 
</ul> 
<p style="color:#595959; margin-left:0; margin-right:0; text-align:left">在大环境的趋势下，我们在 JPress V5 版本中，移除了购物车、在线支付、订单管理、财务管理、优惠券、会员等跟电商相关的功能。但是，商品的展示功能没有移除，企业可以在后台发布产品信息，用户通过官网了解产品后，通过其外链到其他平台完成购买行为。</p> 
<h3 style="margin-left:0; margin-right:0; text-align:left">2、移除了用户中心的投稿、收藏等功能</h3> 
<p style="color:#595959; margin-left:0; margin-right:0; text-align:left">JPress 始于 2015 年，那时是一个公众号盛行的 “媒体” 时代，JPress 也 “担当” 起了行业新闻门户基石的角色，在这种需求驱使下，投稿、收藏等是一个 “媒体” 网站的基础功能。但现在，JPress 定位为 “企业门户”，已经不会再有人给一个企业去投稿，也不会再有人去注册账号收藏一篇企业门户的文章了....</p> 
<h3 style="margin-left:0; margin-right:0; text-align:left">3、新增多站点模块</h3> 
<p><img height="748" src="https://oscimg.oschina.net/oscnet/up-794fe0617ba2cc1727818c15ad28edf0d91.png" width="1080" referrerpolicy="no-referrer"></p> 
<p style="color:#595959; margin-left:0; margin-right:0; text-align:left">随着中国的强大，已经有越来越多的企业在走出国门。他们针对不同国家和地区推出不同的站点的需求，也越来越是明确。</p> 
<p style="color:#595959; margin-left:0; margin-right:0; text-align:left">JPress 为了满足这类企业的需求，特别推出了多站点功能，每个站点可以绑定不同的语言，当相关语言用户访问到网站时，会自动跳转到最为匹配的子站点上去，为用户提供更好、更直观的服务。</p> 
<h3 style="margin-left:0; margin-right:0; text-align:left">4、新增表单模块</h3> 
<p><img height="748" src="https://oscimg.oschina.net/oscnet/up-a28f94485389c6f1660e43ee9b8011cd6ab.png" width="1080" referrerpolicy="no-referrer"></p> 
<p style="color:#595959; margin-left:0; margin-right:0; text-align:left">JPress 表单模块，是一个类似 “问卷网”、“问卷星”、“金数据” 的产品模块，帮助企业在自己的官网上进行问卷调查、数据搜集、用户报名或者在线投票等工作，同时数据不会经过任何 SaaS 或第三方，理论上数据更加私密和安全。</p> 
<p><img height="748" src="https://oscimg.oschina.net/oscnet/up-05ca18189a1f33d8ea3b7ca47f1e2f54f88.png" width="1080" referrerpolicy="no-referrer"></p> 
<p style="color:#595959; margin-left:0; margin-right:0; text-align:left">同时，创建好表单后，我们可以把表单插入到任意的官网文章中，此时，用户在阅读文章的同时，可以参加表单数据的提交，如下图所示：</p> 
<p> </p> 
<p><img height="862" src="https://oscimg.oschina.net/oscnet/up-2981e02ed67398c35ae26a04cc31fe12b5d.png" width="853" referrerpolicy="no-referrer"></p> 
<h3 style="margin-left:0; margin-right:0; text-align:left">5、新增视频模块</h3> 
<p><img height="707" src="https://oscimg.oschina.net/oscnet/up-7298ec90f99e838034aae92debdae7edd70.png" width="1080" referrerpolicy="no-referrer"></p> 
<p style="color:#595959; margin-left:0; margin-right:0; text-align:left">在抖音、快手、视频号横行的今天，在 5G 网络、高速网络家至户到的今天，视频俨然成为一种最为直观的信息载体和媒介。</p> 
<p style="color:#595959; margin-left:0; margin-right:0; text-align:left">JPress 的视频功能，直接打通了阿里云 VoD 和 腾讯云 VoD，在支持本地视频的同时整合云端成熟产品和技术，为企业提供了更好的视频管理模块。在发布文章和页面时，我们可以轻松地把云端视频嵌入到网页中，多维度<span style="color:#595959">地</span>介绍公司及其产品。</p> 
<h3 style="margin-left:0; margin-right:0; text-align:left">6、新增招聘模块</h3> 
<p><img height="700" src="https://oscimg.oschina.net/oscnet/up-c1cba65654e467634378b72e99b64c91e57.png" width="1080" referrerpolicy="no-referrer"></p> 
<p style="color:#595959; margin-left:0; margin-right:0; text-align:left">一个企业最为重要的资源，便是人才，而招聘则是最直接获取人才的渠道。通过公司官网招聘，不仅仅是极为有效的办法，而且是最为 “正规” 的办法之一。</p> 
<p style="color:#595959; margin-left:0; margin-right:0; text-align:left">JPress 的愿景是成为每个政企门户的首选系统，那招聘则是一个最为基础的模块。</p> 
<h3 style="margin-left:0; margin-right:0; text-align:left">7、新增模板拖拽</h3> 
<p><img height="700" src="https://oscimg.oschina.net/oscnet/up-154a56bd882ef08642c43311a30fe184443.png" width="1080" referrerpolicy="no-referrer"></p> 
<p style="color:#595959; margin-left:0; margin-right:0; text-align:left">JPress 的模板拖拽设计功能，和目前市面上的拖拽设计功能完全不一样，JPress 拖拽的 “UI 组件” ，是来源于模板的提供，也就意味着：每套模板所拥有的 “UI组件” 是完全不一样的。这样极大的方便了模板的作者，模板作者完全根据自己的设计风格来提供 UI 组件，而也无需适配系统，使得 JPress 模板足够强大和灵活。</p> 
<p style="color:#595959; margin-left:0; margin-right:0; text-align:left">JPress 模板作者，只需要在现有模板的 html 上，添加一行代码：<code>#blockContainer('容器名称')#end</code>，即可让模板拥有了拖拽设计的能力。</p> 
<p style="color:#595959; margin-left:0; margin-right:0; text-align:left">在开发 JPress 模板的过程中，我们还编写了超高性能的 js 模板引擎、编写基于 bootstrap 拖拽设计的构建工具，也都已经开源：</p> 
<ul style="list-style-type:disc; margin-left:0; margin-right:0"> 
 <li> <p>1、fasty:  一个极快的 JavaScript 模板引擎。开源地址：https://gitee.com/fuhai/fasty</p> </li> 
 <li> <p>2、bsFormBuilder: 一个基于  Bootstrap (v4.x) 的表单构建工具。开源地址：https://gitee.com/fuhai/bsFormBuilder</p> </li> 
</ul> 
<h3 style="margin-left:0; margin-right:0; text-align:left">8、已有模块新增更多运营支持</h3>

<p style="color:#595959; margin-left:0; margin-right:0; text-align:left">在文章分类中，添加了 “是否推荐”、“是否置顶”、缩略图、装饰图 等功能。</p> 
<p><img height="700" src="https://oscimg.oschina.net/oscnet/up-42923a123ea5d0b362816dcaf82b6fac188.png" width="1080" referrerpolicy="no-referrer"></p> 
<p style="color:#595959; margin-left:0; margin-right:0; text-align:left">在文章中，添加了 “是否允许评论”，“是否推荐”，“是否指定”，“是否热门”，“是否头条”，“是否允许搜索”... 等辅助运营的细节功能优化。</p> 
<h3 style="margin-left:0; margin-right:0; text-align:left">9、安全问题进一步提升</h3> 
<p style="color:#595959; margin-left:0; margin-right:0; text-align:left">安全，我们视为 JPress 的生命。</p> 
<p style="color:#595959; margin-left:0; margin-right:0; text-align:left">JPress 的愿景是成为企业门户的首选系统，那么必然把产品安全作为重中之重。我们甚至把安全问题，提高到功能和用户体验之上。JPress v5 在安全方面有做了许多的增强和改进，此处不再细节举例....</p>

<h2 style="margin-left:0; margin-right:0; text-align:left"><span>JPress 企业版</span></h2> 
<p style="color:#595959; margin-left:0; margin-right:0; text-align:left">JPress 是基于 LGPL 开源协议进行开源的，许多企业想基于 JPress 二次开发而不受限于 LGPL 协议的限制，可以联系我们购买。</p> 
<p style="color:#595959; margin-left:0; margin-right:0; text-align:left">同时 JPress 企业版目前有许多市级政府门户、地方国有银行、国内顶级的上市汽车集团在使用，在安全方面经历了这些机构邀请的国内顶尖安全团队蹂躏，目前已经成为打不倒的小强，其中许多安全方面的优化也回馈到了开源版本。</p> 
<p style="color:#595959; margin-left:0; margin-right:0; text-align:left">另外，企业版还提供了 3 级等保评测、文章内部和公司部门绑定、基于 Flowable 工作流的文章发布审核流程、在线的审核流程拖拽设计... 等等功能，有需求的同学欢迎联系我们。</p>
