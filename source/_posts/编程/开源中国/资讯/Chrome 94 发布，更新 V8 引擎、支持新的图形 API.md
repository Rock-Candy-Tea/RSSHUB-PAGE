 <p>Chrome 浏览器如今每四个星期就会推出一个新版本，Chrome 94 已于今天（2021 年 9 月 22 日）正式到来，该版本的更新内容如下：</p> 
 <h3>新标签页上的 Google Drive 文件</h3> 
 <p><img alt="https://www.howtogeek.com/wp-content/uploads/2021/09/2021-09-20_11-31-42.png?trim=1,1&bg-color=000&pad=1,1" height="287" src="https://www.howtogeek.com/wp-content/uploads/2021/09/2021-09-20_11-31-42.png?trim=1,1&bg-color=000&pad=1,1" width="700" referrerpolicy="no-referrer"></p> 
 <p>今年 Chrome 浏览器在新标签页中加入了 Cards（卡片）设计。起初，它们会包括菜谱和购物建议等内容。在 Chrome 94 上新增了能够快速查看 Google Drive 文件的快捷方式。</p> 
 <p>新的卡片被称为 "From your Google Drive"，它显示了最近打开的三个文档，该功能可以手动关闭。</p> 
 <h3>HTTPS 变得更加突出了</h3> 
 <p>Chrome 90 将 HTTPS 作为默认选项，这意味着如果网站支持 HTTPS，它将加载 HTTPS。Chrome 94 通过 "HTTPS-First 模式" 使其更进一步。</p> 
 <p>HTTPS-First 模式试图将所有网页升级为 HTTPS。如果 Chrome 无法用 HTTPS 加载网页，在继续使用旧的 HTTP 标准之前，你会看到一个全屏警告。</p> 
 <h3>改进图形处理</h3> 
 <p>Chrome 94 开始测试一个新的 WebGPU API，允许 Web 开发者利用你的计算机的图形能力。WebGPU API 将对在浏览器中运行的游戏特别有用。</p> 
 <p>WebGPU API 与旧的 WebGL 和 WebGL2 API 有些不同。浏览器可以利用你的计算机的本地处理技术，其中包括苹果的 "Metal"、微软的 "Direct3D" 和 开源的 "Vulkan" 标准。</p> 
 <h3>Android 版 Chrome 获得 "Material You" 主题设计</h3> 
 <p><img alt="https://www.howtogeek.com/wp-content/uploads/2021/09/Screenshot_20210920-115817.png?trim=1,1&bg-color=000&pad=1,1" height="460" src="https://www.howtogeek.com/wp-content/uploads/2021/09/Screenshot_20210920-115817.png?trim=1,1&bg-color=000&pad=1,1" width="700" referrerpolicy="no-referrer"></p> 
 <p>Chrome 93 开始引入了 Android 12 的 Material You 主题。Chrome 94 版加入了更多新设计语言。</p> 
 <h3>桌面端的新共享菜单</h3> 
 <p><img alt="https://www.howtogeek.com/wp-content/uploads/2021/09/2021-09-20_11-42-53.png?trim=1,1&bg-color=000&pad=1,1" height="385" src="https://www.howtogeek.com/wp-content/uploads/2021/09/2021-09-20_11-42-53.png?trim=1,1&bg-color=000&pad=1,1" width="700" referrerpolicy="no-referrer"></p> 
 <p>一般情况下，移动端的应用程序有内置的分享工具，可以方便地与其他应用程序分享内容。Chrome 94 通过 "共享中心" 将类似的功能带到桌面。</p> 
 <p>启用该功能后，用户会在地址栏中得到一个新的共享图标。点击该图标会出现一些分享的快捷方式，包括复制链接、投屏，以及快速分享至 Facebook、Twitter 等社交媒体。</p> 
 <p>目前需要通过进入 <code>chrome://flags/#sharing-hub-desktop-app-menu</code> 来启用该功能。</p> 
 <h3>其他</h3> 
 <ul> 
  <li>Chrome DevTools 现在支持 80 多种语言</li> 
  <li>设备列表新增 Nest Hubs：现在可以选择在 Nest Hub 下查看网页样式与设计</li> 
  <li>新的复选框可以反转网络过滤器</li> 
  <li>Scrollbar-Gutter 属性：允许开发者在内容扩展时防止布局变化，并在不需要滚动时避免不必要的变化</li> 
  <li>Chrome 94 整合了 9.4 版本的 V8 JavaScript 引擎</li> 
 </ul>
