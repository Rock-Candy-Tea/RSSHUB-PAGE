                                                                                            <p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><span style="color:#333333">Semi Design 是现代、全面、灵活的设计系统和 UI 库，由字节跳动抖音前端与 UED 团队设计、开发并维护，是一款包含设计语言、React 组件、主题等开箱即用的中后台解决方案，可用于快速搭建美观的 React 应用。</span></p> 
<p style="color:#000000; margin-left:0; margin-right:0; text-align:start"><span style="color:#333333">目前 Semi Design 发布了 v2.2.2 版本，此版本带来以下变化：</span></p> 
<ul style="margin-left:0; margin-right:0"> 
 <li>【Fix】 
  <ul style="margin-left:0; margin-right:0"> 
   <li>修复 Transfer 在 type 是 groupList 场景下, title 属性传入 ReactElement 类型导致key-warning</li> 
   <li>修复 DatePicker 范围选择 preset 日期设置为 null 或 undefined，选择日期后面板没有关闭问题<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FDouyinFE%2Fsemi-design%2Fissues%2F338" target="_blank">#338</a></li> 
   <li>修复 dateRange 类型 DatePicker，triggerRender 传入时选择完日期面板没有关闭问题<span> </span><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FDouyinFE%2Fsemi-design%2Fissues%2F422" target="_blank">#422</a></li> 
   <li>修复 InputNumber 精度格式化在受控模式下不正确问题</li> 
   <li>修复 IconFastForward 拼写错误</li> 
  </ul> </li> 
</ul>
