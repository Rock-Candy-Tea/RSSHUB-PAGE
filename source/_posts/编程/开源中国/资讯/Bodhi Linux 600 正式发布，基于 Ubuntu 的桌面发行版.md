                                                        <p>有“菩提 Linux”之称的 Bodhi Linux <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.bodhilinux.com%2F2021%2F05%2F12%2Fbodhi-linux-6-0-0-released%2F" target="_blank">发布</a>了 6.0 版本。</p> 
<p>Bodhi Linux 6.0 基于 Ubuntu 20.04.2 LTS (Focal Fossa) 构建，主要变化包括：</p> 
<ul> 
 <li>Arc-Green 主题经过重大修改后，现已具备动画背景</li> 
 <li>更新的欢迎屏幕和其他多项调整</li> 
 <li>Bodhi Linux 6.0 登录页面现已具有访客功能</li> 
 <li>新增 Plymouth 主题</li> 
 <li>改进 Moksha 桌面环境并添加部分新特性</li> 
 <li>改进对非英语语言的支持，默认安装 gnome 语言工具</li> 
 <li>……</li> 
</ul> 
<p>自 <a href="https://www.oschina.net/news/114418/bodhi-linux-5-1-0-released" target="_blank">Bodhi Linux 5.1 </a>发布以来，其他值得注意的变更包括将 PcManFm 替换为定制版本的 Thunar。此版本的 Thunar 现在支持在 Moksha/Enlightenment 桌面上设置背景图片。除了 Thunar 补丁外，新版本还修补了 Leafpad 和 ePhoto。Leafpad 修复了截断文件的问题，这个问题导致它被从 Debian 和 Ubuntu 软件仓库中删除。ePhoto 被打了补丁，以修复它在加载不在“用户主目录”文件夹中的图片时出现的问题。另一个值得注意的变化是采用 Chromium 作为新的默认浏览器，</p> 
<p>此外，过去使用 Bodhi Linux 的用户可能还会注意到，新版本已经用开发团队创建的自定义脚本替换了 apturl-elm，此脚本现在使用 policy-kit 和 synaptic 作为后端，默认情况下会禁用 snaps 安装。</p> 
<p><a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fwww.bodhilinux.com%2F2021%2F05%2F12%2Fbodhi-linux-6-0-0-released%2F" target="_blank">下载地址和详细更新内容介绍点此查看</a>。</p> 
<p>Bodhi Linux 是面向桌面的基于 Ubuntu 的发行，其特色在于优雅及轻量级的 Enlightenment 窗口管理器。该项目从 Enlightenment 的软件开发仓库中直接集成并预配置最新的 Enlightenment，提供了模块特性、高度的可定制性及主题选择。缺省的 Bodhi 系统很小巧，仅有的预安装程序是 Midori、Terminology、EFM (Enlightenment File Manager)、ePhoto 及 ePad，而更多的软件可通过一个基于网页的、称为 Bodhi 软件中心的软件安装工具获取。</p>
