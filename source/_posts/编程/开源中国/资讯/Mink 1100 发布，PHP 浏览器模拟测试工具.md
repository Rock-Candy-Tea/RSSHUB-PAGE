                                                                                            <p>Mink 是开源的 PHP 浏览器/控制器 模拟测试工具。Mink 可以模拟 Web 应用和浏览器之间的交互，测试 Web 应用的行为是否正确。</p> 
<p style="color:#333333; margin-left:0; margin-right:0; text-align:left">Mink 1.10.0 现已发布，此版本更新内容如下：</p> 
<p style="text-align:start"><strong><span><span><span style="color:#24292f"><span><span><span><span><span><span><span><span><span><span><span><span style="background-color:#ffffff"><span><span><span>新特性：</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></strong></p> 
<ul> 
 <li>增加了对<code>symfony/css-selector</code>6的支持</li> 
 <li>添加<code>WebAssert::elementAttributeNotExists</code></li> 
</ul> 
<p style="text-align:start"><strong><span><span><span style="color:#24292f"><span><span><span><span><span><span><span><span><span><span><span><span style="background-color:#ffffff"><span><span><span>删除：</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></strong></p> 
<ul> 
 <li>删除了对 PHP <7.2 的支持</li> 
 <li>删除了对<code>symfony/css-selector</code><4.4 的支持</li> 
</ul> 
<p style="text-align:start"><strong><span><span><span style="color:#24292f"><span><span><span><span><span><span><span><span><span><span><span><span style="background-color:#ffffff"><span><span><span>Bug 修复：</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></strong></p> 
<ul> 
 <li>修复了 phpdoc 中一堆参数没有正确标记为 nullable 的问题</li> 
</ul>
