                                                                                            <p>Budgie Desktop 10.6 引入了对应用程序分组的改进、对单实例应用程序跟踪的改进、对某些KDE 应用程序的更好支持，以及经过改进的通知系统，通知系统现在是独立的，不再依赖 Raven 通知中心。</p> 
<p>此外，Budgie 10.6 对内部的主题和面板添加了各种改进，例如大多数对话框的边框半径、Raven 小部件和面板弹出内容，面板内小部件的统一配色方案，更好地支持 GTK 主题，提高了可读性，更精确的电池电量图标，时钟小程序改进，以及更新的默认面板配置。（如下图，但是俺真看不出啥区别..）</p> 
<p><img height="658" src="https://oscimg.oschina.net/oscnet/up-1b29cee95561a3b0620925fcd2746fb05b9.png" width="700" referrerpolicy="no-referrer"></p> 
<p>除以上更改外，此版本还将所有 GNOME 控制中心引用重命名为 Budgie 控制中心，重命名 Budgie Desktop View 架构，修复 Raven 的各种问题，并解决用户在以前版本中报告的其他错误。</p> 
<p>完整的更新公告可在 GitHub <a href="https://www.oschina.net/action/GoToLink?url=https%3A%2F%2Fgithub.com%2FBuddiesOfBudgie%2Fbudgie-desktop%2Freleases%2Ftag%2Fv10.6" target="_blank">发行公告</a>中查看。</p>
