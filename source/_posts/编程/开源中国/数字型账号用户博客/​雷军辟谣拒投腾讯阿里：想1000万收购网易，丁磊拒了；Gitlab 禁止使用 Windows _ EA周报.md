     <p style="margin: 5px 16px;line-height: 2em;"><span style="letter-spacing: 2px;"><strong style="color: rgb(74, 148, 231);"><span style="font-size: 16px;">AppLovin 开价 175 亿美元收购 Unity</span></strong></span></p> 
     <p style="margin: 5px 16px;line-height: 2em;"><span style="font-size: 15px;letter-spacing: 2px;">游戏软件公司 AppLovin 向同行 Unity Software 提交收购要约，开价 175.4 亿美元，全部以股票完成交易。之前 Unity 已经宣布要收购 AppLovin 的竞争对手 ironSource，这笔交易可能会告吹。AppLovin 开价每股 58.85 美元收购 Unity，比 Unity 周一的收盘价高出 18%。合并之后 Unity 将会持有新公司流通股的 55%，相当于拥有 49% 的投票权。上个月 Unity 宣称要以 44 亿美元收购 ironSource，随后 AppLovin 聘请顾问提交自己的收购要约。如果 Unity 想与 AppLovin 合并，公司董事会可能会终止 ironSource 交易。（IT之家）</span></p> 
     <p style="margin: 5px 16px;line-height: 2em;"><br></p> 
     <p style="margin-top: 5px;margin-right: 16px;margin-left: 16px;line-height: 2em;"><span style="letter-spacing: 2px;"><strong style="color: rgb(74, 148, 231);"><span style="font-size: 16px;">百度无人驾驶网约车起步价 16 元，里程单价2.8元/公里</span></strong></span></p> 
     <p style="margin-top: 5px;margin-right: 16px;margin-left: 16px;line-height: 2em;"><span style="font-size: 15px;letter-spacing: 2px;">百度周一表示，已获得重庆和武汉的批准，可以向乘客提供全无人驾驶网约车有偿服务，这些网约车没有司机或安全人员。百度是首家获得此类许可的中国公司。该公司在一份声明中表示，将在武汉指定的13平方公里范围内从上午 9 点到下午 5 点提供这项服务，并在重庆的 30 平方公里范围内从上午 9 点半到下午 4 点半提供此服务。目前百度 " 萝卜快跑 " 已在部分地区推出自动驾驶付费出行服务，以里程和时长作为计费单位，起步价 16 元，里程单价 2.8 元 / 公里。有网友表示，这也太贵了。</span></p> 
     <p style="margin-top: 5px;margin-right: 16px;margin-left: 16px;line-height: 2em;"><br></p> 
     <p style="margin-top: 5px;margin-right: 16px;margin-left: 16px;line-height: 2em;"><strong style="letter-spacing: 1px;color: rgb(74, 148, 231);"><span style="letter-spacing: 2px;font-size: 16px;">员工在下班后用微信工作期间发病，认定工伤</span></strong></p> 
     <p style="margin-top: 5px;margin-right: 16px;margin-left: 16px;line-height: 2em;"><span style="font-size: 15px;letter-spacing: 2px;">日前，广州铁路运输中级法院二审的判决引发关注。</span><span style="font-size: 15px;letter-spacing: 2px;">在案件中，广州某贸易公司的员工石某在下班后使用微信继续工作，在工作期间不幸突发疾病，最终身亡，被二审法院认定为工伤。</span><span style="font-size: 15px;letter-spacing: 2px;">二审法院指出，石某的微信聊天记录显示，其经常下班后用微信回复工作信息。</span><span style="font-size: 15px;letter-spacing: 2px;">可见，石某回家后继续处理工作是常态且具有连贯性，属于《工伤保险条例》中规定的「工作时间和工作岗位」的延伸。</span><span style="font-size: 15px;letter-spacing: 2px;">因此，在此期间突发疾病死亡的，应当视同工伤。</span><br></p> 
     <p style="margin-top: 5px;margin-right: 16px;margin-left: 16px;line-height: 2em;"><span style="font-size: 15px;letter-spacing: 2px;">对此，深圳市总工会官方发文表示：</span><span style="font-size: 15px;letter-spacing: 2px;">“微信等具备实时发送文字、语音、图片、视频功能的通讯工具低成本、高效率，职工可以随时随地在线处理工作，一定程度上打破了传统的线下坐班、处理工作的时空限制。</span><span style="font-size: 15px;letter-spacing: 2px;">因此，对于相关法律法规的理解，也应当立足于当前社会经济发展状况，作出公平公正、兼顾劳动者权益的解释。</span><span style="font-size: 15px;letter-spacing: 2px;">”（爱范儿）</span></p> 
     <p style="margin-top: 5px;margin-right: 16px;margin-left: 16px;line-height: 2em;"><span style="font-size: 15px;letter-spacing: 2px;"><br></span></p> 
     <p style="margin-top: 5px;margin-right: 16px;margin-left: 16px;line-height: 2em;"><strong style="letter-spacing: 1px;color: rgb(74, 148, 231);"><span style="letter-spacing: 2px;font-size: 16px;">周鸿祎否认360不能卸载：提供完整卸载过程</span></strong></p>

     <p style="margin-top: 5px;margin-right: 16px;margin-left: 16px;line-height: 2em;"><strong style="letter-spacing: 1px;color: rgb(74, 148, 231);"><span style="letter-spacing: 2px;font-size: 16px;"><br></span></strong></p> 
     <p style="margin-top: 5px;margin-right: 16px;margin-left: 16px;line-height: 2em;"><strong style="letter-spacing: 1px;color: rgb(74, 148, 231);"><span style="letter-spacing: 2px;font-size: 16px;">Argues JSON 作者认为应该退役 JavaScript</span></strong></p> 
     <p style="margin-top: 5px;margin-right: 16px;margin-left: 16px;line-height: 2em;"><span style="font-size: 15px;letter-spacing: 2px;">Argues JSON 作者 Douglas Crockford 认为最流行的编程语言 JavaScript 如今已成为进步的障碍应将其退役，但他同时承认目前不存在 JS 语言的替代。根据 StackOverflow 的调查，逾 65% 的开发者使用 JavaScript，第二名的 Python 只有 48%。Crockford 认为 JS 的问题是它日益臃肿，社区对增加功能而不是修正其缺陷更感兴趣，JS 和其它恐龙语言一样成为了进步的障碍。Crockford 称取代 JS 有两大难点：第一是还不存在 JS 替代品，第二需要所有浏览器都采用它。结果只能是“祝好运”了。（Solidot）</span></p>

     <p style="margin: 5px 16px;line-height: 2em;"><span style="font-size: 15px;letter-spacing: 2px;">Gitlab 被发现有一项禁止使用 Microsoft Windows 的公司政策。微软旗下的 GitHub 是 Gitlab 最主要的竞争对手。Gitlab 表示它允许员工使用 Linux 和苹果 macOS，而禁止使用 Windows 的理由是它是份额最高的桌面系统，因此也是间谍软件、病毒和勒索软件的最主要目标。Windows 家庭版的安全性难以保障，因此不符合公司安全准则。苹果的电脑是首选，如果熟悉 Linux 也可选择戴尔公司的 Linux 笔记本电脑。（Solidot）</span></p> 
     <p style="margin: 5px 16px;line-height: 2em;"><strong style="letter-spacing: 0.5px;color: rgb(74, 148, 231);"><span style="letter-spacing: 2px;font-size: 16px;"><br></span></strong></p> 
     <p style="margin: 5px 16px;line-height: 2em;"><strong style="letter-spacing: 0.5px;color: rgb(74, 148, 231);"><span style="letter-spacing: 2px;font-size: 16px;">以全栈信创产品，安全赋能智慧能源！普元出席中国核电信息技术高峰论坛</span></strong></p> 
     <p style="margin: 5px 16px;line-height: 2em;"><span style="font-size: 15px;letter-spacing: 2px;">近日，第十届中国核电信息技术高峰论坛（NITF 2022）在浙江海盐成功召开。享有“亚洲首个专注于核电信息技术发展的交流平台”之誉的论坛，现场汇聚了多领域专家深入探讨“智慧核电 助力双碳 赋能未来”。作为值得客户信赖的基础软件技术公司和信创产业链的关键成员，普元受邀出席论坛。普元数智研究院资深顾问左磊，结合企业数字化转型与行业信创发展趋势，发表题为《打造企业数据资源平台 提升企业数字化运营能力》的主题演讲，与现场专家交流普元基于全栈信创中间件产品的领先优势，为客户提供安全可靠的数据管理与治理产品与解决方案，从源头解决信创应用迁移的数据问题，以普元数据治理运营一体化方法赋能核电企业智慧转型的实际应用案例。  </span></p>
