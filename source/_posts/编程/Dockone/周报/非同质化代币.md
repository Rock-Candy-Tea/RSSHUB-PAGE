<br>【编者的话】以太坊官方于近期在他们的官方文档里上传了一篇介绍 nft（ non-fungible tokens ）的文章，本文翻译自该篇文章。<br>
<br>NFT 目前正席卷数字艺术和收藏品领域。数字艺术家们正亲眼见证着他们的生活发生着翻天覆地的变化，这要归功于新上线的 crypto-audiences 的大卖。一些知名人士也纷纷加入进来，他们发现了与粉丝交流的新契机。但是数字艺术仅仅只是使用 NFT 的其中一种方式。实际上，它们可以用来表示任何独特资产的所有权，比如数字世界或物理王国里持有某项物品的凭据。倘若 Andy Warhol 是 90 后的话，他可能会把 Campbell's Soup 制作成一个 NFT。Kanye 把 Yeezys 投放到以太坊上也只是时间问题。而且，终有一天你可以通过一个 NFT 来证明对自己汽车的所有权。<br>
<br><h2>什么是 NFT ? &#123;#what-are-nfts&#125;</h2>NFT 是一套可以用来代表一些独一无二的商品的所有权的 token。NFT 让我们可以将艺术品、收藏品甚至房地产之类的东西代币化。它们在同一时间只能有一位正式的主人，并且受到以太坊区块链的保护 —— 没有人可以修改所有权记录或者根据现有的 NFT 复制粘贴一份新的。<br>
<br>NFT 代表的即是不可替代的 token。不可替代品是一个经济学术语，你可以用来描述诸如家具、一份歌曲文件，又或者是你的电脑。这些东西无法和其他东西互换，因为它们具有一些独特的属性。<br>
<br>另一方面，可替代商品，它们是可以互换的，因为它们是由其价值定义的，而非某些独特的属性。举个例子，ETH 或者美元是可以互换的，因为 1 ETH 或者 1 美元可以换成另外一个 ETH / 1美元。<br>

<br><h2>资产互联网 &#123;#internet-of-assets&#125;</h2>NFT 和以太坊解决了当今互联网存在的一些问题。随着一切变得更加数字化，我们有必要复制实物的一些属性，比如它们的稀缺性，唯一性，以及所有权证明。更不用说数字产品通常仅在其产品范围内有用。比如，你无法转售你已经购买的一份 iTunes mp3 给别人，又或者说，即便有市场，你也无法将一家公司的忠诚度积分兑换成其他平台的信用额度。<br>
<br>与我们大多数人今天使用的互联网相比，这便是一个拥有 NFT 的互联网的样子...<br>

<br><h2>NFT 的工作原理是什么? &#123;#how-nfts-work&#125;</h2>NFT 具有一些特殊的属性：<br>
<ul><li><br>每个铸造的 token 都有一个唯一的标识符。</li><li><br>它们不能与其他 token 1:1 直接互换。例如 1 个 ETH 和其他 ETH 没什么两样。NFT 却并非如此。</li><li><br>每个 token 都有一个持有者，而且其他人可以轻松地验证这一信息。</li><li><br>它们依托于 Ethereum，可以在任意基于以太坊的 NFT 交易市场上买卖。</li></ul><br>
<br>换句话说，如果你拥有一个 NFT：<br>
<ul><li><br>你可以轻松地证明你拥有它。</li><li><br>没有人能够以任何方式操纵它。</li><li><br>你可以出售它，在某些情况下，这可以让原创作者赚到转售使用费。</li><li><br>又或者，你可以一直持有它，只要知道自己的资产是由以太坊上的钱包保护，那便可以放心地睡下。</li></ul><br>
<br>再者，如果你创造了一个 NFT：<br>
<ul><li><br>你可以轻松地证明自己是创作者。</li><li><br>你可以决定它的稀缺性。</li><li><br>你可以在每次出售时赚取使用费。</li><li><br>你可以在任何 NFT 市场或者私下里卖掉它。你不会被任何平台绑架，也不需要任何中介。</li></ul><br>
<br><h3>稀缺性 &#123;#scarcity&#125;</h3>NFT 的作者可以决定他们资产的稀缺性。<br>
<br>举个例子，想想一次体育赛事的一张门票。就像一个活动的组织者可以决定出售多少张票一样，NFT 的作者也可以决定有多少份副本。有时候它会是一个确定的数字，比如 5000 张通用门票。有时候则会是铸造一些非常相似，但是又略有不同的东西，比如带有指定座位的车票。在某种情况下，创作者可能想要创造一个 NFT，其中只有一个被当成是一个特殊的稀有收藏品。<br>
<br>以上情况，每个 NFT 仍将只有一个持有者，而且拥有一个唯一的标识符（如传统"票据"上的条形码）。NFT 的潜在稀缺性很关键，而且这取决于创作者。创作者可能打算让每个 NFT 都是完全唯一的，以满足其稀缺性，或者他们也有理由生产数千个副本。记住，所有这些信息都是公开的。<br>
<br><h3>特许权使用费 &#123;#royalties&#125;</h3>一些 NFT 在出售时会自动向其创作者支付版权费。这仍然是一个待完善的概念，但是它是最强有力的概念之一。每次出售 NFT 时，<a href="https://eulerbeats.com/">EulerBeats Originals</a> 的最初持有者都会获得 8% 的特许权使用费。还有一些平台，像是 <a href="https://foundation.app/">Foundation</a> 以及 <a href="https://zora.co/">Zora</a>，支持帮助他们的艺术家们收取版税。<br>
<br>这是完全自动化的，他们的作品会在人们手上流转，因此创作者们只需坐下来赚取版税即可。现如今，弄清楚版税是一个体力活，而且缺乏准确性 —— 许多创作者都没有得到应有的报酬。而如果你的 NFT 已经编入版税，那你永远都不会错过。<br>

<br><h3>为创作者带来更大的收益 &#123;#nfts-for-creators&#125;</h3>今天，NFT 的最大用途是在数字内容领域。这是因为现如今该行业已经走向崩坏。内容创作者眼看着他们的利润和获利潜力通通都被平台吃掉了。<br>
<br>一位艺术家在一个社交网络平台上发布作品可以为平台带来收入，平台会向艺术家的关注者们兜售广告。作为回报，他们可以得到相应的曝光率，但是曝光率并不能带来真金白银。<br>
<br>NFT 为新的创作者经济提供了动力，在这种经济生态里，创作者无需再将他们内容的所有权移交给他们用来发布内容的平台。所有权自然而然地融入到了作品本身。<br>
<br>当他们出售内容时，钱会直接打给他们。如果新的所有者随即便把 NFT 卖出了，那么最初的持有者甚至还可以自动获得版税。由于创作者的地址是 token 元数据（无法修改的元数据）的一部分，因此每次交易都可以保证这一点。<br>

<br><h4>复制粘贴问题 &#123;#nfts-copy-paste&#125;</h4>反对者们常常提及这样一件事实，他们一般会拿出一个 NFT 艺术品截图的图片来证明 NFT 是 "愚蠢至极" 的。"瞧，现在我免费得到那张图片了！"他们自鸣得意地说道。<br>
<br>嗯，是的。但是，在谷歌搜索到的毕加索的《格尔尼卡》的图片是否可以让你骄傲地成为这件在艺术史上价值数百万美元的画作的新主人呢？最终得到的真迹，它的价值也即等同于市场为它所带来的。一件作品的内容越是被人截屏、分享和广泛使用，那么它所获得的价值也就越大。<br>
<br>拥有可供鉴定的真迹总会比赝品更有价值。<br>
<br><Divider /><br>
<br><h3>提升游戏的潜力 &#123;#nft-gaming&#125;</h3>NFT 已经引起了游戏开发者的极大兴趣。NFT 可以提供游戏里物品的的所有权记录，促进游戏内的经济发展，而且能够为玩家带来很多好处。<br>
<br>在许多大众游戏里，玩家可以购买物品以供他们在游戏里使用。但是，如果该物品是一个 NFT 的话，你便可以在游戏结束后通过售卖回点本钱。如果该物品增值了的话，你甚至还可以额外赚一笔。<br>
<br>对于游戏开发者（作为 NFT 的发行者），每次在公开市场上重新出售某项物品时他们都可以赚到一笔版税。这就创造了一种更加互惠互利的商业模式，玩家和开发商都能从二手的 NFT 市场上获利。<br>
<br>这也就意味着，如果开发商不再维护该游戏了，玩家收集的物品仍将归他所有。<br>
<br>最终，你在游戏里打造的物品也可以超过游戏本身的寿命。即便游戏不再维护了，你的物品也将始终处于你的控制之下。这意味着游戏里的物品将会成为数字纪念品，而且在游戏之外也有价值。<br>
<br>一款虚拟现实游戏，Decentraland，甚至允许玩家购买代表虚拟土地的 NFT，你可以按需使用这些土地。<br>

<br><h3>让以太坊地址更好记 &#123;#nft-domains&#125;</h3>以太坊名字服务借助 NFT 帮助你的以太坊地址提供一个容易记忆的名字，例如像 <code class="prettyprint">mywallet.eth</code> 这种。这意味着你可以要求某人通过 <code class="prettyprint">mywallet.eth</code> 而不是 <code class="prettyprint">0x123456789.....</code> 转给你 ETH。<br>
<br>这和一个网站域名的工作原理类似，它会让一个 IP 地址更加容易记忆。而且，像域名那样，ENS 名字通常根据长度和相关性确定其价值。使用 ENS，你不再需要一个域名注册表来促成所有权的转让。取而代之的是，你可以在一个 NFT 市场上交易你的 ENS 名字。<br>
<br>你的 ENS 名字可以:<br>
<ul><li><br>接收加密货币以及其他 NFT。</li><li><br>指向一个去中心化的网站，例如 <a href="https://ethereum.eth.link/">ethereum.eth</a>。<a href="https://docs.ipfs.io/how-to/websites-on-ipfs/link-a-domain/#domain-name-service-dns">有关去中心化网站的更多信息</a></li><li><br>存放任何信息，包括个人资料信息，例如电子邮件地址以及推特用户名。</li></ul><br>

<br><h3>实物 &#123;#nft-physical-items&#125;</h3>实物的代币发展并不如他们的数字孪生兄弟。但是有许多项目在探索房地产、独特的时尚物品等实物的代币化。<br>
<br>由于 NFT 本质上是一张契约，因此，终有一天你将可以使用 ETH 购买一辆汽车或一栋房子，然后以 NFT 的形式拿到契约凭据（在同一笔交易里）。随着事情变得越来越高科技，人们不难想象这样的一个世界：以太坊钱包将会变成你的车子或者房子的钥匙 ———— 加密的所有权证明可以用来开锁。<br>
<br>随着像汽车和财产这类贵重资产得以在以太坊上标识，你可以将 NFT 用作去中心化贷款的抵押品。如果你手上没有现金或者没多少加密货币，但是拥有一些有价实物的话，NFT 应该会挺有用处。<a href="https://ethereum.org/en/defi/">有关 DeFi 的更多信息</a><br>
<br><Divider /><br>
<br><h3>NFT 和 DeFi &#123;#nfts-and-defi&#125;</h3>NFT 世界和<a href="https://ethereum.org/en/defi/">去中心化金融(DeFi)</a>的世界正开始以一系列有趣的方式合作。<br>
<br><h4>NFT 支持的贷款 &#123;#nft-backed-loans&#125;</h4>现如今有一些 DeFi 应用允许用户通过抵押来借钱。比如，你抵押了 10 ETH，那么你便可以借入 5000 DAI (<a href="https://ethereum.org/stablecoins/">一种稳定币</a>)。这可以确保放款人得到偿还 —— 如果借款人不偿还 DAI 的话，抵押品将被转给放款人。然而，并不是所有人都有足够的加密货币用作抵押品。<br>
<br>有些项目已经开始探索使用 NFT 作为替代的抵押品。试想一下，有一天你买了一个稀有的 CryptoPunk NFT 回来 —— 它们以今天的价格能够卖到 1000 多美元。通过将其作为抵押品，你可以使用相同的一组规则借贷。如果你无法还清 DAI 的话，你的 CryptoPunk 将会被当成抵押品转给借贷方。这可以最终与标记为 NFT 的任何物品一起使用。<br>
<br>再者，在以太坊上这个实现起来并不困难，因为 NFT 和 DeFi 这两个世界采用的是同一套基础设施。<br>
<br><h4>部分所有权 &#123;#fractional-ownership&#125;</h4>NFT 创作者还可以为其 NFT 制作 "股票"。这让投资者和粉丝有机会拥有 NFT 的一部分，而不必购买整个产品。这为 NFT 铸造者和收藏家提供了更多的机会。<br>
<ul><li><br>部分产权化的 NFT 可以在 <a href="https://ethereum.org/en/defi/#dex">DEX</a> 上像 Uniswap 那样交易，而不仅仅只是在 <a href="https://ethereum.org/dapps?category=collectibles">NFT 市场</a>。这就意味着可以有更多的买家和卖家。</li><li><br>NFT 的整体价格可以由其部分的价格来定义。</li><li><br>你可以有更多的机会拥有你关注的物品以及从中获利。拥有 NFT 是很难定价的。</li></ul><br>

<br><h2>以太坊和 NFT &#123;#ethereum-and-nfts&#125;</h2>以太坊可以玩转 NFT 的原因有很多：<br>
<ul><li><br>交易历史和 token 元数据是公开可验证的 —— 证明所有权的历史不再困难。</li><li><br>一旦交易确认了，就几乎不可能操纵该数据来 "窃取" 所有权。</li><li><br>NFT 的交易可以是点对点的，无需利用平台中介，如果走平台的话，它们会大量克扣作为补偿。</li><li><br>所有以太坊的产品都共用一套相同的 "后端"。换句话说，所有以太坊的产品都可以轻松地相互理解 —— 这使得 NFT 可以跨产品移植。你可以在一个产品上购买一个 NFT，然后在另一种产品上轻松出售。作为一个创作者，你可以同时在多个产品上列出你的 NFT —— 每个产品都将拥有最新的所有权信息。</li><li><br>以太坊永远不会崩溃，这意味着你的代币将始终可以出售。</li></ul><br>
<br><h2>NFT 对环境的影响 &#123;#environmental-impact-nfts&#125;</h2>NFT 越来越受欢迎，这意味着它们也受到了越来越多的审查 —— 尤其是在碳足迹方面。<br>
<br>这里我们要阐明一些事情：<br>
<ul><li><br>NFT 并没有直接增加以太坊的碳足迹。</li><li><br>以太坊保证用户的资金和资产的安全性的途径目前是能源密集型的，但是这一情况将会得到改善。</li><li><br>一旦情况得到改善，以太坊的碳足迹将会得到 99.98% 以上的改进，这足以使它比现有许多行业更加节能。</li></ul><br>
<br>为了进一步解释，我们将需要更多的技术知识，不妨耐心听我们讲下去...<br>
<br><h3>不要将其归咎于 NFT &#123;#nft-qualities&#125;</h3>整个 NFT 生态系统之所以起作用，就是因为以太坊本身是去中心化的，而且是安全的。<br>
<br>去中心化意味着你和其他所有人都可以验证你对某件东西的所有权。所有这一切并不需要信任或授权给任何可以随意实施他们自己的规则的第三方机构来保管。这也意味着你的 NFT 可以在许多不同的产品和市场间流转。<br>
<br>安全意味着没人可以复制/粘贴你的 NFT 或者窃取它。<br>
<br>以太坊的这些品质使得这样以数字化的形式拥有某些独一无二的物品，以及给自己创作的内容拿到一个合理的价格的这些做法成为可能。但是这也是有代价的。像比特币和以太坊这样的区块链产品目前是能源密集型的，因为要保持这些品质需要消耗大量的能量。如果能够简单通过重写以太坊的历史来窃取 NFT 或者加密货币的话，整个系统就会崩溃。<br>
<br><h4>铸造 NFT 的工作 &#123;#minting-nfts&#125;</h4>当你铸造一个 NFT 时，这里会有一些事情发生：<br>
<ul><li><br>它需要被确认为一个链上资产。</li><li><br>持有者的账户余额将会得到更新以包含该项资产。这让随后把它卖出去或者鉴定为自己 "所有" 这些事情成为可能。</li><li><br>确认上述内容的交易需要被添加到区块里，并在链上 "永存"。</li><li><br>网络中的每个人都需要将该块确认为 "正确"。这种共识消除了人们对中介机构的需求，因为整个网络同意你的 NFT 存在并且属于你。而且它已经上链，因此任何人都可以检查它。这是以太坊帮助 NFT 创作者获得最大收益的方式之一。</li></ul><br>
<br>所有这些任务都是由矿工完成的。他们让整个网络的其余部分知晓了你的 NFT 的存在以及谁拥有它。这也就意味着挖矿必须是足够困难的，否则任何人都可以声称自己拥有你刚刚铸造的 NFT 然后利用欺诈的手段转走所有权。有很多激励措施可以确保矿工诚实行事。<br>
<br><a href="https://ethereum.org/en/developers/docs/consensus-mechanisms/pow/">有关挖矿的更多信息</a><br>
<br><h4>通过挖矿确保 NFT 的安全 &#123;#securing-nfts&#125;</h4>挖矿的难度取决于以下事实：在链上创造一个新的区块需要消耗大量的算力。重要的是，链上会不断产生新的区块，而不仅仅只是在需要它们的时候。它们每隔 12 秒左右出一次块。<br>
<br>这对于以太坊的防篡改机制来说至关重要，它也是使得 NFT 成为可能的品质之一。块越多，链就越安全。如果你的 NFT 在 #600 区块上创建的，然后黑客试图通过修改其数据来窃取你的 NFT，那么所有后续区块的数字指纹都会改变。这就意味着任何运行以太坊软件的人都将立即检测到并阻止它的发生。<br>
<br>然而，这也意味着我们需要持续不断地消耗算力。这也意味着一个包含 0 次 NFT 交易的区块仍将耗费大致相同的碳足迹，因为创建它仍然会消耗算力。其他非 NFT 的事务将填补区块上的这一空白。<br>
<br><h4>区块链是能源密集型的，就目前来说 &#123;#blockchains-intensive&#125;</h4>因此，是的，通过挖矿创建区块会产生碳足迹 —— 而这对于像比特币这样的链来说也是一个问题 —— 但是直接责任不在 NFT 。<br>
<br>许多矿工使用的是可再生能源或是一些偏远地区里未经开发的能源。还有一种观点认为，NFT 以及加密货币正在推翻的行业，其本身也有大量的碳足迹。但是，仅仅因为现有行业的糟糕做法，并不意味着我们不应该努力去做得更好。<br>
<br>然后，我们的确有在做。以太坊正在不断发展，致力于让以太坊（同时也就意味着 NFT ）变得更加节能。这一直是我们计划里的一部分。<br>
<br>这里我们并不是想要为挖矿对环境造成的影响作出什么辩解，而是想解释一下，一切将会如何变得更好。<br>
<br><h3>更绿色的未来... &#123;#a-greener-future&#125;</h3>自从以太坊问世以来，挖矿所带来的的能耗问题一直是开发人员和研究人员关注的重点领域。我们一直以来的愿景则是尽快将它替换掉。<a href="https://ethereum.org/en/eth2/vision/">有关以太坊的愿景的更多信息</a><br>
<br>这一愿景目前正在变为现实。<br>
<br><h4>一个更绿色的以太坊: Eth2 &#123;#eth2&#125;</h4>以太坊目前正在进行一系列升级，我们将其称之为 Eth2，它将会用 <a href="https://ethereum.org/en/eth2/staking/">staking</a>（质押）的手段代替挖矿。这将会把算力作为安全机制的方式移除，因此也可以将以太坊的碳足迹减少约 99.98%<sup>[^1]</sup>。在这个世界里，投资者投入资金而不是算力来保护整个网络。<br>
<br>以太坊的能耗成本将会变成运行家用计算机的成本乘以整个网络中节点的数量。如果网络中有 10000 个节点，而运行一台家用计算机的成本约为每年 525kWh。那么整个网络每年将需要 5,250,000kWh<sup>[^1]</sup>。<br>
<br>我们可以据此拿 Eth2 和一个像 Visa 这样的全球服务来做个对比。十万次 Visa 交易将会消耗 149kWh 能量<sup>[^2]</sup>。在 Eth2里，相同数量的交易将会耗费 17.4kWh 能量，约占总共<sup>[^3]</sup>的 11%。这还没考虑与 Eth2 并行进行的许多优化，比如像 <a href="https://ethereum.org/en/glossary/#rollups">rollups</a>。10万次交易可能只会用掉 0.1666666667kWh 能量。<br>
<br>重要的是，这在维持以太坊的去中心化和安全性的同时提高了能效。市面上其他一些区块链可能已经率先使用了某种形式的质押机制，但是它们是由少数几个利益相关者来保护的，以太坊拥有的则是数以千计的支持者。越是去中心化，整个系统就越安全。<br>

<br>_我们已经给出了一份和 Visa 的基本比较，以一个熟悉的名字就 Eth2 的能耗方面给出一份基准理解。然而，实际上，根据交易次数进行比较并不太准确。以太坊的能量输出是基于时间的。哪怕以太坊在这一分钟到下一分钟做或多或少的交易，能量输出将依旧保持不变。_<br>
<br>_还有一个要记住的重点，那就是，以太坊不仅能够完成金融交易，它还可以作为应用程序的平台，因此我们不妨拿它和包括 Visa，AWS等在内的更多企业/行业进行更为公平的比较吧！_<br>

<br><h4>1. 与挖矿相比减少 99.98% 的能耗 &#123;#fn-1&#125;</h4>根据如下数据源可以计算出从通过挖矿保护的系统转变到质押保护的系统，能耗降低了 99.98%：<br>
<ul><li><br>以太坊挖矿每年要消耗 24 tWh 电能 - <a href="https://digiconomist.net/ethereum-energy-consumption">Digiconomist</a></li><li><br>普通的一台台式机，运行权益证明所需的全部资源，每小时大概消耗 0.06 kWh 的能量 - <a href="https://www.siliconvalleypower.com/residents/save-energy/appliance-energy-use-chart">硅谷功率表</a>（某些地方估算值更高，为 0.15 kWh）</li></ul><br>
<br>以上计算方式假定运行以太坊的用户从不让电脑进入睡眠状态或是关闭他们的显示器。<br>
<br>日常使用量：<code class="prettyprint">0.06kWh * 24 = 1.44 kWh</code> 每人每天<br>
<br>年度使用量：<code class="prettyprint">1.44 kWh * 365 = 525.6 kWh</code> 每人每天<br>
<br>如果有一万人运行以太坊（目前的数字为 8000 ish）的话，那么运行整个网络的成本为 <code class="prettyprint">10,000 nodes * 525.6kWh = 5,256,000 kWh</code>。<br>
5,256,000kWh 也即是 0.005256tWh。和挖矿带来的 24tWh 能耗相比，它是其所需能量的 0.0219%，换句话说就是减少了 99.98%。<br>
<br><h4>2. Visa 的能耗 &#123;#fn-2&#125;</h4>十万次 Visa 交易的成本是 149 kwH - <a href="https://www.statista.com/statistics/881541/bitcoin-energy-consumption-transaction-comparison-visa/">截止2020年，比特币网络每笔交易的平均能耗与 VISA 网络的对比，Statista</a><br>
<br>截止2020年9月的年度报告，他们共处理了 140,839,000,000 次交易 - <a href="https://s1.q4cdn.com/050606653/files/doc_financials/2020/q4/Visa-Inc.-Q4-2020-Operational-Performance-Data.pdf">Visa 2020年Q4财务报告</a><br>
<br><h4>3. Eth2 处理十万次交易的能耗情况 &#123;#fn-3&#125;</h4>据估算，Eth2 将允许整个网络每秒处理 25,000 到 100,000 次交易，<a href="https://ethereum-magicians.org/t/a-rollup-centric-ethereum-roadmap/4698">目前 100,000 是理论上的最大值</a>。<a href="https://twitter.com/VitalikButerin/status/1312905884549300224?ref_src=twsrc%5Etfw%7Ctwcamp%5Etweetembed%7Ctwterm%5E1312905886327664640%7Ctwgr%5E%7Ctwcon%5Es2_&ref_url=https%3A%2F%2Fwww.coinspeaker.com%2Fvitalik-buterin-ethereum-layer-2%2F">Vitalik Buterin 在 Eth2 上每秒潜在交易量</a>。<br>
<br>最起码，Eth2 允许的交易量会是今天的 64 倍，目前约有 15 笔交易。这也是引入分片链（额外的数据和容量）的数量。<a href="https://ethereum.org/en/eth2/shard-chains/">有关分片链的更多信息</a><br>
<br>这意味着我们可以估算出处理 100,000 次交易所需的时间，因此我们便能够将其和上述 Visa 的例子进行比较。<br>
<ul><li><br><code class="prettyprint">15 * 64 = 960</code> 次交易/秒。</li><li><br><code class="prettyprint">100,000 / 960 = 104.2</code> 秒，处理 100,000 次交易。</li></ul><br>
<br>在这 104.2 秒的时间里，以太坊网络将会消耗如下数量的能量：<br>
<br><code class="prettyprint">1.44kWh 日常使用 * 10,000 网络节点数 = 14,400kWh</code> 每天。<br>
<br>一天有 86,400 秒，因此也就是 <code class="prettyprint">14,400 / 86,400 = 0.1666666667kWh</code> 每秒。<br>
<br>如果我们将其乘以处理 100,000 次交易所需的时间：<code class="prettyprint">0.1666666667 * 104.2 = 17.3666666701 kWh</code>。<br>
<br>也就是相当于在 Visa 上进行相同次数交易所消耗的能量的 <strong>11.6554809866%</strong>。<br>
<br>别忘了，这是基于 Eth2 每秒能够处理的最小交易量而得出的数据。如果 Eth2 达到其每秒 100,000 次交易的潜力，那么 100,000 次交易将会只消耗 0.1666666667kWh 的能量。<br>
<br>换句话说，如果 Visa 以每 100,000 笔交易 149 kWh 能耗的成本处理了 140,839,000,000 笔交易的话，那么它一年的能耗便是 209,850,110 kWh。<br>
<br>Eth2 一年的能耗将会是 5,256,000 kWh。届时，它所能处理的潜在交易量将会是 788,940,000,000 - 3,153,600,000,000 次。<br>
