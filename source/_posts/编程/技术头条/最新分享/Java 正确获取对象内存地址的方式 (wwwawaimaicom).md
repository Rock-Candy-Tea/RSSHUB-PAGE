在本教程中，我们将了解如何在 Java 中查找对象的内存地址。

在此之前，我们需要说明的是，运行时数据区的内存布局不是 JVM 规范的一部分，完全由实现者自行决定。因此，每种 JVM 可能回用不同的策略布局内存中的对象和数组。

在本教程中，我们主要讨论 HotSpot JVM。
