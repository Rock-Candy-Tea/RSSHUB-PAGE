DNS（Domain Name System）是域名系统的缩写。它是 Internet 的基础之一，用于将域名映射到 IP 地址，就像手机电话簿里将人的名字对应到电话号码一样。当我们在浏览器中输入 Web 地址时，就可以通过 DNS 查询将其转换为关联服务器对应的 IP 地址……
