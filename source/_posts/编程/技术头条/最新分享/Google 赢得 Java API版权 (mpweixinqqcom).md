Google 和 Oracle（甲骨文）在法庭上就 Android 操作系统中 Java 编程语言的使用问题争吵了十年。美国最高法院在周一以 6 比 2 的投票结果裁定，Google 对 Java 代码的使用被归为"合理使用"，没有违反联邦版权法 —— 而非 Oracle 辩称的"纯属剽窃"，而这一裁定也是此案件的最终结果。
